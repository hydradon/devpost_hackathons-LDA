Inspiration

Hey guys! Minotaur was conceived for people who are looking to be more environmentally conscious of their daily power consumption.

What it does

Minotaur is a scalable IoT platform for tracking smart outlet power metrics. Users can group devices together using specific network channels and set consumption limits (in watts) of groups and individual devices.

How we built it

For the backend, we opted to use django for communication with our wqtt API and docker for hassle-free deployment. For the frontend, we used the Vue CLI and BootstrapVue for easy integration of Bootstrap components.

Challenges we ran into
