Inspiration

We wanted to make a light-hearted implementation of an API

What it does

It creates a Pokemon team with the proper move set and nicknames and characteristics of the Pokemon

How we built it

By first creating a prototype, we went from there to continually add components until we completed the website

Challenges we ran into

Challenges with the XML requests and integrating other ideas into the website

Accomplishments that we're proud of

Successfully implementing the API, integrating GitKraken, and creating a complete website

What we learned

API usage and how much XML pains me

What's next for Pokemon Team Generator

More polish with things like average stat comparisons, ability to create a weak or strong team, and custom nicknames
