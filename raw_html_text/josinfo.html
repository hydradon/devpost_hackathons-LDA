Inspiration

in Yogyakarta there is a community named "Info Cegatan Joga (ICJ)" this community has a group on facebook, and user of this group always post matter related facility in Yogyakarta. But, their information is only visible to other user in this group, not addressed to the goverment. I created an app that connects the ICJ community through application that can connect to goverment web sites center.

What it does

to make information from ICJ to goverment

How I built it

I built it using Facebook Login and Facebook Analytic.
