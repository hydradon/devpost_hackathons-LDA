Inspiration

My goal was to make an app, similar to apps like Venmo, using the Stellar network, which takes advantage of decentralized currencies to make money transfers faster and easier. This was inspired by conversations I had with the folks at the CapitalOne and BlockStack tables!

What it does

Users are able to log in with BlockStack. Then, the app asks the user if they want to create a profile for the Stellar Network to exchange currencies, and once they agree, a Stella client generates keys for a test account, and they can simulate real-exchanges with other people.

How I built it

Front-end is written in React using the BlueprintJS framework, with a MongoDB instance holding the public and secret keys encrypted with Google Cloud's Key Management Service and Cloud Functions API.

Challenges I ran into

Learning so much in so little time! Testing out so many different APIs, and realizing some of them don't even work for me was a bit frustrating as well.

Accomplishments that I'm proud of

I'm slowly learning how to put together projects much quicker, even on my own!
