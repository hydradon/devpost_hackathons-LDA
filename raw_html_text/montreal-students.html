Inspiration

We wanted to work with Android Studio and, as students, we wished to make something other students could make use of.

What it does

Montreal Students is an application that lets you log into different University/College portals all in one place.

How I built it

We used Android Studio and Java. Working in a team, we used Git to be able to share our progress.

Challenges I ran into

We ran into a few challenges such as downloading the USB drivers to test the application. Another challenge was having to change projects halfway through and using a new IDE, having never used Android Studio before yesterday and one of us having to learn Git.

Accomplishments that I'm proud of

We have something to present! Having changed ideas so late into the Hackathon, we did not think we would be able to submit a project.

What I learned

We learned how to make quick decisions as a team and divide tasks equally.

What's next for Montreal Students

We would like to implement the login and security password properly. Additionally, we would like to finish developing the application in hope that Universities/Colleges will collaborate with us to bring this project to life.
