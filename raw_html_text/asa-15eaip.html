Inspiration

We were inspired by the fact that we do not see a lot of facial recognition on computers compared to its use on cellular devices.

What it does

It uses your face as a requirement to login your account, which requires pictures of you to also sign up to the page.

How we built it

We created the python code that implements the recognition program and database storage as well as the front end code for the website.

Challenges we ran into

Integrating the front end to back end. The pop-up window for the login posed a problem and we had to get rid of it.

Accomplishments that we are proud of

Learning how to integrate the frontend and backend through thorough research.

What we learned

Sending requests via XMLHttprequest to backend. Using k nearest neighbors in order to perform facial recognition.

What's next for ASA

Acquire experience for future hackathons and other events. Try to implement pop-up windows to front end.
