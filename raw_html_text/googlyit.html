Inspiration 

 There's an API telling us every single numerical point of interest on a human face? Better exercise some special beautification procedure that we have in mind...

What it does 

 Attach Googly eyes on every single human being detected in a picture. IT ALSO DO FACE SWAPS

How we built it 

 Android Studio & Microsoft Cognitive Service API

Challenges I ran into 

 Unpredictable Wifi connection. Android bitmap Memory Leak. Diet Monster not strong enough

Accomplishments that I'm proud of 

 First ever hackathon - DONE WITH STYLE

What I learned 

 We picked up our rusting java skill and also discovered the integration of cloud APIs into android Applications.

What's next for GooglyIT
