Inspiration

Sugar Secure was designed to provide a way for diabetics to securely store their personal data so that it can not be breached

What it does

Sugar Secure uses blockstack to securely store a user's blood glucose data in Gaia Only the user or those they authorize are able to see the data

React + Blockstack
