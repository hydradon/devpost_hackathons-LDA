Inspiration

Many students sell SAT books on the Facebook Groups, but TJay provides an organized website to exchange these resources to benefit the school community.

What it does

Although it's incomplete, TJay is meant to transfer educational books among students and in addition, incorporate a tutoring program.

How we built it

We used a database to store "items"

Challenges we ran into

Being new to using html, javascript, and firebase (firebase being the most difficult), in general, it was difficult to get the code to work without spending lots of time.

Accomplishments that we're proud of

Despite not finishing, it was great to see the database work and being able to store the values.

What we learned

We learned a lot of javascript formatting and code as well as how to use a database.

What's next for TJay

We plan to continue working on TJay and hopefully make use of it in school.
