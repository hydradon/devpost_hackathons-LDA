About

  • Back-end: Developed using WEB API, Code First Entity Framework and SQL Server Azure on database. I decided to use this archtecture because it can be uncoupled from front-end. All business logic is centralized on WEB API layer and all clients (like mobile apps) just call methods and receive json with required information. this archtecture made the solution easy to plug with anything

  • Front-end" A mobile app in Ionic Framework (AngularJS, HTML5, CSS3, Responsive Design) ready to connect with any JSON services.
