Inspiration

As intense workaholics, we're always in the zone and with noise-canceling headphones, it can get difficult to hear when your phone gets a notification. We wanted to make a hack that lets us enjoy our music and relax without having to constantly check our calendar.

What it does

Our hackathon project takes upcoming events on a user's Google Calendar and provides an audio reminder on the Bose device.

How we built it

We built this hack using the Google Calendar API, flask, IBM Watson Text-To-Speech API, and of course, the Bose Soundtouch Audio Notification API.

Challenges we ran into

We ran into numerous challenges as we went about generating the audio files from dynamically generated text and interfacing with the Bose API to serve up files as required by the API endpoints.

Accomplishments that we're proud of

Throughout this hackathon, we learned to dynamically generate audio files using the IBM Watson Text-To-Speech API and utilize the Bose Soundtouch Audio Notification API.

What's next for Bose Remind

Next, we would want to interface with other services like Todoist, Outlook, iCalendar, iStudiez, etc.
