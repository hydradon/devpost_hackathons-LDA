Inspiration

I was inspired to make this app home page after my friends had the idea of the app. So I worked on the side and just created a home page for them.

What it does

Just has a scrolling page website linked to git-hub with pictures, text, and clickable icons.

How I built it

Learned a bit CSS, HTML, and how to handle bootstrap during the hackathon and used the knowledge to create a simplistic static website.

Challenges I ran into

How to connect the html to git-hub.

Accomplishments that I'm proud of

Making a website!

What I learned

I learned a little bit of a lot of areas such as JS, NodeJS, CSS, HTML, bootstrap, git-hub, text-editors like atom, and much more. Also how much fun hackathons really are!

What's next for relink134.github.io

Don't know yet, might just continue to upgrade it or something, maybe learn how to make clickable icons redirect the user to new page, or even make an even more interactive website.
