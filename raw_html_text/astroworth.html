• AstroWorth Front-End Interface

Inspiration

We were inspired to try and find a solution to humanity's dwindling resources, and space fit the bill and the theme.

What it does

AstroWorth uses a WebGL API to simulate the orbital paths and relative challenges of potential asteroid mining sites. It also predicts the profit in mining an asteroid, as well as its mineral composition.

How we built it

We used a combination of Python, R, and Stata in order to run supervised learning algorithms on current data to predict the potential profits in mining asteroids in a novel fashion. We also used Javascript and WebGL in order to build our interactive frontend.

Challenges we ran into

There were some limitations in the currently available data which introduced a challenge in defining competent models.

Accomplishments that we're proud of

We're proud of successfully tying in so many different fields of study in order to build a compelling product.

What we learned

We learned a lot about building frontends and statistics.

What's next for AstroWorth

AstroWorth would greatly benefit from more available data - that is sure to come in the future.
