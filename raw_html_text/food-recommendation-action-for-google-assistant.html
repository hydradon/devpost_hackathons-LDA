Inspiration

We have a Google Home and I love cooking. Therefore, we were thinking of an action on Google Assistant that can recommend what to cook for the day.

What it does

When a user ask: "What should I cook today?", Google Assistant will ask follow-up questions to recommend the user a good dish to match with the user's preferences.

How I built it

We built it using the Diagflow on Google Action and deploy using Google Cloud Platform.

Challenges I ran into

We had to quickly learn the Google Action Console and debug the problems to make the smoothest and most complete conversation.

Accomplishments that I'm proud of

We actually did it in the time limit of less than 24 hours.

What I learned

How to customize an action on Google Assistant!

What's next for Food Recommendation Action for Google Assistant

Google Assistant can add the ingredients to cart for the users after it recommends the food.
