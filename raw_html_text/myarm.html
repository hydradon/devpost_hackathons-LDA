Create a 3D-printed robotic arm which can remotely emulate your own

We used an accelerometer and a touch sensor to map the motion of the wearer's hand to the robotic arm which can be found here( https://www.instructables.com/id/Pocket-Sized-Robot-Arm-meArm-V04/ ) and a touch sensor to close the claw.
