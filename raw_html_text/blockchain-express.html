Inspiration

My frustration with traditional shippers and carriers, controlling everything.

What it does

Blockchain Express is Decentralized Peer to Peer mobile Dapp running on the Ethereum Blockchain connecting senders with carriers with absolutely no intermediaries.

How I built it

Using Truffle, Web3.js, Solidity smart contracts, Testrpc, Angular and the Ionic framework.

Challenges we ran into

Making the UI integrate with the Solidity Smart Contract. Was finally able to do it with the Web3 Javascript API.

Accomplishments that we're proud of

Got the end-to-end functionality running on the Test Blockchain network within 24 hrs.

What's next for Blockchain Express

Automated Refunds and Receipts from Drones and IoT tracking devices.
