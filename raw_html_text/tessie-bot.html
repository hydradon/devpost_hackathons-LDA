Inspiration

The interaction channels between customer and banks are limited. Banking APIs enable us to explore great prospectives for reinventing more convenient ways via mobile social applications such as Telegram.

What it does

Tesse the bot (on Telegram) will help bank customers to check bank accounts and get location-based banking information in an interactive way.

How I built it

Explored data features in Capital One's Nessie API, prototyped using Jupyter notebook, and finally moved the project onto Google Cloud Platform

Challenges I ran into

Some issues using webhook and domain customization

What's next for tessie-bot

  • Voice commands
  • Summary reports and machine learning model based recommendations
