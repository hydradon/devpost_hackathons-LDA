Inspiration

Cost of monthly subscriptions/services

What it does

Uses passwordless authentication to allow you to track and revisit your monthly expenditures online. Passwordless authentication is achieved by only relying on the email and sending a one-time use authentication token to the email each time we need to log-in so that there are no passwords to steal.
