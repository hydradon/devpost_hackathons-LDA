Inspiration

Cipher Tech Inc, enough said.

What it does

It has multi layer unconventional encryption for text.

How we built it

Used a Java password generator we made to get the passwords for the lock file and for the Autodesk action limiter. We also used the image addition and editing tools on Autodesk to code in the locks inside the part file.

Challenges we ran into

Making the password generator actually generate passwords and then figuring out how to actually lock a folder in windows and make it unable to be detected or accessed without the proper lock file password.

Accomplishments that we're proud of

Actually managing to finish a decoders/reverse engineers worst nightmare.

What's next for Encrypt It

Layer by Layer encryption, for EVERY layer like an onion.
