What it does

Designed and implemented an interactive and challenging gaming platform by using dynamic question-answer sessions, resulting in a fair quantification/assessment of one's abilities to search using independent choice of keywords in google.

How we built it

Our team used HTML/CSS for web layout, design and format. Further programming functions and/or UI were implemented using JavaScript.

Challenges we ran into

-Transitioning certain overlaps between HTML/CSS and Javascript

-This was our first hackathon, so we ran into task scheduling challenges.

Accomplishments that we're proud of

We were successfully able to implement planned functionalities by following established timelines and sticking to them, which led to an incremental confidence level and easier implementation levels.

What we learned

-Better knowledge of used frameworks/languages

-A creative aspect of lucrative technology and its importance in setting trends

-Teamwork and effective time-management strategies

What's next for QueryBee

QueryBee 2.0 which will allow for feedback using data analysis and big data initiatives which will help us further improve our efforts to making this project addictive to general masses.
