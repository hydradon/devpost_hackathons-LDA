Inspiration

Nowadays, the stock market is quite fluctuating and people lose money. With the big data analysis techniques, we believe we can predict the future trend of the stocks based on news.

What it does

It's an iOS app that provides the best guidance on stocks. By analyzing the recent news and current stock tend, we aim to predict the future.

How we built it

  1. The mobile end is developing on iOS with Swift.
  2. The backend is deployed on Amazon EC2 and the database is running on Amazon RDS.
  3. A script is running daily to search the recent news using Microsoft Bing Search API and then passed into Alchemy API to run sentimental analysis.
  4. After that, the we cross reference the sentimental results with the current stock trends obtained by Yahoo YQL API.

Challenges we ran into
