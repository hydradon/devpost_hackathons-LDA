• Serenity.ai - All in one Event Management and Networking System

Inspiration

The lack of an a wholesome experience for event organisers where not only can they post about events to gather audience, but also understand the prospects through the dashboard presented using big data analysis.

What it does

It is a community building system, where we can come together to analyse and network with the available data on the events, sentimental analysis of the events, to better the events.

How we built it

We built a dashboard using powerBI, chatbot for interaction followed by a UI/UX application and webpage.

Challenges we ran into

Accomplishments that we're proud of

Integration of a full stack system, including the front end as well as the back end work, to give a wholesome experience and better idea of our project.

What we learned

We learned how to connect better as a team, and stitch the whole project together after the different components had been implemented.
