Inspiration

This game was largely inspired by Pong.

What it does

The objective of the game is to bounce the ball on the spinning rods into the high-lighted wall. Every 5 bounces the difficulty increases and the color scheme changes.

How I built it

I built it using the CoronaSDK

Challenges I ran into

The particular object creation method I was using was causing some memory leaks.

Accomplishments that I'm proud of

Publishing a game on the Google Play Store and IOS store.

What I learned

I learned how to make a hybrid mobile game.

What's next for Rodball

If people are receptive, I might add an additional mode to the game, and perhaps even build a continuation with further improvements.
