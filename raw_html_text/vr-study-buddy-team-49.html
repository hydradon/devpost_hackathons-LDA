We built an Alexa skill that can dynamically take in any deck from Quizlet and help users study using Python and Google Cloud. We combined that Alexa skill with a 360 degree photo of a classroom inside an Oculus Rift to help users immerse themselves in the environment they would take a test in, which is stated to help increase memory recall through context-dependent memory ( https://en.wikipedia.org/wiki/Context-dependent_memory ).
