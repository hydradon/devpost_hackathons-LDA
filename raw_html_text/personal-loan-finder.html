Inspiration

When making a huge purchase, people may need a loan. We are trying to help those people find the best loan offer.

What it does

It is a google chrome extension to help you explore the payment options of personal loans when you are trying to make a big purchase.

How we built it

Chrome extension;HTML;JavaScript

Challenges we ran into

Many chrome extension problems because it is our first time to build it.

Accomplishments that we're proud of

It's a smooth chrome extension that can detect when people may need a loan and present the offer to them.

What we learned

How to build a chrome extension

What's next for Personal Loan Finder

We can train a model to make the detection more accurate.

We can add personal behavior to a database to analyze them.

We can optimize UI and UX.
