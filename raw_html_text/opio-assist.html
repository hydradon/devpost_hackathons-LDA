Inspiration

The inspiration for Opio-Assist came when I read about soldiers in the Vietnam war getting addicted to Heroin and then returning to America and <1% of the soldiers remained addicted. Environmental triggers were the factor at work and those weren't present in the US

What it does

Opio-Assist has three main features: Inform - This serves as a database of all the information that a recovering addict would need to continue on the path to recovery Community - this is a public forum and chat room that enables users to chat with each other and supports private messaging for all involved Alerts - this screen lets users make location based alerts to warn them when they approach an area that could be triggering.

How I built it

I built Opio-Assist as a three part iOS app to get the main features in place without compromising on any useful necessities. I built the app almost entirely in Swift, making use of the Storyboarding features and also several objective C libraries

Challenges I ran into

Some challenges I faced included finding an effective way to integrate alerts in the app. At first I planned to have location and also time based alarms, but then I remembered that alarms are a feature that every phone has anyways so I axed that and stuck with location. This took several implementations but I finally found an effective way to display alerts and let users add new ones.

Accomplishments that I'm proud of

I'm proud that I built the whole thing in only 12 days by myself. With more time I would have been able to make something far more feature rich but given the time span I think that this is a very feature rich app with the important pieces that a recovering addict would need to carry forward.

What I learned

I learned how to make map based features in Swift that work effectively and provide new features including the rings and annotations.

What's next for Opio-Assist

Opio-Assist will be live on the App Store soon and will be followed up with updates, links to get new users onboard and promotions for encouraging adoption and more use for those who really need it.

How the insights are addressed

Insight 1: Insight 1 is covered by the “Side Effects” and “Medication” sections of the inform tab. Both include links to the various sites with specific information.

Insight 2: The “Parenting”, “Family”, “Relationships” and a few links in “Side Effects” and “Medications”. These also include links and a few worksheets

Insight 3: “Relapse” and “Relationships” sections

Insight 4: “Relapse”, “Doctors” and the geo-notifications feature that users can take advantage of to have their phones alert them if they go too close to an area that may be triggering for opiod use.

In addition to informational links several of these sections include links to find local meetings of AA, NA and other support groups On top of that, Opio-Assist includes a live forum/chat room that includes topics for all of these concerns so that in addition to users being able to find information on each of these topics they can also chat with each other to share tips, provide support and encouragement and know they are not alone. Making friends in their own position is vital to finding recovery and avoiding relapse.

Insight 5: Users are encouraged to make use of the calendar and alarm features of their phones to handle their schedules. Replicating these features seems unnecessary.
