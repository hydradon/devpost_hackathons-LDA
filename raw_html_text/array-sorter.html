Inspiration

In CS we learned about sorting algorithms but the way that it was explained to use made it unclear. We decided to try and remedy this by creating an app to teach different sorting algorithms.

What it does

It demonstrates sorting algorithms.

How we built it

We brainstormed the idea beforehand and used Eclipse and the Java Swing library.

Challenges we ran into

One of our members had a lot of difficulty with the WiFi, so we did most of it offline. We had many ideas we wanted to do, but we knew that we had to stick to goals that were achievable within 24 hours. We are also Foundations Computer Science (a beginner CS course) students, so we only knew basic Java.

Accomplishments that we're proud of

We got it done in time, which is pretty impressive. We also learned many new things.

What we learned

  • How to use Java’s Drag and Drop
  • More, different ways of sorting arrays
  • Hackathons are fun :)

What's next for Array Sorter

  • Improving the UI
  • Implement more sorting algorithms, such as radix sort, quick sort, etc.
  • Make the visuals clearer and easier to understand what is happening
