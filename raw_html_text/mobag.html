Inspiration

Everybody loves to travel, but nobody likes to carry luggage.

What it does

Follows you Guides you Alerts you if your bag is out of range Communicates to you via bluetooth, cloud messaging etc

How we built it

Using Native android, Arduino Module

Challenges we ran into

Cracking arduino and app communication seamlessly

Accomplishments that we're proud of

  • Seamless communication (2 way) between smart bag and android app
  • Ultrasonic sensor detection and data communication
  • Push notification through arduino

What we learned

  • Arduino 2 way communication with app
  • GPS module with arduino

What's next for MoBag

  • Plan big with bigger features and challenges
