Website on Maternal and new born child healthcare Motivation: Our website has been developed to provide essential information on obstetrics and new born.It wiil also provide information about various tests that a woman may undergo before she conceives and during her pregnancy.Moreover the user will be able to find nearby test centres and hospitals.

Our basic idea is to create a local host server in xampp and store the data entered in phpmyadmin. The front end has been developed using html,css and javascript, back end is done using php.

Future Scope:We have planned to build a mechanism that would allow her to directly book an appointment.
