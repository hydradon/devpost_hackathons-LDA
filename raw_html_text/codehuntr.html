Inspiration

Finding the right projects to contribute to is a pain. Finding the best developers for your project is nigh impossible. CodeHuntr works to connect the right developers to the right open source projects.

What it does

CodeHuntr is a fun way for developers to connect with interesting projects and for project owners to connect with the right developers. Swipe right to like or left to pass on the developers or projects CodeHuntr recommends. If someone likes you back, it’s a match!

How I built it

The client side user interface for the Android app was build using Android Studio. The server side was developed on a Firebase database hosted on Google Cloud. The ranking of developers and projects was determined using predictive analytics of Azure machine learning.

Challenges I ran into

There was difficulty deciding on the right APIs to use. There was an issue where one of the applications of the API was not functioning and help of the mentors that supported the API was needed.
