Inspiration

  • Students lead busy lives balancing academics, extracurriculars, and more
  • Negative Feedback Loop
    • Trying to keep track of everything = stress
    • Stress = memory loss
    • Forgetting an important event = more stress

What it does

  • Overview
    • User can keep track of upcoming events in one place
    • Organizes events into separate categories for user convenience
    • Ultimately builds student’s organization skills
  • Home Tab
    • Mental health check-in
    • Quote catered to user response

How we built it

  • IDE: Xcode
  • Language: Swift

Challenges we ran into

  • Difficulties
    • First hackathon
    • Working with unfamiliar almost new language
    • Formatting issues

Accomplishments that we're proud of

  • Making a beautifully crafted and aesthetically pleasing app in such a small time period
  • Create an application on XCode that has potential to be on the App Store and available for millions of users

What we learned

  • With hard work and determination, and entire app can be created in eight hours
  • Creating a successful application not only requires making the application, but also has business and entrepreneurship aspects of pitching the app and its idea

What's next for Organize It!

    • School Loop
  • Calendar feature
  • Social Networking
    • Profile tab
    • Make friends, allow chatting capabilities
  • Motivational features
    • Levels, points, badges
    • Encourage student to continue setting and meeting tasks/goals
