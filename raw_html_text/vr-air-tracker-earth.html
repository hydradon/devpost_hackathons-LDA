Inspiration

Eli wanting a simple and intuitive way to view flights his parents were on.

What it does

So far, we are able to view a model of the earth, rotate it, and scale it up or down to act as a zoom.

How we built it

We acquired a computer with the needed GPU and USB3.0 ports then used Unity and C# to create the Earth Model and user control

Challenges we ran into

The first two computers we tried did not work and attempting to wrap a 2D coordinate plane to a 3D sphere.

Accomplishments that we're proud of

We have a working model of the control part of the program.

What we learned

Manage our time better and think more ahead about the required components.

What's next for VR Air Tracker Earth

Hopefully we can continue expanding it and make it work in the future.
