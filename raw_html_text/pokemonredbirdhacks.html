Inspiration

20 year anniversary celebration dedicated toward this spectacular game. We wanted to challenge ourselves to see if we can learn a new language as well as create a new application comprising a team with different sets of skills.

What it does

This app allows users to open up Apple maps and simulate encountering and catching pokemon. Users have different choices as to whether they want to try their luck on catching, attempt to lure with bait, aggravate or flee.

How we built it

We built the app using Swift in Xcode designed for iOS9 beta.

Challenges we ran into

The biggest challenges were populating the pokedex by passing parameters between different view controllers and having views switch based on different actions that the AI performed.

Accomplishments that we're proud of

Animations, staying motivated the whole time, team bonding, and creating a functional game in an entirely new language!

What we learned

If we can imagine a function, sprite, animation, or design; we CAN implement it. Nothing is impossible.

What's next for PokemonRedbirdHacks

Developing multiplayer trading, battling system, and avoid being sued by GameFreak/Nintendo!
