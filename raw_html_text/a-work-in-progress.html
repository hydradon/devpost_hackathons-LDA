Inspiration

Being in college and dealing with the challenge of writing a paper.

What it does

A side-scrolling game where you avoid obstacles.

How we built it

We used Unity in conjunction with C# in order to get all of the functionality and we used Photoshop for all of the designing except for the main character.

Challenges we ran into

We had to change game engines more than 12 hours in. Originally we were using Java and libgdx, but we figured out that using Java was not the best or fastest way to accomplish our game and decided to switch to Unity in hopes of having a better game.

Accomplishments that we're proud of

We got it to work in less than 12 hours. Our graphics are of pretty good quality.

What we learned

Unity C# A little bit about how to use github A small amount of libgdx and how Java can be used for game development

What's next for College Pains

More levels--our original plan involved levels for every year of college but due to time constraints we were only able to do Freshman year. Better animation/different main character
