Inspiration

  • Build the next generation mobile home buying experience using technologies like Alexa, AR/VR or Machine Learning and you can win up to four Alexa devices and a visit to our office to meet the tech leadership team.

What it does

  • It allows the home buyer to tour the house in a VR environment, so they can be in the comfort of their own home
  • Instead of hover at the object
  • The buyer can also drag and drop the furniture to review how it looks, which they can adjust the furniture's positions.
  • Future consideration: save layouts to allow the buyer to compare the different layouts and houses

How I built it

  • Our team built it with unity 3D, C#, and Oculus

Challenges I ran into

  • We try to do mobile but none of our computer support Oculus software. Therefore, we have to install Windows 10 for that.
  • We originally tried to install Windows 10 it in a virtual machine, but failed because my laptop has integrated graphics card that cannot integrate with the virtual machine.
  • The Oculus's hardware is not compatible with our computer's GPU
  • first time using unity software and VR, so there's a huge learning curve

Accomplishments that I'm proud of

  • Despite of all those problems, we still find a way to overcome these obstacles and make it work
