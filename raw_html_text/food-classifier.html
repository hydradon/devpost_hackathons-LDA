We decided to do this project because we had the common problem of consistently encountering foods and meals that we had no idea what they were called. So the idea sprung into our heads to create a web application that completed this task for us, by identifying food based on images and providing users with macro information regarding those foods. The biggest problem we encountered when developing our application was the difficulties we experienced in combining front-end and back-end implementations, as well as using the Google Cloud Vision API.
