We wanted to challenge ourselves by attempting to build a complete NES emulator. We read machine code from rom, interpret it, emulate it on a virtual processor, and read and write to virtual ram. Unfortunately, we didn't have enough time to get the ppu to work, so we can't see graphics. We learned a lot.
