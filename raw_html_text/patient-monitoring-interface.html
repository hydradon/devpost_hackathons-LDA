In Hospitals we see doctors not provide well attention to patients, due to many of patients are died, or paralyzed.So in order to help patients we developed application called patient monitoring interface which help patient as well as doctors.

What it does

Patient Monitoring interface is a Full-Pledge Hospital App,This app Helps the Doctor to Monitor the Patient effectively, we especially designed this for ICU patients,because they need more attention from doctor, we use different e-health sensors which get different readings from patient and broadcast that readings on doctor android phone in graphical form, its also generate notifications alerts about health of patient on priority basis.

How i Build it

we used different sensors along with arduino device, to get readings, and we create C# Application for hospital, and designed android app for doctor to get interact with his patients..

What's next for Patient Monitoring Interface

Currently we used four sensor(Heart Rate, SPO2 oxygen,pulse rate, blood pressure) to monitor patient health but we plan to use 12 sensor in future, which create our app an full pledge Hospital app.
