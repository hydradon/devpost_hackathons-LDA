Value Proposition

Our purpose is to create a society where all people are safety change-makers, not only safety consumers. By disruptive innovation and refactoring citizens' collaboration, we can extend the quality and capacity of emergency services in order to bring the safety level for a new level!

What it does

Our PoC aims at quickly building a solution that is focused on public & individual safety.

Challenges We ran into

Our team was challenged to start everything from scratch (Ideation - Design - Development) into a super early PoC.

Accomplishments that We would proud of

We would be proud of our app if that could be tested in the real environment and saved someone's life...

What we learned

How to start a project from Zero to Hero in 27 hours.

What's next for SafetyPie

A lot of improvement should be done, but if the idea would be validated by people and jury, we will do the following:

-> Test in the real environment + reduce all hard coded data :)

-> It definitely needs deep market research in order to bring a clear value proposition for the customers.

-> Add features as localization sharing, develop the full voice assistant for disabled people and the elderly.

-> Add TomTom GeoFencing API

-> Add messaging feature

-> Add real-time data regarding crime or danger situations.

-> Develop a WeChat mini program of the app

-> Build a community around the app (groups/articles/safety advice)
