We programmed the delta robot to play a human in Tic Tac Toe. The user inputs the space they want to move to, and the Tic Tac Toe algorithm responds with the most optimal possible move. We built it using arduino and python. While doing so, We ran into a few challenges which causes minor numerical errors which we were able to fix. We are really proud that the robot works and we've learned how to make these technologies work together really well. In the future, Delta robot will operate off image processing and play autonomously with the user input being a regular pen and paper game versus the computer input. It will feel just like playing a friend!
