We were inspired after seeing fitness enthusiasts in NUS not being able to work out to their fullest potential because they do not have a spotter. Therefore, we made a Gym Buddies Telegram Bot @NUSGymBot, which allows NUS Students to schedule gym sessions to go to NUS gyms together.

Our telegram bot aims to tackle this issue by allowing fitness lovers in NUS to connect with each other and work out together. Our target group for this bot is NUS Students as we believe it is a bot that has a niche place in the fitness community within NUS. No longer will you have to gym alone because of your hectic timetable, and no longer will you be afraid to get that one more repetition in fear of the weight crushing you!

Functionality:

  • Allows the user to see which other users are going to which gym at what time
  • Users are able to directly contact them as the telegram handle is provided and directly linked
  • Users are able to insert and remove themselves into the various gym locations and time slots
  • Holds a constantly updated SQL database to keep track of users and timeslot

Benefits:

  • Allows us to connect NUS students with the same passion for fitness
  • Increases safety for student gym-goers by allowing their newfound Gym Buddies to spot them and vice versa

This was built using Python and SQL, specifically the Telegram Bot API in Python. Our user database is hosted using MySQL. We managed to build the bot within the stipulated time. As it is our first hackathon for our entire team of 3, accomplishing this feat means a lot to us.

We intend to host NUS Gym Buddies on a cloud server and add additional features such as providing users with workouts so as to reduce the activation energy required for users to head to the gym more regularly.
