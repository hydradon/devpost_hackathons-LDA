Inspiration

An experiment to create a wordpress-like approach in bitcoin space.

What it does

Has a bitcoin processing engine, with pre-installet apps and some integration options. Everyone one can request an app and offer a bounty. Everyone can contribute and get paid. Softtware itself helps beginners to start with bitcoin-driven app development

What's next for pay2exec

Looking for contributors, so library of apps can grow over time!
