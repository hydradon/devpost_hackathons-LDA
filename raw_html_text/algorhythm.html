Second team member: Alex Wiley, BCU

Inspiration

Many parties with music not matching the mood.

What it does

Measure the liveliness of a room and generate music genres (and soon music playlists) from this data.

How I built it

Using Java for the initial version along with a custom fork of a webcam library, modified to detect levels of motion in a frame.

Challenges I ran into

Starting the hack with only 12 hours to go, and as a solo project. Before luckily gaining a second team member in the latter stages.

Accomplishments that I'm proud of

Writing the computer vision code without the use of openCV.

What I learned

Never convert to web based in the last 4 hours.

What's next for Algorhythm

Complete the web based version with full spotify api integration.
