Inspiration

The game Make It Rain was a big influence on this idle game. In fact, the thought stemmed from Make It Rain, except you through change.

What it does

It's an idle game where the goal is to make it hail! Collect all the change you can and compound as you go.

How I built it

Using Unity3D and all the game creation made easy features that it has to offer.

Challenges I ran into

Creating all the art and music for the game in a short window of time. The art definitely turned out interestingly.

Accomplishments that I'm proud of

Completing a full WebGL game in a short period of time! :D

What I learned

This is less so what I learned and more so what I re-learned: I love making video games, and I should spend more time making small games again.

What's next for Change Chunker

Potentially clean up some things and add more features. I definitely want to convince an artist friend of mine to maybe help me clean up the art on this game. Potentially an app version may be in the works in the future.
