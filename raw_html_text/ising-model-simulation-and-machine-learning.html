The Ising Model

The Ising Model was the first successful model of ferromagnetism and is a widely studied model in statistical mechanics.

Our Project

We simulate this intricate model with visual aids to help users learn about the model and it's particularities. We also train an AI that learns to classify ordered (ferromagnetic) from disordered (paramagnetic) states.
