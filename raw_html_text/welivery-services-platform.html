Inspiration

I got this idea from my previous boss, he was running a Chinese restaurant before. The critical problem he facing is unable to delivery food to the customer on time. He hope there would be a platform can help them to arrange the order, also include delivery food.

What it does

Cooking Assistant Arrange Order by arrival / pick up time Combine Order by common dish

Food Delivery C2C Food Delivery (Uber like) Delivery Route Optimize

How I built it

Cloud Services Microsoft Azure - Mobile App (App Back-end) Microsoft Azure - Azure SQL (Database) Microsoft Azure - Mobile Center (App Analysis / Crashes tracking / Distribution) Microsoft Azure - Notification Hubs (Cross-platform Notification Push) Microsoft Visual Studio Online (Continuous integration / Source Code Management / Bug Track)

Software Platform Xamarin Forms (Cross-platform Development Framework)

Challenges I ran into

The Authentication Processing (a small bug of Microsoft Mobile App Client) User Interface Design

Accomplishments that I'm proud of

Automatically Development System Implementation, include Azure configuration, SSL certification, CI and Mobile Center Crashes tracking.

What I learned

More deep understand of Xamarin Forms, and Azure functionality

What's next for Welivery Services Platform

Implementation of Payment system and detail delivery function. For long term development, the machine learning would be implemented, to improve the order arrangement and to provide bulk fresh indigenous to restaurant. Yep, supply chain. Also, Big data analysis would be implement after around half year, to do some basic predict of sales.
