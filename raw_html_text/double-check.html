Inspiration

My Dad had trouble reading a prescription container. So Thinking of him I built this app

What it does

Scan any prescription bottle and return the text from the picture. With the help of Microsoft Vision API , we were able to translate the bottle to a different language if need be.

How we built it

We used Angular 2 and Ionic 2 to build our hybrid framework application.

Challenges we ran into

A hybrid framework like Ionic limits us from creating a more native application and it's feature.

Accomplishments that we're proud of

It works

What we learned

Microsoft Vision API, CORS, OCR
