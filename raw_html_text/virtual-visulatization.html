Inspiration

Understanding data can be a difficult task. Data is not always intuitive to look at and must be simplified in order to view on traditional graphs. This can lead to a misunderstanding of the data, or make it much harder to understand how to data is organized.

What it does

Virtual Visualizer allows you to view data using virtual reality technology. We have created data sets that allow users to move around in a 3d environment and inspect data from all sides.

How I built it

We built this tool using Unity and It's Oculus Rift support. We also utilized data from Kaggle to view some interesting data sets that were verbose and have real-world applications.

Challenges I ran into

We are fairly new to Unity and C#, with no experience developing for the Oculus rift. Learning new technologies is an exciting but sometimes frustrating adventure, but we were able to overcome the challenges we ran into and develop all of the features we wanted.

What's next for Virtual Visualizer

Creating new features that allow the user to even more intuitively and with better UI/UX. It would also be good to integrate tools to add in your own data set for analysis to allow it to be a full-blown tool for users to utilize.
