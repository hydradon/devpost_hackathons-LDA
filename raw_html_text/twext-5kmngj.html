Inspiration

We barely used API's in our projects. Being high-schoolers, we tried something completely new - first with the Alexa, then using a bunch of different API's in a flask web server - and finally using the Twilio API to perform twitter functions.

What it does

Our hack can search for tweets by texting "search {item}" to the number: (201) 257-5564

How we built it

We used the Twilio API & Twitter's API

Challenges we ran into

Trying to host the node.js server and integrating the services.

Accomplishments that we're proud of

Connecting two awesome services together to make something useful.

What we learned

How to use API's, so much more JavaScript...+ node.js, express servers.

What's next for Twext

We are integrating so many more features so that you can access twitter from just SMS!
