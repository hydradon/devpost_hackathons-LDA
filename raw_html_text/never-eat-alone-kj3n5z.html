Inspiration

So Valentine's day hurts us a lot. Being single is hard, especially you have to eat alone sometimes. So Never Eat Alone comes so that nobody will eat alone. This platform helps you find your friends who are also planning to go to the dining halls at the same time periods you are. This way Never Eat Alone saves you the hassle of asking your friends one by one and see who is free to grab a quick meal.

What it does

You pick and choose your dining hall and time preferences, and then it connects you with other friends who are also down for the same

How we built it

The platform we decided was a website, therefore we used mostly tools for web-development such as css, html, bootstrap. The most interesting we used today is that we learned how to implement Firebase to manage our user accounts.

Challenges we ran into

  • Centering the form tag into HTML
  • How to implement user sign in
  • Using Firebase (made it fully functional!)

Accomplishments that we're proud of and what we learned

  • Learning to use Firebase from scratch
  • Learning HTML and CSS from scratch
  • Signed up and used Github for the first time
  • Connecting lonely Bruins

What's next for Never Eat Alone

  • Create an internal messaging app
  • Link to other social media platforms
  • The info page of the dining halls will show the menu and the hours of operation
  • Mobile app
  • Make sure we or other Bruins never eat alone
