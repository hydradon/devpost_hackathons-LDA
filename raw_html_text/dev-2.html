Inspiration

We were inspired by the problem that professionals in our field don't see any value in government recruitment platforms, so we decided to find a way to create value for them.

What it does

DevDev is a platform designed to accelerate developers' professional development. DevDev offers developers:

  • An easy way to keep track of their skills
  • Suggestions on what skills to learn to reach career goals
  • Information about industry trends
  • Suggestions on where to learn required skills

Where's the money?

Developers are highly sought after in the job market, and we'll be essentially gathering a large database of developer skill profiles. This data can be used to connect people with jobs.

How we built it

We started with tons of ideation on a white board with sticky notes. After we nailed down the concept we went though a product design cycle and started sketching page structure on paper. We agreed on general structure of the page and then started building the page in Sketch. Sketch files were imported to Invision for a prototype and presentation.

How do we get the data?

We harvest the developers sites including Github, Linkedin and Stack-Overflow to get data about skills. We use recommendation system techniques to suggest the user skills he may know or may want to learn. Recommendation system techniques such as content based algorithm can be used to suggest the skills to learn.

We also use clustering techniques to suggest the role of the user based on his skills. For example, if the user knows AngularJs, ReactJs, and nodeJs than he/she maybe a full-stack developer. Or we suggest the user if he/she learn the particular skill then his role may change to other role. Role changes?

Once we find his/her interests we suggest the online course (MOOCs) or open source projects to learn the particular skill.

How do you find the courses related the skill?

Maybe crawl the MOOC websites and check whether the particular skill is mentioned as required skill in that particular course.

Challenges we ran into

It was really hard to come up with a concept that seemed feasible and was still a fresh idea.

Accomplishments that we're proud of

We were able to come up with a good idea that could work in many other industries too.

What we learned

A lot about the current recruitment ecosystem. It was also fun to see other designers' process.
