Inspiration

GrowNYC

What it does

KYF enables GrowNYC and their customers to track and trace the source of produce.

How we built it

We built it using Hyperledger Fabric, Hyperledger Composer and Chainpoint. At each checkpoint through the supplychain we write a Chainpoint UUID to Hyperledger. These checkpoints can be visualized on a Mobile device with a Google Map Integration.

Challenges we ran into

Modeling the API, connecting systems.

Accomplishments that we're proud of

Linking together the different hash Ids from Chainpoint and Hyperledger.

What we learned

We can combine permissioned and public blockchains using hashes and txn Ids to secure data.
