Inspiration

The inspiration to work with a completely new framework which has little support online and create an amazing app which can help the society

What it does

It uses Apple's Researchkit Framework to collect data from various sensors like accelerometer,Tapping ,Spatial Memory and Heart rate. The data is then visualized using ORKCharts to provide stunning data visualization.
