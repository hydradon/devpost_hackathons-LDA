This October weekend at SD Hacks 2017, our team put our brains together to build an innovative web app. Allow us to introduce NutriLink, the future of nutritional DNA sequence analysis. Our web app utilizes the Genome Link API along with Edamam’s recipe database API to create a customized nutritional profile that suggests healthy and delicious recipes tailored specifically to the user based on the characteristics of their sequenced genome.
