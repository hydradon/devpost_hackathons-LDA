Inspiration

  • Travel is expensive. But with timely information, one can make significant saving.

What it does

  • Helps people fly to popular travel destinations in US for holidays on budget.

How we built it

  • Used Expedia Flight API along with Google Map API to find the cheapest flights to various destinations in US, and mapped them.

Challenges we ran into

  • Took us some time to place multiple markers with label on Google map widget.

Accomplishments that we're proud of

  • Completed the project on time - Yay!
  • This (tool?) is actually helpful. We'll use it ourselves :)

What we learned

  • Expedia API, Google Map API etc.

What's next for Voyager

  • Supporting countries other than US.
  • Finding the cheapest dates to fly to a given location.
  • Nice user interface
