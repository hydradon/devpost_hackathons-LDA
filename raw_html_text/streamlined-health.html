Inspiration

We realized it was hard to get access to online doctors from the patients perspective. Also we realized that many doctors that are not in big companies like Sutter often get shut down.

What it does

It allows the Patient to get into contact with a list of doctors that are currently online and have a chat with them through email. It also allows the patient to book an appointment online. Also it shows Doctors alist of patients that they got requested.

How we built it

We used html and GitHub pages.

Challenges we ran into

unfamialriy with incorporating javascript with html files.

Accomplishments that we're proud of

We were able to finish the website with limited information on web design.

What we learned

We learned that other options are available for creating websites other than HTML, such as java applet.

What's next for Streamlined Health

We will expand the features to include greater access and bigger storage by using servers.
