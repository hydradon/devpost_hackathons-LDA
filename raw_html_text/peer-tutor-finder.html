Inspiration

We were inspired by the peer tutoring program of our school, TJ. It provides students with the opportunity to get help from peers that have gone through the same process as them.

What it does

It allows people to connect and learn from each other. It allows one to develop knowledge in anything that they are struggling with, or just to help one with time management issues. Working with a peer can motivate one to work harder and more efficiently. It also saves a lot of money because of how expensive tutoring services are today. Peer tutoring gives you a free, life-lasting relationship with a peer as well as allows you to improve your work ethic and lifestyle.

How we built it

We used c9 to share our code between ourselves and used css, html, and js to develop the front-end of the site.

Challenges we ran into

We faced the challenge of figuring out how to input surveys into our site and creating an interactive user friendly site.

Accomplishments that we're proud of

Creating a comprehensive and cohesive site that is easy to use.

What we learned

We learned how to implement Google maps API into a html website

What's next for Peer Tutor Finder

We want to use Big Parser to find locations that are near both peers in order to find a prime place for meeting.
