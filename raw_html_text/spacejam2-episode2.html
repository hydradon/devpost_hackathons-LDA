Inspiration

based on starblast.io and the old arcade game asteroids.

What it does

Flight simulator where you can shoot asteroids to gain materials to upgrade your ship.

How we built it

Using unity2d, and scripting GameObject behavior with C# script files.

Challenges we ran into

Syncing up our versions of the project, and being able to fix important features such as movement.

Accomplishments that we're proud of

Creating a functional game and accurate zero gravity simulator

What we learned

How to create games in Unity, and how to code C# scripts.

What's next for SpaceJam2 - Episode2

We'll use this knowledge going forward to make more games. This was more of a test of our skills.
