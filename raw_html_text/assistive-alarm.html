Inspiration

For my capstone project, I am working with a physically handicap man to create wheelchair accessories. He mentioned he needed a way to call for help in the middle of the night. The goal would be to connect this to the existing Amazon Echo in his home so he could call for help.

What it does

wirelessly controlled alarm working over wifi network

How I built it

Wired up circuit to arduino/particle

Challenges I ran into

I'm not very skilled in software so programming has been a challenge. Also, it was hard to upload code to the particle over the very crowded wifi network.

Accomplishments that I'm proud of

Got the alarm circuit to work, got particle to work.

What I learned

First time using the particle photon, learned all about it.

What's next for Assistive Alarm

Next for Assistive Alarm is to get the device user friendly and connected to the homes of physically handicap.
