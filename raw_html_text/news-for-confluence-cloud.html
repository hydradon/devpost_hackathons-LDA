• Engaging News page

  • One click to feature a blogpost

  • Restrict who can feature blogposts

Communication is an important part of collaboration and when business teams want to use a platform like Confluence Cloud they need access to intranet-like features that can help them do that.

This add-on creates an engaging News page that displays blogposts in a visual layout, highlighting any images that have been included in the source post. Blogpost authors don't need to do anything for this add-on to work, except to flag the blogpost as being featured - the add-on does the rest of the work including trimming titles and excerpts (if there isn't an image to use).

To help larger teams or companies using Confluence control which blogspots appear on the News page, the ability to feature blogposts can be restricted to a set of users.

News for Confluence Cloud is a relatively simple add-on, but there is immense scope to expand and extend the initial feature set.

This add-on will be published on the Atlassian Marketplace as a free beta release. If there is enough interest, I will consider adding more features and turning it into a commercially supported add-on.
