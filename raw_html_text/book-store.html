Inspiration

In school days I happen to many books each year. This website is to help students get books for free from other students.

What it does

Buy and add books on website

How we built it

The tech stack includes HTML, CSS and JS. We have used react library. IndexedDB is also used in the project.

Challenges we ran into

Accomplishments that we're proud of

  • Collaboration

What we learned

Learned and implemented some concepts of react.

What's next for Book store

It is still incomplete and has a lot to complete. Some of the things are responsiveness, authentication and some of the functionality in buying and adding books.
