CasualConvo

Chat with a girl in virtual reality in an out doors setting in the comfort of your living room!

This project features Google Cloud API calls to translate speech to text and text to speech in order to facilitate real time feedback from the Cleverbot chat API. Cloud API calls are done within an express application with a locally run server, which then formulates mp3 files based on the bot responses. These are decoded within Unity before Unity records user input and sends that back to the express server for speech-to-text operations. The VR system used is the Oculus Rift.
