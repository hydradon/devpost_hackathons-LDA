What it does

Removes the entry-barrier to the blockchain for the layman user. Ethernal allows you to write eternal messages on the blockchain.

How it is built?

A smart contract on the Ethereum chain is holding all the messages. Anyone can send a transaction to add an additional message.

What's next for Ethernal

Add a credit card payment system to avoid the Metamask step.
