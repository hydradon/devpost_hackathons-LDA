• Speakr - Learn to Talk

WEBSITE: https://s3-us-west-1.amazonaws.com/speakr-app/speakr.html

Inspiration

  • Our group was formed through mutual friendships. We really did not know what to expect because of our lack of experience at hackathons. However, through brainstorming based on the available api's we thought of a unique idea that can help many students and practacing speakers ## What it does
    • gives live feedback on your speech and facial emotions to help improve speaking habits

How we built it

  • We created a server that handles all the machine learning which the front end(website) is hooked up to. The front end has access to the microphone and webcam

Challenges we ran into

  • Some challenges we ran into is running the server in an effecient manner so that server responses occured in a fast manner
