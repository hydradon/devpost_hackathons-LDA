Inspiration

I always stained my clothes and never knew how to clean it. I always had to text my parents how to clean them. Now, with our web app, there is a convenient location that holds all of the information about doing laundry and cleaning stains.

What it does

This web app efficiently displays every stain and every clothing type and each way to launder them.

How we built it

We used a bootstrap template called "Freelance" to easily design our website. Afterwards, we gathered all the information related laundry and stains. Afterwards we attempted to code a log-in system that would track each piece of clothing using javascript.

Challenges we ran into

Firebase was difficult to learn and use in the span of one day. Thus, making the database hold all of the timing data for each user was not feasible.

Accomplishments that we're proud of

We are proud of our website design.

What we learned

We learned that database coding is difficult. It is very difficult to edit a template's code. We learned how to present in an effective manner.

What's next for Cleaning Central

Our next goals will be finishing the tracking feature and constantly updating our web app with hot new features and information.
