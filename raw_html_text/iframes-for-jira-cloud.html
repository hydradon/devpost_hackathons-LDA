iFrames for JIRA is a tool that makes it easy for any user to safely embed external data into JIRA Dashboards.

It was inspired by a community member that asked for help building something like this. After figuring out how to solve their problem, I wanted to make an add-on that made this easier for non technical users.

This add-on provides a Gadget which users can embed into Dashboards. This gadget accepts a link to an external resource and then displays it inline.

This was built using ACE with Node and deployed to Heroku.
