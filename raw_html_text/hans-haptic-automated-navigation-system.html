Inspiration

Challenges in the field of indoor navigation, as well as the need for a smarter way for the visually impaired to navigate the modern world.

What it does

Internal navigation - it uses an accelerometer to bridge the gaps between imprecise GPS data. It tries to predict the position of the user where GPS does not provide sufficiently accurate location data.

How we built it

We used the maps API to locate the user.The Arduino driven system uses the sensor data to help in navigation

Challenges we ran into

Integrating the hardware and the maps syncing

Accomplishments that we're proud of

we got a fully functional prototype running

What we learned

We will be creating a crows sourcing platform and also make the code setup open source .
