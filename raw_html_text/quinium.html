Inspiration

As students we take notes but we're usually too busy to review them. So we created an app that generates quizzes based off our notes.

What it does

You take a picture of your notes and it passes to a AI transcoder also known as the Google Cloud API, it then parses the data to the Quizlet API, where we grab questions and then render the questions into the app.

How we built it

We used Node js, React Native, Quizlet API 2.0, Google Cloud API.

Challenges we ran into

Learning to format with React Native, getting accurate data from parsing

Accomplishments that we're proud of

Making a nice GUI with React Native, getting accurate data from parsing

What we learned

React Native, How to use the Quizlet 2.0 API

What's next for Quinium

Exporting the quiz to document format to a cloud service so you can distribute or print the generated quiz, multi language support, and integrate math recognition.
