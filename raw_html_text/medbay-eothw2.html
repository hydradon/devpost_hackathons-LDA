Inspiration

We felt that the modern method of keeping track of your medicines is greatly outdated.

What it does

The app allows you to input the medicine you are currently taking. This will allow you to manage your medicines, get more information on each medicine, set up reminders to take your medicine, and allow you to get refills delivered to your house.

How we built it

To build this application we used Android Studio to develop the app and Java to unit test elements of the backend. We also used Sketch to help design the user interface and used to open.fda.gov API

Challenges we ran into

Android Studio is a completely new software for all of us Fetching the data requires an asynchronous request
