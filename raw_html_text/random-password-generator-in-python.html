Inspiration

My inspiration was a website that had a bunch of python project ideas. I saw a random number generator and decided to make my own as the hackathon project.

What it does

My password generator first asks the user if he wants spaces, tabs, etc. Then it asks how many passwords it should make. After that, it asks for the length of the password(s).

How I built it

I just coded it in python 3.6 since that was the only language I knew.

Challenges I ran into

I ran into a challenge that was I had to type each thing on the keyboard separately in the list. I overcame this by using string.letters+string.digits+string.puncuation.

Accomplishments that I'm proud of

I'm proud of learning import string and ascii.

What I learned

I learned more in python by spending a lot of time in it.

What's next for Random password generator in python
