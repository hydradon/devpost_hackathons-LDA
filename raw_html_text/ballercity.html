Inspiration

Nous croyons que la participation citoyenne est la recette d'une ville plus intelligente et plus agile. À travers une interface simple et purifiée, les citoyens pourront avoir accès plus facilement à plusieurs services municipaux tels que l'inscription à une garderie et aux activités sportives. Nous offrons également la possibilité de chercher sur une carte interactive, les garderies et les activités sportives qui respectent critères (distance, prix, etc.)

What it does

L'utilisateur peut :

  • Trouver les garderies proches de son domicile et qui respecte le maximum de critères;
  • Inscrire son enfant dans une garderie;
  • Trouver des activités sportives proches de son domicile;
  • Inscrire son enfant à une activité sportive;
  • Signaler un problème routier;
  • Visualiser les problèmes routier signalés par les utilisateurs en temps réel

How we built it

Un front-end web basé sur Angular 4 qui communique avec un backend en express Node Js pour obtenir/interagir avec les différentes données qui proviennent d'une base de données non relationnelle.

Challenges we ran into

Obtenir des informations fiables sur les adresses physiques, la limitation sur le nombre de requêtes possible sur les API, la compléxité des API, Aymen, utiliser MongoDB pour uploader des fichiers JSON, l'UTF-8

Accomplishments that we're proud of

Angular 4, comment utiliser les API (Poste Canada, Google Maps, Twitter, Google Roads)
