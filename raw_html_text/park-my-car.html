Inspiration

The struggle of wandering around a parking lot trying to find a free spot, resulting in wasted time and annoyance.

What it does

Our site provides an user-friendly interface which allows the user to view the available spots on each level of the parking lot, and reserve a free spot.

How we built it

We used HTML and CSS for the interface and PHP and mySQL to interact with the database. We also used javaScript for making the website more interactive.

Challenges we ran into

Installing all the necessary software. Linking the database with the interface.

Accomplishments that we're proud of

We reached our modest goals..

What we learned

Sleep is for the weak.

What's next for Park My Car

Developing an Android app so that the user can easily access the interface on the go. Creating timers so that the user will know when a parking spot will be made available.
