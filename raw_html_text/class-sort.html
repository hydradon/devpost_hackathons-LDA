Inspiration

The uoZone website is such a nightmare to navigate, we decided to devise a new method to manage your classes on your own terms. It is also an organization planner centered specifically around our education without being bogged down by trying to manage every aspect of your life at once.

What it does

The user can add or or remove courses, and for each created course, three functions can be performed. The user can upload files or images to the class to keep as a reference for later, notes can be inscribed for each individual class, and reminders can be set for future deadlines.

How we built it

We built it using android studio with the front end in .xml and the back end in java.

Challenges we ran into

The majority of our group had never used Android Studio before, so a lot of learning was involved. Setting up the system for notifications and uploading files was a challenge, as it required knowledge of programming libraries unique to Android.

Accomplishments that we're proud of

Achieving all the goals we set at the beginning of the Hackathon, and learning how to develop apps for Android.

What we learned

We learned how to use Android Studio.

What's next for Class Sort

More options for customizing reminders, a better UI for viewing files, and more detailed notes.
