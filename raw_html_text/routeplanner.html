Mål: Å hjelpe folk som kjører elbiler å planlegge reiserute med hensyn til ladningstasjoner og severdigheter de ønsker å se. Du kan gå på museet mens bilen står på ladning osv.

Løser problem: Hjelper å finne riktig ladningstasjon: for rask ladning for de som har dårlig tid og vil betale for det, og for de som liker å lade bilen sin billig og bruke tid mens de venter på noe interessant.

Data som skal brukes: NOBIL API for laderstasjoner (1) og norwegiana/riksantikvaren/etc (2) (planen var a bli ferdig med del 1 på hakaton)

Mulighet for utvikling: inkludere data og tilbud fra hoteller, campinger og restauranter som ligger i nærheten av reiserute, laderstasjoner og points of interest.
