What it does

This app is able to generate routes for various people working in delivery based on what is loaded on truck. Route is update after each package leaves car. All thanks to beacons.

Challenges I ran into

Using javascript canvas in android app.

Accomplishments that I'm proud of

Beacon detection, javascript canvas, selfie duck stick(tool used to locate our seats), huge part of app was written on tablet.
