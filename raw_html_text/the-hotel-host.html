• Alexa powered check-in/check-out and concierge

Inspiration

When we arrived in London we went straight to the hotel. We were exhausted, but hotel checkin took 45 minutes. We decided there had to be a better way!

What it does

Using Alexa, users can navigate the check-in/check-out process and offer concierge recommendations, all with their voice .

How we built it

We built it as an Alexa skill using Amazon lambda.

Challenges we ran into

We ran into difficulty with the initial Alexa setup, and with asynchronous calls from lambda. We also had difficulty integrating Esri.

Accomplishments that we're proud of

Getting Alexa to behave and have a seamless extended voice transaction with us.

What we learned

How to set up a custom skill on Alexa!

What's next for The Hotel Host

Perhaps we'll demo our product to a hotel technology manager.
