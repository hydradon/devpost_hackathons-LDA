• Needle Pick Up Program
    • Problem: Discarded needles need to be collected and safely disposed of.
    • Current Solution:AIDS Program South Saskatchewan receives reports of needle locations from the general public via the phone, however there are issues around when the phone isn't manned or with the lack of exact location of where the needles are.
    • Proposed Solution:Web app to report discovered needles:
      • Records user location
      • Optional photo upload of the needles
      • Optional description
      • Optional contact information to follow up on the completion of it
      • Reports are collected in a admin view for staff to follow up with and report the number of needles collected / collection status (succeeded, couldn't find the needles, etc)
