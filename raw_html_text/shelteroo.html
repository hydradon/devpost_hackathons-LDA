Inspiration

We wanted to create a web application using frameworks we were unfamiliar with. We decided to go with our idea because in a large city like New York there may be many people who are interested in helping pets.

What it does

It allows shelters to look for volunteers, and for people to search for shelters to volunteer at.

How we built it

We built this using SQL, Flask for the backend and html and css for the front end.

Challenges we ran into

We were all relatively new to most of the frameworks we used.

Accomplishments that we're proud of

We were able to learn a lot and considering we tried using languages and frameworks we never used before.

What we learned

We learned how important time management is. In addition, we learned how to use an API and work with multiple frameworks.

What's next for Shelteroo

We have to fix a bunch of stuff. For starters some of our links don't work. In addition, while we used our API to calculate distance, we don't show the actual map.
