Inspiration

Toutes les villes offrent de nombreuses possibilitées d'activitées qui ne sont pas toujours connues des citoyens. Notre application Web permet donc de trouver les activités offertes dans le secteur.

What it does

L'application web localise les activités, loisirs et événements qui sont situés près de l'utilisateur. L'utilisateur peut choisir le rayon dans lequel les activités sont détectées ainsi que le type d'activités (comme les spectacles, les sports, les arts, etc.)

How we built it

On a fait une application web en HTML, CSS et Javascript pour le Front-End ainsi que PHP et MySQL pour le backend.

Challenges we ran into

-Normaliser les bases de données
-Transformer toutes les adresses de la base de données en coordonnées terrestres
-Choisir une technologie connue de toute l'équipe
-Partage des bases de données entre les membres de l'équipe
