Inspiration

Tremco's technology - they need a mobile app! I am here to deliver!

What it does

Allows people to choose a location and mark down on a map of their building layout, specifically where a leak is.

How we built it

Pure swift 4, built on iOS with a backend php server for testing / demoing purposes

Challenges we ran into

This was Mica's first time writing anything besides html - it was a wild but fulfilling ride from 0 to backend server!

Accomplishments that we're proud of

Building a fully functional application within 24 hours, building a backend server for the very first time starting from no knowledge of code beyond html

What we learned

How to communicate effectively and how to build a production level application very quickly!

What's next for Super Leaks
