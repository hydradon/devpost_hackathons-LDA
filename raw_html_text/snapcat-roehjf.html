Inspiration

YikYak but for positive feedback.

What it does

People can spot and post cute cat photos taken recently within a certain radiums around them.

How we built it

We used React Native, Firebase for authentication and IMB-Watson to verify that the photos taken are photos of felines before posted.

Challenges we ran into

Setting up the authentication with Google, setting up React Native in general, running the Android emulator, dealing with Gradle.

Accomplishments that we're proud of

It works and is adorable. Proud of making a mobile app as we are more comfortable with Web apps.

What we learned

How to work as a team, getting more comfortable with React Native (it was our first time using it), some insight on developing mobile applications.

What's next for SnapCat

To be able to check cats around a specific location around the world. Checking all the cats that you voted for and spotted. Being able to share these posts with your friends.
