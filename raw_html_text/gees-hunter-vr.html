Inspiration

Aston University loves their geese ever so much, so a game about hunting them was the only option!

What it does

You aim at gees and shoot using the oculus rift headset and remote. You gain score points for every goose hit.

How we built it

We built the game using Unity with programming in C#.

Challenges we ran into

Wouldn't be able to list within this fields word limit. Public/Private variables, sensor accuracy and key binding to the Oculus remote to name a few.

Accomplishments that we're proud of

Starting of as two guys with little experience in C#, VR and Unity and then finishing having built a game using all three of these.

What we learned

C# basics, making use of Oculus SDK/Hardware, using unity.

What's next for Geese Hunter VR

Expand game functionality and remove some potential bugs.
