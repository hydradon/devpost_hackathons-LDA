What it does

  • Find your voice, showcase your skills
  • Users to sign up and start discussions/conversations by uploading a video, discussing a topic, tagging it and sharing it. Other users "Chip In" by uploading their own videos to continue the discussion that the user had initiated
  • Users will create a profile and list their interests. Any videos that are tagged with their interests will appear in the user's feed.
  • Search videos to hear what users have to say about any topic - search options include location, rating, tags, trending.
  • Clarifai API is used to intelligently analyze videos and visualize the content using tag clouds
  • Video comparisons using the tag clouds will be created using metrics such as geolocation, event meta data
  • Rank and endorse topics and "chip ins" to boost discussions
  • Find inspiration through like-minded people and follow them
