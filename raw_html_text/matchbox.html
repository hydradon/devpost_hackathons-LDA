We wanted to play with Google Cloud Vision API but we also wanted to find a way to make our project applicable and close to many young hearts <3 Users just have to swipe on their own a few times, and in no time at all our application will learn their preferences and swipe for them!

We used Python for everything. We used Tinder API to get all of the information necessary for this project. We analyzed the images by using the Google Cloud Vision.

We threw everything inside of a sequential learning model using Keras and that allows this application to learn as you swipe.
