Inspiration

Both team members enjoy the look and taste of mushrooms.

What it does

Determine if a mushroom is edible by taking a picture of it.

How I built it

Using IBM Watson studio's Object Detection toolbox, we were able to label a mushroom's features such as its color, shape, and surface. Then, we feed these data into the AutoAI model trained using a Kaggle dataset in order to determine if it is edible or not.

Challenges I ran into

1) Figuring out IBM Watson Studio 2) Lack of mushroom knowledge 3) CORS

Accomplishments that I'm proud of

1) The product actually somewhat works

What I learned

1) IBM Watson Studio's features 2) More about machine learning

What's next for Eat The FunGus

1) Mobile version 2) Better UI 3) More data for training
