Inspiration

We were in ECHA at uAlberta and we found out that we couldn't make a game. So we were like "Yo Leap motion in a car!".

What it does

It allows the user to run automated tasks in a vehicle's infotainment functions with gestures allowing the users to focus on the road.

How we built it

We were like yo, leap motion. We need to connect it to a phone. First we explored direct connection with an android phone. However, with a lack of public sdk privileges, we had to explore another means of communicating gesture data with an android phone. Our solution was to route the data through the cloud enabling communication with the device.

Challenges we ran into

Reliably accurate gesture identification and server optimization.

Accomplishments that we're proud of

We decided on a project and followed it through without hesitation.

What we learned

We learned the fundamentals of group collaboration and splitting up the work in a coding project.

What's next for Gest

We are going to transition from the cloud and establish a direct hardware link to enable a more reliable and streamlined experience.
