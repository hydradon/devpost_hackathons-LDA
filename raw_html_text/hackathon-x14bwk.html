Matrix Reload

Matrix Reload is the Hackathon project built by the fabulous Full Stack Team for the UCI Data Hackathon

You can run this demo by the following command line 

sbt run

Then you can open your browser to check out the two apps

  1. localhost:9000

Interactive Signal Strength/Quality Temporal-Spatial visualization demo

  1. localhost:9000/app

Interactive App Usage Temporal-Spatial visualization demo
