Inspiration

Data analysis can help in taking the right decisions. For sponsors, it is not always clear which footballers provide the most value per cost. By looking systematically at performance and popularity data, this process can be supported and great value can be generated.

What it does

We gather data from social media and statistics sites and present it in a clear manner.

How we built

We used regex to parse html and javascript.

Challenges we ran into

Cambridge Analytica as well as paywalls made scraping data from the internet very difficult. Also, we didn't have a front end developer, so we focused more on scraping data.
