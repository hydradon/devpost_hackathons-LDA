Inspiration We wanted to create a classical game using the a new way to play.

What it does One would use gestures , left, right, up, down to move the snake throughout the grid to eat the food.

How we built it We used Arduino boards and C++

Challenges we ran into We had lots of problems with connecting the LED matrix to the arduino. The matrix required 12 pins which would render us unable to add any other features. Hence we use shift register to minimize the pins to only 3 pins. This allowed us to add the scoreboard
