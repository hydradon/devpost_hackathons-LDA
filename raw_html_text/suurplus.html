• SuurPlus

Inspiration

Our community members who struggle to find food at an affordable price and a stable rate.

What it does

It connects people with unwanted food with those looking for free food in the hopes of reducing waste and reducing the rates of food insecurity.

How we built it

We built it using html and javascript. The programs we used included git, node js, and the terminal. We designed the UI using Ucraft.

Challenges we ran into

Learning javascript in a short amount of time. Also, embedding html into UI.

Accomplishments that we're proud of

We are proud of effort we made to attack an issue so prevalent in our community. We are also proud of what our web page looks like.

What we learned

We learned you cannot learn javascript in the amount of time given.

What's next for SuurPlus

Making the javascript more functional and interactive with the UI.
