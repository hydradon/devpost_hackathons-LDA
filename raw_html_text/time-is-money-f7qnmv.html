"# time-is-money"

This is a project by Alexis, Henry, Nick, and Rob for Mhacks on Feb 20-21, 2016, all of us freshman at the University of Michigan. For our first hackathon, we created a web chrome extension app looks at the text in a web page and replaces all money values (numbers after a '$' sign) with the number of hours you would have to work to earn that much money. For example, if you earn $10/hour and you visit the amazon page for an item priced at $15, it will replace the '$15' text with '1.5 hours'. We created this to make people more aware of their purchasing decisions. We hope that people who use this app will learn to spend more wisely and learn to better value their time :).
