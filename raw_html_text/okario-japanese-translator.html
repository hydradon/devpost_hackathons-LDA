Inspiration

I wanted to improve on methodical translators like Google Translate.

What it does

It converts text in Japanese to a rudimentary English.

How I built it

I used the Python server on repl.it!

Challenges I ran into

Many words had multiple meanings, and the program needed to differentiate from context.

Accomplishments that I'm proud of

The translator can translate basic sentences.

What I learned

File I/O operations, parsing through text files.

What's next for Okario Japanese Translator

Increasing the sample size for words, enlarging the kanji dictionary.
