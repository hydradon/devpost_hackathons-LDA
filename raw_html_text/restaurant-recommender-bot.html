Inspiration

We just really like food and we always want to try new food!

What it does

A chat bot that can recommend you nearby restaurants to eat or to make a reservation.

How we built it

We used Node.js for the programming language behind the chat bot. We used Microsoft Bot Framework and Language Understanding Intelligent Service (LUIS) to power the chat bot. We also used Bing Entity Search API to recommend the user nearby restaurants.

Challenges we ran into

Figuring out how to design the conversational flow of the chat bot and actually implementing the chat bot using Node.js.

Accomplishments that we're proud of

Finishing this chat bot with natural language processing!

What we learned

How Microsoft APIs are so flexible and versatile!

What's next for Restaurant Recommender Bot

Probably more features to be integrated into the chat bot, so stay tuned!
