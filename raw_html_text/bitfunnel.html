Inspiration

The true inspiration comes from seeing projects such as hyperledger. I believe we are truly making a better version of this.

What it does

Anchors IoT data to Bitcoin. Verify IoT data wasn't tampered with. Focusing on Oil Industry

3 node Kafka cluster in AWS, 3 node zookeeper, Data comes into the cluster and then gets pegged to the Bitcoin blockchain, receipts are available and then proofs after blocks have been written.

How we built it

A lot of different technologies, Tierion to anchor to bitcoin.
