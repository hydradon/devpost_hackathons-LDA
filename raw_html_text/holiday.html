Inspiration

As travel enthusiasts, we all had the similar problem of being unable to choose a 'dream' vacation- other websites we had found were based around questions that we felt were too broad and impersonal. With Holiday, we strove to provide a way for people to get their perfect vacation by factoring in not just their vacation preferences, but their personality using the Meyer Brigs test.

What it does

Holiday is a HTML-based website that suggests the ideal vacation spot(s) for the user according to their Meyer Briggs personality type.

How we built it

We used HTML, CSS, and Bootstrap Library to design the website. We also imported Google Maps API to produce a map.

Challenges we ran into

The Google Maps API was the most difficult feature to work with.

Accomplishments that we're proud of

We are proud of succeeding in using the Google Maps API and the end product.

What we learned

We learned how to use APIs and how to make an interactive website.

What's next for Holiday

We may add more features and publish it if it receives enough interest today.
