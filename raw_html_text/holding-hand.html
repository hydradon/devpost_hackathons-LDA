Inspiration

We are very concerned about the recent number of sexual assault cases in Ames. So, we developed something that would make it easier for people to reach out for emergency

What it does

With a touch gesture, user will enter an Aware Mode where the user will have to keep holding the touch screen. Releasing it will trigger a 15 seconds countdown. When the countdown is over without further user's response, automated emergency messages will be sent out to selected contacts with information about the user's location. User can cancel the countdown by entering personalized pin, signifying that the user is back on being safe.

How we built it

  • Android Studio
  • Google Maps API
  • Java

Challenges we ran into

Getting the Location was a challenge as the standard way to do it using the LocationManager was deprecated in Android Studio 2. We decided to use the Google Maps API for a more complete set.

Accomplishments that we're proud of

We are proud of been able to help people in need. The ability to send multiple text messages with a simple click of a button.

What we learned

The challenges of compacting everything in one app and making all the different components communicate and work together.

What's next for Holding Hand

Make it more specific for zones, including Iowa state University. Even contacting 911.
