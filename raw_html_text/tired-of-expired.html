Inspiration

We were inspired by our own tendencies of forgetting about leftovers and food and trying to minimize our own food waste.

What it does

This website keeps track of food items in your fridge and when they will expire. It notifies you when something is expiring soon so you can plan your meals accordingly to minimize your food waste.

How we built it

We build this using Javascript, HTML, and CSS.

Challenges we ran into

Our biggest challenge was the learning curve of learning a new language, since none of us have created a website before or coded in Javascript/HTML.

Accomplishments that we're proud of

We are proud that we were able to make a website that is able to properly store user input, alert the user about the food's expiration, and display a list of foods in the fridge.

What we learned

We learned a lot about web development and how frustrating it can be, but how amazing it felt when we reloaded the page and our HTML code was working properly.

What's next for Tired of Expired

Right now, we have reserved the domain tiredofexpired.org for this website. In the future, we want to be able to categorize the food items by category, such as food group, by expiration date, by date the food was added, and any customizable tags that the user wants to create for themselves (such as meal, etc.). We also want to add a feature that suggests foods you can make with the food items you have logged in your fridge.
