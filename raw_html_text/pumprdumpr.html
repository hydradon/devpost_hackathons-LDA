Inspiration

Usually only the rich with big portfolios had the capability to manipulate penny stocks by pumping and dumping them. That's not fair, so we created PumprDumpr

What it does

Someone can organize a pump and dump for a specific stock, ad raise money from others. People sign in with their brokerage account, and our program buys it all at once, waits for the stock to go up, then sells it all at once.

How we built it

Our web app is built on node.js with the pug view engine. It is hosted on Microsoft Azure on a .tech domain.

Challenges we ran into

When we deployed our app on Azure, for some reason it was giving an internal server error even though it was working locally. Turns out our startscript used nodemon instead of node and it didn't have nodemon installed. One of us wasted half the hackathon working on this bug :)

Accomplishments that we're proud of

We are most proud of our own fuzzy search that will filter out stocks that you don't want. Another thing is converting a huuuge CSV file to JSON so we could find elements and iterate through it extremely easily.

What we learned

We learned a lot of different skills that we could use at every next hackathon, such as setting up Microsoft Azure, learning OAuth, essentially creating our own API that we could use to get stocks, among many other smaller skills..

What's next for PumprDumpr

We believe that PumprDumpr could revolutionize the penny stock industry, and helping companies by increasing their stock prices.
