Inspiration

We are a group of math students and we love abstract and crazy ideas. We decided to develop this game as it's a continous game that depends on diferent timelines.

How we built it

We developed the WebGL engine in static javascript and html and then wrapped all with node (express), bootstrap and mongodb. We used heroku and mLab for the free deployment and exposition.

Challenges we ran into

Mind-lowing algorithms about how we could store all the movements done in the present. Also the challenges we made were pretty difficult lol ( level 4 ).

Accomplishments that we're proud of

We managed to find the weekly PIN of The Game, that UNIX timestamp...
