Inspiration

We wanted to create a tool that helps DJs in creating and organising their music catalog.

What it does

It helps DJs in creating their set looking for similar songs from his music catalog.

How I built it

We built it using the umg dataset and the spotify api to create the system recommendations. We used python with django, and psql to create the backend.
