Inspiration

There are a couple of apps on the App Store that try and do goalie rentals well. However, they all feel amateurish to me. I wanted to build something more robust.

What it does

People who need goalies can post their game on the app, and goalies can accept the games. In a finished state there would be a payment system involved as well.

How I built it

Django provides a RESTful backend to manage all of the data. All of the front-end is Swift using iOS frameworks.

Challenges I ran into

Everything to do with iOS. I've never built an app to completion before, so there was a very steep learning curve.

Accomplishments that I'm proud of

I built an app that functions fairly well!

What I learned

So much about app development!

What's next for Puck Saver

I can't wait to get home and fix up all of my little hacks in the app, and make it look much more professional. I'd like to eventually have it up on the App Store, and have a full web app and website to complement it.
