Inspiration

People tend to forget their memorial moment. Lucky Photo reminds us with happy photos automatically.

What it does

When people re-visit memorial place, Lucky Photo will send SMS notification with most happy photo that was taken before.

How we built it

For this service, we used bunch of platforms

  • Google Photo
  • Nexmo
  • AWS

Challenges we ran into

  • Automatically crawl Google photo, when user take happy photo.
  • create index with Geolocation info and emotion using Amazon Rekognition
  • send notification via Nexmo, when user re-visit the location.

Accomplishments that we're proud of

We made people lucky

What we learned

Using platform is quite cool for rapid deployment.

What's next for Lucky Photo

  • develop mobile app
  • More situation trigger (not only location).
