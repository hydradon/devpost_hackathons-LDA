Don't Sleep and Drive

A mobile application that analyzes if you are tired, drunk or high before driving. A tap away from knowing if you can drive. Try it out !

To start the mobile app, just run: 

cd client && npm start

Then, plug connect with the Expo Client on your phone.

To run the backend, run: 

cd backend && npm start
