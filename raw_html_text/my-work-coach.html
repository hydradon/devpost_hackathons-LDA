Inspiration

I've been asked by many people I know how I get so much done while working from home. Following a process of setting goals, prioritizing tasks, and then not getting distracted or interrupted makes working from home a more efficient and effective process.

What it does

My Work Coach is there for when you get distracted, to help pull you back to work. It also provides a structure fo setting goals, defining tasks, and eliminating distractions. Ask for random questions or inspiration that will help you think about how you work. Or go through the question flow to eliminate blockers and distractions.

How I built it

The skill is built using the Amazon Lambda serverless backend.
