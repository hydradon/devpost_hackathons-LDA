Inspiration

Developers create unique code all the time! What if that code was able to generate unique music? That's the idea that inspired us.

What it does

As you type or code, piTunes plays music based on the letters, numbers, and characters you type!

How we built it

We used the raspberry pi and interfaced it with the sonic pi.

Challenges we ran into

We had a hard time getting it up to domains.com since the link is broken

Accomplishments that we're proud of

We are really proud of our punny identity! Along with that, we are inspired by the potential to produce unique music while typing unique code.

What we learned

We learned that consistent failure leads to success

What's next for piTunes

We want to take it to the next level and add more sound kits or sets of sounds per letter to see the kinds of melodies that can be produced while writing unique code.
