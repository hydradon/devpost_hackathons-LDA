Inspiration

On university campuses there is a lot of waste thrown out solely because people are unsure whether what they have is recyclable or not. We designed this hack so that people have access to a resource that can tell them instantly what they can do with their trash

What it does

Take a picture of the trash you have and the Western University Recycle Bot will use its machine learning algorithms to let you know if what you have can be thrown in the blue bin or not

How I built it

We build this bot using the Clarifai API to help with machine learning, as well as the Python and Flask for the texting services

What's next for Western University Recycle Bot

We need to continue improving the machine learning algorithms in order to add an even larger database of objects that are recyclable or not.
