Inspiration

We were inspired by fishbase.org's "Fish Identification" feature, but we wanted to make it more user-friendly.

What it does

User filters down possible species until they've found their fish.

How we built it

Using Node.js, scraping fishbase.org, angular.js frontend.

Challenges we ran into

Learning how to scrape, designing (we aren't designers)

Accomplishments that we're proud of

The simplicity of how it works, and how it looks.

What we learned

What's next for Find-that-fish

Add in feature for measurements Identification by location
User can select a fish that he/she has caught and submit the data into a database for industry and research analysis.
