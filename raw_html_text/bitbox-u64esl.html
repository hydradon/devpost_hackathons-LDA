bitBox

Server based File Management in PHP

bitBox Data Management System is basically a File Organizing Server based Web App which helps an authorized user to upload and download files to and from bitBox Server anytime anywhere.

The main functionality of bitBox is its File Storage service. bitBox allows users to store files on bitBox servers hence the user data can be shared on any number of devices. Currently bitBox is able to support large number of File Formats ranging from Image(jpg, png), Video(mp4, mkv) all the way to Text Files (.txt) and even Compressed Files(.zip, .tar.gz). bitBox is equipped with a very intuitive and minimalistic UI to deliver fast services and create a great User Experience. bitBox implements Authorization and Security through Session Management and User Authentication by SQL Database.

bitBox also has content sharing functionality which means you can share variety of files among numerous users just by knowing their username.
