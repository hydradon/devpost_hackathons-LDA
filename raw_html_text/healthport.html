Inspiration

We wanted to make an easy way for people to manage their appointments along with easily accessing information about their current health practitioners

What it does

You can add profiles for your health practitioners and you can book appointments which show up on the page.

How we built it

Using HTML, CSS, and Embedded JavaScript we worked together on a service called Cloud9 which allowed us all to edit the project simultaneously

Challenges we ran into

We had a few bugs that took quite a while to fix, some notable challenges were:
-extra appointments and practitioner posts being created
-How we were going to design the site
-How much we needed to learn about HTML, CSS, JavaScript

Accomplishments that we're proud of

-Getting the site completed and presentable

What we learned

-We all gained lots of practice with HTML, CSS, and JavaScript

What's next for HealthPort

There are multiple things we could add that we feel would bring our site to the next level.
-When you click book it would automatically fill in the informative of the practitioner you picked
-An option input in the job title for the practitioner so that does not have to be manually added
-A calendar pop up with times for booking an appointment
-When you book an appointment it would send a text message to the number saved to that practitioner
