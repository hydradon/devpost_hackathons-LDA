Inspiration

We were inspired by the issues of motivation to exercise in our communities. We want to incentivize and empower people to take control over their own health and well being as well as give loved ones the opportunity to help in a productive manner.

What it does

Bet will take an inputted name and reroute to an account page. This name is stored in a local Python .txt file. The user will then have the option of challenging their friend to a specific health challenge. The user may also accept or decline challenges on the left side menu.

How we built it

We used Flask to build a Python back-end and used HTML in order to create the interface.

Challenges we ran into

We have never programmed in HTML or Flask before and taught ourselves over the course of the hackathon. We have also never designed a website before in any language.

Accomplishments that we're proud of

We learned basic HTML and Flask

What we learned

We learned about how to design a website and about all of the different components (both back-end and front-end) that go into its creation.

What's next for Bet

We need to create a functional data base that is not on a local file. Right now a local Python file is written in order to store the user information.
