MESA 3 EQUIPO: CODEFIGHTERS

Inspiration

Hemos visto muchos proyectos utilizando Alexa y nos inspiro que tan avanzado es. También nos inspiro la forma en que podemos interactuar y como podemos utilizar programación para generar herramientas educativas.

What it does

Es un juego con temática medieval fantástica en donde a través de preguntas de matemáticas, usuarios vencerán monstruos, dragons, esqueletos y más. Es un juego educativo controlado mediante comandos de voz.

How we built it

Utilizamos la herramienta Amazon Skill Kit y Node.JS para generar la lógica y el gameplay.

Challenges we ran into

Colaborar en Alexa Developer Console y poder comunicar código de JS a un REST API.

Accomplishments that we're proud of

Generar la lógica de las preguntas y hacer un sistema de randomización para brindar una experiencia única a cada usuario.

What we learned

Utilizar el API de Amazon Skill Kit. Los componentes diferentes para generar nuevas interacciones con Alexa y como utilizar NodeJS para un rápido desarrollo.

What's next for Aventura de la Sabiduria 3

Planeamos conectarlo a un API que generamos nosotros mismos para poder conectar la Skill de Alexa a más plataformas.
