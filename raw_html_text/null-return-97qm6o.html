Inspiration

We were inspired by the Arvato challenge to build a toll that tackles a problem their business faces every day.

What it does

Null Return makes it possible for individuals to return items purchased online to any retail store or branch. There are two ways of returning: either through a web interface or through SMS.

How we built it

The application is build using the Arvato API for storing and retrieving orders. Some return information is stored in a separate AWS database.

Challenges we ran into

Setting up an SMS service was problematic since there are limited providers in Switzerland.

Accomplishments that we're proud of

Creating an application from scratch and hosting it on AWS.

What we learned

How to integrate SMS services to an application.

What's next for Null Return
