Inspiration

Actualmente México se enfrenta a un rezago académico en lo relacionado a matemáticas. Según estadísticas en 2012 México ocupó el último lugar entre los países de la OCDE en la prueba PISA. Es por esto que quisimos crear una plataforma en la que los niños puedan aprender matemáticas de una manera interactiva.

What it does

Reta a los niños a la resolución de problemas de tal forma que desarrollen su pensamiento lógico, aplicando "gamificación" para hacer más atractivo el proyecto para los niños.

How We built it

Utilizamos lenguajes de desarrollo web: MySQL como motor de base de datos, PHP para el back-end, y HTML5, CSS3 y Javascript para el front-end.

Challenges I ran into

Al no utilizar frameworks el desarrollo se hizo más lento y pesado. También tuvimos ciertos problemas de conexión.

Accomplishments that I'm proud of

Hacer el sitio sin el uso de frameworks. Nos dimos cuenta que eramos capaces de hacer el proyecto desde cero.

What I learned

A desarrollar un proyecto en equipo utilizando herramientas de desarrollo colaborativo (GitHub).

What's next for Logramatica

Publicar la plataforma y darla a conocer.
