Inspiration

Alexa doesn't have many languages built into it, and I wanted to increase the userbase of alexa by making her commands more accesible for limited english speakers

What it does

Translates a normal Alexa Query to spanish

How we built it

Using all Amazone AWS services

Challenges we ran into

Writing the file to S3, getting the audio to be pronounced correctly in native tongue

Accomplishments that we're proud of
