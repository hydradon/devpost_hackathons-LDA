Inspiration

I absolutely love Retro games. From playing Super Mario Land on the original Gameboy, to Yoshi's Island on the GBA anything retro, give it to me.

What it does

A platform for sharing web games. Think Miniclip, built in 2018.

Can you imagine what the games you love would have looked like if they had first been created in 2018?

Well now, you don't have to!

From Trump Invaders to Kanye Snake, experience the games you know and love, with the modern figures of authority you don't!

How we built it

All of the games are built using p5.js (processing) for rendering, and vanilla JavaScript. The website is built using a disgusting table layout in HTML5 and CSS3.

Challenges we ran into

Working alone I wasn't able to complete as many original games as I would have liked.

Accomplishments that we're proud of

Coming up with original twists to existing games and link them to current themes in business, news and politics.

What we learned

How to register all key events in JavaScript and disable built-in functions for key presses.

What's next for 90s Steam

Hopefully a transition into a personal portfolio for games.
