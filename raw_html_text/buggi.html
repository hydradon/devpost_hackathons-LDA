Inspiration

Watched a YouTube about house critters that play a vital part in your houses ecosystem

What it does

It analyzes whether or not the bug you take a picture of is a threat or not.

How we built it

We used the Clarifai API to use machine learning to identify the types of bugs when you send it a photo. There is also a VR environment is which you can hang out with a recently taken critter. It uses the GIPHY API to generate the gifs for the boxes

Challenges we ran into

Setting up Clarifai API, GIPHY API, having images render.in A-Frame

Accomplishments that we're proud of

Learning how to use A-Frame, Clarifai API, and GIPHY API

What we learned

How to use A-Frame, Clarifai API, and GIPHY API

What's next for Buggi

Integrate the core parts together, give you tips on what to do if uncommon insect is identified
