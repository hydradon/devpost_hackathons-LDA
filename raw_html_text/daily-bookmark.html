Inspiration

Daily Bookmark is a simple secure app for your daily bookmark needs powered by blockstack. When you use a Blockstack decentralized application (DApp), the data that you enter into that application is something you own and control.

How I built it

The frontend for this app is powered using React js. The login is powered using blockstack login. The storage for this app is powered by gaia.

What's next for Daily Bookmark

I plan to add more features to make this app robust.
