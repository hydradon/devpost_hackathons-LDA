Inspiration

People are taking more photos everyday, but these pictures often go forgotten within a few months. Nowadays, most photos are not printed, so reminiscing over old memories is much harder. We wanted to give people access to retrieve their old photos without the hassle of going through files from years ago.

What it does

Users can upload their pictures and decide a date to retrieve these photos. Our website will email the user on that date with a hashed key to access their memories.

How we built it

Used HTML and CSS for the front-end and connected it with Node.js and Express to handle the back-end.

Challenges we ran into

Both members of our team had no prior knowledge of Node.js and Express and very little knowledge of HTML and CSS Connecting front-end and back-end

Accomplishments that we're proud of

We are proud of reading through documentation of languages that we did not know before and creating a web app.

What we learned

How to use HTML, CSS, and JS to create a web page that can store files locally.

What's next for CaptureCapsule

We want to learn how to store images to a database and deploy our application.
