Inspiration

Not everyone can manage their limited budgets effectively, so why not build a service to help them out?

What it does

Bundy is a web services that helps you in better managing your budget.

How we built it

We used HTML with the Materialize framework to develop the front end, and used Javascript and Firebase to develop the database and back end.

Challenges we ran into

Learning curves when it comes to developing the back end. To some of the team members, it was their first time in developing the back end.

Accomplishments that we're proud of

An aesthetically pleasing website, learning an entire back end platform from scratch.

What we learned

We learned material about Firebase on the spot, in a limited time span.

What's next for Bundy

We hope to expand it, and include more useful features as we grow our service.

We put the Bun in Bundy .
