Inspiration

Having a challenging project for newbies :)

What it does

This banking app allows users to create an account and log in. The app remembers a user by creating a text file and saving their information in the system so that a user can log in and pull up their account information and account balances from the last time they logged off.

How I built it

We built it with three team members working together remotely.

Challenges I ran into

Working together in separate locations is tough.

Accomplishments that I'm proud of

Were proud that our app has some graphics, animation, color, and that it actually creates a memory system for returning users.

What I learned

We learned valuable new skills such as working in teams as well as advancing our technical skills.

What's next for Banking Application

Finish it off! Unfortunately we were not able to complete it all but we got a big chunk of it done. The source code can be compiled on a Mac to get the most out of it.
