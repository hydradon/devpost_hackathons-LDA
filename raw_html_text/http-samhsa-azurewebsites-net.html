Inspiration

My idea is that the web is accessible to everyone so I built a website instead of a native mobile app.

How it works

You just go to http://samhsa.azurewebsites.net/ and you can see all the beautiful content fast.

Challenges I ran into

Accomplishments that I'm proud of

First time I have ever built an offline first website

What I learned

How to design not just for mobile users but for people with low end phones so that they have a fast experience.

What's next for http://samhsa.azurewebsites.net/
