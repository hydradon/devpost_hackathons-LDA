Inspiration

We spend a lot of time online every single day, so we wanted to analyze our digital footprints we leave online. With facebook it is especially easy to learn about a person's life, so we made some deeper analysis on the photos and posts we have on facebook.

What it does

Our website allows users to log in with their facebook account and then parses the different data sets that the facebook graph API provides. We analyze interests and values with the Clarifai image recognition API. We analyze emotions with Watsons Alchemy Api that returns scores for five emotion for what the users posts on his or her facebook wall. We also display all the location data that pictures and posts provide on a google map and rank the most popular locations.

How we built it

We worked together on setting the facebook graph API calls up, having a basic bootstrap page up and a github project. Then we split up to work with the different APIs.

Challenges we ran into

Assynchron API calls, lots of data, varying data

Accomplishments that we're proud of

We got three different data analysis working. We were surprised how interesting the results actually are.

What we learned

We didn't have much experience with JavaScript or working with the APIs we chose before HackGT. We also got to see what we post about ourselves over the years.

What's next for istalk.me

We look forward to shock other people with how much personal information a simple facebook profile reveils. To expand the project, we want to add more social media platforms like Instagram, LinkedIn etc..
