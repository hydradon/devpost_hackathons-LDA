Inspiration

I wanted to make a program to be rude to others.

What it does

It takes every single file in a directory, creates 100 new folders for each one, renames the file, and shuffles it into a random folder. It even puts a small text file in each folder to make it impossible to find by sorting out empty folders. Perfect for ruining someone's files.

How I built it

Typing words that the java compiler recognizes.
