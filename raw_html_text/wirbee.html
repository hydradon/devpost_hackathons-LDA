Inspiration

Sabe aquela imagem, paisagem bonita de algum lugar que voce nao conhece? ou viu em alguma rede social de algum amigo e ele nao escreveu onde era? WirBee irá descobrir onde é esse local e irá preparar o seu pacote!

What it does

Faça o upload da foto que mostraremos pacotes de viagens para você, com a mesma paisagem, para o mesmo local! ou faça buscas por sentimentos ou temas.

How I built it

Integração com API Sabre para busca de roteiros, passagens aéreas, valores, e outras integrações para a conciliação das imagens (match), como a api GettyImages listagem de imagens na internet e CamFind API para encontrar outras imagens a partir de uma fornecida.

Challenges I ran into

Conhecer as apis certas para utilizar, design da aplicação e criar uma nova experiencia para o usuario

Accomplishments that I'm proud of

Ter aprendido a trabalhar com as APIs Sabre Dev Studio, um rico ambiente de documentação, ao mesmo tempo criar uma nova forma de procurar uma nova viagem com base em uma fotografia.

What I learned

A trabalhar com as apis Sabre e Getty Images

What's next for WirBee

Finalizar o match com a api CamFind e criar opções de alugar veiculos, reserva de hoteis e integração com a api MasterPass para meio de pagamento. Tambem a integração com as redes sociais (Facebook e Twitter), acessando o album de fotos publico do usuario, e varrer para onde seus amigos viajaram, e sugerir destinos parecidos, estimulando o usuario a trocar novas experiencias com seus amigos, buscando referencias do local.
