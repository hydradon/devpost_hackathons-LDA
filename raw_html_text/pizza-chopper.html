"When I saw the theme I instantly had the idea of a pizza chef themed game. I want to become a game designer so this was the best practice. In the game you are an unknown chef encountering and defeating pizza monsters. The game is a side scroller where you meet an enemy and a battle starts, after the battle the user is allowed to continue. Every battle is dependent on time based attacks; once you attack you must wait to attack again and so does the enemy. The code was made with Java on Apache NetBeans IDE 11.2. We tried to make the game run in real time and ran into a lot of errors. There were also problems when it came to using Github,since we were all new to it, when it came to pushing, pulling, and branches. We also needed to teach and learn from each other as we went along. We all learned how to utilize Github and Java more effectively. Almost all of the members were just getting back into Java after a long time, which made our start really slow. After submitting this here I might comeback and polish up all the bugs that I had when making it during the Hackathon " - Gaven O'Hanlon
