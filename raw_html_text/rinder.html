Inspiration

After always struggling to find where to go with our friends, we thought of making an app to help make our decisions for where to eat and places to explore.

What it does

Currently it gives users the option to look through different entertainment locations and provides the address along with an option to call the location.

How we built it

We used React to build the app, Node.js for the back end and used Yelp API for extracting data.

Challenges we ran into

Working with React was new for some of the team members, so they had a very steep learning curve to work with. We were having issues with the asynchronous API calls and with designing the UI.

Accomplishments that we're proud of

Coming into the hackathon, we were mostly inexperienced with React, but we were able to learn fast and implement some basic features.

What we learned

Invaluable experience working with React (HTML and CSS), git and debugging. (Also that ping pong is really fun!)

What's next for Rinder

Making the UI more refined and turning it into a mobile app, along with adding more settings so users can customize their experience.
