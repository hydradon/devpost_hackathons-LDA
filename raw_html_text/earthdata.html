Inspiration

Create safe and secure way to access and manage environment and earth data about your location, city and surroundings without revealing you personal information or whereabouts

What it does

At present It gets the data about the Air Quality of hundreds of cities using OpenAQ data. Currently data of 40+ countries are available. Your identity and personal information is maintained safe and secure

How I built it

Using Blockstack Authentication , user can logon into the app and then access the Air Quality information

Challenges I ran into

Implementation of Blockstack Authentication and also filling the data using OpenAQ API

Accomplishments that I'm proud of

The data is presented in a neat form

What I learned

Using OpenAQ using Node and Javascript
