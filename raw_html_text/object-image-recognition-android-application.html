Inspiration

We made this app for visually-impaired people who have trouble determining whether if a food product is fresh or not without physically analyzing it.

What it does

It uses object recognition to determine if a food product is fresh or not.

How we built it

It was made on Android Studio (Java) using Open CV.

Challenges we ran into

The app isn't able to run as we made everything from scratch and couldn't make it run at the final time.

What's next for Object & Image Recognition Android Application

We'll be able to make this run and hopefully, upload it to the Play Store with a motive to help visually-impaired people around the world.
