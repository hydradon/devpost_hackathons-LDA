Inspiration

I always have a hard time choosing what for lunch/dinner when I am in a new place. Without some time on research, it is hard to discover the unique food in a new place. This app is aimed at simplifying this process.

What it does

It gives you the options for searching the dishes that best fit your taste when you use the app. By giving it some parameters like food type, price range, food style, and location, you will be given a list of dishes with vivid images just to open your appetite.

How I built it

I built it while learning from tutorials and googling.

Challenges I ran into

Some bugs where too little information is provided to debug, which took some time and some re-dos. When comes to make the site pretty, lacking good ideas and some conflicts in css with bootstrap takes away much time. Creating a big set of mock data took some time and debugging as well. I did this with python.

Accomplishments that I'm proud of
