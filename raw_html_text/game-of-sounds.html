Inspiration

We wanted to create a game inspired by the memory of the sounds and memories of each one

What it does

It's a game. We listen a sound extract and we have to guess the sound. More than 150 audio questions from popular culture: music, movies, TV shows, famous people, animals, objects, places. At each turn Alexa gives a clue before the sound extract. You can play from one to four.

How we built it

I built it with alexa skill kit

Challenges we ran into

Alexa presentation language is hard to integrate

Accomplishments that we're proud of

The game is very fun and we hope that the users will have a lot of fun playing

What we learned

We learned alexa presentation language

What's next for Game of Sounds

For the future we want to add more and more sounds, and always more fun
