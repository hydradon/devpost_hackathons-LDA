Inspiration

Creative custom pixel shaders seen in videogames and other media

What it does

Allows you to create custom shaders and modify its' values on the fly in real time

How we built it

We used C++ with OpenGL and the Oculus SDK

Challenges we ran into

Integrating the VR headset as well as a functional user interface

Accomplishments that we're proud of

We have several working proofs of concept that integrate all we wished to accomplish

What we learned

Getting images to appear on a virtual reality headset is not as easy as the documentation would lead you to believe.
