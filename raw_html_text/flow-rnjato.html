Inspiration

I wanted to build something that I knew I would find beneficial.

What it does

Flow has two modes, one for studying and one for social media. The study mode provides resources for productivity and will eventually contain a series of motivational quotes and a to-do list. When the user is done working, they can switch into the social mode. This mode will have links to Youtube, Steam, Reddit and other fun sites. It's all about convenience.

How I built it

I built Flow in Webstorm using a combination of HTML, CSS and Javascript

Challenges I ran into

I came into this Hackathon only knowing languages that were not optimal for bringing my vision to life. The idea for Flow came after the hacking began and I dedicated myself to learning as much as I could about web development in the time allotted.

Accomplishments that I'm proud of

I'm honestly proud to have something that looks nice considering that I have never done this before.

What I learned

I learned that Google, Stack Overflow and Youtube are excellent resources. The technical problems I encountered along the way made me better at searching things up, problem solving and finding solutions

What's next for Flow

Flow will be further developed in the months ahead to become a full web app.
