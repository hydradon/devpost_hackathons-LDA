Inspiration

Difficulty finding pickup games

What it does

Assigns random teams with next game for random basketball courts in the region

How I built it

Polymer for a web front end and Python Flask for a database backend

Challenges I ran into

Traditional SDKs not working with webcomponents

Accomplishments that I'm proud of

Having a fully fleshed out REST API

What I learned

Web programming is very complicated

What's next for BasketBall Pickup
