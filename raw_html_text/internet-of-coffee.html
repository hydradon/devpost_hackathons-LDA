Inspiration

We wanted to connect something that hasn't been connected before.

What it does

An espresso machine can be remotely switched on with the use of an Android device.

How I built it

We used an Intel Edison to constantly check a Firebase variable for an op-code activation from an Android device, which would then use a servo motor to physically flip the espresso machine switch.

Challenges I ran into

  • LCD screen
  • Shield
  • Firmware
  • Servo resetting to zero

Accomplishments that I'm proud of

In an unorthodox fashion, we connected a hardware device to the internet purely through mechanical means.
