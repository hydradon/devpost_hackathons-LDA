Inspiration

I enjoy optimization games such as this and I thought it would be fun to create one

What it does

A traffic simulator. You control the stoplights and cars as you try to direct each car to its destination. Unfortunately not all of the features have been implemented

How I built it

Using xcode and swift

Challenges I ran into

Having cars merge in an efficient and realistic manner

Accomplishments that I'm proud of

When cars are merging and it looks like it will end in a disaster, the cars will work it out themselves

What I learned

A lot about the Swift language

What's next for Expressway Empire

Implementing the ability for cars to turn

Built With

  • swift
  • xcode
