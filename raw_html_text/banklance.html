Inspiration

Like many personal finance apps that help consumers to connect all their accounts in on place. Banklance is focused to bring account balance alerts together for ease of bank communication.

What it does

A web app that allows you to seamlessly add all your bank accounts. Balance data is then retrieved in real-time and displayed on the dashboard.

How I built it

Bank accounts are added via Plaid Link which supports thousands of financial institutions. Python Flask and Jinjia2 are used for the system.

Challenges I ran into

Unable to modify balance info since it's in sandbox environment.

What I learned

How to access financial data such as transactions, account number, etc thru Plaid's API endpoints.

What's next for Banklance

Moving from sandbox to development which allows live users to retrieve real credentials and account data. Adding SMS support for notifications.
