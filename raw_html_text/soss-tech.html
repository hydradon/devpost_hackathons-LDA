Inspiration

We realized that there's a two-part problem with OSS (open source software). 1. Devs don't have the time to support all the projects. Devs don't have the money to incentivize further development. We're solving the second one.

What it does

It's a super simple command line tool you can install. It lives atop of pip, npm, gem and other and keeps count of all the packages you install. When you go to our website, you can see a dashboard view of all your packages, and the count. From there you can filter projects by package manager type, and easily donate.

How we built it

React app hosted on S3 and backend node express server hosted on AWS.
