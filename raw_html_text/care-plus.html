Inspiration

The demand from the growing elderly population with chronic diseases puts enormous stress on existing caregiver supply. If a caregiver is involved in an elders care, we consistently see improved patient outcomes.

What it does

We want to use Smart Assistants that are affordable and always available to create a daily check-in with an elder. Our system analyzes the answers from the elder and prompts their caregiver to follow up on concerns.

How we built it

We used the developer resources on actions-on-google and Google cloud platform to build custom a intent-based voice assistance for surveying the elderly people living alone.

Challenges we ran into

Designing a robust system that worked for low resource environments. Sourcing a validated question set that would have a real impact on patient outcomes for the initial launch.

Accomplishments that we're proud of

We developed a working application framework that now needs expert training data to be a full fledged product.

What we learned

There are many ways to ask a question that have varying degrees of success. We learned we could apply machine learning to focus on asking better questions, creating a more defensible product.
