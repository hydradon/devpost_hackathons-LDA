Inspiration

Every year Universities around the globe receive thousands of applications. Our Sentimen analyzer optimizes this process to a great extent and helps to find suitable candidates for admission

How it works

We use NLTK in python and Meta mind API to perform sentiment analysis. We perform word to word analysis (local) as well as gloabal analysis on data.

Challenges I ran into

Using web app to take user input and have data visualization

Accomplishments that I'm proud of

We were able to summarize the letter of recommendations into key few lines which can save lot of time for reviewer.
