What is all this fuss about Financial Pypes?

This tool allows financial researchers to quickly build and test machine-learning prototypes and boost developer productivity using a friendly UI

  • Productivity Dashboard
  • Streamlining hypothesis testing

How we built it

Backend using Python with flask and some python magic (running on AWS). Frontend using Js with react, GoJs, ...

Accomplishments that we're proud of

A functional proof of concept of an alpha-gen model implemented in Fipy (see images and code), that can do things like notify people on their email when they should invest due to alpha high.

What's next for Financial Pyplines (FiPy)

Adding more block functionality (starting with statistical operations like regressions and evolving to more complex ones like neural networks) Improving the UI to be easier to use (dropdowns, ...)
