Inspiration

Our current C++ course has a group project that has a large impact on our overall grade. A quick brainstorm lead to us agreeing that mismanagement of time was usually the biggest culprit in group projects. This program we designed would be aimed to aid in solving this problem for projects of any subject and size.

What it does

Helps group projects track individual progress relative to start and due dates.

How I built it

The front end was made using javascript, html, and css. The back end was coded in python using flask.

Challenges I ran into

The largest problem we ran into was a segment of code that would be skipped over when run.

Accomplishments that I'm proud of

We are proud of our creative domain name that has a direct correlation to the program itself.

What I learned

I learned to code in Python for the first time. It was nothing like the C++ syntax I was used to and the more advanced functions were difficult to understand.

What's next for ProjectsAreSQL

We plan on adding in visuals and more indepth statistics for each user.
