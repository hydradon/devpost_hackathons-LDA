Inspiration

We came up with our project so that it can potentially win the Best Human Centered Hack from Centene. We decided to make our project for this fake persona Centene had provided named Dog Lover Dominique who has trouble trying to find more time to hang out with her pups.

What it does

Our bot makes sure to record, remove, and remind you of any task in a casual messenger setting.

How we built it

We built it using node.js and using the discord bot developer portal.

Challenges we ran into

Since none of us had not really programmed a discord bot before, we had to learn from scratch, therefore not giving us enough time to fully create more utilities.

Accomplishments that we're proud of

We are proud that we have created a functioning discord bot

What we learned

We learned how to program in javascript a bit more now and how to set up a discord bot.

What's next for Discord_Bot

We would like to implement more useful utilities in our bot.
