• Parce que ta sécurité nous tiens à coeur

Inspiration

Après avoir manipulé plusieurs jeux de données de DonnéesQuébec, notre attention s'est attardée sur les rapports relatifs aux accidents de la circulation, et quel effroi quand nous avons, après plusieurs recherches, appris qu'en 2016 le nombre de décès liés aux accidents de la route a augmenté de 12% par rapport à l'année 2015. Une envie de changer les choses est alors née en nous, et c'est ainsi que "Ta Route" naquît.

What it does

Ta Route est un outil de recherche d'itinéraires selon un niveau de risque relatif à la fréquence d'accident dans un secteur donné. Le trajet le plus sécuritaire est proposé à l'utilisateur en contournant les zones à risque.

How we built it

L'application est bâtie sur une architecture multitier et multiplateforme, les données sont toutes gérées dans une couche de données.

Le front-end est composé d'un mélange d'HTML/CSS et de JavaScript, le back-end repose entièrement sur JavaScript avec l'utilisation de Node.Js et d'Express.Js pour assurer la partie serveur et communications HTTP.

Nous avons développé un Web API RESTful OData4 pour centraliser toutes les données (provenance données Québec et Crowdsourcing de notre application). La raison de ceci est la facilité d'accès aux données ainsi que toutes les options qu'OData nous offre. Un autre avantage d'utiliser un API C# est le caching, ce qui nous donne une latitude de performance quand on touche à de plus gros fichiers.

Challenges we ran into

Le fichier d'accidents ne comporte pas de point géographique, mais seulement des emplacements, donc nous avons du géolocaliser les accidents possédant des numéros ainsi que des noms de rue.

Accomplishments that we're proud of

Toutes les données recueillies sont accessibles dans l'API, donc nous gardons le concept de données ouvertes, en temps réel, facilement consultable par les municipalités, en format JSON.

What we learned

Nous sommes une équipe formée sur place, donc nous avons dû orienter notre fin de semaine selon les compétences et les connaissances de chacun.

What's next for Ta Route

Ta Route a pour objectif de devenir une plateforme de crowdsourcing permettant aux gouvernements et société de transport d'accéder aux statistiques d'utilisation des transport des citoyens.
