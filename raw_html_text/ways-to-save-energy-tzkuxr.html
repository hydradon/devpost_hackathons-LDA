I was inspired to participate in this hackathon because I wanted to find a way to use the Alexa skills to help others. This hackathon allowed me to do that. I ended up making my Alexa skill related to ways to save energy around your home. When you say, "Alexa, open saving energy" it tells you a way to save energy. I built it using the Alexa developer console. When I submitted my skill the first time, my invocation name was, "how to save energy" so after submitting it the second time, I changed my invocation name to, "saving energy." I'm proud that I was able to create an Alexa skill that could help others. I would like to add more facts and ways to save energy to this skill. I really enjoyed creating this Alexa skill and I'm glad that I could help others who are using it.
