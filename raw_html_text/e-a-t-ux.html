Inspiration: We teamed up to build something creative, fun, and we want to offer a great experience to families in need as part of the National School Lunch Program.

What it does: This form fills out everything you need electronically, without having to write everything by hand, and we can skip the snail mail method altogether.

How we built it: We built it using custom, from scratch HTML, CSS, and a few Javascript plugins under the MIT License.

Challenges we ran into: We spent hours every day planning ahead, researching what kinds of user experiences people are looking for, and we wanted to build something that would be easily accessible for everyone. We built something that would work great on both desktop and mobile platforms.
