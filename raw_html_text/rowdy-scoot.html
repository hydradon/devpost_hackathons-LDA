Inspiration

There many different e-scooter companies active in San Antonio, each with a different proprietary app. Rowdy Scoot aims to aggregate all available scooters into one map view to make it easier to locate one.

What it does

Currently shows Bird and Lime scooters. Allows the user to get directions comparing the speed of walking to a destination versus detouring to grab a scooter for the trip.

How we built it

Javascript, Mapbox & Mapbox Directions API, Domain.com

Challenges we ran into

Bird and Lime do not provide public API to retrieve scooter locations. We had to rely on reverse engineered documentation for this. We would need to seek permission to use these APIs before moving further with this app.

Accomplishments that we're proud of

We got it working enough to serve a practical purpose

What I learned

Hackathons are pretty cool but I think we're all about to pass out from exhaustion

What's next for Rowdy Scoot

Needs support for more scooter brands and a whole lot of polish
