• PROFect Match for perfect PROF

Inspiration

The FCE on CMU website is not as user-friendly as we expect, and we all have a need to choose our favorite professors.

What it does

The user inputs a course number and our website outputs a chart showing professors' ratings and the weekly workloads.

How I built it

We used javascript and Chart.js.

Challenges I ran into

We had to learn Javascript as a new language.

Accomplishments that I'm proud of

We successfully created a product and we learned a lot of new cool stuffs.

What I learned

We learned the MCV model for iOS apps, basic concepts of Swift and Firebase, and how to create a webpage using Javascript.

What's next for PROFect Match

More data involved; better visualization; different types of graphs; go public.
