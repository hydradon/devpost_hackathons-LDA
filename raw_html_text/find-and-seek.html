Inspiration

The YOLO9000

What it does

Puts bounding boxes around objects that the YOLO9000 model supports.

How we built it

  • Built off of sample project on Forge https://github.com/hollance/Forge

Challenges we ran into

  • Didn't actually start this hack until 4 hours until submission time because we realized last minute that the other idea we had wouldn't work. Would have been able to do way more if we even had a few more extra hours

What we learned

  • How the YOLO 9000 model was trained.
  • iOS's Neural Network framework

What's next for Find and Seek

  • Some next steps would be to be able to search for a specific object in that category (e.g. searching for a particular person given an image of their face or searching for a particular water bottle.
