What inspired your app?

What if you could have an app that gave you the shortest path between 2 places in NUS.

What does it do?

The app uses the google map API along with data provided by us to allow for indoor routes

How is it built?

We used Python for the backend and HTML, CSS, JavaScript and AngularJS for the frontend

What were the challenges faced?

Due to lack of time, we were only able to implement the app for a small portion within NUS(the area in an around SoC)

What did you learn?

We learned how to use the Google Maps API, and also developed familiarity with AngularJS and Python as a back end language

What's next for FindNUS

Hopefully we will be able to improve upon the project in the future
