Inspiration

Every time, when I was in need to time in some other country I was getting the help of Google Search. I know it is simple but each time it is time consuming. So I was thinking of a way to get it inside UiPath

What it does

It displays the current time in all the countries

How I built it

With the help of timezoneinfo and GetTimeZones I developed this workflow.

Challenges I ran into

Selecting proper TypeArgument for ForEach
