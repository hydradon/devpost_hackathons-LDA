What it does

Users can input song names and singers and it will print lyrics or users can enter lyrics and find songs where these lyrics have been used. Also has connection with Spotify and Pandora.

How we built it

Used an API that allowed us to get method and a database filled with songs. Create a java file and then used node.js to connect the front end html, css, javascript with the backend java.

Challenges we ran into

Printing the output of java program and using node.js to connect front end and back end.
