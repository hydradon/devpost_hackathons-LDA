Inspiration

I want to make a chat bot just for fun.

What it does

When you submit a python code snippet in a Slack channel, the bot will take the file from Slack and do a static code analyzing with pylint. It will then return a static analysis report of your code.

How I built it

I built it using python-rtmbot, a bot engine made by Slack and pylint for code analysis.

Challenges I ran into

A lot of the challenges with authentication.

Accomplishments that I'm proud of

I successfully built my first chat bot.

What I learned

How to build a chat bot and deal with all the authentication stuff.

What's next for PythonBot

Add more functionalities. My initial idea is to recite poems based on Python code, but that will be for the future.
