Inspiration

We wanted to build a platform that gave professors the ability to receive live feedback from the students as they are teaching.

What it does

Students are able to rate their professor's progress as they go through the course in real time. With the ability to collaboratively chat with each other and share notes. You can view past reviews from students who have taken the class with a professor as well as see the notes that they've uploaded.

How I built it

Hosted on an Linode VPS, using nginx+php5-fpm stack, the backend DB is Clusterpoint. The framework is Laravel. Chat/collaboration functionality is provided by Moxtra.

Challenges I ran into

Working together as a team and only having one production server to work off of.

Accomplishments that I'm proud of

Getting the chat functionality working as well as being able to use the Clusterpoint DBAAS.

What I learned

There are a lot of cool ideas we had in mind, it was hard to pick out which features we could realistically create within the 36 hours we had. It helps to understand the general basic features that is required to get a MVP out and work from there.

What's next for KoClass

There are so many more features we have yet to add, we only wish we had more time to work on it. Study guides and tests upload ability as well as class recordings are some of the features we were looking to add.
