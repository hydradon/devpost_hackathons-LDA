Inspiration

Our passion for travel and our desire to make planning a trip an easier experience for everyone.

What it does

Travel Planner allows users to star places which are interesting to them, and later create an itinerary from the starred list.

How we built it

We used Html, Css, JavaScript, Adobe Muse

Challenges we ran into

One of the biggest challenges was trying to implement the JavaScript correctly to get google maps API.

Accomplishments that we're proud of

We are proud of making a site that people would be interested in using in order to plan out destinations they want to visit

What we learned

We learned how to use JavaScript and retrieving the API data from google maps

What's next for Travel Planner

The next step for Travel Planner is to sharpen the functionality of the website
