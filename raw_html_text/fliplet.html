Inspiration

we want to create a application to help student improve the academic goals

What it does

Our Application offers interactive flashcards that will dynamically flip over to reveal answers. This will help students learn in a new and engaging way.

How we built it

We built it using Reactjs for our frontend. We try to store it on our data on Google FireBase database but ran in to issues, so we use and array objects.

Challenges we ran into

We ran into set up a configuration to our Database. We tried to fetch data from our Firebase DB but it would only return the response body with no data. We ended up getting our account temporarily shut down due to the large volume of requests we were sending to the Database. Also learning CSS from scratch is harder than expected. All the elements a while to align. We still have an issue with up scaling and down scaling the screen size. We also had a hard time learning the event of javascript. no one had any idea what is happening and why is it breaking because we all decided to learn a new language that no one knew how to do from scratch.

Accomplishments that we're proud of

We are proud of building our first running application from scratch while learning a new language and a new frame work of java script. Our team had excellent communication skills on throughout the entire project.

What we learned

Reactjs, nodejs css,.

What's next for Fliplet

We plan on adding new categories for different subjects. We also plan on allowing users to add images to further increase engagement and also adding a Backend database to handle storage.
