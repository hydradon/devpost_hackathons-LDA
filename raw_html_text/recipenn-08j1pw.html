How we built it

We created a Google Action on Google Assistant and built an Android app to input items into the ingredients list and to get recipe suggestions. The back end was built with Node.js, Express.js and MongoDB.

Challenges we ran into

We mainly struggled to come up with a good idea, going back and forth from complex ideas to simpler ones, but in the end we chose a project that we thought was doable in the leftover time that we had while still being challenging.
