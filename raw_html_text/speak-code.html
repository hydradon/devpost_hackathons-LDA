Inspiration

Coding is for everyone from differently-abled people to amateurs to business leaders.

What it does

The idea to give everyone an opportunity to dive into the programming field. Presenting ease for differently abled people to code by speaking through a microphone, it also adds the fun element to coding by enabling even amateurs to interact with the computer in a less daunting manner.

How we built it

We leveraged existing emerging Google Cloud API to convert speech to text and built the Java program. We further extended the scope of the application by applying syntax and semantic analysis using Stanford CoreNLP.

Challenges we ran into

Integrating the API and the CoreNLP framework into the application.

Accomplishments that we're proud of

In spite of challenges with time constraints and integrating new technology, we pulled through to deliver on our ideas and principle that EVERYONE :-) can code.

What we learned

Team Work, Women empowerment and different technology stack.

What's next for Speak Code

Integrating NLP on a deeper level by developing gazetteers to train the application in a better way. We plan to explore other platforms similar to Dialog Flow.
