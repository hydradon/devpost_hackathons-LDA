MAL_updater

Updates your MyAnimeList Currently Watching list from a popup/extension

I created this chrome extension for personal use and decided to throw it up on github.

I got inspiriation from blitzprog's anime release notifier and from "anime tracker" on the chrome web store.

I do not own the MyAnimeList logo. The myanimelist logo and MAL logo are property of MyAnimeList, LLC.

I do not intend to make any money off of this extension.

This is not an official chrome extension of myanimelist.net.

Thanks to dagwaging for teaching me xmlhttprequests.

All the user has to do is input their myanimelist.net username and password from the options page. The extension saves this data in the browser's local storage and uses it to retrieve their currently watching list. They can increment or decrement the episode counter for each anime they are watching. They can also input a specific number for the anime.
