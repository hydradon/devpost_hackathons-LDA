Inspiration

Built out of laziness since editing the details one by one was a pain

What it does

Can edit details of audio files such as album and artist names and can also play music

How I built it

We built this program in java

Challenges I ran into

We had to learn how to work with audio files, a new library, as well as how to use github and how to seamlessly combine our team member's efforts

Accomplishments that I'm proud of

We are proud of being able to learn so much about the new classes we discovered throughout the project

What I learned

We learned how to use github, as well as how to better integrate a library to our program. We learned about MediaPlayers, and other new classes we had not known about yet.

What's next for Harmony Music

We hope to add a playlist creator/editor, as well as add in a visual song queue that you can change as a user, as well as a shuffle mode.
