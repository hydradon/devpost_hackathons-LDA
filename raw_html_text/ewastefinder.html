Inspiration

Technology in landfills secretes harmful chemicals into the planet. E-recycling is the only way to properly dispose of electronics, but there aren't many applications for this purpose.

What it does

Takes a user's location and presents electronic disposal centers near the user.

How we built it

Using Java in Eclipse IDE.

Challenges we ran into

The Andriod IDE wasn't properly configured onto the laptops we used, and prevented us from making progress on a mobile application or with Maps API.

Accomplishments that we're proud of

Creating a database ourselves. Finishing the program despite the obstacles encountered.

What we learned

How to set up a server with Mysql

What's next for EWasteFinder

To develop a mobile application, a nicer framework, visual representation of locations, and allow businesses to easily include themselves in the database.
