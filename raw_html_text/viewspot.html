Inspiration

We loved to travel with our families and want the best possible vacation experience.

What it does

It helps people find a customized vacation that they will not regret having and allows them to directly book their vacation.

How we built it

We used android studio to develop the app and add colorful pictures and graphics.

Challenges we ran into

We wanted to create a map to show the attractions at the beginning but the map api would not work for us so we redid our plan for the project and did not use it since javascript was really tough.

Accomplishments that we're proud of

We learned a lot about android studio and we did not know how to program very well, and we were able to develop an app despite not knowing much, thanks to the help of the mentors.

What we learned

How to use android studio, integrate pictures, texts, buttons, and code the java behind it

What's next for ViewSpot

We would love to create the map and tripadvisor url for each destination.
