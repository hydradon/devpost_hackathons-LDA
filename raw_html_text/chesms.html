What it does

Lets you play chess with friends using SMS.

How we built it

Put together Twilio and a chess library.

What's next for CheSMS

Playing some chess.

How to use it:

Text +12065677559 "Hi" and follow the instructions.

Commands:

  • move <move> - move to a position. Ex: move e4
  • moves - list all possible moves
  • undo - undoes the last move
  • anything else redisplays the board
