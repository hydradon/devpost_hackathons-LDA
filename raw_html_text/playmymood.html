Inspiration

People tend to want to listen to songs that match their current mood, and we wanted to make it easier for people to do so.

What it does

The web app basically gives you the option of uploading a picture, detects the mood and gives the user a Spotify playlist based on their mood

How we built it

We used Microsoft Azure facial recognition API and Spotify API and integrated them on a basic website using 'glitch' platform.

Challenges we ran into

Using these APIs for the first time, creating a secure connection to spotify accounts.

Accomplishments that we're proud of

We have a working prototype

What we learned

How to use an API, glitch and a better understanding of Javascript.

What's next for PlayMyMood

Improving the UX design and better algorithm for song emotion and algorithm that relies on rating system to measure our performance. For the future of this project, we intend to make it possible for users to be able to Login using their spotify details, which could help us manipulate their personal playlist based on their mood too
