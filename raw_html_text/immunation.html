Inspiration

They say prevention is the best medicine. And vaccines are our best line of defense against common diseases such as the flu. However, due to misinformation, people do not avail themselves of this medicine, and many die every year.

What it does

Our app seeks to make vaccines more accessible. We want to inform patients about the vaccinations that they should get, and where they can get it.

How we built it

We built the app using Node.JS and Javascript for the backend. Our database uses Mongodb. Our frontend is bootstrap, HTML and CSS. We used github for version control and Heroku for deployment. We also used NYC open data API to get information.

Challenges we ran into

We had trouble connecting to the database. But we got there in the end.

Accomplishments that we're proud of

We are proud that we were able to create a prototype for an app that we think can benefit a lot of people.

What we learned

We learned a lot about hosting an app on Heroku. We learned the importance of teamwork and communication.

What's next for immuNATION

The sky is the limit for immuNATION. With more data we can expand the app and give users more information. We can also connect with healthcare providers like pharmacists and doctors.
