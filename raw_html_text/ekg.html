Inspiration

We were inspired to create a more cost-effective heart monitoring system after seeing the costs of everyday hospital equipment.

How it works

We used an Olimex EKG shield for the Arduino to filter the serial input to the computer. We read in the serial input using a chrome extension and graphed the data points using CanvasJS.

Challenges I ran into

HTML formatting

Accomplishments that I'm proud of

Programmatically identified all major points of a standard electrocardiograph waveform
