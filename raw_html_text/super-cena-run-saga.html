Our inspiration for our game was Geometry Dash and the No-Internet Game.

It is a 2d game that continuously runs and John Cena must avoid Randy Ortein by jumping.

We built this game using the Git Hub website and help from Isiah the StuyHacks mentor.

Some challenges we ran into when creating our game were placement and movement of objects.

We are proud that we actually created a working website that we were able to modify being beginners.

What is next for Super Cena Saga is for the project to be completed.
