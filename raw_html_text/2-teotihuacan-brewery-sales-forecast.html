Inspiration

Machine learning is one of the fastest advancing and most important technologies today. We really wanted to learn more about it.

What it does

Uses machine learning regression to forecast three months in advance of sales with diferent factors involved.

How we built it

We decided to use Machine Learning .NET Core for this because it allowed us to train a model quickly and also connect it with the web smoothly.

Challenges we ran into

We started out using Python but had trouble understanding main libraries like TensorFlow.

Accomplishments that we're proud of

We were not really much into machine learning before these 24 hours. We learned a lot and overcame a lot of issues.

What we learned

We learned Python and its powerful libraries but also to use what's available for us in the current situations.

What's next for #2: Teotihuacan Brewery Sales Forecast

Definitely a lot of scaling can be done thanks to .NET Core. Upgrading it to a web service, website, desktop application or even mobile with Xamarin will be amazingly smooth.
