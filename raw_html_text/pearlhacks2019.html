PearlHacks2019

Inspirational Card Matching Game for Women in STEM

Programming for this game was done by Jessi Whiteside . All fun facts were researched by Hayley Rose as well as all images gathered and edited.

This was the first time I had ever touched web development, html, java, or css, but I love a challenge!

We wanted to create something fun but also empowering to women, and we know that there are a lot of women in the history of STEM that people don't know about! So, we decided to design a game that could teach the user more about these inspiring women!

We hope that you enjoy our game and maybe learn something about the women who inspired it! We hope to become the technological inspirations that these women were someday!
