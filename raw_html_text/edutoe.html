Inspiration

To make learning fun with Tic Tac Toe for Kids.

What it does

The app allows kids to play tic tac toe and learn at the same time. In order to choose a particular position in the 3x3 matrix, kids need to play mini-games to get their mark on the matrix sheet.

How we built it

We built it using the Android-Studio IDE including languages like java and xml in it.

Challenges we ran into

It was challenging to link each position of the 3x3 matrix boxes to a new mini game and come back to the same page with updated scores for each player.

Accomplishments that we're proud of

We are proud that we were able to think of such a unique and intellectual idea that would improve the learning for kids and were able to create our Tic Tac Toe game using Android Studio for the first time.

What we learned

We learned a lot about how to create an app using Android Studio and also prepared the back-end mini games that would be linked to the 3x3 matrix of the Tic Tac Toe.

What's next for EduToe

I think it needs to be completed as it is a great idea that would make learning for kids fun and entertaining. Once, the matrices are linked with each intellectually challenging mini game, it would improve their productivity of learning.
