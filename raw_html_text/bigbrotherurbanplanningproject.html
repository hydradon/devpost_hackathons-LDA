CHALLENGE: IDENTIFY STREET INFRASTRUCTURE

  1. Our team targets the problem of generating a database for that contains the infrastructure along the streets.
  2. This will hopefully generate a data frame that can help StreetCaster program in Boston.
  3. Providing a data set on the location of the infrastructure and what it is.

APPROACH

  1. The coordinate of the beginning and the end of the street is provided by the user.
  2. The framework will interpolate between the provided geo-coordinates. In order to ensure that the generated points will lie on the road, the Google Maps API will be utilized.
  3. For each of the new coordinate, images of the street will be obtained via Google Street View API. At each point, a set of four photos is fetched to ensure a 360-degree field-of-view.
  4. The photos will be uploaded and processed on Google Cloud Vision API to identify the objects inside each image.
