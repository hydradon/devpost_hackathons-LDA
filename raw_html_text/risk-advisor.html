Inspiration

Provide education and create awareness among common user in India for various risk occurs in someone's life.

What it does

It gather basic demographic details from user and calculate risk factors for different categories. Based on user input and calculated risk factors it filters available market product available in life insurance. User can compare trade-off between filtered products and can choose best products suits to his need. As part of hackathon it only include life insurance risk advisor.

Demo: http://step-risk-advisor.mybluemix.net/ Code : https://github.com/talCrafts/risk-advisor

How we built it

Application is built up using Polymer and Spring boot framework. IBM bluemix trade-off analytics service is used to provide trade-off between various life insurance products.

Challenges we ran into

We faces issues integrating spring boot and tradeoff analytics. We tried watson-sdk services but faced issues with watson api which were modifying the request. Enabling trade-off servlet also conflicted with other spring boot controller.

Setting up basic UI with polymer to start initial integration.

Accomplishments that we're proud of

We were able to put up a POC with polymer, spring boot and bluemix tradeoff-analytics. We were able to put up our design of helping common user to advise life insurance projects.

What we learned

  • Learnt lot about IBM bluemix watson services.
  • How to work as a team in hackathon.

What's next for risk-advisor
