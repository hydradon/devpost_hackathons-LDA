Be bomberman. Aaron and I were trying to come up with a project idea and we went with virtual reality. Some would call trying something completely new to you for a hackathon a dumb idea, but we chose to do this project because we were excited to learn something new. Coding with VR has been a fun and educational experience. When using the proper controls with this game you feel immersed. Around every corner you have to be expecting an enemy. You constantly glance behind you. You try to statistically predict enemy paths. You try to memorize where you placed bombs. You try to memory paths to safe spaces. It is all the tedious attention demanding fun of the bomber game series, except you are literally put in bomberman's shoes without an eagle view. As you hunt in this maze, little do you know that you are being hunted. Vivebomer.

We came up with the idea and everything then a couple hours ago we decided to check youtube to see if its been done. I say we did it better. Look forward to our live demo!
