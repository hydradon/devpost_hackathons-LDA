Inspiration

Have already seen similar addins for other software and found an opportunity to create one for Fusion360

What it does

Application creates bill of materials with part thumbnail as a html file. It`s opened with default internet browser so the user can print it directly or save as a pdf file.

How I built it

C++, Fusion API, a few html words

Challenges I ran into

Screenshooting each unique part, assemlby traversal and adjusting proper image and table sizes

Accomplishments that I'm proud of

simplicity of BOM file creation and printing

What I learned

Fusion API

What's next for Thumbnail BOM

Maybe adding some other output formats like .doc
