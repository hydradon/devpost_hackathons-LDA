What it does

In this multiplayer game you have to customize your spaceship and challenge your friends in x4 multiplayer on the same screen. Each user have to use it's android device as joystick. All the players have to be connected to the same network.

How I built it

It is built with unity3d (game) and android studio (joystick)

Challenges I ran into

The challenge is to create console game experience on PC (Pc as console and android devices as joysticks with touch screen an possibility to save your data)
