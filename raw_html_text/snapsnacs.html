Inspiration

It's very annoying to record everything we eat by entering the name of the food or entering data one by one

What it does

Use socially-interactive way to motivate users to keep track of what they eat; Make calorie consumption easy to record; Humorously remind users of health diets

How we built it

For the backend, we used python and Flask to send the photo to the google cloud platform for processing, and to the food database for more nutrition info about the food.

Challenges I ran into

The most challenging part of this project is iOS development because we are all new to this area.

Accomplishments that I'm proud of

We made a successful backend

What I learned

Flask, Google cloud platform, swift, json

What's next for SnapSnacs

We will try to analyze more food and learn more about machine learning. We expect to enable it to take one photo per meal (of all foods) and detect every food and search for them all at once. We can also update the design of current interfaces and include new interfaces such as monthly calorie track, snacks recommendation and junk food-healthy food conversion to make it more user-friendly
