Through the research provided by My Best Helper team, finding out that their clients were using their product mostly through the website instead of with the mobile app, we tried to improve the user experience and user interaction by creating a responsive, easy-to-use website that has the look and feel of a mobile app, to make the user experience something more enjoyable and fun to use.

Instructions for running the project are in the readme file of the project on bitbucket.
