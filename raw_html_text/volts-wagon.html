What we built

We built a lightweight web application using ReactJS that allows a user to identify potential revenue, problems, and issues that may arise with electrical vehicle charging stations across Hawai'i. We needed to keep in mind of a friendly and user to use user experience. With material design, we were able to incorporate modern design practices while using libraries such as ApexCharts and D3.js to help model and predict our data.

To start the project locally : 

git clone https://github.com/HACC2019/area-51-raid-squad.git
cd area-51-raid-squad
npm install
npm start
Open your browser to http://localhost:3000
