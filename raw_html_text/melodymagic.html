MelodyMagic

Hackathon Spring 2018, music generation

We created an algorithmic music generator for generating music and visualizations. This project creates a four-part never ending song (melody, harmony, bassline, and percussion) using python. Additionally, we developed a few modules that can connect to the core music generator:

  • Music Visualizer written with python and Blender
  • LED Matrix visualization using an ESP8266 embedded processor
  • Optical motion input using webcam and opencv
