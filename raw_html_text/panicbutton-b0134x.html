Inspiration

We wanted to make an app for college students which allows them to easily make calls and send text messages when they are in dangerous areas

What it does

It uses andorid NFC and on a single click it sends out a mass text message to all the people in the contacts you designate in app, and on double tap will call your main emergency contact which is set in app

How we built it

We built it using the twilio api to send out mass text messages, and then used android studio to develop the NFC interactions

Challenges we ran into

Adding the permissions to the android app and setting up the sql server

Accomplishments that we're proud of

We were able to get sql integration, use the twilio api, and have NFC functionality

What we learned

We learned how to work as a team, and split off roles in a way that was productive, and allowed us to make multiple parts of the program simultaneously

What's next for PanicButton

We plan to improve the UI and UX experience
