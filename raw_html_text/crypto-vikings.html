Inspiration

Decentralize and eliminate intermediaries in the world of music

What it does

DApp to buy and play music in a decentralized ecosystem

How we built it

Architecture

Frontend:

Angular 7.1.0

Material Design 3.0.1

Backend:

Solidity

Libraries

uPort (login users)

Swarm (deployment dapp resources: songs and pictures)

Deployment

Frontend: Heroku

Backend: Ethereum

Smart contract: https://ropsten.etherscan.io/address/0xc236ba65a0ed1420bd6513f1964c8e76d0236ca2

Challenges we ran into

Swarm is good for deploy resources like pictures and sounds but it's very difficult to deploy angular front app. We have to use Heroku.

Problem with Blockchain slow

We have to use cache for use the resources in swarm (pictures and songs)

Accomplishments that we're proud of

Use Ethereum like backend Use Swarm like storage open source

What we learned

About Angular, Ethereum, uPort, Swarm and IPFS
