Inspiration

When you see a lost animal on the street and have no simple way to report it.

What it does

An app which helps you report lost animals or see a list of found animals to and from the SPCA.

How we built it

Using the Microsoft Bot Framework. Please use the Bot emulator to run.

Challenges we ran into

Going through the documentation for microsoft bot framework.

Accomplishments that we're proud of

Learning how to work stronger as a team through Git.

What we learned

The importance of teamwork when going through documentation and sharing our knowledge with each other to save time.
