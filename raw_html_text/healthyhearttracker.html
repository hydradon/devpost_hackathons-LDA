Inspiration

Easily interpretive tool to check lifestyle factors leading to risk condition before you can see your doctor!

What it does

Help users to assess relevant heart attack risk and track down the factors leading to it by answering questions related to their weight, exercise habits, alcohol use, etc.

How I built it

We used Python and R for data pulling, parsing and processing. For assessing relevant heart attack risk we trained a logistic regression model. We use d3 , GitHub Pages , CSS and javascript to implement the dashboard.

Challenges I ran into

  • Data Integrity Issues
  • Plugging train model to the dashboard for interactive predictions.

What's next for

Generalize the model to assess the relevant risk associated with various health conditions Develop advanced version of the dashboard
