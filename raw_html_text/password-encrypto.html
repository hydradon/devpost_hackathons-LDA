Inspiration

As i was going through college application, I had to apply many colleges, create accounts and make unique passwords for them. Every time I visited those site again, I used to forgot the password.

What it does

It stores all passwords, user names and site information securely into a image and you use only one master key to access all data.

How we built it

We used crypto steganography to encrypt and decrypt password into image, we use MySQL to enter site information and usernames while using tkinter to interact and provide better interface to users.

Challenges we ran into

This was our first hackathon and we had never been into such events. While building our project, we only had general idea about python programming. During event, we figured out and ran into different python libraries like numpy, pandas, crypto steganography, flask, mysql, tkinter etc. and that's us using those for the first time.

Accomplishments that we're proud of

We had two college freshmen who are over from ULM and NEPAL , one from UTA doing masters in CS from INDIA, we had master graduate female programmer from texas, US. So it was various diverse ideas to cope up with. we put our different information, with different backgrounds and experience we finally figured out fully functioning program.That's what we are proud of.

What we learned

We learned various python libraries, especially connecting front end and back end accessing database.

What's next for Password encrypto

We see its prospect on daily life and on commercial use . We can even expand it to store thousands of passwords and use biometric as master_key. We can host this software easily over windows application, WEB-APP, mobile application etc.
