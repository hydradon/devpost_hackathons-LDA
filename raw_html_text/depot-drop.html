Inspiration

I came up with this idea because I wanted to include a home depot bucket in a game.

What it does

You get to play as a Home Depot bucket and collect Nintendo Switches that fall from the sky. It works using only a desktop and you use the keyboard to navigate the menu and move the bucket left to right.

How I built it

I built it using the lidgdx game development framework. I used Java and developed it using Android Studio.

Challenges I ran into

I ran into challenges trying to design the user interface for the game and getting the images the way I wanted them.

Accomplishments that I'm proud of

I was able to incorporate keeping track of score and enforcing a "5 misses you are out" policy.

What I learned

I learned how to build a simple game using libgdx. This weekend was my first time using the framework.

What's next for Depot Drop

I look to add more features to Depot Drop
