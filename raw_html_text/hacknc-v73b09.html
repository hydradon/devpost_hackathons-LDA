Star Catcher

This project includes Home page with slider images, Product page that allows the user to filter, Footer with social media links and contact form, Backend API server and database server, Responsivepages for various device types and Accessibility and security. a MongoDB database is designed and deployed on desktops to accommodate the ecommerce database.
