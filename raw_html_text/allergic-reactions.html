Inspiration

Our inspiration for this project came from common misdiagnosis of allergic reactions.

What it does

Our skill helps identify identity different types of allergic reactions and how to prevent them from happening in the first place.

How we built it

We built it using Amazon Web Services Blueprint.

Challenges we ran into

Some challenges we ran into included testing the skill in our virtual Alexa machine.

Accomplishments that we're proud of

We are proud that we were able to create our own skill from scratch.

What we learned

We learned important medical facts about allergic reactions along with learning how to create our own Alexa skill.

What's next for Allergic Reactions

We plan to release more skills with more content related to allergic reactions.
