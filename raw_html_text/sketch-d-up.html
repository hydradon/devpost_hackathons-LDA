Inspiration

Those with creative minds who do not possess the technical skills to design mobile apps can now express their ideas through art and design

What it does

Uses the YOLO computer vision algorithm to look at an app layout and builds a cross-platform flutter app in real time.

How we built it

We created 2000 sample drawings to trains a model to recognize different components. We stored information in firebase about each component so the Yolo algorithm would then communicate with it and then used Dart.
