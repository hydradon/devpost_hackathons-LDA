Inspiration

I've always had a hard time choosing gifts. This app aims to reduce that stress

What it does

It (is supposed to) takes a users preferences into account and matches them with products

How I built it

We used python/flask to create a basic web app and implemented a SQL database to store the information. Bootstrap helped with the styling. We used torch to help perform semantic comparisons and match up attributes

Challenges I ran into

Accomplishments that I'm proud of

We learned how machine learning can really help us with our projects and we implemented it
