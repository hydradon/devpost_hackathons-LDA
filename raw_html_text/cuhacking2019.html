cuHacking2019

This is a file sorting and managing project for cuHacking. This project was created by Dhruva Rajkumar, Kareem El Assad, Isaac Leung, and Mathew Siu.

The program prompts the user with a GUI which allows them to input several keywords. Those keywords are then used to parse and sort a set of files assigned by the user. Those files are then placed into a specified directory.

Developed in Python.
