Inspiration

Our team was inspired by the good that not only video games can do for others but videos can do for ourselves. We came up with this app in order to further illustrate that augmented and virtual reality can be used to serve a greater good be on the screen that virtual reality and augmented reality may be used for social good.

What it does

You buddy is an application that creates an augmented reality partner. The user would interact with the digital partner and spend time laughing and loving life.

How we built it

We built it with unity3d and Microsoft hololens.

Challenges we ran into

So the challenges we ran into was how to use the API ran to Hardware Mount functions and Internet outages but despite this we still proceeded.

Accomplishments that we're proud of

We're proud of finally being able to utilize Microsoft API in Unity's game engine.

What we learned

We learned how to implement api's into unity game engine and we hope to be able to do more with this in the future.

What's next for Yo Buddy

We hope to privately continue to work on your body and hopefully be able to demonstrate how augmented reality may be able to serve children.
