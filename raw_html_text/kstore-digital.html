Inspiration

The KStore is a vital part of the theatre department at Krannert Center for the Performing Arts. It has an extensive inventory that all of the sub-departments can purchase from for theatre productions. This project was inspired by real-life difficulties navigating the inventory.

What it does

It allows users to search the KStore inventory, keep track of their budget, and add daily receipts as they make purchases for shows. Previously, the only way to go through the KStore inventory was by scrolling through an unsearchable extensive PDF by hand.

How we built it

We built a prototype with Sketch and began the process of converting it to a usable web app made with Python and Flask.

Challenges we ran into

We quickly realized that the scope of the project would take vastly more time than we had available! We discovered the importance of creating an organized plan while figuring out what we could feasibly accomplish.

Accomplishments that we're proud of

We both tackled things that we had never tried before! Christina had used Flask and Bootstrap a little bit before this project, but not to this extent. Eleni had never coded in Python before. Neither of us had used Sketch before and we both ended up liking it a lot!
