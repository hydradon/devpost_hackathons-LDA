The main inspiration was a parkour game with blocks, and we decided to go further by adding realistic nature-like features

You control a block which jumps with the space bar from platform to platform, and moves using the arrow keys

We used Unity, which required the use of C

Nobody in our group knew C# and we didn't know that Unity needed it, so we had to learn it on the spot

We are proud that we were able to create what we made with such little knowledge of C#. Also, this was the first time any of us had used Unity, so it was difficult to navigate around screens.

We learned a lot more about Unity and we learned how to code in C

We will try to increase the smoothness of the app and increase the sizes to make it more interesting.
