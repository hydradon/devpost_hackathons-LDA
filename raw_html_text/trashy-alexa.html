Inspiration

We wanted to make an app for alexa that would tell you whether or not you could recycle something. However, this already existed, so we gave alexa a bit of attitude.

What it does

Assists user in what can or cannot recycled based off the troy area.

How we built it

We used they alexa development tool kit and aws lambda functions.

Challenges we ran into

None of us had created an alexa app before, we were having difficulties to alexa, and we had initial problems with the set up.

Accomplishments that we're proud of

We have a working project that serves as a baseline product.

What we learned

We learned how to create custom skills for alexa and how trouble shoot with the alexa developer tool kit.

What's next for Trashy Alexa

It is currently just for Troy, but we would like to expand it to include more areas.
