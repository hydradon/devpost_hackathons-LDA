Inspiration

Many hours spent browsing the depths of YouTube. Why couldn't we browse more efficiently?

How it works

We query YouTube's Data API for videos relevant to a category or search phrase the user is interested in and randomly select a few videos from the results.

Challenges I ran into

Depreciated APIs which provided useful functionality

Accomplishments that I'm proud of

  • Great user interface
  • Ease of use for new users

What I learned

Usefulness of jQuery Shockwave Flash is an age old relic

What's next for TubeWalk

We are aiming to allow users to select one of the four videos to guide their path through YouTube and bring them to content they care about faster.
