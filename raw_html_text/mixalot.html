Inspiration

Have you ever needed to kick back and relax after a long day at work? Have you ever been at a social gathering and just not known what drink you want to make. Mixalot is the app to help you with all of your cocktail mixing needs.

What it does

Mixalot works with a Cocktail Database API that allows users to search for ingredients, glasses, drinks, liquors, mixers and various cocktails as well as contribute to the community database to allow for further expansion (Coming Soon!)

How we built it

We built it utilizing Xcode and Swift 4.

Challenges we ran into

We ran into several challenges such as making sure our UI was handled asynchronously and loaded in a timely fashion.

Accomplishments that we're proud of

We are very proud that we were able to interface successfully with an API and present it in a useful manner in under 24 hours.

What we learned

We learned a lot about Firebase by Google, accessing API via JSON and PHP, as well as the inner workings of Swift and Xcode.

What's next for Mixalot

We have big plans to expand the community crowd sourcing aspect as well as uploading to the App Store later this week!
