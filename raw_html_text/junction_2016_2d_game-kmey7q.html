Depending in the element mode (fire, earth, water, air) you are, you have to catch blocks scrolling down from the top of the screen. Every kind of block has a specific effect. Earn tokens, unlock characters, and switch from a mode to another suddenly.

Cross platforms app developed with Android Studio, using Java combined with the library libGDX.

Game Jam sponsored by Supercell

Completely from scratch, no knowledge about developing game before coming here.

How to use LibGDX, 8bits graphics and we got to use a lot of logic.
