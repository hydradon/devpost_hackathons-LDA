Inspiration

The inspiration for this idea came from the video game "Fallout" and it's iconic device called the Pipboy. We thought of a way that we could make this idea feasable for use to display and record engineering based data.

What it does

It can take notes, and do everything a normal computer can do and more.

How I built it

we used pins, an arduino, a raspberry pi, and a lot of zip ties to build and create this device.

Challenges I ran into

The 2 meter cable was quite a challenge to work with, which we ended up scrapping also finding parts such as screws, getting sensors to run properly and having enough time were other roblems we ran into.

Accomplishments that I'm proud of

We made it work.

What I learned

We learned about how to program sensors and build a product in a very short period of time

What's next for Mr.Smith's PAC (Portable Arm Computer)

Further testing in classroom environments, to see if there could be any changes or improvements to make for it
