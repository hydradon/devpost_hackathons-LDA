Inspiration

Having mute relatives who require the assistance of sign language translators. Allows for the removal of the middleman.

What it does

We used ClarifAi's API to recognize pre-recorded hard gestures.

How we built it

ClarifAi's image recognition software

Challenges we ran into

Accomplishments that we're proud of

How to enable machine image recognition applications
