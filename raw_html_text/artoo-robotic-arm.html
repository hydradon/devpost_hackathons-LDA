Artoo is the first robotically assisted selfie-stick that posts pictures straight to Twitter. Control the selfie-stick with a bluetooth PS4 remote to take wild selfies anytime!

Using stepper motors, motor driver and an arduino micro-controller, this robot assisted selfie-stick can take pictures for you!

We had issues with the gears that were supposed to rotate the robot. Our supplies were not ideal and we didn't realize a bluetooth shield made some arduino pins ineffective.

We are proud that we were able to get the PS4 control to control Artoo. Doing so allowed us to control the robot and post selfies straight to twitter!

We learned how to interface multiple stepper motors to a single arduino in parallel with a bluetooth controller!

We plan on making an Artoo 2.0 that will feature feedback control and will have the precision to pick up and move chess pieces so it can be connected to a computer that will play us in chess! Computers are no longer limited to the cyber world with Artoo around!
