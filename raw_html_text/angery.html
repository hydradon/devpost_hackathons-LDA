Inspiration

Life can be very stressful. Often, we may feel angry at events beyond our control. Therefore, our team made this application to help users improve their tolerance and patience! (╯°□°）╯︵ ┻━┻

What it does

This web application is created to be incredibly annoying and triggering. Try out the application and play the game to find out why! By using this web application, users can improve their ability to tolerate unfortunate circumstances and manage their anger!

How we built it

We used React as the main frontend framework for the website. The game was made using Javascript, CSS, HTML and Canvas. We used integrated Firebase and MongoDB for our database. ༼ つ ◕_◕ ༽つ

Challenges we ran into

We ran into issues with API calls with CORS, and this cost us a lot of time. Not all of us were proficient at the technologies that were needed to make a web application and game.

Accomplishments that we're proud of

Despite the challenges, we were great at managing our anger and stress level. Some of our members also managed to pick up new technologies along the way.

What we learned

We became more proficient with Git workflow, React framework, Bootstrap, NoSQL databases and many other useful development tools. Also, we learnt how to stay calm and manage our anger.

What's next for Angery

We wish to build more annoying games to put on the website, so that users can have a variety of annoying games to choose from. We hope that this will help users control their anger better!
