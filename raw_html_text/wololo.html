Inspiration

The general public lacks tools for accessing open data in a simple and friendly way. 3rd parties can obtain a public interest in quality of life only through polls or public initiatives.

What it does

A user inputs an address and retrieves quality of life statistics about that place. The application uses user queries to build up a database of places of interest including the returned statistics which can be used to improve services in sectors that are of the highest concern to citizens.

How we built it

Microservice architecture deployed on Google Cloud Build (very new feature) using microservices written in Python and Golang. The frontend is implemented in Flutter. We used publicly available open datasets provided by Here, Mapakriminality.cz, OpenAQ. Our data are stored in BigQuery and analyzed by Google Datastudio.
