Inspiration

YP, one of our team members, suggested the idea. When he was little, he saw poetry as nothing more than a random concatenation of words. As he grew up, he realized that there is pattern in language that makes poetry beautiful. He suggested making a program that "auto complete" a poem based on the meaning and rhyme. We agreed that this is a beautiful project we are passionate about.

What it does

Predicts words that would likely fit inside the incomplete poem that users has inputted.

How we built it

We scraped the top 500 poems from Poem Hunter and analyzed the collection using frequency analysis and simple correlation. Specifically, we look at words that that likely to occur together in a poem

Challenges we ran into

Everything. For example, in order to process data and generate a csv file we need, Matthias ran his computer for half an hour before the program terminated in error--the program cannot encode the character "|". Publish an anthology using suggested words from the program. But before, we may expand the program to include grammatical considerations. We may also try using other input data to train the machine so that it may "suggest" word for input other than poetry.
