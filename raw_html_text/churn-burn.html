Inspiration

To provide better service to existing customers.

What it does

It's a web application that uses machine learning to predict the churn for T-Mobile customers. It also evaluates where the additional cellphone towers may be needed.

How we built it

React.js, Python, Microsoft Azure ML & Could, Node.js, sockit.io

Challenges we ran into

Uploading large dataset and populating them on a dynamic map.

Accomplishments that we're proud of

The web app ability to show data in real time.

What we learned

Using the Azure machine learning platform.

What's next for Churn-Burn

Implementation of live data feeds and QoS parameters.
