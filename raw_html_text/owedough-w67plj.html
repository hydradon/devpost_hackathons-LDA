Inspiration

Being out with friends has made us come to the realization that we need a minimalistic and efficient way to keep track of the money owed between individuals.

What it does

Our web application allows individuals to add debtors (people who owe you money) and creditors (people you owe money to). It enables users to delete from their dashboard once a debt is paid.

How we built it

We used Node.js for the backend and raw HTML, CSS, JavaScript/jQuery for the frontend. We used firebase for our data management.

Challenges we ran into

Learning how to use Firebase was a challenge as none of us have experience with it.

Accomplishments that we're proud of

Implementing a real-time database.

What we learned

We learned how to use Firebase.

What's next for OweDough.

Providing a complete breakdown and history of a user. We also would like to link interac's API for an easy way to e-transfer creditors. Additionally, receiving reminders (through email and/or notifications) to pay off creditors.
