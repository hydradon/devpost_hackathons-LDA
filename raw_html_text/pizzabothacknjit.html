Inspiration

We are all lonely creatures who sometimes need a friend to talk to. This bot aims to help relieve some of that loneliness by sounding more human.

What it does

PizzaBotHackNJIT will respond to questions or phrases given by the user.

How I built it

It was build using Microsoft Azure, HTML, and JavaScript

Challenges I ran into

The main challenge ran into was that the bot was a QnA bot, which means it was limited to only answering questions given to it. My team circumvented this problem by expanding the scope of the questions.
