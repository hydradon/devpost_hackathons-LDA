4 OF CLUBS!!!

Inspiration

  • Finding flats in London is really difficult, especially with housemates commuting to different locations. It would be easier to put all the commutes into an app and get areas of London that would be good for all flatmates to live in.

What it does

  • The app takes these commutes and calculates an area of London to live in the best is the best fit for everyone

How I built it

  • With great difficulty

Challenges I ran into

  • A lot, especially technical challenges with the complexities of using and managing the calls to APIs

Accomplishments that I'm proud of

  • Constructed a solid plan from the beginning and manage to implement a significant section of that vision

What I learned

  • JavaScript's Promises are very powerful, but also very confusing…

APIs

  • We used a number of different APIs in the project, Google Maps for the locations and TFL for the tube station and bus stop locations. We also played around with the JustEat API and tried to return the top 5 takeaway restaurants near to the best area but had some difficultly with the API.

What's next for Comutr

  • Redevelop the Javascript to be more efficient or potential move a large portion of the javascript to the server side
