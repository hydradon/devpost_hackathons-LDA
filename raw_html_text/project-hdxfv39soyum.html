何のためのアプリ？

・「会社名」 ・「顧客の新発売製品」 ・「担当者の好きな野球チーム」 こういった評判を見える化し、 顧客訪問時のアイスブレイクにつなげるためのアプリです。

利用したもの

・Salesforce ・Twitter API ・Watson(Language Translator) ・Einstein Language

動作の概要

①Twitterからつぶやきを取得 ②Watsonでつぶやきを英語に翻訳 ③Einsten Languageで感情を分析 ④グラフにして表示
