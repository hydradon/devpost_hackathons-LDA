Inspiration

We wanted to find a new alternative to Bluetooth for info transmission

What it does

It transmits byte code between devices through frequency key mapping, and we wanted to have it work through Apple watch

How I built it

We built it by creating a sound file that would be synced to a file, and have a receiver on the other end.

Challenges I ran into

We found a race condition big in WatchOS2, so we failed to play a file from the apple watch.

Accomplishments that I'm proud of

Saving the buffer data into a file required a lot of low-level system magic

What I learned

Never develop with systems currently in beta.

What's next for Freqkey

Remake it once Apple releases their bug fix.
