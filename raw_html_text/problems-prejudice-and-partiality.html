Inspiration

Seeing all of the amazing women in tech who fought hard to get where they are now, it made me almost disappointed that nobody ever knew of what we had to face to get there.

What it does

It is a simple, informative website which opens people's minds to the outright bias in the world.

How I built it

Using a template, I reconfigured the website with my own code (mostly html and some css) and made it an easy to understand interface.

Challenges I ran into

Some of the previous code that was there became useless, so I had to scrap it all and start the "Advice" page from scratch.

Accomplishments that I'm proud of

This is also my first hackathon, so I am proud that I was able to finish my project and submit.

What I learned

I learned to not take the easy way out and take time to be picky because that's what makes the website yours.

What's next for Problems, Prejudice, and Partiality

After this hackathon, I hope to proceed to update the website every time I come across a new struggle that women face in tech.
