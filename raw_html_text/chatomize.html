What it does

Chatomize is a platform that allows developers to build and customize a Chatbot for Facebook Messenger

Inspiration

The demand for building ChatBots have grown tremendously and we noticed many developers were looking for ways to get started with building their own ChatBot. So we decided to build a platform that can help developers who are interested in getting in building one.

How I built it

I built this using Javascript and Ruby on Rails.

What's next for Chatomize?

We plan to add new features that will allow users to automatically connect to their Facebook page and create their ChatBot using our platform.
