Inspiration

An influx of horror news stories about kidnappings and disappearances inspired us to create a safety app, for those situations that you never know might happen.

What it does

Listens for chosen spoken keyword that will send a text message to the specified phone number.

How we built it

We used Java in Android Studios to build this mobile safety app.

Challenges we ran into

Finding a suitable voice recognition that is lightweight and works offline was one of our toughest challenges. We eventually settled on one, but the documentation was lacking so it was difficult to do too with it with our knowledge. Most of our group had limited experience in using Java and mobile development, so learning a new language and new concepts in a very short timeframe was quite the challenge.

Accomplishments that we're proud of

Most of us had little to no experience with mobile development, so to be able to create a working mobile app with no previous experience was very rewarding.

What we learned

We learned a lot about Java and about Android development.

What's next for SafeSound

  • Implementing multiple keywords for multiple phone numbers.
  • Implementing a phone call feature that also calls a loved one in dangerous situations.
  • Including user's location in the sent text message.
