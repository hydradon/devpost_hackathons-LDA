Inspiration

Those who are fashionably challenged or color blind don't have the ability to color coordinate outfits based on a certain item of clothing they would like to wear.

What it does

User can use the built in camera to take a snapshot of their outfit and tell the application to either match with the pant or the shirt. The application will then speak back to you, telling you what color would best complement the other.

How we built it

Programming in Python using Google Cloud Video Intelligence API and Microsoft Azure.

Challenges we ran into

Having to learn the Google Cloud Video Intelligence API and Microsoft Azure in a short amount of time while creating a mobile application that implements them.

Accomplishments that we're proud of

Ability to speak to the application and get a valid and accurate response back.

What we learned

We learned how to use and incorporate Google Cloud Video Intelligence API, Microsoft Azure, and Python together.

What's next for NowYouSee

Next steps would be to implement a wider color selection and have the ability to create a full outfit depending on external factors (e.g. weather conditions, fashion trends, etc).
