Inspiration

Past experience losing money investing in stocks

What it does

It allows you to select and stock and interval to create a prediction for the stock

How I built it

I pulled data from quandl and created a linear regression model in scikit learn and then graphed it using matplotlib and made an interface in tkinter

Challenges I ran into

Parsing the data into the needed data and getting the graph to be correctly formatted in the gui

Accomplishments that I'm proud of

It is fully functional just a bit ugly

What I learned

I learned a lot about machine learning and python

What's next for SmartStock

I hope to move on to other forms of machine learning such as nueral networks
