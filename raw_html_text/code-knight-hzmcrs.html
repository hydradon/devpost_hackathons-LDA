Inspiration

What it does

How we built it

Built with Unity, assets designed in Scratch

Challenges we ran into

Window size, board rotation, music composition, window quality

Accomplishments that we're proud of

Enemy AI, Code Block functions, music composition

What we learned

Music composition, animations in Unity

What's next for Code Knight

More advanced code block functionality (combining code blocks, deck building), boss fights and different
