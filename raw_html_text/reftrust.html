• RefTrust

Inspiration

Conventional feedback system has no history, no overview and no authenticity. eg, job references which must be requested every time one changes jobs

What it does

It helps employees keep a record of the recommendations they received simply by request sending a request to their employer with their public keys

How I built it

We built it on top of Bitcoin cash using Bitbox and React

Challenges I ran into

Getting the transactions on Bitbox

Accomplishments that I'm proud of

Teamwork Understanding of `Blockchain Solving a real problem Getting different technologies to work together Meeting new people

What I learned

Bitcoin cash The internal structure of Bitcoin Cash

What's next for RefTrust

To continue working on it and then expand the use case
