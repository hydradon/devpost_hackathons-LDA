Inspiration

Our inspiration for this application was to have a chance to work with virtual reality, which we normally aren't able to try

What it does

Essentially you are put into a maze where you overcome obstacles using Myo technology(hand gestures).

How we built it

We used the unity platform to create the world, and integrated it with the Occulus and Myo

Challenges we ran into

Learning how to implement the sdk's, set up the occulus, set up the myo, working with unity; In fact every step of this project was a brutal challenge as none of us had any sort of experience in it, and neither did the mentors.

Accomplishments that we're proud of

We got it to work, and we were able to have the full motion experiance

What we learned

We learned that with perseverance, anything can be developed

What's next for Maize Runna

A bigger more functional maze, with music
