Inspiration

Travel agents are boring. The idea for the hack was to discover flights in 3D with my gestures.

What it does

LeapTravel is a 3D Engine that allows you to discover new travel destinations with your gestures.

How I built it

I used javascript with a WebGL interface.

Challenges I ran into

Using WebGL for the first time

Accomplishments that I'm proud of

Integrating LeapMotion with a WebGL interface

What I learned

Building WebGL interfaces.

What's next for LeapTravel

Live global travel data feed.
