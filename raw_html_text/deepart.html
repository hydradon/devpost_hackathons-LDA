Inspiration

I like making art and I wanted to see what art I could make using AI.

What it does

It used Magenta js to combine two images using the style transfer method.

How I built it

I built it by first making a basic web page and styling it then researching tensor flow until I discovered magenta a wrapper for tensor flow that had a style transfer algorithm.

Challenges I ran into

Most if not all of the documentation I came across wasn't for Javascript I spent hours trying to find proper integration methods that weren't roundabout and used python or node.js.

Accomplishments that I'm proud of

I'm glad I completed it.

What I learned

How to use Magenta and combine images.

What's next for DeepArt

Add more images. I wanna see how many images I can combine to get new pieces of art.
