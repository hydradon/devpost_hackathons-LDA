What it does

We use blockchain technologies treating the ledger as a compilation of true statements e.g. 'The sum of all the natural numbers is -1/12' And the transactions are proposals e.g. 'Yev' proposes 'a=b implies b=a' The proof-of-concept is the proving, which, for the sake of this mock-up, is manual But, in the future, the poc would be a machine learning algorithm which logically evaluates the proof

How we built it

A python flask application hosted on Microsoft Azure
