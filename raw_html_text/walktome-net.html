Inspiration

Based upon the general prompt and my personal goal to develop full-stack web dev skills.

What it does

Tracks users weekly millage based on their input and with goal tracking.

How I built it

Using HTML/MySQL/PHP LAMP stack and a linode private server with a domain from domain.com

Challenges I ran into

Actually having to learn how to do the HTML/MySQL/PHP stack myself and construct the website from bottom up myself.

Accomplishments that I'm proud of

Having a live website that lives on a server that actually has a domain name attached to itself that isn't ran on the domain host nor my own machine.

What I learned

Again learning the LAMP stack and full-stack web development.

What's next for walktome.net

Proper back-end for weekly resets and a friends list feature.
