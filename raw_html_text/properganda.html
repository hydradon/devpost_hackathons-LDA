Inspiration

Mass media is one of the most effective ways to reach the general public. Over lately there has been a lot of unrest over the idea of AI being able to use deep generative models to create artificial media (Fake News). The idea is to develop a tool that can combat these tactics so the everyone can enjoy cat videos without any political bias or subliminal messages.

What it does

How we built it

A SaaS application deployed to the Azure platform, we leverage a variety of Microsoft Services our application's functionality:

  • MongoDB Atlas on Azure, Azure Deploycenter for Continuous integration and deployment from our github repository.

  • AMS Video Indexer API, to extract metadata that explains what is contain in the video (brands, celebrities, emotions, etc).

  • Azure Web App and Kubernetes services to make to applications scalability and performance.

Challenges we ran into

Using the Video indexer API was challenging as the documentation was not straightforward.
