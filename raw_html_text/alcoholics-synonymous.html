Inspiration

Were all students who enjoy bar crawls, but lack the money and time to plan them.

What it does

Calculates the best pub crawl based on current user position (or custom position using postcode API) and optional specified budget (overall, and per pint).

How we built it

Using a big dataset found on Kaggle, containing data on all 51,000+ UK pubs. Designed a Java algorithm to calculate quickest route. Android studio to build the app, Google Maps API for waypoint mapping.

Challenges we ran into

Didn't come up with the idea until 9pm!

Accomplishments that we're proud of

It works to a good standard given the time we had.

What we learned

Working with large data sets, route optimization, working with API's like Google maps and postcodes.io.

What's next for Pub Crawler
