AlexaGenie

Add a new skill to Alexa to keep track of a wishlist We made a database use AWS dynamoDB for the wishlist. The wishlist contains staffs the user wants to purchase and a budget for how much money the users have.

User can add budget by saying "Alexa, ask Genie to add XXX to my budget" User can add to wishlist by saying "Alexa, ask Genie to add XXX to my wishlist" User can access what is currently on wishlist by saying "Alexa, ask Genie what's in my wishlist"
