What it does

It's a software that that sends e-mail. You must have a account to login. When you have permission, the software shows 3 forms. The firts one you can configure the e-mail that always will be the sender; The second was made in order to configure the message. It has a HTML editor, field to identify the receiver and a field to fill with json code. This Json code must be in a pattern, actually, this code must be a array of Json. For exemple: [{"Title":"First Photo", "Url":" http://devpost3.assetspost.com/assets/shared/devpost_social_icon-c58b1e7e9a690be35641fe1608647048.jpg"} ]. The HTML editor is able to WYSIWYG the images inserted by json ; The final form has a few informations of my self.

How I built it

Database: MySql - ID: SqlYog; To model the database: PowerDesigner; For the programming part, i used a tool called Maker All. The programmer do the job by flowchart instead of code. So the programming is up to 60% faster than other languages.
