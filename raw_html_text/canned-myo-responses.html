Inspiration

None of us had ever done serious pure code before as we all are more familiar with embedded. We still wanted to do something hardware related, and when we saw the Myo we threw out ideas until we came up with something easy enough to do in a weekend with no experience.

What it does

It allows the user to set several canned responses to the different gestures that the Myo is capable of recognizing out of the box. The Python script sets the text in the Lua script as a workaround for the fact that the Lua script is not allowed to access the filesystem as a requirement of working with the Myo.
