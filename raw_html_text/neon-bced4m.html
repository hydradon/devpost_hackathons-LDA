Inspiration

In a world full of hackers and bots, there is a need for fair trade while maintaining security and trustworthiness. That is why blockchain will allow us to provide this service for auctions. With the help of the API from Luno, we wanted to increase the integrity of auctions.

What it does

While auctions could be done by bots and users can never be sure if they are fast enough to buy an item. We have created a system that allows a user to do safe and secure auctions. It changes the way the world sees auction.

How we built it

We really liked Luno's API for python so we used it to send money from the bidders to the server and from the server to the seller.

Challenges we ran into

Our team was fully capable of making this project however we did not have enough time.

Accomplishments that we're proud of

We came up with a very nice idea and we think that it has lots of potential to be something really big.

What we learned

We learnt a lot about blockchains.

What's next for Neon

More security and UI.
