Inspiration

The martello-mystery challenge!

What it does

Uses a timeline to allow users to search through data based on time and show where members of a building were during and around the time of a murder

How I built it

We used JavaScript, HTML and CSS for our front-end

Challenges I ran into

  • Understanding the layout of our SVG file in order to draw on it
  • Room dimensions were difficult to find and code starting points onto
  • Connecting multiple libraries together
  • Working with new frameworks and languages
  • Handling dynamic data

Accomplishments that I'm proud of

  • Getting a working timeline
  • Getting the location map to work
  • Learning Javascript!

What I learned

  • Javascript
  • How to incorporate an interactive timeline into data analysis
  • What an SVG file is :)
