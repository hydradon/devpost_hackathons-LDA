How it works

An add-on to Google Docs is added as a side bar. The bar contains a list of relevant quotations retrieved from a database to a specified input by the user.

Challenges I ran into

Originally, we planned on using the Google Books API to retrieve quotations from literature. However, the API was not constructed in a method friendly to our goals. In the end, we instead used a Cornell University database of scientific, math, and computer science papers called arXiv.

Accomplishments that I'm proud of

We got a lot done in not a lot of time.

What I learned

Computers are better at faking intelligence than I am.
