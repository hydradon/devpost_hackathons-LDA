Inspiration

Choosing which credit card to use can be a headache. Takes time to search and compare Visa has APIs on credit card offers globally. Was thinking to connect VISA APIs with Sabre.

What it does

Agents can easily access to lists of credit card promotions with airlines, hotels, car rental.

How I built it

With RedApp SDK , Typescript

Challenges I ran into

The APIs in the RedApp is limited to a sandbox env and cannot display remote content easily.

Accomplishments that I'm proud of

Able to create a working prototype .

What I learned

it is my first time to code in Typescript.

What's next for CreditCardsOffer

Connect with Visa developer APIs and able to display up-to-date offers
