Inspiration

And lo, he did say "compare the tags" - Ryan

What it does

Compares the tags in a photo you submit to the tags in a target photo

How we built it

We used react-native for the base of the project, Google Vision's API to get information about an image, Firebase to store information, and attempted to use Snapchat's login API for user verification

Challenges we ran into

We needed to use Google Vision's REST API instead of their example code. It was our first time using react-native, so we had trouble translating Snapchat's SnapKit API to it. Most of us had limited javascript experience and none of us had react-native experience.

Accomplishments that we're proud of

We managed to get most of what we wanted to do working within about half of the time we had available because we had to switch frameworks 8 hours in.

What we learned

Firebase is awesome, react-native is a thing, plan for and research the technologies ahead of time

What's next for NapHunt

Add in the game logic, get the snapchat login working, deploy to a URL.
