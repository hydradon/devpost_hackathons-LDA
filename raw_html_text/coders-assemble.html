Inspiration

CS2150

What it does

An webpage that allows the user to interact with functionalities of assembly code and learn step-by-step how the stacks are moving with each assembly function. A useful tool for people who are interested in learning assembly code visually.

How we built it

Using HTML, CSS, and JQuery.

Challenges we ran into

Creating a custom text editor.

Accomplishments that we're proud of

-We have a code editor simulator and our tutorial slides look really cool!

What we learned

How to use CSS to style HTML pages!

What's next for Coders Assemble

We want to continue to work on this app and provide the full functionality of all the opcodes in assembly.
