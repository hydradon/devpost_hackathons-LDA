Inspiration

We often struggle trying to figure out how to make our course schedules better, so we want a tool that can help us optimize them.

What it does

Allows user to import their schedules and add/drop courses, then we use a neural network to find out the best schedules possible.

How we built it

We used AWS services for the back-end and Javascript & React for the front-end
