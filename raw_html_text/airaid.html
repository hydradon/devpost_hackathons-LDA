Inspiration

We were always tired of going through huge lists and web pages while finding an answer to our basic frequently asked questions

What it does

So here is our attempt to solve basic questions through automated response where answers are taken from American Airlines' website. For now it answers basic questions about carry on and check in baggage. It even accepts voice input

How we built it

We built it using Amazon web services using Lex, Lambda, EC2. We created a cross platform application which works as android application and website. Which can be futher integrated with slack, facebook messenger, and Amazon Alexa as well.

Challenges we ran into

We had problems on narrowing down the technologies we plan to use as not everyone was not familiar with a common technology platform.

Accomplishments that we're proud of

We were new to AWS like Lex and Lambda, and yet we were able to develop something concrete and gain expertise. We are very proud of our first chatbot.

What we learned

We learned the underlying technologies responsible for creating automated response.

What's next for AIrAId

We plan on adding an avatar for giving responses with an aim to make our interface extremely user friendly and interactive. We even aim at integrating it with Amazon Alexa.
