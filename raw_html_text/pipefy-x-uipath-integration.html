Inspiration

I have been working with Kanban (Pipefy) on my projects and I decided to integrate my RPA Platform (Uipath) with it to easily manage my process through my development. Pipefy already have a API and this gave to me the ideia to create a Custom Activity which does this Web Request without much work.

What it does

Custom Activities for Integrating Pipefy with Uipath platform. With the Custom Activities it's possible to manipulate and iterate through all the process, cards and phases from kanban. It's very simple to manage and control the Activity action.

How I built it

I built it using C# language and the API Web Requests from Pipefy gallery.

Accomplishments that I'm proud of

I received a letter from Pipefy CEO which says the Activity is a “strongly impact for user’s experience”

What I learned

Learned more about Kanban, How to create Custom Activities and how to manage API Web Requests.

What's next for Pipefy x Uipath Integration

My goal is to often update my Custom Activities to achieve 100% of process of the Kanban "automated"
