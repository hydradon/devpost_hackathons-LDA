Inspiration

I've always loved video games, and I've recently wanted to try making my own.

What it does

It is a platformer in which the character is constantly moving right, so it is up to the player to react and dodge obstacles.

How I built it

I built this entirely in Godot Game Engine, which runs off own their own language, GDScript, which is based off Python. All assets were drawn by me and the sound effects were taken from copyright free websites.

Challenges I ran into

I had absolutely no idea what I was doing in Godot, so the first several hours were me looking up how to make a game and understanding the engine.

Accomplishments that I'm proud of

I actually have a product that is semi-complete, which is more than I ever thought I'd have after being completely overwhelmed at the beginning.

What I learned

I learned how a game engine works and learned how to make assets in a project.

What's next for Double Jump Hero

Better, higher quality assets and more levels.
