Inspiration

In a search for a disability information source we found nothing and we decided to fill that void.

What it does

Aggregates user input of accessibility information for venues and presents this information in an easy to use and contribute format.

How we built it

Android Application with a postgress and go driven backend.

Challenges we ran into

Google Maps, Effectively working with locations in postgress, building an api and client simultaneously

Accomplishments that we're proud of

Google Maps, Geofencing capabilities

What we learned

Google Maps, Postgress, Geofencing

What's next for Aditum

Hosting the public api so other's can contribute and develop apps
