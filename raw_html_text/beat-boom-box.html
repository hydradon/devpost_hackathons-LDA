Inspiration

The project is based on the fall of boomboxes and the rise of walkmans.

What it does

The project was made using android studio, and runs on android devices. The player has to time their swipes to when the boombox lights up blue in order to send a sound wave to defeat walkmans.

How we built it

We used android studio, with collaboration on github

Challenges we ran into

We ran into trouble with drawing the graffiti. In addition, this was our first time using github to collaborate with multiple people.

Accomplishments that we're proud of

We are proud of the resulting atmosphere that the game manages to develop

What we learned

We learned how to work with multiple programmers using github, as well as drawing graffiti and various other 90's related artwork. In addition, we advanced our android programming skills through this project.

What's next for Beat Boom Box

We would smooth out the frames, and have music play a more important role in our game.
