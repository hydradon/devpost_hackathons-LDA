Inspiration

We are novice cookers, and we don't know any recipes, so we made this for people like us.

What it does

smaRtEcipe takes a recipe name and tells you exactly what ingredients you need to make it (that you don't already have).

How we built it

We built it with IBM Watson and loopback and atom. Additionally, our original idea was to make it interact with the Samsung Smart fridge and it would tell you what ingredients you don't have.

Challenges we ran into

It was hard to use Watson and it took a while to understand how to use loopback.

Accomplishments that we're proud of

Getting Watson to work with the Conversation Interface and putting it on the command line.

What we learned

What loopback and Watson are and how to use them.

What's next for smaRtEcipe

We believe that we could have it in every kitchen, right there to help you make a nice meal. Also, we will incorporate the Samsung Smart Fridge API.
