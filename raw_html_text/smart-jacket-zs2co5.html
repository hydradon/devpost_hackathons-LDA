Inspiration

How you ever chosen the wrong jacket? or left out in the cold? or heat? Ever wished you knew what to wear wherever you go or whatever you do? Smart jacket is here to help

What it does

Smart jacket knows the weather outside and if its suitable for you to wear based on your schedule and location. It notifies via a text if its suitable for your travels.

How we built it

Technologies used

  1. NB-IOT Dev Kit from T-Mobile and Twilio https://iot.t-mobile.com/twilio/
  2. Twilio SMS
  3. Twilio NB-IoT APIs
  4. Azure Functions (HTTP Triggers)
  5. Breakout SDK
  6. Push Button and Ultrasonic sensor

Challenges we ran into

  1. Twilio APIs requests and response for the web hooks were not well defined.
  2. Working with flaky sensors was tough.

Accomplishments that we're proud of

  1. Working prototype in a day.

What we learned

  1. Basic NB-IoT and use cases.
  2. Usage of Twilio APIs
  3. NB-IoT has huge scope and there are lot technological problems we can solve using this.

What's next for Smart Jacket

  1. Additional sensors to predict schedule and gather more useful information about user (heart rate, body temperature, calories burn, etc) to build meaningful insights and interactions for customer.
  2. Commercialize and partner with the clothing vendors like Nike, Adidas, etc.
