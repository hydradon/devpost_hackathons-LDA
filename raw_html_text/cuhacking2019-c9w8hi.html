To run

Go to the project folder
Install npm if you dont have it, npm install npm@latest -g
In a terminal run npm install-g@angular/cli
then ng serve --open
It should open in a new browser

The QUESTIONS html (just run by clicking on html file)is where the user would have to go to fill a questionaire and based on their answers they are redirected to another travel page. This travel page shows information about a city they should visit based on their answers on their answers in the questionaire. (this is everything that it should do but it doesn't yet).

Everything is hardcoded in and we haven't coded it to be dynmaic so might not fit all screens
