Inspiration

How often do we search for adresses, emails, phone numbers, linkedin names of friends and friends of friends?

What it does

Well, everybody keeps his or her data store with Gaia, while the web app is synchronising the information of the others.

How I built it

Easy Single-Page-App build with reactjs, react-router-dom, blockstack and material-ui.

Challenges I ran into

How to retrieve file from other users gaia storage, how to search for other users, that do not yet use cdetacts.

Accomplishments that I'm proud of

building an UI with Material-UI, understanding gaia-Storage

What's next for Detacts

many features are important. Implementation of requests -> one can only see my info, if I can see his/her info as well.
