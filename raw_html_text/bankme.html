Inspiration

The inspiration for the program was originally inspired to manage subscriptions, eventually we felt a money planning app with the banking app all in one would be a useful application.

What it does

Financial planning with subscription management, banking online, investment checking, ATM location, checks user profiles, using Zelle all in one application. It also shows your days until payday.

HTML, CSS, JavaScript
