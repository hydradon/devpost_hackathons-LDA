Inspiration

Women across the globe are often denied access to reproductive health products and sex education. Our mission is to link the women to the stores so that they can access products, while also providing information about contraception and how to create your own materials.

What it does

The application first takes the users location using Google Places Pointer API, then it shows the distance between the closest stores. A user can select a store and access information on products, stock availability and pricing. There are also addition features that provide information on contraceptives and preventative measures.

How we built it

We programmed in the Android Studio environment using mainly Java and XML

Challenges we ran into

It was our first time using the Google API and it's very difficult to perfect the interactivity.

Accomplishments that we're proud of

What we learned

We can learn a lot in 24 hours, have fun, and create something we all believe in.

What's next for Red

Hopefully we can develop it further with various language options, providing tutorials, more information and maybe make the app more user-friendly.
