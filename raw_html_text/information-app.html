Inspiration

From the problems presented yesterday, we were able to pin-point a recurring theme. For information to reach from one point to another, there is a lack of: 1- Stable, secure and persistent communication channel. 2- Information storage base. 3- Insurance of credibility, reliability and within-time delivery

This inspired us to think of many near reallife example scanrios from our NRC collaborators.

What it does

Empowers beneficiaries to realize and access their legal rights

How we built it

Communication Channels: App: Mockups using Sketch to communicate scenarios. Website: Website and server using Python(Flask), MongoDB and Ngrok SMS: Twilio API communicating through Python(Flask) server

Challenges we ran into

1- We missed one "/" in the code and spent one hour debugging it. 2- Many technologies were used for the first time. Understanding, Implementing and Making it work was quite challenging. 3- Trying to take out the basic problems out of all scenarios and combining them.

Accomplishments that we're proud of

Assisting IDPs and refugees in conflict zones

What we learned

1- Extent of problems and crisis that refugees have to face which was a strong source of motivation throughout. 2- Design thinking and how to prioritize problems 3- Many technical skills

What's next for Information App

=> On field testing (Testing it in one of the area. Iraq, Ukraine or Venezuela with local settings) => Large scale implementation of the infrastructure => Educate people => Implement following features:

  • Analytics from the data collected
  • BOT to respond for FAQs
  • Add audio as a channel
