Inspiration

In the dystopian future where the news are influenced by corporate media and designed to serve their interests, subliminal messages hidden in texts lets people become corporate puppets. This needs to be revealed before it's too late. We are the light in the dark, bringer of sentiment revelation, the messiahs of the information age.

How we built it

With the help of python we integrate the IBM Bluemix AlchemyAPI sentiment analyser for news into an easy to use iOS app.

Challenges we ran into

The sentiment of news is always negative, so we changed to analyse entity sentiments to compare the news We ran out of Bluemix tokens API restrictions and hard to understand documentations No data

Accomplishments that we're proud of

it's kinda working and an inspiring idea contributing to world wide education.

What we learned

There is an API for everything, you just need to know how to use it.

What's next for Opinio

True integration of the Bluemix news feed, a working search function with recommendations and pleasing UI.
