Inspiration

We saw the Fracture ARKit challenge and thought it would be fun to try out ARKit and Swift for the first time!

What it does

As of now the app initializes and finds adequate space to place the object, then tapping the object will switch the image scene.

How we built it

Using ARKit, Swift, and RealityComposer, we built a small file that operates off of a Storyboard, View Controller, and .rcproject. In Reality Composer, there are 3 scenes that used a block object for the frame, the image(s), and two billiard racks for the stand.

Challenges we ran into

Only one of us had ever used Swift before, and learning it and ARKit in 36 hours was not exactly doable. Even though, we were unable to meet our goal we still improved along the way.

Accomplishments that I'm proud of

We can say that we created an app, utilized Augmented Reality, and got to know each other better.

What we learned

ARKit is incredibly complicated, but oh so useful when done right. We only have ~360 days to learn it all for the next Swamphacks.

What's next for ARnold

Version 2.0 will include movable objects and the option to upload custom photos & pick other frames.
