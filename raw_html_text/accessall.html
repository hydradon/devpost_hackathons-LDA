Inspiration

We felt as though our app can promote change in public spaces by encouraging them to be more accessible for people with disabilities.

What it does

It allows users to rate the accessibility of a location (restaurant, parks, etc.) to increase awareness for the lack of accessibility and encourage said locations to better suite their businesses for people with disabilities.

How We built it

We used React, NodeJS, Google Maps API

Challenges we ran into

Challenges we ran into was being able work with the google api and implementing it with React in a short amount of time. More so, sticking to what we know rather than picking up something new.

Accomplishments that we're proud of

A well thought out product, and the passion we have and continue grow for accessAll

What I learned

Teamwork, React, Better developing ourselves as engineers.

What's next for AccessAll

Better implementation of Google Maps API, Interactive UI, Adding Assistive Technologies, Adding features for ALL disabilities.
