Inspiration

Eye and Skin diseases are major problems which almost get neglected in the initial stages every time. if detected in the initial stages, they can be treated with very less cost and suffering

What it does

Uses Google Cloud Machine Learning tool to train the model and predict the status of input image.

How we built it

Created front end using HTML,CSS and Python and Django framework. Machine learning model (CNN) was created using Google Cloud Vision and integrated it to frontend

Challenges we ran into

Data collection, Integrating frontend and backend

Accomplishments that we're proud of

Local website is created. successfully integrated frontend and backend

What we learned

Django framework, website designing, creating and training models on google cloud

What's next for Eye and Skin disease detector

Hosting it on a web server, creating a mobile app
