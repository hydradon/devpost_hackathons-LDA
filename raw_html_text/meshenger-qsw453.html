Inspiration

Are you ever travelling and bored in a location with low network connectivity? Meshenger solves your boredom by letting you communicate without a network with those around you.

Current Features

  • Ability to chat with individual users around you
  • Anonymous chat
  • Timestamp: date/time
  • List of connected users
  • New message notifications
  • Team Members: 5

How I built it

Meshenger is an Android application built using the RightMesh API and programmed in Java.

Challenges I ran into

Working with an unfamiliar API and with finicky android phones was a challenge.

Accomplishments that I'm proud of

We accomplished being able to send a message to an individual and receive responses. We established a good-looking chat UI.
