Inspiration

The Pebble is awesome. Punching is awesome. Falcons are awesome. So we decided, using dark arts, to craft the trifecta of awesome: Falcon Punch for Pebble Time.

What it does

Falcon Punch is meant to make learning about nature super awesome. Punch as fast or as slow as you want, then learn about your Punch Animal. Punch as fast as a running bear? Learn some bear facts. Also, with Force of Nature mode, match the speed of the animal displayed on the Pebble for an added challenge. Can you punch as fast as a running tortoise? How about as fast as a cheetah? Get as close as you can!

How we built it

Using the Pebble web development platform, CloudPebble, we were able to quickly edit and demo the code as frequently as we needed to make Falcon Punch a reality.

Challenges we ran into

Accelerometers are tricky business. As is remembering how to actually apply high-school physics to punching. These were our most daunting tasks.

Accomplishments that we're proud of

It actually works, which is pretty awesome.

What we learned

Punching fast is significantly harder than anticipated.

What's next for Falcon Punch

More gametypes, more education, and more punching!
