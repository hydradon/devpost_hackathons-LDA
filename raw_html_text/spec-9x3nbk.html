iOs app for scouting soccer players using the indoor estimote sdk

The application does an indoor mapping of a soccer field, using the estimote indoor location app. then gets uploaded from the cloud to the application. To start scouting, simply click on start scouting button to start seeing the statistics. At the end, of the 60secs timer. You get a summary of the players performance.

We are building an internet of things platform that will connect to various wearable technologies used by casual and professional athletes. This will enable the sharing of activities and performance tracked by the wearable devices in real time creating an online social structure connecting athletes from across the globe.
