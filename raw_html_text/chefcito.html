Inspiración

Quisimos desarrollar chefcito para evitar el desperdicio de comida, para las ocasiones en las que contamos con ingredientes pero no tenemos idea que cocinar.

Que es lo que hace?

Toma una lista de ingredientes y genera una lista de recetas posibles con los ingredientes de la lista. También puede reconocer imágenes o fotos de productos para evitar el proceso de escribir los ingredientes,

Como lo construímos

Esta construido en node.js y esta orientado a conexiones por medio de API´s, empleamos google vision api para el reconocimiento de objetos, amazon web services con virtual private networks para la infraestructura, wit.ai para pattern matching de palabras y facebook messenger
