Block Analytics

Inspiration

The lack of current insights really led us to help other block explorers.

What it does

Provides much needed insights and analysis of the contract ecosystem that current block explorers don't provide.

How we Built it

Stood up cloud infrastructure for storage and created simplified RESTful api and served clean data for a front end data-viz insights.

Challenges

API rate limiting pushed us to cloud storage. Internet made cloud storage and computing very difficult. Lack of experience and relatively new-comer required lots of learning.

Accomplishments to be proud of

We learned, we built something amazing, and we made new friendships.

What we learned

There's way to impact the etherium ecosytem in new ways, lots of great people, and lots of ways to contribute in this space.
