I wanted to help the visually impaired in their daily lives.

This is a wearable device allowing users to be notified if they get too close to the nearest object relative to the device

I used an Arduino kit and a breadboard along with an ultrasonic sensor to detect when an object goes closer to the wearer than the specified threshold in the code.

In the future, the uses of this device can be expanded upon by adding a buzzer and a push functionality as a part of the notifications system so that the person can press the button to turn the buzzer and seek help in the event of an unsafe situation.
