Inspiration

I coded a front end web app for my team's machine learning model, however my teammates were not able to complete their code, therefore I had to create something simple in python to not waste the hours I spent on learning and coding.

What it does

Counts the number of appearance each word has.

How I built it

I used html and css to build a simple website. I used jquery to communicate between my website and my flask backend. Then I deployed it to google cloud.

Challenges I ran into

I did not have any experience in any of the things I have done in the past two days, therefore everything was a challenge to me.

Accomplishments that I'm proud of

The fact that I was able to build a web app and learning some many new things without any prior knowledge.

What I learned

I learned a lot about front end development and many languages.

What's next for Word Counter

I will try to finish the machine learning model and use this front end for what it was actually built for.
