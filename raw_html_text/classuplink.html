Inspiration

Students have trouble focusing in class for almost 7 hours each day. If a student is absent, they miss the lectures and lessons that the teachers give, and can never get back the full experience of the classroom. Our idea is to recreate the same classroom experience for everryone.

What it does

This application ,which is meant for classrooms, converts the teachers voice into text for later review. This helps the students with studying.

How we built it

We used android studio ide with gradle and java tools along with help from the Speech api

Challenges we ran into

Integrating Drive API to our project which would have been much beneficial for the problem we sought to solve.

Accomplishments that we're proud of

We are proud of converting speech to text effectively and converting it into a storable text format

What we learned

We learned how to use Android Studio IDE extensively and using

What's next for ClassUplink

We will try to make more our project more user-friendly by adding efficient connectivity with G-suite and share features. We plan on increasing our application's reach so that it helps more and more students.
