Inspiration

In a world where unhealthy habits arise from technological innovation, we decided to build an application that not only promotes healthy living(Both Mentally and Physically) through exercise but also stimulates community building through pickup sports.

What it does

It allows users to create and join varying different pickup sports games around there community.

How we built it

We built it using React, CSS, Firebase.

Challenges we ran into

Some technical issues with Firebase.

Accomplishments that we're proud of

The idea was a pretty solid one, the UI is responsive, and a lot of hours went into making it.

What we learned

Some of us picked up some basics on how to create a project using React.

What's next for TeamUp

Creating a fully functional prototype.
