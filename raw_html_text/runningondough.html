Everyone want to travel the world with no worry about how much money is in their pocket but sadly everyone is on some kind of budget. Building this web app, it helps to solve two big issues that family/friends can over come when travelling on a budget. Firstly, it gives dates that works with everyone's schedule which eliminates that long stretched task of finding a date. Secondly, it helps to divide up the cost of how much money should go into the 4 main categories which are transportation, food, entertainment, and living arrangements.

Built with

It was built with Python, Flask, Sqlite3, HTML, CSS, jQuery, JavaScript

Challenges we ran into

We were all new to using Flask, so we had a bit of trouble setting it all up and getting it to work.

Accomplishments that we're proud of

We learned a lot about how to build web applications and how to integrate a database with Flask, and we got it to work!

What's next for RunningOnDough

We plan to use the Yelp API to include recommendations for each category that are withing the given budget plan.
