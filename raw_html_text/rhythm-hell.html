Inspiration

Other rhythm games like audiosurf, guitar hero, and rhythm heaven

What it does

It is a fun third person rhythm shooter game.

How we built it

We built the game in the Unity game engine using c# for programming and Maya for art.

Challenges we ran into

Figuring out how to handle varying beats per minute and song types. Alongside that, creating a functional player controller, good AI, and a reactive environment.

Accomplishments that we're proud of

We are proud to finish the game and have it in a working and fun state.

What we learned

We learned how to better work with audio files, materials, and AI in the Unity game engine.

What's next for Rhythm Hell

More polish and refinement of the music insertion process.
