Saving the environment by using technology. Implementing Water Reuse Credits through block chain, smart contracts and a lot of other technologies. We came up with the idea of building a complete dashboard for the regulator in order for him too monitor all the activities as well as different communities reusing water. We came up with algorithms that could prevent tampering of water meters for accurate results to prevent data fraud. We used different clustering algorithms in order to come up with a solution. We manually went to data sheets in order to look for features and patterns that could indicate a fraudulent meter reading. We went ahead and implemented the blockchain perfectly with smart contracts as well on the hyperledger platform. We were able to maintain data privacy of industries which could trade certificates by creating a pool of WRC which can only go through the Regulator
