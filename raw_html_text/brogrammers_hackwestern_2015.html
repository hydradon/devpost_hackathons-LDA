WHAT

date-mate.net provides a fresh approach to those who have problems deciding what they would like to do on date night. Provides locations and upcoming events from a list of activity filters the user is interested in.

HOW

Utilizes Google Maps Geocoding API to determine the coordinates you add. Utilizes Google Places API and Eventful API to query for local entertainment, establishments and upcoming venues. Front end is written in HTML/Javascript with the back end written in PHP powered by Laravel 5 framework.
