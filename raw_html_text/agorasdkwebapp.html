OpenDuoLive-Web

This Agora application that will help you get live video broadcasting and chat messaging integrated directly into JS applications using the Agora Video SDK + Signaling SDK.

With this application, you can:

  • Join broadcast as an broadcasting member
  • Join broadcast as an audience member
  • Chat with broadcasters and viewers

Running the App

npm install npm audit fix npm run build cd dist

The Web Application is hosted and available at the following location - https://34.220.202.49/
