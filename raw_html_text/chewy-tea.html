Inspiration

  • Yelp but watered-down/streamlined for boba
  • Hillary Clinton

What it does

  • At the moment, the app merely allows the client to authenticate their Google account in order to reveal a map (currently locked on Riverside) that searches and marks on a map nearby boba cafes/shops.

How I built it

  • HTML for the layout with CSS for styling and Javascript in order to handle things such as user authorization and authentication and Google Map inclusion

Challenges I ran into

-HTML formatting and learning all the syntax on the spot

Accomplishments that I'm proud of

  • The map appearing/disappearing with the mouse over on the boba mascots.
  • Google Authentication working as intended

What I learned

  • Basic HTML and CSS syntax
  • Basic Javascript syntax and how to incorporate into HTML
  • Basic API integration and parsing with Google Maps/Places Javascript API

What's next for Chewy Tea

  • Chewy Tea is currently only hosted off of a local server that ran on my laptop through Python. By applying something such as Firebase or Flask for a backend server, more functionality is potentially planned such as checking in to local boba places and keeping records by Google Account.
