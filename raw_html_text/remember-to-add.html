There are a lot of memory games out on there on smartphone devices. However a lot those of games have intensive courses that can easily get exhausting to complete. That is where Remember To Add comes in. This game, designed using Processing IDE and Java, displays cards with addition problems and their answers. The player/trainee must match the expressions with the solutions. Remember To Add is unique because it you aren't bounded to a strict course that can wear you out. Whenever you have the free time you can train your brain to Remember To Add!
