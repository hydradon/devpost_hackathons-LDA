We want to use AWS-comprehend to analyze the emotional tone of text.

Emotext analyzes the positive/negative/mixed/neutral tone of the input. The input can either be text passages entered by the user (in multiple languages) or text parsed from image urls.

We built it using NodeJS with Express and MongoDB Altas. We used AWS-comprehend for NLP, tesseract for image parsing, and we deployed our app using Heroku.
