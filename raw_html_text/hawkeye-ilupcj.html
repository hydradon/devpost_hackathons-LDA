We've all lost our kids for a few moments in a public space. Most often we find them within a few minutes but there are cases where they truly are lost. We wanted to develop a smart city solution which will take advantage of crowd sourcing as well as camera feeds to locate a missing person

Uses social media and other crowd sources (ex: twitter) and video camera sources to locate a missing person.

Utilizing AWS cloud-formation, EC2, S3, Lambda, node.js and Python

Permissions to execute and save data. Access to cameras, SDK mismatch

We built this in less than 8 hours at AWS re:invent 2017
