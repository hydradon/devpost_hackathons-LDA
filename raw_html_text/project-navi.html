Project Navi was inspired by the assistant fairy in The Legend of Zeld. Much like Navi, Project Navi is there for you when you need it, automating many of the common functions necessary at home. An automated doorbell uses facial recognition to identify guests. Project Navi is integrated with Spotify via voice commands, so users can listen to music, and Project Navi features an intractable chat feature with the Chatterbot AI.
