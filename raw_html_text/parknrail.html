Inspiration

We want to decrease the heavy traffic jam in Switzerland cities while being economic and environment friendly :)

What it does

Our web platform helps the clients to find the most cost and time efficient route using the parking plots and the railing rides to reach their destinations. Our users can book parking and railing ticket using the platform. Then, they will receive the QR code to give them access to the parking plots.

How we built it

Initially, we created our web pages using the html5 and CSS (Materialize CSS as a framework).

Challenges we ran into

We did not find it easy to incorporate the real-time data from the online databases.

Accomplishments that we're proud of

To work, learn and grow together during the 24h of challenge.

What we learned

We mastered using the different components of the framework. We learn how incorporate databases into our web development project

What's next for ParknRail

To prototype it with car users between Geneva and Lausanne
