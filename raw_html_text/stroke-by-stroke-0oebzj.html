What it does

stroke-by-stroke is a technical software idea for digital ink. The sofware is based on modern web technologies and can run worldwide. There are different modes availible:

1) learn mode – ideal for learning letters, words ...

Every user can learn symbols (or letters, math, symbols, signs and words) with the software stroke-by-stroke. The user see how letters, math, symbols, signs and words are written in the correct way. The software can show which is the first stroke and supports the user / student or the child with further informations.

2) pattern mode – where is this part in the image? You can add some pattern in the background of the canvas and the software ask you … „Where is the bone XYZ of the human?“ Or „Where is the left arm of the people?“ Than you must draw in the right target.

You can create a course for kids, students or different groups and check the input informations.

3) creative mode – you are an artist? Draw, paint

Use the canvas to show your image, draw it with more colors and save your result as image or WILL file on your local device.

The editor of some course can draw the symbol, letter or word and can add some more informations. (Text, images, Videos ...)

How I built it

The software uses the WILL SDK, HTML5, CSS3, PHP5, JavaScript, jQuery, and the Chart.js

Every strokes / inpt can draw in a canvas, the canvas are saved as image on the file server. The software can compare the actually input with the saved images of the editor (teacher). Than the user got the result of the accuracy.
