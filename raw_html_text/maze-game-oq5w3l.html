Inspiration

To gain experience with making a maze game in python and breath-first search implementation.

What it does

The game automatically creates a grid and places the player and exit point in a random location. The designer clicks on different locations inside the screen, creating or destroying walls. The program constantly checks that the player has a viable route to the exit. The designer cannot completely block the player. It does so by using Breadth First Search, which is a simple path-finding algorithm.
