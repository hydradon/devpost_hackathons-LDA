Inspiration

Over sleeping and sometimes phone running out of battery.

What it does

I sounds an alarm or frequency based on the speaker attached. It can be either used for home, or at facilities like US Military dorm rooms.

How I built it

It is build using Intel Edison board, since there was no motion sensor available, light sensor is used to prototype the demo.

Challenges I ran into

There was no library available to connect to azuaze using Intel Edison.

Accomplishments that I'm proud of

I submitted the project this time, and it is also my first time to do demo

What I learned

I learned to use Intel Edison

What's next for Smart motion sensing alaram

Hoping to use the language that uses the library to connect to azure, and also motion sensor instead of light.
