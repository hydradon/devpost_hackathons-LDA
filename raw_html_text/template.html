Inspiration

Being heavily influenced in our local chapter for best buddies, we are motivated high school students looking to raise as much as we possibly could for autism research. We decided to come to a HackTJ to use it as a platform to create an influential app for caretakers of kids with social problems. In the app, we have specified lists with everything a caretaker might need when taking care of a kid with autism. We know that parents and new teachers have a hard time introducing kids with autism to designated caretakers. Therefore, our goal is that any random person given permission of the kid's aaware settings is able to open the app and know the kid's preferences from favorite toys to food allergies and emergency contacts.

What it does

The user is prompted a login page, where specified hint structures guiding the user through registration. The user is able to add their child's personal preferences. Now, when any teacher or care-taker is linked to the child's aaware settings, he or she is able to understand the child's wants.

How we built it

We utilized Android Studio as our IDE of choice to build the Android application. In addition, we employed Git and connected a Github repository. We were able to use git pull, git push, and git commit to share our code and work efficiently on the project.

Challenges we ran into

  1. Implementing the user inputs into firebase using Android Studio. We decided to follow our plan to scrap it and continue innovating through java classes with private data-fields.
  2. Debugging problems with login panel buttons.
  3. Implementing animation design elements before the login page.

Accomplishments

  1. Slide-up animation for the opening panel.
  2. Seamless design and layout
  3. Organizing and saving a user's data, which one can access through their account
  4. Connection between our app and a GoFundMe charity to raise money for autism research ## What we learned
  5. Android Studio Interface
  6. Dependencies
  7. Debugging to figure out path errors

What's next for aaware

We plan to incorporate a cloud-based server, in order to securely host all of our data regarding a user (including their name, password, email, etc.). We may accomplish this by using the REST Protocol and connecting to an application server that may host a SQL server with many databases to utilize. In addition, we would like to utilize innovative technology, such as blockchain, to further increase the security of our application. Our main goal is to raise awareness to autisim, through our app and spreading the word abroad.
