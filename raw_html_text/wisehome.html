Inspiration

** Democratize IoT for every home in the planet with power consumption behavior from devices and people **

What it does

In a World where there is no shared information on people behavior in very specific way, being one of the most important as necessary to do all other jobs and works on daily basis, the Power Consumption behavior is a must to be included in a real time environment where a cloud platform need to be aggressive fast and secure to store and also analyze this Big Data. But those information need to be correct address to not disclosure on people true identity or could lead to a massive problem such as detailed personal behavior, to mitigate this problem a extreme security control has been implemented to grant not only a true cryptograph in real time, but a security in the data storage, this technology lead us to develop a blockchain based solution that could be addressed inside a standard database system like SQL/ORACLE, this solve the problem for data replacement and data fragmentation over the difficult to concentrate data in a unique place. The hardware was developed to construct the blockchain direct on the device but also deployed in a FOG concept. Fiesta platform can address all the data collection to permit a complete solution as SaaS to deliver the filter to the Power Generation companies to understand when, how, which and whom is using the power in real time. As an example, we can use the Basque Country with the largest city Bilbao as an addressable test bed. Target: Residential Consumers Goal: Measure real time power consumption and identify what is the hardware/device using power Set: Identify costumers behaviors Data: Collect data in real time AI: Analyze data in 3 blocks 1) Device using power by brand-model 2) How much power is used 3) Behavior and predictable action

Implementation

WiseHome board is a device where it is installed in a seamless interaction with the infrastructure, whenever there is an in-wall box for power outlet or light switch just remove it, install the WiseHome board and reinstall the outlet/switch. Once installed by the APP the user just adds the device to their own Wi-Fi network or a device could come with LoRa/SigFox enabled solution that the used just need to ADD the device to be unique controlled by him, so no Wi-Fi need to be used (those options depends on dealing with costs).
