Purpose

Simply to extend the boundaries of a typical piggy bank to allow the product to do more in terms of promoting financial literacy in millennials

What it does

The main features include a wallet(which allows users to adjust savings, see their total investments and expenses) , a game induced dictionary randomized and optimized for each user to learn certain finance terms and jargons, forum for discussion and a news feature to keep users updated on the most important financial news specific to their interests.

How we built it

Simply used javascript, flowlab.io, news API and adobe xd for prototyping

Challenges we ran into

Having to fully build both applications on web and mobile. Using javascript to build animation. And also visualizing the game phases in adobe Xd. Trying to build the front end using React.js when we had no prior knowledge of the framework.

Accomplishments we are proud of

We do have a complete prototype of the mobile interface as well as a hosted web interface showcasing the most important features. Learned to work in a completely random team where we each understood our tasks and executed perfectly.

What's next

Building a more complete app that will interface with several bank APIs and use machine learning to understand users in terms of choosing which game content induces more users. Updating the website with a more streamlined game design and multiple avenues to learn about financial literacy.
