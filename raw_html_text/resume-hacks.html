Inspiration

Applying for jobs can be quite tedious as we must constantly change whats on it for different positions. With Resume Hacks, your resume is generated for you based on job key words.

What it does

Generates your resume by analyzing job description and selecting relevant experiences.

How I built it

Implemented with React and node.js, Firebase, and Google's natural language processing.

Challenges I ran into

Connecting Firebase to React and node.js,

Accomplishments that I'm proud of

Learning React and jode.js during the hackathon and implementing it.

What I learned

API calls, Firebase, node.js, React

What's next for Resume Hacks

Integrating and improving Google's natural language processing implementation.
