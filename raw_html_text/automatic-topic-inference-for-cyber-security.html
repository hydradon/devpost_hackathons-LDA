What it does

Generates a compact overview of what are important cyber security topics

How we built it

We used python with word2vec and k-means clustering to produce categories of topics. We display the the data using the simple webserver Flask

Challenges we ran into

Processing the data to get usefull information, dealing with noisy data.

Accomplishments that we're proud of

The effectiveness of our tool.

What we learned

Unsupervised learning is really important to find out what information is hidden in your data

What's next for Automatic topic inference for cyber security

Polishing the data processing pipeline and presenting the data in even simpler and better interface
