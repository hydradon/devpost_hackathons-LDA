Inspiration

We wanted an app that most people could find useful.

What it does

It reads instructions for you while you cook.

How we built it

We built it using the Nuance Nina API and Android Studio.

Challenges we ran into

Integrating the Nina API was a bit challenging at first.

Accomplishments that we're proud of

Learning how to integrate the Nina API.

What we learned

How to use the Nuance Nina API and how we would eventually use it to power our final Nuanced Cooking app.

What's next for Nuanced Cooking

Full core functionality, and later inputting recipes via OCR and potentially user sharing of recipes through the app.
