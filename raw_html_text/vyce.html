Inspiration

Smoking is an issue that harms many people in the city. We would like to help people better themselves, their health, and their life.

What it does

It incentivizes people to quit by giving them the resources to quit, motivation to keep them going, and coupons to save money as they continue to meet their goals.

It also would upload user data to a cloud where we would be able to run statistical regressions on the data. This would allow us to observe smoker habits and the paths people take towards quitting. This would help us create models that would give researchers insight onto the main causes of addiction and the best ways to quit.

How we built it

We used Android Studio to create the App as well as AWS to store user data. We would use a Python module that allows to use R to understand the data that we collect. We used Google Maps API to create the map feature.

Challenges we ran into

The AWS server was set up and running but was to complex to implement into the app. If we had more time we would have been able to make the app work with the server.

Accomplishments that we're proud of

Creating the app in general. Learning more about Android Development and Networking.

What we learned

AWS and Android Studio. Teamwork.

What's next for Vice

We plan on meeting in the summer to continue the app and polishing it up after finals. It should be released by the end of summer.
