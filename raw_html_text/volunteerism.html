Inspiration

We always wanted to volunteer but we weren't sure of exactly all the opportunities out there. We created this website to allow more people to connect with communities through volunteering.

What it does

Allows users to create an account, see volunteer events, create their own, invite other users by email, etc.

How we built it

Front-end: HTML, CSS, JavaScript, Bootstrap Back-end: Python, Django, PostgreSQL Server: Ubuntu 18.0.4 on an Amazon AWS EC2 machine, using Gunicorn/Nginx

Challenges we ran into

Tons of bugs everywhere on basically everything. Bugs with Django, Python, getting the web server to work, getting the SQL to work, getting the HTML/CSS/JavaScript to appear correctly.

Accomplishments that we're proud of

Setting up a functional full-stack website with a:

  • Visually appealing front-end
  • Actionable back end code
  • Deployed to live server on AWS

What we learned

  • Full-stack web development is highly precise process that requires many different moving parts and integration of many different skill sets.

What's next for Volunteerism

We will refine the GUI and add more functionality (remove users from your event, have multiple event managers, Google Maps integration)
