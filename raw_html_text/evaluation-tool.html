Inspiration

We thought it would be cool for judges to be able to use it for evaluating various teams

What it does

The Evaluation Tool is a program for judges to use to evaluate a group of teams for an event.

How we built it

We used the BlueJ IDE and wrote in Java

Challenges we ran into

We started our project in C++ but then realized it would be better to do it in Java. We were originally going to use a Java GUI but ultimately used pop ups instead.

Accomplishments that we're proud of

Finishing on time

What we learned

How to implement pop up boxes and it was our first time collaborating with other people on one project

What's next for Evaluation Tool

Possibly adding additional options and a screen to display various statistics based on the input values
