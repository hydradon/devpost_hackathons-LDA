Inspiration

Sometimes it can be hard to make friends or find people to go out with on the weekends

What it does

It shows you other users that are currently logged in on a map, as well as bars in your area.

How I built it

Using Xcode, Firebase, Geofire and Google Maps API

Challenges I ran into

Building the UI and all of the constraints

Accomplishments that I'm proud of

The interactive map was difficult but I felt really good once I finished it

What I learned

How to work with Geofire API

What's next for theNightlife

Hopefully get some business owners on there and get more people to use it
