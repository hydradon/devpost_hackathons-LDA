Inspiration

We wanted an easily accessible display of updated news and information for women in STEM to stay inspired.

What it does

When installed as a Chrome web extension, it sets your new tab page to the women-in-STEM-themed dashboard we designed. This displays the time and date, a twitter feed of daily articles about women in STEM, a randomized inspirational quote from a powerful woman in STEM, and links to resources and organizations for women in STEM.

How we built it

We created a manifest.json, html file, css style sheet, and javascript to create the web extension, then tested it by installing it on our own Chrome browsers.

Challenges we ran into

Some of us were beginners with web design and HTML, so we had to learn from scratch! Javascript was also unfamiliar to most of us.

Accomplishments that we're proud of

We taught ourselves new skills with HTML, CSS, and Javascript, and met some other powerful female hackers from around the country!

What we learned

How to make a chrome web extension!!

What's next for Steminist Homepage Web Extension

We'd like to make it more customizable, so each woman can design it to fit her interests.
