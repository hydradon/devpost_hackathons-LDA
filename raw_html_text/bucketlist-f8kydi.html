Inspiration

We've all made bucket lists before and let's face it, we've never actually gone through with them. But why? We usually just forgot about it...

What it does

BucketList is a smart bucket list planner that checks the prices of your trip and notifies you when the prices decrease. We use machine learning to find other trips that might be of interest to you as well as coordinate multiple destinations in the same trip.

How we built it

We built this using a Flask back-end and an Android front-end.

Challenges we ran into

Hooking up the front end and the back end proved to be much more difficult than we thought it would be.

Accomplishments that we're proud of

Successfully integrating machine learning into our app.

What we learned

We learned how to integrate machine learning into an actual app.

What's next for BucketList

Adding more features for the machine learning and adding a community aspect around the app
