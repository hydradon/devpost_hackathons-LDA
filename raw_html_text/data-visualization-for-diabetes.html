Inspiration

Diabetes is a prevalent problem in the United States, as well as in this world. We wanted to visualize how countries are affected by diabetes, as well as create a campaign to raise understanding and awareness.

What it does

We've inputted huge JSON files indicating the prevalence of diabetes in different countries around the globe. We've tracked three years of diabetes prevalence data, and we believe that we can use this to show how widespread diabetes really is.

How we built it

We created the landing page via basic HTML and CSS; we then used webgl and three.js to generate a globe that takes in JSON data and outputs bar-like visualizations.

Challenges we ran into

We've had many concerns with syntax behavior when it comes to using webgl and three.js to generate our model; in addition, we've overcome some issues regarding the input of videos on HTML.

Accomplishments that we're proud of

We are proud that we were able to generate data visualization in such a beautiful manner. There are many different ways to produce data that engages the users, and we are happy that our method is one that does that.

What's next for Data Visualization for Diabetes

We hope to use what we learned at PennApps to continue creating learning visualizations and models.
