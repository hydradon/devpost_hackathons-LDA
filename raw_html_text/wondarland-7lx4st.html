Inspiration

So many of today's devices and applications detract our attention from the people around us. WondarWho emerged from the desire to have technology direct our attention back to the world around us.

What it does

WondarWho is your catalyst for connecting with people around you. User's bio's bridge the gap and provide a little bit of information to start conversation. All of the content is displayed through augmented reality.

How we built it

We built WondarWho using Azure's facial recognition, ESRI's ArcGIS, and Unity.

Challenges we ran into

Integrating ESRI’s SDK with Unity caused a few compatibility issues. Choosing a facial recognition API with the features we required.

Accomplishments that we're proud of

We were able to match the theme of the Hackathon, while creating an application that will be meaningful in the long-run.

What we learned

We learned about the importance of dependency management.

What's next for WondarWho

Upgrades in the UI, face-tracking for localization of content, integration with ESRI 3D Terrain, automatic Azure bucket training, and incorporation of Teleportal.
