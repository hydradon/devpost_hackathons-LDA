Inspiration

We wanted to create a way that for people to easily reduce their plastic usage.

What it does

Our app shows restaurants near you and displays their level of eco-friendliness right on the map. The eco-friendliness data is crowdsourced from people who go to restaurants and leave reviews and take pictures showcasing the use of things like compostable bowls and cutlery.

How we built it

We built the front end with HTML and CSS, the back end with flask for python, the maps with the Google Maps JavaScript API, and the logo and icons with Gravit Designer. We hosted the project on a VPS through AWS Lightsail and used Domain.com to get the greenspot.tech URL.

Challenges we ran into

We had a small team and most of the technologies we used were new to us so we had to spend a lot of time debugging.

Accomplishments that we're proud of

We set up the server and DNS settings correctly and it's running live at https://greenspot.tech . We also used the Google Maps API correctly and were able to customize the look of the maps and pins and create custom interactions with them.

What we learned

We learned how to use many technologies and set up an app on an AWS VPS.
