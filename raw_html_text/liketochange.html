Inspiration

We see social media everywhere, everyday, in every form. What if we could use this powerful platform to actually support a cause?

What it does

Like to Change is a platform where users can upload pictures to raise money for a cause of choice since for every like, our company would donate five cents to the respective cause. We plan to get this money from corporate sponsors (perhaps in return advertise them), as well as personal gofundme's.

How we built it

We began our project by creating a list of ideas for our website. After agreeing upon one idea, we created a list of desired features and drew sketches of our website. We split up the website into different parts such as the UI, database, and functions. We developed the framework of the website (HTML), and proceeded to design it (CSS). We developed all the different aspects of a proper social networking platform allowing users to easily and efficiently access all possible tasks. We made many of our different functions with Javascript and PHP.

Challenges we ran into

We faced numerous challenges while working on this project. One of our biggest challenges was time, as we only had 24 to finish while battling sleep deprivation. Although we did successfully overcome obstacles, we had some trouble writing code for some aspects of our ambitious plans (database). However, at the end of the day, we learned a lot while we were working on this project and were able to pull through by making a functional website that we are all very proud of.

Accomplishments that we're proud of

We are proud of developing a functional website, which successfully allows the user to interact, while educating the user on different global issues.

What we learned

We used the pros and cons of each language to build an overall website, which combines various languages. This project helped improve our understanding for each of the languages. Also, we struggled on developing a database, and although we did not manage to complete it, we learned a lot through tutorials, and we hope to eventually grasp the concept enough to implement it in our website. Moreover, a huge project like this requires teamwork, and we learned how to effectively communicate as a team.

What's next for Liketochange

We would like to make our initiative more accessible to users by creating a mobile application version of liketochange.tech, so users on any platform, whether it be a phone, tablet, or laptop, have the ability to donate and create change. Additionally, we would also like to implement a database for the charities and provide the opportunity for people to become ambassadors for these specific organizations. These ambassadors would promote their respective cause.
