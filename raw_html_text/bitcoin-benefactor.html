Inspiration

Doing social good while making a fun game.

What it does

Allows the user to role play as a wealthy philanthropist bent on granting his entire fortune to organizations working for the greater good. Bitcoin Benefactor attempts to reprogram the average user's consumerist mentality by suggesting that giving can be just as satisfying as getting. At the end of the game, the user is linked to a series of actual charity websites and encouraged to make a real donation.

How we built it

With the Swift sprite kit on Xcode.

Challenges we ran into

Learning a completely new programming language.

Accomplishments that we're proud of

Learning Swift and finishing the application in time.

What we learned

How to use Swift to make iOS applications, how to design pixel art, how to work with image and audio files in the Xcode SDK, and how to run iPhone simulations.

What's next for Bitcoin Benefactor

Making it into an Android application (and worldwide fame).
