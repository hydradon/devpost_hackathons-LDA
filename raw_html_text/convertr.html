Inspiration

We saw a problem with having to search for the many different conversion formulas all in different places, so we decided to solve this problem by putting them all in one, easy to use app.

What it does

It will convert any base measurement to any other specified measurement.

How we built it

We built the framework with HTML5, styled it with CSS3, and built the logic of the program with Javascript.

Challenges we ran into

We tried to compile the app into an android APK, but ran into problems with Phonegap.

Accomplishments that we're proud of

Finishing this project!

What we learned

We learned a TON about Javascript; we're still new to the language.

What's next for Convertr

Next, we'll be adding calculations for many other different formulas
