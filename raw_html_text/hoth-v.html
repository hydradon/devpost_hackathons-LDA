Inspiration

* The hundreds of flyers all around the campus. * We hate having the images of the flyers in our phones which we eventually forget all about. We created this app so that it can easily obtain the date, time, venue and other useful information in a simplified format and automatically add it to our calendars.

What it does

  • Uploads the image of the flyer.
  • Obtains the date, time, venue, event name and other information from the image using Google Vision API
  • Displays all the flyer content in a simplified format

How I built it
