Inspiration

This is based on our original Bucket Boy Game at https://devpost.com/software/bucket-boy#updates and https://github.com/matthewlewis112/GameRepo/tree/master/KidC

What it does

This is a game about a boy escaping a prison, then seeking adventure outside.

How we built it

We built this game in Unity and used GIMP for sprites and assets. We started with a new code base. Unlike our previous demo, we implemented a visual color change for the player sprite, multiple powerups, added new graphics and audio assets, and implemented new entities like the Nano Bot enemy.

Challenges we ran into

Learning to work with Unity more in depth, and developing a level.

Accomplishments that we're proud of

The final scene of the demo and scripting the power-ups

What we learned

C#, Unity, and importance of sleep.

What's next for The Bucket Boy

Level and story development
