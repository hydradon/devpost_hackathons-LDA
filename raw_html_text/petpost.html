Inspiration

Kate owns a cat and we both love to use Instagram so we decided to make our own version for pets!

What it does

Make an account and post pictures of cats and see other people's pictures of cats.

How we built it

Using the Django framework with AWS Cloudfront serving images from an S3 bucket. We used bootstrap for some css styling as well as some custom written css. The end website is hosted on Heroku.

Challenges we ran into

Setting up Cloudfront and S3 as well as sending and receiving images took longer than expected.

Accomplishments that we're proud of

Using some minor aws software to display static and media assets. Creating a responsive website

What we learned

The process of using a CDN for images and files. How to use Bootstrap for a project.

What's next for PetPost

Implement more in depth features and finish styling the website. Setting up our domain from the Hackathon!
