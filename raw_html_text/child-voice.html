Inspiration

An web Application for NGO created using Content management system with WagTail Framework

NGO Name - Child Voice India

What it does

  1. It has admin page which allows to edit content and publish the content without help of developers 2.Completely Opensource. 3.Simple and Easy to edit contents 4.Allow then to send email and get notified 5.Gather Details of Volunteers 6.Allows volunteer to provide funds using PayU gateway
  2. Responsive web application can be easily orientated for mobile view

How we built it

Using WagTail,DJango,Python,Javascript,HTML5,CSS#,PayU API
