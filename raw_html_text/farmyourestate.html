Inspiration

Based on the StateFarm mission, we wanted to build on the "recover from the unexpected" part to aid users to face any adversity they may encounter. Essentially, helping people become more self-sufficient

What it does

It is an incentive based tool to increase communication between customers and agents to help solve any unintended problems that could arise

How we built it

We developed the front and back-ends natively using Swift 4 on Xcode.

Challenges we ran into

We faced issues attempting to deploy the application onto a device, so we had to implement it through the simulator offered by Xcode.

Accomplishments that we're proud of

We developed a well-functioned app within tough time constraints of 24 hours. We also enjoyed the user interface we applied onto our app.

What we learned

It was the first opportunity we received to work with Swift and Xcode natively to develop our app.

What's next for FarmYourEstate

We hope to set a standard for customer and agent relationships and hopefully allow other companies to build a better system together and to help things go right.
