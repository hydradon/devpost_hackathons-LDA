Inspiration

People have some pretty bad speech habits. We're here to correct that.

What it does

A mounted arm will raise up and slap you in the face if you say too many trigger words (such as "like").

How I built it

Use Google cloud's voice recognition API to detect trigger words. Communicate with an Arduino to drive the appropriate motors when a slap is needed.

Challenges I ran into

We tried using a Raspberry Pi at first, but it was too slow for the voice processing, so we had to move over to doing the voice processing on a computer and then the motor controls on an Arduino.

Accomplishments that I'm proud of

We made progress.

What I learned

Raspberry Pi's don't have spectacular processing power. Testing should happen sooner in case things break.

What's next for You like jazz?

Better mobility (right now it's tied down by power supplies and a computer).
