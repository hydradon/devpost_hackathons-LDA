Inspiration

We were inspired because we found it difficult to hold onto and save all of the people we were meeting in college, especially during new orientation

What it does

Our projects is a contacts app that lets you record down basic information about people that you meet so you can actually remember them in some way shape or form. This is all about keeping in touch with others!

How we built it

We built it using XCode and Swift.

Challenges we ran into

We were completely new to iOS development, so we found it incredibly difficult, especially learning swift as a new language.

Accomplishments that we're proud of

We are proud that we able to learn so much in such a short period of time, especially learning how to code with xCode and swift. Performing segues between screens was very difficult for us but we managed to figure it out

What we learned

We learned swift and the basics of xCode

What's next for Contact Network

We want to plan on expanding the app so that it has more features such that you can actually record down more information about people you meet
