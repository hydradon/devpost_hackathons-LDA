Investr

Investr offers a unique all-in-one investment advice experience packaged specifically for young adults and those new to investing. The application is comprised of three components: Investr Learn, Investr Analyze, and Investr Connect.

Investr Learn

Provides an immersive curriculum regarding personal finances and investing. There are little assumptions made regarding previous knowledge, so its an excellent choice for fresh beginners. The course specifically covers the topics of investments, risk, returns, stocks, bonds, mutual funds, and exchange-traded funds.

Investr Analyze

A unique machine learning algorithm is used to create a realistic financial forecast given only a ticker with the help of an API to pull stock price and fundamental financial data. Using this information, a multi-variable linear regression algorithm is leveraged to find a correlation between the stock price and other important indicators such as earnings per share, book value per share, and dividend yield.

Investr Connect

In an effort to extend learning beyond the application, Investr helps users find a Financial Advisor to discuss more complex deals and provide a more realistic portfolio.
