Inspiration

We were fascinated by the challenge which was formulated by Sitra. At the very beginning we had no idea how to deal with this problem because all of us had never before faced such complex problems. We are interested in this topic because as a researchers we understand that sometimes when we are looking for some information, we cannot even now that any particular data source (that can be very useful and interesting for research) can exist. That is why we decided to take part in this challenge and get into the inspiring world of scientific data and innovation.

What it does

EasyData is interactive catalogue for health and welfare data and tool for forming new innovative data combinations.

You can search for any information and the tool gives show, which data exists. It creates an innovative playground of different data sets and connections between them, suggesting you new data sets you can use and where you can find it.

It aggregates observations in different open data stores' datasets into simple metadata objects and allows to search in them.

Our proposal is each dataset from all the databases are aggregated to model which consists :

Origin Title Variables Keywords Language

When user inputs a search query, we make a search on Title, Variables and Keywords and present the results retrieved.
