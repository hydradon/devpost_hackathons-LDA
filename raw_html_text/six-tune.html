Inspiration

My partner loves to jam, but hates the tedious process of tuning each string of his guitar individually, so we created a solution.

What it does

Traditional tuners require the user to pick one string and tune it before moving onto the next, ours lets you strum once and tells you which strings need to be tuned and which they need to be tuned.

How I built it

We used angular.js coupled with html5, vanilla javascript and the web audio API to harness the capbabilities of html5.

Challenges I ran into

We were challenged processing the signals from the guitar strums into six distinct harmonics to then be able to analyze.
