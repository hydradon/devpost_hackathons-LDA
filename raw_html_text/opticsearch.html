• Optic Search

Optic Search is an image search engine. Upload any image, and our system will use machine learning to identify it, add searchable keywords, and index it in our database. All uploaded images are then publicly searchable.

Technologies:

  1. Clarifai (Deep learning API. Extracts keywords from images.)
  2. Algolia (Search API. Provides fast and reliable search functionality.)
  3. Google Cloud (We're using their servers to host the project.)
  4. MongoDB Stitch (Serverless functions to handle image upload and host them in AWS S3.)
