Inspiration

Yearning to spare innocent lives from the dangers they would otherwise be unaware of, such as the widespread death that resulted from oleander consumption in Sri Lanka.

What it does

Informs the user if the plant they took a photo of is poisonous or not

How we built it

Created our own plant database API containing poisonous and nonpoisonous plants, which we used to classify the user's input using machine learning software on XCode.

Challenges we ran into

Creating a plant API large enough to be accurate in plant identification.

Accomplishments that we're proud of

It worked!

What we learned

API, machine learning, how to use XCode

What's next for Snapplants

Location pinpoints of recent poisonous plant activity from recent users to inform other users of nearby dangers
