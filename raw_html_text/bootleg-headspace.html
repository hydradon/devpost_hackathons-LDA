Inspiration

Meditation is nice and cool, so I decided hey why not build my own meditation software(Well's it's actually an app), but you get the gist.

What it does

You play how many minutes you want to meditate and what environment you want to meditate in.

How we built it

We built it in Unity with some simple C# code.

Challenges we ran into

First was teaching my other two team members on how to use Unity. The second was how to implement what type of screen goes with each minute(i.e.: fifteen minutes in a beach area.)

Accomplishments that we're proud of

Getting the application to run. My team members knowing how to use Unity

What we learned

For me, not anything new. But my team members learned how to use Unity
