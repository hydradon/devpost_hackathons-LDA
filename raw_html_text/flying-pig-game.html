Inspiration

Our little game is inspired by a hot Japanese game.

What it does

Players can control the pig (which is the object) by using their voices. The louder, the pig can fly higher. We also set up some columns that make the game more difficult. If the pig crashes on the columns, game is over.

How we built it

We built it with python3 and pyaudio

Challenges we ran into

The volume package that we were intended to download online are all outdated, so we have to build a simple voice input function by ourselves.

Accomplishments that we're proud of

All of our team members only took intro to programming before, and it is our first time to participate in a hackathon. We are happy that we finished making this hilarious game within such a short time frame.
