Inspiration

Usability testing is expensive

What it does

Uses cognitive services to allow for data driven usability testing at low costs

How we built it

Control browser with selenium, desktop app on kivy, API & database in node, dashboard on React. Images of desktop, user, and voice are processed for sentiment, emotion, and level of confusion or confidence. Summaries are extracted from transcripts.

Challenges we ran into

Original idea was chrome extension — extensions don't have enough permissions to access the media devices we needed
