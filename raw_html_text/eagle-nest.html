Inspiration

We wanted to bring everyone together in a centralized platform for the BC community.

What it does

Connects both students and organizations with each other to foster a connected campus.

How we built it

Using Parse as the backend we used Swift & Xcode with 3rd party libraries to develop an application that can be used across all iOS Devices.

Challenges we ran into

Autolayout issues, difficulty in setting up Git version control. Encountered issues using 3rd party libraries we had never been exposed to.

Accomplishments that we're proud of

A clean, sleek, and polished UI/UX. It is a lightweight, yet, efficient app.

What we learned

3rd party libraries, Swift/Obj-C, Parse, and most importantly to work together effectively as a team.

What's next for Eagle Nest

Adding more campus functionality.
