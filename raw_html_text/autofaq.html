Inspiration

T-Mobile is all about customer care - so we thought to provide even more answers to even more questions.

What it does

Customers search for questions and the database tallies up how many times that question has been answered.

How I built it

Jazz Python API and NodeJS website

Challenges I ran into

Jazz query return values: Apparently "data" and "input" are unconditionally the only supported JSON keys.

Accomplishments that I'm proud of

Got a website to interact with a Python API which reads from a SQL database.

What I learned

Capabilities and limitations of server less architectures. Little bit of AWS.

What's next for AutoFAQ

Update the database on the fly.
