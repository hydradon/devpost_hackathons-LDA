Inspiration

Created for RIT's 2016 iOS App Challenge

Original inspiration was a gif of a girl chopping bamboo with a samurai sword ( http://i.imgur.com/X3DKg.jpg ). We wanted to create a game based around using attacks at strategic angles to deal damage to enemies. We thought this was a fitting idea and would play in nicely with the multiplatform theme.

What it does

This was a game built for a hackathon. It's intended to be a quick and simple 20 second interactive game.

How we built it

By coding with Swift

Challenges we ran into

Lots!! We ran into a lot of challenges with our game logic and we happily worked through them.

Accomplishments that we're proud of

Impressive user interaction, art assets and a compelling final game

What we learned

Teamwork, swift and developing for iOS

What's next for Woodland Warrior

Possibly continuing to develop and utilizing motion controls for the "slash" effect
