Inspiration

The NationalKidney Foundation of AZ has a lot of paper work and manually enters each field by hand which can be very time wasting and tedious.

What it does

We took all of the paper forms and turned it into an online portal to automate the submissions process.

How we built it

SailsJs generated RESTful API, & Yeoman Gulp Angular generated frontend. Bootstrap used for mobile scaling

Challenges we ran into

Where what data goes. Time

Accomplishments that we're proud of

The end result, how it tied together so well.

What we learned

Angular 1.5 components, nodeJs, gulp
