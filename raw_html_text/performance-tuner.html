Idea of developing a performance tuner striked me when i heard few of my projectmates discussing on how to deal with slow processing system.Target was to develop a user friendly and a single platform to clear all the unwanted data like cache,cookies,temporary internet files,temp data etc.

Performance tuner helps the user to handle with these unwanted data. Initially it handles only these data,but in the coming versions am looking forward to add more features into it.
