Inspiration

Interested in doing something with the Last.fm API and integrating into a platform we love (the Linux command line)

What it does

The game imports your scrobbled albums from Last.fm and shows their pixelated album covers in the terminal. The aim of the game is to guess the name of the album, with the help of a track from the album played while the album art flies across the screen.

How I built it

Using Python3 and ANSI escape codes for the terminal to achieve efficient rendering methods and fancy graphics (like the explosions!).
