Inspiration

Ever wanted to have a more interesting gas experience? Change it up, sometimes when going to gas stations I look for other things. Now I know if they will be available. Know my gas types, food, inclusions, everything !

What it does

We get gas station information, what they have, what they offer, when they're open, and anything else is included at the gas station, offers, promos, car washes, perks, food, and anything else.

How we built it

Using Ionic, Angular, Javascript, Html, Css, Express

Challenges we ran into

Using Angular was a challenge within itself. Thankfully, the Ionic documentation made the Angular part a bit easier. However, now the Ionic framework was a challenge within itself.

Accomplishments that we're proud of

We're proud that our API call works. We're proud that our logo works. We're proud that we get to submit this devpost on time.

What we learned

We learned that React-Native should not be used while it is still in beta. we also learned that the Ionic framework mostly supports Angular; therefore, we learned Angular.
