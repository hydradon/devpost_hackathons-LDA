• Welcome To cliffhanger

  • This is the Game creator where you give your game a name.

  • The game list will show all of the games that have been created. You can join any game at any time.

  • The creator of the game will start when they are ready. A prompt will be shown when they start.

  • When the game starts a prompt is shown and the players create a story based around it.

  • As shown here, a prompt such as "you will be assimilated" is given and players must elaborate off the topic.

  • Here is a photo of a group playing.

Inspiration

Cards against humanity was a fun online game, but it didn't give people an opportunity to add their own twist to the story.

What it does

It allows players to create their own story, line-by-line with a prompt.

How I built it

Node.js, websockets, express, gulp, jquery, animsition, bootstrap, handlebars, mongodb

Challenges I ran into

Time limit, William had to go home and sleep. Victor and Jacob went through HTML tutorials, and the whole group learned MongoDB with Luke's instructions.

Accomplishments that I'm proud of

The game was fun, we spent a while playing it.

What I learned

HTML and MongoDB

What's next for cliffhanger

We'll add some security and validation improvements, then launch it as its own website.
