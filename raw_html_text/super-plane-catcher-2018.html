SUPER PLANE CATCHER 2018

Inspiration

The inspiration for this project came from aircraft databases, the Hollom Trading Card Game and Pokemon Go! Ideas from these inspirations were combined in our project.

What it does

It is a session based program where users catch and trade aircraft cards of varying value with different users around the world.

How we built it

Three main parts were constructed, the html/javascript web app, python server, and MySQL database. The were then combined to create the finished app.

Challenges we ran into

Server communication to other parts of the project proved to be quite troublesome, but eventually partially solved.

Otherwise the main challenges of the project were inconsequential towards the completion of the project.

Accomplishments that we're proud of

We are proud of the integration of open data in our project, and how we use the API to retrieve the data.

What we learned

We learned how to use GitHub as a cooperative team, and helped eachother understand new coding techniques an practises.
