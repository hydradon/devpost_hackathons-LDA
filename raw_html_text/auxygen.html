Inspiration

The hassle of requesting music at a party.

What it does

Allows users to easily add songs to a Spotify playlist through a web application so the songs can be added to a real time queue.

How we built it

For authentication we used node.js and a Spotify Javascript library with ReactJS.

Challenges we ran into

Authentication issues and CORS

Accomplishments that we're proud of

What we learned

How to troubleshoot for problems you've never heard of.

What's next for Auxygen
