Inspiration

Real project for my community kitchen which displays nutritional values to users to make them more aware. This is because there has become obesogenic environment which is due to processed foods and unhealthy food availability. This will help users to be more aware of nutritional values and live a healthy life.

What it does

-It display nutritional values to users -Allows admin to submit food nutritional values -Allows admin to Schedule weekly food schedule -Allows users to order quantity of food -Authentication

How I built it

I built this app using android studio. For the database I connected it to firebase

Challenges I ran into

I was finding it difficult to display and retrieve data from the database and displaying it in different adapters such as recycler view and spinner.

Accomplishments that I'm proud of

I have a working app which stores and retrieves data:)

What I learned

I learned how to be a better programmer and better problem solver. This experience has made me capable of figuring out any problem I have.

What's next for CommunityKitchenApp
