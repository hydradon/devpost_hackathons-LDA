Inspiration

dVAC is a decentralized vaccination registry using Interplanetary Identifiers (IPID) for decentralized Public Key Infrastructure built on top of Interplanetary File System (IPFS).

What it does

IPID is a Protocol Hack to cryptographically resolve Decentralized Identifiers (DID) to the Interplanetary Name Space (IPNS) which serves as a permissioned distributed hashtable.

How I built it

At first I had hoped to use the Hyperledger project Indy to accomplish this, so instead, after reading and understanding how DIDs resolve to DDO, I realized that this could be accomplished using existing infrastructure with much lower barrier to adoption.

Challenges I ran into

Apparently, the WebRTC protocol is not compatible with webkit and therefore it did not work in browsers that that use it as well as the Apache Cordova iPhone app wrapper.
