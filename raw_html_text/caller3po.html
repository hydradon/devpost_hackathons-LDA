Inspiration

Phone calls can be frustrating if you're not a native English speaker. Whatever purpose a new English speaker may use this app for (e.g., speaking to customer service), it is our mission to make their lives easier.

What it does

Caller-3PO offers:

  • translations of live phone calls with captions
  • a complete conversation history in text format, with the option to translate

The conversation history is a versatile feature and may serve very useful for attendees of a large conference call to keep track of what was said!

How I built it

React-native front end Python back end SQL for persistency

Challenges I ran into

Dependency on synchronous I/O, and associated performance costs Using a non-dedicated server to host backend

Accomplishments that I'm proud of

Having a conversation between 2 instances of the application and watching the translations happen in real time
