Inspiration

Google Stadia's new game streaming service.

What it does

Through a browser tab, stream your controller input to a target device running a game, where a virtual controller is created. The target device streams back low latency video of the game.

How we built it

Golang server, WebSockets, WebRTC

Challenges we ran into

Streaming video is tough. So are sockets.

Accomplishments that we're proud of

"It works on my machine!!"

What we learned

Need to continue to learn a lot more about game streaming to beat Google.

What's next for Not Stadia

Make the video streaming happen through browser native calls and not OBS.
