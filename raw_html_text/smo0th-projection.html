Our project is an OCR for the automation of form digitization and creates insight from the digital version of the forms.

This was built using multiple languages from back end Python to front end web development which allows the non profit to view their information.

Sleep deprivation hindered some of the development but sleep is also a challenge.

We built a simple, robust and effective product for automated digitization of handwritten paper forms.

We learned that often the simplest methods are the best when it comes to solving complex problems.

We plan to add mobile photography and scanning along with the addition for other form styles.
