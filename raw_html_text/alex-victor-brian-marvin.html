Inspiration

What it does

Allows civilians to post to a public board about issues, and sends the most relevant/popular posts in a weekly newsletter to the representative of the area.

How we built it

Back-end server running using Node.js, front-end design done with HTML/CSS, and page executions done with JavaScript

Challenges we ran into

Our custom registration in order to use the Phone2Action API caused us to run into several roadblocks that were incredibly time consuming.

Accomplishments that we're proud of

Successfully implementing a registration/login mechanism

What we learned

How to use the Phone2Action API, how to create a custom sign-in system, and when to ask for help

What's next for LegiTalk

Automatic sorting/grouping of posts, upvotes/downvotes, search for posts, more advanced interface, more secure login
