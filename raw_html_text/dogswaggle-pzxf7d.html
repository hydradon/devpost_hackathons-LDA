Inspiration

A curated list of experiences based on your dog breed. Because a social dog is a happy dog!

What it does

We list all dog friendly restaurants, clinics (via Kinship API), pet friendly travel destinations (via United Airlines API).

How we built it

We’ve built a native iOS app with Swift. Used Kinship API, United Airlines API. Used Node.js + CoffeeScript + Heroku for integration with United Airlines API.

Challenges we ran into

We wanted to spend more time on it. Using United Airlines API was :)

Accomplishments that we’re proud of

We built a high-fidelity app within 48 hours. We learned new technologies (Swift) we were not completely familiar with. Designed a beautiful UI. Integration with United Airlines API - converting it’s SOAP API into a pretty RESTful JSON.

What we learned

New programming language - Swift Learned SOAP API

What’s next for DogSwaggle

More partnerships, monetization, pet insurance policies, IPO, happy dogs!
