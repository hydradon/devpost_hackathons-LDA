Inspiration

Lupi esta inspirado en mejorar con el uso de Tecnologías de Información un area de negocios poco explotada, los "Foodtrucks". Este servicio de comida es bien conocido a nivel mundial debido a que son fáciles de reconocer en cualquier area en la que estén atendiendo, ya sea por el gran tamaño del camión o la publicidad que tienen impresa o la buena calidad de la comida, sin embargo, los (chiefs) de estos restaurantes mobiles se arriesgan diariamente debido a que tienen que confiar en su instinto para poder sobrevivir porque desconocen si el negocio perdurará establecidos en alguna zona en especifica y es por esto que existe mucha incertidumbre de inversión para estos tipos de negocios.

What it does

Nosotros atacamos dos problema de incertidumbre que los Foodtrucks tienen:

  • El usuario: mediante el uso de una aplicación móvil los usuarios podrán saber en donde se encuentran sus Foodtrucks favoritas y/o cercanas.
  • Los Foodtrucks podrán utilizar nuestro servicio inteligente en la nube que les permitirá con un mínimo esfuerzo el reconocer las zonas propensas (hotspots) para poder venderle a sus comensales mas leales.

Toda esta información almacenada es utilizada por nuestro servicio en la nube que aprende diariamente de los gustos y preferencias de los comensales, por lo cuál, mientras más gente utilice nuestra aplicación mas fácilmente podremos descubrir estos "spots" importantes en las ciudades del mundo y ofrecerles un mejor servicio a todos los fanáticos de la buena comida.

How we built it

Para poder construir este proyecto utilizamos diferentes plataformas, lenguajes y servicios:

  • Usamos AWS para construir el API que es consultada por nuestras aplicaciones y WEB usando PHP y el framework Codeigniter en conjunto con la librería Craftsman diseñada para el mismo.
  • Para la aplicación móvil usamos Android siguiendo la guía de estilos de Google:Material Design.
  • Para la aplicación WEB usamos Javascript en conjunto con AngularJS para correr los procesos interactivos por parte del cliente.
  • Nuestro servicio inteligente (del cuál estamos orgullosos) está diseñado con el algoritmo de Rede Neuronal Artificial Multicapa en parte Java y Python para integrar los resultados de procesamiento de datos que consume nuestra API.

Challenges we ran into

Sufrimos de constantes desafíos durante el proceso de desarrollo del proyecto, durante el reconocimiento de patrones para poder entrenar nuestra RNA tuvimos que desarrollar simulaciones de entrenamiento en diversos escenarios, lo cuál consumió mucho tiempo y esfuerzo. Además podemos comentar que fue un reto el poder comunicar todos nuestros desarrollos y unificarlos en una sola aplicación totalmente funcional.
