Inspiration

We were brainstorming for ideas and came across that we all had been in situations where we had poor internet / no internet connection and actually needed internet to do some important task.

What it does

Send Samaritan a text message to

  • get directions
  • google search
  • post a tweet
  • read tweets
  • read mail You can get the response via text or voice.

How we built it

We used Nexmo as our communication platform. We used python for all our coding. We used Api.ai as our bot. We used Google search and directions api, and Twitter api.

Challenges we ran into

Integrating with Nexmo voice. As we all were working on different parts, integrating all the parts was a challenge.

Accomplishments that we're proud of

We did it !!!

What we learned

For all of us, it was the first time using Nexmo, bot AI. For some of us, it was the first time coding in python. But we had lots of fun and yes, learned a lot !!!

What's next for Samaritan

We are planning to make it more robust and scalable. We are also planning to add more features such as reading news, integration with Facebook etc.
