Inspiration

We're all passionate gamers and wish we could contribute whilst gaming.

What it does

Enables users to work hard for charity by playing hard on the go.

How we built it

We developed the app by Android Studios and the website using BootStrapped CSS and HTML.

Challenges we ran into

We struggled with integrating open-source games into the client application

Accomplishments that we're proud of

We registered and constructed a function website and application

What we learned

Library management is both moody and vital to the success of any application

What's next for Penny Arcade

We intend to implement leader-board functionalities and weekly fund-raising competitions. Additionally, we want users to be able to submit their own games to implement into our application. In order to do so, we would have to devise a way to ensure the security of such games to protect the users and content creators.
