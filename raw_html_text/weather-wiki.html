Inspiration

What most of the weather app does is that they have the limited availability of location and the weather forecast is provided with half-hour delay and that too not accurate.

What it does

It gives you the correct information of the weather of your town and with the delay of no more than seconds.

How I built it

I built it using API's which were updated regularly and also which provided me a lot of API requests without the server getting down.

Challenges I ran into

The biggest challenge was to work with API as I never did it before and after that finding an API was a huge task for me.

Accomplishments that I'm proud of

I finally made the API to work for me in my desired manner and that made me happy.

What I learned

I finally learned how to use API, how to make full use of them and more about parsing.

What's next for Weather Wiki

Next to weather wiki is predictions accurate enough that it is the sole app left in the world which people would like to use.
