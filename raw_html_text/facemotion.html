FER Sense

Motivation

Computer vision has enabled a number of possibilities including sentiment analysis of individual's faces. This opens applications in improved customer service, therapy, care and management.

How It Works

Every couple seconds, the application takes a photo of an individual's face and analyses its emotions to make live insights into the person's emotions including their happiness, sadness, fear, anger, surprise and neutral. It the transforms this information into visual graphics and also able to tell the user the emotional breakdown.

How We Built It

The application was built using Indico API for sentiment analysis, Python and also HTML. We also used flask and a text to speech API.

What We Learned

We learned that there were a number of technological barriers in linking various APIs to programs in order to create rich, interactive user interfaces and tools which are useful to individuals. However, the largest lesson we have learned that is the importance of leveraging individual team member's expertise and skills to develop a minimum viable product.
