Inspiration

That feeling when the Stack Overflow link is broken.

What it does

How we built it

We used a rails API running on AWS, which receives and sends data to the extensions.

Challenges we ran into

Accomplishments that we're proud of

We finally got AWS to host the server.

What we learned

How to make extensions on Chrome and Firefox.

What's next for m1ssinglink

We want to expand our service and potentially make phone apps for it.
