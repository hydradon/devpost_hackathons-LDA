Inspiration

An interest in incorporating music in algorithmic systems, and seeing what could be done using a machine to create something typically produced using human creativity.

What it does

It creates a randomly generated MIDI sequence that can be used in multiple music production applications. It allows the user to use limited customization options to tweak their MIDI sequence.

How we built it

Using python, and an additional python library from github, which was behind the brunt of the work required to convert python language to a MIDI sequence.

Challenges we ran into

This was actually our first true experience working with python code, which made this a very informative experience for our group.

Accomplishments that we're proud of

We were able to learn python with no prior experience in it and MIDI protocol.

What we learned

We learned to use object oriented design with python and reference MIDI protocol

What's next for CitrusHack2018

Well that's a great question, but this has inspired us to do another hackathon in the future.
