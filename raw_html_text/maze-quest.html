Inspiration

As inexperienced coders, we decided we would create a simple project that could teach us a little more about html and javascript languages.

What it does

Our project is a simple maze, consisting of barriers players must navigate around, solely for entertainment purposes.

How we built it

We first created a layout for the maze, a figure to travel around it, and then created the code for the movement and barrier functions.

Challenges we ran into

As inexperienced coders, we struggled immensely with the barrier functions.

Accomplishments that we're proud of

Our movement commands were smooth and worked well.

What we learned

We learned much about html and java, including syntax and logic that is applicable for us.

What's next for Maze Quest

A next step for us would be a randomly generated maze to navigate through.
