What it does

This program, given the files on a folder or a bunch of random files, stores them to different locations according to their formats. The user can choose which formats are ordered to which folders in their own computer. Butler has been made from scratch in 32 hours by 3 first-year university students, and is compatible with Windows, Linux and OSX.

How we built it

The entirety of Butler has been built in Python 3.5.1.

Accomplishments that we're proud of

We managed to make it work in three different OS, with barely any experience whatsoever in application development

What's next for Butler

We'd like to add a background functionality that would make Butler silently check for files in a given location, and manage them thereafter
