Inspiration

To try out Alexa APL and also for the prizes involved.

What it does

Rock Paper Scissors is a popular hand game. Challenge alexa in the fun game to improve your ranking and enjoy what will be your favourite pass time soon. This skill maintains user stats and these can be queried by asking alexa for the same.

How I built it

I used Amazon Alexa ask-cli to build the skill and added APL documents and screens using the APL Authoring tool offered in the Alexa developer Console

Challenges I ran into

Adding APL feature to the skill. But APL resources and docs, and videos on twitch channel helped a lot.

Accomplishments that I'm proud of & What I learned

Was able to learn how to build APLs for Alexa Skills

What's next for Rock Paper Scissors

I will add ISPs offering for the skill and add more user engaging features, may be a restricted rock paper scissors version.
