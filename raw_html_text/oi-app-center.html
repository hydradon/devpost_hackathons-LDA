Inspiration

In a decentralized world, managing meta data for apps should be in control of the app user and app publisher. There was an afford by One Platform Foundation to solve this issue in the Android world. OI App Center is the tool for the decentralized world!

What it does

It manages app meta data like comments for users and like product updates for publishers

How I built it

It is using Gatsby and Blockstack

Challenges I ran into

Make Gatsby and Blockstack work together nicely.

What's next for OI App Center

Add support for app store data, improve design.
