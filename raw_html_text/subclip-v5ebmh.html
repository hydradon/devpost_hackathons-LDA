Inspiration

Say you donated to your favorite streamer and they gave you shoutout right after. That was an awesome experience for you, and sometimes you want to go back relive it. That's what SubClip is for: after you subscribe, we create a clip of the stream, and keep a link for your future viewing pleasure. Show your friends' your streamer shout outs!

How we built it

Front end

Javascript

Backend

Node server handling authentication requests from the streamer. Dynamo DB is used for storing authentication tokens and clip links.

Challenges we ran into

We weren't able to show videos in the extension, so we had to use a panel in order to link out of the current view.

Accomplishments that we're proud of

Authentication per-channel via broadcaster tokens.

What we learned

The dev rig is awesome. Webhooks are very useful for live updates.

What's next for SubClip

We want to add different donation types, iterate on the UI, and add more configuration options for streamers.
