Inspiration

As a part of the Association for Women in Mathematics, we wanted to do something math related. We know how annoying plugging equations into Wolfram Alpha, and thought that it would be easier for Google Home to do that for you.

What it does

Finds derivatives and integrals, after you tell it the information it needs. It's also supposed to send the answer to snapchat and makes a filter out of it.

How I built it

Using Dialogflow, an api maker in Google Cloud Platform.

Challenges I ran into

Getting the program to output the answers.

Accomplishments that I'm proud of

Being able to code it in a language that none of us know.

What I learned

javascript in Dialogflow is a modified version of it and is very annoying and confusing to use.

What's next for Math Wolf

Try to finish it and make it functional!
