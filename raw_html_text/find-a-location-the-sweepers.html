Inspiration - Our inspiration is the problems our parent faced when coming to Canada.

What it does - It allows immigrants to access businesses that speak the immigrants native language.

How I built it - First we designed some sketches, the we implemented these sketches on Processing (a java based API)

Challenges I ran into - Thinking of the solution and the problem and some struggles with the code.

Accomplishments that I'm proud of - Being able to build the application on Android.

What I learned - We learned how to find problems in our daily society and then come up with solutions to these problems.

What's next for Find A Location - We will improve the design and functions of the application and remove all the small bugs.
