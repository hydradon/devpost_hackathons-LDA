Inspiration

We love traveling. I am bad at planning.

What it does

tripMUC offers an interactive way to plan a weekend trip to Munich by suggesting current events, sightseeing, shopping and other activities. The user can create a time table for the future journey.

How we built it

We created a simple 3-step form with HTML, CSS and JS. The data is from the Munich website and the open data portal. When necessary we crawled the data with python.

Challenges we ran into

Too much data, too little time. Put enough for a prototype.

Accomplishments that we're proud of

It looks pretty and fits into the official branding.

What we learned

  • Parsing HTML is error prone
  • Aggregating data is time intense
  • There are a lot more things in Munich than we thought

What's next for tripMuc

It would be nice to keep it up to date and add more data but it would take a lot of time.
