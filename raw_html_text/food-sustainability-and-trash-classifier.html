Food scarcity in developing and underdeveloped nations has been increasing day by day and as per statistics provided by FAO, nearly 30% amount of food wastage globally and this ultimately leads to starvation.

Basically, we developed an application with two features:

  1. Garbage Classification
  2. Food Wastage Solution

We built the frontend part using the react-native framework and backend logic is developed in python using the flask framework. The object detection part is done using google cloud vision API.
