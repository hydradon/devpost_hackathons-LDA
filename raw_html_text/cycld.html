Cloud Vision API enables developers to understand the content of an image by encapsulating powerful machine learning models in an easy-to-use REST API.

What does it do?

  1. It classifies images into thousands of categories (such as, “sailboat”)

  2. Detects individual objects and faces within images

  3. Reads printed words contained within images

Future Plans: Amongst being able to measure if an item is recyclable, we envision our product to be able to accumulate data from a user’s recycling habits (with granted permission via user agreement) and track their carbon footprint.

There can be an incentive to incorporate our app in a user’s recycling routine if given the opportunity to have corporate sponsors.
