The core of this app is the Microsoft Language Understanding Intelligent Service. We made a smart chat bot using the service. Empowered by machine learning, it can answer various questions after feeding it with data. It has the power of understanding human language (NPL). That is, no matter in what way you ask it, it will find the key in your question and generate an answer. The chat bot currently support Android, desktop and web version.
