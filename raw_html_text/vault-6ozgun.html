Inspiration

As experienced volunteers, we found that it is sometimes not intuitive to find the positions that are available for the events we would like to take part in. There also seemed to be no efficient procedure to recruit more experienced volunteers when it came to crucial tasks. Thanks to ConUHacks and the challenges given by CGI, DRW, and GSoft, we were driven to create this website tackling that a problem we have all felt frustration towards.

What it does

It allows companies to manage and create volunteer events as well as gives volunteers the opportunity to find events easily and participate in them. Additionally, as a volunteer or event manager, you can see statistics such as your performance in past volunteering events.

How I built it

We used a combination of Vue js for the front end and node js for the backend with MySQL for the database.

Challenges I ran into

The lack of time. We had a hard time choosing which features should be prioritized for the demo.

Accomplishments that I'm proud of

We have a pretty solid API and we have implemented almost all of the features we initially had in mind on the interface.

What I learned

Better management skills and choosing which features to prioritize.

What's next for Vault

There are a few features left to integrate for volunteers and managers to be able to work seamlessly using this interface. It would be awesome to implement a way for people attending this event to use this platform to sign up and to get event details.
