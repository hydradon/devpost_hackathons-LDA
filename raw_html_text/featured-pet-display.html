We wanted to make a versatile app that could be used on Facebook or by web link.

This app presents a random featured pet from the current Pets Fur People database to be placed on a Facebook page or be placed on any web page. Each time this app is refreshed, it randomly presents a different pet from the Pets Fur People database. The user can then click through to see the detail of that featured pet.
