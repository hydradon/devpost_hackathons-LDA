Inspiration

I wanted to make a game that could interact with the player in an interesting and fun way.

What it does

It's a game that gets angrier as the player progresses. It tries to hide the continue button as the player keeps clicking it.

How I built it

I used XCode to build this game for iOS. I used all the basics that the software provided.

Challenges I ran into

I had to figure out smart ways to connect two different screens in an interesting way. I had to channel my creativity in order to finish this.

Accomplishments that I'm proud of

I'm proud that I could make it stylish. At first, it was plain text on a screen. Now, it has color to it and looks better.

What I learned

I learned how app development for iOS works. I also learned the similarities between different phones and the ways to code for them.

What's next for Continue

I really want to extend the game into more intense levels. I also want to add in some animations to add to the fun.
