Inspiration

Unhealthy diet plans and dietary confusion, inspired us to develop an easier way to create healthy habits.

What it does

It provides multiple meal plans to guide users through their weight loss journey.

How we built it

We built it with html and java script.

Challenges we ran into

The most prominent challenge that we faced was to overcome our inexperience, referring both to coding in html and participating in hackathons. The difficulty of learning while creating a project created a challenge all on its own.

Accomplishments that we're proud of

We are proud that we succeeded in making a functional website that includes the basic information that we set out to provide on the website.

What we learned

From a technical perspective we learned the fundamentals of html as well as java script. In more of a general way, we grasped the teamwork and time management required to create a project like this.

What's next for Health A

Health A will transform in an application that users can consult for everyday use, as well as providing additional information on health, diet, and fitness.
