Inspiration

Create an integration that will link Woocommerce and XERO with no need of configuring and creating ssl keys . Users require only few clicks to install and configure this integration .

What it does

Woocommerce orders are saved as invoices in Xero . Integration supports any type of tax can be applied in Woocommerce and creates that tax in XERO on the fly .

How I built it

PHP

Challenges I ran into

Woocommerce supports applying different taxes based on regions in client address . Creation of this taxes on the fly was most challenging .

What's next for Woocommerce to Xero Integration

Iron out any remaining bugs and convert it as Partner app for XERO so users don't need to authorize app every 30 minutes as now .
