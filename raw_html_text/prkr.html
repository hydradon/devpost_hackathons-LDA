Inspiration

When Hummam's father got a parking ticket, he decided that he wouldn't let any Montrealer get a parking ticket ever again, and created an application to decode the cryptic signs on most Montreal streets.

What it does

Take a picture of any parking sign, and the application will tell you if you can park there, and for how long.

How I built it

The mobile frontend is made with Flutter, and the Python backend runs on Google App Engine. The backend sends the image to Google Cloud Vision to be analyzed. The application also uses data to present fun facts about parking in the city.

Challenges I ran into

Trying to get Firebase working (we gave up in the end)

Accomplishments that I'm proud of

Getting the Cloud Vision API working

What I learned

We learned a lot about the Google Cloud Vision API

What's next for prkr

Better accuracy and faster serverside optimization. Maybe in the future the machine can be implemented in the Montreal parking app.
