Inspiration

Stock data is hard to visualise and inaccessible for a less mature audience, we have changed that.

What it does

Allows users to gamify the stock market, buying and selling stocks with the use of gestures.

How I built it

Leap motion, Javascript backend, Swing + Awt frontend, CapitalOne API, Jackson

Challenges I ran into

Fighting with Jackson right up to the deadline.

Accomplishments that I'm proud of

LeapMotion grabbing gestures and full interactivity.

What I learned

How to use the leap motion, more advanced vectors and scaling.

What's next for StockMotion

Web based.
