Brief Overview: Our program adds additional words that fit into the existing dialogue of the pasted code. We used an API by Xerox to determine the part of speech (with context) of each word. With this, we were able to decide where to place additional adverbs and adjectives.

What's next for Word Count Optimizer: Using the same type of strategy (and additional use of the API) for further language processing and determing which words to cut.
