Inspiration

Meeting people in general can be challenging for some. Our app helps give an alternate medium for meeting new people. Tagalong is about using technology to break down barriers and connect people with similar interests and encouraging them to go out and do something together.

What it does

Tagalong allows users to join or host a group-event in the spur of a moment.

How we built it

Our development team decided to interface Tagalong in two ways. The first is a mobile application environment which features a beautiful UI. The second interface we implemented to access the application is a text messaging bot. Simply, text FIND to find group events or HOST to host group events to 213-438-9280 (We change the number every now and then so it might not be updated after submission). The bot implements Tropo, our provided SMS platform, and Python as a backend language for routing user input to, Zapier, the backend framework. The app landing page was built with HTML, CSS, and JavaScript.

Challenges we ran into

The biggest challenge we faced was implementing Tropo's Scripting API with a backend for two-way SMS exchanges.

Accomplishments that we're proud of

Accomplishments that we're proud of include the creation of an amazing UI/UX for our mobile app interface as well as eventually implementing Tropo's API seamlessly into our application.

What we learned

During our experience we successfully distributed work among proper individuals based on experience and skill-set. Some of team members learned how to use new technologies such as: UI design with Sketch, Python/Backend frameworks, and RestfulAPIs.

What's next for tagalong

We believe tagalong has the potential to successfully eliminate a pervasive issue among demographics that find meeting new people to not be an easy task. Future implementation includes machine "understanding" of regular English sentences.
