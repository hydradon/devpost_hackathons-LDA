Inspiration

We found the idea of Mixmax very appealing and wanted to do something with the SDK.

What it does

This project is an enhancement to Mixmax that allows the user to place charts in their emails. When the enhancement is opened, the user is shown an easily modifiable spreadsheet where data for the chart can be placed. At the moment, the user can make scatter, bar, and pie charts.

How we built it

We used JavaScript, html/css, Node.js, and the Mixmax SDK.

Challenges we ran into

One of the main challenges we ran into was trying to figure out a bug related to cors.

Accomplishments that we're proud of

This project did not crash and burn.

What we learned

We learned how to use the Mixmax SDK.

What's next for Charts on the Go

We can implement more chart types and figure out how to further simplify the design of the front-end.
