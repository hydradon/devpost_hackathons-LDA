Inspiration

We wanted a hardware target and the keyboard looked cool

What it does

It can be used as a volume indicator for .wav files - by default we play one of Tchaikovsky's track. It can also be used as a very crude sound synthetiser, and includes a small "follow me " game.

How we built it

With the greatness of Python alongside lots of keystrokes.

Challenges we ran into

Using the SDK with Python all the while having to use Windows.

Accomplishments that we're proud of

It works. Kinda.

What we learned

PythonPath + Windows = pain and suffering
