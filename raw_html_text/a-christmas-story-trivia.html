Inspiration

I LOVE watching A Christmas Story every year. I wanted to create a skill that would allow me dive into A Christmas Story more and expand on my Alexa skills.

What it does

The skill is a trivia game for the movie A Christmas Story.

How I built it

I used the Alex Skills trivia game template.

Challenges I ran into

Finding enough trivia to make a good game that can be played multiple times.

Accomplishments that I'm proud of

This is my best trivia game! I hope you enjoy it.

What I learned

I learned how to build a trivia game in the Alexa Skill environment.

What's next for A Christmas Story Trivia

Perhaps more questions this year!
