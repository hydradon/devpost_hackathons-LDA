Inspiration

Being freshmen, we always need help. This web-app is a simple way for students like us to collaborate with study buddies.

What it does

Study Buddy puts students in touch with peers in their class through an online chat and canvas.

How we built it

Using Node.js and GitHub we were able to use the necessary APIs to make this project possible

Challenges we ran into

Formatting as well as traversing and transferring data with Node.js

Accomplishments that we're proud of

We are very proud of the course page and how we were able to use a Rutgers API to gather student data.

What we learned

we learned a lot about GIT and bootstrap.

What's next for Study Buddy

We plan on adding a live video and audio feed as well as access to real, qualified tutors. We are hoping this provides assistance to students like ourselves.
