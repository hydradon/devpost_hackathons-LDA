Inspiration

While studying phone calls and messages becomes distraction and to prevent that, it helps to know if checking phone at that moment is important or not.

What it does

Bulb glows to a specific color based on the incoming call which is either individual or added to a group.

How we built it

Using Android Studio, Telephony manager API and Ilumi bulb API

Challenges we ran into

Connecting the phone call state (Telephony manager) to bulb to execute at the same time

Built With

  • android-studio
  • ilumi
  • ilumi-sdk
