Inspiration

Give the ability to people to create trusted data for the city and help the home buyer or renter to make a good decision before acquiring a place to live.

What it does

For the first version, we focus on detecting pest in the city of New York. With certified devices, we are able to monitor pest activities in building and report it to a blockchain. Data will be used for landlord to be notified when the building have a pest problem, to the city to coordinate pest eradication.

How we built it

Our proof of concept use ESP8266 with Ultrasonic Sensor to detect Rats passing near by. It send information trhow Wifi/Rest API to Microsoft Cloud Azure. Data are stored in quorum database. A node.js server is able to send data to a web map for visualization.

Challenges we ran into

Connect logic app ESP 8266 - 3v HC SR01 - 1v Inserting smart contract to database

Accomplishments that we're proud of

We made ESP8266 with Ultrasonic Sensor work. We made the map work We made the smart contract to work with our database

What we learned

Using D3.js to make map for visualization.

What's next for IoT Pest Detector

Certify sensor / device Different pest detection Rats, Cockroach, Mosquitoes, Ants, ... AI for detection accuracy 3rd party usages / Open data
