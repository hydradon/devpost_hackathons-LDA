Inspiration

One of our teammates is afraid of ferris wheels so we decided to take that to our advantage and just have fun making a random fun game.

What it does

This is a two dimensional retro game where the main character is trying to run away from the Ferris Wheel of Misery.

How we built it

We used pygame and python to develop the game.

Challenges we ran into

We spent spent two hours just trying to download the software, then got stuck on the game play and character movements. We were learning as we went, and so that ended up costing us a lot of time because our code got really jumbled and unorganized.

Accomplishments that we're proud of

We made a movable background and enough scripting for the beginning animation to work. We also learned a lot more about Python and Pygame, and now recognize different ways that next time we can make things more organized and better working.

What we learned

We started learning more about how to use pygame, and that it is important to stay very organized. We learned how to work past a lot of our mistakes, recognizing our errors and problems. We also learned that labeling everything clearly makes work a lot easier.

What's next for Ferris Wheel of Misery

In the future, the game will have its game play finalized and have more animations to include the plot and pineapple power ups.
