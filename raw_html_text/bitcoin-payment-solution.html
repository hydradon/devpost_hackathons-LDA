• Pay with BitCoin!

Inspiration

We wanted to find out more about processing BitCoin Payments

What it does

Creates a BitCoin request using an existing API (coinbase), and uses an actual CoinBase wallet to create a request

How I built it

C#, CoinBase API, HostedCheckout used as a launching point

Challenges I ran into

Finding an API compatible with VS 2013 that had the functionality we needed. (It would have taken too long to get a license to a newer version of VS)

Accomplishments that I'm proud of

Actually integrating to CoinBase and my BitCoin wallet.

What I learned

We all learned a lot about BitCoin payments

What's next for BitCoin Payment Solution

Would need to use oAuth2 for authentication to wallets other than my own. Transfer code that is in HostedCheckout to our own API. No redirect or text entry needed once 'Submit' hit - would integrate to user's wallet & merchant's wallet seamlessly. An API would enable our other payment platforms to also accept BitCoin.
