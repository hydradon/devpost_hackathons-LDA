Space lapse is intended to be an interactive chronology of every manmade object that has entered space. At the moment it runs a basic simulation of the planets in the solar system, and in the future it will display every spacecraft on the trajectory it followed as it went to space.

I built it from scratch using Three.js, a javascript framework for displaying 3D objects in browser. Textures of planets, and orbital parameters were sourced from several space databases on the internet.
