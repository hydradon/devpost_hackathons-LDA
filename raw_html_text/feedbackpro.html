Inspiration

Getting customer feedbacks is not that difficult, but the data might be hard to collect, maintain and analyze. We have built a system to help small business owners or startups to collect and analyze customer feedbacks in an easy way.

What it does

  1. Collect customer feedback. Users can use our online form, web API or mobile app to collect cusomter feedbacks. Our mobile app even support speed recording!

  2. Analyze customer feedbacks. We use IBM Watson's sentiment analysis API to analyze customer feedbacks and generate a score, which help help better analyze the data and trends.

How I built it

We have built a functional backend using Node.js and Mongodb that allows data collection and analysis. Our public API can be used by our customers. We have also built a mobile app using ionic, which can record customer's voice and parse them into text and then save them into our system.

We have used IBM Watson's AlchemyLanguage API to do our analysis.

We used a domain .tech domain name.
