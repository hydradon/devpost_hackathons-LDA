Inspiration

As a team of freshman and transfer students, there are activities and events we want to be involved in but we are not aware of all the activities that go on around campus.

What it does

Aggie Calendar allows organizations to add their events in a master calendar for all students to see and join. Students can add these events to a personal calendar.

How we built it

We used Android Visual Studio written in Java

Challenges we ran into

We found that dates are hard to format within a calendar

Accomplishments that we're proud of

The entire app was made by inexperienced beginners to android development.

What we learned

We learned planning is important and goes a long way

What's next for New Aggie Calendar

Fixing bugs, allowing new events to be created, a informations page
