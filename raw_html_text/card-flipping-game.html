The main aim of this project is for learning, and relaxing in break time. I was inspired by the reputation of the game Arena Of Valor, that most of my friends are their fans , so I build game involve to their generals, so that I and my friends can play it for fun. Concurrently, I can train my developing skills with HTML/CSS, Javascript and Vue.js.

This project was built in 2 days using HTML5/CSS, Javascript and Vue.js. The biggest challenge is I had no knowledge about any JS framework except jQuery. But after a few days studing Vue.js myseft, I won it!

And now I want to share my archivement for everybody. This game was deploy on cardflipping.000webhostapp.com so everyone can enjoy it.
