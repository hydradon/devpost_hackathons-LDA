Inspiration

It is the most common skin condition that affects over 50 million people a year.

What it does

This device will monitor your skin blemishes to accurately determine the effectiveness of the product you are using.

How I built it

Using a Spectrometer, Arduino, breadboard, Raspberry Pi, and some wires. We were able to develop an AWS api based on lambda. This lambda function utilized aws app sync to establish a subscriber client based model for live data on a web app. This data is subsequently stored in a database.

Challenges I ran into

Communication between the raspberry pi and Arduino. Developing the U.I to display live data.

Accomplishments that I'm proud of

Fixing the communication error between Arduino and RPi. Learning how to format a react page.

What I learned

There is a substantial lag between data being sent to the server and user being able to view it.

What's next for EpiTech

Continue developing and incorporating a camera for machine learning algorithms and hopefully exapand application by looking into skin diseases.
