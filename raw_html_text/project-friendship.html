What it does

Giving back to the community. we tried to build a app that would connect mentors with the students according to their preferences.

How we built it

We build it by creating a backend using clojure with compojure-api and a front end with jquery that consumes that api.

Challenges we ran into

Inability to serve the api on the local wifi as well as being new to clojure.

What we learned

We learned a lot about developing in clojure as well as learning to build a front end in javascript that consumes an api.
