Inspiration

It is hard for professors to get visual feedback from the students during classes.

What it does

Our app helps the professor get an incentive of the students behavior during their lectures in order for them to know how to improve the content.

How we built it

We used Angular 8 along with Google cloud platform to build this platform

Challenges we ran into

  • Passing the video feed from students to professor's view.
  • Analyzing the data we get back from the vision API and use it in our purpose.

Accomplishments that we're proud of

  • record video and send it to google cloud API
