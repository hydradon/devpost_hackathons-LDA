Inspiration

In medically under-served areas/countries, there is an extreme shortage of CT scanners which are essential for all sorts of diagnoses and for surgical planning. X-ray machines, however, are already available across almost all areas.

What it does

We therefore address this humongous gap of resources on medical imaging by creating a software which would reconstruct and display the 3D structures from just a few plain film radiographs.

How I built it

With Matlab codes, Open Source CT images, medical knowledge, oh, and, with PASSION!

Challenges I ran into
