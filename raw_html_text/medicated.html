Inspiration

My grandma was prescribed multiple medications and she did not know if she could take any ibuprofen for the pain she was feeling, thus the idea for this project was born.

What it does

The project can help show medications that shouldn't be combined together, as well as allows users to submit their own bad combinations of medicine(mostly doctors). The app can also help alleviate some symptoms by suggesting medicines to take. After that, we move on to a medical facility locator, which, used on other platforms could be further advanced with more usage from an API.

How I built it

We used appLab to construct a layout along with coding front-end and back-end.

Challenges I ran into

We had a very hard time implementing API to locate health facilities in Applab because it was not supported, so we improvised how the API would run on another platform.

Accomplishments that I'm proud of

We were able to come up with a useful idea that would help many people who struggle with over-the-counter medicine.

What I learned

We re-learned javascript while coding this.

What's next for Medicated

We hope to be able to move this onto other platforms in the future to implement several API's.
