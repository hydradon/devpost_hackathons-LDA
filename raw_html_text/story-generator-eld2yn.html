Inspiration

We once heard someone told funny and random stories.

What it does

It generates a random story with user's typed name inserted.

How we built it

Specify the HTML templates and use random number function to fetch one of them.

Challenges we ran into

We would like a way to integrate many HTML templates so that it generates dynamically.

Accomplishments that we're proud of

We were able to finish it and had it worked!

What we learned

Familiarized the modern techniques of web design.

What's next for Story Generator

Dynamically generates stories, allows users to upload their own story templates, and allows to share to website.
