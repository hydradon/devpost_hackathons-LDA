Inspiration

Same that https://devpost.com/software/daxes-hacks-youtube-channel I really like that Channel and I would like to get last updates from channel in my Alexa.

What it does

Tell you the number of subscriptors to my Channel 'Daxes Hacks', and can tell you the last video uploaded. If you are using a compatible device (with screen) you can see the thumbnail for the video and the video description.

How I built it

Using a Lambda bucket running NodeJS

Challenges I ran into

Certificate my Skill

Accomplishments that I'm proud of

I developed a common code, where you can build your custom skill for your or other Youtube Channel, only changing .env parametters, and using the same entire code.

What I learned

APL, Still now I only was building Speech and Cards Skill Responses, but now I could add APL Responses, differents for each device.

What's next for "Luisito Comunica" Youtube Channel

I would like stream uploaded videos to the Channel, allow search videos and list recent uploaded.
