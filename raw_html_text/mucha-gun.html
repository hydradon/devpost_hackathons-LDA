Inspiration

Starting with wonder how a First-Person Shooter game would look like in a smartwatch, the goal was to make it with as minimum element as possible with as much fun as possible. The visual idea inspired from mobile game called Slashy Heroes but with more bright theme instead of spooky.

What it does

Set yourself into classic arcade world where you busting bogeyman with guns and candies!

  • Select your favorite weapon
  • Each weapon have different type of use
  • Use the bezel, swipe, or tap on both side screen to aim the enemy
  • Upgrade your Health, Power, Reload Speed, and Max Ammo through countless wave
  • Enemies got harder the longer you play
  • Survive, get the highest score to unlock more guns
  • Collect Pink Bolts through gameplay to purchase guns

How I built it

I develop games by my own using Construct 2, a perfect tool for non-programmer like me. First I create the game itself, then I export it to Tizen platform. The exported file then moved to the Tizen-IDE / Tizen Studio which then will be packed into .WTG file to be able to run on Tizen Smartwatch such Samsung Galaxy Watch. Construct 2 can use Javascript in order to use various function on the device.

Challenges I ran into

  • Finding unique control for each weapon
  • Different enemy behavior
  • Balancing difficulty & reward
  • Minimum UI/UX element
  • Tweening animation
  • One man dev :')

Accomplishments that I'm proud of

The game was nominated as 3rd Best Wearable Games on Indonesia Next Apps 2017.

What I learned

I really learned so much about code & system management. During working in this project there were lots of missplaced codes & functions which confuse myself when adding new features or do some optimization.

What's next for MUCHA Gun

Rebranding and re-release it with easier & understandable title, also to add little story, character introduction, more guns, areas, enemies, and boss level.
