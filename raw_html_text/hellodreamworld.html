Inspiration

We wanted to create a fun math game for kids!

What it does

This program guides the player through a text-based maze full of spooky encounters with iconic Halloween characters. In order to win, the player must beat the monsters by correctly solving custom elementary level arithmetic.

How we built it

We coded the entire program on Eclipse using Java.

Challenges we ran into

Only two members of our 8 member team had previous programming experience in Java.

Accomplishments that we're proud of

Created an awesome game while still utilizing everyone's talents.

What we learned

How to code in Java as well as how to contribute to a team with seemingly minimal programming knowledge.
