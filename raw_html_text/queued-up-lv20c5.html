Inspiration

Wanted to listen to music we all liked

What it does

creates shared queues where anyone can use an extension to add a song from music site to queue

How we built it

Hosted data in MongoDB on Google Computing Engine and created flask app to work as music player. Used swift to create an extension to add songs from common music playing apps like Spotify or Apple Music

Challenges we ran into

Figuring out how to configure database with cloud Understand Swift

Accomplishments that we're proud of

It works completely and is really useful!! Lots of our friends want it.

What we learned

Flask, cloud architecture, swift

What's next for Queued Up

Develop to to have more advanced UI and work with amazon music
