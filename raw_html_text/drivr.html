Inspiration

Argument between significant other and I about whose the better driver. Now we'll know.

How it works

Phone is planed in a secure location in the car. User activates the app and begins driving. The accelerometer sends data to our algorithm, which is converted to a value that is visualized on a graph in real time. At the end, the vales are averaged to give the user a final "driving/smoothness" score.

Also have an auto-track mode that can run in the background. The app tracks the users drivers, and stores the data locally. The app uses POI and User Activity function to track when the car is driving.
