Inspiration

Problems of traffic flow during the pass of the years.

What it does

It detects cars, and according to priority calculated with the number of cars on the street and if the street has an emergency car.

How we built it

We built an algorithm and implemented it with unity for demonstrating the functionality of it.

Challenges we ran into

Build a realistic demonstration in this short time.

Accomplishments that we're proud of

An algorithm that works in must cases.

What we learned

Go from the easiest to the complex features in order to not run out of time.

What's next for SmartTraffic

Add Deep Learning and AI to enhance performance and use real life API's and data.
