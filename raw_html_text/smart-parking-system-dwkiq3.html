What it does

It shows all the nearby parking areas with distance and realtime update about each spot either its occupied or vacant

How I built it

Smart parking system is build using a Rasberri pi and an ultrasonic sensors, with raspberry pi as backend, and mobile to show empty spots

Challenges I ran into

Some of the challenges were to get the sensors to detect the car. With a reliable accuracy. One of the challenge was to use the right hardware. Initially we developed the system with intel Edison microcontroller board. It was hard to send data to the firebase database using it. So next we decided to use the Raspberry pi.

Accomplishments that I'm proud of

The Realtime update on mobile apps

What I learned

connectivity of sensors with raspberry pi, and pushing the data to firebase

What's next for Smart Parking System

Further we can replace the sensors with cameras monitoring the cluster of parking spots with Pattern recognition, to detect weather the spot is taken or empty.
