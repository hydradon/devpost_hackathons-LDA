Inspiration

We wanted to create a game where we could try to take photos that matched a description, like a photo scavenger hunt.

What it does

You take a photo, trying to match it to the list of tags. The more tags you match, the more points you score!

How we built it

We used Android, Clarifai to tag the images, and Firebase to store the database for the tags.

Challenges we ran into

We had problems getting Firebase to work correctly, getting a good-looking UI, and piecing the parts together.

Accomplishments that we're proud of

Learning the Clarifai API, learning more about UI Design, learning Java, having a working final product

What we learned

Java, API Usage (Clarifai, Firebase)

What's next

PvP mode, global leaderboards, an actual scoring algorithm
