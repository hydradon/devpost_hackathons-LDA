Inspiration

I keep track of all my expenditures on a register and sum them up at the end of the month.

What it does

This alexa skill does just that. I tell it to add some value to my wallet and it sums up and also keeps track of how long has it been since I first started this skill.

How I built it

I built it using node.js and aws lambda functions.
