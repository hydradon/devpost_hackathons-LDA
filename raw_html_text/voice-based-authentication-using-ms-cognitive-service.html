Inspiration

To provide secure way of bio-authentication . There are different biometric authentication methods available in the industry but relying on single metric is not always secure. Voice Authentication can add additional layer to existing system , hence creating it more secure.

What it does

It uses the voice sample to authenticate the user.

How we built it

We used Microsoft Azure cognitive speaker recognization api to distinguish voice sample and created a authentication system which leverages the same.

Challenges we ran into

Converting the microsoft supported wav file format from voice samples collected from multiple sources and devices

Accomplishments that we're proud of

Working system with application in website login.

What we learned

We learned how create and consume various APIs using secure way and leverage the same in multiple system using microsservices.

What's next for Voice Based Authentication Using MS Cognitive Service

We can use this application in various other domains like, IOT, Remote Login,Smartphone Authentication, etc.
