It simplifies and displays important weather information across the US, making it easy to process and read. It also gives suggestions on how to act.

Java, HTML, CSS, OpenWeatherMaps API, ZipCodeAPI, BreezometerAPI, maven, Springboot.

Too many request errors (409), bad request errors, 404 whitelabel errors.

Its a working product and is fully functional.

We learned how to divvy up work for everyone and think of new ideas on the fly.

The future:

Make the range of application extend to global zip codes. Collect data on skin types in order to better inform the user on how to protect themselves from UV rays, pollution, and so on. Better design (we only had a couple hours). Post relevant and useful products for adapting to different situations. Add forecasts.
