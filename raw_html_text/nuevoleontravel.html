Inspiration

We wanted to do a cool app, and the project caught our attention.

What it does

It is an app for tourists visiting Nuevo Leon. They can browse sites of interest, hotels, restaurants and events.

How we built it

Using Android with Material Design. We also Google's Places API to track the places that the user visits. We also built a web interface in PHP using the Laravel framework to visualize user data.

Challenges we ran into

Troubles with the Google Places API, also with some layouts of the app.

Accomplishments that we're proud of

That the app looks pretty good, and it works smoothly.

What we learned

A lot about Android development and the Places API, Material Design, XML, Laravel.

What's next for NuevoLeonTravel

Maybe some more development of it, I think that it has a bright future.
