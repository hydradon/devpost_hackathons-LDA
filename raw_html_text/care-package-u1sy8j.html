What it does

The web application provides simple UI that aims to be easy for an elderly person to use. It allows the user to keep track of their daily medications, journal entries, exercises, and survey of how they feel that day.

How we built it

We store the user data into a cloud database using AWS DynamoDB and JavaScript.

Challenges we ran into

There were many technical difficulties, as this is the first time for our team to be working with web services and databases.

Accomplishments that we're proud of

We learned new technologies and challenged ourselves.
