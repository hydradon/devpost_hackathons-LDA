Inspiration

We wanted to make traveling smooth inside and outside the airport

What it does

Guides a traveller through their flight details, their airports, and their destinations--flight, fun, and future. We not only keep track of the traveller's itinerary and airports but also prepare them on how best to appreciate, enjoy, and respect the culture of their destinations

How we built it

React Native

Challenges we ran into

Web scraping in JavaScript, the fetch API, and getting React Native set up

Accomplishments that we're proud of

Pulling together a solid proof of concept

What we learned

A lot about APIs, React Native, and JS web scraping

What's next for Pocket Trip

Hopefully full development
