Inspiration

While I was revising for my exams, i had a lot of problem due to the need to memorize a lot of informations in the same time so i decided to search for solution helping memorize faster. I found a font called "sans forgetica" : Sans Forgetica is a font designed using the principles of cognitive psychology to help you to better remember your study notes.( https://sansforgetica.rmit/ ) I tried it by myself and it was really beneficial, I also found that psychologists and scientists published a research telling that these three colors: blue, orange and green are very helpful - according to the article: ( https://www.shiftelearning.com/blog/how-do-colors-influence-learning ).

What it does

Starting from that we had the idea to make a web plateforme using this font and those colors for student where they can edit their revision, share them online and so as by making less effort learning them.

How we built it

It is a website that can provide revision cards of all topics depending on tags, with students all around the globe

Challenges we ran into

providing a soft and a minimalist solution

Accomplishments that we're proud of

The new idea that will help a lot of students during exams revision (so basically we will be using our application in order to learn more subjects)

What we learned

Teamwork and developing with new technologies such as php and some concepts concerning cognitive psychology

What's next for memorize

We aim to build a mobile version and add the feature of image recognition to scan notes written by hand and made them automatically with the special font and colors.
