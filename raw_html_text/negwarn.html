Inspiration

Negativity is the killer in any conversation. Especially Telegram group chats. If you can get rid of the negativity, you can make the conversation more productive and more friendly!

What it does

Monitors Telegram chats and warns you when your they're becoming too emotionally erratic. Uses the Haven OnDemand sentiment analysis service.

How we built it

Node.js based Telegram bot server which uses the Haven OnDemand sentiment analysis api to work out if the chat is being too emotional.

Challenges we ran into

It's difficult to put people down without being offensive. As a result, we implemented the --offensive flag!

Accomplishments that we're proud of

It works, and the bulk of it was working within 10 minutes of hacking!

What we learned

The Telegram bot api! It was a load of fun working with it, and I'd definitely use it for a hack again!

What's next for negwarn

Ports for WhatsApp, Facebook Messenger, you name it! negwarn will be on all your favourite messaging services!!
