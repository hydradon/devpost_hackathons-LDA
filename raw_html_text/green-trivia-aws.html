Inspiration

Mistakes in recycling has always been causing great economic loss. We want to build a application to teach users recycling and environment-protecting skills in a fun and interactive way.

What it does

Our hack is call "Green Trivia". It is an Alexa skill working with Amazon Echo on AWS. Green Trivia asks a series of questions to a user and score the user.

How we built it

Green Trivia consists of a voice interface written in node.js and a driving Lamda function.

Challenges we ran into

In limited time, our tasks include system design, getting familiar with node.js and gathering multiple choice questions about recycling.

Accomplishments that we're proud of

With great teamwork, we were about the process the workload and make the application run smoothly. We found very good resources online which meets our needs.

What we learned

Always try to simplify the design. A good decision in system design can save many work hours.

What's next for Green Trivia (AWS)

we need to add more questions to Green Trivia and include question-types other than multiple choice.
