Inspiration

For many college students, course scheduling is such a struggle. The school's scheduling system is very hard to navigate and use through the comfort of your smartphone. Most students just wait until they have access to a computer to schedule their courses. As a result of this problem, some students even end up scheduling late, not getting the classes they want or need.

To solve this problem, we decided to make a simple scheduling app for college students.

What it does

It helps college students schedule their courses.

How we built it

We used Android Studio, Java, Android 6.0 Marshmallow (API 23), Android Documentation and Slack (for communication)

Challenges we ran into

GitHub conflicts, design decisions, learning how to use Android Studio, scraping courses from WCU's course website, etc. Making a schedule is hard since you have to make sure there are no conflicts, sometimes classes get filled up, and sometimes you can't choose between different classes.

Accomplishments that we're proud of

Built an Android app.

What we learned

We learned how to build an Android app using Java and Android Studio.

What's next for Scheduler App

We would like to make UI more intuitive, attractive and fast.
