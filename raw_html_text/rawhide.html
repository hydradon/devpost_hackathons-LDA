What it does:

Maintains the genetic profiles of cattle on the blockchain and, allowing genetic herd management and fractional ownership.

How we built it:

We used existing Ethereum standards for tokenizing such as ERC-721 and ERC-20 that together resemble ERC-1155 for more efficient transactions. We used a Truffle Box to create an Ethereum compatible web dashboard that hooks into our smart contracts written in Solidity. We used IPFS to store genetic profiles and other metadata off-chain.

Challenges we ran into:

In development, there are many ways uses of technology coupled with new blockchain advantages that can be used in this industry- narrowing down a focus area on what could be done done right now was most important.

Accomplishments that we're proud of:

Giving ranchers a new business model and control in a traditionally volatile and high risk operation.

What we learned:

We learned a lot about the new blockchain laws passed in Wyoming, specifically the Series, LLC that we can use in our cattle management.

What's next for Rawhide:

We'd like to further decentralize the application such as the minting process. We'd also like to use Zero-Knowledge Proofs to allow data to be masked and monetized for data analytics.
