We wanted to build software that would allow users to print to the CMU Andrew printing system without installing any software. This project allows users to re-queue printjobs, print without drivers and also print from mobile devices. This is hosted on a raspberry pi running raspbian.

How to Use

Users create an account associated with their CMU id. Then users upload any files they want to print and queue them via the online interface.
