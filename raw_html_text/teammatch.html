Inspiration

The inspiration comes from years of frustration of teamwork projects done alone. We all have been there before and that is why we decided to built an app that helps you find teammates for projects

What it does

It helps you find teammates for projects. When the user signs in, the user is asked to put down his interests/goals. The algorithm then searches for potential matches in the data base based on theses interests/goals. The display shows one potential match at the time, the user has the choice to either "swipe left" or "swipe right". When there is a match, it is sent to a pool of others matches where the user can start a conversation. Also it is worth nothing that during the choosing process, the picture of the potential matches is not shown. However, other information such as interests/goals, age and gender are displayed.

How we built it

We tried to build it with technologies such as angular, firebase and other typical web development languages.
