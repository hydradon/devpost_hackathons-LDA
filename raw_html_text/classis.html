Inspiration

We found SIS very frustrating to use, so we wanted to simplify it.

What it does

ClasSIS allows one to select classes and put it onto their schedule by clicking on the classes

How we built it

We focused on coding with Angular, HTML, CSS, and TS in order to develop the aesthetic and functional features of the webpage.

Challenges we ran into

Our greatest challenge was creating an interface which adapted and followed user input/interaction. We were able to overcome some aspects of this dilemma. However, there are some elements that remain not completely resolved.

Accomplishments that we're proud of

We are proud of developing a formatting method to display the classes. We also learned a lot about Angular, HTML, CSS, and TS.

What we learned

We learned the basics of Angular, HTML, CSS, and TS programming. As well as how interactive websites systems are developed.

What's next for ClasSIS

Next, we intend to implement the entire Lou's list set of courses into the system and also add a better UI for the service we plan to provide.
