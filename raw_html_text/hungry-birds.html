Inspiration

How can we help kids improve their focus, listening and problem solving skills using a voice enabled device like Amazon Echo.

What it does

Hungry Birds is a game that aims at teaching kids the constructs of programming and logical thinking, using a fun and engaging experience.

The objective of the game is to help the bird reach its nest. Make sure the bird stays on the path filled with worms and corn. Whenever the bird eats a worm or a corn, it can perform two movements. You can select from four types of movements, fly, sit, right and left. Your job is to select two movements, for the bird to perform, whenever it lands on a worm or a corn square. One wrong move, will take the bird off the path and will get caught by the fox. So, select your moves carefully. Good luck!

How we built it

We built it using Alexa Skill Kit, Java and unparalleled passion.

Challenges we ran into

Creating a graphical and voice experience for a game like this without the ability to show the characters moving.

Accomplishments that we're proud of

Our creative solution to create a game experience within the constructs of the Alexa Skill Kit.

What we learned

Building skills for Echo Show.

What's next for Hungry Birds

Creating more game templates to teach kids, programming constructs like loops, conditional statements etc.
