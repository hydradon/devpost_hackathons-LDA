Inspiration

This app was built for people who want to be a part of the music at a party. This way, people can vote locally on their channel (a code for their specific playlist) and then choose which songs will be played. It's a way to make something easier for the host of a party and more engaging for others.

How I built it

It's built with Couchbase, Node.js, Express and Angular.js. On top of this, I decided to make it a hybrid app using the ionic framework, which streamlined the app development process significantly.

Challenges I ran into

Dealing with newer versions of Couchbase Node SDK, and server. Learning the ionic framework for the first time.

Accomplishments that I'm proud of

First full app I've designed on my own, and I look forward to improving it more and more from this point onwards. There's still more to be done, but this is a great start in my opinion.
