Inspiration

Our inspiration stemmed from the fact that one of us wants to play Rubik's Cube, but he doesn't have one. So we decided to build one so that people can play it on their computers.

What it does

It simulates the real Rubik's Cube. Users can play Rubik's Cube on their computers (both Windows and MacOs).

How we built it

Basically, we write code in C# and complete the game on the Unity platform. Finally, we build the game into an executable in Windows and MacOs.

Challenges we ran into

We had some problems writing methods to rotate the cube.

Accomplishments that we're proud of

Building a 3D game is much more difficult than building a 2D game. We are really excited about and proud of building such a 3D game.

What we learned

We learned how to use the Unity platform to build a game. We also become more familiar with how to contribute to Github.

What's next for Rubik's Cube

For this game, we will try to make it more convenient for users to play (improve operations for rotation and adjusting the angle of view). Besides, we will try to improve the user interface to make it cool and fun. In the future, we will try to build more original games using the Unity platform.
