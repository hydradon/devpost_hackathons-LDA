Inspiration

lots of people really get confused and totally have no clue where they can get various services like hotels, medical centres, gas stations, banks etc.

What it does

it will help users to get the nearest service centres [hotels, medical centres, etc] that are around them and it will also provide a means of getting detail information of those service centres.

How we built it

this mobile app is built using React Native Framework which will enable us to build an app with native experience. and we will use Google map API in our app.

Challenges we ran into

we were challenged on deciding how we can make the app cross platform, so that users can get access to it using different platforms.

Accomplishments that we're proud of

we already have full prototype of the app in hand so far.

What we learned

we learned some concepts of the popular framework ReactNative.

What's next for Tekuame

building an app the will help peoples get information regarding service centres that are around them.
