Inspiration

iClickers are a must buy for any college student, but they're expensive and annoying to carry around. Our team sought out to rid that problem by placing the questions and answers on your phone!

What it does

Anyone can create a poll and anyone can take a poll (given they have the link). Poll creators also have the option to enable location monitoring so that students must be in the vicinity of the classroom in order to answer questions. Answers are ultimately saved and can be displayed.

How we built it

The application was built and hosted on IBM Bluemix. We used technologies provided to use by IBM, such as the Watson Internet of Things and their CloudAnt databases. Aside from that, we used common web development tools to make the web application unique.

Challenges we ran into

The platform was very challenging to use, and very buggy. The servers also went down during the hackathon period.

Accomplishments that we're proud of

Despite all the challenges we faced, we still manage to pull off a viable product in the end. We hope to make it better in the future.

What we learned

We learned how to create a web application using Bluemix as well as some web development tools.

What's next for LocalTools

We would like to get the database fully functional in addition to improving the UI. After that, we would like to create a login system so that later on, we can invite professors to test our software out, as an alternative to iClicker.
