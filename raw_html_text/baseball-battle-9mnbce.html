Inspiration

Gambling baseball go hand in hand. Who doesn't want to earn some spare $$ with their sports knowledge.

What it does

Pits users against each other for their hard earned cash. If you answer the question correctly before the opponent, they get the $$, if the opponent answers the question correctly before you, your opponent gets the money.

How we built it

It's build with Node, Graphql, mongodb, mongoose, & iota.

Challenges we ran into

Ran out of time.

Accomplishments that we're proud of

Built a mostly working graphql backend (minus a couple necessary functions), have a basic front end, parsed data for baseball trivia questions

What we learned

Learned some front end and react stuff, experience building graphql resolvers, how to clean data for trivia.

What's next for Baseball-Battle

finish the front end, shore up the backend, ensure iota transactions are working
