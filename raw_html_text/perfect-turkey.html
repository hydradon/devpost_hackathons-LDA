Inspiration

Decentralized Applications for Everyone!

The killer dApp that brings mass user adoption won't be a single application, but instead will be an ecosystem of user-respecting applications that cover a range of needs. Unfortunately individual applications, even when using a common identity standard, still feel separate. For users new to dApps and crypto, we aim to ease the transition into new password-less systems and decentralized applications by providing a handful of useful applications that can be used on a day-to-day basis, as well as an entry point into the larger ecosystem of decentralized applications.

What it does

  • Dapp launcher
  • encrypted contact list
  • encrypted file manager

How I built it

Webby is built on Blockstack authentication and storage, as well as the app.com app-mining api. Application framework is Vue and heavily customized Vuetify. Webby is a mobile-first application. We wanted to target the devices that users have with them all of the time to enable quicker access to the decentralized Internet.

What I learned

Building a single responsive application is still almost the same as building a separate desktop and mobile application. There's no magic bullet to building apps that look good on desktop and mobile.
