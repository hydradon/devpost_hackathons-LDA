A custom crowdfunding platform to help charities. Uses Cryptocurrency miners on a website to generate money.

What it does

What mine4charity does is allow anyone to form a team to help charities. These can be schools, clubs, organizations, etc. After a team is created members can join the team using custom codes (6 digit integers). After a member joins a team they can choose a charity to help. For the demo, I provided five charities. Each charities option links to a custom webpage that in the background mines a cryptocurrency. These mined currencies go to the charity after the fundraiser is over.

How we built it

As a full stack javascript developer, we used the MEN stack (MongoDB, Express, and Node.js). We built a website that mines cryptocurrencies using coin-hive, an open API.

Challenges we ran into

We had some trouble installing a cloud instance of MongoDB.

Accomplishments that we're proud of

With just a few minutes of mining, we already started to generate profit (albeit not much), but when multiple students, mine for multiple days, the money generated will be significant.

What we learned

I learned a lot about how to mine cryptocurrencies and became a better overall developer by increasing my full stack JavaScript skills.

What's next for Mine4Charitiy

The goal for Mine4Charity is to expand. The more teams and people using our platform, the more money we can generate for charity.
