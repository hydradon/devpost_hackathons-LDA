Inspiration

We wanted to make it easier for people to communicate using a speech recognition engine

What it does

It parses what you say through Cortana and sends a message to the desired recipient over different social networks.

How we built it

We built a universal windows app that connects Cortana to an azure server that uses nightmare-js to scrape the different social networks' messaging systems.

Challenges I ran into

NODE.JS! PhatnomJS has a bug so it's actually impossible to use that for Facebook implementation. We didn't realize this until this morning. We ran into trouble implementing the Universal Windows app with Cortana.

Accomplishments that I'm proud of

We set up the server successfully. It took a while!!! We were very close to making Cortana interact with the user.

What I learned

I learned that I need to learn more Node! It's awesome.

What's next for Cortana Messenger

We still have to fully implement our app with Cortana and find a way to circumvent the PhantomJS bug and Facebook Messenger security measures at the same time.
