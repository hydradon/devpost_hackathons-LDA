How I built it: We used the Spotify API and a nodejs / expressjs web framework.

Challenges I ran into: Getting Spotify authentication to work with our website, back-end algorithm issues.

Accomplishments that I'm proud of: We finished!

What I learned: Start simple with good practices and increase complexity once it works.

What's next for Disney Movie Mixtape: Increasing algorithm complexity & movie scope.
