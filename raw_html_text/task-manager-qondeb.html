Inspiration

To be able to create a web-based task reminder system for a group of people working on sensitive/important topics that require a high level of attention at all times. Part of motivation to integrate Twilio was for the alerts.

What it does

Essentially being able to create, manage and assign tasks within a group of people.

How I built it

Frontend: Angular 4, Backend: Django (websockets)

Challenges I ran into

The old version of Angular I used was in conflict with latest node packages. Also getting Twilio to run.

Accomplishments that I'm proud of

That I got to work on this project and it was fun. Weekend well spent.

What I learned

Brushed up on Angular after a long time.

What's next for Task Manager

Complete Twilio integration, expand features, create groups, the possibilities are many....
