Inspiration

Help people with different capabilities to join a technological world.

What it does

Detect hand gestures to help people with different capabilities to engage with the world and the new household technologies.

How we built it

The system is conformed by a Machine Learning Algorithm that recognizes the hand gestures and creates an technological environment for everyone, the data transmission via Bluetooth and a text to speech program that advises the user (people with sight impediments) of the surrounding obstacles on the path.

Challenges we ran into

To process the data obtained by the algorithm Apply machine learning algorithms for the detection of different hand gestures Convert the data obtained by the machine learning algorithm in C++ into characters and transfer it via Bluetooth

Accomplishments that we're proud of

Obtain reliable data about the hand gesture the user makes. The successful transition of the data and control of different devices.

What we learned

To Apply machine learning algorithms for specific the solution of highly specific and challenging problems

What's next for Personal-Inclusive-Assistant

To allow a larger amount of gestures that can be detected, create a user interface with visual aids and auditory aid
