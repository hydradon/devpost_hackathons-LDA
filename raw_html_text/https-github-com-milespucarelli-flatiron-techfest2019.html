Miles thought of an extension to help people who are unused to the web navigate it more easily.

This extension highlights links, navigation elements, and inputs so that users can identify what parts of a site are interactive. The extension also highlights ads, warning users to ignore them.

Pure javascript. But a translation feature is to be added in Node.

Chrome extension has many limitations and is not meant to be used with modern JS frameworks.

MVP is complete, and advanced features are in process.

How to program a chrome extension
