Inspiration

Every one of us has a variety of hobbies and things we are interested in. When meeting new people, it can often be difficult and take time to discover all the things you have in common.

Perhaps that stranger you shared a bit of small talk with also loves the same obscure video game that you do. Perhaps your friend of six months is also a fan of the same music band; you just never realized since that topic never happened to float up in conversation.

Jungo is about taking initiative and seeking out birds of a feather to have meaningful relationships with right from the beginning.

What it does

Users can login to Jungo and search for and select their interests. They can then discover which other users have also selected the same particular interest, as well as the users that they share multiple interests with.

How we built it

We built this website by using the Facebook Graph API as the source of the possible interests. Keywords in the search bar locate corresponding Facebook pages. We used MongoDB as the database for storing the interests that users select to perform queries and aggregations. Everything was hosted on Azure, with our application running in a virtual machine and MongoDB in Docker.

What's next for Jungo

Some possible features in future include: filtering of search results, a recommendation engine, ability to chat with other users, location-based filtering of users.
