Inspiration

University life is hard. There is a prevalence of distress over time management, grades, and people are overall not internally motivated to learn. Our overall goal is to inspire learning by placing emphasis on the process of study and not the destination of a grade.

What it does

It first asks for a goal of the study session and samples your mood. It displays a simple Pomodoro timer with some prompts for self-care during the 5-minute break. There is a walkthrough of each prompt, then asks if the timer should be reset or the study session should end. After the session, the user is asked if they met their goal and samples their mood again. These data are stored as statistics for the user to see progress over time.

How we built it

We used a website platform with python, javascript, HTML, and CSS.

Challenges we ran into

This was our first hackathon, and we had limited knowledge of coding. We had a very slow start in getting the code written.

Accomplishments that we're proud of

One of our teammates, who had no experience in coding for a website, was able to create a page with the timer and self-care prompts.

What we learned

We all learned the backbone of a website and what it takes to develop a website by manual code.

What's next for Study Buddy

Being able to track user progress (mood, goal completion). We did not have enough time to figure out that approach.
