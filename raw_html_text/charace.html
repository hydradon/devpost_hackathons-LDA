Inspiration

There's not enough fun incentives to donate to charities. It is also sometimes quite difficult to decide which charity needs the most support. If only there was a fun and rewarding way to donate...

What Charace does

Charace provides a unique and entertaining way to support your favourite charities as well as seeing which one needs more support.

How we built it

Most of the website's functionality is seen in frontend and was made with HTML, Javascript (using jQuery) and CSS (using Bootstrap)

We used Just Giving as a donations payment processor.

Backend is running on Python on Flask framework.
