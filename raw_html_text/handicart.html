Inspiration

We wanted to improve the shopping experience for the visually impaired people and to help them avoid buying food items that could cause any allergic reactions

What it does

With the help of azure and Matrix lab Creator board, we were able to implement a complex and powerful hardware solution which is rather easy to use.

How we built it

Matrix Creator along with an RPI camera are the frontend of the project. This enable us to have a rather simple to use interface for the user. Azure is doing all the heavy lifting on the back. From facial recognition to object recognition to language translation. Everything is hosted on azure.

What's next for HandiCart
