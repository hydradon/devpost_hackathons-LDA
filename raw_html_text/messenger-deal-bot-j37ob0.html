Inspiration

Getting deals and a review requires using multiple apps. I wanted to build an app that does both. Also, making it a bot will make the service instant and direct.

What it does

Given a city, state and type of deal the bot will return a deal and a review from yelp about the business holding that deal.

How I built it

Using heroku to host a website that holds all of the logic for the bot and getting my data from lesserthan and yelp API.

Challenges I ran into

I started with Slack bot and ran into a load of trouble, eventually switching to Messenger. Also, connecting a review to the correct deal was a challenge as well.

Accomplishments that I'm proud of

Having a working demo.

What I learned

Bot building.

What's next for Messenger Deal Bot

Add more features in order to get as many deals as possible.
