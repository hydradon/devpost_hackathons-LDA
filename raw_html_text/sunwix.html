Inspiration

A lot of the software functionality has been built already: generating a quote, calculating solar project revenue, getting utility data, etc. But, every time an entrepreneur or an engineer in solar company needs this functionality, they build it from scratch. There is no platform for clean energy entrepreneurs and engineers to share their solutions: APIs, widgets, datasets, etc.

What it does

It allow users to upload, download and review APIs, widgets and datasets. This will help entrepreneurs build apps and platforms faster.

How we built it

We build a small web version of a SaaS platform.

Accomplishments that we're proud of

We finished a working demo. We finished all backend work and most of the front end work. This is 80% production ready (MVP).

What's next for SunWix

Make it live and share it with fellow entrepreneurs and start using it in our own projects and daily jobs.
