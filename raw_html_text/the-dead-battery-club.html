Inspiration

Everybody has a dead phone battery, but no way to tell friends that it's dead!

What it does

Simple, at the press of a button -- you can tell your friends your battery is dead.

How I built it

  • ESP8266 WiFi Module
  • DigitalOcean for hosting our DNS server/backend
  • Domain.com .CLUB (deadbattery.club)

Challenges

Current state of the device is it runs off of USB power. Implementation as a product would require a Lithium-Ion battery and a switch to trigger the device. The actual device would tweet a message.

What's next for The Dead Battery Club

Get a working prototype. At its present state it works, but not well. The DNS server successfully forwards to the server running on DigitalOcean, but little to no processing is done.
