Inspiration

Amazing GTA 5 mods made by modders.

What it does

Added a gun game mode in GTA 5, in which players get new guns by killing NPC's and staying alive.

How I built it

I used Visual Studio as my IDE. The entire mod is written in C#. The mod heavily utilizes the game's library.

Challenges I ran into

Testing the code in game. Figuring out the game's library.

Accomplishments that I'm proud of

Created a new game type. Created my first mod for a game.

What I learned

C#, Visual Studio, GTA 5 library, game physics.
