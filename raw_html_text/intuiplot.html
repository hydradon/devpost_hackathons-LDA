Inspiration

GGplot is hard. The syntax is hard to deal with to do the simplest tasks.

What it does

We provide a user-friendly data visualization alternative that allows the user to intuitively plot the data, just how they want to.

How we built it

We used Python backend and the ggplot library as the base for our graphs. We hosted on a localhost.

Challenges we ran into

We couldn't use Heroku. Image hosting was troublesome. We couldn't add in all the functions we wanted to.

Accomplishments that we're proud of

We solidified the baseline of our product and get the foundation working.

What we learned

This is a bigger project than we expected, and we had to break down the project to what was doable.

What's next for Intuiplot

We will add more plot functions and more customization. We will also incorporate natural language wit.ai API and have prediction options. Also, outlier detection and NA handling.
