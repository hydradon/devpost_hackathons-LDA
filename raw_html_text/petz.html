Inspiration:

Some of us have been to East Asia where there are many stray dogs and cats roaming the streets, most of which are malnourished. We felt that we had to take initiative and make sure that these dogs and cats, along with other animals, are properly taken care of by trustworthy members of the community.

What it does:

Our app, Petz, is a network which allows people to adopt or give pets for adoption with a clean and simple UI.

How we built it:

We used Swift and Xcode to develop the Petz, an iOS app. We also used Sketch to create aesthetic mockups.

Challenges we ran into:

We were unable to complete the login authentication via the Firebase API and a properly filterable search engine via Machine Learning.

Accomplishments that we're proud of:

We developed a somewhat stable UI with an easy-to-use design. We also delved into using iOS tools which we utilized to our advantage.

What we learned:

We learned how to improve upon our skills with the Xcode kit and Sketch. Aside obtaining technical knowledge, we also learned how to work better under a time constraint and pressure.

What's next for Petz

  1. Geolocation API - to secure location of users
  2. Firebase API - to verify credibility of users
  3. Machine Learning - to filter search results based on user's given parameters
  4. Chat System - to allow communication between pet owner and potential adopter
