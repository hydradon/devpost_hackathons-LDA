Inspiration

Our inabiliity to decide what to eat in social situations.

What it does

It allows members of a group to vote anonymously on their food of choice and displays the results.

How we built it

We developed a webapp using html, css, and javascript, and used FireBase to store user data.

Challenges we ran into

Communicating with the database. Creating an attractive UI.

Accomplishments that we're proud of

Using FireBase, a technology that we've never worked with before, to read and write data.

What we learned

How to utilize FireBase in our web applications.

What's next for Vouper

Scalability - improve roomID algorithm in order to allow for more possibilities. Lobby - Assign head role to another member in group should the host disconnect (Allows for smoother).
