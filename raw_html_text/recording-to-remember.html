Inspiration

One of our group members, Andrew, is an aspiring actor and he had the idea to create something that could help him remember lines easier, better and more efficiently.

What it does

This project records the user and the user is able to play back their audio as it saves to their desktop. They can record for however they want and they can share the audio with other actors/actresses if they choose. Users can use this app to critique their voices.

How we built it

We built it using python and python packages. The packages we used were tkinter, pyaudio , and wave.

Challenges we ran into

The challenge we ran into was coming up with an idea to start the project. Another challenge was trying to save the audio file onto the user's desktop and having the user able to name it whatever they wanted.

Accomplishments that we're proud of

We are proud of coming up with this idea as we think we can improve this idea more if we had time. We are also proud of working together and sharing the workload.

What we learned

We learned that coming together as a team made this project fun instead of stressful. We are proud of learning off of each other.

What's next for Recording to remember

If we had more time, we could try to incorporate a video so that the users can see how they look and sound when they are running through their lines. They could also upload the videos to have other developing actors/actresses critique them. Users would be able to add notes and blockings to their scripts.
