Inspiration

Frustrated PartyAnimals , Sick and tired of organising failed parties, now look to solve their problems through this app!

What it does

An Android app that helps people organise parties, reunions, get togethers, etc in an efficient manner, all in one place.

This app allows the user to create a party, hence providing a location, date, time and the list of guests.

It also allow him to track and "nudge" his guests in order to remove the inconvenience of calling and asking each guest.

It also helps to keep an account of all the events he has been invited to.

How we built it

We used Android Studio as our IDE to build this app. Then we also used the Google Maps API to provide users location-based services. Along with this, the Parse SDK was used to maintain the back-end database of the app on a cloud platform.

Challenges we ran into

It was quite challenging to implement the Parse SDK to create a proper relational database.

Accomplishments that we're proud of

We managed to successfully create a fully functional prototype with a cloud-based database for the first time.

What we learned

Teamwork and time-management.

What's next for Party House

We will try to implement some more features advanced features such as:- optimal meeting point suggestion using high performance algorithms and cab sharing between multiple guests.
