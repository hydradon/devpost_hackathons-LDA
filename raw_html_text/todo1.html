Most of the day to day information we handle have a geo-location component. What my app brings is the possibility to map the infrastructure buildings, floor maps, and map our devices using geographic coverage. From here we can build tools to analyze information through geographic theme visualizations.

In my app, I've already implemented the core functionality to build the "geo" part and some basic device analysis using attributes coming from Spiceworks.
