Inspiration

We love playing the card game Mao and we wanted to make an online version of the game.

What it does

It simulates a round of Mao and allows one player to play with the computer.

How I built it

We built it with java. We created the main structure, then added the class Card into the main structure. Then we added the shuffling and drawcard method. Then we added the rules.

Challenges I ran into

We didn't know how to make GUI on java so we had to look it up and learn

Accomplishments that I'm proud of

Finishing the project

What I learned

How to make GUI and how to make a class and call the class in another class

What's next for Mao Card Game Simulator

We will add more rules to the game and we will make the interface look better and more user-friendly.
