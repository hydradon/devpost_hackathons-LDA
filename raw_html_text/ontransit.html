OnTransit is a mobile website that crowdsources vehicle positions to generate real-time arrival predictions. This app loads in transit schedule data from the Valley Transportation Authority. It then presents a list of nearby stops or trips to users of the mobile website. If a user opts into sharing their location by saying they are on a transit vehicle, the real-time arrival information will be calculated for that transit trip. It will also then show at any stop that the transit vehicle will travel to. The real-time predictions only work with the help of transit riders like you!
