Inspiration

Wanted to build an application that helped farmers realize larger crop yields, while at the same time incentivizing better and more sustainable agricultural practices and ecological stewardship.

What it does

Uses imaging, and hopefully eventually, sensor-equipped drones to regularly monitor farms and determine field health through machine-learning data gleaned over time from work with farmers. Farmers get incentivized via proof of work - either by improving their farms or by providing feedback data to the network.

How we built it

Largely front-end at the moment

Challenges we ran into

Logistics - we will likely need to build our own coin (on an existing smart coin like EOS or Ethereum) to make this all possible.
