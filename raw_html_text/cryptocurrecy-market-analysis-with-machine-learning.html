Cryptocurrency Trading Meets Machine Learning

We used multiple machine learning models to predict the near future prices of cryptocurrencies. We used this data to determine programmatically the optimal time for buying and selling currencies. Our algorithm can be applied to almost any currency and operates on the Bittrex Exchange.

During this project, we learned a lot about machine learning, regression and classification models, and practical applications for different model types. (As well as utilizing trading API's)
