Inspiration

We live in New York and want to look for places to do fun stuff near us at any given moment.

What it does

We choose a recreational activity and lists areas by distance and shows us the map.

How we built it

We used Android Studio with firebase and obtained json files from NYC Open Data.

Challenges we ran into

We had to write a lot of source code so we ran into a lot of bugs.

Accomplishments that we're proud of

We created a working app using json files from NYC Open Data to create a robust app.

What we learned

Debug early and often, and plan early.

What's next for NYGO

Update About page and make the pictures cleaner.
