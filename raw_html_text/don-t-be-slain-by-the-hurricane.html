Inspiration

What it does

Point-and-click game designed to teach people how to prepare for a hurricane.

How we built it

Angular 4, Bootstrap, HTML 5 Canvas

Challenges we ran into

  • Closing shapes
  • Room transitions
  • Point in polygon algorithm
  • Pop-up dialog box

Accomplishments that we're proud of

  • Room transitions
  • Approximating element shapes
  • Naming of project
  • Gamification of hurricane education

What we learned

  • Use Gimp to "glue" multiple pictures together
  • Use Canvas in Angular
  • Build a program in Angular/get more comfortable with Angular cli & bootstrap

What's next for Weather the Storm

  • Add more educational factors to hurricanes
  • Add animations to progress bar and "weather the storm" button
  • Add more natural disasters
  • Improve UI/change each picture as we click on it
