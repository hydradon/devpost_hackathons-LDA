The visual style of this game was inspired by seasonal greeting cards. We wanted to play with the idea of two characters unknowingly influencing each other's environments.

We built the game using Unity. Graphics were painted and created in Photoshop. We used a mixture of Unity and Photoshop for the animations.

We aim to polish this up and release it.
