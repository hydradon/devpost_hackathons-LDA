Inspiration

With the constant development in tech such as AI & ML, we decided to create an interface that could expose the younger generation to concepts about computer science and programming.

What it does

With constraint, specific orders and limited time, our game can help the younger generation understand the basics of programming through the boba building process. We also built a special Code Jam edition barista for this version.

How we built it

We built the interface using PhaserJS and designed everything by scratch by Figma.

Challenges we ran into

First time using PhaserJS while integrating the designs we wanted into it.

Accomplishments that we're proud of

Learned PhaserJS and we were able to integrate the designs smoothly.

What we learned

PhaserJS and the documentations

What's next for Boba Builder: Code Jam Special Edition

Integrating more cs concepts and constraints for users to practice.
