Inspiration

The Zitcom case speech.

What it does

Allows you to compete with your co-workers in the gym. Each time you go to the gym you get a score depending on the amount of work you put in. A leaderboard shows the top performers along with other interesting statistics.

How we built it

Using .net core, react and arduinos.

Challenges we ran into

Getting the sensors to properly detect when an object is lifted.

Accomplishments that we're proud of

That the leaderboard is live updated, when a person lifts the object. Using requests to and from the .net core application.

What we learned

React, arduino, some networking

What's next for Zitness

An angel investment of $1bn from Elon Musk
