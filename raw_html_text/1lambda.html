AWS Lambda

What it does

Microservice architecture platform built on the 1&1 Cloud

No Servers to Manage

1lambda automatically runs your code without requiring you to provision or manage servers. Just write the code and upload it to 1lambda.

Continuous Scaling

1Lambda automatically scales your application by running code in response to each trigger. Your code runs in parallel and processes each trigger individually, scaling precisely with the size of the workload.

Hit Based Metering

With 1lambda, you are charged for every time your code is triggered. You don't pay anything when your code isn't running.
