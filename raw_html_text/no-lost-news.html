No Lost News

A workflow tool providing fast curation and feedback mechanisms. Designed to be the user interface to a machine-learning powered intelligence brief recommendation system.

Features:

  • Easy categorization of information
  • Feedback loop for training machine learning
  • At-a-glance report status via color-coding
  • Can deep-dive into Intelligence Reports and share comments

Inspiration

Taking a cue from how Reddit organizes news, No Lost News was designed to be the "front page of the internet" for breifing book builders.

What it does

No Lost News is a powerful interface to categorize incoming news and train machine learning models

How we build it

create-react-app and firebase.io
