Inspiration

The consistently limited comprehensive sexual education for the youth in school.

What it does

This is a quiz that offers ten (10) questions about things you may not know about sex. Once the quiz is complete, it will let you know your score and inform you of some extra resources for you to learn more.

How we built it

We used Android Studio to create the quiz action, the score keeping function, and the results. This ultimately would result in an app.

Challenges we ran into

Some challenges were initially learning how to use the program, learning how to use GitHub collaboratively, and getting the test to be able to restart.

Accomplishments that we're proud of

A COMPLETED APP!!

What we learned

We learned how to use Android studio and more about each other.

What's next for How Sexy Are You??

Hopefully, with the next update, HSAY will be able to have working links to its informational sources and maybe a different section on body positivity.
