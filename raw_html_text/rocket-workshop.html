Inspiration

We ran a Git workshop at the University of Oklahoma, and we had no idea how to gauge everyone's progress on the workshop. Many students feel imitated to raise their hand to receive help, so we developed Rocket Workshop, a platform where students can ask any question and receive instant help during the workshop.

What it does

Makes running educational workshops more efficiently.

How we built it

We used React, Firebase, Redux, and Draft.

What's next for Rocket Workshop

Make it to where it has production value so that it can be used school-wide.
