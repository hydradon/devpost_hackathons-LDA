Inspiration

I have a small business and couldn't track my income, expenses, projects, clients and so on in one app without using some expensive finance app, so I decided to develop mine.

What it does

The App helps user to manage all their clients. User can assign a project to a client and follow up the status of the project in the app. The App allows users to record their expenses and income and also manually track the money they have in bank and cash at hand. The App has module where user can take note. Another cool feature about the App is that user can generate financial report ( profit and loss) as pdf.

How I built it

I used the latest version of ionic3 to build the App. I used Typescript to code the components, html5 for the view, css3, and cordova plugins for the native functionalities.

Challenges I ran into

It was very tough for me to figure out how to generate report dynamically in pdf format. It is dynamic in the sense that I can get report from a certain date to another.

Accomplishments that I'm proud of

I brought Client , Project, Marketing, and Finance Management into one App

What I learned

It was my first time using ionic3 which has lazy loading features. I also learnt how to generate report in pdf format using app.

What's next for Affiah

I am looking for a way I can improve the functionalities and deploy it fully or sell it to a finance institution.
