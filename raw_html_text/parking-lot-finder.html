Parking Lot Finder is a web application that gives you a list of parking lot options close to your destination. All you have to do is type in your destination (it can just be some of the address or the zip code, the search bar is pretty flexible) and it will load a list of parking lots will appear. The list is sorted so that the best ratings show up at the top. When an option is clicked, it will readjust the map to create an optimal driving route from starting point (your current location) to the selected parking lot, and then an optimal walking route from the parking lot to your destination. Clicking through the options will readjust the route.
