• Be A Better Roommate

Inspiration

If you have not had a negative roommate experience, count yourself lucky! While personality test are effective in pairing like minded individuals, they don't account for one's habits!

What it does

It allows roommate to assign, notify, and grade the progress of tasks completed.

How we built it

The project is being mocked using wordpress, while individual components are made using meteor.js .

Challenges we ran into

Technical knowledge was lacking for the whole team. However, we were still able to create a representation of our business idea.

Accomplishments that we're proud of

Creating something without knowing how to use it!

What we learned

Creating a SAAS product from scratch is very tedious.

What's next for [b]Roomie

Complete SAAS development of the project. Additionally, a mobile application would benefit it.
