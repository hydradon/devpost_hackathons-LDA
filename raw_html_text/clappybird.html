Inspiration

I got pressured by Sam Agnew to build something.. so I put together something in a few hours.

What it does

When either clapping or yelling, the little bird flies upwards.

How I built it

I used an open source version of FlappyBird I found on github and looked up various real-time mic libraries in python.

Challenges I ran into

I at first thought to create my own game from scratch without any knowledge of game production, but I thought using FlappyBird might be more fun.

Accomplishments that I'm proud of

I managed to "hack" this together in 2 hours. The rest of the time was testing and tweaking the sound level filter.
