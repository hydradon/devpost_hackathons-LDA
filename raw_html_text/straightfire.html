Inspiration

What it does

You drop your mixtape on the haters because it's so fire.

How I built it

Everything was written in python using pygame.

Challenges I ran into

Wanted to write another controller using a websocket and swift, but it was more trouble than it was worth. Pygame is supported differently between Ubuntu and OSX. We had to write a bunch of janky stuff to ensure OSX was stable while running the game.

Accomplishments that I'm proud of

Our game is playing with both keyboard and xbox 360 controller input.

What I learned

First time using pygame, so everything used which was pygame related was learned.
