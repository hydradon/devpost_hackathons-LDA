Inspiration

When listening to wonderful songs and appreciating movies, emotions are elevated, but often it is difficult to communicate that excitement to others. Also, the same way of feeling depends on the time and place you are listening. We wanted to make it easy to express emotions in such a case.

What it does

emShare provides the user experience that emotional sharing and other users' rehearsal experience.

How we built it

We used AWS for system construction of SNS. In addition, we used Cloudinaly 's API for media posting. This will allow you to upload content faster.

Challenges we ran into

First of all, during Hackathon, we built a structure as the basis for web applications.

Accomplishments that we are proud of

  • Maximum use of media Cloudinaly API was utilized.
