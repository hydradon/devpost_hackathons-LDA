Inspiration

Create a platform to engage demographics with the 2020 election through technology.

What it does

It gives the typical user data driven insight about presidential candidates of the 2020 election, which is a very important event for all U.S. citizens.

How we built it

Vue was used for the front end. Asp.net and c# was used for the backend. We got tweets from the presidential candidates twitter feed. We fed those tweets onto google's natural language api. Then we created analytics from the results from the api.

Challenges we ran into

We had a difficult time getting everything started at first, such as making sure we had everything we needed in order to program. We had difficulties getting data from the google's api.
