Inspiration

Nuestra principal inspiración fue conocer la problemática que existe respecto a las empresas que necesitan del servicio de un programador y no saben donde encontrarlo, o también puede ocurrir que un programador no encuentre algún trabajo/proyecto a desarrollar, o para quién.

What it does

Se desarrolló una plataforma web vinculada con una base de datos para el alojamiento de perfiles de ingenieros o expertos en el área de tecnologías de la información o desarrollo de software, para ser un intermediario entra las empresas y estos últimos. Ayuda tanto a las empresas como a los programadores a satisfacer sus necesidades y solucionar sus problemas.

How I built it

Se usó para el back-end la tecnología de node.js para montar el motor del servidor, se usó el servicio de microsoft azure para el host de la aplicación web. Se uso github para el manejo del repositorio; se usó la librería de express, para el front-end se uso html5.

Challenges I ran into

El aterrizamiento de las ideas y la implementación e integración de las tecnologías

Accomplishments that I'm proud of

El diseño principalmente, Maquetación y arquitectura de la aplicación web De la solución que va a ofrecer nuestro proyecto

What I learned

Algunos aprendimos a programar, a diseñar, a formar un buen equipo interdisciplinarios, a tomar importantes decisiones, resolver problemas bajo tiempo y estrés

What's next for WILD GEEK

Implementarla completamente en la web para poder llevarla al mercado, hacer publicidad para la aplicación y potenciarla y mejorarla para que de un mejor rendimiento día con día.
