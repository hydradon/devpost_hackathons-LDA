Bowling FTW

Contributors: Kurtis Dressel, Jacob White, Peter Gawecki version: 1.0.0

== DESCRIPTION == Bowling FTW is a webpage that calculates players' bowling scores dynamically. First, input the number of players participating in the bowling game. Each time the player rolls the ball, they input the score into the webpage. Then it calculates the score automatically, and moves to the next player. Allows up to 5000 players!

Hello World Hackathon Project Languages: HTML, JavaScript, CSS
