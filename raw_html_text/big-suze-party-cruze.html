Big Suze Party Cruze

Category: Transportation

Used Data Sets: NYC-vehicle-collisions.csv

External Data Sets: https://www.health.ny.gov/statistics/cancer/registry/appendix/neighborhoods.htm

Team Members: Marc Mekhanik, Liam Chung, Raphaelle Tseng, Cheng Lin

This is our submission for McGill CodeJam 2017: DataDive.

We took data from NYC vehicle collisions from May 2015 to February 2017 (roughly 600 000 documented collisions) and analyzed them using the pandas python library.

The user can input a ZIP code, and select the New York borough they reside in. The GUI will output the number of casualties in the ZIP code, and the ZIP areas in the borough with the most and fewest collisions.
