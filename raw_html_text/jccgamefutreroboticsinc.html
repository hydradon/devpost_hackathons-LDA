Inspiration

A large variety of sci-fi movies .

What it does

Lets you build a robot, run away from that robot, and then stop that robot destroying humanity!

How I built it

We built this game by creating textures in GIMP, modelling and texturing assets in Blender, as well as programming and constructing game inside Unity.

Challenges I ran into

Programming difficulties, minor texuring issues.

Accomplishments that I'm proud of

We are proud of what we have achieved in a short time, including all the various assets and levels we were able to create and the various programming system which were implemented.

What I learned

We have further expanded our knowledge of programming, game design philosophy, and C sharp programming.

What's next for JCCgameFutreRoboticsInc

More Time = More Content, maybe a full release of the game, who knows!
