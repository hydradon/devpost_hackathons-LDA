Inspiration

To help the elderly connect with others

What it does

It allows the elderly to share stories through an application

How we built it

We used Corel Draw to create images that set the template for the prototype. We then tied all of the application pages together and created interactive buttons with Invision.

Challenges we ran into

It was very difficult to decide which ideas were too complicated to implement for seniors and which ideas were not.

Accomplishments that we're proud of

Our website is colorblind friendly and simplistic for senior users.

What we learned

You must be careful with how you organize a webpage because there are a lot of design choices that could make it confusing or feel cluttered.

What's next for Tome
