Inspiration

Since none of us actually owned a dog we thought it would be interesting to create a dating site from their point of view. We thought that a dog dating site would be original and funny, yet be achievable as an idea within 24 hours.

What it does

It displays a homepage with twelve dog profiles including pictures and basic information. We have buttons at the top that take you to a submission form for users to input data about their own dog. As of now the back end stuff has had some minor errors in testing so we were not able to implement it into our website. Another button takes us to the about us page where we wrote about the project and the authors. And then home button takes you home!

How we built it

All the front end stuff was built using html and css.

Challenges we ran into

The back end javascript and databases were really difficult to implement into our project.

Accomplishments that we're proud of

We had a great time making our FIRST website. We learned a lot about html and css, and some javascript.

What we learned

How to use html, css, and some javascript.

What's next for DoggoDoggo

We're going to implement true functionality of our product. We will have all the buttons working and hopefully more features. A complete system allowing for user input and filtering of dog profiles.
