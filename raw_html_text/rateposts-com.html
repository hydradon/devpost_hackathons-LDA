Inspiration

I wanted to create a photo rating app which helps to eliminate low quality pictures on the internet. Users will get paid in Satoshi / Blockchain tokens for their contributions.

What it does

Private Image uploader which ensures worksafe and quality posting.

How I built it

Built using Laravel 5, Php, mysql, vanilla javascript, php gd.

Challenges I ran into

Images would sometimes not upload, had to fix folder permissions. Like system for each image had to be tracked.

Accomplishments that I'm proud of

Ability to create a photo sharing app, that emphasizes quality pictures on the internet.

What I learned

Images are better stored in a CDN.

What's next for rateposts.com

Integrate blockchain rewards and use ML to identify quality photos.
