How I built it

We used a Meanstack and primarily socket.io to send live messages between the host and all the other users

Challenges I ran into

It was tough implementing Oauth and live streaming, we have to set a timer on the host, update the db everytime time had passed, and we did it in the most doable way we could do it which just did a post request for the time on every user.
