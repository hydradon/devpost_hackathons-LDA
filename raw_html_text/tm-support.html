Inspiration

We want to alleviate the headache that a T-Mobile customer faces when waiting to speak with a representative about their issues.

What it does

The project is a built-in feature for T-Mobile that allows customers to either solve the issues themselves by following simple steps recommended, or call a representative.

How I built it

We used Python for the back end, and Javascript and CSS for the front end, along with multiple frameworks and APIs.

Challenges I ran into

We had difficulties trying to create a good connection between the front and the back end.

Accomplishments that I'm proud of

The UI looks wonderful and the database schema is carefully designed.

What I learned

We got to learn different sets of skills from each other since everybody's got experience in different fields.

What's next for TM-Support

We show the customers how to solve their own issues in the least amount of step in the least amount of time by integrating Machine Learning.
