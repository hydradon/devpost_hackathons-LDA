Inspiration

To make it easier for new or casual investors to be able to communicate with and learn from others willing to help them.

What it does

Allows people to follow friend's portfolios so that they can see what might be working or what might not be.

How we built it

We separated into two sub-teams; one for frontend development and the other for backend development. The frontend team used HTML, CSS and JavaScript to design the website template and make it interactive, while the backend team used JavaScript, Node.js and the Alpha Vantage API to build the server and fetch real stock market data to be used by the functions we developed for running various calculations.

Challenges we ran into

Integrating the database queries into the code so that it updated as it should.

Accomplishments that we're proud of

Being able to get the database working as it should; the design of the website; and the innovation and name of the project.

What we learned

We learned more about the stock market and the data being stored at real time, and we learned about working with technologies which some of us have never used prior to this project.
