BrickHack V Hackathon 2019

Never before have we used any of the technologies except for HTML and CSS before today.

Google cloud platforms vision API was leveraged to classify pictures. Wegmans API was leveraged to search for important information regarding food choices and places to find the best prices. NPM was used to manage packaging and include frameworks such as express for routing.

Unfortunetely, a lot of routing issues occurred and once they were resolved it seemed as though cors errors were encountered as well. This meant that the back end was virtually unonnectable to the front end.

However, the most important thing is that everyone on this team came in knowing very little and came out learning a whole lot!
