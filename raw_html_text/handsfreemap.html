What it does

Our app allows you to use a map without requiring you to touch your device. This is done using voice commands such as "zoom in" and "move left".

How we built it

We used Android Studio along with the Google Maps API and built in voice recognition libraries.

Challenges we ran into

We originally wanted to use LEAP motion to allow for the user to manipulate the map using hand gestures, but we ran into complications with time while attempting to make it work.

Accomplishments that we're proud of

Our app successfully recognizes simple commands and follows accordingly.

What's next for HandsFreeMap

We may try to use LEAP motion to make for simpler use as well as adding the ability to search for routes to a destination.
