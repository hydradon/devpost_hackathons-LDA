Inspiration

This started off as a rape prevention application, but then we made this for general crimes. Many times, people do not know they are in a dangerous area where crimes have been often.

What it does

Shows a map, alerts friends when the you are feeling uncomfortable by sharing your location, and automatically calls the police when the button Help! is chosen. This also has a recording feature when you are feeling uncomfortable.

How I built it

We used Android studio, the Twilio API for messaging friends your location, Google Maps API, and attempted in using the Facebook API

Challenges I ran into

We could not get the Facebook API to work.

Accomplishments that I'm proud of

The application works for the most part.

What I learned

Time management.

What's next for Safe Alert
