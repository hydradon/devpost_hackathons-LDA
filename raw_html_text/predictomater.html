Inspiration

I built Predictomater because the Bloomberg APIs provided so many different types of data, that it was difficult to correlate it all.

What it does

Predictomater calculates a score and recommendation for the security it has been set to analyze. Currently, Predictomater only supports analysis on Western Texas Intermediate Oil Futures.

How I built it

Predictomater was built on a LEMP stack utilizing the Bloomberg APIs.

Challenges I ran into

The Bloomberg APIs had very sparse documentation.

Accomplishments that I'm proud of

It works! (albeit on a single security)

What I learned

Server DevOps (The entire stack was configured manually through the command line)

What's next for Predictomater

Expand to more securities and conquer the world!
