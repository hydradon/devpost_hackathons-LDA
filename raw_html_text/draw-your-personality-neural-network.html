Inspiration

Most people are in favor of using thing that was created especially for them. What if there would be a t-shirt that expresses person's' personality?

What it does

We've trained a neural network that draws one's subconscious after the analysis of one's social network profile. We put this picture on a t-shirt and ship it to the user.

How I built it

Feedforward neural network and c++ desktop application (better if the web hosting is blocked by Roscomnadzor).
