Inspiration

Cada vez la delincuencia es problema mas grave, siendo los asaltos los delitos mas comunes a uno le roban el celular por ejemplo y no tiene forma de avisar o denunciar al momento. debido a eso se plantea esta aplicación.

What it does

La aplicación se conecta a través de bluetooth con el reloj fitbit ionic y en el momento en que se pierda dicha conexión se mandará una alerta a través de SMS a números de emergencia establecidos y se publicará un Tweet mencionando a la policía local

How we built it

Haciendo diseño al mismo tiempo que el back end, empezando con el manejo de la base de datos y posteriormente conectarlo con el front end

URL

La url del dominio es http://www.stayhelpy.com

Challenges I ran into

Rutas y conexiones de twitter y programación de fitbit

Accomplishments that I'm proud of

primera vez que programamos el back end en node.js y aunque fue confuso pudimos realizar las operaciones necesarias para el funcionamiento correcto de la aplicación

What I learned

Hacer backend con node.js introducción rapida a como es programar una app para fitbit

What's next for 48:Helpy

Poder apoyar a mas personas que sufren de estos delitos en el estado de Puebla y posteriormente en todo México.
