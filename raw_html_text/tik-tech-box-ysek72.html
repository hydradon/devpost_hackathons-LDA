Inspiration

Most of inexpensive existing pill organizers do not meet the needs of patients on multiple prescribed drugs. The difficulty in remembering and managing a complex medication regimen makes medical adherence especially demanding for them.

What it does

Tik-Tech-Box is an interoperable system between a smart pill box and a web app that aims at helping the user maintain a complicated medication schedule by reflecting the real-time status of each of many drugs.

How we built it

Along with an interactive web app, we made a prototype of the pill box with SolidWorks, 3D Printing and Arduino.

Challenges we ran into

Establishing wireless connection between the pill box and the web app. Making queries from Firebase in the web app in real time.

Accomplishments that we're proud of

Prototype design and functionality: The box and the PC controller of the Arduino communicate smoothly. Web app interface: Simple and comprehensible. Business cycle: Maximizes bi-directional information exchange between CVS and individual customers to reinforce medical adherence.

What we learned

Controlling LCD from Arduino. Cloud database set-up in Firebase.

What's next for Tik-Tech-Box

Adding machine learning capabilities. Refine real-time interoperability. Connect web app to CVS database.
