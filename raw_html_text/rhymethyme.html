Inspiration

Rhyming helps to teach individuals how language works and helps them to experience the rhythm of language. It's a great tool for young children!

What it does

Rhyme Thyme is a fun game skill for all ages! Alexa gives you a word and you respond with a word that rhymes! Easy right?

Just say, "Alexa, play rhyme time".

How I built it

Using Node, Typescript, AWS Lambda, Azure Storage

Challenges I ran into

Sometimes Alexa doesn't quite get exactly what the user says which results in her thinking it was a wrong answer :).

Accomplishments that I'm proud of

Building a fun, interactive skill

What's next

Finishing the multiplayer functionality so users can compete with others at home or around the world!
