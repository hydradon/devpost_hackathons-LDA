Inspiration

We wanted to build something cool, functional, and useful after the hackathon was over.

What it does

This application allows you to control your music with the swipe of a finger.

How we built it

We used a leap motion sensor and java to create our project.

Challenges we ran into

Fine tuning the sensitivity of the sensor was a little difficult and time consuming

Accomplishments that we're proud of

Our program 100% works (to the best of our knowledge)

What we learned

Look for existing solutions, APIs are a great great thing.

What's next for Aounify

We're going to implement a neural network to add custom gestures not supported by the leap motion API. We will also add a user interface that allows the user to interact with their program in a more traditional manner.
