Inspiration

I had trouble finding some downloaded pictures for a powerpoint and wanted an simple alternative to macOS's Finder.

What it does

It allows the user to browse through their system and see their photos.

How I built it

I learned how to use qt and used what I knew about C++ to build this image browser.

Challenges I ran into

Learning how to use qt from scratch took a few hours and the the amount of classes and their functions can get overwhelming.

Accomplishments that I'm proud of

Having the pictures not show up in their original sizes since some of them were too big to fit on the screen.

What I learned

I learned how to make a GUI for a program and how to link buttons and functions together.

What's next for FileBrowserQ

Allowing it to access other file types and make it look more professional.

Built With
