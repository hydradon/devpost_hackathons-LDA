Inspiration

Used to work in a restaurant and saw a lot of food being throw away daily all this when millions of people die of hunger.

What it does

If you have more than you can consume than just take a pic of food and the app will determine your location from image using exif and it will be uploaded to server to share your location with those in need. If your GPS is turned off no problem you can click a button and share your location.

How I built it

I used exif library to extract LatLong from Image then use this information to create a map using google-maps api. Hosted this JSP app using Tomcat server while using Java & JavaScript as main language.

Challenges I ran into

Figuring out how exif library works, getting maps to work.

Accomplishments that I'm proud of

Successsfully completing the app

What I learned

Integration of maps api, exif Library and JSTL tags
