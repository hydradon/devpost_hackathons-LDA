Inspiration

Sometimes, driving can be very frustrating because of reckless drivers on the road. We wish to communicate with other drivers on the road for a safer driving environment. In order to avoid distractions on the road, an armband hand gesture reader is used to send communications.

What it does

Recognizes hand gestures and shows the corresponding message using a LED display in the back windshield to communicate with the driver behind.

How we built it

Built using LUA, python, Myo own language, Myo Armband, and raspberry pi. The Myo armband reads the hand gestures and sends the data to the Bluetooth dongle. The dongle is connected to the raspberry pi where the data is analyzed. Then the Myo connector app opens a python script. The python script reads each hand gesture and translates it to the appropriate message. Finally, the raspberry pi will show the message to an LED display on the rearview windshield.
