Inspiration

We always wanted a lightweight app where we can see our matches and standings with no questions asked.

What it does

It gives a list of competitions and allows to view the fixtures and standings of all of those matches.

How we built it

We used CocoaPods in Alamofire and SwiftyJSON to get data from an open football API online and created the app using Swift in XCode

Challenges we ran into

We had trouble filtering and getting this data and coding in Swift as it was our first time doing all of these things.

Accomplishments that we're proud of

We are proud of having a working build as we have never coded in Swift or used XCode before.

What we learned

We learned how to use Swift and Cocoapods along with JSON in the best way we could.

What's next for Eurofoot

We hope to add more competitions and features to the app.
