OceanVessels

Problem Statement 7:

Ocean Vessels is a Mac OS X Application that utilizes the internet connecton speed, the large data storage and the speed of a desktop or laptop computer. It parses information from many vessel databases and provides a unified search platform. After initial processing the application is 100% usable offline.

It is currently accessing the data from:

IOTC, Indian Ocean Tuna Commission 56,131 vessels

Alaskan CFEC - 6,842 vessels

WCPFC - Western and Centeral Pacific Fisheries Commission - 5,664 vessels

It is easily extensible to have more data feeds and also to provide a mechanism to read in future files.
