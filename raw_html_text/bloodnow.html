Inspiration

Our original inspiration for the project came from our team just wanting help out in some way. We wanted to engineer a solution that would benefit society as a whole. After several weeks of brainstorming, we came up with BloodNow.

What it does

Start saving lives with BloodNow through the use of a proximity based location notification system to enable philanthropic individuals to donate blood to individuals in dire need. Local hospitals will send out requests with specific requirements for a certain type of blood that they may need to save a life. Thus, this request will notify any individuals in a certain radius that someone is in need. Then, depending on meeting the requirements, that individual could help save a life!

How we built it

Using React, React Native, Firebase, Twilio, Google iCloud, Node JS, Heroku, Express, and TravelTime API, we customized this application primary to carry out the functionality listed above.

Challenges we ran into

Data synchronization and compatibilities/dependencies

Accomplishments that we're proud of

We are primarily proud of the calling system that we implemented to ensure user privacy. With our implementation, people can help those in need and ensure that their privacy remains intact as well.

What we learned

Teamwork, how to deviate from the comfort zone of language such as just HTML, CSS, and PHP, and that we need more sleep.

What's next for BloodNow

Additional features such as filtering results, privatized logins for hospitals, expanded categories beyond necessarily blood.
