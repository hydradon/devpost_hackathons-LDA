Inspiration

Ecommerce on mobile feels outdated with brands creating app that are confusing and hard to use. We will create an app that uses simple swipe gestures to express that you "love" or "hate" a product. It will finally make ecommerce fun and easy to use.

What it does

Using the envy, inc. product API we will create a tvOS ecommerce app so that you can shop for products right from your Apple TV.

How I built it

Native tvOS app using Swift 2.0

Accomplishments that I'm proud of

You can swipe through thousands of products right from your Apple TV! Thats so awesome!

What I learned

Apple's new tvOS is a great platform for developers to reach new users. The Siri navigation and rechargeable touch remote adds a new layer of interactivity for developers.

What's next for shoeenvy

We want to push out shoeenvy as a universal app that works on iPhone, iPad, Apple TV, and Apple Watch.
