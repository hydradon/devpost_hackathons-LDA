Inspiration

In today's society, organizations are in constant for updated information including big data. We were interested in how this information affects real-life experiences such as Stocks!

What it does

The twitter bot posts daily tweets about potentially volatile stocks based on headlines of news articles. It also directly messages subscribers about stocks they are personally interested in.

How we built it

We built the program using Python, Mysql, Mysql lite, and various APIs.

Challenges we ran into

Some challenges we encountered were deciding which language to use, figuring out which APIs would work the best with what we were trying to accomplish, and how we wanted the finished product to appear our users.

Accomplishments that we're proud of

Accomplishments we are proud of are using multiple APIs, being able to adapt to unexpected problems that arose, and actually getting the project to work!

What we learned

We learned how to navigate the terminal/command prompt, how to use multiple and new APIs, as well as work with different programs such as git!

What's next for Chuck: the stock bot!

Something we would want to improve upon with our project is to notify the subscribed user when a stock they are following falls below a certain threshold.
