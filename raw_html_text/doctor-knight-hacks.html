Inspiration

Usually People freak out when they look for diagnostic on internet and they do not know what to do next. Doctor Knight Hacks is able to provide a professional diagnostic base on your symptoms and at the same time it will schedule an appointment for you.

What it does

Alexa's skill that will analyze your symptoms and based in a professional API will make a diagnostic and let you schedule with doctors around your location. Finally the schedule will be set in your google calendar.

How we built it

We built our backend using Amazon Lambda , our back-end connects to different APIs that provide possible diagnostics, real doctors information around the area and finally google's api to insert event in a calendar.

Challenges we ran into

Synchronizing all the APIs with the Alexa's intents. it was quite a challenge filtering all the API's responds to connect them and make this idea to for flawlessly.

Accomplishments that we're proud of

Been able to implement a serverless using Python and putting together several apis.

What we learned

We learned how to create RESTful services with Python and Node.JS

What's next for Doctor Knight Hacks

Been able to increase the dictionary of symptoms to provide more accurate diagnostics
