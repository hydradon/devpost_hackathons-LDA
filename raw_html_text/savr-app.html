What it does

The Savr app allows users to keep track of food items and consume them before their expiration dates — saving money for shoppers and reducing food waste.

How we built it

Prototype was built using InVision. Website was coded in JavaScript. APIs used were the Google Cloud Vision API and the EDAMAM Recipe API. Android app was developed in Android Studio using Java.

Challenges we ran into

Familiarizing ourselves and setting the app up with the Google Cloud Vision API was fairly tricky.
