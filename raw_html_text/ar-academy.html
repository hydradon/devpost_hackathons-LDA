Inspiration

We wanted to make learning fun, so that's what we did.

What it does

AR Academy offers a new interactive way for students to learn using the power of Augmented Reality.

Using the microsoft hololens, AR Academy can utilize 3D holograms to help students visualize their coursework.

How we built it

We used Unity and the hololens SDK to create a representation of chemical bonds in 3D

Challenges we ran into

Hololens would not allow any external plugins, so we had to create everything from scratch.

Accomplishments that we're proud of

Creating the system without using any external plugins

What's next for AR Academy
