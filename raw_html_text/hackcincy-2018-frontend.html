Inspiration

Giving is hard. What if it didn't have to be? Connecting with the community can prove even more difficult. What if it didn't have to be? Enter The Pantry, an online platform that connects helpers to those in need.

What it does

Our online platform gives users the ability, not only to donate at the click of a button, but also to choose exactly what food and essential items they are able to donate with ease.

How we built it

The platform is built with the React framework and Google Map and Places API integration on a Node.js server.

Challenges we ran into

Different experience levels, which really ended up being a good thing. We all learned so much from each other regardless of our knowledge coming in Saturday morning.

Accomplishments that we're proud of

  • We built a web app!
  • We all can walk away from this knowing way more than we did when we came in.
  • Google Map and Places API integration for distances, locations of homeless shelters
  • Some of us coded in React for the first time ever

What we learned

  • React
  • No, seriously, React
  • Version control
  • How to build a project with a team
  • Time and product management without a project manager

What's next for hackcincy-2018-frontend

The next iteration would expand to include donations from businesses with excess food to contribute, decreasing food waste.
