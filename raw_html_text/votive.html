Inspiration

A spark of an idea, that had practical applications; while allowing us to venture into new learning experiences.

What it does

Allows an administrative user to create a poll including a group of individual users to vote on

How we built it

Created the front-end of the app with flutter and dart. Then connected it to a POSTgreSQL database using the aqueduct framework.

Challenges we ran into

Learning and setting up the different frameworks.

Accomplishments that we're proud of

We came into this project with very little knowledge of these resources, as well as no experiences in app development. We learned a lot and made things happen.

What we learned

Connecting a database to an app is not simple.

What's next for Votive

Expansion of the analysis tools.
