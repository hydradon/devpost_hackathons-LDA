Inspiration

I saw that the current methods of communication for paralyzed people could be improved upon. The low-tech solution was very slow and the higher tech solutions were complected, large, and very expensive.

What it does

It cycles through various text options on the screen and when the user wishes to enter the selected text they blink and the person they are communicating with taps the screen.

How I built it

I built it using Android Studio.

Challenges I ran into

Had trouble keeping the input text box updated. Ensuring that the overall aesthetics were visually appealing

Accomplishments that I'm proud of

Created a initial prototype to demonstrate the overall concept of the design

What I learned

Android development, teamwork, design process/ design refinement

What's next for Blink to Speech

Next we would implement the full functionality into the app. We would also like to create a setting menu to provide various themes so that the user can choose an option that provides the best contrast for them. Additionally, in the settings we would allow the speed at which the selector moves to be customized so that the user can use the app at a speed that is most convenient for them.
