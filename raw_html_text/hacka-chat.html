Inspiration

I have trouble approaching others about forming teams, so I figured other people might too. By being able to talk to other hackers, we can form teams and possibly friends more easily.

What it does

It is a chat server that also allows you to show general information about yourself, and see information about others that is related to hacking. Users sign in with either Twitter, Google, Facebook, or GitHub. It also tests your compatibility with other hackers.

How I built it

I used Atom with Firebase, Bootstrap, and various APIs to form the chat function and profiles.

Challenges I ran into

I had never developed a dynamic website before, especially not one with a chat function.

Accomplishments that I'm proud of

I am proud that I got so much done and I learned a lot of information that can use later.

What I learned

I learned how to develop a dynamic websites with multiple different, complicated functions.

What's next for Hacka-Chat

I would like to add more information to the profiles and allow for private messaging between hackers.
