• HateCheck

Vizualizes hate and where the hate is from and helps to analyze sources of hate speech. using Natural Language Processing.

The project visualizes our results in a world map with a heat map that shows if a searched term has been mentioned in a negative fashion several times as interpreted by Natural Language Tool Kit for Python (NLTK).

The project can be adapted to different source material fairly easily.
