  • Doação com um clique

  • Doações são catalogadas

  • Fatura pode ser gerada e pagamentos consultados

  • Status da fatura pode ser atualizado diariamente ou ao desejo da parte repassadora

Inspiração

O processo comum de doação pode afastar um possível doador ao encontrar um processo que demanda um esforço maior do que este está preparado. Caso a doação estivesse apenas a um clique de distância, com valor imperceptível, o ganho em volume pode demostrar um aumento significativo nesta inserção de um novo doador ao primeiro contato filantrópico.

O que faz

Após um pequeno cadastro lojas virtuais, tais como as de e-commerce, podem coletar intenções de doações de seus compradores. O valor pode ser simbólico, podendo ser baseado no arredondamento do valor para o próximo real inteiro. Estas intenções e seus respectivos valores e origens são persistidos para que a loja possa então receber uma fatura via PayPal para que o repasse das doações seja realizado.

Como foi construído

Após a modelagem dos campos necessários foi criada tabelas no banco de dados MySQL que serviram de base para que geradores de código pudessem construir interfaces entre a linguagem e a camada de persistência. O projeto usa uma proteção a alguns endpoints autenticados por meio de um token JWT que pode ser invalidado quando desejado pelo desenvolvedor parceiro. Tanto o front-end e back-end foram programados usando a linguagem F# dentro do ecossistema dotnet core, usando o transpilador de F# para Javascript Fable, juntamente com bibliotecas que são suportadas por React.

Pedras no caminho

A documentação para integração com PayPal continha muitas informações sobre diferentes tipos de funcionalidades, atrapalhando encontrar a melhor função para a emissão das ordens de pagamento.

Incompatibilidades entre pacotes Java injetados via Spring Boot tornaram o processo doloroso, inviabilizando o uso da linguagem para a versão final.

Conquistas alcançadas

O projeto atingiu um nível de maturidade adequado, estima-se que o caminho restante para uma implantação real seja curto. O processo necessita de pouca interação humana para seu funcionamento, possibilitando um recebimento de um grande volume de doações.

O que aprendemos

Nunca subestimar a complexidade da implementação de um projeto; mesmo um projeto com poucos processos pode nos levar a enfrentar vários desafios.
