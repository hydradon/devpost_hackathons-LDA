Inspiration

To help students find the empty classrooms around campus any time.

What it does

Find an empty classroom in any building by searching room number, time or date.

How we built it

Use C as backend and angular framework for front-end and Uwaterloo OpenData API.

Challenges we ran into

Getting used to angular framework and parse the extremely large json files.

Accomplishments that we are proud of

Successfully implemented the web app with angular.

What we learned

How to work on a project in a really tight schedule.

What's next for UW-Focus

Integrate Google maps API.
