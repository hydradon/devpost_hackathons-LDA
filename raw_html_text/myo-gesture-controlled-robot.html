Inspiration

Being able to control things with simple gestures could be crucial to helping people with physical motor skill disabilities to be able to perform everyday tasks.

What it does

The code takes in the myo data and according to which position your hand i in the robot will drive forward, backward, left or right.

How I built it

I used an arduino connected to 2 servo motors that were then connected to two wheels on the robot.

Challenges I ran into

I had a lot of problems with having the right resources for motors which hindered my progress.

Accomplishments that I'm proud of

Using the myo armband was super fun!

What I learned

I learned how to talk to the Myo through the arduino.

What's next for Myo Gesture Controlled Robot

I would like to develop the robot more with better motors. My initial plan was to have 2 dc motors with an Hbridge motor driver. Also, some additional sensors could include a wall sensor, and an image processing sensor for facial or object recognition.
