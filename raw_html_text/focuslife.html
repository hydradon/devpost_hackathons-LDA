Inspiration

I spoke to the representatives at Sacramento Autism Spectrum & Special Needs Alliance. They said it would be helpful to have technology to help special needs students to stay organized and be successful at school.

What it does

This app allows a student to list school assignments. It currently displays three hardcoded assignments for classes.

How I built it

I read some tutorials for iOS app development and used them as a template for my project.

Challenges I ran into

I did not know about app development prior to this event, so I had to learn it all from scratch.

Accomplishments that I'm proud of

I made a simple, hardcoded display.

What I learned

I learned how to create a simple app in swift.

What's next for FocusLife

I would like to allow the user to input classes, include due dates, sort by subject and/or due dates, tap the labels to see a more detailed description, check off which assignments are done, input and view their class schedule, and include some way for teachers to verify that the student took note of all the assignments.
