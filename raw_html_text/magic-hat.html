• Magic Hat

Inspiration

My first Alexa Skill - built as part of the Amazon Alexa Skills Challenge: Kids

What it does

Suggests names for your pets. Has generic names for any pet and some custom names for specific pet types like, cat, dog, horse, and unicorn.

How I built it

Using the Alexa Skills Kit from Amazon

Challenges I ran into

Trying to get the slot value for synonyms of a word

What I learned

I learned alot in a short period. 2 days from concept to first beta version.

What's next for Magic Hat

Expand lists of pet-specific names and types.
