About HackClicker

This is a clicker game in the spirit of HackPoly. This project was created in an effort to learn how to build a web application from scratch using AngularJS and Node.js.

Inspiration

The food. The tickets.

Challenges

I was not able to refactor my app.js file into the separate controllers and services that I wanted to without components breaking. Tuning the cost of the ticket rate and its increment rate to mimic the rates of current clicker games was a lot more difficult than I thought.

Installation 

- npm install http-server
- npm install
- gulp sass

Running

http-server ./ -a localhost -p 8000
