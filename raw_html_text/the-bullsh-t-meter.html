Inspiration

Startup websites all seem to be "Revolutionizing the world by providing innovative digital solutions and encouraging progress through technological innovation."

What it does

We built a tool that analyzes how many of these cliches ("innovative", "digital solutions", "saving the world") are present in a website.

How We built it

We built the tool on Django python framework, with a ReactJS front end.

Challenges We ran into

We initially hoped to use the Watson API to integrate more advanced Natural Language Processing, but were unable to do so because of difficulty and time constraints.

Accomplishments that We're proud of

Our last-minute pivot into a new idea, when our initial idea proved too much to do.

What I learned

Building an Ethereum marketplace is incredibly difficult :)

What's next for The Bullsh*t Meter

We plan on refining this into a product that startups can use to improve the quality of their websites.
