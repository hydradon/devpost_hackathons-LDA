Inspiration

Nothing but the best experiences from interacting with customer service departments to help solve our problems

What it does

Encourages callers to speak to professionals about any problems or repairs they might need in their life and redirects them to the specific department certified to help

How we built it

Built a flask app and utilized the Twilio API to create a call center and would gather and recognize callers' speech and provide an appropriate response based on the request of the caller.

Challenges we ran into

Figuring out which APIs we wanted to work with and then implementing them together. It was a trial and error process. Another challenge we had was to create a directed graph that would cause callers to go into an infinite loop with our helpful service.

Accomplishments that we're proud of

Learned more about the twilio API and twiml verbs in order to create an interactive environment with the callers.

What's next for Infinite Loop Thingamajig Supply Inc.

Hopefully corporate restructuring will allow even more quality service departments
