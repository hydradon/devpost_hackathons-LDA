Inspiration

I was inspired by Silicon Valley Season 4

What it does

It allows users to take a picture of food and gives back a list of recipe of how to make the item

How I built it

I built it with the Google Cloud Vision API and Endamam Recipe API

Challenges I ran into

I had trouble setting up the Cloud Vision API and learning how to use it.

Accomplishments that I'm proud of

I am glad that the Android Application works in the end to a certain accuracy

What I learned

How to use Cloud API in junction with other API

What's next for SeeFood

SeeFood hopefully will evolve into a Tinder of Food where users could swipe right or left to learn how to cook certain food with the set out target of certain calories or other nutritional values
