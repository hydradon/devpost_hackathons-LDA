Inspiration

Wireless camera systems are fascinating!

What it does

Sends a MJPEG stream to a SmartCam server via WiFi. Also shows the minimum - maximum / average temperature values in ThermalRadiometricKelvinImage mode

How I built it

I integrated the MJPEG encoding from the SmartCam Android client to the FLIR ONE example app.

Challenges I ran into

I had to adapt the MJPEG encoding to ARGB8888

Accomplishments that I'm proud of

I achieved it with relatively few changed LOC

What I learned

Using libjpeg

What's next for FLIR ONE to SmartCam

Support switching between the different view modes on the fly if possible
