What it does

Text Tommy is an invisible app* built around an "expertise at your fingertips" service model. Somewhere in-between freelancing and specialized service. Get whatever you need on demand with just one text message!

*Invisible apps are apps without an interface.

How we built it

Built the front-end on vanilla HTML/CSS and used jQuery to take user-input (their phone number) and send it to the backend PHP scripts. PHP scripts utilized the Twilio API to convert phone numbers into HTTP signals, which could be manipulated server-side.

Challenges I ran into

Learning how to use PHP and set up servers. Figuring out how to put everything together (PHP, jQuery, HTML/CSS, Twilio API).

Accomplishments that I'm proud of

Overcoming the above challenges.

What we earned

How to code in jQuery and PHP, and how to set up a server on Digital Ocean and navigate it in Unix. Most importantly, how to build front-end WITH backend!
