Inspiration

We realized that a great amount of time is wasted every day through distractions. In order to be more productive and efficient, we created this website.

What it does

The website consists of a timer that calculates distracted time and focused time. Currently, we don't have an app, but have a prototype. The prototype disables other apps in order to create a focused environment.

How we built it

We used javascript, css, and html.

Challenges we ran into

  • We were unable to make the navigation bar stick to the top, but we figured it out.
