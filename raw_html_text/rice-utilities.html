Inspiration

As a group of primarily Rice students, we felt that there was a need for a rice utility app for android and felt that we could fill the void.

What it does

Map: Shows map of campus, building locations, class locations, points of interest, and bus locations in real time.

Happening Today: Displays a list of current events happening on campus

Bus Notifications: Notifies user when a selected bus is a quarter mile away along the bus route from a particular bus stop.

Shoutout: Allows users to post short messages that are shown on the map at their locations, and to see messages that other people have posted.

Links: Various links to rice websites that open in the app.

How we built it

We built Rice Utilities in Android Studio, but also used Google App Engine to host our web server.

Challenges we ran into

Getting the app to run a service that notifies the user when a bus is a quarter mile away along the bus route from a bus stop of the user's choice.

Accomplishments that we're proud of

Finishing the bus notification service and creating a polished user interface.

What we learned

Android is hard.

What's next for Rice Utilities

We plan on continuing to add features to the app (including an upvote/downvote system in Shoutout) and including more Rice related resources in the future.
