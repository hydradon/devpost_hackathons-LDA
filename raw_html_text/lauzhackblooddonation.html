Inspiration

This application improve data understanding of each blood picture. Not everyone is medic and understand all the specific items in blood picture.

What it does

I created application which recieve blood picture from doctor (It's second application) after that i showed in graph full blood picture and then can user analyze his result. I trained neural network for detection some problems in blood picture. This neural network gets results from blood picture. After that user can see his recommended plan.

How I built it

I built it in Visual Studio with Xamarin

Challenges I ran into
