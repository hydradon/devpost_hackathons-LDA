Inspiration

Many of the clubs at our schools use Facebook as the main means of communication, but as some students do not use facebook, it can be rather problematic, so we created this app.

What it does

It is a platform for students to communicate with clubs. They can join different clubs and post messages so the other students in their club can see it.

How we built it

We built it with Swift and used Firebase for the authentication and the database

Challenges we ran into

XCode is buggy, swift likes to change syntax every version which caused confusion, Firebase's asynchronous nature caused weird bugs where code was not ran chronologically. Also code signing is a huge pain.

Accomplishments that we're proud of

We made a slick looking UI

What we learned

We used Firestore for the first time, we used Hero, a UI framework, for the first time, and this is the largest iOS app (in terms of number of scenes) either of us have made.

What's next for organizr

Add push notifications, allow pictures in posts, and making it multiplatform, both android and web.
