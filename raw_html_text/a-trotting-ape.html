Inspiration

Inspired by the company name A Thinking Ape, we create an ape as the game hero.

What it does

It's an endless running, where Kong the ape must run as far as possible to save its life from Canada's winter by running through the city, where it can take a shelter.

How we built it

It's built in 48 hours on Unity using best programming patterns such as an event based logic, pooling and service locators. Mixing high quality art techniques such pre rendered sprites, spritesheet and validation by concept arts.

Challenges we ran into

Manage and create all resources to create an complete mobile game in only 48 hours.

Accomplishments that we're proud of

We are proud to deliver a complete, fun mobile game.

What we learned

We learned how to overcome troubles for example a dropped out member, sickness and infrastructure issues. We learned what is a manhole.

What's next for A Trotting Ape

We are polish it better and let it be available on Google Play Store.
