Inspiration

This is our first hackathon project and we started with the simple goal of utilizing one API.

And the Nuance Nina API turned out to be perfect match for us, and we used it to create a chatbox that answers questions related to McHacks 2018.

What it does

It answers questions related to data that users put in. In our case, we used the McHacks FAQs. It's a fun way to get answers to questions (No scrolling!).

How we built it

We built it using Nuance's Nina API

What we learned

JavaScript, HTTP request

What's next for QuikAns

We wanted to built a full-fledged interactive chatbox but we decided to start small with QuikAns. Our next goal is to make it speak! :)
