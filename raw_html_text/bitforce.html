Inspiration

Portability of on demand infrastructure as-a-service. Resource management.

How it works

Device discovering using arp-scan. Infrastructure as-a-service using Docker Deployment. Service Monitoring based on RAM usage of devices. Disaster Recovery based on RAM usage of devices and Docker service allocation.

Challenges I ran into

Docker deployment on remote system. Service Monitoring on devices in network.

Accomplishments that I'm proud of

Dynamic Resource allocation based on current RAM usage of devices. On demand decision making of service installation.

What I learned

Usage of Docker for service containerization. Discovering devices in network.

What's next for Bitforce

Exploring better methods for Device Discovery and Service Monitoring.
