• DocuHouse

Inspiration

I'm buying my second house and buried in various emails, miscommunications, and scattered deadlines across many different parties (agents, loan officers, escrow companies, etc.).

What it does

We developed a simple documentation repository to bring together all documentation in one place. No more looking up emails. You can comment on uploaded documents, set deadlines in a calendar, edit and share task lists/deadlines, and chat live with your real estate agent/loan officer.

How I built it

We built it using a combination of HTML, CSS, Java, a few APIs. We also used WIX for a simple landing page.

Challenges I ran into

We are all beginners and spent the first half of the day trying to set up our github repository and write very simple lines of HTML code.

Accomplishments that I'm proud of

We managed to add a comment section and live talk feature.

What I learned

We learned a lot about version control in Github and how to work on different components of the site at the same time.

What's next for DocuHouse

Perhaps we will build out this demo further and raise money to start the company!
