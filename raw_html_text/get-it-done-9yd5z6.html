Inspiration

After browsing Facebook for three hours, I decided that I needed an app to help get my life back on track.

What it does

Tracks your time spent on websites Visualizes your data, both in the short term and in the long term Allows you to set a limit on how long you want to spend on unproductive sites

How we built it

We built this application with Google Chrome's extension API. We used jQuery, Bootstrap, and ChartJS.

Challenges we ran into

We had problems with accurately recording duration that is spent on a given tab though we did eventually figure out a strategy to doing so.

Accomplishments that we're proud of

We accomplished the goals that we first sought out (getting all functionality down as well has having some time to work on visualizations)

What we learned

We learned more about the Google Chrome Extension API and debugging within Google Chrome.

What's next for Get It Done

We plan to provide more customization to the user when it comes to the visualization of their data.
