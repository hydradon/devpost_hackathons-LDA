Inspiration

After seeing countless videos of how litter negatively affects wildlife, the team was determined to create a project that would create the same emotional response as we had.

What it does

We hope that this project brings awareness to the dangers of littering in our environment and shows how confusing it can be for wildlife.

How we built it

We used Unity3D and C# to make our project into a small video game.

Challenges we ran into

Using the repository was a problem for our team. Git gave us a few problems and different branches created merge issues.

Accomplishments that we're proud of

The art, as we never thought we personally could make anything as pretty as this. Also the emotional impact the ending has.

What we learned

How much litter actually affects sea turtles in particular, which is why we choose the animal. One of our members had their first experience with Unity as well.

What's next for Shellfish

Making the game more detailed and complete. We would like to include more stages with different affected animals, as well as have a more concrete leveling up process.
