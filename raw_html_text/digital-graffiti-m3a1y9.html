It was based on the 5 guys corkboard concept, but to be run on a projector or tv for companies. It can add a bit of interaction for any business, and creates an element of personalization for the patrons.

Personally, I struggled with wrapping my head around kotlin as it was my first time using it, and Klayton struggled a bit with reacclimating to using Dart and Flutter. It was quite a fun project, and it was nice to not have to worry about integrating with external APIs
