Inspiration

Other leading language learning apps do not focus enough on the concept of gendered words.

What it does

Offers flashcard-based practice tools for vocabulary building in a variety of languages.

How we built it

We used create-react-app to lay the foundation, AWS amplify to provide user authentication and build a Rest API from our DynamoDB tables, DynamoDB to store the word data and we wrote the app in React.

Challenges we ran into

Learning the AWS API was our biggest challenge. We had difficulty creating multiple tables in DynamoDB as well.

Accomplishments that we're proud of

Created a sleek, lightweight demo using completely new technology in under 24 hours.

What we learned

We learned how to use AWS amplify to automate tasks like authentication and facilitate the building of a Rest API.

What's next for Polyg.io

Expanding word database, supporting more languages, creating new quiz options, allowing users to track progress, implementing community features such as sharing progress and competing with friends.
