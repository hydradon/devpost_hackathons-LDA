Inspiration

We love TV shows, but we hate choosing between platforms.

What it does

Takes user preferences and provides an algorithmically derived streaming platform recommendation

How we built it

Designed a survey with 11 questions on user preferences. With the help of Tkinter, a python library to build GUIs, quantified the user's responses and finally, used a point system to match user data with a streaming service provider.

Challenges we ran into

Finding a consistent, reliable way to quantify responses

Accomplishments that we're proud of

Finishing our first Hackathon and staying motivated the whole time. Having a final product we can continue to build on.

What we learned

Basic GUI building How to work with frameworks like matplotlib to visualize data

What's next for ManageMe

Further developing our algorithm to improve accuracy Web-scraping information from streaming platforms instead of using manual input.
