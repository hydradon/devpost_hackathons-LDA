SmartShares

Cap Table Management with the Blockchain

Project Overview

SmartShares allows any company to manage their complex cap table. At any time the company can: 1) Get an overview of number of shares outstanding 2) See who owns what percentage of the company 3) Automatically update vesting schedules 4) Easily issue new shares 4) Simply track if shares are sold in secondary market

To run the app 

yarn install
npm start
testrpc
truffle compile
truffle migrate
