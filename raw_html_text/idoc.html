Inspiration

Mayo Clinic and Web MD scares me way too much when I try to look up some symptoms.

What it does

It asks the user for some initial data, and then it analyzes that data and provide the users with questions using Artificial Intelligence. Once it gathers enough data, it provides the diagnoses.

How I built it

I used the Infermedica API and a custom wikipedia API.

Challenges I ran into

A lot of the API's I was originally going to use all require me to contact the company first to receive the API-KEYs so I end up discarding a lot of my features.

Accomplishments that I'm proud of

I am proud that I actually got the questions to work and especially the wikipedia API as well.

What I learned

I learned how to use Natural Language Processing to process user inputs.

What's next for iDoc

I am going to integrate the DrugBank API once I have the API-KEY and offer the users suggestions on over the counter drugs.
