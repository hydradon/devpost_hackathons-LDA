Inspiration

We developed ballpoint to fill a need for more structured productivity and concentration on text-editing apps like Google Docs.

What it does

Enables users to set goals and create checklists that match the type of composition they hope to write.

How we built it

ballpoint is deployed as a Chrome extension, and was built by interfacing with the Chrome API. We also isolated front-end from back-end for increased efficiency.

Challenges we ran into

Adjusting the the asynchronous nature of JavaScript.
