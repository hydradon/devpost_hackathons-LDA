Inspiration

We enjoy playing games when hanging out with friends and dring some beer so we wanted to mix up little games in much bigger one !

What it does

First fill in your infos for the game ! Then, find your partner, clues and solve the enigma ! If you lose a little game be prepared to do some funny actions !

How we built it

We used React Native for the app, Dialog Flow API and Python for the exchanges and the game and socket.io with flask to exchange between all of it !!
