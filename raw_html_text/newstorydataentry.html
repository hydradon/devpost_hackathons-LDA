Inspiration

We heard about New Story's struggle about offline data transfer and felt we could develop an app that help ease this process.

What it does

The app is able to create new families for New Story's project. The data is stored locally in .csv format.

How we built it

We used Android Studio

Challenges we ran into

The original intent of the project is to create a SQLite data form in the Andriod port and send these data to cloud MySQL. However, we have problem implementing this feature because it required proficient knowledge of backend and PHP development.

Accomplishments that we're proud of

As a team with no experience of Android development, we finished a functioning app within 24 hours.

What we learned

Android, SQL, UI

What's next for NewStoryDataEntry

Look to implement SQL feature and data retrieval into Android port.
