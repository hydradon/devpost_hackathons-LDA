Inspiration

People use earbuds for countless of hours but don't realize the consequences it can have on their long-term health conditions.

What it does

It tracks earbud usage and provide a graph/ statistics on the amount of time spent using earbuds

How I built it

using xcode and swift

Challenges I ran into

using swift and xcode to code for the first time.

Accomplishments that I'm proud of

debugging and making it function as we had planned.

What I learned

how to use swift and xcode and work as a team to create a successful and useful project.

What's next for Unplug

integrating a volume meter to measure the volume of audio playing and send notifications if volume is too loud. send reminders after a certain amount of time to take a break from earbuds.

Built With

  • xcode
