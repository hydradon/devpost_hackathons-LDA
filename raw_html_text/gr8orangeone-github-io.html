Inspiration

I am the graphic designer for the club and I wanted to redesign our aestetic and try to make the design more modern and interesting

What it does

this site integrates our events and contact and currently has place holder media as I do not have any but hope to get some and not give up any of the videos. There is also a dedicated mobile design along with the desktop one.

How I built it

I mostly used Adobe Muse to create the site however I am using git hub to host the site for now.

Challenges I ran into

I had not used muse very much before this event but now would consider myself proficent. I also encountered a number of issues along the way trying to get a sticky header, and keeping the website acessible on a number of different resolutions and devices.

Accomplishments that I'm proud of

I feel that having really learned muse I can continue to create some great sites. I am proud that I was able to create such a nice site in such a short amount of time and I feel with more time it will only get better

What I learned

I learned how to use get hub a bit better and I also learned muse as well how to make a visualy apealing site for a number of platforms
