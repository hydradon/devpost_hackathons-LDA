Inspiration

To make learning fun

What it does

Augmented reality app that helps you learn a language through recognizing objects and giving you translations for those objects.

How we built it

Machine Language is an iOS application built using Xcode utilizing Apple's machine-learning and augmented reality application program interface.

Challenges we ran into

Trying to get the API to be consistent with its recognition of real-world objects while trying to create the best learning environment was a bit of a challenge but ultimately we feel as though we figured it out and arrived at a happy medium by making the words that were unrecognizable still a learning experience.

Accomplishments that we're proud of

We feel like we arrived at interesting ideas for future apps that will be able to make learning fun and immersive.

What we learned

Through the process, we've arrived at interesting concepts about how augmented reality can be used to help students learn. We also feel like we've made leaps and strides driving into apple's ARKit and AVKit and learning ways to alter the environment in interesting and interactive ways.

What's next for Learn

We look to make more augmented reality applications that make learning seamlessly.
