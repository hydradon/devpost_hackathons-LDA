Inspiration

An environmental data sensor project for indoor gourmet mushroom cultivation aimed at reducing the cost of instruments, making the process more and involved and providing data for reference in future projects.

What it does

This project will measure Air quality, Humidity, Light, and Temperature data for the cultivators use. when a project is completed the data averages will be provided so the cultivator will be able to best determine location suitability for results of yield.
