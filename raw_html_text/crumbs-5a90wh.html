What it does

Our Android App collects and pushes out data to its users. Data is aggregated across users, and the user-driven collection process is incentivized through the use of virtual currency.

How we built it

The front-end code is built using Android Studio. The back-end is built using Play framework with MySQL.

Challenges we ran into

To incentivize users, especially those in war-torn areas, to remember to use this app even after a certain disaster.

Accomplishments that we're proud of

Dynamic quest surveys Geolocation-specific surveys

What we learned

Incentivizing people is difficult to use an app, especially when their life is under threat. Coding for 20-hours straight turns even the strongest to zombies, but it's also a great team-bonding experience.

What's next for Crumbs.

Opening it up to the public using GitHub Market validation
