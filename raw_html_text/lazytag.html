Inspiration

We wanted to share pictures anonymously with people but we're too lazy to go through the effort of tagging the picture as is common practice in this day and age. So if an app could do that for us it would be a load off our minds.

What it does

It allows users to upload an image which is automatically tagged using the Clarifai API. You can also search for pictures which correspond to the most popular tags among all users.

How we built it

Split the main screen, camera functionality and server functionality into three separate applications and then joined them together.

Challenges we ran into

Git problems, AWS problems, Firebase problems, lack of sleep problems, etc.

Accomplishments that we're proud of

We managed to get a lot of these problems built and solved from the ground up

What we learned

How to use a third party API and database services

What's next for LazyTag

If we aren't too lazy, we'll try to squash a bunch of bugs, add voting and search functions
