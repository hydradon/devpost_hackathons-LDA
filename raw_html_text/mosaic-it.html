Inspiration

We wanted to make a collage of our pictures for our student organization that spells its name, but we could not find software to do it. We decided to do it ourselves.

What it does

Mosaic it takes pictures, sorts it by how dark the image is, then uses the darker images to make a background and the lighter images to make words.

How we built it

We built it using Python, OpenCV, and numpy.

Challenges we ran into

We lost a team member. We were having trouble with the software we chose.

Accomplishments that we're proud of

We had to create an algorithm to sort how dark an image is and put images in an array from darkest to lightest.

What we learned

We learned how to use Python, OpenCV, and numpy.

What's next for Mosaic it

Expanding functionality to different size and colored images. We also wish to implement multi-language text.
