Inspiration

Why not create a game in the Insta/Facebook filter which you can spend hours to play with and even share with your friends?

What it does

It is an Augmented Reality game which uses the movement of your head and your mouth reactions as game controllers. It is a space game, you can control the ship by turning your head and got to dodge the falling planets. you can play it in your own environment or the space itself, just switch the background by opening your mouth..!

How I built it

I built it using Spark AR Studio. I challenged myself not to use a single line of code, and hence this game is made without a single line of code written in it.

Challenges I ran into

The biggest challenge was the time challenge, along with making it done without a single line of code. The patch editor is so powerful that it made it happen.

What I learned

I learned to use Spark AR Studio, specially the patch editor so that we ACTUALLY can build awesome stuff without writing code.

What's next for Space Run

To be deployed and shared across users to have fun and make it a multiplayer platform integrating fb login somehow.
