Inspiration

We are both competitive collegiate dancers, so we are very active. But we aren't really interested in investing a lot of time and effort into fitness, we just like to dance and we want our bodies to be able to support that. We created this website for people like us!

What it does

Provides resources for people to get information about foods and simple exercises they can to do maintain an active lifestyle in a healthy way. The website also provides resources to address simple injuries.

How we built it

HTML and CSS

Challenges we ran into

Getting our files up on the domain name.

Accomplishments that we're proud of

Getting a website up that actually looks pretty good (last time we tried to make a website, it was one plain text page..)

What we learned

A lot about HTML and CSS as well as how to transfer files to a domain

Who knows? But we really like the idea.
