Inspiration

Wanted a "siri" like application to work exclusively on facebook messenger conversations especially considering that a lot of people use messenger to communicate these days.

What it does

Displays message from friend, then lets you speak to the bot instead of typing which automatically gets sent to them as text. This will let you multi-task efficiently as you don't have to waste time + get more distracted by manually going to the page and hitting reply while you could just do it over voice.

How I built it

Node.js + facebook messenger API. The implementation of the speechAPI is still in process.

Challenges I ran into

Was new to node.js so learning it and trying to make this work was a challege.

Accomplishments that I'm proud of

Got a good understanding of node.js and working with the messenger API.
