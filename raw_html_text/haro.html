Inspiration

In the event of natural disasters, resources gathering and distribution should be coordinated. Our project makes it easy for resource seekers to post requests in the local community, and resource providers to provide resources.

What it does

There are basically two types of users for this Web App, user and volunteer. User will send resource request to the server. Volunteer will take the request and deliver the resource.

How we built it

We use React, Redux, Express and Google Map API.

Challenges we ran into

We are not quite familiar with the tech we used, so it took us some time learn and implement everything.

What we learned

We used the opportunity to learn some cool front end tech. We learned how to use Redux and react router, how to use Google Map API.

What's next for HARO

Convert the React App into React Native App Some features are not well implemented yet. For example, volunteers and users should see different home pages
