• Overview

The user gives the following input:

  • Topic
  • Two media outlets
  • Start and end date for analysis

Our product, Point of View, does the following:

  1. Collects relevant articles published within the provided date range by scraping Google search results
  2. Performs sentiment/key-word analysis with IBM Watson's Natural Language Understanding API
  3. Outputs an interactive comparison graph showing how each media outlet covered the topic
