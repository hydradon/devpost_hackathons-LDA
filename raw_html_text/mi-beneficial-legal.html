Inspiration

Non-English speaker has a hard time filling out the immigration forms, so our non-profit wanted to create a website that's similar to TurboTax where you fill out the information on the website and once you're done, a filled- PDF will be outputted.

What it does

Our web app will first prompt the user to choose the form they want to fill out. The app then will ask the user to answer questions from the form, and the questions will be shown one by one. After all the necessary questions are answered, the app will input the answers onto the form's paper format. The user, then, can download the form to be mailed to the embassy or other immigration services.

How we built it

We used React to build the UI of the web app, and Java for the back-end development.
