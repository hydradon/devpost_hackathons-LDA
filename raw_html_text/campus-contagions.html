Inspiration

We feel that campuses are a breeding ground for rapidly spreading disease. Having a platform where individuals can track symptoms and disease trends in their area can help people to make more health-conscious decisions.

What it does

Our application serves as a dashboard for students to search for symptoms they may be experiencing, look for nearby doctor's clinics, and see where symptoms have been supported on campus.

How we built it

We built this web application using python flask and using the google maps and webmedic apis. Our database hosting was done in google cloud.

Challenges we ran into

We had no familiarity with python flask, so that was a learning experience for all of us. The google maps api does not provide the ability to map directions to nearby clinics, which was very frustrating.
