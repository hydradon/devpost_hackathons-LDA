Inspiration

Our inspiration is the current education system. With increasing technology people are implementing training coupled with new technology which helps the students learn in a better way

What it does

This app basically helps the kid to learn the basics of alphabets and number system

How we built it

We built it using amazon sumerian and window's speech recognition

Challenges we ran into

Our primary challenge was implementing a fool proof speech recognition system

Accomplishments that we're proud of

We are proud that we have implemented the speech recognition and changing the scene's entity based on the speech input from the user

What we learned

We learned some intermediate technical aspects of amazon sumerian and window's speech recognition.

What's next for Kids Trainer

Our future target is to add more courses for the kids to help them learn in a much better way
