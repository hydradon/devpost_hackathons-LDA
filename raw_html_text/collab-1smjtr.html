Collab is a web application meant to make it easier for students to study.

By using Collab, students can create and join study groups with friends closeby or far away. Through Collab, students can message each other about anything that they are studying and they can even search Wolfram Alpha by simply typing "wolfram ..." .

Built With

  • react
  • ruby-on-rails
