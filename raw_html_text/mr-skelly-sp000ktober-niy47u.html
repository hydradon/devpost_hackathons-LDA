Introduction

In the spirit of Sp00ktober, we decided to create a basic 2D platformer that was Halloween-themed. It runs exclusively in Jquery, Javascript, HTML, and CSS, and so it can be run on most modern browsers.

Game Mechanics

The player is capable of moving left, moving right, and jumping. Additionally, it can aim a gun to fire at enemies along the way, which all have a set amount of health. The player loses health when colliding when enemies, and gets a Game Over if they die prior to reaching the end of the level. If they reach the end, they win! Levels are randomly generated with enemies and platform layouts.

Production details

We started off from a generic 2D Platformer in JavaScript, and eventually added more nuanced functionality and game mechanics. Every piece of art seen in the game was custom-made by members of the team.

Challenges we ran into

Collisions were rather time consuming and difficult; adding new images also proved to be tedious at times.

Accomplishments that we're proud of

The game has 3 complete themes with completely random level configurations every run. Every sprite image in the game was custom-made by the team. It works! :)

What we learned

JavaScript is not the ideal environment for games; another platform and some preparation would have suited this project better.

What's next for Mr. Skelly: Sp000ktober

Likely extensions to this game include boss battles, maps, dialogue, and a storyline.
