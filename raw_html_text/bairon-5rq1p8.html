What it does

Bairon is a human-ai collaborative text editor. Bairon is able to give inspiration and help overcome writers block by offering suggestions based on what you have already written (Similar to Gmails new auto-compose feature).

How we built it

The backend API is written in Ruby on Rails using GraphQL. The frontend is currently a react web app and we are considering mobile platforms in the future
