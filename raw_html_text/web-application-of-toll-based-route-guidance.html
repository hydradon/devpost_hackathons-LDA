Inspiration

Tolling is reactive to traffic conditions with a time lag Unexpected events blocking certain entry and exit points Toll revenue loss Transportation inequity No user-based information

What it does

Helps toll users to choose a route based on their departure time, targeted arrival time, and their budget

How we built it

We used open source tools which includes R, Google Maps, and leaflet.

Challenges we ran into

Historical data is not tidy and explanatory enough and dataset is small.

Accomplishments that we're proud of

We developed an offline prototype.

What we learned

Start early.

What's next for Web Application of Toll-based Route Guidance

Mobile app.
