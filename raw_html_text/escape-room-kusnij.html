Inspiration

I am very interested in working with video games in the future, however, since I had no prior experience with code, I didn't want to start on an all out video game, but instead I have used text-based games and was interested in trying it.

What it does

By inputing certain words you are able to navigate a small room from which you are trying to escape.

How I built it

Using java and eclipse, I just used strings and functions to progress the narrative.

Challenges I ran into

I had to learn java as I went, and I got very confused at times when I ran into more complex issues.

Accomplishments that I'm proud of

It actually runs, and although it's not very long or perfect it still does what it is supposed to.

What I learned

I learned java and coding.

What's next for Escape Room

I am considering taking what I've learned from developing this game and using it to make another more complex game with a similar basis.
