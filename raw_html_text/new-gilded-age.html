Inspiration

We wanted to quantify the level of income disparity that has continued to grow nationwide and across the globe over the past few decades. We were inspired by the description of how poor the common person was when compared to the top 1% during the Gilded Age of American history during the 19th century.

What it does

Creates graphs and charts that helps visualize the level of income disparity in an area.

How We built it

We used income inequality data from the World Inequality Database and created a data source using the OmniSci Cloud database. We hosted the website using Amazon AWS and used javascript to link the database and the data visualizations to the website.

Challenges We ran into

Integrating the OmniSci tools with the schema that we used to design our website

Accomplishments that I'm proud of

Created a database and used the tools provided by OmniSci in order to transform that data into a readily understandable format.

What We learned

Building a website and integrating a database with that website design.

What's next for New Gilded Age

Expand our current database to focus on more specific regions. Increase the amount of queries that it processes and integrate predictive tools to show future trends in income inequality.
