Inspiration

We were inspired by the Ask for Angela campaign, which encouraged bar patrons to use a code word to discreetly notify staff that they feel unsafe. We brought this idea to a mobile application, where the user can discreetly alert businesses or other nearby users.

What it does

Sends an alert to nearby business-owners or other users in the database with the information of the current user to help the person in need.

How I built it

We used Android Studio.

Challenges I ran into

Integrating different aspects was the most difficult. Getting the Google Maps api into the app was also challenging.

Accomplishments that I'm proud of

We figured out how to manage the different versions of tools we had, which included the ui, sms messaging, and Maps aspects.

What I learned

We learned to develop android apps using android-studio, as well as using Google Maps API, sending sms messages, and managing data.

What's next for Angela

We hope to implement live location tracking integration for users and businesses. We also hope to have cloud storage for our user database.
