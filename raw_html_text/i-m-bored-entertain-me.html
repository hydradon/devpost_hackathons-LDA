Inspiration

Lifestyle websites such as The Smart Local, The Weekender & Eventbrite

The Problem

  1. Information not from a centralised source
  2. Users have to manually mix and match activities when planning for a day of activities

Solution

A centralised platform where users can view all the nearby ongoing activities and events while an added functionality to generate a dayplan.

  1. Specify Outing Parameters
    2. To start off, users will have to specify some basic information about their location and food preferences. This information will be processed and all possible activities will be populated to the user.
  2. View all possible activities/food
    3. The activity dashboard will consist of all the activities the user can do.
  3. Generate a day plan
    4. A day plan will be generated for users, with details for each particular timeslot.

Front End

  • Web application built using Vue.js framework.

Back End

  • Python backend utilising flask library for handling requests.

Database

  • Firebase firestorm for real-time data updating and query.
