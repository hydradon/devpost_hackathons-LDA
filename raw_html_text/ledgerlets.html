• Interface for signing files

  • File's signature uploaded and waiting in the queue to be submitted to the blockchain

  • File's signature captured on the blockchain

Ledgerlets lets consumers store a proof of existence of their documents on the ethereum blockchain in a secure manner and for FREE!

It was built with a combination of node.js as the web service and python using the web3 library to do all of the magic behind the scenes

This service is now on http://www.ledgerlets.com . It provides basic functionality of using the blockchain to store proof of existence of documents for consumers but planning to also extend it to enterprises with support for private chains and additional file information
