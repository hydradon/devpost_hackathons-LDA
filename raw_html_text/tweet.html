Inspiration

https://www.mapd.com/demos/tweetmap/ really inspired us. We found exploring the world with social media is pretty cool.

What it does

Our project takes a hashtags and keywords and in real time maps the locations that that hashtag or keyword is sent from. This information can be used to identify how they are doing in the market and compared to their competitors.

How we built it

We built our app with python to make a sub api of the Twitter api, and then we used that sub api in our React app.

Challenges we ran into

Creating our own api and deciding on what technologies we were going to use were the hardest parts. Pushing streaming data to front end.

Accomplishments that we're proud of

Making our first api!

What we learned

We learned a lot about working as a team with Github and Slack. Front end-Back end realtime data interaction.

What's next for Tweet

Not sure at the moment, but it was definitely an interesting project
