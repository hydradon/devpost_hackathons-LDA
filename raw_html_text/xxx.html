Inspiration

Being at a party and not being able to control what music you’re listening to.

What it does

Android app that searches youtube for videos to add to a playlist. Plays video on phone screen.

How I built it

Hard Work and Caffeine! Used Github, Android Studio, Youtube API, Android Device, and Adobe Illustrator for this project.

Challenges I ran into

Little sleep, over thinking a line of code, Adobe Illustrator crashing… The normal daily grind of every coder and designer.

Accomplishments that I'm proud of

Originally we wanted to use ChromeCast as part of the project. After running into unforeseen obstacles though, we dropped that part of our project. However, we were able to collaborate, discuss, and improvise for our project. Without good communication we would not have been able to conquered that hurdle.

What I learned

We learned so much more about Chrome cast, Androids, HTTP Requests, JSON, APIs, Client Server communication, Teamwork and Development Setting, UI / UX Development.

What's next for CastBox

Have multiple users, integrate Spotify instead of YouTube, have server work correctly, integrate Chrome Cast, thumbs up feature.
