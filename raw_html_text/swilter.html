Inspiration

Comments sections on some websites are often filled with offensive and ad hominem comments which don't contribute to any meaningful conversation.

What it does

Implements a comments section that can be plugged in to any website and includes a built-in filter for profanity and ad hominem comments.

How we built it

JQuery, Browserified Node, Firebase, Azure and Giphy APIs, all packaged up as a Cloudflare App.

Challenges we ran into

Sleep.

Accomplishments that we're proud of

Decently beautiful design for a hackathon project

What we learned

Promises are cool. Git is hard.

What's next for Swilter

We want to publish this on the Cloudflare app marketplace!
