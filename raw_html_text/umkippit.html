Inspiration

At Zeiss, there is the problem of an element moving after the mending process. There is a critical translation and rotation that is not allowed to happen. If it is outside of a predefined quality score, it needs to be rejected.

What it does

Extracts the transformation matrix based on the before and after images

How we built it

Using python with computer vision libraries like opencv

Challenges we ran into

Filtering out unwanted background structures, finding and filtering the right corners
