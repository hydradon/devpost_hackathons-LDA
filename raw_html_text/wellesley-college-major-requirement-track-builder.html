Inspiration: Difficult to navigate current course browser and hard to find department major pages

What it does: It is designed to outline the courses needed to fulfill different majors at Wellesley College, graph/layout the next x number of years that students have at Wellesley

How we built it: We tried a lot of different programming languages and software, but finally decided to use javascript and html as we were the most comfortable with them

Challenges we ran into: Not knowing python, the website builder didn't accept javascript

Accomplishments that we're proud of: we created a working page with input and output

What we learned: We learned more about python and other coding languages/software

What's next for Wellesley College Major Requirement Track Builder: We hope to expand the builder so that it is available to all Wellesley students and majors, create a more sleek design that is easy to follow and understand as well as dynamic
