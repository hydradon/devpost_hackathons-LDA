Inspiration

Have you ever had a social event that got off to a slow start?

Or an awkward business meeting and didn't know what to say?

No more! With our revolutionary new ice breaking technology jump right into getting to know each other... and your embarrassing tweets.

(or see how well you really know your favourite celebrities).

What it does

Twitcher is an icebreaker game where you are presented with random tweets and have to guess who has written them. You decide which users the tweets will be selected from, so the game can be played with a group of people and used to get to know each other's tweeting tendencies!

How we built it

The back end was built using Node.js to interface with the Twitter API and communicate with the front end via WebSockets. The front end was written using HTML5 and CSS3 with additional JavaScript functions to handle client requests and manipulation of the webpage.

Accomplishments that we're proud of

We won an award! American Express kindly chose our project as the winner of the Icebreaker challenge and we had great fun making it. Hasan stayed up all night making the game look amazing and we're very proud of him for this :)
