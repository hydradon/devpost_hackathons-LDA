I wanted to use image data from Urthecast's API, so I made a jigsaw puzzle where the user can select an map tile to make into a puzzle. The image will keep zooming in until the user selects "Generate Puzzle". It uses the in-orbit Theia camera image data.

It is meant to be a light, fun project!

Note: it uses a pre-existing puzzle JS file to make it easy to create puzzles (SnapFit).
