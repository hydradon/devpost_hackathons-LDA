Inspiration

One of the most frustrating situations while doing good work worth doing is using tools that don't pull their weight. Inefficient workflows can create a bulwark to reaching people in the need of assistance.

What it does

Connecting multiple data streams into one easy to use app. Leverage GraphQl Querys to bring in demand information to the fingertips of admins providing data to funders

How we built it

Scaffolded an intuitive GraphQL Database Authenticated admin routes using Firebase Collect New user info in React with materialize UI

Challenges we ran into

Prioritizing MVP

We are very proud of our

GraphQL Database which is fully flushed out and has powerful filtering fulfilling proof of concept

We learned

The demands of a project of this scope within this timeframe.

What's next for CareConnect

increasing the utility of Data Results, exporting in CSV
