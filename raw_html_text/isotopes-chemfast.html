Inspiration

Protests and Chemistry

What it does

Connects people to events around themselves such as a Chemistry tutoring class. The game allows the audience to gauge interest in the event.

How we built it

The protest app was built through Swift and the game was built through Java

Challenges we ran into

Deciding on which platform to execute our plan

Accomplishments that we're proud of

Being able to have the user interact with our idea on multiple platforms

What we learned

Multiple platforms can come together to execute a single idea

What's next for Isotopes-ChemFast

Hopefully, we can connect more events and games to allow the user to gauge interest in the events they will attend.
