Inspiration

As all 3 members of this group are active musicians, we thought of the importance of developing this skill as children. With our ability to program and the theme of art, we wanted to create a tool/game to help the youth develop their visual recognition and pitch relativity.

What it does

This application has 3 difficulties with 3 varying sizes. Here, you will be tasked to match the melodies with the appropriate choice or choices.

How we built it

This application built using Java.

Challenges we ran into

The Java GUI is a new concept for us and we spent a lot of time debugging. On top of this, our design process was failed to contain important information that was vital to how we would work and communicate in separate parts.

Accomplishments that we're proud of

This application, overall, is engaging and musically entertaining. We are proud of how our skills applied to an application we find would be very beneficial to the youth today.

What we learned

Through this experience, we learned the importance of communication and roles responsibility.

What's next for BeatRepeat

We plan to make this into a proper iOS/Android application for production.
