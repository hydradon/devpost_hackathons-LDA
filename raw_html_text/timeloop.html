Inspiration

Time management.

What it does

Helps teens save time by keeping track of how much homework they have. We used the school's averages for each class to accurately calculate student's homework time. We also planned to include time spent doing extracurriculars, and also things like eating and sleeping.

How we built it

We made a chrome extension to perform the calculating and schedule displaying, while also getting data directly from school loop so that students wouldn't have to input their class data. We also had a website made with HTML5 and CSS.

Challenges we ran into

Getting the chrome extension to read data from school loop.

Accomplishments that we're proud of

It can read data from school loop.

What's next for TimeLoop

Planning to have it learn more about the user with machine learning. If a user consistently spends more time on a task than the predicted time, the extension will learn to allot more time for that task.
