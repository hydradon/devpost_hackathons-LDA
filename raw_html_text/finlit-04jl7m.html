Inspiration

We were inspired by the impact of the new era of banking services, becoming local centers for business and wanted to bring these benefits to underserved communities where the services are needed most!

What it does

This project optimizes the location of financial literacy workshops based on the public need for them based on a machine learning model constructed to analyze data.

How we built it

The application is cloud hosted on DigitalOcean and uses Django to serve a web page. It also uses a sms chat bot using the Twilio API as a way to reach out to those without constant access to computers or smartphones.

Challenges we ran into

Accomplishments that we're proud of
