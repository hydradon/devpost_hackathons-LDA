Inspiration

We all know how annoying it is to go for purchase or to the favorite restaurant and notice that a place is very busy.

What it does

Crowdy shows you how crowded a location is at the moment. Places can be shared with friends. Favorite Places can be chosen.

How I built it

Crowdy is built as an Android App.

Challenges I ran into

There is no Google API to get the Live Visit Data of locations.

What's next for Crowdy

Crowdy will find locations in a named radius.
