Inspiration

Many students forget homework and assignments. In addition, some events and due dates are scheduled in such close proximity to one another that it becomes difficult to remember them.

What it does

The Scheduler is THE collaborative scheduler that uses info from many people to make a complete list of all upcoming assignments for every class in the school. People can check homework by class and by due date.

How we built it

We used the node .js module discord.js and a few others along with atom's teletype and github

Challenges we ran into

We've never used javascript before, but we like learning. We also could't get Facebook developer to work properly.

Accomplishments that we're proud of
