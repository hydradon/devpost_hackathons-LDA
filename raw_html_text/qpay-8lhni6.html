Inspiration

Snapchat QR code can add a friend quickly and precisely. Venmo can quickly do mobile payments but risk fat-finger mistakes. Apple Pay is fast and no need to enter credit card info or carry the card, but it requires additional hardware.

What it does

2 factor authentication built into the process. Removing the inconvenience of 2FA, and instead making 2FA the convenience of this process.

How we built it

Open up Android Studio and vim, then some 36 hours later, it was a blur.

Challenges we ran into

First time developing on Android, 'twas a blast.

Accomplishments that we're proud of

There's an actual working backend server that's handling all the requests and not hacked together BS.

What we learned

Android dev hard, web dev easier, actually Android documentations long, makes me yawn.
