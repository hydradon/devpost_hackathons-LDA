See sharp graphics uses C# and Unity to create an online environment called the "Purple Room" where people from around the world can come and mingle. The Purple Room is currently a basic program where a user can move around and explore the 3D landscape, but it has the potential to expand to include more user to user interactions, separate chat rooms for online learning, conferences, etc. This program differs from existing programs in that it takes place in virtual reality using an oculus rift. Again this is a basic model, in the future the Purple Room can expand and become more accessible through making the program run on a Google glass.
