Inspiration

Many years of browsing websites lent me inspiration to make this satirical text adventure comedy.

What it does

Plays a puzzle-solving adventure through various websites of the internet, complete with mouse-drawn artwork.

How I built it

I used Quest, a free engine for text adventure games, custom Java coding, and MS Paint for artwork.

Challenges I ran into

I ran into problems with custom Java code not behaving properly. I was able to resolve this by taking a different, and altogether smoother method.

Accomplishments that I'm proud of

The first game I have made solo! Usually I work with a team, so being able to finish a game on my own is very rewarding.

What I learned

I learned never assume that programming will be easy. I also learned important facets of game design and implementation.

What's next for Execrable Exchange

I intend to polish the game and also make the next chapter.
