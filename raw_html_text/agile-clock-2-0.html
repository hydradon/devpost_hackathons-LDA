Inspiration

As a Scrum team, we need had problems with efficient time management on our meetings. We use to work with JIRA Agile board and we consider that we need to see the clock during the meeting.

What it does

Agile Clock is time clock counter placed on the JIRA Agile board.

How I built it

It is jQuery script with configuration saved inside JIRA add-on.
