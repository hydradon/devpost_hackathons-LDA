Inspiration

We thought it's cool to play songs that are very close to exactly how you feel at the moment

What it does

Provides a range slider for user to select how happy or sad they are out of 10. (0 is saddest and 10 is happies). then plays songs.

How we built it

We used angular5 for frontend with typescript. and used python and flask for backend and API calls.

Challenges we ran into

We weren't fimiliar with http get and post request using angular and spent a lot of time on it. Also using flask was new and making the right API calls from and to front end was a real challenge.

Accomplishments that we're proud of

Front end completed 100% and backend is almost ready. but it still needs some final touches.

What we learned

We learned how to use http post and get request using angular. also learned about flask.

What's next for Emotion Slider

Maybe putting some final touches to make it work.
