Inspiration

We wanted to learn more about APIs and how to use them. SmartCar API seemed like the perfect opportunity for us since our whole team is made up of first timers. The simplicity of the API helped not overwhelm us.

What it does

The program displays basic statistics about a vehicle given that the owner authenticates the application and the car is compatible

How I built it

Python and HTML using the smartcar api and the openweathermap api

Challenges I ran into

Learning the syntax for python and learning how to apply the api

Accomplishments that I'm proud of

We got it to work and as first timers that was huge for us

What I learned

We got experience with APIs and a widely used language

What's next for SmartStats

More stats!
