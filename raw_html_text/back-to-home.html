Como funciona? La persona entra al auto , se sienta y se pone el cinturón de seguridad, sopla en el alcoholismo y si esta ebrio el auto no enciende y si no el auto funciona normalmente

Podrías explicarlo? Para que el auto encienda tienes que tener la puerta cerrada, tener el cinturón de seguridad puesto y no debe de sobrepasar el promedio de nivel de alcohol estándar para conducir un automóvil, de otra forma el automóvil no podrá ser usado

Como haces para verificarlo? Tanto la puerta como el cinturón de seguridad cuenta con placas de presión que verifican cuando el auto tiene las puertas cerradas y tienes el cinturón de seguridad puesto, si no cumples con estos 2 parámetros no te dejara hacer la prueba de alcoholismo, esto para verificar que solo la persona que esta sentada en el asiento del conductor y que tiene el cinturón de seguridad puesto hará la prueba de alcoholismo.

Si te quitas el cinturón de seguridad el vehículo se apagara eventualmente dándote oportunidad a orillarte y detenerte en forma segura, y se tendrá que volver a hacer el proceso para volver a encender el automóvil

Extras que podría tener:

Este modo solo funciona cuando es activado por el usuario, pero teniendo en cuenta la tecnología que se esta implementando en los nuevos autos se podría mejorar el sistema usando una función de giroscopio y acelero-metro el cual podría verificar sin necesidad de que actives el modo seguro que estas conduciendo de una manera incorrecta o rara mandándote alertas para decirte que si sigues manejando así el auto se detendrá eventualmente en cierto tiempo y tendrás que verificar tu estado de ebriedad.

+(Cuando conduces en estado de ebriedad ocurren varios factores como cambios bruscos de velocidad y de dirección, esto es lo que detectaría estos sensores)
