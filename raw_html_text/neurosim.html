Your inspiration

We wanted to see how our body and minds would react in stressful situations using virtual reality

What it does

NeuroSim tracks your vital signs while you are subjected to different environments in virtual reality in order to train yourself to react more calmly in stressful situations.

How we built it

The virtual reality is done using the Oculus Rift and we used a Muse for brain wave sensing

Challenges we ran into

Visualizing the raw output data from the Muse was a challenge at first but we were able to eventually figure out which waves were relevant to our project

Accomplishments that we're proud of

The website looks really clean and we were actually able to get real data or someone experiencing fear in virtual reality

What we learned

We learned how to set up an Oculus, and how to use a Muse headset. We also improved skills in web development.

What's next for NeuroSim

Moving forward we would want to take into account more brainwaves when building our model and possibly include more sensors for heart rate and other factors
