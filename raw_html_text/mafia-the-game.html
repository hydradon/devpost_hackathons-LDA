Inspiration

We wanted to make Mafia much more accessible to more people. Lobby system inspired by COD.

What it does

Our Web Game replaces the need for cards. In this day and age, everyone has a cellphone so this game would be easily accessible to anyone who wants a fun game to play.

How we built it

We initially thought of a real-life game that could be converted into a game on the internet, and Mafia was perfect for this.

Challenges we ran into

We ran into several issues with the database because none of us were experienced in MySQL

Accomplishments that we're proud of

We managed to get the game lobby system working perfectly. In addition, we figured out how to use MySQL to meet our needs.

What we learned

Making a game is a lot harder than we initially thought, especially with limited experience in programming.

What's next for Mafia: The Game

We plan on adding many new features, improving the graphics and user interface, and lastly, we plan on improving the performance of the site.
