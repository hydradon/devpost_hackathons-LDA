Inspiration

Inspired by having fun and having a good time, this project gives a much needed laugh to weary flight travelers!

What it does

When you go to flywithfriends.online, it shows you how you can have fun flying with your friends.

How I built it

Using html and firebase, I built it with much dedication and enthusiasm. I bought the domain name on domain.com!

Challenges I ran into

It was hard for firebase to verify the ownership of the domain. It took some time, but eventually we overcame the obstacle.

Accomplishments that I'm proud of

I am proud of deploying a full website to a custom domain!

What I learned

I learned how to deploy a website using firebase. I also learned some more HTML!

What's next for Fly with Friends

Next, we will try to make it even funnier to provide more relief to flight travelers.
