Inspiración

Automatizar el proceso de licitaciones para estar al tanto de todos los proyectos que se desarrollaran en nuestra ciudad o estado y así conocer en que se gastan nuestros fondos.

Qué hace

Mediante una aplicación y un servidor donde concentramos la base de datos, se desarrollaran consultas así como una interacción en tiempo real para automatizar el proceso de una licitación. Desde la perspectiva de la empresa tu podrás ofrecer tus servicios para desarrollar la solución para cierta licitación. Como servidor publico podrás compartir la o las licitaciones necesarias y encontrar una empresa de calidad para resolver la problemática. Y como ciudadano obtener la información como cocer el costo final de las obras terminadas y de la misma manera calificar la obra y así mismo a la empresa.

Cómo lo desarrollamos

El equipo se dividió básicamente en 4 actores: 1. Quien desarrolla la aplicación. 2. Quien monta el servidor. 3. Quien diseña la pagina web. 4. Quien crea y gestiona la base de datos. Para estos trabajos usamos las siguientes tecnologías: 1. Android en Android-studio para el desarrollo de la app. 2. UbuntunServer para el servidor. 3. HTML5 para la creación de la pagina web. 4. MySQL para la base de datos.
