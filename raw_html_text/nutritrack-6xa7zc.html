Inspiration

Calorie counting is tedious enough. We often forget about tracking the nutrients that our bodies require to be healthy!

What it does

We made an interactive chatbot that accepts input from users and performs calculations based on the input, tracking various nutrients such as calories, fat, and iron.

How we built it

The chatbot was coded using Python on the Telegram messaging platform

Challenges we ran into

APIs, having no idea what we had to do

Accomplishments that we're proud of

Trying new things and not being deterred by failure

What we learned

Setting up APIs should be done beforehand and to have a solid plan before coming to the hackathon

What's next for Nutritrack

Audio input, more nutrients, more platforms
