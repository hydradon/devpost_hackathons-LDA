Inspiration

Simple way to view popular articles and news on political figures and their tweets, as they tweet

What it does

Scrapes twitter for tweets, puts tweets into google, scrapes search results, jquery reads the results and puts them on a page

How I built it

Ruby, watir, html, jquery, bootstrap

Challenges I ran into

I tried to create a method to find keywords in tweets but it did not consistently work. Tried to use ajax with jQuery to automate updates of tweet information, but it also did not consistently work.

Accomplishments that I'm proud of

Finished a working project

What I learned

File reads with jQuery are very simple and useful, finding the exact google search results I want

What's next for What the hell are they talking about?

Allowing the user to input a twitter url and scrape tweets from any account, making everything generally more user interactive, also I need to refine the google search to give better results.
