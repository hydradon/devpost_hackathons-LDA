BTNG Hackathon: Bridge The Gap

Sharing knowledge with aspiring tech talent in Nigeria

Motivation

"Brain-drain is a big issue with respect to the development of emerging economies. Skilled people are known to leave their countries of origin, looking for better paying wages and higher quality of living, often times this is to the detriment of the developing countries which require human capital to develop." ( https://btnghackathon2019.devpost.com/?mc_cid=7d39ab3b4b&mc_eid=f67e0d75c0 )

Bridge The Gap is a web application that offers mentorship as a service. The goal is to provide skilled people with the resources necessary to navigate their career in the tech industry.

Working professionals from the UK will provide sessions at least once a fortnight with their mentees (residents of Nigeria) to make sure they are on track to achieving their career goals.

How to use

Please follow these instructions to use the app:

  1. Clone this repo
  2. Open in IntelliJ
  3. Run the program

Technologies

The project is written in Java in IntelliJ.

Hackathon User Stories 

As a user,
So that I can find a mentor,
I'd like to sign up and login to Bridge The Gap.

As a mentee,
So that I can connect with a mentor,
I'd like to see a list of mentors for my career path.

As a mentee,
So that I can find other mentors,
I'd like to search mentors by topic

As a mentee,
So that I can keep track of my progress
I'd like to visit a page to see feedback from my mentor.

As a mentor,
So that I can choose which mentees to connect with
I'd like to be able to accept a connection request from a mentee 

As a mentor,
So that I can choose which mentees to connect with
I'd like to be able to reject a connection request from a mentee
