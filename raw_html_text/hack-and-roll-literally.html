Inspiration

This game is inspired by the name of the Hackathon, Hack And Roll.

What it does

It’s a two player fighting game that involves two players standing on a ball and knocking each other off the ball to get to the end point across the map.

How I built it

We built the game using Unity engine.

Challenges I ran into

We had problems with game mechanics itself as it involves different moving parts.

Accomplishments that I'm proud of

We proud to be able to make write a unique camera that is able to scale dynamically.

What I learned

It is important to plan early so that there is sufficient time to complete the work.
