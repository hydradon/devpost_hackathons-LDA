What it does 

  Mr.Pill contains prescriptions for the elderly, and dispenses the right amount at the right time.

How we built it 

 We created a model of Mr. pill in inventor, then printed the parts. Then for structure, we used straigented paper clips, because printing it all would take too much time. We then messed with the timing of the servo so it only dispensed one pill.

Challenges we ran into 

 The biggest challenge was probably getting the machine to only dispense one "pill" at a time. We also had problems with what we were using for the "pills".

Accomplishments that we're proud of 

 we are proud that we got a working model, even if it didn't complete all of our original aspirations.
