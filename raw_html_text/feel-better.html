Inspiration

Sometimes you miss your friends. Seeing the faces and kind words of the people who matter can remind you of how loved you are. :)

What it does

FeelTheLove compiles positive Facebook posts from your friends to you using pretty animations.

How I built it

I used JavaScript to load data from the Facebook API/generate HTML content, and CSS keyframes to animate the script-generated divs.

Challenges I ran into

Asynchronous calls without promise chaining can apparently effect functions such as setTimeout, making animation via JavaScript difficult for my implementation.

Accomplishments that I'm proud of

I was able to implement the animations using CSS instead of JavaScript.

What I learned

I learned how to use the Facebook SDK for JavaScript, access and display data using APIs, and animate divs using CSS keyframes. I have also been exposed to promise chaining in JS, and will refactor to include it in the future.

What's next for FeelTheLove

Facebook approval (so anyone can use it), Integrate promise chaining, Sentiment analysis using APIs (may require refactoring for node.js)
