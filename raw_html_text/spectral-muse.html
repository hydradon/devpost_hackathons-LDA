Inspiration

I wanted to use my brain. I wanted to make some cool graphics stuff.

What it does

Muse device connects to your phone. Metal renders some graphics. Muse streams some data into the metal graphics pipeline. Colors. Psychedelic shapes. You know, the Werks.

How I built it

I've been working on a graphics library with Metal, called Spectra and working with the Muse peripheral seemed like fun. I've been doing a lot of similar stuff with audio lately and the interface seemed similar.

Challenges I ran into

I needed to write more Objective-C than I wanted. Because of that, I couldn't really interface with libMuse.a as easily as i would have liked. There's probably a better way to do this though.

Accomplishments that I'm proud of

What I learned

Keep it simple. This seems to be a lesson that I miss, all the time, everytime.

What's next for Spectral Muse

It's gonna be hard to work on, since I don't own one. But a device like this would be a lot of fun to mess around with.
