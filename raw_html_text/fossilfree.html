Inspiration

If a person owns a fossil fueled car, they can find a gas station literally anywhere. But electric car drivers, or drivers who use other kinds of fuel, do not have an easy way of determining where a local fuel source is. This app is intended to help with that.

What it does

Determine your location, and provide you with a map of alternative fuel sources near you based on goverment-provided API data

How I built it

Using react, a goverment provided open data API endpoint, node

Challenges I ran into

react state management, component design, CSS

Accomplishments that I'm proud of

google api integration, figuring out a solution for the info windows to support click events from the buttons

What I learned

  • Hackathons are hard!
  • Done is better than perfect
  • Google API usage with a react app is problematic, given its async loading nature

What's next for fossilfree

  • personalization, create an account, save your favorite locations
  • review a location
  • plot your route
  • details view for a station
  • currently responsive - react native version!
