Inspiration

We wanted to create a game that was at our skill level (beginner) but also very fun and addictive

What it does

You use the arrow keys to navigate your way through a designated area for the snake and try to eat the apple. Once you start to get bigger and bigger you have to be mindful to not accidentally eat yourself (resetting the entire game)

How we built it

Using HTML coding and javascript

Challenges we ran into

Running code for different details such as the length of the snake and the size of the border

Accomplishments that we're proud of

Completing and finishing our first HTML code game on time

What we learned

Hackathons provide insight and extra knowledge (if you don't know a lot) on hacking and creating projects with code

What's next for Eat The Apple Avoid Yourself

A bigger more complex version, including a bigger border so the snake can double its size without too much difficulty
