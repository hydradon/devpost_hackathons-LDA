Inspiration

One of our team member wants to listen to his beautiful voice, so we decide to develop a app with similar function.

What it does

This app has a function to record and play the voice of the user.

How we built it

Xcode. We design the UI and connect to ViewController.

Challenges we ran into

We are totally new in Swift4. And we learn it and build the project. From nothing to something. Like, We don't know how to implement the save function for the record.

Accomplishments that we're proud of

My teammate is very very satisfied that his voice can record and have fun on it.

What we learned

Teamwork. We know that we can save a lot of time by working with others. Also, we can get a lot of inspiration from our teammates.

What's next for record_demo

We will try to implement the save function, and add more function on it. We can do the random generator of text. We can add the demo recording. We try to implement a database to connect with our app. So that we can save a lot of audio in our phone.
