Inspiration

What it does

An image classifier and web scraper that determines the rarity of Pepes found

How we built it

We used the 4chan API to webscrape the thumbnail images and TensorFlow to build a neural net of Pepes. We show our images on a site created by using Ruby on Rails.

Challenges we ran into

Our first classifier ran slowly, so we had to make another classifier that worked quicker and fed its results to our first classifier.

Accomplishments that we're proud of

Creation of a relatively sophisticated neural net that was trained in < 20 min

What we learned

What's next for Pepe Rank

Deploying using AWS and using our domain peperank.com
