How it works 

 ChalkChalk is a doodling web app that can doodle on a blackboard or on your own uploaded images! ChalkChalk uses a super cute Pikachu mouse and you can choose different colors to doodle! Let's be creative!

Challenges I ran into, button interaction 

 Hackathon has been very challenging for us as programming beginners. We taught ourselves JavaScript, learnt CSS and Html in Hack Week, and learnt how to use everything with google.

Accomplishments

We are very proud of uploading file without using servers and making the doodling really costumized!

What I learned

I have learned a lot during the Hackathon. We have not only learned the basic structures and functions of Javascript, CSS, and HTML, but also learned to lookup online sources to help us find useful func.tions
