Inspiration

A Reminder for those who are struggling with Cancer that they are not alone. In a fashionable portable manner.

What it does

A portable purse that blinks every time new tweets are received showing support for cancer patients.

How I built it

The purse was confectioned by a local leather artist. The hardware was build using Arduino Flora, BLE and customizable LED's. Push notifications sent using node web app and with the help of the Twitter API and mobile App using Swift.
