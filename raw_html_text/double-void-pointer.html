Inspiration

I made a pixelizer last semester. This is bigger and better.

What it does

Made with JavaFX, it can use nine iterations of the Effect interface. The filters open a new window which views the image dynamically as the sliding values change. Also included are the option to place a random cat sticker at a random point on the image, and a "blend" tool which can combine two filters and apply them. Our project is capable of loading any jpg or png off of the computer, and can save edited images as png anywhere as well. If development continued, we would look into facial recognition software, as well as the filters available in openCV.
