What it does

The main purpose of the app is to get help from people who know what they are doing during emergencies. During an emergency, you can ask for help, connect to a doctor, a lifesaver, or a person with a license to do first aid. For example, let's say you saw someone unconscious on the floor. You can use the app to call for help from around and ask for immediate attention.

How I built it

Using React Native, Google cloud services, MongoDB

Challenges I ran into

The algorithm for finding the closest people that can help and making communication in between those people was a real challange.

Accomplishments that I'm proud of

The mobile application is built with React-Native, which is ready for cross-platform deployment. We deployed the server in the Google Cloud Platform virtual machines, with a MongoDB.
