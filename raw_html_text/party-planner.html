Inspiration

Our inspiration for this project was to explore the scope of the Walmart API in a fun and resourceful way.

What it does

This helps you plan a party in an effortless manner! Specify the type of party and the theme (or have no theme at all)!

How I built it

We built this app using the Walmart API along with angular.js, node, and HTML/CSS.
