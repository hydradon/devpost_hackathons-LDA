Inspiration

I've worked for most of the top 10 corporate travel agencies in Brazil. At every single one, the main source of fraud, under-billing and friction with clients was the emergency hotlines.

What it does

This is a SaaS style application for incident management.

How I built it

I used Python, Django and the Django-Jet app.

Challenges I ran into

The Sabre SOAP API doesn't work out of the box with popular Python SOAP clients. I've ended up using low level HTTP requests.

Accomplishments that I'm proud of

I've finished several hours before the deadline.

What I learned

I hate SOAP.
