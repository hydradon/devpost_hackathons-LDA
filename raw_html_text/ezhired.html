Inspiration

When job hunting season gets busy it can get difficult to keep track of all of your deadlines. "ezHired" make it easy to keep all of your job application needs in one place. Never miss a deadline again!

What it does

Allows users to organize companies and positions they are interested in by their stage in the application process.

How we built it

Built using React in the front-end and Meteor.js in the back-end.

Challenges we ran into

This was our first time working with Meteor.js which as it turns out, is very difficult to debug.

Accomplishments that we're proud of

Working hard and diligently to create a UI we're proud of.

What we learned

How create a web application using React and Meteor.js.

What's next for ezHired

Adding additional features and replacing Meteor.js with either Firebase or Node.js.
