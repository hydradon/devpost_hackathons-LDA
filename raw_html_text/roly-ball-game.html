Inspiration

It was both of our first time attending a hackathon. Neither of us actually knew how to code, but as our understanding for the topic grew, we decided to attempt to make an game/app with unity.

What it does

The game is designed with the white sphere as the player. The arrows dictate the movement of this player. The goal of this game is to move the player in such a way that the sphere collects all of the small, rotating yellow coins.

How we built it

We used the Unity tutorials and we developed and modified it to our own custom style and version.

Challenges we ran into

transferring the code from one computer to another making the ramps line up running the scripts without errors adding a color to the game objects

Accomplishments that we're proud of

Making our first app/game

What we learned

learning how to code and use Unity to make a basic game

What's next for Roly-Ball Game
