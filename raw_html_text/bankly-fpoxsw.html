Inspiration

  • People dont pay attention to how they spend their money. If we can help them allocate their money such as on groceries, clothing & electronic appliances. We aim to help people manage their money better so that they can utilize that to better their life.

What it does

  • Our application tracks users expenses.
  • Provides users analysis based on their spending habits.
  • Checks their recent purchases / statements
  • Scans products using the UPC Barcode scanning library.

How we built it

We built it using Android Studio. We had use retrofit for the api's over at the Nesse's Documentation. We also had to create an simple api that would be able to identify the product based on the UPC code.

Challenges we ran into

The Nesse API figuring out how payments and deposits work.

Accomplishments that we're proud of

We are proud of being able to incorporate the nesse api with the custom api that we built to send the price of the product to purchase using the nesse api

What we learned

We learned more about the nesse api and it's capability.

What's next for Bankly

We are hoping to modify the total expenditure by the user into multiple categories (Food, Bills, Etc) to help provide our users accurate data on their money expenditure.
