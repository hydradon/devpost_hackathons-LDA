Inspiration

Witnessing the alarmingly large levels of depression and anxiety in our schools, we decided to build an app that focuses on easy and quick methods to help with mental health.

What it does

It provides important information and resources to struggling individuals and expands the knowledge of the user. It also allows users to chat and interact with each other, finding ways to unwind and therefore increasing their chances of a quick recovery.

How we built it

We used HTML and CSS to build the website on glitch.com.

Challenges we ran into

Neither of us had any coding experience, and we spent the first 6 hours on a project we decided not to pursue, therefore wasting a lot of time. Also, initially, the coding domains we tried did not work on either of our laptops.

Accomplishments that we're proud of

We completed a website, which neither have us have done or knew how to do before this hackathon.

What we learned

. We learned a lot about HTML and CSS, including how to use dropdown menus, how to change the color, fonts, and size of pages, and how to create linked buttons. We also learned how to create an interactive chat box (scroll down to bottom of page to see).

What's next for Exemption

We would like to officially publish and spread the word about our website with hopes of improving other's lives.
