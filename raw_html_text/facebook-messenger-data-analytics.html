Inspiration

We were interested in looking at statistics of our facebook data and wanted to present the results in a meaningful manner. We also wanted to spread awareness about the mass amount of information social media websites have on their users.

How we built it

First we had to congregate the data from facebook in order to process it with our script. We used python to scrape the data from the 500+ json files to a single csv file. We then imported that data into the MicroStrategy desktop app to process and visualize the data.

Accomplishments that we're proud of

Creating a Python script to scrape and process the data into a usable .csv format.

What we learned

Powerful python scripting commands. Creating HyperIntelligence cards and data analytics using the MicroStrategy desktop app.

What's next for Facebook Data Analytics

Application for multiple social media, development of a web app to allow for users to test their own information.
