Inspiration

Keeping track of an event such has McHacks on multiple social media can a hassle so we decided to combine all the major social media website into a single page for a better overview of the event. We are meme lords.

What it does

The user send a search term and the server fetches posts from Facebook, Instagram and Twitter

How we built it

We build our application using Node.js and Express.

Challenges we ran into

One of our goal was to retrieve public post from Facebook. Unfortunately, Facebook's API does not allow searches to their public feed API due to privacy reasons. This forced us to use other search criteria for Facebook so we decided to go with Pages.

Accomplishments that we're proud of

We barely have any experience with Node.js but we still managed to complete our project.

What we learned

Node.js, API calls, Github command line, Bootstrap

What's next for Triforce

Better result displays and loading screen while the results are loading.
