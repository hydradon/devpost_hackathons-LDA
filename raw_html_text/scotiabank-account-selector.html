Inspiration

Considering that this Hackathon's theme is "An idea that can be banked on", we decided to focus on a topic that will help build financial literacy to the general public. Sometimes, it's difficult for a young adult or a student to choose the right bank account with the confusing terms used in websites. We wanted to make a user friendly interface which lets the user input their financial criteria and output a suitable banking account to fit their needs.

What it does

An online site prototype built for Scotiabank. This will be an official research tool for potential customers. It is an online helper for potential customers to identify which type of services they prefer Quiz type: A customer can answer the questions and the site will match them with the possible services.

How I built it

Used python to build interactive user interface. Html was used as a visual to make it even more simple and user friendly.

Challenges I ran into
