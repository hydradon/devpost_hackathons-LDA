Inspiration

The knowledge of touchscreen interfaces with a java software implementation.

What it does

Provides processes for the user by room within their house.

There are currently three programmed rooms; kitchen, bedroom, and bathroom.

The following is a list of the rooms and their menu abilities: 

 - Kitchen : Timer for Cooking
    - Bedroom : Alarm Clock
 - Bathroom : Fun Fact Slideshow

How I built it

Software : Java JDK and custom Unix configuration file. Hardware : Raspberry Pi, 2x 3.7V Lithium Ion batteries, and 10.1in. Waveshare touchscreen.

Challenges I ran into

Developing a completely embedded file system and wireless physical interface.

Accomplishments that I'm proud of

Providing a system that does not rely on internet, nor does it require a wired power supply.

What I learned

Embedded file management, and battery power implementation.

What's next for Home Entertainment Automation

The addition of a gps locator that interfaces with the Raspberry Pi will allow for a geographical map to indicate when a user moves to another room; such that, the system can automatically transition menus to the new room in which the end user has entered.
