Inspiration

Sami and hotels, and Mihai and hotels, and Whitbread challenge.

What it does

You can check in hotel without interacting with human being. You can do everything by speaking with Amazon Echo. Also you do not have to call or go downstairs to order room service. You can do it using text messages.

How we built it

We used Amazon Echo and AWS lambda to host Alexa skill. We also used Twilio API to handle text messages.

Challenges we ran into

Asynchronous JS and setup of Amazon Echo

Accomplishments that we're proud of

We finished and it works!

What's next for Room service

Add more available options and interactions, connect everything to hotel booking backend.
