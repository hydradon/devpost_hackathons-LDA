For too long transit agencies have relied on mediocre yet incredibly expensive real-time information systems. Now is the time change that. Transitime provides a fully functional information system that can use any source of GPS location information, it an be used for very large systems, and can be used by small shuttle systems all the way up to huge transit agencies in developing countries.

For more information on the project see the transitime.org website, the live maps for VTA buses and VTA streetcars , and the code on the Transitime GitHub page.
