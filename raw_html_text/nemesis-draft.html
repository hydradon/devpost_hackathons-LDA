Inspiration

We've been playing League of Legends for a very long time, and we wanted to make an app to assist us in making better choices in game.

What it does

Nemesis asks the user for their in game name. It then uses their personalized information, as well as information scraped from the internet, to compile a list of champions which have the highest win rate in their current situation.

How we built it

We used Riot Games API to gather specific champion and user information, and we created a web crawler to get data on champion counterplay and synergy.

Challenges we ran into

Our largest issue was figuring out how to data mine for the information necessary for our algorithm to compute the optimal solution.

Accomplishments that we're proud of

We made a custom algorithm based on a large amount of data and input. Crafted a staff out of stickers and Red Bull. Easily 14 cans tall.

What we learned

We learned how to make a web page crawler.

What's next for Nemesis Draft

Variants for different game modes. Tips and strategy for champion recommendations. Suggested item builds. Customize suggestions based a team role.
