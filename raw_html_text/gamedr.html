Inspiration

Games, Internet, Mobile Apps

What it does

Allows gamers to choose a game and match with players around the world, get put in a chatroom together, and make friends.

How we built it

We use Android Studio to build the app and Azure API to support the communication between players.

Challenges we ran into

The chat pushing notifications from Azure to all the players in the same chatroom was the biggest challenge.

Accomplishments that we're proud of

We got the basic functionalities of the app to work on Android and we successfully store user data onto the cloud by using Azure API.

What we learned

We learned how to build a local database on Android, how to use Azure API, and how to run Azure server scripts.

What's next for Gamedr

Finishing our app and publish to the app store.
