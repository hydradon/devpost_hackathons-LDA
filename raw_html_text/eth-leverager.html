Inspiration

By using MakerDao, we can gain extra leverage on our Ethereum.

What it does

This tool opens up a CDP, locks in ETH, pulls out a set amount of Dai, then buys more ETH. This ETH is then used to populate another CDP, from which more Dai is pulled. The depth of this process is specified by the user.
