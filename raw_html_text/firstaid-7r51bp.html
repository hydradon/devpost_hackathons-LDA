Inspiration

We struggle in school, and spending hours Googling just makes the struggle harder. We wanted to make an app which would alleviate the pain of Googling for the perfect resource, and instead would return the most relevant information from a user's query.

What it does

It is a minimalistic web app which provides the student with the most relevant results from their query.

How we built it

We built the web app using Python and the Flask micro web-framework.

Challenges we ran into

Google blocked our API requests due to us requesting far too much.

Accomplishments that we're proud of

We are proud that we overcame some immense hurdles and got through this rollercoaster of a night with a functional product.

What we learned

We learned that obstacles are everywhere, and that one must learn to accept such things.

What's next for FirstAid

We plan to add more resources, improve the AI, and upgrade the overall UI and UX.
