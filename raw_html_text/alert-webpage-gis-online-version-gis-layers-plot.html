Inspiration

Deliver a real time hydrology data to the public with GIS layers on top of base maps.

What it does

It is a part of flood warning web page to the public in the county. This web page give a real time hydrology data to aware of a potential flood event in the Santa Clara County. It gives following information. real time hydrology data; rainfall, stream flows and current reservoir status NOAA's forecasted rainfall data District GIS layers including Lidar contour lines
