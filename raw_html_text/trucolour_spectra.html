Inspiration

It's really hard to find a good foundation or concealer match for my skin tone. Sephora has a face iq, but Sephora products are expensive and does not include drug store products. I want to be able to find products that match my skin tone and be able to compare for the best online prices.

What it does

App that will allow you to find your perfect shade every time. TruColour will also allow you to search for products based on your skin tone.

How I built it

Android Studio Java Gradle Git & GitHub Google Vision API Canva LunaPic Marvel App Google Search Stackoverflow

Challenges I ran into

Getting Google Vision API to work.

Accomplishments that I'm proud of

Being able to connect Google Vision API on our app.

What I learned

Have fun and build!

What's next for TruColour_Spectra

Use Semantics3 API to search for products across different site such as Walgreens, CVS, Sephora, Target, Ulta, etc. Improve Google Cloud Vision parsing to match against pantone colors Search based on the pantone colors for better accuracy Expand search to all cosmetic products Chatbot to help customers find their products
