Inspiration

I travel a lot and always want to online map that records where I've already visited. And a map that I can easy add data and link some pictures that are the highlights in my trip.

What it does

You can use different color to hightlight any countries or cities on this map to record where you have visited. And a share button will create a link to a webpage that shows your map. You can share that link on any social media and your friends can see the customized map to know where you have been to!

How I built it

Using React.js with datamaps powered by D3.js

What's next for iTraveller

I would like to continue working on this project to add the features that I planned to add. For example, creating a link to share this map is not done yet.
