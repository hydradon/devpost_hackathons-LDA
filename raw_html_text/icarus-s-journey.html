Inspiration

The Greek myth of Icarus and Daedalus, in which Icarus flies too close to the sun.

What it does

It's a game where you play as Icarus, trying to keep flying and to avoid obstacles.

How I built it

I used Pygame to make this game.

Challenges I ran into

I wasn't able to find the proper images so I had to make them in MS Paint, which isn't the most convenient or visually-appealing option.

Accomplishments that I'm proud of

I learned a lot of new things and created a program that I didn't think I would be able to before today.

What I learned

I learned many new things from this project, like how to create start screens for games. I gained a deeper understanding of Pygame.

What's next for Icarus's Journey

I'd like to polish some of the rougher details that stick out, like the end screen.
