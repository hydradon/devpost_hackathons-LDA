Inspiration

We wanted to build a magic/humor hack for inflatable dino costumes! Wolfram seemed to offer some cool options for image recognition and analysis, and this gave an opportunity for some fun MS Paint skill flexing.

What it does

We did not get to finish, but it would ideally give each person in a group picture a nickname with a pronounceable bigram random word generator, and give them a wizard persona based on their perceived emotional state.

How we built it

We used a domain from Domain.com to build the site, and did most of the program with Wolfram One.

Challenges we ran into

We struggled with learning the ropes of Wolfram One, and as such couldn't put together the project in its entirety.

Accomplishments that we're proud of

Worked all the way through the hackathon, came up with an idea, and stuck with a team.

What we learned

We learned that sleep is actually pretty important for human function, as well as good old dihydrogen monoxide. Furthermore, we'll certainly look into APIs that we are more likely to be able to use effectively next time.

What's next for Mystical Chum Analyzer

Perhaps the project will find another life eventually, but for now it remains as a collection of wacky art assets and an oddball domain name.
