Inspiration

Online shopping is rough. Confusing sizes, hard to locate size charts and varying measurements from different brands are just the surface of troubles people run into when shopping online or when they have no time to waste trying on clothes. Size-Me is an application that reduces those pains.

What it does

The application takes user inputs on 3 simple measurements and is able to generate their clothing size from a variety of stores.

How we built it

We built the application as a website using html, css and javascript.

Challenges we ran into

Having little to no experience with web design.

Accomplishments that we're proud of

We almost finished.

What we learned

How to efficiently record data, how to manage time, how to html a bit, javascript
