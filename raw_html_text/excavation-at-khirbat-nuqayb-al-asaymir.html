Inspiration

All the excavations found at KAH are exciting and we want to demonstrate all of them through VR.

What it does

Visualization of all the data through the excavation of KAH such as photos of artifacts and models of ancient sites.

How I built it

Using Unity3D.

Challenges I ran into

Finding an intuitive and easy-to-understand user interface to display all the information.

Accomplishments that I'm proud of

Building suh educational application in such a short time.

What I learned

Learned a lot about Khirbat Nuqayb al-Asaymir. The process of archaeological excavation.

What's next for Excavation at Khirbat Nuqayb al­-Asaymir

Extend to more historic sites. Model those artifacts so that user can see them in details. Scale this project to a more general application so that anyone is able to visualize their excavation data in VR with less effort.
