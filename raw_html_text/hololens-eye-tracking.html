Inspiration

Using HoloLens is really cool, but the limited field of view provides challenges for the gaze based system. Eye tracking would allow users to select things without shifting their entire head.

What it does

Tracks the users eyes and reports the result back to the display

How we built it

Using Unity, a raspberry pi, a pi camera, a fork and some electrical tape to mount it

Challenges we ran into

Using the HoloLens emulator and development suite was very challenging

Accomplishments that we're proud of

Overcoming early challenges and bringing together hardware and software

What we learned

It's hard to use new technology that doesn't have a lot of documentation and resources
