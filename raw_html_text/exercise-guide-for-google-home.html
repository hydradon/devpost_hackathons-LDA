Inspiration

We wanted to do a simple exercise app that would guide users through workouts that they can easily do at home.

What it does

Using the Google Assistant app (or ideally Google Home), it will ask users for a day to focus on, push, pull, or legs, and will give exercises based on the chosen day. Then, it will tell them reps, sets, and rest times for each exercise.

How we built it

With great effort (in learning Node JS and different APIs)

Challenges we ran into

Learning the API for Dialogflow and Google Cloud (completely new for us) Trying to communicate with our Cloud Datastore database. Trying to find a supported timer Trying to connect Google Home Mini to a wifi network Setting up and downloading libraries to deploy our program

Accomplishments that we're proud of

Finishing with a functional product

What we learned

Node.js, connecting Google Home commands to a backend through Google Cloud Services, using Google Cloud Datastore

What's next for Exercise Guide for Google Assistant

-Ideally creating an app or website so users can upload their own exercise regimes -Being able to orally communicate to Google Home to make your own exercise regime -Implementing difficulties for the programs
