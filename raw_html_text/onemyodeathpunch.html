What it does

This is a connector script for the Myo Armband enabling control of the game OneFingerDeathPunch using just two arm movements.

This project was primarily used as an excuse to learn about programming the Myo and scripting in lua, and I don't really intend for it to be released until I get a chance to bugfix it.
