Inspiration

One day, when all five of us were playing our own songs aloud simultaneously, we realized that we were falling down the path to insanity. We needed a better solution. So we built one, and thus, JamPacked was born!

What it does

JamPacked utilizes personalized playlists from Google's Music API to create a public playlist catered to the needs of all parties. This way, everyone can enjoy their own music despite the differing genres.

How we built it

We made a web-application (ease of access by all parties) with Javascript, HTML, & CSS. Apache, PHP, SQL, and Linode was used to establish the server, and Matlab and Java for data analytics and AI.

Challenges we ran into

Incorporating Google's API, connecting the HTTPS server for GET & POST, getting the domain name, and converting the playlist files into playable playlists.

Accomplishments that we're proud of

The fact that we were able to take an idea and actually make it happen (for the most part) is a huge accomplishment for us. More specifically, making a secure website within 36 hours.

What we learned

We learned more about networking, server data, shell, and web-development.

What's next for JamPacked - Your Personal Party DJ

Better automation and incorporate a real-time ranking system for playlists. We would also like to use QR codes to simplify the user-experience.
