Inspiration

We had so much fun exploring the Microsoft Cognitive Services, we actually decided to create a game out of it!

What it does

Collect weapons and defenses, activate them by a specific emotion to beat your opponent!

How we built it

We created a custom C++ game engine and connected through Node.js to the Microsoft Emotion API.

Accomplishments that we're proud of

Pretty hard to connect Node.js and C++ (we ended up using a custom REST service and the Microsoft C++ REST SDK).

What we learned

Definitely the Microsoft API, lots of C++ and everything around it.

What's next for Face.off

Lots of fun for everybody ;-)
