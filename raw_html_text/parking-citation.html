Inspiration

We wanted an informative application that would be useful to drivers of Houston, be they commuters or passer-bys.

What it does

The application shows people indepth information about parking citations in the neighborhoods of Houston.

How we built it

We built this using JavaScript and the Google Maps API.

Challenges we ran into

Making sure that our dataset was accurate and reflected the parking citation situation as clearly as possible. Challenges included data that was not always consistent.

Accomplishments that we're proud of

We were able to accomplish every target goal.

What we learned

We walk away from this Hackathon as better developers with a stronger knowledge of dataset interation, as well as query modification.

What's next for Parking Citation

There are lots of additional features that can be added to the project, including expanding datasets. We hope to see this project continue to develop and grow.
