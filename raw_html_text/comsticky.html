• C_sticky

Inspiration - _italics _ A layman finds it difficult to read a source code, in order to make a layman understand what kind of code is it all about , we usually write comments under neath it or in front, but it's still messy.

Hence with respect to code the comment is written in the right with a label.

What it does - bold comments in the code are separated by a division.

Challenges we ran into - we were trying to make it more interactive for the developer and the code reader, but for that we have to develop the whole IDE.

What's next for ComSticky - We are designing many different kind of sticky notes, so instead of the label, a sticky note can take its place. This would look more appealing to the developer and software engineer.
