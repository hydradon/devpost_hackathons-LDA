Inspiration

I couldn't really find an app similar to it. Me and my friends want to be able to quickly create and join our gatherings and parties.

What it does

Add an event, and its location and details are shown on a map of Gainesville.

How I built it

I have a Raspberry Pi at home that runs the php and mysql server which holds all event data. The Android application allows users to interactively post new ones.

Challenges I ran into

I had no experience of this type: implementing a server and interacting with it, incorporating the Google Maps API, and attempting to adhere to material design standards.

Accomplishments that I'm proud of

It mostly works.

What I learned

Overcoming the challenges above.

What's next for Eventr

User profiles.
