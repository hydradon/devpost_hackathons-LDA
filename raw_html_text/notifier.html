Inspiration

We are dependent on the Internet. Our work, welfare and lives are directly connected to the internet. However, sometimes it happens that we have no access to it when it is necessary. My project is aimed to solve this problem. My inspiration was to create a project that would make us less dependent on the internet, and also fast way to work with mailing services without Internet connection.

What it does

This project sends notifications about new messages on the Gmail via SMS.

How I built it

I built it using primary PHP. I also used Nexmo API for sending SMS messages and Google API for the access to users' inbox.

What I learned

I learned how to use Nexmo and Google API's as it was the first time when I used them.

What's next for Notifier

There is still many things to do. I plan to make it less dependent on the internet, which means removing necessity to remain main web page open. I also plan to add integration with Twitter and features, like sending messages directly via SMS.
