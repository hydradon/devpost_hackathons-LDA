Inspiration

We wanted a platform where friends can share deep stories in a virtual camp fire kind of setting. I love telling ghost stories, having deep conversations and watching videos (whats better than youtube?).

What it does

Create your own voice and video room (public or private) and have a lot of people join! Video is streamed using webrtc and there is countless limits on this platforms.

How we built it

We built this using webrtc, react native, nodejs, php, html, and youtube apis.

Challenges we ran into

We wanted to implement a 'Pass the stick.' feature where someone has priority to talk and others are muted. Webrtc was sometimes a challenge to work with and we had to check browser compatibility.

Accomplishments that we're proud of

Getting a voice and video chat platform on the web where others can text chat and watch youtube videos! Whats better than watching youtube with friends? People can create private rooms and these chats can be used for countless purposes like venting or just having fun.

What we learned

We learned how webrtc changes the way video chats are displayed in web browsers. Also we learned how to create an event driven chat app.

What's next for FireSide Interactive Chat

We plan to continue this and add features for users such as custom events in the chatroom, organization participation, community and social change involvement.

Also we plan to add fun displays for video chats such as colors for usernames, custom power ups such as custom commands in the chats.
