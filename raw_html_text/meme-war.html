Inspiration

There are so many memes and internet jokes out these days that everyone has their go-to favorites. Being able to create your personalized memes and go head to head with your friends sounds like a great way to heat up friendships.

What it does

create tournament style knockout rounds and uses the players to vote on whos personalized meme is the funniest.

How I built it

Using javascript react and node js we created a web that where users can join or create a party session and upload their memes.

Challenges I ran into

We were all novice hackers so everything was a challenge. Learning to use git as a team was a huge challenge. Connecting frontend to backend was also a huge challenge.

Accomplishments that I'm proud of

everyone in our team cam out knowing way more then they did Friday morning. We were also proud of the fact that in the end we had an idea and were making stable progress.

What I learned

How to organize a team and use git effectively. how to use libraries along with react.

What's next for Meme War

Completing and seeing possibilities of mobile launch
