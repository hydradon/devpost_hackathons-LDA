Inspiration

In the tech industry, we are known for being the laziest. Thus, the best in finding shortcut to do things. Katalon Studio team is a group of test automation enthusiasts and we try to find the answer for this concern: "What if Testing Team can do everything in one place and doesn't have to jump back and forth between tools."

What it does

An add-on for JIRA so that business users can involve with development team. Project Managers or Business Analyst can compose requirements on JIRA under the BDD's format. These requirements will be synced with Katalon Studio (a free test automation tool) so that test script can be generated and associated accordingly.

Basic functions and features of this JIRA add-on are as following:

  • Compose business requirements with Gherkin syntax
  • Integrated with Katalon Studio to generate automation test scripts
  • Track development process for requirements
  • Automatically get execution results from Katalon Studio
  • Query issues based on test execution status
