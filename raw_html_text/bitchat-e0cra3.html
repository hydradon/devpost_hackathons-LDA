Inspiration

I wanted to send messages to fully encrypted and secure messages to my friends on the Blockstack platform.

What it does

BitChat provides secure and decentralized messaging for everyone with a Blockstack ID.

How I built it

I built it with React. I stored the users' messages using Gaia.

Challenges I ran into

Polling for new messages in an efficient way.

Accomplishments that I'm proud of

Reaching an optimal solution for fetching new messages for the users.

What I learned

How to maintain a socket connection between two or more clients.

What's next for BitChat

Adding groups and support for sending images and other media
