Inspiration

I jot really sick of filling out the same basic information into every job application so I wanted to write this.

What it does

Fills in basic information for greehouse.io and lever.co job applications.

How I built it

I used a variety of web technologies including jQuery and Materialize.css. If I did it again I would actually use a real build environment for it.

Challenges I ran into

I started dealing with some JavaScript/CSS libraries that required other build tools. I'm not much of a front end guy so I'm kind of lost when it comes to developing this kind of thing.

Accomplishments that I'm proud of

I was able to fill out some fields. I learned a bunch about WebExtension standards.

What I learned

JavaScript is stupid. Front-end development is a mess.

What's next for Auto Job Filler

I plan to get it working on more job sites and on more browsers.
