Inspiration

We were inspired by the Blueprint Learnathon. During the learnathon, a lot of new coders, like me, couldn't understand the complicated syntax associated with coding languages. We wanted to make it easier for new coders to learn to code, so we made a project which allows new coders to understand the concepts behind code without being familiarized with the syntax
