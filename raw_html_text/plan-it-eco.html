Inspiration

We want a better future for ourselves and the planet. It is important that the envrionment is clean for not only humans but other species to ensure a stable ecosystem for generations to come.

What it does

It connects the organizer and the users so that they can host events where people decide if they want to go or not. Incentives would be a potentially good feature to this project.

How we built it

We built this by using HTML, CSS and Javascript with some features of bootstrap and node.js. We connect the event to a database where it would be stored and shown when a user needs to see an event.

Challenges we ran into

We did not have enough broader coding experience, and knowledge of the language SQL so we decided to use JSON. With MySQL, we could have finished this project faster and more efficiently.

Accomplishments that we're proud of

Creating a functioning website with limited knowledge and without SQL, as well as with members with limited coding experience. We made due with what we were given, and learned a lot about what makes a website and its functionalities. We feel happy about finishing this project even if not fully.

What we learned

We learned a lot about how inefficient Javascript is. We also learned a bunch of HTML and CSS for future web development projects. We learned a lot about the coding environment and what it means to work under pressure, as a team, and towards a project that has its applications in real life.

What's next for Plan-It Eco

We plan extend this into an app and add a working account database with SQL this time, password encryption and user profiles. We want to add incentives and hopefully have mainstream organizers, also governments can sponsor certain events. We will update the event list and make sure that we update the CSS and look of our web. We have a lot of things to add, but we look forward to making Plan-it Eco something better and friendlier for everyone and marketable.
