AskAPI is a natural language query engine for APIs.

There's a huge amount of highly useful, real-time business information that's available from internal APIs and widely used web services, but it's not easily accessible to business users.

AskAPI makes it possible for non-technical users to ask natural language questions, or use search terms, and get current business-related information from the relevant APIs. As an API-driven web service, AskAPI transforms search terms and simple questions into REST-based API calls that return the results a user is looking for.
