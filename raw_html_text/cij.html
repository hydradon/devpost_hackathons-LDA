Inspiration

We came up for this app idea because sometimes you are feeling down, and need a nice compliment or funny joke. Other times, you want a funny insult. This app is perfect for those instances.

What it does

First, input your name. Then choose the crying-laughing emoji if you want a random joke, the blushing emoji if you want a compliment, or the mad emoji if you want an insult. Then, you can hit the redo button for another, or go back if you want to change options.

How we built it

CIJ was built using swift on xcode. We used framework UIKit.

Challenges we ran into

Our main challenge was transferring data between view controllers.

Accomplishments that we're proud of

We are proud of creating an aesthetically-pleasing app, easy user-interface, smooth graphics, smooth transitions.

What's next for CIJ

More jokes, insults, and compliments, as well as other features.
