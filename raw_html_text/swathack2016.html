Inspiration

We wanted to figure out how to get people into gyms more - social gamification is our answer

What it does

Tells whether the user is within coordinates known to be a gym, and updates a leaderboard accordingly

How I built it

Used the Ionic2 web framework to create a web app, that exports to a native mobile app for android

Challenges I ran into

Enabling location services on Android, basic Ionic CSS

Accomplishments that I'm proud of

Learning a new, pretty powerful web framework

What's next for Gymify

Actually making it a project that works -> accounts, links, groups, and everything. Right now it's a frontend with a global controller for one user. We want to develop a backend, possibly in node.js or RoR. Also, prettying it up.
