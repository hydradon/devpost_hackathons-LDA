Inspiration

Manual processes on various QA, complex online user application form and submitting are bad experience for the applicants, and also bad performance for the whole verification and validation process.

What it does

For POC, a chatbot could be created to answer VISA application questions and help the applicants to apply for VISA by intuitive conversation instead of complex web application form. We also integrate MOVEit into our KYC process to comply with the regulations.

How I built it

  • Use OpenEdge database to store user information, KB
  • Create chatbot using NativeChat
  • Use Kinvey as chatbot backend to prepare JSON and invoke Corticon decision services for application validation and processing
  • Implement the rules in Corticon for application validation and processing
  • MOVEit for KYC docs submission and storing securely.
  • WUG for SLA management.
