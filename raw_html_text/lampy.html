Inspiration

We both believe that mental health is an important topic that does not get talked about enough nor has enough resources for people who may need help. Therefore, we developed a mobile application that allows anyone to keep track of their daily moods and talk to a virtual assistant. Our app allows people to get the help they need through many relevant organizations.

What it does

Lampy is a mood tracking iOS app. It asks the user daily how they felt that day. Moreover, our lampy provides relevant tips on how to manage your emotions while also giving users the access to mental health organizations to get the help they need.

How we built it

Lampy is built using Swift4 & Xcode.

Challenges we ran into

Not being super familiar with Swift. There were a few features we wanted to implement, but just didn't know how/have the time. For example, our profile page is incomplete, so we had to set a temporary name variable to get the app working.

Accomplishments that we're proud of

For two people that aren't familiar with Swift or Xcode, we got a lot done in 8 hours.

What we learned

We learned how to code in Swift and the basics of Xcode.

What's next for Lampy

We want to transform the tracker into a calendar that displays the user's mood for every day.
