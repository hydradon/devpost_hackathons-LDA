Inspiration

We were inspired by the food challenge to make a game on not being wasteful with food.

What it does

The game works by making us collect pieces of food and raising the score as food gets collected. You are given lives and when the food disappears after a time limit your lives disappear.

How we built it

Food Collector was built with processing that ran java code using various sprites, pieces of music and a background to make it look more appealing.

Challenges we ran into

The biggest challenge that we ran into was the timer mechanism. Our timer had to be built with a new class that used the sprites to individualize all of them.

Accomplishments that we're proud of

We're proud of the way it turned out, and the timer mechanism that proved so difficult in the beginning.

What we learned

We learned a lot on how sprite creation and game design works.

What's next for Food Collector

The next step for Food Collector is to properly relay the message that we want, which is to conserve food. While the game is functional, it is still quite incomplete, and could be ambitious for bigger food conservation goals.
