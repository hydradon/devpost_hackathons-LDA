Inspiration

It's always helpful to see how much data you're using, especially on limited connections. Also, your data can be disseminated over wide areas without your knowledge. This extension helps give you insight into where your data is going, and how much you're using.

What it does

Tracks your data usage across tabs using Chrome's logging and a proxy server to determine response size.

How I built it

Chrome extension and Go proxy server running on a Linode instance.

Challenges I ran into

Making the proxy server is pretty hard, actually.
