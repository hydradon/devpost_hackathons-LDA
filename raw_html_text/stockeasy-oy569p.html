Inspiration

Financial Data is fascinating how it changes instantly and how we can use computers to take advantage of these changes

What it does

Provide a basic interface to trade stocks

How I built it

Wolfram handles the backend while Javascript handles the frontend to provide an easy on the eyes interface that presents a few mathematical scripts to provide data analysis in helping you make the decision to buy or sell

Challenges I ran into

Learning the Wolfram language, and figuring out how to make REST endpoints work for a client-server connection to consistently update

Accomplishments that I'm proud of

Getting some things to work (Having a graph to display data, and an appealing website)

What I learned

How to use Wolfram and how to establish REST endpoints using Wolfram

What's next for StockEasy

Adding more features and data analysis to help users make decisions on whether or not to buy/sell stock
