What it does

Tweet any of the following:

'''@asg_027 @realDonaldTrump #haiku'''

*Tweets back a haiku made from tweets from the user mentioned (@realDonaldTrump)

'''@asg_027 @HillaryClinton #grade'''

*Tweets back the reading level "needed" for the person's tweets (based on the Gunning Fox Index)

'''@asg_027 @realDonaldTrump @HillaryClinton'''

*Tweets back a "roses are red/violets are blue" poem from tweets from the mentioned users/

How I built it

Uses Twitter API, REST and STREAM. Currently tying to host it on heroku.
