Inspiration

We wanted to build a game in Unity for the first time since we thought it would be interesting. So we built a VR game! We were inspired by other VR games and desktop games, such as keep talking and nobody explodes.

What it does

This game generates mazes randomly for a VR user and audience to explore. Beware! There are enemies about as you collect orbs to reach your objective. They know where you are, but you might not know where they are!

How we built it

We used Unity to build the app, along with Visual Studio and Git.

Challenges we ran into

Considering we had to learn unity for the first time ever, we had some trouble getting the ball rolling. We had many merge conflicts when working on the code and assets with our team. There were also issues with map generation, figuring out AI, and pretty much everything we wanted to do. We also ran into issues of sleep, so writing this is pretty hard.

Accomplishments that we're proud of

WE MADE A GAME IN VR! IT ACTUALLY WORKS!

What we learned

We learned many aspects of game development in Unity and C#. We also learned about game brainstorming and sticking together.
