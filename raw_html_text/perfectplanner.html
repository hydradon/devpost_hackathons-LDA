Inspiration

We wanted to make trip planning much less stressful and quicker.

What it does

By inputting the dates of your trip, locations of your trip, budget, and food preferences, PerfectPlanner plans the trip for you!

How we built it

We used the node.js as our back-end while utilizing the Amadeus and Yelp APIs to retrieve data regarding travel and display everything using handlebars.

Challenges we ran into

Trying to work with the asynchronous nature of Javascript gave us a few headaches when we were trying to utilize the APIs to retrieve data.

Accomplishments that we're proud of

We were able to successfully retrieve data using the APIs and process and display the travel information based on the preferences of the user.

What we learned

We learned how to effectively use Restful APIs and build a website using handlebars.

What's next for PerfectPlanner

We plan to allow the user to input more preferences, display a bit more options for the user to choose from, and suggest places of interest, hotels, ground transportation, and destination tips for the user's trip.
