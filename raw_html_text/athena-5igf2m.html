Inspiration

We wanted to provide a convesational platform that can help students learn about engineering topics and events around them.

What it does

Athena is a Facebook Messenger Chatbot, that allows to find out what events are around you that you might be interested in and learn about Computer Science topics by asking directly as a message.

The user can ask, "show event around " to get insterting events for the user around it.

Also, The use can ask "I want to learn " to optain relevant courses to that topic where he can expand his knowledge.

How I built it

We developed our server in Flask and deployed it to a Heroku instance. We use the Eventbride and Udacity APIs to provide the relevant user information to the user.
