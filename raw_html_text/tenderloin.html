• User interface for safest path to get to the zoo from home

  • Visualization of crime densities in San Francisco

Both of us have been to San Francisco multiple times in the past few years. While there are many nice areas, San Francisco contains a number of neighbourhoods which you would likely want to avoid. This idea inspired us to create "Tenderloin". Tenderloin is an application which helps you pick the safest walking path between two destinations. It makes the decision based on existing crime data and aims to avoid areas with a high density of crimes. Our application was built using react and node.js while the data processing was done in python. It also leverage the Google Maps API for options of different possible routes.

Data

We used the San Francisco Crime Classification data ( link ) to populate our heat-map.
