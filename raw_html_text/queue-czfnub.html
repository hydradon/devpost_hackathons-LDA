Inspiration

We attended a friend's birthday party which was originally intended to be hosted at BJ's restaurant. Because of the inefficient organization of queueing customers, we weren't able to eat. We had to change plans at the last minute.

What it does

Assists with restaurant reservations by managing customer lists and allowing users to digitally reserve tables.

How I built it

We used UnityEngine and ported it to iOS.

Challenges I ran into

Several of our members lacked technical experience, so the majority of our time was dedicated to learning

Accomplishments that I'm proud of

We learned something new.

What I learned

We learned how to use implement priority queues and design basic front end.

What's next for Queue

We want to make Queue scalable so that any restaurant in any location can take advantage of this service. We also want to make improvements to front end design.
