Inspiration

I always wanted to add extra information to issues. They are only relevant to me so I can't add them as a comment. I also need a mechanism to search, sort this notes. For example before a meeting I want to take notes on the issues and pass over them in the meeting.

What it does

It allows you to add private notes to issues. You can organize notes with different colors. You can see list of your notes or filter them.

How I built it

I build it with Atlassian Connect using EC2 and Firebase.

Challenges I ran into

Security domains created by iFrames makes it difficult.

Accomplishments that I'm proud of

It has a nice UI. It is fast and really helpful tool.

What I learned

I have learned a lot about atlassian connect, firebase.

What's next for Jira Note Taker

I have some ideas but I don't want this thing to get complex so I'm not thinking of adding lots of new features, but I will listen user's feedbacks and proceed accordingly.
