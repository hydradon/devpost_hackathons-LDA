Para proyecto de Hackathon MTY 2019, decidimos aceptar el reto de una empresa que su problemática era tomar un set de datos con compras, ventas y movimientos de empresas junto con información de los vendedores y compradores. Nosotros diseñamos una aplicación web que te muestra un resumen de las compras, ingresos egresos en varias gráficas, y también permite llevar un registro de próximos movimientos

Este proyecto se creó usando Node Js, React, y para los datos MongoDB, nuestra aplicación web es una web page application lo que hace que su tiempo de respuesta sea muy rápido
