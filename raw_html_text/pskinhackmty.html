Pskin

Inspiration

Nos inspiramos en ayudar a la gente, conectándolos y compartiendo sus experiencias con la comunidad, haciéndolos sentir en familia y apoyados.

What it does

El usuario podrá tomar una foto de un problema que éste tenga en la piel, la aplicación le dará una predicción del
problema que éste tenga y podrá compartirlo con los demás usuarios.

How we built it

Se utilizó el IDE Visual Studio con Xamarin Forms, una base de datos en MySQL, una API en ASP.NET y el servicio de Custom Visión

Challenges we ran into

Hacer una interfaz y un diseño amigable con una buena experiencia de usuario.

Accomplishments that we're proud of

Será una ayuda extra para las persona que no cuentan con los recursos para poder realizarse un diagnostico en una clínica.

What we learned

Aprendimos como crear APIs, como consumirlas e integrarlas a una aplicación móvil, así como también aprender a trabajar bajo presión.

What's next for Pskin

Se pretende que la aplicación sea escalable y conecte a diversos usuarios por el mundo.
