What it does

It takes a picture of you and writes a Haiku about you

How we built it

Amazon Alexa speaks to our Lambda function which hits webserver1 and asks it to tell the browser that's pointed at it to take a picture with the webcam. We then send this picture off to clarifai for processing. We then take the keywords and send them to webserver2 which uses nlp to build a haiku. The final message is then sent back to the echo to be spoken.

Challenges we ran into

It's extremely difficult to get a computer to finish a thought in a set number of syllables
