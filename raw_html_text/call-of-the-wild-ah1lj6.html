Inspiration

On campus, we have a huge problem with groups of people hoarding bird scooters inside their homes. The public can't access them, and chargers cant charge them

Call of the wild, sets out to change that by identifying hoarders by using nudge theory to convince them to release the birds into the wild

What it does

Using gps data we find clusters of birds using the HERE.com API and attempts to make the birds chirp on command

How we built it

We used HERE.com api to make a custom map of the birds

Challenges we ran into

Finding birds in the wild to test Getting the bird to chirp on command, with more practice and bird treats we expect them to learn this skill

Accomplishments that we're proud of

We are proud if the impact this application will have in the real world

What we learned

How to use HERE.com API

What's next for Call of the wild

We hope to work with other companies to prevent this inhumane hoarding and #UncageTheBirds
