• Game Architecture

Inspiration

Twitch plays pokemon sparked this idea. We are a fan of old school text RPGs. We wanted to build a game that required community effort to complete.

What it does

Send commands in the twitch chat to interact with the game.

How we built it

Java chatbot connecting to Twitch IRC servers sending messages back to redis db. Ruby worker and game orchestrator are listening in a pub/sub channel sending commands to both the chat bot and socket io interface.

Challenges we ran into

Getting phaser.js to work. Twitch stream delay.

Accomplishments that we're proud of

Connecting to twitch IRC. Getting tiles to display in phaser. Connecting all the services together.

What we learned

Java, IRC, web sockets, and publish/subscribe architecture, and javascript </3.
