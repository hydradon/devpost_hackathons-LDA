Somos la Mesa #25

Inspiration

A la hora de cocinar es molesto estar checando una hoja de papel o la computadora para seguir una receta nueva, que mejor herramienta que un pebble en nuestra muñeca, para tener una vista paso a paso del proceso que estamos realizando.

What it does

PCook esta ideado como una aplicacion de movil desde la cual el usuario puede crear y guardar sus recetas favoritas en pasos sencillos, enviando la receta a pebble desde la aplicación se obtiene una vista paso a paso en el smartwatch

How I built it

Se realizo una sencilla aplicacion en android para poder crear y editar recetas, ademas de guardarlas, se intento ligar a una base de datos web, pensando en que las recetas pudieran ademas estar al alcance de todos los usuarios. Se trabajo en la cloudpebble para verificar que se mostrara paso a paso alguna receta el proyecto no se pudo concretar, aunque si se logro visualizar la receta en el pebble.

Challenges I ran into

Aprender C, JavaScript Manipular hardware desconocido por el equipo: Pebble

Accomplishments that I'm proud of

Si se logro manipular el hardware

What I learned

no es nada sencillo programar en un hardware desconocido y lograr algo de la noche a la mañana, el propio hardware tiene limitantes que aveces se desconocen y eso produce que el desarrollo se venga abajo y la curva de aprendizaje se prolongue.

What's next for PCook

nos gustaria poder terminar el proyecto, sin embargo no contamos con el hardware. Tambien nos gustaria trabajar con algun modelo distinto: como el round o el steel para añadir alguna funcionalidad de dictado a la aplicacion.
