Inspiration

When we went camping together, we would often not know what some camping materials are because we are not too familiar with them. We had trouble finding what spots to visit, so we had to keep going around asking people.

What it does

GoHelp scans an image and outputs news headlines about the product or about the events happening in the location.

How I built it

We built it using Xcode, JSON, Swift.

Challenges I ran into

It was hard to get the data that was in the picture into news that was related. We also had a hard time setting up json. We ran into many dead ends so we had to trace ourselves back a few steps and work our way around all issues that occurred.

Accomplishments that I'm proud of

We are proud that we have made an app that can help many people in the world when they go to explore the world.

What I learned

We learned that even though there are API's included there is still a lot of work that you need to do to get the app running the way you want it.

What's next for GoHelp

Next steps are:

  • create a faster algorithm to get news
  • put news in different categories
  • be able to favorite news
  • send news according to each individual's "most frequent searches" by using search engine optimization
