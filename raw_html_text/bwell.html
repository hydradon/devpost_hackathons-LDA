Inspiration

Seeing friends and closed ones struggling from diabetes inspired us to create a solution to have an ever growing problem. When we decided to delve into the issue, we discovered that 463 million adults were living with diabetes out of which over 140 million adults had some form of disability.

What it does

Our app provides users a more efficient way of monitoring their diet for diabetes. This is done through recognizing meals and snacks using Clarifai AI, as well as Google Cloud Speech API.

How we built it

We utilized the react-native framework to create a mobile IOS application. This app connects to NodeJs and a mySQL database.

Challenges we ran into

Getting the audio to work was a challenge because of the limited knowledge we had. However, we were able to pull all of our tech together before the deadline.

Accomplishments that we're proud of

We're proud of being able to get all of our translation technologies and APIs set up within the deadline that we had set out for ourselves originally.

What we learned

We learned about how to use various APIs such as Edamam, Amazon S3, Google Cloud API, etc.

What's next for bWell

Improve build quality for mass market.
