Inspiration

Learn blockchain and leverage the technology in hiring industry

What it does

Authenticates and calculates diversity ration for a employee to promote diverse pool of candidates

How We built it

Used Java Spring boot for the UI and JPA + Hibernate for Backend. Locally run Blockchain on private P2P network

Challenges We ran into

Setting up a local blockchain and diversity ratio calculations

Accomplishments that we are proud of

Implementing blockchain to streamline hiring process

What We learned

How important diversity is and how we should aim to promote it with technology

What's next for Unblock

Going public with the P2P network and signing up students and employees to use the system
