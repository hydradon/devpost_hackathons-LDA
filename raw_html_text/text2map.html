Inspiration

After traveling so often we realized that getting directions and travel information was impossible without a data plan. Hence, we decided to create an app that uses SMS to get directions to a specified location, nearby wifi hotspots or restaurants.

What it does

Using Twilio to connect the Android application to the server, you can still get directions to wherever you need to travel to without taking a hit on your data plan.

How We built it

Node.Js, Twilio, Google Maps API and Android
