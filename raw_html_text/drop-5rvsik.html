DropSolutions 

DropSolutions is a Chrome extension built in JavaScript, HTML, & CSS. The extension keeps track of the time spent on websites that are not wasting the user's time. These websites are input manually by the user. If the user enters a site that they are not supposed to, the timer stops and calculates how many gallons they accumulated to donate to an impoverished community. In our demo, it is set to 1 gallon per 5 seconds. Ideally, however, it should be a gallon an hour. 

This extension would ideally work best when it is partnered with a charity organization like UNICEF that will help us bring our idea to life. 

In the future, we plan to add a pop-up window that automatically alerts the user that they are entering a time-wasting website and that no further gallons will be donated. The user will be given the option to bypass the message or close the tab and continue being productive!
