Inspiration

The style was inspired by the mobile game "Duet" and the tendencies to multitask as an enriched student.

What it does

It challenges the player to use both hands to progress individually at the same time.

How we built it

We built it using python and pygame. We split up the tasks of making graphics and programming. The menu and game were then further delegated to different people.

Challenges we ran into

The challenges that we ran into were coming up with the idea of the game and polish the code up in the end.

Accomplishments that we're proud of

Completing the program within the time, coming here without any idea of an actual game and having it come all together in the end.

What we learned

We learned how to effectively work as a team utilizing everyone's strengths to create a final project

What's next for Through the Stars

What's next for Through the Stars is having more detail and higher quality graphics to enhance our game.
