BlockChainBanking

The inspiration behind BC banking is to implement the blockchain in a useful and modern manner. Its main purpose is to provide a secure, and reliable transaction system for the future. The reason that it is so secure is due to the immutability of the blockchain. For our project, we created our blockchain from the ground up using python, flask, and HTML. Throughout the creation of our project the main issues we faced we debugging issues. Problems would seemingly materialize from thin air and many hours had to be dedicated to the squashing of these bugs. A large problem that was also prevalent throughout the development of our project was version control, in order to function correctly all of our systems had to be on the same versions and this was not always the case.
