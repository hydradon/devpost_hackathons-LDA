Inspiration

Ensuring that a new generation of international travelers save money for budgeting and banking.

What it does

Moneybeee is a web application that helps people who are traveling abroad save money on unnecessary exchange fees.

How I built it

  • NodeJS Express web application
  • Hosted on a Google Compute Engine node
  • Firebase backend
  • Google Vision API

Challenges I ran into

Integrating all of the different components

Accomplishments that I'm proud of

Everyones teamwork and effort was amazing given that some it was the first Hackathon a lot of us.

What I learned

Each member of the team learning a lot while developing their specific component.

What's next for Moneybeee

Implementing Blockchain technology and incorporating Smart Contracts to ensure that users are not being ripped off.
