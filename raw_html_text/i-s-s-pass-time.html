Inspiration

Building a multimodal Amazon Alexa skill that uses the new Alexa Presentation Language (APL) is my inspiration for developing this project.

What it does

I.S.S. Pass Time presents the next and future pass times for the cities including Istanbul, Ankara, New York, Chicago, San Francisco, London, Paris, Rome, Berlin, Mexico City, Mumbai, Nairobi, Tokyo, and Seoul.

How I built it

Alexa Presentation Language (APL) is used to build this project and an API ( http://open-notify.org/Open-Notify-API/ISS-Pass-Times/ ) is used for getting international space station pass times information.

Challenges I ran into

Challenge I ran into is to learn Alexa Presentation Language (APL).

Accomplishments that I'm proud of

Accomplishments that I'm proud of are learning Alexa Presentation Language (APL) and developing a multimodal Amazon Alexa skill that uses the new Alexa Presentation Language (APL).

What I learned

I learnt using Alexa Presentation Language (APL) for developing multimodal Amazon Alexa skills.

What's next for I.S.S. Pass Time

I may add some other information about space like how-many-people-in-space or international space station current location to the project.
