Inspiration

Building web applications can be a major hurdle in the development of a small business, especially when none of the founders are experienced programmers. Our app alleviates this issue by allowing non technical users to drag and drop existing Microsoft Azure and Standard Library APIs, including computer vision, sentimental analysis, twilio, and Google Sheets.

How we built it

We used Python and Flask for hosting and for all scripts.

Challenges we ran into

Integrating all the individual APIs was more challenging than we anticipated.
