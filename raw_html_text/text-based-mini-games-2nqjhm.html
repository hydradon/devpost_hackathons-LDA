Inspiration

Love the look of Twilio's programmable text API.

What it does

Text-based text games including hangman and RPG game.

How we built it

Flask server on Heroku connected to Twillio with webhooks to deal with texts sent by the user.

Challenges we ran into

Connecting to Twilio.

Accomplishments that we're proud of

Two fully working games.

What we learned

Twilio's API and using webservers.

What's next for text-based-mini-games

More mini-games like battleships. More validation for stability.
