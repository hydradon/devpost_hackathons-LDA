Inspiration

Augmented Reality and difficulty of moving furniture

What it does

It displays 3D models of furniture over a certain design (in our demo this was a Fritos bag), in order to assist with planning and organizing moves. It also allows the models to be scaled in size.

How we built it

We built this in Unity, using Vuforia, and ran it on Android

Challenges we ran into

We had not used Unity before and as a result had to start from scratch learning it

Accomplishments that we're proud of

We produced a working app by the end

What we learned

We learned more about how to use Unity

What's next for FurnitureGO

We plan to implement additional models and increase the number of possible transformations
