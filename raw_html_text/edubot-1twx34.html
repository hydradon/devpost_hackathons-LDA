EduBot

Education Bot for students to use, helps keep track of progress and encourages anyone to learn something new.

basicbot_v1.9 - Able to run lessons & provide daily exercises for students.
To-do-list.py - Creates a to-do-list for student to plan what they want to do
PIL_test - Text to Image converter

What you will need

  1. Download basicbot_v1.9
  2. Download the CSV and images files
  3. Put them in the same folder
  4. Run basicbot_v1.9
