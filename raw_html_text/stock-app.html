Inspiration

We were inspired the stock market and nature.

What it does

Our intent was to provide a well documented summary of the stock market at any given moment.

How we built it

We used MySQL for our database, and Java as the background to fetch and store data. Then we used GraphQL to access the data from the front end. Our intent was to create a website using React as well.

Challenges we ran into

Our largest challenge was using GraphQL. Writing queries to return data types is very tricky in Javascript

Accomplishments that we're proud of

We ran multiple servers to test, but did not get any meaninful results.

What we learned

MySQL , and a decent of GraphQL. Server setup was also vital

What's next for Stock-App

Finish querying stock api's to gather data, finish front end, finish GraphQL queries
