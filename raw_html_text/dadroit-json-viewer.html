What is it

Dadroit is a brilliant tool that threats JSON as a Data format, not a plain text. It provides you quick outline view with the help of tree representation of JSON Data from root to last nodes. You can browse and query JSON like an enterprise DBMS. Integration with Blockstack help the developers to syn their info, settings, sharing them with team members in a safe and private manner.
