Inspiration

Didn't know what to make for dinner one day, didn't have a lot of money to spend.

What it does

Users are able to post recipes and have it appear in a feed.

How we built it

We built this application using Angular JS for the front end, and Java for the back end.

Challenges we ran into

Linking the back end to the front end was a huge problem that we ran into. Also inexperience in Angular.

Accomplishments that we're proud of

Creating something that we believe has long term scalability.

What we learned

More Angular, Linking wegmans api, working in a team environment.

What's next for brokemenu...v2!

Hopefully a lot more. We hope to integrate the Wegmans API fully in order to allow users to see if the groceries needed to create certain recipes are available at a Wegmans store near them, as well as integrating the price of the each ingredient and calculating it. We'd also like to improve the overall design of it.
