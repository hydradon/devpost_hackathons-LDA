Inspiration

Our group is a huge maze runner fan. (book and movie)

What it does

It's a maze in which you use a vr headset (example. google cardboard) to navigate the maze and WIN!

How we built it

  • we used unity to make the 3D aspect of the game possible
  • we coded in c#

Challenges we ran into

  • no mentor
  • new to developing world
  • never done VR developing
  • unity crashed several times

Accomplishments that we're proud of

  • learned c# in 12h
  • made something that ACTUALLY WORKS

What we learned

  • there are a lot of compatibility issues with different devices on android
  • a lot of time is required to fully optimize apps so that they don't crash

What's next for MazeRunner3D

  • better compatibility with other devices (android devices and VR headsets)
  • want to be able to run on HTC vive at a higher resolution
  • publish our apk file onto the google play store
