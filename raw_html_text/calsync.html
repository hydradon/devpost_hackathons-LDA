Inspiration

We desired to come up with a way to easily and efficiently add events to our calendar.

What it does

CalSync allows the user snap pictures of posters, flyers, and documents, and by extracting the dates and locations from it using optical character recognition, automatically schedules an event in your calendar, with your approval.

How we built it

We started with an Android application written in C# (with Xamarin), and later supported it with a Flask backend which does all the heavy lifting (OCR, test processing and such).

Challenges we ran into

We faced some difficulty getting used to C# and Android.

Accomplishments that we're proud of

Despite having to abandon an earlier idea, we managed to make substantial progress on CalSync and created a working app in under 24 hours.

What we learned

Utilizing optical character recognition tools, developing Android mobile applications, and setting up a web server using DigitalOcean.

What's next for CalSync

Incorporate event details using text recognition, add voice command, and scan documents with multiple dates.
