Inspiration

In Cadger, we tried to simulate the world where heavier objects fall faster than lighter ones (Toyota + Audi is heavier than a Hummer so it accelerates faster). The goal of this game is to get furthest down, and the cars faster than you can be a threat since a collision from the top can hurt your car. But there still is hope. A car above can absorb HP from the car below, which means you can heal yourself by destroying cars that are slower than you!

What it does

This is a game - as a player, you have to compete with other cars to survive and go down as far as you can.

How we built it

We used Adobe Photoshop CS6 for all the drawings and implement the game mechanism through Unity.

Challenges we ran into

We first tried to use Java FX, but due to the time limitation, we switched to Unity.

Accomplishments that we're proud of

Pretty pictures, smooth movements? The very fact that we made the decent game within the set time.

What we learned

.... that we can build the nice little game within 36 hours!

What's next for Cadger

There are more things that could be added on to the game such as bonus stages, bonus items, and more :)
