• CharityWorld

Inspiration

Généralement, ne connaissant ce que feront les organismes a but non lucratif de notre don, nous sommes des fois méfiant. Ainsi, nous avons pensé à développer une plateforme permettant d'augmenter la confiance des donateurs à travers la vue de l'impact de leur don.

What it does

Nous avons développer un système en temps réel pour informer un donateur de l'impact de ses dons. Afin de l'inciter à en faire plus.

How we built it

  • Angular
  • Waston IBM
  • AWS (Lambda function, API Gateway, Python, Mysql)

Challenges we ran into

  • Utiliser l'intelligence artificielle
  • Intégrer plusieurs services cloud.

What we learned

  • Développer un produit présentable en 26h.
  • Utiliser l'API Waston de IBM

What's next for CharityWorld

Les prochaines étape du projet CharityWorld sont :

  • Améliorer le design du site web.
  • Obtenir les sentiments des personnes sur les photos.
  • Confirmation des ententes et partenariats.
