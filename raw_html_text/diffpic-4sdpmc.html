Inspiration

We wanted a cool way to visualize changes in an image over its commit history.

What it does

Allows you to easily see the changes for a picture in your GitHub repo's commit history. You can see pixel changes, rotation, and resizing.

How we built it

It's almost vanilla Javascript.

Challenges we ran into

See "How we built it"

Accomplishments that we're proud of

UI is simple and it's easy to see the changes in an image.

What we learned

It's fun to make single page web apps.
