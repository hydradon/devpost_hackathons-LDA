What it does 

 Allows users to create a posts about an issue they care a lot about. Other uses may inspect these posts and click an "I Support" button which sends an email to their corresponding constituents about the issue for them.

How I built it 

 Front end is built on react.js . Back end is built on express.js with mongoDB.
