Inspiration

None of the members of our team had ever dipped our toes into AI, machine learning, or voice recognition. Since we were graciously provided voice credits for some very interesting APIs, we jumped at the challenge to build something ambitious.

What it does

Motiv.AI allows users to track habits and goals and provides motivational messages and positive reinforcement to keep them on track.

How I built it

We built Motiv.AI using VoiceFlow for the voice user interface, DialogFlow for natural language processing, MongoDB for storing user data, Python for back-end, and Google Cloud Functions for integrating everything together.

Challenges I ran into

Initially, extracting the intents using DialogFlow had a steep learning curve. Training the model to properly understand and extract meaningful user information was especially challenging.

Accomplishments that I'm proud of

We are proud of the fact that we were able to combine several unfamiliar services, and still managed to build something functional.

What I learned

We learned that artificial intelligence is far more artificial and way less intelligent than we thought it would be.

What's next for Motiv.AI

We would like to expand the range of goals and habits supported. We would also like to move some of the language processing from VoiceFlow to DialogFlow.
