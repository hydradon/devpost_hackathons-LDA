Inspiration

Majority of the students like us have huge amounts of loan to pay off for school and soon, we might have some more housing etc. This makes it really important for us to make sure that we are keeping a tab of what we spend our money on and how much we are saving for the future. This application could make that really easy for us in the future. At the same time, every person wants to make a significantly purchase at some point in his life but doesn't know how to save or cover up for it. This app will also help us save money to make a big purchase in the future.

What it does

It incorporates the major functionality available in the Intuit Mint software package. In addition to that, FinanceAdvice helps you to figure out how to save money to make a big purchase in the future, with the least possible cut in you current budget.

How I built it

We're currently hosting the project on a Linode server and have our domain name financeadvice.me registered with Namecheap. We had to prepare the server for website hosting and have added in the Nginx framework so it can handle more concurrent users. We also have a MySql server storing all of the user's information, and we have some dope Node.js and javascript.

Challenges I ran into

DNS servers taking a lot of time to synchronize Learning new frameworks like for Nginx, MySQL and Node.js Creating our cloud server on Linode and accessing it

Accomplishments that I'm proud of

We created an actual service that works in real time with users.

What I learned

Node.js, MySQL, HTML, Javascript, Nginx, cloud hosting,
