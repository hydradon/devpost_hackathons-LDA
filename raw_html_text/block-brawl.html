Inspiration

We wanted to make a unique video game that could also be used to reflect society in subtle ways.

What it does

This game is a two player fighting game where players win by jumping on top of the other or by blowing them up with a missile. We have also included multiple cutscenes to reflect to the story of the two characters.

How we built it

We used python and the pygames packages, along with gamebox to create the game. All images were also made using paint.net

Challenges we ran into

Accounting for gravity Walls being changed by increase in size implementing power ups

Accomplishments that we're proud of

Successfully adding five power ups that work properly Adding a modular system to allow for an easier addition of powerups to the game Gravity System

What we learned

How to make a gravity engine How to make a two player real time game Coding with other people

What's next for Block Brawl

Mobile app with online multiplayer with a ranked ladder to encourage competition and continued play.
