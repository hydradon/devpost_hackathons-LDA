Inspiration

Problem solving games could not be vivid and cheap at the same time. With augmented reality, we can combine their advantages.

What it does

Our project maps a real-world object with a 3d model, providing an augmented reality which can be exploited in games such as room escape.

How I built it

After readling documents in Unity3d and EasyAR SDKs, we built our application using Unity, Visual Studio.

Challenges I ran into

We are not familiar with AR or Unity3D, so there is a learning curve when it comes to actually building the project.

Accomplishments that I'm proud of

We explored a new way of playing room escape.

What I learned

Programming experience in the environment of Unity3d and EasyAR. The importance of teamwork.

What's next for Immersed experience: an economical solution using AR

From our point of view, we can incorporate HCI technologies into our application, bringing the immersed experience to an elevated level.
