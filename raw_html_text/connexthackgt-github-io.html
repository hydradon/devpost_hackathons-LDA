Team Search Queeries for HackGT: We came together as a group of mostly beginner web developers with the intent of creating a web application that would have the potential to be further developed in the future.

Drawing inspiration from the director of Georgia Tech's TEDx, we entered HackGT with the idea to create a collaborative web application that would allow for the free, safe, and accessible exchange of ideas, because groundbreaking ideas are what will shape the next generation.

On the front page of Connext, you’ll be able to find a personal data visualization of all of the skills and their corresponding level to allow the user to plan how to shape their skills sets. Furthermore, each users profile page allows access to their personal scheduled meet-ups, provides an opportunity to review their self-described levels of expertise as well as areas for development, and additionally consumers can see their previous impacts made by meeting up with other users. Lastly, check out the discover page to review potential meet up opportunities with individuals set within your designated mile radius.
