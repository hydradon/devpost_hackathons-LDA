Inspiration

My friend is colorblind and sometimes when I send him screenshots, he doesn't understand what he's supposed to be looking at, so this will help.

What it does

It helps people take screenshots so that people with red-green color vision deficiency can see.

How we built it

The project was built using Java and lots of love.

Challenges we ran into

We couldn't directly access the OS easily through Java, so we had to change our project midway.

Accomplishments that we're proud of

All of the features that we added after we thought we finished our project. Originally, we only had the option to screenshot for protanomaly but we also added a feature for a regular screenshot and deuteranomaly, as well as a feature to go back to the instruction screen and to scroll through the current session's screencaps.

How to use

To run this program, double click on the ProtanopicScreenCapture.jar file. A new window will open with the instructions. To take a color adjusted screenshot, press 'F6' and the window will display the modified screenshot. The screenshot is saved to the MyScreenshots folder. To quit, press 'Q'.
