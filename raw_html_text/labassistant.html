labassistant

A virtual assistant which can automate the Customer enquiry about the timings, location, services etc of the Lab so that the Lab need to man a CSO to answer the queries. Also, the Virtual Assistant can assist admin with Server maintenance tasks as starting and stopping an EC2 instance via Chat
