I didn't know Javascript before this, so a lot of learning, but I got through and I'm happy with what I've made so far. I learned a lot about game development, and about Javascript, and a bit about HTML.

What's next for Monster Lazer Shooter of Awesome

I might continue development and add features, however I'm quite happy with this first release. Feel free to fork the repo and create your own version of the game.
