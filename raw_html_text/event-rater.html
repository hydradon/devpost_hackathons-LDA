Inspiration: Thought it would be fun to get the rating of events.

What it does: Insight from relevant tweets for the search term, and an overall rating on a scale of 1.

How I built it: Used Bluemix for both hosting, and sentiment analysis of twitter data. Web framework was django.

Challenges I ran into: To filter the relevant hashtags for the search term.

Accomplishments that I'm proud of: Finished a presentable project in this short time. :-)

What I learned: Bluemix and hosting a live project.

What's next for Event Rater: Better algorithm for relevant hashtags retrievals.
