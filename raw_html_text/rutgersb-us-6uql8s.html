Inspiration

Getting from point a to point b can be tricky, so I wanted to make it less tricky.

What it does

Simply put, you enter where you are and put in where you want to be and the website takes care of the rest.

How I built it

I spent most of my time working on the backend using the google maps geocoding API and some fun unexpected math! The website was thrown together quickly using flask.

Challenges I ran into

Zero web development experience, minimal coding experience, no experience using google maps APIs, it was a lot of work for one person

Accomplishments that I'm proud of

I made something that works, mostly!

What I learned

Backend is easier than I expected and frontend is harder than I expected.

What's next for rutgersb.us

Website improvements, better error checking & input validation, autocomplete, recognition by building codes, & more! Stay tuned
