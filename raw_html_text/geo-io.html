Inspiration

Wanted to implement the user's location to create a fun multiplayer game experience

What it does

You have to "eat" others players around you and avoid larger players. The goal is to get as large as you possibly can.

How I built it

We used Java in Android studio to create the application. Then we used the google-maps api to plot the users location and other player's location on the map.

What I learned

We gained a increased understanding of Android Studio and app development.

What's next for Geo.io

Increase the available range you can travel, and more game features to partake in
