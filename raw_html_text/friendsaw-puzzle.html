Inspiration

We wanted to build a simple interactive game that could be played between friends . A puzzle is a great way of doing that specially when it involves your friend. So we started building Friendsaw where users will play with profile picture of their friend and compete for the best time to finish the puzzle. The developers community challenge was also a huge motivation into working with the project.

What it does

When you hit click to play on a friend's shared timeline, It loads a jigsaw puzzle with that friend's profile picture. The best finishing times to solve the puzzle can be shared in timeline .

How we built it

We used facebook instant games to host our HTML5 game. We used the FBInstant javascript SDK to run our game and use facebook shares to make the game reach out to everyone.

Challenges we ran into

The first challenge was building the game in HTML5 . Being freshers and having limited knowledge of javascript and HTML5 , building the game gave us enriching insight. Understanding the facebook development practices was also a challenge. We learnt how the facebook instant games SDK worked and integrated our game into the platform.

Accomplishments that we're proud of

We were able to understand the mechanisms of HTML5 game development and facebook instant games in a short span of time. We can proudly say we can now build better and more interactive games within facebook.

What we learned

We learnt the expanse of facebook instant games and the level at which we could make a game interactive. In the beginning of the project we had no idea if the game was implementable following facebook SDKs. But we came to learn the tools available to us to build such a project.
