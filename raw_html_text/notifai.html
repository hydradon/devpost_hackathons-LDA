Notifai

Notifai is your social assistant, that is always watching your facebook images so you don't have to. Using clarifai's computer vision platform, we were able to train a machine learning algorithm to identify images that you may not want on your social media accounts. After identifying these, it is simple for users to remove the offending image.

We hope that notify will allow people to more efficiently monitor their online presence and effectively deter online harassment.

What's Next for Notifai?

In the future, we hope to integrate the application with facebook webhook notifications to allow notifications to be sent immediately to users. We also want to add a feature that automatically de-tags a user, and sends a notification, giving them the option to re-tag themselves or not.

Issues we encountered

We encountered difficulties with implementing the webhook api and notifications. All of the apis that we had planned on using turned out to be deprecated. We went so far as to even resurrect a Facebook application from 2 years ago so that we had access an older version of the API
