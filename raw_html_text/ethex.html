Inspiration

Wanted to create an easily accessible way of exchanging cryptocurrency in a peer-to-peer fashion.

What it does

Listing of cryptocurrency offerings and ability to transact with a listing.

How I built it

Used Solidity to develop a smart contract, then used a react app to run the web app.

Challenges I ran into

Tutorials used were based on older versions of Solidity and did not work with the most recent edition - had to do a lot of searching on Stack Overflow.

Accomplishments that I'm proud of

Created a smart contract and web app.

What I learned

How to create smart contracts.

What's next for EthEx

Additional features - potentially order book functionality.
