Inspiration

The show Westworld is so awesome I needed to count the seconds until the next episode in 4K on my LG Smart TV!!!

What it does

The user enters the name of any show and is displays information about the next episode and a countdown.

How I built it

Used the LG WebOS SDK which has a custom IDE and emulator. Used an api from https://www.omdbapi.com/ for getting show data.

Challenges I ran into

Needed to refresh my AJAX skills.

Accomplishments that I'm proud of

Developed on a new platform.

What I learned

How to make apps for LG's WebOS.

What's next for Show Countdown

Use the same framework for other projects and apps for LG smart TVs.
