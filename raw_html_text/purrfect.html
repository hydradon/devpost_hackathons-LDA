• Purrfect

Inspiration

Losing your pet is a very emotionally draining experience. Presently there is no dedicated forum for finding lost pets or reporting stray animals.

What it does

People who find stray/pet animals can upload pictures of the animal along with their personal information so that the owners/foster homes can contact them and take their pets back.

How we built it

We used Python's Django Framework to build the backend and frontend of the web app.

Challenges we ran into

Finding the correct framework to connect backend to frontend. We wanted to use PHP for this application, but we thought of developing data visualizations which are more interactive.

Accomplishments that we're proud of

We were able to learn Django framework and deliver a solution which might help a lot of lost animals to reach back home.

What we learned

We learned how things which appear daunting at the beginning have a very simple solution.

What's next for Purrfect

As mentioned above, we can deploy Geo location-based interactive map to find lost pets, derive insights regarding lost pets across the states in the US and enable a peer to peer lost animal reporting system.
