Inspiration

We wanted to make it easier to find a parking spot for everyone.

What it does

It shows people looking for a parking spot where a free one can be found. It does this using a pressure sensor to sense if there is a car in the parking lot or not. It then relays the information back to the cloud using a LoRa IOT device. The device is low power and therefore only needs a small battery but can last for years without a replacement. The data collected is then shared on a website/app so people can find free parking spots.

How we built it

We used a LoRa radio and an Arduino to send the data to the cloud. We then used Google maps API to map it on a website where it maks free parking spots.
