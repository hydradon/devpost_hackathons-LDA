Inspiration

We wanted to make a game in which the characters were out professors for fun.

What it does

This game allows you to choose between two characters and fight on a single map. The game is designed in such a way that we can add new characters with little extra work as well as new levels and improve machanics. First player plays with an Xbox 360 controller and the second player plays with the arrow keys and wasdq

How we built it

Using the Unity game engine
