What it does It provides a web platform for the admin to administer and keep up with the last updates, barriers, and timeline of the project. It allows the user to upload remarks, notices and reports. While notifying the admin of these updates on a progress bar. It provides a competitive advantage of providing a system generated progress and executive reports.

How we built it Using web applications. The front-end developed by HTML, CSS, Bootstrap and JS. On the other hand, the back-end is developed by PHP Myadmin and MySql database.
