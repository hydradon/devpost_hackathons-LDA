What it does

Join the game with your friends at Dubious Detectives and you'll each be sent a different character with a set of traits. You're all detectives but among you there is a murderer that could be one of your friends. You'll each recieve a text from an anonymous source with some traits about the murderer so you'll need to work together to find the murderer. But be warned, though, one of your friends could be the murderer trying to decieve you with false information so watch your back! If you guess wrong you won't survive!

How I built it

  • twillio to handle input and output to users
  • google cloud to host the site and api server
  • python flask server
  • angular website

Challenges I ran into
