Inspiration

We wanted to created an application that would automate a lot of the resume screening process so that hiring managers can spend more time talking to applicants instead of looking at resumes.

What it does

Our app scans a resume and pulls out both hard and soft skills that a person the employer seeks. It then calls the user and asks them behavioral questions in order to judge their interpersonal skills. The speech from the automated behavioral interview is analyzed by an algorithm.

How I built it

We built our app app as a web application. The front-end was built with AngularJS. The NLU algorithm was programmed in Python and incorporating a variety of different publicly available libraries. Our backend was built using MongoDB and Node.js.
