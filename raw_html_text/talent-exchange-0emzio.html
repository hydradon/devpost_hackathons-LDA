Inspiration

We wanted to make a collaborative community where talents can be exchanged.

What it does

Members can earn points to improve renown, or even exchange for money.

How we built it

We used HTML, CSS, JavaScript and a variety of plugins including underscore.js

Challenges we ran into

Our team lacked backend experience, so we simulated database data using javascript objects.

Accomplishments that we're proud of

We were able to make the data dynamic so we can add them any time, and display them.

What we learned

Even though the implementation failed, we set foot in Node.js and socket.io.

What's next for Talent Exchange

Implement a backend data system.
