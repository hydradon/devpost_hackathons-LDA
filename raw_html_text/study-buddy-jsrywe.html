Inspiration

Our struggles during studying with people who do not have similar work habits as us.

What it does

Connects students of similar grades and courses so that they can study together.

How I built it

We used HTML, CSS, and Javascript. We stored the input into a local storage and compared different data.

Challenges I ran into

The majority of our group had never used HTML, CSS, and Javascript before.

Accomplishments that I'm proud of

Using different languages to make a product.

What I learned

I learned how to use HTML, CSS, and Javascript.

What's next for Study Buddy

Having more users use the program at the same time.
