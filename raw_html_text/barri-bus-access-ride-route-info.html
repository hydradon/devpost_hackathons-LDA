Inspiration

No such applications exist for the local city bus of our locality

What it does

It gives real-time schedule of busses and the routes they will follow, also if the bus is running late or delayed, a notification will be directed to the application user

How I built it

Using google maps api for tracing the route the driver chooses and linking the database of the busses timings and schedule

Challenges I ran into

Integrating google maps api and linking the database

Accomplishments that I'm proud of

Creating a rough outlook of the application and the idea

What I learned

How to apply the knowledge of technology for good purpose

What's next for BARRI (Bus Access Ride & Route Info)
