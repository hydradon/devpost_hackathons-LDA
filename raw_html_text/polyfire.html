Inspiration

The team was inspired to create a product related to the hackathon. We had one goal, to make a simple game which would represent our polyhacks pride.

What it does

The game is clone of a classic arcade game called asteroid, but with polyhacks vibe. The game is at about at 90% functionality with a few more bugs to iron out.

How we built it

We used GameMaker Studio. We had never used the program nor developed games prior to polyhacks.

Challenges we ran into

We ran into several problems regarding collision with different objects, respawning of lives for the player, and populating the correct in-game score.

Accomplishments that we're proud of

We were successfully able to produce a prototype of our game with lot of bugs.

What we learned

We learned how to develop games, work as a team with one goal in mind, and pushing ourselves to the limit. We encountered several problem but surely we found a way around them. We also learned a lot about what we live and don't like about the development portion. Some of us enjoyed the sprite making and others enjoyed the coding portion.

What's next for PolyFire

For the future, we plan on updating our game and implementing a better scoring system and taking our game to the cloud, so that everyone can enjoy PolyFire!
