Inspiration We wanted to create an app that allows a user to contact their political representative about environmental issues, as well as monitor his/her own environmental impact. Due to the nature of our idea, we decided to use the Phone2Action API to be able to contact local representatives.

What it does In order to record a user's environmental impact, the app - through a questionnaire - will calculate the total carbon footprint of the user. We also implemented a feature for members of the community to directly alert their local representative about any concerns in the surrounding area. This "concerns" feature allows for politicians to easily hear the people's thoughts and work with them.

How we built it We used Android Studio and Java to create the app. We also integrated the AirNow API to get air quality data and Phone2Action to get legislator data in certain regions.

Challenges we ran into We had issues converting code from one laptop to another, so we plan to use Github to collaborate in the future.

Accomplishments that we are proud of We are proud of our user interface design in addition to the development of the carbon footprint formula through our form.

What we learned We learned many ways to integrate API into Android applications and display the data from the JSON structures.
