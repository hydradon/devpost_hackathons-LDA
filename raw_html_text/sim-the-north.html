Inspiration

Some of my friends did not have the opportunity to attend this year so i decided last minute to make a "Hack the North Simulator".

What it does

Puts you under the stress of developing an app for hack the north.

How I built it

We didn't want to use any libraries so we just wrote it all in straight javascript and html.

Challenges I ran into

Well, writing the entire thing in Javascript and Html is a little challenging, especially since we could have used a game engine to accomplish something a lot better, a lot faster.

Accomplishments that I'm proud of

That the core logic of the simulation works as intended!

What I learned

Writing an interactive web app is hard work and requires a lot of fixing of dumb bugs and things. Your internet browser is very lenient with javascript so finding errors in my code were difficult.

What's next for Sim the North

Hopefully we can turn this into a fun and interactive game tingy.
