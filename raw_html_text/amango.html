• Multiple accounts, easily managed!

Both of us have lots of accounts that we constantly have to log in and out of, and decided we wanted an easy way to manage all of them.

Amango is a native browser built with electron that uses browser sessions to keep track of all of your accounts.

We built it with HTML and CSS and Javascript and made it into a native electron app for easy cross platform compatibility and to make it into a native desktop app.
