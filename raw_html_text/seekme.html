Inspiration

Fire alarm assembly points

What it does

Verifies if a building is evacuated by checking who is at the assembly point. Notifies who is missing and contacts them via sms.

How we built it

When alarm signal comes, Intel-Edison scans for the user Bluetooth devices available to check if they are at the assembly point.

Challenges I ran into

Getting built-in Bluetooth device to work.

Accomplishments that I'm proud of

Sending sms.

What I learned

Running linux shell commands and bash-scripts from C++.

What's next for Smart Evacuation System

Creating a web-interface for configuration. Building up a network of several Evacuation points. Finding users across a larger area. Build in the redundancy checks to improve the security(e.g. network failure).
