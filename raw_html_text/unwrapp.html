Inspiration

We want to help people stay informed about what they are eating. Especially college students who are prone to things like the freshman 15.

What it does

It scans the nutrition labels of any food item and compares the ingredients to harmful ingredients that we have inputted in our database. We inform the user of potential harmful ingredients. In a future version the user can also input their dietary restrictions which the app can take into account as well.

How we built it

We built this app as an ios app on xcode using the language objective-c. We used tesseract for our ocr and firebase for our database.

Challenges we ran into

One of the challenges was getting tesseract to accurately decipher the picture message. Another challenge was retrieving and comparing the data from the firebase database with the data from tesseract.

Accomplishments that we're proud of

We got the ocr tesseract to work and read the image.

What we learned

Not use objective c as many of the functions are deprecated. We also would use a SQL database for the next version of the app.

What's next for unwrApp

We are going to implement the camera, add dietary restrictions feature, and improve the user interface of the app.
