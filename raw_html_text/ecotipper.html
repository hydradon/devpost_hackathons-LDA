Inspiration

Sustainability is near and dear to my heart, so I wanted to highlight that in this project.

What it does

This app allows you to calculate your carbon footprint with an embedded webpage calculator, and then provides useful tips and facts about reducing your environmental impact. It also shows an inspirational quote about the environment that changes on a timer.

How I built it

Xcode and Swift

Challenges I ran into

This was my first time using Xcode, and first time working with Swift, so the learning curve was really steep. Embedding the website with the calculator was also quite a challenge.

Accomplishments that I'm proud of

I successfully created my first ios app, and now feel confident managing layouts and basic Swift code.

What I learned

I learned the basics of navigating between views, how to use auto layouts, and the basics of Swift

What's next for EcoTipper

I would love to add more dynamic content, like resources to get involved in sustainability based on user location, or a way to enter the calculated carbon footprint score and display scores for all users (creating a sort of "competition" for who has the lowest impact)
