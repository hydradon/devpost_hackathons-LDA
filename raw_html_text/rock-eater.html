Inspiration

We wanted to create a game using one of the newest technologies, and experimenting with virtual reality

What it does

It is a game in which the player must run to prevent the falling rocks from hitting him. In the meantime, the speed of time is controlled by the player so the faster the player moves, the faster the rocks will fall, making it possible to develop strategies by using this tool.

How we built it

We used unity3D and Oculus SDK to implement the best experience for our players

Challenges we ran into

Discovering how a completely new SDK works, and facing the challenge of not having computers with the best characteristics

Accomplishments that we're proud of

Creating a playable experience with oculus in only 36h!

What we learned

How working with the Oculus camera is different from the unity basic camera and how to make a game which is playable both from our laptop as from Oculus.
