Inspiration

There have been numerous accidents caused by people driving shared vehicles, such as Rekola, Lime etc. The cities try to ban these services, but we though there is a way to keep them, while making the authorities happy.

What it does

Connects to various shared transportation providers and offers notification when the vehicle is driving in an unauthorized area or too fast.

How I built it

We built a demo app for Android, used machine learning for classification of the driver's action from camera feed.

Challenges I ran into

We failed to built a server with C#.

Accomplishments that I'm proud of

Machine learning classification works with accuracy above 95%.

What I learned

Keep it simple.
