Inspiration

We were thinking about Marta's archaic method of obtaining ticket passes with wasting plastic cards and physically having to go out of one's way to load money onto their Breeze passes. We figured if we could create an app that would possess all the functionality of a breeze card, it would save people a lot of time and save the plastic that went into constantly creating new breeze cards for new travelers.

What it does

It is meant to allow users to register and log into their unique account where a QR code when scanned charges the user's balance for the marta pass. The use of the Fiserv API validates the user's payment method and allows the user to load money onto their account to use for marta passes

How we built it

We built it using Android Studio with the Java programming language. We integrated Google Firebase to retain user's information that can be generated into a unique id / QR Code.

Challenges we ran into

We had some issue splitting up the work on github. After testing the API calls, it was a challenge integrating the code to request the API into the code for the app

Accomplishments that we're proud of

We were able to create some functional front ends with a clean look and design that user can interact with.

What we learned

We learn that working with RESTful API's are hard

What's next for Metro Breezy

We want to somehow connect the user's information to a payment api
