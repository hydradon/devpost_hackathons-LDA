Inspiration

The project is inspired by services like SocialWall Pro, that combines multiple social networks in one super sexy feed.

What it does

Due to our usage of Kubernetes and clustering technology we built a very robust system.

How I built it

First we setup a kubernetes cluster with six raspberry pi 3's, two network switches and some LEGO. Then we built the software for our "social wall" and dockerized it to then deploy it on the cluster.

Challenges I ran into

Setting up the kubernetes cluster was quite difficult and we had a couple of local network problems.

Accomplishments that I'm proud of

Our failsafe, docker-powered cluster in their LEGO boxes.

What I learned

We learned a lot about high availability services and installing/configuring them.

What's next for winBox

  • We could add more interfaces to other social services (currently we only crawl twitter, gmail and reddit)
  • Add more Raspberry PI to the cluster
  • Add an easier way to configure our software
  • Add "fun" features like emoticon parsing
