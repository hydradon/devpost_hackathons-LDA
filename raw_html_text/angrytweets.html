Inspiration

We wanted to create a service that helps journalists understand how their readers feel about a specific trend and helps them target their audience

What it does

It pulls the most trending and most recent tweets for a specified location and analyzes them for emotions with IBM Watson

How we built it

It is built using ES2017, HTML and CSS

Challenges we ran into

The biggest culprit was visualizing the data without being able to get informations about each tweets exact position
