Inspiration

While large cities like Prague and Brno have strategic plans to become smart cities, we must not forget that 80% of the population lives elsewhere

What it does

Using AI-powered self organizing maps, we can find surprising similarities between cities and municipalities in order to connect those that are facing similar problems.

How we built it

AI Algorithm

  • Python
  • Julia
  • Libraries like SciPy and Matplotlib

Backend

  • Java
  • Spring

Frontend

  • AngularJs
  • Corpis maps

AR App

  • Kotlin
  • Vuforia AR API

Challenges we ran into

Mostly frontend and backend integration plus showcasing in Corpis Maps

Accomplishments that we're proud of

Working out the logic of self organizing maps

What we learned

We are thrilled to have learned and tried out the self-organizing maps approach, which was a completely new concept for us.

What's next for SMAPRT

The tool can further be refined to include more and better datasets, but it can already be of some use for municipalities and NGOs wishing to analyze cross-country municipality data in Czechia
