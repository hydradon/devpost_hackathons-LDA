Inspiration

We had a long line while trying to get into the event.We thought it could be a great idea to create an app that can register people who want to check-in thorugh network location.

What it does

It creates a check-in app that minimizes the amout of time it takes to check in people during a event.

How we built it

We used Android Studio to create the app. We used the database from fire base. We used the network location tools found in google.

Challenges we ran into

Trying to get the right network location. Putting the project together.

Accomplishments that we're proud of

We were able to finish this project.

What we learned

We learned how to used databases, network location, and Android Studio

What's next for BitLine

Improve the portability, fix some bugs, create a better UI. Improv the ability the amount of students who can are logged in.
