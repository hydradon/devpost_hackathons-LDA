Inspiration

Inspired by old arcade games and Alexa's open API, we decided to put them together and create endless fun time for friends and family.

What it does

Play text based games using Alexa and voice commands.

How we built it

We used a web scraper to get text from http://www.web-adventures.org in order to not just have 1 game but a library that you can play right away. We create an Alexa skill that we deployed on AWS Lambda. The skill takes speech input from player and reads the response from the game.

Challenges we ran into

Scraping a website and coming up with an FW looking generic way for all games Creating Alexa skills - dealing with complicated non-english game names. Using AWS Lambda and creating player interactions that made sense.

Accomplishments that we're proud of

All of the above

What we learned

We started with 0 experience on Friday - installing it for the first time. So we learned about: Scraping, AWS and especially Lambda and FaaS, Alexa's API and more

What's next for AlexaSkillHackTM

Play games from other websites. Use AI so Alexa can make up her own games.
