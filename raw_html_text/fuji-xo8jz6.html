What it does

Fuji is a web app that allows users to journal at ease and even use speech to text to document their day to day activities. User's can save and edit their journal entries

How we built it

We built Fuji using flask for our backend, html/css/js for our frontend, and rev.ai for speech to text transcription.

Challenges we ran into

We had difficulty setting up streaming transcription from rev.ai as most of the documentation was for asynchronous loading.

Accomplishments that we're proud of

We're proud of getting the api running and introducing two members to the world of hackathons!

What we learned

We all learned how to use python to make webpages using flask, basic html/css/js, and how to use rev.ai.

What's next for Fuji

We plan on making Fuji a browser extension where one can journal on their home screen. From there, one would be able to save it as a PDF or import it into Google Drive.
