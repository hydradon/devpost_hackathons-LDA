Inspiration

Singapore's Weather Is Very Unpredictable, As such we decided that it would be interesting to develop web app on the chrome store that allows us and anyone to know the weather in detail before heading out

What it does

It works like all chrome app, select it from the top right and a drop-down will come out where you can see the latest temperature readings every 3 hours and for the next 3 days

How we built it

We started off with Codepen to build the HTML and CSS but moved on to Atom to include the API's and Javascript, using Open Weather API, Google Fonts & Codepen

Challenges we ran into

We wanted to add in PSI information from the Governement Dataset, However, the API and links were not working and We wanted to add a share button for them to see the app but we could not do so as it was a chrome web app and not a website/webpage

Accomplishments that we're proud of

We built the app and complied the datasets ourselves, had a load of fun and stress building the web-app in a limited time constraint, which was done from scratch

What we learned

Javascript, Understanding API's and building a web app in general

What's next for Weather@SG

Adding the PSI information, share button(share to your friends!) & Uploading it to the Google Chrome Store
