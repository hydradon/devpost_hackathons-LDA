Inspiration

Meeting up with friends is always a struggle and people are constantly running late

What it does

Allows you to create a meeting and track users the day of the event to know when people are near

How I built it

Using XCode on OSX and html CSS and javascript for the website

Challenges I ran into

Setting up the server proved the hardest challenge.

Accomplishments that I'm proud of

Being able to push to the server from both the app and website

What I learned

Patience is the biggest thing when working with multiple users.

What's next for MAD Hackathon

I would like to see this be a yearly thing.
