Inspiration

As students at Cupertino High School, involved in clubs such as Tino Investors and FBLA, we were hoping to improve the economic market by creating an app that predicts stock prices using machine learning.

What it does

Using polynomial regression, we are able to analyze previous stock data and predict future prices as well as realtime API data.

What's next for TradeX

TradeX hopes to improve its machine learning accuracy by increase the amount of data along with several features in the app that allow people to buy and sell stocks along with the ability to create your own portfolio.
