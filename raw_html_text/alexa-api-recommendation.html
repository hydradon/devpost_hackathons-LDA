Inspiration

It is always a tedious task to find the correct API for your work, most of the time is wasted on surfing the correct API.

What it does

This Alexa skill will tell users the top5 APIs suited for their category.For instance, if a developer is working on a fitness-centric application and he wants an API for his app.So he just need to say "Alexa tell me an API for fitness".Then Alexa will suggest you the best and most popular APIs in fitness category.

How I built it

In the backend, we are running the data mining scripts for fetching the data for different categories.
