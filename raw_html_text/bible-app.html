Inspiration

When I searched App.co,I did not see a Bible dapp, and I was galvanized to build one.

What it does

This dapp allows people to read the Bible.

How I built it

I built my front end with React and my back end with Node.js. I used Blockstack Authentication to authenticate users and Gaia storage to store saved bookmarks (soon to be implemented).

Challenges I ran into

I could not use Blockstack Authentication properly because CORS wasn't enabled for my dapp. I fixed this by finding the relevant information on the Blockstack's website that showed me how to fix the issue by hosting on Netlify.

Accomplishments that I'm proud of

I'm proud of building a dapp that can help people get closer to the things that they find important.

What I learned

I learned how to use React and Blockstack to build a dapp.

What's next for Bible App

Currently the app has only one version, The King James Version, so I will be adding more.
