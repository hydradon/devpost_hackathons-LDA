Inspiration

Inspiring to have better code quality and improve coding style through collaboration.

What it does

Allows the user to post code snippets in various programming languages for other users to post reviews about coding style.

How I built it

I built the frontend in React using redux to manage the state of the app. I used draft-js with the prismjs library to perform code highlighting for different programming languages.

I built the backend in sailsjs which provides different endpoints to save reviews and code snippets and other tasks.

The login system is done through github oauth.

Challenges I ran into

I ran into a lot of problems when trying to implement code highlighting but using draft-js in the end made things easier.

What I learned

Using draft-js for the first time. More experience with the redux workflow.

What's next for Code It

Code iT is going to be used internally inside a company called Think-iT to help them in their programming bootcamp to help code reviews
