Inspiration

In order to get a definitive and straightforward answer, you have to use third party websites such as NameCheap.com to see if a domain name is available to purchase. Just because a domain can't be found doesn't necessarily mean it isn't taken (e.g. crazymonkey.com).

So I thought, why not cut out the middleman and just list on the domain itself if it's available or not.

I felt that there was a lack of websites providing accurate information specifically about their domain's availability. With that in mind, I was compelled to create a website that does just that.

What it does

This website tells you if it's domain name is available.

How I built it

I used HTML and CSS. Shoutout to Khan Academy for teaching me that stuff.

Challenges I ran into

Coming up with a design that was simple yet elegant was quite challenging. Many websites have an interactive user interface, but adding such design features would clutter the web page and detract from the website's authority. Also, the color composition of the background was quite diverse and contrasting, so finding the right text color was perplexing.

Accomplishments that I'm proud of

Not only does this website tell you if it's domain is available, but in the unfortunate case that it isn't available, it will suggest to you another domain to see if that's available.

What I learned

Vertically aligning stuff is hard.

What's next for IsThisDomainAvaiable.com
