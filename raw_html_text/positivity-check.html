Inspiration:

We know that every teenager, including us, deals with stress, and we wanted to find a way to relieve some of the pressures that come along with school.

What it does:

Our program asks the user a series of questions and adapts to those responses to cater to their mood, using jokes and inspirational quotes in the process.

How we built it:

We built this program using python.

Challenges we ran into

Since this is our first time coding, we were not sure of what we were capable of creating or the means of doing it.

Accomplishments:

We are proud of learning how to code using python in 8 hours and having a project here to present.

What we learned:

We learned how to code and that even though it seemed intimidating at first, we were able to overcome this apprehension by practicing and attending the workshops.

What's next for Uplifting Chatbot:

We are hoping to run it on our friends and classmates to improve their mental health.
