Inspiration

There are a lot of food, beauty products and other things expired in our home but we will not notice until once we need to use them.

What it does

It takes a picture of receipts and return all the expiration dates of packaged food and beauty product. At the beginning, the backend would return the expiration date with the default stayingDays (The number of days the product would stay in the supermarket). Later on, with more and more user reports of actual stayingDays, the estimation would be more and more accurate.
