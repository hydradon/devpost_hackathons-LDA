In order to raise awareness about the importance of nature. We have developed a simplistic simulation of an enviroment with Humans, Trees and Food. The project in this state would be a nice base for a more complex game. The game element isn't implemented, just the core mechanics, the user has no acces to the simulation parameters as of yet. Check out the images to see the parameters
