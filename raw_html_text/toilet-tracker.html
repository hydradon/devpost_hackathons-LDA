Inspiration

Caroline thought of a problem that she faces in her own life: sometimes bathrooms can be an unsafe place, especially for women.

What it does

Toilet Tracker uses Google Maps and crowd sourced information to rate restrooms based on the level of safety, cleanliness, and overall experience. Users can search for and leave reviews on restrooms.

How we built it

Lots and lots of youtube and Stack Overflow.

Challenges we ran into

None of us had ever used Android Studio, or even developed an app, so we had to learn quickly along the way.

Accomplishments that we're proud of

It was our first time using gitHub to share code in a bigger project. Also, we used Google API, which we were not familiar with, but we ended up liking it a lot.

What we learned

We learned how android uses an activity based system for managing processes in Windows. We learned that a little too late.
