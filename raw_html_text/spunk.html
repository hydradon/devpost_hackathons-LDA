Inspiration

In today's world, due to the rise of social media, cyberbullying has become a common thing. So, we researched about different mediums available for helping and motivating people who have been bullied and we found out that there is no well-known platform which addresses this issue specifically.

What it does

This app provides a platform for people

  1. to share blogs,
  2. ask questions how to handle and avoid bullying situations,
  3. provide assistance, advice or motivation for people in need
  4. provide emergency contact numbers

How we built it

We built it using Android Studio and started with UI and database connectivity in parallel.

Challenges we ran into

Database connectivity issues

Accomplishments that we are proud of

  1. Connecting app to phone app in the mobile
  2. Loading browser through app
  3. Tried to accomplish basic functionalities in short span of time, even though it was our first Hackathon.

What we learned

  1. Exploring different layouts available in Android
  2. Learned Android architecture

What's next for Spunk

  1. Implementation of sentiment analysis using machine learning models to automatically avoid people from posting abusive/negative comments/posts
  2. A content recommendation system which will enable to extract the tags from posts and provide appropriate links for support from web
  3. Adding emergency contact facility
