• Plant Bot - Allows data capture and control of IoT devices with Alexa

Inspiration

The idea is to create a voice enabled ubiquitous Enterprise communication that enables machines, humans, and enterprise solutions to talk and act.

Enterprise IT landscape is witnessing impact of disruptive technology from consumer world. Next generation workforce comes with the natural expectation of usability at work.

Secondly, increased complexity in building these Smart Products puts newer challenges for Digital enterprise. The traditional tools such as legacy software, Email, and spreadsheets though prevalent today, do not meet future expectations.

What it does

Plant Bot showcases how enterprises can use Alexa and IoT to address critical business challenges.

Plant Bot enables following key features that are demonstrated in the video:

  • Voice enabled hands free interface
  • Real time interface based on telemetry device data
  • Present contextual data, saving non-productive time of search and aggregate data
  • Interface to enterprise software, extract relevant information and present in actionable format

How I built it

Plant Bot is built with Amazon Alexa Skill Kit, serverless computing using AWS Lambda, AWS IoT and Raspberry Pi

Challenges I ran into

I ran into multiple challenges as this was my first Alexa skill

  • Alexa custom skill configuration
  • Setting up Alexa development using Lambda (with Java, Eclipse plugin)
  • IoT Shadow object troubleshooting

Accomplishments that I'm proud of

Architecture is fully Cloud native Serverless, secure, and responsive. In the end there is Joy in accomplishing what I envisioned.

What I learned

Use of Alexa Skill Kit.

What's next for Plant Bot

I would be showcasing the solution inside my organization. There is opportunity to develop a few integrations of Alexa with our own enterprise solutions.
