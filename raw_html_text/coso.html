Inspiration

Having been to a lot of social events connecting with people on multiple platforms proved to be a hassle and waste of time, seeing this we want to make it work in one button.

What it does

Stores usernames in a database for each user and creates something similar to a business card working on a website makes it cross platform and allows users to befriend each other on multiple platforms at once.

How we built it

We built it on the web using HTML and Java script, using python for the back end and SQL for the database

Challenges we ran into

Connecting the front end and back end, issues with the domain

Accomplishments that we're proud of

Having a proof of concept ready to present, while not being aesthetic it shows how effective our app is at delivering what we want it to. A service that saves a lot of time for people.

What we learned

We had to overcome challenges learn to prioritize functionality over design to get something working in the limited time frame.

What's next for Coso

Making the page look nicer improving the code
