• Magic box concept

Inspiration is mathematics fun game to create magic square with number picked by a friend.

What it does is it creates a box/Square with numbers imprinted which all together leads to sum equalling to user input when we count horizontally, vertically and diagonally.

Challenges I ran into was to display output using MessageBox activity of Uipath

Accomplishments that I'm proud of doing simple but effective maths trick using Uipath

What's next for MagicBox is we can do more of tricks and riddles and make Uipath more fun to work with
