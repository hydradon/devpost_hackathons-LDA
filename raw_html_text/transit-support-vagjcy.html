Inspiration

This skill is for those users which are not comfortable in operating Maps on smartphones. So I decided to build and provide a simple voice interaction model to find the local commute around them.

What it does

It renders the data from Google-Maps API and tells you about the local commute available for you. It gives you entire information which is required while travelling locally. It tells you about the timings, cost of the fare and varied options available for you such as Tram, Bus, Train etc.

How I built it

I used the Directions API of google maps. And filter the data accordingly to give it an understandable and seamless voice experience.

What's next for Transit Support

A better interface can always be improvised.
