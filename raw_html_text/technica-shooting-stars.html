Inspiration

Zombies; we wanted people to learn how to triage via playing a game in which they save people.

What it does

You have to find the Golden Star

How we built it

Using Unity and applying new skills abut VR, we were able to create a virtual reality game.

Challenges we ran into

We had a variety of skills we had originally and having to learn new skills. We all learned about VR software. There were also technical issues that may have been able to be avoided. Compatibility between devices.

Accomplishments that we're proud of

Figuring out the VR software.

What we learned

We all learned C# and practice with Unity and SketchUp.

What's next for Technica- Shooting Stars

Hopefully, the game could be used to teach logistical or critical thinking to real day situations. Keeping the Gold Star, players could learn to save money if every time they cross a black star they lose money. Going back to the zombie idea, players can learn how to react in emergency situations.
