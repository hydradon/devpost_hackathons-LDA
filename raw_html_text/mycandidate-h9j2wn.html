Inspiration Candidates seem to take on a lot of different stances, and most people aren't even aware of what those stances are. Of course being election session we want to see more people voting.

What it does This application gives an unbiased view, you pick the stances which affect you the most and we tell you the candidates which support your view points.

How we built it Using the meteor.js framework, it has a mongodb backend and a blaze front-end.

Challenges we ran into Not all of our computers could run meteor.js.

Accomplishments that we're proud of- Learning a entirely new framework.

What we learned- How to implement code live.

What's next for MyCandidate - Hopefully we get people to use it and we can change some peoples view point on which candidate they should support.
