I worked on a project to count estimated number of mangoes on a Green Mango tree and their position on That tree. So i collected a dataset of about 1000 images, labeled them and trained a SSD Inception V3 model using tensorflow Object detection Repo. We have data for 5 to 6 categories of mangoes. That we use for estimation.
