What it does

Aida allows women in developing countries to learn about their options for contraceptives and find local clinics, so they can effectively plan their births. In doing this, they can give a better life to each of their children and focus on their own careers and lives more.

How we built it

We used twilio and Java.

Challenges we ran into

We ran into an issue with the Google Maps API and with sending one message at a time.

Accomplishments that we're proud of

We are proud of finding a workaround to some issues surrounding the use of HTTPServlet.

What's next for Aida

We would like to make the Google Maps API work and make a function for people with some extra money to donate to Aida's cause through SMS.
