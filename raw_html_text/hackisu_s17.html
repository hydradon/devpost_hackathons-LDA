hackisu_s17

A genetic algorithm neural network that learns to play Mario. This is done from scratch, utilizing only an open source emulator and a Mario ROM that we ripped from a cartridge. There are many levels of input. The Mario game is mapped and translated according to schematic guides. This process was done through reading ram and constructing a map from it. The neural network uses mutation to create new layers. Must acquire your own ROM to test ;)
