Inspiration

We felt that car rides could be more fun if there was a mutual playlist with songs we all love!

What it does

It creates a playlist with songs shared in common between both parties.

How we built it

We had two teams, a frontend and backend team. The backend was build using react, javascript, and the spotify api, while the front end was built using html and css.

Challenges we ran into

The biggest challenges we ran into were OAuth tokens and integrating the front and backend into one product.

Accomplishments that we're proud of

We are proud that the application is fully functional.

What we learned

Web Dev is hard.

What's next for Matchify

Properly integrating the front and backend (non-terminal based).
