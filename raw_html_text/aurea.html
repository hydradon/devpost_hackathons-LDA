Inspiration

Making it easier for teachers to take attendance!

What it does

It automates the attendance of events just by getting close to the region.

How I built it

Using Particle Swift I combined Bluetooth IBeacon technology to allow users to be tracked while inside a determined region.

Challenges I ran into

Had issues using old libraries for Bluetooth support, but was able to make platform dedicated therefore fixing the solution.

Accomplishments that I'm proud of

The simplicity and how useful this information can be!

What I learned

Particle, Bluetooth Beacons
