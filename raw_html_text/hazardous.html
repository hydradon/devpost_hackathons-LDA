Inspiration

Hazard base detection prevention at work.

What it does

This is an app that analyzes videos featuring work operations and to send events through Azure IoT Hub that can be viewed in Azure Web App.

How I built it

Using the very well explained user guides at Azure website. Used tensorflow models and OpenCV methods for feature matching.

Challenges I ran into

Docs are good so everything went smoothly deploying the app/python sender.

Accomplishments that I'm proud of

Processing a the video fast enough to consider this real-time solution so the events can be sent as soon as possible to Azure IoT Hub.

What I learned

New tech stack to connect devices is a good thing!

What's next for Hazardous

More GPU capacity and perhaps running on GPU VM. Also implementation of new detections.
