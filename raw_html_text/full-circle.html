Inspiration

Current processes for exchanging contact info are tedious and inefficient, especially when trying to exchange information between all members of a group. This usually requires everyone contributing to one document (piece of paper, facebook message, google doc, etc.), or manually exchange info pairwise. We are here to change that.

What it does

Our goal is to streamline the process of connecting with multiple people across multiple platforms, optimizing it to be quicker and more efficient. Users will be able to quickly add and connect with each other all on one centralized hub. We help users connect full circle.

How I built it

We built an android app using the Facebook graph API and Firebase as our tools. We first made wireframes based on what we thought was most important to our future users - simplicity and ease of use. After discussing and refining our features to cater towards our target users (students trying to connect in educational or social groups), we moved on to developing the app.

Challenges I ran into

Learning how to develop an android app and using the Facebook Graph API and Firebase.

Accomplishments that I'm proud of

Actually implementing a back-end and using APIs that we had no idea how to use before.

What I learned

How to work well in a team and learning new technologies!

What's next for Full Circle

For future iterations - Group key/password for joining instead of just a hashtag, visualizing connections, refine the design, interaction, animations, integrate more social media platforms: LinkedIn, Github, Twitter, etc., improve code quality
