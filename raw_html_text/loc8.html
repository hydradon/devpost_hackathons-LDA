Inspiration

Social Networking

What it does

Allows users to share their locations, along with a status message, with friends

How we built it

Android studio, java, xml, firebase, esri

Challenges we ran into

It was our first time building an app with firebase with real time data and updation and so it took a while to get things started. we also had to understand how to use esri's api which took a while as well.

Accomplishments that we're proud of

We worked all night to create a working social app and overcame issues related to map and geolocation.

What we learned

We were able to learn a lot with firebase, maps, permissions, and how to integrate api's that are unfamiliar.

What's next for Loc8

Expand social media capabilities like chatting with friends and being able to create posts
