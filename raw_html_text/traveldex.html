Inspiration

Why jump around the internet booking travel and taking notes when you can do it all in one place?

What it does

Traveldex lets you book travel through Airhob ( http://airhob.com ) without leaving Office 365. Right now, it works in Word Online and for Flight Search only.

How I built it

Following some online tutorials to get set up.

Challenges I ran into

Just the learning curve of building an Office 365 add-on. This was my first add-on so it was a great learning experience.

Also couldn't get Booking from Airhob to return a valid "ticket". Will try to fix that first.

Accomplishments that I'm proud of

Getting it all to work successfully!

What I learned

A lot about building Office 365 add-ons and interacting with the Office apps through add-ons.

What's next for Traveldex

Expand to all the Office applications!
