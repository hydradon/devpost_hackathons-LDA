Inspiration

We wanted to experiment with hardware.

What it does

We developed a terminal game that iluminates the keys of the keyboard, and you have to press them.

How we built it

We used the Logitech SDK with c++ in Visual Studio.

Challenges we ran into

Logitech challenge.

Accomplishments that we're proud of

We had never worked with hardware before, so it was a lot of fun to make a game with it.

What we learned

We improved our C++ coding skills, learned how to use Visual Studio and key events of C++.

What's next for Light it up

An improvement could be make it an online game, where you can challenge your friends!
