Inspiration

We were inspired by our daily struggles as University students to spend appropriately. When you get busy with school and homework, it's easy to forget how much you're spending but now with Saveify, we have created an effective solution.

What it does

Saveify asks a series of questions based on a variety of expenses an average University student would experience and provides a recommendation on how they can save money, by comparing their expenses to the average Ontario University student.

How I built it

We built this website using HTML, CSS, and Javascript.

Challenges I ran into

Majority of the team members were not familiar with JavaScript so we struggled in implementing it into our code and connecting it to our HTML and CSS files.

What I learned

We learned the basics of JavaScript and how to execute a javascript function when clicking a button in our HTML page.
