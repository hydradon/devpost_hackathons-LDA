Inspiration

Real life physical bots does some task for you. We wanted to build a chat bot that does the same.

What it does

After you authorize and configure it and it sends messages on your behalf on a regular interval on pre-configured set of themes e.g. General, Love, Motivational etc.. to the person you have selected!

How we built it

Telegram App SDK, C#, Azure, React, Bootstrap

Challenges we ran into

Getting to impersonate the sender and getting the Telegram APIs to work. Learning React!

Accomplishments that we're proud of

E2E working product

What we learned

Chat protocol, network flakiness, more robust fallback and retry mechanism

What's next for LoveBot

More intelligent heuristics and message repository and take more conditions like previous chat history and current location into consideration.
