Inspiration:

Don't Tap The White Tile

What it does:

When the user clicks the rounded rectangle, it turns into a circle. If the user doesn't click all of them under a certain time limit, they lose.

How I built it:

Using jQuery, Javascript, HTML5, and CSS3 And cloud9: c9.io

Challenges I ran into:

Although we're not completely done, we ran into a few challenges with creating basic functionality, such as the timer and the transition from rectangle to circle.
