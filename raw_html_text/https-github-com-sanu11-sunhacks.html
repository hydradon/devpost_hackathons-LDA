Inspiration

Lot of times the internet connectivity in the rural areas isn't good enough for people to search their queries on internet. Our aim is to provide them with the results they need for even though they dont have good internet connectivity.

What it does

Our product provides them with the internet query results they ask for using their cellular network without having internet connectivity.

How we built it

We have used Twilio for calling purpose. The user calls the number registered with Twilio. Twilio maps the number to the URL publicly available on the internet that runs the google assistant api and returns back the voice results to the user through Twilio.

Challenges we ran into

Integration of Twilio and google-assistant api

Accomplishments that we're proud of

Giving users the chance to access internet results through cellular network
