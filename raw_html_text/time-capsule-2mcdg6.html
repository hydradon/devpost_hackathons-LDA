Inspiration

Remember those time capsules that you used to make as a kid? When I was in elementary school, we would collect trinkets and write notes to our future selves, then put them in a sealed container and bury it for set periods of time, usually a few months to a year. After the time passed, it was interesting to uncover the contents and remember what we were doing and feeling when we first buried them. The Time Capsule app is the digital version of the old-school time capsule: it allows a user to write notes to their future self and set the date and time at which they would like to receive the message.

What it does

Write a note to future you. It can be anything: well wishes and positive thoughts, or solemn reflection and a reminder that you made it through some tough times. Basically, whatever you think will inspire you in the future.

How I built it

I used Sketch to create static images of the user interface for the app.

Challenges I ran into

This was my first time using Sketch, so I was unfamiliar with the tools and resources available. But it was a lot of fun figuring it out!

Accomplishments that I'm proud of

Pulling off my first iOS app design.

What I learned

How to import and edit Sketch templates and elements.

What's next for Time Capsule
