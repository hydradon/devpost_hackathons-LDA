Inspiration

Training dogs is though, and can be boring. Why not add a twist by letting Alexa help?

What it does

Dog Trainer for Alexa is an Alexa skill that allows you to train your dog, together with Alexa. After teaching the trainer your dog's name and sex, the trainer will start

It makes training your dog fun again, allows children to train the dog in a playful way. The fact that Alexa's voice is exactly the same on every training helps the dog to understand and grasp the commands faster, similar to a clicker training.

The code

Dog Trainer for Alexa is built with the Alexa Skills Kit, with a Python AWS Lambda. The source can be found on GitHub
