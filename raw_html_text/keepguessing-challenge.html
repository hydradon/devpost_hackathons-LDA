Inspiration

Actually I was learning game development in college, suddenly one of my friends adviced me to build a game development project in HTML

What it does

In this game, you have to match the same pair of an object hidden inside the grid. You can play multiple times in your free time. It also gives an appropriate message after each pairing or completing the game.

How I built it

I built it using Html 5, CSS and JS.
