Inspiration

We couldn't decide which challenge to do or hack. After hours of researching and talking to other companies, we decided to do something we have never done before. The Qualtrics Challenge seemed a great opportunity to visualize some data of our choice.

What it does

We as a team were able to get some data on firearms and visualize it using python, and the Plotly API.

How we built it

The final product was built by scraping some data files (.csv) and then parsing through the data with Python. Once we had out data, we used the Plotly API to help build the visualizations.

Challenges we ran into

Finding accurate data. Parsing some junk data. Implementing Plotly API.

Accomplishments that we're proud of

We are proud that we were able to host the final product using Azure Virtual Machine. Also proud of the fact that were able to actually learn how data visualization works.

What we learned

We learned a bunch of of Python. We also learned how to work with a new API form Plotly.

What's next for Firearm Visualization Data

We hope to continue and get more data to compare and cross reference with other studies.
