Inspiration

Whenever we make new friends or coworkers, we always get asked a question: "Where do you want to go to eat?" Whether it's out of politeness or genuine indifference, we often find ourselves awkwardly trying to think of restaurants that everyone would enjoy. Here's where our app takes over.

What it does

Our app will take in a set of three preferences: your maximum price point, the furthest distance you'd be willing to travel, and your minimum review rank (ie 1 star or 3 star restaurant).

How we built it

We built this app using JavaScript, HTML, and CSS as well as using Firebase to host the application.

Challenges we ran into

As the both of us are fairly unfamiliar with web development, we had to learn a lot along the way. In addition, the frontend was particularly challenging for us to make because we often had to search everything up.

Accomplishments that we're proud of

We also had never used Firebase before, but thanks to the tutorials we were able to learn quickly.

What we learned

We learned a lot more about web development, such as a greater understanding of what the server versus the client handles.

What's next for Let Us Eat!

We will continue to develop this app, mostly to improve the frontend.
