Inspiration

We were partially inspired by Dr. Heinz Doofenshmirtz while we were thinking of a project to pursue... Hence the Money Exchanginator was born. We wanted to use the Twilio after seeing the demo and also wanted to try applying the banking system.

What it does

A user can exchange money with other account holders at the Doofenshmirtz Bank inc.

How we built it

Capital One API for the banking system, Twilio for user input for transfer requests, Android Studios to make an Android app.

Challenges we ran into

Troubles with using Capital One API Bugs with Twilio Using Android Studios for the first time.

Accomplishments that we're proud of

Learned Android Studios Used APIs for the first time Tried using Twilio to send and receive messages

What we learned

How to use Android Studios. Just keep on coding.

What's next for Money Exchanginator

Add more functions (check balance, etc.) Better app design, user interface Available for more phone types (sizes, ios)
