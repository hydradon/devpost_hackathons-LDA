Inspiration

All queues around the world

What it does

It allows user to join the line using his smartphone or another device via web client or telegram bot we and analyse how he can spend spare time until he is first in this line. It also allows a specialist to manage a list of queues.

Solution

Our app contains API, web interface and a telegram bot; Via bot and web interface user can join the line, find out his position in this line. Via administration web panel creator of lines can manage queues and analyse their work (i.e. average time).
