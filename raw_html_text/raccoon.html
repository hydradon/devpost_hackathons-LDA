Application Overview

Analyse applicant's interested/concerned thing and motivation for the workplace By interview note.

Target

Human Resource Agent

Background

Person's interested/concerned things is changing constantly.

By communicating with the person, we can analyse the concerning/interesting things.

But it's very skilful thing.

Human Resource agent have to know them quickly and correctly without working with them.

As some of indexes for many wishes of applicant, this app helps them.

And then, they can find the best workplace for the applicant
