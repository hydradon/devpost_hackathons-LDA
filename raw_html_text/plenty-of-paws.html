Inspiration

We set out to develop a project to connect single felines across the globe. As pet owners ourselves, we understand the struggle that cats face when it comes to finding a mate. As a stereotypically solitary creature, cats are often unable to make meaningful relationships in the real world. Concerned pet owners are at a loss as to how they can help their beloved family members to find their purrfect match.

What it does

Plenty of Paws is a website where cats create a profile. They talk about themselves and what kind of cat they are looking for. Members can email each other to connect.

How we built it

Html5, CSS

Challenges we ran into

As beginner programmers, most of our group has not worked with html or css before. Not having an understanding of javascript or php also lead to some features being impossible to create.

Accomplishments that we're proud of

This was a great first step into learning how to use html and css. We managed to create a prototype navigation bar from scratch.

What we learned

The group learned the basics of html and css in order to create a working website.

What's next for Plenty of Paws

A working "Create Your Profile" page where cats can submit their dating profile.

A login page for the website.

A messaging system that works on the website instead of email.

An algorithm that matches cats based on compatibility.
