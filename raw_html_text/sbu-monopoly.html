Inspiration

As avid Monopoly enthusiasts & proud Stony Brook students, our aim was to combine the two and create something truly - far beyond!

What it does

You can play online monopoly (up to 4 players) on our server

How we built it

We spent long tedious hours working on developing the front end (the board) to look pretty using react.js and scss. We then made sure to create an Information board, and have logic being served through Node.js on the backend.

Challenges we ran into

Handling doing turns for players and creating the front end for the board were extremely challenging.

Accomplishments that we're proud of

We are very happy with our layout and also our amazing choices of locations ;)

What we learned

Front end is hard, and tedious but as long as you're having fun, time flies by.

What's next for SBU Monopoly

  • Making sure that we can execute more complicated monopoly rules - like buying and bidding properties, and handling money via a Bank
  • Creating a Server to run the game instead of a local computer - and have multiplayer using multiple devices
