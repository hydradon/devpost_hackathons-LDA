Inspiration

Wanted to make a way for students to spend their time as much as possible.

What it does

Gives an easy way for UTD students to freelance work to other students.

How I built it

Used React for front-end, Flask for the backend, and MongoDB for database.

Challenges I ran into

Learning React, and starting late.

Accomplishments that I'm proud of

Able to connect backend and database successfully.

What I learned

How to properly use backend technology to connect to database using Flask and MongoDB.

What's next for Poor Favor

Implementing more features.
