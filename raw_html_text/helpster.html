Inspiration

There are so many different methods to organize your life but they lack a certain consistency across different devices. This aims to solve that problem.

What it does

It helps you organize your day while also allowing you to view statistics relating to your interactions on a day to day basis to help you become more efficient.

How I built it

The app is a node backend with a react front-end. It also has an api we made for communicating with DialogFlow as well as twilio for cross-device interactions

Built With

  • express.js
