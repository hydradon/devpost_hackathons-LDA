The Problem

Most of the pictures we take have something wrong with them; whether it be a blurry image or an unhappy face.

The Idea

We wanted to create a hands-free approach to editing people's faces and decided that a web application was the best way to go.

Implementation

We created a simple and intuitive ReactJS application which connects to a Microsoft Azure AI service that identifies a person's face in an image. We then used the characteristic data from the Azure service paired with our own algorithm to edit a person's face into a happier version.
