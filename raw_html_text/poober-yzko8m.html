Inspiration

Our group had a common problem. We are out in public and we need to use the restroom. We thought to ourselves, "There must be a better way!" This is where poober comes in to save the day!

What it does

Poober allows you to find bathrooms near you ASAP.

How we built it

We used a flask server using the Google Maps API and NYC Open Data to develop the site. Then we hosted it on poober.tech

Challenges we ran into

We had a lot of problems setting up poober.tech and managing the DNS problems. We also had problems with the API due to limits put on the API

Accomplishments that we're proud of

Making the world a better place.

What we learned

We learned how to set up a domain and getting it to work and host our pages.

What's next for poober

Next, we will make our poober app to make finding bathrooms even easier!
