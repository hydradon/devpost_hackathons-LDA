Inspiration

Imagine you are in France and you want to plan your trip according to attractions and restaurants in your vicinity. Being able to quickly see an array of activities to do and places to eat, that are easy to access, is vital when planning an itinerary for any trip.

What it does

This app shows you nearby, restaurants, hotels, schools, hospitals, stores, and local boutiques, and allows you to add them to an itinerary to look back on later.

How we built

HTML, CSS, Javascript, Web APIs

Challenges we ran into

Using Google Maps API

Accomplishments that we're proud of

Implemented the Google Maps and Google Places APIs

What we learned

We learned how to use local storage

What's next for You Maps
