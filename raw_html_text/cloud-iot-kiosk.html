Inspiration

Light switches, but better.

What it does

Controls smart devices using the droplit.io cloud! Using a Raspberry pi and lcd with touchscreen, you can create a kiosk for any zone, with the intent of adding on to every room in a house or office.

How I built it

Using Visual Studio Code
