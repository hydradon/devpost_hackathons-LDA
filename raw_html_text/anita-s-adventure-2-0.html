Inspiration

The intro video at the beginning of the Boilermake Hackathon inspired us to create this adventure based game featuring Anita.

What it does

Our program tells a story through a game environment.

How we built it

We decided to build our game using Scratch due to the ability to easily create a graphical user interface. The graphics were created in the Google Drawing web app.

Challenges we ran into

Drawing on a mousepad Troubleshooting code Finding an idea that we were capable of coding in one weekend

Accomplishments that I'm proud of

Creating a fully functional game

What I learned

Before turning to Scratch, we learned a lot about Django and Flask from online tutorials. While using Scratch, we discovered the array of possibilities with a seemly simple language.

What's next for Anita's Adventure 2.0

Perfecting small details, adding more sounds, adding additional features, settings, customizations. https://scratch.mit.edu/projects/362505850
