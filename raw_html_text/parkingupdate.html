Inspiration

I was inspired to build this app because I see the need for a more efficient system to help people find free and affordable parking in LA.

What it does

It takes city data to determine if you are are parked safely in LA, and reminds you when you should move your vehicle to avoid a ticket.

How we built it

We had our UX designer wireframe it. We have two dev's build a static mock.

Challenges we ran into

Converting the data into objects in the app. ( We needed more Dev power )

Accomplishments that we're proud of

We found the data that is crucial to building this app.

What we learned

We learned how to work together on one idea. We learned to use an empathy map and a business model canvas. We learned to utilize each other's skills to keep building on the same idea.

What's next for ParkingUpdate

The backend build. We need a full stack developer to help us code this into IOS and Android. We are also considering making a Web App that will be able to convert to both markets.
