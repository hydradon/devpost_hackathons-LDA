Inspiration

Social media is an important part of today's lifestyle. 72% of college students say they get the majority of their news and facts from social media apps. So we'd like to help make it a smoother, seamless experience between different apps and websites by allowing you to store any interesting posts or websites as personalized playlists. The stored posts and websites can be given a description and ranked to be able to viewed at a later time. They can be sorted by categories for a more streamlined experience.

What it does

Allows users to store web links in an efficient manner to be able to be accessed later on.

How we built it

Using Android studio as the IDLE and java as the language of choice. We used the Firebase database to manage and store user information and personalized added posts.

Challenges we ran into

Not having real time synchronization

Accomplishments that we're proud of

The utility and design of the app

What we learned

Multiple onClickListeners running in various activities can cause an overload on the UIThread

What's next for Post Playlist

Being able to use your own categories and having in-app capabilities to add a post
