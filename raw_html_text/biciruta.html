Inspiration

Thousands of bike riders are use to look for new places to go, better if its far away from the busy cities, and they are even more tempted to ride bike while traveling to other countries or cities, but they don't have a community to do so.

What it does

Biciruta messenger bot, helps people find user created bike routes based on their location, level of expertise and preferences

How I built it

I've created a small app so the users can create routes for bike rides, so the messenger bot will communicate to the backen, after asking some basic information to the user, it analyzes user's data, and recommend routes based on his information

Challenges I ran into

Making messenger bot answer based on actual data Creating an SSL certificate Deployment Learning how the technology works Applying to a real problem

Accomplishments that I'm proud of

Connecting a real database with the messenger bot, so the information is smart, and also I've learned about PNL Even if the messenger bot is the most important thing, I've created the entire map editor for the user to create the routes.

What I learned

How to use PNL (Natural language processing) How to use location in messenger bots How to use wit.ai

What's next for Biciruta

The app should be able to send plain map images about the routes and of course by using new future features of messenger app, will be great if the map itself can be shared. So the biker can follow the routes instead of just going to the website or having just a plain map, It will be even cooler to build features to share rides that you are having with your friends and not just for yourself, will be great to create groups who can track many people on the same biciruta plan
