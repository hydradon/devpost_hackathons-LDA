Inspiration

While many believe technology is a limiting factor in social interaction, we wanted to use it in a way to enhance social interaction.

What it does

Giphy Guesser is a simple phone game using Giphy which involves having a player choose a .gif relevant to a word or phrase that they choose and having other players guess the chosen word or phrase after viewing the given .gif image.

How we built it

We used Expo.io to easily create an app which runs on both iOS and Android while calling the Giphy API to fetch relevant .gif images. The app is made with React-native which works well with iOS and Android.

Challenges we ran into

We had a few issues with deciding what frameworks to use, but Expo.io was very easy to understand and use. We also ran into issues using various other APIs and technologies which limited us to using those we were about to work with.

Accomplishments that we're proud of

We scrapped another idea halfway through the hackathon so we didn't have as much time as we would have liked to build this idea. However, we got a working proof of concept which we are pretty proud of.

What we learned

Preparing more ideas for hackathons ahead of time would be super beneficial for situations where we decide an idea is too difficult or found to be unviable.

What's next for Giphy Guesser

Better multiplayer features over bluetooth, wifi. etc, More gamemodes and variations, Smarter and more relevant Gif suggestions.
