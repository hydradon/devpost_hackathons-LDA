Inspiration

Our society struggles to think of new and interesting things to do so we end up doing the same things each week and we wanted to change that

What it does

It finds an activity for you to do based upon specified search criteria

How we built it

We built it using HTML 5 CSS 3 and JS using Sublime

Challenges we ran into

Finding out we had to pay to use the Google Maps API

Accomplishments that we're proud of

The app looks beautiful and professional

What we learned

We learnt how to implement map APIs into an application as well as how to think about the security of a venue based service

What's next for wheel

Continuation of the search and review capabilities of the app as well as the accessibility features
