What it does

This mobile application can identify fish species. It can provide information of fishing zones. Provide information of fishing seasons. Identify found gear by qr code. Mark your gear on a map. Look up vessel information in a database. News and push notifications for news items.

How we built it

We have built a C# .Net backend and connected it to a HTML5 Hybrid application for front end. The backend is fully customizable and flexible to implement unlimited functionalities without boundries. The mobile application is available for iOS and Android.

Challenges we ran into

It was a challenge for us to analyze an image caught from camera or file input and process it on the phone. We solved that by sending the photo to the backend and do the heavy lifiting server side.

Accomplishments that we're proud of

Image analyzation is an accomplishment that i'm proud of. This is a feature that is handy for most fishermen to categorize their catch.

What we've learned

We've learned to think outside the box and never give up on crazy ideas!
