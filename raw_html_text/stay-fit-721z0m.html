Inspiration

Introduction of Alexa presentation language and my personal interest in fitness. And obviously this competition.

What it does

This is a fitness skill For easy Home workouts.

Welcome to Stay fit. This is a fitness Skill.

Here I am providing the user with the options of 4 categories cardio Abs full body and yoga all these exercises can be done at home, without any equipment. exercises are generated by the system the user just have to select the difficulty level from easy, medium and hard. Hope users will enjoy this skill and stay fit.

Note:This is the first version of the skill. I will keep improving this skill based on the user feedback.

How I built it

I build it using Node js for coding AWS lambda for hosting the code and S3 to host the resources

Challenges I ran into
