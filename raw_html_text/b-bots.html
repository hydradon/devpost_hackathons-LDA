Inspiration

The inspiration for this came when a friend struggled carrying things through sand, so having an automated carrying machine would make this task easier.

What it does

It automates the movement of any object without having a person present.

How I built it

I use AutoCAD Inventor to develop the entire base, I used a raspberry pi to connect the ultrasonic sensors, motor and camera, wrote the sensor logic in python and created the web app for the controls.

Challenges I ran into

The Li-Po battery I was using puffed up, so I will be unable to demonstrate the balance, but I will be able to display the sensor data.

What's next for B-Bots

Developing a full platform for multiple robots to function at the same time.
