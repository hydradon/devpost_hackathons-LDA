Inspiration

GIPHY!!! Bringing the world closer with gifs!

What it does

We wanted to translate a video and its body parts (title, description, captions, and comments) into a stream of gifs.

How we built it

We built the backend in php to grab the youtube video that the user has chosen on the frontend, we grab the video's info using the YouTube API and then make calls to GIPHY's search endpoint with certain phrases/words from the title, description, captions and comments.

Challenges we ran into

Implementing the Youtube custom search feature on the site so users do not have to leave.

Accomplishments that we're proud of

It all works

What we learned

We improved all of our HTML, CSS, JavaScript, and PHP skills

What's next for GIFifai

Reddit support!!! We want to turn sub-reddits into a stream of GIFs that describe them!
