Inspiration

Following the recent election we made this fun website soundboard to relieve tensions and stress.

What it does

It is an online soundboard that allows users to play sound bytes from Donald Trump, and combine then into sentences.

How we built it

Using dreamweaver, html5, css, and javascript.

Challenges we ran into

We ran into many challenges creating the feature to allow users to create a queue of words to play in sequence while ensuring a proper amount of delay between words.

Accomplishments that we're proud of

Creating a project at our first Hackathon!

What we learned

We learned how to work together as a team, and also improved our skills in html5, css, and javascript.

What's next for DJTrump

We will continue to update and improve the website to provide more features.
