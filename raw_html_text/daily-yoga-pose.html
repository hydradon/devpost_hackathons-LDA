Inspiration

Being a software engineer for years, it took me some time(and some health problems) to realize that we need to take care of our whole system, not only the weight. yoga helped me a lot to fix my problems, but it wasn't always there, and there were times that I couldn't find time for a course. So if I had a device with a screen, the first thing I would like to have would be something that makes yoga easier to access.

What it does

It generates a Yoga pose and describes how to practice it in details.

How I built it

I have read the tutorials on how to use the APL, did some design on the online APL tool. Generated the json file (needed to modify it though) I used Git for my own source control and coded with python. Served via heroku.

Challenges I ran into

The APL editor had some problems, sometimes I had to download the json, modify it and upload it back, because the tool was not always editable. I tried too hard to find pictures of the poses that are completely free to use, and I had to research in many different yoga sites to make sure I use the right words to describe the positions.

Accomplishments that I'm proud of

Apparently APL is very new, so being an early bird makes me feel proud. If the application gets activated by many people, I will remotely affect many people's lives in a very positive way. That makes me proud.

What I learned

APL.

What's next for Daily Yoga Pose

Because of the time constraints, I couldn't include most of the Yoga poses. I'm planning to include at least whole basic poses.
