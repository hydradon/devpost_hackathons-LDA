Inspiration

Saw student faint in University Dining Hall, due to low blood sugar.

What it does

Allows members of an organization report medical emergencies to the organization.

How we built it

A little of everything. The files submitted include HTML, JS/jQuery, Swift and MySQL. We can improve them by replacing all the JS with Ruby and doing the mobile stuff in some cross-platform framework.

Challenges we ran into

Databases.

Accomplishments that we're proud of

Watch us nail this presentation!

What we learned

Ruby on Rails. MySQL. API deployment. Don't do things in JS just because everyone else does. If it looks easier in Python, do it in Python.

What's next for GoldenTree

A mobile app for use within an organization, allowing quick emergency alerts. Use data to analyze likely health risks for members of organization. Allow authorized users to request health information regarding other members.
