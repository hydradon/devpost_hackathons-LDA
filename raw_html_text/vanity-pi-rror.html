"Mirror mirror on the wall..." - Sleeping Beauty or something

The Vanity Pi-rror was created out of an interest in IoT devices which enable ordinary items to become something greater. An monitor covered with an acrylic sheet and magical one way mirror film provides a poor man's display for the raspberry pi running a php backend with nginx that serves HTML and JavaScript files. The mirror displays general information such as the time and weather, but also offers a personal experience for the user by utilizing a raspberry pi camera board and microphone for facial and speech recognition.
