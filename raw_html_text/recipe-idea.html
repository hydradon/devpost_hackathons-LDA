• Recipe Idea Amazon Skill card in Alexa app

Inspiration

* Sometimes, you face the question - what to prepare/cook today? Just Ask Alexa: Open Recipe Idea. * If you like what you hear, you can find more info on up to date discussions on reddit.

What it does

Suggest user what to prepare today and what community recipes are actual

How I built it

I have built it using Python and hosting it in Amazon cloud as Lambda service

Challenges I ran into

Porting my python code from Raspberry to cloud. Facing issues with Reddit API (sometimes it becomes to slow) and will implement caching.

Accomplishments that I'm proud of

Using python on Amazon infrastructure

What I learned

How to use/host Lambda service and easily publish/update python code.

What's next for Recipe Idea

I am working on caching for faster response. When Amazon team enables links in Alexa app (cards) I will add direct link for each recipe
