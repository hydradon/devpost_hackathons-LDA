Inspiration

We are working on building an immersive coding educational platform. We started with Alexa because it’s ease of use and great interaction model. The idea of waking up and saying “Alexa, ask Devhulk to teach me something” seemed like a great way to challenge yourself as a young person to learn programming terminology.

What it does

After asking Devhulk to teach you something you will be read a programming term like “variable”. You will then be told the definition and how to use it in a sentence.

How I built it

This is a lambda function with modified terms to be easier to understand from a young adult/ new to programming perspective. The Alexa skill is called Devhulk.

Challenges I ran into

Changing complex technical definitions into more easily understandable definitions, that are still accurate.

Accomplishments that I'm proud of

45 terms that will aid in your programming journey!:)

What I learned

Voice interaction is powerful.

What's next for Devhulk

Continue to add terms and release video content that complements the skill. That way it really starts to create a more immersive learning experience.
