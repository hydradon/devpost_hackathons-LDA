Inspiration

In our daily lives, we experience a lot of stress and we wanted to create a simple app to help tackle this issue.

What it does

Our app provides a few simple actions for the user to help them destress, such as visual breathing exercises, positive message of the day, and calming sounds

How we built it

We used Android Studio to code our app using java and xml.

Challenges we ran into

We had challenges with our very time working with APIs and SDKs.

What's next for Destress

We are working on adding one more feature using Text-to-Speech and Tonal Analysis to create a sentiment analysis for a sentence.
