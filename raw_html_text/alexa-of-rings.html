Inspiration

Bringing all kinds of gamers together, many people can't physically play traditional video games, using the Alexa allows everyone to experience a great adventure

What it does

It uses the Amazon Alexa to interact with the game, such as creating a character and fighting battles. We a built neural network to learn how you play and adapt the game to how well you are playing.

How we built it

We used Flask-Ask a library for python to interact with the Amazon Alexa, we also built the neural network in python. All systems were connected together using a ngrok server

Challenges we ran into

Challenges we ran into was coding the game logic. Amazon Alexa is not built well to run an adventure, so we had to use unorthodox techniques to get around the problems we faced

Accomplishments that we're proud of

We extremely proud of creating our first functional neural network, and programming the Alexa for the first time

What we learned

We learnt more about python, machine learning, flask-ask and how to create a skill on Amazon Alexa

What's next for Alexa of Rings

We want to scale the whole system up. We want first create a full adventure for players to experience, add more training data to the neural network and creating a live GUI
