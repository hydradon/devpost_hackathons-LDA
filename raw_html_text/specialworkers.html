specialworkers

proyecto para hack puebla 2017 - temática Grupos vulnerables

Es una aplicación web con la finalidad de ofertar empleos a personas con alguna discapacidad, que nace de la necesidad de la falta de herramientas que satisfagan de manera optima el correcto uso con este tipo de personas.

El usuario que se registre podrá registrarse para la vacante de su interés, así mismo las empresas podrán ofertar diversas vacantes.
