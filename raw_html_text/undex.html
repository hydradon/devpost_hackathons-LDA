Inspiration

Looking through the provided dataset we found an interesting set containing unoccupied houses. Needed an idea as we had recently scrapped the old one, we decided to run with it.

What it does

Find places that are unoccupied(according to council data) that are within certain distances of types of buildings.

How we built it

Python backend serves all the data and works out best places. Javascript frontend displays data.

Challenges we ran into

Massaging the data into a nice format, working out a heuristic solution to an NP problem. (Lat, Long) -> (x,y) was difficult.

Accomplishments that we're proud of

Getting it finished in 8 hours.

What we learned

Web technologies, how well we can operate under fatigue.
