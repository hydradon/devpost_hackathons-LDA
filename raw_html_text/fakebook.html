Inspiration

What it does

Fakebook lets you use a site that is like a clone of Facebook, and post as your friends.Also using Algolia you can search all of these posts.

How we built it

We split up the task with Iva and Callum on the front end, Adam working on Facebook login, and Aaron working on the Database.

We also used vagrant to make our environments identical.
