Inspiration

We noticed that it was hard to find like minded individuals to program with and it was also hard to find groups to compete in hackathons with before hand.

What it does

This application allows users to search for people with similar skills or skills that they would like to learn and request to connect with them. If a connection is made, both users' social media accounts (Facebook Messenger) is displayed so the users can communicate with each other.

How we built it

We built this application using Node.js and React Native. We used Node.js for the backend and React Native for the front end.

Challenges we ran into

Since it was our first time using React and Node we ran into problems figuring out how to set up the environments, and we also had trouble integrating the programs. We got stuck on sending and receiving data.

Accomplishments that we're proud of

Coming into this hackathon, two of us had zero experience with node and react and after building this application we have enough understanding to use react and node again.

What we learned

We learned more about the user interface and also how databases work and store data via using react and node.

What's next for Find a Friend

We would like to continue developing this application so that we can publish it and people can start to use it to connect with other like minded individuals.
