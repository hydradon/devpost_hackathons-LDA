Inspiration

A presentation by Environment Hamilton during the opening ceremony.

What it does

It should present opentreemap data along with inhalemap.com data, and reward users for planting trees in polluted areas.

How we built it

Using Android Studio and StackOverflow.

Challenges we ran into

The Google Maps API turned out to be a pain to work with for us, due to our lack of experience as well as the differences across versions. This was further compounded by attempting to get the current location.

Accomplishments that we're proud of

Reverse engineered part of the inhalemap website's data entry points at long last.

What we learned

There's always more to learn, but be careful when picking up projects when the original creator is MIA and the technologies used are unfamiliar.

What's next for Environmentally Friendly App

Adding features, more testing, rewards for planting trees weighted according to surrounding air quality.
