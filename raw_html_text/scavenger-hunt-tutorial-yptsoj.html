• Scavenger Hunt Tutorial

Inspiration

I wanted to create a game where people could learn about T-Mobile's self service options and their Team of Experts.

What it does

Scavenger Hunt Tutorial mimics the HOME tab of the T-Mobile App with green Easter eggs for the user to find.

How I built it

I built this game using Gamemaker Studio 2.

Challenges I ran into

I had to mimic the touchscreen application with the user's mouse and nothing else. I also had trouble deciding what kind of game to create in the beginning and how to incorporate the Team of Experts.

Accomplishments that I'm proud of

I completed a workable prototype in a short amount of time.

What I learned

I figured out how to move the camera view with mouse controls.

What's next for Scavenger Hunt Tutorial

Incorporating the remaining tabs.
