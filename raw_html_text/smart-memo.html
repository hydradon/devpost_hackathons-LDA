Inspiration

We're lazy to write down or type into the notes sometimes.

What it does

Our app can help you reduce your laziness of taking down the notes by just taking photos. The system will extract only the texts on behalf of you from the photos taken and even from your photo library. Our app provides translation function as well.

How we built it

We built it using 'Swift' language and we embedded 'Google Cloud Vision API' for extracting text and 'Kakao Translate API' for translating. For database, we used 'CoreData'.

Challenges we ran into

Search function was quite hard to implement / Collaboration via Git was a bit troublesome - wasted time to fix the conflicts.

Accomplishments that we're proud of

We implemented most of the requirements and technical specifications we have set in prior.

What we learned

Very importantly, we have learnt how the co-operation works during teamwork. Learned the general workflow of Git.

What's next for Smart Memo

We will do code refactoring after this Hackerton and we can improve its performance.
