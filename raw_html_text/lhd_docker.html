OneLyf

OneLyf is a Python Django application running in a Docker container. The custom image uses port 8000.

This is a natural disaster management application. Here all the disasters are listed. Each of this has a map associated with it, where the the other users would have marked which locations are safe, danger, or in which application help is needed the most.

This application is also linked with news API, twitter API, and google Maps API for getting relevant data about each disaster
