Inspiration

Star Wars themed Hackathon! :)

What it does

The fight against the Galactic Empire rages on, a Rebel X - Wing fighter jumps out of hyperspace to be met by an on-going battle within an asteroid field.

How we built it

Using an arduino in a lightsaber we get accelerometer and potentiometer readings to control the X-Wing via a node.js server.
