Inspiration

Tinder has been the most common dating app for a long time. Finding a match on tinder is not easy and adding visual impairment doesn't help. Our goal was to make tinder more user-friendly for everyone to find a match.

How we built it

Amazon's Alexa Skill Kit.

Google API

AWS Lambda (Backend and Natural Language Processing)

Python (Backend for website)
