Inspiration

Adding holes to cylindrical walls is a common modelling task. It would be nice to have a tool to automate it, make it easy to create, easy to modify.

What it does

The "Hole On Cylinder" comes with a command dialog, allowing user to select a cylindrical face in Fusion 360. Then the script will automatically create a hole on the cylindrical wall per the size, location, orientation defined by the user in the command dialog.

How I built it

Followed Fusion 360 API Help docs and related samples.

Challenges I ran into

The main challenge for me is to understand the relationship between the coordinate systems in sketch space and in model space, to make sure that sketches that are created by the code are placed at the correct location. This is challenging because cylindrical faces selected by user can be at any location.

Accomplishments that I'm proud of

It normally takes multiple steps (multiple work planes, multiple sketches) to manually create a hole on a cylindrical wall. With this app, users will only need to select the cylindrical face and provide a few input parameters and get the hole right away.

What I learned

Fusion 360 API and related 3D knowledge.

What's next for Hole On Cylinder

Get the hole size data from some kind of data services.
