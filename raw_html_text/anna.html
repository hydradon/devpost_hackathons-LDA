Inspiration

In the world of a digital advertising, many companies still use offline posters to advertise their products. People see them only for a few seconds, which means that without an immediate interaction the content will be quickly forgotten.

What it does

HoloBee allows people to get the most relevant artist related information from a billboard that advertises music concerts.

  • Get the latest artist's playlist
  • Get the location of the venue
  • Get the latest tweets from an artist
  • Buy a ticket

How I built it

  • Using Google Cloud Vision API , we extract the text (artist's name) from a billboard.
  • Content Pool - to get the latest tweets from an artist
  • Qwant API - to get the list of tracks
  • Songkick - to get the information about avenue and then calculate the destination using Google Maps API
  • Unity Game Engine + Vuforia AR SDK AR Map - AR

Challenges I ran into

Apart from the tricky API handling, combining all the responses into a single AR experience is a complicated challenge

What's next for Anna

Move from using Vuforia AR SDK to Google ARKit
