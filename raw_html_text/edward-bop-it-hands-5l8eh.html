Inspiration

The Bop-It basically began the Industrial Revolution. To honor this historic item we've created a virtual version of the favorite past-time instrument.

How it works

The setup uses LeapMotion to track your hands; LeapMotion then works with Java to tell the player what motions to make, and a score is given based on how well you can keep up. All of the code is written in Java, including the user interface.
