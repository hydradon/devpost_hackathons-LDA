Inspiration

Our inspiration was handshake because handshake is linked to LinkedIn.

What it does

The app encourages people to do more community service, in order to gain points and accomplishment badges that connect to LinkedIn and gain employers.

How we built it

We used JavaFx to create the design. We used the language Java and used SQL.

Challenges we ran into

We ran into many challenges. We had a lot of errors in our code, and linking our code to the database was difficult. We were successful in designing the layout of each page, but had trouble connecting one page to another.

Accomplishments that we're proud of

Our design layout on Java Fx, icon designs, trying to use high level code as beginners.

What we learned

We learned a lot about designing an desktop app through Java Fx, and we learned SQL and complicated code we are not familiar with.

What's next for Connecteer

We need to improve the backend development of our desktop application.
