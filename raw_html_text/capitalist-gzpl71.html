Inspiration

To learn more about the stock market

What it does

It is a simulation of trading stocks and helps teaches younger kids about the stock market. It does this by having simple news events that affect the stocks and has limited options for stocks to prevent kids from being overwhelmed from having too many choices.

How we built it

Unity and C#

Challenges we ran into

  • Only two people knew how to use Unity, and only three people knew C#
  • Many bugs and numerous null pointer exceptions

Accomplishments that we're proud of

  • Everyone learned a little about C# and Unity
  • Pretty nice graphics
  • Have most of the logic implemented, but we had trouble displaying the results on the screen

What we learned

  • Game development is difficult
  • Front-end development it highly difficult as it involves combining so much data together
  • Teamwork is important
  • How to organize code classes and the hierarchies
