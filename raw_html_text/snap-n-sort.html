Inspiration

We all know we should recycle, but we don't. This is because we are lazy and need instant gratification to recycle. Our app solves this.

What it does

You upload an image and we'll tell you if you can recycle it. If not, we'll offer a better alternative. You can also earn points for recycling and get rewards for good recycling habits. You can also see nearby where you can recycle.

How we built it

We used the google-vision API for image recognition. We wrote our code in Python and used Flask as our framework.

Challenges we ran into

Our interactive map was difficult due to the limited amount of data available - we will work on this!

Accomplishments that we're proud of

We are first-time hackathoners and never met until yesterday - so we are very proud of our app!

What we learned

We learnt a lot about how to build an app and how to integrate several APIs and cool ideas into one project.

What's next for Snap-N-Sort

We want to add a leaderboard to our rewards section and we also want to be able to hover over images and tell you instantly the nearest place to recycle this item.
