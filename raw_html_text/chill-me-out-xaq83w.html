Inspiration

Make your dev life great again!

What it does

We have made an INTELLIJ-ent stress level tracker that provides live feedback based on a person's facial expressions

How we built it

Our solution comprises of two pieces

  1. Developing a model for predicting the stress levels
  2. Creating an Intellij plugin that invokes the model and triggers a response

Challenges we ran into

  1. Creating the exclusive dataset for the model.
  2. IntelliJ plug-in SDK.
  3. Synchronizing the response of ML model with IDE's plugin in real time.
