Promptu is a collaborative platform used to raise awareness around social issues using speech-to-image technology. Users interact with the installation through a web-app on their phone in which they are prompted with a question and provide an answer using their voice. The answer given is then represented as an image and is displayed on a digital canvas. As more users add their answers to the canvas, it grows to create a unique collage of the collective mindset. The user is also given insight into the social issue behind the prompt.

PLEASE NOTE: For this uploaded version we needed to remove several image models from the processing directory in public. There used to be 345 different object files each with 500 variations of image data (i.e. vertices). This file resulted in a final project size is of 90MB so several items from this directory had to be deleted.
