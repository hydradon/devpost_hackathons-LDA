What it does

  • Collates and stores your scanned pictures of your receipts and analyses the text using OCR.
  • Placed in a central repository for you to view later on and analyse.

How we built it

  • Python server running Flask (REST Api)
  • Tesseract OCR to recognize text in the receipts
  • Vue.js to handle the data from our API

Challenges we ran into

  • Local network issues (for testing on mobile devices)

Accomplishments that we're proud of

  • Learning Vue.js and developing an application we personally would love to us.

What we learned

  • The framework and methodology behind Vue.
  • The importance of knowing the areas/market of our application.

What's next for Receipt'd

  • Adding additional functionality.
  • Streamlining the service into an easier and more connected process.
  • Integration into different services.
