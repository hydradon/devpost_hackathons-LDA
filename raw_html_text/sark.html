Inspiration

Hiren's bootcd. We decided to make the life of IT professionals a little easier by making a easily configurable GUI for TronScript, which is a an open source batch suite used for cleaning and disinfecting computers. The script includes features like rkill, mbam, TDSSKiller, Stinger, registry backup, WMI repair, sysrestore clean, oldest VSS set purge, create pre-run System Restore point, SMART disk check, NTP time sync, etc.

To learn more about the script, visit Link ![this wiki]

What it does

Disinfects, repairs, patches, and optimizes your windows PC. The program allows you to easily configure TronScript before executing. This allows the user to tailor the program to their needs.

How I built it

We built it in java using JFrames and the swift library.

Challenges I ran into

Working with JFrames and swift was a hassle. Allowing the user to configure the program to their needs before running.

Accomplishments that I'm proud of

Making a GUI in Java, I would not do that again.

What's next for Sark

Making it auto-update the program before running since TronScript is constantly being improved. Expanding the features of the program to give the users more flexibility tweaking the script. Allow the user to add custom scripts to fit their needs
