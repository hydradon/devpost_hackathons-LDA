Inspiration

During technical interviews, we are often asked about CS side projects we have worked on. We have found that we don't have many substantial projects to talk about during these interviews. We wanted to create an online platform for students to share their project ideas, and find a crew for their project. The Crew Search will help students build their resumes as well as build connections with other students. Students will be able to get experience collaborating in groups - something that will be extremely useful as we move into 'the real world'.

What it does

Connects Simmons students with each other to work on projects that they are passionate about. The projects can be categorized by goal-oriented topics such as Mental Health, Social Good, Web Dev, etc.

How we built it

Javascript, jQuery, HTML, CSS

Challenges

We are pretty new to web development, so this project was a big challenge for us. We learned a lot about the ins and outs of web dev and got to experiment with new languages.

How will we improve The Crew Search?

  • integrate a database for the user info and project info
  • work on the design to make it more user-friendly
  • expand project to other schools
  • allow alumni to become mentors on projects
  • implement a recommendation system, so that users can be matched with other users based on similar project interests
