What it does

It skips (sometimes). Hates usb3.

How we built it

Throwing code together to get mpv to play and skip, and read from a TTY on the host end. Artistically crafted, bespoke C++ code on the 'duino.

Challenges we ran into

X86 and ARC was a marriage made to be broken. An AVR would've been more reliable, and Serial over USB turned out to be really flakey.

Accomplishments that we're proud of

It works when it feels like it.

What we learned

Some microcontrollers are very regressive and anti-progress when it comes to the latest universal serial bus specifications and implementations.
