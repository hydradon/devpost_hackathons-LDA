Inspiration

Looking at the data sets on data.world about growing homeless populations. We want to leverage this data to tackle the problem of homelessness.

What it does

  1. Visualize homeless populations across the country.
  2. Aggregates data from charities (shelters, food banks, etc.)

How I built it

Javascript front end, RESTful APIs (Jersey), MySQL backend, scaled with Spark

Challenges I ran into

Integrating new teammates, Coordinating mobile data input to display Finding open shelter data

Accomplishments that I'm proud of

Being able to deliver a working prototype in a short time frame

What I learned

Being able to work as a team to accomplish a project within time constraints Spark, HERE APIs

What's next for MapRefuge

More datasets Prediction modeling Aggregating drone data Leveraging homeless skills through the platform
