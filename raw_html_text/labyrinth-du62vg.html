Inspiration

The inspiration for this game was the association between darkness and the unknown. In order to keep the game mechanics simple, we looked toward minesweeper for ideas.

What it does

Labyrinth is a game about exploring the unknown! We were challenged to explore the unknown in unique ways by Ubisoft. The game is set in a dream reality that is nonsensical and perilous! You control your character to avoid deadly traps in order to navigate to the way out!

How we built it

To build this project, we utilized the Ubisoft API and its game engine, which provided all thr resources we needed to create the environment and mechanics on Labyrinth

Challenges we ran into

Many challenges such as bugs and difficulties in understanding c++.

Accomplishments that we're proud of

Despite the lack of knowledge of c++ prior to this hackathon, everybody on the team contributed meaningfully and collaborated to successfully build a game.

What we learned

We learned how to use the Ubisoft API, as well as gained some insight and experience regarding game development, and had an opportunity to learn about c++;
