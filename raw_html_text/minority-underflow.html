Inspiration

Our team is composed of 4 individuals who are minorities in tech and all come from different backgrounds and have different experiences related to being a minority in tech- many good, but some bad. We all thought that having a way to confront people who have made unnecessary comments to or about us would be a really great tool. So we built that this weekend!

What it does

There is the functionality to post questions, add responses to questions, read the About page, and see questions on the homepage. There is also a Resources page which has information on different organizations and conferences for minorities in engineering and tech.

How we built it

We used JavaScript, Node.js, HTML, and Bootstrap/CSS.

Challenges we ran into

We had issues with the domain but got them resolved. We also had issues with the database and Bootstrap, but got them resolved.

Accomplishments that we're proud of

Having our website deployed on a server with Heroku.

What we learned

We learned how to make a website from scratch and how to use Bootstrap and JavaScript this weekend.

What's next for Minority Underflow

We want to add an upvote system, similar to Stack Overflow's, as well as a page with an individual's questions.
