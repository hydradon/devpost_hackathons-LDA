Inspiration

There is a heavy discussion, if or when Bitcoin will rise massively or blow up eventually. For both sides, it's intersting to get to know indicators as soon as possible.

What it does

Tells you when bad news rise, marketplaces go down or the trades change a lot - but it's a first hackathon version yet only.

How I built it

We build an engine using sentiment analysis and deep learning - and inform you via realtime alerts.

Challenges we ran into

Connecting all bits and pieces as well as keeping it realtime.

Accomplishments that we're proud of

Finding many interested people.

What I learned

Always keep the first version as simple as possible - then stack up.

What's next for BitcoinBubbleBurst

  • adding more data sources
  • creating a improved analytical model
  • adding more alert channels like sms and chatbot
  • Subskription model with extra features
