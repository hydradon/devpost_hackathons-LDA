Inspiration

Home Automation can be cheap

What it does

Our product is capable of controlling output signals based on maximum values set via the internet from a web interface.

How I built it

The product is built using an Arduino 101 and a Particle Photon to communicate through wifi.

Challenges I ran into

Finding the best way to send data received through wifi on the the Photon.

What's next for ArduinoAutomation

We would like to be able to store our readings in a database to monitor trends of temperature and light data.
