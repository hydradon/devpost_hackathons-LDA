Inspiration: We have always struggled to get to the west side without getting getting a sketchy ride and we want to make life nicer for Us and others Like us

What it does: It is a Long distance Ride share app that allows users to easily find rides and give rides

How we built it: We used android studios as the IDE , Java and XML as the main programming Languages and Google Cloud Services especially Firebase as back-end

Challenges we ran into: We lost 2 teammates within 30 min of start, never used Java Before, Never used Android studios before. Ran into issues Google Cloud Services.

Accomplishments that we're proud of Finishing a Good Demo app for the time we spent, implemented Google Cloud Services.

What we learned: Java,XML, Android Dev, App Dev

What's next for RideAlong: Facebook integration, More Google Cloud integration, improved UI, More in-depth User Profiles, getting Human Users.
