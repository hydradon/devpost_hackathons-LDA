Inspiration

Promoting hawker culture and wanted affordable meal suggestions

What it does

Locates hawker centres near user

How we built it

6 hrs on learning how to compile with dependencies. The rest was built on "free" electricity and Wi-Fi. We took a bottom-up approach to building it. At least we think we did.

Challenges we ran into

Lack of detailed Java API for Telegram

Accomplishments that we're proud of

We managed to produce a working bot that is close to deployment.

What we learned

How scrub we are. How inexperienced we are...

What's next for HawkAi

Improve on data completeness (we lack price ranges, menus and opening hours, etc) Providing access to such data that we lack
