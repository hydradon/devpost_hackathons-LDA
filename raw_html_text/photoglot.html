Inspiration

A desire to dissolve the language border using photo recognition that directly delivers the identification in a language of the user's choice

What it does

Recognizes a picture taken in real-time and returns the result in a language of the user's choice

How we built it

Using google cloud API's running through python, we connected it to a web app with a simple interface designed through html, css, javascript, and django

Challenges we ran into

Accessing the user's camera was difficult as well as communicating between the script and interface

Accomplishments that we're proud of

Successfully completing the project (with a nice UI too!)

What we learned

Implementing APIs, working with UI

What's next for PhotoGlot

Other languages, permanent learning app, move to mobile dev
