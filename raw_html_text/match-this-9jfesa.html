Our Inspiration

Our inspiration was the PoseNet tensor flow model and the drive to use it to create something creative.

What It Does

Match This! tracks your body using PoseNet and compares how well you have done compared to a predetermined set of poses. Participants have three seconds to create the correct pose before it determines their score and moves on to a new one.

How We Built It

Using vanilla JavaScript and HTML, along with some CSS for styling.

Challenges We Faced

Performance with PoseNet, as it takes quite a while for the API to return and update each pose.

What We Learned

To set more realistic goals when coming up with shortterm projects.
