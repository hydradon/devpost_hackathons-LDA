Inspiration

We have over thousands of songs in our collection that is not enough for storage in a smartphone.

What it does

It wireless streams music files from an SD card to the smartphone app flawlessly.

How I built it

Php Apache Server hosted in Edison, Android, iOS

Challenges I ran into

We couldnt figure out basic music player functions, but eventually wrote algorithms that did.

Accomplishments that I'm proud of

We made both Android and iOS apps along with a caching algorithm that lets the user makes the listing of the files fast.

What I learned

iOS coding, Java, installing Edison's Yocto Linux image

What's next for Euphony - Wireless Music Streaming
