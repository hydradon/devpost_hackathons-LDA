Inspiration

We wanted watch to tell us whether we left our phone.

What it does

It vibrates whenever the connection is lost with the phone.

How we built it

We used CloudPebble API with C language and some on-line support websites.

Challenges we ran into

We could not use the Pebble JS API to be able to use a Worker (background running up on pebble watch), we had to switch code to C

Accomplishments that we're proud of

We were able to accomplish this without the need of separate mobile app

What we learned

Some C language (we never programmed in this particular one), Pebble API, Pebble Watch (we also never used one before)

What's next for Leash

Quicker response time with the disconnection (signal strength checker), a better GUI, clean code, add lock functionality for the phone
