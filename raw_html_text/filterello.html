Inspiration

Filterello hopes to solve the "too many cards" problem highlighted by some Trello users by providing tools to enhance card visibility in a Trello board.

What it does

Filterello enhances your visibility into your Trello board:

  • View recently created or updated cards. Access recent cards from the board view.

  • Flag cards for follow up. View flagged cards from the board view. Manage your personal flags from the card view.

How we built it

Filterello is built using HTML, CSS and JavaScript and utilises the Trello Power-Up API to interact with the Trello platform.

Challenges we ran into

Navigating the Trello Power-Up client library APIs and seeing which ideas we could implement during the hackathon.

Accomplishments that we're proud of

Developing a Power-Up that would be useful to a Trello user without compromising on the simplicity and flexibility of the Trello experience.

What's next for Filterello

New features coming soon to Filterello:

  • User-configurable quick filters accessible with one-click from the board view
