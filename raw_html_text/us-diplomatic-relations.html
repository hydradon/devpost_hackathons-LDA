Inspiration

Creating a dynamic map to show the United States' diplomatic outreach

What it does

A world map with a scrolling timeline to show established diplomatic relations between the US and countries through history.

How we built it

We used a Python script to scrape data from the state department website. Then used JavaScript to create a dynamic map that will show the data we collected.

Challenges we ran into

The state department website was excruciatingly difficult to work with, the data was never in the same id or in the same spot.

Accomplishments that we're proud of

We were able to successfully scrape the data from the website and create a website with the dynamic map.

What we learned

We learned a lot of Python and JavaScript. Coming in we knew close to nothing in both these languages, and now we have become familiar with the basics.
