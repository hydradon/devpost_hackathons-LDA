michdud.github.io

Functions Allows users to view movies and ratings by other users about their gender and racial diversity. Users can click on a movie title to view the individual movie's page, as well as entering ratings for gender and racial diversity.

Purpose Help people examine the media they consume, and see how their perspective differs from others'. Not designed to be an objective measure of a movie's "goodness," but a way for people to understand the impact of media in a context different from their own.

Tools We used HTML + CSS with a Bootstrap template, some Javascript + JQuery, and a Firebase database to store the movies. We used a MovieDB API to find movies.

Challenges It was difficult to manipulate the data from the database using HTML and Javascript.
