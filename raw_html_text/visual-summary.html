Inspiration

One of our teammates came up with the idea.

What it does

Takes a document and searches for keywords and uses those keywords to find a picture summary.

How we built it

We built the program with Java Swing for the GUI, and Java to process the files and gather keywords. Additionally, we used the Google Image search API as well as parsing JSON using third-party libraries.

Challenges I ran into

Originally we were going to make a web app, but we didn't know how to create .jsp files in Google App Engine to store the information. In the end we decided to make a Swing interface to build on the back-end coding we completed already.

Accomplishments that I'm proud of

We were able to finish our project in 10 hours.

What I learned

We learned more about web design and development.

What's next for Visual Summary

We would like to develop the web app in the future if we did not have the time constraints we had here.
