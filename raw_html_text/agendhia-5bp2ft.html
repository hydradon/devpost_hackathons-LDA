Inspiration

One thing so many college students struggle with is time management. Often times, we are pressed with so much work that we need to figure out ways to manage time properly, especially if your working a part-time job like so many college students, while still trying to live an active social life. Agendhia attempts to provide a solution to this very common problem by providing users with a platform to write out their daily schedule.

What it does

Agendhia attempts to provide a solution to this very common problem by providing users with a platform to keep track of their daily schedule, with the added bonus of receiving rewards based off your performance in our own mini RPG game that we incorporated.

How we built it

We built it using Google’s Firebase for user authentication as well as keeping tracking of user data. We also used Swift and Xcode to build the front end of the application.

Challenges we ran into

We struggled a lot with authentication and learning the ins and outs of Google’s Firebase. We also struggled navigating the nuances of Xcode, and learning various aspects of Swift, which we were unfamiliar with.

Accomplishments that we're proud of

We are proud to have come up with awesome idea. We genuinely think that it can help a lot of people and think that the mini game is a fun, engaging addition that will encourage people more to do tasks.

What we learned

We learned the ins and outs of Xcode’s interface, various swift concepts, and firebase concepts.

What's next for Agendhia

We hope to expand more on our idea, especially the game aspect. We hope to incorporate several more pets and implement a combat system in which you utilize your pets to defeat various dungeons.
