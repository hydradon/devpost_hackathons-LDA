Penny FYT

What it does

Penny For Your Thoughts is a penny-based social media in a web application platform. The users will be able to react to the posts meaningfully, and to make an impact for what they care for. This project has been created for Dubhacks Fall 2016

How we built it

The code utilizes HTML, CSS, Angular, PHP, Stripe, and Firebase. The APIs we used include Stripe and Firebase .

Inspiration

We wanted the users to make meaningful likes, by allowing the users to load their money to their account and to send a penny each time they like or comment on a post.

Challenges we ran into

We faced some challenges around micro-payment services such as Stripe.

Accomplishments that we're proud of

We used a couple APIs(firebase and stripe) in order to implement our project.

What we learned

It was a great chance for us to use some APIs that we have not worked with before. We also practiced the languages we already knew, such as html, css, angular.js, and php, and polished our skills into the product.

What's next for Penny For Your Thoughts

PennyFYT will become the next standard for business model in social networks. We also want to aggregate multiple social media platforms into our Penny FYT platform, so that the users can use one platform for all kinds of posts.
