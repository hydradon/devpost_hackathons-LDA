Inspiration

Background results concentration loss, sleeping disorder and cardiac arrhythmia. We aim to change that through active noise cancelling (anti-sound) devices built into noise polluting units such as AC, refrigerator and other appliances.

How we built it

We've built an arduino-powered device, including a microphone and a speaker. Programmed with Python, it is able to invert the audiofile it receives and than emits the anti-sound accordingly.

Challenges we ran into

The set-up and consistency have been the most difficult tasks.

Accomplishments that we're proud of

Our software is top-notch.

What we learned

How to run a python scirpt on an arduino powered device. How noise pollution works. How to simulate soundwaves in octave.
