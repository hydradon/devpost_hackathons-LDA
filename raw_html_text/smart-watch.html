Inspiration

We wanted to build a SmartWatch and took inspiration from JP Morgan's competition prompt to make the world a better place for entrepreneurs.

What it does

It is a simplistic device that has the ability to be personalized and accommodate features that entrepreneurs need in order to be productive in their day to day life.

  • Tells Time & Date
  • Stop watch
  • To Do List
  • Stocks
  • Secure

How we built it

  • The actual programming of the watch was done through Arduino IDE was tested on a Arduino Uno.
  • The website to explain the features was done using HTML and CSS.

Challenges we ran into

  • Hardware issues to make the device secure. In future implementation we plan on using a fingerprint sensor to improve the security.

Accomplishments that we're proud of

  • The two time functions on the watch: time and the stopwatch.

What we learned

  • We learned how to use QR Codes, make a website using HTML and CSS.

What's next for Smart Watch

  • Fingerprint Sensor.
