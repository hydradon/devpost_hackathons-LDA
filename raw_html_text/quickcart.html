Inspiration

We wanted to build something that would make the shopping experience easier and require less overhead or waiting.

What it does

Walk into a store, scan a cart, grab your items. You'll automatically be charged to your account!

How we built it

Using Angular, Android, Arduino, and a few other languages, we were able to combine hardware and software to make a physical cart that would be able to collect and transmit data to be analyzed on our manager web app and mobile phone app.

Challenges we ran into

Server and parsing issues came up along the way, but the most common problem was dealing with unsoldered nfc reader tips.

Accomplishments that we're proud of

We were able to fully implement a working model! There were essentially no hard coded components.

What's next for QuickCart

With backing from Macy's we want to be able to actually put this in stores.
