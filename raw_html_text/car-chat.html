The inspiration for this app is based on my experience when my car transmission light went on and I don't know how urgent the case was because my car seems to be running fine.

What it does - the app recognize if the repair is urgent and needs service soon

How I built it - I built it with IBM Watson Assistant

Challenges I ran into - I have not used Chatbot application before so the challenge is learning how to use and implement the program. IBM engineer Upkar was very helpful showing me how to add nodes and intents.

Accomplishments that I'm proud of - The chatbot worked!

What I learned - I learned how to use IBM Watson Assistant

What's next for Car Chat - add more features
