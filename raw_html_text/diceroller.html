Inspiration

We were inspired by the RPG Dungeons and Dragons, and were encouraged to create the app due to the lack of dice rollers available for the Samsung Gear S2.

What it does

This app generates a random integer within a range determined by the user's dice selection. The user can choose to roll a 4-sided die, 6-sided, 8-sided, 10-sided, 12-sided, or 20-sided die

How we built it

We used java in Android Studio

Challenges we ran into

We were unable to make the app work in android wear, and we were unable to make android studio run on linux

Accomplishments that we're proud of

We were able to make the app work with mobile.

What we learned

We had previously not had any experience with Android studio, and one team member had no experience with Java, so this experience provided a lot of opportunities for both team members to gain experience with android development and Java.

What's next for DiceRoller

We hope to make DiceRoller compatible with android wear and to add animations for each of the rolling dice.
