Inspiration

My curiosity and love for nature and plants. I also wanted to learn more about machine-learning.

What it does

It uses machine learning to identify and classify types of flowers.

How we built it

In Xcode built an iOS application using Core ML framework in Swift SDK to integrate machine learning models into the app.

Challenges we ran into

Compile error when implementing the framework.

Accomplishments that we're proud of

Was able to classify five different types of flowers and deploy a fully functional app.

What we learned

About to train a machine learning model and how to implement Core ML in Swift.

What's next for Flower Power

Hopefully, we get to add more plants and flowers so people can enjoy and learn more about plants.
