Inspiration

Our inspiration came from solving our problem of never knowing what our friends or classmates were up to.

What it does

Agenda Blenda takes two or more course schedules from college students and overlaps them to determine the shared class, study, and available time.

How we built it

We built each page of the app in Android Studio and created a server with Firebase, which is how we connected all our schedules. The website for the app was created with HTML and CSS.

Challenges we ran into

It was our first time using Firestore, Firebase, and Android Studio.

Accomplishments that we're proud of

We managed to launch the login.

What we learned

We learned that we should choose a simpler idea next time and build on that if time permits.

What's next for Agenda Blenda

We're hoping we can expand Agenda Blenda to incorporate schedules for just the user and between roommates as well.
