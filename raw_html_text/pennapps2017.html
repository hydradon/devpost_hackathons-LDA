What it does

For students at UPenn, they can just open the app, press a button, and a security escort will on their way to pick up the student. For security escorts, they can start the app and see a list of escort requests, and confirm the request that is closest to them.

How we built it

Front end using react-native and back end with Python using simpleHTTPServer with our custom protocols.
