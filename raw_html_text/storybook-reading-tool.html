Inspiration

We are inspired by the youth of today and want to assist in their reading development.

What it does

Users can repeat on-screen words into the microphone and receive feedback on their pronunciation.

How I built it

We built this project using Java, Eclipse, and the Microsoft Speech API.

Challenges I ran into

We ran into challenges using GitHub, downloading the Speech API, and using graphics.

Accomplishments that I'm proud of

Successfully implementing the Microsoft Speech API, implementing graphics, and collaborating on GitHub.

What I learned

We learned how to use Microsoft APIs, graphics, cognitive services, and the power of teamwork.

What's next for Storybook Reading Tool

Update graphics, worldwide deployment, update efficiency.
