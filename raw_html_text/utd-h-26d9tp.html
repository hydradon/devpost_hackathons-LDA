Inspiration

We realized that on a college student's list of priorities, taking care of your health is the last one. With busy schedules and back to back classes, college students deteriorate their health very quickly.

What it does

We created U+D H to help solve this problem. This program consolidates all health-related information on campus onto a website to help students when they get sick. Depending on the season and the campus, our "going on now" tab gives an idea to the students about what to watch out for. Additionally, emergency contacts for the ERs closest to campus are stated. We also started on functionality for a student to log in and track their health using the Google Calendar API.

How we built it

We built this application using angular.js and angular material.

Challenges we ran into

We ran into challenges when implementing the GoodRX API to find the cheapest prices on prescriptions because GoodRX did not approve the request for a Client ID by the end of the hackathon.

Accomplishments that we're proud of

We are proud of our UI.

What we learned

We all learned Angular or how to better navigate Angular to create a fully functioning application.

What's next for UTD H

It is our plan to get U+D H to implement a cloud database for all of the people that opt into our program. We would like to get authenticated to use the GoodRX API to find the cheapest prices for over the counter and prescription medication in the area.
