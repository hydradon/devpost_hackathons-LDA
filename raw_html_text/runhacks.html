Inspiration

Hackathons are really difficult to run smoothly, all the way from check-in to submission. RunHacks helps organizers by introducing several helpful services to help make sure the logistical procedure of a hackathon is as pleasant as can be.

What it does

RunHacks offers a way for organizers to see all their participants, an integrated thread forum system, a push message system to get information out to all participants, and a sign in system for the participants and volunteers.

How it's built

This is a web app using Angular.js for the front end and mongoDB and node.js for the backend.

What was learned

These environments were new for most of our team, so picking up extra frameworks and libraries, especially widely used ones is always a nice skill.

What's next for RunHacks

General polishing and security updates.
