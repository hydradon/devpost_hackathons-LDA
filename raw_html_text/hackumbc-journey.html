hackUMBC-journey (Now, hackUMBadChoices)

VR game project created in 24 hours for the UMBC hackathon. We came up with an idea to develop a Web VR game with multi platform integration. It runs in any modern browser (IE, I'm looking at you) and you can make it run in your VR headset as well, including Google Cardboard, Oculus Rift, HTC Vive, you name it! The domain is set up at domain.com: umbadchoices.com
