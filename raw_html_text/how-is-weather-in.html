Inspiration

I was trying to go to hackrice last weekend, but could not go there since it was flooding then. I was unsure about the road conditions on the way since I was driving on my own, and I did not want to take chance. So if there was something from where I could see most recent pictures of road conditions and live information, it would have made it easier to decide.

What it does

It gathers weather information (photos, description) from crowd and displays it if someone is trying to know weather condition of that location. It is not fully built though. Since we were only two and arrived late, we could only work on backend.

How we built it

We built it using Node.js, express.js, mysql, aws-s3, aws-rdbms, html, css.

Challenges we ran into

Figuring out how to store uploaded images Integrating backend and frontend since we tried to work with totally new technology for us. We tried to work on react-native first, but could not figure out.

Accomplishments that we're proud of

The idea itself. Although we could not implement react native, we learnt a lot about how it works. Could implement the idea on backend.

What we learned

Learnt about AWS-s3, react native. Learnt how backend works.

What's next for how-is-weather-in

Built a nice front-end for mobile app and integrate it with backend and expand app with several other functionalities.
