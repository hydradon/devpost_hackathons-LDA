Inspiration

A more specific tool for getting accurate computer science related material through a text messaging service.

What it does

Ask StudyBuddy conceptual computer science questions by sending a text message from any phone and receiving a text response answering the question.

How we built it

Node.js using express, Twilio API, Watson Natural Language Classifier, Retrieve and Rank and Watson Dialog service.

Challenges we ran into

Connecting the Natural Language Classifier with Dialog service. Document creation and structure for best results.

Accomplishments that we are proud of

Connecting all the services together, making StudyBuddy live.

What we learned

node.js, Bluemix, Natural Language Classifier, Retrieve and Rank, Twilio API, and Watson Dialog service.

What's next for StudyBuddy

Curating StudyBuddy dialog documents, student account, student feedback system.
