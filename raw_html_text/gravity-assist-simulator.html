Inspiration

Interested in exploring interactions between spaceships and planet gravity and how this can be automated.

What it does

We have created an environment for training and testing different reinforcement learning models to learn what models are best at space navigation.

How we built it

We used openai gym to create a new module. In addition we also used pyglet to render the current state of the environment.
