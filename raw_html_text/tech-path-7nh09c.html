Inspiration

We find lots of people interested in building softwares but don't really know how to go about it and the various career path that is available in the IT world, so we thought of building a solution that will help solve that problem.

What it does

Tech path enables beginners to get started in learning programming by pointing them to online resources to help them get started in their career.

How we built it

Tech path is built by integrating watson conversation to the messenger platform using the botskit framework.

Challenges we ran into

We spent lots of time sketching the conversation flow to enable the Chatbot respond intelligentlyk9.

Accomplishments that we're proud of

We are excited to integrating watson conversation to our chatbot, and hope it will put beginners to the right path in the Tech ecosystem.

What we learned

We learned how to collaborate as a team towards achieving every tasks given on this project. we have also learnt how to build a chatbot.

What's next for Tech Path

We look towards adding more features like images buttons and image gallery to the chatbot for a better user experience.
