Inspiration

Hiring is often an arduous task for tech companies, and Acorn Talent aims to simplify the recruiting process by prescreening applicants through multiple choice testing.

What it does

Businesses: Ability to create questions, tests and view the leaderboard for their tests. Users: Complete tests, receive notifications from businesses

How I built it

Express and Node on the backend, using pug as a template engine to speed up the creation process.

Challenges I ran into

Building the forms, creating an authorization function to determine whether a user is a business, among others.

Accomplishments that I'm proud of

Created a (semi) working product in 3 hours!

What I learned

Build fast.

What's next for Acorn Talent

Short answer questions and code submission
