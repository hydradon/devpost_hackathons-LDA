Inspiration

Wanting to learn how to use API.

What it does

Using CloudVision to search for websites of a pic uploaded to the app, and directs the user to that website so they can buy it.

How we built it

Through android studio.

Challenges we ran into

Google API not being compatible with other app development platforms we tried using.

Accomplishments that we're proud of

Using the API so that it's functional.

What we learned

How to use API, program more in android-studio, and the limitations of different app development platforms with small communities of people using them.

What's next for Picturio

Making it more compatible with social media, rather than just the phone's gallery.
