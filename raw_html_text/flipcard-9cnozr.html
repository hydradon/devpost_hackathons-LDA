Inspiration

Quizup

What it does

FlipCard allows instructors to make quiz/test style multiple choice questions called 'Cards' and store them in a 'Deck'. He assigns the deck to a classroom where he hosts tournament style competitions. The tournament

How I built it

We used Google Cloud Firestore noSQL database to store all objects. Google Firebase for user authentication and management. express.js for the server and react for the frontend. Github for source version control. Google App Engine to host the entire project.

Challenges I ran into

Our team had no previous experience using a noSQL database and struggled overcoming relational database design thinking.

Accomplishments that I'm proud of

What I learned

Google Cloud Firestore & noSQL, Firebase Authentication management, and Google App Engine.
