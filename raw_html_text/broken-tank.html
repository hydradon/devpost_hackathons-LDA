Inspiration

Simple, easy to use multiplayer experiences that don't require much thought and can be enjoyed by many people.

What it does

Pits players via online against each other in small planets trying to destroy each other.

How I built it

It's built on Unity3D 5.2 and JSON

Challenges I ran into

Getting the players to stick to the planets correctly using physics and trying to manage the server side data.

Accomplishments that I'm proud of

Getting proper game functionality on mobile device and server interaction functional.

What I learned

Multiplayer aspect of a game, server side management.

What's next for Broken Tank

Enhance multiplayer aspect, multiple sessions. Improve stability and server connection with Android devices. Rework physics and user input.
