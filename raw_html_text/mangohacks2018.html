Inspiration

I was inspired by some of the board games that I play, with interesting world-building mechanics.

What it does

It's a dungeon-crawling action board game with RPG elements, set in a haunted house

How we built it

This is built entirely in java, using no libraries.

Challenges we ran into

The hardest part of writing this program was probably the rotate method, where we had to understand how to calculate a given point in radians

Accomplishments that we're proud of

Actually getting the graphical interface to work

What we learned

We learned how to manage a medium-sized project through Git and delegate tasks to the team effectively.
