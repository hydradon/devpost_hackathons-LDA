Inspiration

We had inspiration from Waze as well as a desire to utilize a public crime data-set.

What it does

Computes criminal analytics for the street you walk an to see how unsafe it is.

How we built it

We used Java, XML, Python, and Android Studio to construct our mobile application.

Challenges we ran into

We originally wanted to create a neural net that could generally predict/estimate the likelihood of a crime occurring given that we knew the infrastructure present as well as the time of day. However, we ran into the issue of not finding accurate resources. We also didn't find crime data sets that showed when crime didn't occur.

Accomplishments that we're proud of

We are proud that we all picked up android Studio within a day.

What we learned

We learned a lot from the event such as Android Studio, Learning to use google cloud platform for hosting methods/programs we want accessible from anywhere.

What's next for NaviSafe

We will expand our app, polish it of then put it on app store.
