Inspiration

Music and urban geeks want to somehow use the city to visualize music and help people connect with each other through music.

What it does

Using an android app, we analyze music using the Spotify API which is then mapped on the screen using MapBox.gl. The data is stored on Firebase real time which is then sent to all the users connected to the service.

What we learned

Planning ahead helps.
