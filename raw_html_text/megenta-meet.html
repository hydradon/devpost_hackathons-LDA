Inspiration

Personal experience. We have a lot of days where time is wasted through meetings and we feel that this is a real problems that needs solving.

What it does

We have a console to display all available meeting rooms to quickly decide if a meeting room is free. We use a rasperry pi to detect if there is a person in the room using Amazon Recognition and update the console.

How we built it

Pi cam interfacing with angular application using firebase.

Challenges I ran into

Tmobile network ssh.

Accomplishments that I'm proud of

Flushed out UI

What I learned

Firebase and dynamic driven application

What's next for Magenta Meet

Meeting sign ups. Expand tracking to parking.
