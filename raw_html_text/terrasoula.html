Inspiration

The need for an easy solution to plan and layout ground mounted PV in complicated terrain.

What it does

Takes random ground terrain and compares it to a normal PV installation on flat ground. It calulates the adjustment required to avoid shading and outputs the new coordinates.

How I built it

MS Excel.

Challenges I ran into

Programming with MS Excel, No ability to get 3D Geoslam Data, and remove a 2D Slice.

Accomplishments that I'm proud of

The simple prototype works for varied user imputs and gives adjustments in x,y form.

What I learned

Teamwork. There is a need for this technology.

What's next for TerraSoula

Contact software developers, search for funding.
