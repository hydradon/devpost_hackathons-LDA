Inspiration 

 These days kids are more attached to devices instead of just games this skill enhances their knowledge while having fun with which they will be motivated and they will come back to the skill to learn again.

All parents dreams are to help kids in education and help them to get knowledge. And this skill is going to help both parents and kids. Making life better and to make next generations future bright.

What it does

Alexa game that teaches kids state capitals, state nicknames, country capitals while giving a fun riddle or joke for every right answer.

How we built it

Alexa Skills Kit: For the voice interaction, intent schema, slots and utterances. AWS Lambda: Node.js backend to handle the interactions, utterences. AWS S3: For voice file.

Challenges we ran into 

  Alexa plain text to SSML. Plain text format couldn't play voice file.
        Voice file has to be at a particular format otherwise it won't play on echo dot.
  Format of JSON for for the questions.
  Format of the JSON file for the fun riddles, jokes
  Testing

Accomplishments that we're proud of 

  Accomplished a goal of helping kids to learn while having fun giving them some fun jokes, riddles to keep their brains sharp, focused and engaged.

What we learned 

  New SSML tags.
        Lambda
  S3
