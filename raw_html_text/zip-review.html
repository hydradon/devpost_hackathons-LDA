Inspiration

We want to give companies the ability to understand customer feedback in order to improve their product and features.

What it does

Users can click a social media platform, which will redirect them to a search interface. From here they can type in keywords such as the product or features, and the correspondence feedback.

How we built it

We used HTML and CSS to build this, as well as Google's Customized Search Engine.

Challenges we ran into

The challenges we ran into were creating a domain name, and learning how to make a website from scratch.

Accomplishments that we're proud of

Creating a working website in a limited amount of time.

What we learned

We learned how to use HTML and CSS, as well as Google's Customized Search Engine.

What's next for Zip Review

We want to add the feature that allows users to filter search results, for example, they can search only for bad reviews.
