Inspiration

I wanted to have an easier time finding what courses I should sign up for so that I could complete all of my requirements.

What it does

It ranks classes that fulfill requirements based on how many other requirements it also fulfills.

How we built it

We parsed several text files off of Rutger's websites which we compiled into a database which eventually lead to a website.

Challenges we ran into

The text files on rutgers were not very consistent and there were constant issues with our cloud9.io development environment.

Accomplishments that we're proud of

Finished this project despite spending 5 hours on one weird bug.

What we learned

Flask is a very finicky framework - especially on cloud 9.

What's next for Course Helper

Improving the website and ranking algorithm.
