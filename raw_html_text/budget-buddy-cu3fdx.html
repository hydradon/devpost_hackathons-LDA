Inspiration

A few of us are particularly bad at managing money and need this extra help

What it does

Budget Buddy is a tool that will help users curb their bad spending habits by showing them how much their expenses can add up to over a longer period of time

How we built it

We used html, css, javascript, and bootstrap to create this

Challenges we ran into

Figuring out what to do about the backend and if we should use a database

Accomplishments that we're proud of

Submitting this project

What we learned

A lot of people spend a lot more than they should

What's next for Budget Buddy

We need to figure out how to make the program even more useful by adding tips to the tips section
