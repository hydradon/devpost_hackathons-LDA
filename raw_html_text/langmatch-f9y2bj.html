Inspiration

We wanted to provide a self help application for learning new languages but try to go about it a different way. We were very interested in getting more familiar with GCP and wanted to explore what they had to offer.

What it does

Interactive language learning app using Google Vision to allow user to take pictures and retrieve labels which user can match to objects in picture to learn how to identify them in another language

How I built it

We used React Native to run the app and used Node.js and SQL for the backend as well as talking with the Google Vision API and Translate API

Challenges I ran into

First time using Google Vision API

Accomplishments that I'm proud of

Getting more experience with React Native and GCP

What I learned

More knowledge of what GCP APIs can offer and Node.js
