Inspiration

Kids are really engaged in alexa now a days, hence i thought to develop a skill on Maths quiz so kids can test their basic maths knowledge.

What it does

A Simple maths quiz , which ask you 20 questions, people using alexa with display can really enjoy and answer the questions. this skill uses touch display to select the answer as well as voice .

How I built it

Build With Node js language as server side in aws lambda as back end. and alexa apl to display the questions and answer in a button format design .

Challenges I ran into

Really it takes hard times to design the front end to give the best and right design to alexa users.

Accomplishments that I'm proud of

Really happy to part of this hackathon to submit a skill on APL.

What I learned

Learned Alexa APL design properly will be releasing more APL in india.

What's next for Maths Quiz

An Update will be given with more functionality for testing their knowledge and engage customers.
