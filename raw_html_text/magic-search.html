Inspiration

Limited search functionality in Fusion 360.

What it does

Magic Search allows you to search objects by name and type, and select and deselect objects from the search results. Supports component, body, sketch, joint, rigid body and feature object types.

How I built it

Magic Search is an add-in written in JavaScript using the Fusion 360 API.

Challenges I ran into

Learning the intricacies of the Fusion 360 API in order to build the add-in UI and logic.

Accomplishments that I'm proud of

Being able to complete the project on time.

What I learned

Writing a useful application in JavaScript.
