Inspiration

  • security is so important for all people!

What it does

  • recognizes faces and unlocks door if correct; locks door and notifies owner if unrecognized person attempts to unlock.
  • makes it easy to unlock door even if hands are full with bags/groceries

Accomplishments that we're proud of

  • it is very secure
  • utilized Twilio API for phone alerts

What's next for smartdoor_impenetrable

  • sky's the limit
  • train with larger image sets to increase confidence scores
  • incorporate hardware (currently just simulating opening/locking door with animations)

Built With

  • opencv
  • python
  • twilio
