Inspiration

Ability to have transaction in pennies without high fees through cryptocurrency enables anyone charge premium for specific content like blogs, coding sites like leetcode etc.

What it does

Premiumify is javascript SDK which allows anyone to make any content on web go behind a paywall using cryptocurrency api by Hedera Hashgraph.

How we built it

Design -> Framework -> Code -> Premiumify

Challenges we ran into

Hedera is extremely new and unstable right now. Most of the issues faced were around Hedera Go SDK. Spent hours making hedera work in google app-engine with no avail. Spent hours making hedera work in Go CGI app with no avail. Finally worked with python cgi calling go code from system commands which is extremely bad hack which should improve as Hedera matures.

Accomplishments that we're proud of

We were able to hack first prototype!!

What we learned

Working with new product comes with its own challenges

What's next for Premiuimify

Make it legit by having full-fledged cloud service where content creators can keep track of everything premium and analyze what works best
