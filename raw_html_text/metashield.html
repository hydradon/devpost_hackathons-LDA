Inspiration

We wanted to find a way to improve the foster care system using technology.

What it does

MetaShield acts as a database solution for foster care systems, providing up to date information from a centralized source, about foster homes, foster children, and foster agencies.

How we built it

MetaShield, in its final form, was built using PHP, MySQL, HTML5, CSS3, JavaScript/JQuery, with the assistance of the Bootstrap front-end framework.

Challenges we ran into

Originally, MetaShield was going to be a blockchain project, however we ran into several problems which prevented us from completing the project on schedule. Therefore, we diverted to a database solution instead.

Accomplishments that we're proud of

We completed the project even though it was not the way we intended.

What we learned

Despite the fact we did not go with blockchain, we gained lots of useful knowledge in blockchain in our efforts to implement it into the project.

What's next for MetaShield

A few aspects of MetaShield still need to be implmented. For example, MetaShield currently lacks a login system, and a special interface for children. With more time, the user interface would be improved on as well.
