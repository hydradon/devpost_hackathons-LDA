Inspiration

The traditional route of learning a new language is in a boring classroom that is filled with students. The LangBot promotes one-on-one learning and a less intimidating environment. LangBot gives opportunity to create more relationships with others building a stronger social community and environment where everyone can better understand each other.

What it does

LangBot is a skill made through Amazon Alexa which helps users master the pronunciation and conversation of many foreign languages. The user can ask the LangBot to translate any phrase in the language of their choice.

How we built it

We used the Amazon Alexa skill kit to build the program and implemented the Google Translation API through RapidAPI.

Challenges we ran into

  • getting the API to work with the code
  • learning how to manipulate Alexa
  • programming with javascript and nodeJS

Accomplishments that we're proud of

  • Successfully applying an API
  • Understanding how Alexa and creating a new skill works

What we learned

  • Programming and debugging in new languages.
  • How to work efficiently under a high stress environment

What's next for LangBot

We are working on giving LangBot the ability to have conversations with its users in any language. This allows the user to develop in conversational skills. For the future, we plan on integrating a quiz game that allows the users to challenge themselves and what they've learned so far.
