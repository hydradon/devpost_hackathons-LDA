Inspiration

We wanted to be able to solve the problem of accessibility to financial technology, plus personal stories of developers.

What it does

Allows users to input their information (income + expenditures), and gives them recommended amounts of money they can spend for that month, based on percentages that they want to save.

How we built it

Using C++

Challenges we ran into

Implementation with storing the data, preserving information and calling that information to print out a report.

Accomplishments that we're proud of

Building a "database" to store all of the user data

What we learned

There always multiple iterations of a project (design process), adversity is necessary to arrive to a good solution!

What's next for BAA: Budgetary Analytical Advisor

Implement a full chatbot, finishing translations so it's more accessible to people of different countries who speak different languages.
