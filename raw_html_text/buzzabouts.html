Inspiration: we love Buzz, to the point where we wanted to know when we were close to him on campus. To solve this, we created BuzzAbouts, so that people were able to crowd-source where Buzz was.

l

What it does: BuzzAbouts adds pins to a map, wherever the user touches on the map, and each pin represents a Buzz sighting.

How I built it: android studio, Google Cloud

Challenges I ran into: UI/UX, creating a database

Accomplishments that I'm proud of: correctly using an API for the first time

What I learned how to: use an API

What's next for BuzzAbouts: the backburner
