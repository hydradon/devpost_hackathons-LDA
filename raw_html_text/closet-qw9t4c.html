What it does

This app gives you a chance to enter where you are going for that day and upload various items of your closet to the app. Then, based on this information, it recommends an outfit for you to wear. It also gives you chances to change certain items or generate a new outfit.

How we built it

We used Android-Studio to program our app. It uses java and xml.

Challenges we ran into

The biggest challenge we ran into was figuring out how to combine all our code into one big code that runs properly. Our biggest accomplishments were working with languages we weren’t familiar with.
