Inspiration

Have you ever think about how to visit cities in a different ways? So now you can do that while having fun!

What it does

Simple, it's only an app to play while you are make tourism somewhere trying to know the environment.

How we built it

We used android studio to develop most of the project with the different tools that it provides. After that, we used another technologies like node.js and API calls.

Challenges we ran into

We needed to use API's that we never used before and get a different qualities such as learn a lot of git (and crying with the fail merges)

Accomplishments that we're proud of

This is our first hackaton and we could finalize our basic purposes, make an app :)

What we learned

A few things about API's for logins in android, how to create and improve a server with an API REST using nodejs (but we didn't had enough time for use it correctly) and also we improved our abilities with android and git.

What's next for the project

We are thinking about going to another way, we learned many things but it is better to create some different stuff like hardware (like drones).
