Inspiration

The inspiration for this idea came from the dating app Tinder. Tinder allows you to find people near you that you can date. So, we thought, why not make something that allows you to find gamers near you with whom you can team up with.

What it does

Exouzia stores profiles with information such as the user's game of choice, location and level of interest. Then it takes these things and finds gamers closest to the user and matches them up.

How We built it

We wanted to build a website where people can go, login and get matched up. So, we used HTML and CSS along with PHP and JavaScript.

Challenges I ran into

The most challenging thing was learning PHP and JavaScript and implementing them right away to make a functional website.

Accomplishments that we are proud of

We were able to build a functional proof of concept.

What I learned

PHP and JavaScript. And also that web development is really interesting!

What's next for Exouzia

  • Improving our searching algorithm

  • Improve the UI and make the website more interactive

  • Set up a notification email system for the user

  • Giving the users the ability to create events for get together and tournament
