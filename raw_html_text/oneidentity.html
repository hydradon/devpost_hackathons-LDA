Inspiration

Minos, one of the teammates, had to move to multiple homes (~3) in a time period of 2 months. Having to change addresses and billing information for all his subscriptions was more than just a hassle.

What it does

One Identity is a webservice that allows people who are "on-the-move" to change their addresses and billing information to all their subscriptions in ONE PLACE.

How we built it

Much was done in JQuery, Javascript. Bootstrap framework was utilized. HTML & CSS was used to create the demo pages. The Azure database was taken into account. Parse helped keep track of the account information.

Challenges we ran into

Weaving everything together

Accomplishments that we are proud of

Minos- Using Azure Database. Bright- Deeper understanding of JQuery Odile- Binding Javascript with HTML5 and CSS

What we learned

Learned to bring all our previous knowledges together.

What's next for OneIdentity

To fully furnish this service
