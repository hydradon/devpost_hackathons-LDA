Inspiration I brought a google cardboard, and we wondered if we could make a virtual reality game in game maker.

What it does It is an entertainment app that simulates parkour elements and speed running in virtual reality 3d environment.

How I built it Using game maker language to take device tilt and touch input to create a virtual reality game for google cardboard

Challenges I ran into A lack of access to the device accelorometer values. The website with all gml documentation was down

Accomplishments that I'm proud of We made virtual reality in Game Maker, which to most people would seem impossible or ludicrous

What I learned Given enough determination, you can make anything with any set of tools.

What's next for Pirate Dragon's Lair More difficulties, branching paths, interactive objects, and the google playstore
