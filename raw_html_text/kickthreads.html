Inspiration

Being a fan of hip-hop culture, I have been interested in sneakers for a long time. KickThreads allows sneakerheads to check if their outfit complements the sneaker colorway.

What it does

It uses IBM Watson's Visual Recognition to check if the outfit item like a t shirt has similar colors as the sneaker.

How I built it

I used custom classifiers in Watson to create different classes for the sneakers I own and used the iOS camera to pass the image to Watson and display the feedback on iPhone

Challenges I ran into

Collecting training data for Watson.

Accomplishments that I'm proud of

Creating an app which uses the IBM Watson framework

What I learned

I learned about machine learning and visual recognition. I also learned about complementary colors and recognizing them using Watson

What's next for KickThreads

Solidify the core visual recognition and pairing system.
