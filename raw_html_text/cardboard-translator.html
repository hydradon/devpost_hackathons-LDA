Imagine you are in a foreign country, and you can't read any of the signs. That's where Cardboard Translator comes in. Cardboard Translator is able to turn any English text into Spanish. Using Google Cardboard and an Android phone's camera, we can create an augmented reality provides the translations to your peripherals. Using Vuforia, we are able to recognize text in the real world. We can then process that text to provide it in Spanish. This is mostly a proof of concept, as it does not matter what languages you are translating between.
