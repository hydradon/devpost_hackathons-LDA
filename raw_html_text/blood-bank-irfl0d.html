Inspiration

Whenever I look to the Thalassemia patients I was thinking I just want to do something for that so those people give me motivations to build such an app in order to help them

What it does

In Blood Bank a user can register as blood donor or blood patient by entering some details on Arcgis map when people who need blood open an app so it will show all blood donor and blood patient on map within specific range when user click on that mark they can see donor or patient details they can easily filter the map based on donor and patient. It easily update the current location using and update the result accordingly

How we built it

we built it using the android studio and arcgis map and it use

Challenges we ran into

The tough part in that project is converting the x and y point in latitude and longitude to draw the exact location on map

Accomplishments that we're proud of

Integration of Arcgis and study of it

Working with Arcgis
