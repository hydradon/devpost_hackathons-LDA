Inspiration

Coming from parents who have immigrated to the US, we all struggled with them helping us with homework. Not because they were not smart, but because there was a language barrier from Spanish to English.

What it does

Our project translates English speech to multiple languages.

Our Vision

Kids can have their parents help them with homework by reading the problem aloud and Alexa will then translate. The parent(s)/guardian(s) can then help them with the problem at hand and not let language barriers hold them back.

How I built it

We utilized Rapid API (Google Translate) to build a skill for Amazon's Alexa.

Challenges I ran into

We ran into the challenge of integrating the Rapid API with Alexa. We would love to see Amazon improve their API integration support.

Accomplishments that I'm proud of

As rookie hackers, we are proud for coming here and trying our very best. This is a project that is near to our hearts and look forward to utilizing it.

What I learned

This was our first time using Alexa and APIs.

What's next for Alexa's Titan Translator

We would like to improve the translator by finding a different translator and Alexa's accent by integrating Polly.
