Inspiration

Seemed attainable for first year programmers, and has real world international impact!

What it does

Collects, stores, and displays data on Ghanian schools using Google Maps API

How we built it

Blood, tears, and sweat. Also, node.js for the backend server, java script and HTML5/CSS for the user interface.

Challenges we ran into

Learning how to build a server, and struggling with version control in GitHub.

Accomplishments that we're proud of

Development of server went really well for our first time implementing something of this scale! Very proud of making a functional web app in such a short amount of time that looks pretty good.

What we learned

Learned basic web programming very quickly, along with implementation of API's.

What's next for Ghana Get It

Login pages for data security, modify functions to update school information, and seperate web page for data dump.
