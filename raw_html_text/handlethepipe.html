Inspiration

This is a major problem for the water maintenance/supply industry workers. The current solutions all require hands which isn't ideal especially when you're trying to restrain a pipe in a ditch.

What it does

Provides a hands-free heads-up display for the dangerous field work of restraining water pipes under pressure. The alexa skill takes several parameters such as pressure, trench type, & pipe length to feed accurate restraint calculations to the worker while they are on the job.

How I built it

The game environment is built with unity & vuforia which listens to the webserver which is built in node.js which listens to commands from the Alexa skill hosted on AWS Lamda

Challenges I ran into

Alexa is still a tricky mistress and took up most of the project time.

Accomplishments that I'm proud of

Successfully performing get requests from the unity environment to retrieve data from a web server.

What I learned

More about Alexa & Unity then I ever would have thought before coming to this event.

What's next for handleThePipe

UI polishing and successful connections between all three entities or possibly two entities (removing the web middle man) & building the project to a suitable AR device that preferably has alexa built in.
