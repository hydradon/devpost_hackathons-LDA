Inspiration

So many forms of social media exist in the world, and there isn't a good way to access it all very easily. It takes way too much effort to add someone to all of your social media because there is so much out there.

What it does..

Kado allows users to connect all of their social media into one platform, and then exports a QR code. When users scan said QR code, a window pops up and users are able to follow or send friend requests to all different types of social media of their friends at once.

How We built it

Kado is built in Java using the Android SDK, as well as using a barcode api called zxing and several other apis for social media such as instagram, linkedin, facebook and twitter, and is built off of the Parse framework.

Challenges We ran into

Several challenges were run into including the lack of the ability of several APIs such as facebook and instagram to send friend requests through the API, which led to them having the undesirable effect of redirecting to Facebook rather than automatically sending the friend request. In the end we only managed to have enough time to implement the twitter API

Accomplishments that I'm proud of

Making a working product, even if it wasn't as robust as intended by the end of 36 hours

What We learned

How to use several new APIs in Java and more about Material Design and Android

What's next for Kado

Adding more support for more APIs as well as perhaps NFC support.
