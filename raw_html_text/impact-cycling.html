Inspiration

Our team wanted to do something to help the environment, and we love bicycles. The project was a natural fit.

What it does

Impact Cycling is a web app allows the user to keep track of the environmental and financial benefits of their biking habits.

How we built it

We built Impact Cycling using Python/Flask, SQLAlchemy, html, css, and JavaScript. We collaborated using GitHub.

Challenges we ran into

We encountered challenges in using common module versions across the team, configuring our AWS server, and learning the SQLAlchemy library.

Accomplishments that we're proud of

We managed to build a working multi-user web app with a database!

What we learned

We learned many skills that will help us with future projects!

What's next for Impact Cycling

We plan to make our database system more robust, improve some of the calculations done for the user, and adding more features! Some features we would like to add are automatic logging of cycling activity (via Pebble/phone location), and some gamification mechanics. We also want to grow our user base!
