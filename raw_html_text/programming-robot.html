Inspiration

We got inspired by our mother in law because she wanted to have a more affordable way to teach programming to children

What it does

It lets the user enter different movement he wants the robot to perform on a Android cellphone and sends these movements via bluetooth to Arduino

How we built it

We used C++ to build the Arduino program, a HC-06 to use bluetooth with the Arduino, motors to drive the robot and the circuit to control the motors. The application was designed with java on Android-Studio

Challenges we ran into

Bluetooth settings problems and power supply from the batteries problems

Accomplishments that we're proud of

The bluetooth is working perfectly and the robot works for most of the wanted functions

What we learned

Always says twice the time you think it is gonna take you to perform a task

What's next for Programming Robot

Cleaning the app, debugging the java code and tweeking the length of the signal to control the movements of the robot better
