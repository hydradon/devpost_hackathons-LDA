Inspiration

We spouted random ideas and then we thought of those educational games associated with Dora, but why not add a recording feature?

What it does

Allows users to practice Spanish while listening to actual pronunciations of the Spanish equivalent of the English word.

How we built it

We used Unity and C# and a lot of spaghetti code.

Challenges we ran into

  • Getting Claudia to avoid always using Unity's Update loop and writing wrapper code around her work to make it more flexible.
  • Not picking the tasks we actually wanted to do, which slowed us down.
  • Creating a flexible audio map to map multiple buttons to the same Audio clip within an hour.

Accomplishments that we're proud of

  • Created a quirky game

What we learned

  • Lots of things
  • Teaching Unity, System Design/Implementation, other ways of thinking (random, but methodical rather than only just methodical)

What's next for viacom-dora

  • Post your scores on Facebook
  • Develop a more flexible and robust Microphone plugin for Unity with custom Microphone drivers
