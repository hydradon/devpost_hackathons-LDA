Inspiration

We aimed at solving challenge #7 using Machine Learning and map data from Brno city, but in a way that can be applied to any city and not limited to Brno.

What it does

Our project analyzes the city POIs and compares different routes through the city to suggest the best places to build bike lanes from proposed development plan.

How we built it

We prototyped using Python and Jupyter and we used map data from Here API with confirmation through Google Search / Maps. More data sources can be added.

Challenges we ran into

Acquiring data was the hardest part, as not all cities provide the data we needed.

Accomplishments that we're proud of

It works

What's next for Nabla

Multiple things on the list: adding more data sources, improving the machine learning algorithm, improving the visualization of suggested bike routes
