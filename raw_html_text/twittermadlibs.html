Twittermadlibs allows you to create mad libs from any public Twitter account. You can see what a Kanye West, Donald Trump, or Elon Musk mad libs would look like.

You can use the bot by sending a tweet to it on Twitter. The tweet should be "@madlibbot1 @USERNAME" where USERNAME is the username of the user who's tweets you want the madlibs to be pulled from

This app uses Tweepy to pull tweets from the Twitter API. After cleaning the data, twittermadlibs sorts each list by parts of speech (i.e. singular nouns, comparative adjectives, etc). It then chooses words to place in the mad libs, and returns a fully completed mad libs.
