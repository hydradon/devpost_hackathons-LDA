Inspiration

We were inspired by the idea of making a mini adventure around the TAMU campus.

What it does

Our program takes in user input in the form of making a choice on what to do next and uses it to output the consequence of their choices.

How we built it

We built it using Glitch, which used the programming languages HTML, CSS, and JavaScript.

Challenges we ran into

We ran into the challenge of not being able to figure out how to use switch-case syntax. We also had trouble getting the buttons to work.

Accomplishments that we're proud of

We were able to finish the program. We are also proud of our counters that we got to function properly.

What we learned

Roshini learned how to code in Javascript and I learned how hard it was to make a text adventure game.

What's next for Aggieventure

The next thing for Aggiventure would be the addition of more choices and events the user can go through.
