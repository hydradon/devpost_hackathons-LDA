Inspiration

In today's high tech world, many people don't have time to volunteer and keep up with their fast-pace lifestyle at the same time.

What it does

Using the user's Google Calendar Account, the web-app finds free-time in the user's schedule and find the perfect volunteering opportunties nearby.

How I built it

It is built with HTML, CSS, JS, Bootstrap, and Google Calendar and Sign-in API

Challenges I ran into

We had trouble customizing the Google Calendar API

Accomplishments that I'm proud of

We were able to upload it to a host website.

What I learned

We learned how to use Google Calendar API
