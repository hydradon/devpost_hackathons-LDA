Inspiration

Pangolins are adorable, and they are also critically-endangered, so I was inspired to make a pangolin-themed environmentalism game for the social good prompt.

What it does

It's a website that has a basic minigame (where a pangolin dodges bullets with arrow keys) and some background information about pangolins.

How I built it

I built this with HTML, CSS and JavaScript.

Challenges I ran into

I had to self-learn a lot of JavaScript for this, and my code is not completely error-free...

Accomplishments that I'm proud of

I'm glad that it somewhat works!

What I learned

I learned (perhaps too much) JavaScript.

What's next for Protect the Pangolin!

I need to fix errors and add a scoring system.
