Inspiration

We wanted to build this application to better facilitate the process for people to connect to medical experts based on their pain history.

What it does

You can select the body parts that hurt you and it will record it in the database. This way, when it's been a while since you've been hurt and you now want to seek a doctor to see what's going on, it'll recommend it to you the correct doctor.

How we built it

Magic! We first decided on a framework. ReactJS was chosen as it 's fun and challenging! It is also one of the best frameworks for single page web applications. Additionally, a database was implemented using Firebase.

Challenges we ran into

None of us knew how to use ReactJs or Firebase. Most of the time was spent learning and discovering the way these frameworks

Accomplishments that we're proud of

There is a working website. The body parts can be clicked and hovered upon for further medical evaluation, followed by an appointment database that can be expanded to store medical history.

What we learned

We learned that sleep is very important. We learned ReactJS... and everything mentioned above! :D

What's next for BodyCheck.Tech

We would further clean up the design as none of us are very good at graphical user interface. Additionally, we would make it cohesive and user friendly.
