Inspiration

The local non-profit theater, Dreamwrights

What it does

It allows auditionees to apply to audition and then allows the theater to view and print those applications in real time

How I built it

Using premade plugins on WordPress

Challenges I ran into

I don't have very much experience with web design, so I wasn't able to custom build much.

Accomplishments that I'm proud of

The layout and the fact that is works

What I learned

How to use WordPress, and a little HTML

What's next for Provino, Auditions Made Simple

Connect it to a better hosting site, make it mobile friendly,and add the ability to add notes about an audition to an application.
