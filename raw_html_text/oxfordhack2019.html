• our FitBit app! yellow screen means there's warning nearby

OxfordHack2019

Every year, a lot of people die or hurt because of the disaster such as flood, fire, collapse of building, earthquake etc... Even in some cases, the body can not be found. So, we decided to make the application which can get the life signal and if the heart beat is too less or 0 then, can send the location information to save life or at least to find the body. It was our first time to use fitbit, so it was so hard but we learned a lot from it. The difficulty was that fitbit can not send data directly to the server. But, we can find the way to send some important data
