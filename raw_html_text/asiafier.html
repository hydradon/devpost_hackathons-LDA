Inspiration

You tired of people asking "What kind of Asian are you?" No? Well, do we have the tool for you!

What it does

Asiafier is a "revolutionary" new software that classifies images of East Asian people using a convolutional neural network!

How we built it

We built Asiafier in a python environment using tools from Google's Tensorflow library and Twilio's API. The neural network is built off of Google's Inception model.

What's next for Asiafier

We hope to expand the functionality and accuracy of our classifier. In the future, we hope to include more classifications than just East Asian ones.
