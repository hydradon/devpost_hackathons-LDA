Inspiration

We like the popular Bitcoin App "tippin.me" but don't like that it's custodial over funds. We wanted to build a version on Ethereum that's fully user-centric but also user-friendly.

What it does

Creates profiles for sending tips to people.

How we built it

Web3-react

Challenges we ran into

We gave up on our previous idea the night before submissions.. at 10pm.

Accomplishments that we're proud of

Persevering and submitting an interesting project, even when we had to abandon a project upon learning that many popular DeFi projects are not very conducive to open development at the smart-contract level.

What we learned

We learned a lot about DeFi lending from our original idea. Once we pivoted, we learned a lot about some of the more user-facing components of modern blockchain apps; how to build powerful dApps with React, how to interact with the Fortmatic wallet, and how to create simple fiat-crypto onboarding flows with Wyre.

And as always with any hackathon, we were reminded again about the importance of good documentation and developer tooling for projects that want developers to build on them.

What's next for hoardin.me

Front-end work and incorporation with some sybil-resistance options such as social media apis.
