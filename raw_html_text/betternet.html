What it does

When traveling, one can search for certain attractions depending on your choice, and once you find it, my google chrome extension can store several links of your plans for the next day/week.

How I built it

javascript and google maps API

Challenges I ran into

Accomplishments that I'm proud of

What I learned

how to use an API with javascript and how to build a google chrome extension

What's next for BetterNet

we plan on taking this web app to mobile devices and to move the google chrome extension to the mobile google chrome app store as well.
