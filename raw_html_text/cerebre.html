Inspiration

One of us got inspired to make this app after completing a "Wheel of Life" questionnaire at a workshop.

What it does

Cerebre tracks student's moods and activities everyday and extracts patterns to make recommendations in order to improve students moods.

How I built it

As an Android app using Android Developer Studio

Challenges I ran into

I had never made an Android app before, so everything was a challenge to some degree. Using external libraries and understanding documentation were the hardest part.

Accomplishments that I'm proud of

Albeit unfinished, we have a semi-complete product to show. Not having made an app ever before, this is quite an accomplishment.

What I learned

If you're persistent enough you'll get there.

What's next for Cerebre

A complete rewrite.
