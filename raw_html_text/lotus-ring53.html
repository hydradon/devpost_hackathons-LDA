Inspiration

How do we connect ourselves to the network of devices in our lives in an easy way?

What it does

Lotus lets you review all the information being collected about you in one easy way.

How we built it

We built a backend in Python that is able to easily access APIs and connected that to a Javascript frontend so that it can be easily accessed by any device.

Challenges we ran into

Running a server from the Hackathon without uploading the code to a hosting site.

Accomplishments that we're proud of

Working together with multiple languages and being able to connect them seamlessly.

What we learned

We had never used Javascript before and we had never connected two different programs running on two different devices.

What's next for Lotus

Expand the program to use better APIs and make it smarter.
