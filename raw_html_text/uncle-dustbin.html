How we built it

we wrote a line tracing program for the mbot in Python. The mbot would trace the lines and go to different rooms to collect trash.

mBot identifies a black track on a light colour surface to make mBot move along a track.

we use some engineering so that we can can mount and unmount the dustbin head so that it will be interchangeable.

The following introduces four situations where mBot is relative to the black line, which can be dealt with using the following strategies:

  1. if it is on the black line, it keeps moving forward;
  2. if it deviates from the black line, it turns left or right in time to keep itself on the line;
  3. if it is not on the black line, it turns back to find the black line.
