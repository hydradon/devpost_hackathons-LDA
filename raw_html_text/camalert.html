Inspiration

We heard about the challenge from Skymantics for tracking of people using a Flir stereoscopic camera and thought their ideas were very interesting.

What it does

We decided to try to use the camera for tracking people and give an alert when the person enters a certain area to warn them their fight is leaving soon and also send a message to the gate that the passenger will be at there soon. We used FracTel for the message alert.

How I built it

The program was built with python using a python server to receive information from the Flir camera and processing of the information from the camera.

Challenges I ran into

The data structure that the Flir camera used to send information was protobuf which is extremely difficult to work with, this was huge hurdle that dominated the majority of our time.

Accomplishments that I'm proud of

We were able to get information from the protobuf file and use it to track people.

What I learned

We learned about the Flir camera system and protobuf data structure also we learned.

What's next for Need More Coffee

Working on better processing and utilization of the information from the camera.
