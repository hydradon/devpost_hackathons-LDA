Inspiration

Having many bad experiences with waiters at restaurants that often spoil the customer's experience. We decided to fix this problem using technology and tools available to us such as google firebase.

What it does

It helps people order at restaurants and avoid error in ordering the wrong item. And provides customers and the kitchen with real-time information about the orders.

How we built it

We used firebase, javascript,html and css.

Challenges we ran into

The time was too little and we were too ambitious in our visions for this project. Also lack of experience was challenging.

Accomplishments that we're proud of

Making a working qr code data and successfully using firebase to get,upload and use data.

What we learned

Javascript,Html,CSS,Firebase,Problem-solving

What's next for RoboWaiter

Adding a real-time queue so that the user knows their place in the line. A an ETA so that the kitchen can update the user of the status of their order.We want to improve the user experience and add it to as many restaurants as we can.Add reccomendations based on customer preferences
