Inspiration

Seen many freshmen face problems while making schedule, which makes it difficult for them to choose the location for their classes

What it does

It lets the student know if he has enough time to get from one class to another easily

How we built it

It was built using python and google map's distance matrix api

Challenges we ran into

Understanding methods to access the api

Accomplishments that we're proud of

Learned to work with apis

What we learned

What's next for Warn the Walk

Integration of warn the walk with lionpath can significantly ease the scheduling process and make the student aware of the distance between buildings on campus.
