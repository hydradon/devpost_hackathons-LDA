Inspiration

HackMan is entirely inspired by HackRiddle 2019 itself. From the characters to the aesthetics, we tried to incorporate as much as we could into HackMan. Overcome obstacles, shoot code at the bugs that crash you, don't let your energy get drained, and make it all 24 hours.

What it does

HackMan is a simple sidescroller game that is surprisingly difficult and pleasing to the eye.

How we built it

We used Java and JavaFX. Sprites and other visual aspects of the game were hand-drawn in Procreate.

Challenges we ran into

The real challenge happened before the coding began: forming and agreeing on an idea in the first place. Rejecting beloved ideas like Is It an Avocado? (an app that tells you if it is an avocado). We also ran into some other challenges, such as animating the sprite, object interaction, and making quality graphics, but those weren't so bad.

Accomplishments that we're proud of

Finishing.

What we learned

We learned how to make sprite sheets, how to implement a hit box, and most importantly, how to love each other.

What's next for HackMan

We're not sure. We just want to sleep.
