Inspiration

What it does

Users can upload a picture of their pet and ideally their friend's pet. Then users can vote on which pet they think is cuter and view the results.

How we built it

We used Android Studio, Firebase, and Java.

Challenges we ran into

We ran into many challenges, many involving sending data to and receiving data from the cloud. This was the first time using Android studio or Java for all of us and so it was a learning experience.

Accomplishments that we're proud of

We are most proud of connecting our backend database located on Firebase to our front-end. There were many times that we felt like giving up so making the app itself work was a big accomplishment, as simple as it is.

What we learned

We learned how to make a simple Android application, connect it to the internet, and how to code in Java to some degree.

What's next for TopDog

Making the UI not ugly, adding user authentication so one person doesn't have to upload both pictures, fixing many bugs
