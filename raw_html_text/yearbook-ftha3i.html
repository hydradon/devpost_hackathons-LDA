Inspiration

One day, while eating the dining hall, something ticked in my brain, saying, "Wow, I really suck with names." So, I thought I would fix that problem, because I am a computer Scientist, it what we do. Then, it evolved to not only learning their name, but doing some good at the same time.

What it does

It allows the user to put compliments into a multitude of groups in an organized and efficient, but happy, way.

How I built it

I used Notepad++ and JScript and PHP to jump back and forth between buttons, giving them functionality by keeping a hidden variable in the background at all times, while erasing and replacing what is on the current screen.

Challenges I ran into

Making the PHP and JScript be able to talk to each other in a way that made sense took me hours to complete. Also, examples are hard.

Accomplishments that I'm proud of

I managed to get all the code working and it also looks really nice.

What I learned

I learned how to create functional buttons that move the website along.

What's next for Yearbook

The goal is for the user to eventually be able to put a name to a face, while giving them a nice compliment in the meantime.
