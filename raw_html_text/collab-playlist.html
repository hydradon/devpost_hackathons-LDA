Inspiration

being in the car and wanting to edit someone's playlist

What it does

allows a user to create a playlist and then share the access code with other people which allows them to add songs to the playlist.

How I built it

with Spotify SDK and Firebase

Challenges I ran into

using the Spotify SDK because all the documentation is written in Objective C

Accomplishments that I'm proud of

The app being approved by the apple App Store

What I learned

How to overcome obstacles with little documentation

What's next for Collab Playlist

adding more music features and editing features
