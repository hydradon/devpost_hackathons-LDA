What it does

Matching colorful blocks before you hit the wrong color. Play in 4 different levels. Game starts in easy mode. Amp it up with medium mode. Challenge your reflexes with hard mode. Test your skills to the max with INSANE mode. Each game gets progressively harder. Easy to pick up, but very hard to master.

How I built it

It was coded in Swift which was learned to create this application.

Accomplishments that I'm proud of

Addictive game play with functional advertisements in game. Keeping the game free for download.
