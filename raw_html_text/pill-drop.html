• This is the prototyped device

Inspiration

We want to help others who have a difficult time keeping track of their pills.

What it does

This device takes one pill from the hopper and dispenses it after pressing a button on the web app.

How we built it

We used 3D printed parts, an Arduino Uno with some servos and node.js along with java for the code.

Challenges we ran into

We had a tough time connecting it to the web so we have it plugged into a computer which then connects the device to the web.

Accomplishments that we're proud of

We got the project working with the very nice 3D prints.

What we learned

We learned how to come together as a team and create an awesome health hack

What's next for Pill Drop

We plan to continue working on this project with the integration of a fully functional mobile app and a new enclosed 3D printed housing for the mechanics of the device.
