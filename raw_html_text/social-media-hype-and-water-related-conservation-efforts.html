Inspiration

We knew we wanted to do something with JavaScript from the get-go. We all wanted to learn more about JavaScript and also wanted to fit the theme and use social media and other APIs. So we integrated all these ideas into one and came out with our project.

What it does

What our project is supposed to do is scrub Twitter take the coordinates through JavaScript and plot all of them on a map.

How we built it

We used JavaScript to host a local server and then used Mapbox to make a map and tried to plot points on it. We also used the Twitter API to scrub twitter for certain hashtags and got the GPS coordinates for plotting them on the map.

Challenges we ran into

Trying to plot multiple points on the map that wasn't the user location Getting approval for API keys in a timely manner Scaling issues for various size screens

Accomplishments that we're proud of

Learning JavaScript quickly Learning Firebase

What we learned

JavaScript Mapbox API usage Firebase

What's next for Social Media Hype and Water-Related Conservation Efforts

Adding functionality for plotting tweet locations, adding options to customize which tweet keywords are searched for.
