Inspiration

The inspiration for this project was to utilize the Johns Hopkins API's to create a course list. Students can then list the courses they are signed up for and contact other students in the same course to form study groups.

What it does

Helps students organize study groups and collaborate.

How I built it

Built using Node.js, Google Firebase, and Hopkins Hub API's

Challenges I ran into

Issues regarding parsing of data and data types from Firestore.

Accomplishments that I'm proud of

Beautiful UI, integration with JHU HUB. Course searching, Use of Database.

What I learned

Learned about databases, fetching of data, parsing of data, and integration into a single webapp.

What's next for StudyFire

Further work to enable full feature set for optimal simplicity and functionality
