Inspiration

We were inspired by our electromagnetism class, where we don't understand a single thing that's going on.

What it does

PHY9C teaches students physics (electric fields for now) in the most interactive way possible.

How we built it

We built it with JavaScript, WebGL and HTML5 Canvas.

Challenges we ran into

We had trouble understanding how the TouchPad worked, because there were no APIs or documentation available. Luckily, we had talented Synaptics engineers to help us.

Accomplishments that we're proud of

We're proud that we got to work with a cool piece of hardware.

What we learned

We learned about electromagnetism, WebGL, and that the GOP's color is red while the Democratic Party's color is blue.

What's next for PHY9C

We're going to include more simulations such as periodic motion, waves, and ideal gas motion so that students can learn those topics in a fun and interactive manner.

We're also probably going to make this thing less politically charged. No pun intended.
