Inspiration

Because of the unpredictable nature, power utility companies have trouble managing the power grid when there is a gap between power generation level and consumption level. Our team tried to solve the problem by utilizing smart home devices which can be managed automatically

What it does

When the power consumption doesn't match the power generation, Pother in power grid system create a request power adjustment through smart contract. Pother in smart home adjust the power consumption automatically and get compensated by Ether according to their contribution.

How we built it

Incorporate Smartthings API with solidity

Challenges we ran into

None of us is a good blockchain/solidity developer

Accomplishments that we're proud of

Proof of concept, exposing smart home functionality through custom developed API

What we learned

smart contracts are significantly limited in functionality compared to our expectations.

What's next for Pother: Smart Power Grid Management

Find blockchain/smart contract developer and create MVP that can attract investors
