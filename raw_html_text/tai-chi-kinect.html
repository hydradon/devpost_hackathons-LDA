Inspiration

Physical therapy

What it does

Allows the user to practice Tai Chi with the help of a robot instructor.

How we built it

We used the Microsoft Kinect to capture video data of us performing tai chi moves in order to train the classifier to recognize the individual motions. The scene was built with Unity terrains. An animated instructor was created by matching video data to animations.

Challenges we ran into

The Kinect is very finicky. Animations were also difficult because we have not really worked with them before. Getting the different technologies - Unity, Kinect, Myo to play nicely with each other.

Accomplishments that we're proud of

Live animation of humanoid model based on Kinect motion capture.

What we learned

Animations with Kinect.
