We set out to make an app that brought social media down to it's essence. MainGo harkens back to the likes of AOL instant messenger with user statuses as the main form of communication as a simple medium. Aside from this, the main feature is a map view that showcases your friends list as avatars overlaid on a google map. The avatars are surrounded by a color halo which corresponds with their status emotion. The Avatars on the Map can be clicked to reveal the users status. User status' can be viewed in a list form as well.
