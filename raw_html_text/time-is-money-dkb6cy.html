Inspiration

Originally an IOS app idea however due to limitations we decided to start with a web app

What it does

Mobile website that calculates how much money you make in a normal shift of variable hours. It also calculates how much state income tax you will pay based on what state you live in.

How we built it

We used Sublime and basic .html uploads to Google Chrome!

Challenges we ran into

This project included learning all new languages for both of us. We had a lot of fun and it really helped for both of us to help and encourage the other. Noah had never touched CSS before and I chose this specifically for the opportunity to get better at JavaScript.

Accomplishments that we're proud of

We learned SO MUCH from this project. We made our first ever web page and we think it looks pretty cool too! Its interface is minimalist and we managed to incorporate some JavaScript and CSS bells and whistles to make it look more professional.

What we learned

We learned HTML, CSS, GIT and Javascript. We also worked on our creativity and teamwork skills.

What's next for Time is Money

More features including: ->reading time money tracker (half done) ->mobile device friendly ->calculations based on graduated federal income tax and other deductions ->calculations based on input expenses that would estimate the hours of work needed to make ends meet ->user accounts, saved data
