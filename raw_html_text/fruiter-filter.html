Inspiration

Have you ever been the recipient of unwanted fruit photos? We've all received our share of unsolicited fruity images, whether they be bananas, cucumbers, or yes, even eggplants. We wanted to develop an answer to this problem, to keep the citrus out of our eyes and in the produce aisle.

What it does

We built three separate demos:

  • a chat program that warns you about incoming links;
  • a web browser extension that blurs fruitful photos;
  • an HTML generator.

How we built it

The chat client is written in Python using sockets; the browser extension is written in JavaScript; and the HTML generator is written with C++ and JavaScript. The three share a core that calls into the Clarifai API.
