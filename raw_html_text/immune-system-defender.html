Inspiration

Last year, we made Tower defense game, which was mostly a failure

What it does

A fully functional tower defense with graphics, three different types of towers and enemies, and game play elements such as money, waves, and increasing difficulty.

How we built it

We built the framework completely from the ground up, using the most basic form of graphics and creating our own engine. We used Java and the abstract window toolkit from the standard JDK.

Challenges we ran into

Synchronization with the different people in our team

Accomplishments that we're proud of

We created a pathfinding AI for the enemies we created the entire thing completely without any libraries or external software

What we learned

work distribution

What's next for Immune System Defender

More content, such as towers and enemies.
