Virtual Vomit

Inspiration

General interest in Virtual Reality

What it does

Tests the user's ability to perform simple tasks while withstanding exacerbated issues commonly experienced while using virtual reality.

How we built it

Using Unity for the development of the game and creating models in Solidworks and Blender

Challenges we ran into

The nocebo effect while testing the game. Knowing what the players will experience caused the effects to be lessened on the testing and development team.

Accomplishments that we're proud of

The aesthetic design in the game.

How I built it

Using AWS

Challenges I ran into

Accomplishments that I'm proud of

What I learned
