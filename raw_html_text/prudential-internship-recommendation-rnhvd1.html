Inspiration

As students we struggle with searching internships. This is the perfect tool to learn more just talking to Alexa.

What it does:

It provides information about Prudential Financial and lists their internship opportunities.

How we built it:

We used AWS Lambda to upload our lamda_function and connected it to the alexa development console.

Challenges we ran into:

developing the lamba_function program .

Accomplishments that we're proud of:

Getting to know more about AWS and their potential.

What we learned:

Alexa-skills-development and AWS.

What's next for Prudential Internship Recommendation:

Implement requests directly to the Prudential Website using HTTP or an API for the most up-to-date information. Build this model into a product available for other companies that are offering internships.
