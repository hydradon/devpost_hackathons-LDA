Inspiration

Trying to get into CMB350 - Foundations of Brewing

What it does

Checks if a course has spots, and if not, checks every half hour and emails the user when a spot opens up.

How I built it

A python script that checks the BB API, compares the number of members to the course limit

Challenges I ran into

Getting the script to run on our AWS server, so it doesn't actually work from our BB module, but the script itself does work.

Accomplishments that I'm proud of

Getting the notification system to work!

What's next for Course Availability Notifier

Getting BlackBoard to add a property to check a courses capacity in the API
