Inspiration: The usefulness of graphs and the ease of using graph databases

What it does: Allows a user to create a network of friends, friends of friends, etc. and easily find these people in the network. Specifically, it allows a user to find a person of a certain occupation in their network

How I built it: Using Python Flask, Neo4J, Cypher, Bootstrap

Challenges I ran into: Using the Python Neo4J package

Accomplishments that I'm proud of: Creating this web app individually

What I learned: How to use Flask and Bootstrap

What's next for Finding Connections: Building on it so that information can be updated also
