Inspiration

We were inspired by our old and hard to use school website and how hard it was to navigate and access data from it.

What it does

It takes all the information on the website and condenses it to make it easier to find.

How we built it

We used Swift and Java

Challenges we ran into

Finding information from our school's website. Also, adding all that data to the app.

Accomplishments that we're proud of

Gathering all the data we needed for the application and making the map that is scrollable and able to pan.

What we learned

How to make images scrollable and able to pan. Navigation side bar.

What's next for LAHS App

Talk with the school to see if any additional information could be added.
