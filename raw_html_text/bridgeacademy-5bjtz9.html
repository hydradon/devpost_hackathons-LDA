Inspiration

The lack of motivation among younger generations. Promoting tech for women

What it does

It motivates children to study and learn about different subjects related to tech and science, offering them coins which they can exchange for prizes, accessible tech for everyone! :)

How we built it

We've built it using Flask (Python), HTML, CSS and JSON.

Challenges we ran into

We'd never used Flask before, so that was challenging, but in the end we found it quite useful.

Accomplishments that we're proud of

Building an app with a language we never used before.

What we learned

Flask.
