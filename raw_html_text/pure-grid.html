Inspiration

Identify a low-cost cleaning solution for large numbers of tightly packed solar PV systems.

What it does

The application of protective films to protect solar panels from dust and the implementation of a centralised cleaning service

How I built it

The system require the installation of a roller for each row of solar panels and a set of clamps that keep the interchangeable plastic layer in place. The system has two repositories: one for clean sheets and another for used sheets. Once the repository for used sheets is full, used sheets are taken to an off-site location for cleaning.

Challenges I ran into

One of the main challenges was to determine if there is a material that is suitable to use it as an extra layer. The plastic need to have high light transmission and resist outdoor conditions.
