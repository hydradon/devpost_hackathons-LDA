This app is a "smart walking stick". It is designed to help those who are visually handicapped and do not want to carry a walking stick. It gives the blind user a 3-dimensional understanding of the world through plane detection and is able to calculate the distance and displacement from a marked position, all of which is controlled through Google's Speech To Text API. We used Microsoft Azure cloud to perform facial recognition based on an S3 bucket tied the native cloud's API. We also allow the user to move around and visually updates the 3-dimensional world in real-time to allow the user a visual and verbal aid.
