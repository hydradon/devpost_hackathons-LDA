Inspiration

My friend Blake recently discovered that dabbing was a thing. As such, he has been doing it... a lot. As such, I wanted to make a simple reminder for him to know just how many times people have seen him dab.

What it does

Every time someone on the site views the gif, it adds one to the counter.

How I built it

Domain.com links to a tamu personal server that has this page running.

Challenges I ran into

I didn't know about the differences between server-side and client-side variables, so I had to rewrite the counting code to be able to handle multiple people accessing the site and still give the same numbers.

Accomplishments that I'm proud of

I'm just happy that there is a published site

What I learned

The basics of php, ajax, html, and jquery.

What's next for Blake Dabs

Add the ability to count more than one user at once.
