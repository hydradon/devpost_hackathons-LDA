Inspiration

Chinese doctors always have no time to explain details to their patients about tests, exams and treatments, which makes patients confused. Since there is no standard medical information platform in China yet, it will be helpful if there is an easy access to medical information for Chinese patients.

What it does

Our app provides Chinese patients with easy access to accurate medical information.

How we built it

Using online-OCR tools to extract key words from diagnosis sheets. Using AC auto machine to match key words in dictionary. Translate medical information from English to Chinese with Google translater.

Challenges we ran into

We have difficulties in getting enough medical information.

What's next for Hospital Helper

This version is only a demo. We are going to implement it next.
