Inspiration

The excitement of the NPO and the need for teachers to be recognized and paid more than they are currently.

What it does

Provides teachers with the ability to post videos to a subject competition that are voted upon to receive monetary rewards.

How we built it

Using C# and Xamarin for the application building, using Flask and Python to build the necessary APIs, and using Airtable as our database.

Challenges we ran into

OAuth; Server communication; UI;

Accomplishments that we're proud of

We have a rough working demo. Color scheme of the app is consistent with the NPO's. Created 3 APIs.

What we learned

Do not spend too much time on one thing. It can hold you back from producing.

What's next for Million Dollar Teacher Project

Add payment options. Upload videos. Expand the user profiles to include more than just the teacher.
