Inspiration

Some of our team members like to play along on their guitars with their favorite music, but it can be a pain to keep pausing to search for tabs or chords.

What it does

Chordinate automatically finds guitar tabs and chords for the song you are currently listening to on Spotify, and it automatically updates as you change songs.

How we built it

The frontend is Angular, the backend is Express, and we interface with Ultimate Guitar and Spotify for data.

Challenges we ran into

Authenticating with Spotify took longer than we had hoped. We also didn't start working until 5-6 hours after the event started because we couldn't think of an idea.

Accomplishments that we're proud of

The tool works exactly as expected and is genuinely useful.

What we learned

We had worked with the Spotify API before, but this time we learned how to interface with their authentication.

What's next for Chordinate

We want to add a lot of basic features to make the tool more useful, and then we want to sync the chords with the music to make it even more useful.
