Inspiration

We wanted to find a way to improve safety through leveraging technology.

What it does

Operator+ automatically extracts valuable data points from a 911 call's dialogue, therefore reducing sources of human error by the 911 operator.

How we built it

We used a type of AI (Natural Language Processing) we analyze and extract keywords from a 911 call's dialogue.

Challenges we ran into

We found it challenging to work in a fast-paced environment under a time crunch. A technical challenge we faced was setting up the programming environment of this project.

Accomplishments that we're proud of

Building a fully fleshed out pitch.

What we learned

How to work more effectively as a team (delegating tasks, the importance of communication, etc.).

What's next for Operator+

We want to finalize the technical aspect of our project altogether during our reading week.
