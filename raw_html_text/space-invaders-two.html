Inspiration

This project was inspired by the games Missile Command and Downwell. The art style is my take on Downwell's monocolor sprite style and the player shoots down enemies much like in Missile Command.

What it does

It runs in the browser. It should provide entertainment for those who are curious for a few minutes at least. The game revolves around keeping city buildings standing and racking up combos.

How I built it

I built this with native Javascript and HTML5, using the canvas element to render the game.

Challenges I ran into

Simply thinking of an idea and following through with it was a major hurdle. A more technical challenge was figuring out how to scale the game with the browser window. The limited art style required a different approach to avoid blurry pixels and unclear graphics.

Accomplishments that I'm proud of

I built this within two days, when normally a project like this would take me weeks. Additionally, I learned about Javascript features that I never used before.

What I learned

I learned what it's like to participate in a hackathon, how to power through and manage my time, and that prototyping and testing are more important than finding the perfect idea.

What's next for Space Invaders Two

Given the motivation, I might expand on this idea and make it into a full-fledged game.
