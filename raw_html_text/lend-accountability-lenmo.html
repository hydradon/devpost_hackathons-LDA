Inspiration

Money lent to people but never returned back to the lender. Wanted to introduce an accountability system that makes lending less stressful.

What it does

Lend money, see people's borrowing history. Rating system that is available to all, so people can manage risk.

How we built it

We designed the UI with proto.io. We also ported one page to XCode.

Challenges we ran into

Implementing Hedera Hashgraph

Accomplishments that we're proud of

Functional & Clean UI, Great end product

What we learned

Collaboration, Making apps, prototyping, UX considerations, blockchains/DLTs

What's next for Lend Accountability (lenmo)

Port to IOS/Android. Add more functionalities. Make a web app.
