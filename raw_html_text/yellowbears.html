Inspiration

To help YB with a new interactive MatchCenter, so all the fans in the stadium, at home or on the road can participate and have a good time. With different games, bets where you can win MOHF Coins and more information about the players, it offers a lot to explore. Enjoy the games with a new perspective.

What it does

We developed a new part for the already existing MatchCenter. You can win MOHF Coins in Challenges, have a look at the fashion corner or play small games.

How we built it

We build the Frontend with Vue.js. For the styling we used Tailwind CSS. The backend is based on Node and we worked with REST API and XML. Connectivity between the Frontend and the Backend is done with Websockets. For the Database, we used a MySQL database.

Challenges we ran into

Working with the XML file that wasn't really what we needed. Problems with the Backend and the Database because we don't have that much experience with it.

Accomplishments that we're proud of

Even though we are a big team with 6 members, we worked really good together. We got a vision of what we wanted and what everyone will do. In the end, we got on a product, on which everyone contributed.

What we learned

New insights into WebSockets. Working with Tailwind. Node.
