Inspiration

I have a child and I see that it takes students a long time to get the data to solve the exercises. So I thought that with Alexa it could be made easier.

What it does

It gives you data about the elements in the periodic table

How I built it

I built it in javascript / node.js, using ask-sdk, apl and lambda functions.

Challenges I ran into

I had experience with ask-sdk, but apl has been a challenge, because it is something complete new, and I have to get the feeling.

Accomplishments that I'm proud of

I'm proud of getting the skill looks good in all the devices. It was very challenging.

What I learned

I have learned about apl and working with multiple devices.

What's next for Periodic Table

The next steps will be to add more data.
