Inspiration

We want to enhance learning/education experience while incentivize user to go into local businesses

What it does

Makes learning fun to encourage students to learn via Amazon-Alexa enabled devices. The question and answer would be dynamic. If they answered corrected, they would received a special coupon via SMS and direction to the business and therefore drive demand to local businesses.

How we built it

We used Amazon-Alexa, Nexmo, AWS (Lambda) coding in Node.js

Challenges we ran into

We ran into trouble when we initially build the Nexmo interface using Python while the Game is in Node.js. But that did not work well so we resort to refactored our code using NodeJS to NodeJS.

Accomplishments that we're proud of

We are proud of our first time hackathon’er / student, who learned a lot about AWS and Nexmo technologies.

What we learned

We had not incorporated SMS into our previous hackathon projects, and we took AWS to a new level.

What's next for Myth Quiz

We would expand the number of Quiz into different areas and interesting questions to allow richer experiences to excite students using Amazon-Alexa devices. At the same time make a process to allow easy business sign up to expand options.
