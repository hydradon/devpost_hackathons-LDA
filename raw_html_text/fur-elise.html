Inspiration

A visit to the blind community - based on the real needs of visually impaired

What it does

Converts the most common musical score format into speech, giving the blind access to an unprecedented variety of music.

How I built it

We used Angular with Javascript, Flask with Python, Google Cloud Service, IBM Watson, etc.

Challenges I ran into

There are a lot of very detailed rules in music scores, making it very hard to make a comprehensive speech converter.

Accomplishments that I'm proud of

  1. We're able to capture all the nuances in music scores
  2. UX of our website is specially optimized for the visually impaired; nonetheless, the UI is exquisitely designed.
  3. Many handy functions, customizable for the users.

What I learned

Web development, xml processing, HTTP requests, etc.

What's next for Fur Elise

Implement an automated pipeline to search for sheet music in the future
