Inspiration

We were inspired by the data visualizations shown on the webpages for the city budgets of Oakland, Philadelphia and Boston.

What it does

The project consists of an interactive Sankey diagram detailing the appropriation of budget for the 2015 fiscal year for the city of New Brunswick, New Jersey.

How I built it

We used the Google Data Visualization API to create the Sankey diagram, and experimented with various D3.js functions.

Challenges I ran into

It was hard to adjust the Google API, so that we could have more freedom over the Sankey diagram.

Accomplishments that I'm proud of

We managed to quickly create a functioning data visualization for a large amount of data, that greatly resembles the products that we were attempting to recreate.

What I learned

We learned that the proper choice of design and programming tools can reduce a difficult task into a more manageable one.

What's next for NB Financial Data Visualization

We plan on creating a database to hold various city expenditure statistics, and auto-generating the Sankey-diagrams for the future.
