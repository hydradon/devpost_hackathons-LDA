ProfHacks 2019 Project Contributors- Stephan Duroseau, Scott Stubbs, Jack Zheng

PhilathFund is a scalable proof of concept crowdsourcing application that allows users to connect with and aid people in need. From building homes for victims of natural disasters, to helping to preserve wildlife, PhilanthFund covers it all.

Technology Stack: -Amazon Web Services (Amplify, Cognito, DynamoDB, S3) -GraphQL -React.js + Node.js -HTML/CSS/Bootstrap -Stripe API

Features: -Secure login with hashed passwords and usernames using AWS cognito. -Secure card transactions utilizing Stripe. -Utilized a nonrelational database, DynamoDB. -GraphQL allowed for us to query our data. -Hosted website on cloud using AWS amplified. -Analytics on current investments and projects. -An augmented reality tour of the project.
