Inspiration

To help eager learners learn new languages by learning a word each day

What it does

Sends out a SMS/calls to the registered number(nexmo verified) with a random generated english word,Its meaning and its french/spanish translation

How I built it

Using Angular as the front end and nodejs as the back end server

Built With

  • nexmo
  • node.js
  • oxford-english-dictionary
