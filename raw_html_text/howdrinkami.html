How Drink am I?

What You Need to Know

Created for hackCWRU 2018 - We aim to provide a convenient solution to prevent unsafe levels of alcohol intake. Using a simple linear regression model, we can correlate the delay from a base reaction time with blood alcohol levels. The first time the user opens the app, they will be asked to complete a simple test that takes no longer than 30 seconds in order to establish a base reaction time. When a user accesses the app after drinking, they can take the test to estimate their blood alcohol concentration.

Getting Started

HowDrinkAmI can be accessed via mobile site (kclejeune.github.io/HowDrinkAmI) or android application.
