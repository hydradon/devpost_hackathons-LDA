What we plan to solve

We'd like to give people reasons to come to bars.

  1. User can see what's going on inside SmartBar quikcly.
  2. User can have more fun with other bar attendees and obtain various perks through augmented reality game called "AngryBuds".
  3. User can drink safely by setting limit easily.
  4. Bar owners can visualize consumption easily and collect consumer's drinking patterns.

How we'll solve

We will make augmented reality games. Although Angry Birds concept is used, the app can modify theme such as AngryQB, AngryGolf, AngryBaseball, AngryBowl, etc.

  1. A bar customer enters a SmartBar.
  2. Footfall sensor collects image with timestamp.
  3. The SmartBar recognizes user profile via AngryBuds app.
  4. AngryBuds app changes background scene that matches bar atmosphere through light sensor and sets up Pigs with current bar users.
  5. Customer purchases glass or bottle of Budweiser. POS will signal AngryBuds app.
  6. AngryBuds adds a Bud with SmartBar sensor information such as volume, temperature, pressure, etc. For example, A bud's color changes based on temperature.
  7. A dedicated bigscreen TV will start broadcasting the game.
  8. Bud flings to Pigs who are the current attendees at the bar.
  9. There can be a king Pig that user can pick as a future soul mate and ask for contact info in Cupid mode. User may choose king Pig as his rival in War mode. For example, Boston Redsox fan may use NY Yankees image.
  10. The crushed Pig can accept/reject the contact request. If the Pig revenges next with the Bud as target, contact information establishes automatically in Cupid mode.
  11. AngryBuds app keeps the record of drinks purchased including user's favorite beer.
  12. The app can set the limit in consumption and can call Uber if the user is completely drunken.

Our measurement of success

We measure success through adaption by end-users and bar owners.

  1. Read relevant sensor values using RESTful APIs.
  2. Visualize sensor values through gamification.
  3. Popularity in app store and number of downloads.
  4. People make a good connection through this app.
  5. Bar owners customize app to make their background more fun.
