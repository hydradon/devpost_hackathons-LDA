• Telegram Bot StartApp

Inspiration

Calendar applications have always been hard because you constantly have to remember to add events #popular people.

What it does

We have created a telegram bot to make this a seamless process by simply texting the bot and your events will be added to your calendar.

How we built it

Uses Google Calendar API and Natural Language Processing to add events into the calendar

Challenges we ran into

Oauth2 in telegram is a pain. You have been warned

Accomplishments that we're proud of

Solving the Oauth2 problem, and making it work

What we learnt

Time management and delegating work. Choice in music is extremely important ;)

What's next for CalendarBot

Improving NLP, checking for conflicting events, able to add recurring events
