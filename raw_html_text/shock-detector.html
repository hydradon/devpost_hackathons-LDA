Inspiration

Many packages that have been shipped don't reach their destination in a supposed state. So we wanted to make something to track that.

What it does

This shock detector detects the motion of the package and reports to the web server when it reaches a certain limit. It sends the date and time of the event. Then the package owner/carrier can get information about when the event actually happened.

How we built it

We used android studio.

Challenges we ran into

We have the web server which sends the info to cell phone of the user but it kept on crashing so there were problems connecting with the app.

Accomplishments that we're proud of

This was our first time of making an android app so we are proud that we at least did something.

What we learned

To use the tools like android studio.

What's next for Shock Detector

We will keep on working and see how it can better. We want to include the ability to send the data directly to the server of the carrier so that they could decide if they want to send the data to the user or keep it to themselves for research or so on.
