What it does

Your Grandfather's python reads python code written onto punchcards, and saves it to a server to be executed anywhere.

How we built it

We built Your Grandfather's Python using javascript, python, and opencv. The backend uses AWS Lambda functions.

Challenges we ran into

We ran into issues with converting the image of the punchcard into binary code, and sending the data from the phone taking the pictures to the server for processing.

Accomplishments that we're proud of

First attempt at computer vision.

What we learned

We learned the basics of feature detection in opencv, about python, and how to configure AWS Lambda functions.
