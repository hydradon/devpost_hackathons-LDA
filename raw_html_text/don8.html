Inspiration

Donations, contributing to the community. We wanted to give back to the community, but more so make it easy for the general public to give without having to spend much effort doing it.

What it does

An easy way to donate on a daily basis, small amounts with a big change, without impacting the customer's bank balance to a noticeable degree.

How I built it

The application is built using the latest API's available on Fusion Fabric Dot Cloud. We utilize a vast range of technologies such as API's, Machine Learning, Java, HTML, JavaScript, CSS, JQuery, AJAX and more.

Challenges I ran into

There were some issues getting one of the FFDC API's to work due to authentication issues, however, with the help of our expert team here in Finastra, we were able to solve the problem quickly.

Accomplishments that I'm proud of

In the short time we had, we were able to produce a fully working, functional and impressive looking application, that was only possible due to good management in the form of Agile working, Scrum and short Sprint deadlines, that utilized good teamwork and dedication.

What I learned

The Don8 team learned a great deal from this experience, for example, the various API's that FFDC has to offer.

What's next for Don8

There is so much more that can be done to take Don8 to the next level, this is not an application that just ends here, we have so many more ideas that can help the donation/charity industry.

About ML

To attract the user and show them the preferred organization that they may want to make donations to we will use an ML module:

The ML module inputs data such as the age of the user and the location, (all static data we take from the user when s/he signs up for our app). The ML algorithm will learn also from new users data and can fit his suggestion to the new users accordingly. The ML module is using a MultiOutputClassifier build on a RandomForestClassifier(all implemented on python with sklearn).
