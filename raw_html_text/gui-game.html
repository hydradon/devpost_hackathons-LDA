Authors • May Jue • Eleni Lazaridou
See also the list of contributors who participated in this project. Description of Nim The game of Nim is a two player strategical game. The game has a pile of a specific object. Each player take turns and remover a number of objects. The purpose of the game is for one player to get the last one. Getting Started The user plays against the computer in a Tkinter environment. The computer uses mathematical logic in order to win. The players should make wise decisions, as they have the benefit of always start first.
