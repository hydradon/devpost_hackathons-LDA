Inspiration

Everyone hates downloading new applications on their mobile-phones. We wanted to give users an opportunity to use information about their city without needing to download a new app.

What it does

Uses Reddit's /r/montreal subreddit as a data set to serve photos, news, and historical information. We also pull from an open weather API to get weather for your exact location. It presents all information in a messenger bot to the client.

How we built it

Our dependencies our PRAW (Python Reddit Wrapper), Messenger API, Open Weather, and Flask. We used Python to build the entire application.

Challenges we ran into

First time using gitHub as a full team of 4. We had no sleep.

Accomplishments that we're proud of

Getting the device location to properly work and send weather information as well as get a quick replies feature working.

What we learned

How to serve flask on a server, how to use python for a full stack connected with various APIs such as PRAW and the Messenger API and Open Weather API.

What's next for MontrealGO

MontrealGO becomes InternationalGO and we used what we have developed to build a generic framework where it can be accessed throughout the world.
