Inspiration

First of all, what inspired us was the story and the challenge itself . Gathering data and connecting it to clause was exciting as well as fun.

What it does

Our program gets the given data and deliver it in a more readable way. Thus, the user can be able to solve the case.

How we built it

First of all, we converted the data from json to cvs to be able to examine it better.

We used pandas library to sort through the data to see important aspects of the data.

We decided to use JavaScript to fetch the data and wrote a program which illustrates the key points.

We used React framework to build an user friendly UI/UX.

Challenges we ran into

The challenge we had was that we could not choose easily the language which we were going to use because at first we thought that using Python to analyze the given data is easier but realized that making user friendly UI/UX is not easy to achieve in Python so we decided to use React instead.

Accomplishments that we're proud of

In conclusion, we all believe that we have a pretty awesome project which we spent many hours to achieve. The team effort spent in this project was incredible. We had fun while building this program and continued building up our teamwork and creative skills.

What we learned

We realized the significance of the teamwork and creative problem solving skills in such a event.

What's next for DataEctive

In the future, we can work on our program to make it working for all given data instead of one single given data.
