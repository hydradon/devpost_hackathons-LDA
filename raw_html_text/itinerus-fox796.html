Inspiration

When a friend from San Francisco visited me for a day, I spent more than 2 hours trying to create a schedule that would highlight all the many things Puerto Rico has to offer.

What it does

It allows you to select the touristy places you want to go and generates an optimal itinerary of how to visit these.

How I built it

Using a Python Django backend and a JQuery frontend; we pull relevant venues from the ViewPR API, relate them to TripAdvisor listings, and using Leaftlet.js we expose these in an easy to use map-based view.

Challenges I ran into

Trip advisor data wasn't complete in ViewPR.

Accomplishments that I'm proud of

Implementing an efficient optimizing algorithm and having a map-view UI that is fast, snappy, and responsive.

What I learned

Developed even further our web development skills. Learned about Puerto Rico's tourism industry.

What's next for Itinerus

We would like to create more interface tools to filter the results from the API.
