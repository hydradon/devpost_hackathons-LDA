Inspiration

We were inspired by Snapchat's convenient follow protocol. This drove us to create a product that connects across all social media (its only twitter right now, but we've built the framework for other social media sites).

Challenges I ran into

Accomplishments that I'm proud of

We followed a proper git flow, mastered maven repository and its 3rd party plugins, we reaffirmed our knowledge of gradle and android-studio

What I learned

Proper implementation of 3rd party jars and using multiple maven plugins on a single application.

What's next for followUp

We hope to implement Facebook, Instagram, GitHub, Google+, and various other social media outlets in order to better connect our users.
