With limited tools and supplies how could we give back to the community?

Using an old repurposed Linear actuator Jaime had around from a flight simulator project, we found our vision. With a Linear actuator with 250 pounds of force, it could be possible to create a leg prosthetic that could modulate in length.

Slowly we built on the idea, and decided that this would need a dedicated website to promote it, along with some sensors to make this prosthetic really stand out. 2 Distance sensors and one gyro/accelerometer unit we had a game plan.

Through the power of endless console errors on chrome, and circuit redesigns we created our first iteration of WalkR, our low cost solution to amputees.

Visit the github repo and the live heroku website to get more info.
