The Inspiration

In the beginning of the hackathon, our team kept coming up with APIs with specific use-cases. We wanted to make an API that was 1. flexible enough to have a broad and powerful impact on a variety of applications and 2. something that we would actually use after the hackathon.

The Result

After much thought we settled on Perish, a way for users to set constraints on their files and have the files deleted after the conditions are met. The files are encrypted for security and hashed to ensure integrity. The users uploading and downloading files are completely anonymous and the files are gone forever after the uploader's specifications are met. This API is perfect for secure, safe file sharing and the range of applications for this API are limitless. We are happy with how the API turned out and hope that people will find it as useful as we do!
