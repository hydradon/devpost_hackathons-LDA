Inspiration

Our inspiration comes from pixel-arts, online games and having fun ;)

What it does

It is an online game in which you can draw your own Pixeling and interact with other Pixelings (players). Game chat and other features still need to be implemented.

How we built it

It was 3 am and we didn't know what to do, we were tired. Then suddenly we came up with an idea of making this game. We were very excited and started developing straight away.

Challenges we ran into

We needed to build an application from scratch and come up with basic game features. It took quite a while - that was the main challenge.

Accomplishments that we're proud of

We are proud of presenting our game in it's beta state, hosted on Google Cloud Platform. We believe that this game can deliver great fun to it's users. It turns out that it is a lot of fun to draw you own character and play around with it.

What we learned

  • We learned to work together more efficiently.
  • We learned that Java is sometimes easier to use as a server than nodejs.

What's next for Pixel Game

  • Add some game modes
  • Add a game chat
  • Improve overall design
  • Add the ability to leave notes for others
