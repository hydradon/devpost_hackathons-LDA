I consider this as another milestone in my career as a developer as I take the Alexa skill coding to a new level by using the Amazon Presentation language with C# .

This project brings this popular word game into a variety of Amazon echo devices creating great user experience by utilizing the new objects presented in APL that makes multi-modal interaction possible.

With the combination of Visual Studio IDE and C#, embedding the APL JSON objects into C# classes makes it quicker to develop and debug multi modal applications.

Judging from the enormous documentations you need to start off , learning to code from Display Interface to APL
makes coding for user interface more exciting and challenging again for Alexa skill developers.

I have learned that with APL, you can now create more flexible, customizable and reusable display templates that will make the Alexa skill more enjoyable for users to interact with.

I am looking forward for APL to support video streams and HTML rendering in order to come up with more alexa skill projects in the future with this new great feature.
