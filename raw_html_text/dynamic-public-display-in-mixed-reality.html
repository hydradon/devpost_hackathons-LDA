Projection mapping in dynamic fashion in hologram and Mixed reality

What it does

Using VIVE and motion capture capacity, and coordination calibration in Unity 3D scene and model. I project image on a box which on top audience and see the hologram floating in the dark, and see the projection mapping on the wall and the box do dynamic story telling when the sale person moving and rotating the box in different location and orientation

How I built it

Build a box with some transparent and some opaque windows (could be on demand later)so people can see the real object in the box or see me do thing. put a VIVE tracker aside to track its location and do 3D model calibration so I can do dynamic projection mapping precisely. Accord the need, I can design auditory and visual effect when the object in different location
