Inspiration

We play many games which were fun, so we thought of what made a game fun. A game that looked professional and never-ending.

What it does

An android game where we touch left or right part of the screen to control a 3-d car. The obstacles set on the terrain are meant to be avoided.

How we built it

We used unity and c#

Challenges we ran into

The interface was confusing and it was hard for all of us to work on the program.

Accomplishments that we're proud of

We had no clue of how to use c# or unity so it was interesting to figure out how to program and make a simulation.

What we learned

We learned game development and how to use unity to build an android game.
