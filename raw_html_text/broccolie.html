• BroccoliE

Inspiration

Avoiding fraud, Identity theft, and the recent Equifax data breach.

What it does

A homemade Polygraph test that incorporates the use of EEG, Eye-Tracking, and Voice Analysis

How I built it

Created with data generated from the Muse Headband and Eye-tracking data obtained from the webcam. This data is then given to a Tensor Flow model to be analyzed
