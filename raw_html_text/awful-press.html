Inspiration

Some of us work closely with political events and processes daily, so we felt it's important to analyze publicly available sources of political debate to understand what is really happening, why, and how to avoid an era where the biggest liar always wins. ;)

What it does

Our app shows the user quotes. The user needs to guess, if the quote is real or fake. Afterwards the user is given feedback on his input. We also analyse user's micro-movements to rate his or her confidence level.

How we built it

We used React Native to build the mobile game app, Node.js and PostgresQL for the backend, and Python and TensorFlow to analyze the Politifact.com data

Challenges we ran into

  • Inconsistent duplicates of statements while scraping 12,500 statements
  • A missing comma
  • AWS sucked. :(
