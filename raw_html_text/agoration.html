Made by Miha Bhaskaran and Laasya Chukka

Inspiration

Teens who suffer severe anxiety issues.

What it does

Allows people with this mental illness, Agoraphobia, overcome their fear and open them up to opportunities outside their "safe-zone".

How I built it

Front End: HTML, css Sentiment Analysis: Keras with TensorFlow (Python) Back-End: Some Java

Challenges I ran into

Coming up with an idea was pretty hard. We spent a lot of time trying to make a chrome extension to no avail. Working out the data for the sentiment analysis program was also equally challenging. Another challenge was implementing our sentiment analysis into our program (we still haven't achieved this but we hope to in the future). But the hardest thing was agreeing on the right color scheme!

Accomplishments that I'm proud of

The sentiment analysis program's functionality and implementing the video api successfully.

What we learned

Don't waste your time on trivial things (like deciding on color schemes!!).

What's next for Agoration

Integrate Sentiment Analysis into our program and not separately. We also plan to use facial recognition into our program to detect the mood of the user so we can connect with them better. Lastly, we plan make the program into an android app so that it's accessible to anyone who has Agoraphobia.
