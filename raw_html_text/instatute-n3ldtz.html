Inspiration

Being students in a busy life, we know how hard it is for us to manage our schedules and be in touch with our teachers. So, we thought it would be best to integrate both of these things using a Web App.

What it does

It does several things. It allows for both teacher and student login, gives an easy way to pick our time table by using Google's distance matrix API, and also check grades.

How we built it

We used Python and attempted to use PyQt to make an interactive webpage.

Challenges we ran into

Being beginners in programming, the biggest problem we had to tackle was the use of APIs. We had several problems finding the keys and secrets, but in the end, we figured it out.

Accomplishments that we're proud of

We're proud of our achievements regarding the file management system and the algorithm developed, as it is an intricate system that can store and sort through files, and the algorithm never fails to provide an optimal solution.

What we learned

We mainly learned how to use APIs, and also learned more about PyQt and JavaScript.

What's next for Instatute

Instatute can be developed with a front end interface, which can make it easier to use and more appealing to the eye.
