Inspiration

We wanted to make an application that would allow us to simultaneously learn and enjoy, and our entire group loves the Starship bots,so we made a game on them

What it does

This is a food delivery game where you control a bot that must deliver lunch to a hungry President Purdue.

How we built it

We built this Unity as a 2D plat-former game

Challenges we ran into

We initially struggled to adapt to C# but we overcame this after toiling through the night

Accomplishments that we're proud of

We're proud of learning C# and Unity on the fly

What we learned

We learned teamwork and how to develop a new game

What's next for BoilerShip

We had really ambitious plans for this project but couldn't incorporate everything in this limited time-frame. We plan to add these after the hackathon
