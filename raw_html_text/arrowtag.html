Inspiration

Nentico Lodge's implementation of RFID check-in.

What it does

Implements a universal rapid check-in standard.

How we built it

Using OALM API references and QR and RFID tutorials and datasheets.

Challenges we ran into

Understanding the new API and figuring out the mess of NFED and RFID memory formatting.

Accomplishments that we're proud of

Agreeing on a standard and devising the operation of a potential program to use the standard.

What I learned

RFID has immense potential with its form factors and capacity.

What's next for ArrowTAG

Development of a program to use the standard for check-in and eventual deployment.
