Inspiration

I wanted to make a simple skill for small children and thought puppies would be the perfect theme

What it does

This skill makes use of choice and deduction for the child. They will need to examine the puppy or just put up flyers to see if they can find the puppies owners.

How I built it

Using Alexa skill kit, lambda and Javascript

Challenges I ran into

Since I'm fairly new to Javascript some of the issues I had to research.

Accomplishments that I'm proud of

I think it's a fun little skill that hopefully some kids will enjoy.

What I learned

Alexa skills and programming in general is a fun thing to invest time into even for a hobby.

What's next for Lost Puppy

I have some changes and maybe a new skill that will be similar with kittens that will be a little more advanced for slightly older children.
