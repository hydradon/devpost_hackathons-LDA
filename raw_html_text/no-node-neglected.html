Inspiration

-Martello is an awesome company that we are definitely interested in, they proposed a very exciting challenge that we were curious to tackle. Also, our program is an innovative solution to problems modern households face.

What it does

-It is a web application that continuously collects and visualizes all of the devices connected on a home network. IT provides useful information and data and uses it to create the representation.

How we built it

-First we drew a basic layout of how we want it all connected. We then wrote the web application using javascript and html.

Challenges we ran into

  • One thing was we are all fairly new to javascript, so we basically learned the entire language including node.js in one night. There were lots of errors that required our attention, which in turn wasted a lot of time.

Accomplishments that we're proud of

  • We managed to get it to work, we made an awesome visualization with a really easy to use UI, we were able to answer all of the challenge questions while implementing most.

What we learned

  • We definitely learned how to delegate and complete tasks as a team. We learned Javascript, node.js, and angular.js all in one night. We also learned how to manage and put together an online web application.

What's next for No Node Neglected

  • We would like to implement new innovative features such as remote control of different devices connected on the network.
