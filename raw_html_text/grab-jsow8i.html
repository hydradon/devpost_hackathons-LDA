Inspiration

Inspiration came from the idea that it can just a little bit more of someone's day, likely someone who doesn't see well.

What it does

Recognizes text from images with text and allow the user to see it over the image or without, whatever helps them.

How I built it

Using ionic framework and work with Microsoft Optical Character Recognition API.

Challenges I ran into

sending binary image data in a different form to be accepted by the api and challenges of javascript scoping issues.

Accomplishments that I'm proud of

I learned much more about debugging and many of Microsoft's services.

What I learned

I learned much more about debugging and many of Microsoft's services.

What's next for Grab

Being able to translate into different languages to help non english speakers understand text through a simple snapshot.
