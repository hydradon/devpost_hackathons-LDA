Inspiration

Everyone these days ignores push notifications and we wanted to create something that was engages the user rather than waiting for the user to login.

What it does

A user can set goals for running, biking and swimming, with TriFit every user is a triathlete! Each day the user logs how far they swim, run and/ or biked. TriFit engages via multiple communication channels with motivational words of wisdom and imagery.

How I built it

We built TriFit with Andriod-studio. The database is SQLlite and the front end is Java and xml.

Challenges I ran into

We ran into some issues (due to time constraints) with the database.

What's next for TriFit

We will be integrating TriFit with Twilio so we can send the gifs and messages as texts.
