Inspiration

People tend to ask others for their thoughts on a subject so we wanted to make it easier than the current Q&A applications on the web.

What it does

A revolutionary Q&A web app.

How we built it

Used open source web framework Django 1.9 with Python 3.4. Implemented REST web service and MVC web application.

Challenges we ran into

Determining the right tools that fit our project with respect to the time constraint.

Accomplishments that we're proud of

Project status is complete with the functionalities we discussed at the beginning of the hackathon.

What we learned

Attended WebDev and API workshop. We learned about Django and ended up using it for our full stack development.

What's next for echo
