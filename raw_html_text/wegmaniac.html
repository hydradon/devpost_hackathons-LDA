Inspiration

Coming from a data science background, we have always enjoyed exploring data and coming up with different ways to analyse it. So we decided to use the Wegmans API to make something cool out of it.

What it does

So a user would click a picture using our React-Native app of any food item he or she wants. The app would tell which food item it is and in which store of wegmans its located along with its name and price.

How I built it

We used React-Native to use the app in all mobile platforms. We have used Vision API from GCP to detect objects in the pictures.

Challenges I ran into

Integration of react native with wegmans api

Accomplishments that I'm proud of

A successful app displaying almost accurate results
