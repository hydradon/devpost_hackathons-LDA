Inspiration

My kids kept asking me to quiz them with math questions.

What it does

The app creates an endless number of addition, subtraction, multiplication, and division problems to practice using the Amazon Echo. It's a great alternative to math flash cards.

How I built it

I used Amazon Web Services, including Lamda, S3, Cloudwatch, and the Alexa Skills Kit.

Challenges I ran into

Figuring out how to debug with the Alexa Skills Kit.

Accomplishments that I'm proud of

My kids can have fun learning math on a screen-less device.

What I learned

Developed and published my first custom skill on Amazon Alexa in less than 48 hours.

What's next for 1+1 Math

Next steps are to create a more adaptive game with different levels that challenge users with harder questions as the game continues. Then, create a multi-player, turn-based game where users can compete against each other. Finally, develop user accounts to store previous game results and help players and parents identify which type of math problems the player can improve upon.
