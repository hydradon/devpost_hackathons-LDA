Inspiration

To be honest, we're not the healthiest people. Also, compared to our other idea, WebMD VR, FitMate didn't sound too bad.

What it does

The mobile application allows users to meet people around them and find a workout buddy! Our system matches users based on their location, and their workout preferences. Workout buddies can chat with each other and stay up-to-date on each other's progress.

How we built it

We used Xcode, Swift, and Fire Base to build our app.

Challenges we ran into

Since we are first time hackers, we struggled with source control, creating a database, and learning the development process of a mobile application.

Accomplishments that we're proud of

We are definitely proud about building a functional mobile application using new technology in only 24 hours.

What we learned

We learned how to write code in Swift and use Xcode. We also learned how to use Fire Base for our application, which was very cool!

What's next for FitMate

We are planning to implement other features that we did not have enough time to do such as machine learning to suggest better routines for workout buddies!
