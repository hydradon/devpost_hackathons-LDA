Inspiration

We really hate having to change the aux cord in a car.

What it does

We make a live playlist that allows a party to vote on songs to play. Songs that have high up votes are moved to the front, songs that have too much down votes are removed from the playlist.

How we built it

We used Android Studio to program it.

Challenges we ran into

MySQL Database would not connect.

Accomplishments that we're proud of

We're proud of thinking of this idea, and since there are approximately 7 hours left, there's plenty more time to do even more.

What we learned

AWS, calling database in Java, YouTube API

What's next for SOUNDLIST

Fixing bugs, making it more robust, adding additional minor features, adding other online video/music providers.
