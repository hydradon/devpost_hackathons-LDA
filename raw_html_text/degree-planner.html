Inspiration - We were frustrated with advisers being too busy or making errors in course planning for students.

What it does - Students enter the courses they have taken, and we tell them what they still need to take with a few options.

How we built it - It is a simple Microsoft Razor project.

Challenges - We ran into Learning a new technology.

Accomplishments that we're proud of - Displaying and sorting our database entries.

What we learned - We learned syntax of C#, Razor, html, and css. We also learned how to make a sql Local database, how to implement version control with github, and the basics of the Model View Controller architecture pattern.

What's next for Degree Planner - We will continue to work on implementing functionality and improving the design pattern.
