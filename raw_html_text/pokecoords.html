Inspiration

My love for Pokemon, the mobile game and the desire to help out the Pokemon community were my inspiration.

What it does

It allows users world-wide to input and view locations of legendary Pokemon.

How I built it

I built it by using Android Studio and linking the project to a Firebase console to store my data.

Challenges I ran into

Create a method to call upon the Calendar in order to access the current date and time. To then compare it to the time the user inputted, in order to eliminate the raids that have already passed.

Accomplishments that I'm proud of

Completing the challenges I faced.

What I learned

How to link Android Studio to Firebase and how to grab current time data.

What's next for PokeCoords

To adjust the designs and view to appeal to the public.
