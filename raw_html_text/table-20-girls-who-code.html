Enable GWC Partner Engagement using 12 AWS Services

The challenge for our team was to create a way for Girls Who Code to be able to modernize their interactions with corporate sponsors through their HQ portal. The team developed a tile style display that can be filtered by sponsor type, upcoming, and allow the user to add their own events. An addition request was for either a Girls Who Code admin or corporate sponsor to be able to view their employee participation and statistics.
