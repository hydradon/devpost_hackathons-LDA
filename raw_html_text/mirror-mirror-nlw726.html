Inspiration

After reading about HeforShe and hearing Emma Watson's opening speech, we thought that we would take our own personal spin on how to achieve gender parity.

What it does

Mirror, mirror is a cross platform application that seeks to empower the younger generation in realizing their full potential through continuous positive messages. We've built a working website and would like to push out an app as well as physical product into the market.

How I built it

We are first time hackers and we built a website using brackets. We also used arduinos and other hardware to try and prototype the mirror.

Challenges I ran into

We are first timers and spent a lot of time brainstorming as well as figuring out code & hardware. Everything was a learning process for us. It was frustrating to have an idea but not know how to fully implement it.

Accomplishments that I'm proud of

We were able to build a website with live webcam feature that we hope to expand upon in the future.

What I learned

A lot. But seriously, everything we did this project we learned during this time period so that was really awesome.

What's next for Mirror Mirror

We've detailed that in our submission, but we're hoping to build an app as well as launch a physical product.
