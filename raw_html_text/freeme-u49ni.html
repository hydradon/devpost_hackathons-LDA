Inspiration

After realizing how frustrating it is to arrange something as trivial as a hangout with friends, we came up with the idea for FreeMe.

How it works

FreeMe is a simple iPhone app that solves an age-old problem. Rather than asking friends when and whether they're free, everyone can just broadcast their availability using the app. Slide the switch to the right to indicate that you're currently free, set how long you'll be free using the timer, and add a personal status for more details.

What's next for FreeMe

Polish the frontend app, secure the server from vulnerabilities, and make an Android version.
