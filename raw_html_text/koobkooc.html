Inspiration

The Inspriation came from Ben's mom talking about a fridge that will tell you the ingredients you have. We decided to mess around with the Amazon Echo an create a reverse cook book.

What it does

Koob Kooc is an Alexa Skill that will take your inputted ingredients and give you recipe ideas at the ease of asking Alexa.

How we built it

We built the Skill using a Lambda fuction on Amazon Web Services, along with javascript

Challenges we ran into

Our main challenge was for the Echo to come out with a legitimate recipe available.

Accomplishments that we're proud of

None of us have hacked an Alexa before or messed with Java Script, the fact that we learned all that we did in less than 24 hours is astonishing and I am thrilled what I will learn next.

What we learned

Alexa SDK, Javascript, Connecting an external API to an Amazon Service

What's next for KoobKooc

After today we may grow this application and make it become a published skill for all to use on Amazon Alexa
