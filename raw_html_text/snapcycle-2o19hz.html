Inspiration

The inspiration for this project came from one of the speakers who challenged us to make an app that recognizes if objects are recyclable or not

What does it do

Our app lets you take a picture of an object and it will return what it thinks it is with a confidence score and let you know if it is recyclable

How we built it

We built this project on Xcode using swift and used CreateML to make the model

Challenges

Some challenges we ran into were that we had little experience building iOS apps and we also had little experience with ML

Accomplishments

We are proud that we built a functioning app that actually uses ML in the background

What we learned

We learned how to build a UI that users would like and we also learned how an ML model is made

What's next for SnapCycle

We hope to add more functionality such as a tracker that shows how much you are helping the earth with what you are recycling
