Inspiration

We think the Yelp experience isn't social enough. That is, it's still hard for groups of people to decide where to eat.

What it does

Fignest alleviates this problem by choosing a restaurant based on the photos selected from a set of unknown restaurant options. The restaurant with the highest number of photos selected wins!

How we built it

Node/Express REST API server with Socket.io for real-time communication. iOS app using Swift.

Challenges we ran into

Developing on these platforms for the 1st time. Limited time to complete the project.

Accomplishments that we're proud of

All of the above :)

What's next for Fignest

More gamification, real-time communication, and improved UIX. Follow here: https://github.com/nlujan/fignest
