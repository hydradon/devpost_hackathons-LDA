Inspiration

What it does

InTouch lets caregivers setup prompts they would like seniors to respond to, as well as parameters to define the ways prompts are delivered, the degrees of flexibility with which responses are required, as well as triggers for prompts lacking responses.

How I built it

We start by designing a mock UI using Sketch. To receive input from stakeholders, we walked through the InVision prototype with representatives from Heart of The Valley. We then developed the app on iOS using Swift. The app's backend services are built on Parse and Twilio.
