Main idea

We wanted to create an easily searchable catalog of the events during Edinburgh Fringe Festival. We wanted the app to recommend the events matching your interest.

Why

There is a bunch of the events data and there is a potential to make it more interesting and appealing visually. Maybe we can learn something from that.

Key features

  • Search engine (with Speech to Text)
  • See what other people like
  • Subscribe to Events

Challenges I ran into

  • Integration of Events API
  • Integrating Speech Recognition
  • Displaying Events Information

Accomplishments that I'm proud of

  • Up and running React client side application and backend with database

What I learned

  • How not to do React ;)
