What it does

Play pac man using a phone's tilt to control movement. Only one life so use your portal to stay alive

How we built it

Java and Opengles 2 on android studio.

Challenges we ran into

Had to finish it overnight. Creating a method for collision detection. Using accelerometer.

Accomplishments that we're proud of

Simple collision math for walls.

What we learned

How to use the accelerometer. Collision detection.

What's next for Pac Man with Portal

Balancing speed to make it more game like. Multi threading for speed improvements. Better collision math. Fixing portal bug.
