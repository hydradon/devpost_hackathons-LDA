When They Gave Up

Hackathon project that analyzes participants and determines when they gave up based on Github commits

Issues I encountered

I was initially going to source a list of hackethon participants from devpost but I unfortunately got blocked by devpost.com for web scraping. So I had to pivot my idea a bit to simply analyze Github repositories instead and try to determine when a developer has given up based on commit messages instead.
