• The one and only- Donald

Inspiration

Starting with the NodePromptManager project, I decided to built my own bot with it. Since 'The Donald' (love him or hate him.. apparently it makes no difference to him or his popularity) is in the limelight right now, I decided to built a bot in honor of our future President! NOTTTT!!!!

What it does

TheDonald is a simple Prompt App bot written in Node.js+jQuery that spits out the most outrageous statements said by Donald J Trump. This bot receives a message from an SMS/Slack/Prompt website and returns a new Donald Trump quote to the user.

How I built it

This was built on top of NodePromptManager using Node.js + Javascript + Microsoft Azure.

Accomplishments that I'm proud of

I came out to Hacktech16 determined that I would work on an iOS app. Once I got here, I was surprised to see that there were only a handful of iOS devs. Hence I decided to risk my weekend on web-dev and I am glad I paired up with Peter Kaminski . I picked up Node.js and Javascript and was able to built this bot by myself.

NodePromptManager -- starter template for building Prompt bots with Nodejs

We spent the first half of HackTech working closely with the founders of http://promptapp.io to learn more about their development platform, which had just been released the day before the hackathon. With no documentation specific to NodeJS, we felt the need to create a boiler plate class for Node--allowing devs to go from 0 to 100 with Prompt bot development. Real quick. Credit for this also goes to Peter Kaminski
