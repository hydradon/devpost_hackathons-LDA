Inspiration

Studying anatomy from a book can be dull and boring - why not make learning more interactive and fun!

What it does && ## What's next for Body Touch

The system was intended to be a 3D model of the human body and be able to interact with the model using the Leap Motion. By pressing on the various body parts the name of that part would appear and by double tapping the model would reveal the underlying layer of muscles and tissues that could be explored as well.

Challenges we ran into

We originally wanted it to work with the Oculus Rift and spent a lot of time getting the environment set up. The device was not necessary for the project to be completed and that time could have been spent on working with the leap motion.

What we learned

We learned how to use unity and how to develop with LeapMotion :)
