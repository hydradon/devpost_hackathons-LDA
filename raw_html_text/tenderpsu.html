Inspiration

We always here stories of people not knowing where to go out to eat, so we thought we would make an app to help them solve that problem.

What it does

It uses the Amazon API Gateway, and Amazon Lambda to create a http and interact with Dynamo db.

How we built it

Using Android Studio and Amazon Web Services

Challenges we ran into

We overscoped the complexity of the backend API.

Accomplishments that we're proud of

Better knowledge with AWS and mobile development

What we learned

We learned a lot about Amazon Lambda and Dynamo db, and a little bit of Android Web Development

What's next for TenderPSU

Incorporate the API into the project and display restaurant locations.
