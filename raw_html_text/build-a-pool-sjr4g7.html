Inspiration

We are all frustrated students in our last years of university but we want to go out with a bang! To do that, we've created build-a-pool !

What it does

It allows students to optimize their university experience by choosing an appropriate path determined by powerful sentiment analysis.

How I built it

We built it by doing web scraping and sentiment analysis with Python and figuring out a way to display huge datasets in HTML!

Challenges I ran into

Creating the visualization and web scraping pre-requisites/co-requisites written different manners were two of many hurdles we overcame.

Accomplishments that I'm proud of

This project was ambitious and, my team and I, are proud of the overall end result.

What I learned

We became PROS at BeautifulSoup with Python and displaying large datasets!

What's next for build-a-pool

Improving some of the UI and expanding for other departments would be neat (:
