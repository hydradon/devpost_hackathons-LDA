Inspiration

The presentation for OCCC made it apparent that they require a scheduling solution that is accessible for all hours of the day. The solution that we came up with is pretty much an out of the box solution that can easily be implemented using the current infrastructure that is provided by the State of Hawaii for departments on the with a little bit of modification for security reasons.

What it does

The main purpose of the website solution is to provide some form of scheduling that is also mobile friendly. Because the website's template for the state is not too flexible (not sure if this has changed but we have the impression that the theme is pretty set based on the other websites for the various departments) if integrated with the currently unified departmental website solution for the state (WP Multisite), the theme used was made look like the state website but the implementation is slightly different.

How I built it

The solution we provide utilizes the WordPress core and two key WordPress plugins that are constantly being updated and supported. Our solution was built to ensure that it would require low maintenance and be available for a long time without the need for a team of developers.

Challenges I ran into

There is very little to no information about regulations and so forth for websites such as these. Accomplishments that I'm proud of

The main accomplishment we are proud of is that we can provide something that is open source and somewhat simple. The current IT individuals can use the system as it sits with a few security modifications.

What I learned

Our team learned that there are areas that can really benefit from a simple solution that does not require a lot of manipulation to make work using open source. Areas of the City and State systems need items that are very approachable and not tons of systems that lock people into a solution that makes them reliant on one source of development.

What's next for OCCC Simple Scheduling Base

The next step would be to finalize the solution and integrate it with the current state's WordPress implementation and then add more plugins to extend the solution even further. Some ideas would be to work with close field location based beacons, for instance, that allow for further instructions to guide visitors within the property.
