• Face/Off

Inspiration

Inspired by Face/Off movie.

What it does

Dynamically detects face in captured webcam shot and uses it to replace face in any target image.

How I built it

TensorFlow 2.0 BodyPix Real-time Person Segmentation javascript API GitHub https://github.com/niravanu/faceoff

Challenges I ran into

The next challenge is to extend this solution to change face in any video in real time.

Accomplishments that I'm proud of

One week effort to understand TensorFlow BodyPix API and to implement this idea

What I learned

TensorFlow 2.0 BodyPix API

What's next for Face/Off

  1. Make merged face look more natural (instead of copy paste kind of look) to the target
  2. Extend this solution to change face in any video in real time
