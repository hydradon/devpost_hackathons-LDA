Inspiration

As college students, we have all had moments of extreme stress. We wanted to create a unique, user friendly virtual environment where students can relax.

What it does

FeelingFeed allows the user to choose feelings they're experiencing and topics they're interested in. FeelingFeed then generates a feed of randomized content specified to their needs

How we built it

We began with our idea of having a page of clickables where a user can select items they relate to. Then, we dove into what that would look like and how that would translate into generating a personalized feed. We wrote FeelingFeed in JavaScript and HTML.

Challenges we ran into

We struggled with implementing APIs into our program.

Accomplishments that we're proud of

We are proud that we have created something that can help people feel better.

What we learned

We learned a lot about working with APIs, Bootstrap, and jQuery.

What's next for Feeling Feed

In the future, we want to expand on FeelingFeed's feelings and interest choices, and dynamically capture up to date content to display for users.
