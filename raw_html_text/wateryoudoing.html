Inspiration

We were inspired by some of the problems that we as Californians are directly challenged by; namely, the chronic droughts that affect all areas of the state.

What it does

This project is a game that encourages water conservation by keeping track of the amount of water the user consumes each day compared to a water usage goal input by the user. Whatever remainder there is is used to "water" a virtual plant so that it can grow. The user is encouraged to use less water so that their plant can grow faster.

How we built it

We used github and Cloud9 to collaborate our project. We used javascript, HTML, and CSS to code the project, as well as Krita to design the graphics.

Challenges we ran into

There were many issues in pulling/pushing, etc in Cloud9. There was also the issue as to which Java interface we wanted to use, and eventually settled on Github. There were also formatting issues for the buttons.

Accomplishments that we're proud of

We are proud of persevering through our project, and attempting to resolve all of the issues.

What we learned

We learned that Derk is good at coding

What's next for WaterYouDoing

A sensor to keep track of a home's water gauge so that the water calculations are more automatic, more in game rewards and activities to engage the user, and more information on our information page.
