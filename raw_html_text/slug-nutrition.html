Inspiration

Inspired to by one of our team members to do something with image recognition and ocr that was once thought to be impossible.

What it does

Our mobile app has the capability to use image recognition to take a picture and determine the type of food in the picture while pulling data from a web server to retrieve nutritional information. Also, it has the capability to read a nutritional facts stamp using ocr and display that information to a user.

How we built it

We built our hack using a computer vision api from Microsoft Cognitive Services. We then incorporated Android Studio and a web server to put our own innovative twist on it.
