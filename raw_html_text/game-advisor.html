game-advisor

Think about the times when you are searching for a game to play. Imagine the stress you have been through when you look at the overwhelming number of apps in the app stores. You want to try something exciting but you don’t want to waste time on the games that you might regret. So, we came up with a solution. We build a website and it has several types of games laid out on our main page and a quiz that can decide what types of games the user would use by answering a set of questions. According to the quiz, users can identify which categories they would like to have a try.

Through this project, we learnt about the basics of web development. We gained more knowledge about the programming languages HTML, CSS and JavaScript. We practiced and improved our teamwork skills and learnt how to make team collaboration smooth and efficient.

We used HTML to build the basic structure of the web page. We utilized CSS for formatting and making the website more aesthetically pleasing. Lastly, we used JavaScript for user interactions, including gathering and analyzing user inputs and generating corresponding outputs.

The most challenging part of the project was to deal with user inputs. We had to collect and save the answers to the questions that the user inputs. Then, we have to analyze the most suitable game category for the user, make a recommendation and link the user to the specific category. This process was a bit challenging for us as starters for web development.
