Bio-Me

Rapid oral microbiome detection prototype submission for Hack HPI 2017 Health Tech Hackathon. We outline an end-to-end solution pipeline for collecting, processing, and analyzing personal medical data with a smartphone app using existing biological technology.

Why at-home oral microbiome? It provides insight into personal microbial flora diversity. We developed components that can be extended to create a mobile app for personally monitoring health and visualizing one's data.
