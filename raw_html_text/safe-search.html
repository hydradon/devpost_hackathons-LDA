Inspiration

We stayed at a very sketchy motel and we wanted to create an app that could show how safe an area was.

What it does

The user can select a point on the map and then creates a heat map uses tracts in the selected county.

How I built it

We used arcGIS to create the map and render the heat maps.

Challenges I ran into

Creating the polygons to represent the counties and tracts on the map was difficult.

Accomplishments that I'm proud of

We are proud that we were able to create the polygons.

What I learned

We learned how to use the arcGIS sdk to create detailed maps.

What's next for Safe Search

Using more data and spending more time developing algorithms to generate the heat maps.
