What it does

Smshare is providing deep integration with ecommerce platforms: opencart, shopify, prestashop, etc...

Smshare allows store owners to send SMS to their customers on various events. Smshare offers 2FA for opencart.

Smshare is available for store owners all around the world. Some of them already started using Nexmo.

How I built it

This is built using Java technology and apache http client.

Challenges I ran into

Nexmo api is one of the best api. However, it lacks the ability to detect the SMS type (Text VS unicode)

We had to parse the SMS content looking for unicode characters, if there is at least one unicode character, we add the unicode parameter to the API call.

Accomplishments that I'm proud of

With one click and few keystrokes, store owners can start sending SMS from their store using Nexmo reliable SMS gateway.

What I learned

Smshare works with multiple SMS gateways. Nexmo gateway is very reliable especially for gulf countries whereas other gateways failed to deliver SMS.
