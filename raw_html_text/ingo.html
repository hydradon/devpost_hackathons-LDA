Inspiration

Bringing people together - not wasting food - social integration of new residents - helping get more information about events near you

What it does

You may place any kind of event into our environment, accessible via smartphone (iOS and Android), natively programmed, or even publish it in social media.

Whether it is

  • a restaurant, that wants to sell the remaining food, they have shortly before the closing hours, for a reduced price
  • a private institution (club, restaurant, cinema) wants to inform about currently running ads or campaigns (reduced pricing, ladies night, costume party)
  • a public institution wants to have a fast way to publish the new opening hours or common information
  • a private group (recreational activities) that searches for participants
  • a new person in the city wants to find groups that share the same interests
  • a person visiting a city needs a simple way to find reduced prices, events or just groups to hang out with

all of this is possible via google assistant or simply by using our app or the website.

How we built it

Apps (iOS and Android) were written with React Native and build with gradle Backend was written in GoLang and build in a docker-in-docker build An API Gateway in AWS was created that provides ssl termination and redirects to the backend

Challenges we ran into

Some animations that were intended to be put in were found harder to implement into the react-native. We had to create certificate signed by an official authority even when we were in the test phase.

Accomplishments that we're proud of

Additionally to creating an integration with google assistant we were also able to develop an ios and android native app, a website and a backend. All of these components adhere to the google design pattern, except the gradient ;)

What we learned

Teamwork and communication are as important as technical know how.

What's next for InGo

Implementing more dynamic functionality in creating events, utilized by a AI. Adding security concepts to the architecture. Only People with a business account should be able to publish event for this business.
