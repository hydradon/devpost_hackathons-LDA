• MyToolbox

Inspiration

I've collected a large amount of tools - and I'm never sure if I should by one that a friend might have. I also would like to share mine with friends - use it or lose it I say. So I built this Alexa skill for you to add almost any common tool to a virtual toolbox. You can then have a secret code to link to friends and ask Alexa if they have a particular tool.

What it does

It stores your tools (ladders, drills, saws etc) in your virtual toolbox. It allows you to search for a tool (for those of us who are forgetful) It allows you to add friends and search for a tool a friend might have

How I built it

Using the Alexa interface, a VPS to host a RESTful interface, running an Apache and PHP and MySQL database.

What's next for My Toolbox

I'll put in a few more useful features - locations of tools. Don't you hate it when you forgot where a tool might be? I'll put in something that will remind you of all the places you store your tools!
