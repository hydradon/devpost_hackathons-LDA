Inspiration

Immer, wenn man eine Entscheidung treffen möchte und selbst nicht das entsprechende Know-How hat, möchte man eine Vielzahl an kompetenten Leuten fragen. Mit diesem Portal kann man sowohl Freunde und Verwandte, als auch die gesamte Community fragen was die beste Entscheidung für das Problem oder Vorhaben ist.

What it does

Andere Menschen können dir dabei Helfen, zum Beispiel ein neues Auto oder Laptop zu finden. Dabei hat jeder die Möglichkeit die Antworten der Anderen zu bewerten wodurch der beste Vorschlag gemeinsam gefunden werden kann.

How I built it

Das Front-End setzt auf TypeScript mit React und verwendet eine PouchDB. Das Back-End ist in Java geschrieben und verbindet die PouchDB mit einer CouchDB. Das Back-End wird letztendlich als Docker gepackt und deployed.

Challenges I ran into

Einen Server zu finden, der schnell und einfach zu konfigurieren ist und allen Anforderungen entspricht.
