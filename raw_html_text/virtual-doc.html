Inspiration

We were interested in making diagnosing and buying affordable medication easy with the use of machine learning. We want it to be used by all demographics, so we planned on integrating with social media and mobile devices.

What it does

It's a chat bot that talks to the patient and asks deeper questions to make a diagnosis. Once the machine learning model takes the input from the user, it will search the internet for the cheapest and most suitable medication for you condition.

How we built it

We started with getting a simple message to our phones. Starting from there, we had to organize how the questions will be asked and how the user can be prompted for their diagnosis. Then we had to build a data set from scratch to account for every possibility that could occur through our machine learning model. Then we would integrate with other APIs (for medication, social media, etc)

Challenges we ran into

We really had to condense the scope. It was a bit more ambitious in the beginning.

Accomplishments that we're proud of

We made a bot that is capable of diagnosing.

What we learned

New machine learning methods, twilio integration.

What's next for Virtual Doc

Either horizontal expansion in other sectors or talking with other doctors to make the diagnoses more accurate.This technology can be applied to almost anything, including a Virtual Banker, a Virtual Social Media Marketer. We are really excited to see where it goes.
