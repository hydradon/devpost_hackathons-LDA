• A word frequencies' pie chart for a simple PDF file

Couldn't make it to your 8:30 lectures? Running out of time to study? Don't worry, my friend. We are here to help! My partners and I came up with an idea that can let students study in a more productive way.

We developed a program which has the functionality to sort out all the key words from a pdf file. Moreover, after sorting out all the key words, it will then generate a pie chart to represent how frequent each key word appear in the input file.
