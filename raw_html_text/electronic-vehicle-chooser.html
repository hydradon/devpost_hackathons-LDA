Inspiration

We wanted to help people choose their EV to help them to live a green life etc.

What it does

The EV chooser helps you pick an EV.

How I built it

We used react front end with a Flask backend to enable python ML.

Challenges I ran into

Regarding connecting the front end and backend we had a lot of difficulty because there were issues installing flask correctly in the virtual environment, and the path variables were not being set correctly. Additionally, we did not have enough data to train our machine learning model, so it proved to be difficult.

Accomplishments that I'm proud of

Proud of setting up the back end and front end, with a machine learning model.

What I learned

We learned how to basically deploy a machine learning model on flask and react.

What's next for Electronic Vehicle Chooser

We plan to get more data and then make it more full fledged.
