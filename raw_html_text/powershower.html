Inspiration

Long waits at dorm showers as well as increasing need to conserve water during recurring Californian droughts.

What it does

Provides an easy-to-use shower timer and displays remaining time on the stall door, as well as relaying it via WiFi to a downloadable application that shows the daily shower record as well as a waiting list.

How we built it

Used two Arduino Nanos with two display timers programmed using the adafruit library and bluetooth to relay timer data, as well as a WiFi module.

Challenges we ran into

Difficulty implementing libraries for bluetooth and WiFi modules and difficulty soldering.

Accomplishments that we're proud of

Bluetooth connection, general compactness, keypad interface.

What we learned

Arduino, bluetooth, network connections, PCB soldering.

What's next for PowerShower

Improved application functionality.
