Inspiration

The inspiration behind development of this project is to make use of IOT and provide the world a solution with Artificial Intelligence and Machine Learning.

What it does

It is an automated and voice controlled banking system.

How we built it

With the help of technologies like AWS, Microsoft API and Capital one API

Challenges we ran into

We thought of incorporating the face matching as a password for this banking system but because of the json parsing problem and a second API integration. And finally we came up with some alternative solutions.

Accomplishments that we're proud of

Completion of API integration and successful working of Alexa

What we learned

We learned how to connect multiple devices

What's next for Automated Banking System with Alexa
