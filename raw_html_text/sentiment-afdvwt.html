Inspiration

There's so much Twitter data out here, what if we could actually analyse this real-time data, visually. It could give us an overview of whats going on, without having to read 1000's of Tweets per second

What it does

Analyses real-time Tweets, uses IBM Watson to calculate sentiment, and more, visually displays the results using D3.js

How I built it

Node.js, D3, Twitter, IBM Watson, HP, JavaScript
