Inspiration

Registering for thins is slow and tiresome, alternatives require you to link to social media. Why not have the ease of both in one?

What it does

Upload a photo and fill in the form and never have to fill one in again. If a site uses our api it allows for an image to be uploaded and it will fill in all the details.

Your details are stored nowhere on our servers, completely contained in the image itself so there is a greatly reduced risk of data leak.

How we built it

Using flask as a server it allows the upload of photo data and personal info,then uses steganography to encode the image with the data. The site uses a script to call our api which will decode the image.

What's next for Pictyour

Improving the efficiency and speed of transfer, as well as encrypting the data for extra security. A widget to allow sign in via the pictyour, or a web browser extension to auto fill forms.
