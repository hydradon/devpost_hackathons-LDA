We arrive here with the idea of do something awesome with the Myo armbands, so we used leap motion because there were no Myos. :(

In #LeapingInTheSpace you can control your spaceship with your own hand; also you can destroy asteroids only by closing your hand.

The game was developed using Unity3D Game Engine, C# and the Leap Motion Hardware; it was very challenging working with six parameters, specially with the 3 different rotations, looking for a 3D aspect.

Finally we would like to share our game with the leap motion users and later on try to merge the project with the oculus rift.
