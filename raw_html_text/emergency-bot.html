Inspiration

We wanted to create an app that would allow people to feel like they had some sort of safety net when they might be going into dangerous situations (e.g. walking home late from a class, going to somewhere new alone, etc.)

What it does

The app was supposed to integrate a simple button pressing UI with a Messagebot that would send a message to a contact from facebook. Unfortunately, we didn't have enough time to connect the two.

How I built it

Used Facebook APIs

Challenges I ran into

Setting up Heroku servers and facebook api was difficult to navigate.

Accomplishments that I'm proud of

Fixing the stuff that didn't work

What I learned

I learned how to use Facebook api and json. Learned how to use Xcode better, and use Contacts API.

What's next for Emergency Bot

We hope to connect the better UI, and implement message sending out from volume buttons.
