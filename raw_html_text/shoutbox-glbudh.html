Inspiration

We saw the need for a chat box that cannot be altered nor shut down. This does not exist today, so we built it! Shoutbox is a fully decentralised peer-to-peer chat: one that is not hosted on any third parties servers.

What it does

Enables every user to run their own in-browser node & send messages in a fully decentralised way

How we built it

Typescript, libp2p
