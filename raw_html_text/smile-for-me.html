Inspiration

We were inspired by Uplift’s mission to educate society today about the pervasiveness of rape culture as well as biases and microaggressions that multitudes of women face today. We wanted to create an interactive and engaging experience for a user to become aware and gain insight into these often forgotten issues.

What it does

It simulates the every day life of a working woman and allows people to be put in the same situations that women may face today.

How I built it

The program was built with Javascript, CSS, and HTML

Challenges I ran into

None of us really knew javascript, so we struggled mostly with specific coding or syntactical aspects.

Accomplishments that I'm proud of

We didn't know much (I mean, any!) Javascript, yet we still managed to code a working program. Additionally, we managed to put together different elements, like graphics and audio, to make the simulation more immersive.

What I learned

We learned JavaScript and a lot more about web development in general. We learned more about opportunities available by incorporating different design tools, like the tablet. We also learned a lot more about collaboration and the importance of something like GitHub to make the process significantly more streamlined.

What's next

We would like to add more effects to engage the audience, such as text effects. Pairing this website with a virtual reality device is a possible next step. Additionally, we would like to polish our program so that the program works well on different devices
