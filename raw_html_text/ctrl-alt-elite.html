Inspiration

To help enrich immigrant experiences by solving their problems. The first step to solving problems is communication and if an immigrant does not speak the language, the problems are even more difficult to solve.

What it does

INTERPRETME connects immigrants with language translators (uber for translators) and offers 3 types of translation services within the app: conference, in person, video/telephone.

How we built it

Java Spring Boot for the service layer with React and Material UI for the UI and deployed on a Heroku (free) server.

Challenges we ran into

Unique Ideas, Time Constraint.

Accomplishments that we're proud of

What we learned

New catch phrase: "Lose sleep, add hours to your day". We learned that much more can be accomplished when you don't sleep ... with only the minor side-effect of going crazy.

What's next for INTERPRETME

The possibilities are endless: integration with social media, text, and chatting features, just to name a few.
