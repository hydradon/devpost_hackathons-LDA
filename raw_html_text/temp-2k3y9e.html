Earth Hacks - Third place winner!

Inspiration

Not everyone has the time to volunteer, likewise, not everyone can afford to volunteer.

What it does

Cleanthis.space connects environmentally conscious donors and eager community members to help out in our public places. We envision a platform that gives events like beach cleanups, trail restoration, and tree planting a lot of exposure to give them the most impact.

How we built it

We used React to make a progressive web app. On our back end is a Raspberry Pi running a Caddy server. Cloudflare proxies the DNS for our domain.com domain.

Challenges we ran into

It always takes so long to get everyone's development environment set up. Sometimes things just don't want to play nice.
