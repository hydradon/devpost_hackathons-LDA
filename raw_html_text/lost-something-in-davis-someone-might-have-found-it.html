What inspired us

Everyday I check Wildfire and Facebook only to see some people asking others for lost Aggie Cards, AirPods, and bikes, and others posting abandoned Hydro Flasks and ID Cards.

What it does

It is a website that people can post and/or help others find their lost possessions. People can choose to reward others who help recovered their items.

How we built it

We used HTML, CSS, and Javascript for front end development, Node.JS and Express.js for backend, and MongoDB as database.

Challenges we ran into

We found it hard to post images to MongoDB, and we tried to use Google Cloud Vision API to analyse images. We also tried to stay patient to the poor WiFi.

Accomplishments that we are proud of

We only slept for 2 hours on average.

What we learned

Collaborating in GitHub

What's next for Lost Something in Davis? Someone Might Have Found it!

User Authentication, collaboration with Davis Police Department to combat bike thieves, mobile app development
