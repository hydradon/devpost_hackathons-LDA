Inspiration

Creating a useful, health centric web app

What it does

Takes in user input containing health data and goals and returns a weekly meal plan according to the user's specific needs

How we built it

We used python for developing the backend and drawing from the Wegmans API, and used JavaScript, HTML, and CSS to develop the frontend that accepts the user input

Challenges we ran into

We had issues connecting the idea of recommended daily intake of nutrients to filtering out potential recipes from the data. Initially, we also had an issue ensuring all of the data was pulled from the API and usable in the format that we needed.

Accomplishments that we're proud of

We are proud of creating the beginnings of a web app that could have real utility as a completed application.

What we learned

We learned how to manipulate data using an API and ways to connect data sets and a front end of a website.

What's next for Leggo My Weggo

Possible expansion into a fully functional application with options to add to cart and place orders based on meal plans.
