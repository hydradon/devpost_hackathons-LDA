Inspiration

The idea was to build a network to connect out-of-school children to donors worldwide and provide a secure, verifiable and transparent mechanism to donate funds for school children. It allows verification of the money and provides real-time progress of the student and so promotes social ownership. This directly falls under the UN Sustainable Development Goal Number 4: Quality Education.

What it does

The app allows secure donations, with complete record and transparency using block chain network.

How I built it

The application is built as a mobile application and at the heart lies the block chain network. Hyperledger Composer and Hyperledger Fabric for the Business Network. HTML, CSS, JavaScript for the Front End.

Challenges I ran into

  1. Developing a block chain network
  2. Designing a simple and clean user interface
  3. Integration of the block chain to the front end

Accomplishments that I'm proud of

The block chain network is ready to be deployed on any PC that has Hyperledger Composer and Hyperledger Fabric.
