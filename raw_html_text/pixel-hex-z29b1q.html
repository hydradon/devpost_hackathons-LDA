Inspiration

The inspiration to make Pixel Hex came when 3 students from RIT were interested in sharing color in an easy hex format from photographs. An easy to use interface makes the app great for almost any profession.

What it does

Pixel Hex is a useful tool for developers, fashion designers, artists, and photographers to match their favorite color from the real world. Select a photo, click on the color you want to save, and then you will get a hex code and have the color sent to your watch instantaneously.

Why hex codes? Simply because hex codes are an efficient way to find a color by a RGB value from a photo. Once you have that code, you won’t be deceived by those trickster colors that pretend they’re the one you want but they’re actually off by a shade.
