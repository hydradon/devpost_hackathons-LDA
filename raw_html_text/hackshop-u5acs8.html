Inspiration

We wanted to make it easier to shop for things.

What it does

Uses google vision api to analyze photos and redirect users to similar shopping options online

How I built it

Used google vision api, etsy api, and scrapped the amazon website while also using python flask javascript and html + css for front end

Challenges I ran into

It was very difficult to scrape amazon because the api wasn't available to us as developers. Instead, we needed to use beautiful soup to scrape the actual site.

Accomplishments that I'm proud of

I'm proud of the fact that I was able to integrate so many hard-to-use tools together.

What I learned

As a team, we only knew the basics of flask, javascript, and html before we began development. We were able to learn a lot about machine learning analysis and the actual use of apis.

What's next for HackShop

We plan on continue expanding and potentially integrating the amazon api when an api key becomes available to us as developers
