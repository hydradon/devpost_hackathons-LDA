Inspiration

A future intern that I will working with over the Summer inspired me to create a VT tailored calendar importer when I saw that he had created one for UMD!

What it does

It imports students school schedules from the VT schedule system to their personal Google calendar with the click of a button!

How I built it

Forked a repository from GitHub, created a VT web scraper, and stackoverflowed all over the place looking up how to make Google api calls!

Challenges I ran into

Some schedules (I discovered late) have recitation days that ruins the original format of my parser, so I had to fix that and it was ugly :D

Accomplishments that I'm proud of

I preservered and fixed the said challenge!

What I learned

A lot about javascript (scope challenges, html specific code), making api calls from an external service

What's next for VT GCal

Test with more student schedules, upload to the chrome store for all Hokies!
