WiHelp

Inspiration

There is so much corruption in the legal system, especially when dealing with immigrants. We saw that there was a 58 percent of corrupted lawyers when reading over some articles.

What it does

It connects an immigrant with the most trusted lawyers

How we built it

We built it using React.js, Node.js, and Firebase as the database

Challenges we ran into

  • How to deal with various users
  • Understanding the technology stack
  • Working in a really big group

Accomplishments that we're proud of

We have designed the prototype of how a user will use our application. Also, we got the language translation working

What we learned

The most interesting thing we had learned throughout the hackathon is how much corruption there is in the legal system when dealing with immigration. Seeing this we see how technology can also help deter and resolve these issues.

What's next for Watson Immigration Help (WiHelp)

Finish developing the platform as well as find potential partnerships to help make this happen.
