Inspiration

In today's world, it's easy to impulse buy everything. Sure, you should be able to treat yourself once in a while -- but how much money are you really spending?

Keeping track of your income and expenses can be a hassle if you're using a spreadsheet. Flow aims to make that process easier and more accessible to everyone.

What it does

Flow easily keeps track of your income and expenses in just a few button clicks. No one wants to spend 3 minutes entering data manually into their phone every time they buy something.

How we built it

We used react native in order to have duo compatibility with IOS and Android. This is so any college student, no matter what device they use, are able to utilize it in their life.

Challenges we ran into

Learning how to use react native :(

Accomplishments that we're proud of

We are proud to complete our first project using react native. React native is a very interesting and useful technology for duo compatibility with Android and IOS.

What we learned

We learned how to use react native, split up tasks, and excel where we work best. In a coherent team, we were able to create a functioning app for the Hackathon and for our future lives.

What's next for Flow

Implementing a backend and create a web app so it will be compatible and transferable with desktops and laptops.
