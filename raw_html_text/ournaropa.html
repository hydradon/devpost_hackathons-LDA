Inspiration

I love reading, but I don't love buying books! So whenever I hear about a good book, I end up manually checking each of the three libraries in the Boulder area that I have access to: Naropa, CU, and BPL. I thought to myself that it couldn't be that hard to search all three at the same time!

What it does

Search Naropa's Libraries, CU's Libraries, and the Boulder Public Library all at once

How I built it

As a mountable Ruby on Rails engine, with Materialize for design and layout.

Challenges I ran into

Staying up for 36 hours without sleep is pretty rough... :)

Accomplishments that I'm proud of

It works!

What's next for OurNaropa

I've been working on a unified events calendar for Naropa and I'm hoping to also create a better course planning tool than the one we currently have at school.
