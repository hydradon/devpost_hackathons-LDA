Inspiration

We grew up in low income neighborhoods and graffiti was always a problem, there is "tagging" almost everyday on a house or wall. Street art is sometimes mistaken as graffiti and will get erased or painted over.

What it does

Differentiate the difference between the graffiti and street art.

How we built it

we used machine learning and fed it a bunch of images to be able to differentiate the difference between graffiti and street art.

Challenges we ran into

We didn't know tensor flow nor did we know machine learning so it was a steep learning curve trying to learn it all.

Accomplishments that we're proud of

learned how to use machine learning to train images.

What we learned

machine learning/ tensorflow

What's next for Graffiti Locator

the next thing would be making it global and being able to use it all over the world to differentiate the difference. We also plan on making it crowdsourced to be able to have people detect and report the graffiti
