Inspiration

We wanted to design a web app that allows a user to get a fitness plan tailored to their capabilities and lifestyle that is simple and clean.

What it does

Generates a fitness plan for a user based on their goals, experience level, and number of days they want to exercise each week.

How we built it

It was created using the MEAN.JS stack along with Materialize for CSS styling.

Challenges we ran into

Setting up the database and understanding what was needed from the MEAN stack for this project was time consuming.

Accomplishments that we're proud of

The clean interface and algorithm for determining a user's workout plan.

What we learned

Making a MEAN stack application can be very difficult to tackle efficiently.

What's next for git fit

Add more options for users to get a more specific workout plan. Also add the ability to allow the creation of custom workout plans and include professional workout plans from top fitness programs.
