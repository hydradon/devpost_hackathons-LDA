Inspiration

Parking is a big hassle, we want to make it easier.

What it does

Scan the NFC card, then it reads if a car is in the parking lot. If so, records the information onto the database. Will calculate prices after the car drives away (does so automatically)

How we built it

Servers connected to pi and dragon board that send information through requests.

Challenges we ran into

Dragon board compatibility and integration.

Accomplishments that we're proud of

WE GOT IT WORKING!

What we learned

Integration can be as hard as implementation

What's next for NFCity

Block chain based encryption, apple/android pay
