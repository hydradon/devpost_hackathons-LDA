What it does

Read out sensor values from a Thunderboard and transmits them in a nice, readable format

How we built it

Using a Raspberry Pi and a thunderboard connected over Bluetooth LE

Challenges we ran into

Thunderboard has some connection issues (Connection between Thunderboard and Laptop is impossible, wasted 7 hours on that) Thunderboard firmware is rather fragile

Accomplishments that we're proud of

We can read out the sensor values.

What we learned

Take it easy :)
