Inspiration

Made by introverts, for introverts (everyone, really), Studious aims to help connect university students to each other for study sessions in a discreet manner.

What it does

Allows users to post about study groups, find other users through search, and join them. We hope to implement a room reservation system as well.

How we built it

Uses Electron (desktop reservation management), and JavaScript to create a platform that enables powerful search and database functionality. We experimemented with using Django (database) & Algolia (search), and will migrate to Firebase Cloud Firestore for our backend database system.

Challenges we ran into

Setting up the database for post storage was incredibly difficult.

Accomplishments that we're proud of

We all learned a great deal and that's what matters the most.

What we learned

We learned a lot about JavaScript's quirks, JSON, API integration, along with fundamentals of the Electron framework.

What's next for Studious

  • Map: Automatically locate nearest university, drop pins for latest updates on homepage
  • Search: Autofill options when typing
  • Home: Dynamically update to latest sesh postings
  • Reservation Management: Coming soon...
