Inspiration

The rubix cube from Statefarm.

What it does

As time progress while you try to solve the rubix cube, the rubix cube provides commentary on your performance. The longer you take the more rude the cube becomes. It provides comments every 15 seconds, and as time progresses the more "spicy" the comments become.

How we built it

We built it using React JS.

Challenges we ran into

Implementing the cube to be interactive with the user was a challenge we ran into.

Accomplishments that we're proud of

Adding the voice feature while the user solves the cube was our main goal which we were able to accomplish.

What we learned

We reinforced our understanding of React JS, and taught another team member about these fundamentals along the way.

What's next for Sassy Cube

Being able to re-shuffle itself repeatedly after a certain amount of time has passed while the user is trying to solve the cube.
