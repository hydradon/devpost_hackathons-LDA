Inspiration

Code for good! We wanted to increase voter participation and registration.

What it does

Allows the user register to vote by scanning their driver's license.

How I built it

We used BlinkID sdk to use the scanning aspect. Once information has been pulled from the license, we send the information to Rock the Vote API to generate a pdf that has been filled out with the user information.

Challenges I ran into

Working with Rock the Vote API's poor documentation. We had trouble getting the information from the BlinkID scanner.

Accomplishments that I'm proud of

Ambitious idea that could make an impact on the election.

What I learned

Learned more about HTTP requests and JSON objects.

What's next for Cast My Vote

Get more functionality. Improve UI/UX.
