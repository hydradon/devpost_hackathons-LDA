Inspiration

Real life experience of our team members

What it does

We tried to build android application that allows patients in hospital to scan their wristband qr code and see their scheduled examinations

How we built it

We built android app using Android Studio, we used ZXinq library to capture and decode QR code and Back4ap for model database

Challenges we ran into

We have never developed anything in Android Studio so we pretty much had to learn everything from the start

Accomplishments that we're proud of

It kinda works :)

What we learned

Our main goal coming here was learning and thats also why we chose a project that we did not have any experience in. It was a bit frustrating in the beginning but after we saw that our app kinda works, it was worth it.
