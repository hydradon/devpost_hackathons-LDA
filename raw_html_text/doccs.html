Inspiration

Difficulty finding documentation especially before a close deadline.

What it does

Doccs allows related documentation to be embedded inside the document itself. Users can use markdown to style text.

How I built it

Doccs was developed in Visual Studio 2015 using HTML, CSS and JavaScript.

Challenges I ran into

Accomplishments that I'm proud of

Completing development and compiling marketing and promotional material in time for the hackathon deadline.

What I learned

Developing an Office add-in.
