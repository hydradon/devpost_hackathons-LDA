For AUHack we have developed a sensor system documenting data relevant for a farmer's everyday use. With this sensor system the farmer can see the humidity if the soil of his fields and air, plus the temperature and wind speed.

At the same time, the system doubles as a sensor measuring vibrations in the earth. Our idea for this is for this data to be public. This way the public will be able to see if there are any possibility for earthquake forecast financed by farmers giving them the opportunity ti help others whilst streamlining their farming and using less water.

Our domain for the Domain.com challenge: farmerstalking.com
