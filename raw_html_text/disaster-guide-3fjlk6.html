Inspiration

How can we help people in the event of a natural disaster?

What it does

It allows users to access all necessary information about a natural disaster within their current location without having to find multiple websites on one topic. The information on disasters and nearby hospitals can be updated live through a python file that we created.

How we built it

HTML and Javascript using Google Maps API

Challenges we ran into

It took us a while to figure out how to add markers and custom markers, and getting the user's current location to the Google Map.

Accomplishments that we're proud of

Getting custom markers to work using an array from a custom JSON file.

What we learned

We learned Javascript and how to use an API that fetches information from a service like Google Maps.

What's next for Disaster Guide

The next step in improving Disaster Guide would be to use our python file to update information live so that users will have accurate, current data about natural disasters in the area and where to go for help or relief.
