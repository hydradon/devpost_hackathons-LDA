How to use this app:

1.) Login using Facebook
2.) Click on Create NewsLetter Button and scroll down
3.) Select who you are sending the NewsLetter to (Eg. Facebook Friends or Group Members)
4.) Select NewsLetter Subject(Eg Weekly, Bi-Weekly or Monthly NewsLetter); 5.) Select Facebook Greetings Swag Card that you will send along with the NewsLetter
6.) Enter all the Facebook Groups Post Titles and their Clickable Links one by one on the Text box and Send as Newsletter.
7.)Use Facebook Share button and Facebook Send Dialog Button to share on your Facebook Timeline and to a specific Facebook user in HIPAA Compliance Mode respectively.

FaceBook Technology Used:

1.) Facebook Login Graph API: Used to authenticate Users
2.) Facebook Shared Button: used to allow user to share content to her Facebook Timeline
3.) Facebook Send Dialog Button: used to allow user to share content to a specific user in HIPAA Compliance Mode.
