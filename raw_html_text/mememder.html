Inspiration

Who needs men when you have memes??

What it does

A program that sends you memes via text tailored to the events you planned out on Google calendar.

How we built it

Use Google Calendar API to get events from Google Calendar in email. Event names are then sent to IBM Watson's Tone Analyzer API. The generated result is sent to Meme API to get meme corresponding to the sentiment of the tone. Finally, the meme is sent via Twilio API.
