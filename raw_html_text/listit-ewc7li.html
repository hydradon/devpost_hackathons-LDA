ListIt

Mangohacks 2018 project

A Visual Todo List WebApp.

Started with the ambitious goal of building a full web application with a new framework and little web development experience.

Due to this inexperience, many technical hurdles were encountered along the way. We quickly narrowed the scope to simply implement the main feature: Adding an icon to your list item via laptop/phone camera.

We successfully learned the basics of the framework and got the key features working.

It was rewarding to be able to accomplish this, considering our level of experience and we will continue to evolve the web app beyond it's current state, so that it can be a useful tool for organizing lists in a visual way.
