Inspiration

Growing up I remember my mother applying for my siblings and I to be on the free lunch program. I felt it would be a great way to give back and support a system that has helped so many families.

What it does

It is an easy to use electronic form broken down into components which save data and takes the user step-by-step through the requirements of the application.

How we built it

We designed this application using Angular JS and UI router to render the form templates. We set up a database with a User, Adult and Child model. This data was then captured and saved to be reviewed on the final page of the application.

Challenges we ran into

Syncing states with data and CSS.

Accomplishments that we're proud of

We mapped out a plan, improved on it and continually progressing on the project.

What we learned

We learned more about how to work with a team and building a multi-template form in Angular

What's next for USDA Free and Reduced Lunch Application

Next, is to begin testing, continue to improve product and start signing families up for the free lunch program.
