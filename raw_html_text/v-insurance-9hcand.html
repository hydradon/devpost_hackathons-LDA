Inspiration

Instead they prefer to get work done from small chat window-Facebook. So let’s get started on Facebook Messenger https://zephoria.com/top-15-valuable-facebook-statistics/

What it does

Using FB-Surance Messenger bot you get insurance Quote, can buy insurance policy and can download policy documents.No need to fill multiple forms. Just chat with Bot and done.

How we built it

Built using Facebook messenger platform, nodeJs, Express, API.ai and Solartis Insurance API
