Inspiration

We feel that the online education portals in Singapore does not meet the student's needs. Hence we decided to incoporate AI technologies to aid the students' studies

What it does

Uses AI to automatically create a time table for students to follow and also creates a logical diagram of data based on their conversations with their teachers.

How we built it

We use nodejs as the server and bluemix api for the AI to create the web application

Challenges we ran into

It was difficult to implement the AI as it was our first time using such technology

Accomplishments that we're proud of

Manage to tune the AI to an acceptable accuracy

What we learned

Learned how to use bluemix api

What's next for StudyLah!

Add more features and gather feedback from students in Singapore
