Inspiration

We wanted to design and create a game from scratch using Unity's game development platform.

What it does

Players assume control of a character that is spawned in a procedurally generated maze. They can shoot other capsules to kill them.

How we built it

We used Unity's game development platform and integrated SDKs. We coded in C#.

Challenges we ran into

Learning Unity essentially from scratch, and collaboratively working on software (linking git)

Accomplishments that we're proud of

Learning the basics of Unity and a little bit about C#. Finishing many core aspects of the game.

What we learned

Unity & C#

What's next for MaizeBalls

Multiplayer capability, extra features
