Inspiration

The inspiration behind this application is to solve the critical problem faced by blood banks / hospitals to get voluntary donors when in need.

What it does

The solution is in two parts

  1. On boarding of voluntary donors

  2. A mobile app is created to get donors registered on this service. And they will be notified when there is requirement blood donation which they can fulfill.

  3. Finding the right set of donors when in need

  4. Hospitals / blood banks can send request via mobile application to reach out the possible donors for their current requirement ( for e.g platelet donation)

How we built it

  • We built this application using various technologies like "ionic" for building mobile applications, "symfony" for creating web-apis, and cool orchestration framework called "docker"

Challenges we ran into

  • Push Notification Issue Push notification callback was not getting called when application was running in background. Finally we found a workaround to solve it.

  • Fine tuning the algorithm to select donors for the request sent by the hospital.
