Inspiration

When I select my courses, if I want to get professor's rate. I need to copy the name and pasta to the search bar in rate my professor. Sometimes there are many professors with the same name but different school so I need to search again, which is really inconvenient.

What it does

Get link of the Rutgers professor when your mouse hovers the name of the professor and you can click it to see the professor in ratemyprofessor.

How I built it

It is a Chrome extension built with html, css and javascript(main).

Challenges I ran into

I need to get hover action and get the link of the professor and in our school.

Accomplishments that I'm proud of

It can search many professors and much more convenient than before.

What I learned

At first, I want to use ratemyprofessor api but it doesn't work so I need to find other ways. Like get a link of the professor.

What's next for WebReg Helper

Direct add rate in the webreg and course planner.
