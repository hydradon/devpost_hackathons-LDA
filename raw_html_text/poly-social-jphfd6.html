Inspiration

We wanted to do something so students could have an easier time finding clubs that work with them, as well as study groups that work with their needs and their schedule.

What it does

Poly Social provides one platform for students to be aware of the events that are happening on campus. It also provides an environment for them to engage with other students to create study circles.

How I built it

We built an interactive mock-up using Adobe-XD. From there, we used Android Studio to create the app.

Challenges I ran into

There were many challenges my team and I ran into while using Android studio. We had never used it before, and it took quite a lot of getting used too. One member is also very new to java, and none of us had really used xml before.

Accomplishments that I'm proud of

I was able to create a navigation screen that went through four different screens and created the UI for the screens. While not all of the screens were functional, I was pround that we were able to figure out the UI and work towards a working app.

What I learned

I learned a lot about mobile devolopement and how to use layout and UI.

What's next for Poly Social

Hopefully adding extra funcitonality and being albe to created a very easy to use way for poly students to get connected with eachother.
