Inspiration 

It can be tedious to schedule events through email, since you got to decide on a time then
go to your calendar and then set it up.

How it works 

Alfred allows you to schedule an event directly through your email, all you do is write a 
command inside your email message and once you send the email the reciver will know when it is and
an event will automatically created on your calendar.

Challenges I ran into 

Coordinating between different API and the permissions/tokens 
they need, parsing the responses, timezones.

Accomplishments that I'm proud of 

The project is done using all free software

What I learned 

contextIO/google API

What's next for A Project 

Improving parser and commands to match natural language better, and adding more functionaltiy
to manage your events through email.
