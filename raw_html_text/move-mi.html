move-mi

Windows Only

Entering for Health/Wellness Hack and the Use of Adobe Tools

A series of games using the Leap Motion designed to aid people with muscle atrophy in regaining fine motor skills while still remaining entertaining.

Built with C# in Unity using the Leap Motion SDK.

Electron used for game launcher.

Adobe Photoshop used for all graphics that were not built-in Unity assets.

Domain Name if we were to make it a web app: http://www.GitMeMoving.com

*.exe files are found in the ElectronApp/app folder, if Electron is not installed.
