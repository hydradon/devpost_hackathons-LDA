Inspiration

I inspired in the UiPath Automation Ideas Contest.I interested to develop these custom activities. This package is having the two activities called Zip and Unzip. The activities are having the provision with password.

What it does

1.Zip - Compress the files and folders as zip file. 2.UnZip- Extract the compressed file(zip file) in a specific folder

How I built it

Using the C#.Net and Ionic.Zip library
