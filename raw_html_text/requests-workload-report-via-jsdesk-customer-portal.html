Inspiration

Each month our customers were asking:

  • "How much time have you spent on our request?"
  • "Do we have any support time left?"

What it does

This is a Server JIRA Service Desk add-on. It provides real-time work logged information directly to the customer via the JIRA service desk portal. Information provided is configured by project. Customers can filter and store the results in CVS file.

Challenges we ran into

There is no standard way to show additional information on JSD Portal.

Accomplishments that we're proud of

We've done it :)

What's next for Requests Workload Report via JSDesk Customer Portal
