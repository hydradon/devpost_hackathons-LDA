Inspiration

We were inspired by how much we used to get bad grades and could've really needed a tutor to help to make our moms proud.

What it does

After you select which course you're seeking help with, it links tutors with tutees in an Uber-esque fashion, showing a list of nearby available tutors in real time, their location, and estimated walking distance. We also used Twilio to create an AI-powered chatbot that users can text to find nearby tutors for a subject.

How we built it

Had two people working on frontend, two on backend. Used Vue, Express, MongoDB and Node as our stack.

Challenges we ran into

Learning on the fly, collectively having little experience in Web Dev apps. Google Maps API had a weird issue w/ billing.

Accomplishments that we're proud of

Built a fullstack without having all the knowledge to do so prior to this weekend.

What we learned

How to use Vue, how to collaborate.

What's next for TutorTracker

Adding tutee and tutor profiles.
