What it does

Allows journalists to ask for truths to be verified using the incentive of Eth. Anonymous users can create rings signatures from users that have been previously verified that are a part of a similar affiliation (e.g. governmental organization), in order to present truth, while hiding the whistleblower.

How we built it

Using ring signatures to do pseudo anonymous release of information.

Challenges we ran into

Gas limit and size of the contract, use of robsten network.
