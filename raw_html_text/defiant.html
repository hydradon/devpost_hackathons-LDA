Inspiration: Inspired by old style side scrolling shooter video games from the classic nes days

What it does: old style side scroller shooter with random spawning 3 enemies, each with a different form of attack

How we built it: we made it in unity with c

Challenges we ran into: complications with game over screen, getting the various enemy behaviors to work right, proper collision detection and reactions, bullet trajectory stuff, adding sound effects, explosion animation.

Accomplishments that we're proud of: we made a game

What we learned: how to make a game
