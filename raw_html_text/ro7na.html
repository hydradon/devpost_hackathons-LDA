it all started with a trip with my friends, we went in the car visit three different places, we noticed that there is a lot of pictures of lost children or mentaly retarded people, and we thought why there is no application to help those families to find their members quickly, we heard about LHD and it is a chance to start realizing this idea

what it does

findTheLost is a simple application that allows families to post a picture of their lost child (or any one of the family) and search among the pictures of found people posted before, if someone found a lost child in the street for example, he takes a picture for him and post it in the app, findTheLost will compare the image posted with the children's pictures posted by families (using azure face recognition services) , if there is a high ressemblance rate, the app will notify the family about it, and send them the picture posted with the information of the person that posted it,

How we build it

we decided to realize this idea as a mobile application, for this we choosed ionic 3 to make it multiplatforme, and we used azure api to make image processing and face recognition, consuming this api using asp.net core

Challenges I ran into

we had a some difficulties in connecting our ionic app to the asp.net backend
