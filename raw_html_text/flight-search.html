Inspiration

Idea was to build a flight search which finds cheapest flights between 2 airports through any number unspecified airports to allow cheap travel for adventurous people

What it does

Just lists flights from an origin

How we built it

Used Skyscanner API to get the prices for the cached flights.

Challenges we ran into

The number of cached flights we were able to get from an origin was limited and we did not have enough time to finish everything

Accomplishments that we're proud of

It works to some extent,

What we learned

How to learn Python and APIs

What's next for Flight Search

Use Ryanair api and make optimisation through multiple cities. Develop a proper UI for better user experience.
