Inspiration

This problem was posted by Pfizer. Archhacks being our first hackathon, we thought this problem was more generic and doable.

What it does

It enables a patient to enter their medical record electronically using a web app and refer, use it later

How we built it

We chose MEAN (MongoDB, Express, Angular, Node) Stack.

Challenges we ran into

We could not persist our data in mongoDB. We explored and tried few other options (Angular,Java,Spring) that didnt work out. Integrating frontend and backend.

Accomplishments that we're proud of

We were not familiar with any of MEAN stack framework. But we were able to setup the project and create four pages and preserve data using Node JS. Even though we couldn't implement the end to end functionality, we are proud that we were able to create a working app at end of the Hackathon.

What we learned

Hackathons are a fun filled way to learn new technologies and practice those we are familiar with How to create a MEAN stack web app Time Management Project Integration

What's next for Pfrecords

We hope to implement more features that we have brainstormed. (Cloud Storage, Access across Multiple providers)
