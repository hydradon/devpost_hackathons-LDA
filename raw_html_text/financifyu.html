Inspiration

We were inspired by online banking in general and trying to make sense of financial information.

What it does

FinancifyU provides insight about your financial data. Using Capital One's Nessie API, FinancifyU retrieves all sorts of financial data linked to you and put its in a nice format for viewing. This includes upcoming bills, recent withdrawals, and data visualizations.

How we built it

We started with a nice HTML template for our site. As the site was being developed, we also wrote Python code to handle Nessie requests. As both progressed, we used the Flask framework to tie everything together.

Challenges we ran into

We ran into issues trying to host our site and trying to get all the pieces we wanted to work together.

Accomplishments that we're proud of

Having a nice looking site that we can demo!

What we learned

Domain registration sucks and Python has a function for everything.

What's next for FinancifyU

Expand our demo's functionality and add in a 2-Factor Authorization process (hopefully passwordless)!
