Inspiration

Having dealt with being treated differently than men and belittled by everyday comments, and made to feel as not important, we wanted to educate others about feminist ideals in a way that was light-hearted and educational.

What it does

It simulates a "battle" between our protagonist, Feminisa, and a character who has been brainwashed by the evil enemy, The Patriarchy. The battle is not won by fighting with fists, but by educating the enemy about feminism, especially in the sense of intersectional feminism.

How we built it

The game was programmed using the Unity game engine, using both C# and JavaScript. The art was produced in PhotoShop using a Wacom tablet.

Challenges we ran into

Learning how to use Unity was the biggest challenge, as we had limited experience with it.

Accomplishments that we're proud of

We taught ourselves how to navigate the interface of Unity, how to code for a game, how to troubleshoot, and how to create art within a certain theme.

What we learned

The basics of Unity, how to control game flow, what to add to an interface to ensure the user knows what actions to take, how read and control user input for a game.

What's next for Feminisa

Building upon the story, more levels, and fine-tuning the educational aspects (possibly teaming up with a Women's and Gender Studies professor).
