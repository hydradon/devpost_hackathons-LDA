What inspired us?

We were inspired by r/roastme, which we actually used

What does it do?

It takes an image captured by the user's webcam and compare's their face to faces posted on Reddit's r/roastme. From there, we will return the top-rated comment (determined by upvotes and comment score)

Who worked on what?

Zachary and Noah worked on the front-end/ui for the app. Isaiah worked on building the API with AWS. Angela contributed to all parts of the project, which was impressive because this was her first hackathon! She learned skills in HTML, python, and we were all amazed at the speed at which she learned these new technologies. Isaiah implemented docking.

What bugs did we encounter?

AWS seemed to have a lot of bugs, and the documentation was not very good at finding how to fix the bugs.

What did we learn?

We learned how to use aws as a backend and use atlaskit for front end components. We learned how to efficiently compare faces and how to scrape reddit for posts.

What's next for Roast Bot?

We need to scrape more posts for Roast Bot, and users need to use like/dislike to improve the quality of roast to improve the accuracy of roasts.
