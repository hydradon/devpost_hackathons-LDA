Inspiration

The Smartcar API makes it easy to interact with vehicles to provide value to drivers in ways only conceived through mobile devices. For example, the Smartcar API helped us log a drivers longitude and latitude to then map past routes through a single page application.

What it does

Tracks driver's routes to improve a driver's travel time by proposing new routes on days with heavier traffic.

How we built it

Flask on the Backend acts as a REST API and React is used on the front end to render a map of the drivers past routes.
