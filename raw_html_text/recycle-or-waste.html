Inspiration

Often unsure about which bin an item can go into, and whether or not it can be recycled. Usually have to search online for an answer, or worse guess or throw straight into the general waste bin.

What it does

Recycle or Waste allows you to provide an item that you'd like to throw out (eg. a plastic bottle) and it'll reply stating which bin should be used.

How I built it

Using AWS Lambda and the Alexa dev console.

Challenges I ran into

  • Providing enough data to make the skill useful - still a work in progress, but continuing to add more data.
  • Upskilling in how to build an Alexa skill, as it's the first one I've built.

Accomplishments that I'm proud of

Shipping my first Alexa skill that offers a function that I use often.

What I learned

  • Lambda
  • Alexa skill development
  • Node.js
  • What items can be recycled and what can't

What's next for Recycle or Waste

  • Expanding the product database
  • Offering more tips and tricks on how to reuse un-recyclable products in different ways
