Inspiration

It always inspired me the opportunity of doing something for others. I love coding and I tought I could use it to help kids to find the same interest that I found with STEM related topics. Now, I see this as a way to show the beaty of science to others.

What it does

This skill lets you play an adventure game where you will have to overcome different STEM challenges, in a funny and creatuve way, so that children have fun while learnign!

How I built it

I built it using the Amazon Alexa resources, from AWS Lambda, to DynamoDB and S3. With all the neccesary tools that helped me build a continous and enjoyable user experience.

Challenges I ran into

I had to learn to do many new things, since I am new with Alexa, like creating a memory for every new session to keep the progress in the game, hosting the content of the skill in another web site and adding sounds and music to the skill.

Accomplishments that I'm proud of

I am sure that this skill will help many kids around the world, but also, that it helped me grow in my professional formation.

What I learned

I learned to code in a new way, not only using new tools, but a different type of user experience, a voice user experience.

What's next for Space Hero

I am planning to keep adding more levels and creative ways to make STEM fun to kids.
