Inspiration

As an assistant Alexa should become your personal secretary, too.

What it does

It dials calls for you on your regular phone. So you don't have to search, open the dialer, and search for the number any more. Instead talk to the switchboard skill.

How I built it

I made a voice interface to PBXes virtual phone systems. That's where you can connect your smartphones etc. over the standard SIP protocol to.

Challenges I ran into

Lots of, first I had to learn about account linking to connect to the right virtual PBX. Second, the skill has to understand all kinds of place names e.g. the local surf club to find the corresponding phone numbers.

Accomplishments that I'm proud of

The skill can really understand nearby place names, contact names, and numbers. So I can voice dial everyone just like talking to a good secretary.

What I learned

I learned about current limitations some really huge and some smaller of the Alexa skills kit.

What's next for Switchboard skill for PBXes

I am waiting for an API to run the actual phone calls over echo devices itself, rather than via smartphone.
