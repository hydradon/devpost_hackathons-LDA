Inspiration

People always get lazy when watching TV so I want to start making them to workout in front of their TV

What it does

Coaching video with gesture repetition counting using the siri remote or third party hardware sensor

How I built it

Using tvOS SDK + bluetooth sensor from mBient lab

Challenges I ran into

New tvOS SDK, still a lot of bugs. Also connecting tvOS with external hardware.

Accomplishments that I'm proud of

Building the first tvOS app

What I learned

Learn how to make tv app with gesture control

What's next for TVFitness

Publish on Apple TV Store
