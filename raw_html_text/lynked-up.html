Inspiration

Many people are without degrees or formal education, but need a job. Here's the solution

What it does

It let's people enter their expertise and experience in almost any field, based on which they are offered local jobs. You can request for any job you need help to get done, and you can be paid to get done a job for someone else.

How I built it

We used the environment CodePen and the languages, HTML and CSS.

Challenges I ran into

Linking one page to another took us a lot of time to figure out. Inserting Text Input took much research to execute.

Accomplishments that I'm proud of

Being able to complete a full skeleton of a potentially very useful website is very impressive for us.

What I learned

Much basic coding about HTML and CSS and the logistics of a website

What's next for Lynked Up

Expanding so users will be able to access this website
