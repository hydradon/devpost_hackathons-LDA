Inspiration

Many people we know have dietary restrictions that make it complicated to cook for everyone, this app is the app we needed to choose a recipe that works for everyone.

What it does

Allows the user to input several dietary preferences and restrictions and generates recipes that fit all parameters.

How we built it

Built with react native using spoonacular API to select recipes based on restrictions.

Challenges we ran into

Challenges we ran into include learning react-native, and understanding the inner workings of spoonacular API.

Accomplishments that we're proud of

Creating a responsive recipe list based on multiple variations of inputs from the user.

What we learned

How to user react-native and integrate the library with an API.

What's next for SnackCheck

SnackCheck will incorporate a google vision API to let the user take a picture of a food item, and display information on the safety of consuming based on their dietary preferences and restrictions.
