Inspiration

We got inspired from Nosedive episode of Netflix's Black Mirror series and real life personal experiences.

What it does

It provides a intelligent unique identifier rating value by aggregating data on multiple sources included it's own rating system.

How we built it

We built a server with Django, an Android client App, and a website! We used sms authentication creating a very quick register/login. We registered rated.ai during the hackathon, but due to domain registrar delays, we had to register everyrate.io for the presentation.

Challenges we ran into

Creating a unique database of entities in a couple of hours by fetching rating data based on businesses and places ID’s on multiple data sources such as Google Places and Foursquare. Second challenge was to simplify UX at the highest level.

Accomplishments that we're proud of

We created a fully featured product from scratch in less than 15 hrs and we think it will help users solve a real problem.

What we learned

API’s are becoming so powerful that you can build real products in hours.

What's next for everyrate.io

Integrating unstructured information through text analysis from multiple sources such as e-commerce and social media to better define each entity unique identifier rating value. Add people database and transactional ratings by fetching public profiles, users social media friends and rated.ai platform users.
