Inspiration

Have you ever Googled "wings near me" and gotten a seemingly infinite search result of the same restaurant? Well, Wing Drop is the one stop shop for the best wing drop!

What it does

It is a way to crowd source the best local wings in one place. Google just doesn't do it for anybody anymore and all the reviews don't seem accurate.

How we built it

We build it using Flutter and the Google Cloud Platform for the Maps API

Challenges we ran into

We did not know how to use Flutter or Dart. It was a big hurdle for us to overcome.

Accomplishments that we're proud of

Our design team did a killer job with the design and logos.

What we learned

We learned new tools, new languages, and the importance of having a good idea.

What's next for Wing Drop

Next is to get people to use it! To use the data acquired to make the app better. We also want to expand to different products as well.
