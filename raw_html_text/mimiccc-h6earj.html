Inspiration

We were inspired by what we thought we could do. We all have a background with Dungeons and Dragons and Tommy has a background with GoLang. We worked together to tie it all together.

What it does

Its a simple bot that responds to commands introducing new players to concepts from D&D.

How we built it

We started out looking up how to create a discord bot, and then found a freeware github on discordgo api.

Challenges we ran into

We originally wanted to make this more conversational, however, we couldn't make a scanner work properly to read the next line in the discord server via GoLang. We also ran into an issue storing variables for a longer time, we eventually got around this, but it didn't leave us enough time to implement the database we wanted to use.

Accomplishments that I'm proud of

It's a working Discord Bot. Its using GoLang.

What's next for Mimiccc

Spells Function, a full list with stats Database integration for full character sheet generation within discord.
