Inspiration

Some billboards and magazine ads show some great locations, but they don't show where it is, now there's a way to know where it is and actually know how to get there from within one app

What it does

Snap a photo or upload one to, and using Vision API the application will not only return information about the place, but also how to get there

How I built it

Azure vision API and API from an online travel agency

Challenges I ran into

Not all places are recignized and not all places has a major airport near by

Accomplishments that I'm proud of

Creating a practical application that implements Azure Vision API

What's next for Snap and Fly

Improving the performance when calling these APIs
