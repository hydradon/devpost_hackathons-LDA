Inspiration

Picture is worth a thousand words. Animated is worth a million words.

What it does

A chrome extension to allow users to record any web page and save the result into an animated gif. The initial use case for this extension is for UI/UX designers, developers, or product managers to save good user flows. Currently there are no web or native apps to streamline this process.

With Frames, user can record anything on the web, process the recorded image sequence, and then save and share the animated GIF all within the browser.

With Gfycat, the users will have a platform to collect and share all their inspirations in a animated form.

How I built it

Gfycat Api, Chrome extension, React/Redux, Nodejs

Challenges I ran into

Enhancing image processing performance on all image sequence.

Accomplishments that I'm proud of

Created a fully functional app in 24 hours!
