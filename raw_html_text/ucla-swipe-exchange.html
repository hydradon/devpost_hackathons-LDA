• Home Page Where Sellers Can List Offerings

  • Buyers Can Select Restaurant & price

  • All Offerings Available for Buyers

  • Receiving the Seller's Phone Number From Twilio API

We wanted a simple interface that allows users to buy & sell dining hall food passes & an online e-commerce site was the perfect way to do so. Thus, we utilized AWS DynamoDB coupled with Flask as the backend to store user data and utilized twilio Messaging API to connect users through text message, allowing for easier purchase & selling of dining hall food passes
