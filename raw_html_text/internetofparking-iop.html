Inspiration

Almost everyday we have this problem: we need to park our car but we don't have the time to waste searching for a parking lot. InternetOfParking aims to solve this problem.

The inspiration is a practical case that is implemented using our vision for the product:

"Allow the access to ‘geographical and physical services’ (parking slots) managed by a Partner (Parking Area Manager) on a Digital Channel for all the Subscribers/Consumers of a Telecom Operator (CSSP)"

What it does

It helps the user to find a parking lot with empty spaces. It also enables the parking lot owners to have more customers.

How we built it

Using C#, ASP .NET, TMForum API, Huawei API and City of Lisbon API.

Challenges we ran into

Working with people that we didn't knew before, with a short deadline.

Accomplishments that we're proud of

Using TMForum API, Huawei API and City of Lisbon API in a simple manner.

What we learned

The using of mature APIs is easier, efficient and effective, rather than develop everything by ourselves.

We also learned that this is the future.
