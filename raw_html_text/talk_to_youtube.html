Inspiration 

People like talk when they are watching videos.

What it does 

It is a Chrome app that recognizes the user's speech and generates text under a YouTube video player.

How I built it 

I made use of webkitSpeechRecognition API and textillate text animation API

Challenges I ran into 

Making Chrome app running on YouTube site is not that easy.

Accomplishments that I'm proud of 

Making Chrome app running on YouTube site.

What I learned 

 It is better to plan ahead when doing a hackathon.

What's next for talk_to_youtube 

 Record text and generate record files on user's disk.
