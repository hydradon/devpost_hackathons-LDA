• Wake App - Home Page

Inspiration

Say you set a song you like as your wake-up alarm. After a while, don't you get sick of it? Or your brain might begin to ignore it. The idea behind Wake App was to change up your alarm sounds.

What it does

Wake App allows the user to set an alarm at a specified time. The alarm sound is selected randomly from a library of music.

How we built it

Wake App was built using HTML5, CSS3, and JavaScript.
Third party code used included that for the alarm clock.

Challenges we ran into

All four of our team members are freshman students currently taking CS1050. We had very little experience with coding besides our limited knowledge of C programming.
We had trouble (but eventually figured out) how to embed playable audio. It took a long time to make the random selection of songs from the library work.

Accomplishments that we're proud of

Successfully creating a website with functionality.
Having a working clock. Adding Javascript to make website interactive.

What we learned

We learned/reviewed HTML. Gained experience with JavaScript and CSS.
Experienced working in a team on a deadline, solving issues as they came up, and learning as we went.
