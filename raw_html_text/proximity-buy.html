Inspiration

Our inspiration is shopify it self. We want to build the future of the commerce, and that can happen in the physical world.

What it does

Its a mobile advertising system where Developers can integrate our SDK to their apps and easily start to earn money with ads and the stores can create campaigns for their products and the costumers near by will be aweare of it by using the beacon technology.

This product can be a feature for the shopify's costumers who also has physical stores. Shopify's clients can buy it as a monthly plan and the company would send the beacons to their stores address plus the admin panel access for the management of their campaigns

How we built it

We build the ios native version using swift and iBeacon library. The server side was PHP + mysql for the API and angular.js + bootstrap for the client.

Challenges we ran into

Too many things to do, trying to learn about the new technology. Work in team and avoid bugs by planning in a short time and too much to do.

Accomplishments that we're proud of

Created the app sdk version for ios and the api + client server for creating campigns for stores and monetization for developers.
