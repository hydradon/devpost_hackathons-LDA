Inspiration

Inspired by the current controversy regarding political bots on social media sites such as Facebook, we took on the challenge of analyzing Twitter handles to determine the presence of automated bots.

What it does

Our web extension provides an analysis of Twitter handles to determine whether the user is a bot or just a troll.

How we built it

Python was used to analyze the data. The extension was built in HTML, CSS, and Javascript using Sublime Text.

Challenges we ran into

There was some trouble at first with implementing the Twitter API.

Accomplishments that we're proud of

We're proud of data we scrapped from the Twitter API.

What we learned

We learned how to create an extension and we're pleasantly surprised by how simple it is.

What's next for Bot or Not

Further investigation into machine learning and the mysterious world of Tensor Flow.
