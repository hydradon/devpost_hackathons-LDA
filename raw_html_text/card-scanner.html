Inspiration

We wanted to use an OCR program for something useful.

What it does

Card Scanner reads in names, phone numbers, and emails from business cards, and also links the user to the LinkedIn profile.

How we built it

We used Android Studio to create the app.

Challenges we ran into

We couldn't get the LinkedIn API to work.

Accomplishments that we're proud of

We managed to make the app work.

What we learned

We learned how to use Android studio to make apps.

What's next for Card Scanner

Improve the UI, add faster recognition, better accuracy.
