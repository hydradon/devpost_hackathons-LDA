Inspiration

Recently I've been fascinated with the different ways people communicate when they're virtual vs real. Anonymous vs Transparent. You're a lot more free to be yourself when you're not being held accountable. So I thought I'd make an app about it.

What it does

Anonymously chat, one-on-one with the world around you. Confide in someone you normally wouldn't. Please be nice!

How I built it

Solo Dolo

Frontend was in java and android studio. Backend was a restful api built in golang and cloudhosted... Made some pixel art, Ditched that, went with colors

Went piece by piece. Laid down basic framework. Then built, Jumping between front and back end and weaving it together. First 12 hours went to Android Studio (fragment adapters and recycler view pains)

Map everything out but let the ideas flow and evolve. Had a lotta fun too!

Challenges I ran into

My initial-partner skipped out on the hackathom so that was a complication. I think it actually mighta helped though, i focus well under pressure.

This was my first time using Android Studio. Most of my issues came from unfamiliarity.

Fragment Controllers (Android Studio Stuff) .. Asynchronous Errors

Superglobals helped a lot. I ended up centralizing around a singleton class for organization's sake.
