Why?

Have you ever felt a desire to cook something, but things like complicated recipes, long cooking time and weird equipment stopped you from doing it? Well, with Feed Me, this is not a problem anymore!

All you have to do is to answer several simple questions and your perfect match will be found!

How?

We created a web application, using Flask framework and JS, HTML, and CSS to make it pretty :) While working on the project we run into lack of free recipe API's, but we overcome this challenge by creating a local database.

What's next?

In the future, we're planning on enlarging our database or maybe writing a web scraper. Also, we would like to implement an AI that will train while you're using an app more and more and showing your more personalized recipes. In addition, we will try to sneak in questions about the user's preferences, allergies, medical conditions, and habits, while the user is surfing our app.

To learn more and try it out, please find us near the TeamRoom (next to the football table)!
