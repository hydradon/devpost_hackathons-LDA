Inspiration

What it does

It is a real-time translation that can allow two people to text each other and automatically sends a translate response in the user's native language. For example, if I am talking to a Spanish native speaker, the app will send the user a translated version of their response to me.

How we built it

We used React-Native along with Send Bird API and Google translate API

Challenges we ran into

Building the backend server of the messenger app.

Accomplishments that we're proud of

Happy we built this project in React-Native.

What we learned

We learned how to use React-Native and how amazing it is.

What's next for Project Bable

Have a working version of this app and expand into a group chat.

The future of Bable

Bringing unity to everyone across the globe without worrying about language barriers. We also want this product to be used for international business.
