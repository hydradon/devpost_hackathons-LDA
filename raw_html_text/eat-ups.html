Inspiration

The inspiration comes from the New Year's resolutions people often make to get fitter.

What it does

It is an endless game that challenges players to help Teddy get fitter by doing chin ups.

How we built it

We built it using HTML's canvas element. We created and manipulate game object using JavaScript.

Challenges we ran into

We ran into problems with making our game compatible with phones.

Accomplishments that we're proud of

We're proud to have made a complete game using JavaScript, a language we are all unfamiliar with.

What we learned

We learned to use JavaScript to make a game.

What's next for Eat Ups

Not sure what will come next for Eat Ups.
