Inspiration

O Social Trip é um site que tem o objetivo Inspirar destinos de viagens a partir da integração com os “likes/curtir” do usuário na rede social Facebook.

What it does

Ao entrar no site, o usuário deve clicar no botão Facebook e logar com a sua conta. Ele solicitará permissão para acessar e... Pronto!!! Com um click, o perfil de viajante do usuário é traçado e são sugeridos diversos destinos de viagens que tem a ver com as preferências do usuário!

Além disso, clicando em Detalhes, o usuário terá todas as informações sobre voos e passagens para aquela localidade

How I built it

Tecnologias/Ferramentas/APIs: Node JS, MongoDB, GIT, OpenShift, API de integração com o Facebook, API de integração com o Sabre.

Challenges I ran into

Integração com o Facebook e com a API Sabre

What I learned

Integração com o Facebook e com a API Sabre

APIs Sabre utilizadas

Lead Price Calendar para traçar roteiros entre aeroportos.
Busca de aeroportos próximos a localização atual do usuário.
Buscar aeroportos nas cidades que a aplicação sugere como destino.

What's next for Social Trip

Implementação de sistema de compras de passagens. Melhorar algoritmo para traçar o perfil do usuário a partir dos dados encontrados no facebook.
