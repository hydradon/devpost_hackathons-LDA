Inspiration

Binary emotion prediction.

What it does

Analyzes text to determine happiness or non-happiness. Also has a short quiz to provide the same feedback & give an inspirational GIF.

How I built it

Using Google's AutoML to upload datasets to train & then predict text. Basic HTML, CSS, and JavaScript for the web.

Challenges I ran into

AutoML took a while (several hours!) to train.

Accomplishments that I'm proud of

Using Machine Learning to in a relatively practical application.

What I learned

Machine Learning.
