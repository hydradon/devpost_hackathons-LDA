Inspiration

Resident Roommate is having trouble waking up early in the morning. He usually oversleeps and has trouble waking up in time. It is known that oversleeping can cause many medical problems like diabetes, heart disease, and increased risk of death. In order to prevent this, we made the Adaptive Interactive Deep Sleep Program.

What it does

It allows the user to input in how many hours they wake want to wake up and our program determines the person's sleep stage that the user will be going to wake up and plays the appropriate music based on it.

How I built it

It was built using Python

Challenges I ran into

Compatibility issues, Program Syntax incorrectly working

Accomplishments that I'm proud of

Creating a General User Interface (GUI) that allows users to interact with the program

What did we learned

How sleep cycles work and Python can predict and act appropriately based on this data.

What's next for Adaptive-Interactive-Deep-Sleep-Program

Implementation to website and application and improving its graphic design for the GUI.
