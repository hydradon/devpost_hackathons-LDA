Inspiration

Want to remember when Christmas Day is, and how long it is until Christmas Day

What it does

Tells you when Christmas Day is, and how long it is until Christmas Day. Plays a little jingle. Shows a festive image.

How I built it

Using ask_sdk_core and ask_sdk_model libraries in python running in a lambda instance on AWS.

Challenges I ran into

Required to check apl_support = handler_input.request_envelope.context.system.device.supported_interfaces.alexa_presentation_apl for device APL support - this isn't really documented anywhere.

Accomplishments that I'm proud of

Integrating with lambda, s3 and alexa

What I learned

APL works well
