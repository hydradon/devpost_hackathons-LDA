Inspiration

Inspired by street children who need help on a daily basis. Anyone in the world can see them, this was hard to implement as they don't have one place to stay, hence Refugee VR as refugees need help and they are constantly situated in one place.

What it does

Refugee VR displays information about a specific refugee camp. This information includes:

  • Basic information about the camp, e.g. location, when it was established, the current population, what's most needed by refugees.
  • The situation at the refugee camp, that's how the refugees have adapted to the environment
  • A video to explore the refugee's situation
  • A way to link users around the world with the help on UNHCR

How I built it

Used react360 library to develop the whole system. I tested it out on the web and it works fine although it ain't in the completion stage.

Challenges I ran into

  • I didn't find any 360deg videos or images for refugee camps
  • Time Constraint, as I knew the challenge late

Accomplishments that I'm proud of

Users can find information about one refugee camp currently and this is somehow helpful

What I learned

I took this project as I was leveling up on react360, it has helped me to understand its concepts and implementation.

What's next for Refugee VR

Linking up with the google map API, preferably google street view API, so that users are able to select the tagged refugee camps around the world and view its information
