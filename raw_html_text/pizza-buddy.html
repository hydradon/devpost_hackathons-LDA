Inspiration

We wanted to create a way for pizza to bring friends together, as well as streamline the pizza ordering process.

What it does

Our pizza bot acts as the intermediate between two friends and listens for keywords. When the user types "PIZZA" into the chat, the pizza bot will automatically order their favorite pizza from the nearest dominoes.

How I built it

We used Node,js to create a facebook messenger bot that constantly listens for the users to input "PIZZA". The pizza bot uses a python script to call the Dominos pizza api to order the users favorite custom pizza, from the nearest Dominos, and has it delivered to the their location.

Challenges I ran into

Coming up with a solid idea, and not having much experience with javascript.

Accomplishments that I'm proud of

Getting the basic functionalities of the pizza bot to work, and being able to order pizza at any time.

What I learned

We improved our skills with Node.js and using apis.
