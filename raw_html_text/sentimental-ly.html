Inspiration

Our goal was to build something awesome and learn about cool new APIs!

What it does

Our web app, Sentinmental.ly, shows a histogram of all posts in a certain page with a sentiment value between -1 and 1.

How I built it

We used Facebook's Graph API to collect public posts from Facebook pages. We then used the indico api and d3 to analyze and visualize the positivity in the different posts on a page.

What's next for Sentimental.ly

Aggregating the sentiment in the comments in each post, as well as incorporating Facebook's search plugin.
