Inspiration

We like to hunt trophies and thought a social view on the challenge/trophy situation could be fun

What it does

The app is about setting goals / challenges for you and your teammates to compete against each other.

How we built it

We built the mobile application with React Native and Exponent. The backend is built with Nodejs, GraphQL and MongoDB.

Challenges we ran into

Graphic user interface design is very hard. Keeping the spirit up

Accomplishments that we're proud of

Having built a mobile application that runs on both Android and iOS.

What we learned

React Native skills have been improved and it have been fun hacking with GraphQL

What's next for Challenger

Going public in the different app stores
