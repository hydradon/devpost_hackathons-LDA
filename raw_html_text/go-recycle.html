Inspiration

We were inspired to create this project because our school is notoriously bad at sorting trash.

What it does

Our web game helps teach people how to properly recycle.

How we built it

We used Node.js for a simple backend and HTML/CSS/JS for frontend. Bootstrap and p5.js (processing for JavaScript) helped significantly.

Challenges we ran into

Most of us had never used p5.js before, so learning how to use it was a big challenge. Learning JavaScript was another challenge we faced.

Accomplishments that we're proud of

We are firstly proud that we finished it. We are happy to say that we learned a great amount in completing this project. The project has potential to bring change, another fact which we are very proud of.

What we learned

The most important thing we learned was how to collaborate as a team via Github. Learning Github can sometimes take forever, so we are very happy that it didn't for us.

What's next for Go Recycle

We would like to add more complexity to the game in the form of more levels, a database, and improved webpage styling.
