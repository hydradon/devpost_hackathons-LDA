Inspiration

Have you ever received a text from a friend and could not tell if they were being sarcastic or not? It can be really embarrassing when you misinterpret sarcasm. Use this free tool to check if the text is sarcastic! ¶

What it does

Trains and uses a Naive Bayes classifier to classify phrases as either sarcastic or not

How I built it

Used python in jupyter notebooks with pandas, numpy, and sci-kit learn libraries

Challenges I ran into

Using iPython widgets, implementing Naive Bayes classifier

Accomplishments that I'm proud of

Did this project in a couple hours

What I learned

You can use html markups in jupyter notebooks! (I know, I am dumb)

What's next for R U Serious?

I hope to expand this into a mobile app that uses voice recognition as well as sarcasm recognition
