Inspiration

Many students in the Bronx don't get that much money to spend after school.

How it works

How it works is that we use a mobile app for students to find areas around their schools where they can get something to eat with whatever money they have.

Challenges we ran into

The challenges we ran into with this project include finding an idea that would fit this app and how we would integrate the foursquare api while building this app and how students will be using it.

Accomplishments that we're proud of

An Accomplishment that we are proud of is coming up with the project idea and how the mechanics of this app would work.

What we've learned

We learned that we need a determined team that will not give up.

What's next for Student Specials

Next is helping students be able to save money when needed.
