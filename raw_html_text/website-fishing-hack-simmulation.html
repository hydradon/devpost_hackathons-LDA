Inspiration

With a sudden increase in the past decade of the use of the free web, it is important that we educate ourselves on how to protect ourselves from people stealing our information. In order to do this we created a simulation of a program hackers might use to show how easy it is to steal information.

What it does

It uses php to and java script to save our password information to a database on the dev server, On the link that we submited unfortuently due to issues with payment of the domain, the site would not let us force download our keylogger which is a jar file onto to the users device.

How I built it

We used PHP, and Html for the site and saving the password information, for the actual keylogger prototype we used java and incorperated jframe.

Challenges I ran into

We could not set up a fully functioning keylogger which allows you to record all key presses on the computer, however, our version does save the key presses that the user enters into the Jframe application window.

Accomplishments that I'm proud of

Using a vaierty of coding languages, and working together with our team

What I learned

We learned how to host domains for websites for free as well as how to set it up so that we can log multiple users informations.

What's next for Website Fishing Hack Simmulation

Get a hidden keylogger which tracks all key presses into a text file in our database.
