What it does

When your you're going into labour, Mommy Update will notify your loved ones automatically through your smart watch! You can share the great news because the application will send a text once your calories expended has gone above the set threshold. So new moms can feel secure knowing that their family has been notified.

How we built it

We used the Pebble API and AWS (SNS and IoT).

Challenges we ran into

We originally planned to use the FitBit API, but we realized we didn't have a FitBit. We assumed the Pebble had similar functionality but we were mistaken.

Accomplishments that we're proud of

Correctly connecting our application with the AWS API
