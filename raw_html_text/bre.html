Inspiration

Learning about UPC codes and barcodes in our AP computer science course inspired us to make our own algorithims for encryption. Other assignments such as "kathmandu" and "butcher" were total inspirations as well.

What it does

It takes a message and goes through 3 layers of encryption, with the first rearranging letters, the second replacing characters with a chosen symbol or key and the third is our secret formula.

How we built it

Primarily using the bold Java Eclipse IDE* bold *, utilizing the GUI classes.

Accomplishments that we're proud of

First time programming a GUI for a java program and running it as an executable jar software, developing our own algorithm and doing all this with no WI-FI.

What we learned

Cryptography, GUI creation, and cohesion of classes.

What's next for BRE

Making more layers for encryption, stylizing the UI, possible transfer to mobile application, etc. We're just glad we're done.
