Inspiration

We were inspired by student life, many students are looking for fun and a break from exams, homework, and studying. Eventure seeks to ease the tension and relieve boredom by facilitating social gatherings.

What it does

Eventure allows users to promote a social event and invite users in the vicinity. The event creator will also have the option to restrict the event to certain ages & sexes. Each event will have a marker on the map and when a marker is tapped, a small window will show containing a picture (posted by the creator) and a short description detailing the event. For more information, the user can tap the window and the full description will show containing the following information: How many people are there, how far it is, and feedback from attendees.

How we built it

We built the app using Android Studios, implementing Google Maps API to include a dynamic map.

Challenges we ran into

Problems connecting Android Studios to Github or to work on all the PC's was the most challenging issue we ran into. Many hours were spent just getting the IDE to work.

Accomplishments that we're proud of

We are proud of the methodology we used to pull Eventure from the depths of our minds!

What we learned

We learned how to harness the power of the Android Studio IDE to create apps to better the community!

What's next for Eventure

We plan to complete the UI, add real-time fraud reporting, and possible link Uber and Lyft as transportation options.
