My goal with this project was to make it easy for people to get into the bitcoin lightning network and test out their new wallets

Lightning Faucet Provides a way for users to receive free bitcoin on the lightning network

I built this web app using nodejs

The main challenge with this project was limiting the frequency of user withdrawals so that everyone could receive a fair share

The website has become popular quite quickly which I am very pleased about
