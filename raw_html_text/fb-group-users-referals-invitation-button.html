Inspiration and What It Does

Introducing FB Users Referrals Button to tackle/engage all Unanswered Questions in FB Group Posts.

In most FB group discussion some questions are left unaswered simply because they was nobody who currently has an answer to it. What if you know someone who can attempt or solve the problem. This is where this FB Referral/Invitation concept comes in.

This is not a Shared Button on Timeline but a Specialized button attached to each Post Status Updates on the app that will send the Post/Question link directly to the person you know that can asnwer the question posted.

To achieve this, our application uses Facebook Send Dialog, Email and SMS components.
You can automatically send an Invite by clicking on the Invitation/Referal button attached to the post discussion to invite someone who can answer the question.

In this way, all groups post questions will be attempted and answered, it will drive more users engagement since the Invitee will now become part of the process thus keeping the group discussion awesomely going forward...

Building Engaging Updates

For Instance in Mathematics or Chemistry Group Discussion on Facebook, Most Post involved are Mathematics, Equations, Organic Molecular Structures, Chemical Equations, Engine Drawings etc that were screenshot and post on Facebook.
Now am reading the Posted Question and discover that this question has not been answered while I have Andrew, John etc on my Facebook Friends who can answer the question correctly.

Clicking the Referal/Invitation Button will give me an option to send the Post Question link only to Andrew, John etc. on Facebook prioritizing them that their Presence is needed on certain Facebook question. This will ensure that the question Posted will be attempted and answered thus growing the groups on Facebook going forward.

How to use this app:

1.) Login using Facebook
2.) All Group Post has a tag (Do You know someone who can answer this question?. please Share a link to this Post by clicking Referal/Invite Button
3.) Share the Post link to the Invitee Facebook Messenger using :
A.) Facebook Send Dialog Button
B.) To his Email or
C.) To Mobile Devices via SMS telling him that is contribution is urgently needed a particular Post on Facebook..

FaceBook Technology Used:

1.) Facebook Login Graph API: Used to authenticate Users
2.) Facebook Send Dialog Button: used to allow user to share content to a specific Facebook Friends. in HIPAA Compliance Mode.
