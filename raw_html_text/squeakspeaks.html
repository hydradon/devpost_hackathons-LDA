Inspiration

Rubber Duck Debugging. Its that moment when you are explaining your code to an inanimate object (or buddy that doesn't code) and while you are explaining it, realize your problem.

What it does

The user can 'text' the duck, and it will ask you general questions prompting you to explain your code. These are random responses that are preloaded.

How we built it

We used Android Studio and developed in Java.

Challenges we ran into

None of our team had used Android Studio or had any experience making apps, so it was hard to figure out how to start. We also had (are having) trouble implementing the chat.

What we learned

How to make apps for android, and how to yell at GitHub.

Accomplishments we're proud of

We learned how to use new technology!

What's next for SqueakSpeaks

Finishing implementing the chat, using a tone analyzer for the duck to respond in a relevant way. Maybe use Alexa or a way to talk to it and have it respond.
