Inspiration : There is no good movie manager apps for the movies on your hard disk

What it does : It scrapes through movies on local disk and gets all information about that movie from imdb, and creates a good browsing experience for the user

How we built it : Using electron API and nodejs

Challenges we ran into : Getting the proper movie names from the file names

Accomplishments that we're proud of : We were able to get an end to end solution within 24 hours

What we learned : lots of wisdom on handling asynchronous calls

What's next for ProtonMovieManager : Incorporating Machine Learning techniques
