Inspiration

When our complex Markov Chain Analysis project hit a major speedbump, we turned our focus to Amazon Alexa.

What it does

The Markov Chain Analysis project was going to develop sentences based on character probability from a sample text taken from the internet. The Alexa skill answers the question "Did we finish our RowdyHacks project?"

How we built it

For the Amazon Alexa Skill, we used Amazon Web Service and Amazon Developer to create Lambda functions to answer this question.

Challenges we ran into

For the Markov Chain Analysis program, we were getting logic errors with the arrays, and with the sentence structuring. For the Alexa Skill, using Javascript instead of our preferred language, Java, was quite hard. Two of our group members left us very early on as well.

Accomplishments that we're proud of

It works and were happy to have something running to turn in.

What we learned

Enough JavaScript to write small programs and logic.

What's next for Plan B

Develop a plan for the next hackathon and stick to it instead of hopping from project to project.
