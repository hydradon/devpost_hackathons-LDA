Inspiration

College students are looking for easy and quick ways to be more health-conscious. Energy drink consumption has been on the rise after the introduction of Redbull in 1997. We made a vending machine that only dispenses a drink or a snack if the user is able to meet certain health criteria such as maintaining a heart rate & respecting snack limits.

What it does

"Amending with Vending" allows for individualized profiles for each user based on their lifestyle. Only when certain health parameters are met, snacks or drinks are dispensed.

How we built it

We used an esp-32 to control six micro-servos, an OLed Display, and take input from a keypad. In addition, we display user information on a web server hosted by the esp-32.

Challenges we ran into

The biggest obstacle was not having experience with the esp-32.

Accomplishments that we're proud of

Achieving an operational vending machine, getting it to be personalizable, and our primary goals.

What we learned

Servo control, esp-32 as a webserver, impeccable modular coding

What's next for Amening with Vending

Integration with rising wearable electronics and health-related applications Upgrade the components with industrial parts to increase capacity/functionality Get it going around the world
