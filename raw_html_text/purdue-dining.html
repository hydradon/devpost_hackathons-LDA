Inspiration

My sister is a freshman at Purdue and they have many tasty places to eat.

What it does

My skill lets people ask Alexa what food is available at the various food court stations.

How I built it

I build it using the Alexa web skills portal and the lambda developer console. It uses JavaScript code to call a web API provided by the university.

Challenges I ran into

Learning how to do a voice interaction was very new to me and also working with the Alex device.

Accomplishments that I'm proud of

I'm proud that this will help my sister and is useful.

What I learned

Programming for Alexa is fun and can help people.

What's next for Purdue dining

I'd like to let people ask for vegetarian options specifically.
