Inspiration

Taking notes in class and sharing notes after class with your study group can be very difficult, and its difficult to pin point where a knowledge gap exists in a lecture.

What it does

Notebridge is a collaborative note-taking platform that allows you to take notes with your friends and ask them questions for clarification. Taking notes collaboratively allows for pooling of knowledge and lets your friends explain a concept to you in a new way, across a single platform. You can also track and export the most insightful points from your notes.

How we built it

We built this application using ionic framework for cross platform development.

Challenges we ran into

We couldn't create nested components for the questions and notes divide feature in the note editor.

Accomplishments that we're proud of

Using authentications plugins to manage users, and also using firebase DB to handle the notes and user properties. Building in features to track cyberbullying and harassment using our reporting features.

Cross Platform Responsive Design made with Angular.js and Firebase DB. Updates in real time, limited distraction from lectures Automatically imports list of classmates Flag features provide comfort and safety 5000 points earned == 25% off on AWS SimpleDB, AWS S3 Cloud Storage. Take advantage of curated, crowdsourced notes to optimize study time.
