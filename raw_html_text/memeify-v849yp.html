Inspiration

We like memes. And so you do, we created this platform for a continuous, infinite and free entertainment.

What it does

It scrapes memes from all over reddit and show them in out application!

How we built it

We built it using Python for the backend and scraping. And HTML5/CSS for the front-end, it uses a combination of server-side and client-side rendering.

Challenges we ran into

First we tried to build a meme recommender system, but our computer couldn't handle it so we cancelled the idea.

Accomplishments that we're proud of

We scraped over 20k memes! We learnt Flask. We built a fast and cool website

What's next for Memeify

Add user login Add social features Add localiation, so we can choose popular memes by place Split it into microservices architecture.
