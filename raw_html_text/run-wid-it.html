Inspiration

Spur of the moment inspiration from the prompt to solve a problem plaguing the tourism sector.

What it does

It connects tourists with the events, places of interest, activities and transportation in the country. The website can also be used by locals to find deals. Uses a vocabulary of tags to find similar events which in the end, tailor the selection to the user's need. Provides a means for the user to pay for tickets they require. Uses Google Maps to show the user the location of events, places of interest and activities.

How we built it

We built the website using drupal 8 to enter the content and its relationships and the styling was edited using a basic text editor (Sublime)

Challenges we ran into

We had challenges gathering data and adding Google Maps functionality.

Accomplishments that we're proud of

We're proud of the interactivity of the site with Google Maps as we were told it might have been too difficult to add that functionality.

What we learned

We learnt data gathering/research techniques as well as content editing and design.

What's next for Run Wid It

Run Wid It will hopefully be able to develop into a larger, much more robust website with a larger database of events and things to do. We would also like it to become a mobile app for easier use.
