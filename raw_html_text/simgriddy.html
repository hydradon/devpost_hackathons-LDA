Inspiration

Games can be used as educational tools and “choose your fate” games such as “Oregon Trail” have become very popular.

What it does

A user starts the game by choosing one of four characters that have varying advantages and disadvantages in a crisis scenario. The game begins and the user is able to explore the map and take actions to prepare for a looming disaster. After the disaster hits, the user receives a scorecard that provides the results and provides resources for further learning.

Challenges we ran into

Games are complex and require a lot of logic. We were quite ambitious.
