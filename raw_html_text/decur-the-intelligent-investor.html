Inspiration

The app was born out of frustration of a financial loss and inability of the general public to invest in the currency derivatives market since they cannot predict the maximum loss or risk.

What it does

It helps in predicting a bunch of technical factors using GARCH model. The factors which are predicted are Volatility, Variance and Maximum Possible Loss. The input factors are Spot Price, Return Return on the previous day, Confidence Percent, Variance(of the previous day), portfolio worth. It can be used with any currency pair whatsoever.

It also plots a graph between all the factors we mentioned.

How we built it

We built it using Android Studio and using high level discrete math and manipulation and improvisation of the GARCH (generalized autoregressive conditional heteroskedasticity ) model

Challenges we ran into

Implementation GARCH model for different currencies and simplifying extremely complex equations. Handling more than 60 variables

Accomplishments that we're proud of

Simplifying the and enhancing a 70 year old model and coding it for the first time in Java as an Android App.

What we learned

Android App Development, Simplifying highly complex equations, Financial Analysism Fintech(Financial Technology), Loss prevention

What's next for Decur- The Intelligent Investor

The Decur platform will be expanded to a web version as well as live data analysis and ability to trade on its own.
