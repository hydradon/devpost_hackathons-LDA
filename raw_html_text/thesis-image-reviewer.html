Inspiration

My thesis is due tomorrow and I need a break.

What it does

It downloads every single thesis submitted in 2017. It get a percentage percentage of images downloaded from the internet for each thesis.

How I built it

I used the Vision API from Google Cloud together with some web scraping.

Challenges I ran into

Finding images inside the PDF's, had to check the binary file until the image format would be found.

Accomplishments that I'm proud of

I wanted to play with one of the Google Cloud APIs

What I learned

I learned how to scrap websites more efficiently.

What's next for Thesis Image Reviewer

Hopefully I'll graduate and not have to think about theses for a good long time.
