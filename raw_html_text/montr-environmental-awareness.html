Inspiration

Because we are all the change!

What it does

Montr is a web application that monitors and communicates daily changes in weather. The ultimate mission of Montr is to raise awareness of Global Warming’s effect on everyone on the planet - not just those who have ready access to tracking resources and technologies.

This system is notably very sustainable and enables long term use. Montr will run as a web application on hyper-portable computers that are inexpensive, solar powered, and independent of a wireless connection. This allows for a more ready sharing of information throughout communities, regardless of their ability to invest heavily in technology.

Challenges I ran into

I did not have time to build a fully functioning web application from scratch. So, instead, I focused my energy on the Recurrent Neural Network and build the web page with WIX.

What I learned

I definitely took away - how to manage my time efficiently! I knew that I could design a more depictive prototype if I spent less time on run-time errors in a fully function JS web app!
