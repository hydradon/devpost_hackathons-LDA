Problem

Creating an application that will provide families with the best choices for selecting academics and extracurricular activities that best fit their families need. Mobile friendly website will also assist new Montgomery residents, and residents that maybe moving or has a child that is transitioning to new grade levels.

Solutions

Mobile friendly website that helps users navigate the MPS system.

Features

  • view schools
  • view activities
  • view location
  • view cost
  • view times
  • view age appropriate activities
