Inspiration

It was a long night of playing poker and not one of us could be bothered to count chips any longer.

What it does

Automatically counts stack sizes and pot size.

How we built it

We used a unique mapping of resistance measurements to stack sizes and used an Arduino to process and display the information. The chips prototypes were laser cut. The mapping from resistance to chip value was done on python.

Challenges we ran into

The laser cutting did not size our chips correctly and it was very delayed. We did not have the adequate materials to make the chips have proper conductive surfaces. We did not have enough time to properly implement the circuitry. The unique mapping was a very hard problem to solve.

Accomplishments that we're proud of

It actually worked!!!!!

What we learned

Start work on the hardware early and request supplies before the hackathon. Use software to plan out circuitry before implementing it.

What's next for Self-Aware Poker

Have proper chips and develop a more integrated system to enhance the poker experience.
