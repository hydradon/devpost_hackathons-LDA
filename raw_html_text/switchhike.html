Inspiration

A lot cars you see on the street only have one person inside. All the other spots are technically wasted.

What it does

SwitchHike automatically offers remaining spots in your car, but only when it’s convenient for you.

How we built it

python backend with flask, Ionic mobile app (iOS and Android)

Challenges we ran into

Data connectivity

Accomplishments that we are proud of

international, newly-formed team able to come up with completely new idea and realized it in time. Having fun!!!

What we learned

Think about your objectives before starting a project!

What's next for SwitchHike

Make it a personalised system using Machine Learning
