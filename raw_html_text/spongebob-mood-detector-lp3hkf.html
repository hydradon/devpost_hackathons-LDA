Inspiration

We wanted to create something fun for our first hackathon. This is a simple interactive application and we are happy that it works!

What it does

It detects the mood of a user based on the answers to the questions.

How we built it

We used python to code the actual logic behind the program. We used raspberry-pi for the display and arduino to move the servo motors.

Challenges we ran into

Python was slightly new for us. We had to use several hours to learn to code efficiently.

Accomplishments that we're proud of

We are very happy that it worked.

What we learned

We learned a lot about the basics of hackathons. We are sure we will be able to come up with better projects in the upcoming hackathons.

What's next for Spongebob-Mood-Detector

We can add a feature that would control the LEDs and make them blink based on the moods.
