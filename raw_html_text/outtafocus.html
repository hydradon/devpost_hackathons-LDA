Inspiration

We're all guilty of wasting too much time mindlessly browsing Facebook instead of working on homework or other assignments.

What it does

This Chrome Extension aims to solve the issue of wasting time on distracting websites by allowing for a user-configurable blacklist to be set-up to slowly fade in a blur over the blacklisted website in order to keep the end-user on track.

How we built it

HTML, CSS, Javascript, Adobe Illustrator, and Google Chrome.

Challenges we ran into

Integrating JavaScript functions into the Chrome Extension's HTML files.

Accomplishments that we're proud of

Most of us had very little prior knowledge of HTML and CSS. This was also all of our first time building Google Chrome Extensions and we gained valuable experience throughout the process.

What we learned

HTML, CSS, JavaScript, How to build a Chrome Extension, Task Assignment and Time Management.

What's next for OuttaFocus

Further learning in the fields of HTML, CSS, and JavaScript in order to move forward in completing the feature-set necessary to complete our project.
