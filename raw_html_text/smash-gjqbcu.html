Inspiration

Our inspiration came from the original Super Smash games, and the countless hours we spent playing them. We thought making this game would take us back to the 'good old days'.

What it does

Our game is a two player, fighting game on a platform based map. The users control two characters, and move/fight with them. The last man standing wins.

How we built it

We started working on it after the MasseyHacks IV workshops. We used java with Sublime Floobits so that we could work on the code simultaneously. After we got out sublime working, we worked throughout the night on mechanics, sprites, collisions and much more. With a little help from out mentors of course.

Challenges we ran into

In the beginning of the project, we had a lot of trouble getting the basic moving functions started. Specifically, making the jump and fall look accurate according to gravity. After that we worked on making the user be able to jump through the appropriate platforms. Finally are last and biggest issue, was the classic disappearing MegaMan. At times our second character would disappear from the map, however several frustrating trials later, and one integer change, we solved it.

Accomplishments that we're proud of

We're most proud of our sprite animations and movement graphics. It took a lot of time to find, edit and make the game look fluid, but the end result was worth it.

What we learned

During this project we learned a lot about how to code in terms of object oriented programming in java. It taught us how to solve problems and code graphics in a way java would use efficiently. This was a great way to get our skills warmed up for our school projects and to have fun.

What's next for Smash

Hopefully with a little more time and planning, we can add all the extra features we wanted and polish it up. Our project was a great start for this type of game, but still can be made a lot better!
