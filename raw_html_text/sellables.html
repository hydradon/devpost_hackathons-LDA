Inspiration

People LOVE stickers. Stickers are easy to share and show off. A few people LOVE non-fungible tokens (NFTs). NFTs are easy to sell but hard to show off.

This project combines the two worlds together.

What it does

This is an iMessage app that queries the OpenSea API ( https://docs.opensea.io/reference ) for the NFTs and collectibles you own. For now, you configure your address in settings, but eventually some wallet integration (like with Squarelink, see https://docs.squarelink.com/#introduction ) will be nice!
