Inspiration

Our Inspiration was to learn more java, and we thought that we would learn through a diffrent approach than usual courses by making games with java.

What it does

It is a retro arcade containing 3 classic arcade games programmed in java plus bonus feature

How we built it

We used a combination of images and java. To do this we used the Eclipse IDE and Paint.net.

Challenges we ran into

We were originally thinking of making it multiplayer, but when it came to the part where we had to start networking we found it would very difficult to make it multiplayer

Accomplishments that we're proud of

We are proud of being able to combine 3 games into one appp

What we learned

We learned java skills like swing and thread multitasking.

What's next for Retro Arcade

We want to make the games multiplayer and add more retro games
