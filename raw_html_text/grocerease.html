Inspiration

All the perfectly good food that we throw out every week :(

What it does

GrocerEase allows a user to input grocery items, and the app will notify you when they are close to expiring. The item list is linked to a Firebase database that updates in real time.

How we built it

Created an Android app using Android Studio and integrated the Firebase Java API

Challenges I ran into

Android and database architectures, small team, choosing an idea

Accomplishments that I'm proud of

Creating our first Android app with successful API integrations, making something practical for everyday use

What I learned

Android development, backend/database management, Agile development

What's next for GrocerEase

Look pretty <3 Tapping notification opens the app Dynamic notifications (X will expire in X days) Prefilled text when editing an item Searchable items/filters Sorts items by expiry date, type Pictures for food items Shopping list that integrates purchased food into item list Gets recipes based on food in item list
