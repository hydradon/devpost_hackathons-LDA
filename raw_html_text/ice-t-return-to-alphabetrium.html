Inspiration

I got inspired after watching the second season of Rick and Morty. I knew that there was more to Ice-T's story to be told (this is the alternate-universe version of his story!). I decided to try to make a game about Ice-T in under 4 kilobytes of code to take on Datto's 4k code challenge.

What it is

This is a retro-style shooter game! You play as Ice-T, a frozen letter from the planet Alphabetrium. After a long exile on Planet Earth, you return only to find that your entire planet has been taken over by rogue Magma Letters, from the same tribe as your very father! Driven to get to the bottom of this, you must fly through Alphabetrian airspace, clearing out the evil letters as you make your way back home.

How I built it

I built this using pure JavaScript, CSS3, and HTML5.

Challenges I ran into

Of course the space limit was a challenge, but after some manual code shortening I ended up well within limits.

What I learned

I learned how to implement a game using javascript and HTML canvas element.

What's next for Ice-T: Return to Alphabetrium

More levels, a boss battle, an ending? These are all possible!
