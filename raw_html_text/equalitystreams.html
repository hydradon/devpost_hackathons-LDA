Inspiration

My inspiration for this project came from interacting with smaller streamer. There were a lot of streamer who had quality content but not many views. This could be very intimidating for smaller streamers and can turn them away from streaming on Twitch, or streaming all together. So I drafted up a new discover page for streams in which you can see random streams from streamers of all sizes.

What it does

This project is a prototype of the discover page for twitch. Here we show off 3 random streams, of a certain game, and 3 streamers of different viewer counts. We have the large streamers, the medium streamers, and the small streamer. They are displayed together in order to allow viewers to chose what exactly they want from a stream, i.e. the interaction famous with smaller streams, or the professional streams famous with most popular streams

How I built it

I built this prototype through python and django by using the python-twitch-api wrapper for the Twitch v5 api

Challenges I ran into

I did not know how to code with Django at the beginning of this project, and now i know enough to display many different. I also couldnt get it to deploy on a server, so for now, it is just a local site parts of a twitch stream

Accomplishments that I'm proud of

I really proud of being able to use the api and python to interact with many different streams from a ton of different sizes.

What I learned

I learned how to use the twitch api and how to create a small website with django

What's next for EqualityStreams

I'm looking forward to filling out this prototype with a game search bar, a css/bootstrap file (to make it a hole lot prettier), and an overall clean and usable looks. It is my hope that this website can be a new homepage where people can view streamers of all different viewer counts.
