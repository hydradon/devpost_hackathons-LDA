Inspiration

Have you ever felt lonely and bored in a place without your friends? Then SoChirp is the solution, an app that helps you meet new people.

What it does

SoChirp uses soundwaves to enable you to communicate with people in your proximity and allows you to interact with them through chat and social games.

How we built it

We built it using Android Studio and Chirp API, which allowed us to modulate data into soundwaves. Then we had to unpack the data on the receiver's side using the same protocol.

Challenges we ran into

Coding the data in a specific format and transferring the data between phones in multiple stages as the API only allows you to send 10 characters per message.

Accomplishments that we're proud of

Finishing a BETA version of the app in 24 hours, coming with a nice and helpful idea

What we learned

We developed our android and Java skills and learned to use a new fun API.

What's next for SoChirp

Expand it with more games and features (sending pictures,videos,etc) that allows interactions between people, but this require a more advanced version of the API.
