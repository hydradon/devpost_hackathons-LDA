Inspiration

We are inspired to make something that every drivers can use to ease their long journey.

What it does

Our Smart Glass is a portable device that can display the current temperature, humidity, speed, time and navigate users to their destination. With simple GUI, it will be user friendly which only necessary information being displayed.

How we built it

We are going to use program a Raspberry Pi and connect it to a few external sensors to detect temperature, humidity and track the current time.
