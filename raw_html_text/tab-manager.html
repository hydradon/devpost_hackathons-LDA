Inspiration

The idea came from one of our members always keeping his tabs open for future reference, but never clearing it, eventually leading to a congestion of tabs on Chrome which uses alot of computer resources.

What it does

This tab manager extension loads the active window's tabs as icons, and provides different sections. The icons can be dragged to different sections. Clicking on an icon brings the user to that tab.

How we built it

The extension was built using React framework and Javascript.

Challenges we ran into

Along the way, we ran into many challenges:
