What it does

We created a tool where the user can pick a certain suspect and see all of the events related to them. The user can select a certain event, this will map the suspects location on to the floor plan and will also map all other suspects location at the event time that the user selected. We also implemented a raw data table which is filterable by all fields and is sortable and filterable.

How we built it

The tool was written in JavaScript and we used the Vue framework.

Challenges we ran into

The main challenges were just learning the ins and outs of the Vue framework, and finding a good balance between possible visualization ideas and the time remaining in the hackathon.

Things we wished we could finish

Update the styling, added more meta data to the visualization map (time data, and event data for the other suspects).
