pizza

Pizza is a command line browser for Piazza. Quickly and easily browse Piazza without leaving your terminal.

While it's still very much in-beta, you can try it on most Unix-compatible machines by running:

pip install git+git://github.com/prafulshankar/pizzasearch.git

It requires Python 2.
