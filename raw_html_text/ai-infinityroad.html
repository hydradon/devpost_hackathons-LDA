Inspiration

We like road trips! But it is so much work to plan a great road trip. We decided to make an AI which will do the work for us.

What it does

I will find a tailored road trip based on your preferences.

How we built it

We used a genetic algorithm to select the best locations that you should visit. The fitness function is able to take custom restrictions into regard.

Challenges we ran into

Traveling Salesperson Problem, Google Maps API (limited number of requests, even after we gave them a credit card!)

Accomplishments that we're proud of

Teamwork

What's next for [AI] infinityRoad

To deal with Google maps' restrictions, add more custom restrictions for the planner, ... profit
