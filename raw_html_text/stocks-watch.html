Inspiration

We really wanted to do something with finance and something interactive. So we figured that a web app that notifies users when their stocks drop/rise would be perfect.

What it does

Stocks Watch helps monitor the activity of your stocks and sends text notifications when a stock rises or drops below a certain point. Users will add their stocks and set the price they want to be notified with.

How I built it

Stocks Watch is a web app built using Ruby on Rails on the backend and bootstrap on the frontend. The Twilio API is used to send text notifications to the user and Yahoo Finance (stock quote) to monitor the stock prices in real time.

Challenges I ran into

We had many challenges setting up the environment and database in the beginning since most of our members did not have experience working with Ruby on Rails. The next big problem we had was getting the login authorization to work properly.

Accomplishments that I'm proud of

It was amazing getting the login authorization to work, seeing the API grabbing the correct stocks, and just watching the product grow.

What I learned

Gained a lot of experience using Ruby on Rails, Yahoo Finance API, and Twilio API.

What's next for Stocks Watch

In the future, Stocks Watch should be able to:

  • make predictions on the stocks using Alchemy API
  • customizable text messages
  • autocomplete for stock symbols
  • mobile version
