data-cleaner_app

For a data scientist, cleaning and organizing data takes up 60% of the time, according to Forbes.
To try and solve this problem, we created a web app which takes the data in csv form and detects the invalid data in each column and gives the user the control to remove it. It also gives user the option to define the data based on limits and data types.
