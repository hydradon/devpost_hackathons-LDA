Inspiration

We were inspired by friends, family, and ourselves. We felt that many people could benefit from an assistive mental health chatbot.

What it does

In My Feelings is a chatbot focused on helping you with your mental health. Through passive questions and data from API interfaces, the chatbot gets to know you, and learn what might make you feel better.

How we built it

In My Feelings was built using DialogFlow system for chatbot development. It uses natural language processing and machine learning to identify the user's statements and react appropriately. The agent uses API data on the number of steps taken, the number of calories burned from activity, weather, and time spent on the phone. The AI uses this data to provide actionable insights to help the user improve their life.
