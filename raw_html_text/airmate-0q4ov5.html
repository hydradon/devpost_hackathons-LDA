Inspiration

The need to find something to do with our time while waiting in an airport for a plane

What it does

AirMate pairs you with other people who are waiting for the same flight in the airport. This guarantees that you can make new friends and make use of all your time in a fun and interactive way. This can also provide opportunities of adventures with strangers in other countries.

How we built it

We built it with a framework for PHP called Laravel, and we integrated the SkyScanner API

Challenges we ran into

Some trouble implementing the SkyScanner API and the chats for the airports

Accomplishments that we're proud of

We managed to implement all the main features that we planned for this hackathon

What we learned

We learned how to use the SkyScanner API

What's next for AirMate

Integrate the hotel and car rental services from the SkyScanner API
