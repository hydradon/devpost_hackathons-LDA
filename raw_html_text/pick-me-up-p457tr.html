Inspiration

We were inspired by #trashtag and wanted to continue the benefits that we saw from this movement.

What it does

The web app allows users to take a before and after photo of a polluted area that they clean up, earning in game points. If they rank well enough on the leader boards, they could get cash prizes.

How I built it

We made an express app and used node.js and html to build the back and front ends, respectively.

Challenges I ran into

We did not know node.js, html, or how to use express.

Accomplishments that I'm proud of

We proud of the fact that we were able to make a cohesive application in the time frame with multiple restarts and not knowing almost anything about web app development beforehand.

What I learned

We learned how to code node.js, html, css and how to correctly use api's from companies such as Google.

What's next for Pick Me Up

We are considering making this a fully-fledged app and launching it on the play and app stores.
