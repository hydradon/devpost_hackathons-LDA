Inspiration

I wanted to make a place where anyone can share their ideas quickly and easily in a novel and interactive format.

What it does

Leadrboard allows you to post ideas and upvote whatever post you want in a fun and interactive manner. You can see what other people are upvoting with update animations that are synced in realtime.

How I built it

I used Firestore Realtime Database to host each post and manage the real time updating, and javascript, html, and css for the front end.

Challenges I ran into

I ran into many dead-ends with limited documentation on the recently added Firestore database and learning how to make every person's copy of the posts all live and synced.

Accomplishments that I'm proud of

I'm really proud of getting this working live and getting it hosted live online on a .tech domain name.

What I learned

I learned a lot about javascript and the process of creating and hosting a website from scratch.

What's next for Leadrboard

Next, I would like to add features to support more people such as spam protection and features that makes it easier to see newly added posts.
