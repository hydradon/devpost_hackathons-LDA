• Worker Safety

Inspiration

Worker Safety in the warehouses, factories.

What it does

Locate/relocate someone or something Improve workplace safety by deploying alert buttons and detecting falls More efficient use and allocation of key assets, + reducing fraud and lost items Analyze paths & locations, test new layouts, improve business (e.g. retail)

How I built it

Using .Net Core and MS Azure IoT Suite

Challenges I ran into

Machine Learning of Azure IoT

Accomplishments that I'm proud of

Able to send device data to Azure IoT Suite and display alert on UI when worker falls down.

What I learned

Azure IoT Suite with device twin communication.

What's next for WorkerSafety

From cloud send alerts to device.
