Inspiration

PrarieLearn as a testing system is very efficient. However, as with all good things, there is always a deficiency. This system does not have a GradeBook to view your grades after you're done with your assessments. This proves quite frustrating to the creators. We know a lot of peers who echo this sentiment too. Hence, I started working on this GradeBook that should be an essential part of any academic testing suite.

What it does

The GradeBook is an efficient way to view your grades on all the assessments you have done on PrairieLearn.

How we built it

We worked with the maintainers of this project at HackIllinois 2018, in order to learn the the build details of the project and get it set up.

Challenges we ran into

None of us really knew either javascript or SQL. However, with the help of our amazing mentors and some messing around with the code environment, we were able to build a proof of concept GradeBook.

Accomplishments that we were proud of

Worked on a big codebase with languages and technologies we were never exposed to before.

What we learned

How to work with new languages on the fly, and a bit of knowledge about javascript and sql.

What's next for GradeBook for PrairieLearn

We want to make the interface more intuitive and beautiful. We also want to make the interface fully customizable so instructors can use it as their courses primary gradebook.
