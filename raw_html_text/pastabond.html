We found that finding great housemates that are respectful and compatible is difficult and a lot of times, depends on luck. We wanted to find a way to match people who are compatible with one another and find houses together.

The website lists the individual potential housemate - Spaghetto, and with questionnaires answered, users will be able to see a compatibility percentage and interact with them through the chat room function.

How we built it

  • Front-end - React
  • Back-end - ASP.NET Core
  • Database - MongoDB

Challenges we ran into

  • selecting a tech stack because there are so many languages, tools and frameworks over there
  • setting up MongoDB to make it work with ASP.NET Core
  • managing to overcome knowledge gap in some of the techs in short period of time

Accomplishments that we're proud of

What we learned

  • how to set up an ASP.NET Core project
  • how to use React-Bootstrap
  • how to make MongoDB and ASP.NET Core work together

What's next for PastaBond is to refine the look on the chat room, adding bill spliting and complete list of the questionnaires.
