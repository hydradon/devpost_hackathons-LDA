Inspiration

The need to connect with a person on each and every social and professional platforms in just a few clicks without a need to carry a physical business card drove us into building this project.

What it does

With this app one can create digital social and professional cards which can be shared with anyone having the app in just a scan. Companies can also create their own custom organisation cards.

How we built it

The front end was built using React.js and Ionic Framework and the backend api was built using Node.js and MongoDB.
