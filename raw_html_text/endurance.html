Inspiration

We were inspired by a friend who used to complain about the manual process of tracking his workout statistics, so when given the opportunity to hack with the Myo Band, we instantly saw the potential of a fitness tracking app.

What it does

Endurance give the user full statistics about their fitness schedule in a very simple, easy to use manner. It does this by reading the EMG(electromyographic) sensor data from the Myo Band.

How we built it

We used the existing Myo iOS SDK and created our own specialized algorithm to recognize different exercises.

Challenges we ran into

Some of the challenges we faced were : configuring, calibrating, and capturing the raw sensor data from the Myo Band.

Accomplishments that we're proud of

We made it.

What we learned

We learned a lot about Myo.

What's next for Endurance

We have some interesting ideas in how to further the app by adding new exercises.
