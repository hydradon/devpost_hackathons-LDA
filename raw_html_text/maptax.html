Inspiration

Taking the advantage of iOT(internet of things) devices to calculate the accurate time spent on different businesses and add significant value to professional life.

What it does

It is a web and iOS app that can track the location and can calculates the milage a user did for multiple jobs there by providing user a clear estimate of deductible tax for separately.

How I built it

We built an iOS app to track the location of the user which posts the latitudes and longitudes of the user's location into aws RDS(mySQL). We created an Amazon EC2 instance for Ubuntu and created a web server in it. Our web client uses this data and gives report to the user which has the estimates of the miles ran and calculates the tax deductibles separately for multiple jobs.
