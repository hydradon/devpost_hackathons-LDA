Inspiration

We wanted to make a game that could easily be controlled by anyone with just a camera

What it does

It recognizes which way you move your fist then it moves your character in the same direction

How we built it

We used pygame and a gesture recognizer module

Challenges we ran into

Our map kept building incorrectly and caused many problems

Accomplishments that we're proud of

We are proud that we could make a game and map that can be mostly run by moving your fist

What we learned

We learned how to use pygame with other libraries and make it a joined game

What's next for Maze Runner 2.0

We hope to have many levels in the future plus different characters.
