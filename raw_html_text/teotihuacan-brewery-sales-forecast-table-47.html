Inspiration

Our inspiration were all the different companies that currently are in constant progress and investigation to obtain better results in artificial intelligence. Also we are interested in the way technology is evolving to predict different scenarios and get the best possible result in analysing data.

What it does

Our neural network iterates multiple times to learn by the historic data and with all the information clean is able to predict future sales quantities and displayed in a web UI showing the results in a friendly manner through graphics.

How we built it

We build the neural network using python and for the queries using MySql. For the UI we use JavaSfript and for the management of the graphics we decided to use Ant Design. Lastly AWS is used to the management of the server of the database.

Challenges we ran into

The data received was too much large, so the amount of data is difficult to process by our neural network, resulting in lack of precision.

Accomplishments that we're proud of

Due to de short period of time doing such a big project we are proud to be able to finished the project and being able to present it.

What we learned

We learn a lot about the mapping of data and the way neural networks learn and works. Also we discover new frameworks and ways to work in team.
