The challenge posed by CACI

You have been hired to test the security of two wireless network systems. The client has placed three codes within the system behind various technologies to test the security of the system which must be e-mailed to the client as soon as you discover them (to track the time to break) at caci.cyberchallenge@gmail.com . Each team has a designated code and the first two characters represent your team's designated code. If you are team 1 your code will start with 'T1', team's 2 code will start with 'T2', etc. An example of the code is shown below:

T1-ALDKF T2-KENFS T3-JUPLQ

The client will provide you with the SSID of Network A (CACIRocks!) which contains details about the challenge.
