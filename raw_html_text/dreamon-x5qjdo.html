Inspiration

We enjoy sleeping and its benefits. We want other people to sleep more.

What it does

It eases users into calm environment and asks the user to sleep. If the user utilizes their laptop/desktop device, the website sends an alert.

How we built it

We utilized HTML for structure, CSS for design, and implemented an application with JavaScript.

Challenges we ran into

We had difficulty implementing different languages into the same program; however, we overcame those issues.

Accomplishments that we're proud of

We are proud of developing an eventlistener and an aesthetic GUI.

What we learned

We learned front end web development and new languages.

What's next for DreamOn

We plan to create a login system and a database that tracks user times. Users can access a graph displaying statistics for fellow users too.
