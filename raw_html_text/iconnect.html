There was a urgent need for a change in the recruiting process. No one wants to create a profile for each job and apply. This was a great opportunity to contribute to this area.

The iOS app facilitates easy application process. User can look into the job description and apply on the fly. User might wish to save the job notification and look at it at a later point of time. We also provided a watch app that aims to target the user base.

Primarily, we used the iPhone core SDK, WatchKit and Parse to build it.

Integrating WatchOS2 with Parse, as the SDK that is available from Parse lacks documentation.

We were a team of two people, and still managed to pull off with our goals.

Use the real world data and provide job opportunities for the users at their finger tips. We see a lot of potential in this app.
