Inspiration

There seem to be a lot of misconception about savings and investments out there, and I wanted to make something using Capital One's API to help demystify it.

What it does

Gathers user data such as income, expenditures and savings in order to tailor specific advice. Allows user to see expected returns over a certain time period given a certain rate in the market.

How I built it

Used React, JavaScript, CSS, HTML, Nessie

Challenges I ran into

Getting the Nessie API to work was more difficult than I expected and getting cookies to work properly so I could store data took a long time.

Accomplishments that I'm proud of

It works pretty well!

What I learned

Learned web development for the first time

What's next for Jessie

Better visualization in the form of graphs and a more dynamic website
