Inspiration

Build something with Linkit ONE development board. Also try other things like API and services provided

How it works

When it turns on it establishes a GSM connection and uses a LCD display to print out reports. then when the connection is established we can control the color of the LCD or let it play music. There is also a physical pin that when disconnected triggers a music to play. The only way to stop it is by paying trough Braintree.
