Inspiration

With a love for structured poems and code, we set out to combine our passions and create a beautiful self-generating haiku program. We could not wait to see what unusual yet amazing creations lay in store for us.

What it does

The Haiku generator randomly selects words and creates an exciting new haiku.

How we built it

We used python and a package called arcade to generate the images.

Challenges we ran into

Neither of us had coded in a group before, so it was a little weird trying to work on different things and then put them together, but it was a great experience. We both really enjoyed it.

Accomplishments that we're proud of

We were very unsure as to whether the poems would make sense as they printed out, but it actually works pretty well most of the time.

What we learned

We got to work with a graphics generator and learned a lot about how to create different images in a package called arcade. We also learned that there is indeed crossover between linguistics and computer science.

What's next for Haiku Generator

Maybe one day it will have more themes and a more exciting print screen, but for now we must let it show itself off.
