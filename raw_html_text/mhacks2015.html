Inspiration

We wanted to learn machine learning for the longest time, and this project gave us something that would be somewhat practical.

What it does

Based on your geographical location it notifies you if you are statistically at risk of being obese

How I built it

We used azure machine learning to find out the percentage risk you are at based on your location

Challenges I ran into

The azure machine learning API was refusing to do anything CORS, which gave our team issues for some amount of time.
