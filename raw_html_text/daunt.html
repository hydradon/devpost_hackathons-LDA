Inspiration

We thought it would be interesting, fun, and a great learning experience to exercise our teamwork, communication, and coding skills.

What it does

Has the user go through a dungeon crawler-esque video game.

How we built it

Using Python and the IDE PyCharm.

Challenges we ran into

Developing the idea itself after attempting multiple other projects.

Accomplishments that we're proud of

The basic functionality of the game.

What we learned

A program does not only need people with coding skills but as well as a sense of creativity.

What's next for Daunt

Multiple classes to choose from for the player, possibly more enemy types, different stories, leveling up feature, and more.
