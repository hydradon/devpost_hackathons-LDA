Inspiration - Our inspiration was to create a fun website for people to interact with their friends online!

What it does - Allows users to rank specific items, from songs to foods and compare with their friends/friendly online users. Has an account service.

How we built it - Using React for frontend and Firebase as a database and substitute backend.

Challenges we ran into - Formatting with CSS, pulling data from Firebase, and authentification with Firebase.

Accomplishments that we're proud of - Creating our first React project!

What we learned - React and Firebase.

What's next for Rankings for Friends - Improving the UI, and improving on the topics and Firebase data pulling aspects.
