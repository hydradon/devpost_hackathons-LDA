Hacker's Calendar

An app that gives users a list of upcoming Hackathons taking place around the world. It integrates with the users calendar and allows them to add Hackathons straight to thier calendars. The data is fetched from Hackalist's API which can be found here https://github.com/Hackalist/Hackalist.github.io and my app can be found here https://play.google.com/store/apps/details?id=com.shiv.hackerscalendar
