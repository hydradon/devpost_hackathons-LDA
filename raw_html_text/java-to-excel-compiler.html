There are a few parts:

  1. A bare-bones assembly language. There are 19 instructions in total.
  2. A spreadsheet which runs programs in this assembly language. It is pure, spreadsheet functions, single-static-assignment - no macros, iterated computation or proprietary shenanigans.
  3. A compiler from a very limited subset of Java (int variables, some arithmetic/bitwise/comparison operations, if/else, while loops, function calls) targeting the assembly language.
