Inspiration

Algerian hackers usually get to know the next hackathons using Facebook, and that's possible if they know the club that organizes the hackathon, or maybe some of their friends whom are interested in the event.

So HackDz is a mobile application willing to help these hackers to find all the nearest next hackathons or competitions.

What it does

it list all the available hackathons, you can filter by date, city, prices and more.

How I built it

For the interface we used Flutter, a great framework to fastly build a mobile application. For the backend we'll be using Firebase.
