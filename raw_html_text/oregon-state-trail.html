Inspiration

The Hackathon theme was "Rewind" and that made us think about the original Oregon Trail PC game from our childhood.

What it does

It's a text based game with random events that happen in between class weeks. There are times when the user has to pick whether to study in the library or computer lab and so forth.

How we built it

C++, repl.it

Challenges we ran into

Realizing that there were a lot of things to consider to make an interesting and well balanced game. Also, real life keeping us busy and not having a lot of time.

Accomplishments that we're proud of

Getting a working game going despite lack of time. :)

What we learned

Oregon trail was a complicated game for its time and it was fun to think about the challenges the original game makers must have had.

What's next for Oregon State Trail

Would really like to add some graphics and a "game" part to it, sort of like the hunting feature in the original.
