Inspiration

We wanted to write something that could be used by anyone and everyone. We also disliked the cyber bullying and it's effect on the youth.

What it does

It is a web service that functions as an API, receiving a message as its only parameter and returning an integer back to the sender. That number is the likelihood that the message was offensive. We also built a demo to showcase what the API can do
