Inspiration

The desire to classify a random group of photos into specific groups that fit certain criteria.

What it does

Given various images and the number of groups the user wants, it classifies them into the number of desired groups.

How we built it

Using C# and microsoft visual studio while working with Microsoft Cognitive Services.

Challenges we ran into

Converting the data that we got from microsoft to data usable by the algorithm that classifies the photos given their categories.

Accomplishments that we're proud of

Using a machine learning algorithm named K-means.

What we learned

How to use Microsoft Cognitive Services and specific details about developing in C#, as well as how to implement the K-means algorithm.

What's next for HackMty_02

Improving the User Interface as well as accepting user designed categories to fit to their liking.
