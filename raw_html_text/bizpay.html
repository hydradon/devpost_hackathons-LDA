Inspiration

The inspiration that we had to create this website was the need to provide businesses a way to receive payments form their clients and pay their employees through "ATH-Móvil" in an easy, low-cost and seamless way.

What it does

Bizpay is a website that allows businesses to receive payments from its clients and pay all its employees through a website console. It is specifically built to help make these processes easier and more organized using "ATH-Móvil’s" platform and API’s. It is basically a concept of what we think “ATH-Móvil business” should look like in its web client and the features it should include.

How we built it

We built it by first, designing the website based on a dashboard interface. Then, we integrated programing languages such as HTML, CSS and JavaScript to build the apps interface. After that, we made a database of information and integrated “ATH Móvil’s” API to make it all work.

Challenges we ran into

The biggest challenges were integrating the API, building a solid business plan and designing the webpage.

Accomplishments that we're proud of

We are proud of every single aspect of the webpage. Everything works amazingly.

What we learned

We learned a lot about integrating into our website frameworks such as polymer, design guidelines like material design, API’s, how to get the API to work with our project and polished our other abilities further such as programing in CSS, HTML and JavaScript.

What's next for Bizpay

The next challenge for Bizpay is integrate it into "Ath Móvil" and expand the different options the platform will offer in conjunction with “ATH Móvil”.
