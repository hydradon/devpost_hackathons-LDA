What it does

Marks a location on the map with an arrow that is viewable through a Augmented camera app

How we built it

Swift, ARKit

Challenges we ran into

convergence from actual map location to x,y,z point on Augmented Reality scene.

Accomplishments that we're proud of

This is the first time ever developing in Augmented Reality. I feel like we accomplished more than we thought were able to finish what had in mind for the project

What we learned

Read more about what you're planning to do in a hackathon. Though having never touched an AR library prior to this hackathon, this project was very enjoyable yet challanging

What's next for AR location marker

Make it into a chat app that send AR location Improve location detection Use it with applications like Lyft and Uber
