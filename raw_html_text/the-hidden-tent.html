Inspiration

The passion for innovation and a certain type of resourcefulness

What it does

It enables android phones to control any computer from anywhere

How we built it

We created an android app that sends instructions through our node.js server with the aid of a websocket to enable the desktop app to fire keystrokes on the target computer

Challenges we ran into

Scaling the mouse movements from the phone screen to the computer screen, hosting our server on aws, etc.

Accomplishments that we're proud of

Making an app that can remotely control any computer.

What we learned

We learnt how to combine different languages to create an application. We also learnt how to deploy a server on aws and route our own domain to the online server

What's next for the-hidden-tent

Continuous improvement to create a sale ready solution to facilitate remote access
