Inspiration

The most common cancer in the United States is skin cancer, it is also one of the easiest ones to prevent. By providing people a free way to test themselves and determine if they should visit a physician or not we will help prevent the most common kind of cancer in our country.

What it does

The application uses Google Cloud's Machine Learning in order to detect whether a particular skin lesion uploaded by the user is likely to be cancerous.

How we built it

We built it using GCP Machine Learning API and HTML integrated into Python Scripts.

Challenges we ran into

Having no experience with Machine Learning or Google Cloud, we all had to climb a steep learning curve.

Accomplishments that we're proud of

Having successfully trained the ML algorithm.

What we learned

Sleeping is nice.

What's next for Skin Screen

The next step for Skin Screen will be an upgraded functional web application and more importantly a user friendly mobile application.
