Inspiration

Create an iPad application that helps bring families closer together while cultivating a child's creativity.

What it does

The application allows a user to create their own story book with their own hand drawn images or uploaded images. The application also allows a user to add their own text at the bottom of each page and record a voice over to read the story back to them.

How I built it

This application was built using Obj-C in the Xcode editor for the client side application. The server side application is an IBM Bluemix hosted application that allows data to be stored in the IBM Mobile Data Cloud.

Challenges I ran into

The challenges that my team ran into were integrating IBM Bluemix and utilizing the Google Authenticated API.

Accomplishments that I'm proud of

Some accomplishments that we as a team are proud of are working together to accomplish a common goal and integrating IBM Bluemix into our application.

What I learned

My team learned how important it is to stay focused and work as a team.

What's next for My Story Book

The next steps for My Story Book are to finish the Google authentication and to add more features to the application.
