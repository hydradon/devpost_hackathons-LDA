Inspiration: College students love to use flash card like quiz games to study so I made one completely from scratch

What it does: It picks a random financial literacy question from the bank and you have to guess the definition/answer

How I built it: Android Studio, Java, XML

Challenges I ran into: Making the random question generator work

Accomplishments that I'm proud of: Getting Android studio to finally work

What I learned: Using actionListeners in Java using Android Studio

What's next for FinLit Quiz: You can share your percentage to social media!
