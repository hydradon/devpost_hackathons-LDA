Inspiration

We found health code violations on Louisville Open Data and thought it would be cool to track within the city.

What it does

pulls information regarding health code violations in Louisville and displays it on Google Maps.

How we built it

We used laravel on top of PHP for our back-end and Angular for our front-end.

Challenges we ran into

1 slow computer, learning curve

Accomplishments that we're proud of

We have a finished product.

What we learned

became more familiar with typescript and Angular

What's next for Inspectors Had-It

Search functionality, colored icons based on score, spreading out nationally.
