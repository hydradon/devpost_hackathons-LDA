Inspiration

Infini Modular/Mobile Suit is a project to create an open modular whole body wearable, multipurpose IOT platform for creating tactile sensations, and measure skin conductivity parameters, and muscular signals by using specialized sensors and mesh wireless connectivity.

What it does

It will be able to work as:

Electrophysiological scientific research tool for skin Gaming and entertainment Relaxation therapy Brainwave entertainment Aid for disabled persons Fine motricity therapy aid Major skin wound sensibilization therapy Sex toy

the hardware mockup presented here, uses a servo as tactile generator and a wirelessly connected mobile phone accelerometer as data source. Also is demonstrated local control of the device. The software mockup, introduces to the first commercial mobile application proposal compatible with the IOT hardware platform, currently on development.
