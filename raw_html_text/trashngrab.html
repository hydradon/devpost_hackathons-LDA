TrashNGrab by the Gilbert Strang Squad This application offers the possibility for producers to offer leftovers to other consumers by giving their location. Producers and consumers can sign up, and then sign in. The system send an email when signing up. Producers can then offer items of a category. Consumers can follow selected categories and are advised by email when a new item of that category is offered by a producer. All items should be displayed on a map API using the location given by the producer.

The front-end is Angular, using typescript, html and css. The server/back-end is in Express using mainly typescript.

Most parts of the application are completed, like the server, yet some links still need to be added to the UI.
