With chat knowledge we aimed to introduce an easy way to analyze chat information and represent it in a beautiful materialistic design. It works rather simply – the user first authenticates using their desired social media, which we process according to the GDPR requirements. After that the user can see a few things about their chats in the boxes in the main page. In one box the most common words in the chat can be seen represented by a word cloud, in another box few there is a chart representation of the overall moods of the chats in the different days and in a final box there is a detailed overview of the chats statistics such as the number of messages sent.
