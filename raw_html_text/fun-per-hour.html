Inspiration

What it does

Gives an event admin a event code. Attendees of the event can text this event code to a Twilio number to subscribe to a text survey. This survey will ask them their "fun level" from 1-10 every hour.

How we built it

We split our team in two. Half of us made the front end view of the site. Half of us made the server code. Add in long all nighters and very intense water pong games.

Challenges we ran into

I spent an hour on an error caused by a letter being lowercase instead of uppercase.

What we learned

Only one of us had experience with PHP, JavaScript, and AJAX. Now all of us know how to do the majority of this project on our own.

What's next for Fun Per Hour

We we'll finish it and test it at very long events that last 7-10 hours so see how much fun people have per hour.
