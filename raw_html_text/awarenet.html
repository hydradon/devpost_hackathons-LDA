Inspiration

In light of recent hurricanes and inclement weather that hit many locations in the south of the USA, our team decided to create an app that would notify a user of any severe weather changes in the location that they chose to monitor not just in their area.

What it does

Notifies a user of any severe weather changes in the location that they chose to monitor not just in their area.

How we built it

AwareNet's UI was built using Android Studio and also sends notifications to the user utilizing Google Firebase. It has a MySQL RDS hosted by AWS. For providing all the updates on weather, we wrote a web crawler that collected data from FEMA website.

Challenges we ran into

The main problem was the fact that we found out that Android-studio developed apps and MySQL dbs cannot connect directly.

Accomplishments that we're proud of

What we learned

We learned A LOT this weekend. The only familiar part was Java as a programming language and MySQL. Everything else was new to us.

What's next for AwareNet
