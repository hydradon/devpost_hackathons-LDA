Inspiration

I Got the inspiration for Jiffy by thinking of discord bots and we though it would be cool to make a bot for slack.

What it does

It gives you the hackathon schedule and suggests what meals you should eat from plated.

How we built it

Jiffy was built with Javascript and node.js also with botkit to interface with the slack API.

Challenges we ran into

We ran into a big challenge where for some reason jiffy was not replying to messages it was programmed for.

Accomplishments that we're proud of

We learned more about data structures, functions, third party API's and how to pair program.

What we learned

We learned as beginners to program in JavaScript and Node.js.

What's next for Jiffy

Jiffy is hopefully going to be to able to reply to any questions you will need and even schedule appointments for you.
