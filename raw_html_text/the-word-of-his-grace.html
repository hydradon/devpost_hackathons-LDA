Inspiration

The motivation and inspiration for this skill is to provide voice assistance to the handicapped and visually impaired.

What it does

This skill can read Bible verses given the book, chapter and verse numbers. It finds applications in church and home. settings.

How I built it

I used Alexa Skill API to build the interaction model and bible API provided free by https://getbible.net/api .

Challenges I ran into

Getting Alexa to recognise correctly, bugs in the the interaction model, converting JSON output from the bible API, etc.

Accomplishments that I'm proud of

I can help create the future of voice user experiences, especially for the handicapped and visually impaired.

What I learned

I learnt a lot: VUI best practices, programming, debugging etc.

What's next for The Word of His Grace

Continue development, add variations, more bible translation, etc.
