Inspiration

Need for cheap, expedient tech help with simple problems

How it works

Crowd-sources tech support from users for users within geographical proximity of each other.

Challenges I ran into

Google Authentication was almost impossible. Lots to do between web app and iOS app combined.

Accomplishments that I'm proud of

Accomplished Facebook Authentication, created functional web app where users can post and accept tasks.

What I learned

Even the base features of a Web app require hours of work. Also, Google's API is awful.

What's next for NinkiT

Greater sophistication for the iOS app, support for Android. Possible extension for Apple Watch.
