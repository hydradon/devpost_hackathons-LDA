Inspiration

What it does

The web app lets you plan a group vacation. It is a convenient and economical solution that smartly finds for your group preferable destinations within everyone's budget.

How I built it

We built it using the django python web app framework with foundation templating as frontend. We developed a beta machine learning component which learns from user interactions and suggests better options to future visitors.
