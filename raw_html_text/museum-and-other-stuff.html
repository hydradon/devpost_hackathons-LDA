Inspiration

We were looking to experiment with Unity and VR, and ended up creating a small level; using the HTC vive, in which a lot of various things are presented. We also ended up coding our own grabbing and moving fuctionality for objects in C# using the vive controllers.

What it does

It showcases a few of the things that you can do with the vive and unity, although at a very basic level. Teleportation, handling objects, throwing

Challenges I ran into

we ended up running into problems with the implementation where the triggers on the vive were not interacting properly with the objects and had to debug to make the entire grabbing and handling script run.

Accomplishments that I'm proud of

achieving teleportation and the object handling script, even thought there aren't that many objects that are available to manipulate, just the paintings on the

What's next for Museum and other stuff

probably expanding on what we have and looking to perhaps change up the grabbing implementation so that it works better with the physics and throwing interaction.
