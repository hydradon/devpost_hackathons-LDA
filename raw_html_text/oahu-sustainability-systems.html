oahu-sustainability-systems\

A metrics based data analysis tool developed with the intention to display actionable Key Performance Indicators using creative and dynamic visuals for a more sustainable Hawaii. Oahu Sustainability system Intends to provide an interface for what if analysis for both legislative and communal action based off of data provided by the aloha dashboard.

Our solution is to develop a tool that tells the story of progress in building Sustainable Communities.
