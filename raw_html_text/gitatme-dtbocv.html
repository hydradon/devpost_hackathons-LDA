Inspiration

I was inspired by the reddit & twitter bots of the world. Also, I read git commits from the people around me and my own team. Stuff can get pretty zany.

What it does

It goes through all repos related to a hackathon and then pulls git commit messages from them.

How I built it

I built it using python, github, and twitter. It's current home is on AWS!

Challenges I ran into

It still crashed very few hours.

Accomplishments that I'm proud of

It's currently running on it's own with no intervention from me. They grow up so fast. ;>

What I learned

I how to make a program to be on twitter.

What's next for gitatme

Support for more hackathons, search terms, and machine learned selection versus a random selection method.
