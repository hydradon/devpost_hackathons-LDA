Inspiration

Existen en el mercado actual algunos productos para sensar humedad y controlar el sistema de riego en diferentes sistemas de producción agrícolas pero con el inconveniente de un alto costo y una disponibilidad insuficiente en el mercado nacional. Nuestra propuesta es un dispositivo encargado de la medición y almacenamiento de información relevante para el control de los sistemas de irrigación agrícolas con una gran adaptabilidad a los requerimientos del cliente ya que es un sistema modular fácilmente escalable para la medición de otras cantidades físicas importantes para el cliente. Además se tiene la posibilidad de construir bases de datos de las mediciones obtenidas y almacenadas con el dispositivo, mediante la red con una interfaz propia gestionada en linea.

What it does

Mediante sensores basados en semiconductor se obtienen mediciones periódicas y confiables de temperatura y humedad (entre otras). En base a la información obtenida, el dispositivo puede controlar sistemas de irrigación o extracción para mantener un equilibrio en las condiciones del cultivo y evitar así la proliferación de hongos o bacterias.

How we built it

Para la construcción del dispositivo se empleó una tarjeta de desarrollo "arduino uno" para desarrollo rápido. Ademas los sensores y circuitos adicionales para el uso de la pantalla se montaron en un proto-board para obtener un primer prototipo de la idea en cuestión.

Challenges we ran into

Uno de los primeros retos fue el control y sincronización de los sistemas de medición y almacenamiento usados en el dispositivo. Otro reto importante fue el desarrollo de las interfaces para la gestión de las bases de datos generadas con el hardware prototipo.

Accomplishments that we're proud of

El primer logro es que logramos en un tiempo corto un prototipo funcional y bien depurado. Además se logró terminar la interfaz de gestión basada en html5 y php.

What we learned

Aprendimos a comunicarnos con personas de otras áreas de la ciencia y tecnología y obtener un bien común reflejado en un prototipo con diversas aplicaciones. También aprendimos a trabajar bajo presión en un tiempo bastante reducido cumpliendo los objetivos iniciales del proyecto.

What's next for Termo-higrómetro diferencial

Ya que se tiene el primer prototipo de la idea, el siguiente paso es la implementación de un prototipo de prueba para su uso en el campo. Para ello se pueden utilizar diferentes materiales para fabricar un chasis resistente a la intemperie y condiciones extremas de temperaturay humedad. Además es posible con este primer prototipo el estudio de viabilidad del uso de la comunicación inalámbrica en el campo. Así como estudiar otra de sus posibles aplicaciones tales como el control de proceso de curtido de pieles que es una de las principales actividades económicas de León Guanajuato.
