What it does

Our web app crowd funds student loans with our smart savings account.

How we built it

We used the AISP API provided by Finastra's fusionfabric.cloud. This gives us access to users' account details with their consent. We analysed the data and built our entire infrastructure with C# and the .NET framework. For the user interface and experience we used JavaScript, CSS3, Bootstrap4 and HTML5.

Challenges we ran into

We had to build a web app instead of a mobile app because the ffdc AISP API returns a web url for consent. We could have created a pop-up or iframe within a mobile app, but we think that provides a bad user experience.

We struggled setting up the API using .NET.

Accomplishments that we're proud of

We made it work!

What we learned

To engage as many experts as possible early on as we run into trouble. Agile methodology is best for hackathons.

What's next for nineTeen

We love our product, so we are going to build it for the world. Finastra provides a whole suite of APIs for our entire workflow, so we have a one stop shop for everything. We will spend the coming months validating our idea.
