Inspiration

About 94% of the Quebec population can carry a conversation in French. We want to help Anglophones learn French quickly and efficiently so that they can take part in contributing to Quebec society professionally, academically and culturally.

How we built it

Frontend: HTML, CSS, JavaScript Backend: GO, Google Cloud, REST API

Challenges we ran into

Inputting files on the Front end and translating them to the backend

Accomplishments that we're proud of

Working with REST API and successfully converting text between French and English smoothly, given audio files.

What we learned

How to work with REST API, Google Cloud, GO.

What's next for Franglais

Record audio on the spot rather than uploading already-recorded audio files.
