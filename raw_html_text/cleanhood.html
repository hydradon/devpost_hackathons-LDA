Inspiration

Every now and then we all come across something that annoys us, disgusts us, a nuisance. But often we don't try to fix it, but instead learn to live with it. You walk across your neighborhood everyday on your way to work and it's riddled with obscene graffiti, you walk across a street riddled with garbage from the nearby food carts.

You notice these things, and they annoy you for a few days. But soon, you stop looking.

What it does

Cleanhood is a solution to fix these nuisances in life and to stop ignoring them! We created a crowd sourced aggregation platform that allows a user to simply take a pic of a neighborhood nuisance, geotag it and request volunteers to sign up for a clean up drive.

How we built it

We built a native Android application backed by Node.js and MongoDB. We used Firebase and Twilio to support real-time notifications.

Challenges we ran into

Integrating our diverse tech stack.

Accomplishments that we're proud of

Creating an amazing crowd-sourced platform!

What we learned

Using Expresss and Node.js with native Android applications along with support for real-time notifications with Firebase and Twilio.

What's next for CleanHood

Build a passionate crowd-sourced user base that takes to the streets in a storm and builds on the clean-up drive.
