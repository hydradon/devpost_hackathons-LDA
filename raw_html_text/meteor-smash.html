Inspiration

We had a VR headset so we used it.

What it does

It is a Virtual Reality First Person Shooter game. Be the hero you've always known you were and shoot down the Asteroids before they crash down and destroy the planet!

How we built it

Created 3D Models using Blender, ported the models into Unreal Engine 4 and worked blueprint magic to create the game. Audio was recorded via our on site professional studio (in a dark, quiet corner of a lecture hall).

Challenges we ran into

Having to alter batch scripts to get the game to build and upload onto the phone. Ensuring 3D models were optimised for efficient rendering (This meant creating our own 3D models rather than downloading existing ones). Many algorithmic challenges with creating the game (such as getting UE to recognise GearVR motion controller input).

Accomplishments that we're proud of

We created a FULLY FUNCTIONAL VR GAME IN LESS THAN 24 HOURS!!!

What we learned

How to create efficient 3D models specifically for the desired project. How to interface with Samsung GearVR through Unreal Engine. How to develop and set up android games.
