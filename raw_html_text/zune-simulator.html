Inspiration

Zune was always better than the iPod

What it does

Simulate a Zune interface for your musical desires

How I built it

Used react to create a UI that mocked the Zune interface and device and connected it to the Spotify API to login and play music

Challenges I ran into

Enabling the Spotify auth flow and interacting with the API was more complicated than expected

Accomplishments that I'm proud of

Pretty satisfied with the way this turned out. Fiddling around with the Spotify API was annoying but overall very satisfying once it worked.

What I learned

Read the API and Spec documentation VERY closely, pay attention to EVERY little fine print.

What's next for Zune-Simulator

Enable Albums, Artists, Genres and all the other great features Zune has to offer
