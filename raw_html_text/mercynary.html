Inspiration

Mercy from Overwatch. We didn't want the game to about resurrecting the hero. Old school action games

What it does

It's 2D action game, where you determine the fate of the battlefield by resurrecting dead troops.

How we built it

Game was build with Unity 5 2D and scripted in C#. Game's art was created using Photoshop and Paint Tool SAI.

Challenges we ran into

The biggest design challenge was creating a game about conflict where player cannot fight at all. There were some problems with version control, but they were solved pretty quickly. We also realized late that the deadline was so early, so we had to cut many features we had planned. We also decided not to stay up all night and rather get good night's sleep, strongly believing crunching is not the answer.

Accomplishments that we're proud of

Coming up with coherent and unique idea. Managing to deliver a working prototype we can be proud of. Having a diverse team where we respected each other's ideas and opinions. We are also proud that we were able to create an art style that was consistent and worked well together.

What we learned

What's next for Mercynary

We have plans to keep polishing the game and adding features we had planned from the start.
