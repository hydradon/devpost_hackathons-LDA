• Code That Song!

Inspiration : We both love music! So we wanted to do something creative and unique!

What it does : Displays a list of choices to a user and plays a song intro based on choice. User must press enter continuously to play each note of the song

How we built it : Using Java and Eclipse

Challenges we ran into : importing the Jmusic reference library. Using eclipse for the first time

Accomplishments that we're proud of : It works!!

What we learned : trial and error is the key!

What's next for Code That Song : making longer songs, better timing, being able to play more notes at one time!
