Inspiration

The inspiration for this project was to do something with the Nest. Since there was a camera and a thermostat, and the camera was taken, I chose to use the thermostat.

What it does

Modifies nest thermometers and their temperatures using the leap motion controller and and web platform.

How I built it

Used gestures from the leap motion to interact with angular.js's stored variables, and these communicate with Nest's api.

Challenges I ran into

1) Nest has a very limited development kit (still giving me problems) 2) Hacking alone for an entire project is difficult

Accomplishments that I'm proud of

1) Hacked the entire thing alone

What I learned

Lots to do with the leap motion and using some other company's apis.
