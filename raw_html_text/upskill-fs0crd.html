Inspiration

We were inspired by not being able to find certified and qualified tutors in our daily lives

What it does

Upskill is an app that helps students find tutors with subjects that they need help with. Students can search for Tutors located nearby for convenience, can personally message tutors to plan meetups in person, and can also sign up to be a tutor.

How I built it

We used JustInMind to create the prototype, and Sublime Text with HTML and CSS to create the website.

Challenges I ran into

We had issues with bringing our ideas to life via our code, and improving the functionality of the app.

Accomplishments that I'm proud of

We are proud of using something that we weren't very familiar with (HTML) to further challenge ourselves.

What I learned

We learned the importance of teamwork and using collaboration to our advantage.

What's next for UPSKILL

We would like to improve UI and functionality by creating a better messaging service.
