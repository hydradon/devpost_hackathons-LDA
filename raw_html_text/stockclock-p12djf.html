This app was coded over 24 hours as part of HackTJ. We were inspired to keep this app because it is very hard to be able to look at stock and be able to predict for the future.

This app will calculate a score based on the three best stocks to invest in with High, Medium, and Low Scores.

We built it using XCode and the Quandl API

We ran into challenges with both the BigParser API and the MongoDB API, and getting dependencies to work on XCode.

We are proud of making an easy to use UI and a easy to use app.

I learned that we need to be prepared before hackathons by installing new APIs. I also learned a lot about how to organize my code and use different methods.

We plan to get the database working in the future for better predictions and a better algorithm.
