Mono

In such a fast moving era, poor sleep habits can jeopardize health and lead to future issues. Mono helps to solve these issues by providing an easy-to-use sleep tracker that encourages long and healthy sleep.

Under the hood

Mono's app is made using React Native. In addition, it is powered by a REST API written in Go, using MongoDB to store logs of sleep.
