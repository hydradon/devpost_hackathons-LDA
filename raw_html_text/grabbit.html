Inspiration

A new way of playing around with geo-fencing

What it does

Shares files with users inside of a local radius

How I built it

Using the MEAN stack and gmaps.js we were able to build an applications which could host files and dectect who is around the users.

Challenges I ran into

MongoDB cannot store files, so the hard part was figuring out how to represent the data

Accomplishments that I'm proud of

Being able to implement a map api for the first time

What I learned

How to implement a MEAN project using proper folder structures

What's next for Grabbit

Expand on it to deal with static location files (permanent files stored based on location)
