Slackit

Slackit are plugins developed for Revit and Rhino that enable them to communicate with team members via Slack. This allows the model to alert users if something goes awry and giving a programmable interface for model management and maintenance.

Current functions

Messages are sent to a specified channel (no groups or DMs yet)

Random gifs (via GiphyAPI) are embedded into the Slack attachments by keywords search

Messages included so far:

  • Synch to Central started
  • Synch to Central completed
  • Central model was opened
  • User closed the file without synching to central

Future development

Include groups and DMs Add screen capture and post utility to Revit ribbon (current one doesn't work) Ability to post messages built into the Revit ribbon

More messaging options

  • File size
  • Open/sync takes too long
  • Audit type issues (revit warnings & errors, non-standard practices)
