Inspiration

With all the natural disasters that happened in the US recently we want to offer people a way to easily find community services in order to help out those in need.

What it does

CommuNYC is a web application where a collection of community services are posted to allow volunteers to easily sign up and lend a helping hand.

How we built it,

We built the frontend of CommuNYC with React.js, the backend with Node.js and Express.js, and MongoDB as our database.

Challenges we ran into

We had to learn React.js and how to integrate it to communicate with our backend.

Accomplishments that we're proud of

We were able to learn how to use React and connect the frontend to the backend to allow them to communicate with each other.

What we learned

We learned how to connect the React.js frontend to the backend.

What's next for CommuNYC

We plan to implement geolocations to allow volunteers to search for events nearby.
