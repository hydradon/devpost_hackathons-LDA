Inspiration

We recognized that the homeless are heavily ostracized for society and wanted to create a platform to help them connect with one another and share resources.

What it does

It is a website that allows users and volunteers to sign up and view posts in a proximity/ratings based system. We support tags for the posts that will help users find social support, resources, or other things they may be looking for.

How I built it

We built it on html,css, and javascript to perform the basic functions.

Challenges I ran into

There were challenges with our own inexperience with the languages as well as not having knowledge about homeless people initially.

Accomplishments that I'm proud of

We are proud of being able to make a project that has a bit of our functionality as well as keeps with the spirit of our idea

What I learned

We learned all the languages we used in this project and that we all have a lot of room to grow in terms of technical skill.

What's next for Home-In

For the project to reach a workable prototype, it needs to be run on a server or mainframe so it can collect user data and calculate proximity on posts to display a relevant feed to the users. Also using mainframe we can run analytics to locate the most affected areas and focus efforts and resources to improving conditions.
