Inspiration

In meetings with some developer we have no easy way to track all the time for all developers.

What it does

When you start the plugin you can concentrate on the content of the meeting. By choosing an other issue "Meeting Time for JIRA" automatically starts tracking the time. And after the meeting you can edit the time, choose users for logging, add/remove issues, .... It helps you!

What's next for Meeting Time for JIRA

Its planned to extends the plugin with some functionality like choose groups for logging time, pause a meeting, text notes...

Enjoy the plugin!
