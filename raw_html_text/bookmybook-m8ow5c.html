Inspiration

Being hailing from Asia countries, we felt Textbooks here in US are not in our budget and have felt other CSU Fresno students share our feelings. To overcome this problem, we thought of developing the Web app allowing the students/professor to upload books, trade books, make a wish list.

What it does

Bookmybook.com tagrgets to solve the real life problem... Expensive Text books! Well, we came up with the Web app which welcomes student as well professor for free to upload, trade books and the bonus is our app gives a chance to the professor to post notification about the text book well before the blackboard alerts.

How we built it

We built using the various technologies such as Node JS, Google Book API, Twilio API, HTML/CSS3, Bootstrap

Challenges we ran into

We faced a challenges while the designing normalized data base, low latency framework where user can see the accurate result involving natural language processing.

Accomplishments that we're proud of

We are proud of integrating the application involving various technologies and making our software engineer strong.

What we learned

Teamwork, Co-ordination , Innovation, Motivation ! and about new things Google Book API and Twilio API etc etc

What's next for bookmybook

Students can follow the respective processor and stay alert about the notifications. As well as we are planning to expand to all CSU systems.
