Inspiration

We wanted to help our colleagues to find information wherever they are by voice command.

What it does

Lets us trigger searches by voice command with a custom trained hot word.

How we built it

We used a Raspberry-pi device on which we installed and configured google assistant sdk.
We changed the "Ok Google" hot word, with help from many of our colleagues by training it to answer to "Hey Zitec" using Snowboy.
Using a specific function, a PHP is called in command line that makes it possible for us to add events to the Microsoft calendar

Challenges we ran into

Configuring the Google Assistant SDK with Snowboy
Adding events in the Microsoft calendar
Creating a Trivia Game controlled by voice command. The application is still awaiting approval from Google

Accomplishments that we're proud of

Even though at 8 PM we had to delete and restart from scratch we are proud that not for a minute did we think of quitting.
We are also grateful to the colleagues that helped us pull through when we were in a jam.
We are proud that we managed to use 4 technologies that are new to us and that we learn during the ZiHack

What we learned

How to configure a Raspberry-pi device
How the Google Assistant works and how to personalize it
How Actions on Google works by creating a Trivia Game controlled by voice command with Google Assistant
How to add events to the personal Outlook calendar by external service

What's next for Zitec Assistant

With this application, we want to be able to interact with the e-mail client by voice command
