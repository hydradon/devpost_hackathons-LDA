Inspiration

Civic theme of this hackathon.

What it does

Allows bone marrow recipients or companies representing them to locate populations of people with similar ethnic backgrounds across the U.S.

How we built it

JavaScript, HTML, CitySDK, GoogleMaps API

Challenges we ran into

Working with the CitySDK and integrating Google Maps with the US Census API.

Accomplishments that we're proud of

Working with languages unfamiliar to us.

What we learned

Using poorly documented APIs/SDKs can be frustrating but rewarding in the end.

What's next for Donor Link

Improving website style and obtaining latitude and longitude data from US Census tracts for plotting on a Google Map.
