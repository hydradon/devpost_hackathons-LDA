Inspiration

A few weeks ago we had a misadventure with a "hidden" fire hydrant. Result was a fine of over 60$. In order to avoid such unfortunate events.

What it does

We developed the prototype for an app which will let you know when the location where you parked may result in a parking fine.

How we built it

We used Montreal's Open Data to access the list of parking posts and fire hydrants. We made a front-end in JS which will let you know if you are in a location that can trigger a parking violation.

Challenges we ran into

Developping for an Android was a major difficulty with Android Studio and/or Xamarin. We built it temporarily using JS with the intention of porting as soon as possible to Android and then iOS.

Accomplishments that we're proud of

It works (especially back-end) and is able to test if there are any hydrants/interdictions nearby

What we learned

Learned to use JS and a bit of Xamarin
