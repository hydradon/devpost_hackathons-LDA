Inspiration 

We took two beautiful things that we all love and combined them for something greater.

What it does 

Our app integrates the Doge Meme into traditional Japanese poetry. With a photo upload comes a powerful haiku using tags from the image(from Clarifai) and major Doge keywords.

How We built it 

We incorporated Clarifai with a Node.js server to tag an image the user inputs. We take the tags and sort them by syllable so that we can use our algorithm which generates a haiku. The haiku incorporates a database of doge-words, as well a string replacement function to maximize the doge-ness to their best advantages.

Challenges I ran into 

We ran into many challenges while figuring out different algorithms on sorting the syllables into the strict haiku format. Then we fought against the server to structure the backend in a way that we could gather the data and spit it out in doge speak.

Accomplishments that I'm proud of 

Above all, it works. I’m especially proud of the fact that this could be easily expanded. We essentially built a framework that anybody could create a "*blank*-themed Haiku generator" (ie computer nerd). Also, we thought of the idea.

What I learned 

I learned how to actually structure my files and work with a larger team than I am used to. Our key to success was brainstorming and making it fun.

What's next for Doge-Ku 

Take over the world with Haiku generators? Want to build Haikus like a computer nerd or an astronaut? Cool! Us too! Just give us some time and we will let you generate your own.
