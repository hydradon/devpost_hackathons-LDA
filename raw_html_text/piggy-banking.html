Inspiration

This application was inspired by my own wants for a simpler personal finance app. I wanted to be able to keep track of my money without getting too muddled in specifics and take advantage of blockstacks decentralization to keep the details of the finances private.

What it does

How I built it

Initially attempted to do this as a mobile app but due to technical difficulties switched gear to a web based interface, the platform blockstacks is best supported. Uses reactjs for all functionality

Challenges I ran into

mobile SDKs are in beta leading to limited documentation to learn the api and the eventual movement of the application to a web based platform

Accomplishments that I'm proud of

successfully getting the blockstack api connected to the react app and creating my first react application

What I learned

basic android development blockstack api basics reactjs basics

What's next for Piggy Banking

flush out basic functionality and add data analytic and budgeting tools. Eventually migrate application to mobile as well.
