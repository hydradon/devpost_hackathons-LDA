Inspiration

I wanted to see what interfacing with a large dataset through microsoft's API would be like.

What it does

Use the filters to produce Highcharts graphs. Use the linear regression line to help you understand the data. Compare graphs side-by-side.

How I built it

Mainly js utilizing highcharts great graph-generation.

Challenges I ran into

The API had very little documentation which made life difficult.

Accomplishments that I'm proud of

Using geolocation, using in-depth jQuery's ajax promise-deferred functionality, making a one-page, visually aesthetic app

What I learned

Alot about the JS deferred object, how to use microsoft's api, how to use highcharts.

What's next for Farming for Our Future

Implementing the preset graphs as actual filters that can be changed.
