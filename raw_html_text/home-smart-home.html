Inspiration

Making Smart Homes actually smart.

What it does

It removes the need to manually control your home systems.

How we built it

We detect specific states using sensor data and choose the best course of action accordingly.

Challenges we ran into

Finding the correct sensors and lack of processing power.

Accomplishments that we're proud of

Addressing a real problem that many people face.

What we learned

Smart Homes ecosystem is huge, messy and challenging.

What's next for Home Smart Home

Developing a super intuitive system that is for all age groups.
