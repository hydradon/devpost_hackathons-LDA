Inspiration

Targeted at a college demographic, we aim to address the issues of food wastage and unhealthy eating habits.

What it does

Din Dinz takes in the ingredients in your fridge and generates quick, easy, recipes tailored to your needs.

How we built it

Built using html, css, javascript and jquery front end and SQL lite, python backend. In the future, we aim to extend this to other mobile platforms.

Challenges we ran into

Minimal previous coding experience, particularly back end development Architecture the flow of the system and the app Making it a real-time system How to make it reset state Making it pass data through pages How to get input while maintaining state

Accomplishments that we're proud of

Simple UI Interface Web scraper/Data base

What we have learnt

New languages such as html, css, javascript, web scraping Learnt the connection between front and back end development Gained an understanding of the process in creating a product in industry Collaboration skills from exposure with peers and working as a team

What's next for Din Dinz

Transform it into a functional app which supports both Andriod and iOS platforms with functionalities such as a AI scanner to detect ingredients in your fridge, social integration which allows you to share ingredients and cook with others, filters for both nutrition and dietary, collaboration with external companies such as Coles and Woolworths to track the quantity of your ingredients.
