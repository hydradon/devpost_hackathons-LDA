What it does

Our application does not allow dreams to only be dreams. It also provides a platform for predictive analysis through deep learning as well as a visualization front end.

How I built it

Built on a scalable Linode VPS, the application runs on Java and connects to both mySQL and MongoDB databases. The visualization is coded using HTML and Javascript (D3). The front and backend of the project is highly modular and communicates via JSON and provides a basic API for third party integration.

Challenges I ran into

Strange servlet bugs and Integrating new technologies that we were not familiar with. Also, Gigantic JSON files.

Accomplishments that I'm proud of

Actually making a deep neural net!! Creating a user friendly interface and integrating the the two.

What I learned

How to make a deep neural net! How to use Java webserver. Integrating many different programming languages and technologies.

What's next for M.O.L.A.S.S.E.S

More database api's, csv loading
