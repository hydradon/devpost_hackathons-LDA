Inspiration

We realised that most drivers cannot find the nearest carpark, while reaching their destination. This becomes a hassle as the driver has to go around in circles just to manually look for a carpark.

What it does

We decided to create a easy to use telegram chatbot that uses the driver's current location and periodically finds a path to the nearest chatbot when it is turned on.

How we built it

We used the smartnation's carpark api to retrieve the available parking lots and filter carparks with at least 5 lots available. Then the coordinates of the nearest carpark and the user's current location are used to develop a path that can be view on an online map.

Challenges we ran into

We couldn't get the user's coordinate information through the telegram api.

Accomplishments that we're proud of

We manage to clean and process the smartnation carpark availability api.

What we learned

What's next for ParkingBot

We will fix the bugs regarding user location and upload it to Heroku where it can run on the cloud. As such, users can use chatbot to find the nearest carpark with available lots.
