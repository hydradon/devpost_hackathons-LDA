Inspiration

As students with busy schedules, we wrestle to keep up with our daily struggles. We decided to create an app to allow fellow students to communicate with each other and share notes.

What it does

Join students through the joy of note sharing.

How we built it

Using a Node.js back-end with Express routing and Socket.io for event-based communication, we were able to connect to our MySQL database. In the front-end, we decided to go with React Native with NativeBase and React Native Elements UI kits. For navigating within the app, we chose to go with React Navigation and Moment.js for our date formatting.
