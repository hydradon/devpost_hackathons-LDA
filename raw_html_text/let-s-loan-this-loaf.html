Inspiration

We wanted to find a solution relating to money and financial literacy. We were interested in the impact of community microloans and wanted to make something that facilitated that.

What it does

We created a website that you can log into and view a dashboard that shows the total amount of money available in the emergency fund. From there, you can record a contribution to the fund, view your transaction history,

How we built it

We used python with Flask connected to a database that stored the information for each transaction and account. We bought a domain name for the website and set it up so anyone could reach it. We used HTML and CSS to make the web pages.

Challenges we ran into

We originally wanted to use real money to transfer into the shared account, but we did not have enough time to explore this option fully and ended up creating a database to track the transactions instead.

Accomplishments that we're proud of

This was the first time that most of us we exposed to creating websites from scratch, and we had a lot of fun customizing the site!

What we learned

What's next for Let's Loan This Loaf

Adding more functionality and creating a way for you to directly contribute the money from your bank account.
