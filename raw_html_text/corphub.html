Inspiration

We wanted to help startups have their complete press presented to them all in one place.

What it does

It scrapes the internet for queries relating to the user's input to gather all relevant news and present it to the user.

How we built it

We webscraped with Python and implemented it into Django, and then created the UI with HTML and CSS

Challenges we ran into

We ran out of requests per second when we were making the functionality

Accomplishments that we're proud of

The design the and the functionality of the application is nice and smooth

What we learned

We learned about requests, and the limit that networks have per second

What's next for corphub

Implementing more features to search multiple companies at the same time, and having a faster response time
