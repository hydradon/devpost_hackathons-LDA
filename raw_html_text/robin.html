What it does

Centralized source for Immigrant services to increase synergies between service providers.

How we built it

Python backend on top of an SQL database, with a front end made with some HTML and CSS (bootstrap).

Challenges we ran into

At first we were just going to create a service listing and help them find local services, but halfway through Saturday, we realized there was an existing one of those out there through United Way's 2-1-1 phone number, and the database listing offered by http://www.getgoodmap.com/

Accomplishments that we're proud of

Successfully pivoting our service to expand upon existing database and offerings to provide new beneficial functionality.

What we learned

Learning the services that already exist can be difficult.

What's next for Robin

WeChat OAuth integration Have existing service providers become part of our service provider network Further improvements to user interface to accommodate for ESL individuals.
