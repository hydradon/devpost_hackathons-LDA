• Field of play

Inspiration

Boredom from flashgames based on traditional inputs like arrow keys.

What it does

Players wear the Muse Headset; relaxing slows down the obstacles and makes it easier to collect all the pellets.

How we built it

We use Three.js to render the objects, and the JavaScript APIs for the Muse and Leap Motion to interface with the hardware.

Challenges we ran into

Connecting to all the hardware properly

Accomplishments that we're proud of

Translating EEG and motion input to a graphical display

What we learned

Rendering 3D graphics, interfacing with the Leap Motion, and basic JavaScript

What's next for Mind over Matter

Quiz-style games giving players who concentrate an extra advantage.
