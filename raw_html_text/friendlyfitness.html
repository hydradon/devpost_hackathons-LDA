Inspiration

We thought about how America has an obesity problem. Much of this could be attributed to the fact that we underestimate how much exercise we have to do to burn off how much we eat. If people knew it took, for example, an hour to walk off that donut, maybe they wouldn't eat donuts every other day.

What it does

The user enters in the number of calories the user consumed or wishes to burn, along with the user's age, height, weight, and gender assigned at birth, and the program tells the user how many minutes of different activities the user can do to burn off those calories.

How we built it

Java

Challenges we ran into

It was difficult using graphics in Java, especially creating text boxes that could accept user input, clickable buttons, and formatting everything. We are currently still unable to make all of the text line up nicely.

Accomplishments that we're proud of

Finishing the project in less than 24 hours.

What we learned

The process of coming up with an idea and executing it.

What's next for FriendlyFitness

Having a more aesthetically pleasing UI; expanding the program to determine something like how much food the user should eat, based on their physical traits and lifestyle.
