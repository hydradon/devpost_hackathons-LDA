Inspiration

Our inspiration was the need for customer representatives to better assist customers.

What it does

Super Rep uses Speech-To-Text and NLP technology to suggest solutions based on what the customer is saying.

How we built it

We built it using Node.js and Socket.io for the backend and React.js for the frontend.

Challenges we ran into

Sending the real-time audio data to the socket and then handling the data with node.js.

Accomplishments that we're proud of

IT WORKS!!!

What we learned

We learned about how to use sockets for real-time data transfer and small array manipulation techniques.

What's next for Super Rep

The next step is for T-Mobile to implement it into their current toolkit.
