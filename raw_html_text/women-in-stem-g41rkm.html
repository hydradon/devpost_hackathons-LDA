Inspiration

We were inspired by the design and spirit of the techiesproject and we also believe that increasing the number of women who take STEM subjects to be an important cause

What it does

Provides a resource for women who might be interested in pursuing a STEM major. We have profiles of women who are currently majoring in a STEM subject and we have a brief interview with them. We also provide information about majors

How we built it

We built the website on Django a python framework and used a SQLite db to store the information and used html, css, scss and js for the front-end of the design.

Challenges we ran into

Neither of us had use Python/Django before so it took adjusting and we had some git repo problems at occasions.

Accomplishments that we're proud of

We're proud of this project as we believe it to be addressing an important social issue and we are proud of how the platform is taking shape

What we learned

Django on python

What's next for Women in STEM

We want to take this project forward and expand on it as we believe it is important and is a topic which needs to be more readily discussed/addressed.
