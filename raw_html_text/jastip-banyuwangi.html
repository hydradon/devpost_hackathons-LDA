Ide ini dibuat untuk memajukan UMKM dan pariwisata di Banyuwangi, kami berencana untuk membuat jasa tirip, dimana user bisa menitip buah tangan user yg sedang berpergian dibanyu wangi. Pastinya akan ada transaksi dana menggunakan API BRI. Dengan menitip user yabg ada di banyuwangi terdapat percakapan 2 arah yang akan menambah sosial masing masing user (menambah kenalan atau teman atau mungkin jodoh. Keuntungan bagi UMKM dan Pariswisata Banyuwangi pastinya akan terekspos dengan aplikasi ini. Kedepannya apabila aplikasi Jastip Banyuwangi ini suskses, kami akan membat aplikasi tersedia bagi seluruh UMKM dan Pariwisata di Indonesia.
