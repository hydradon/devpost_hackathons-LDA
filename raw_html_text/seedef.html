Inspiration

Help people who struggle with new languages and wants faster access to the meaning of words,

What it does

Extracts the text from an image and provides the list of words that you might want to look it up. When a word is chosen, the definition will be given

How we built it

At the backend, we worked mainly with node.js using some node packages such as ocr, hashset etc, and some online apis. The front end was mainly React.js

Challenges we ran into

Using new technologies that we have never used before created challenges.

Accomplishments that we're proud of

Being able to extract text from the image and providing the definition of the words that user might want to select.

What we learned

How to integrate multiple online APIs and also node.js especially React.js

What's next for SeeDef

Create a better user interface that us more descriptive to what this product can do.
