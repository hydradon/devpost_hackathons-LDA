Inspiration

Businesses needs to know where their customers are located for effective marketing,improving customer service and better supply chain management.

What it does

It plots the locations of customer transactions from QuickBooks on Google Maps.

How I built it

Java and JavaScript are used to access the API's from Intuit and Google Maps, respectively. JSON is used to interface between the Java and JavaScript components.

Challenges I ran into

We needed to find a way to transfer data between components.

Accomplishments that I'm proud of

We were able to access API's using multiple languages and combine the components.

What I learned

We learned more about using API's in multiple software components.

What's next for Customer Locator

Creating an intensive data analytics application for creating hot zones based on individual items sold and providing search filters for time frame.We are also planning to use sensex data and other available marketing data to create hot zones about the present competition.
