Great Reads is an online ereader .

This project was inspired by the lack of clean online ereaders. We wanted to offer our version of a ereader that was both visually appealing and simple to deliver a GREAT reading experience.

The user can log in with a Google account and start uploading a collection of ebooks. Books can be opened and read right away. Great Reads offers font style, font size, line spacing and page colour customizations for optimal viewing pleasure. Other features include search and arrow keys page navigation.

We started off adventurous and decided to code the project with Meteor.js. However, with no prior experience with the framework, progress was much slower than expected. Halfway through the hackathon, we switched to Ruby on Rails which we probably should have started with.

Potential future features include bookmarking, library sorting and library sharing.
