Inspiration

Make use of every single city people to gather several information about the great city they live in and applying gamefication components to motivate people to contribute.

What it does

App lists challenges for users to take pictures and tag infrastructural points of interest, e.g. fire extinguishers or defibrillators. The users obtain reward points after they successfully tagged new points of interests.

How I built it

We have built our WebApp using the Ionic Framework and our backend service runs a Flask web server that uses Microsofts Cognitive Services API for Machine Learning like image tagging.

Challenges I ran into

Frontend - backend communication, especially http requests with images as payload.

Accomplishments that I'm proud of

We managed to marry frontend and backend.

What I learned

Javascript is a pain in the a** but Python and Flask are lovely.

What's next for TagMunich

We plan to build a rewarding system for active users. Possible rewards would look like free minutes for MVG bicycles or even restaurant coupons.
