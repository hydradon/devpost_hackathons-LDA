Inspiration

We were inspired by the need of people to visualize and see how cars are spread and what they can do in order to take their MOT test.

What it does

It predicts whether a car will pass the mot test or nor and visualizes the total number of cars for each country tested.

How we built it

We built it using a machine learning algorithm and an Open Source given by Harvard with a free license for the visualization.

Challenges we ran into

The connection between the python and nodejs servers.

Accomplishments that we're proud of

We managed to finish the visualization and the prediction both in the same time and getting past all the problems.

What we learned

We learned about RPCs, flask and other back-end technologies plus about what other data visualizations APIs are out there.

What's next for CheckIt

We will try to implement an algorithm which will predict when you will have to take your next technical test.
