Inspiration

Studying and working on assignments with other people can be very helpful, but sometimes none of your friends are near by.

What it does

Connects people who are in the same courses to form study sessions together.

You can create a new study session or join an existing one, filtering by classes.

How we built it

We built an app using Flutter with Android Studio.

We also built a Flask/MySQL webapp on Google Cloud Platform to store and retrieve data for our users.

Challenges we ran into

We're very inexperienced Android developers and we used Flutter for the first time this weekend. Only one of us has any server side experience and he’s not great at SQL.

Accomplishments that we're proud of

Our App

Android noobs learned and built an application with Flutter! Evan learned he never wants to do mobile ever again.

Our Server

It runs and handles requests like a champ! Migrations, connections from local machines,

What we learned

We learned so much about Android development (mainly Evan and Josh), Flutter, REST APIs, server models (mainly Tim), and asynchronous functions. Also Google Cloud Platform is great!

What's next for StudyUp

We want to add friends in the app, a log in page, and even integrate with College room booking services.
