Inspiration

After using an existing image manager on Blockstack, I noticed the pictures were actually not being stored in Gaia nor was that communicated. They were also available to the public if you had the fileName.

What it does

It allows you to create albums, upload photos. Image downloading, deleting coming soon!

How I built it

I built it using Angular 7 and TypeScript. I am using Clarity VM for UI.

Challenges I ran into

Found storing images to be hard and slow. Hoping to learn more about how I can store this on Gaia faster, currently using base64 encoding.

Accomplishments that I'm proud of

Getting it to finally work, I have not worked with images before and found that to be a challenge.

What I learned

Learned how to work with images and how Blockstack auth and storage work.

What's next for Secure-Photo

I will be building out the album feature, image organization with deletion and ability to download. Looking into ways to increase performance so looking at Gzip client side and image compression. As well as multiple file upload with drag and drop.
