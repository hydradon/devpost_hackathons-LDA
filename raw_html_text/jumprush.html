    Development Phase

Inspiration

The game that Inspired me is the game everyone has played, Flappybird. I wanted to change it up to give a different feeling but keep it as interactive.

What it does

It is a game where you use incoming blocks as stepping stones to avoid falling down into the water.

How I built it

For the characters and game objects I used free assets from Unity. I built this game with Unity and Java. I used Unity to build the game design and Java to write scripts for the game objects.

Challenges I ran into

Working alone on this project has been difficult where most of my time has gone towards research. Another challenge was working with Unity as it was foreign to me and getting the scripts right/

Accomplishments that I'm proud of

I am proud that I have made my first game app. I am proud of the time and effort I took into this project and being able to learn new things.

What I learned

I have learned advanced java code and how to use unity in making sprite animations, moving objects, and character movement.

What's next for JumpRush

What is next is Adding different game objects to make the game more interactive and smooth. Also, adding firebase admob and publishing to the app store.
