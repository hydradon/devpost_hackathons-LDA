Inspiration

HackNJIT's 2019 theme is Pizza. Building an online pie-themed multiplayer game will allow to spend a fun time with friends.

What it does

Collab It! - challenge your friends in the pizza knowledge! Build, cook, guess you favorite Pie types online!

How I built it

I used Angular 8 to build front-end part of the projects and used Socket technology for real-time online gaming experience. Hosted via Google Cloud, uses MongoDB database instance on the Microsoft Azure.

Challenges I ran into

Setting up server on the Virtual Machine.

Accomplishments that I'm proud of

Complete one-gaming-room application.

What I learned

I've learned how to set up full-scale web application.

What's next for Collab It! - Guess the Type of Pizza

Adding multiple game rooms, drawing game are in plans for the future. Also, based on the experience obtained from that project socket technology will be integrated into a web portal Scrawlless ( https://scrawlless.com/ ).
