Inspiration

Help people reach content easily and help them to enter programming world or improve their programming skills.

What it does

Learn Programming bot Helps anyone to learn programming easily, Just talk to bot and choose department to begin with and choose lesson. After every lesson bot asks user some question to improve his knowledge. Also, bot includes Chat Room to let users communicate with each other and share knowledge, ideas and more features included in bot. and Last but not least bot comes with two languages ( English - Arabic ). - French soon.. -

how I built it

Learn Programming bot built with: Node.js, Express.js, Mongodb. and some libraries like ( nodemailer, etc..).

Challenges I ran into

Easily deliver information to users whose are new to programming.

Accomplishments that I'm proud of

Bot chat which connect users to each other, How to do section which help users to build web apps and more, Multiple languages in bot.

What I learned:

build softwares more efficiently with fewer lines of code to increase the speed of implementing and handle requests.
