Inspiration

Lack of utilized data on real estate in Toronto

What it does

Using open source crime, traffic accidents and paramedic data this web app provides you with a rating of how well a postal code stand with respect to the rest of Toronto

How I built it

beating our heads into every open source data repo that exists

Challenges I ran into

matching data locations (i.e crime data is in relation to "hoods" which doesn't correlate to neighbourhoods)

Accomplishments that I'm proud of

Finishing with (a little) time to spare

What I learned

ingesting, interpreting and applying map data

What's next for Score TO

Analysis of future trends for neighbourhoods and deployment/ productionization
