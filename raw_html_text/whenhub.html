Inspiration

The inspiration was to create an interface which allows the user to navigate across the content gallery with as fewer clicks as possible.. And one which gives a holistic view of the entire library in one snapshot..

What it does

The multi axis visualization allows a user to jump across content categories for various attributes in a single canvas..

How I built it

The stills were created using Photoshop and the mockup animated prototype was built using Marvel app..

Challenges I ran into

The main challenge was to create a design which accommodates longer strings of text..

Accomplishments that I'm proud of

The subtle colours, circular iconization and single canvas design..

What I learned

The many new features added to Marvel app, especially the hotspot duplication, hyperlinking and embedding..

What's next for WhenHub - Multi Axis Visualization

The conversion of these designs into functional code maybe..
