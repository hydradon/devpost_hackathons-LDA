Inspiration

I really just wanted to see if I could build it. We knew nothing about this sort of program, so we figured we would give it a shot.

What it does

Prints keystrokes into an output file. Cannot detect all keystrokes, and cannot detect shifts.

How we built it

With the help of internet tutorials, and our own intuition.

Challenges we ran into

For the longest time, I couldn't get the output on multiple lines, but eventually we figured out I was using the wrong command and character.

Accomplishments that we're proud of

We learned a LOT!

What we learned

How to protect ourselves from this sort of an attack.

What's next for Simple Keylogger

We don't plan on doing anything with it, but who knows? Maybe we will!
