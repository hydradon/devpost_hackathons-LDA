Inspiration

Exchange Article for Article instead of Article for Money , rational:

  1. You don't feel like wasting money as you use your unwanted items to exchange for something you interested in.
  2. Sensible resource allocation among the society and reduce waste effectively.
  3. Connect neighborhood/community, promote social harmony.

What it does

The website allows people to exchange used second-hand goods with neighborhoods, the exchange process flow allow people to exchange their goods efficiently.
We apply machine learning method to build a recommendation system that provide user-specific goods recommendation on ‘Browse’ page based on user's behaviour/ preference. We also proposed an automatic item matching system, which is able to perform automatic item matching for exchange using machine learning based on big data.

How we built it

The front end of the website is built using HTML, CSS and JavaScript; the back end / server is built using Django with Python.

Challenges we ran into

We don't have enough time to generate training data for machine learning model, however, the framework is constructed.
