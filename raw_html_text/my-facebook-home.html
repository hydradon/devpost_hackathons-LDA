Home automation at the tips of your fingers through Facebook messenger. How much simpler could it get?

We used the Facebook messenger API to create a bot that helps you automate your home.

Functionalities

  • Retrieving indoor temperature
    • Using an MPU 6050 temperature sensor hooked up to an Arduino.
  • Monitoring security cameras
    • Multiple security camera capability using webcam.js.
  • Notifying events
    • Simulated door bell using a button soldered to an Arduino with an accompanying LED.
  • Light conversation
    • It's polite!
