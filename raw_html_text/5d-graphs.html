Inspiration

Visualizing Data in interesting ways

What it does

Takes healthcare data and graphs it in VR

How we built it

Takes data from the data.cms.gov endpoint. Pushes it into a flask api then that api is called by unity to plot the points in VR.

Challenges we ran into

The data.cms.gov api went down for a few hours during development and the team had no experience in unity.

Accomplishments that we're proud of

Getting the flask api setup and actually seeing the data points in VR.

What we learned

How to use unity and a lot of python knowledge.

What's next for 5d Graphs

Actually graphing 5 dimensions.
