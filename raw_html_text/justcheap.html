Inspiration

We're broke. As so many other college students. Therefore, we offer a way students can keep track of sale items on many popular clothing sites.

What it does

It fetches the datas from gilt (online shopping website) API and presents the top results according to user's search.

How we built it

For backend, we use javascript to fetch data. For front end, we use html&css

Challenges we ran into

Most of our team members only know java, so it takes a lot of time for us to learn a new language. We also never use API before.

Accomplishments that we're proud of

Being able to fetch the data and display it. Building front end with languages that we are new to

What we learned

A little bit of languages. Basic structure of web development.

What's next for #justcheap.

Improve the front-end. Fetch data from more websites. Actually get the sale items, not just the normal ones. Ideally, the user might be able to purchase right on our website.
