Inspiration

Altria Chosen because of interest artificial intelligence and its ability to analyze data.

What it does

Reads twitter data related to a subject picked by a user it gathers tweets then outputs the average sentiment.

How we built it

Built with javafx to display the graph, used twitter4j library to read data from the twitter api, and then used google cloud language processor to gather sentiment.

Challenges we ran into

Originally used a library for the NLP written by Stanford but it ended up giving poor sentiment values so had to switch last minute to google language processor

Accomplishments that we're proud of

That the application actually works.

What we learned

How to use the google cloud platform in general. Understanding of Sentiment Analysis as a way of viewing perception of a company.

What's next for AltriaHackAnaylzeTwitter

Would like to add the ability to search twitter before and after.
