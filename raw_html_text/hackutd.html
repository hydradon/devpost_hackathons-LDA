Inspiration

With anime fever in the air, and lots of hardware to rent, we decided to make a sushi chef simulator using Myo arm bands.

What it does

Sushi Chef Simulator 2016™ is an, action packed food preparation simulator designed to create a realistic sushi making experience. The game is cooperative, allowing one player to chop the sushi meat, while the other collects the falling sushi.

How we built it

The project was created using the unity game engine, and is controlled using 2 Myo arm bands. Game resources were made in SketchUp and Audacity.

Challenges we ran into

Only one person from our team had experience using Unity before, and there is a serious learning curve involved in learning to use Unity.

Accomplishments that we're proud of

After multiple collective attempts to use Myo armbands at various hackathons we were finally able to use them in a project.

What we learned

We learned how to develop for Myo armbands, and also learned more about how water and physics work in Unity.

What's next for Sushi Chef Simulator 2016™?

We also planned to add phases to the game other than meat chopping including cooperative fishing where one person operates a harpoon and the other reels in the fish, and cooperative box lifting.
