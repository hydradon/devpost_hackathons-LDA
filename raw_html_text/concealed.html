Inspiration

Personal experiences and what we see happening around us.

What it does

The game provides a user interactive "pick your path" type story where the user decides what will happen during the story to provide a sense of relatability in emotion and experience between the user and the character.

How I built it

We built this game by first collectively discussing an appropriate and engaging storyline, then using photoshop to hand draw and color each and every image shown in the game, as well as using JavaScript for the "pick your path" scenarios to accomplish our goal.

Challenges I ran into

Understanding the JavaScript syntax and familiarizing with CSS and HTML to help enhance user experience.

Accomplishments that I'm proud of

Hand drawings of all the scenes, and collectively finishing the project.

What I learned

We learned to not slack and finish our work by diving head first and driven, persevering through tough obstacles, and patience in the strenuous drawing process.

What's next for Concealed

We hope to add more scenes and address more ongoing problems in our world to create a relatable scene for the users to play, relate, and hopefully learn from.
