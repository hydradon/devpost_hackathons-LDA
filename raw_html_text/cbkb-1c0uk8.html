Inspiration

We wanted to win at code golf competition, and Emojis have the widest character set that we can all understand.

What it does

By using the latest advances in Javascript technology, we compile your CBKB in the browser and then offer you the ability to share it with the world!

How I built it

The parsing is done using the compiler compiler Jison .

Challenges

Developing a syntax for pattern matching and its equivalent code generation.
