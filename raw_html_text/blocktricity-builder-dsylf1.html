Inspiration

We have been working on the Blocktricity (blockchain based automated demand management system) for over a year now but we are facing trouble to get the customer to collaborate in order to build it.

So we decided to design a platform for the participants to share and manage their data first and then simulate and optimize their assets.

What it does

It displays the participant and its community data in the the dashboard tab.

It manage assets and contracts in the profile tab.

How we built it

We built the interface with React on top of data provided by the spnsors. We simulated a user and its community.

Challenges we ran into

The main challenge was to set the data for visualization and simulate a user case. We also work on the storage valuation.

Accomplishments that we're proud of

We did IT !
