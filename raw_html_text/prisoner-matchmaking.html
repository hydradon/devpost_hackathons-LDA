We realized that it offers great benefit for people coming out of prison to get back into the workforce. There are many companies which are interested in hiring ex-prisoners, and this service will connect the businesses with them.

This website connects ex-prisoners with job opportunities. Businesses can subscribe to the service and can find ex-prisoners to employ.

It took us a long time to set up our backend. We used SQL and PHP to create a database which would include the names of the people and businesses.

We are proud of getting a good-looking website and creating a functional SQL database.
