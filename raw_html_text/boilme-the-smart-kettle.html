Inspiration

We are inspired by the internet of things.

What it does

BoilMe is a smart kettle that can be controlled by a webinterface, from here you can start the boiling. Furthermore it provides realtime data of tempature, waterlevel and estimates time for your hot water.

How we built it

We have had to figure out the way we are going to measure temp, water level. We have made temperature sensor from basic components. Measuring water level was done using force sensitive resistive plate. In order to establish wireless communication we have used Spark Core from Particle, but also tried Esp8266 wifi device with Mbed. The platform holding kettle was designed by us and cut using laser cutter. To control power mains we use relays.

Challenges we ran into

Setting up the whole HTTP server on embedded devices, which was easy with Spark Core, but hard with Esp8266 + mbed. However we had usage challenges with spark core, issues with connectivity and stability.

Accomplishments that we're proud of

Website app, laser cut platform

What we learned

HTTP protocol and how actual servers works behind it. How to set up HTTP server on embedded device.
