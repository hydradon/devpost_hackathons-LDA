Inspiration

Control all time the price of the new cryptocurrencies and virtual coins... it's funny for me

What it does

Show the price in euros or US dollars of the main virtual currencies like Ethereum, Bitcoin, Ethereum Classic, Litecoin, Voxels

How we built it

With XML and Javascript, JQuery and AJAX and the Atlassian SDK to create the plugin for JIRA Server

Challenges we ran into

Different providers and API's to obtain the same values. We use http://coinmarketcap-nexuist.rhcloud.com/

Accomplishments that we're proud of

The dashboard works fine, the values are refreshed well!

What we learned

To create a new Dashboard and put in a plugin! Now seems easy :D

What's next for Jira BlockChain Viewer

Make the config panel to set values, add new virtual currencies, etc.

IMPORTANT NOTES

While Atlassian approves the Atlassian Marketplace submission you can download the gadget in XML format (the first link). Go to ADD GADGET in your DASHBOARD, then MANAGE GADGETS and install the URL of the XML file. That's all!
