Inspiration

Inspired by the hackathon theme for cyber security we tried to build on an existing idea for encrypting pdfs.

What it does

We used HTML and python to create a website were any individual can copy and paste text, which is transferred into a file that is shuffled to be encrypted and unshuffled when a password is entered.

Challenges we ran into

How to connect our python programs and HTML files. Enabling HTMl to access private data files.

Accomplishments that we're proud of

Making a website for the first time.

What we learned

Applications of coding. Maybe in the future we could create a password creator.
