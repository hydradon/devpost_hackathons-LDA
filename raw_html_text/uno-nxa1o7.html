Inspiration

How we built it

We used python 3 to code all the files

Nothing much, just complications in coding the nuances of the rules

Accomplishments that we're proud of

Creating a working computer system that chooses the cards to play

What we learned

I learned how to explain what I'm doing in such a way that an inexperienced person can follow, and my partner learned how to code in python

What's next for Uno
