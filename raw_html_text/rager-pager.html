What it does

It gives you the littest hotspots in town. Hit the Fire and we'll drop the fire. You can even see in real-time fires being dropped all over the world

Challenges we ran into

  • The Google provided clustering module did not have the features we wanted, so we used an updated module provided by another party. Credits are given in the file and obtained from GitHub.
  • Each of us made edits to files that relied on outdated parts of each other's code. We had to create a framework of functions we would use, then we re-merged all of our code.

Accomplishments we're proud of

  • Obtaining a SSL certificate and using HTTPS

What we learned

  • How to accomplish something
  • Servers are finicky
  • Servers take a while to update to the latest version
  • Must be able to take away privileges without having to close someone's tab
  • Having multiple places be able to read/write a file gets confusing

What's next for Rager Pager

  • Real-time fade away of flames after 30 minutes, Moderation in terms of comments
  • Expansion into college campuses
  • Incorporating Button with Uber, allowing users to take an Uber to the littest hotspot of their choice
  • User set filters that only let you see hotspots that have a certain amount of litness or below a certain amount.
  • Animation of fire
  • Load screen
  • Only allow users within a certain distance to add comments, in order to prevent rival hotspots from slandering each other
