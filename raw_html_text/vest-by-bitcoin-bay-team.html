• VEST

Inspiration

We were inspired by the power of Blockchain to create a finite number of digital assets – we wanted to challenge ourselves with using those same digital assets to back physical assets.

What it does

Creating a dApp that allows a user to mint then pair, a Unique digital token with credentials to a physical asset in this case a limited run of T-Shirts

How we built it

Our PoC dApp UI was built with a Material Design UI library, a QR Code scanner library was used to scan the IDs of our T-shirts . The smart contracts are all published on the Rinkby Ethereum Testnet. Web3 was used to bridge the the Frontend with the Smart Contracts. The planning and UX of the project was coordinated via Figma.

Challenges we ran into

Learning Material UI in one afternoon. Pivoted several times on Friday night to Saturday morning on the purpose the of app we want to build and the problem's we wanted to solve.

Accomplishments that we're proud of

We're so proud that we put this all together in ~36 hours.

What we learned

Blockchain Hackathons are amazing
