Inspiration

Castlevania and Metroid

What it does

Plataformer with manual Scene selection, ground and air enemies that follow pathing, custom sprites and animations, tileset, movement, jump, double-jump, health system that goes down when falling off map, game over screen letting you restart map or exit game (only able to exit once exported), unfinished enemy collision.

How I built it

Using C#, piskel and aseprite.

Challenges I ran into

Double-jumping, collision mapping.

Accomplishments that I'm proud of

Actually finishing a competent movement system, created all possible sprites for the game, functioning enemy paths.

What I learned

How to use C# with Unity and a multitude of methods that can be used in Unity.

What's next for Yeti The Wizard

Hopefully finish the game when I have more time and sleep.
