• Online editor and chat

Inspiration

We are three friends that make projects for fun, at work we use daily tools like Google Docs and Sheets for share documents and collaborate with clients and collaborators, so we think we can make a similar tool for developers!

What it does

Collaborator.io is an online editor, where you can create files and code and have this available in the cloud, if you wish you can share your code at fb with friends so they can enter to the app and collaborate with you in real time. So everybody can see the changes of every friend at same time. Also we have a chat so you can have a direct channel for speak with everybody working in the file.

How we built it

We read from the contest three days ago, so we use create react-app for bootstrap our project. We use firebase for have a real time db with the file information. Also we add facebook login and share button for share the code with friends. Finally we use node.js for the chat.

Challenges we ran into

We have few time so we have too create a simple but good mvp.

Accomplishments that we're proud of

Our work team and organization in few days.
