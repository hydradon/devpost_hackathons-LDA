Inspiration

Between aging elder and constantly growing population we need better health care access for all. Thus, we strive to design something to make a difference.

What it does

Collect data from a patient. The patient can then send the data (heart rate, temperature, etc.) to his/her doctor for evaluation.

How I built it

The NodeMCU collects the data from a temperature sensor and an electrocardiogram and sends it to the Firebase data base. The patient can then see the data collected in real time on his or her phone via a our application

Challenges I ran into

Learning to use React-Native and learning to debugging Debugging the electronics for the NodeMCU Debugging the firmware for the NodeMCU Learning to use Firebase Debugging Network connection errors

Accomplishments that I'm proud of

Making the real time data acquisition work for the first time. We could see the heart rate and temperature in real time.

What I learned

See challenges section. We pretty learn from all those challenges
