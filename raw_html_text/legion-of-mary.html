Inspiration

I have always started with Legion of Mary prayers. My first Android and iOS app are based on the Legion of Mary prayer. So when I start learning Alexa I started with the same concept. It is like the Hello World program we all start when we learn any programming language.

What it does

It starts with the daily prayer. Users can choose different prayers from the list. It is primarily targeted for the Legion of Mary prayers

How I built it

Using Amazon provided tools in the console and sample programs.

Challenges I ran into

Some of the texts are not recognizable by Alexa

Accomplishments that I'm proud of

Added exception conditions to make seamless flow.

What I learned

Alexa voice programming

What's next for Legion of Mary

Add the prayers in multiple languages support by Alexa
