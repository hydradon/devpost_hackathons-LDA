Inspiration

Before this hackathon, we never really worked on building a bot for a game. We thought it would be an interesting challenge to work on during GUH.

What it does

The bot play the game MS Tanks.

How we built it

We used Python to build the bot.

Challenges we ran into

Time Constraint. We had different ideas, and it took a while to settle on one. Latency with the server impaired the precision required for the target tracking function.

Accomplishments that we're proud of

Coming up with a mathematical model to hit moving targets on both straight and curved trajectories.

What we learned

Machine learning isn't always the answer.

What's next for Bot-o-matic 3000

Some of us would like to use machine-learning to make it better.
