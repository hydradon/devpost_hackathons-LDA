Inspiration

Travelling in a foreign country can make you feel like an alien on another planet. If only there were some free and easily accessible way to find out what an unfamiliar object is...

What it does

Wazzat uses your Android device's camera or photo gallery and Computer Vision (from Microsoft Cognitive Services) to recognize (tag) the unknown object in question. The app then provides the user with general information about the object (or the most relevant tag) from Wikipedia.

How we built it

With Java (Android Studio), help from stackoverflow and mentors.

Challenges we ran into

Implementing the Computer Vision API, parsing information from Wikipedia using its API.

Accomplishments that we're proud of

Figuring out how to develop an Android app with Android Studio.

What we learned

Ideation is easy. Implementation not so much.

What's next for Wazzat

Implementing a tag hierarchy to make results more accurate and relevant.
