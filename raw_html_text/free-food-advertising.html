Inspiration

Who doesn't like free food? The need to advertise events on campus inspired us to build this small web application that benefits students and event organizers.

What it does

Displays current event locations on google map. Allows organizers to enter the events.

How we built it

We integrated Google map and markers with Javascript to display all the current events and PHP for accessing database on AWS.

Challenges we ran into

Connecting PHP and Javascript.

Accomplishments that we're proud of

Using Google maps API with AWS. Places multiple markers with infowindow.

What we learned

Connecting PHP and Javascript.
Using AWS for accessing database.
