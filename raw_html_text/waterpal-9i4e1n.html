Inspiration

We wanted to provide a way for people to stay hydrated around NYC.

What it does

Display water fountains on a map for people to get clean drinking water.

How we built it

We used the MERN stack and heavily used the leaflet API to provide information for nearby drinking location. We populated our database with an NYC dataset. Also with love.

Challenges we ran into

Learning new APIs and trying to incorporate as many APIs as possible while still maintaining the spirit of the application.

Accomplishments that we're proud of

Working successfully as a team.

What we learned

An application does not need to have many features for it to provide value to the end-user. If anything, a simple and streamlined user-experience is key to making the application attractive.
