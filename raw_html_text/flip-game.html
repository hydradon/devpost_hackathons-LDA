Inspiration

We just want to test out the unreal engine and see if we can create something fun from it.

What it does

It's like the memory test, you can see the numbers on each block at the beginning and they will becomes invisible once you hover your switch over. Then you need to flip the numbers in order and try to score as much as you can by memorizing the correct locations of numbers.

How we built it

We used the UE4's blueprints system

Challenges we ran into

Blueprints are completely new to us and there are lots of things to adjust to.

Accomplishments that we're proud of

We made an actual game (even tho far from perfect)

What we learned

Experience with UE4

What's next for Flip Game

Improvement gameplay and more customizable options. And a menu.
