Inspiration

To reduce Carbon Footprint

What it does

It's an app that show you you calculate the kilowattage/wattage usage of your household utilities.

How I built it

I built it using Xcode, and Swift, Apis from data.energystar.gov

Challenges I ran into

It was very hard at first to parse all the JSON data I was getting from the API, I was able to do that later with the help of the apple developers,

Accomplishments that I'm proud of

I have finished all three view controller that would display infos for TVs, Computers, and Lightbulbs.

What I learned

I have learned a lot, since I build all my project alone. I have learned how to parse json data properly, how to make a detailed view controller, how to populate a label from an API and the list goes on...

What's next for Carvest

What next for carvest is more Sections, hopefully two more with one for Air Conditioner and maybe some other device. Plus the existing sections' brands will have more info about them so users can compare products based on that information.
