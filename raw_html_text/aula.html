Somos un equipo que trabajamos en una universidad que tiene un porcentaje muy alto de estudiantes extranjeros. Algunas titulaciones las impartimos en inglés pero la mayoría se imparten en español. Para ayudarles a mejorar su conocimiento de español y poder mejorar así su integración académica y social nuestro equipo de transformación digital hemos creado esta skill que estamos seguros les puede ayudar de una forma lúdica y eficaz a mejorar el nivel de español.

Con nuestra skill, el usuario puede interactuar de varias formas: puede elegir aumentar vocabulario, practicar la escucha, hacer test y aprender gramática con ejercicios orales. La skill Aula+ está pensada para mejorar cualquier nivel de español. Queremos preparar ejercicios de nivel básico, medio y avanzado aunque en esta demo, tenemos preparados ejercicios de nivel A1

Cuando el usuario se conecta por primera vez, Alexa le pregunta si quiere hacer una prueba de nivel, o realizar directamente ejercicios de un nivel determinado. Alexa ofrece escoger entre los niveles A1, A2, B1, B2, C1 y C2. Cuando el estudiante elige un nivel determinado, entonces Alexa le da 4 opciones: gramática del día, vocabulario del día, ejercicios de audio o pruebas de nivel. El usuario puede interactuar con todos los tipos de ejercicios preparados. Una vez termina las prueba propuestas, Alexa le envía una mensaje a su correo electrónico con la puntuación obtenida. Hemos visto que esta función podemos desarrollarla con la API Game ON SDK pero en estos momentos no es posible implementarla puesto que se necesita autorización por parte de Amazon. No obstante, la hemos desarrollado con el envío de la información a una base de datos externa que usaremos para explotar con Power bi (herramienta para crear cuadros de mando) Mediante la API Reminders, Alexa recuerda al usuario a una hora determinada que tiene disponible la lección del día. La Skill está conectada al servicio de Dynamo BD para mantener el listado (clasificación) de todos los usuarios. Si el usuario quiere, puede preguntar por su situación en el ranking para saber el grado de avance

En un futuro nos gustaría desarrollar esta skill en más idiomas y contemplamos la posibilidad de usar pantallas para realizar más tipos de ejercicios.

Nuestra skill ha sido construida directamente en la función Lambda usando el almacenamiento de Bucket S3. Además para el envío de los mensajes al correo electrónico, hemos usado Microsoft Flow y para crear los cuadros de mando con los resultados, hemos trabajado con Power BI. Para las métricas de uso de nuestra skill se crea automáticamente una hoja de cálculo en Microsoft Excel que luego analizamos con Power BI.

Hemos aprendido que las posibilidades con los asistentes de voz son muchas y que además el uso de esta tecnología va a ser esencial en un futuro. Las nuevas generaciones ya están acostumbradas a buscar todo tipo de información a través de voz. Por otro lado, queremos investigar y profundizar en las posibilidades de añadir APIs para mejorar todo tipo de interacciones con el usuario.

Por último, nos gustaría añadir que nos ha encantado participar en este evento por su interesante reto y su gran organización. Muchísimas gracias!
