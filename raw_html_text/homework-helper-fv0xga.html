Inspiration

One of our team members had a sibling that did not enjoy doing homework because of the difficulty.

What it does

Our skill helps kids to do their homework by solving math and spelling problems.

How we built it

We used the AWS Lambda and Amazon Developer Console to set up the skill.

Challenges we ran into

We had trouble with setting up our Math function and getting it to convert a string to a math equation that JavaScript could solve.

Accomplishments that we're proud of

The team was really proud when we got our app to run for the first time.

What we learned

Our team was able to learn a new method called the eval() method which solves a math problem in a string.

What's next for Homework Helper

We wish to add more problems that kids would ask to the app.
