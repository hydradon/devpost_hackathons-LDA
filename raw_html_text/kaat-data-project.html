Inspiration

Higher institutes of learning benefit greatly from having a diverse faculty, staff, and student body. However, this data is difficult to find and understand. Here, we plot the diversity data graphically to hold schools accountable to creating a fair and inclusive environment and make it easy for students worried about fitting in to select a school.

What it does

Demographic Data of Texas Universities plots the time-series demographic data of the student body for several Texas universities in an easily digestible format.

How we built it

We found the data at data.gov, removed unnecessary columns with pandas, and graphed them using Matplotlib. The html files were created and then hosted locally using flask.
