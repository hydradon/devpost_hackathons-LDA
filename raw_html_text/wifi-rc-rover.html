Inspiration

Combine various environments to control a rover via webpage and see video feedback via ip camera.

What it does

The raspberry pi hosts a webserver that serves a webpage with buttons that can be used to control the rover. The arrow keys can also be used on the web page. The phone on the rover streams video data through an ip camera which gets read by a python script that can be used to identify objects as people, tables, chairs, and more.

How we built it

Raspberry Pi, Rover, H-bridges, phone, and more.

Challenges we ran into

Busecure did not allow us to connect to the raspberry pi, so we had to make a hotspot out of our computers and phones to connect to the raspberry pi while having an internet connection. We also had to learn enough javascript and web programming to get what we needed.

Accomplishments that we're proud of

We got it to work!
