Inspiration

We love video games and we thought of making a problem-solving virtual environment.

What it does

It is a "escape the room" simulator.

How we built it

We built it using Unity.

Challenges we ran into

A lot of challenges including hardware compatibility since it only runs on windows and we both had macs, finding the right plugins to use, and learning a brand new programming language.

Accomplishments that we're proud of

We were able to create a virtual environment for the first time ever.

What we learned

We learned a lot of C# and the Unity software in general.

What's next for Escape the Room

Better puzzles, more fluid mechanics, and smoother animations.
