Inspiration - The problem we want to solve

There are always issues to manage in any type of property (e.g. leaking taps, missing or broken appliances or equipment, …).

If you manage several properties:

  • How can you efficiently capture all the issues in all the properties you are in charge of?
  • What is the best way to coordinate everything, manage your team's activities and make sure everybody is on the same page?
  • Where can you keep track of all that has been decided?

All these are significant pain points for people like property managers, real estate agents, contractors or property investors.

What it does

We are building a property management App. Unee-T is a Software as a Service (SaaS). Our goal is to increases productivity, transparency and customer satisfaction.

We have a narrow functional focus on case management and Inspection Reports .

Unee-T allows user to:

  • Capture issues easily.
  • Coordinate and communicate to agree on a solution.
  • Organize work to be more efficient.
  • Keep track of what has been done and remember what was decided.
  • Generate Inspection Reports and Handover Reports directly from a mobile phone or a tablet.
  • Avoid disputes and disagreements.

With Unee-T, users can easily search and retrieve information or generate reports from all the issue recorded.

How I built it

Unee-T is a web app built with a Meteor front end, Mongo DB and MySQL/Aurora DB. We chose to built a web app and not a native app to

  • Maximize platform compatibility: Unee-T works on iPhones, Android phones, Computers, laptops, Tablets.
  • Minimize the "frictions" for new users: there is nothing to download or install: an invited user just has to click on a link
  • Minimize the memory space taken by Unee-T on a user's phone: there is no app to download or install.

We use AWS as our infrastructure to make sure we have maximum flexibility and scalability. We use AWS Lambdas and REST APIs to interconnect the various building blocks that we need.
