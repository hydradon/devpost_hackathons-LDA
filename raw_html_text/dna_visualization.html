Inspiration

This is an open-source project for LifeFoundry. Because Biology PHDs always spend tons of time on type in DNA series, we want to save their time.

What it does

As a result, we wrote a web program that automatically identifies amino acids with a long string of DNA series.

How we built it

We use HTML5, CSS, JS and d3 library for the data visualization.

Challenges we ran into

d3.js has ambiguous documents about how to implement hover function so we spent lots of time build from the ground with on-click event.

Accomplishments that we're proud of

The diagram is easy for readers to read.

What we learned

We learned a lot about d3.js and techniques for data visualization.

What's next for DNA_Visualization

We want to build a bar chart which is another form of visualization on the same dataset that helps reader to understand the DNA sequence.
