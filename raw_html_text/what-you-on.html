Inspiration

This application was inspired by the challenge in planning events with friends.

Challenges we ran into

We had never used google cloud functions and this proved to be a struggle to get the first function up and running. Additionally we had no experience with MongoDB and had a lot of trouble with connecting to it through google cloud

What's next for What You On

We plan to expand the functionality to include a lot of the best functionality from all calendar applications. Additionally we hope to add an automated system of event planning for friends
