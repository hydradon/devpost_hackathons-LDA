Inspiration

We wanted a non-JS option for compiling, testing and deploying smart contracts for long, so here it is.

What it does

Compile and tests Smart Contracts, along with integrations with security tools.

How we built it

We built it using golang, goblin for testing, automatic generation of testing templates, and integration with MythX

What's next for Insulin

Become plugin to several IDEs
