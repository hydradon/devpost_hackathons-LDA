Inspiration

Alcoholism remains a leading cause of death and suffering. Despite the swift advancement of society in the realm of technology, many individuals still struggle with the age-old issue of addiction.

Tracking progress in sobriety and trends in alcohol consumption have been shown by groups like AA to be effective tools combating alcoholism, so our solution is to apply AI towards this issue to help track and detect alcohol consumption.

What it does

Sobriety is a mobile app that uses the latest computer vision methods to provide a probability index of how drunk a subject in the camera is. This emphasized feature is embedded within an account system that helps log the user's history and track other facets of sobriety.

How I built it

The front end was built using react-native, and the back end seamlessly communicated with the Firebase NoSQL database. The machine learning module, trained using Google Cloud AutoML, was hosted on the Cloud and communicated with the device using REST APIs.
