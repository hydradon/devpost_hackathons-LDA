Inspiration

There is a large number of people who have died/severely injured because of delayed ambulance response. Specifically, the ambulances typically got caught waiting for red lights and for traffic to clear.

What it does

Our program simulates what it would be like if the ambulance were attached with a functional device that would allow traffic lights to anticipate the arrival of the ambulance and clear the traffic ahead of time.

How we built it

We build a program that simulates traffic and an ambulance that is trying to get to its destination using the device.

Challenges we ran into

1-Implementing the interaction between cars and traffic lights. 2-Implementing the collision between car. 3-Implementing an algorithm that allows the traffic lights to change to green when the ambulance approaches.

Accomplishments that we're proud of

1-learning how to use a new programming language (JavaScript) in order to build a webpage. 2-Coming up with an idea that has the potential to save many lives.

What we learned

1-JS 2-P5 library 3-effective teamwork/work distribution

What's next for The Green Carpet

We believe that the idea of The Green Carpet has the potential to be implemented in ambulances across the country. However, there are some challenges need to be overcome. Being that technology required is relatively low-tech, we have to ensure that The Green Carpet would be implemented with the top security measures.
