Inspiration

We were inspired to make a game which helps players learn what you do at a cornhacks event.

What it does

It is a text-based game which offers several choices depending on the actual schedule plus a few interesting embellishments.

How I built it

We used C++ to program the code and had a team of writers for the majority of options.

Challenges I ran into

It was difficult to coordinate work on the specifically, so we were limited to one computer doing coding at a time.

Accomplishments that I'm proud of

I am proud of how easy it is to add decisions and expand the storyline.

What I learned

I was refreshed on how to use pointers in C++, learned how to do structs in that language and also various ways to pass arguments and get data outside of the specific C++ language.

What's next for The Genuine Cornhacks Experience

We will be presenting today and there is a potential random number generator we will add for the program.
