Inspiration

There are many sites in the Internet that offers to teach programming languages , but those aren't quite perfect yet. Those sites teach students programming subjects by using such easy and simple examples , which is good and necessary at first but the level of examples needs to be pump up a bit. Like focusing on Competitive implementation and complexity of a programming language , there are some sites that offers that features as well but then again they only focus on competitive programming.

What it does

We are working on an idea in which we merge the course learning with competitive implementation. Student learn programming language with an automated track system which records the progress of student while learning. Student shouldn’t just learn “Basic” programming language , they should focus on other ways of implementing those programming knowledge. At last the student is able to make his/her own project or they can contribute to an open source project and work with other peoples.

How we built it

We created a website by using HTML, CSS, JAVASCRIPT, PHP, , MySQL in which we created a track based learning system which helps students learn programming languages in a procedured way and not just language but also how to implement the programming knowledge in competitive field to improve programming skills and also at the end a project session is included to finalize the course.

Challenges we ran into

Accomplishments that we're proud of

What we learned

What's next for Track system based learning [CodeTractors,Education]

We're going to merge the programming language course with competitive learning and project creating so that whoever learns from this system are fully skilled.
