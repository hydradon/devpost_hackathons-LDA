Inspiration

We realize that on most community-based platforms does not support on-the-fly services. What this means is if a platform is missing a feature, users will have to send feed backs and wait for software update. However, features that users need aren't always big ; In majority, users seek only minor tweaking . Our project aim to provide a solution to this problem.

What it does

In general, we will allow users of our platform to implement plugins for our website that does some arbitrary computation. This plugin system will support web, so ideally plugins will be in javascript.

How I built it

We built a REST API in VB.NET and a Database and JSON data parser in C#. Peer-to-peer client in VB.NET

Challenges I ran into

Anonymity is a flag in our project. We consider anonymity an important feature but is not required. Security was the biggest concern for us. We discussed about solutions to prevent our platform from becoming malicious or botnet-like .
