Inspiration

With inceasing population day by day and so the crime rates.We want to solve the typical issues like identity criminals on the run ,find missing children and also the victims of human trafficking or to keep an eye on a person of interest. It may be used by all the police station. when missing are listed to database, police can become alert as soon as they recognised by facial recognition.

What it does

A web portal that will help police to search missing persons or criminals through the installed CCTV cameras across the city in a real time. If a person is detected on the images/videos taken by camera it will inform respective police station in a real-time with his location and time. Police can see the ‘last seen’ of the person and navigate to that place to catch the suspect and a User friendly facebook Chatbot for status intimation

How I built it

We created a ML based system that uses MTCNN and face embedding model to recognise missing persons. This ML based system is interacting with firebase and firestore using REST API. A web interface using REACT, EXPRESS, NODE which helps users view the current status of the case. It uses JWT token to establish an industrial level security. Along with this, we have also built a dialogueflow chatbot which is integrated with facebook messenger which helps users interact and get the status of the case with the help of its caseID.

Challenges I ran into

It was quite hard to find dataset of faces of Indian people.

We created and successfully deployed a chatbot that helps users know the status of the case. We have successfully instreaming the live data to and from firebase to python. Our biggest achievement is putting everything together on a single plate in less than 24 hr under immense pressure.
