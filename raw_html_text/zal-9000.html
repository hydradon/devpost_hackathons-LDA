Inspiration

We just thought that searching in a shop catalog should be as close as possible to speaking with a real shop assistant.

What it does

No checkboxes or filters or controls. Just speak or chat with him about what you are looking for, interactively.

How I built it

We built a backend which takes care of interpreting the phrases coming from the user into articles suggestions.

Challenges I ran into

Natural language processing is a very overwhelming task; even in a narrow domain, the ambiguity can destroy any attempt for deterministic methods.

Accomplishments that I'm proud of

Finding a couple of interesting (although probably not novel) ways of interpreting user queries and developing it as an API rather than a monolithic product.

What I learned

Building GUIs with Python really sucks.

What's next for ZAL 9000

Depends on people's feedback!
