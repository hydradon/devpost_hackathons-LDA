Inspiration

We made this website because we are often very lazy to type the code whenever our professor writes it on the board.

What it does

The website, Image to Code, turns a simple image of handwritten code into actual code displayed on your computer!

How we built it

We built it the program using Imgur and Google Cloud Vision APIs and Python. We built the website through HTML and CSS.

Challenges we ran into

For everyone on the team, we never used APIs before and have little knowledge of Python. 4/5 of the team members also never been to a hackathon until Cutie Hack 2018.

Throughout the event, we attempted to use Microsoft, Imgur, and Google Cloud Vision APIs to read the text in the photos; however, we only successfully got the Imgur API work and implemented into our project. We also attempted to use the Microsoft API, but it never worked, so we opted for Google Cloud Vision API. We got the Google Cloud Vision API to work, but we were not able to figure out how to incorporate it into our website.

Accomplishments that we're proud of

We learned how to collaborate in a time-pressured environment. We also learned what are APIs and how APIs work.

What we learned

We learned how to approach our next hackathon! We also learned about APIs!

What's next for Image to Code

Connecting the Google Cloud Vision API with the website
