Inspiration

Arcade emulator for the modern age.

What it does

Brings together all the arcade games from your computer into a single, unified interface, which can be accessed by phone.

How we built it

Electron desktop application and Android Studio mobile application.

Challenges we ran into

Both Electron and Android development were new to us, so learning and integrating both sides was a challenge. Mapping phone axis input and buttons to game controls.

Accomplishments that we're proud of

Writing our first Electron and Android apps :)

What we learned

Electron, Android development and choosing what is to be part of an MVP.

What's next for Wayback Arcade

Multiplayer!
