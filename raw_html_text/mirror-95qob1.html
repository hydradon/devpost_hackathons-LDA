Inspiration

We were inspired by the interview process and recent developments in AI regarding facial recognition.

What it does

Mirror is a platform that will allow an interviewer to better understand how well their interviewee handles questions using facial recognition and analyzing emotional data gathered from the interviewee. Mirror collects this data from the interviewee and displays it with very pretty graphs.

How we built it

We build this app using Node.js and with API's from Microsoft Azure.

Challenges we ran into

The whole project had and still has a bit of problems. One of the biggest challenges was getting the video chat to work and after a lot of debugging we were able to get some nice video functionality.

Accomplishments that we're proud of

We're very proud of how we used Microsoft Azure to get real time analysis of the interviewee, the graphs also look pretty.

What we learned

We learned a lot about constructing a video chat platform, displaying data using Chart.js, and using Microsoft Azure.

What's next for Mirror

The next steps would probably be brushing up and styling the website used for calls. IBM Watson integration for personality insights was something we were trying to put together and we think it would leverage the platform to a very effective service.
