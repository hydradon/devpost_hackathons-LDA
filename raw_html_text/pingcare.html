Inspiration

Small maintenance issues - burned-out lightbulbs, damaged bricks, malfunctioning projectors - are often left unfinished but now we realize how widely applicable this can be!

From small to large, problems arise everyday and sometimes without notice. It is Ping Care's job to address issues no matter the size.

What it does

Ping Care is a real time web app that illustrates the density of problems as small as maintenance requests to as big as WiFi outages!

How we built it

For the front end, Marc used html, css, bootstrap, and jquery. For the functionality, Emmett used javascript, the Google Maps API, and inkscape For the backend, Alex used python, django, django REST library, MySQL, Apache, and hosted the website on a digital ocean droplet

Challenges we ran into

Learning some javascript and django on the fly. Having good communication about our first major group project and merging everyone's code. Also, REST API's were hard to work with

Accomplishments that we're proud of

We were able to create a functional web app with no prior experience. For two of us, this is our first hackathon, and we are so happy to create something that would be helpful to the world and to proudly call ours :)

What we learned

We Learned everything listed above as separate entities and combined those things into one. We worked as a team giving each person a vital part of the project.

What's next for PingCare

Having multiple user interfaces for different hierarchies and companies. Making more user input public and friendly for others using the app
