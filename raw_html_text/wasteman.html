We were inspired to do an environmental hack because of the current state of the world. Our mobile app is designed to harness the power of machine learning to help users determine what items can be recycled. The app was designed to be cross platform and built in react-native with a TensorFlow model in the backend. Our strong suit in this project was definitely the deep learning model with 87% accuracy rate. Through this project we learned a lot about app development and building something from scratch!
