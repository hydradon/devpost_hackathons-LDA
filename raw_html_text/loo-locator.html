Inspiration

I've been victimized by dirty washrooms

What it does

Finds washrooms located throughout the city. Delivers ratings and information of washrooms. Allows users to rate washrooms.

How we built it

Used Python and a dataset by City or Toronto. Visualized the various locations of the washrooms Created pseudo ratings to align with locations

Challenges we ran into

Little knowledge of mapbox, using the data, cleaning the data

Accomplishments that we're proud of

We have user interaction to give a rating to a location We can find the location of the closest washroom to you

What we learned

The difficulties in analyzing data
