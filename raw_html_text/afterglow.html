Afterglow was inspired by the neon colors of Tron and the endless fun of a planetary defense game.

Afterglow is an endless survival arcade game where the player aims to try to beat the high score by tapping and destroying obstacles falling towards Earth.

We built Afterglow with Unity 3D coding in C# and Blender for the 3D modelling and texturing.

We are proud that we were able to create this game in the short 24 hours that we had. From nothing, we built an actual mobile app, which we are very proud of.

We learned countless things through this experience! We practiced coding and we all got a lot more familiar with Unity and Blender.

What's next for Afterglow? We plan on releasing a cleaned up version of the game with more features in the future on the Google Play Store!
