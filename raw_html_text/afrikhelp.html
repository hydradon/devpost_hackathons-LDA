What it does

Afrikhelp builds a community of africans who are ready to provide support in form of funds for campaigns.

How I built it

I built afrikhelp with codeigniter php framework, MySQL database, Paystack Api and facebook share api.

Challenges I ran into

I had issues implementing facebook share api but i figured it out later on.

Accomplishments that I'm proud of

I built a crowdfunding platform for Africans.
