Inspiration

We were inspired by the game Space Invaders but wanted to make a kid themed, multiplayer version to occupy children on long flights.

What it does

Uses websockets to communicate data between clients to transfer bullets from one screen to another, calculates hit detection and maintains scores.

How I built it

Built using Phaser, a browser-based JS game engine, along with a web server and websockets to communicate between clients.

Accomplishments that I'm proud of

Getting a working product in just 24 hours!

What I learned

How to communicate between multiple clients and a server using websockets and use Phaser.

What's next for Scoob Invaders

4-player mode, mobile support, UI improvements.
