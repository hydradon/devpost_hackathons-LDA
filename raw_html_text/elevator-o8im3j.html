What it does

Gives the user a challenging puzzle game to enjoy

How we built it

We spent most of the time coding on Gamemaker, but branched out for some game art and the music.

Challenges we ran into

Each mini-game had its own challenges, but one of the biggest challenges was properly timing naps so we could maximize productivity.

Accomplishments that we're proud of

We took something that would originally have taken months, and compressed it down into a weekend project. It only took 25 hours to completely finish this project.

What we learned

"Coding puzzle games is a puzzle in and of itself. How meta." -Samee Mohan
