Inspiration

We wanted to Experiment with some Hardware and chose a Pinball table as a fun Project for this.

What it does

It has two paddles to move the ball arround. They can be controlled by two Buttons.

How I built it

With some Arduino compatible boards from Infineon, two servo motors and a lot of Stickers.

Challenges I ran into

Understanding the Motor Control.

Accomplishments that I'm proud of

It works!!

What I learned

A lot about Motor Control and microcontrollers.

What's next for HackaFlipper

More Elements like Bumbers. A score and a sensor which registers if the ball got through the paddles.
