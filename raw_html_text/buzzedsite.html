Inspiration

Compare how tech students balance their life.

What it does

It is a questionnaire that plots the user's social vs academic balance using ga tech related questions.

How we built it

We programmed in javascript, html, and css to create an interactive website and utilized google firebase to store user data for comparability. We linked the two together in javascript. We deployed the domain using heroku.

Challenges we ran into

We had to learn how to store data using databases and learn how to deploy the website

Accomplishments that we're proud of

Overcoming our challenges and actually deploy our website

What we learned

How to store data and the importance of storing data

What's next for BuzzedSite

Add more questions, more memes, better design and add some textfields to use machine learning.
