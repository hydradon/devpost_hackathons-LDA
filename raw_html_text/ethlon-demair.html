Inspiration

As someone who speaks english as second language, we want to provide people with the ease of communication with their physician.

What it does

It is a service that helps patient to find someone who speaks their native language who can help translate important information from their physician.

How we built it

We built the front-end using react react-bootstrap css and developed the back-end using node.js express.js mongoDB and javascript

Challenges we ran into

Setting up the environment for both back-end server and client side, and figuring out how to collaborate on github as a team

Accomplishments that we're proud of

We finished most of what we planned for this hackathon

What we learned

Some of us learn new technology such as react and github

What's next for Ethlon

We would like to add user authentication and some API that we can use to help further improve Ethlon
