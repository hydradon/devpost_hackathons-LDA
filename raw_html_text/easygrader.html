Inspiration

Scantrons and their sheets are expensive, and multiple choice does not challenge students, so we created an automated system for grading free response questions

What it does

Analyzes a worksheet with numbers on it, and grades the answers against an answer key

How I built it

Using Xcode and swift to create the front end and photo capture, and a Firebase backend running the machine learning algorithm

Challenges I ran into

Making the ml model take in photos sent from an iphone

Accomplishments that I'm proud of

Creating a python wrapper for firebase to successfully run a machine learning algorithm

What I learned

How to use Keras for ml

What's next for EasyGrader

To expand it to helping homework, and gradebooks
