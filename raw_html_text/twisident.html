Inspiration

We wanted to bring the President's tweets to the world more conditioned. Our website is hosted here https://www.ocf.berkeley.edu/~joshuazlin/trump/Taris4.html

What it does

Using a database of our President's previous tweets, our program attempts to use machine learning/natural language processing algorithms to understand the underlying content, and create a new version that it more politically correct, and less dramatic.
