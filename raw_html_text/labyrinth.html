Inspiration

We had the rough concept of this game in the back of our heads for a while, but it developed over the hackathon into what it is today.

What it does

Labyrinth is a 2.5D game that puts you in a labyrinth full of hell hounds and you have to escape it.

How we built it

We used unity as our game engine.

Challenges we ran into

Firstly we had to learn how to use unity, Secondly we ran into a problem where it was next to impossible to implement a feature that we wanted because of the architecture of our game, and lastly we had a few clashes between the group members but in the end it was all sorted out.

Accomplishments that we're proud of

At the end of the hackathon we had a working fun game that we actually enjoyed playing and competing on.

What we learned

We learned how to use unity, build a game and work as a team.

What's next for labyrinth

We want to implement a multiplayer mode and more play styles such as fighting and collaborating.
