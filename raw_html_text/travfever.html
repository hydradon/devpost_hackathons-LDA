Inspiration

Walking to Imperial College; we realised we were always late. This app serves to remedy this.

What it does

  • Provides real-time weather checking functionality and map functionality within one app with user-friendly interface
  • Direct to different pages and back using buttons

How we built it

  • Dividing the task into individual parts; i.e. weather, transportation map & a user interface (app)
  • We used Android Studio
  • Learnt how to use Android Studio
  • Worked on our partial goals
  • Combined our results into an app

Challenges we ran into

  • Using TfL API to collect data turned out to be too challenging, so we resorted to using data from Google-Maps instead
  • Not knowing all of the programming languages used and having to learn them (Android Studio java)

Accomplishments that we're proud of

  • Creating a Maps Interface

What we learned

  • How to use Android Studio
  • How to work together to build a complex project
  • An appreciation for the work that goes into developing apps we take for granted

What's next for TravFever

  • Add a timetable and link it to maps: maps will warn the user automatically when it is time to leave for a certain destination (given in the Timetable functionality that will be added)
