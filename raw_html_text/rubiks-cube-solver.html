Inspiration

I like solving Rubik's Cubes and I wanted to see if I could make a program that could solve one.

What it does

It gives you the steps and turns you need to make on a Rubik's Cube in order to solve it.

How I built it

I made algorithms that would check what had to be done to solve the cube, preform the moves to keep track of what's happening to the cube, and after it's solved, it would output all the moves made in order to solve it.

Challenges I ran into

Some parts of solving the cube are fairly intuitive, so it was difficult to translate some steps into code.

Accomplishments that I'm proud of

I was able to make a program that successfully solves a Rubik's cube.

What's next for Rubiks Cube Solver

I hope to make it into a mobile app or a web app so a user can enter in values for their scrambled cube.
