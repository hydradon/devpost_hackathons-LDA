Inspiration

r/memeeconomy

What it does

It allows users to by and sell dank memes using an online currency called karma. Users can also look up memes and explore future investment opportunities by seeing their values and percent change in value over past 24 hours.

How we built it

We scraped data from google trends api and parsed it and then used javascript to make the functioniality and css and html for the webpage.

Challenges we ran into

Getting the webpage to look clean

Accomplishments that we're proud of

Scrolling bar at top with feed of memes.

What we learned

Friendship is forever, heroes never die, memes are a serious business. Not having teammates makes things difficult.
