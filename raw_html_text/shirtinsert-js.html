Inspiration

We want to shop for clothing without actually having to go places. Thus, we need an app that allows us to visualize what it's like to wear a shirt without having it on hand.

What it does

It uses machine learning to project the shirt onto your body.

How we built it

We used react-native to build the UI. The algorithms, based on opencv, run on a node backend, served with express.

Challenges we ran into

Installing react-native and getting the libraries to work together. Working with Unix based systems.

Accomplishments that we're proud of

Fixing a unix-based system without access to a GUI.

What we learned

Have build tools set up before the hackathon.

What's next for InsertShirt.js

Better machine learning algorithms and more AR integration.
