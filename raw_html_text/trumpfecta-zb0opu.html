Inspiration

Some lack of sleep and stupid domain later and the Trumpfecta was born. Having always come to hackathons with my friends to enjoy ourselves we normally try something new or come up with an idea 4 or 8 hours into the hackathon. The trumpfecta's inspiration really stemmed from the fact we had so many technical difficulties with a backend server running googles visual cloud api.

What it does

The Trumpfecta Project brings you three different ways to experience the Donald.

WhereIsTrumpGonnaBuildThatWall.com - is a website strictly dedicated to informing people around the country and the world where Trump plans to build a wall. This was done with Domain.com

TrumpIfy Extension - Replaces every image on a webpage with a lovely photo of Mr.Trump. Its light weight and easy to install and get running!

Mr.Trumps VR Build a Wall - In this game you play as Donald Trump working to separate a landmass by building a wall. There is even more added realness when using the oculus.

How we built it

Unreal Engine 4 was used to build the VR experience, this allowed us to have a high framerate yet still beautiful game. The Extension was built in JavaScript and HTML. While the domain was purchased for free through Domain.com

Challenges we ran into

Microsoft Azure server set up was our biggest obstacle. We were not able to get it to communicate properly with our extension. So the original plan of an extension that grabs images from a webpage and looks for landmarks using googles visual cloud then gets the location data of the user through geolocation. Then it would provide the user with flight plans and destinations where the landmark was located. ATM locations and also Banks will be located close by.

Accomplishments that we're proud of

We come to these hackathons not to win but to enjoy ourselves. My friends and I aimed high starting halfway through the night without any knowledge about backend and we were not able to complete it. But we were able to complete 3 separate things all pertaining to a theme.

What we learned

To research and come up with an idea in a timely manner or you end up with a bunch of trump themed projects.

What's next for TrumpFecta

Multiplayer Support for Build a Wall and added wave support. TrumpIfy-will receive a wider range of images and resolutions so that a trump can fit in any situation he is faced with. The website- perfect fixing nothing
