How I built it

Our Front End was in React using Reach Router to do paging. ## Challenges I ran into Because we put the server and front end in the same repo, we ran into issues where refreshing would not hit the right endpoint. Solution was to catch the tail end of the routes with a catchall- solving the problem ## Accomplishments that I'm proud of Completing the product in a balanced fashion using teamwork and most of all, coming up with a solid idea that is genuinely useful! ## What I learned Everyone should be on the same stack ## What's next for Ridebid More integrations with other apis
