Inspiration

I wanted to try creating an iOS extension and a custom keyboard was the best place to start.

What it does

Seamlessly connects travelers with locals.

How I built it

Using the iOS IDE, firebase mentors for firebase set up and the different talks and workshops

Challenges I ran into

I thought of the idea pretty late into the game so it felt more like a 20 hour hack as opposed to a 36 hour hackathon. Also working by myself meant I had to do everything so I couldn't work on the app and set up the server because I'm only one person, Haha.

Accomplishments that I'm proud of

It works.

What I learned

There are a lot of difficulties that go into working by yourself on a new task, but you learn a lot more about the whole process and the project development is more flexible with one schedule as opposed to two.

What's next for KeyFind

Making a few tweaks and adding Data viz to the map.
