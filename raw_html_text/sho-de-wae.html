Inspiration

Seeing as how the recent internet meme Ugandan Knuckles always uses his catchphrase "Do u no de wae?". I thought it'll be funny if he were to be the voice of a GPS.

What it does

Shows you directions to popular hackathons in a funny way

How we built it

We used text editors to create HTML/CSS/JS and interacted with Google's Map JavaScript API.

Challenges we ran into

Making the location data usable, changing the markers with custom images, path drawing not rendering

Accomplishments that we're proud of

Getting custom images to work, able to use previously unusable data, first time using API

What we learned

HTML, CSS, JavaScript, BootStrap, Google Maps API

What's next for Sho De Wae

Allow for custom destinations, adding funny dialogue
