• Go Green

Inspiration

Go Green Alexa skill is about giving an easy tip of the day, which can help protect the environment and conserve natural resources for current & future generations.

We are encouraging people to listen ONE tip of the day and try to act on it.

What it does

This skill tells you "Go Green" Tip of the day. You just have to invoke the skill by saying - Alexa, open go green and you will get the tip of the day.

How we built it

Using amazon alexa skill kit, aws lambda product, javascript and node.js

Challenges we ran into

We had very less time to learn and develop an alexa skill but we found amazon provided services platforms an out of the box cloud native experience which helped us in focusing only on development and meet the timeline.

Accomplishments that we're proud of

Learned and finished skill development and publishing in a week.

What we learned

  1. AWS products such as IAM, lambda, dynamodb etc.
  2. Alexa skills kit.
  3. How to publish skill privately and publicly.

What's next for Go Green

We are planning to add below features in future releases:

  1. Use dynamodb table to store go green tips and read dynamodb tables from lambda.
  2. Add more details to each tip so that if user is interested, they can learn more on demand.
  3. Add more intents to the conversations.
