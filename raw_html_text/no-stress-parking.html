Inspiration

Many of us have a driver's license, and we should all know how annoying it is to find a parking spot when you try to chase on something. So we decided to build this app to help people find a parking spot nearby.

What it does

We created an IOS app that displays available street parking near the current location of user, and also gives information regarding the parking. We also generated a Machine Learning (ML) model that can identify an empty parking spot from a given street view.

How we built it

We used Swift for iOS development, and OpenCV and Tensorflow for our ML model.

Challenges we ran into

  • Finding data to build the ML model.
  • Working with JSON in Swift

Accomplishments that we're proud of

Building an ML model that can identify empty parking spots from any given image.

What we learned

  • How to build a ML model
  • How to work with JSON in Swift

What's next for no stress parking

  • Request real-time street view for our parking recognition model
  • Integrate our ML model with the app
