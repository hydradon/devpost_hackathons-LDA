Inspiration

Interest in drones and boids

What it does

Supports a server of threaded drones to simulate drones flying together and visualize them using Unity.

How we built it

Networking through Python, data management through mysql, and visualization using Unity

Challenges we ran into

Connecting across, database concurrency, managing many threads, using Unity for the first time

Accomplishments that we're proud of

Visualization works! Networking is solid, none of our stress tests fail

What we learned

We definitely overestimated how quickly we could tackle the back-end

What's next for Drone visualization environment managed by a database

Implementation of pathing and swarming algoriths
