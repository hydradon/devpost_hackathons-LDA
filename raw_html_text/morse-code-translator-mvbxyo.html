Inspiration

I thought that a computer could translate morse code because it is difficult for a person to translate it.

What it does

The user blinks morse code and the computer translates it into the actual letters and words

How I built it

I used Python, and dlib and cv2 libraries to build it

Challenges I ran into

Making a timer so the program could recognize different letters between the dashes and dots

Accomplishments that I'm proud of

I was able to continually increase the speed of my blinks, and i learned some morse code.

What I learned

I learned some morse code and how to set timers in python

What's next for Morse Code Translator

Maybe i'll make a website and put the translator on it.
