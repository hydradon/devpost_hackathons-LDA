Inspiration

A sedentary lifestyle is bad for your health. Recent studies show that sitting too long may contribute to memory loss.

What it does

The Move It skill reminds you to get up and move every hour for 8 hours.

How I built it

I had to hack it. This skill is implemented as a prerecorded audio skill.

Challenges I ran into

It took me forever to come up with a decent idea. Also, I've never built an audio skill before.

Accomplishments that I'm proud of

None really. This skill is a last-minute hack.

What I learned

There's still a lot I don't know about building Alexa skills.

What's next for Move It

I want to make the skill customizable.
