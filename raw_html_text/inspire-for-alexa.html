Inspiration

With the new innovations in personal companions, the team was inspired to provide an experience that would truly improve the lives of those who Alexa interacts with.

What it does

Alexa uses a custom built skill called "Inspire" that allows her to respond to a number of queries to help inspire and relax you. Currently, Alexa can play soothing sounds, recite an inspirational quote from a database of 40, and can engage the user in calming breathing exercises.

How we built it

Using S3 and Lambda provided by Amazon Web Services (AWS), the team created a custom Skill written in JavaScript that allows the user to query Alexa with multiple different intents, prompting an appropriate response.

Challenges we ran into

At first, the syntax and hierarchy of organization, as well as the setup of AWS and Lambda was confusing and hard to pick up. Additionally, the differences in existing demos made it hard to zero in on a style of design for the code and flow of the conversation.

Accomplishments that we're proud of

Coming into this hack with no knowledge of how Alexa worked, and very little experience in JavaScript, the team is very proud of overcoming the challenges and developing a working skill that makes interacting with Alexa all that move valuable.

What we learned

We learned how to program new skills for Alexa, as well as how the system picks up on different intents, making the conversation flow more smoothly.

What's next for Inspire for Alexa

In the future, the team wants to extend the different tasks that Alexa can preform in the Inspire skill so that it can make a genuine impact on the lives of those around her.
