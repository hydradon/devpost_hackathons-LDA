Inspiration

Our main inspiration started when we were in Massachussetts one looking around for good food. Not knowing about the area, we decided to call up our sibliings to figure out a nice place to dine at. During this time, we had then intiallty came up with the idea of creating an app to share our past traveling experiences with our relatives.

What it does

This app allows for users to pin point on a map locations that they want to save for later access. The pins would be stored with a name for the location and a description to allow for the user to add comments.
