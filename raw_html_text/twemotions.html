Inspiration

Twitch provides an interactive experiance for its users. And, songs are the best way to enhance user experiance. Songs helps the user relate to the streamer. This made us come up with TwEmotions: a web app that plays background music based on the streamers real-time emotions

How we built it

We built it using Microsoft Azure's Emotions api. We took the screenshots of the screen, which included the streamer's face. Detecting the emotions on the streamer's face, we changed the background song.
