Inspiration

Stressed people who are facing challenges need to hear relevant stories about others that were once in their shoes, facing the same issue.

What it does

Experience helps you know other people's experience facing the same challenges you're facing, how they defeated them or how they succeeded in not making them an obstacle. It also makes it possible to share experiences about a city you've visited or a place you found remarkable.

How I built it

Experience is a web application built with Django and MongoDB.

Challenges I ran into

This was my first time using Django with a non-relational database and my first time using MongoDB, so I had to give some time to learning MongoDB and using it with Python and MongoDB compass.

What I learned

I learned a lot about non-relational databases and when they offer some convenient solutions opposed to relational databases.

What's next for Experience

I'll probably work on enhancing the user experience, and making the platform open for a wider variety of challenges.
