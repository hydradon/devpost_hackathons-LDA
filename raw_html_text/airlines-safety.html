Inspiration

Some of the flights on sky scanner includes airlines which have very poor safety ratings. We want the users to travel safely so have removed the airlines which have poor safety ratings

What it does

Uses the IATA API for getting airline safety ratings for SkyScanner flights

How we built it

We used IATA Api to get the safety ratings along with airlineratings.com. We pulled all the flight data between two airports from sky scanner and got their respective safety ratings. Then display that to the user

Challenges we ran into

Using the pusher api to update the data in real time.

Accomplishments that we're proud of

Make air travel safe again.

What we learned

Never fly with Air Canada Express, Bangladesh Biman.

What's next for Airlines safety

We are going to make the UI nicer and implement this as a chrome extension. Only show results above a certain safety rating
