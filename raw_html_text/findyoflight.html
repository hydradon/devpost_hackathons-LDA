Inspiration

Wanted to find cheap flights around the world, with no specific time given, just starting airport!

What it does

You can enter your nearest airport's 3 letter code and click the Find Yo Flight! button and it will search for flights from your airport all around the world, showing them to you in order of cheapest to most expensive, with images accompanying most of the tiles.

How I built it

I used javascript and the SkyScanner API

Challenges I ran into

I had a lot of challenges with getting the images for each location. At first, I tried to use Wikipedia, but it did not work well. I also had some challenges in pre-deleting the flights that are no longer available.

Accomplishments that I'm proud of

I am proud that I built this site in less than 2 days and that I wrote all the code myself (with some advice from the mentors)

What I learned

I learned more about how to work with various APIs, like Skyscanner's API. I also learned more about Cross-Origin-Resource-Sharing

What's next for FindYoFlight

I may continue to improve it and fix bugs, but it may just be as it is
