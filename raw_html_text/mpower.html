Inspiration

Seeing and hearing about how so many young refugee girls are unable to get a quality education.

What it does

Diana is a web, Messenger, Skype, and SMS AI-powered chat bot that uses Quizlet to teach any topic of interest on the go.

How we built it

Primarily using Node.js + API.AI on Heroku (see tags for other parts of tech stack).

Challenges we ran into

Trying to build for so many platforms while making full fleshed-out features on each one.

Accomplishments that we're proud of

Implementing so many new technologies we didn't know beforehand.

What we learned

How to use so many APIs and platforms.

What's next for Diana

More learning features and expansion to more platforms!
