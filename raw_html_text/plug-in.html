Inspiration

Have you ever wanted to do something, but didn't have a friend to do it with? We have -- which is why we wanted to build PlugIn, to help users connect with others in their nearby community and make new friends.

What it does

PlugIn provides an interface for users to create open events to others in the community. Users can search by tags to find events in that category and message the users within those tag communities.

How we built it

We utilized XCode with Google Maps API and Firebase to create an iOS application and manage data like events and user profiles.

Challenges we ran into

XCode seems particularly difficult to debug because its messages are not very detailed. Issues with the map really slowed down our progress.

Accomplishments that we're proud of

We have capitalized on a useful idea that will help people explore their community and interact with others in a new way.

What we learned

API's are not as easy as you think they'll be. We also learned that it is important to communicate and work together to utilize each of our strengths.

What's next for PlugIn

We hope to keep making PlugIn better and better by smoothing out all of its processes. We want users to feel like making an event and interacting with each other is fluid and natural. We also hope to develop an android version to allow even more people to use PlugIn!
