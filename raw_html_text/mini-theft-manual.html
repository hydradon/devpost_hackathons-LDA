Mini Theft Manual

Submission for Brickhack V

Project

We wanted to make a multiplayer bumper cars game with some sort of unconventional input. We used a plate with two color splotches on it as a steering wheel, using a computer vision algorithm to detect its orientation and angle to send as input to the server. The entire server architecture is written from scratch with node.js, using socket.io to handle client-server communication. The hardest part about this was figuring out the computer vision algorithm. We tried a variety of different techniques from image segmentation to contour recognition in order to try and detect the color splotches, but what ended up working best was using template matching to identify the color splotches.
