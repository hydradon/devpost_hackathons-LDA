Summary:

  • We tasked ourselves at creating a web based app in order to help learner drivers complete their logbook. This is currently a tedious and manual process with high potential for cheating.
  • A mobile app has the potential to save numerous hours of time for learner drivers. This means that drivers can spend less time stressing about pleasing VicRoads and more time learning how to drive.

Aims:

  • Use the GPS location of a user's phone to determine nearby traffic and weather conditions.
  • With these conditions, automatically fill out each logbook entry.
  • Export and import trip information from an online database.
  • Allow drivers to select from a list of supervisors and car registrations as well as set default profiles for trips.

Result:

  • We were a group of electrical engineers with a lot of hardware experience but limited coding experience. However, we felt it would be a good to stretch ourselves outside our comfort zone and work on a software project.
  • We built the web app from the ground up using Javascript, CSS and HTML, ensuring that it was as simple as possible for users to use. We also used Firebase for the results database and web hosting.
