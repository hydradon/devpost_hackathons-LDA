Inspiration

Have fun with music, more fun when in a party with your friends!

What it does

Linked with your host and guest, join the same party, shaking your phone with the pattern set by the host to control the house's music.

How I built it

We use Parse as the backend server to handle user login system as well as the interaction between different devices. As long as devices are connected with internet, communication is super smooth between the host and guest iOS devices.

Challenges I ran into

None of our team members have been developed in iOS with Swift language before, also the Spotify API is written in Objective-C and lack of documentations

Accomplishments that I'm proud of

We finally made it works!

What I learned

Programming skill is important, while the team cooperation is much more essential

What's next for SpartifyApp

Add more interactive actions to play with our iOS device in the party room!
