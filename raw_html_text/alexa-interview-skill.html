Inspiration

The inspiration for this project came from this groups' anxiety about upcoming interviews and the desire to get more practice answering interview questions.

What it does

Using the Alexa Skills Kit, we are developing an interactive "interviewer" so the user can hold a mock interview in the comfort of their own home.

How we built it

Using the Alexa Skills Kit, we created the interaction model based off of how we think an interview would go. With the Lambda function, we created a backend that would store the user's responses.

Challenges we ran into

Since the majority of our team has had no experience programming before, a lot of time was spent learning basic programming concepts (dictionaries, lists, if/else statements) so all members could contribute to the writing of the lambda function.

Accomplishments that we're proud of

We are proud that the first project we have created is something that can be useful to people.

What we learned

Working as a team, we can accomplish things that we couldn't accomplish alone.

What's next for Alexa-Interview-Skill

In the future as we expand our programming knowledge we hope to build more functionality into our skill. In particular, we would like to make our skill more dialogue based instead of a one-sided conversation.
