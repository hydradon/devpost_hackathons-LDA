Inspiration

Our inspiration was Twilio, and our interest in football.

What it does

This SMS service gives information to the user about football, which include team standings in leagues, recent scores, and the upcoming fixtures.

How I built it

It was built using python and Twilio, with the use of API-FOOTBALL API.

Challenges I ran into

Providing a robust way to access the different APIs and provide the user with the necessary information.

What's next for FootyNotify

Add extra features such as specific details about their chosen team.
