WasteFood

-Compostor uses computer vision and machine learning to identify your pictures.

-WasteFood is fully powered by Javascript libraries and frameworks.

-To deploy and host my application I used Zeit/Now as its cloud platform.

-The client side was done using Nextjs which is a Reactjs framework.

-Compostor was built using ml5 which is a tensorflow.js wrapper.

-I used an S3 bucket to host the static files such as .css, .png and .html files .

-As for CSS libraries, I used Tailwindcss and Bootstrap.

-I used a .tech domain for the application.
