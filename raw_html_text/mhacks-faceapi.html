Facial recognition with SMS notifications for successive failed unlock attempts.

Uses Microsoft Azure's Face API for adding authorized faces and identifying/verifying others. If too many unlock attempts are made with an unrecognized face, a text message is sent to a phone number notifying the user. This is done using the Twilio API.
