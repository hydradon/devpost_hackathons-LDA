Inspiration

I've always a wanted a simple, easy to use desktop app for monitoring prices of items that I plan to purchase.

What it does

Want helps users save money by monitoring Walmart.com online pricing for saved items and sending notifications when prices drop.

How I built it

With Visual Studio 2015 and lots of C# and XAML.

Challenges I ran into

I had lots of trouble with some UI stuff as well as the Windows.Storage APIs, but I was able to get all of my issues worked out.

Accomplishments that I'm proud of

Integration with the Walmart API and the relatively good design for the time I had to work on it.

What I learned

There isn't much documentation out there for Windows 10 apps (yet).

What's next for Want

I'd like to add a bit more polish and make this a production app very soon. I'm thinking about integrating with other services as well.
