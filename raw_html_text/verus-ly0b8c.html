Inspiration

Negatively emotional bias change the way we consume media, we want to bring awareness to that.

What it does

Scans articles for negativity, returns negative phrases, keywords and overall rating. Showing biases in our daily media.

How I built it

Using iOS and IBM-Watson Alchemy Language API.

Challenges I ran into

Watson API isn't as good as you would think.

Accomplishments that I'm proud of

Group members had the opportunity to learn new technology.

What I learned

Watson API isn't as good as you would think.

What's next for Verus

A different API for more accurate results.
