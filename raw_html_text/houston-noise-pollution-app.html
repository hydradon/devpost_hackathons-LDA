Inspiration

We wanted a way to map the decibel levels of noise pollution around the city of Houston.

What it does

Individuals can contribute to the database by visiting the website and clicking contribute. The microphone records the decibel reading and submits it to the database.

How I built it

We used react for the client side page and node to host the server for the database.

Challenges I ran into

The heatmap framework was difficult to implement as was getting the microphone to pick up the decibel reading.

What We learned

How to implement mapbox with react.

What's next for Houston Noise Pollution App

To implement it across browsers and devices and to have people drive around town and use the app!
