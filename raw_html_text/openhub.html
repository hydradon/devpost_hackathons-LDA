Inspiration

Have you ever wanted to contribute to contribute to open source projects but never knew how to get started? OpenHub is a the app you’ve been dreaming for !

Join a community of like minded software developers and start contributing to some of the coolest open source projects out there.

What it does

OpenHub matches open source projects with people looking for a project to contribute to. The user can define languages and topics of interest to get project recommendations.

How we built it

We build a web app in react with typescript and a backend in Scala.
