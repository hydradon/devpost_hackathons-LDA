Inspiration

Quality Education, No Poverty, Zero Hunger

What it does

ChairtySharity enables you to give charity without paying a penny, by simply sharing your presence at retailer's location, on social networks.

How I built it

CharitySharity was built for Android devices using Facebook open graph API to share retailer's location on Facebook. Google Firebase was used to store data, sending notifications and Crash Analytics.

Challenges I ran into

Facebook open graph API version 2.1 was not allowing me to share more than three characteristics at one in a post (e.g. sharing status, tagged friends, location and Picture at once). I discussed this matter in Facebook development discussion page a lot. Later on, in version 2.6 and above this problem was solved.

Accomplishments that I'm proud of

As I was working alone on the development of this project, so working on this project gave me confidence and mobile development experience using different SDKs other than Android.
