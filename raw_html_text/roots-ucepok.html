Inspiration

Roots assists with new music discovery by looking _ backwards _, rather than forwards, based musical influences.

What it does

Pick a current artists, and discover musical influences for that artists. For example, clicking Beyonce could lead to Whitney or Mariah, whose songs would play.

How we built it

With love.

Challenges we ran into

Trying to find the data around influencers. The best dataset (Rovi) has been offline since 2013, so instead we had to implement custom scraping.

Accomplishments that I'm proud of

We are proud of finishing our idea, being able to learn so much about the music industry, and meeting some great people!

What I learned

-About the current state of the music industry and some of those innovations -Spotify APIs -Web scraping using Cheerio -Angular animations -Masonry grid layouts

What's next for Roots

Clean up the front-end a little bit, but I think we will actually use it for fun even after this hackathon.
