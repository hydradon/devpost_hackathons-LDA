This app provides you with a quick state-by-state snapshot of the cereal grain industry in the United States since 1960. It highlights data sets from the major cereal grain crops such as wheat, oats and barley and visualizes it on a map.

It plots data taken from the commodity "Quick Stats" from the National Agricultural Statistics Service (NASS). The UI is fairly straight forward. Choose from a drop down menu on the left to select a commodity. Choose from one of three data sets from the menu at the top. A slider bar at the bottom allows to scroll through the years.

Grain Gabble was built with the d3.js, a javascript library which specializes in data visualization, using with HTML, SVG, and CSS technologies. A lot of the effort involved taking raw data, modifying it, and whipping it into shape before it could be used by D3, or any other data visualization library
