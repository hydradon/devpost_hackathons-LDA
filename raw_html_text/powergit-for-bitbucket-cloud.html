What it does

PowerGit provides the most wanted features for BitBucket including commits, contributors and pull requests statistics, code search and Git LFS support.

How I built it

PowerGit is build with:

  • NodeJS
  • Elasticsearch
  • MongoDB
  • redis

What's next for PowerGit for BitBucket Cloud

  • More Graphs
  • API access

To install, please go to https://bitbucket.org/account/addon-directory , select your account/team and find PowerGit in that page
