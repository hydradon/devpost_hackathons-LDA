Inspiration

We have been inspired by Carlos Matos' presentation about BitConnect.

We highly recommend anyone using the extension to watch his presentation: Carlos Matos Presentation!

We do not affiliate with BitConnect in any way. This extension is open source and was made with an intent to learn and enjoy hacking.

What it does

Carlos Matosifier is a chrome extension that replaces keywords to quotes that Carlos mentioned during his presentation. With added features, such as changing the images of any webpage to images of, non other than, Carlos Matos.

How we built it

We used JavaScript and HTML to come up with this unique idea from scratch. We search specific string across the whole page and change them to Carlos Matos sayings.

Challenges we ran into

Replacing texts was very challenging, at one point we ended up not only replacing simple texts but also styling properties which would break the whole page.

Accomplishments that we're proud of

This was our first chrome extension ever!

What we learned

We've learned how chrome extensions work and how to use JavaScript and HTML to replace words and images in webpages.

What's next for Carlos Matosifier

Further improvements, better background, more images, more sounds, more strings replaced.
