Description

This software is meant to run in the background while a developer writes and tests code. When an error message is generated, we automatically take the relevant parts of the error message, search Stack Overflow, scrape the best answer from the first question returned, and print the result in the terminal window. This allows the developer to seamlessly troubleshoot problems without needing to ever open a web browser and copy paste the error.
