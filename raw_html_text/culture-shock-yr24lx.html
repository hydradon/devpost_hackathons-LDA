Inspiration

Our team wanted to create a way for people who are unaware about certain beliefs,customs, or practices from another country to be able to understand and learn about them properly.

What it does

It teaches anyone entering a country about the country's laws, phrases, and do's and dont's.

How I built it

This android app was developed via android studios using java for the back end of the app. Photoshop for the app logo, and xml for the front end of the app.

Challenges I ran into

Certain challenges arose throughout the development of this app. Not being familiar enough with android studio and Photoshop consumed large amounts of time when we needed to troubleshoot issues and bugs that we did not know how to solve. Luckily there were mentors and others that were able to guide and help us to the right direction.

Accomplishments that I'm proud of

Overall we gained knowledge about the tools we used to create this app. Specific accomplishments we as a team are proud of are being able to implement a real time database. Also it felt great to be able to make a functional application in android studio.

What I learned

As a team we learned that developing things are not as easy and quick as it seems to be since there are many different pieces that must be put together in order to create a working app. Also that it is good to work together in order to solve the issues you encounter.

What's next for Culture Shock

We will try and implement the app to a larger audience and scale so we could inform others about more countries.
