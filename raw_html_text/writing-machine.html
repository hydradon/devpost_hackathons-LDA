Inspiration

What it does

We managed to make an automated writer that would copy the image of handwriting written on an adafruit TFT display, several times.

How we built it

Using Servo Motors and Arduino Uno

Challenges we ran into

We had a hard time looking for a compatible material to bulid the arms of the writer. There is some difficulty in mapping the handwriting points to angular and rotational data that are utilized by the motors.

Accomplishments that we're proud of

Successfully obtaining handwriting data using tft display module

What we learned

Having a clear objective about the project the team would like to do is a very helpful strategy to effectively work as team during the event

What's next for Robotic Writer

Add computer vision to be able write text from images and videos
