Inspiration

I wanted to create a voice therapist that gives you advice. I wanted to have the user input custom problems for a real person to write a response that alexa reads- but I ran out of time so this version gives you general advice for now.

What it does

It gives you little tidbits of advice.

How I built it

I used the voiceflow platform to organize and design the skill.

Challenges I ran into

Getting the ISP portion of the skill working was a challenge as the voiceflow platform does not show the full testing.

Accomplishments that I'm proud of

I am proud of the voice skills that I've built from this challenge. Going from no experience with voice apps to building a few unique alexa skills was interesting, challenging, and a lot of fun.

What I learned

Natural language processing has come a long way. I think we can definitely build out a more robust therapy option through alexa.

What's next for Mini Therapist

I would like for the user to be able to submit custom issues that a real person can reply to.
