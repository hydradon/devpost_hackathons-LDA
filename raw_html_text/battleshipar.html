BattleshipAR

BattleshipAR is an AR battleship game created for Swamphacks 2020. At first, we wanted to make an AR version of the game Battleship. As we were looking for 3D models for ships, we discovered 3D models of Star Wars ships and that changed our whole idea. We decided to make a 1v1 battle game between Star Wars ships. We learned a lot about Vuforia which made building the AR game very fun. Some of the challenges that we faced were getting multiple ships showing at the same time and getting the attack buttons working. Moreover, a few hours before this was due, as we were finishing some final touches, Vuforia went down and we were unable to fully complete our idea. This will not be the end of our project as we are planning on continuing to work on it so we can make it better and better.
