Inspiration

Out-of-office emails, but for Messenger.

What it does

Waits for a new message to be received on Messenger and responds to it based on your Google Calendar. If you're busy

How I built it

We used Python to connect to Facebook Messenger and the Google Calendar API. If a user is currently in an event, the bot takes over and responds for them!

Challenges I ran into

Installing some python packages is _ somewhat _ challenging when there are conflicts between version dependencies.

Accomplishments that we're proud of

What's next for AFKChatBot
