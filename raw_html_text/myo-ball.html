• Myo Ball in Action

Inspiration

We wanted to make Wii games for the browser. The Myo Armband is the perfect wii-mote replacement. We also love the feud between Lebron James and Steph Curry so we made a basketball simulator in the browser for these two titans to duke it out.

What it does

Two players are required-one representing Lebron James and the other representing Steph Curry. Using the Myo Armband, each player attempts to shoot basketballs into the basket. Player turns alternate. The first player to score 3 points wins the game!

How we built it

We built a physics and graphics engine in Javascript using WebGL. These engines take data from the Myo Armband and simulate the arm and ball movement. We used node.js as the server and networked via socket.io. Finally we hosted the server on Microsoft Azure so everyone can join the fun!

Challenges we ran into

Trying to coordinate networking, game logic, and graphics proved to be a difficult challenge.

Accomplishments that we're proud of

Solving our challenges. Game aesthetics.

What we learned

Networking, node.js, Microsoft Azure, ray marching

What's next for Myo Ball

Myo Boxing, Myo Hockey, Myo Bowling, Myo Baseball
