Inspiration

Research suggests that early exposure to sexually graphic material is likely to have a negative influence and a potentially harmful impact on children. It can be quite stressful and they will sometimes find ways to act it out to try to make sense of it. There is only software that block the whole site; however, our plugin is able to filter out the image instead of blocking the whole site.

What it does

Another demo with explanation: https://www.youtube.com/watch?v=U73ZDtbI3OQ

Parents can filter out adult content using our chrome extension. It can prevent children from viewing explicit images. Moreover, it also helps porn addicts who are struggling with pornography since it helps them to prevent triggers ( https://www.reddit.com/r/NoFap/comments/2utqnr/heres_how_to_avoid_triggers/ )

We also use Azure for machine learning as a database so users can choose what they want to see or prevent. Moreover, it helps improving our software performance.

How we built it

We use Clarifai API to recognize adult images. Azure was used for machine learning.

Challenges we ran into

We have challenges setting up the API and have troubles connecting to the Microsoft SQL Server from Azure.
