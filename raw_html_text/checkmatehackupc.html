CheckMateHackUPC

· This is an android app designed as a system for the elderly (check) and their trusted friends or family (mate). It sends notifications to the elderly person asking them to check in (so the name check), and if they don’t do it after a certain amount of time, it will send an email to everyone on their friends list.

· Go through the application

· If we had more time, we would send notifications to the mates when their check is not okay, instead of an email.

· If we had more time, we might also make it possible for a user to be both a mate and a check so the elderly could be a mate for their elderly friends.
