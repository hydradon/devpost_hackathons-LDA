Inspiration

Lack of resources for people that primarily rely on ASL (eventually more sign languages).

What it does

Converts spoken Languages to sign languages, using image files for displaying signs

How we built it

Using Google Cloud for interpreting speech and converting it to text. Using NoSQL for converting text to signs. It was our hope to expand the limited sign resources using machine learning.

Challenges we ran into

We simply did not have the JavaScript experience to make much headway in one day.

Accomplishments that we're proud of

What we learned

Selecting a language and framework that works for our goals.

What's next for Speech to American Sign Language

Preparing for future hackathons with this experience, possibly StASL will make another appearance then.
