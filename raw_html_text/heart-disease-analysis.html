Inspiration

We were looking for something that could have societal impacts and after searching through a couple of dataset databases, this one stuck out.

What it does

Analyzes four datasets downloaded from here containing patient information and tries to find relationships with heart diseases.

How we built it

We've built two versions, one with a neural network, and one with KNN clustering.

Challenges we ran into

The dataset had a lot of missing values so we had to come up with ways to make sure our data could still be analyzed.

Accomplishments that we're proud of

Being able to have a functioning system that has a pretty good confidence/accuracy was the best moment.

What we learned

A lot about how to implement machine learning and how to deal with messy data..
