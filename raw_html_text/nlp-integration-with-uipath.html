Inspiration

We wanted to take our self-service sales robot to the next level adding AI, Natural language processing specifically.

What it does

It recognizes natural language and returns the intent behind it.

How I built it

I created an NLP instance using wit.ai an open-source solution that provides a REST API to access it. nad UiPath studio to create the front end.

Challenges I ran into

Training the NLP defines success, understanding how that particular kind of AI works is the key to this.

Accomplishments that I'm proud of

Integration NLP to our self-service solution giving the ability to users to ask a question about their sales orders using natural language.

What I learned

How NLP engines work, how to integrate with UiPath, what you send what you get back and how to use it afterward.

What's next for NLP Integration with UiPath

After this small snippet, we can basically add NLP capabilities no any robot we create, we are currently thinking in a chatbot for specific purposes.
