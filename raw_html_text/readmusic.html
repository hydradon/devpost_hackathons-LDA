Inspiration

What it does

user input a book URL and the app will show a summary of the book, as well as a playlist of 10 emotionally relevant songs.

How we built it

Challenges we ran into

Using API and its limit with the high volume calls Deploying Python Web App with Flask for the first time

Accomplishments that we're proud of

Being able to pull the summary of most of the books and the book's basic information Being able to get music data and create a playlist to user account (with user permission and authentication) Being able to learn to create Python Web App with Flask Using an API for Emotion Analysis with results of 5 different emotions

What we learned

How to quickly find and utilize a REST API Python Web App Deployment with Flask (Very tricky and have a steep learning curve) Other than Sentiment Analysis, we can have many other ways to process language.

What's next for ReadMusic

Build a bigger, better database Improve criteria to match books and their suitable songs/genres Option for user to enter different output as well as scan barcode Option for user to navigate playlist to Spotify web app Design and User Interface Deploy App on Cloud
