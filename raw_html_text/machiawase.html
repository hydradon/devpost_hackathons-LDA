It is an application used for meeting with friends. I think that there is one thing that friends do not know where they are. With this application, you can find it immediately! Welcome, Pull request!

What it does

友達がいる方向をアプリ上で表示してくれます！ It displays the direction in which the friend is on the application!

How I built it

リアルタイムに位置情報を取得し、それを画面上に表示するところです。 Where location information is acquired in real time and it is displayed on the screen
