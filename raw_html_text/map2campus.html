Inspiration

Many students, never having visited the campus, did not when where to enter the campus to arrive in the Dining Hall. Searching for Queens College led to the USPS address, not helpful

What it does

This web app uses longitude and latitude of the exact entrance around campus

How I built it

We built a HTML/CSS pages for the section of the campus then to Campus pages for selecting the entrance

Challenges I ran into

There was trouble with the Google Maps API First time reading heavy documentation

Accomplishments that I'm proud of

Using the Google Maps API Deploying the app on the web

What I learned

The Google Maps API UI/UX

What's next for Map2Campus

Adding more campuses, a modular database, User submission with GPS coordinates
