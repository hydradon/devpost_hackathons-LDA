Inspiration

We wanted to create a quick and efficient way to help people come defeat the difficulty of coming of with meal ideas.

What it does

Scans available ingredients in the fridge to suggest healthy meal while informing users of calories and other nutritional information.

How I built it

Project was built in a android studio environment (using Java) and relied on the TensorFlow API for object detection.

Challenges I ran into

Challenges included integrating features with android studio, finding the relevant models, and parsing data as required.

Accomplishments that I'm proud of

Managed to learn enough Android Studio Development to build a fully fledged and deployed (locally) application.

What I learned

Android Development Environment, Implementation of Environment (TensorFlow), and network, communication and teamwork skills.

What's next for Quick Kook Up

Hopefully additional features which will improve user interface, support for healthy and exotic recipes.
