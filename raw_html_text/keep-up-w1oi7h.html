Inspiration

I wanted to build a game that was simple, addicting but super challenging.

What it does

Keep Up is a high score based game in which the player tries to juggle two balls simultaneously for as long as possible.

How I built it

Unity Sdk

Challenges I ran into

It was initially challenging to figure out how to secure the backend so earns are exploited.

Accomplishments that I'm proud of

Going from concept to launch with a new SDK within 3 weeks.

What I learned

Kin is a substitute for micro transactions with added rewards from the KRE.

What's next for Keep Up

Build a multiplayer mode where players go head to head.
