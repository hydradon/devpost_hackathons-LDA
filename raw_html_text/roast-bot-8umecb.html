Roast_bot

A Discord bot to harass and roast individuals who say something the majority of people disagree

Commands List

Commands Description Usage
'!help' Displays a page similar to this list '!help'
'!about' Displays information about us, links, and about Roast_bot '!about"
'!startvote' Starts a vote against a user for 3min '!startvote @mentioned_user'
'!addroast' Adds a roast to the list of roasts '!addroast Yo mama joke '

Getting Started

Prerequisites

  • Python 3.6.8 or LOWER (dependency doesn't support 3.7)
  • discord.py 1.0.0.a (install manually via their github page)
  • Discord server (with Administrator privilege for user to install bot)
  • Discord Dev Portal

Configuration

Assign 'token' value for server

Run 'bot.py' (contained in the main directory).

Use '!help' for more info within your Discord server

Happy roasting!!!
