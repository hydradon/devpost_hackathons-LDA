Inspiration

We created QWER Films to demonstrate the diversity of the LGBTQIA+ film community. We believe that queer individuals deserve to have their stories told - and beyond the archetypal gay and lesbian characters in mainstream Hollywood media. We built QWER Films to help the queer community find movies that speak to them. We did this in hopes of influencing production companies to make more of these films.

What it does

Upon launching the website, users can search for reviews and the ability to filter movies and TV films by popularity, date of release, and representation.

How we built it

We used html/css, and javascript within the react framework. To store our data, we used firebase, domain.com, and google cloud. We gathered insight from other individuals from the LGBTQIA+ community and used secondary data gathered online.

Challenges we ran into

We initially had trouble finding a dataset for a list of LGBTQIA+ movies. This was also the first time any of our members worked with React in depth. Deploying our website was also a challenge we were able to overcome with the help of mentors.

Accomplishments that I'm proud of

We designed a website in React! We were able to successfully integrate a dataset that users are able to interact with and filter through.

What we learned

We learned how to split up the work and work in a fast-paced environment. We also learned a new framework - which is a good object oriented approach to building web apps.

What's next for QWER Films

We want to incorporate meaningful filters through extensive user research and user testing. This is a meaningful first step to gathering more use cases and learning more about the user in general.
