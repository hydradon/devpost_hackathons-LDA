Inspiration

Coming into the weekend, we decided that we would take a look at the various challenges/technologies and to the best of our abilities, create a hack that utilized multiple technologies.

What it does

It allows the user to find their exact location via their smartphone and the vehicle's location and determines if they are in the same location without actually providing coordinates for the phone in order to protect user privacy.

How we built it

We built this mobile application with Java, Python, Android-Studio, Flask, and Smartcar's API.

Challenges we ran into

Initially, we discussed Microsoft's azure being heavily integrated into the application, but as we made progress we realized that it did not seem as necessary as it was initially perceived to be and so it was eventually cut from the hack. This was a challenge because we had to change what the main focus of the app was and we landed on the location of the smartphone relative to the location of the vehicle would be the direction we would go in.
