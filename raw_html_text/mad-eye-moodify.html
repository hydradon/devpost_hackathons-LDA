Inspiration

Most of us have way too many songs in our Spotify library and playlist. It takes time to go through and create a playlist based on a mood. So we made it faster.

What it does

It takes tracks from the user's Spotify library and saved playlists and then creates a playlist based on a user selected mood.

How we built it

We used the spotipy python package to get data from Spotify and filter the tracks. We then built a website to host the program.

Challenges we ran into

We ran into challenges with Oauth2 and getting authorization set up.

Accomplishments that we're proud of

We're proud that it works.

What we learned

We learned how to use the spotipy library, as well as Oauth2.

What's next for Mad Eye Moodify

Next, we'll refine the moods and add more options.
