Inspiration

Sometimes I want to watch movies or listen to music, but I draw a blank on what to watch or listen to. I can get suggestions from Netflix, Amazon, Spotify, and more, but they get old after a while. The app is named after the first alphabetical element.

What it does

Actinium links you to all of your Facebook friends who also have Actinium and lets you send them a single request for a movie suggestion, a tv show suggestion, or a music suggestion.

How I built it

We used Android as the platform. We used the Facebook SDK for logging in and handling users. We used Parse as the backend for the app.

Challenges I ran into

Figuring out how to use the Facebook SDK to login users and get access to their friend lists. The learning curve of using Parse as the backend.

Accomplishments that I'm proud of

Developing an app that, although it needs improvements, is functional after 24hrs.

What I learned

How to use Facebook SDK to login to an Android app and manage users as well as their friend lists.

What's next for Actinium

Finish adding Parse backend to allow users to see the past recommendations they have received with ability to delete from the list. Use rotten tomatoes API to help friends suggest movies and provide relevant information on recommendations. Clean up UI to make it look nicer. Continue adding new features. Make backward compatible to pre-lollipop devices. Fix bugs.
