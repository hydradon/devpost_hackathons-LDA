FinVisor is a personalized financial adviser android application that keeps a track of your gross income and net spending. Managing personal finances is a tough task for many people. Credit cards and debt make it easy for people to overspend, so our app provides users with financial advice based on their monthly expenses and balance so that they stay informed at all times and make good decisions. We developed the back end of the application using Android Studio and then exported it into MIT App Inventor for the UI design. We also wanted to create a website for the project but we weren't able to pair our code with the front end of the website. Also, we made an AI Chatbot using IBM Watson but we weren't able to connect it with our website and application within the time frame we had to develop both part of the project.
