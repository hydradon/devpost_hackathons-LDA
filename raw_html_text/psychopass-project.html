Inspiration

College search process is extremely complex. It requires many factors to be considered. One of of the most important ones is how a person matches college community.

What it does

We developed a system that recommends a college based on personal phycological characteristics.

How I built it

We built it using iOS SDK and IBM Bluemix personality insight

Challenges I ran into

Bluemix was not working for a while ;)

Accomplishments that I'm proud of

An elegant and simple to use interface

What I learned

It was our first iPad app as well as the first time using Bluemix SDK. It was kinda difficult to access API using NSUrl calls, so we spent a lot of time doing research about it.

What's next for Psychopass Project
