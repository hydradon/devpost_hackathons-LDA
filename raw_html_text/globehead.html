Inspiration

We were inspired by Booking.com's Travel themed game challenge.

What it does

It's an endless runner game, with coins to collect and enemies to avoid.

How we built it

With Unity 2019 Version & SumoPaint (an online bitmap image creator)

Challenges we ran into

Unity's scenes don't play nice with git.

Accomplishments that we're proud of

It works! We were able to add options to make it more convenient for ourselves as developers as well as accessibility settings so our game can be more inclusive.
