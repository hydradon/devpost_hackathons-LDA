Inspiration

We enjoy getting our daily exercise by working out at the gym. However, we also enjoy online gaming. This Role Playing Game has a physical exercise component to motivate us to work out more.

What it does

Demon Abs uses a tilt switch from Arduino to detect sit-ups. It's a digital sensor that can detect changes in orientation due to its built-in shaker. There is also the main game which is coded in C++. When playing the main game, the outcomes of the game are narrated depending on the player's choices in-game.

How we built it

We built this with C++ and an Arduino Kit

Challenges we ran into

It was hard to incorporate a software solution for the sit-up counter and the code for the main game was rather tedious.

Accomplishments that we're proud of

We are proud that our project provides a unique game-play experience for the prospective gamer. It also encourages us all to work-out to see results, both in-game and in reality.

What we learned

We learned how to create complex if-then statements to tell a story, and how to use Arduino wires to power the tilt-switch.

What's next for Demon Abs Online

This is just our prototype. We will improve upon our sit-up counter and game at the next hackathon.
