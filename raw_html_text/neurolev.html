Inspiration

An escape from daily stress or possibly introducing VR into mental health industry.

What it does

Monitors EEG using openBCI and runs visualizations based on your current state of mind.

How I built it

We used Oculus DK2, OpenBCI EEG sensors on the hardware side and Neuromore and OSC on the software side.

Challenges I ran into

Combining them all together and learning how to setup and use the hardware.

Accomplishments that I'm proud of

Trying hardware hacking for the first time.

What I learned

Neuromore, OSC and setting up hardware.

What's next for Neurolev

Add more visualizations, adding more data sensors.
