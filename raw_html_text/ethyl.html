• The device

Provide an easy to use wearable device that is backed by a community of fellow recovering alcoholics and supportive members.

What it does

The wearable device takes ambient alcohol readings from the air after the user is prompted to perform a breath test. Future models would include a skin sensor that measured alcohol content off the skin to make testing automatic.

The web platform tracks the users progress and provides a support network for them in order to help them through their journey to sobriety.

How I built it

The wearable was built using an ESP8266 Wifi enabled microcontroller, an alcohol sensor, and a number of other electronics supplies.

The web platform was created using Bootstrap, HTML, CSS, and Javascript.

Challenges I ran into

Getting the electronics of the wearable to work in such a short time period was very difficult.

Accomplishments that I'm proud of

The live demo with the wearable device sending data to our web pages to be displayed.
