Inspiration

There are many people in the world with unlimited phone minutes that could use this to browse Wikipedia and other educational websites.

What it does

You text a number a URL and it calls you back with the contents of it. Headings and then the paragraphs under them.

How we built it

Used Twilio to do voice/text stuff. Used wikipedia api and general web scraping for rest of websites.

Challenges we ran into

Strange encoding bugs.

Accomplishments that we're proud of

We got it to work!

What we learned

Work on 3 projects and then submit the last one.

What's next for Galaxy Explorer
