Inspiration

It gets hectic when it comes to getting train tickets to travel during New Year in China/Taiwan. My friends are from Taiwan and they have personally gone through this war of getting the tickets to travel home. We came up with a simple solution to have slight advantage when getting the tickets.

What it does

The script will provide users with intuitive interface when users will input few required information to buy a ticket. The app will then proceed and direct them to payment page. This will give them a slight advantage against other ticket buyers

How we built it

flask framework to build a basic web
web scraping technique by python package selenium
Google Firebase to store the user type in data

Challenges we ran into

To locate specific objects from websites and setting up database
Running web scraping on both linux and windows system
Running web scraping on both Firefox and Google Chrome browser
How to make the python program into google chrome extension

Accomplishments that we're proud of

Writting code which could actually helping us do task easier
Getting things to work

What we learned

We learned that running scripts on multiple platforms could give people challenges.
