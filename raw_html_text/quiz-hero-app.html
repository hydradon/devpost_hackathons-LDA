Hero App Hero App is an Android Application for online quiz of many categories connected with an online database. It has a signup activity where client first came and register himself then and id will assign to user and via this id if any other user signup then his score will increases. User select of his on kind of category and select a quiz of that category also select level for the quiz so that the quiz starts and question are asked related to that quiz then at the end its score are calculated and he has an option of restart that quiz or go to main menu.
