Inspiration: My own experience of being able to search for menu prices of restaurants easily.

What it does: Searches for restaurants around you that have menu prices for under your budget

How I built it: Using Xcode, swift and eatstreet api

Challenges I ran into: The biggest challenge for me was trying to find an api with restaurant prices. It was a lot harder than expected.

Accomplishments that I'm proud of: Being able to create this app in such a short period of time and also ship it to the App Store for the first time.

What I learned: learned a lot about swift and how to use Xcode and also how to deal with database apis.

What's next for SmartEats: For the future I'm working on implementing directions so users can set directions to the restaurant they want to go to. Also working on adding a feature to filter out menu categories.
