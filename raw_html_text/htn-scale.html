Inspiration

The need comes from the desire to keep track of your household inventory.

What it does

Tracks household items by weight and provides insights on the usage of these projects.

How its Built

Angular.js, php web application supported by a Firebase data centre which is connected to the IoT particle scales which track the weight of items. Items are kept track by scanning in barcodes on a mobile device.

Challenges I ran into

-setting up spark core/particle -latency on cloud servers -poor support for spark core/particle on windows -learning javascript and json for communications -soldering own sensor connectors -maintaining dependencies for web apps -http POST

What's next for Sparrow

-advanced analytics for consumer behaviour -machine learning to understand purchase trends -commercial use cases for sensors -shared item tracking
