Inspiration

We were inspired to create this application from our experience in Algorithms classes. There were many times where we would write out algorithms but had no way to assert it works without manually running through the code. As a result, we created CodeScanner that allowed us to quickly take a picture of an algorithm and compile it with the language of our choice.

What it does

Allows the user to take a picture of code and compile directly from your phone.

How we built it

We used Google Cloud Platform's Vision API and Cloud Functions along with React Native and Expo.
