Inspiration

Sometimes I want to know what is the synopsis of a film, the score or a recommendation. But I don`t want to find it on my mobile, I want to have this information while I'm lying watching TV. So I decided to create this Skil for Alexa.

What it does

This helps to get information about films and TV Shows gets its ratings and, in addition, this can recommend you a film.

How I built it

Using Alexa SDK for .NET, .NET Core, Azure, APL C# Library, effort, and dedication :D

Challenges I ran into

To get working dialogs. To get working APL interfaces.

Accomplishments that I'm proud of

Creating an awesome Skill.:D Learn about Alexa.

What I learned

To use Alexa SDK How to build voice interfaces. About Voice First Design.
