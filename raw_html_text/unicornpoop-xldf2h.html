Inspiration

With all of the news stories coming out about trans people getting attacked or harassed in public bathrooms, we think the ability to find gender neutral bathrooms is vital for the community at large.

What it does

Unicorn Poop would track the user's current location and allow them to submit their location as a geo-tag that marks a potential gender neutral bathroom. This would also include whether or not the bathroom is free for public use or if it requires money to access.

How we built it

We decided to create it with html and embedded javascript. We used Google App Engine to get an API that allows us to utilize location services. Implemented graphics were created using the adobe suite.

Challenges we ran into

Our team is brand new to web development, so most of this project was a challenge. Initially, understanding the relationship between the different programs in the stack was a hurdle to overcome. Then, we had troubles with properly utilizing Google API to our advantage. Finally, we came into the challenge of setting up a database and the communication pathways between the website and the database.

Accomplishments that we're proud of

We are proud of the fact that we were able to come into this as newbies and still be able to work as a team to overcome the challenges we've faced.

What we learned

We have learned how to quickly adapt to new environments and how to teach ourselves new techniques and languages.

What's next for UnicornPoop

Possibly learn more and functionally create a finished product for the community.
