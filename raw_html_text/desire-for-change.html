Inspiration

We saw the requirements of the sixth sense NGO and it was very interesting for us. So we decided to hack the problem.

What it does

A platform for connecting donors and volunteers directly to the students ( orphans / disabled / underprivileged ) helping for their career development and social welfare. It offers the users to post their needs and get help accordingly. Creating events for volunteering (teaching, learning activities).

Anybody can offer and anyone can seek help

How we built it

Open source solution using Drupal CMS

Challenges we ran into

Given the time frame and the complexity of the problem statement.

Accomplishments that we're proud of

We were able to create a stable working model.

What we learned

Never give up until you hack the problem.

What's next for Desire for change
