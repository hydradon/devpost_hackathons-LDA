• Movie Rating

Motivation

I've built this skill for myself but I am sure it will be useful for others. I often found myself googling for movie rating before making decision whether it worth to watch.

What it does

You ask Alexa skill for movie eating using movie name. Alexa skill with provide movie rating (score)

How I built it

It was built using Alexa Skill Kit and Lambda function using Node.JS

Challenges I ran into

Finding source of movie rating but I've found great "free" service which exposes API.

Accomplishments that I'm proud of

It solves problem :)

What I learned

I've learned a bit more about Java Script, Dialog Management and APL

What's next for Movie Rating

Include capability to provide more details about movie, not just rating (score). For example, cast, maybe even read reviews.
