Inspiration

Inspired by the upcoming Star Wars movie, we chose to make a game based off of the millennium falcon and the most common and widely recognized spaceships.

What it does

It is a video game that allows the player to move around the screen and collect powerups in order to defeat the most enemies and bosses and achieve the highest score.

How we built it

We built the software portion of the project with Python 3 in conjunction with the Pygame library, in addition to Arduino UNOs.

Challenges we ran into

We were unable to setup the ultrasonic sensor by the specified project due date, due to the absence of our teammate who mostly handled hardware.

Accomplishments that we're proud of

We were able to create a full-fledged and polished game (software-wise) that even us, the creators, were able to enjoy playing.

What we learned

We learned a lot of processing power restrictions and how limited some programming engines could be.

What's next for Space Blazerz

With a website we had previously designed, we will try to utilize pyjsdl to embed the game onto a website.
