What it does

The idea is to create a central repository to store all of your workout data, and then build an API on top of this to make it easy for other services to provide you with further insight.

What I learned

Building this by myself, without any experience in django beforehand was definitely a challenge. I was able to complete a few of my goals, creating a backend for holding all of the workouts.

What's next for workoutDB

We will be integrating with Strava's API and looking for other opportunities in this space.
