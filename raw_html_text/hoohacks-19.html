An intelligent query recognizer and live chat service. Get information on any topic, watch videos -- anything goes, really!

Inspiration

Ever had someone post something on a chat log and wonder what exactly they were saying? Worry no more!

Whenever users chat in Hivemind, any user can click on ANY prior chat log message and receive a smorgasbord of analytics -- sentiment analysis, image description, etc.

How to use:

Navigate to the site. Register an account, then login.

Enter any query (or YouTube URL!) into the input box, then press [ENTER].

Select any query for more details, including details such as sentiment analysis or Wikipedia abstracts.
