Inspiration

We wanted to create a Where's Waldo game that would track the user's time to find Waldo. We would provide hints where we black out half of the image where Waldo wasn't, in turn the user would be penalized with seconds added to their final score. We also wanted to implement a leaderboard.

What's next for NoWaldo
