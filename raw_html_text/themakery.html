The Makery

This is a project for hackathon PennApps XVIII created from the AngularJS WebPack Starter .

The username and password for an employer account: employer hackathon The username and password for an intern account: intern hackathon

Installation commands:

  • Clone project bash git clone git@https://github.com/damian2221/TheMakery.git cd TheMakery
  • To run front-end:
    • Install NPM packages bash npm install
    • Run dev version bash npm run dev
    • Go to http://localhost:3333
    • ...or build production version bash npm run build npm start
  • To run back-end:
    • Create a database from ./themakery.sql
    • Set up your database configuration in /backend/config.php
    • Go to the back-end bash cd backend php -S localht:8000 -t backend/
