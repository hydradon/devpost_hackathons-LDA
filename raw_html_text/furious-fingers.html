Inspiration

We wanted to create a fun experience for you and your friends - a game that you can play while you're together. We decided to make a series of fun mini games that are quick and easy to play.

What it does

The app allows you to connect with up to 3 friends and play fun games together, without the use of any kind of network.

How we built it

The app itself was built in Swift, relying heavily on MultipeerConnectivity to allow for communication between devices. The web app which shows global ranks and allows for writing/retrieving from the database was built in Python + Flask.

Challenges we ran into

The MultipeerConnectivity presented many challenges in broadcasting and reacting to events. It was difficult to make the interactions between devices seamless. Creating new games that were quick and enjoyable was a challenge as well.

Accomplishments that we're proud of

We used a new technology that is not widely used to create a fun experience for ourselves and other people that want to an interesting activity to do when with those they care most about.

What we learned

We learned a lot about project/time management as well as lots about the new APIs and frameworks we used to build this.

What's next for Furious Fingers

We plan on continuing development and adding more and more interesting mini games.
