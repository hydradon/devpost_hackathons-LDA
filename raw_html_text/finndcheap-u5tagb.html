Inspiration

Tired of manually searching flights.

What it does

It allows you to never worry about a flight, it recommends you the best options and warns you of any bad weather!

How we built it

With Java, Telegram API and Finnair API.

Challenges we ran into

Lots of nulls pointers exceptions.

Accomplishments that we're proud of

Being able to deploy in a real server the bot being available to everyone in the world.

What we learned

To create a bot for Telegram completely from scratch.

What's next for Finndcheap

Implement better recomendation searches, buttons and more options for the user.
