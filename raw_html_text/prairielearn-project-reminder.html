• Example of text messages send to a phone

Inspiration

The idea came from one of the proposed project by the current PrairieLearn developers. We saw the headline on the PrairieLearn website and decided that this project will be fun to do.

What it does

Currently, this project has an interface that allows users to add phone number and send text to their phone. Since we are using a trial account, only one of our team member's phone number works.

How we built it

We used the Twilio library to achieve sending text to phone. We also used PostgreSQL to get the data from the database.

Challenges we ran into

We have problems pulling from the database because we didn't understand the structure of the data.

What's next for PrairieLearn: Project Reminder

We would like to followup with the PrairieLearn developers to actually make the code work on CBTF Scheduler.
