Inspiration

I have always found navigating the web for the answer I need to be quite annoying at times, so I tried to make a site that helps with that

What it does

Has storage for code snippets, search API for getting results directly off of stack overflow (and the capacity for more forum systems)

How I built it

Using php and mysql with html and css as the gui interface (on xampp)

Challenges I ran into

Getting blocked by stackoverflow for being a bot. Using DOM documents for the first time.

Accomplishments that I'm proud of

Getting a search interface on stack overflow

What I learned

how to interact with other sites as a website

What's next for Hacker M8

setting up better search functionality, and adding more forums
