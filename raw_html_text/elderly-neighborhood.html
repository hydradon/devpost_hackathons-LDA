Inspiration

We realized that many old people have problems especially when interacting with their community, and we wanted to provide them a way in which they can both meet younger members of their community, and give them aid whenever they need it

What it does

It sends out a request on the website for people to either aid, or visit the elderly person, and then younger people can send a reply to join their request lobby

How we built it

We build it using the flask microframework to act as a REST api, and React to handle front end template rendering

Challenges we ran into

The main challenges that we ran into were having the template rendered in React

Accomplishments that we're proud of

The accomplishmeny we are most proud of is completeing the backend, and having a very preliminary login and registration system, and being able to get the items from the query

What we learned

We learned how to use react, to display things

What's next for Elderly-Neighborhood

We want to implement more features, and finish the display, because there are still problems with displaying everything
