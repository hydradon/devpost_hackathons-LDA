Inspiration

The project was to initially make a 3 Dimensional game of snake, but lack of parts left us to having to make a 2 dimensional-type version.

What it does

It moves the character across the 4x4 world.

How we built it

LED matrix controlled by a shift register. The user controls the character using a joystick.

Challenges we ran into

Lack of parts, using only one shift register.

Accomplishments that we're proud of

Able to have a working prototype.

What we learned

How to use a shift register and how to control an LED matrix using only one.

What's next for Joystick_Hackathon_2019

Acquire the rest of the needed parts to make a full version of the game.
