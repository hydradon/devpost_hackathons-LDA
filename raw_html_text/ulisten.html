Inspiration

When I was younger, I had a friend whose mother was deaf. I always admired how my friend could communicate without speaking and I thought it would be cool if I could help people learn and communicate in the same way my childhood friend did.

What it does

We have built an IPhone app that allows people to view sign language terms so they may repeat them to a hearing impaired person.

How I built it

We used Apple's programming language Swift to build the app.

Challenges I ran into

We ran into the problem of how to store multiple videos under the same tag and have them all play through slow enough so the user is able to see them.

Accomplishments that I'm proud of

We are proud of getting the app to show a particular gif when we press a certain word.

What I learned

We learned how to better use Swift and how Dictionaries work.

What's next for UListen

We plan on allowing users to search for certain terms instead of scrolling through a list. We also hope to allow users to input their own gifs, and have a user input text that would turn into a string sign language gifs.
