Inspiration

It began with an interest of IOT, soon became a place where nothing is impossible. Inspiration came from movies where everything can be recognised from videos or footages. Their names, age, occupations and more can be retrieved just solely based on an image.

What it does

Collects data from hardwares like WioLink. And stores into the database, this serves as a platform to do data analysis as more data is being entered.

How we built it

We used technologies like Node.js, Postgres, and documentations for WioLinks to build the groove modules.

Challenges we ran into

Scalability, Visual recognition.

Accomplishments that we're proud of

What we learned

UltrasonicRanger can be used to calculate a few dimensions of object.

What's next for FishDataCollector

Small scale visual recognition to measure the size of fish.
