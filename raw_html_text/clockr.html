Inspiration

It is hard to wake up at a hackathon once you are sleeping. It is hard to wake up from a nap after school. In general, it is hard to wake up. So we wanted to make something that had a purpose

What it does

It's an alarm clock that automatically donates to your charity if you don't wake up on time. You have to manually check off that you got up otherwise it will automatically donate to the charity of your choice.

How We built it

This app was built on swift.

Challenges I ran into

There were a lot of bugs in the beginning. Some of our code only worked for an earlier version of Swift, so we had to use trial and error to get it working again.

Accomplishments that I'm proud of

What We learned

Rohan and Sushanth came into this hackathon with no knowledge of Swift and iOS develoment. At the end of the hackathon, they had experience creating designs and integrating it into an iOS project. From Previous Projects

We were inspired from some of the previous projects we had made. We used code from open source libraries for handling user storage and previous projects for handling time change and notification.
