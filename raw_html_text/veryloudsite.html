Inspiration

veryloud.site is inspired by classic harsh noise music and 8-bit audio generation hackery.

---> Note that this is just a mini project developed in the wee hours of the night! Our main project is https://devpost.com/software/k-choice <---

What it does

It makes a loud noise. Seriously, that's more or less the only thing it does.

How we built it

30 minutes of refining the noise generation algorithm to be as unpleasantly noisy as possible.

Challenges we ran into

Not enough noise without using premade sound samples! :)

What's next for veryloud.site

It'll probably stay up for a year, until the domain expires.
