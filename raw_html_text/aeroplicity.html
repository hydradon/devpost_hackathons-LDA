Inspiration

People always talk about going somewhere on a whim, and sometimes they feel it's not very plausible, but if you showed them the ticket prices in a fast and simple way then it will encourage people to travel more

What it does

Scrapes data from the page for travel destination which is then built into an array that we later call the api on to find the the closes airport to the destination

How we built it

We built it with the google chrome extension and took advantage of the local storage to be able to pass data back and forth. We then used the SkyScanner api with the filter to find the cheapest flights available

Challenges we ran into

Getting the API to work properly, and getting the data to pass back and forth properly

Accomplishments that we're proud of

Getting it to work

What we learned

How to pass data back and forth in the chrome extension

What's next for Aeroplicity

inclusion of different type of routes and more destinations
