Inspiration

Buses are never on time, so we decided to analyse how late they are.

What it does

Scrapes data for Rutgers buses in an XML format, currently overlays the map with position of the bus.

How we built it

Using Python, XML, Google-cloud, Flask, and MySQL

Challenges we ran into

Many.

Accomplishments that we're proud of

Learnt a lot of new things, and integrated many different technologies

What we learned

Its hard to integrate, multiple technologies in a short amount of time

What's next for BusRuOnTime

Make Rutgers bus service great again.
