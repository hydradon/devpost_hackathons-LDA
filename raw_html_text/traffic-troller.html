Inspiration

We wanted to explore Machine Learning and Data Analytics by creating an application that helps lawmakers and traffic authorities in Boston be able to put more traffic lights or enforce more trains/buses in areas that have the most traffic

What it does

Using the dataset from the Boston.gov website, we analyzed the busiest locations in Boston where people got picked up and dropped off which then creates a map to pinpoint locations where traffic lights should be placed

How we built it

Matlab, and watching lots of videos and tutorials

Challenges we ran into

Figuring out the logic to create the script for the large dataset

Accomplishments that we're proud of

Using and implementing MatLab to create a map and pinpoint traffic in a region

What we learned

How to use geoshow and converting the data into a heat map

What's next for Traffic Troller

Use machine learning to predict the probability of an accident happening at specific locations in Boston in the future
