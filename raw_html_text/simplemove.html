Inspiration

Wanted to visualize renting data to assist people moving into a new neighborhood.

What it does

Visualizes Zillow renting data as well as user submitted data stored in an ESRI feature layer.

How we built it

Parsed Zillow data using python scripts. Running on a node.js server. Frontend is using the ARCGIS javascript API.

Challenges we ran into

Manipulating the Zillow data into a usable format.

Accomplishments that we're proud of

Extensive data visualization and large pool of relevant data.

What we learned

We've learned how to manage a node.js server as well as how to use the ARCGIS javascript API.

What's next for SimpleMove

Scaling.
