Inspiration

We hope to support high school students who are trying to find work but unable to consistently work a full-time or even a part-time job.

What it does

Odder Jobs allows students to browse odd jobs, internships, or volunteer opportunities posted by employers. Employers can review the basic profile of applicants and contact them about the job.

How we built it

We used Atom to create a React website using MongoDB, JWT, and CSS.

Challenges we ran into

We had some issues with merging when pushing to Git. Merge errors reappeared even after being resolved. We also had to spend quite some time solving issues with API calls and had issues with backend compatibility between computers.

Accomplishments that we're proud of

The homepage picture is an excellent fit for our website. :)

What we learned

Our group learned some new CSS skills when styling our website and also practiced debugging skills when working on backend issues, especially those regarding the API. We also learned how to better work with input and the API on the frontend.

What's next for Odder Jobs

We participate in the Web Design Club at our school, which Thomas is leading. We plan to discuss the code in coming meetings and can modify the website to practice our skills. We hope to learn to add pagination to job listings, add additional information to employer and student profiles, and improve some of the CSS.
