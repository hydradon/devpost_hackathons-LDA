Inspiration

Wanted to become better programmers, but have fun at the same time!

What it does

you go around as a knight who hack and slash zombies

How we built it

we built using unity as the engine, visual studio as the IDE and piskelapp as the art tool

Challenges we ran into

doing animation for the first time in unity and pixel art, how it all tied with code and works together

Accomplishments that we're proud of

building everything from scratch except the sky box

What we learned

How to manage out time between a large time sink, like the front-end art and creating actual features like AI and combat.

What's next for KnightHack and slash

The game itself will likely not be developed further, but many scripts and art assets will be used in further game projects from our team.
