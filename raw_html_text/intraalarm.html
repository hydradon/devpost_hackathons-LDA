Inspiration

One of our teammates reflected on herself and thought of this to wake up on time.

What it does

Keeps ringing and vibrating until the user types in a matching password. If the entry is wrong, a second pop-up will come on with a longer password.

How I built it

We first set-up the basic skeleton of the UI then built on it

Challenges I ran into

Integrating the react-native-alarm-notification module into the app Setting up the UI Most alarm libraries do not have much support and documentations

Accomplishments that I'm proud of

What I learned

Learn react-native and found out that alarm libraries are limited for ios applications.

What's next for IntraAlarm

Generate a password for each alarm for user to input when it rings
