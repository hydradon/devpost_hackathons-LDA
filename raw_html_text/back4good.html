Inspiration

Technology tend to connect people around the globe as well as isolate people that surrounds you. Back4Good is an attempt to use technology to create new experiences that renew human relationship and foster better communities.

What it does

It chooses and shows a random challenge and a place to foster better family and community relations.

How we built it

It uses python for the backend and flask web framework, a sqlite database to store the challenges and places information and html/css to render the front end.

Accomplishments that we're proud of

Getting to finish the project as we intended it to be was rewarding feeling.

What we learned

We learned everything from how to work correctly and efficiently as a team using github and git to track our work.

What's next for Back4Good

To cause a broad impact.
