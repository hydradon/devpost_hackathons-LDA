What it does

Users upload their resume. They are then sent rejection (or acceptance) emails from the server, with a link to a page summarizing which skills accepted applicants tended to list on their resume. [Not actually fully implemented]

How I built it

Our project is written in NodeJS and python. The server is hosted on Heroku for now.
