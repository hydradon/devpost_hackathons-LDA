Inspiration

We always get a sense of dehydration and we know ti is bad.

What it does

You can track how much water you drink.

How we built it

We used Notepad++ and coded it using Python and HTML

Challenges we ran into

Updating the database in the correct format. Making a table. Updating the total amount of water that you drank.

Accomplishments that we're proud of

We really like how the website turned out and also of how it can register different people into the database and log them in and store their data. We also feel proud of how we were able to update the total amount of water.

What we learned

We learned how to style a website using HTML and also storing databases in comma-separated value files. The most important thing was updating the .csv file via browser input.

What's next for H2O Tracker

Make it so that people can't register with the same name. ScoreBoard to track trends and motivate users Make the password more secure using hashing Table to record history of water drinking Make login page look more colorful.
