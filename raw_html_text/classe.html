What it does

ClassE uses student evaluation data to rate each course-professor pairing to empower students to make informed registration decisions.

How we built it

A server running mysql supplies enrollment and evaluation data as JSON objects using web.py. Our web page consumes these objects and returns useful data to the user using jquery.

Challenges we ran into

The bulk of our project involved parsing university published data into a format that we can perform analysis on. This took hours .

Accomplishments that we're proud of

It works... It actually works

What we learned

Debugging Perl scripts you wrote at 3am will make your eye bleed

What's next for ClassE

We'd like to implement some pretty graphs, and a lot more metrics for professor performance. The data is there, we just need to format it in a useful way.
