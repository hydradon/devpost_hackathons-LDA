Inspiration

Our inspiration was that, being in Waterloo, not many people played sports. We rarely ever had enough people to play basketball, so we made this to premake events ahead of time.

What it does

It marks a location on the map and sends it to your friends so they can see when/where it is

How we built it

We built it using android studio and java on the mobile side, firebase for db, and javascript/html/css for web all using google map api

Challenges we ran into

google map api, countless things could be said about this

Accomplishments that we're proud of

having a finished product that is ready to use

What we learned

A lot just about google map api and the languages that were used
