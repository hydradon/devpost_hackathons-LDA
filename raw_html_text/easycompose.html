Inspiration

I am a musician and always dreamed of having a simple input of notes to sketch ideas and save them / be able to work on them later. Now this can be true. Until now there was no great solution. So I'm happy to work on it.

What it does

Write down musical ideas / compositions with touch on your tablet / smart phone

How I built it

With myscript music api

What's next for easyCompose

Release a great app for iOS and Android for musicians. Hopefully with the help of the myscript-team.

Problems sometimes

Sometimes the api server is not available (returns a 500 Error) which makes the app unresponsive (Unfortunately easyCompose doesn't tell the user that the api server returns an error) which is only visible in the console.
