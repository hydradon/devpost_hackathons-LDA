Inspiration

Centralized exchanges suck because they get hacked, and are prone to front running/manipulation. Decentralized exchanges allow direct transfer of tokens.

What it does

It's a decentralized and open orderbook for 0x.

How I built it

It's built with React/Redux on the frontend, orders are created using the 0x protocol, and propagated over websockets to anyone else listening to the website or websocket API. All source code is open source as well: https://github.com/Everhusk/ZeroExchange
