Inspiration

Language learning textbooks build the learning process on a number of thematic sections. Those sections are generally linked to a specific situation in life: going to a restaurant, traveling, buying something in store. We thought that the process could be made more interactive using modern technology.

What it does

Imagine you go to a coffeeshop in Germany. Lexicon provides you with words and phrases you can use to explain yourself in German. The app understands where you are and shows you exactly what you need.

How we built it

We have used iOS and Swift to build our app. Location is determined using Foursquare API. The data is stored and synchronised using Google's Firebase.

Challenges we ran into

Providing precise location tracking and creating a unique user interface, which we call "The Glow"

Accomplishments that we're proud of

Elegant and stylish UI, Apple Watch support, Responsive Design, High Utility and of course - cute mascot.

What we learned

We have learned to use iOS SDK networking features, Animations and Apple Watch SDK

What's next for Lexicon

Release on AppStore, implementation of premium language packs, Text to Speech features and much more - stay tuned
