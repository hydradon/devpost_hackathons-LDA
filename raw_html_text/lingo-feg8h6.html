Inspiration

Wanting to learn how to more about foreign cultures and bring people together through tech.

What it does

Connects users that want to learn a language with fluent speakers of that language. This fosters a healthy environment to organically learn your desired language and connect with other people at the same time.

How we built it

Back-end was done using firebase and front end was developed with React

Challenges we ran into

Merge conflicts and general lack of knowledge and expertise

Accomplishments that we're proud of

Learn React and built a somewhat functional UI

What we learned

How to use react.

What's next for Lingo

Story Creation Mode, Story sharing options for consumers, Daily discussion topics to incentivize people communicating with one another
