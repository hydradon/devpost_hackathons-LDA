Inspiration

When you leave your house or need a hands free way to get Unitrans info quickly. When you don't want to use your data or data isn't available but sms is still available.

What it does

The Alexa skills and SMS response will tell you information about when the bus will arrive as well as predict when you will get to your destination

How I built it

Using Python/Flask for the back end. Alexa Skill and Twilio makes an api request to my server that is deploy in Google App Engine

Challenges I ran into

Alexa Skills has no debug console so I don't know whats wrong.

Accomplishments that I'm proud of

Something that works

What I learned

This weekend I learned how Alexa Skills and how to deploy using Google App Engine, I also learned a bit of React Native, but that idea was scraped

What's next for Unitrans Info Extenion

Add more to the data set for locations as well as train the mode to understand the stops better
