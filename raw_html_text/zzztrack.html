Inspiration

We were inspired by our and our peers' sleep deprivation this past semester. We hope that people will take sleep more seriously here if they have something that tells them exactly how little time they spend sleeping.

What it does

zzzTrack tracks how long you have been sleeping. Specifically, it counts how much time has passed between clicking a "sleep" button and a "wake" button. It is meant to be open as the user is sleeping. There is an "interrupt" button to pause the timer and a "snooze" button to count time spent half-asleep just before the user gets out of bed.

How I built it

My teammate Annie took care of most of the CSS, while I worked on the Javascript. The sleep tracker subtracts the date, in milliseconds, between the time each button was clicked. We used Microsoft Azure to host our website.
