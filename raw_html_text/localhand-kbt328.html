• LocalHand

Inspiration

We were inspired by social networks that bring people together like MeetUp and platforms that help people solve problems like KhanAcademy, and wanted to combine features of both to help people and small businesses.

How we built it

We used HTML, CSS and JavaScript for the front-end and used Firebase for the backend.

Challenges we ran into

Implementing back-end chat features, using Google Maps API

Accomplishments that we're proud of

Successfully setting up a messenger feature through Firebase.

What we learned

Certain front-end features and API functions can be used in exchange for back-end features and implementations.

What's next for LocalHand

A sign-up page for problem-solvers, pitch for any organizations seeking to pick up the project
