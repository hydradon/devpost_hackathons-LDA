Inspiration

When we went for lunch, we saw Minh waste a lot of food, so we wanted to come up with an idea to help reduce the problem of food wastage

What it does

The bot helps to track the items put inside the fridge and prompts the user when the item is about to be expired. In this way, people will be reminded to use the item ASAP and hence reduce food wastage.

The bot has features to add/check/remove items into the fridge, as well as an added functionality to scan receipts to add a list of items that the user may have bought from a shop.

How we built it

We utilised python to code a telegram bot, using OCR to help with the receipt recognition.

Challenges we ran into

Learning telegram bot within 24 hours and maintaining a proper structure as we tried to expand features over the course of the hackathon.
