Inspiration

We were interested in creating a website that displayed weather graphics in an easily understandable format, marketed specifically towards students in Ann Arbor.

What it does

The website uses current weather data to display graphics based on the temperature and other factors, then recommends a playlist to the user based on the weather.

How we built it

We designed the website using HTML/CSS/Javascript/Bootstrap, and used the weather underground API for the weather data.

Challenges we ran into

We spent a significant amount of time working on the website's animations.

Accomplishments that we're proud of

We were able to develop and integrate complex animations into the website (raining in the background, animated icons, etc.)

What we learned

How to build a website from scratch, as well as how to combine and present data from multiple sources.

What's next for MoodyTunes

Linking it to the user's personal playlists, as well creating a mobile app version.
