Inspiration

I like to collect coins and when I get a new coin I have to look through all of the coins to see if I have the coin already.

What it does

This skill keeps track of what coins you have. It can tell you how many coins you have in any of the following combinations denomination year and mint mark.

How I built it

I used python to build this skill

Challenges I ran into

One challenge I ran into was how to make multiple definitions down to one

Accomplishments that I'm proud of

I am proud of the search function and its ability to search any of the requests given in one definition instead of using multiple definitions

What I learned

I learned about classes, and a little more about coins.

What's next for Coin Collection

Allow for other countries coins, add options for design, error, material, grading/condition
