Alexa Chat Line

Alexa Help Line is a skill for the amazon Echo providing comfort people suffering from depression, anxiety or any other emotional distress.

Activating the "Empathy Bot" will record the response by the user, and analyze the underlying tone. The analysis, performed by IBM's Watson tone analyzer, will flag situations that require intervention. The response will then be directed to a chat line for further support from professional mental health care providers.

Alexa takes the speech of the user, and turns it into text (poorly). The skill (Lambda) sends the text to Watson, using RapidAPI as a wrapper, and analyzes the text, including flags. The analyzed text is sent by SMS to a person chatting with the user via Twilio, flagged for crisis situations if necessary.
