What it does

Allow citizens to contribute health indicators based on current weather conditions. It allows them to view CDC data at the state level, for exposure to various toxic elements such as pesticides. Also, it is possible to view current forecast for smoke, wind speed and current weather advisories that can help to understand and prepare for any health related issues.

How I built it

The app is built using arcgis-android sdk. It uses Firebase to store crowdsourced data. It uses http://tmservices1.esri.com/arcgis/rest/services/LiveFeeds to access current foreccast.

What's next for Toxic Watch

Beta testing before releasing to a wider audience.
