Inspiration

WHAT IF you could identify malicious intent before action?

WHAT IF you could tell someone who’s about to post something really dumb and hurtful to social media: “Your brain signal suggests you are not thinking clearly!”

What it does

We listen to the orchestra of your mind by using the MUSE headset. Then we import this data we receive from the sensors on this device to matlab, then to eeglab, where we can interpret data.

How we built it

Some rough previous knowledge, caffeine, YouTube, ...

Challenges we ran into

Available hardware, clear communication, clearly defining a problem to solve.

Accomplishments that we're proud of

We can listen to your brain!!!

What we learned

We learned a ton about matlab through eeglab!

What's next for The super duper brainwave meter 7000 (EEG profile)

If we could create a pretty and also useful UI that a researcher could easily use, we think this would be incredibly helpful.
