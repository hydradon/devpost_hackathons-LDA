Inspiration

Registration over the summer for courses was difficult because we had to organize our schedule so there were no conflicts.

What it does

Gives you an randomly generated schedule with your classes

How we built it

We built it using Python, Flask, HTML, CSS, and we got the data from the scotty labs course api.

Challenges we ran into

Navigating the JSON file. Developing an efficient algorithm that we can use.

Accomplishments that we're proud of

The Idea to simplify course scheduling, something that can positively impact the CMU community. Succesfully creating a website.

What we learned

We now better understand how to write efficient algorithims. We also are a lot more familiar with Python and web apss.

What's next for Course Scheduler

Fixing the remaining bugs in the algorithm and integrating SIO features into course scheduler
