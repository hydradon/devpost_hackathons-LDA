Inspiration

As students and part time instructors we know how hard it can be to get feedback from students. Especially with the many distance learning options that are available today.

What it does

Realtime Class sentiment analysis and transcription of live lectures to multiple languages. Accessibility added for those impaired of hearing with text to speech.

How we built it

Mongo DB, Express, HandleBars, Microsoft Emotions API,WebRTC, Canvas, Google Translations, Sockets, Bootstrap 4 Logins and JQuery. We built modules of the application and connected them accordingly.

Challenges

Dealing with real time data, transmission of data and handling a lot of moving parts.
