Inspiration

We wanted to make an alarm clock that was effective at doing it's job.

Wakes you up by playing obnoxious music for an alarm, extorts user files, and ridicules them publicly on twitter.

Wrote a python script to run an alarm and delete files. Also uses an API to publish tweets for the ultimate humiliation.

Accomplishments that I'm proud of

We made an annoying application.

What I learned

How to use a twitter API.

What's next for Alarm Shaming
