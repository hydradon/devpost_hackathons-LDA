Inspiration

What inspired us to build this project is the idea that many elderly use tablets, and that many elderly still live with their families but are always home alone. This inspired us to build an application that will always inform the elderly on their tablet in huge block letters on a full screen background important information like taking their medicine.

What it does

Sail allows for a sender to send information to the elderly. No matter what the elderly is doing on their tablet, like watching YouTube, Sail will forcefully switch the app to the Sail page and display the information sent! Upon reading the message, the elderly can confirm he/she read the message.

How I built it

We built it using react-native as our mobile device platform, along with node.js for our messaging server and php/mysql for our database implementation

Challenges I ran into

Challenges we ran into is ensuring that the socket events were sent correctly. In addition, configuring the data on the php/mysql server to allow 3rd party applications to access the data

Accomplishments that I'm proud of

It works! Managed to create a lot of resources during small time period.

What I learned

I learned how to organize and prioritize tasks properly. Despite the numerous other ideas we have, we understand that time is not on our side, so we had to careful in what features we want to push out.

What's next for Sail

We hope to continue working with this and improving it by adding more features such as a detailed schedule of the elderly.
