Inspiration

Everyone complains when an application either requires a player to make an in app purchase to progress or allows others to get ahead through in app purchases, but everyone loves tweeting! So why not make your tweets have worth?

How it works

We used Twitter's API to allow users to redeem their tweets via our own API.

Challenges I ran into

Creating the server and authenticating using Twitter's API were the largest challenges we ran into.

Accomplishments that I'm proud of

Seeing our tweets getting redeemed in the demo application we made was a huge accomplishment.

What I learned

A lot was learned through making the demo application and the Tworth API such as: creating servers, game design, pseudo-wave functions, and UI creation.

What's next for Tworth

The current version of Tworth only searches tweets for keywords, and this could lead to issues when someone tweets "I hate the game insert key word ". In the future, Tworth will be able to analyze the tweets and decide whether or not it deserves the payment.
