This is a chat bot built using the Prompt API which allows you to perform basic linux server management tasks via SMS. With predefined scripts you can run any task you wish. Also can control system services. It's convenient in the case of a server emergency when you don't have access to a terminal. Limits access to authorized users via web authentication.

Example usage:

@myserver [servername] [start|stop|restart] [service]

@myserver [servername] [predefinedscript.sh]
