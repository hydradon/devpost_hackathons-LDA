Inspiration

over the summer Emma was put in charge of a very fast three year old

What it does

It is a child's watch that send a message to a parents smart phone via bluetooth when the child gets too far away

How I built it

we 3D modeled it digitally (on fusion) and mocked up app display on Adobe illustrator

Challenges I ran into

We did not have the tech or manpower to build a physical model

Accomplishments that I'm proud of

getting it actually done

What I learned

software and coding is really confusing

What's next for Tiny Ninja Tracker
