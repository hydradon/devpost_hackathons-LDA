SafeCampus

The app allows students to access a button that notifies specified contacts (for example campus police) their real-time location in times of emergency. For scenarios that are less time sensitive, emergencies can be specified (school shooter, fire, medical, etc). We built the app using the android studio application in the Java language. We used AWS SNS tools and AWS Device Farm for testing.

We encountered some challenges: we had to learn a new software and how it works with android devices. We are proud that we figured out how to properly design an android phone app.

We will be glad to develop an ios user app for user diversity
