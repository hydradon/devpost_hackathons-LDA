Inspiration

Lo que nos inspiró a hacer este proyecto fue que en el área de la medicina existen muchos medios como este para desarrollar prótesis que ayude a las personas discapacitadas, ya sea en la rehabilitación como una prótesis.

What it does

Nuestro brazo robótico se mueve por medio de servomotores y puede agarrar objetos y moverlos de posición, esta controlado por medio de una aplicación en android para dispositivos móviles.

How we built it

Utilizamos la plataforma de IMP app inventor 2 para hacer la aplicación móvil y arduino para programar los movimientos del brazo.

Challenges we ran into

El obtener los grados para capturarlos en la aplicación y así se realizara el movimiento.

Accomplishments that we're proud of

Estamos satisfechos con que lo planeado resultó, logramos que el brazo robótico tomara objetos.

What we learned

Aprendimos a utilizar la plataforma de app inventor 2.

What's next for Brazo Robótico RADS

El siguiente paso es automatizar el brazo y que sea más eficiente.
