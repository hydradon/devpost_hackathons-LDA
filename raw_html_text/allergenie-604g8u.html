Inspiration: In recent news, a 3 year old student died from an allergic reaction. This showcases that teachers need allergy information about their students easily accessible to them

What it does: An iOS App that tells teachers what allergies their students have.

How we built it: We prototyped it using Marvel, but it would be developed using iOS native app development.

Challenges we ran into: Learning about school infrastructure and how documents are stored.

Accomplishments that we're proud of; Completing an entire mockup.

What we learned; We learned about why and how schools need to modernize their data

What's next for AllerGenie:

Learning ways to improve the UX and UI to better solve the problem.
