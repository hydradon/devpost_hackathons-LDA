Inspiration

La necesidad de transportarnos de manera optima en una ciudad mediante el transporte publico.

What it does

Indica mejores rutas del transporte publico cercanas a tu posición que puedes tomar para llegar a tu destino.

How we built it

En Android Studio

Challenges we ran into

Selección de rutas optimas, acondicionamiento a lenguaje Android Studio, API de Google Maps, adquisición de base de datos.

Accomplishments that we're proud of

Se logro seleccionar las rutas cercanas a la posición, editar o agregar nuevas rutas, trazarlas en google Maps, incluir una interfaz para recibir comentarios del usuario.

What we learned

Programación en Android Studio y Java.

What's next for SmartRoute

Mejorar la interfaz de usuario, expandir a cualquier ciudad del mundo, mejorar la velocidad de respuesta.
