Inspiration

We were inspired by the trivia games we played growing up. Our fond memories of playing those different games translated to creating something for kids to learn about the world around them.

What it does

Knowledge Quest provides facts about different "adventures," or topics that kids can learn about. They are then quizzed on the information they just heard.

How I built it

With previous skill development experience, we used Node.js and Visual Studio Code to create knowledge quest.

Challenges I ran into

Getting the data to be read by the game so that it could produce engaging adventures.

Accomplishments that I'm proud of

We worked together to create a new game that is educational for both kids and adults.

What I learned

We chose topics that we were interested in and tried to find some kid friendly, interesting information to fill the adventures with. Even though we were already had a base understanding of the topics, we still learned some new information!

What's next for Knowledge Quest

We would like to add sounds and images so that it could be more engaging.
