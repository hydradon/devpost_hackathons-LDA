• Open an image in a new tab normally.

  • Without Better Image Tab, you only see the image by itself.

  • But with Better Image Tab, you get so many more options.

What It Does

Better Image Tab was created to turn the empty bare bone tab you get with just the the image into something useful. It allows you to change many properties of the image. Download the image as a different file format and many other features. This project was implemented as a Chrome Extension.
