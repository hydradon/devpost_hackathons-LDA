Inspiration

I've always needed a really easy to use note taking app for creating lists, like splitting a bill, or just writing down a bunch of things I want to buy.

What it does

Allows users to create notes in a list, and add values to the notes, then allowing them to add or calculate the total of the items in the list.

How I built it

Swift and Xcode.

Challenges I ran into

I had a lot of issues with CoreData and saving lists properly.

Accomplishments that I'm proud of

I'm just really proud that I shipped an app to the app store.

What I learned

I gained a lot of coding experience, learned that I could really be moderately proficient at a coding language.

What's next for List Easy

I have a lot of features planned for the first update, including image uploading, exporting lists, and paid customization.
