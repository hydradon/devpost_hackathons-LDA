Inspiration

We wanted to bring the time it takes to start a playlist as close as possible to zero seconds.

What it does

Given a picture of your surroundings and authentication to Spotify, our app will generate a playlist tailored to your musical tastes and current situation.

How I built it

We built the app on Django, using APIs from Clarifai, Twitter, Microsoft Cognitive Services, and Spotify.

What's next for SoundLens

We need to host it, which we ran out of time to do. More news on that later...
