Inspiration

We wanted an easier and more interesting way to search the internet through Bing.

What it does

It allows the user to see a snapshot of each web page returned by a Bing search. When the user finds a page that looks promising, they will be able to select it and go to that specific web page in full.

How we built it

We used Unity with an Oculus Rift to build our project environment and then called a jar file packed with a Java based web API calling service to get the snapshots.

Challenges we ran into

Downloads took a majority of our time and technical difficulties with input from the Oculus remote lead to a lot of lost time debugging and unnecessary research.

Accomplishments that we're proud of

We were able to navigate through the first layer of results using our application.

What we learned

Downloads should be taken care of before hand and that some challenges aren't always solvable.

What's next for Search Xavier

To build the next generation of search engines optimized for a future packed with AR, VR, and everything in-between.
