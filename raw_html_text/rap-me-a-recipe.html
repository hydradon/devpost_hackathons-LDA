Rap-Me-A-Recipe

HackUMBC 2017

This hack uses the McCormick API to get a recipe. Then, we use the MusixMatch API to match words in the recipe "description" and to get lyrics for each song. Then, we compare the lyrics with transcripts in Youtube videos to get time stamps for each word. After, we use the Spotify API to build a rap playlist that will play each song at a specific time. The URL is sent to Alexa via AWS and she will rap you the recipe for food on McCormick.com
