Inspiration

So many places to visit, so little time to open windows! It's for me, it's for Krishna's kid, it's for my elderly relatives!

What it does

64 buttons, 64 actions! Open websites like news, weather, the SpaceX YouTube channel, or my favorite GitHub repos. Open native apps like FaceTime or Skype. Configure the buttons via the website. Support complicated developer workflows by making it easy to visit all the disparate webpages and apps as needed.

How we built it

It's a steaming pile of JavaScript, nodejs, and Bootstrap, plus the LaunchPAD USB-connected button thingie. LaunchPAD is intended to be used by musicians.

What's next for LaunchMe.site

The buttons can either have stickers, or they can light up in colors. For my elderly Dad, we can make a photo album with new photos blink red...
