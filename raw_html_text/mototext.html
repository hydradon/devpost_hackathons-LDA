• Welcome

Inspiration

Empowering persons with disabilities in ways that would allow for easier communication and living has been a passion of ours.

What it does

It allows disabled people (e.g: transradial amputees, blind individuals) to be able to text messages without being limited by their disabilities. They wouldn't need to use voice commands. The goal of this project is simplistic texting through muscle movement rather than finger typing.

How we built it

We used a Thalmic Labs Myo for the hardware to recognize the distinct muscle movement to provide commands to a mobile application built using React Native.

Challenges we ran into

It was our first time using a Myo. Our main obstacles was integrating it into a React Native system and also understand how it works.

Accomplishments that we're proud of

We were able to understand the capabilities of the Myo and were able to achieve a significant milestone with the messaging software, despite having little prior knowledge of the Myo.

What we learned

How to operate and develop with the Myo, and integrate it with a mobile application.

What's next for motoText

GyroScrolling, Dictionary implementation, address book lookup, earpiece feedback and texting without being in contact with the phone.
