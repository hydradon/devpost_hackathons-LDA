Inspiration

The inspiration came from the difficulties that people with a disability face and how they must be feeling when dealing with every day situations.

What it does

Showcase a rather difficult platformer but with the main factor of jumping taken out along with speed being heavily cut down, forcing the player to think outside the box to get to the goal.

How we built it

We've built it using the Unity engine with C++ coding along with Photoshop for drawing out sprites and tile sets.

Challenges we ran into

Figuring out how to get the game to work properly and learning the ins-and-outs of Unity. We were unable to complete even a level due to difficulties of learning how to use the engine.

Accomplishments that we're proud of

Got the basis of the game to work (the movement and some aspects like platforms and intractable objects like enemies and trampolines)

What we learned

We've learned the process that comes from developing a game alongside more information on working with Unity to its fullest.

What's next for 2D Game For Awareness

To complete the original game idea or take the concept and implement it into a future project
