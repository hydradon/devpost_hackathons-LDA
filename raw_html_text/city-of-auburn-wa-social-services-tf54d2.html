What it does

It simply finds the nearest Social Service location where ever you drop the marker. The table to the right will populate with the nearest locations and how far away they are. Be sure to click on the red markers to view additional information, or click on the name in the table and it will also show additional information

How I built it

Using ArcGIS JavaScript API and Mapbox JavaScript API

Implementing it into our GIS web application environment
