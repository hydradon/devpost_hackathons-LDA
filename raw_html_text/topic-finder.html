Inspiration

We wanted to find a way to connect multiple platforms of social media to a single site.

What it does

Topic Finder takes a keyword inputted by the user and prints the top results from certain social media websites.

How we built it

Topic Finder was built using Python, a YouTube API, a Reddit API, and html for the front end.

Challenges we ran into

The challenges we ran into were figuring out flask, discovering how to use different APIs, using the Facebook API, connecting front end with back end, taking and printing the results from YouTube and Reddit onto our website, and designing the website to make it easier for the user to read.

Accomplishments that we're proud of

We're proud that we were able to take on a considerably tough project and be able to complete it. We were also able to cooperate and input different perspectives that elevated our project.

What we learned

We learned how to implement both Python and Flask to send and receive user input. We were able to find out how to take results from our python file and transfer it to our result html file for display. We also learned more about APIs and how to use them.

What's next for Topic Finder

We hope to add more APIs to our project and implement more search filters. Also add the YouTube videos to our site.
