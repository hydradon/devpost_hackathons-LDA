Inspiration

Improve upon my friends simple Ethereum Smart Contract with some Bootstrap, JavaScript, and jquery.

What it does

Adds a string to the Ropsten Ethereum Testnet smart contract and retrieves the last string added.

How we built it

Using VScode and lite-server.

Challenges we ran into

I wanted to add a list of all the moods added to the blockchain but I ran out of time before I could figure it out.

Accomplishments that we're proud of

Adding a few features to the front end of the smart contract.

What we learned

Blockchain is hard.

What's next for Ethereum MoodDiary

It will be used as an Ethereum Teaching Tool for Apprent.io
