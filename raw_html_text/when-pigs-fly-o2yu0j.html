Inspiration

We decided to model the no connection dinosaur on chrome and add our own twist to it to educate users about healthy and unhealthy choices.

What it does

The user moves the pig sprite to avoid unhealthy decisions (which will deduct their lifespan) and try to get the healthy ones.

How I built it

We used GameMaker Studio 2.

Challenges I ran into

Since it's our first time making a game as well as our first time using the software, we ran into a multitude of difficulties trying to implement our ideas.

Accomplishments that I'm proud of

It's functional. :D

What I learned

We learned how to make a game, which encompasses the graphical interface and the coding part.

What's next for When Pigs Fly

If we had more time and experience with the software, we would try to make it endless, like the dinosaur game on chrome! Since we can't upload the game, please come to us to demo it!
