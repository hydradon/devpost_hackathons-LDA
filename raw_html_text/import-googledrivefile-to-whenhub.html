Inspiration

This is a web app that will help users to import their Google Docs presentations into WhenHub.

What it does

One of the main advantages the user has is that the app actually automatically imports images taken from the presentation so the original Google content remains protected.

How I built it

Using Google Java API to convert presentation to image, aws (s3) for hosting images and finally storing Schedule and Event on WhenHub API.

Challenges I ran into

A small conflict with Google Java API fixed with Gradle.

Accomplishments that I'm proud of

Nice rendering of PDF to image to WhenHub.

What I learned

Management of Java projects with Gradle. Rendering of PDF to image with Apache libraries.

What's next for import-googledrivefile-to-whenhub

Ability to sync additional content types could be a major improvement for future versions.
