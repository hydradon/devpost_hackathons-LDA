QuizAR

Study terms, models, graphs, and more with the power of AR

What I did In simple terms, I built a web app that helps people with dementia and people who want to memorize vocabulary terms.

How does it help people? I wrote a couple Javascript libraries for camera processing, tracking, etc. during the hackathon to aid the AR that I had implemented

What did you use? I used Node.js React.js, some basic physics, and sensor fusion to create a highly efficient and scalable AR web app that can be used to create a memory palace of Quizlet flashcards that can help people who need to memorize stuff or help people with dementia with recognizing objects.
