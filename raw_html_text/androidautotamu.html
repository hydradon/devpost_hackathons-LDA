Inspiration

We wanted to use commonly available devices to help individiuals hurt in automotive accidents.

What it does

It detects rapid deceleration of more than 2 G forces (Roughly a 45 mph car accident) and promps the user to ensure they are okay. If the user fails to respond the app will (hopefully) call for help.

How we built it

We coded the app in Android Studio and used their android emulator to simulate the sensor data.

Challenges we ran into

It was quite difficuilt for us to get an accurate measurment of acceleration over time as opposted to instantanious acceleration.

Accomplishments that we're proud of

We are proud of our ability to use sensor data in a program!

What we learned

We learned loads about Android API and how to manipulate sensor data!

What's next for AndroidAutoTAMU

Hopefully to complete the interface and to implement the functions that will contact help if the user fails to respond!
