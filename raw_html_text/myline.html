Inspiration

We get tired of waiting in line.

What it does

Gives real-time updates on the number of people waiting in line. User picks from a list of dining areas on campus and will get to know the length of the line.

How I built it

Used Android Studio to build our user-interface and a compatible programming language to get information from the sensor and process that data.

Challenges I ran into

Center-aligning How to track the number of people and coming up with a simpler solution.

Accomplishments that I'm proud of

Using Android Studio and getting a basic functionality.

What I learned

How to use Android Studio.

What's next for Q

Use the data collected to estimate the approximate wait time. Use student schedules to interpret rush hours.
