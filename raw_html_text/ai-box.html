Inspiration

To help those who have difficulty hearing

What it does

When the voice box hears "Repeat this,..." followed by a statement, it will repeat the statement back.

How I built it

We used the Google Voice box kit which came with a Raspberry pi. We coded in python using Google's cloud API

Challenges I ran into

We initially wanted to integrate a voice box with a vision box to detect images and output the name of the image through the speaker.

Accomplishments that I'm proud of

After many failed attempts, we are proud that we were able to get our prototype to function.

What I learned

We learned how to set up a Raspberry Pi wirelessly through a hotspot and ssh client.

What's next for AI BOX

The next step would be to integrate the vison feature to our voice box to help the visually impared.
