Inspiration

Working in company that uses Agile and scrum as there development cycle we incorporate Trello to ease the process.However the constant moving of cards and management of these stories can be a hassle.Goat is the atom plugin that revolutionize how we do Trello and Agile.

What it does

Goat is a new Atom package that integrate Trello in an ease to use GUI. It allows user to see their boards and manage the stories. Easily add comments side by side actual code and move stories throughout their life cycle.

What's next for Goat By Atom

More coffee and definitely more integration between Trello and hacker's favorite text editor!
