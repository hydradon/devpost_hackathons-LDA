Inspiration

No one likes to type and document during meetings espacially scrum meetings. So we use speech recognation.

What it does

Create Tasks to scrum board. Assign task to developers. Change task status. Moderate daily scrum meetings.

How we built it

We built a alexa skill with Trello requests. We do it with the Alexa console and node.js for the backend.

Challenges we ran into

Speech design is very complicated. Not very flexible with speech requestes. Many functions we expected which are not included/provided.

Accomplishments that we're proud of

The requests beetween the alexa skill and trello

What we learned

Find a solution for the not flexible speech design method. Work outside from aws.

What's next for Scrum with Alexa and Trello

Include other Scrum tools. Waiting for better documentations, more (german) functions for alexa
