Inspiration

Lots of people get stressed about what they say - and don't say - to others. Why not try it out first?

What it does

Lets you practice writing messages, delete them, retry, and actually send the ones you like.

How I built it

Using React Native for Android, forked react-native-gifted-messenger and used react-native-send-intent .

Challenges I ran into

First time using react and a lack of material on using react native for Android.
