Inspiration

I used IoT devices to manage a baby, which work independently. I always felt if they talk each other, they can generate useful insight for baby health.

What it does

Eny turns on lights in baby room and parent’s bed room. Open garbage can for diapers by eny. These activities are logged, and visualized nicely :)

How we built it

Create server that keep listening to Eny activity. Server visualize the activity in order to understand if a baby has any issue.

Challenges we ran into

We had a lot of ideas, but majority of them introduces additional work for users. We wanted to avoid such ideas.

Accomplishments that we're proud of

The proposal does not force user to do additional work, but keeps track of important logs for baby health.

What we learned

Hardware has unique challenges compared to software which our team is familiar with.

What's next for eny
