Inspiration

After talking to optum, we decided it would be an interesting project to apply graph theory and linear algebra to solve the problem of distributing nurses effectively

What it does

Inputs a pool of nurses and distributes them to care for patients as efficiently as possible

How we built it

Using linear algebra techniques such as page-rank algorithms and adjacency matrices to optimize the number of places a nurse can visit

Challenges we ran into

Applying numerical linear algebra/algorithms and linking front-end development

Accomplishments that we're proud of

Using multiple Linear algebra algorithms numerically to get a solution we know is efficient

What we learned

Tkinter in python and different numerical algorithms

What's next for Efficient and automated routing of nurses

A more friendly front-end, efficient computing, and better storage/linking to database
