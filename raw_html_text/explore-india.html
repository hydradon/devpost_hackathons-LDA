Inspiration

The inspiration for this project came after I saw a skill by the Alexa team about Space. Because of the new APL feature now devs can show images and videos also. At that moment I thought it would be great to make a skill like that about India because India is full of colors and beauty.

What it does

It shows you images and information about different cities of India.

How I built it

I built it using Alexa Online Console.

Challenges I ran into

Using APL was a little tricky because this is the first time I was using it.

Accomplishments that I'm proud of

I am proud of the fact that I made the skill in time to submit into the hackathon.

What I learned

I learned very much about APL and Alexa working itself.

What's next for Explore India

Adding dynamic features so that it will work with any place the user might want to explore.
