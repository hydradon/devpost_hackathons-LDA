Inspiration

Just as Marco Polo pioneered East Asian Trade, we have pioneered the trade market of the modern world.

What it does

Combines the robust data available from Slice with a visual competent: Google Maps. Allows users and companies to see what items are being bought most frequently online in a particular geographic region.

How I built it

Primarily with the Ruby on Rails Framework. Incorporated Javascript, HTML, and CSS as needed.

Challenges I ran into

Deciding which web development framework to use was one of our biggest challenges. We briefly experimented with Angular and Node before deciding on Ruby on Rails. Correctly using callbacks with Ruby on Rails in conjunction with the Slice API was also a major challenge.

What I learned

This project served as a great intro to Ruby on Rails, and Web Development in general.

What's next for Marco

Full integration with Slice, varied time frame options in order to select trending period (the last week, month, year, Christmas weekend, etc), and integration with Amazon product images.
