Inspiration

Angles is a topic in which every kid faces difficulty while learning. I myself faced difficulties specifically when learning becomes boring from books. An interactive way of learning a topic can actually make learning fun.

What it does

Angle Info is a skill that helps kids learn about angles. It tells them about all sort of angles like right angles etc. It also provide kids information about a specific angle such as 34 degrees.

How I built it

The skill is hosted on AWS Lamda. I used Alexa Skills kit for development. It was a good experience to learn these technologies offered by Amazon.

Challenges I ran into

I had no prior knowledge of node.js

Accomplishments that I'm proud of

I developed a skill that could help kids learn angles interactively so that it's not boring for them.

What I learned

I learnt node.js
Working on projects is actually interesting and fun.
Time Management

What's next for Angle Info

I'm looking forward to adding quiz capabilities to the skill, and making the task of learning a little less boring.
