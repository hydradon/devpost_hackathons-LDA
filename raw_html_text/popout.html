Inspiration

We challenged ourselves to think outside of the box! We became inspired by DC universitys' culture to 'Pop Out' and we wanted to incorporate that theme to help underserved communities.

What it does

Underserved areas in District are not obtaining enough outside traffic, hurting the financial revenue of the area. POPout facilitates events that bring consumers into underserved communities.

How we built it

We used HTML & CSS with repl.it.

Challenges we ran into

  1. Coming up with an innovative idea!
  2. Figuring out how to display the mobile app prototype.

Accomplishments that we're proud of

  1. Clean and crisp front end design
  2. Unique PwerPoint presentation
  3. TeamWork and relationships developed

What we learned

  1. Diversity as a team makes the hackathon more well-rounded!
  2. We can not start with the technology, we must start with the problem that we would like to solve!

What's next for POPout

We hope to expand to the greater DMV areas. We hope to include more dynamic content in our product as we expand!
