Inspiration

We are first year students at BCIT who wanted to try our hand at making something for the first time. This project is very basic but it was as fun experience.
This started from a joke between Tim and Ivan on friday. Tim always wants to eat food and more meats. Ivan joked about harvesting Tim's organs the Russian way. So we decided to make a game where Ivan is trying to make Tim eat healthy before harvesting.

What it does

Tim (whose mouth is at the bottom) wants to eat all meat and sweets, you are trying to

help him stay healthy. Shoot down the garbage food and let him have a good healthy vegetarian diet.

How I built it

This game requires the pygame module

With pygame installed, run game.py to open program

Press enter to start game

Use 'A' and 'D' to navigate left and right

Press space to shoot

Your score will be shown at the very end

Challenges I ran into

We were trying to learn how to use C# and unity, however, there was not enough time to get an accurate grasp of C# language and Unity/Visual studio We also could not properly compile the program into a single executable

Accomplishments that I'm proud of

We managed to make our first collaborative project on Github

What I learned

We gained a better understanding of Pygame, and how to make a game that utilizes collision detection of moving objects.

What's next for Heathydiet dream

We need to make it more graphical, and add nutritional values for each item shot down so it can be more educational.
