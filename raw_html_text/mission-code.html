Inspiration

Brandon Wang wanted a better way to connect non-profits to volunteers.

What it does

This service scans resumes and connects volunteers to non-profits.

How I built it

We built a small web app which could accept resumes in pdf form. We lift the text from these resumes and use NLP to match words from the resume to keywords given by the non-profit.

Challenges I ran into

Getting the node.js server to run was very difficult.

Accomplishments that I'm proud of

I'm really glad that we got the node.js server to work.

What I learned

This was most of our team's first hackathon. We've learned a lot about web-programming and how web-apps are designed.

What's next for Mission Code

Hopefully, we get user accounts set up for both non-profit organizations and volunteers.
