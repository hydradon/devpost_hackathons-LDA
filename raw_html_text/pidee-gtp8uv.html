Inspiration

Self sovereign identity is a blockchain supported identity. Blockchain is associated with big servers and huge amount of energy. We want to show that a blockchain application can run on a small micro computer like a raspberry pi.

A meaningful application would be to open a device

What it does

Demo of Jolocom's SSO credentials on the Raspberry Pi + IOTA payment. Once a user verfies their credentials and pays it triggers the launch of an application.

How we built it

Challenges we ran into

Getting the Rasberry Pi screen resolution set properly. Compiling on the Rasberry Pi. Running a webserver on it. Micro processor integration + hardware setup.

Accomplishments that we are proud of

Getting it working. Coding overnight + building the payment channels and neat design.

What We have learned

Raspberry Pi is a pain in the ass unless you're a total coding baller.

What's next for PiDee

Wide scale planetary destruction and micro payments.
