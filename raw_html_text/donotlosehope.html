Inspiration

An AI powered web application to connect the disconnected souls. For whatever reason, if you’ve lost contact with your dear one during a natural calamity or if a baby is missed, our application is here to help you. Thanks to the power of intelligent decision making systems which lets us get connected even in the toughest of times.

What it does

Search - is a functionality of the site let’s you find the information like phone number, address, email id etc if there is a potential match with the existing people in the database. We use the powerful constitutional neural networks based clarifAI API to find the closest match of the searched image in the database.

Report - is a feature through which a submission of abondoned or lonely individuals can be posted by someone who’s willing to help the lost person. On every submission of image, we train our Neural Network model so that it can identify if there’s a similar image posted through search at some later point in time in the application.
Use Cases - * Pet Finding * Disaster *Kidnap *Accident *Lost someone? -Use the search function.
*Found Someone lost? -Report the details.

How we built it

We've used Python Flask framework and SQL Alchemy in the back-end. Mainly used clarifai API to identify the matches in the model created with new reports submitted through the application. On the front-end, we've used HTML, CSS, Ajax, JavaScript, BootStrap to build this application.

Challenges we ran into

Using imgur api to create a public URL for the image uploaded.

Accomplishments that we're proud of

Seamless integration of multiple APIs like Clarifai, Twilio, imgur to make this application completely RESTful. Successfully training the concepts of the customized model in clarifai API.
