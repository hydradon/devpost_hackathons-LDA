Vision

The vision for this project is to create a farm management warehouse that can

  • Track crops planted by farmers in a certain amount of acres (which we refer to as blocks)
  • Keep a log of all crops that are swapped between blocks and do a by-field crop rotation analysis
  • Using yield data collected from different government organizations, produce a estimate of farmers' yield and a very rough estimate of profit based on by-country profit data
  • Allow farmers to input prior yield and revenue data and allow for anonymous analytic calculations that aggregate data and create metrics to compare one's own farm to
  • Coordinate across local areas to ensure soil quality stays high by crop field rotation analysis
  • Allow auditing per user to see who has added what when

What We've Gotten Done

  • Create a database that can track all of the data as needed above
  • Create a webpage to emulate the intended usage for a single user for rendering and removing entries
