Inspiration

Currently, there is so much access to data in the world. In past years, the problem for many researchers was that they didn't have easy access to data but nowadays it's the exact opposite. To help sort through the data in the world, we developed JMore.

What it does

JMore takes in an audio input and a reading level, and depending on what is spoken on the audio file (for example a physics lecture which is converted using speech to text) and the given reading level, JMore will return a list of links that are relevant to the audio.

How we built it

We built it largely on python and flask.

Challenges we ran into

None of us have used flask before and it was really difficult to understand.

Accomplishments that we're proud of

We are proud that we got it all working in the end.

What we learned

We learned that flask is a versatile tool and are more inclined to use it in the future.

What's next for JMore

Next, we are planning on improving the speech to text and the quality of the links that are shown to the user.
