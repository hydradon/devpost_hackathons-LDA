We were inspired by the goal of Mission Services of Hamilton to create a social impact app that features programs providing a mix of after-school activities, learning opportunities, food programs, mentoring and more to kids from low-income homes. For further this goal, we created an API that provides the set of tools necessary for this endeavor. We are proud of our resilience in the face of endless bugs and have learned so much about the application development process.
