Inspiration

We are willing to contribute to society by making a charity app using the Nessie API provided Capital One!

What it does

It is capable of having your back account attached and used towards donating money to various organizations. The exceptional aspect of CharitalOne is its ability to easily make donations within the chosen period.

How we built it

We were successfully utilizing Nessie API, and React Native framework for JavaScript.

Challenges we ran into

The hardship of getting and sending the requests, learning how to use React Native, and designing.

Accomplishments that we're proud of

We finished the project on time, and we fully comprehended the usage of Nessie API, and found it very useful.

What we learned

We learned a lot about React Native android development.
