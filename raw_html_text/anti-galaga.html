Inspiration

We love the old school pixel style and wanted to rebuild our childhood game. We decided to design a twist from Galaga, where one controls a spacecraft at the bottom of the screen battles against enemy aliens arriving in formation at the beginning of a stage in a battle of the ages.

What it does

Instead of controlling only one spacecraft, user is asked to make a strategy of releasing enemy aliens to destroy the spacecraft and obtain points. There are seven kinds of enemy aliens with different formations and abilities. All of them have maximum available number and need time to be charged. The game ends when time runs out. Moreover, we made the game can be played on both computer and mobile browsers.

How we built it

Built the game with JavaScript and HTML.

Challenges we ran into

Difficulty in calibration of the balance of the game.

Accomplishments that we're proud of

We did not use any outside references to come up with this idea nor implement it.

What we learned

No matter how difficult it is, a game can be built using just JavaScript and plain HTML.

What's next for Anti-Galaga

We are going to add more modes of playing on it.
