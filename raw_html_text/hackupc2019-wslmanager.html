HackUpc2019-WSLManager

HackUPC 2019 Project, It's a WSL Instance manager.

I use WSL at work, I've made a tool called WSLD in the past, that facilitates the import/export of images from docker to wsl and viceversa. Now, I've made this tool that allows to control all instances, and in the future, I'll join both.

All the controls appear as a context menu.

I had to learn XAML/C# User Interface.
