• MusicVerse

Inspiration

Inspired by MagicVerse, the world will be painted by data. If we can create our own music by interacting with our physical space, that would be great.

What it does

You can Jump, Nod, Shake Head, Walk and Tap to create/remix your music in the space and use Bose AR Frame Glasses to hear your music.

How I built it

Using Unity+Bose API

Challenges I ran into

This is my first time using "Music" as the main concept to create an app. Visual will be easier for me since I am a designer. I am not familiar with the "sound" interface design.

Accomplishments that I'm proud of

I am a designer and I only have basic coding Unity knowledge. With Bose Team's tech help and friendly developing tool in Unity, I was able to develop this project on my own.

What I learned

1) Spatial music interface design 2) First time using Bose AR Frame 3) First time code an app from scratch (I used to watch tutorials and change the assets and parameters - my skillsets)

What's next for MusicVerse

Create the actual music world with Bose AR Frame
