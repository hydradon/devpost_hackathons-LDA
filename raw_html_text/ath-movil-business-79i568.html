Inspiration

Requesting money is an efficient and polite way of asking for money you're owed. Whether it's for a job you did or an event you organized. Use ATH Móvil Business to send a payment requests to any client using ATH Móvil.

What it does

ATH Móvil Business will allow any business request payments to their clients using their phone number. Clients will receive a notification with the requested payment information.

How we built it

This mobile app was built using Android for the front-end development, and a NodeJS server with a MongoDB database in the back-end. Authentication and payments are made using the ATH Móvil API.

Challenges we ran into

There were a lot of compatibility issues between the some of the libraries we wanted to use.

Accomplishments that we're proud of

Building a functional API that allows business to request payments and track clients and transactions from a mobile app. Businesses are also able to view comprehensive reports about their client activities.

What we learned

How to develop complex graphs using the Android native language and new technologies that allowed us to optimize the overall functionalities of our application.

What's next for ATH Móvil Business

Making the ATH Móvil App fully compatible with ATH Móvil Business to allow better user interaction between both apps and making the whole experience of paying with ATH Móvil even more seamless.
