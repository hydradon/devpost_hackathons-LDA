Inspiration

Your child has a rash and a fever. She won’t stop crying. You try to Google her symptoms but your phone’s Internet is extremely slow. She keeps crying. Page won’t load.

Heal provides quick, easy to access medical advice. Send your symptoms and a doctor will reply with a diagnosis. All it requires is a phone that can text. No need for data. No need for Wifi access. No need to wait on hold for an available nurse or yell over the crying of your child.

Heal can be life changing for those who have medical questions, but don’t have Internet access and can’t travel to a health center. Our @help prompt is especially useful for those who are hard of hearing but need to make emergency calls.

What it does

Heal is aimed at trying to connect users with a doctor on-demand using Twilio, Prompt, and Wolfram. The user will send prompt a text message such as "@health I'm having really bad headaches," and a text message will immediately be sent to an available doctor. The doctor will then be connected with the user and will be able to send back a text message responding to the client's ailment. Using the Wolfram technology, we run through the doctor's response and pull the definitions for medical terms and send that as a another text to provide the user with more information.

Challenges we ran into:

The major challenge we ran into was using Prompt’s API and connecting it with webhooks.

Accomplishments we're proud of:

We’re really proud of how we were able to incorporate multiple technologies into a sleek, easy-to-use product that addresses a real health care problem.

What we learned:

We learned a lot about using different APIs such as the Wolfram API, Twilio, and Prompt.

What's next:

We want to produce a suite of health-related prompts. Ideas include an @counsel prompt which will assist the user in connecting with a counselor for mental health counseling.
