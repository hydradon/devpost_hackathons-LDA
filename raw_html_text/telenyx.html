Inspiration

We were inspired by the Rogers challenge and the workshop by Wysdom.ai about the tech behind chatbots.

What it does

It simplifies the buying process by using customer's behavioural and transactional data through an intelligent voice enabled chatbot

How we built it

We utilised Google's Dialogflow service to build an intelligent chatbot. We built the front-end using React so that users can interact with the chatbot.

Challenges we ran into

We ran out of time.

Accomplishments that we're proud of

We went from zero to MVP in less than a day.

What we learned

We learnt how to implement AI, create a business plan, conversation design, Design thinking

What's next for Telenyx

We are excited to present our project. If given the opportunity, we can scale this idea to multiple platforms.
