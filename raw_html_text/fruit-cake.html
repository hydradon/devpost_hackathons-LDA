Inspiration - Lack of visualization tools on the SSC

What it does - Generates data visualizations from transaction history on the SSC

How we built it - With difficulty and a lot of stack overflow

Challenges we ran into - Integrating a web based frontend with a python backend

Accomplishments that we're proud of - Using many new libraries such as flask and successfully scraping the SSC transaction information into a local data structure and producing limited visualizations.

What we learned - Lots of Python (only few of us used Python before this project), working in a team, working with git, balancing code quality and speed in order to complete

What's next for Fruit cake - Generate more visualizations, clean up the codebase, new UI
