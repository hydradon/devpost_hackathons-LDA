Inspiration

In organizing any auction needs a lot of money so to save the money, this idea came in my mind to digitalize the auction platform. Help the organizations to save some money which can be invested in other processes.

What it does

It conducts the auction online, i.e it digitalizes the process of auction.

How I built it

It is built by me alone using MEAN stack.

Challenges I ran into

I faced difficulty in deploying the app but successful in finding the way to deploy on Heroku. The live demo found here - link

Accomplishments that I'm proud of

Successfully completed this project alone using MEAN stack.

What I learned

From this, I learned how to keep patience and debug the code and also how to deploy the MEAN stack web app.

What's next for Auctioneer

There are lots of features that will add in the future i,e:-

  1. Integrate payment gateway.
  2. Make it a PWA to use it as a native web app.
  3. Add real-time chat for solving any misconception during the auction with auction manager.
