Inspiration

We wanted to create a game for virtual reality that was fun and beneficial to the individual.

What it does

It's a fun and challenging rhythm game that also provides a workout.

How we built it

We used unity and Oculus SDK. It was programmed using C#.

Challenges we ran into

Learning how to create the algorithm to detect the rhythm of beat in any song.

Accomplishments that we're proud of

Creating an app that looks good and performs well.

What we learned

We learned how to use Oculus SDK to program virtual reality games.

What's next for Rythm Racket

Adding the option to search for any song to play and improving the visuals even further.
