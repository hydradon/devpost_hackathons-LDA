What it does

It has a student login, whereby a logged-in student can leave reviews or comments about various courses from school. If a course is not already on the site, students can add courses, by including the course code and course title, along with all of the course's instructors. Other features include a search bar that allows the user to narrow down his/her desired course, and the ability to edit/delete reviews that you've previously left about a course or prof.

This page is different from sites like ratemyprof.com because students comment on the course itself. All students currently taking a course can then look through the feedback for it and can compare different profs of the same course, all under a single page, rather than having a different page for each prof.

How we built it

We used the MEN stack for the development of the web, along with front-end tools like HTML, CSS, and Bootstrap.
