Inspiration

There is no other artist more focused on process/growth over end-result than Bob Ross!

What it does

Trains a Markov Chain using transcripts from Bob Ross's show, and then outputs text in his affect.

How we built it

We used python for this project

Challenges we ran into

We had to teach ourselves the fundamentals of Markov Chains, and we had to do some digging to find Bob Ross transcripts!

Accomplishments that we're proud of

His voice really shines through in the generated text

What we learned

Markov chains, more python syntax esp about dictionaries
