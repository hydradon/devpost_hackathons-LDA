Inspiration

It becomes difficult for managers to manage their tasks and task given by him to subordinates.

What it does

It is a task and team management app. It has live chat rooms for people to discuss about the tasks

How we built it

It is build using ruby on rails in the backend with socket.io for real-time communication and react and redux for the frontend.

Challenges we ran into

It was difficult integrating the real-time communication on the separate server

Accomplishments that we're proud of

I am proud the app is able to deliver it functionality and express and rails server are able to work properly.It has caching for improved performance .
