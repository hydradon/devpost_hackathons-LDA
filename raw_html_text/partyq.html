Inspiration

Small house parties!

What it does

A party host can create a Queue, which prarty-ers can up/down vote and suggest songs through a web interface or text message.

Prarty-ers that join receive a random giphy profile icon.

How we built it

PartyQ uses a React website, connected to a NodeJS Express server. The backed acts as a cache which connects to Spotify to manage a playlist. Text message input is handled through Twilio.

Challenges we ran into

The Spotify API has documentation issues, which required some network investigation to the Spotify Web Player.
