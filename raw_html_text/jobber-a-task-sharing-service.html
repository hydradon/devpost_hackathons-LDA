Inspiration

An idea for a local startup to serve the Ottawa community.

What it does

Jobber provides a platform in which people can interact with other people that are in need to have tasks done for them or want to do tasks.

How we built it

Leveraging our team's experience, we worked together in building a clean and modern web application with NodeJs, Firebase, and Bootstrap.

Challenges we ran into

Known issue with firebase on auth-state change

Accomplishments that we're proud of

Mostly functional and looks clean

What we learned

Sleep is very important and planning is key

What's next for Jobber: A Task Sharing Service

Hopefully bringing it to a point where it can be deployed. In addition adding a rating feature and in app messaging.
