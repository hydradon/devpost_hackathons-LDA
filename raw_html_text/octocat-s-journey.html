Inspiration

None of us had ever made a game or used Unity before, so no better time to try than a hackathon. We wanted to have a fun, code-related platformer, which is where the idea to make it about Octocat came from.

What it does

Octocat's Journey is a 2D platformer all about getting to the pull request at the end so he can merge it.

How we built it

We exclusively used Unity for the game engine, and used prefabricated sprites.

Challenges we ran into

Complete lack of knowledge of Unity.

Accomplishments that we're proud of

We were able to make a working game, which is way more than initially anticipated.

What we learned

Unity is a fantastic tool for game development, which is simultaneously harder and easier in many ways than we could have imagined.

What's next for Octocat's Journey

We would like to create more levels, add our own custom sprites, and add more attacks from Octocat.
