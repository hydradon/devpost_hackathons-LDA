CryptoBonds is a blockchain application that was built using Hyperledger Sawtooth. The goal of the project is to make a stable coin with its value linked to real life company bonds. CryptoBonds acts as a clearing firm and monitors transactions between the banks and the traders. Traders are able to exchange cryptobonds and other cryptocurrencies with other users.

CryptoBonds is developed by five people at HackIllinois 2019, with the help of various mentors.

Click here to access our front end web interface. Our Django project is deployed on Heroku.

Inspiration

Our Idea was to make a stable coin by using bonds as a way to create a financial application for a blockchain web app.

What it does

Our interface can act as trading, bond issuing, and clearing firm to issue bonds.

How we built it

It is built using Hyperledger Sawtooth. We had a lot of help from the mentors and used as many resources as we could.

Challenges we ran into

Getting everything set up and working was a big challenge. Specifically, setting up the python backend correctly was a bit of a challenge.

Accomplishments that we're proud of

We're very proud of having a product that is demo-able for a proof of concept.

What we learned

We learned a lot about blockchain and how to create an application around the idea.

What's next for Crypto Bonds

We have a lot that we could implement in the future but as it stands our idea is commercially viable with just some more time to fully build out the feature set.
