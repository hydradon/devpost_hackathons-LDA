Inspiration

We want to help people with reading disabilities. The primary focus now is the children at the school age. The common treatment for dyslexia is to read with the children, make them use fingers to read the lines word by word and to explain the syntax after each sentence.

What it does

It essentially does the work that teachers and parents usually do. The children can see the sentence sentiment, keywords and syntax by just clicking on a few buttons.

How we built it

We used Google API and Javascript to build the website.

Challenges we ran into

We have some difficulties finding free-copyright children e-books with interesting stories and abundant images.

Accomplishments that we're proud of

What we learned

Using crawler for raw data and data processing, using express as a bare-bone backend framework

What's next for Happy Reading

We are planning to connect with scholars in neuroscience and psychology and improve the functions of the website.
