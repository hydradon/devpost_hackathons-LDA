Inspiration

Idk, we joke a lot about who the saltiest person is.

What it does

Collect every post and the list of members using the groupme node module, save it to a text file with multiple tildes as delimiters sorted by author, parse it with python and conduct sentiment analysis using TextBlob, sum up and average out the scores, and sort authors by score.

How I built it

A little bit of help from friends, some Googling.

Challenges I ran into

This was my first time using JS async requests, that was painful at first, I ended up restarting the first draft.

Accomplishments that I'm proud of

Just being able to finish this and do something new.

What I learned

JS async functions, Text Blob

What's next for Salt-o-meter

Maybe making the code cleaner and better-documented. Maybe some more professional number-crunching than averaging it out. Sorry, not a stats major.
