Flierr

Flierr is a crowd-sourcing application where people can advertise (or discourage) people from going to a specific location. Users can then rate the event that was listed in order to share with other users if the listed event was real or not.

How to Run

Install Node 6.2.2 and sqlite3.

Steps to install and run:

  • npm install
  • node index.js
  • open http://localhost:3000/ in the browser

Known Issues

When creating an event, the event cannot be upvoted or downvoted unless the page is refreshed. Otherwise, the server will crash and will have to be restarted.
