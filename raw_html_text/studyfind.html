Inspiration

We wanted to make it easier to study for finals, especially for harder classes such as AP or Honors classes

What it does

Our app makes it easier to collaborate and study with others by sharing notes for specific subjects, making the concept easier to understand.

How we built it

We built two mobile apps, one for IOS and one for Android.

Challenges we ran into

Learning how to use UI of Tableview controller and of webview controller. Learning how to use Android Studio and creating Android apps, and learning xml

Accomplishments that we're proud of

We're able to allow users to access the internet through our app using UIWebView in iOS.

What we learned

We learned how to create android and ios apps and how to streamline ideas into one project that is functional.

What's next for StudyFind

We plan to allow users to upload pictures, and have the application run havenondemand OCR API to allow users not only to access notes but also allow them create them through our application. Furthermore, we would like to develop an algorithm which allows users to determine if the source of their notes if credible.
