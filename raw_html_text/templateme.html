TemplateMe!

Effortlessly change JIRA's email notifications and notification schemes!

Create personalized dynamic notification templates with the versatility of Apace Velocity. Assign different notification schemes to user types, issue types or projects through our add-on configuration options.

What it is and what it's for

Replaces JIRA's standard email notifications (created issue, new comment,...) with your own customized templates. These templates can be assigned to a issue type, user type (with or without JIRA license, external user) or project.

What are the benefits and advantages?

  • Very versatile configuration: create templates for different user types, issue types or project
  • User friendly interface: Respecting JIRA's screen schemes so the user is comfortable when configuring our add-on.
  • Easy switch: by clicking one button you can decide to use JIRA's default notifications or TemplateMe! customized emails notifications.
  • Dynamic templates: Using Apache Velocity language you can create dynamic templates to include issue details and other variables provided by JIRA
  • No restart needed: You can change templates as often as you like. No JIRA restart is needed. This is not the case when changing JIRA's built-in templates.
