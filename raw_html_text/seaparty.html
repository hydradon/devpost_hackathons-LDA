Inspiration

We took inspiration from the party game, Astro Party, but changed the theme and the many of the game mechanics

What it does

Sea Party is a fun party game

How we built it

Using a pixel editor for image editing, and Processing with java for the game

Challenges we ran into

Attaching physics to the existing logic made it harder to read/work with.

Accomplishments that we're proud of

Finishing in time and making a playable game

What we learned

How to use box2d, how to edit images/make pixel art, and how to make games with Processing

What's next for SeaParty

Android support, more power-ups, ui upgrade, menu, background change, more maps, more arena objects, etc..
