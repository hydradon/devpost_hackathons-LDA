HackPoly-2016

An eye tracking program that was implemented in c++ and OpenCV. This program was hard to work on as the theory involves a lot of matrices operations and image processing. Qt interface is currently in progress. Thanks to trishume and their webcam gazetracker which already had an existing pupil tracking implementation.
