Inspiration

We were inspired with the problems we all personally faced with accidental food waste from forgetting about food in the fridge and letting it expire or being unaware of upcoming expiration dates.

What it does

This project allows users to keep track of the food they purchase. The purpose of the application is to send notifications to the user when their food is about to expire or needs to be eaten soon and also provide them with a place to keep track of all the food in their fridge.

How we built it

We built the webpage using Adobe Edge (for the animation intro), HTML, CSS, Javascript, and Jquery.

Challenges we ran into

We ran into many challenges. We wanted to make this a mobile application, but since we were unfamiliar with making a mobile application, we decided to start with a website that was well-formatted for mobile devices. It was difficult to do the coding and we weren't able to accomplish all the things we wanted, but in the end we achieved our short-term goals.

Accomplishments that we're proud of

We are proud of the aesthetics of our project. We all also learned some skills we were previously unfamiliar with.

What we learned

We learned a lot about coding and the basics of coding. We also learned easier ways to approach coding problems that were more efficient and saved time.

What's next for Fresh 4 U

We would like to develop this project into a mobile application. We hope in the future it will be more user friendly. Some examples include allowing the user to scan bar codes or use a camera to identify products instead of having to manually type in the food names and expiration dates.
