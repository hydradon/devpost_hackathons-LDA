Inspiration

We wanted to advocate for women in technology by creating a story about a girl, Belle, who had to go through a lot of ups and downs but overall is able to overcome adversity.

What it does

Twitter data sets that were received from Bell are applied to a machine learning model to determine which of the tweets contained positive and negative information, and produces a graph output.

Accomplishments that we're proud of

Some of us are new at coding so just producing something was pretty great. We also were able to produce a machine learning model to apply to the data we were provided by Bell which was a huge accomplishment. Learning html and using bootstrap was big too!

What we learned

We learned how to produce and apply a machine learning model, as well as how to use bootstrap and html
