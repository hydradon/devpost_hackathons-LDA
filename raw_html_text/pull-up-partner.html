Pull-up partner uses Python, Tkinter, Raspberry-Pi, and Arduino to add smart sensors and programming to your pull-up workout. Users can chose from 6 different workout modes such as rep counter, hang timer, and our favorite, the "GET SWOLL" game.

The biggest challenge we encountered while creating pull-up partner was scaling the output voltage of the load cell that we planned to use to measure the weight and force applied by the user. There wasn't enough time to mess around with the differential amplification circuit in order to achieve desired output.

The "GET SWOLL" game is a side-scroller game in which the player needs to jump over obstacles to survive. This fun game is integrated with exercising because player controls his/her sprite's jumping by doing a pull-up, which is detected by the ultrasonic sensor.
