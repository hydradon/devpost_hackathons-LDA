swatchit

Introduction

Swatchit solves the problem of your aesthetic. It goes through your Instagram feed and fetches the color data to generate a color palette. The project uses Node.js on the backend, and HTML and CSS on the frontend.

Swatch it was inspired by Swatch.py, a Python program developed by us at the same time, that parses images and videos for color data to generate an overall color palette as well. This program also runs standalone.
