Summary: Our project aims to streamline the token creation process, building on Wyoming house bill 70. It seems like more and more crypto currencies are coming out daily, and we feel like this shouldn’t be left only to those capable to write their own code. Wyotoken is a token creation utility that will allow anyone that wants to create their own tokens to simply go to our application ( https://wyotoken.herokuapp.com/ ), and enter a few key pieces of information about the tokens, i.e. token name, the organization url, what the tokens value will be set at. From there, our application will create and validate the new tokens and send them to the purchaser for distribution.

Applications: This easy create token process will allow anyone to create and distribute their very own crypto currencies. (i.e. spend tokens to enter UW football games, distribute tokens for movie theatres, use tokens at hospitals for healthcare, either from an insurance standpoint or the hospital itself)
