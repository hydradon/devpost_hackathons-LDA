Inspiration

After a long string of hackathon projects devoted to my new programming language, I decided to go back to game development.

What it does

Last Man Flying is a first person aerial combat game in which you are the last soldier of a fleet that has all but been wiped out. But instead of dying in vain, you take to the skies for revenge.

How I built it

Last Man Flying is built using my own game engine: Kaeon ACE Ascent.

Challenges I ran into

There are several more things I wanted to implement, but overall I'm satisfied with how it turned out.

Accomplishments that I'm proud of

A lot of people seemed to like the game.

What's next for Last Man Flying
