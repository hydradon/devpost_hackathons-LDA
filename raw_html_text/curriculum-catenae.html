Inspiration

We were inspired by the real problem of fraudulent CV entries and BlockChain could solve this problem

What it does

It allows users to create Block-Chain powered verified CVs and HR professionals search for verified experts.

How we built it

Using Kotlin for a web-app, Hyperledger Framework - Sawtooth Blockchain for Business and Vuforia Unity as an extra Augmented Reality feature.

Challenges we ran into

Platform choice for Blockchain solution (see pictures)

Accomplishments that we're proud of

Somehow it works as a whole...

What we learned

We were completely new to Kotlin-powered native webapp development and also learned a lot about the Hyperledger Framework

We shall see! But we will definitely incorporate it in our further projects
