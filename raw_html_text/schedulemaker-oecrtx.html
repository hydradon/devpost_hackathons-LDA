Inspiration

All the students who wanted to keep themselves organized couldn't find an easier way to export their University schedules. Hence, I came up with a simple, easy-to-use Chrome Extension to export their University Schedule into an iCal (iCalendar) format which can be imported into almost any popular Calendar service.

What it does

It exports University Class Schedules of a student into an iCalendar format.

How I built it

I used my previous Javascript experience and learned more Chrome Extension APIs to be able to make this Chrome Extension.

Challenges I ran into

At some points, it was hard to debug the Chrome Extension API errors I was getting, but through a few more tutorials and sample code blocks, I learned how to make this extension finally work!

Accomplishments that I'm proud of

This is the first time I've made a Chrome Extension!

What I learned

I learned that before I start implementing, it's good to practice with some sample code, get the hang of it, and then implement my real project with at least some usefule background knowledge.

What's next for ScheduleMaker

I will be expanding my Chrome Extension to work with any University's Website powered by Schedule Builder, a platform used by numerous Universities to help students schedule their classes with more flexibility in timings.
