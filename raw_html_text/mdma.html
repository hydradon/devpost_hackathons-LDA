Inspiration

My team is really into building mesh network projects.

What it does

A set of nodes talk to each other, and then to a base station, to create comprehensive metrics on variables like temperature, wind velocity, atmospheric pressure, soil ph, humidity, motion, light and others to create an understanding of the environment you deploy the network in.

How we built it

We mostly built it using meteor.js and pinoccio boards (Arduino Megas). We also used some playdoh, a lot of solder, and some wiffle balls from walmart.

Challenges we ran into

Play-doh is apparently conductive.

Accomplishments that we're proud of

The hardware actually mostly works after only 24 hours.
