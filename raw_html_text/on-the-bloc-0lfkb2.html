Inspiration

Born out of the necessity for open honest discussions on all of the most relevant issues. On The Bloc looks to revolutionize the way we connect and communicate using decentralized block-chain technology.

What it does

Allows groups of people to connect in a decentralized way with no censorship.

How we built it

Using a react frontend and a custom spring-boot and mongo backend while using Blockstack for authentication.

Challenges we ran into

Determining how we wanted to store data. Blockstack Gaia was just too restrictive for our needs

Accomplishments that we're proud of

Able to build and deploy a fullstack web application using blockstack in less than 36 hours.

What we learned

How to authenticate with blockstack. How to integrate new technology with existing technologies.

What's next for On The Bloc

Fully build out communication functionality.
