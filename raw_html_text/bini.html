Inspiration

Global Warming is one of the main problems that we are facing at the moment. If this problem is not handled as early as possible, it will be the downfall of all of us in the future. Therefore, we, as the Bini Team, are here to help prepare the future generation to be more aware of how they can contribute by educating recycling.

What it does

Bini Helps toddlers to learn how to recycle in a fun and gasified way. Children will be able to snap a picture of a garbage in an application and then Bini will detect whether the object is recyclable or not using Machine Learning. Moreover, We also provide the game section where our target audience, children, can learn if the object shown in the picture is recyclable.

How we built it

We built the backend using Nodejs and Python and the front-end with Android. For design part, we used adobe illustrator and photoshop for design elements, and built with a sketch.

Challenges we ran into

Uploading Images from Android to the server is pretty challenging.

Accomplishments that we're proud of

We are able to help children aware of recycling in order to make a sustainable society.

What we learned

How to collaborate with people with a diverse background. Learn how to communicate between engineers and designers.

What's next for BINI

A more personalized profile feature where the user can keep their knowledge metrics.
