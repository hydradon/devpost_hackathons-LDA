Inspiration

Communicate through sound

What it does

Posting 15 second audio messages

How I built it

We iterated with Facebook Messenger API to fetch the voice recordings and they posted the recording to the react web app.

Challenges I ran into

-No JS SDK for Facebook -Time crunch - difficulty implementing full set of features within the given time frame of < 9 hours -Integrating Google Speech-to-Text API - the audio file format had to be in linear64 encoding, thus the MP4 file (default format for Messenger app) had to be converted first -Thinking of a good idea for the hackathon

Accomplishments that I'm proud of

We are proud of getting something working.

What I learned

I learned how to use Javascript, React, mongoDB, API's, and github.

What's next for blindBook

Use speech to text recognition and furthermore use the text to generate images for the audio files
