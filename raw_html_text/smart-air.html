Inspiration

Google Cloud challenge, Southwest Airlines Hospitality Challenge, Frustration with cabin crew experiences lacking efficiency

What it does

Provides all functions of the cabin crew at the user's fingertips so that they can interact with a smart chatbot as well as ask for services from a menu

How we built it

Utilized DialogFlow by Google Cloud to train an AI agent for the chatbot, JavaFX for the GUI interface and SQL and java for the back end.

Challenges we ran into

Understanding and implementing the AI from DialogFlow API

Accomplishments that we're proud of

The intelligent Chatbot

What we learned

How to read API documentation and implement them

What's next for Smart Air

Extending the features of the AI and beautifying the GUI
