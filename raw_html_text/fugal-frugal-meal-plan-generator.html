Inspiration

Wanted to use an application that would automatically generate meal plans for a set budget.

What it does

Automatically generates meal plans based off of caloric goals, macronutrient goals, and budget.

How we built it

Backend powered by the Google Cloud App Engine, Python, Flask, and SQL. Frontend powered by Bootstrap, HTML, SASS.

Challenges we ran into

The biggest challenge was devising an algorithm that would optimize the choice of foods for their nutritional benefits while maintaining the total cost under budget.

Accomplishments that we're proud of

Establishing an online web platform, designing a beautiful website, and creating a well structured MySQL database.

What we learned

How to use Flask, Google Cloud App Engine, Google Cloud SQL, construct databases, and Jinja to make HTML templates.

What's next for Fugal (Frugal Meal Plan Generator)

Add recipe recommendations, food allergies, web crawler to retrieve more accurate geospatial pricing data, and create a responsive UI.
