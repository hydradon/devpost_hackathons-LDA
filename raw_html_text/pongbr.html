Inspiration

Our group is often motivated by all types of video games. So when we thought of a video game that could utilize 5g, we thought of a multiplayer game using websockets. One that we hadn't seen and would be really fun is Pong Battle Royale

What it does

Pong is a player vs player pong battle royale game. You can play a game of pong against a friend, a group of friends, or 99 strangers.

How we built it

Websockets, webGL, Unity, C#, nodejs, and javascript

Challenges we ran into

Networking with Unity. Drawing the game for each player join and elimination. Finishing a website for the project in time.

Accomplishments that we're proud of

Working demo, Reliable server with websocket logic.

What we learned

We learned a lot of about networking for video games. A lot about websockets and incompatibility with other polling like libraries like socket.io

What's next for pongbr

Polish the game and release it for multiple platforms. Complete the website for the project.
