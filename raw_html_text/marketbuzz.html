MarketBUZZ - what does it do?

1. Analyzes the latest major news sources and identifies key phrases by associating the article content to additional info on the web.

2. Analyzes the images provided by the articles by categories, facial recognition, description.

Try it out!

Link : [MARKETBUZZ] http://marketbuzz.s3-website-us-east-1.amazonaws.com/

OR.

  1. Clone the repo
  2. In the project directory, npm install . This will install the npm packages for both the Angular client and the Node API.
  3. Open another terminal in the same directory and run the node api : node app.js
  4. Open another terminal in the same directory and serve the angular client locally (port 4200 by default) : ng serve

Angular CLI version 6.0.8. Node 8.11 +
