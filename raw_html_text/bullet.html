Inspiration

Wanted to build a app that promotes social interaction between people while also getting people to move around.

What it does

Allows users to interact through placing a pinned note at a geographic markers.

How I built it

Taking advantage of the APIs provided through Google and combining different libraries to come up with an ideal

Challenges I ran into

FInding a balance between complexity and simplicity in order to get the project done in time.

Accomplishments that I'm proud of

Staying focused even though sometimes it was easy to put everything.off. Persevering through this project even though sometimes it was easy to get frustrated and give up.

What I learned

Learned a lot about Android Studio than before and also about pulling location and interacting directly with a server.

What's next for Bullet

We have thought of many others features to implement for bullet which include a time expiry for pins, a count for the number of people who have visited a pin, number of pins visited by each user, and a map of nearby pins.
