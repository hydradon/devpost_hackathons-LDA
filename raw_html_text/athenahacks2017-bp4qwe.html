Inspiration

We wanted to develop Hungree because oftentimes people have a difficult time deciding what to eat. While Yelp gives reviews for a particular restaurant, it does not explicitly rank the food items on that restaurant.

What it does

Hungree helps users discover good dishes near them. Once the user logs in, they are presented with "buzzing" dishes, or dishes that are popular (have a high up vote to number of reviews ratio). The user can also select a particular restaurant that they may want to go to and see what dishes are popular at that restaurant.

How we built it

We primarily used HTML and CSS to develop our app, with some Javascript as well. We used the Microsoft Azure database to host our images.

Challenges we ran into

Using the Facebook button API, linking our HTML to Javascript, and layout.

Accomplishments that we're proud of

We were able to design a nice user interface and utilize Azure.

What we learned

HTML, CSS, and simple Javascript.

What's next for Hungree

Hopefully we are able to get this website fully functional, with location services to find the restaurants and dishes near you and all of our menus stored on the Azure data base.
