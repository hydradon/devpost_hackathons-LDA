Inspiration

Currently, the process of donating to charities in person is outdated. We've seen this through fundraising with Penn State's Dance Marathon. Canning, which is where one fundraises outside of stores and on the side of streets with a can asking for cash donations, gave us light on the issue.

What it does

Processing donations in Cannr streamlines this process and allows for easier fundraising. It uses QR codes and electronic payments to provide a better experience for both donors and fundraisers.

How we built it

We built an iOS app using Swift with a Firebase backend and utilized Braintree to process payments. The UI was prototyped in Sketch and transferred to Xcode.

Challenges we ran into

Problems with the backend coordinating with Swift; transferring of UI to Xcode.

Accomplishments that we're proud of

Getting payments to work, and creating a UI that flows and provides a good user experience.

What we learned

About using a backend with Swift and about designing and developing mobile apps.

What's next for Cannr

More development, redesign, and improvement of details!
