What's next for Lifelineonvideo

LOV is now available in both APP Store and Google Play. The app is designed to be confidential using a secure videoconference with a counselor. It requires only state, city or zipcode. What's special about this app is that it can be used for many other personal crisis'. From depression, veterans PTSD, eating disorders, suicide, and bullying. The website is www.lifelineonvideo.com
