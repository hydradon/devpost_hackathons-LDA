Inspiration

Using new hardware and our limited set of skills as first years to create a simple arcade game.

What it does

Using hand motions, you can select the game you want to play and control the snake / paddle depending on the game.

How we built it

Using Processing and the Leap Motion Controller libraries.

Challenges we ran into

Integrating the Leap Motion Controller libraries Calibrating the sensors to successfully control the snake.

Accomplishments that we're proud of

A (semi-) working application.

What we learned

How to use the Leap Motion Controller libraries. How to use APIs. Creating and animating complex objects.

What's next for Arcade Game using Leap Motion Controller

Using the Leap Motion Controller for other applications
