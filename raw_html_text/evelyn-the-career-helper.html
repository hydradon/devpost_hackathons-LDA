Inspiration

Artificial Intelligence Cognitive Computing Client to Server architecture

What it does

Career Assist Personal Assistant which helps the user build their career by recommending jobs, videos, articles and skills needed to upscale the user. The Cognitive chatbot suggests career improvement tips to users, recommends books, videos and magazines for the user.

User Interaction -> Evelyn Identifies skillset -> Evelys makes intuitive recommendations.

How we built it

Interactive Chat user interface powered by NodeJs and Java Android. Web App and Android Application connects to IBM Watson, preprocesses users' requests via Watson Assistant API.

Challenges we ran into

Connecting Android UI to the Watson Assistant API. Validating user request and returning the result back to the user via the web app.

Accomplishments that we're proud of

The team is proud of the successful integration The project is continuous, can be amended to include more use cases, and functions.

What we learned

Java, JavaScript, Rest, NodeJs, HTML, CSS, lucid chart, Watson Assistant, AI chatbot implementation.

What's next for Evelyn - The Career Helper

Implementation of more career building APIs. Improving GUI Database implementation of past connversation and data retrieval
