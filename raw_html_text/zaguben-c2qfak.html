Inspiration

I was out hiking when our car got 2 flat tyres. To add to the situation it was -30 C outside and we didnt have internet connectivity to find our route. It would been weird to stop a car, ask them for directions and then try and hitchhike.

The catch was that we still had network connectivity which we couldnt use to find our way. We suffered for hours outside in the cold when our destination was a walking distance away.

What it does

How we built it

Our app uses your smartphone's built-in GPS to send you directions when you are lost or in distress--especially when you're without data/Wifi. Using your phone's cell network, we can direct you regardless of your location.

Challenges we ran into

Getting phone's GPS coordinates.

Accomplishments that we're proud of

Combining the Twilio API with the Google Maps API to be able to transmit directions without internet connectivity and this can have real life applications for people in distress situations

What we learned

We're noob hackers so we learned everything using APIs to creating Android apps to using a server
