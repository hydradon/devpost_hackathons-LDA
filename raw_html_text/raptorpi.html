Inspiration

We were tired of the having to sit through roll call during every class and thought that we could improve the process of checking attendance.

What it does

Scans a student's QR Code and adds the info to a spreadsheet.

How we built it

Connected a USB webcam to a Raspberry Pi and then intergrated Google's Spreadsheet Api

Challenges we ran into

Writing the python code that would ultimately capture the QR Code information and write it to our spreadsheet.

Accomplishments that we're proud of

Before we started this project we didn't have much knowledge about Python and now we have a better understanding of the language.

What we learned

We learned how to program a QR Code reader, write programs in Python and how to bring different platforms together.
