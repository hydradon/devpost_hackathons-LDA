Inspiration

The inspiration was to create a revolutionary way to predict the future of companies and provide cutting-edge analytics tools to modern investors and stockholders.

What it does

Displays graphical information based on recent social media mentions.

How we built it

Flask backend that used sentiment detection services on online articles and tweets.

Challenges we ran into

API for social media and crawling to find news articles was difficult.

Accomplishments that we're proud of

The final, sleek design and easy usability.

What we learned

With a great project, it is never complete. There is still a lot of potential for Mentionr that we would like to develop.

What's next for Mentionr

More social media being aggregated, better analytics tools and a full-fledged statistical analysis environment.
