Inspiration

I started to implement the routing for a Microsoft HoloLens based navigation project. The other person involved experienced setbacks, and decided to pivot. I had no other ideas for this hackathon so I kept working.

What it does

It runs Dijkstra's Algorithm against a collection of Unity GameObjects, and draws a path between them.

How I built it

I kept marching slowly forward, through the pain and the tears.

Challenges I ran into

I encountered a malfunctioning build environment, sleep deprivation, and a whole set of non-cooperative vendor provided libraries.

Accomplishments that I'm proud of

I wrote code I can reuse in the future.

What I learned

Have an idea before arriving at a hackathon. Always keep in mind the complexity of the tools and platform you are building on.

What's next for Incomplete HoloLens Navigation Experiment

I see myself reusing parts of this project's implementation future Unity based projects.
