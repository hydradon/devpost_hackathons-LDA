Inspiration

We were simply trying to come up with an idea for a short interactive game that would be both enjoyable to play and keeping the gamer wanting more

What it does

You start at the homescreen where you are able to choose amongst three garden gnomes, you are then brought to the instructions page where you will learn how to play. Once you are ready for the task, shoot those darn rabbits! You start with four lives and once four rabbits get to your crops, the game is over.

How we built it

We simply imported the pygame library and used it's functions to implement the code we needed.

Challenges we ran into

There were a few methods we did not quite understand right away and had to stay up all night looking them over. Once we figured those tricky methods we were rolling again and the rest was smooth sailing

Accomplishments that we're proud of

The finished product of course!

What we learned

Pygame is a fun and easy library to create simple code for a complex game.

What's next for Garden Defense

We plan on creating it's sequal at the next hackisu, so stayed tuned for more fun and exciting updates!
