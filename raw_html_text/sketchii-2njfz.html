Inspiration

We were inspired by all of the web applications that we use in our day-to-day lives.

How it works

Sketchii uses node.js and sockets to allow you to draw with your friends in your browser.

Challenges we ran into

Getting the collaborative portion to work was difficult.

Accomplishments that we're proud of

We were able to go from a very low knowledge of the tools that we worked with to a working product.

What we learned

We gained a better understanding of the web technologies that we used, such as JQuery and Bootstrap.

What's next for Sketchii

We plan on adding a login system, the ability to change the color that you draw with, and different drawing tools (e.g. a circle tool).
