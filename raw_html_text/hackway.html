Inspiration

I found the messenger bots rather interesting and i wanted to create something on messenger to help me keep up to date with my programming skills.

What it does

1.It allows to write and compile code anywhere, anytime on messenger platform.

2.Hackway has a set of curated programming questions which can be solved by writing code inside HackWay's editor and get immediate results of your code.

3.You can solve a set of multiple answer programming questions and get feedback right away.

4.If you get stuck on a question you can get instant help by sharing it with your friends or groups on messenger.

5.You can also challenge your friends with the same set of quizzes or programming questions you have solved and see whether they can beat you.

6.If you wanted to create a test for a group, say a group of students, you can do that on HackWay. Just generate a random quiz, send it to the group and get notified when anyone solves your quiz.

How I built it

I first created a django restful api backend, a react frontend for the web views and finally i connected them up on messenger.

Challenges I ran into

I ran into challenges when trying to deploy my react app to production. React Router wasn't just working for me.

Accomplishments that I'm proud of

As a programmer, i really enjoy solving the problems in HackWay.

What I learned

I learnt how to create restful apis, how to integrate Facebook login to a website and also create front end apps in react.

What's next for HackWay

I wanna make HackWay a full blown programming educational platform on messenger. Where i can have anyone interested in learning code taking a course and getting a final grade.
