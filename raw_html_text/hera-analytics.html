Inspiration

We have read and understood that many startup companies fail to achieve their goals because they don't select the right demographic place of customer.

What it does

Hera provides help in selecting the location suitable for a company by using analytics and machine learning technology. Said data includes parameters such as corporate tax rates, income distribution, age demographics, etc.

How we built it

MEAN Stack, AWS, Jupyter NoteBook, MicroStrategy, AWS Ubuntu VM

Challenges we ran into

Connecting the website to the remote AWS server, Connecting the machine learning model used in Jupyter Notebook to the NodeJs app.

Accomplishments that we're proud of

We were able to work together and produce a substantial product, despite our different backgrounds.Two of our members had never previously attended hackathons, and one is a chemistry biology major.

What we learned

Routing in nodejs (a model view controller framework), connecting a local website to a remote database, Visual Analytics in Micro Strategy, K-means clustering algorithm for multi variant data, sci-kit learn and pandas library, visualizing using matplotlib.
