• The Bot!

Inspiration

The problem we identified was that people at work are stressed, and no one is doing anything about it.

What it does

Checks on you on regular intervals for negative moods (eg. stressing out at work). CandyBot disrupts these counterproductive emotions by calling you with relaxing music, sending check-up texts, and dispense candy for you . this helps to break the chain of negative emotions and helps you relax.

How we built it

We used a RaspberryPi for the camera and the dispense mechanism. Inside the Pi we use Node.js to start the process of checking stress levels, reaching out to APIs and then use hardware components.

Challenges we ran into

Getting the right parts for the project. Stacking multiple RaspberryPi components. Getting the Node.js server running and connecting the hardware scripts.

Accomplishments that we're proud of

achieving all the goals we wanted for the project was the biggest accomplishment.

What we learned

Clarifai is a great way to introduce oneself to the ideas of machine learning, and it was easy! We also learned how to use hardware components on the raspberry pi such as a Servo, and camera.

What's next for CandyBot

Corporate wise it would be great (easy) to adapt this project as a product. (not using the raspberrypi)
