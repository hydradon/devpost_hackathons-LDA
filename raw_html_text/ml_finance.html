ML_Finance

Machine Learning Approach to Finance.

This program allows people to figure out if they should invest in a stock for a specific company. Using newsapi-python and RSS-to-JSON-API-python, we scrape data of recent articles from trustworthy news sources and filter out looking for the specific company. The program looks at the aggregated data of news articles for particular companies and calculates the sentiment of those articles using machine learning. Furthermore, it markets stock prices, petroleum oil prices, and global currency values into account. We then feed that data along with other relevant data into the sentiment analysis API to compute the projected company's value. Then, it will weigh those and inform the user about the company's projected value and whether or not they should invest in the company. We are using bootstrap and node.js to display the data on the front-end.
