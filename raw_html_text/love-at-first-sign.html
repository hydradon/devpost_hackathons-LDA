Where we started

Both of us have experience with programming, but we've never attempted web development and we decided to give it a try. We thought that Love at First Sign would be a fun and challenging way to begin.

What it does

Love at First Sign is a dating site that bases compatibility on astrological signs. Users enter their information and are offered potential partners based on their Zodiac and Chinese Zodiac signs. If they choose, they may find the next most compatible partner, simply by refreshing the page. Users are also added to the master database so that they may show up on others' pages (if they are compatible).

How we built it

The app was built using flask/python 3 (on Spyder), and designed using HTML (on Atom), CSS and Bootstrap. The database connected to the app was managed using SQLite

Challenges we ran into

We found it challenging to connect the database to the python file, and to work out the sign up page.

Accomplishments

This was our first time using flask, SQL, and HTML, so we were happy with the learning experience.
