• Bunny Tickle

Inspiration

We envisioned a kids skill that took advantage of custom recordings to make up the Bunny Tickle. We want kids to be delighted and engaged with the Bunny voice and lighthearted tickle game.

What it does

The Bunny Tickle skill will count to tickle time randomly at each turn. The user never knows when the Bunny will announce tickle-time. It's a surprise for the user and anyone playing along.

How I built it

I used javacript levering that SSML tags to implement custom voice-over recordings for each Alexa prompt.

Challenges I ran into

Getting the timing between SSML tags took some time to correctly adjust but the controls made available were easy to use.

Accomplishments that I'm proud of

I'm happy we were able to implement a skill almost entirely using fun voice-over that hopefully, our users find entertaining.

What I learned

I learned that however simple a skill might be the qa process takes twice as long as one might think.

What's next for Bunny Tickle

More tickle time for everyone!
