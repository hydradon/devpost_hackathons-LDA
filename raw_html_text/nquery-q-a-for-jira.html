What it does

Natural Language Query tool on JIRA dataset. Ask/Type any question related to your JIRA projects and nQuery will answer it.

How we built it

We built a NLP algorithm that parses the natural query and converts it into JIRA API calls to fetch the data. Bolted on top of our algorithm is an intelligent visualization layer that renders data in various charts,tabular,graphs etc.

Challenges we ran into

  • Understanding the different ways a user can ask same questions. English is ambiguous :)
  • Understanding different JIRA APIs to query data.

Accomplishments that we're proud of

  • Ability to understand user's question with 80% accuracy.

What we learned

  • AI is amazing, we can do a lot more cool things with JIRA dataset.

What's next for nQuery: Q/A for JIRA

  • Will launch as a Add-On in marketplace to help atlassian users discover their data in a cool way.
