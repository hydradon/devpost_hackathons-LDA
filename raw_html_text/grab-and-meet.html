Inspiration

When you studying in the library, a coffee would be great. What about when you too busy to spend time to go to the Starbucks and wait in the endless line? Let somebody also going there helps you!

What it does

Our program helps users to contact others through cell phone message. If one user send a message to a certain phone number and specify his or her needs, and if in the same period of time another user offers to help, they can contact each other through phone numbers.

How we built it

We first watch the demo from Twilio and then try to make a grab and meet platform to help people using text message. We go to the Twilio documentation website to find the tutorior and API description. Then we follow the instruction and implement the Twilio Messaging API using Java.

Challenges we ran into

How to use java web framework and how to link it to the Twilio server.

Accomplishments that we're proud of

A caching data structure to store information for both demand and supply side of help.

What we learned

The usage of Twilio API and how to development Java under Spark Java web framework.

What's next for Grab-and-meet

Bug fix and more functions.
