Inspiration

The problem of Asian Carp invasion.

What it does

A real-time monitoring system that helps field workers detect areas where Asian Carps may have a population bloom.

How we built it

We built the backend side using ASP.NET Web API, Android for the Mobile Application, and HTML and AngularJS for the Web Application

Challenges we ran into

We had problems in finalizing our ideas, we wanted to make sure that everyone understood the needs and the goals of the project. The main road block that we had was pulling from the data source as we could not really extract the data from the images.

Accomplishments that we're proud of

We were able to finish early and was able to finish the two apps way ahead of our designated deadline.

What we learned

That progress for the project will be fast as long as everyone in the group was able to fully understand the mechanics of the problem.
