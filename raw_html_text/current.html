Inspiration Friends making me late to class because of inability to choose outfits quickly.

What it does

This app makes choosing an outfit less stressful.

How we built it

We used HTML/CSS, JQuery, Javascript, Bootstrap and the a weather API.

Challenges we ran into

Saving data from forms so that user could put in preferences and the app would better know what to suggest.

Accomplishments that we're proud of

Successfully implementing the weather API.

What we learned

How to use Bootstrap and API's.

What's next for Current

Working on saving data from the forms so that the app can know where the user lives. We would also want to give the user the ability to import photos of their closet so that the suggestions would be tailored to clothes that the user has.
