Inspiration

We get this idea from when tackling a complicated question, we sometimes solve this question's solution building on prior people's work. However, only the last person who solved this question get all the money. We want a distributed system where you can easily get inpiration from other contributors and all people get a fairly amount of the prize.

What it does

This is a distributed system which make referencing other people's work more easily and the prize is more fairly ditributed by all the contributors with people who you collaborate and also previous work you referenced to solve this problem.

How we built it

We build it based on decentralized storage solution IPFS for the backend and solidity to do the blockchain part which operate certain functions to credit all contributors work.

Challenges we ran into

Solisity is a fairly new language which we don't have enough experience on and we need to put our mind into the smart contract and blockchain to create something inovative.

Accomplishments that we're proud of

We made the IPFS fully functioning and most required functions working in the solidity part.

What we learned

We learned how the smart contract and decentralized storage works.

What's next for SD-Hacks-Blockchain

This project is our first step into the world of the blockchain. This is an interesting field that we definitely want to work more on it and make it become our first side project on blockchain part.
