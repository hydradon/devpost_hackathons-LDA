Inspiration

Many of us try and plan our day out, but often fail because we procrastinate.

What it does

We programmed a webapp that allows you to have a fun way to plan your day or week out, you can do it with friends, family, or even use it at work. It allows you to create a competitive atmosphere, which makes people want to complete tasks and gain points.

How we built it

We used Meteor, with MongoDB, React, and javascript, and jsx. We all split up tasks into parts of specialty for different people.

Challenges we ran into

We had problems storing complex json objects in the database. We also had some trouble with layouts that were responsive across different devices.

Accomplishments that we're proud of

We all worked together, and learned a lot about css layouts. We also used Flootbits to create a live google docs like workspace to code together.

What we learned

We all learned to write our first webapp, as we all only have experience in developing in system languages.

What's next for Outlast

We want to integrate a react-native app so its more intuitive for the phone.
