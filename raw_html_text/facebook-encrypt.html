• Encrypted FB post

Inspiration

If our friends and family are allowed to spam us with baby photos and stuff we dont get, then we will spam them with encrypted posts that only we can understand!

What it does

It allows you to encrypt a facebook status. Only people with the chrome extension and the relevant key is allowed to decrypt and view the post.

How we built it

Using Javascript and Chrome extensions.

Challenges we ran into

Manipulating the Facebook html was a rather difficult and tedious thing to do. This is probably the biggest source of weird bugs and "Features™".

Accomplishments that we're proud of

We had all spend little to no time with javascript and none of us had made a chrome extension before this hackathon. So getting as far as we did was a big win for us.

What we learned

Javascript, javascript and more javascript. Also we learned the hard way that the planning (and sleeping) phase of a project is an essential thing!

What's next for Facebook Encrypt

Only the future will tell... Hopefully less "Features™".
