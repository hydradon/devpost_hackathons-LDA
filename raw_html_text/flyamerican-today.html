Inspiration

We wanted create an engaging experience at the gate for passengers which would encourage them to be early while incentivizing American Airlines flights and program

What it does

Provides the passenger with useful information about their flight and about American Airlines

How we built it

We built it using** American Airlines's Mock Engine API* , **Weather API , and **flightTracker API *

Challenges we ran into

Due to the number of API's we were using, parsing data and storing it locally during a session was proving to be difficult

Accomplishments that we're proud of

As we developed further into the application, we realized that we could add more and due to the modular design of the application, it was smooth and simple

What we learned

How to use AA's Mock Engine and provide a smooth pre-boarding experience for passengers

What's next for flyAmerican.today

We would like to see it being used at America Airlines's gates and terminals
