Narwhal Chat

Narwhal Chat is an app which uses RightMesh's API to automatically connect local users together within a single chat room even without a Wifi or cell-tower Connection. It can utilise the mesh network through a phone's independent WiFi hotspot and bluetooth features to connect multiple users in high density areas into an effective chat room for anonymous communication.

Current Features:

  • Username setting
  • Available list of active users
  • Time stamp of messages
  • Message history display
  • Allowed connection settings adjustment (Wifi-Direct, Wifi-Mesh, Bluetooth)
