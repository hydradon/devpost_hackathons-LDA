Inspiration

We were inspired by a mix of the Space Invaders demo and Fruit Ninja. We wanted to make a game that is fun and casual to play, while at the same time being easy to pick up and lightweight, so that it runs smoothly in the browser.

What it does

Viruses and anti-viruses try to get from the top of the screen to the bottom. Kill the viruses by clicking them before they manage to get to the other side. Don't click the anti-viruses.

How I built it

We wrote the bulk of the game code in Javascript, and designed the webpage with CSS and HTML.

Challenges I ran into

It was hard at first to get all the pages to load, especially since everyone in my team was unfamiliar with Javascript and HTML.

Accomplishments that I'm proud of

We are incredibly proud of persevering and finishing the project in such a short time.

What I learned

I learned Javascript, CSS and HTML and how well the 3 of them work together.

What's next for Internet Invaders

We hope to introduce more elements to the gameplay, such as more game elements, more variation in gameplay etc.
