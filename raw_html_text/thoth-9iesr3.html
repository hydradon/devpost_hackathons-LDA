Inspiration

I was in Toronto, searching for unknown address using my DATA. It was a bad day, it was pouring hard, and all of sudden data stopped. My life was over, I did not know how to get to the place anymore. THEN I came to hackthenorth, and I realized I was among many others who had same trouble, so I decided to make my hack to solve this problem.

What it does

Thoth users ask normal google searches such as address, direction, reviews, etc... to internet without having data network.

How we built it

Thoth, takes request from users using SMS and send it to node.js backend hosted on IBM bluemix using Twilio service. Our backend service is using IBM watson to make conversation natural.

Challenges we ran into

IBM watson API are well documented, but little hard to understand. Twilio API are well documented and easy to use, we invested most of our time to make our backend more intelligent.

Accomplishments that we're proud of

we have implemented app with three functions already running. Our prototype is ready with good UI and basic functionalities

What we learned

IBM watson, twilio, bluemix, android

What's next for Thoth

Planning to add more function in future and release on APP store.
