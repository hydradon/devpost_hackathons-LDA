Inspiration

My father is an amateur runner, and he likes to attend the races that are held in my city, he always asks me for the computer to check the new races that are there every week. Inspired by this I wanted to develop this Skill to ask to Alexa the races that there are in the week and to be able to have this information as fast as possible. I know it will help many people in my city (and country) interested in the running world.

What it does

Tell you incoming races.

How I built it

Using an Amazon Skill Template for Calendar, with Help of Amazon Lambda.

What I learned

How to read a .ics format.

What's next for Alexa Peru Running Skill

Promote the Skill in the running community.
