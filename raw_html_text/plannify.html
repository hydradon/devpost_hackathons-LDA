Inspiration

We had everyday problems about planning out our own lives. At times, it felt so difficult. While planning is an important skill, it is hard for manmy students to be perfect at it. We wanted to fix these problems using an easy-to-use program.

What it does

Our Chrome extension is an enhanced, simple calendar. While using it, seeing the activities, adding new events, and planning out schedules become very simple.

How we built it

We originally built it using an HTML, CSS, JS software. After that, we transferred it using more JavaScript to create a Chrome extension.

Challenges we ran into

We ran into errors while learning new ways of employing JavaScript, like extensions and JQuery. A lot of these methods were unfamiliar to us, and thus we ran into many problems. However, with a bit of teamwork, we managed to plow through those issues.

Accomplishments that we're proud of

We were proud that we were able to create an extension using various stuff that we were not familiar with. Being open to new ideas allowed us to create this amazing project.

What we learned

We learned new ways to implement JavaScript, as well as creating Chrome extensions. We managed to create a really good program through these learning opportunities.

What's next for Plannify

We want to add multiple users that can collaborate on events. Sending messages and reminders are coming up in the near future!
