Inspiration

After a few interactions with the employees working at the bank, we realized that one of the areas where the bank was losing money was identity fraud and decided to tackle it our way

What it does

The project throws a series of questions to the user when he/she signs up in order to build a personality profile for him/her. This profile stays unique to each user and is used in the future to authenticate the user in times of transactions with high stakes.

How we built it

Flask and Python

Challenges we ran into

Getting flask to run smoothly with our work

Accomplishments that we're proud of

Implementing the IPIP personality test

What we learned
