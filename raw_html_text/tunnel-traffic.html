Inspiration

I lived in a town where there was always traffic and I always wished there was a way for me to know which route is backed up

What it does

My project notifies residents when the Hsuehshan Tunnel is backed up. It sends notifications to residents when there is a backup at the tunnel.

How I built it

What makes my project unique is the technologies used. One of the biggest advantage my project has is the ability to recognize when a user is stopped for a period of time and when that user is moving. This allows for accurate readings on congestion.
