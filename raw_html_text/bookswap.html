Inspiration

The textbooks are expansive and college students are broke. Sometimes we ran into a situation where our professors give out homework on textbooks but we don't have them and we cannot get them delivered until the homework is due.

What it does

It is a platform for students to post their books for sell or swap and to find the books they need for different courses.

How we built it

We built the database using parse and we make this iOS app using swift. We also built an interactive website using NodeJS.

Challenges we ran into

Constraints across different iPhone devices. When building our website, data binding is a big problem.

Accomplishments that we're proud of

This app is neat and handy.

What we learned

What's next for BookSwap

  • Add an in-app message system for users to communicate.
  • Universal design across all platforms (Android, Windows, etc)
  • Auto-feed when the books you requested are available.
  • Users book swap history.
  • A recommender system for all kinds of low cost books.
