Readit

The project idea was created when team member josh saw videos on youtube that read Reddit posts through text to speech. We learn how to import libraries, basic Python skills, and text to speech. We built it by utilizing the PRAW library to get a Reddit post through a user-entered link. The program gets the title, author, post, and the top 5 comments. It makes them into a single string, and the gTTS library to create an mp3 file that has the Reddit post. Then playsound will play the mp3 file. For the GUI, we used Tkinter to create the interface.
