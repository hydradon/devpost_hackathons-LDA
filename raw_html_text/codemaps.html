Inspiration

We wanted to create a simple interface where people interested in computer science and programming to meet up and participate in the community.

What it does

Takes local tech gatherings from meetup.com and displays them onto a map. Also allows users to input their own events and place markers. Is a resource for people not traditionally involved within coding communities to get involved. Help women and people of color in coding get involved with communities.

How we built it

Used Android Studio and Java. We used several libraries ranging from google api to jsoup.

Challenges we ran into

This was all new technology for us, so we had to learn as we go.

Accomplishments that we're proud of

We're proud we ended up with a working demo and it was really cool to learn Android development.

What we learned

Android Studio, jsoup, xml, and google maps api

What's next for Codemaps

Learn how to create a server that allows all users to see the same map and build a community.
