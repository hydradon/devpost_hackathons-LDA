Inspiration

Our inspiration was Google Inbox and the way of tracking e-mails there.

What it does

Cameleon delivers a Revolutionary approach to managing issues in JIRA. Inspired by Google Material’s Design, the user interface delivers the sophistication and ease of use that can only be described as Brilliant. Based on Google Inbox, Cameleon offers a refreshingly unique approach to creating, searching, organizing and transitioning issues by keeping things extremely simple, with the emphasis on at-a-glance access to the most relevant issues and requests.

How I built it

All is based on Angular Material framework and Angular itself. Cameleon uses a public JIRA REST API.

Challenges I ran into

We run in many challenges so far. The biggest one is beta version for Angular Material framework.

Accomplishments that I'm proud of

I'm very proud of using ng in JIRA plugin as it is much better then jQuery.

What I learned

I've learned a lot about Angular.
