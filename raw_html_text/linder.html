Inspiration

Books are ridiculously expensive these days, and we all have books that we read once and tossed aside. I wanted to think of a way to allow a community to share their books with one another. Linder is like the sisterhood of the traveling pants--except for books.

What it does

Users maintain a sharing shelf--a list of books they would be willing to swap with someone in exchange for a different book. Browsing through their feed, users have the ability "swipe right" for books they'd want to read. Whenever two users are both interested in one another's books, we have a match.

How I built it

Using the parse api for all backend purposes, the front end employs the familiar tinder user interface for pairing book owners for swaps.
