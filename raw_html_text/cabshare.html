Inspiration

Taxi sharing problem in Morocco.

What it does

CabShare is a simple application which helps users to find and order a taxi. It makes possible sharing taxi with someone else, always showing the most accurate and personalized data just for you!

How we built it

Magda created graphics and design part, Patrick created the Android application based on location services and taxi sockets and Norbert made the backend part as well as servers management and music.

Challenges we ran onto

Time - we wasn't able to ship all the functionalities in given time. Concept - it was challenging to simulate all customer cases.

Accomplishments that we're proud of

We achieved the core functionality and the application is ready to be reviewed!

What we learned

We learned how to solve issues that our technologies create when working together!

What's next for CabShare

In the future we intend to allow users to choose which whom will they share taxi with. We also aim to make the concept of driver application much more functional.
