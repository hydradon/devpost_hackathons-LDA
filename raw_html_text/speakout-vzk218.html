Inspiration

We wanted a way for people to have their voices to be heard as well as being able to find other worthy causes to support.

What it does

We give users a way to see petitions near them or any where around the world, and join their causes.

How I built it

Once a user has signed up for an account, they will be able to create submissions or join a petition! We give user's a way through Here's api to see petitions near them or any where around the world. Using Docusign's api, signatures from other users can be collected and stored for future use.

Challenges I ran into

The hardest part of the project was learning how to manipulate the apis we used and get them to communicate with each other in a clean fashion.

What's next for SpeakOut

We wanted to include searching functionality so you can search petitions with specific tags instead of having to go through the map to find ones you are interested in. I would also like to be able to make it so you can click on the map to add a petition to a particular location.
