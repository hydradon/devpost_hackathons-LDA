Inspiration

Charity donations can be sporadic. What do you give to the homeless if you’re cashless?

What it does

Makes donating money an automatic, easy, everyday event. Makes receiving it cashless and universal. Our platform connects charities and the homeless to your everyday banking app allowing you to donate anytime or on a regular basis.

How we built it

We used React and a Material UI library called react-md to build a Progressive Web App to simulate a TD banking application. On the backend, we have a Java program that interacts with TD's API to get transaction information, and sync information with Firestore (new version of real-time database provided by Firebase). The Progressive Web App listen for new data from Firebase, and update the transaction information in banking app.

Challenges we ran into

The challenge is to figure out work flow of TD's API for pulling transaction data of a customer.

Accomplishments that we're proud of

Being able to create an app that is connected to a bank's API

What we learned

Building a progress web application

What's next for The Charitables

Integration with Presto machines so card balances can be checked. Integration with Interac's API
