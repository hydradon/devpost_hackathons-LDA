Inspiration

All of us are very indecisive about making plans and usually anything we decide fails horribly.

What it does

picks the midpoint between two independent parties and finds locations to meetup at based on your selected preference.

How we built it

used express.js and caffeine pills. .

Challenges we ran into

used the mlh starter template and ran into issues with sequalize. Also ran into issues with google maps api so we did something else completely. Had version 3.0 in the last 6 hours

Accomplishments that we're proud of

we got it work! UI was a little delayed but the functionality is there.

What we learned

we learned a lot about how google api works as well as routing with express.

What's next for FriendsToGo

possibly host it and add a nicer, maybe an actual map to display the meetup points.
