Inspiration

Have you ever played a video during lecture without muting your laptop volume? With this program, you can be sure to not embarrass yourself in class during these moments.

What it does

Compare the current SSID with the settings and set the volume to the preconfigured level

How we built it

We built the program with Python, interacting with the system command line to adjust the volume

Challenges we ran into

Platform compatibility(currently only works on Mac OS X), unable to accommodate multiple WiFi profiles, most importantly: unable to run automatically when WiFi is connected

What we learned

Python GUI with Tkinter

What's next for Volume control using wifi profile

Multiple entries with database, volume control depending on background noise level, run automatically when WiFi is connected
