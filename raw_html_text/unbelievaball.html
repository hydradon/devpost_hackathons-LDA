Inspiration

We built the project based on Super Monkey Ball and were interested in using the Leap Motion controller to power it.

What it does

It is a 3d action puzzle platformer game that involves a player controlling a ball using the Leap Motion controller. The player controls the level's rotation to guide a ball through obstacles and across platforms to the end point.

How I built it

We used Unity 3D and C# to develop the levels and back-end code and scripting. We also used the Leap Motion controller and API to incorporate the hardware into the game.

Challenges I ran into

We ran into the challenge of running the game using insufficient hardware. We also had to learn how to use and implement the Leap Motion controller from no previous knowledge since the hardware was not our first choice.

Accomplishments that I'm proud of

We made a fun, functional game in under 24 hours with no previous knowledge on how to use the hardware.

What I learned

We learned how to plan and create levels, how to incorporate the Leap Motion controller into Unity, and how to incorporate UI elements and game flow aspects of game development.
