Inspiration

a lot of disabled people are left behind such as blind people, over 30 million blind people in the world. Not for every person there is help nearby and that is something we wanted to solve with our app.

What it does

A blind person can request help via the phone by holding down the home button. Then a request is sent to multiple helpers. When a helper accepts a request for help a video call is started. In de video call the helper can point out different objects read something out loud for the disabled person.

How we built it

We used android studio to make (obviously) an android app. Furthermore, we made a web portal for the helpers by using a socket connection to the app

Challenges we ran into

Managing sockets was a big obstacle for, exchanging sockets between the app and the server didn't go really smooth, so there is some room for improvement.

Accomplishments that we're proud of

  • We managed to make a group call via the web portal, which was a big accomplishment
  • a successful group work

What we learned

Teamwork, working under hard conditions (time, pressure, fatigue)

What's next for SenseAid

It needs to be improved, new and better features need to be added. Eventually with the goal to help de blind people as best as we can.
