An Android App to add fun in learning the history of numbers.

You can actually give a number of your choice as input, then you can see the facts related to that number.

We built it using the numbersapi.com (REST API) and also by using android programming going through a series of developments in the app

We finished this app in a span of 2 hours.

We have learnt how APIs can be used in android

We plan to add some more additional features that are provided as services in the numbersapi.com.
