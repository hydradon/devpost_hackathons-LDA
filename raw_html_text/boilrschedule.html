Inspiration

As freshman at Purdue, it is often hard to get used to the size of the campus and the commute between classes and other activities.

How we built it

We used Android Studio to create the Android application. We used Java and Gradle frameworks to build the application, and the Google Maps API. For the sample database, we used Firebase's services.

Challenges we ran into

Getting the Google Maps Intent to run properly and give directions automatically based on the user's schedule was challenging.

Accomplishments that we're proud of

We were able to finish everything we had planned for our application as well as some extra features like the push notifications.

What we learned

We learned a lot about using Intents in Android as well as getting familiar with Google's services for projects that use Firebase and/or any of their APIs.

What's next for BoilrSchedule

We plan on making the app layout much more aesthetically pleasing and intuitive, as well as adding features that allow customizability of the schedule. We also hope to be able to connect the student's career account to allow automation of the full schedule and other activities student may have.
