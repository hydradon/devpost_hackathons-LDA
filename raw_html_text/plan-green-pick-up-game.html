Inspiration

After watching World Cup 2018 this Summer, I was amazed when Japan and the other countries that followed started picking up trash after games.

What it does

Plan Green: Pick-up Game (PGPG), is an app that coordinates individuals to help inspire crowds of people at sports games to help pick up trash after games. The app provides venue location and architecture which allows users to sign up for a section of the stadium in which they will contribute picking up trash. The objective is not to only have the people who have signed up to pick up trash but inspire those who are not participating to help out. There will be admins that can create events that will handle the organization of people.

How I built it

Using Android Studios, we created a basic layout for the interface. In addition, we used an API called FormMaster that allows us to create a form and store the data efficiently. To display venue locations, we planned to use HERE.

Challenges I ran into

None of the team members had any experience in Android Studios, so our first challenge was familiarizing ourselves with Android Studios. As part of that familiarization process, importing APIs and getting them to work was difficult.

Accomplishments that I'm proud of

Coming together and meeting new teammates to work on a great idea.

What I learned

I learned the basics of how to use Android Studio and create a user interface.

What's next for Plan Green: Pick-up Game

Future plans for Plan Green: Pick-up Game include: 1) Support for multiple sports venues 2) Expansion into non-sports related event 3) Reward system for those who have signed up
