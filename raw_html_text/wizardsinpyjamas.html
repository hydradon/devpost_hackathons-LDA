Inspiration

We want to wizard. Now we are.

What it does

WizardsInPyjamas is a virtual reality game that utilizes the Kinect and Myo Armbands to grant the player powers of a wizard! Throw fireballs and call upon meteor strikes upon a flickadawrist!

How I built it

The world was built in Unity and the in-game components are programmed in Unity C# editor. Several of the models were made in paint.net and photoshop. We interfaced the Kinect and Myo to provide position tracking.

Challenges I ran into

Integrating the Myos accelerometer and orientation output with the Kinect position data to create gestures with attack attributes. Collision detection with various objects.

Accomplishments that I'm proud of

Successfully mapping the motion in real life to in game gestures. Writing the code for the player, enemy, update, and battle mode successfully.

What I learned

How to Git correctly. Myo accelerometer detection.
