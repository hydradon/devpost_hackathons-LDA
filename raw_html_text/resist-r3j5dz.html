Inspiration

The recent amount of protests occurring around the U.S. inspired us to create an easy to use platform that helps high school students find events/protests that match the social issues that they are passionate about.

How I built it

We built this app with Xcode, and the programming language used was Swift.

Challenges I ran into

Since we were both new to using Xcode, we ran into many challenges. It was often hard for us to understand the error logs since we only had experience with Java, but using this prior knowledge as well as researching online allowed us to fix many of these challenges.

Accomplishments that I'm proud of

We are proud of making a clean, easy to navigate user interface.

What I learned

We learned the basics of app development as well as the programming language Swift.

What's next for Resist

We want to add a new section to our app that allows users to create their own protests. We hope to do this by allowing users to make accounts on our app.
