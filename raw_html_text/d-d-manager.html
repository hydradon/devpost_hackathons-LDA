Inspiration

I've always had trouble finding a good medium for managing my character, so I thought to myself, "why not try making one myself?"

What it does

It allows you to build up your character and keep your character information up to date as your character grows throughout the campaign.

How I built it

An Android application that was optimized and designed for the newest Android API and Google Pixel.

Challenges I ran into

I ran into some issues initially on how best to design the layout to be user-friendly, while also trying to prevent code redundancy.

Accomplishments that I'm proud of

I was able to put together a nice looking app that has a lot of potential.

What I learned

Reusing activities can help in creating consistent and modular layouts.

What's next for Fifth Edition Manager

I want to build additional features for per-session management too.
