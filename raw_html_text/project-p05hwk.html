Inspiration

Produce the most annoying solutions to everyday problems you didn't know you had.

What it does

Using facial recognition(FR) software based on microsoft azure's face api we built a toolkit to let you play around with some advanced FR concepts. Most importantly it helps lift up your spirits( most of the time)

How we built it

Utilising C# with Visual Studio to use azure's cognitive services to get a live feed of the webcam of your laptop and applying FR functions to it.

Challenges we ran into

Originally we planned to make the application using java, however, due to the lack of documentation from AWS we couldn't move forward with our project fast enough and turned to using C# and Azure

Accomplishments that we're proud of

Advanced FR functions which are able to track your pupils made on the basis of mathmatical calculations mostly from scratch.

What we learned

We learned how to manipulate real-time data in semi-realistic scenarios, along with plenty of new FR concepts. Deduce valuable insights from 2D evironments and implement them into an application.

What's next for project

Add even more functions and possibly branch out to mobile and web development.
