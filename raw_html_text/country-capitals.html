• Country Capital

Inspiration

Leverage the usage of the APL feature in Alexa devices and enhancing the voice experience. Give more information about the country and its capital location

What it does

The Alexa Skill give the Capital of the Country and shares a bunch of other information over the display at hand i.e the flag of the country, Currency of the Country, ISO codes and so on. Phase 1 of the skill has the basic implementation, while I plan to add more about the Capital location in near future.

How I built it

The APL - beta can be taken advantage with the latest ASK SDK. I have used NodeJS and hosted it on AWS Lamda. ThirdParty APIs are built using PHP and hosted independently.

Challenges I ran into

Though the documentation is rich enough, I did run into some trouble with the initial integration steps. However, with more grazing over the web, things turned out to be pretty simple.

Accomplishments that I'm proud of

Getting the dynamic response of each Country on-the-fly is something I was able to achieve and plan well. Getting the Flag details from another third party is a task in itself.

What I learned

APL - has a lot of scope. I soon plan to have my existing skills integrate this bit. Surely, the Alexa Team will add more features once APL is out of beta.

What's next for Country Capitals

  • More of visual information for the Capitals.
  • Details of the Capital location - Attractions/Speciality and so on
  • President and other authorities of the Country and so on
