Inspiration

Mapquest API

What it does

Generates a list of stations via the iHeartRadio API which we were unfortunately unable to get, your favorite genres and your geolocation based on a route to play local stations that is in your favorite genre so you don't have to touch the dial ever again. #DitchTheDial.

How we built it

We started with the mapquest API to try to get a list of the major cities on a route from point A to point B. Soon we realized that the mapquest API does document that and thus we started using regular expression to get the data we want from the turn by turn narrative of mapquest.

Challenges we ran into

Complex compounded regular expressions

Accomplishments that we're proud of

We were able to produce a very powerful set of regular expressions that did what we needed.

What we learned

Regular Expressions and GeoLocation

To be the biggest music app that everyone uses and loves.
