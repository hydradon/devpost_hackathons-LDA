Inspiration

We've each had the issue where we have the time and money to travel, but have trouble deciding where and how to go about it. gmthoh aims to help with this by using Amadeus's API.

What it does

Allows a user to get travel ideas by simply entering a date of availability. Flight and hotel information is presented depending on the user's budget.

How we built it

Built using the ionic framework and firebase for the backend. Basically, lots of Angular.js

Challenges we ran into

This was our first time using the ionic framework, which despite having previous Angular.js experience, was tricky at times to work with.

Accomplishments that we're proud of

We used Amadeus's API in multiple ways in our application.
