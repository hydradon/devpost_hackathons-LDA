Inspiration

Bubblegum and zombies.

What it does

Survival shooter game that takes place in Space, defending the fort against spooky green bubblegum monsters.

How we built it

Unity game engine combined with sketched in Google Sketchup and scripts written within Unity in C#.

Challenges we ran into

Object Collision, script within unity, new concepts with C#, creating 3D Models, and a lot of functionality problems.

Accomplishments that we're proud of

Working though unfamiliar programs, and languages.

What we learned

How to make 3D models, implement the models into Unity, how to use scripts to program unity, how to use different effects and tools within Unity and Sketchup.

What's next for Bubble Bunker

More levels, weapons, enemies, and interactions.
