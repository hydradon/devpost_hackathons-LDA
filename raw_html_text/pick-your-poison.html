Inspiration

We wanted to make a webMD like website. It was intended to be humorous and not to be taken seriously.

What it does

Enter your symptoms. If we find it, we'll match a terrible drug (side effects listed) for it. Else we just tell you to start praying

How we built it

Html/css + a little bit of bootstrap for the frontend. Python flask framework for backend

Challenges we ran into

We only knew rudimentary html/css. No knowledge of bootstrap or how to run python scripts on a web server. Or how to setup a web server, or how to get a domain.

Accomplishments that we're proud of

We googled everything we didn't know, prodded around with the bugs until they gave up and got a pretty nice looking website up!

What we learned

So many bugs are from mundane things. Forgetting to capitalize POST, didn't surround something in brackets, (particularly frustrating was) the browser caching websites so changes don't show up even when you're sure they're there. Personally I learned how to setup a AWS server so that's pretty nice
