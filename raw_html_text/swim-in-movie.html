Inspiration

A friend of mine created a 3d audio visualizer on the web, and I wanted to try put the user at the center of that. Along with this I've always wanted to try out the Oculus Rift.

What it does

Swim in Movie allows the user to play videos in a nice quiet environment.The audio visualizer will take in the Audio from the video and create a visual representation of it. This will allow the user to see the audio visualizer while watching the movie.

How I built it

I created the project in Unity.

Challenges I ran into

There were a few challenges that I ran into. First I had to create a gradient of colors for the audio visualizer, as there was not a simple solution to this. Next I had to figure out how to create the screens, which shouldn't have been as hard as I made it out to be. The most difficult part was creating the ray-casting and having it work alongside the space bar.

Accomplishments that I'm proud of

Having the space bar and audio visualizer work. It seems like a little but the space bar was a bit messy.

What I learned

Using the Oculus on Unity isn't too hard.

What's next for Swim in movie

Next for Swim in Movie is a pdf screen that will allow the user to read while in this environment. This would let the user study practically anywhere.
