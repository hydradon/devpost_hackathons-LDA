Inspiration

As human beings, we all physically live in a 3D world. However, our content are mostly constrained in the 2D space. From TangoFLIR, we are trying to solve this problem by having the image overlayed on top of the depth information. By doing that, user will have more information, and experience the content with true 3D.

How it works

We take Project Tango device and connect a FLIR one camera on it. Using Tango's tracking and mapping, and FLIR one thermal imaging technology. We can reconstruct the 3D scene realtime.

Challenges I ran into

We used a micro USB extension coord, it doesn't work on the device.

Accomplishments that I'm proud of

We achieved the thermal scanning feature.

What I learned

I learned a lot about thermal imaging, how it works and the theory behind this.

What's next for Tango FLIR

I think i will add couple more features into it and make it as a public open sourced project
