Inspiration

The idea was to expand the concept of a digital whiteboard into a digital workspace for team collaboration, with Microsoft Teams and Office 365 integration via Microsoft Graph.

What it does

Workspaces is an app for Microsoft Teams that allows you to create a digital workspace, a space where you can collaborate with your team in real-time, wherever you are. Use it as a whiteboard to draw your ideas, or write your thoughts. Or use it as a pin board to organise people, files or events from other Office 365 applications.

How we built it

Workspaces is comprised of a server and client app. The server is built using C# and ASP.NET Core. The client app is built using HTML, CSS, JavaScript, TypeScript and Vue.js.
