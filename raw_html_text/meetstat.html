Inspiration

Money wasted on ineffective meetings

What it does

MEETSTAT collects and analyses data about company meetings, providing useful insights and best practices for meeting organisers

How we built it

Collecting data about the meetings from the TEEM calendar and using an Arduino MAP MAD-01 device in order to provide accurate meeting start and stop times as well as a counter with the cost of the meeting depending on the salaries of the persons attending

Challenges we ran into

Missing and incomplete data from TEEM app, incomplete documentation of the API, getting the hardware to run properly

Accomplishments that we're proud of

Successful building of the hardware to run independently

What we learned

We have learned how to merge the hardware unit with the software modules

What's next for MEETSTAT

API integration with the rest of the applications(MIRRO, ZIDOX), machine learning to provide insights on historical data,
