  • eSUB and BIM 360 Docs Integration

Inspiration

Everything that we do at eSUB is to mitigate the risk for subcontractors through improved documentation and collaboration. Having access to BIM 360 Docs does just that by opening up access to 3D models for the subcontractor to collaborate directly with the designer. Using a 3D model, subcontractors can address issues before construction begins to save significant time and money due to costly rework and errors.

What it does

Attach a file in BIM 360 Docs to any eSUB document.

What's next for eSUB

More integration features between eSUB and BIM 360 to improve the workflow of subcontractors.
