• Our administration interface allowing for fleet management of bicycles.

Inspiration

We were inspired by T-Mobile's initiative for a green lifestyle. Part of this initiative involves cycling. With T-Mobile's new campus expansion, cycling is taking a front row seat with DIY bike repair stations. We believe healthy commuting habits including cycling to work and being more connected with your transportation and with nature promotes a cleaner environment and a healthier body.

What it does

Our system allows employees of T-Mobile who would like to borrow a bicycle to check them out for the afternoon to cycle to lunch with coworkers, or alternatively check them out daily for commuting purposes.

How we built it

We used the Narrowband Developer Board to collect data about the bicycles, including location and push button sensor, in conjunction with an user-interface based on Twilio messaging system. There is also a web-based based administration panel that can track the location of each bicycle.

Accomplishments that we're proud of

Using the Twilio Narrowband Developer Board to communicate with our API and the system was a really great experience.

Additionally, this is our first hackathon together and it was a fun experience to work together and build something new with this new technology!

What we learned

Integrating multiple systems including the Narrowband hardware, Microsoft Azure Serverless Function hosting, Twilio Flow, as well as our own front-end system.

What's next for CycleShare

We want to build out more social features including group ride coordination, which increases safety and enjoyment of cycling. We also want to introduce cycling statistics for the user including average speed and distance traveled.
