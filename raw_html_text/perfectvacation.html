Inspiration

Traveling is one of our favorite things to do! But we wanted an app that would give us the best deal for us!

What it does

Our app allows you set markers on a map to pick locations of where you want to travel. You can set as many markers and remove one's you no longer want to travel to. Finally, Perfect Vacation displays you're perfect travel.

How I built it

We used Android Studio, Google Map's API, GeoCodeAPI, and Amadeus' API.

Challenges I ran into

Quite a few!

Accomplishments that I'm proud of

We are proud that we were able to integrate three different API's calls successfully

What I learned

For most of us, this was our first Hackathon! We learned how to work with API calls and Android Studio.

What's next for PerfectVacation

If we had more time, we would have liked to add Hotel fares or average food cost to help give a better estimate of the overall trip!
