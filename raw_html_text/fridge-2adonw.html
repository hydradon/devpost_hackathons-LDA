• Fridge

Inspiration

We were inspired by our experiences with food waste at home, when buying too much and having to throw away expired items from the fridge.

What it does

An Alexa skill that keeps track of what you tell it that you put in or take out of the fridge. Also keeps track of expiration dates. A mobile app provides a visual interface to view the currently stored items.

How we built it

Using Amazon Alexa, Node.js and React Native. Also DynamoDB on AWS.

Challenges we ran into

Exposing API from DynamoDB, and interpreting and displaying it in React Native.

Accomplishments that we're proud of

It works!

What we learned

Documentation is really important. Planning integrations is best done early. Refactoring code needs to be done before it takes an unreasonable amount of time.

What's next for Fridge

Integration with IBM's Watson Chef to suggest recipes for items currently stored. Reminders to use items that will expire soon. Option to set earlier reminders for very perishable items.
