Inspiration

Wanted to use the twitter api

What it does

Uses the twitter api to obtain the usage of recent hashtags to create a quiz that tests users on the popularity of different hashtags.

How I built it

It's built using android studio.

Challenges I ran into

The integration of the twitter api using fabric stopped working and we ended up manually getting information using node s to determine information on different hashtags and hardcoding the values in.

Accomplishments that I'm proud of

It is a functioning quiz app.

What I learned

How to work with the twitter api
