Inspiration

When you are having a house party and you wanna let guests pick the music without constantly giving them your phone.

What it does

it gathers all Twitter posts with #hashthemusic and adds it to Hash The Music playlist on Spotify.

How we built it

Using Spotipy, Spotify API, Twitter API, tweepy and Python.

Challenges we ran into

Using Java for Twitter; using JS for Spotify and figuring how to use APIs in the first place

Accomplishments that we're proud of

Some of us haven't used Python and APIs for the first time and managed to actually create something.

What we learned

Spotify has shitty API.

What's next for HashTheMusic

We have to develop some GUI.
