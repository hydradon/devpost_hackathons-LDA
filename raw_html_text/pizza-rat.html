Inspiration

We were originally inspired by the idea of rating restaurants by their sanitary grades. We then figured that we could also pool together some information from the MTA to figure out how bad the experience would be getting there too.

What it does

It's a chat app that only has 3 capabilities, and does it poorly at that. Oh and it has snarky remarks.

How we built it

We built it using node with express.js.
