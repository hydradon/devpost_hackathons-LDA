Inspiration

We were inspired to make a text adventure based on Yale's campus and were inspired to take on Datto's challenge and make it less than 4 kb.

What it does

Allows user to travel around Yale's campus and interact with characters and objects to make it to Yale's award ceremony.

How I built it

We used Python.

Challenges I ran into

Keeping the source code under four kilobytes.

Accomplishments that I'm proud of

Successfully uploading then accessing the game's data online.

What I learned

How to store game assets online and fetch them in real time.

What's next for YHacks Adventure

A graphical user interface to allow better immersion for the user.
