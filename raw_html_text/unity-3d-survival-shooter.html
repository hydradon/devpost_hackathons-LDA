Inspiration

I created a 2D video game using Unity over the summer, which prompted me to create this project.

What it does

We were able to fully recreate the game from the tutorial, as well as adding new enemy spawnpoints, and powerups.

How we built it

We used Unity, and our scripts were coded in C#. Also, we downloaded free assets/sprites from the Internet to create our characters and in-game items, including health boosts and coins.

Challenges we ran into

Errors in code, and small in-game bugs.

Accomplishments that we're proud of

We were able to fully recreate the game from the tutorial, as well as add more.

What we learned

We learned about the Unity interface, and that games are hard to make.

What's next for Unity 3D Survival Shooter

In the future, a level-system could be implemented into our game.
