Inspiration

I need something to help me quickly find a spot in hectic parking lots.

What it does

This web application shows you a map of empty parking spots in a lot near you.

How I built it

I built it using Flask framework. Python for deep learning.

Challenges I ran into

Learning tools needed, implementing functions of source code.

Accomplishments that I'm proud of

Pushing through until the end of my first hackathon.

What I learned

Flask, deep learning, some hardware.

What's next for ParkUMBC

Fully implementing the deep learning and feeding it live parking lot images using a camera on a drone or some other tool.
