Inspiration

News sources will never tell you how bias they are. What if we could? By providing no-click analytics on the latest news, we can present users relevant information based on the news they are reading.

What it does

Presents sentiment analysis on the latest news.

How we built it

News API, AWS Kinesis Firehose, AWS Lambda, AWS Comprehend, AWS RDS, Microstrategy dashboard and Postgresql.

Challenges we ran into

Javascript errors and debugging in the cloud.

Accomplishments that we're proud of

Getting a working data pipeline and data base setup. Our visualizations with Microstrategy's dashboarding software also look great.

What we learned

Lots of complexity in the cloud. Data is extremely complex when going through different pipelines.

What's next for IntelliNews

  • Author bias collection
  • Collecting a bigger dataset
  • Continual updates
