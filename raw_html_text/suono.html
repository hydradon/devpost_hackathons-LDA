Inspiration

We're two teenagers that both love music, and we've both had this issue before! We found that Shazam was an incredibly useful tool but only if a desired song was currently playing in its full form. We made a tool that can identify those great songs that you can't name with just a simple hum.

What it does

Record audio into the website interface, either by humming or by tapping a rhythm (less accurate, hopefully for obvious reasons), and our machine learning algorithm will try to figure out which song it was!

How we built it

Python for all of the back-end machine learning/grunt work. We created a website instead of a traditional application so we used HTML5, CSS3, and JavaScript for the front-end work.

Challenges we ran into

As expected, nothing worked the first time we did it. We had a surprising amount of front-end issues, a lot of which rendered the program useless at times.

Accomplishments that we're proud of

Creating the app and overcoming our challenges
