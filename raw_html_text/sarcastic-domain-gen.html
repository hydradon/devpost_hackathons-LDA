Inspiration

The prize for most unique domain name inspired me to create this domain name generator. But I couldn't just have a basic name generator; it needed character. So I added a sarcasm component!

What it does

Sarcastic Domain Generator (SDG) is a website for generating a unique domain name when you struggle for ideas. In addition, SDG greets you with a healthy dose of sarcasm, because:

Everyone "needs" sarcasm in their life       -Zach Burkhardt

How I built it

I built the website from scratch using HTML, CSS, and JavaScript.

Challenges I ran into

I had never learned how to use JavaScript or APIs before, so I was learning JavaScript and parsing of JSON files along the way.

Accomplishments that I'm proud of

I created a [mostly] functioning website that utilizes new skills I learned over 24 hours.

What I learned

I learned the basics of using JavaScript and APIs. Also, I learned that parsing XML files in JavaScript from APIs is incredibly difficult.
