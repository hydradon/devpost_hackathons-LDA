Inspiration

Aggregation apps

What it does

Create a Real Time Feed for victims in emergency situations, their families and first responders

How we built it

IBM Bluemix, Node.js, JavaScript, HTML, CSS

Challenges we ran into

IBM Bluemix went down, Internet, learning curve on Node.js and Bluemix, issues with Twitter API

Accomplishments that we're proud of

Getting the map to populate the emergency message and having a solid idea to begin with that we can continue to build on after this hackathon

What we learned

How to use IBM Bluemix and the neat integrations it has and more familiar with Node.js

What's next for ermaApp

Further developer includes integrating social media (Twitter), figuring out how/when to eliminate issues that have been resolved, adding SMS capabilities, sort/filter and search options for the local area, etc.
