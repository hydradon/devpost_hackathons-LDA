Inspiration

Small donations can make a big difference. Especially when it's an app like Uber (with millions of users)

What it does

The website provides more information about the Donation program for Uber rides. It also shows the locations of homeless people using the Leaflet API. It also has a donation counter that simulates in real time how much money was donated by how many users.

How I built it

Using Javascript, Html, Css in notepad++

Challenges I ran into

None of us had much experience in web development, using html5, javascript, css. We had to learn web development and create our own website.

Accomplishments that I'm proud of

  • Donation counter
  • Implementing the map in the website
  • Both functional App and a website

What I learned

  • HTML 5 -Javascript
  • Using API's from the app and website

What's next for Uber Donation Website
