Inspiration

We wanted to create an interface for crypto exchange to be done via smart speakers.

What it does

Allows users to buy and sell Ethereum based cryptocurrencies (eth, golem, augur etc) using an Amazon Echo. Potentially very useful for people who value efficency (and trendy tech). Payments are made by leveraging the users Starling bank account.

How we built it

Development was threefold.
1.) Create a cross platform desktop app which acts as a relay between the speaker and the banking server
2.) Develop the infrastructure for communicating details to the speaker.
3.) Use the banking (Starling) API to augment transactions and relay the information back to the smart speaker
