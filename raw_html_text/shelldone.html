Inspiration

It was inspired by command line interfaces.

What it does

It integrates different services and functionalities into a single cohesive interface.

How I built it

I developed a web server with Node.js and integrated it with a front-end using Electron (desktop framework that uses Web technologies) and made real-time communication possible using Socket.IO

Challenges I ran into

A lot! Sleepiness, Heroku's awful free service, my lack of knowledge with OAuth and Twitter APIs.

Accomplishments that I'm proud of

I finished it. Sort of. :) At least it covers basic functionality and features.

What I learned

That it takes time to OAuch (I mean, OAuth). And it is very hard to be ambitious and practical at the same time.

What's next for Shelldone

I'm not sure. Maybe a better (more structured) code and a new (better designed) system architecture (with persistent storage).
