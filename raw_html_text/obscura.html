• What we built this weekend.

Inspiration

We don't entirely trust every website we sign up for, so we wanted to build something to hide our identities from apps and websites in the event of a malicious owner or database leak.

What it does

Obscura generates proxy email addresses to hide your identity from websites, apps, and newsletters. After we forward your email, we immediately burn it.

How we built it

It's built with nodejs/react/postgres with a mail server called wildduck that we proxy messages through.

Challenges we ran into

Setting up a proxy mail server was pretty new for us.

Accomplishments that we're proud of

It works!

What's next for Obscura

We want to get payments and invites working so people can subscribe to our service.
