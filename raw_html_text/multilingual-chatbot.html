Multilingual-Chatbot

A chat bot that does language translation between users

How to Use

Text to a message to: 914-312-4797 in the form: (language code) (message)

Supported Languages

Note: Input language is auto-detected, but output language is restricted to the following
Though not included, the translation api supports these languages

  • (en) English
  • (es) Spanish
  • (fr) French
  • (ja) Japanese
  • (ru) Russian
  • (yo) Yoruba

Help Codes (not case sensitive, no language tag accepted)

  • option
  • options
  • ?
