Inspiration

Sometimes it's hard to find someone to talk to when you're feeling upset. It can often feel overwhelming when you feel like you are the only one going through the thing you are going through. Therefore, we wanted to create an app that would help people connect to others going through similar issues.

What it does

Talk It Out lets users select their emotion, and then it automatically connects anonymously to other people who have selected the same emotion. They can then chat about their problems, hopefully getting solace from others in a similar situation.

How I built it

We built the app using NodeJS, Socket.io, and an AWS web server.

Challenges I ran into

Since we had never used NodeJS or Socket.io before, we had to learn how to use them. This, combined with the fatigue of a 24-hour challenge, made it difficult to fully implement the matching features we wanted. Thus we were limited to a small set of simple emotions, rather than a more sophisticated way of matching people based on their circumstances.

Accomplishments that I'm proud of

Learning NodeJS, Socket.io; creating a good-looking interface

What I learned

NodeJS, Socket.io, some JavaScript, and how good sleep is

What's next for Talk It Out

Adding more sophisticated matching methods.
