Inspiration

The inspiration for this project came from buzz-feed and a mentor from Jam hacks. We noticed that we enjoyed fashion and we were pretty experienced and many people struggled to find their style and find something they can find comfort within. We thought this was a very smart idea as it added our own personal talent as well as helped others.

What it does

The website is built so that the user does a quiz and types of styles are shown to them, where they can think of what they want and plan their ideas through this website.

How we built it

We used HTML CSS and this was our first time coding and we used the first website we thought would help us to understand.

Accomplishments that we're proud of

What's next for Fashion Styles
