Inspiration

It's pretty frustrating to try to use any DApp for the first time. You have to install a browser extension like Metamask, set up a private key, store all sorts of information, sign up for an account at Coinbase, wait for a wire transfer to clear, and transfer that into your wallet, all before you can even consider buying that fluffy unicorn cryptokitty. There's got to be a simpler way.

What it does

It's two lines of code that any developer can add to their DApp. It acts as a fallback for users who don't already have a browser extension like Metamask.
