Inspiration

GoodNuch (short for good nutrition) was inspired by the lazy man's desire for a low effort nutrition tracking app.

What it does

GoodNuch uses the google cloud vision api to read nutrition facts from a label for you, and automatically insert the data into your daily nutrition log, where it tracks your nutrition compared to the daily recommended values.

How we built it

We used the google cloud vision api with android studio to develop a computer-vision-enabled phone application.

Challenges we ran into

We attempted to use a number of other machine vision libraries/apis before we decided on the google api. In addition, getting the application to run and display on an unfamiliar platform like Android was a big challenge.

Accomplishments that we're proud of

We're proud to have created something we think could be truly useful to the Android community. Our application encourages good nutrition and learning to watch your diet, in an attempt to combat the epidemic of poor health habits.

What we learned

We learned a lot about using Android and Android Studio, and look forward to using our experience to enrich our future projects.

What's next for GoodNuch

We would like to polish up the application for submission to the Google Play Store, where real people could actually use it. We would also like add more features, like custom daily nutrition goals, tracking for minor nutrients, and visualization of personal nutrition trends.
