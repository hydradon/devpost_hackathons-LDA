Inspiration

We wanted to provide law enforcement with a quicker and easier first respondent. After running into problems with programming the drone, we decided to have the drone run an autonomous flight using javascript.

What it does

The drone is programmed to run a pre-determined flight. The drone can move up, down, left and right on its own as well as performing a couple of tricks like flipping and waving.

How we built it

We used javascript to program the flight and command line to run the flight to the drone via Wifi.

Challenges we ran into

We were having troubles accessing the video feed and programming the drone to recognize faces and objects.

Accomplishments that we're proud of

We were able to start from scratch and learn how to program a drone without any knowledge of drones. Even though we weren't able to execute our idea, we were able to put on a show using the capabilities of the drone.

What we learned

We learned how to program a drone as well as the commands that program the drone to execute an action. We learned how the drone works and how to hack a drone to run an autonomous flight on its own.

What's next for The Hunt

The next task for The Hunt would be to program the facial recognition software, a trigger that would allow the drone to takeoff, and a sensor to avoid obstacles in its way.
