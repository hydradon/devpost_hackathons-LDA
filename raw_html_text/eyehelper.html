Inspiration

Inspired by the difficulties that the visually impaired faced, I developed a tool to help them understand what they see.

What it does

This website takes an image url and tells the user keywords to help the user understand the image.

How I built it

I built it using Python, HTML and CSS and used the clarifai API

Challenges I ran into

Original idea did not end up working out and started working on this late in the game.

Accomplishments that I'm proud of

Learned and coded in python for the first time. Learned to use the flask and gained a better understanding of HTML structure and POST and GET methods.

What I learned

Learned python, flask, how to use vim and terminal, learned about the structure of HTML and websites and how to use POST and GET methods.

What's next for EyeHelper

Next for EyeHelper is capability for keywords to be read aloud to the user within the app. Future capabilities also will include braille reading capabilities by building a braille clarifai library so that clarifai can learn different words in braille as most people that are blind cannot read braille.
