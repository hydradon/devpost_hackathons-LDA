Inspiration

Being too indecisive on what food to make.

What it does

Finds a recipe for you from a picture of any ingredient.

How we built it

We used Microsoft Computer Vision API to calculate what the ingredient is and Food2Fork API to generate recipes.

Challenges we ran into

We had troubles getting defined ingredients in certain photos.

Accomplishments that we're proud of

It runs and works fairly well!

What we learned

How to implement APIs and working with Java Scripts.

What's next for Lazy Fridge

Addition of ingredients and more consistent API returns with correct results.
