• The 3 Art Boards

Inspiration

Apple introduced a Dark Mode feature in macOS Mojave and I was inspired to try to bring this into the mobile world, into one of my most used applications.

What it does

Themed, concept prototype for Instagram.

How I built it

I used Adobe Xd to manipulate the prototype and design.

Challenges I ran into

properly manipulating all of the icons in the pages/ artboards.

Accomplishments that I'm proud of

Overall the entire concept.

What I learned

there is more than just the programming in app development, and to be able to properly create a very clean UX is a very important factor for applications.

What's next for Instagram Dark Mode Design

to be able to implement this into more and more pages throughout the application.
