Inspiration

As tennis lovers, we wanted to see what it takes to create a truly 3D experience, integrating real-time feedback and interaction between the audience and players.

What it does

Users can play one-one-one or in multiplayer games of tennis. The audience is able to interact with Alexa, who acts as a personal scorekeeper, judge, and commentator for the game. Using accelerometer data from an Android app and a Myo attached to a Tennis racket, we were able detect accurate movement and make this experience as immersive and accurate as can be.

How I built it

We built it using Unity and AWS Lambda for Alexa, and "Azure-base".
