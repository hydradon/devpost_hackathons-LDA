Inspiration

I have designed this application to help drivers and travellers to get a better understanding of the traffic situation in Hsuehshan Tunnel, using a simple interactive and cartographed online map.

What it does

It shows the traffic situation in Hsuehshan Tunnel by utilizing modern cartography and animation.

How I built it

I used the data provided by the challenge and converted it into GeoJSON format. I also combined PHP, Ajax, jQuery, JavaScript, Leaflet mapping library and SVG to develop this application.

Challenges I ran into

The first challenge was making an animated map which changes by time. The second challenge was reformatting data to be used in this app.

Accomplishments that I'm proud of

  1. Designing an animated interactive map.
  2. Finding a way to reformat data.
  3. Creating Visually attractive map.

What I learned

First I learned about the traffic and geography of Hsuehshan Tunnel. Second I learned a lot of programming and developed my abilities in web mapping.

What's next for Hsuehshan Tunnel Traffic Animator

If enough good data is provided, I can improve this application for example by adding met data or connecting it to real time sensors.
