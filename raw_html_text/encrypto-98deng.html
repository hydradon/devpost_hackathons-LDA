Inspiration

Using public computers in libraries and netcafes, I was unable to encrypt files because guest users are unable to download and install software. This project seeks to make encryption possible anywhere with internet access.

What it does

Encrypts files.

How we built it

Using Bottle.py for framework, HTML5 for designing the website, Python 3.4.2 for the encryption.

Challenges we ran into

Mostly technical difficulties in decrypting files. Also difficult to allow files to be altered if user does not have admin access.

Accomplishments that we're proud of

Encryption and net framework fully works, all completed in 24 hours.

What we learned

Set-up functioning website using Bottle and HTML.

What's next for Encrypto

CSS to make website neater.
