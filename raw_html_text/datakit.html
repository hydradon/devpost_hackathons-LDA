DataKit

Our Project

DataKit is an open source data collection and logging platform. Its goal is to free users from proprietary software ( Vernier ) and wasting money ($500+) on expensive hardware.

With DataKit , you can collect data from the variety of sensors already present in your smartphone. Just connect any computer with our client software installed, and plug in your own hardware sensors; DataKit takes care of data transfer, collection, and logging.

How do I use it?

It's simple, easy, and just works ™ It also saves you a bunch of money, and doesn't lock you down to a proprietary platform.

Architecture

  • Sensors:
    • You can connect any number of sensors to a client hub, which connects to a computer or cloud machine with the server software installed
    • If you don't have a client hub set up, just install our mobile app!
      • Our app will make use of onboard sensors on your phone, so you can get started right away!
  • Server:
    • You can deploy our cross platform and open source server application on a cloud server or on your own machine. It's all up to you.
  • Web GUI:
    • Our highly accessible Web GUI lets you select sensors and stream live data from them over the Internet
