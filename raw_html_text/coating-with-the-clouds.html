Inspiration

You have 3 coats. You don't want to be too hot, and you don't want to be too cold. Which one will you wear??

What it does

Based on your clothing and the weather, it will tell you what to wear!

How we built it

Lots of googling and intense work! We all used languages we've never worked with before.

Challenges we ran into

This is our first hackathon, so we ran into several of the introductory errors. We maybe should have collaborated more, because some of the coolest work was unable to be submitted. Next time we'll work on our planning!

Accomplishments that we're proud of

We are proud of learning new languages! None of us really knew css, html, javascript, or php! But we did it!

What we learned

How to learn coding languages, and how to learn them fast!

What's next for Coating with the Clouds
