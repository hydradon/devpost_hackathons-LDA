Inspiration

We wanted to create a game that would help improve users' mental health through receiving supportive messages from the community.

What it does

A user is able to express their current emotional state, which is visible to others, and is able to send uplifting messages to other users who might be struggling emotionally.

How we built it

Montivity was created with Unity, using C# for coding, and the user interface was made with Sketch.

Challenges I ran into

We had some challenges using Unity due to limited experience with the platform.

Accomplishments that I'm proud of

The Montivity team is proud of developing a game that positively impacts mental health.

What I learned

We learned how to create different icons with Sketch and how to design UI in unity.

What's next for Montivity

We plan to branch out and work on different aspects of Montivity.
