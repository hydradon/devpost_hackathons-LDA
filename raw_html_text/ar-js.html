Inspiration

Wanting to create AR applications but not having access to frameworks like the xCode AR kit

What it does

Creates a 3D in Model Augmented Reality. Simply go to the link provided by the QR code on your phone then point your phone's camera at the "Hiro" image.

How we built it

Using a Linode webserver and AR.js framework

Challenges we ran into

Creating a 3D model that will load properly

Accomplishments that we're proud of

Uploading it to a server for public use

What we learned

How to create websites and implement Javascript frameworks

What's next for AR.js

Animation, More codes with different models

Models courtesy of Cesium: https://cesiumjs.org/
