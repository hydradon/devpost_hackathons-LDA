Web Crawler

The object of our project is to mash multiple API's of Wegmans and other 3rd party APIs to make a google chrome extension. This extension can be used on any web page, when the extension is turned on, the extension will look up all the words on the web page and hit the Wegmans' Products API and provide a link to the product in Wegman's online store. This enables ease of business for customers with Wegmans.

Wegmans' API is also proposed to get the nearest location of a store by getting the GPS location from Google Chrome. A third party API - Nutritionix which can be used to get the nutrition content of each food product the user is viewing.
