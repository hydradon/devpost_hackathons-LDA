Inspiration

Updating Customer Relationship Management Software with accurate, up to date information is crucial, but is typically overlooked because it's very tedious. Manually creating a log of an email in Sugar is time consuming, and it is a very click and copy-paste intensive process. We wanted to create a way that would speed up this process

What it does

SugarSync allows a user to sync their client emails from a gmail account to SugarCRM with minimal clicks and no copy-paste involved

How we built it

We built the application using xcode and objective-c, SugarCRM API, and Context.IO API

Challenges we ran into

The biggest challenge we had was figuring out the API calls, especially Sugar's API, and using OAuth

Accomplishments that we're proud of

Finishing the application! We started out with a challenge and little knowledge of what we were supposed to work with. We were learning as we went along, and in the end, built a usable application

What we learned

That creating a software program is an ever-changing process. Features can be added and removed, layouts can change, and you always need to adapt to this change

What's next for SugarSync

We plan on creating an Android version
