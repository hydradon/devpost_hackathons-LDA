Inspiration

  • Choose your own adventure interactive storytelling media (e.g. Bandersnatch Black Mirror)
  • Limited forms of creative expression with current platforms (e.g. GIPHY, stickers, filters); increasingly preplanned and scripted storytelling content going viral (from static memes to Tik Tok)
  • Make learning to code fun, interactive, and shareable with minimal application setup overhead to distribute experiences via native social platforms

What it does

  • Build and remix interactive storytelling media microapps, shared socially (e.g. Snap, Twitter, etc.)

How we built it

  • Leveraged Snapkit SDK to create a native iOS mobile app that enables React Native app to display library of interactive memes, which can be viewed, liked, remixed for easy tweaking and resharing.

Challenges we ran into

  • Integration between native iOS app enabled with Snapkit SDK, ensuring that MicroMemes app could interface and message by posting and receiving messages with data passed back and forth e.g. micromeme to display

Accomplishments that we're proud of

  • Platform to enable creatives to build interactive experiences with limitless customizability

What we learned

  • Coding interactive memes are fun!

What's next for MicroMemes

  • Submit iOS app to Apple App store, release to creative technical communities, see what gets built!
