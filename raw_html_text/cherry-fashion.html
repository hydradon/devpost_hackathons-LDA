Inspiration

We solve the following problem: help people to understand objective feedback on how their styles look -- by community, themselves, professional stylists, and general rules. Targeting market is around 3 billions $.

What it does

Product Concept

  1. Person takes photo.
  2. We evaluate color and proportional characteristic, and provide automatic feedback.
  3. With sentiment analysis (images still recording), we understand if person likes her look.
  4. The look can be shared for estimation of expert/community.

How we built it

Style checking is done this way:

  1. Selenium (download colors).
  2. Deep Learning CNN for types.
  3. Deep Segmentation, remove background.
  4. Color neighboring.

Sentiment analysis is approached via Google/Azure API.

Frontend/Backend are done with Android Studio and Flask.
