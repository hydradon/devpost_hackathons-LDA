Inspiration

Creating 3d models to share information is a powerful tool. This has traditionally been an expensive or lengthy process. The time had come to make this tech available to all users

What it does

Using the parallel processing available on the latest mobile GPUs, camera images are reverse projected onto a high dimensionally textured shell.

How I built it

Interface, compute shaders and cross platform via unity. 3d modelling via blender.

Challenges I ran into

Performance and finding open gles 3.1 devices.

Accomplishments that I'm proud of

Initial feasibility complete.

What I learned

Mobile device performance is ready for this tech on the latest devices

What's next for Holographic Scanning

Polish and publish the modelling application. Implement a social media viewer, sharing
