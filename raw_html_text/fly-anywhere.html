Inspiration

Everyone wants to travel, but it's difficult to make it convenient and cost-efficient, and it's intimidating to do the research. This project aims to remedy that with an extremely simple-to-use application.

What it does

It finds the cheapest flights to the USA's top 50 airports and displays them to the user.

How I built it

Oh man where do I even start.

Challenges I ran into

The Expedia API actually returns prices which are lower than the ones listed on their website, and it doesn't give us a URL for the flight, so when moving from our application to their website, it becomes difficult to continue the same seamless UX. There were so many challenges. So many.

Accomplishments that I'm proud of

I've never taken on a project like this, and I've actually never used Flask period, and never used Python in a project like this. I'm also very happy with the UI of the application.

What I learned

I learned a lot about Flask, though there is still a lot more to learn. I also learned a bit about python and how it can work as a web language, which I had never seen before.

What's next for Fly Anywhere

Totally unsure, possibly hosting it on a domain so that people can use it.
