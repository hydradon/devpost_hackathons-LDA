Inspiration

We decided to build this chatbot because we know that many sales professionals are always on the go and want to have all the information they need at the tip of their fingers

What it does

SalesIQ is an sms chatbot that keeps sales professionals up to date on their meetings and customers.

How I built it

We built this chatbot using the Ringcentral sms api and the Lita chatbot plugin

Built With

  • lita
  • ring-central
