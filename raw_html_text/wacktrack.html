Inspiration

Have to go to individual sites and login to see order status

What it does

It's an all inclusive app to track all your orders currently in the process of being delivered.

How we built it

We made the skeleton of a simple note app where instead of a note it's an Item currently being tracked.

Challenges we ran into

We couldn't automate the process of adding orders. The app in current state can't add an Item.

Accomplishments that we're proud of

We worked with the Google api to access a gmail inbox. Made a web app that barely works
