Inspiration

My inspiration came from Kevin Lewis, who struggled not to eat food containing wheat.

What it does

Scans products barcodes and tells if person can have this particular food based on information they provided, additionally calculates BMI and gives link to workouts for a particular body type. Suggestion of products, which are better for a particular person is in development.

How I built it

Software build on classical LAMP stack

Challenges I ran into

Building this project by myself was more challenging but interesting at the same time.

Accomplishments that I'm proud of

Achieved everything I planned

What I learned

Learned about idea behind barcodes

What's next for MyAllergies

Improve forms, allowing taking more information. Adding analytics, possibly developing mobile app.
