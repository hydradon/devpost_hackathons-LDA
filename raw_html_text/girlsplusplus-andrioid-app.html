What it does

The app allows members and admins to use a user friendly application that helps It allows members to add a profile, check in and provide feedback while it gives admins rights for creating meetings and generating a roster for who attended the meeting.

How we built it

We used SQL to create tables. We used both PHP and SQL queries for hostgator to send in an API request which returns json objects from the Database. We have working front end application for android.

Accomplishments that we're proud of

We learned how to make android activities and how to generate API calls to the database.
