Inspiration

We wanted to maximize time as a student - no need to manually insert each date into the calendar

What it does

Parse info from a picture of the schedule on the syllabus and automatically add events onto calendar

How we built it

We use Android Studio and test it on our phones

Challenges we ran into

learning coding language, new tools, software the quality of the info parsed depends on the AI of the tool used

Accomplishments that we're proud of

We got it to open the gallery, grab an image, detect text on the image, picked out a line that has a date

What we learned

how to use Android studio OCR isn't all the hype! => needs more work

What's next for Calendar

set Reminder in Calendar if event is exam/quiz/due date
