Inspiration

As TJ students lack proper sleep on a regular basis, we thought this app would help incentivize students to sleep more, and it also has applications outside of just students.

What it does

The app allows a user to sign in and for every hour they sleep, half a cent is donated to insomnia studies.

How we built it

We built it in Cloud9 working on the same account with Sean working on Bootstrap, Linus working on making graphs with the data, and Eddie and Josh implementing Firebase.

Challenges we ran into

We found a lot of errors when using Firebase.

Accomplishments that we're proud of

We worked together well and were able to learn new things from each other. We also got more experience with coding something from scratch.

What I learned

We all got better at coding and got more experience with it. We especially got better at Firebase since none of us had used it much before.

What's next for SendSleep

Setup bank account for SendSleep and work out the remaining issues with it.
