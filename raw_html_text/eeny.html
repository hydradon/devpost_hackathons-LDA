Eeny

Eeny is a random-choice decision app using the Google's Places API. We don't expect to win any prizes, but we had a lot of fun developing an iOS app!

Time spent: 19 hours spent in total

Runs on iOS devices To run, compile in xCode and set device location sharing preference to ON.
