Inspiration

To address college student food insecurity.

What it does

Allows students to donate meal swipes and paw points to a virtual food bank

How I built it

We used Firebase to build the backend data systems; google cloud storage, google app engine, Python, and Flask to build the web server, and we used HTML5, JS, CSS, bootstrap, and jQuery for the frontend

Challenges I ran into

We originally intended to make an android app using Apache Cordova, but we were unable to make the Google Authentication service work correctly. We also had issues working with bootstrap and front-end web development.

Accomplishments that I'm proud of 

We successfully created a working web application, with more features than we originally intended.

What I learned 

We learned how to use Google Cloud services.

What's next for HungryTiger 

To infinity and beyond!
