What it does

Produce fillable LaserPro Application and Closing documents w/out having to install LaserPro

How we built it

Modified LaserPro to accept option to print "blank" documents via a command line call. Modified the Automated Lending API to add an additional print option for blank forms. Build a project using swashbuckle swagger api to add end points to request a document list, and print one or more documents as PDF files. Branched the Patient Lending LaserPro docker container and updated it to include LP and ALAPI changes.

Challenges we ran into

Unfamiliarity with C#, swagger and docker containers

Accomplishments that we're proud of

We learned how to stand up a swagger endpoint and modify a docker container
