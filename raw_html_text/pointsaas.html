What it does

The System will enable a web user to provision a supported webapp based on specified location, and URL extension. Target Audience: Sales Representatives who wish to create demos of software at the click of a button, developers who wish to bring up / tear down instances on demand, customers who would like webhosting and abstract the technical details of setting up an app.

How I built it

Website built with Bootstrap and angular.js. Scripts create an instance.
