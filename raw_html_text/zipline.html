Inspiration

Friend who's constantly breaking his budget because he underestimates the cost of going out with us.

What it does

ZipLine first of all provides the user with the ability to carry out price comparison on Uber, Lyft and taxi companies/city taxis, but this isn't all it does. It also enables them to select a destination for a meal, effectively allowing them to calculate the cost for the entire trip.

How we built it

We built ZipLine using APIs from NCR and Google, and using ReactJS, we also ended up using Node.js and Express.js for the backend.

Challenges we ran into

During the development process we ran into some difficulty trying to use the APIs from the two companies in ways they weren't quite meant to be used in. We also had to create a backend infrastructure to resolve CORS restriction issues.

Accomplishments that we're proud of

Building a React app and backend infrastructure from the ground up

What we learned

ReactJS, Node.js and Express.js

What's next for ZipLine

To create a more aesthetically pleasing user interface and incorporate more venues for customer trips.
