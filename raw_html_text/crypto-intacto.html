Inspiration

In an increasingly digital world, plagiarism is abundant in industries such as music and film.

What it does

Using blockchain technology, it encodes files so that modifying any part of a file (author's name, title, etc) renders it unreadable. Users can Intacto-fy files and view them using the Intacto player.

Challenges I ran into

When I got everything working, I got excited and accidentally busted my monitor. I was eventually able to copy my project to a USB drive after two hours of guesswork.

Accomplishments that I'm proud of

-Putting together a UI with almost all the features that a regular music player/ image viewer would have

-Learning and utilizing blockchain-related concepts

- Managing to navigate my computer with a broken monitor.

What I learned

Always save progress online, even when working alone.

What's next for Crypto Intacto

Using compression algorithms to speed up processes.
