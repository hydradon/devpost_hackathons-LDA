Core Idea - Get location based tweets using search tweet API using IBM Bluemix twitter insight for events which are popular in the selected location and analyze those sentiments. At the same time, we can detect what are all the popular events going on at the given location. So based on the current trends at the particular event popularity, a company can provide some offers to particular goods and increase productivity.

Challenges - IBM bluemix does not support sentiment analysis for PHP module.Initially we started building the web page using PHP by pulling tweets from Twitter Stream API,but we came to knew that IBM bluemix didn't support the process and we had to shift using Bluemix's dash DB and insights for twitter API.

What I learned - IBM Bluemix Twitter API insights, dashDB usage

Bluemix Insights for Twitter API will be very helpful tool for sentiment analysis and recommendation projects if its connection with the server is more faster.
