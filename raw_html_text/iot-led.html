Inspiration

With the components available to us we decided an interesting project would be to control the LED lights via the AWS.

What it does

Uses an internet browser to control your IoT LED device color mode.

How we built it

Using AWS sdk, we set up our embedded C device to communicate with the AWS with MQTT protocol. For the construction of the webpage and server, Node.js was used with javascript, css, and html to provide an interactive interface.

Challenges we ran into

We had limited supplies to work with and only one member was able to access the AWS, making it difficult to test certain aspects of the service. We also were not very familiar with most of what we were coding with.

Accomplishments that we're proud of

We were able to learn more than expected about what we were working within a limited time than expected.

What we learned

How to create a Node.js server and get a device to subscribe to the AWS MQTT protocol.

What's next for IoT LED

LED's are popular in many devices and could mean that they could be implemented in many different projects.
