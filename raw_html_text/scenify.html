Inspiration

All of us are new to Montreal and we thought it'd be nice to be able to see all the sights in Montreal. So we created Scenify to help us do this.

What it does

It calculates the most scenic route to get from point A to point B

How we built it

We used Clarifai's and Google Maps' API, combined with our own machine learning library called 3net.js. We designed our own machine learning pipeline to rate images that are aesthetically pleasing. Then we used this to generate a score for each route going from point A to point B in terms of how scenic it is.

What we learned

Implementing deep learning algorithms in a practical situation.
