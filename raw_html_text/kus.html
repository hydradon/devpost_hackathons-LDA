team name

Kus ( bird in Turkish )

Inspiration

Riko のキャラクターがとても可愛いため、その可愛さを生かすアプリを作りたいと考えました。 また、try!Swift 中にもっとカジュアルに参加者同士のコミュニケーションや感想を共有できるものがあれば良いなと考えていたため、それを達成できるものを作成しました。

What it does

このアプリを使っている人全体に対してメッセージを送ったり、受信したりできるアプリです。 届いたメッセージを Riko (Siri) が音読してくれます。 他にも、3D タッチで Riko を愛でることができます。

How I built it

使ってみたかった技術である、

  • Firebase
  • Realm
  • 音声処理
  • 3Dタッチ
  • Rx

など、 try!Swift conf で学んだ要素を盛り込んで、 Riko があなたのメッセージを誰かに送るメッセンジャーアプリを作成しました。

このアプリケーションは Riko の可愛さと参加者同士のコミュニケーションに対する革命を起こします。

Challenges I ran into

ハッカソンらしく、初めて使う技術を存分に盛り込みました。 知見を持っている方もいましたが、全員がそれぞれやったことない技術に挑戦しました。

What I learned

様々な最新技術をどうやって使うか、そして、どれほど簡単に利用できるかを学びました。 具体的にはソースコードを見ていただけるとわかります。

What's next for Kus

それぞれの機能を作り込んで次の try!Swift までにこのアプリをリリースし、参加者同士で気軽に感想を共有できるアプリケーションにしたいとおもいます。
