Inspiration

Lonely nights with beer but no friends ;_;

What it does

Allows you chug with friends and strangers over long distances

How we built it

We used Firebase for the backend and Kotlin for creating an Android. We also used WebRTC for real time video communication.

Challenges we ran into

WebRTC is hard! We also didn't know Kotlin beforehand.

Accomplishments that we're proud of

That it actually works -- at least most of the time.

What we learned

We learned to code in Kotlin, using Firebase APIs and WebRTC communication.

What's next for Bundr

  • Login-system
  • Scale to support more users
  • More reliable communication
  • Heatmap of chugs taking place nearby
