Inspiration

Capital Factories challenge to help connect startups and make logistics simpler.

What it does

Commute Connect is a ride sharing app that connects startups with like minded startups, investors and mentors. Texas has a unique challenge in that all of its cities are far apart. We help business make good use of their transit time by fostering meaningful connections

How we built it

The website is built using HTML5, bootstrap and CSS3 on the front end. The back end uses flask to create dynamic web pages.

Challenges we ran into

We had issues with the back-end as none of are very skilled with flask
