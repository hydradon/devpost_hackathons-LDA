Inspiration

Inspired by the possibility of 5G for brands and causes to create a more compelling world of personalized advertising in the future.

What it does

AR XR Ads reads complex images and serves users with an enhanced digital environment on their Samsung device. Additionally, it provides advertisers an additional media space that uses customer behavior data to improve customer retention and conversion.

How I built it

AR XR Ads is a native app built in Unity for Android. It was built to run on the Samsung S10 5G and was tested. We also built a web based version that works on the latest mobile web browsers.

Challenges I ran into

We had to adjust the demo product to fit within the constraints of the demo space.

Accomplishments that we're proud of

The demo runs smoothly and is very solid.

What I learned

We learned how to find the public endpoint for IBM Cloud Storage. We were able to store dynamic videos on the IBM Cloud, reducing latency and making use of expanded bandwidth.
