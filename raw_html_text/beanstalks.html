Inspiration

Re-inventing money management for kids

What it does

Bank accounts for kids at real banks are not really designed for kids- we plan to fix that.

  • We let kids set and manage saving goals
  • We let them manage, and get credit for chores
  • Most importantly: we connect to a real bank account, and provide a kid friendly interface for managing it

How we built it

Tools and apis:

  • Mobile App Development: Kinetise
  • Bank transfers: Xero API
  • Firebase DB
