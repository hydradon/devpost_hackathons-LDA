Inspiration

We were inspired by the various technologies for object recognition and the need to help the less fortunate around the world. We decided to harness this challenge by creating an app that helps people in daily and social situations better.

What it does

This app uses tensorflow in order to classify images in real time.

How we built it

We used a pre-trained model and made the app speak the object out loud when over a certain degree of confidence.

Challenges we ran into

the classifier we used was not trained that well with everyday objects

Accomplishments that we're proud of

-understanding the various functions of the API

What we learned

  • How to use image recognition ## What's next for Periscope -training a classifier with more daily items -storing faces and recognizing people
