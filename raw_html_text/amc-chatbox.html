Inspiration

Our struggles with picking shows to watch and interacting with others with similar taste in entertainment.

What it does

Allows viewers to rate and comment on a show/movie using words and/or emojis.

How we built it

We didn't build the application with code, but we hope to use Javascript to complete our project.

Challenges we ran into

Learning Javascript and building an application based on our mockups.

Accomplishments that we're proud of

We are proud to brainstorm an idea that can be valuable to the company and can be possibly implemented to increase the social element and viewer engagement to series and movies.

What we learned

We learned how to use Javascript and create a prompt window and using Bootstrap to use Modals.

What's next for AMC Chatbox

A rating system for videos, using emojis, much like FB posts, to indicate the overall favorability from past viewers that can be seen when hovered over the video.
