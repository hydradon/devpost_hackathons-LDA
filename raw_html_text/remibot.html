The goal behind RemiBot is to establish a non-overt way for users to become more self-aware of trends in their mental well-being. RemiBot periodically encourages check-ins through Facebook Messenger where users are encouraged to share how their days are going. This data is then processed using the Text Analytics API through Microsoft Cognitive Services, which assigns each message a sentiment ranking. These rankings are sent to our web service and stored in a database, allowing RemiBot to link users to a data visualization portal. Here, various graphs similar to stock charts display trends in mental well-being and morale.
