Inspiration

What it does -

Analyzes a users tweet history and judges their opinions on the candidates based off of their sentiment in context of their mention of the candidate

How we built it -

Python3.5 with the Flask web framework

Challenges we ran into -

We struggled for hours on how to analyze the text data in a way that would allow for machine learning

Accomplishments that we're proud of -

Getting our code up and running with a web UI

What we learned -

Machine learning on text data is very challenging.

What's next for Political Twitter Sentiment Analysis -

We would like to create an interactive web front-end allowing you to see the text of the tweet scored by our algorithm
