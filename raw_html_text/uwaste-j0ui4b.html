Inspiration

We were inspired by the amount of waste we made during the hackathon.

What it does

The app allows users to take pictures of their waste and tells them how to dispose of it properly.

How we built it

We used Android Studio to develop the Android app, and used Google's Vision API to analyze the pictures taken.

Challenges we ran into

Implementing an effective UI and properly using the Vision API.

Accomplishments that we're proud of

Taking advantage of a municipality's waste disposal database to provide the user with instructions.

What we learned

How to take a picture in an Android app and use the image data.

What's next for UWaste

Implementation of a point system to incentivize users to recycle properly.
