PoSDroid

Aplikasi Android untuk warung kecil

LATAR BELAKANG

Seiring perkembangan teknologi informasi, cash register maupun pencatatan transaksi secara manual mulai digantikan oleh komputer sebagai media point of sale yang biasa disebut dengan aplikasi kasir, software kasir, ataupun program toko. Sebagai seorang wirausaha yang mempunyai warung kecil dan sedang berkembang, tentu juga ingin memiliki pencatatan penjualan yang baik dan efisien. Banyak usaha yang gagal, karena masalah keuangan yang tidak beres atau transaksi penjualan yang tidak sesuai dengan penerimaan keuangan di luar kendali pemilik wirausaha.

Dari hal tersebut di atas, permasalahan untuk warung-warung kecil sangatlah tidak mungkin menggunakan POS berbasis komputer. Untuk mengatasi hal tersebut di atas diperlukan perangkat portable. Smartphone Android adalah salah satu jawaban untuk mengatasi permasalah di atas.

Dari latar belakang di atas maka kami mendesain dan membuat sebuah aplikasi sederhana namun cukup menjawab kebutuhan para wirausaha kecil, aplikasi ini kami beri nama PoSDroid.

FITUR-FITUR POSTDROID

BARANG

Form untuk memasukkan data barang dengan pilihan masukan ID menggunakan manual dan QR Code Scanner.

TRANSAKSI

Form transaksi penjualan barang, input kode barang berbasis manual dan QR Code Scanner. Pada Form ini diberi fasilitas cetak nota transaksi menggunakan Bluetooth Printer.

EXCEL

Data Barang, transaksi penjualan dikonversi menjadi file excel / csv

LAPORAN

Laporan sederhana transaksi penjualan dan stok barang

BANTUAN

Bantuan para pengguna untuk menjalankan aplikasi PoSDroid.
