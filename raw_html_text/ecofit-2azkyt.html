Inspiration

Sustainability of Earth's ecosystem is a fragile entity. We believe that in positive reinforcement we can change our bad habits to reduce our carbon footprint.

What it does

Tracks a user's carbon footprint Suggests challenges to reduce their carbon footprint based on their current carbon emissions Data visualization of the user's carbon footprint

How we built it

Frontend: React Backend: Node.js + Express Google Cloud Platform (AppEngine) Twilio (SMS) Database: MongoDB

Challenges we ran into

We wanted to implement UI path in the automation of our messaging system. Since it was using a Windows server, we ran into some trouble because our app is mainly Linux based. We wanted to implement a mobile app. Since our team is more knowledgeable in web development, we wanted to use React Native which would have allowed us to code using React but have a mobile app as an end product.
