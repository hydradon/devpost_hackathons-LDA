What it does

Safe+ Net is a website built with disaster preparedness in mind. Natural disasters are increasingly common these days; it's important for everyone to know their emergency plans and be able to execute them when disaster strikes. This app streamlines the process of creating a disaster plan.

How we built it

We used Figma to mock up the site and coded it with a combination of HTML, CSS, Javascript, and jQuery. We used the Bing Maps API to help our users find exit routes and used the Firebase API to store user information so it could be inserted into a custom plan at the end.

Challenges we ran into

We had problems integrating Javascript into our Firebase functionality and getting the Bing Maps API to work properly with a custom input panel.

Accomplishments that we're proud of

We successfully integrated the Bing Maps API into our project!

What we learned

Figma was a great tool for our designers to work collaboratively on the UI design. We learned a lot about how to integrate APIs into projects, as some of us had never worked with APIs before.

What's next for safe-ty.net

Continue improving our interface and user experience, to make it as easy as possible for users to make emergency plans. Implement some kind of login feature so that user information stays private.
