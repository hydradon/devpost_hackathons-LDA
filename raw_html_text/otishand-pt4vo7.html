Inspiration

We thought the mouse was too oldschool so we wanted to create a new device for game controlling.

What it does

Uses a few sensors in order to allow you to control your character in a new way.

How we built it

We used touch and flex sensors alongside an accelerometer.

Challenges we ran into

We were planning on using a wifi module but it broke down after we already spent some time creating the setup.

No experience in calibrating hardware components.

Had to use Unity for the first time.

Accomplishments that we're proud of

We managed to get it to work. We managed not to set it on fire.

What we learned

Arduino and Unity.

What's next for OtisHand

Turning the OtisHand into a data-glove.
