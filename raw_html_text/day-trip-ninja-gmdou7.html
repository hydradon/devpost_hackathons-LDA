Inspiration

  • This webapp is inspired by Yelp. Yelp has been very successful in helping millions of people find local businesses by letting users share reviews. Instead of one specific type of search results, what if we could combine multiple Yelp results to form an entire day of trip for users?

What it does

  • Day Trip Ninja is a website that uses Yelp’s API to return multiple fun places based on users’ current locations. With users’ willing-to-travel distance preferences, we will generate random day trips to the users. We will intelligently help users think where to go for breakfast, lunch, and dinner. More importantly, Day Trip Ninja will also help users plan what to do next after each meal.

Technologies

The webapp is built with HTML, CSS, Bootstrap, JavaScript, JQuery, Ruby, and Rails

Our challenges

Yelp's API doesn't accept/return user's budget data so it's difficult to provide a more robust user experience as planned.

Things that we are proud of

  • This is our first time consuming Yelp's API, and we are able to deliver a robust experience by letting users easily get highly customizable day trip info. We do not just randomize a single day of trip to users. We generate a day trip that users can modify until they are satisfied.

What's next for Day Trip Ninja

  • Show related day trips modified by the other users
  • Let users favorite trips.
  • Let users share trips.
  • Let users leave comments to trips.
