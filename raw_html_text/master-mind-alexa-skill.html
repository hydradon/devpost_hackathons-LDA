• Master Mind Alexa Skill

Inspiration

I wanted to have interactive math game that I can play with Alexa.

What it does

Become Master Mind expert, discover the hidden code and beat Alexa! There are 4 positions, on each position there can be a number 1,2,3,4,5 or 6. They can repeat. You have 6 guesses available, and if you are really an Master Mind you will guess always in 5 or less attempts! In order to make easier for user to follow responses I have coded feedback in Alexa app - User can check there all Alexa hints/answers.

How I built it

I have built the skill using Python and Amazon AWS - Lambda function.

Challenges I ran into

How to handle wrong guess and validation of user feedback

Accomplishments that I'm proud of

I am proud of using variations in Alexa responses thus making them more interesting

What I learned

I have learned using SSML tags and how make Alexa sound more human :D

What's next for Master Mind Alexa skill

Working on counting user score and making hiscore web interface, so that all can compete. Hope that I will get Echo Spot soon so I can enable visual feedback
