Inspiration

We hope to create a secure and anonymous platform for video-chat where you don't have to care how you look or sound, since no-one will know your true identity.

What it does

Anonymous videochat that uses real time face and voice transformation, providing anonymity.

How we built it

We created our own p2p server that creates connection between FE clients. FE is running on electron so its fully compatible with linux, mac and windows.

Challenges we ran into

management of UDP sending and realtime face transformation were challenging

Accomplishments that we're proud of

We managed to create a working prototype, which we kinda did not expect at the start and since this is our first time hacking, we are quite happy with what we've created.

What we learned

To cooperate, listen to each other and not to train anything on CPU without building TF for given platform first.

What's next for Lendface

Fix FPS issues and create better compatibility
