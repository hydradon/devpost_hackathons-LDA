Inspiration

It's always time consuming to rerun & monitor the whole workflow again and again for troubleshooting or when need of reproduction. That's where this activity comes useful which records the whole workflow as a video file. Once installed custom activity called 'Recording Scope' is available under 'Screen Recorder' namespace. Workflow which needs to be recorded should be invoked or kept inside the 'Do' sequence of the activity.

What it does

After completion of the execution, a video file is generated.

How I built it

Its been build in Visual Studio using 'Accord.Video.FFMPEG' library.

What's next for UiPath Screen Recorder

Will be adding few more properties to control the frame speed and other details for the custom activity.
