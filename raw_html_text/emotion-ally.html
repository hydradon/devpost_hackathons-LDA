Inspiration

We wanted to provide a way to allow users to collect data about users emotions through messages.

What it does

Provides a visualization for users to see what the sentiment is behind their messages.

How we built it

We used Android Studio, Java, XML, IBM Watson Tone Analyzer, and a graphing library to provide a working application.

Challenges we ran into

Learn Java, Understanding how to connect to external API/Library's, using threads for efficiency

Accomplishments that we're proud of

Creating a working application that, with a little more work, could be deployed to market for users to start using.

What we learned

Java, API/Library calling, Android App development

What's next for Emotion Ally

Improve upon our UI and integrate our functionality and visualization with other messaging services .
