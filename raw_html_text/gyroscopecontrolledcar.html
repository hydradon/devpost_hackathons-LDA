Inspiration

We want to explore various ways to communicate across devices.

What it does

It is an arduino unit that could be controlled by android app which we built with gyroscope library and bluetoothSPP library.

How we built it

First we built the physical parts of the arduino unit, then we wrote arduino code as well as python code to power it. Next we devised the java code in Android Studio to set up bluetooth connection, and then we used the gyroscope library to provide the control functionality.

Challenges we ran into

Bluetooth connection from arduino unit to android app Gyroscope functionalities and imbedded them in our app

Accomplishments that we're proud of

solving the challenges above

What we learned

version control with git, android studio, arduino, python, java
