Inspiration

Nous avons été inspirés par une problématique selon laquelle des psychologues et d'autres spécialistes connexes nous ont rapporté que les personnes ayant une déficience intellectuelle avaient des difficultés à effectuer plusieurs tâches de la vie quotidienne. Une de ces tâches consiste simplement à cuisiner.En effet, ces personnes doivent être souvent grandement superviser pour faire ces taches ou encore, tout simplement incapables d'entreprendre ces tâches.

Ce que le projet accomplit

Ce site web permet de décomposer des recettes complexes en sous-étape plus simple ainsi qu'en illustrant chaque étape afin de clarifier certaines nuances. De plus, ce site permet de réaliser des commandes en ligne auprès d'épicerie afin que les personnes ayant des difficultés a ce déplace aisément en ville, soit par transport en commun ou par transport autonome, puisse avoir accès a des aliments sans avoir besoin de supervision.

comment l'avons-nous construit

Nous avons utilisé le Framework React afin de créer notre site Web. Nous avons également utilisé AWS afin d'héberger notre serveur. De plus, nous avons utilisé l'API de Walmart afin de réaliser des commandes en ligne

Defis rencontrées

Nous avons rencontré des défis avec l'API Wal-Mart puisque cette API n'est pas très bien documentée. De plus, nous n'avions pas fait de site Web avec React donc nous avons dû apprendre ce framework rapidement.

Accomplissements et fiertés

Nous sommes fiers d'avoir produit un site web permettant de rendre disponible des recettes à un public souvent oublié. Nous sommes aussi fiers des apprentissages que nous avons faits en ce court laps de temps

Apprentissages

Nous avons appris à avoir une bonne estimation de nos capacités en fonction du temps accorder. À travailler en équipe et à bien communiquer les situations bloquantes et urgentes. Nous avons également appris plus profondément le framework React ainsi que le framework AWS

Food4you dans le future

La possibilité d'inclure des recettes personnelles des utilisateurs pourrait être un élément important dans le développement futur de Food4you.

----------------------------ENGLISH-------------------------

Inspiration

We were inspired by the problem that psychologists and other related specialists told us that people with intellectual disabilities had difficulty performing many tasks in their daily lives. One of these tasks is simply cooking. Indeed, these people must often be greatly supervised to do these tasks or else simply unable to undertake these tasks.

What it does

This website allows you to break down complex recipes into simpler substeps as well as illustrating each step in order to clarify some nuance. In addition, this site makes it possible to place orders online at grocery stores so that people with difficulties can move around the city easily, either by public transport or by independent transportation, can have access to food without the need for supervision.

How we built it

We used the React Framework to create our website. We also used AWS to host our server. In addition, we have used the Walmart API to fulfill orders online

Challenges we ran into

We encountered challenges with the walmart API since this API is not very well documented. In addition, we did not make a website with React so we had to learn this framework quickly.

Accomplishments that we're proud of

We are proud to have produced a website to make available recipes to an audience often forget. We are also proud of the learning we have done in this short span of time

What we learned

We have learned to have a good estimate of our abilities as a function of time. To work in a team and to communicate well the situations blocking and urgent. We also learned more about the React framework and the AWS framework

What's next for food4you

The ability to include users' personal recipes could be an important element in the future development of Food4you.
