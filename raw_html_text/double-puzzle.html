Inspiration

We noticed that when advertisements had some secret code hidden inside, people try to get that hidden code by hook or by crook. People love to play games and win prizes, so why not try that in Augment Reality ?

What it does

It is a puzzle solving game, where the people will use this AR effect to look at certain advertisements, which will reveal an animation and a 3D object. Inside the animation and 3D object, there'll be hidden code which the participants must find. Once found, they should submit that hidden code to the organizing company's web page or in their facebook page's message inbox and they get to win some prizes.

How we built it

We built the Augmented reality effect using Spark AR and built the 3D object using Blender.

Challenges we ran into

Creating the logic using Patch Editor of Spark AR and various other on screen gestures were a bit challenging. Keeping the size of the AR effect small and under 2 Mb was also challenging.

Accomplishments that we're proud of

We managed to finish this within one week.

What we learned

We have learnt the various possibilities that Augmented Reality can be utilised to add a new dimension in UI/UX.

What's next for Double Puzzle

We will add new ways to hide text or audio, which should make the puzzles even more challenging !
