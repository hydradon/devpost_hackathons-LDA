Inspiration

I wanted a way for people who didn't have access to the internet or have a data plan to have a way to recieve weather information

What it does

You text theTwilio number with the message "weather" and you'll recieve the current weather for that day

How we built it

We used Cloud9 IDE, ngrok, Python, Flask, Twilio

Challenges we ran into

Python 2 and Python 3, combing 2 API's, using Cloud9, learning git, learaning how to use API's, Parsing,

Accomplishments that we're proud of

Learning to parse, setting up a ngrok server, ploished product

What we learned

How to pasre, how to use APIs, how to push and pull from github

What's next for Text4Weather

Receive forecast for upcoming days.
