Inspiration

We have seen the struggles of our peers in getting dates so we decided to make a tool to help out.

What it does

It is a website that receives Tinder profile pictures and a bio as input and produces a list of the best pickup lines as an output.

How we built it

A lot of data scraping, analysis and model building.

Challenges we ran into

Python's low performance in handling queries

Accomplishments that we're proud of

Getting the project to give reasonable results

What we learned

Lots of Python and JS syntax and some APIs like Azure and the one for Reddit.

What's next for Tinderrr

Maybe being integrated to some extent in the actual Tinder app.
