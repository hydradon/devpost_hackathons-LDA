Inspiration

Snapchat has a good selection of stickers however its missing a very popular sticker category, MEMES!

What it does

This app allows you to add wholesome memes from reddit as stickers to your Snaps.

How I built it

Developed an iOS app using Swift. Used Reddit API for memes. Used SnapKit's Creative Kit to create the snaps and connect to snapchat.

Challenges I ran into

Accomplishments that I'm proud of

I got it working!!!

What I learned

I learned how to use the camera on iOS. I also learned how to integrate SnapKit in an iOS app.

What's next for Snapchat meme stickers

Add ability to search for memes. Maybe create a whole marketplace for custom snapchat stickers. Extend functionality to videos and gifs.
