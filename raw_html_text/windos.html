Inspiration

ვნახე საიტი სადაც იყო iOS აწყობილი HTML ში. დამაინტერესა სხვა პროექტები, იყო შექმნილი ანდროიდიც და ბევრი რამე მაგრამ Windows არ იყო არსად აწყობილი ამიტომ გადავწყვიტე რომ მე გამეკეთებინა. ჩემი მეგობარი ალექსანდრე ხარაბაძე დამეხმარა ამ იდეის განვითარებაში და ერთად გადავწყვიტეთ რო ჯობდა რომ Windows ზე შეგძლებოდა ასევე ფაილების ატვირთვა. ფაილების ჰოსტინგი Windows 7 ის დიზაინით!

როგორ მუშაობს?

უბრალოდ რეგისტრირდებით და თქვენთივს გამოიყოფა მეხსიერება რომელზეც შეგიძლიათ ატვირთოთ ფაილები და ფოლდერები. ეკრანიდან გადაიტანთ და Windows Online ის Desktop ზე გაუშვებთ და პირდაპირ იტვირთება!

Accomplishments that I'm proud of
