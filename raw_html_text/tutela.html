Inspiration

Novices and dabblers looking to pick up new hobbies might find professional teachers to be too expensive and online videos to be too passive. Tutela allows enthusiasts to locate talent peers in the community that can offer an affordable, casual, and satisfying learning experience.

What it does

Tutela allows users to advertise skills they have experience in and feel comfortable teaching to beginners. In addition, users can learn new skills from others who know enough to get beginners started.

How I built it

We used the web application format taught to us in the hacktx workshop on Friday and we built a web application that stores information and allows the users to search for tutors.

Challenges I ran into

We were new to flask and not all of us knew python, so collaboration was an issue. Also, we had problems with the concept of rest api because it is more abstract than a new learner might think it is. The web app still doesn't do all that we want it to do.

Accomplishments that I'm proud of

We got something to show to other people that we worked and we are happy to have learned important insights in programming through the really great mentors and corporate representatives.

What I learned

We need to get an organizational structure and a good work ethic in place before we try to accomplish any task.

What's next for tutela

Getting it to work properly and pushing it to an app.
