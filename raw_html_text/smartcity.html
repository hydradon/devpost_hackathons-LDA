Inspiration

The idea is quite simple. Provide, in the spot, activities available near someone while sharing offers information. The inspiration came from another project that we heard of that also came from a hackaton, but it was at a way smaller scale than our idea.

What it does

It provides information about near by activities.

How we built it

We used intel XDK to build the app using HTML code based on Foundation.

Challenges we ran into

At the beginning we truly didn't know what to do.

Accomplishments that we're proud of

Mainly learning from all of the experience and workshops we took during our stay here.

What we learned

We learned the bases of Rails and other web and app developing software.

What's next for SmartCity

After acquiring other skills we would like to take this project to the place it was actually intended to be in the beginning.
