What it is

A chrome chatbot to help students stay on top of their assignments.

Upon first opening this chrome extension, our chatbot, Chromie, greets the user and asks the user to input his/her classes for the semester. The answers are saved in the database for future reference, and from that point, the user can freely add assignments to the extension by talking to Chromie.

How it works

The chatbot uses api.ai to interact with the user, add classes (as entities), and to interpret commands given to it. We originally used HTML5 localStorage to save the assignment list, but switched over to chrome.storage api since it was more flexible. The extension will send notifications every X hours to the student reminding him or her of their tasks with desktop notifications through a background javascript file.

We finished a lot of it, but were unable to put on the finishing touches...This is definitely something our team will continue to work on after the hackathon though!
