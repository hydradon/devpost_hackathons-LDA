What it does

The user is able to place down Benedict(The banana bread man) at certain coordinates and then Kareem the Koala and Derrick the Duck will then go look for him by finding an optimal path. Kareem moves faster on land, while Derrick moves faster on water. Unfortunately several components are incomplete. We still need to add in Kareem and Benedict to the game. Next, we want to use user input to decide where Benedict is placed which we can do with a simple scanner.

How we built it

Using Java and java swing components using the classic Model-View-Controller Design pattern.
