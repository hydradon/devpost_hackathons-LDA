Inspiration

We wanted to provide a website similar to mapliv.com or padmapper.com for but for job search

What it does

Show a map with markers detailing jobs and their description (taken from indeed.com)

How we built it

Used an ajax call to get information from the Indeed API. We also used the Arcgis API to display the information on a map.
