Inspiration

We wanted to create an app that everyone can use with their real life problems. Everyone has experienced being stuck in awkward positions, so this app will solve that.

What it does

This app will make a fake call to the phone so the user will have a reason to get out of an awkward situation.

How we built it

We built it using Android Studio.

Challenges we ran into

Security issues with vibrate, making it work on different types of Android phones, and the design of deceiving others.

Accomplishments that we're proud of

Making a fully functioning app that everyone can use. This was our first Android developed application that our team has ever created.

What we learned

It's very difficult to make an app that deceive others.

What's next for HelpMe

Text message, email, voicemail, and social media features.
