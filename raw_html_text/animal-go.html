Inspiration

The popular game pokemon go

What it does

Fist part: An android app for reporting illegal acts against wildlife animals and rewarding people for it.second part: a system that helps wildlife guards to analyze this data and take action

How I built it

We used java for the android app and asp.net mvc with web api and Entity framework for the backend We used html,css,bootstrap and jquery for the frontend.

Challenges I ran into

Time Sleep, , Complexity of the project How to maintain the link between The android app and the webAP How to handle unwanted data sent by rogue users

Accomplishments that I'm proud of

Being able to actually create such a complex project I such small time

What I learned

Many many things like Web API and Okhttp library for android I also learned presentation skills and how to create a good presentation

What's next for Animal Go

A full implementation and hosting on the web and google play I will ask my friends to download and use it And I will manage and analyze the data that they send
