Inspiration

Budgeting in college is difficult because you don't know what to account for

What it does

It files into categories all of your expenses to make it easier to see where you need to make allowances in your budget

How we built it

Using Java script, it was integrated into an android application

Challenges we ran into

Getting the application to run properly and getting everything to run smoothly visually and logically

Accomplishments that we're proud of

Making a working app as well as having a team consisting of several different major fields

What we learned

How to code an android application that's useful for us as individuals and for society as a whole

What's next for Cash-U

Marketing it to the public community at the campus and further creating it as a global tool
