Inspiration

Help people with speech impairment.

What it does

It provides the user with a set of tongue twisters and pronounces it in the correct way before the user is recorded for practice. Once recorded, the web provides feedback to the user and can pinpoint where the pronunciation error was.

How I built it

The back-end was built using python and google-cloud-speech API (speech to text). The front-end was made with javascript and html.

Challenges I ran into

Using google speech to text API and trying to remove the auto correct.

What I learned

Many things!

What's next for Tongue Twisters

Tongue twister would like to provide educational videos or collaborative feedback to the user after detecting where the pronunciation error was. The end goal is to tailor the assistance to the user.
