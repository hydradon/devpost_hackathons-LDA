Inspiration

What it does

The main part of the app is the backend. Like many big companies, the main product is under the hood and the front follows a minimalist ideology. When users sign up and log in, they are added to a chain of events and they create a block in the blockchain. They are able to loan a part of their block to other people, and request loans.

How I built it

We built the blockchain using SHA-256 encryption on Nodejs. The front-end was built using HTML, CSS, and Javascript.

Challenges I ran into

Firebase has been acting strange the entire hackathon. It would only accept our requests sometimes.

What I learned

We learned how to create a blockchain from scratch and how to implement it.
