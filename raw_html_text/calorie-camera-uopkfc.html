• Calorie Camera

Inspiration

Sometimes we all just wonder how many calories something delicious is, so why not build an app that tells you just that!

What it does

When you hover your phone camera over some food, it will automatically display how many calories it contains and what food it is.

How I built it

We built it using Android Studio and we used Microsoft's cognitive services custom vision and trained it to recognize different food.

Challenges I ran into

However, since we do not have much experience in creating Android applications, we did run into a lot of trouble trying to fit all our ideas in the application. If given more time, we could tidy up the user interface and add more screens.

Accomplishments that I'm proud of

We are proud that we are able to get the app up and running and correctly detecting different foods.

What I learned

We learned how to implement external APIs to build our app.

What's next for Calorie Camera

We plan to expand the number of different food the app can recognise. Also, we plan to implement new features such as a history tab that records previous captures and a function that helps you keep track of your total calorie intake in a day.
