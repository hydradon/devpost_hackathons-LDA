Inspiration

Discussing as a group we realized that as college students there are many possible win-win exchanges we could have with elderly people. We believe youth in general struggle with key soft skills that are vital for their success. Given the prompt to aide the elderly with social isolation we chose to create opportunities for these exchanges.

What it does

Using a detailed questionnaire we create suggested matches for youth with elderly that are looking to teach key soft skills. We then help facilitate the relationship between the two using features such as chat, voice chat, by suggesting and creating activities.

Challenges we ran into

While tackling the backend we stacked on many new technologies along with learning to manage a new workflow. The backend team implemented the chat application for example separate from the UI. This implementation worked with persistence using a MongoDB database although when it came time to integrate the solution with the UI we were short on time and not able to quite get it to work.

Accomplishments that we're proud of

Distilling out problem space, getting insight from all of our mentors and course correcting our solution, and our UX solution.

What we learned

We learned technical design skills, new technology stacks, as well as team working skills such as compromising with conflicting ideas and incorporating differing skillsets.
