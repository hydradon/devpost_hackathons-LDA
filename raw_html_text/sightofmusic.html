Inspiration - Since we were both developing musicians at one point in our lives, we decided to build an app that would make it easier for younger generations to learn sight-reading better than we did.

What it does - it trains the student to identify the note in a specific key signature in a allotted amount of time.

How we built it - We used Swift and iOS to build the app.

Challenges we ran into - We were comparing two values and identified an odd error that involved Optional versus non-Optional comparisons.

Accomplishments that we're proud of - Finishing, learning, and customizability.

What we learned - how to deal with bugs.

What's next for SightOfMusic - expanding to chords and intervals.
