We wanted to see if we could create a computer from the past using current technologies

It counts in binary with 10 buttons on 4 individual bright LEDs

We first sketched the concept, the made a prototype on a bread-board. After that we copied the schematic on Eagle and printed a custom board to house the electronics.

Fitting everything we wanted on a compact board proved difficult so we needed to decrease the size of the pcb wiring.

We managed to design and print our own pcb that fit all of our components and the complex circuit we designed.

Next we will likely try to make it able to add numbers when multiple buttons are pressed since it can only currently add up to 17.
