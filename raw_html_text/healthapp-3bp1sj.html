Inspiration

We wanted to improve efficiency of the health care system.

What it does

Our app allows patients to book appointments from their doctors and we plan to add more features including a messaging system between doctor and patient.

How we built it

We built it using Android Studio and Java.

Challenges we ran into

We had problems connecting client to server most likely because of fire wall problems.

Accomplishments that we're proud of

We are proud of learning how to use Android Studio.

What we learned

We learned how to create an app and to work as a team.

What's next for HealthApp

We plan to develop a messaging feature and allow patients to access their medical histories.
