Inspiration

I started the Hackathon searching for ideas while looking through the challenges set by the sponsors. After a while, I realized that I was interested in playing around with the Mozilla Firefox Extensions. I planned to design an extension which could give you suggestion on how you could save the earth and raises awareness for the Climate Crisis. This led us to the development of ClimateUPC which does that seamlessly.

What it does

It gives you suggestion on how you could save the earth and it raises awareness for the Climate Crisis problems.

How I built it

Used different tools from the frontend side like HTML, CSS and Javascript to built it successfully

Challenges I ran into

I had to research alot about firefox extensions and their functionalities

Accomplishments that I'm proud of

Idea is now a reality and works better than expected.

What's next for CLIMATEUPC

Machine learning algorithms can increase quality of suggestions
