Project Submission For MLH Local Hack Day. Farmo is a blockchain platform that connects Farmers directly buyers on the Ethereum Blockchain Platform. Farmo eliminates the middle-men i.e Finiancial insitutions(Banks) by creating an avenue for the buyers to pay directly to the farmers

  • run truffle compile

  • run truffle migrate --reset

  • npm install

  • npm run start
