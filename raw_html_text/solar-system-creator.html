Inspiration

We wanted to do something fun with the wealth of data available in the NASA Exoplanet Archive.

What it does

This is a dash (python visualization library) application that allows you to create a solar system, and then see which observed solar system in the Exoplanet Database is the most similar to the one you created.

How we built it

We built this application using the dash library in python

Challenges we ran into

Indexing things is hard. Parsing big data is annoying. Dash applications do not like to be pretty. Orbits aren't easy.

Accomplishments that we're proud of

Having something that works!
