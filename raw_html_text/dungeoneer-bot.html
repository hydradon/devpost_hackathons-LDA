Inspiration

We simply came in with a goal to create a discord bot with interactive commands. Ultimately, the goal would be to develop a bot that aids a Dungeon Master in creating and running a D&D game.

What it does

Currently, the bot will roll dies, greet people, and insult you!

How I built it

We used javascript with the discord.js library as well as a google api to analyze comments.

Challenges I ran into

The documentation of discord.js was often difficult to navigate, slowing the process of developing desired featues.

Accomplishments that I'm proud of

We created a discord bot that is hosted on a server.

What I learned

I became more proficient with Git-Hub and javascript while learning how to create a discord bot.

What's next for Dungeoneer Bot

Implement further utility, such as tracking character sheets and managing users in voice channels.
