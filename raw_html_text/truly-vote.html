Inspiration

* We wanted to create an application that would revolutionize the current corrupt and unfair election system. TrulyVote was also created to maximize young voter turnout by allowing voters to vote from anywhere. *

What it does

TrulyVote is a voting system that uses Blockchain to ensure security and anonymity. It allows for a decentralized election and prevent outside interference.

How I built it

We built our application mainly using Blockchain technology and using websockets to authenticate users and transfer data from the clients to the main socket

Challenges I ran into

Although we had many technical issues and bugs, lack of time was our greatest challenge

Accomplishments that I'm proud of

Our team is proud that we were able to work with our diverse skillsets and create an application that can have an impact on our society.

What I learned

Our team learned how to effectively collaborate for a group project.

What's next for TrulyVote

We hope to implement our application at LinkNYC stations as an easily accessible voting centers for everyone.
