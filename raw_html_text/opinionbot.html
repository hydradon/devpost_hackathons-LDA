Inspiration

I wanted to build a way for companies and people to easily have a reliable way to poll anything

What it does

I enter something on the website and it will go through a bunch of tweets and run it against some sentiment analysis and check if people view it positively or negatively

How I built it

I used Flask, HpHaven API, Bootstrap, Jquery, Charts.js and azure to host

Challenges I ran into

Limiting the number of tweets and speed

Accomplishments that I'm proud of

Finishing this, learning more about we

What's next for OpinionBot

Better speed, dynamic charts
