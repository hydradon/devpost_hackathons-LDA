Inspiration

Kids ask lots of questions and love animals, so I thought a game that quizzes you about animals would be great!

What it does

Animal Guesser randomly chooses two animals and a trait (weight, height, speed) and asks which animal is heavier, taller, or faster.

How I built it

I built it using the Alexa skill kit and AWS Lambda (in Python)
