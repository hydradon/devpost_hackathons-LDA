Inspiration

We were inspired by the dilution of information that comes from having just world news. We wanted to offer people a simple way to get local news from anywhere around the world.

What it does

It allows a user to type in any address, country, state or city name and obtain news regarding the area.

How we built it

We created a web application that integrates the Google Maps API and the New York Times API.

Challenges we ran into

Formatting various components of the web application so that they could be seamlessly combined.

Accomplishments that we're proud of

The clean and user-friendly layout of the web application.

What we learned

How to simplify tasks by fully utilizing APIs and frameworks such as Vue for JavaScript.

What's next for Deep News

Pulling information from more local news sources.
