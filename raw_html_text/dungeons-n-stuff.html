Inspiration

Our inspiration for creating this project stems from a few games. The biggest influence by far is Dungeon Crawl by Stone Soup, link , as well as other rougelike games such as Binding of Isaac, Enter the Gungeon, Streets of Rogue, and Dungeons of Dredmor

What it does

Dungeons N Stuff procedurally generates the dungeon as you progress through the floors, increasing the difficulty based on shifting probability distributions for mutators that change the stats of enemies, as well as mutators for the main character.

How we built it

We built our game using python for the logic while pygame handles the graphics.

Challenges we ran into

We ran into a lot of issues with python and pygame errors of a wide variety

Accomplishments that we're proud of

We're proud that it's somewhat playable

What we learned

We learned more about what it takes to develop a video game. As well, we learned about using pygame which we can apply to future game projects.

What's next for Dungeons N Stuff

In the future, we're going to add loot, weapons, mutators for more items, more balanced difficulty increases, etc.
