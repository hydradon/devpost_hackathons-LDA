Disaster Support

Inventory distribution, donation and resource allocation of items for disaster victims

Getting resource to victims of natural disaster can be a difficult task. If all items are located in one area this creates a tough environment since people will concentrate on where the resources are being distributed

Our solution

In order to answer to these needs we've created a mobile application which gives the victim the option to create a request of the item needed, and for the volunteers in the area to supply with the resources available.

How our app work

The home screen give you the option to identify who you are (victim, volunteer, donator). Once established, the respective screen of the individual lets them post items or accomplish tasks buy delivering the items.

Tools used

Google cloud platform

To build container gcloud builds submit --tag gcr.io/distaster-aid/container

To deploy gcloud beta run deploy --image gcr.io/distaster-aid/containre --platform managed

Node and Express for web backend

Flutter for mobile development
