Inspiration - The concept of using chatbots for finding restaurants.

What it does - Finds restaurants and reviews in any location.

How I built it - Using AWS Lex and Lambda, along with the Google-Places-Python API.

Challenges I ran into - I initially started with a different idea and then switched to this idea late in the competition.

Accomplishments that I'm proud of - Building my first chatbot!

What I learned - How to use AWS Lex and Lambda, along with the Google Places API in python.

What's next for SuperRestaurantBot - perhaps restaurant recommendations and ordering service with restaurants.

For testing instructions please see the attached GitHub repo. This bot is a Facebook Messenger based application that can be accessed at this Facebook page: https://www.facebook.com/superrestaurantbot/ Just click on "Send Message" to start interacting with the bot.
