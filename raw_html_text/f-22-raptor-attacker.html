Inspiration

Lockheed Martin arcade game challenge

What it does

It is an arcade style game that requires the user, playing as a Lockheed Martin F-22, to avoid large birds. The users receives points for avoiding the obstacles and for hitting birds.

How we built it

We used Processing and linked to a .txt file to save the high scores.

Challenges we ran into

We struggled with figuring out how to display high scores.

Accomplishments that we're proud of

We're proud that we got all the essential pieces working and created a fully functioning game.

What we learned

We learned how to use Github for collaboration.

What's next for F-22 Raptor Attacker

Now that the functionality is working we'd like to create additional features including allowing the user to select a different aircraft. We could also levels of increasing difficulty.
