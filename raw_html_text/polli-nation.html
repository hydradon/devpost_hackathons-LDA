Inspiration

Wouldn't it bee nice to rule the world? And what with bees dying maybe it's time for them to take their own back.

What it does

Starting from manchester, finds the five cheapest flights using SkyScanner's API. Then picks one of these at random, subtracts the cost from your budget and adds it to the map. Depending on which country you land in, you're assigned a different customised quiz. If you run out of money, it's game over no more polli-nation.

How we built it

A mix of node, mapbox and python.

Challenges we ran into

Mapping proved more complicated that initially thought, one of our engineers had problems communicating with the API so was set back a fair bit. Keeping the map highly interactive proved challenging so react-js was ditched.

Accomplishments that we're proud of

Each of us tackled new web application domains we had never touched before.

What we learned

Sleep deprivation only makes bee puns more appreciated.

What's next for Polli-nation
