Inspiration

I wanted to be able to ask Alexa, "How many home runs have the Cubs hit this season?"

What it does

Provides more advanced MLB statistics than Alexa provides OOTB

How I built it

NodeJS application that calls the MySportsFeed API

Challenges I ran into

Parsing response from MySportsFeed Different times of the year for "current" season (resetting stats for the playoffs)

Accomplishments that I'm proud of

Being able to work with regular season, playoff statistics, etc.

What I learned

What's next for Dingers

  • Make the skill adaptable for multiple teams
  • Allow a user to save a favorite team
  • Allow a user to use more statistics than just home runs
