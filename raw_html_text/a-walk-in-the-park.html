Inspiration

Parking should be a walk in the park.

What it does

This app will show you the parking locations that are near by that still have available spots.

How we built it

We used Current powered by GE's API to help us retrieve data in the parking lots about cars coming in/out. We used MySQL to store our data, and Java to process it. We used Tomcat to set up the server using the Spring framework. We used React Native in the Expo XDE to make the app.

Challenges we ran into

Setting up a server.

Accomplishments that we're proud of

Some team members had never used JavaScript before, and picked it up pretty quickly.

What we learned

All about how the Internet works!
