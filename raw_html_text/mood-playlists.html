Inspiration

We wanted to do something involving music, and 3 hours of coming up with ideas brought upon our project

What it does

Uses facial recognition to determine emotion via valence and curates a playlist based on valence

How we built it

python to connect with Spotify API via spotipy

C++ to create facial recognition program via Affectiva SDK

Boost to convert C++ to python

Challenges we ran into

Authenticating a spotify user account

Accomplishments that we're proud of

The amount of work we put on it and determination. In addition, finally authenticating a spotify user account

What we learned

How to use boost, Spotify API needs a lot work for authentication flow

What's next for Mood Playlists

create more parameters to measure mood
