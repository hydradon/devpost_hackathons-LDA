Inspiration

We really like what Microsoft is doing with cognitive services, and wanted to build a human touch into visual studio.

What it does

When something goes wrong in visual studio, it will try to comfort you based on your emotinal status.

How we built it

We built an extension for Visual Studio and hooked it up to webcam, Emotion API and Speach API.

Challenges we ran into

Visual Studio extensions do not play well with webcamera and unsigned nuget packages

Accomplishments that we're proud of

Never giving up
