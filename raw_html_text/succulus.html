Inspiration

We decided to make a game centered around an inside joke on our floor.

How it works

You use the WASD keys to move the vacuum cleaner around, to chase and attempt to catch the figure. As you catch figures, a new one will appear. This will continue until time runs out.

Challenges I ran into

We didn't know any HTML, JavaScript, or CSS, or how to use a Google API, so we had to start from scratch.

Accomplishments that I'm proud of

The game works.

What I learned

We learned how HTML, JavaScript, and CSS work, along with how to use the Google Maps API.
