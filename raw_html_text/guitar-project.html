Inspiration

When we open our fridge, we don't always know what we want to eat. Inspired also by being hungry, sitting at an isolated table at PennApps.

What it does

Uses machine learning to recognize and identify key food staples. Then, it can provide recipes that the user can easily cook using the ingredients in their fridge.

How I built it

Using TensorFlow, Keras, and Python. Found fruit image dataset online, other datasets (vegetable, condiments, eggs, meat, dairy) created by us.

Challenges I ran into

Creating own datasets from scratch. Learning how to use TensorFlow and Keras.

Accomplishments that I'm proud of

Learning how to use TensorFlow and Keras.

What I learned

TensorFlow and Keras.

What's next

Need to classify multiple objects in one image within the categories we have currently.
