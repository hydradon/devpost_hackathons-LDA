Inspiration

We wanted to make picking up litter something people wanted to do.

What it does

It gamifies picking up litter by breaking it into two teams. Finders find litter, take a picture, and post them on the map. Sweepers hunt for the litter spotted by the finders and "sweep" them by throwing it away. Both of these actions reward points.

How we built it

We used Android Studio to build the front-end of the app and Flask to make the back-end database.

Challenges we ran into

We have difficulties integrating the database into the front-end. We switched to Android Studio from React Native because of technical difficulties.

Accomplishments that we're proud of

We built and designed a functioning app that used a map interface.

What we learned

We learned about how a back-end database operates. We also discovered how to harness the Google Maps API and furthered our knowledge of Android Studio.

What's next for Finders Sweepers

We better integrate the in-game currency, further incentivising users. Also, configuring walking directions to allow easier travel to litter. Possibly going more in-depth on the game reward system through adding teams and levels. Finally, adding the Litter Wall, along with photo taking.
