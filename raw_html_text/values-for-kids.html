Inspiration : Kids nowadays are so glued to smart devices around, we want to design an app for them that would teach them moral values in an interesting way.

What it does:

Parents have to set up accounts and pick the values they want to inculcate in their children It Includes interactive videos with story-telling that has certain questions which child has to answer and gives different outputs according to incorrect answers, Gives tasks to kids to complete for the day. Parental control on how children use the app.
