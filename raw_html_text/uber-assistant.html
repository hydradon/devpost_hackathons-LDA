Inspiration Provide an intuitive interface to book uber and a lot of value adds

What it does User can book uber by "Ok google" on any android phone

How we built it We built it using google api.ai, google actions api, uber api and python flask asthe web hook

Challenges we ran into As this is a nascent area, there is little documentation and sample code. Also there are integration challenges as google actions client sdk is only in node sdk. Further we faced some challenge in training the program

Accomplishments that we're proud of We succesfully used google ai.api and google actions api to process natural speech and integrated with uber api for the first time. A lot of firsts :)
