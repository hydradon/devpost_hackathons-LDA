Inspiration

Agricultural industry remains quite under-developed in the field of technology. Technology brings an opportunity to develop better agricultural practices. The inspiration for this project comes from the desire to create a social impact in the field of agriculture, especially in developing nations, and promote better and cheaper technology for agriculture.

What it does

HiFarmer is an IoT platform that aims to make it easier to setup monitoring system for small scale farming like organic farming and rooftop farming.

How we built it

The software has been deployed using Azure's Ubuntu Virtual Machines, running Node.js application. The statistical data has been stored on MongoDB. The application is accessed using front-end platform built using Angular.js
