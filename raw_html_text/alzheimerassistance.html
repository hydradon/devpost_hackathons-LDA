Inspiration

We wanted to improve the quality of life for both patients suffering from Alzheimer's disease as well as their families.

What it does

It engages in a conversation with a patient trying to figure out what might be the cause of his problem or concern, taking into consideration the fact that it is likely that cause of one's problem is different than usual person, as forgetting of small yet important tasks may be involved.

How we built it

Using Docker. The whole program was built using Python and Cisco Spark website as a mean of communication with the bot.

Challenges we ran into

Docker was really hard to set up at first and Python (as a new language for us) was a bit confusing at the beginning, but with time, we quickly grasped the main concepts of it.

Challenges for: Cisco,Accenture,HackCity18

Accomplishments that we're proud of

Actually building the bot in 24h hours

What we learned

Python, how to use Docker and how teamwork can help in achieving goals

What's next for AlzheimerAssistance

Machine learning and a wider scope of functions such as quite detailed reminders of upcoming events
