Inspiration

We saw an opportunity where we could help a user quickly find a way to the top.

What it does

The user enters in their cards, using our auto-completing search bar, along with a powerful expansion pack feature.

How we built it

We built it using Node.JS, Angular.JS, and Git Version Control.

Challenges we ran into

We started out attempting to use the React Framework. Over a short period of time, we realized that the learning curve may have been a little too steep for us to overcome within 24 hours. We also ran into issues with git version control. Multiple times, with 3 different people working on one project, we ran into merging issues, that took quite a bit of time to overcome.

Accomplishments that we're proud of

Starting and completing a project to the best of our ability in under 24 hours.

What we learned

Some of the team learned Node.JS, while others started out with the basics and caught up quickly and gained the valuable experience

What's next for NetDeckWins

Possible improvements in the searching and matching algorithm. Along with a more robust search platform.
