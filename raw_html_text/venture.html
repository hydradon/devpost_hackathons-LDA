Inspiration

Incentivar jovens a entrar no mercado financeiro.

What it does

Um app para os pais acompanharem o início das atividades de investimento dos filhos. Além disso. é um bot do messenger que auxilia o jovem a realizar as transações financeiras.

How we built it

O aplicativo para o pai foi feito em swift e possui um servidor escrito em python para se comunicar automaticamente com os filhos.

Challenges we ran into

Demoramos para encontrar um projeto com o tema do Hackathon. Então, começamos a programar as 2h da manhã de domingo!

Accomplishments that we're proud of

Começamos a programar as 2h da manhã de domingo e conseguimos fazer o que pretendíamos.

What we learned

Swift, Messenger API
