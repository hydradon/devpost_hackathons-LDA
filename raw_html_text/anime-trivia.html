Inspiration

Trivia games test memory in both a fun and meaningful way. This trivia game uses anime content to engage kids while also training their memory.

What it does

Asks trivia related to anime (Japanese animation).

How I built it

I used Python 3.6, AWS Lambda, the Alexa platform, and opentdb for the content.

Challenges I ran into

I've built trivia games before so this was another way for me to provide content to a different audience.
