• Time to learn time

Inspiration

The other day I was at my cousin's place where my aunt was just done with teaching my cousin how to read time. At that time I realized that they might have been skipping some major aspect of reading/interpreting time.

What it does

The skill shouts some random time phrases which the child is expected to tell it back.

How I built it

With all the hard work and patience.

Challenges I ran into

Since I did not had Echo, it was quite tough for me to test skill with the actual user experience.

Accomplishments that I'm proud of

After I gave demo for this video, 5 of my neighbors ordered echo right there. They were so impressed with the concept that they couldn't resist. On looking at the usage of this skill, they were so happy that now their child can practice time anytime. Someone apart from them would be able to help assist their child learn time.

What I learned

How to work with AWS lambda. How Alexa could be beneficial.

What's next for Time Phrase

Will add more cases where children can not only answer the questions but get hints, learn more about time. I also plan to roll out a version for Echo Show as well for better understanding.
