Inspiration

We wanted to make an AI-empowered workflow assistant to assist you with automated task to help you generate presentations, create voice memos, etc.

We feel that the Google Assistant is more akin to a Google Maid. It doesn't really fit within a professional environment. Our app serves as a logical extension of AI-connected voice assistants.

What it does

Our application's primary value lies in its ability to connect you with your most important tasks. It gives you the ability to create data visualizations based on speech commands. Another application is the creation of lists/memos based on speech patterns. This app also gives you the abiity to shop for items on InstaCart

How we built it

We built it primarily using UIPath and VoiceFlow. VoiceFlow's API was used to collect speech from users and parse it. Firebase was used to route data and prepare it for use by UIPath. UIPath was instrumental in automating tasks such as chart creation.

Challenges we ran into

We ran into a lot of challenges using UIPath. It was particularly difficult to use Visual Basic combined with UIPath Studio. Connecting it to Firebase proved to be a challenge as well.

Accomplishments that we're proud of

Successfully connected UIPath and VoiceFlow

What we learned

We learned a lot of Visual Basic and the amazing utility of Firebase

What's next for Flow AI

Expanding feature set to include richer options for presentation creation, ordering lists on third-party apps such as InstaCart, etc.
