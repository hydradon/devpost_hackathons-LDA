• Rate My Hackathon

Inspiration

We have been wishing this app existed for the past few hackathons we have attended. We were inspired by the hard work and energy the organizers and sponsors put into making these events so useful to our community.

What it does

Allows hackers to rate their hackathons à la the famous Rate my Professors website.

How we built it

We built the backend in Django. Although Flask or Node may have been better choices for this small-scale app, we have been trying to learn Django all semester so it was a great opportunity to put our theoretical knowledge into practice.

Challenges we ran into

We had never used JavaScript on the frontend to interact with the SQLite database in the backend. We solved this by mostly server-side rendering. If we had more time, we would have solved this with AJAX.

Accomplishments that we're proud of

We're proud that we succeeded in only 24 hours.

What we learned

We learned more about Django and JavaScript.

What's next for Rate My Hackathon

We would like to continue to add features and styling to eventually be a clone of Rate My Professor. Social media integration and spam prevention are also stretch goals.
