Inspiration

I wanted to use an app that utilized Google Cloud Platform.

What it does

Takes common phrases and allows users to see the translation and hear it spoken.

How I built it

Node.js, Angular, Google Cloud Platform.

Challenges I ran into

Lack of documentation for the Google Text-to-speech API

Accomplishments that I'm proud of

Creating a project by myself. Having something demo-ready. Overcoming challenges with the translate and text-to-speech API.

What I learned

Google Cloud Platform has a lot of capabilities, and keep trying.

What's next for TravelBuddy

More languages, more phrases, better UI.
