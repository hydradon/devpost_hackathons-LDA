Rent_and_Safety #RAMHacks2018

Our goal was to create an web application which takes in housing data provided by CoStar and predicts the rental cost using machine learning and spacial data analytics kriging. We also analyzed the relationship of features in the CoStar database. We stored our data into MongoDB from which we created an api utilizing MongoDB Stitch. Using api calls from publically available government data, we were able to map flooding and crime to create a heatmap. This application can predict rental costs for housing with unknown market rent price.

Challenges Attempted:

CoStar, MongoDB, Markel, WillowTree

TL;DR: Mapping rental houses and safety through various api
