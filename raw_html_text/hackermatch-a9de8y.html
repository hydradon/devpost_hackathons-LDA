Inspiration

Make it easier for students attending a hackathon to find a team to work with at a hackathon

What it does

A student can create a profile for themselves that includes the coding languages they know, coding languages that they don't know, and what level of competition they are seeking during the competition. The user then will be able to search for team members by the parameters of their choice in order to create the team with the most compatibility.

How we built it

We coded the backend with Java and are planning to integrate the Java code into a slack app.

Challenges we ran into

We spent the first 3 hours of the hackathon with a totally different idea and ended up having to change ideas, java doesn't integrate well with Slack apps

Accomplishments that we're proud of

We got a button on the webpage, we were able to successfully integrate our app onto our slack team, we understand the logic involved in integrating the java code with the slack app.

What we learned

We learned Version Control, more about java, more about webapps

What's next for HackerMatch

Working to get the code we wrote working in a slack app.
