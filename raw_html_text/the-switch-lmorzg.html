Inspiration

we decided to build an emp for security purposes so we can lear to use hardware and software more as a unitary thing instead of us two knowing them individually

What it does

it sends out a short ranged emp that has the potential to short out a computer incase of theft

How we built it

we used a disposable camera and made a very weak emp to show the concept and then linked an arduino to the emp s it could be controlled by the software we built. we also installed a hard switch for a safety measures.

Challenges we ran into

unexpected scripting errors and getting the emp to actually fire.

Accomplishments that we're proud of

we built an emp.

What we learned

how to combine both software and hardware to work in tandem to provide a security measure

What's next for The Switch

compaction, more power, and a remote activating trigger
