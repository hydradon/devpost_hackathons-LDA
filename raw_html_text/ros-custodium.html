ros-custodium

An arduino/esp8266 distributed moisture sensor network.

  • Easily extensible with additional sensors
  • Easily interoperates with various web frameworks
  • Easily self-hostable

Designed to be used with esp-8266 boards (a Wemos D1 mini is pictured).
The sensor is only powered when the server polls the micro-controller. This is to increase the longevity of the sensor prongs.
A sample polling server (python2) is also provided that interfaces with the esp-8266 to demonstrate this ability.
These sensors are meant to be on their own network, unattached to the wider internet. Eventually, I will get around to uploading a wiring schematic.
