Inspiration

We wanted to do something with LeapMotion, so we started creating a game and then we ended up creating a ping pong game.

How it works

The LeapMotion device lets you hold an "air ping pong paddle" and play ping pong only with a computer and a leapmotion device!!!

Challenges I ran into

I had to do some math when creating the terrain. Connecting with the LeapMotion API was a challenge as well.

Accomplishments that I'm proud of

Terrain development, LeapMotion API

What I learned

Unity, LeapMotion API C#

What's next for Ping Motion

We're going multi-player soon! We're working on functionality to allow you to play with your friends online.
