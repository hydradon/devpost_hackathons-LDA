Inspiration

The main feature is adopting blockchain skills to create gaming. We inspire from Tamagotchi. Payers are able to play and feed the cat to become the lion.

How we built it

Using Skale instead of Ethereum Virtual Machine to make transactions faster, and using Portis wallet instead of a common MetaMask wallet.

Challenges we ran into

We have the challenge to make cats level up in Skale and make Portis wallet successful work on. In the end, we solve the problems

What we learned

Skale is much faster than any Ethereum Mainnet and Testnet which is very impressive. When we use the Metamask wallet which we have to press the confirm button while we make any single transactions. In order to remove the annoying pressing button all the time, we adopt the Portis wallet that auto-confirm once we make transactions.
