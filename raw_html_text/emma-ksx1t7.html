Inspiration

We're deploying Docker containers daily and we miss a decent ChatOps to deploy them smoothly.

What it does

It notifies you on Slack when your images are build on Docker Hub. You can chat with the bot and deploy these images on AWS Elastic Container Service.

How we built it

We've build the frontend in React.js. The bot has been build with Node.js, Slack realtime api and the AWS SDK.

Challenges we ran into

Deployment on Elastic Container Service and integration with IBM Watson. Eventually we've dropped Watson because of a lack of time :).

Accomplishments that we're proud of

Automatic deployment on ECS

What we learned

React.js and AWS SDK for the container service.

What's next for Emma

More natural interaction and deployment on other cloud services like Google and Azure. Also use the Tutum API.
