Inspiration

Sometimes people feel bored, sometimes people don't know what to do. We aim to make life more interesting than it is!

What it does

SurpriseMe will give those who feels bored a suggestion what to do next. It may give the user a joke, a book, a film, a music or a restaurant upon the surpriseMe button is clicked.

How we built it

We used python crawler to grab information we need from internet and store them in a JSON file which will be displayed in a web application.

Challenges we ran into

We are first-hackers and don't know how to come up with a brand new idea. Also there is a difficulty using python crawler to find the right information.

Accomplishments that we're proud of

Successfully builded a web application using a .me domain.

What we learned

Teamwork is important and everything takes far more time than what you think it would take.

What's next for SurpriseMe

We want to take a like button and memorize all the choice the user has made and generate a more suitable suggestion based on the likes and dislikes and the user. Also we want to add chatting so that two people clicked surpriseMe at the same time can talk with each other!
