Inspiration

I was inspired by the idea this winter when I noticed that many pedestrian audio signals do not work properly and even if they work the sound sometimes cannot be heard due to traffic.

What it does

The application uses the camera to detect pedestrians. signals and the mobile says"GO" or "STOP" depending on the signal.

How I built it

I trained the algorithm using photos of pedestrian signals that I took in the street and used the prediction API to build the app.

Challenges I ran into

Learning how to use computer vision software and android studio.

Accomplishments that I'm proud of

I am proud that I learned a lot about computer vision and used it to make a useful application.

What I learned

I learned android studio and Microsoft's computer vision program.

What's next for Blind Assistance Application

To use more datasets of pedestrians signals photos to make the program more accurate.
