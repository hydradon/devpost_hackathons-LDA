Inspiration

With some experience with elderly, we were inspired to solve the problem of forgetting to take or overdosing on medication.

What it does

We built a simple machine that takes data from a server in order to dispense medications at set intervals in the proper dosage.

How we built it

Using a Dragonboard and an Arduino

Challenges we ran into

We had many challenges with the Dragonboard wifi, the arduino modules and flask

Accomplishments that we're proud of

Getting the code to work for the Arduino

What we learned

How to use arduino and a dragonboard

What's next for Pillinder

Expand it into a ui and also use predictive technology for detecting when someone might be having a medical emergency.
