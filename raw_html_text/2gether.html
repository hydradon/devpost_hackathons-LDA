Inspiration

Sky TV and AOL

What it does

2togethrr creates and sustains communities

User searches 2gethrr database ( SKY and AOL )

User rates the search results

User then selects an item or items to share with friends

_ For Free content: _ User shares with friends

_ For Premium: _ User buys the item and sends to friend

How we built it

Watson API, SKY API, Node JS

Challenges we ran into

Working with different datasets into one database. Developing several endpoints using variable API's

Accomplishments that we're proud of

We got the project working, with search query

What we learned

Team work. All the API's can integrate and interoperate, without need for silos

What's next for 2gether

To develop it further and stay together
