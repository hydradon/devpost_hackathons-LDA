Inspiration

As per my nature and my past community services that I have done in my life, I really liked the idea to help and develop something for a non-profit organization that is really moving forward to donate and help underprivileged people. I have also done community service for 1 year in India to help underprivileged people and this is where from inspiration to opt this project for this hackathon came from.

What it does

Currently the app displays almost everything that is on website. Home page, how Donii works, Contact page, Donate link, Stories, News as well as Team.

How I built it

It works on Android and iOS as it is developed in Ionic framework using HTML5, scss and cordova as a server. Currently database is not integrated as there were no such transactions required in this app. But this app can be taken to a next level by implementing few more features.

Challenges I ran into

I was not able to configure the ionic server at first but after watching few tutorials I was able to see where I missed few points.

Accomplishments that I'm proud of

I was able to complete such a good application that can work on both the platforms android as well as iOS in few hours without using native functions. Though Ionic uses Native functions of iOS and Android I don't have to go to learn there as I know Angular and HTML5, I was able to build using that technology.

What I learned

I learned how to achieve the functions of native application work by using functions from the external framework.

What's next for SPARK-- Donii Android & iOS app

The next step would be retrieving the live data for News section and finishing up the Contact Us form. As well as to put the Gallery in it which can be updated regularly. In addition, to send notifications as well to the users. This would keep users in the loop and updated. Currently notifications is implemented but has not been tested due to time constraint.
