What it does

This project consists in facilitating the tasks of the persons dealing with the car parks in establishments or private car parks, each user having a car must have an account on the application, and there will also be administrators to manage the accounts of the users, for each captured license plate, an image processing is done to it to be tested if it is in the database or not.
