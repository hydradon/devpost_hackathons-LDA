Inspiration

Just felt like making a game.

What it does

Connects clients through websockets to allow them to play together.

How we built it

We used the Phaser.js game dev framework to implement game logic and rendering. Communication and game syncing was handled through websockets and a node.js backend

Challenges we ran into

Our idea shifted multiple times

What we learned

Games are harder to make than we thought. We learned a lot about Phaser.js
