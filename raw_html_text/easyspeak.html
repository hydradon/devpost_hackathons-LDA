Inspiration

Applications for non-verbal individuals are too expensive and limited in functionality.

What it does

EasySpeak allows non-verbal people to communicate using text to speech, as well as give them a way to connect to the rest of the world beyond the app.

How we built it

Java

Challenges we ran into

Java

Accomplishments that we're proud of

Using google's location API to give EasySpeak and its users a way to interact with nearby locations. e.g. if the user wants to eat at a restaurant, EasySpeak will use location data to suggest nearby options for the user to communicate.

What we learned

How to develop Android apps.

What's next for EasySpeak

Predictive text to speech Using muscles sensors to give accessibility to individuals without fine motor skills.
