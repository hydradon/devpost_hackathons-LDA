We were inspired by

We were inspired by the Best Friends panel and their mission to end the killing of dogs in shelters.

What it does

Matches dogs looking for new homes with individuals looking to adopt

How we built it

Created web page and connected it to a database of users and their pets. We shuffle though these pets and allow users to view pets and their details and "like" or "dislike" them.

Challenges we ran into

We had to figure out how to run nodeJS and mongoose, as well as making sure all team members knew JavaScript.

Accomplishments that we're proud of

We finished this program in 1 evening and taught ourselves how to use databases and JavaScript

What we learned

JavaScript and Database stuff

What's next for woof

We would like to expand our horizons, and maybe one day include cats.
