Inspiration

The US is riddled with gun violence. There are over 1,000 incidents per week. Half end in injury, a quarter end in death.

What it does

Web spider collects data on gun incidents, adds the data to MongoDB and Flask backend of an api hosted on Heroku, then a Unity VR project makes a call to the API to populate the visualization.

How we built it

web spider in python, api in python, flask, mongodb, heroku, unity VR project with cardboard sdk, all custom assets in Blender

What's next for Visualizing Gun Violence

More accurate assets that match the description, allowing the visualization to compound over time.
