Inspiration

Promote healthy lifestyle.

What it does

My Healthy Cart is an Android OS based mobile application that helps you to take care about your dietary and fitness requirements. Based on your allergic problems and existing health affecting diseases, our application will suggest you certain dietary and exercising aspects to maintain a healthy lifestyle.

How we built it

We built the user interface using android OS on android studio platform and connected it with the Django python framework. Database used is postgres.

Challenges we ran into

Collaborating Android and Django framework using request, response parameters using REST API framework.

Accomplishments that we're proud of

Speech-to-text translation using Google Translator API and collaboration of Android and Django Python framework.

What we learned

Android studio, Django python framework, integration of both frameworks, speech-to-text translation

What's next for My Healthy Cart

Providing dietary and fitness suggestions based on 1 or more allergies and diseases. The user will be able to order for dietary requirements from our app and can set a particular time interval to refill the requirements.
