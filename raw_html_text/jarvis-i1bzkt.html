• spare the air

Inspiration

We wanted to work with Alexa on a project together and thought this would be fun

What it does

To check the air status to see if wood burning is allowed in the San Francisco Bay area

How I built it

We used python, flask-ask, zappa to build and install the application.

Challenges I ran into

We had parsing problems with the HTML and I ran in a problem with zappa. I had to install lxml in the virtual environment.

Accomplishments that I'm proud of

We were able to have Alexa relay the information to us.

What I learned

The syntax has to be exact; Otherwise you have problems with the utterances and context.

What's next for sparetheair

This is a completed project but we are looking at other projects to do with Alexa.
