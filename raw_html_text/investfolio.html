Inspiration

Easy to use. Free.

What it does

Uses machine learning to take in stock data to predict future stock prices and determine whether to hold or sell the stock.

How we built it

Using sublime, atom, emacs, codeanywhere, google cloud, python, php we were able to build a database of stock data and import it into an algorithm which exports to an api that we created.

Challenges we ran into

Downloading and finding stock data, computations with large amounts of data, accessing python within php

Accomplishments that we're proud of

Use of apis, mysqldb, and interfacing with python

What we learned

How to interface with mysql using python, interface php with python, working on different parts of a script at the same time

What's next for InvestFolio

Expand the scope of our stock data, incorporate more classifiers into algorithm and increase accuracy, increase efficiency
