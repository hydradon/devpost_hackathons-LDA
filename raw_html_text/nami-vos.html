Inspiration

Mental Illness is a very serious problem, that affects many people around us. Building an efficient data management system for the National Alliance of Mental Illness will allow them to focus on helping more people in need, instead of tracking down tedious paperwork and excel sheets.

What it does

NAMI-VOS is a data management portal for National Alliance of Mental Health to manage their volunteers, donations, program participants, and community members. It provides a centralized, secure, and convenient system for submitting, tracking, and analyzing data that is critical to NAMI's efficiency as an organization.

How we built it

We used NodesJS with a PostgreSQL database, Redis for sessions, Bootstrap for style, and jQuery for UI functionality. The application is also secured with Bcrypt password hashing, encrypted session cookies, input sanitation/validation and AES 256 sensitive data encryption. The live demo is hosted on Heroku.
