Inspiration

What it does

Using a camera and an Vuforia computer vision, Project Anicca presents you a mystery world for exploration in augmented reality. Users can observe game world in 3D and control the game character, Anicca, to find a path out of the game.

How we built it

Project Anicca consists of several major parts: Game logic controller 3D model

Challenges we ran into

Implementing game logic on a transformed, constantly changing plane. Binding the three parts together was a big problem.

Accomplishments that we're proud of

Beautiful world from modeling Game control Combine game and AR

What's next for Project Anicca

A different mode of viewing the game world More sophisticated scenery More game levels
