jukeBot

A smart playlist generator that makes a collaborative party playlist everyone loves. This project attempts to improve creating music playlists for events and house parties that do not have a budget for a professional DJ, by pulling data from the Spotify API for multiple users attending the event.

It is called using the nuance mix-nlu text to speech API and uses weighted scores and top tallies to find overlapping songs from multiple user lists to optimize the playlist.
