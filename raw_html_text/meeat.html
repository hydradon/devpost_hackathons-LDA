Inspiration:

  • Connecting UST community and promoting learning through collective intelligence

What it does:

  • By matching and inviting students to have lunch and share sessions.

How we built it:

  • An Android app built on Azure cloud, Github and Skygear.
  • Collaboration between members that each possess different skill sets and tech stacks.

Challenges we ran into:

  • Constructing algorithms to run through member's lunch schedules
  • Coding on time!
  • Integrating Google API

Accomplishments that we're proud of:

  • Integrating Google Calendar API to our app's scheduling functionality,
  • Understanding and utilizing robust features of Skygear as an infrastructure to launching our app,
  • Incorporating Azure cloud platform to sustain our app's computation and traffic.

What we learned:

  • Skygear's full stack service required to launch an Android app,
  • How to utilize Azure cloud for the build,
  • Collaboration between product manager, front & back-end developers, and designers.

What's next for Meeat:

  • Integrating SSO using school ITSC login accounts and class schedule
  • Develop push notification to announce lunch matches
  • Automatic calendar event creation upon matches created
