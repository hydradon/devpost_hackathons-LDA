Inspiration

We used case studies from Penn State and NC State University to safe a massive amount of time and effort in retrieving books from the library.

What it does

A concept for implementing best practice technology involving a robotic arm and organized space saving shelving is able to find and reach any book in the library within seconds, which frees up space for more student areas on the Fourth Floor of Evans.

How we built it

We designed new uses for the saved space within AutoCAD.

Challenges we ran into

We found it difficult to agree on the needed uses for the new available space, as well as to accurately proportion the new spaces created.

Accomplishments that we're proud of

We were able to motivate each other and get excited about the new things we would be able to have with this new system while at the same time proposing changes to a space that we all personally value on campus.

What we learned

We learned by studying other examples, reallocating the space is better that building something completely new.

What's next for Stack It!

We hope to see some of these changes soon, and to be used and enjoyed by every student who comes to the library!
