Inspiration

The inspiration comes from the idea of the 20-20-20 rule. Every 20 minutes, take a 20 second break and look at something 20 feet away. We were also inspired by Chris's choice of great music.

What it does

This app is used for giving the user a break from the computer, phone, or work. An alarm is set and can only be turned off by some physical activity.

How we built it

We used Android Studio to create the app.

Challenges we ran into

Major challenges include:

  • Ensuring all AS software is on the same version
  • Using SQLite
  • Sensors on a phone
  • Debugging on Android Studio
  • Coming up with an idea
  • Assets and layout

Accomplishments that we're proud of

Good collaboration and focus on the project. We got something done!

What we learned

It was good to review using Android Studio again. Fast pace environment and time management.

What's next for Break-Habit

Keeping track of personal statistics. Optimize the application. Random exercise and amount choices. Keep track of the timer in the background.
