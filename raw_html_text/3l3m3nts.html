Inspiration

We got the inspiration from our curiosity for learning new things. We were thinking about a project that could be interessant enought and also funny and that could show us a big range of thins so we thought the best way, without beeing ambitious, was to make a game which combined both things.

What it does

We createD a game that, it's first idea was to start it with a protagonist as an ice cube which during the game would develope some habilities that would allow it to change its state(ice, liquid and gas) that would simplify it's hability to defeat it's enemies. Which, of course, would be upgraded and more dificult to kill as the game takse part. However, as we were not familiarizated with unity, it takes us a lot of time to get used to it and to develope with agility so we could do a little developed scheme of the game which allows you to experiment the state changes and fight with some enemies. It is more or less what we expected but ther is still a lot of work to do to make it real.
