Inspiration

We were inspired to build this website because it is hard to find great places to take pictures.

What it does

It allows you to find and upload places with great backgrounds for taking pictures for your social media or creating professional videos.

How we built it

We built this by using html, css, and javascript to create the website and using file stack and cloud boost to store the data.

Challenges we ran into

One challenge that we ran into is splitting up the work between us and being able to work together.

Accomplishments that we're proud of

We are proud of being able to create a project that can help so many people.

What we learned

We learned how to use File stack

What's next for Photogenic

The next step for photogenic is to create a working mobile app.
