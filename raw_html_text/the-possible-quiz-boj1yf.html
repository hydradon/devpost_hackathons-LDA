Inspiration

I loved the Impossible Quiz when I was growing up. Although I am not experienced enough to make all the effects, I think the questions are like what you would see when playing it.

What it does

This is a text-based quiz where you type in your answer or multiple choice. It calculates your score at the end and has a replay option.

How I built it

I built this code from scratch using the Java language on the Eclipse software.

Challenges I ran into

I originally attempted to make this a GUI, but an error popped up and I tried to fix it for multiple hours. I had two hours left, and had nothing done. I built this entire program in two hours and managed to get it done.

Accomplishments that I'm proud of

I am proud of completing this alone in two hours . I typed faster than I ever had and barely finished it in time.

What I learned

I learned the basic JFrame code (GUI) and how to use loops to my advantage to type less.

What's next for The Possible Quiz

I may add some questions, and hopefully make this into a GUI!
