What it does

It converts visual text to speech, mainly for the blind.

How we built it

We used a webcam, raspberry pi, python, and google-cloud (text-to-speech,OCR).

Challenges we ran into

Connecting to the Raspberry pi via ssh. Implementing text-to-speech and OCR.

Accomplishments that we're proud of

This can actually help people, and it is cheap enough for people to use.

What we learned

Google-cloud has amazing capabilities, and you do not need to know how OCR and text-to-speech works to make something amazing!

What's next for BlindSight

Combine parts to improve form factor.
