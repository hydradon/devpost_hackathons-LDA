Inspiration....

I always park my car in a huge parking lot where there're hundreds of hundreds of cars near me and I get confused where I park my car.

What it does

We developed an android app that can mark the location and when we get back, it is able to show us which direction we need to go and the distance between the marked spot.

How we built it

We utilized Fused Location Provider API, location, GPS and network service under Android Studio.

Challenges we ran into

At first, we used network service and the location is not as accurate as we thought. Then we tried to use Fused Location Provider API and GPS service along with network service depending on their accuracy.

Accomplishments that we're proud of

1.Fused Location Provider API 2.Make our result accuracy as possible 3.make our app smooth
