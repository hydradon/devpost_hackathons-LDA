Inspiration: We all really like games and we wanted to make something unique.

What it does: Puts a twist on normal checkers by adding two extra players and totally changing the rules.

How I built it: Node, JavaScript, jQuery, HTML, CSS, SocketIO.

Challenges I ran into: Game logistics, HTML headache

Accomplishments that I'm proud of: Getting it all to work seamlessly across multiple devices.

What I learned: Server-Client relationships are very delicate

What's next for CheckersSquared: Maybe a sequel game?

Note: Need 4 players to start a game
