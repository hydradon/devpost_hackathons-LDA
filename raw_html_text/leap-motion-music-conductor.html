Leap Motion Music Conductor

Inspiration

I (Keely) never used leap controller before, and because MLH gave access, I took the opportunity.

What it does

Upper bound of Leap range plays instrument 1, lower bound plays instrument 2. Left-right controls note.

How I built it

Plugged in Leap, looked at api. Created bounds of each note, notes play based on hand.

Challenges I ran into

Originally I wanted to do it in python, but could not get the controller to connect to the python api or get any sound libraries to work quickly on my machine.

Accomplishments that I'm proud of

It is fun to play with and looks cool when using it.

What I learned

How to use a leap motion control and what it is capable of doing. (Also brushed up on JS)

What's next for Leap Motion Music Conductor

I won't have access to one after the hackathon, so it will likely sit on GitHub dormant.
