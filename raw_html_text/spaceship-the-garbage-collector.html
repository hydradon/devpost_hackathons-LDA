Inspiration

To have a game to educate people about garbage classification and environment protection.

What it does

Controlling a spaceship (WASD and arrows) and its collector type (use space to switch) to collect the right type of garbage to provide energy for the spaceship to travel in the universe.

How we built it

With canvas in javascript, github and visual studio code.

Accomplishments that we're proud of

Intereting and challenging gameplay, engaging look and feel.

What we learned

How to build a game with javascript.

What's next for Spaceship | The Garbage Collector

Add more featrues to the game to make it more interesting to play, such as adding a global leaderboard for all players and more types of garbages.
