Inspiration

We like nice clothing but we are universty students. We wanted to be notified of mind blowing clothing deals via Text.

What it does

You input your name, phone number, along with brands you want to track. Our app constantly scrapes the web and texts you amazing clothing deals as it find them.

How we built it

We used:

1) Google Cloud Functions to generate APIs and orchestrate everything 2) Reddit API wrapper to fetch data 3) Pandas to filter and clean data 4) Twilio to notify users via text

Challenges we ran into

We had major pivots throughout the night. We seemed to be constrained of computing power for our first couple Big-Data ideas. We also ran into issues in manually scraping Twitter for large datasets.

Accomplishments that we're proud of

We are proud to have used multiple APIs in the project and expand our learning horizons by utilizing Serverless computing.

What we learned

We learned to be comfortable in using unknown technologies.

What's next for Broke Boi

We would like to finish the implementation of periodic fetches of deal data and be more consistent in texting deals.
