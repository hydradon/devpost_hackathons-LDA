• Findr

Inspiration

It is common for us to forget the movie titles when mentioning it in some conversations with friends and peers. We generally remember cast or some descriptions of particular interesting scenes in the movies. This is when we need the bot Findr

What it does

It finds the movie titles based on a given description.

How we built it

The bot asks for small description of a movie one is looking for. It Lemmatizes the responses from the user and categorise them accordingly. These are then ran against a classifier that finds the probable movies and ranks them. It then presents the results to the user with direct links to trailer or stream the movie online over Netflix or Amazon Prime.
