Inspiration

We wanted to use sensors and see what we could do with them

What it does

Currently it is a mobile application and a separate application that returns data from the sensor, but the goal was to have an application that would tell users if their TV or stove was on in their house

How we built it

We used Swift to build the mobile application side and C to program the sensor

Challenges we ran into

We couldn't run any code using the sensor for a while, and after that we were unable to convert its readings into the correct values

Accomplishments that we're proud of

Flashed code to the sensor and created a bare-bone working UI

What we learned

Working with new hardware is difficult

What's next for What's On?

Hopefully we can link the sensing code with the application and have a fully deliverable product
