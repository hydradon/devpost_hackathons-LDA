Inspiration

For our Cutiehack project, we wanted to do something that allowed us to utilize our introductory skills to the C++ language. We also combined our collective interest for personality quizzes and the Harry Potter series to create inspiration for this project.

What it does

The Sorting Hack is a quiz that determines your placement into one of the four houses in Hogwarts, a fictional wizarding school in the Harry Potter series.

How we built it

We used many functions in C++ to create components for the quiz. We also used part of an HTML template and added some extra CSS and JS to make it look as it does. Lastly, we used IExpress to create executable files and an installer.

Challenges we ran into

Trying to debug the program was a very tedious process, along with making questions that would fit the quiz.

Accomplishments that we're proud of

We are proud that we were able to do something with our current programming knowledge.

What we learned

We should believe in ourselves more since we could create more than what we could think.

What's next for Sorting Hack

We are trying to make a website for it so it can be available for download.
