Inspiration We wanted to use cloud vision api. Wanted to make catalogs mobile

What it does Gives information on Movies, Books, and Video Games by taking a picture of it

How we built it We used the cloud vision api to recognize whether the image was an image of a film, book, or video game. We then used the omdb api to get information on Movies. We got information through parsing Wikipedia with the Media Wiki API to get infromation on non-films

Built With

  • google-vision-api
  • mediawiki-action-api
  • omdb-api
