Inspiration

We all don't like walking home alone.

What it does

Connects people to others walking home on a similar path

How I built it

We used Node.js, JavaScript, CSS and html.

Challenges I ran into

We all had other commitments that gave us a limited amount of time to complete the project.

Accomplishments that I'm proud of

We finished in a limited amount of time.

What I learned

What's next for safewalk

We want to expand our functionality to allow more complicated paths. We want to add 'pop-up' locations that are created based on user demand. We also want to add login with facebook.
