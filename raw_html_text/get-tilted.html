Inspiration

We initially started out wanting to connect multiple phones in a classroom for an educational setting, but looking into physics we eventually just turned it into a game

What it does

Players can enter a room with up to three other friends. They can then race through a track on their phones to see who reach the end first, if they can even survive.

How we built it

We split the work into three parts: network, game engine, and level design. Michael set up the network to connect between players, Silas set up the game mechanics, and Xavier designed the levels.

Challenges we ran into

Originally, other players were going to have different controls and functions, but due to Unity Bugs and lack of a good control scheme, we had to scrap that.

Accomplishments that we're proud of

Multiplayer connections.

What we learned

How to set up multiplayer in Unity.

What's next for Get Tilted

Implement the different controls/functions for different players.
