Inspiration

After hearing about friends that would lose track of how much they drank or were wondering how their night went so wrong, we decided to build this app.

What it does

Allows for users to simply tap a button to keep track of each additional drink they've had. Users input physical characteristics into the settings and the app keeps track of time elapsed since the first drink, all of which are factored into the calculation of BAC. This information is displayed as well as the time that has elapsed since the previous drink and first drink.

How I built it

I downloaded Android Studio as well as GitHub at the beginning of HackRice, and I did all of my development through them.

Challenges I ran into

Basically everything involved with app-development. Not knowing how any component (other than the math) functioned beforehand, we had to learn everything from scratch.

Accomplishments that I'm proud of

A finished product that I can keep on my phone!

What I learned

A lot about Android development as well as software development in general.

What's next for BAC-track

As my first substantial project, I'll make sure to keep adding features like color-progression with BAC, graphical visualization, additional subprograms that will help to keep students safe, etc.
