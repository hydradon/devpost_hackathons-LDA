Inspiration 

The ExxonMobil HackTX Challenge Prompt

What it does 

Adheres to feedback from current ExxonMobil App users and offers an upgrade to the system by providing additional features that will boost sales as well as increase awareness of membership benefits.  Improves upon the navigation within the app to streamline the user experience.  Creates an interface more personalized than any other, by adding specific cars by their make and model which can help with travel planning and how to ensure that ExxonMobil can be incorporated in travel plans, gas dependency and maintenance.

How we built it 

Designed a concept for the front end of the iOS application using Swift and Xcode.

Challenges we ran into 

Brand new experience for every member of our team, as we have not programmed applications using Swift or Xcode before.

Accomplishments that we're proud of 

Created a functional, user-friendly, and efficient navigation system from scratch that is easily adaptable for future design changes and back end implementation.

What we learned 

 Basics of how to use Xcode and the Swift programming language. How to tweak and improve a UI to better serve the UX.

What's next for ExxonMobil HackTX Challenge 

Implementing features in their entirety, including integration with map services, user profiles, a membership service, and payment automation. Continuing to improve upon our concept for the UI.
