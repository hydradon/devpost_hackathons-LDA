Inspiration

OCR is important to expand RPA coverage, because it's one of digitalize technology from real world to computer world. However,at this time, OCR is not 100% accuracy in some cases such as handwritten characters, or language using various characters like chinese character etc.

What it does

This activity package provides three functions to complement OCR and help to improve its accuracy.

How I built it

This activities package was built by Visual Studio 2017.

Challenges I ran into

Effcient algorithm to calculate RMSE of normalized edit distance.

Accomplishments that I'm proud of

This activities are practical.

What I learned

How to calculate edit distance and RMSE.

What's next for OCR Complementary Tools Activities

Better algorithm to estimate from multiple OCR results.
