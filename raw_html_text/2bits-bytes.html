Inspiration

As college students, we rarely take care of ourselves. Always being on our computers, we forget health as time goes by and develop unhealthy habits. Having a reminder for simple tasks such as eating or taking a break helps fight these bad habits.

What it does

The user chooses the time period in which they want to be reminded of the task(i.e. sleeping), and our mascot, Bubu will send a notification after that time period.

How we built it

We had a team comprised of 3 members. While two worked on coding, one worked on illustrating the mascot and various images related to activities the user does. The two who worked on coding, researched on how to build a Chrome extension using HTML, CSS, and Javascript.

Challenges we ran into

Because we were all very unfamiliar with creating browser extensions, we ran into issues related to what is compatible with Chrome. There are still a few bugs in our javascript code, but we tried implementing multiple solutions. Right now, the timers are not working properly, but notifications will still pop up after a time period has ended.

Accomplishments that we're proud of

Actually getting this far as 2nd year students, we never developed an application that can be used by multiple users across one platform. Also, Bubu has been acknowledged by many people as a very cute blob.

What we learned

We were rather unfamiliar with HTML, CSS, and Javascript. Now, we have become somewhat confident in coding in these languages. Also, the illustrator has discovered that creating 16x16 pixel icons is really awful.

What's next for 2bits&bytes

Personally, we wish to fix the bugs related to the timer. In the future, we wish this can become an actual working extension that can be benefited by many.
