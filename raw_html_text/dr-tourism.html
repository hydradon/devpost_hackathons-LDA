Inspiration

Based on the top touristic attractions in the Dominican Republic

What it does

From the comfort of their home or from any Alexa-enabled device, users can ask Alexa about the top touristic attractions and information about those

How I built it

Using the ASK-CLI, applying the Alexa Presentation Language

Challenges I ran into

Handling the correct pronunciation of some of the Spanish words, interchanging English and Spanish in the intent response, as well as supporting non-screen devices and screen-based devices in a seamless fashion, using the same codebase.

Accomplishments that I'm proud of

Creating a clean user experience for those devices that support APL

What I learned

More about creating engaging and useful Alexa Skills

What's next for DR Tourism

Hooking it up to a more robust backend, bring more data about touristic attractions.
