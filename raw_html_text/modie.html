Inspiration

The desire to create a facial recognization based app and our love for movies cumulated into this web application.

What it does

Recommends movies based on your mood/facial expressions.

How we built it

We used the Microsoft Cognitive Services Face API to compute emotions from images and we built a wrapper for the MovieDB API and created a great mashup.

Challenges we ran into

Internet Connectivity/ReactNative-Expo issues: We were not able to test our initial app because of network issues. The lack of a swift version of the SDK of Azure cognitive services API.

Accomplishments that we're proud of

Although we had to switch our base platform thrice due to technical difficulties due to the react native-expo update and the deprecation of the swift SDK of our choice. We still stuck to the idea, pivoted to a web app and a functional app.

What we learned

With three first hackathon participants, we achieved more than we expected to. We feel like if we did not run into the initial obstacles, we would have an even better product that we already do.

What's next for Modie

Create an iOS app with React Native.
