Inspiration

Ever had busy roommates you hardly see, but need to communicate with them about household items? When text just won't cut it - use Google Home to leave messages for each other and track chores lists!

What it does

"Add message for [Roommate]: [Message]" ---> tracks messages and will tell that person next time they're home "Add chore for [Roommate]: [Chore] ----> adds the chore to that roommate's chores list You can also view the current chores lists and messages and add chores/messages on the website, which is updated from/updates to the database :)

How I built it

DialogFlow ---> speaking to Google Home Firebase ----> store data Node.js ----> server React ----> front-end website
