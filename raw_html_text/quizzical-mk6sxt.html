Inspiration

I wanted to compete for bragging rights with my programmer friends.

What it does

Quizzical allows you to create short objective tech quizzes and share them with the world.

How I built it

I built it with Blockstack, React, JavaScript, HTML, CSS, and Inkscape.

Challenges I ran into

Performance when there were several questions.

Accomplishments that I'm proud of

Scoring 100% in the first quiz.

What I learned

I learned how to design gradients in inkscape.

What's next for Quizzical

Observe user behaviour and squash any minor bugs.
