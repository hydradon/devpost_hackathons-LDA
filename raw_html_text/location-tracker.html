Inspiration

We wanted to be able to locate our phone and send the coordinates to our own server.

What it does

It get the phone gps coordinates and publishes them to a mqtt broker, the server is subscribed to the broker and receives these coordinates.

How we built it

We created an Android app to get the gps coordinates, we used paho library to connect to a mqtt broker, the server is based on flask.

Challenges we ran into

Getting the GPS location proved to be a challenge.

Accomplishments that we're proud of

We managed to get the GPS coordinates and to set up the server.

What we learned

How to use the Geolocation services on android and how to set up a flask server.

What's next for Location Tracker

Creating a front end service to make it easy for a user to see the current location of the phone using google maps.
