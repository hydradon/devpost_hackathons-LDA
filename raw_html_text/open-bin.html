Inspiration

We have seen that people recycles incorrectly all the time in the schools, parks etc.. So we thinked that they need an extra help to do corretly this task. And we create this amazing experience to correct this problem.

What it does

It´s detects proximity of the people to each bin and speaks to the person to check if the material is the correct for that bin, if it´s the correct, people say "Go" and the bin opens the door of the correct bin to put the correct material inside, And make it an awsome experience for the people and helping with the recicling task.

How we built it

We uses an arduino 101 that uses proximity sensors to detect proximity of each bin, also we uses an audio module to talk with the person and a speach recognition module to detect people answers, also we use a motor driver and DC Motors to lock and unlock corretly the bins.

What's next for Open Bin

We want to implement computer vision to detect the material shape that people is going to put in the bin and also a central of data using wifi shield to have the status of the bin in real time.
