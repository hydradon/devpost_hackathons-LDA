Inspiration

When we were little, we were fascinated about animal and loved learning more about them. Now that we are older and have the expertise to build an Alexa app, we want to find a way to allow kids to develop their passion for animals while having fun and learning with a simple quiz.

What it does

Animal Guess Game is a simple quiz game to help children associate animal name with their sound. After entering the quiz, users can guess the sound of an animal. There is also an information mode where users can ask the app what sound does a certain animal make. With this app, children can easily familiarize themselves with animal sound while learning more and entertaining themselves!

How we built it

We used the build-in interface on developer Amazon Alexa to develop the VUI. The backend is hosted on AWS Lambda.

Challenges we ran into

Alexa only recognize words in the English dictionary. Therefore, we had to come up with creative ways to imitate the sound of animals.

Accomplishments that we're proud of

Finding a way to overcome the limitations of Alexa, including how Alexa’s vocabulary is limited and how speech recognition is not exact.

What we learned

We learned a lot about Alexa skills and how they work in general. We also learned how to use the interface in developer Amazon for Alexa and how to host lambda functions on AWS.

What's next for Animal BUZZ

We will add more animal data and diversity the animals. We also hope to improve the voice interaction so children can have more natural conversations.
