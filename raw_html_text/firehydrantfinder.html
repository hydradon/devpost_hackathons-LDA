Inspiration

The need for accurate information especially in emergency situations is very acute.

What it does

This application provides a way to view and maintain fire hydrant information. For the first responders, it provides an accurate way to find the closest viable fire hydrant. They also can look at any critical notes associated with the fire hydrant.

How we built it

We are using ReactJS front end with Google Maps API
