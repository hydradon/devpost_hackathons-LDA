Inspiration

The past is full of memorable moments captured in only black and white. From pictures of Einstein and US presidents, to the first kiss on camera, photography was fundamentally different without colour. That's why we decided to add colour to the past.

What it does

Turns grayscale images into coloured ones.

How we built it

We built the project using a neural network for colouring the pictures and a python server for uploading them.

Challenges we ran into

Choosing the right parameters and configuration for the neural network Deciding what features to use for training Processing power limitations -> Tensorflow can't use GPU on Windows Lots of preprocesing for the data Large training times Normalizing the data to bring it into the RGB space

Accomplishments that we're proud of

It works ok for a couple of images You can see the original shape of the object Python server works (yay)

What we learned

Working with TensorFlow A lot of Machine Learning Building a server in Python

What's next for DeepColour

We will add the possibility of uploading larger images.
