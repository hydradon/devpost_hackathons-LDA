Inspiration:

Our inspiration was to create something that made lives easier... or life hacks. So we decided to create a program jam packed with tools to allow a student to cruise through life.

What it does: ##

The SAS LifeHackz Chat Bot is a convenient method for users to receive information that can be useful on a daily basis. This information includes: Randomly Generated Youtube Life Hacks Videos, Weather from world cities, Dictionary, Calendar/Birthday reminders, Currency Converter, and a News Article Summarizer

How we built it:

We used JavaScript and Slack to create our ChatBot

Challenges we ran into:

The ChatBot would return the same line multiple times, which we were able to fix by limiting the line statements to 1.

Accomplishments that we're proud of:

Integrating several API's into a Chatbot and having them run when the user wants that particular tool.

What we learned:

We leaned how to use API's and how to integrate them into a chatbot using Slack and JavaScript.

What's next for SAS Chatbot:

We plan to use Dialogflow to implement the SAS Chatbot in Facebook Messenger, Google Hangouts, Discord, etc.
