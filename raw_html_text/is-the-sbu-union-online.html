Inspiration

We were inspired to build this project after getting a code to register a domain for free. We realized that a plethora of students on campus do not know whether the renovations at the Union are finished or not. This website will provide them with that information.

What it does

This website alerts visitors whether or not the SBU Union is open or not in a concise fashion.

How we built it

We built this website after registering a domain on Domain.com. We used InfinityFree to host our files.
