Inspiration

I was inspired to create this software because computer vision is intriguing and I wanted to explore the possibilities.

What it does

This application allows you to scan a vehicle license plate and retrieve user data of the driver. The use case is if two StateFarm customers have a collision, they can easily exchange information and notify their agents.

How I built it

I used Unity3D with the Google Vision API for OCR and Vuforia SDK for displaying AR content.

Challenges I ran into

Creating image markers at runtime to track was a difficult task due to limited capabilities of available SDKs.

Accomplishments that I'm proud of

I'm proud of the ease of use and accuracy the application has.

What I learned

I learned how to implement Google Vision API for computer vision purposes.

What's next for EasyReport

Adding incident report generation with more detailed reporting features would improve the application.
