The service is an APP interacting with server applications powered by TM Forum APIs and Huawei BES API for SMS. The APP requires during installation an initial user registration to provide: --- TMF API customer registration (FiWare) Name Social Security Number (SSN) Address User Mobile Number Gender Age Country --- enables to direct the request to emergency services in a country basis Personnel Medical Doctor Mobile Number An optional complementary person Mobile Number In case of emergency: User presses the APP button in the mobile phone The APP collects geo location APP collects hospital data (list of Hospitals in the region) --- Lisbon City API APP calculates nearest hospital to reach APP sends SMS with location and SSN to Hospital --- Huawei SMS APP UI confirms Message Received
