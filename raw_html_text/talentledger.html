Inspiration

In a world where new technologies are constantly disrupting industries, we wanted to create a hack that not only used these new technologies but also helped businesses onboard the necessary talent in order to keep up with increasing competition.

What it does - TalentLedger for employees

With TalentLedger you can upload your current resume to our application. From there, we will use machine learning to classify the content and learn more about you. You can send a URL to your former institutions (alma mater, companies, etc.) where they can review your claims and verify that you worked or studied at X and did Y. Our application then logs their confirmation on the blockchain. Now your skills and academic pursuits are verified on the blockchain.

What is does - TalentLedger for employers

As an employer, you can use our application to find verified prospects for your company. The resumes submitted are classified with natural language processing so that you can easily find prospects in your specified industry. For example, if you are looking for a _ Computer Scientist _ that uses _ Ruby _, our program will output verified resumes aligned with your needs.

How we built it

We used Python3 and deep learning to create the natural language processing classifier in the Google Cloud. To interact with the blockchain we used the Ethereum Virtual Machine and node.js to write and deploy smart contracts in the language solidity. The front end was created with Angular.

Challenges we ran into

The main issue we had was integrating our stack. Rohan R. was using a machine learning stack, Brian was using a blockchain stack and Rohan V. was creating the front end on Angular. Getting everything connected was difficult.

Accomplishments that we're proud of

We are proud that we were able to get high confidence levels when it came to matching the resume to the specified industry. We were also proud that we were able to interact with the blockchain from the front end.

What we learned

We all learned a lot, the technologies we used are fairly new and none of us consider ourselves "experts". What is important is that we practice our collaborative problem-solving skills while also learning about new technologies.

What's next for TalentLedger

We would like to test our software on the client facing side. Getting feedback from the end user is important, our technology should be able to be used by anyone, including those that are not tech oriented.
