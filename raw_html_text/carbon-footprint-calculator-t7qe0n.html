Inspiration

Ecosia is a search engine that uses the profits from searches to plant trees where they are needed most. We were inspired by how it integrates sustainable practices within a task everyone does daily.

What it does

The carbon emissions calculator is a navigation platform that aims to educate individuals by integrating sustainability awareness into one’s everyday transportation routine.

How we built it

We built it with HTML, CSS, Javascript, and [API] Google Directions API ( https://developers.google.com/maps/documentation/directions/start )

Challenges we ran into

To bridge both design and coding together as well as integrating Google Directions API.

What we learned

We learned a lot on teamwork; fostering and supporting each other's skills and talents. Developers also tackled some technical issues on Google Directions API.

What's next for Carbon Footprint Calculator

We want to improve the clarity of Google Map by adding/tracking carbon emission on top of the path. We want to improve the aesthetic look of design (maybe to add some simple animation). In addition, since our work is web-based, we want to convert it to mobile so everyone can access it.
