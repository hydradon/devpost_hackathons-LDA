Challenges we ran into

Navigating Android Studio is quite a learning curve. At one point we accidentally pushed a lot of garbage files onto the master branch on github and we had to spend a lot of time figuring out how to undo that.

Accomplishments that we're proud of

Making an app from scratch. Also Andrew spent all night to build our entire map graph and connect all the nodes, which is pretty lit.

What we learned

Learn how to use .gitignore y'all, work on a branch and don't eff up the master branch.

What's next for CUintheTunnels

Expansions! Add rooms, tell when to change levels or use stairs, add step by step directions, notable features like bathrooms.
