Inspiration

Sitting in an auditorium, we found each other over our love for technology and decided that the Myo Armband seemed like a great tool to use.

What it does

Pro Motion is a tool that allows the Myo Armband to control a GoPro using web url's.

How we built it

The Armband sends gesture details to a PC, which sends we URL's to control the GoPro over the GoPro's WiFi network.

Challenges we ran into

We initially had the intention of attaching an electric motor to a GoPro mount to allow for a swivel feature using gestures. However due to network difficulties and inventory constraints, we were unable to get the swivel working using an Arduino.

Accomplishments that we're proud of

Getting the program to run!

What we learned

We learned the importance of early planning, and how to allocate tasks within the team.

What's next for Pro Motion

Implement the motor swivel mount to allow for remote surveillance features in future iterations of the system.
