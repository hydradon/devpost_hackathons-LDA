In a time and age where technology makes it so easy to move money around, why not bring some retro flair to the timeless concept of giving for the greater good? :)

And also: https://www.youtube.com/watch?v=aYIwPu50Fic

What it does

Users who are hosting an event can create an event page with a playlist. Guests can pay to add songs to the playlist through Venmo (pricing is determined by the event host upon event creation), and the proceeds will be donated to the charity of the host's choice.

How we built it

The app is built with Node.js and Firebase in the back end, Venmo integration for payments, Facebook integration for login, and Soundcloud API for searching and streaming music.

Challenges we ran into

  • setup and deployment
  • the time constraint
  • finding an appropriate payment API

Accomplishments that we're proud of

We missed the first day of hacking and still managed to finish implementing our core features! Also integrating several different APIs in such a short amount of time without having much prior experience.
