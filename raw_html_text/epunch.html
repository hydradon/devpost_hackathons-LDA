Inspiration - We wanted to take on the Radar API challenge and our goal was to use a location based service to make peoples everyday life easier.

How I built it - We built it in the Android Studio using Java and we referenced the Radar API.

Accomplishments that I'm proud of - I am proud of how we worked together as a team throughout the event.

What I learned - I learned a lot about APIs and web hooks.
