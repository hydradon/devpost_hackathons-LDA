Inspiration

There is a lack of apps that focus on promoting eco-friendly behaviour and sustainable small companies over larger companies. We thought that users might be interested in learning more about the eco-friendliness of a company before purchasing products from them. We also thought that having goals to work towards to be more eco-friendly would help promote more of this type of behaviour.

What it does

The app allows the user to scan a company's logo with their phone to generate an overlay in AR that describes the eco-friendliness of said company. The app also allows the user to work towards goals to be more eco-friendly.

How we built it

The front-end was built with Swift for iOS. AR was implemented using arkit. The logo classification was done with coreml.

Challenges we ran into

First time working with AR. Ran into multiple issues trying to use a tensorflow model in iOS.

Accomplishments that we're proud of

Getting AR to work in conjunction with a logo classifier machine learning model.

What we learned

We learned how to work with AR, and coreml.

What's next for grow

Based on their scan history, personalize recommendations of eco-friendly companies to each user.
