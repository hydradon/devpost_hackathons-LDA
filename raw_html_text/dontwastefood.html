Inspiration

I always wanted to have a positive impact in my community through technology. So i wanted to build an app that where i could help the homeless people in Atalanta brainstormed for days and i was able to come up with DontWasteFood

What it does

It drops markers on maps and adds a description box.The description includes the pick up time,address,description.Using the information provided food banks can send volunteers to pick up food and later donate in to the homeless.

How I built it

I used google maps API to draw a map,on click JavaScript change marker and description box information.

Challenges I ran into

The map wouldnt load time to time,and the markers wouldn't respond onclick or change colors.

Accomplishments that I'm proud of

Completing the project less than a day

What I learned

Stronger understanding of google maps api

What's next for DontWasteFood

Expand to not just food banks but other services that need to be connected in real time.
