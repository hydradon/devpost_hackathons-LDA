Inspiration

Wanted to learn react js.. and i thought if i could find a bot that would help me with the tutorial and i did find the bot but it was tiresome so i needed an easier way to find bots and there came Botstore.

What it does

Botstore helps one to find a bot of there choice and see its ratings and even rate it themselves. With these rates a developer can know what to correct and what to improve.This in turn makes it easy for the developer's bot to be known by users.

How I built it

Used React for the view as it is easy to implement with the states. Integrated it with a Nodejs API for the back end using MongoDB as our database.We have used heroku as a cloud platform for hosting.

Challenges I ran into

Accomplishments that I'm proud of

What I learned

What's next for Botstore

Marketing the web app to make it known have more bots uploaded. Make a plugin to be included in the bots such that when using the bot after some time using it a pop up appears with the Botstore web app URL where you can rate the app..this will make it act as a customer service for the bots.
