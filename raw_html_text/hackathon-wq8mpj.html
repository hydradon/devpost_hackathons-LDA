Inspiration

A vision of a more accessible world, that brings the information needed by every day citizens to their fingers tips.

What it does

Enables users to view predefined maps such as listings of accessible locations. Also allows users to create custom maps where they can add locations to a map and share with friends.

How we built it

Using android studio we created an android app that uses the Google maps API to deliver a personalized map experience.

Challenges we ran into

The Google map API requires a fee to expand above basic functionality, so we implemented the features currently available free of charge.

Accomplishments that we're proud of

A stylized user interface providing a straight forward and functional experience

What we learned

How to overcome not knowing anything about app construction and the various challenges associated with integrating API's in a custom user interface.

What's next for OmniMap

Users able to create personalized maps with set locations and share these maps with the community and friends. Crowd sourcing more map categories and utilizing computer vision to recognize and add accessible locations, and to update other categorized maps.
