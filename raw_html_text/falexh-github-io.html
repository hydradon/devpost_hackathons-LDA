Inspiration

My inspiration was the amazing cats that are able to jump very high. Also, all of the tutorials by Shiffman on Youtube for p5.

What it does

Allows a user to click a box and add randomized cats!

How I built it

Using Javascript + p5, as well as HTML and some CSS

Challenges I ran into

The cats images had to be adjusted a few times. As well as the mass of each object.

Accomplishments that I'm proud of

The fact that I got multiple cats to bounce in a box!

What I learned

That working with cats can be frustrating.

What's next for falexh.github.io

Updates! This is only the start of the bounce house. There will be new modes coming soon!
