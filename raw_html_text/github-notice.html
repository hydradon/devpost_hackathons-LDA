Github Notice

This is a chrome extension providing you a shortcut to what people are doing on github, with just one click.

Usage

  1. To add a user to follow, click add in settings and input their github account (case doesn't matter!) and you can view their recent activities in activities view.

  2. Import your followings on github by clicking import and input your github account username.
