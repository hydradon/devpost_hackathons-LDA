A simple javascript project that uses the deckofcards API to create a game of war. Documentation: http://www.deckofcardsapi.com/

A human player plays against the computer, each of them drawing one card on every turn. The card with a higher value wins. This continues until either player is left with no card remaining, in which case the other player wins the game.
