Inspiration

We wanted to work with virtual reality and wanted to implement a game. We decided on creating a VR Doodle game because we though it would be a fun challenge to create a two-dimensional game in a three-dimensional space.

What it does

When you enter the game space, a tutorial level will demonstrate the basic actions. In this tutorial, users will learn how to create weapons and objects to overcome obstacles and defeat enemies. All the created objects can be grabbed and thrown.

How we built it

We developed everything using Unity.

Challenges we ran into

Keeping objects from falling though the game floor.

Accomplishments that we're proud of

Free Draw for weapons. Blue Screen of Death.

What we learned

More features Unity offers.

What's next for Doodle VR

More user levels. Better weapon creation.
