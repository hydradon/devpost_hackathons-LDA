Inspiration

We wanted to make a simple game you can play with a group of people. It involves moving around and having fun with your friends. This simple game combines the elements of tag, manhunt and all those running games you and your friends had so much fun playing.

What it does

Just spin up the app and create a room for your friends to join, or join an existing room with a room code.

The rules are simple:

  1. Each player is randomly assigned another player to capture.
  2. When within arms range of the assigned player, hit the button on the screen to capture!
  3. You are now given the person assigned to the player you've captured for you to capture.
  4. Last man standing wins!

How we built it

The app is written in Java with Android Studio.

Backend is using Flask written in Python.

UI designs made in Figma.

Challenges we ran into

Coding the UI into XML within Android Studio was quite difficult and took a lot of our time.

Handling the rooms and joining them proved to be difficult as it was our first time using Flask.

Getting the ultrasonic signals to work in Android and receiving them was our biggest challenge. There aren't any easy to use and recent libraries for this and none of us are familiar with signal processing.

Starting the Google Cloud server to hose our python server.

Accomplishments that we're proud of

Created a fun game in 24 hours.

Learned new technologies.

What we learned

Creating a backend server with Flask and making a complex app in Android Studio.

How to use Google Cloud.

What's next for Bounty

Adding a GPS and boundaries so people can't just run away forever.
