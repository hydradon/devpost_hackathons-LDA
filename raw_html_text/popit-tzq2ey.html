Inspiration

We wanted a Chrome extension like Momentum but one that we could access without opening a new tab.

What it does

It has 5 widgets: Weather, Top News, To Do List, Your Top Sites, and Trending Songs

How I built it

We used HTML, CSS, Javascript, and a couple API's to pull data.

Challenges I ran into

Learning how to save preferences for a Chrome extension.

Accomplishments that I'm proud of

Building everything with very little Javascript knowledge, including implementing drag/drop reordering.

What I learned

How to build things in Javascript

What's next for PopIt

We want to work on saving users' preferences
