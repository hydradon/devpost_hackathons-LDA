Vowel Removements

Team Scratch Idiots' HackSheffield 2017 project Vowel Removements is a web game based on the 'missing vowels' round on the tv show 'Only Connect'. the rules are simple:

  1. You're given a category, for example "Shakespeare's plays"
  2. You're given a word or phrase from the catagory, with all the vowels remove and the spaces moved around. For example, 'Much ado about nothing' would become 'mc hdb tn thng'
  3. you have to guess what the original phrase is based off the scrambled version

you have a decreasing number of seconds to guess each word, so you'd better be quick!
