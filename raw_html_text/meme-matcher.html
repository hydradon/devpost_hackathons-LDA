Inspiration

People love MEmes so we decided to use this to connect people

What it does

It gives you memes to look at you can like a meme or not like a meme depending on your choice of memes it matches you with and individual so you guys can talk if you guys would like.

How I built it

We built the application using HTML CSS and JavaScript. We used the GIPHY API to pull GIFS from its API loaded them to an array and displayed them for users to interact with them to begin collecting information.

Challenges I ran into

We had trouble for a while fetching the GIFS from the API and we also had some trouble creating a server(we didnt solve this).

Accomplishments that I'm proud of

We have a decent first project for a first hackathon we had fun

What I learned

I learned JavaSrcipt learned more about how to use APIs and learned how to use the Fetch API

What's next for MEme Matcher
