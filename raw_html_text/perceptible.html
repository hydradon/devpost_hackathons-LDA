Inspiration

Our initial line of discussion was mainly on how distracted people are in the current world. Often times, emotions go unnoticed and tension builds up over time due to lack of communication. We initially proposed the name "Honey I'm Home", after the famous one-liner vast majority of people relate to on day to day basis. But even this ideas just reinforced our initial thought process. We realized a technology that could identify human emotions could greatly benefit people on the autism spectrum.

What it does

The current application model takes webcam input and provides an emotional analysis of a person described by an emoji.

How we built it

We used .net-core framework to create a web application and used JavaScript to create a real-time video data feed which would then be fed to an emotion recognition API called "Affectiva".

Challenges we ran into

Where do we start? Our initial idea of android development wasn't successful due to the API's ineffective runtime sequence resulting in massive wait times. We then found a good number of other sources to do the same. The concept of creating a real time webcam input was really struggling since it was not something that I've ever done before. Also, the platform wasn't something that I could find help or documentation to implement on.

Accomplishments that we're proud of

  • Realizing the potential of this application
  • Overcoming the technical difficulties and arriving at a stable product
  • Creating a good and simple UI.
  • Using Affectiva to get the emoji
  • Making the JavaScript run webcam stream ## What we learned We learned that innovations can arise from the simplest of the ideas. And that, when you set to solve one problem, there is quite a few of others you end up tackling. Patience is a good virtue to posses and above all, there is always time for Spike-Ball. ## What's next for Perceptible
  • Continue research on the interface and the experience.
