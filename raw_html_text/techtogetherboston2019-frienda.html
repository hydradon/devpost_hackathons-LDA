Facebook Messenger Bot

A chatbot to help tackle mental illness by providing the user an anonymous platform to open up. It also tracks user progress overtime and suggests professionals in the location of choice along with doing a sentiment analysis of the transcript to show the user's improvement when asked.

Getting Started

  1. Clone this repository and cd into it.
  2. Execute npm install to download dependencies.
  3. run node index.js

Pre-requisites

  • Node.js and npm
