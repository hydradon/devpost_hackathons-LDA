Inspiration

There are a lot of NGO's outside and there are so many kind-hearted people who want to help the needy. But the people don't know how to approach the NGO to help the people or when and where they can access the people in need. So, the ServeTheNeed aims at empowering the people who want to help others and make a better society.

What it does

We allow them to know about the nearby NGO and also train them about sustainable living. In the same way, we also allow NGO's to know about people who are nearby them and who wants to help others.

How I built it

I built the entire application using swift in Xcode. I used google-maps-API to display the nearby people on the maps. I used Dialogflow which is a conversational UI which helps them to understand the application and the goal of the ServeTheNeed. I've used Firebase as a database.

Challenges I ran into

I tried to implement the messaging between the NGO and the normal citizen but I was blocked because I don't have an enrolment in Apple Developer Program to create APN keys and use Firebase messaging.

Accomplishments that I'm proud of

  1. I have finished the project in a way so that anyone can use it and build upon it.
  2. I'm glad that dialogflow is properly integrated into the app.

What I learned

  1. I learned about the dialog flow and more about sustainable living also.

What's next for ServeNeed

  1. I want to extend this by adding messaging between the users.
  2. Include payments so that people can directly transfer the money to an NGO from the app.
