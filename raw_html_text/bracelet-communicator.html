Inspiration

Personal experience with inefficient service in restaurants, and wanting to improve that communication for both the customer and waiter.

What it does

The bracelet communicator will allow the customer to notify the waiter of requests, so that their communication can be enhanced.

How we built it

Using Arduino 101, Arduino IDE, the nRF Connect iOS app, with the coding language C++.

Challenges we ran into

We were all beginners with the software and hardware, so it was challenging to start and know what steps to take next. Getting the Bluetooth connection working, syncing the different environments, learning C++, and understanding the Arduino.

Accomplishments that we're proud of

Programming in an unfamiliar language, building the Arduino board, being able to combine different features, applying new skills that we learned from workshops and mentors, troubleshooting issues successfully.

What we learned

We learned C++, how hardware, circuits, leds, and how to use resources to find solutions.

What's next for Bracelet Communicator

Improved connection and useability. Expand to other markets such as health, hospital, senior care, etc
