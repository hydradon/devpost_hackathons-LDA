• Hi I'm Dr.Simmons.

What it does

It allows for facebook users to cope with depressed, anxiety and other forms of mental illness by creating a non-judgemental environment where they can only express themselves. The chatbot is split into a therapy session and an educational game on depression. The game utilizes open-ended answers and allows the player to follow their own path and explore an impoverished city as an individual dealing with depression.

How we built it

We built a node.js server on Heroku's cloud platform to act as a chatbot for our Facebook page, Dr.Simmons. The server acts as the middleman between the chatbot and wit.ai. By integrating wit.ai, we were able to give unrestricted user responses values and respond accordingly.

Challenges we ran into

Setting up the messenger chatbot to communicate with wit.ai seemingly. Training the chatbot to receive answers and respond to determine dual entities.

Accomplishments that we're proud of

Developing a chatbot that does machine learning.

What we learned

How to develop a chatbot with node.js and wit.ai as the backend and using facebook messenger as the user interface.

What's next for Dr.Simmons

Moving into other platforms, like twitter, gmail, Alexa, Google Home, Improving AI to expand into more mental illnesses, such as bipolar disorder Using therapy sessions to accurately portray responses.
