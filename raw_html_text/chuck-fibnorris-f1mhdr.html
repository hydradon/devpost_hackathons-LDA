Inspiration: Initially we thought of developing an endless runner game or a tile game but the second we though of the legendary figure Chuck Fibnorris we knew that was the game for us.

What it does: It's basically an arcade game where you blow alien-robot thingies up with binary data coming out of your finger.

How we built it: Java and a lot of hours.

Challenges we ran into: We had problems creating the menu, setting some sprites and mostly controlling the interactions of the player with platforms in the game.

Accomplishments that we're proud of: Implementation of a menu, collisions with platforms...

What we learned: A whole lot of things we didn't know about java.

What's next for CHUCK FIBNORRIS: Who knows?
