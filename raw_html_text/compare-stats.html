Inspiration

Nous voulions avoir accès à des données qui sont en temps normal caché

Qu'est-ce que ça fait?

Compare Stats est capable d'aller chercher des données publique à partir de différentes API pour pouvoir les afficher de manière à pouvoir comparer différentes villes. C'est donc un outil d'analyse très pratique pour une multitude d'utilisation.

Comment c'est fait?

Compare Stats est une application HTML5. Un serveur NodeJS récupère les données à partir de différentes sources et les enregistres sur une base de donnée MySQL. Le serveur peut par la suite réorganiser les données pour qu'elles deviennent comparable.

Difficultés

Nous possédions peu de connaissances techniques pour développer une application de ce type. Par conséquent, nous avons manqué de temps et plusieurs fonctionnalités sont pour l'instant incomplète.

Ce que nous avons appris

La majorité des compétences requises pour développer cette application ont été apprises durant l'évènement!
