Inspiration

Smart watches are specialized for one of three categories; the sick, the athletic, and the classy.

By fixating so avidly on these demographics, developers miss out on an extremely important portion of society - those that are too young to be athletic, classy or sick.

Infants!

What it does

WearA'more pulls vital signs from wearable tech (in this case, the Microsoft Band) and displays it on a smartphone, within Bluetooth range. It also tracks these metrics and establishes patterns, pushing notifications and warnings if vitals become comparatively erratic.

How I built it

We used the Band SDK to pull heartbeat, temperature and movement metrics, and built the rest of the app from scratch with the Android SDK.
