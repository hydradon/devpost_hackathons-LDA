Inspiration

As a university student not on a meal plan, sometimes you just want to know what you can make with what you have.

What it does

Take a picture of ingredients and obtain a curated list of recipes all requiring nothing more than what you had in the photo.

How we built it

We used the Clarifai API to get food words, which were then sent to the Edamam API to fetch recipes.

Challenges we ran into

It was challenging to figure out how the Clarifai API worked (it actually drove Daniel insane). But somehow at 2:34 am, he got it working. Eric and Kevin did an excellent job with the Edamam API and also displaying the recipes in an improved UI.

Accomplishments that we're proud of

What we learned

For one of us, this was his first Android app. For others, it was the first time using API's and computer vision!

What's next for SnapChef

We can optimize our searches and clean up some random images that clarifai detects.
