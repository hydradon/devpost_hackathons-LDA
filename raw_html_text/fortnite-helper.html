Inspiration

What it does

This project will show how a user is playing and whether they can be disturbed or if they are on a winning streak and should be left alone. In addition to that some other goals are to display daily challenges as well as past performances on top of updating data for an app to show.

How I built it

I created a node server to query a third party fortnite api for latest playing statistics and deployed that to a Raspberry Pi Zero wireless.

If the user statistics were trending downward I would then send an HTTP request to another node server running on the Rasperry Pi that would then send some data over bluetooth to an Arduino Mega, and display a helpful message to anyone in the vicinity.

Challenges I ran into

The bluetooth communication was a lot tougher to get going that I expected. It isn't quite as well coded as I would have liked, but I plan on fixing that with my hack day partner/son over time.

Accomplishments that I'm proud of

We had a working prototype at the end of the hackday which is much more than I expected to accomplish!

What I learned

Given the right tools a lot can be accomplished in a short amount of time!

What's next for Fortnite Helper

Refactoring and possibly printing a case for the LED screen.
