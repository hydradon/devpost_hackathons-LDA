Inspiration

I was looking for something challenging and new. I couldn't find any apps on the app store the could identify a prescription pill in one step.

What it does

You give program a picture and it spits out with it thinks it is with a confidence tag.

How we built it

We built it using tensor flow.

Challenges we ran into

Understanding how to correctly use tensor flow and how to get it to give us the correct answer.

Accomplishments that we're proud of

We are able to predict with confidence if a picture is a pill in our database.

What we learned

We learn a lot about machine learning. Also, what makes a good classifier or not. Most importantly we learned that with a little faith and hard work you can get a lot done in a day and a half.

What's next for Pill Classifier

We want to create an iPhone app that would allow users to take a picuture of the pill and it would shortly return the result of what we think the picture is.
