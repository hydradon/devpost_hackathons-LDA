Inspiration

To give more flexibility to end user so that productivity increases and one can concentrate on real tasks instead of searching of which button to press/click for required action. They can simply speak the command name and it is done for them.

What it does

It recognizes speech and then execute respective commands so we don't need to have these basic/obvious command buttons in our viewer window, giving more space to other complex command buttons.

How we built it

We used HTML5 built in feature in Google Chrome browser to detect and use microphone and recognize spoken text commands.

Challenges we ran into

Support for other browsers seems complicated and we are working on it.

Accomplishments that we're proud of

It is perfectly working without any glitch and we are planning to incorporate NLP into this application.

What we learned

How to use HTML5 capabilities with modern-age web application and extend features to make end user's life more easy.

What's next for Voice 360

Support for other browsers and NLP integration seems to be on the top of list for us to work upon.
