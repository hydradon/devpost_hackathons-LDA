Inspiration

I wanted to create a Xero alexa mashup in a morning and record a video

What it does

You can ask Alexa "What is my invoice status" it will respond with how many invoices have been paid. How many are outstanding and how much money is owed. You can also ask for what is at risk and it will tell you all invoices over 60 days

How I built it

With c# aspnetcore. AWS Beanstalk

Challenges I ran into

Some changes in aspnetcore on how it handles certificates.

Accomplishments that I'm proud of

Having a working project in a morning.

What I learned

It's fun to break out and create new projects. Hopefully people can use it to create their own apps.

What's next for AlexaXeroAspnetCoreMashup
