Inspiration

Often times, I would find myself lost at Wal-Mart. I wonder if I can build a solution that can help other to avoid the same situation.

What it does

It allows the store to save the locations of specific products and allows users to get such product's location easily.

How I built it

This application was built using Java and calls Wal-Mart's API.

Challenges I ran into

It was a bit trouble to get images into the application. In the end, we decided that the low resolution nature of the images that Wal-Mart provided fails to warrant the sacrifice in performance that we will be taking in order to have them.

Accomplishments that I'm proud of

We were able to build a demo that is relatively light-weight and perfectly functional.

What I learned

I've learn how to call APIs and parse their results.

What's next for Walmart Store Helper

I hope to have the chance to create a mobile app that customers can use in store.
