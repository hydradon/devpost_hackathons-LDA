What is StoreFront ? StoreFront is a web application created to enhance the current FIT forum where students market products and services. This application is slickly designed with a responsive interface that makes it easy to use. It will mitigate the clutter of emails being spammed to your inbox.

How was StoreFront made:

The framework and SDK used:

Cordova Ionic WebStorm AngularJS Front-end/back-end: AngularJS was mainly used for the front-end. The back-end was implemented using Python Flask Framework. The mySQL runs on Amazon web services (AWS).
