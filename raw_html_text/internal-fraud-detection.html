Inspiration

Common form of fraud, least-explored is internal fraud compared to external (consumer) based fraud

What it does

Reads transaction data to check for discrepancies, plots the differences into a simple, easy-to-read form for managers. Targets beliefs to change behaviours using MINDSPACE framework

How we built it

Used html, CSS, REACT.JS to build website, Python to build discrepancy detection

Challenges we ran into

-Difficulty in acquiring transaction data related to internal fraud -Time constraint

Accomplishments that we're proud of

-Interdisciplinary result

What we learned

-Internal fraud detection is difficult to detect due to most methods being too expensive/overly done

What's next for Internal Fraud Detection

-Applying machine learning to detect patterns in fraud
