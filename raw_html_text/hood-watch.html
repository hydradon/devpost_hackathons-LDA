Inspiration

Better public safety in the community and allow users to connect more easily.

What it does

Allows users to ask for a carpool to a destination, rent a bicycle, organize a meet up with friends/community members, group chat live with all members of the community, a list of local organized events, and a list of the information of all community members.

How we built it

We used java and implemented JButtons and JFrames to design an applet with the stated functions. We used some APIs to aid the coding and we also implemented a HashMap to store all of our data.

Challenges we ran into

Efficiently producing the app due to an incompatibility with known languages and creating a polished plan to work with.

Accomplishments that we're proud of

Finishing in a timely manner with a well thought out design.

What we learned:

We learned more about the Java language, API's and applets.

What's next for Hood Watch:

Implementing in Cupertino and expanding to other places.
