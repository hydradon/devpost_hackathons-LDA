Inspiration - We wanted to try and make a game that featured real time updates from the web.

What it does - It takes data from a cryptomarketing site and updates the values of the coin, and the magnitude of the change determines whether or not the obstacles help or harm the player.

How we built it - With C# and Unity

Challenges we ran into - Many difficulties with translating Python into C

Accomplishments that we're proud of - That it actually works!

What we learned - A lot about Unity and C# that we had not encountered before.

What's next for Bit Bouncers - Rest.
