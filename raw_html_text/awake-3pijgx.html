We were inspired by classic first-person horror games such as Slender and Outlast. However, unlike typical horror games we wanted to add something a little different with a dynamic, user-based experience.

What it does

Awake is an experience determined by your heart rate. Using a FitBit, the game takes in heart beat data and makes game play more intense as the player's heart rate goes up.

How we built it

We used a pre-built forest in Unity, then added enemies and obstacles. For the backend, we made use of FitBit's API to gather Bluetooth information.

Challenges we ran into

Getting Bluetooth data from the FitBit in real time was challenging, along with multiple bugs we found with AI movement. Over time, however, we figured these out.

Accomplishments that we're proud of

We learned how to parse data from FitBit and convert it into useful metrics.

What's next for Awake

More game play options and a more polished environment
