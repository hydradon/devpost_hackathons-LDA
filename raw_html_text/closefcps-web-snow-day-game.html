Inspiration

There's always so much hype over potential school closings that we thought we could make a fun game to see what the majority of people predict.

What it does

The website tallies votes over the status of school, and affords weight to these votes based on past accuracy to display the chance that school closes

How we built it

We built the program in Cloud 9, using a combination of php, html, and python for algorithms

Challenges we ran into

A lot of syntax with php, as well as setting up databases

Accomplishments that we're proud of

The website is online and functional

What we learned

A lot of php syntax, how to make databases, html design

What's next for CloseFCPS (Web snow day game)

We're thinking of adding a leaderboard for the highest performers!
