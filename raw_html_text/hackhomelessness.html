Inspiration

The St. Patrick Center Hackathon. Well...the first one with 7 people. (We had 2 devs show up and 5 "project managers"...you can guess how that turned out.) I learned of the need for rapid rehousing. I've been looking for opportunities to be involved since and knew that there was a way to build an infrastructure on an app to serve multiple needs.

What it does

CoordinatedSTL allows all members of the CoC to be on the same page about who their clients are and what info is accessible. It also keep track of any Point of Contact that any CoC member has with them. The software allows users to create a location with a certain number of units (rooms) and beds for each unit to properly manager their emergency shelter needs. However, we've built it in a way that it also works for landlords. It equips them with powerful tools that make them want to use the software, find customers, and in turn, make a difference.
