Inspiration

I was inspired by the kids that I teach for app development, during one of the camps I ran they tried to combine the image recognizer and a bingo game using App Inventor.

What it does

Displays a bingo card on your phone and you can scan items in, if it matches you will fill up a square, get a straight or diagonal line and you win!

How I built it

I used Android Studios to build the app, then used google cloud computer vision to set up the object recognition.

Challenges I ran into

Getting Google Cloud to work with Android Studios

Accomplishments that I'm proud of

Using Android Studios for the 2nd time without breaking my laptop

What I learned

How to create object recognition using Google Cloud

What's next for Objective Bingo

Fully implementing the object dectection
