Inspiration

HackKings 2.0 is so hyped up that each batch of tickets has been selling out within an extremely small amount of time.

What it does

Kings Bot is a Eventbrite bot that constantly reloads the HackKings ticket page until tickets become available. Then it reserves a ticket for you to finish filling out the registration forms.

How I built it

Initially tried to use a headless Nodejs web crawler module but then realised I actually needed an api. Then switched to a chrome extension.

Challenges I ran into

I had never made a chrome extension before. Also, since the HackKings ticket page went up, Eventbrite have decided to change their ui and class/id naming conventions along with it so testing was quite hard.

Accomplishments that I'm proud of

I made a chrome extension.

What I learned

How to make a chrome extension. Also Eventbrite has really bad bot protection. I must have refreshed that one page at least 10,000 times over the past 24 hours.

What's next for Kings Bot

Make it work for any Eventbrite ticket page!
