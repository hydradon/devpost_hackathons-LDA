Inspiration

When Alexa starts talking without invocation I get annoyed. This silly skill addresses those unwanted intrusions.

What it does

This skill forces Alexa to apologize for interrupting. Alexa, 'shut it' responds with " I am so, so sorry."

How I built it

I started with the myFactSkill and Space Facts skills featured in the skill development kit. Then, I removed all facts except one. This is a very simple skill.

Challenges I ran into

This was a learning exercise for me in using node.js and in using the skill development kit.

Accomplishments that I'm proud of

I am surprised to see that people besides me use this skill.

What I learned

By minimizing the lines of code I was able to follow how the code works better than my past projects.

What's next for Alexa Dev Review for Shut It

I don't have plans for this skill's future. If the reviewer has a creative suggestion, I am open to it. Thanks!
