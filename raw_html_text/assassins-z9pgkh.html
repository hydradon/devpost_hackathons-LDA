Inspiration

We were unhappy with the state of mass multiplayer GPS games. They encourage teamwork and good feelings. We wanted to create a game that brings out the darkside in you.

What it does

Mobile GPS game which allows the user to assassinate other users within a nearby proximity. Includes powerups such as invisibility, decoys and shields. Allows a live overview of all active users.

How we built it

Using node.js, Socket.io, Google Maps API, mongodb

Challenges we ran into

Problems with getting cordova to actually build the app. Problems with closing sockets.

Accomplishments that we're proud of

Working mass multiplayer game with a good launchpad for potential improvements and ideas.

What we learned

Lots about socket programming and utilising GPS.

What's next for Assassins

Implement more advanced upgrades and a scoring system. Monetization.
