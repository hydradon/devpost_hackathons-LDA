Inspiration

After hearing the challenge banorte proposed, we decided that we wanted to create an easy way for any user to detect and prevent phishing attacks directly on their browser.

What it does

The user selects a link and, after selecting it, opens the extension to reveal if the link is safe or not. The user can also check emails in order to prevent getting scammed.

How we built it

The Chrome Extension is built in JavaScript and the UI seen in Chrome by the user is made with HTML y CSS.

Challenges we ran into

Getting an email and good website dataset for the extension's Deep Learning.

Accomplishments that we're proud of

We managed to create a simple way of protecting digitally-naive users from phishing with JavaScript y React.

What we learned

We learned to work with React y Node.js and how to make a Chrome extension. We also improved our knowledge of JavaScript.

What's next for AntiLure

Do further improvements to the extension and promote its use.
