Inspiration

We want to make the world a better place, by pushing efficiency in the workplace. Arguably, billions of dollars are lost due to typing inefficiencies in the US alone. We are happy to provide the basis of our advanced financial analysis on request.

What it does

Analyze people's typing behavior and let them know if something can be improved. By analyzing people's typing behavior in real time and providing ad-hoc feedback, we provide incentives for people to converge towards a more efficient workstyle. See for yourself: https://www.youtube.com/watch?v=CtP7PS5FZmQ&feature=youtu.be https://www.youtube.com/watch?v=Uiztak_799o&feature=youtu.be https://www.youtube.com/watch?v=JVprhJgBVr8&feature=youtu.be

How we built it

Using C++ and OpenCV, images of the keyboard are shot using a webcam and processed by our cutting-edge real time optimized finger positioning system.

Challenges we ran into

Finger detection and webcam/keyboard synchronization. Also: we don't run into challenges, challenges run into us.

Accomplishments that we're proud of

Using C++ and OpenCV in a hackathon.

What we learned

Work under tight constraints, and never ever use C++ in a hackathon.

What's next for Order of Magnitude

Learn from people's typing patterns and see if there are even more efficient typing styles.
