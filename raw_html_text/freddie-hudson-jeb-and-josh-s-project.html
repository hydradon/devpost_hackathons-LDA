Inspiration

With the many candidates for Senator that go up for reelection every year, it can be tasking for a voter to sort through the masses of representatives up for election. With the struggle of the everyday consumer against bias and misinformation, it is crucial for there to be a location where voters can easily sort candidates by what their views on hot-button issues are.

What it does

RepFind tells you the Senators that are up for election in your state while also informing you of their stances on issues that matter to you.

How we built it

We used Python, Php, Java, and HTML to build the site.

Challenges we ran into

Trying to interact with files with HTML and php Trying to integrate Java and Python into HTML (We have yet to fully solve this obstacle)

Accomplishments that we're proud of

We were able to integrate both Senators and 2020 presidential candidates into the system.

What we learned

How to web scrape Php HTML formatting/hyperlinks

What's next for Freddie, Hudson, Jeb, and Josh's project

Expansion of the project into more political roles, such as governors and local-level law-makers, and a more flushed out website with better and cleaner CSS.
