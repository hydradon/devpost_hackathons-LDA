Inspiration

Some waste such as pizza boxes are often put in the recycling bin, but do they really belong there? Many Canadians are recycling wrong, causing contamination and resulting in more garbage.

What it does

Anna helps you sort out your waste by telling you if you should put them in the garbage, recycling or compost bin. Just ask her by saying "Hey Anna"!

How we built it

The web app was built using Angular and the Web Speech API for speech recognition. The server uses Node.js with Express and processes speech inputs with Watson.

The Amazon Echo integration is made using Alexa Skill Kit and uses AWS to host its functions.

Challenges we ran into

Finding a dataset large and creating a satisfactory UI.

Accomplishments

Building a software that has the potential to reduce waste and help inform people.
