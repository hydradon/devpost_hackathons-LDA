Inspiration

We wanted a quick an easy way to access our own collection of gifs and images that we can quickly paste into chat windows at our leisure.

What it does

It an interface that lets up upload our images and tag them. These images are stored on an s3 server and are processed by some middle ware that lives on node app running on scalingo. Its this middle ware that auto categorises the images and also uploads the images to s3.

How I built it

It uses a mix of technologies, react and electron on the desktop and utilises the algolia cloud for quickly searching through and obtaining the perfect image
