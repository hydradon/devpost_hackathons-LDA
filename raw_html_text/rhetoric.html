Inspiration - We wanted to create a quick way for people to learn about government without extensive Google searches

What it does - Uses a web-scraper to pull trending political discussions and uses Twitter API to the find the latest insights

How I built it - Python for web-scraper, Android for front-end

Challenges I ran into - We originally tried to use Alexa, but the skill setup was troublesome

Accomplishments that I'm proud of - Creating web-scraper, and implementing Twitter API

What I learned - Connecting general public to government is not an impossible task, and that people are excited about seeing tangible change

What's next for Rhetoric - Expansion to Amazon Echo and Real-Time tweeting and web-scraping.
