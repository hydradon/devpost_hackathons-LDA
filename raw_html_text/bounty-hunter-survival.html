Inspiration Making a game that appeals to everybody

What it does A fun game for all ages to try to beat or improve their score.

How I built it Used Libgdx for cross-deployment amongst, Android, iOS, HTML & Desktop

Challenges I ran into Getting the Character Sprite to change images, based on character direction. Getting the Zombies rendered on the screen was tough because, there is no clean way to do it

Accomplishments that I'm proud of Getting the Title Screen, Main Menu and instruction screens done and implemented. I also got the character sprite and the bullets working and displayed.

What I learned the complexity of what was supposed to be an easy concept and the great extent of troubleshooting and the adding on of features and enhancements make the project more difficult.

What's next for Bounty Hunter Survival. Adding touch features for android and iOS. Adding proper background screens, possible expansion of enemies, which would be bonus enemies for certain levels, more characters, that the player can choose from, each with their own strengths and weaknesses. Working on the algorithms to generate and monitor zombies intended behavior in an automated way. Fully implementing the scoring system and high scores. Allow the player to enter data, to be traced with game scores.
