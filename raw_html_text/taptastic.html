Inspiration

We love Tap Tap Revenge on our phones. Tap Tap Revenge is a game where you are able to tap on the screen according to the beat of the song, However, the game only offers a number of songs. We want to be able to play tap tap with any song in our playlist.

What it does

Taptastic analyses a given song, extracts its information, such as beats and rhythm, and then produces a tapping game that you can play on Chrome.

How we built it

We first set up music visualization to study about beats and rhythm. This helped us to develop an algorithm to convert a song's beats and rhythm into tapping arrows.

Challenges we ran into

We were both very new to all the tools that we used for this project. In addition, we also had problem with time constrain.

Accomplishments that we're proud of

We were able to successfully implement our algorithm and got the game running.

What we learned

We learned a lot about web game development.

What's next for Taptastic

  • Improve the algorithm even more
  • Sync the given music with the tapping arrows
  • Polish the user interface
  • Port the game to more devices
