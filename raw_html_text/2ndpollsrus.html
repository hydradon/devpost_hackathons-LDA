Inspiration

We were looking for an idea so we decided to poll in our room. BOOM! IDEA!

What it does

It allows you to respond to polls and look at polls.

How we built it

We used Android Studio and Node.js in order to make the application, and split up based on our talents.

Challenges we ran into

We had trouble with manipulating Android Studio and learning how to use it at first, since most of us had no experience with it.

Accomplishments that we're proud of

We set up the database for questions and user data, along with post gets and put requests for the data.

What we learned

We learned a lot about basic app development and making databases.

What's next for 2ndPollsRUs

If we learn to better implement this, we could actually turn this into an actual app.
