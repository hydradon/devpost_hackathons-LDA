Inspiration:

We saw our Massey Debate website, and wanted to start our journey through web development by creating a new Massey Debate site based without any template to be a 100% learning experience.

What it does:

It shows people interested around Essex County (Windsor, ON) what debating in Massey's club is all about. It also informs members about previous topics, and information for applications and competition.

How we built it:

We used a combination of HTML and CSS to build our website.

Challenges we ran into:

We were inexperienced with HTML and CSS, so everything from basic syntax to good flexibility for all display sizes was a challenge to us.

Accomplishments that we're proud of:

We made the website quite aesthetically pleasing, and it is something that we can look at and appreciate.

What we learned:

We learned how to best use objects in situations, and that for different occasions a template for a website is a lot better to use for bringing more useful features to the website besides providing text to the reader.

What's next for Dabaou-iiazaar.github.io:

We want to make a server that will access a database to hold all user accounts to provide more information for club members, such as their next opponents or even virtual debates.
