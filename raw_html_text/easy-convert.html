Inspiration

Currency conversion apps on the market are hard to use. You need to input the numbers yourself.

What it does

Converts currency on price tags etc. by simply taking a picture of them.

How we built it

We built an iOS app using the tesseract OCR API. We also included some data analysis features using python.

Challenges we ran into

We had some issues scanning image files for numbers. Auto-layout was a big hassle for us.

Accomplishments that we're proud of

Neat and fast conversions 30+ currencies to choose from Underlying data analysis system that allows merchants to make an inference of their sales.

What we learned

Design and build UI ahead of time. Assigning tasks and proper collaboration is really important.

What's next for Easy Convert

Better UI, more accurate text detection.
