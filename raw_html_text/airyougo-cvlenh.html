Inspiration

Air Quality has been a major concern in the recent years. Due to increase in pollution i.e. increase in the concentration of pollutants which has led to many air-borne diseases like Chronic obstructive pulmonary disorder(COPD), Lung Cancer etc.

What it does

So through our application, we tend to provide users the information regarding the quality of air in home address and the work address and also provide precautionary measures. Along with AQI, the application gives the temperature of the selected places.

How we built it

We made an Android application with the server in node.js. We used AirNow API for getting the AQI value and openweather API for getting the temperature.

Challenges we ran into

We faced many challenges during the course of the development of this application. For example the AirNow API provides the AQI index for Ozone, PM2.5 and PM10, but at many places the data is not given or it is partial data, so we had to tackle all such cases.

Accomplishments that we're proud of

we learnt many new APIs and after using them we were able to make the application which aims to provide useful information to people regarding the air quality and hence preventing many airborne diseases.

What's next for AirYouGo

We would like to work on it even after this hackathon and improve it and put it on playstore so that people could get benefit from this.
