Inspiration

We wanted to make a website that would help us educate on environmental and health issues.

What it does

EcoTrak is a website that computes carbon footprint, calories burned, and produces a detailed map on climate change.

How we built it

We used HTML5, CSS3, JavaScript, Python Flask, arcGIS API

Challenges we ran into

How to create a localhost using IIS

Accomplishments that we're proud of

Learning how to implement JavaScript with an HTML5 file. Finishing the website. Working consistently, Displaying great teamwork and leadership.

What we learned

How to implement APIs in a website. How to create a website. How to work in a collaborative environment.

What's next for EcoTrak

Adding more applications. Making it more attractive. Create mobile versions for it.
