Inspiration

It's an app to help people come up with nice outfit ideas in the morning.

What it does

Analyzes user's closet and queries the clothes, finds out what the person likes using user input and machine learning, and recommends nice outfits!

How I built it

We used Javascript for most of the scripting, Watson API for machine learning and Node Red for a lot of workflow management.

Challenges I ran into

Node Red is extremely finnicky, It was a challenge.

Accomplishments that I'm proud of

We used machine learning to develop something useful!

What I learned

I learned how to design front end using CSS and how to apply machine learning to different algorithms and softwares.

What's next for Hangr

Integration with Alexa would allow our application to be used by blind people, eventually the style identifying algorithm could be public, and part of big data.
