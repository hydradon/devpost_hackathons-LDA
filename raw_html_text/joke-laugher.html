Inspiration

I find that frequently, nobody laughs at my sick jokes or puns, so I decided to create a skill for Alexa, so somebody will always appreciate my jokes.

What it does

After opening the program with "Alexa, laugh at my joke", you say your joke, Alexa will respond, then you finish with the punchline. Alexa will always laugh.

How I built it

I used Amazon's Alexa skill-building utilities, as well as AWS's lambda functions.

Challenges I ran into

Since I'd never done anything like this before, it took a while to understand how the skills work, how to develop a conversation, etc.

Accomplishments that I'm proud of

I was able to create a proper conversation between the user and Alexa.

What I learned

I learned how to use Node.js more effectively, as well as develop a skill for Alexa.

What's next for Joke Laugher

I hope I can use what I learned to create more skills, and more cool projects.
