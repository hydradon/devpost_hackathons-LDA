Inspiration

Sometimes, We can not hear what is said during announcements, or what's for lunch in the cafeteria, so we decided to make this information more easily accessible.

What it does

You send a message to the bot, and it replies with the required information.

How I built it

Originally, we decided to use web hooks to connect an external software to the bot, but due to difficulties, we were not able to do that. We then used Chatfeul, which makes it easier to update and set options for

Challenges I ran into

The web hooks did not send information between the code and the bot.

Accomplishments that I'm proud of

We found a way to make it easier for admins to change and add new information.

What I learned

It is extremely hard to set up web hooks when working as a beginner. We also learned how to use git and heroku commands.

What's next for INFORU

Expand to more schools, and add more options for Admins to be able to change the information. Make it easier to update new info.

This was our first Hackathon for most of our teammates, so we learned a lot about coding and the many different platforms we could use.
