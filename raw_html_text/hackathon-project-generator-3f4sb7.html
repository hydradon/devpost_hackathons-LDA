Inspiration

It was 1am I needed an idea because my other ideas were falling apart. I was talking to the people around me about what problems we could solve in a lame way so I created the worst possible hackathon project generator I could come up with.

What it does

It recommends hackathon projects sometimes. Features over 1,000,000 project idea combinations (when it works)

How I built it

I build it in python using TKinter for the UI.

Challenges I ran into

I tried to integrate Twilio so I could crowd source ideas for keywords to put into the keyword the lists I am using, but I was unable to get the Twilio api connected to my program.

What's next for Hackathon Project Generator

Moving it to a web app so it could be more easily accessible or to a TI84 calculator so it can be less accessible. I would also like to add the Twilio api to completely crowd source ideas
