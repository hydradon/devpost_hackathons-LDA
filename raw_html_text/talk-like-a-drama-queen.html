The skill helps English learners improve their vocabulary by learning words from phrases used in movie scripts. The skill can work by itself or as part of streaming experience.

case 1: When using The Skill only user will be able to learn English words that have been used in Amazon Prime TV show or Movie of user's choise. List of words will be predefined. For the competition purpose scripts from the movie "Quail Lake" by Roman Wyden have been used throught obtaining required copyright.

As a part of learning process user will hear and repeat script line where particular word have been used. After practicing words user will have oportunity to test his knowledge. During test he will be required to guess word by it's meaning. The Skill can be used anywhere in the house, my favorite location is kitchen - during some boring cutting work or washing dishes. Also can be used during any type of boring house work like laundry. Some users might prefer it as bedroom before sleep experience.

case 2: The skill can be extended to work with streaming experience. Additional module will be created which will allow user to add words to his list while watching his favorite TV show or movie on Amazon Prime streaming service. This list will be used by Alexa skill "Talk like a Drama Queen" as discribed above. In this case user will get access to the streaming extention service and Alexa skill "Talk like a drama queen" as additional paid feature to his Amazon Prime Video subscription.
