Inspiration

We wanted to design a game that was all in browser. After much deliberation the idea that stuck out the most was something akin to a drum battle. We took this idea and turned into a pattern matching game instead of beat matching.

What it does

Much like the game simon, this game pits multiple players up against each other in turn base pattern memorization
