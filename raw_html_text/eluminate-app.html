Inspiration

As Eluminate, we aim to support girls who are deprived of an education, and allow every individual to be able to support every girl in need in various ways outside of financial aid.

What it does

We have created a game that donates money for points earned, a support network for people to help, specifically, a certain girl through her journey, an ambassador program for the youth to get involved, and a tips page to summarize all the ways any person can help.

How I built it

We used the MIT App Inventor features and coded our app through the system.

Challenges I ran into

There were issues and challenges with consistent WIFI, and being able to add all the features in due time.

Accomplishments that I'm proud of

We are proud of the impact we can create with our app, and the fact that we were able to create an app that is inspiring to others in the given time frame.

What I learned

We learned aspects of code, as well as the issues that can be solved through awareness if we take action.

What's next for Eluminate App

If we receive funding, we plan to add this app in the App Store because we not only believe and have passion for it, but wish to see it through till the end.
