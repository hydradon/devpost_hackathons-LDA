Inspiration

Old school platformer with a twist.

What it does

It entertains the player. The main mechanic of our game is throwing a ball and catching it to progress the level, ultimate frisbee style. Dropping the ball causes the player to reset to wherever they were before throwing the ball.

How we built it

We built it for the 10th hack ISU using unity.

Challenges we ran into

This was our first time using unity and our first game. We did not have much time to figure out what we were doing which lead to complications.

Accomplishments that we're proud of

It works! This is our first official project.

What we learned

We learned how to use unity to create a game.

What's next for Moonlight - video game

We would like to improve the game mechanics and animations. We would like to add more levels and content.
