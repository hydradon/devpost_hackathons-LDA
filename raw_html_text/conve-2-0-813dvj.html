Inspiration

For the MOBI Grand Challenge, Connectivity is the future and is a building block for Autonomous Vehicles.

Connected Vehicle Market is estimated to be around 200 Billion USD by 2023

This solution opens up new market areas but brings a lot of technological challenges as well.

What it does

To build a Global Infrastructure Network for mobility industry where any OEMs, Users and 3rd party service providers can interact in a trust-less & secure fashion.

How we built it

-Using React and Solidity

Challenges we ran into

-Scheduling Meetings that fit multiple time zones. -Everyone having the same direction.

What we learned

  • Communicate consistently and often
  • Ensure everyone is confident on project tasks and direction.
  • Less planning, more doing
