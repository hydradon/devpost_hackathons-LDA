Inspiration

Wanted to make a fun learning app with cool pirate graphics!

What it does

It allows teachers to set math problems for students and see each students progress through the problems. On the other side it makes the math for fun for the students!

How we built it

Used a flask framework and a lot of javascript for the animation

Challenges we ran into

Less knowledge of javascript and short on members

Accomplishments that we're proud of

Making a fun app!

What we learned

A lot of javascript and adobe illustrator

What's next for Pirate Learning

Better problems and more varied formats more data tracking and better displays!
