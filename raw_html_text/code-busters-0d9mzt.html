SecureSearch SecureSearch is a software that helps routers manage different Wi-Fi connections from a various number off phones. SecureSearch acts as a router and has 3 different clearance levels. Level One has access to different types of HTTPS websites like Google, Bing, etc. To get to level 2 you have to enter a password. You get the password by gaining trust on the main router. The 2nd level has access to different types of social media accounts and various games. The 3rd and final level includes complete unrestricted access to any website on the internet regardless of safety concerns.
