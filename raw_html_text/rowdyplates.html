Inspiration

If the government can do it how hard can it be

What it does

It has the ability to read and save information from license plates in real time and can pull meaningful correlations from the data

How we built it

Through hosting on a rapspberry pi and extensive python and nodejs libraries

Challenges we ran into

finding a good machine learning platform that accurately and reliably pulls data from images but thanks to my man Matt Hill who apparently single handedly developed and maintained openalpr; the best license plate recognition software in the game

Accomplishments that we're proud of

reaching the goals that we set out to accomplish

What we learned

never use nodejs, if an api is over 3 years old just don't do it even if it looks so good, dont subprocess what you can thread, also don't sub process never ending loops

What's next for RowdyPlates
