Inspiration behind this program was when me and my teammate wanted to create a hack that was sort of destructive and full of fun!

My team's virus starts music and memes in the background after you reach a certain score in Tetris while "stealing" the user's password when they put their password into our virus.

We built it using java and our Tetris project.

Coding the fake alert screen using AppleScript because it was a new thing to learn.

We are proud of being able to create a program that is able to automatically open programs in the computer and display alerts and notifications to the user.

AppleScript is not for everyone and we should come prepared for any challenges like AppleScript.

Instead of making the virus harmful, we should make tetris virus useful by helping the user automate mundane tasks.
