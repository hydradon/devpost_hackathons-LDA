Inspiration

I've build a few other quiz skills using Alexa, but this is the first one that I've designed for kids as the main audience.

What it does

After the skill is launched (by saying "Alexa, open Creature Quiz" into an Alexa-enable device), the user can play a quiz by attempting to answer a random assortment of questions.

How I built it

I based the JavaScript code on a couple templates that Amazon provides.

Challenges I ran into

I had trouble with the first template that I used, so the final skill is based more on another template instead.

Accomplishments that I'm proud of

I'm proud that I was able to pivot from the first template to the second template.

What I learned

I learned that some of Amazon's templates are more difficult to use than others are.

What's next for Creature Quiz

The code is set up so that I can continue to add new questions into the mix to keep the game fun.
