Inspiration

We wanted a solution to paperless receipts that was swift, easy, environmentally friendly and emulated effortless services like Apple Pay

What it does

Allows the transaction receipt to be sent to a phone via nfc reader and stored automatically within an app with items

How we built it

We created a heroku server to store the uploads of json files representing the receipts themselves, then pulled the data from the server using an app built with the ionic framework using typescript

Challenges we ran into

The nfc readers were very difficult to get to upload the data to the server, but eventually, we figured it out with some intensive and hair-pulling debugging

Accomplishments that we're proud of

Actually getting the nfc reader to work and finish with transfer being smoothly emulated between two phones

What we learned

Had never used typescript or the ionic framework, so we learned how to use those tools
