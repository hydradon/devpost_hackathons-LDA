My car had broken down, but it was unable to be used, since no one in my family had time to actually drive it to the mechanic.

It allows a mechanic to remotely access the car and fix it

We built it using Google's firebase and firestore as well as the Smartcar API

None of us were experienced in firebase or knew how to integrate APIs, so this called for a lot of learning in a short amount of time

We are proud of the fact that we were able to integrate firebase as well as writing our own Google Cloud function, as well as communicating with Smartcar to unlock the car.

We learned how to integrate firebase as well as firestore and Smartcar's API

In the future we plan on implementing the full functionalities of firebase, connect the Firebase with the Smartcar API currently there, and integrating the iOS mobile app to get it to work cohesively
