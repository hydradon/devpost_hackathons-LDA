The modern day developer has an impossible list of different commands to remember. Most developers have to look at stackoverflow at least a few times in a 8 hour work day. There are CLI commands, terminal commands, and each language had it's own set. There are so many, that it's easy to forget even those that are considered simple. To counter this issue, with Duper Alexa office routines, we can ask for commands in a natural way. Alexa will then find and run the command for us. It can execute these commands with a round trip lookup through saved paths utilizing dynamoDB. It's as simple as saying, "Alex git clone my awesome repo."
