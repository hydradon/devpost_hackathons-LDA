Inspiration

We wanted to help people become more aware of how much water they use while showering.

What it does

This is a stop watch that is connected to the internet. You start the watch by pressing the button before getting in the shower, and then stop it by pressing the button after getting out.

How we built it

We built it on a Photon internet connected microcontroller.

Challenges we ran into

We ran into lots of trouble with the hardware side of the project. Specifically getting the lcd display to work while not having an available "grove" port for the display cables to plug into on the Photon.

Accomplishments that we're proud of

We're extremely proud of the hardware, and proud of the programming that was done for this project in C.

What we learned

We learned more about how to hook up LCDs to hardware, as well as how a pull-up resistor functions.

What's next for Go With The Flow (Shower Stop Watch)

We hope to add a social component to the website, where you can compare your environmental footprint with others.
