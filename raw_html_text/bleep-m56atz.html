Inspiration

We want to create a web experience where everyone has a comfortable web experience.

What it does

It provides users a safe way to browse the web by filtering unwanted content. By creating a Chrome extension which reads each web page and identifies explicit or offensive images and text, we are able to filter and replace components of each webpage with friendlier content.

How we built it

The project utilizes the Google Cloud Platform Safe Search recognition library. In addition, it also uses text scanning to detect words. In addition, it depends widely on the chrome extension libraries, which allow for the storage of user preferences and access to various webpages. Lastly, it also utilizes the Giphy API to provide substitute content.

Challenges we ran into

We found the GCP docs quite challenging, as well as navigation of the DOM, which was complicated because of the constant manipulation.

Accomplishments that we're proud of

We're proud of pulling together several different technologies to create an immersive, safe experience for users.
