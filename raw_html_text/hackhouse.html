Inspiration

Cloud technology to solve a real world problem where group members don't have a convenient shared todo list

What it does

Connects people together with their Gmail account names in order to collaborate on todo lists for group projects, groceries, trip planning, etc.

How I built it

Using Google firebase as a real-time database, the app is very responsive and drives the information displayed on each user's screen

Challenges I ran into

Learning NoSQL DBMS.

Accomplishments that I'm proud of

The app passed all of the tests we ran as well as how quickly it communicates with the server and updates the information on screen.

What I learned

I learned that planning is everything. We took the time to properly design our solution before any coding took place and it led to an enjoyable and successful coding experience.

What's next for HackHouse

User profile and stats. Things like how many open todo's each user has. Their top collaborators, etc.
