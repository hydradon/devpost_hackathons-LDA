Inspiration

Jared inspired us by purchasing a winter jacket that didn’t fit. His mother wouldn’t let him buy another since he lost his receipt. He had to wear a jacket that didn’t fit until he grew into it because of this. Our project would eliminate the problem by keeping digital copies of receipts. We decided to go one step further by displaying purchase data in a sleek, user-friendly way.

What it does

Our project read pictures of receipts and uses data from them to track and analyze the consumer's spending habits, as well as present this information in a sleek, user-friendly way

How we built it

Programmed using Python 3.6.4 Google Cloud Vision API’s text detection software Kivy library for graphical user interface Pie charts designed with matplotlib library Logo designed with logomakr.com

Challenges we ran into

using Google Cloud was a challenge, it was very difficult to use based on its complex authentication process

Accomplishments that we're proud of

That we were able to make a unique algorithm to decipher and categorize the information from receipts

What we learned

We learned a lot about how the python packaging system works, as the python framework Kivy (which we used for the frontend)

What's next for CompleteReceipt

Spending reduction advice and the ability to set budget goals, an integration with Google Maps to visualize where purchases are made, and visualizing data on monthly changes in spending habits Log expenses for businesses to simplify expense reports
