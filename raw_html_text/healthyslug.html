Inspiration

Health is important for academic success and well being. We crafted HealthySlug for fellow new year resolution slugs by providing an easy way to follow several healthy diets: basic, paleo, keto, and bulking.

What it does

HealthySlug recommends healthy diet abiding foods options depending on desired meal and dining hall.

How we built it

We classified menu information scraped from the UCSC dining hall website using an algorithm comparing carb-fat-protein ratios. Using Google Firebase's real-time database, we linked the foods to an android studio app.

Challenges we ran into

It was difficult to use Google Firebase database to share data to the app.

Accomplishments that we are proud

Pulling each individual component together to make a coherent app! We learned a lot about scraping the web, parsing that data into useable types, nutrition plans based on certain health goals, using a database to communicate with an app front-end, and making a front-end UI with Android Studio.

What we learned

Developing software in a team can be challenging and good communication is key to success.

What is next for HealthySlug

We plan on implementing personal profiles for tracking daily values and calories count.
