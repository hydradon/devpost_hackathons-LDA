Inspiration

Someone jokingly told Zach to incorporate a pinky into our project idea, and we rolled with it.

What it does

It was supposed to find users who have similar goals and matches them with each other, who can hold each other accountable.

How we built it

We used React Native, Javascript, Expo, and Node.js.

Challenges we ran into

Setting up the framework/learning the framework took more time than expected. Connecting the database with the app was very difficult given our use of Expo which proved to cause more problems than it solved.

Accomplishments that we're proud of

We learned how to use React Native, we created a logo, we didn't give up even though we were strongly considering it, we made a beautiful interface which could be a full-fledged app given more functionality, incorporated forms, sign in, and creation of a profile,

What we learned

How to use React Native, Javascript, Google Cloud Platform, and Expo.

What's next for Pinky Promise

We would implement the functionality that we were planning, such as the matchmaking feature, implementing a database, and messaging among matches.
