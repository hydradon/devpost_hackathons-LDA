Inspiration

Many a times we want to control our devices remotely.

What it does

Using Smart Extension Board, plug in your devices, and control via a mobile app.

How I built it

Using Arduino Uno, ESP 8266, GSM sheild. Server side scripting using Ruby. Mobile app using Java and Javascript (Ionic)

Challenges I ran into

Being new to electronics, our team faced many difficulties in assembling even small things to work. In this process we ended up by burning 3 of our components.

  1. Power switch 12V 5A
  2. ESP 8266
  3. MultiMeter

Because of which we couldnt make Arduino communicate to web service.

Accomplishments that I'm proud of

We were able to make relay work. ie. We were able to control each socket on extension board using our program.

What I learned

Learnt a lot about IOT, came to know about different cool IOT hacks.

What's next for The Great Gatsby

There's lot more to come by this team, once we pass the basic hurdles of IOT.
