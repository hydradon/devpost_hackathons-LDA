Inspiration

With the both of us being animal lovers, we thought it'd be cool for other people to be able to share their cute animal photos for others to see. After all, what is better than looking through cute animal photos if you're feeling depressed, lazy, or just need to waste a couple minutes? Some of our inspiration was drawn from the WeRateDogs™ Twitter account. However, we found it only limited to dog lovers. We wanted to create a more diverse and open platform for all animal photos to be shared.

What it does

We wanted to create an almost Instagram-like platform for people to share and rank other animal photos. Right now, users can create an account and start uploading photos.

How we built it

We created a domain using domain.com and utilized Amazon Web Services and React to create the database management system.

Challenges we ran into

Learning how to use AWS and React.

Accomplishments that we're proud of

Utilizing a content distribution network for global presence. Also, creating a website that have useful analytics features.

What we learned

AWS is VERY powerful in terms of scalability and analytics.

What's next for Pets Ranked

Creating the ranking system and allowing other users to view photos.
