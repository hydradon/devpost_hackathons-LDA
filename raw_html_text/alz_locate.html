Inspiration

After personal experiences with Alzheimers and Dementia, we were inspired to develop a service that would help individuals suffering from these kinds of mental illnesses have more independence while allowing their primary caregivers to live at ease by having access to their precise location.

What it does

This web app is used by both the primary caregiver and the individual suffering from the mental illness and allows caregivers to outline parameters which the user can stay within. If those with the illness exit the set radius, the caregiver is notified that they exited their set parameters, and the program draws a path which the icon can follow showing direction.

How I built it

We used Django, Python, JavaScript, HTML, CSS, Google Cloud API

What's next for FindmyFam

Better implementation of the API's and not as much to be hardcoded.
