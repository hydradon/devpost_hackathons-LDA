Inspiration

We were inspired by the five-minute journal method used in modern practices for mental focus.

What it does

The five minute blog is a more accessible, convenient way for users to log in their thoughts and find focus for the upcoming day.

How we built it

We designed and built our prototype using Sketch and InVision. To implement the features of the website, we primarily used HTML, CSS, and JavaScript.

Challenges we ran into

The biggest challenge of our hack was troubleshooting how to implement an archive section where users can access their previous posts as well as public posts by other users.

Accomplishments that we're proud of

Our biggest accomplishment was being able to implement our idea in a very short and limited timeframe.

What we learned

We learned how to use and implement Javascript, HTML, and CSS.

What's next for The Five Minute Blog

In the future, we plan to implement a more complex database for The Five Minute Blog which would allow users to view public posts and blogs, and be able to organize the archive section in an efficient manner.
