Inspiration

Inspired by the endless potential of virtual reality

What it does

Provide another more accurate simulation of shopping through the web.

How we built it

We used Aframe and Blend to build the VR components. The other software technologies like node and react were used for other logic pieces. We also incorporated the Square API for payment processing.

Challenges we ran into

VR is hard to debug! 3-D modeling does not render properly most of the time. The logic in our app isn't necessarily simple either.

Accomplishments that we're proud of

Being able to create a product that provides a value add within this time period.

What we learned

VR is fun
