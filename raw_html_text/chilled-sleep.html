Inspiration-We knew that if you sleep between certain temperatures, you'll get a more restful sleep and feel better.

What it does-Chilled Sleep monitors your sleep quality and changes the temperature of your room to better suit you.

How we built it-Using node.js we wrote a program that links up with the nest thermostat.

Challenges we ran into- We had a lot of trouble setting up a virtual nest on our laptops and getting it to communicate properly. We also had to learn new coding strategies to get a working prototype.

Accomplishments that we're proud of- We have a working prototype of the software needed for Chilled Sleep to function. We have set up a virtual nest (thermostat) and a server, that can be used for future development.

What we learned- We all learned new ways of programming, and how to set up a cloud-based server that can be accessed by everyone. We've also gained a lot of valuable experience in Javascript, Node.js, and local servers.

What's next for Chilled Sleep- The next step that we are working on is finishing the app for smartwatches and polishing the UI.
