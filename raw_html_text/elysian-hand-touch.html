The main goal was to eliminate an unnecessary level of abstraction between the human and computer and supplant that with a greater human computer interface experience.

It reads motion input data from the Leap Motion sensor and creates output for the hand, arm, and finger movement. Also simple Scripts provide access to google maps where the individual will be able to interact with the map without the mouse.

Software applications drive the hardware and virtual components of the total system.

Finding the right sensor hardware and building on beta code that was written with varying languages, API's, and libraries.

When motion output was successfully achieved from the Leap Motion sensor and then again when I was able to build a light GUI on scripts that ran with multiple online components and small programs.

Working with teams is crucial even if parts of a team move in different directions.

The next phase is the development of more complex programs that leverage the motion sensor technology and various cloud development environments.
