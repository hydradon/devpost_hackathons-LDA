Inspiration

SourceIt is a crowdsourcing extension that allows users of the Sense360 API to crowdsource data to use for projects via a REST API.

How it works

Users add a few lines of code to the Sense360 API and the data is crowdsourced to the server in the background. After, anyone can access this anonymous information via a REST API and can use the data to discover where the activity is anywhere in the world.

Challenges I ran into

I had some challenges getting the SDK to play nice with Google, but after I made the decision to do the processing on the server side I was able to make it work perfect.

Accomplishments that I'm proud of

My girlfriend had the idea for the app a few years ago but we knew that we needed something that could get the location without killing battery. Sense360 gives the opportunity to do this and it is great to see the SDK come to life now after these 2 years.
