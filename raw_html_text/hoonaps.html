Inspiration

We wanted to create an app that shows students around campus good places to nap. Locations can be added by any student so that a large collection of locations is collected.

Motivation

UVA expects students to strive for excellency. Because of this, there is a lot of coursework and pressure to participate in extracurriculars, often leading to poor scheduling and bad sleep schedules. We felt it would be convenient for students to be able to determine good spots around grounds to take a nap during empty time blocks between classes.

What it does

HooNaps shows locations around UVA, added by students, that are comfortable or suitable for napping. Locations are shown as markers on Google Maps, including a name, building, floor, description, among other information.

How we built it

Django was used as an integrated framework for this app. Google Maps was integrated, with the ability to add and remove custom markers. Python is the underlying language.

Challenges we ran into

None of the group members knew how to use any of the technologies used for this project before today's hackathon, especially Django.

Accomplishments that we're proud of

We created a mostly demo-able project with almost no background knowledge beforehand. The app is something we could see ourselves, as well as other students, using during daily life at UVA.

What's next for HooNaps

User authentication/authorization, location approval by administrators
