Inspiration

The horrendously confusing signs at Marketplace were the inspiration for our project. Those signs are so unhelpful and items are almost never correctly sorted.

What it does

The user uploads a picture of a questionable item, and heapSORT will tell the user whether it's trash, compost, recycle, or donate-able item.

How we built it

We built our website using Flask to integrate Google's Vision API with our back end and front end.

Challenges we ran into

The most difficult part of our project was creating a website using Flask for the very first time.

Accomplishments that we're proud of

We successfully built a website using Flask!
