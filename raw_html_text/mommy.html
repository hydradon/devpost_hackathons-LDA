Inspiration

There are very few apps on the playstore which holistically covered the pre-natal care that a woman might need & so this was pretty much n uncharted territory. Thus, as soon as we saw this theme, a myriad of ideas came to our heads and so we went out and made this project.

What it does

It does 3 things:

  1. Provides a lot of handpicked articles for information related to pregnancy divided into different categories
  2. Helps to keep up with the timeline to vaccination
  3. Provides a chat section for consultation with the doctor in case of minor side-effects Apart from this, it also provides an emergency button for the times when a woman goes into labor and she is alone in the home.

How I built it

  1. Front end was created using XML
  2. Back-end was taken care by PHP & MySQL
  3. For everything else we had android studio and java.

Challenges I ran into

  1. Implementing a timer using PHP for the timeline section
  2. Implementing the chat section

Accomplishments that I'm proud of

Implemented the chat and timer sections using php alone (whose code was very ambiguous online. And so this was pretty much my idea in itself).

What I learned

The various functions related to date and time in PHP.
