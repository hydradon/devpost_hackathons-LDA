O aspecto social é uma dimensão muito importante na criação de uma experiência positiva de viagem. Exceto com alguns grupos específicos, não existem muitas oportunidades para conhecer outras pessoas que tenham os mesmos interesses durante as viagens. SmartMatch é uma solução web e mobile que ajuda na busca de pessoas que querem viajar e que compartilham os mesmos interesses, auxilia na agregação dessas pessoas e na compra de serviços turísticos com objetivo de criar comunidades por afinidade. O projeto até o instante mostra as potencialidades, mas precisa de bastante trabalho para otimizar as funcionalidades e bastante contato com os futuros clientes de SmartMatch para alinhar os interesses e para criar ainda mais valor para todos os parceiros.
