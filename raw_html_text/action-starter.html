Inspiration

Filing a lawsuit can be a daunting task for many people. We wanted to make a platform that would make it as easy as possible to organize one.

What it does

You can start or join a class action lawsuit.

How we built it

We used React for the front-end to handle complex view logic and built a back-end using Node.js with data being served from a MySQL database.

Accomplishments that we're proud of

We didn't start with any boilerplate and had to scrap our first idea five hours into the hackathon. We're proud we were able to come away with an MVP app!

What's next for Action Starter

We will give admin status to users who start class action lawsuits so they may edit or add hearing dates accordingly. All participants will be notified of updates related to the lawsuit. We will create lawyer accounts so plaintiffs can be easily matched with lawyers and add geocoding capabilities so users can see what actions are trending in their area.
