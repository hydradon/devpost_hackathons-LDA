Inspiration

We were inspired by the game Superhot, which also plays with temporal mechanics. We wanted to incorporate this into a 2D, cooperative environment.

What it does

This is a 2-player game in which you must cooperate to succeed.

How we built it

We used Java and libgdx to program this game, and we used hand-drawn assets on paper in order to create the art.

Challenges we ran into

We automatically generated the levels in this game, which was difficult to make work at first, but it paid off in the end.

Accomplishments that we're proud of

We successfully implemented parallax scrolling backgrounds and functional enemies to our game, in addition to original artwork and mechanics, and came out with a pretty well-polished game.
