Inspiration

Studying solo is difficult, so why not study with a friend? No friends? No problem! There's Alexa, the best study buddy there is that knows all the answers to test banks. There's also a Slack bot for those who would prefer a quiet interactive study session.

What it does

Help students study from test banks through interactive quizzes.

How we built it

Alexa: Voiceflow Framework to build Alexa Skill

Slackbot: Stdlib Slackbot to build interactive messages with our Studybot

Challenges we ran into

Not knowing how the APIs worked Limitations of Voiceflow Framework Alexa struggled to hear certain letters (a->hey)

Accomplishments that we're proud of

Working Prototypes

What's next for iStudy

More Courses and comments on why each answer is wrong, etc. to aid learning Automated Parsing of test banks More Platforms like Google Home, Facebook Messenger Bots, Discord Bot, etc...
