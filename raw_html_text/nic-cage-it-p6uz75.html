Inspiration

My friends and I enjoy Nicolas Cage, so we wanted a project inspired by him.

What it does

It takes a user-uploaded image by utilizing a jQuery face-detection library and superimposes images of Nicolas Cage onto the faces within the photo.

How we built it

It's simply built with vanilla HTML, CSS, and JavaScript (with a bit of jQuery). It also utilizes git for basic version control.

Challenges we ran into

We spent a large majority of the time figuring out the proper use of JavaScript closures utilizing anonymous self-executing functions (particularly within loops).

Accomplishments that we're proud of

Honestly that it more or less is functional.

What we learned

More experience using JavaScript, particularly with the HTML5 Canvas API as well as DOM manipulation.

What's next for Nic Cage It

Maybe a bit more optimizing, but not too much more.
