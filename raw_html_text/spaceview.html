SpaceView

Website that lets children seamlessly learn about NASA's data with just one touch and a few words

Space View is an educational website that uses data from NASA's Earth and Mars Rover API's to allow people to develop an interest in space exploration by viewing and hearing about some of their publicly-available data. The application utilizes a speech-to-text API and a text-to-speech API's in order to improve the overall user experience and make viewing NASA's world-class images as easy as just saying a few words. It also harnesses the power of Google Maps's street address to geographical coordinates conversion system in order to allow users to view NASA's satellite images of street addresses, which was only possible through NASA alone if one had already known the latitudinal and longitudinal coordinates of the street ahead of time.
