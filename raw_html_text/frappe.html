Inspiration

We work with coffee farmers and wanted to test new technologies that focus on usability in order to increase their user experience and "Kill the Fox" for non-tech savvy users. That way we can send tip over to farmers with no friction on the blockchain side.

What it does

Allows the user to login with any browser using SMS or Gmail Accounts. Once the user has logged in he is able to buy cryptocurrencies or send it as a tip to someone else.

How we built it

We used truffle boxes to create a boilerplate, then we integrated Torus for Gmail Ethereum Accounts, Fortmatic for SMS and Wyre for buying cryptocurrencies.

Challenges we ran into

Each framework has a different testnet and handling the web3 provider get's difficult with the lack of documentation

Accomplishments that we're proud of

Having a working proof of concept

What we learned

Use different tools to help coffee farmers
