Inspiration

We wanted to make something smart, fun and cool!

What it does

In the game setting of dots trying to eat other dots, the player plays against the AI, allowing the AI to train itself and become a master at the game!

How we built it

Using Processing 3.4, Java, Google Cloud Console, Google Cloud Platform and shell script.

Challenges we ran into

No time to train the neural network...

Accomplishments that we're proud of

...however we were able to set it up. We created computers with the capability to have the network to run indefinitely and train against itself/other players.

What we learned

EVERYTHING (seriously, we didn't even know how to use Processing before this, or that Google Platform even existed).
