Inspiration

We realized that when home alone, people usually blast music or find other ways to kill time. What if you could talk to Alexa like a friend?

What it does

You can ask Alexa to suggest a TV show and it will tell you where you can watch it.

How we built it

We used the Alexa Skills Kit and AWS Lambda to configure this, with the actual skill written in Python.

Challenges we ran into

We struggled to find a reliable API for a database of TV Shows and Movies and other details to make this more complex. Due to time constraints, we could not implement as much functionality as we would have liked.

Accomplishments that we're proud of

We have something to demo! We learned how to make a skill on Alexa with no prior experience.

What we learned

We learned that we need to learn JavaScript to make developing skills in the future easier. We also learned the structure of Alexa's skill kit is pretty intuitive which will make future developments more straightforward.

What's next for AlexaImaginaryFriend

We hope to successfully implement an API to make Alexa's knowledge more extensive. We also want to add more topics of discussion and randomize her responses.
