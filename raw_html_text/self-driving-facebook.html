Inspiration

It was inspired by the difficulty of maintaining an active facebook presence.

What it does

It fixes this problem through posting and liking for you! Just do a little work to gather training data from your facebook page, then this uses markov chains to generate comments and a Naive Bayes algorithm to determine whether to like or comment on every post based on your previous patterns.

How we built it

We built it entirely using javascript and jquery.

Challenges we ran into

We ran into many challenges interacting with facebook (liking and commenting posts programatically and gathering data).

Accomplishments that we're proud of

Creating a machine learning algorithm to decide what a user would or wouldn't like

What we learned

About using javascript to interact with facebook and how to implement machine learning.
