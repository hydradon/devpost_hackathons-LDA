Inspiration

Puzzle games like Portal, Platformers like Super Mario Bros.

What it does

Uses Photon Unity Networking to connect ten friends to race each other to the end of the level.

How we built it

We used the Unity game engine to build the level and physics. We used Photon to make it multiplayer.

Challenges we ran into

Originally, we planned to export to mobile to end with a cross-platform, mobile, competitive multi-player game. Unfortunately due to challenges in getting mobile input to move the characters, we were unable to export to Android.

Accomplishments that we're proud of

We are proud of the technical knowledge we gained in intensively using Unity for the first time. We also effectively split up our time and effort among a team new to Hackathons.

What we learned

We learned a lot of technical knowledge in using the Unity game engine, and how multi-player gaming works.

What's next for Multiplayer Platformer

Hopefully with more time, we can effectively export to mobile devices. We will use the experience gained at Hack Western to implement more games and participate in teams effectively!
