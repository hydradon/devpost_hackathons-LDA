Inspiration

Occasionally my kids have difficulty agreeing on an activity or game to play. We used to settle these disagreements by drawing slips of paper from a hat. But that's so old-school nowadays... We can now let Alexa determine the next activity!

What it does

When prompted the skill will have Alexa choose a random activity from a library of over 50 different options.

How I built it

It uses AWS Lambda and the Alexa Skills kit.

Accomplishments that I'm proud of

The skill had over 100 unique users in its first 30 days of availability.

What's next for Boredom Buster

I'd like to update it to allow for users to customize the activities.
