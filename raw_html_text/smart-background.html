Inspiration

We were inspired by building a webpage for a smart mirror. As the project developed we realized we didn't have the skills or time so we settled with creating a page that tells time and weather in your local area!

What it does

Tells time and weather in your local area

How we built it

We implemented some API's into some HTML code and made a local website

Challenges we ran into

Literally everything. We came into this with no experience with HTML, CSS, or many coding programs. Our biggest challenge by far was implementing API's into our code. It took us a long time to figure it out.

Accomplishments that we're proud of

The images that dynamically change with the weather

What we learned

Basic website design using HTML, CSS, and experience with an API

What's next for Smart Background

Adding more APIs! (and implementing it into our smart mirror)
