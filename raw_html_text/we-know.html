We were inspired by the growing media coverage of sexual harassment coupled with our experiences as women in college surrounded by the damaging effects of sexual assault in our community.

Our web app takes in anonymously reported data, stores it on a server for data analytics, and refreshes to page with real time data visualization.

We built it on HTML with JavaScript functionality and PHP for data analytics.

We first attempted to use the Google Cloud Platform but faced errors with IP address sharing when uploading user data to the cloud.

We're proud that we've created a clean, cohesive brand and that within two days, we have a functional web app online.

We learned how to connect the back end technology that we learn in school to front end design to create beautiful web apps.

We want to expand our data analytics in the future to better identify patterns of bad behavior and to increase the quality of information about sexual assault on college campuses.
