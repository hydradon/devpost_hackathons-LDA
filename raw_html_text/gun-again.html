Inspiration:

Mad Max; the want to learn more about VR and a paid tutorial for Blender that we didn't want to spend money on.

What it does:

Gunner is a VR shooter game where you ride in the back of a truck and shoot vehicles in front of you.

How we built it:

The game is made in Unity and runs using C#. We built the models using Maya and touched them up with sculpting from Blender.

Challenges we ran into:

Git was giving us issues for about an hour including a folder that wasn't supposed to be.

Accomplishments that we're proud of:

How much work we could get done in 24 hours. It was a good test to see how far we could get.

What we learned:

Git hates SteamVR, we learned how to set up A LOT of the VR stuff.

What's next for Gunner:

A workshop, upgrades, variety in weapons and vehicles, new terrain, new enemies and more bosses!
