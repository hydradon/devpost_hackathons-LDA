DEVELOPER CRAWLER

We know that finding the right developers can be tough, that's why we created Dev Crawler to do the heavy lifting for you. Our prototype searches through the most proficient users of any language and the top contributors of repositories to find the best fit for your thriving company.

Challenges we ran into

We were hoping to go a bit deeper than just the GitHub API by also finding the GitHub user's LinkedIn account, therefore supplying employers with the developer's job experience as well. Unfortunately, LinkedIn's API was not as welcoming as GitHub's, and much time was wasted just trying to get an API token for LinkedIn to no avail.

Also had a great working GUI and python application, but we were just short of linking them to work together in the end as was initially intended.
