Inspiration

We are a students and we didn't find any app can help us to improve ourself in the college life.

What it does

the user can add the courses and the task, also the deadline for the task.

How we built it

Front-end tools: HTML, CSS, boostrap, javascript, jquary, ajax, and ejs. Back-end tools: node.js, mongoose, express.js, body-parser, and json manipulations. Databases: MongoDB and MLab.com.

Challenges we ran into

login authorization, saving data as json files, playing with css, and modify node js.

Accomplishments that we're proud of

we learned how to use this technology by ourself and we was solving every problem we faced while we builded this website.

What we learned

  • working as a team.
  • progressing from understanding theorems to actual applications.
  • solving problems.
  • debugging some syntax errors and logical errors.
  • learning new technologies to help us improve the project.

What's next for AceCollege

  • adding authorization ability.
  • adding "HelpMe": it will allow the user to find someone near to him and help him to solve the problem.
  • adding users score, for each task the user complete tasks, vice versa if the user didn't complete certain tasks within the deadline.
  • adding a simple AI, which will give advices for the user to improve their college experience, such as recommending to visit x instructor.
  • (It is kinda of impossible) connecting our databases to universities' databases to limit users' input and to have accurate and fast data that we can modify.
