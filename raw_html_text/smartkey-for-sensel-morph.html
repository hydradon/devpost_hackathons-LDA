Inspiration

Not everyone types perfectly on a standard keyboard and people have different typing styles/keyboard layouts. Why not make something that will adapt to how people want to type?

What it does

The user calibrates their keyboard and then can type based on the layout they imagined.

How I built it

We used python and the Sensel Morph API
