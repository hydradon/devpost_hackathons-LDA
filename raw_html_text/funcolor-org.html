Inspiration

I wanted to teach my teammate web dev in html, css, and JavaScript. This website is what we made while she was learning.

What it does

Tells time and allows the user to pick from a number of pastel backgrounds for the site.

How we built it

HTML for initial framework, javascript for time and color swapping functionality, and css for aesthetics.

Challenges we ran into

We could have made something actually useful, but both Natalie and I had to leave after a while.

Accomplishments that we're proud of

Learning html, css, and javascript so quickly, and building a javascript heavy site in just 5 hours.

What we learned

Web dev and graphic design

What's next for funcolor.org

It is up on funcolor.org for all to have fun with!
