Inspiration

A lot household items are wastefully bought and barely used ever again. Eventually these items will end up in a landfill. It would be more efficient if you could rent such items on demand instead.

What it does

A rental platform (web)app that reduces the buying (and hence wastage) of uncommonly used household items.

How I built it

Meteor framework with react

Challenges I ran into

1) I couldn't figure out how to implement a search function for users to search the mongoDB collection. It needs external libraries (like mongoose) which I did not have enough time to learn about. 2) I couldn't get something basic like adding Names & Contact details to the accounts

Accomplishments that I'm proud of

What I learned

How to use meteor with the react library

What's next for Rentability

The future goals include research into recycling
