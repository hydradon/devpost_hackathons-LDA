Inspiration: The need to help students save time!

What it does: Inform people about seat occupancy and allow people to reserve seats

How we built it: Using Javascript, C++, HTML5, photon particle, breadboard and self-made pressure sensor

Challenges we ran into: Lack of an actual pressure sensor and having the webpage obtain data from the micro-controller

Accomplishments that we're proud of: We managed to create a device that can receive commands from a webpage which actually solves a huge student problem. It's gonna help thousands of people

What we learned How to build our own pressure sensor and using javascript

What's next for SeatSense We're gonna integrate more features such as the ability of the app to analyse books that users read and how frequently they read and also suggest similar books.
