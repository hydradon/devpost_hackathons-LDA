Intro : I had the idea for this tiny extension while working on Google docs, including a picture required to open a new tab and it disturbs you work.

How it works : It could not be simpler to use, after clicking the extension, just type what you want to search and you'll get the image that you want directly in the popup. You can then easily copy it.

How it's made : Javascript for processing the query and fetching the images. The query is given to a custom search engine from google cloud platform.

Problems : Reading the JSon file because of Chrome security permissions

What I learned : Javascript, JSon, Google API

What's next : Portability to Firefox and encrypt the search engine id used in javascript
