Inspiration

Space O2 is a game where you plan plants on other planets. All because the lack of awereness of enviorment. VR Games only look like tech demos. They aren't addictive enough for the player to be feasible.

What it does

It's the first Incremental VR game. The gameplay is minimalistic and you only need to play it for seconds, being able to leave it on idle mode as if it was "playing for you".

How I built it

I used the Merge VR headset, Controller and API to create the experience.
