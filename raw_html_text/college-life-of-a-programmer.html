Inspiration

We are inspired by the game "Chinese Style Parents" on steam. However, instead of cultivating the main character, this game allows the player to explore a life of a typical UCSD CS student.

What it does

This game simulate the events and experience of college programmer life to let people know the basic "programmer daily life" in college and help programmer to best know how to be successful due to some choices for events

How we built it

This project is built based on two main segments. We use the android studio to design the android application.for our game , including the basic information, help and the game link. At the same time, we use unity to build the game itself, the background and scenario of the game. Of course, we use some other tools for ancillary help, including Photoshop, GitHub.io…...

Challenges we ran into

The integration between unity project and android studio project was the first difficulty with encountered.

Accomplishments that we're proud of

We have successfully created the game that truly reflects the life of a UCSD CS student based on our experience and imagination.

What we learned

Only one of the member knew how to use Unity, but all team members have worked to learn to use Unity in order to complete this project. Also, we have sharpen our skills in using Android Studio.

What's next for College Life of a Programmer

We will firstly add the new brunches and events to more accurately simulate the college programmer life. Secondly, we would build a log in parts for a formal game. Finally we think our images and scenario could be improved and some shortcut icons for users to play game better
