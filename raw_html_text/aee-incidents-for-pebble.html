Inspiration

Danny del Rio et al. AEE Incidents Frontend

What it does

It shows the municipalities of Puerto Rico and the number of incidents reported in that area.

How I built it

I built it by hooking to the AEE API made by Danny del Rio et al. and displaying it in a menu on the Pebble Smartwatch.

Challenges I ran into

JavaScript is a very tedious language.

Accomplishments that I'm proud of

I was able to build a functional app in a language I had never tried before!

What I learned

Google is a man's best friend.

What's next for AEE Incidents for Pebble

I would like to publish this app to the Pebble App Store, make it display some more info in regards to each Incident report, and push notifications to the user whenever a new incident is reported.
