How we built it

We decided to implement our idea as a webapp using Flask to host our server. We believe that a webapp makes our tool accessible to all. The frontend is built using Bootstrap flask with html files.

The convolutional neural network is implemented in python using tensorflow.

Challenges we ran into

We ran into plenty of challenges trying to learn new technologies. Due to using new languages, we ran into lots of issues with syntax and implementation of our concepts.

Accomplishments that we're proud of

Not easy to learn Python, HTML, Javascript in 24 hours. We manage to solve most of the obstacles using whatever ressources we could use. By learning and applying our newly found knowledge to the project, we surely deserve a cold beer after the demo.
