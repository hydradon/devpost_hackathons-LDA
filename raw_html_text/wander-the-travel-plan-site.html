Inspiration

We were inspired by travel websites such as TripAdvisor and food websites such as Yelp.

What it does

Wander is a website that allows users to create an itinerary and a planning list. Users can categorize attractions according to price and rating. The website aims to create an inclusive community for travellers to rate attractions and share their itineraries with one another.

How we built it

We started prototyping the website through HTML.

Challenges we ran into

Choosing an idea and sticking to it.

What we learned

It is important to choose an idea early so that you save more time to work on the development of the project.

What's next for Wander - The Travel Plan Site

Implementing Google Cloud API services to autofill for countries and retrieving data for the attractions (ie: operational hours, price, time spent)
