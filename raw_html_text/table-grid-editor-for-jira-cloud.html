Are you missing tables on JIRA Cloud? Table Grid offers you the best solution!

With Table Grid you can easily create, view, and edit your data in tabular form.

Track milestones, releases, line items for a quote, employee history, products in use, and many other things that require structuring of data within an issue.

Column types currently supported: strings, numbers, integers, dropdown selects.

In upcomming versions of TGE for JIRA Cloud we're providing database connections, more column types and formulas.
