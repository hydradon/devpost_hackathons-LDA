Inspiration

Our inspiration was to build a system to control home plants and also little urban crops. One of the principal features of our hack has been the usage of low cost hardware components.

What it does

Our hack monitorizes different environment variables and takes decisions based on a learned method. As for instance the system is able to water or turns on the fans whenever i'ts necessary. Furthermore on the webpage or using the android app is possible to see analyticals on real time regarding the quality of the atmosphere of the plant and, for instance, turn on the fans.

How we built it

We built the device with some boards and sensors that we have at home and recycling some things. For example, the cardboard used as a frame we found at the trash... So many thanks to the staff to buy chips the carton box was very useful for us ! The core of the device is a BeagleBone board which control the display and an Arduino to control the sensors and the internet interface.

Challenges we ran into

One of the main challenge has been to develop with the BeagleBone board, because isn't the definitive version, it has several bugs and there aren't a lot of information or libraries. Another challenge was to find a structure for assemble all the parts.

Accomplishments that we're proud of

We are very proud for building a full functional device in less than 36h. and also proud of build a functional web page to monitorize and control all the system.

What's next for IoT Garden

We want to fix some bugs, add some features, improve the design and search investors for commercialize the product.
