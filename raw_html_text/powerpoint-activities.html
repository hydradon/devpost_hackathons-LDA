Introduction:

This package contains three activities.

1.Insert Slide - Using this activity , can able to add a new slide in the existing PowerPoint file.

2.Insert Picture - Using this activity, can able to add picture in specified slide.

3.Delete Slide - Using this activity , can able to delete the file.

Future:

Have to add more functionalities in the activities.

Flexibility and Usability:

This is the power of UiPath . It is very flexibility to the developers to develop the custom activities. After the development, it is very easy to use it. I have Developed activities in C#.Net.
