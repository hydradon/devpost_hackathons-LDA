Inspiration

We wanted to create a fun way for users to get a quick look at most commonly used words in reviews at restaurants.

What it does

Aggregates reviews from Yelp and displays them into a word cloud.

How we built it

We used Django

Challenges we ran into

Yelp's API was very uncooperative, and half the team had to learn how to use Django

Accomplishments that we're proud of

Learning Django and navigating through Yelp's restrictive API.

What we learned

We learned how to handle the abstraction between the data and the view.

What's next for Mood Food

Implementation of a geolocation feature to allow users to simply see the word clouds of restaurants near them.
