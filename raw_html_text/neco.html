NECO

NECO is a simple survivor finding system for people and/or animals.

The system consists of server-side application and client-side application. Client-side application is supposed to be installed some small gadget with GPS and camera and the gadget is supposed to be attached on the collar of cats, dogs and so on. The camera is turned off at usual state. In the time of disaster, the application sends pictures with its location information to the server every 20 seconds, The server analyses and tags the picture. If someone or some animal is in the picture, the system reports it with its location to the headquarters for disaster control.

NECO also means cats in Japanese.
