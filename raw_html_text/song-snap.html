Inspiration

Wanting to send song recommendations to friends with a fun voice message to make it more personal.

What it does

As of right now, it lets users log in and start playing a default Spotify playlist - in the future we hope to add in the ability to find / add friends, record personal messages to send with the song recommendations, and other things.

How we built it

We used IntelliJ and Android Studio, programming in Java.

Challenges we ran into

Embedding the Spotify Android SDK and figuring out how implement the user database.

Accomplishments that we're proud of

As this was our first app, we're proud that we got the links between pages and the Spotify SDK working. All in all, it wasn't too bad for a first try!

What we learned

We learnt how to make an Android app and do UX design.

What's next for Song Snap

We plan to finish implementation of the recording and sharing functionalities.
