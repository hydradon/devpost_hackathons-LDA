LED Weather Indicator

Hack UIowa Project

LED Weather indicator was developed on an ESP32 based board using a neopixel ring. The project connects to openweathermap's API to get the current weather and displays different patterns of LEDs for various weather effects. A button allows the project to switch between temperature and weather. A second button will cause the project to refresh the weather data.
