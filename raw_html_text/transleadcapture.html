Inspiration

Automation is the future and we truly believe that most of the monotonous tasks should be done by machines and humans should be more productive.

What it does

It is basically going to collect your Full Name, City, State, Country, Email and Products that you are interested in. It will also tell you about the product offerings.

It also will confirm the information and say that it is going to send it to the agent.

How we built it

We used API.AI for the intents and entities ( basically the dialogue that the user would have with the agent/bot )

We understand that a lot of people use Facebook messenger; Guess what!! You can chat with the bot in Facebook messenger and get all the information.

It is also integrated to Google Home, so you can talk to it.

It is also integrated to Twilio ( However, we are not sending any messages, you can use it as a bot ; But why download another app when you use Facebook already )

Challenges we ran into

Sending email needed a server running and proper communication. Rest all was easy;

Accomplishments that we're proud of

Developed for Google Home for the first time.

What we learned

Building Bots, Webhooks, Google Home, Heroku
