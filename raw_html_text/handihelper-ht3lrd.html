Inspiration

We were inspired by how much trouble wheel chair impaired humans had to go through just to explore

How it works

We use the google places and google maps api so find locations and a open source java city guide database to gather more data about the accessibility to location near you for handicap

Challenges I ran into

Getting data from data base and ui for the google api's

Accomplishments that I'm proud of

Being apple to implement the database in a simple easy way

What I learned

@4 hours is so short to come up with an innovative idea

What's next for HandiHelper

Adding computer vision to identify crosswalks and paths to help the blind not veer from it
