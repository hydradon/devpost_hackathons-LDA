Inspiration

Currently in Bogota, the city where we live, approximately six thousand two hundred tons of waste is buried in the garbage dump of which 40 percent are recyclable.

Losing a billion Colombian pesos, which is the commercial value of recycling, thousands of families live from this business.

If we were more aware of this problem, we would help the environment and the families that dedicate themselves daily to this work.

What it does

A filter that shows which materials go in each bag, which objects are recyclable and which are not.

How we built it

We use Spark to create the, filter through an image traker, Blender to create 3d objects and animations.

Challenges we ran into

The creation of an fbx that will work 100% well. It was complicated pro sometimes did not export well.

Accomplishments that we're proud of

Finishing the challenge, it was difficult to take time for things from work.

What we learned

More on how to develop filters for Facebook, we think they work for instagram but on that platform it is not compatible with the image tracker.

What's next for recyfilter

hat people use it and that we generate a true conscience, show that there is beyond recycling, the help to the environment and to the families that are dedicated to this work.
