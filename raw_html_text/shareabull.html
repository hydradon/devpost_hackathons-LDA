Inspiration

Everyone has had a questionable, financially irresponsible roommate. Shareabull prevents rent mishaps by distributing responsibility for rent to all tenants and creates a seamless way to pay month to month.

What it does

Sharebull divides the rent for a living space to all tenants living there, makes paying for rent easier, and reminds users month to month.

How we built it

We used HTML/CSS with Bootstrap and integrated the Paypal API for payment (for now).

Challenges we ran into

We had difficulty settling on a project and did not leave ourselves enough time to create a finished project.

Accomplishments that we're proud of

We learned to create with Bootstrap for the first time!

What we learned

Time management is key.

What's next for Shareabull

Continued development!
