What it does

It shows data visualization of car crashes that happens at Southern states of US. Allows filtering for people by state. It is useful for one preparing to travel to new cities in South.

How I built it

We cut the data down to certain information from certain states, in excel and macros and we uploaded to tableau and organized using python scripts.

Challenges I ran into

Data was too large to work with tools like Google Data Studio.

Accomplishments that I'm proud of

One accomplishment is that I learned the concepts quickly.

What I learned

I learned to publish data to servers globally.

What's next for State Farm hack

We are embedding it into a website URL.
