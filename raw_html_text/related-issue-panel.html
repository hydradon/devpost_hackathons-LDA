Graty add-on provides web-panel for JIRA issue with list of related issues by extended JQL query

Multiple panels may be added per issue Configurable field layout for table-view panel Display related issues on map with routes and distances Autocomplete Address CustomField Supported two map providers: Yandex and Google Сlustering support for yandex-map Auto calculate routes and distance Permission schemes for panels Extended JQL syntax as example:

project = TEST and cf[10000] ~ @issue.key

panel with this query displays issues from project TEST where customfield_10000 like issue key.
