Inspiration

The inspiration for this project hit us when we were spending the night before the hackathon in a hotel. We found it really inconvenient having to take turns paying for food, gas, and board and trying to remember who owes what, why, and where. Then it hit us, we can create an app to split the bill!

What it does

Split Pay manages the users expenditures and allows you to split the bill with other accounts on our platform!

How we built it

We build this app using sinatra to server the pages and api calls, plaid to access the users bank account, and bootstrap to make the webapp presentable.

Challenges we ran into

The greatest challenge we ran into was using plaid, given that part of it's use is in the front end utilizing javascript. None of our team members were really experienced with javascript so some of our features are left incomplete.

Accomplishments that we're proud of

We were really proud when we could access the bank accounts that we provided, and that our web app starting looking much better towards the end.

What we learned

We have learned how to manage time, how to overcome various challenges of the apis we utilized, and to persistently try to solve the problem at hand.

What's next for Split Pay

The future of split pay for the near future is to finish implementing and integrating some front and backend code to make a more streamlined user experience, then perform various optimizations and security checks, after which we will publish the app.
