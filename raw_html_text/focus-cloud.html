Inspiration

We want something to log our working activity easily and that is not our phone-they're too distractive!

What it does

It logs study activity, alerts when people keep studying or resting for too long and analyzes study patterns.

How we built it

We built the tool starting with streaming sensor data to a computer via serial cable. After that, we added more logic functions to the board and control it from the computer.

Challenges we ran into

Cable communication and logic designs.

Accomplishments that we're proud of

Accurately record study activity and perform an analysis of the data.

What we learned

Advanced embedded programming, serial communication, and project management.

What's next for Focus Cloud

Advanced analysis of study activity to guide more efficient working.
