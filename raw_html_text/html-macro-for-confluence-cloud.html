What it does

Use the HTML Macro for Confluence Cloud add-on to add snippets of HTML to your page. These can include HTML, JavaScript and CSS which will run just inside the macro itself, but cannot affect the larger page.

  • Add HTML to your pages (optionally sanitized)
  • Add JavaScript to your pages
  • Embed HTML/JavaScript from other websites
