Flash! is a self-learning SMS-and-Web chat-bot. Its aim is to reduce the time it takes to manage one's hospital appointment and also retrieve vital hospital service information.

Flash! is marketable due to the following facts:

  1. Text messaging is the most used data service in the world.( http://www.nielsen.com/content/dam/corporate/uk/en/documents/Mobile-Consumer-Report-2013.pdf )

  2. 4.2 billion+ people text worldwide. ( https://www.textrequest.com/blog/is-mobile-dependence-actually-a-bad-thing/ )

  3. 81% of Americans text regularly. ( http://www.pewinternet.org/2015/04/01/us-smartphone-use-in-2015/ )

  4. 97% of American adults text weekly. ( http://www.pewinternet.org/2015/04/01/us-smartphone-use-in-2015/ )

It will reduce costs and save every stakeholder time. Flash!'s Web implementation is a cheap alternative where SMS fails. It can become a global sensation too.

With enough funding, I hope to have Flash! on all the major messaging platforms in order to reach every niche. Welcome to Flash! where stress is tucked away and healthy living is embraced.
