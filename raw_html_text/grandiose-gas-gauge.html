Inspiration

What it does

It detects if dangerous gas is present in your area.

How we built it

We built it using an Arduino and a gas sensor.

Challenges we ran into

We are not very good at soldering so that was a big challenge. Also, our temperature sensor broke.

Accomplishments that we're proud of

We came up with some pretty solid code that can easily be built upon in the future.

What we learned

We learned a lot about how gas detection works.

What's next for Grandiose Gas Gauge

Adding more features like data-logging and keypad to select from multiple different sensors and calibrations.
