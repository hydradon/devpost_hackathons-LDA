Inspirsation

Many students are trying to find places to go and study any time. Not only students even employee want to get things done.

What it does

You can create account and see the cafes that you can go and study. And drink coffee at the same time. Main thing of this website it there is a news page where you can see the updates from each location. It works like a social media platform

How I built it

Using django which is a web framework. And html, css, js ofcourse. For text editor I used Atom.io

Challenges I ran into

Time, I couldn't finish it. But almost done

Accomplishments that I'm proud of

Nice design and simple user interface.

What I learned

Listing items, sign in page, how to create a sign up page. Edit profile page and lot more

What's next for My CoffeeHouse

I am planning to grow this project and open it to public. I want to make advertisement on college campuses. And ask the cafe houses if they want to put advertisement to my website. And make newsletters that shows events of each cafe they are having to entertain people.
