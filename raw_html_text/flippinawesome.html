Inspiration and Mission

Founded in 2019, Flippin Awesome's mission is to advance the state of the art by building tools that will help people solve problems, make discoveries, and create new breakthroughs. We were inspired by a problem we all face in our daily lives as students - memorizing speeches and text for presentations, exams, and the like.

What We Learned

We learned how to use HTML, CSS, and JavaScript to create a functioning game hosted on a website.

How We Built It

We used HTML, CSS, and JavaScript and connected our application to a web domain using Surge.

Challenges We Faced

We experimented with a transcription option using the Rev.ai API, but the API did not appear to work within our Node.js framework. We hope to get the transcription feature working for future iterations. We also faced some challenges with the design, animation, and brainstorming process, but we overcame those.

What's Next For Flippin' Awesome

We hope to implement a transcription feature such that the user can input their text by speaking into a voice recorder. We may also expand the size of the gameboard and accommodate text in different languages to make our application accessible by an even wider base of users.
