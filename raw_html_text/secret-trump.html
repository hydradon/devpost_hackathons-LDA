Inspiration

Secret Trump is designed to be a personification of all the issues currently being felt in the current American political cycle. It is a direct reflection of the chaos, insincerity, and apathy many people currently have with the established. It is all these things compressed into a fun mafia'esque' game.

What it does

It allows players to step into the shoes of their political rival or ally and attempt to take over Congress either through electing policies or by killing the main protagonist, Trump. The game can be played currently in two ways. Either as the traditional board game as first envisioned or as an online app on the website.

How I built it

The board game is based on the original game Secret Hitler. The board game was designed from scratch but based on the gameplay and layout of the original game.

The web version is based on code provided under the creative commons licence. The assets and text of the game were modified in order to fit in better with the theme of the redesigned game. This was done in a way to still allow online play with those playing with the Secret Hitler version.

The website which hosts everything was created with Wordpress with custom code and a theme.

Challenges I ran into

AWS was a service that was incredibly difficult to get my head around. A lot of my time was spent doing research on how to solve problems constantly popping up. I also found it a challenge trying to reverse engineer the code for the online game without breaking the source code.

Accomplishments that I'm proud of

Being able to eventually do both were perhaps my biggest accomplishments. Having the site successfully hosted and being able to create the reworked online app were huge achievements in themselves.

What I learned

The beginnings on how to utilise AWS in terms of hosting and launching an application. I learned the importance of coming back to a problem if I can't initially find a solution to the problem. I also learned just how complex it can be to launch an efficient application or website.

What's next for Secret Trump

Refining the site and working further on developing the money mechanism will be needed for future development. As well as this, I'm planning to do heavy promotion on places such as reddit as we get closer and closer towards the American elections. Contacting media outlets around the time would also be a tactic I would used to build further traction
