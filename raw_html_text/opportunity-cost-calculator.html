Inspiration

Over 60% of Americans have less than $1000 in their savings account and thought of making a project that could allow people to understand that investing can be easy, powerful, and convenient.

What it does

The code is meant to take in banking information of an individual using the Capital One API and calculate the opportunity cost for different types of investment strategies.

How we built it

We built it using HTML, CSS, and PHP. The algorithm was solely built on PHP.

Challenges we ran into

It was very difficult to develop the algorithm along with integrating the API to work with PHP.

Accomplishments that we're proud of

We are very happy that it was able to work at all considering the complexity of the code and our lack of experience with API use.

What we learned

We learned to integrate the Capital One API with PHP as well as develop a cost model for compounding interest with reinvested gains.

What's next for Opportunity Cost Calculator

We can include more factors to determine better investment strategies and risk assessments for our customers.
