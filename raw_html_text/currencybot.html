CurrencyBot

What it does

This Bot converts amount from one currency to another currency. This is integrated with both Slack and Twilio.

How I built it

This bot is built using the AWS Lex & Lambda Services. Lambda code is developed in Python.

Challenges I ran into

The biggest challenge I ran into was installing the needed libraries (requests) for python and zipping all the code into one file and using this file as my lambda function.

Accomplishments that I'm proud of

I am completely new to AWS Lex & Lambda but when I came across this competition few weeks back, I thought of giving it a try and the tutorials provided by AWS helped me a lot in completing my first bot.

What I learned

The functionality provided by Lex (NLU + ASR) & Lambda services is amazing. Developers can just focus on their core logic and don't need to worry about these deep learning capabilities.

What's next for CurrencyBot

It is limited in functionality now as it supports only few currency types. I would like to change the code so that it accepts any currency type provided by the user.
