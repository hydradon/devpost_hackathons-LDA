Bounding Balls

Bounding Balls is a game that is unlike any game you have ever played. Taking a minimalist approach for all visuals, you use the phones hardware and CoreMotion to control everything in the game. Move all the balls into the score zones to get the high score.

One of the other features you can only get on a mobile environment is a session can be as quick as one minute. So you can play the game anywhere and have a quick distraction in the elevator or waiting for the train.

Each level presents it's own challenges, so get the highest score you can and see if you can beat your friends. new levels are added in updates, so keep an eye out for new challenges and power-ups.
