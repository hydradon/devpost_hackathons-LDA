Inspiration

We wanted an easy to use portal to find local events and be able to to filter them based on what kind of event they are

What it does

It gives you an events portal and map that can find local events near you. It also gives you the ability to add events, specifying date, time, location, organization, etc.

How we built it

We built it with blood, sweat, and tears in android studio!

Challenges we ran into

None of us knew how to use android studio

Accomplishments that we're proud of

We actually got some functionality in and were able to create a decent project despite having 2 members not being cs majors

What we learned

How to use android studio better

What's next for Core

adding user authentication, event details pages, and location services to find local events
