Inspiration

Younger Brother was struggling to find Waldo.

What it does

Finds if Waldo is in an image or not.

How I built it

I used Custom Vision connected it to my program using C#.

Challenges I ran into

The API was really bad and I was not able to figure out if Waldo was in the image or not using the predictions.

Accomplishments that I'm proud of

Fixing my issues with the API and figuring out how to get around not being able to read a JSON file.

What I learned

A lot about how to get around faulty API's as well as how to connect a program to an online project.

What's next for Where's Waldo

Finishing developing an iOS app as well as being able to pinpoint where Waldo is.
