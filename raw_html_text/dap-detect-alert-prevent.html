Inspiration

We want to prevent mass shootings and the more guns there are the more danger society is in.

What it does

Security system that detects possible danger which is alerted and prevented before violent situations occur.

How we built it

Using Python, Azure, and Twillio to design audio, visual, and facial features implying danger.

Challenges we ran into

Connecting all aspects into one program, using webcam to identify the situation, and troubles with the trigger key algorithm.

Accomplishments that we're proud of

Collaborating to accomplish three vital aspects of the security systems.

What we learned

How to navigate through Twillio, Nod-Red, and Azure.

What's next for DAP (Detect, Alert, Prevent)

Detected alarming sounds, more accurate scoring system for facial recognition, being able to identify objects, such as a gun, hidden under or in other objects like a backpack.
