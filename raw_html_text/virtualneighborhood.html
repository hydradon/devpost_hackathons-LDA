Inspiration

We wanted to be able to travel to see any place by just saying the word in virtual reality. At the same time we wanted to be able to get information about the location. So we set out to do this and we accomplished this goal

What it does

We utter the ID of any customer and based on the ID you are transported to the location of the customer. You will see the location in virtual reality and also be able to see data regarding the location.

Based

How we built it

We used unity and the Oculus SDK for the virtual reality visualization. Then we used google maps to get the location and generate a virtual reality render of the location. Then we used google voice to text API to go where you want based on voice commands. Then we set the data given to us by Fannie Mae and other data to generate visualizations based on data.

Challenges we ran into

Overfitting of the Risk Analytics engine

Accomplishments that we're proud of

Getting voice recognition to work accurately. Getting a virtual render of street view.

What we learned

We learned more about googles cloud platform and the enormous potential it possess. We learned more about how the Oculus SDK works

What's next for VirtualNeighborhood

Better data models for better predictions and more data visualization.
