Inspiration

Needed something to help manage chores between roommates

What it does

Users can:

  • See roommates
  • Add groceries to the list
  • Other users can see the groceries on the list and pick something up
  • See the chore list, and whos on what
  • Add a chore to the list, and indicate who is doing it
  • Mark a chore as done
  • See the rent amount

How I built it

Started with an express app, and just kept going. It's pretty simple, it was just time consuming to add all the parts myself.

Challenges I ran into

Making a Google Cloud Compute Platform is overly complicated

Accomplishments that I'm proud of

Built it in a day
