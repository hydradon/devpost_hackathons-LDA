Purpose

Ocean was built to ease the carpooling search process. Our group wanted to create an application that was easy to use for both riders and drivers to connect with each other. We achieved this through building search queries with postgreSQL backend.

Breakdown

  • Leveraged stdlib to create our API
  • Implemented postgreSQL database to store trip and user details with stdlib
  • Built web application using html, css, and javascript to interact with back end logic
