Inspiration

What it does

Rendezvous allows you to select your friends and their hometowns and find the cheapest place to meet up for everyone based on flight prices. It also can show you which of your friend's hometowns would be the most inexpensive for everyone to visit.

How we built it

We used Javascript, the Google QPX API and HTML/CSS/Bootstrap to build the background logic, fetch the data, and build the interface, repectively.

Challenges we ran into

Currency conversion was a big problem. Another big challenge was learning how to use Javascript and the API's on the fly.

Accomplishments that I'm proud of

Interfacing with API's and making real data appear was a big step, as was making the data presentable in a user friendly format.

What we learned

We learned a lot about Javascript, which we really didn't know very much about going in. Additionally, we learned about using API's, which nobody in our team had used before easier.

What's next for Rendezvous

We hope to continue working on Rendezvous together in the near future.
