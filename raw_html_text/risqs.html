Inspiration

The inspiration comes from the idea to make accessible and easy to everyone identify the risks that could happen to your work, home or other place that is important to the user.

What it does

The project RisQC is responsable for data analyses to determine an average of risk based on different aspects of the informed address, like distance from hydrants, inundations areas, fire-fighters houses and calculate the risk involved to that place.

How we built it

BackEnd - Java , Hibernate and SpringBoot FrontEnd - Angular 2 Database - MySql

Informations used to analyse

ZONES INONDABLES À RISQUES ( https://www.donneesquebec.ca/recherche/fr/dataset/vque_78 ) ZONES INONDABLES RÉGLEMENTÉES ( https://www.donneesquebec.ca/recherche/fr/dataset/vque_77 ) BORNES-FONTAINES ( https://www.donneesquebec.ca/recherche/fr/dataset/vque_28 ) CASERNES DE POMPIERS ( https://www.donneesquebec.ca/recherche/fr/dataset/casernes-de-pompiers )

What's next for RisQC

The next steps for the project RisQC is to improve the capacity of data analyses and find more data to be analysed making the system more fast and reliable. Another point is to integrate with differents alert system by all Canada, making it able to predict catastrophes and send alerts before it really happens .
