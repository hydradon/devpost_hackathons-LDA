Inspiration

Llamas are awesome.

What it does

Thoroughly entertains you for the next 5 minutes

How I built it

Primarily used the LEAP Motion javascript API and html5 to build it. Bundled everything into a github repository and linked it to a custom domain so that it can be accessed online from anywhere.

Challenges I ran into

Limitations with the accuracy and consistency of the LEAP Motion hardware created a few bugs in behavior.

Accomplishments that I'm proud of

Being able to submit a presentable product that will surely be at least entertaining for many people.

What I learned

How to map 3d movements in space to 2d graphical animations in the browser.
