Inspiration

I suffer from anxiety and depression, and I would benefit from an app that gives me metrics about my mental health.

What it does

When a user registers, they're given a quiz to determine weaknesses that would be best targeted to improve mental health. These metrics are not only useful to patients, but also healthcare professionals.

How I built it

React front-end with PhoneGap to port it easily to Android, iOS, and Windows devices.

Challenges I ran into

It's hard to set up dev environments.

Accomplishments that I'm proud of

Everything you see was done in a few short hours.

What I learned

Mobile dev, gamification.

What's next for Headrest

Robust DB and data analysis
