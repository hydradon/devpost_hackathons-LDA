Inspiration

We wanted to make it easy and fun to make your own memes with your iPhone frontcam.

What it does

Memster generates memes out of your selfies using artificial intelligence.

How we built it

We got a lot of memes by scraping meme sites and ran all these images trough Amazon rekognition and Microsoft Cognitive Services to analyze these pictures and store the texts describing the memes. We then built an API which takes an selfe (and analyzes the face and checks up on the meme DB to see which meme text will fit the most and return a meme with that.

Challenges we ran into

  • How to scrape the sites
  • How to get the most fitting text to the image.

Accomplishments that we're proud of

  • Making so much of the concept in so little time.

What we learned

The internet doesn't like scrapers..
