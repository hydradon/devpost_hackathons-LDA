• Switcheroo implementation

Inspiration

We came up with the idea for Switcheroo because we wanted to create an affordable home automation solution that doesn't rely on expensive smart outlets.

What it does

Switcheroo is an app that allows the user to wirelessly control their home appliances from their smartphone. The user can make channels on the app, which maps to individual relays. They can then toggle the devices that are connected to the relay on and off.

How we built it

We built this system by creating an Android app on Android-Studios, which connects via bluetooth to a relay system. The bluetooth module allows the user to send control signals to an Arduino Mega, which then controls the relays. The relay was then wired up to a GFCI output, allowing for the control of household appliances.

Challenges we ran into

The main challenge that we ran into was making the app connect to the bluetooth module.

Accomplishments that we're proud of

Our proudest accomplishment is making an easy to use app, that can intuitively communicate with external hardware.

What we learned

We learned how use electrical, relays, bluetooth technology, and android studios.

What's next for Switcheroo

Next we plan to refine our application by adding accounts, which will allow the user to save their channels. We also are going to design a more commercial hardware apparatus that will make setup easier.
