Inspiration

The inspiration for our idea came from the recent events that occurred by hurricanes.

What it does

Our app takes the location the use enters and goes through past data to present the user with a possibility of what natural disasters has occurred at the past. The app ideally runs off-line and thus in an event of a catastrophe it will aid the user in figuring out what to do in that time and help them get through the crisis in a better fashion.

How we built it

We built the application using Android Studio, using Java.

Challenges we ran into

We had never used Android studio before and it was a challenge figuring out how to work it.

Accomplishments that we're proud of

We are proud of the fact that we got the main GUI to work and most of the code. Although we ran into many code problems that hindered our final application we did learn a lot from this experience and are proud of it.

What we learned

We learned how to use Android studio better and learned a lot more about Android development.

What's next for The Disaster Handbook

The next thing would be to get even more information for even more disasters and also expanding the data set to show more specialized data points.
