What it does

Allows users to use with an AR capable browser to load a webAR page we made. The page has a dropdown menu that lets the user select from several models and then when they tap that selected model will be placed right in front of them.

How we built it

With Javascript in an html page, building off Google examples and using models from Poly.

Accomplishments that we're proud of
