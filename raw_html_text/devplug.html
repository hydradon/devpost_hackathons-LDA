Inspiration

The inspiration for building devplug came reallly, after seeing how much developers,especially start-up developers suffer to know whhat's the best way for them,so we thought of a solution that can close that gap and provide an environment that will allow every one connected to it or plugged to it flourishes.

What it does

Devplug connects developers among them, to mentors, give them access to tutorials,and help them build teams to work on projects.

How I built it

Devplug is build by a team of backend and Frontend developers , we've use Facebook tools like Login,but the system is entirely built in Java

Challenges I ran into

We're connecting developers globally in one platform.

Accomplishments that I'm proud of

We have local developers who are already interested in the system.

What I learned

To be patient, to have eager to learn, to deliver under pressure,etc.

What's next for DevPluG
