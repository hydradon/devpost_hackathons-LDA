Inspiration

It would be nice and neat if you could easily advertise your Github repos to all your friends with neat stickers for Snapchat.

What it does

This API takes your GitHub repo information, and it displays and transforms it into a sticker for Snapchat users.

How We built it

We built this with Java in Android Studio.

Challenges I ran into

Snapkit has no beginner documentation and no example apps for Android. There were only 3-4 GitHub repos that attempted to use Snapkit for Android. Snapchat doesn't display its errors to its API calls. Snapkit would not work for me. We have never used Android Studio, so we had zero idea what we were doing. The error trace in Android more often than not came back with a runtime error, but didnt link itself to my code so we couldn't narrow down bugs that made the app crash.

Accomplishments that I'm proud of

We made an android app that accesses the GitHub API

What We learned

We learned Android Studio.

What's next for Snappo

Hopefully to get a redo and to get Snapkit fully integrated for android and maybe ios.
