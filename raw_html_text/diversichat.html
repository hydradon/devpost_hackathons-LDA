• DiversiChat chat

Inspiration

We wanted to make a more inclusive and diverse way to chat with people across the world.

What it does

Send messages with DiversiChat skill on Alexa and it sends your message to diversichat.netlify.com as the chatroom.

How we built it

We used Express.js and Firebase as a backend hosted on Heroku. We implemented a RESTful API to POST messages from Voiceflow API integrations and GET messages to the React.js frontend.

Challenges we ran into

Figuring out how Voiceflow API integrations work and getting everything to connect to each other as a full stack was a challenge.

Accomplishments that we're proud of

Everything is not localized. You can use this from anywhere. Also, it's our first hackathon for two members and we successfully deployed a full stack app.

What we learned

We learned how to work with APIs better and how to coordinate together as a full-stack team.
