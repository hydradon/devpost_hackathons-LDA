Inspiration

All of us were brought together by our passion to save the environment. Andrew, one of the members of our team, thought of the idea to help consumers switch from gas/diesel cars to a more environmentally friendly option. We slowly fleshed out the idea and now have a working app.

What it does

It uses the Google Maps API to track the distance a car travels in a trip and uses data to see how much money the user spends on gas using their diesel car.

How we built it

We built it using android studio, Google Maps API,

Challenges we ran into

Using the Google Maps API and being able to track location in the app as well as the distance between two points.

Accomplishments that we're proud of

We're proud that we have a working app.

What we learned

We learned how to make an app and a website.

What's next for Eco-nomics

Hopefully, we can use the website and the app we developed to create a cross-platform program. We also want to expand into other areas like electronics usage and water usage in the office and figure out where we can save people money in those areas.
