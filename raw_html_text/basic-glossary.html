  • Show the definition

  • Request Define a the term if not already defined

The Basic Glossary Plugin is a very simple glossary implementation. The basic concept is to create a space using the Glossary Space Blueprint where terms are defined. The glossary space is identified as a glossary space by having the space category glossary. Regular spaces are linked to the glossary by having a shared space category. When a use highlights a term or phrase on a page and selects the glossary icon, a search is performed in all of the linked glossary spaces for that term. Any matching pages are then displayed to the user in a popup. If the term does not exist, the user is invited to defined it should they have permissions or to request the admin of the glossary space to define it.
