Inspiration

Internet of things. Connecting household tasks to the internet.

What it does

It is a device that could recognize an individual and implement actions based on voice. For the hackathon we will demonstrate how it can play music based on facial recognition.

How we built it

It was built using a raspberry pi that uses a camera for facial recognition to implement voice activated tasks. Amazon Alexa played a huge role in this as well as IBM Watson.

Challenges we ran into

Putting the website online. Website is to display the security camera, record video and speech to text.

Accomplishments that we're proud of

Integrating all of these technologies creating a robust product that implements voice activated actions with the added layer of facial recognition security.

What we learned

Integrating the hardware we made with the different APIs from different companies difficult.

What's next for Alfred My Butler

The sky is the limit for this project. Alfred could be connected to all devices in the house. Not only will it implement actions based on voice but it will implement actions based on text as well.
