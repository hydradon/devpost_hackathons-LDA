Unihack 2017 - our submission Alex, Jacobus, Joshua

Whats in the fridge? This is a webapp that can tell you what is in your fridge/pantry from a photo and can give you ideas about what to eat from that

Inspiration We enjoy food but we never know what to make with what we have

What it does Figures out what recipes you can make with what you have in your pantry and/or fridge

How I built it Accenture machine learning api, bottle web server, ngrok port exposing, bootswatch for html styling, github

Challenges I ran into Making the Accenture API work for our use and connecting the service

Accomplishments that I'm proud of That we managed to produce a working, functional prototype with api calls and all

What I learned The use of other API's is important, and teamwork skills
