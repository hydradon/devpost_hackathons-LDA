Inspiration

No one likes eating alone! New to an area? Travelling? Find someone in the area to eat with!

What it does

Connects you with someone looking for a lunch buddy. Keeps your information anonymous to protect your privacy.

How we built it

We built it as a web app with OAuth (FB authentification),JavaScript, PHP, and web dev tools (html, css, bootstrap).

Challenges we ran into

We only had two working members on the team. We are both experienced with only backend. We spent a large amount of time trying to understand cordova to build an android/ios app however it just wasn't working great. After a while, we resigned and moved onto focusing on just getting the web app and information processing logic prioritized. We intended for a user to be able to search on google maps a location and use that to search for restaurants but the API didn't want to play nice.

Accomplishments that we're proud of

We got a website of sorts up with its own domain name and everything!

What we learned

OAuth, App Dev is terrifying, GoogleMaps APIs are handy if you can get them to work with your own code.

What's next for LunchBuddies

implementing the google maps, restaurants tab, better css layout....and someday an app.
