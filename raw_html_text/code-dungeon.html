Inspiration

Get more people involved with coding in an engaging and fun manner!

What it does

"Code Dungeon is a game. a good game. the greatest game." - Trump

For real, Code Dungeon is a programming adventure game which encourages beginners to learn and improve their knowledge of basic programming concepts.

By engaging the user with an interesting and story-driven experience we hope to pique the interest of novices and enable them to learn in a fun and rewarding way.

How we built it

We built Code Dungeon entirely in JavaScript so that it could be enjoyed via the browser.

Challenges we ran into

We found that there were many features we had initially planned to add that ended up being scrapped due to time constraints. However, we did implement the core functionality we had originally aimed for and hope to build upon the project in the future.

Accomplishments that we're proud of

We are proud that we managed to build a fully working game from start to finish in the time limit, and very happy with how it turned out in regards to the user experience.

What we learned

Improved/Reinforced JavaScript development skills, learned to prioritise core functionality over smaller features (even if they would be much more enjoyable to implement :( )

What's next for Code Dungeon

We would like to completely finish and polish our prototype and continue building levels to teach more of the basic principles of programming.
