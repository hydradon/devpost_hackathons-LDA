In various emergency situations like earthquakes as well as search and rescue missions, it is crucial to be able to identify individuals in a safe manner. Using the FLIR lepton and Arduino, we built a robot and developed software that uses thermal imaging to autonomously follow a person while keeping a reasonable distance away. This technology not only has applications in search and rescue missions but can also be used in military situations where it might be necessary to have an autonomous and intelligent back up robotic vehicle.
