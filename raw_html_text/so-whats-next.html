Inspiration

As I was getting into tech and learning how to code I began to notice that there are lots of online resources to learn coding, but very little help with where to start or what to do with that knowledge. I wanted to foster a space where people could have access to a code compiler/debugger, as well as suggested real life projects that people want to see happen.

What it does

Our website provides user profiles, educational tools, forums, and an embedded debugger IDE (OnlineGDB) all in one place.

How we built it

We utilized Wordpress plugins, and did lots of editing/customizing source code for our own needs. It now looks and functions almost entirely differently from theme started with.

Challenges we ran into

We began with four members, ended with two.

Accomplishments that we're proud of

Our product really works and looks much nicer than any other project we've submitted in Hack-a-thons before.

What we learned

How to use Wordpress, communicating about specific code with a team of people. We also got practice with php and css.

What's next for So, Whats Next?

So...Whats Next? Next steps would be connecting Github and LinkdIn to user profiles, and getting more in depth with educational resources.
