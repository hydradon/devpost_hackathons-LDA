Inspiration

Makes users able to see Bamboo notifications in the Flowdock flows.

What it does

This plugin integrates Bamboo with Flowdock. It allows any Bamboo notification to be posted to a specific Flowdock flow. This plugin allows you to take advantage of Bamboo's:

  • flexible notification system (i.e. tell me when this build fails more than 5 times!)
  • commenting (comments show up in the chatroom so everyone can see someone's working on the build).

How I built it

Using the Atlassian SDK and Flowdock Push API.
