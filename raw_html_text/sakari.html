Inspiration

We started out with trying to think of uses for the Oura ring. While walking around the venue we discovered the Tieto smart building system and thought that we could integrate our health data with Tieto's office sensors and get some interesting results.

What it does

It measures employee stress levels and uses that data combined with smart office building data to enhance the working experience in the office.

How we built it

Using the Tieto API and stress data we created an additional layer for the Tieto application.

Challenges we ran into

Wifi problems and Tieto API mood swings slowed us down considerably as most of the development time was spent waiting for results from the API.

We used HTML5 canvas for this project but none of us had any previous experience with it.

We faced some difficulties retrieving the raw data from the wearable devices because of the subscription plan issue.

Accomplishments that we're proud of

Innovative and perspective idea. Getting to know the intelligent building setup. Using new APIs.

What we learned

New APIs, HTML5 Canvas, how to get data from different wearables.

What's next for Sakari

Collaboration with Tieto ideally.
