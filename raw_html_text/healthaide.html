Inspiration

Personal experience in the health care system, multiple accounts of stress and worry over miscommunication.

What it does

Focuses on allowing healthcare providers to better keep track of your healthcare activity, allowing them to act on time. Allows communication between hospitals and the patients in order to better keep track of their activity and status.

How we built it

Using angular framework to build a structured frontend environment and connected it to an api created with express.js in order to save metadata about important statuses.

Challenges we ran into

Making sure communication would not intrude in too much personal information and structuring different aspects within angular.

Accomplishments that we're proud of

Being able to understand angular on a deeper level and being able to build a backend api as such a fast pace.

What we learned

More performance enhancing method and proper structuring of an angular frontend. As well as understand of data trying to integrate Omnisci API into an angular environment.

What's next for Healthaide

Properly implementing Omnisci API Visual and ensuring the application can handle different databases and work together seamlessly.
