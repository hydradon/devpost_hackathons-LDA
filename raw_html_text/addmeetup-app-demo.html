I've got Inspiration from Meetup.com and Udemy.com courses.

AddMeetup App helps users post and manage their meetups. Users can also post their updates, create or comment other people threads.

It is built with MEVN Stack (MongoDB, Express, Vue.js & Node.js).

Hosting it on the cloud gives me head-each (moving it from local development to heroku cloud hosting)

I'm proud of making this meetup application part time.

I've learned how to build a full-stack scalable application from Start to Finish: all the necessary tasks needed to build, test and deploy a MEVN Stack App to Heroku or Any Other Cloud Platform.

I am planning to Add more features to AddMeetup Application like custom view, manage bookings and payment options.
