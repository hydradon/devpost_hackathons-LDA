Inspiration

I was inspired by my experiences at Technica 2016.

What it does

Currently, the game is rather buggy, but it has two areas (a coding room and a food room) and a few NPCs you can talk to. You can get coffee. You can start on the main quest line (which is fairly unfinished.)

How I built it

I started from scratch in a program called Scratch, and went from there. I made some graphics in Scratch's designer.

Challenges I ran into

Time limitations and limited knowledge of coding is what set me back the furthest.

Accomplishments that I'm proud of

Getting so far in so little time and doing sprites without a drawing tablet.

What I learned

I learned about time management.

What's next for Hack-a-Thon RPG

Working minigames, more dialogue, and more coffee!
