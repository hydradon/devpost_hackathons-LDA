our inspiration was doing a wifi updated interface that showed the weather of a place

What it does: Our code gets the measurements of a temperature and humidity sensor and displays them in a constantly updated web interface

How we built it: Using Arduino code for the esp32, and html css, for the interface

Challenges we ran into: Making the network connection

Accomplishments that we're proud of: The constant updates of the measurements seen in the web server, it updates itself every 15 seconds or it can be manually updated by reloading the page, as long as youre connected to the network
