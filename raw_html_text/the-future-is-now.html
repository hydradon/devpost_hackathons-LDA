Inspiration

When we walked outside and had a discussion about how the climate had been volatile lately, we had an inspiration about what we could do to help the cause for stopping climate change.

What it does

Our product is a website that caters towards kids for the purpose of educating them about climate change in an interactive way. Using location data, we show how their area is being affected and how it's going to be affected in the future so it's easier for the children to relate to.

How we built it

We built this website using Javascript and the NOAA API.

Challenges we ran into

We developed two systems independently without planning beforehand for later integration. This led to difficulty later when integrating the code together for developing the final product.

Accomplishments that we're proud of

Utilizing specific location data to bring a personalized experience for each user.

What we learned

We learned how to integrate APIs within a javascript framework. We also learned a valuable lesson in discussing the framework beforehand to mitigate these issues later down the line.

What's next for The Future is Now

In the future, we aim to bring more features to the website to inform children in a fun and interactive way about the climate change crisis currently going on, using the location data for each user for each of these new features.
