Inspiration

Artificial intelligence has always been a fascinating topic, and creating an unstoppable killing machine in the virtual world of Grand Theft Auto IV was too fun of an opportunity to pass up.

What it does

It goes on unstoppable rampages.

How we built it

We used Tensorflow's Object Detection Models to recognize in game objects such as cars, people, and traffic lights.

Challenges we ran into

Setting up and using Tensorflow was a huge challenge given that we haven't had any experience with it in the past.

Accomplishments that we're proud of

By the end of the hackathon, we had finally created an AI that could occasionally hold its own against the citizens of Liberty City.

What we learned

We learned a lot about using Tensorflow to classify images as well as programming in Python.
