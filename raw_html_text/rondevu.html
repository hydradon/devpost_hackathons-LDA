Inspiration

We were inspired to make Rondevu because we are tired of going to the same parties. We know that there are better parties out there and all that is missing is an app to find them.

How it works

The way we want it to work is similar to Reddit and YikYak, where users can upvote and downvote parties based on their quality.

Challenges I ran into

We ran into challenges with the navigation bar as well as parsing JSON.

Accomplishments that I'm proud of

We are proud of accomplishing most of the basic functionality and a respectable UI given the time we spent on it.

What I learned

We learned a lot about android design as well as backend development.

What's next for Rondevu

We plan on greatly improving the UI, adding more functionality to prevent project X parties, and we plan on adding other events to it.
