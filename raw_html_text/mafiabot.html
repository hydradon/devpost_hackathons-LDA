Inspiration

To provide more interactive experiences with viewers on Twitch. Inspired by all the games in this genre including: Town of Salem, Werewolf, Mafia, Avalon.

What it does

MafiaBot allows viewers to queue into a game of Mafia with each other and perform all the functions of a regular Mafia game through a chatbot.

How we built it

We used python and the Twitch Chat API and code samples.

Challenges we ran into

Converting the logic of a speech-based game into text and integration with Twitch API

Accomplishments that we're proud of

A basic game with satisfactory functionality.

What we learned

We gained knowledge of the Twitch API and how to create chatbots.

What's next for MafiaBotBotBot

We would like to fix all the bugs, roll out more features, and cover all corner cases.
