Inspiration

Helping the elderly with a way to interact with modern technology. Creating a new way to managed speaker functionality.

What it does

Uses computer vision and gesture identification to control the functionalities of the speaker.

How we built it

We used OpenCV and Leap Motion Controller as our libraries, and we used Bose Soundtouch API and Leap Motion SDK as references.

Challenges we ran into

Leap motion controls were inconsistent and hard to manage and connecting with the Bose speaker.

Accomplishments that we're proud of

Pushing through!

What we learned

How to solve problems, including ones with Bose's API/speaker and XML HTTP requests.

What's next for Controlling Bose Speaker with Gesture Recognition

Adding additional features and improving recognition.
