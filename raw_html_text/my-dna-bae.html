Inspiration

The advances in genomic sequencing and biotechnology over the past several years have been significant, often bordering on science fiction, but are underutilized by ordinary people outside of health or ancestry-based contexts. My DNA Bae is intended to demonstrate what can be done with the genomic technology available at present and provoke thought on the benefits and risks of finding one's bae based on their DNA.

What it does

Using 23andMe's freely available APIs, My DNA Bae selects users based on their genetic traits and integrates those traits into their dating profiles.

How we built it

The user interface and front-end was built using HTML, CSS, Bootstrap, jQuery, JavaScript, and JavaServer Pages. The back end was built using Java for the application logic and SQL for the database.

Challenges we ran into

Fully integrating OAuth tokens and our SQL database with the overall system within the given time constraints

Accomplishments that we're proud of

Implementing responsive design principles across desktop and mobile devices, using dynamic jQuery effects to provide a compelling user interface, and creating a strong visual design paradigm and user experience

What we learned

Developing large-scale applications within a short time constraint requires making trade-offs on what functionality to prioritize for the initial release

What's next for My DNA Bae

Fully integrating 23andMe APIs/OAuth and our user account subsystem/database.
