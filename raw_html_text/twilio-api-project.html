Twilio-API-Project

Used the Twilio API to create a program to send/receive text messages using a real phone number and a virtual phone number.

-build.gradle - Implemented the Twilio SDK using Gradle -SendSMS2.java - Program to send text messages to virtual number -ReceiveSMS2.java - Program to receive text messages from virtual number
