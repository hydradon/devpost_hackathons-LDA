Inspiration

We were inspired from our personal experience of waking up early to go to school, only to find out that school was closed or buses were cancelled and we missed an opportunity to sleep in.

What it does

This application checks the twitter account of the Peel District School Board to determine whether school is closed or not. If the Peel twitter account makes a tweet saying so, then your alarm will be automatically turned off.

How we built it

We used python to write the program, python-twitter to check the twitter account of PDSB, and a dragonboard to create the alarm.

Challenges we ran into

Some challenges we ran into were being able to run python code on the dragonboard and installing the necessary packages to do so.

Accomplishments that we're proud of

An accomplishment we're proud of is being able to successfully connect the dragonboard and the python program.

What we learned

We learned that how to use and install modules with pip install, as well as how to use the PUTTY command line.

What's next for Snow Day Alarm

Currently, the alarm only works for students within the Peel School Board and we hope to expand this to other regions. We also hope to improve the accuracy of determining whether the tweet indicates schools are closed or not.
