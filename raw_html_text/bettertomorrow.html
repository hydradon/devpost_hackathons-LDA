Inspiration

I was inspired to make different component of the website due to different reasoning. I made the financial page for my friend who have a hard time staying away from retail shopping. I made the recycling page for those who are like me who are ignorant of how carefully recyling has to be done. And finally I made the mental health page from my own experience of having panic attacks and trying to cope with it.

What it does

Better finance -Uses google custom search api to limit search results from google for under specified limit. it also does not allow advertisements so people are not going down an endless rabbit hole of retail therapy. Better finance -Is a simple HTML game that helps people understand basic recycling and what goes into what bin. Better mental health -Is for people who are having a panic attack. It has various component so that since solution to panic attacks are not one size fits all.

How I built it

I used HTML, Javascript, CSS for the website structure. PHP for the contact page and google custom search API for the search engine.

Challenges I ran into

My initial plan was to do a web scrap to find different prices of online items and build a database that limit the appearance of that. I was unable to do so due to time restrictions.

Accomplishments that I'm proud of

I am very happy with how the website turned out so far in such short notice.

What I learned

Even though I was not able to implement web scrap, I was able to learn a lot about it and would love to implement it in the next hackathon.

What's next for BetterTomorrow

I will try to make an actual game from the recycling page instead of an HTML one.
