Inspiration

Similar food products out there. Hard to choose.

What it does

It searches similar products that users type in and compares the nutrition components. So people can choose different products based on needs.

How I built it

I used html, javascript, and jQuery to request data from Nutritionix API and plotted out the variables of interests.

Challenges I ran into

I have never learned the program language I used here before. So I spent a lot of time searching how to do what I want.

Accomplishments that I'm proud of

Complete a project from scratch alone (knowing nothing).

What I learned

Some basics about web development and how to use API.

What's next for NutritionixAPIWebApp
