Inspiration

Saying happy birthday is just a click away,now let's buy secret gift with the contribution of common friends

What it does

Connects with your Facebook friends and grab your Facebook friends' birthday. Reminds you who's birthday is happening first. Allows you to either pick a gift, and share it secretly with common friends and share the cost among all participating friends equally.

How we built it

Mostly JavaScript and Facebook API

Challenges we ran into

Facebook API integration and Data extraction with Graph API

Accomplishments that we are proud of

Successful integration with Facebook API

What we learned

How to master the Facebook API

What's next for GiftEd

Gift recommendation based friends' data !
