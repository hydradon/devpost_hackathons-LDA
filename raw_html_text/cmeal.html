Inspiration

Falta de tiempo (para comer entre clases, horario laboral, etc), filas y lleno desmedido en restaurantes.

What it does

Al generar un usuario, se permite el acceso a la página con un mapa que marca tu ubicación y muestra los restaurantes más cercanos. Al seleccionar algún restaurante mostrará su menú, y promociones permitiendo escoger cantidades y ordenar la comida vía internet, tiene como reserva una tarjeta de crédito que proporcionará el usuario para cargarle el importe, o la opción de pago en efectivo al llegar al restaurante. Al haber realizado el pedido desplegará el tiempo aproximado de preparación.

How we built it

Se desarrolló en HTML, CSS y Javascript, usando HTML como estructura, y modificando sus elementos con CSS y la librería de Javascript JQuery, se utilizó la API de Maps para desplegar el mapa.

Challenges we ran into

El lenguaje de programación orientada a objetos como Javascript, cuenta de varias funciones ya predeterminadas, desconocidas por el equipo que debimos investigar.

Accomplishments that we're proud of

Desarrollo de una página web, interactiva.

What we learned

Dichos lenguajes de programación, el conocimiento de una estructura ligada a diversos archivos, trabajo en equipo, manejo de versiones.

What's next for CMeal

Mejorar el back-end y front-end del proyecto, hacerlo más efectivo, tal vez desarrollar una aplicación móvil.
