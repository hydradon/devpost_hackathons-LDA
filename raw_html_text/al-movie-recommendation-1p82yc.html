Inspiration

As amateur coders, we wanted to test our skills on a practical application of python by creating a flask based web application that recommends movies to users based on a movie that they liked.

What it does

The application retrieves movie listings and its associated information and uses an data analysis algorithm that outputs 5 movie listings that are similar to an movie that the user inputed.

How we built it

Using python and flask and hosted on html.

Challenges we ran into

Inexperience in flask and html coding

Accomplishments that we're proud of

We got the output needed!

What we learned

Patience and determination
