Dog Days

Unity game for Kent Hack Enough

Short, simple, exploration RPG where a player can run around as a dog to solve puzzles by interacting with the surrounding area.

The game was developed using Unity and C#.

All sprites for the project were made during the hackathon by members of the group using GIMP and Piskel.

The full Github Repo can be found here and has a development log of the project.

An executable build is attached to this devpost project, feel free to try the game for yourself. Enjoy!

Download releases of the project from the project release page
