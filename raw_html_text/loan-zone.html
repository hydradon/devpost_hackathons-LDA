Inspiration

We wanted to put a fresh spin on the first prompt and synthesize the process of loans with the modern trend of peer-to-peer services.

What it does

Allows users to create profiles through which they can post loan offers on a market place list or pick the loans which best fit their needs.

How we built it

Java Swing was used to program the GUI and Java was also used to program the objects used within the application.

Challenges we ran into

Large segments of code had to be rewritten due to miscommunication between the different objects and code chunks in our project.

Accomplishments that we're proud of

Typing a combined 450 lines of code between only two people, persisting through multiple hard-to-decipher bugs, and completing a systematized app for a creatively productive idea.

What we learned

Organization and clear structuring is very important to precisely executing code and develop the potential for future expansion of projects

What's next for Loan Zone

Expand the amount of detail in user profiles and loan listings, implement a filter function to sort the market by, and expand the application to iOS and Android with online network connection.
