• Facial cursor

Inspiration

Gestures can have a major impact on laptops just like on mobile phones.

What it does

Detects a face and finds the landmarks for mouth and eyes. these landmarks are used as a mouse. Apart from that, hand is used as gesture to control various applications

How we built it

It is built using dlib facial landmarks detector on OpenCV using python

Challenges we ran into

Handling noise and unnecessary data. Performing multiple tasks at the same time.

Accomplishments that we're proud of

A new contribution which might be useful in the future

What we learned

How can we use basic facial points to perform large number of tasks.

What's next for Mouse-Free and Gesture Control

Using gesture based Keyboard. Using hand gestures to perform many more tasks.
