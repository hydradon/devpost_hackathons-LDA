Inspiration

I always lose against my friends when playing farkle so in order to never lose again I created an AI to help me win everytime.

What it does

It allows the user to input his or her current situation in the dice game Farkle and uses statistics to recommend the best possible move

How we built it

Using the Android API and Data structures, we created a program that runs the statistics to recommend the best move
