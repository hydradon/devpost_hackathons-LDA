Inspiration

We learnt from some videos and articles that how so many people facing mental issues should try and have a talk with someone, we got an idea like how about the immediate help for the people is their own virtual assistant!

What it does?

Alexa acts as a Virtual Therapist where it tries to start a conversation with the person trying to reduce the level of the issue he is facing. It plays songs, reads stories and also makes calls for the person in order to reduce their level of stress and depression

How we built it?

We built it using the Voiceflow and AWS interfaces specially made for Alexa, where we created special intent and choices for the person talking with Alexa. We also set up an external API with Alexa where it fetches random jokes from a website to lighten up the mood

Challenges we ran into

We faced problems while coming up with the responses for Alexa to respond to the user. We also faced trouble when we were trying to classify the inputs from the users (for ex: user can show affirmation in many ways: Yeah, Yes, Sure, etc)

Accomplishments that we're proud of

We are proud to say that Alexa can now solve over three mental issues by so many different ways. (Music, Story, Calling, Jokes, etc).

What's next for Virtual Therapist

The Virtual Therapist will try and expand its base by including more mental issues and by increasing the number of ways with which she can solve the issues.
