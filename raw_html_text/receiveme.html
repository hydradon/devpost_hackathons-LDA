ReceiveMe! Custom email processor for JIRA

What it is and what it's for

ReceiveMe! provides with a custom email processor that automatically create issues or comments on existing issues, keeping the original body text format. If the email sender does not have a valid JIRA account, then ReceiveMe! will create a JIRA account that doesn't take up any JIRA license.

What are the benefits and advantages?

  • User-friendly interface: We've respected the JIRA schemes configuration convention, so that the user feels comfortable configuring our add-on.
  • Easy creation of issues: Each new email sent to the pre-configured email address will automatically create a JIRA issue.
  • Inline image handling, text format and attachments: The issue description will maintain the text format, include embedded images and attachments the same way they were added to the originating email.
  • Easy user registration: Each email sender with no licensed JIRA account will be created as an unlicensed JIRA user.
