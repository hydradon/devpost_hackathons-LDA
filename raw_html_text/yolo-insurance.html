YOLO Insurance

You only live once, so make it count.

What it does...

A Web Application that compares and contrasts Insurance plans... Giving customers the ability to rate those plans. Plans are reviewed, and rated by users... the ranking then causes the plan to propagate to the top most view for other users.

How we built it

We used Node.js, html, css, and javascript to build the web app. We also used Firebase to store User, Company, and Plan data.

Challenges we ran into

Data Scalability. Filtering efficiently.

Accomplishments that we're proud of

We are proud to have produced a clean web app the is connected to Firebase.

What we learned

We grew in coding skill and team work.

What's next for YOLO Insurance

Depending on the ranking, maybe further development.
