Inspiration

I wanted a simple way to parse text, as recently I've been quite interested in Natural Language Processing(NLPs) and text parsing is one method

What it does

Parsr parses text in a simple terminal window. It's minimal, clean and simple.

How I built it

I used the open source Spacy library and Python.

Challenges I ran into

Due to organization issues with my team I ended up building this project on my own in a few hours.

Accomplishments that I'm proud of

Having learned more about NLPs, Python and spaCy.

What I learned

How spaCy works. How NLPs work.
