Inspiration

During a large gathering like refugee camps or relief camps, getting separated from your family is a common problem faced by many people. A few years ago, when a natural disaster struck in the northern part of India, many people got separated from their families and reunited years later. This was mainly because of the kids and people with certain disabilities were not able to provide enough information about themselves and their family. This application can be really helpful in these kinds of scenarios.

What it does

Using this application, we can either detect a lost family member or family of a lost person using facial recognition and provide the last known location of that person.

How we built it

Using Azure Face API and python

Challenges we ran into

Exploring different features of Azure Face API.

Accomplishments that we're proud of

We are able to detect whether a person is present in a group image & provide the last known location from the set of images in which he/she was present.

What we learned

How to make better use of the Azure Platform for building our application more robust.

What's next for Relief camp reunite

Currently, we built an application using only a few images. Testing in a real-time scenario is one of the future goals. Additionally, we can also include images uploaded on Social media and news channels specific to a particular event, to increase the scope of the search.
