Inspiration

Work outs are boring and hard to fit into schedules. So we decided to make a short interval workout app.

What it does

Provides short workouts for the busy person. Ranging from simple to pro.

How we built it

We used the Unity engine to build up our U.I. We used C# to code for buttons and functions.

Challenges we ran into

We ran into problems with the Unity canvas. Also we had to research a lot of code. For example, we wanted code for a GPS tracker. However the GPS tracker longitude and latitude conversions failed.

Accomplishments that we're proud of

Were proud of the fact that were here competing. Were proud of the fact that we were able to create a prototype for a app.

What we learned

We learned how to code in C#. We also learned how the Unity engine works. Plus how to create our own sprites with Firealpaca.

What's next for MobileFit

The future of MobilFit is a more complex and detailed. MobilFit will expand to beyond the workplace. MobilFit would include more then just simple exercises and will expand to diet, and more intense workouts. We wanted it to be like a reminder to workout.
