Inspiration

PCMS - Create a useful application for helping customers buy ingredients for meals

What it does

The application defines ingredients for a specified recipe and tells the customer where these ingredients are located within the store

How we built it

It is built using Springboot talking to a mySql database. The front end was built using JavaScript and html

Challenges we ran into

We intially tried to write the UI using cordova with framework 7, but it turns out that didnt work out

Accomplishments that we're proud of

We managed to finish a usable UI

What we learned

Dont use framework 7.

What's next for FindMyMeal.

Adding geolocations for items within the store. Add in the ability for customers to add their own recipe's based on items within the store.

Smart phone support - package the UI to specific device implementations.
