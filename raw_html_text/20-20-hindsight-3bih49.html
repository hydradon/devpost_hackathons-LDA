How we built it

We built our project using tensorflow and google vision to track humans. We then used python and opencv to capture the video.

Challenges we ran into

We ran into trouble having tensorflow work and actually recognize that people were in front of the webcam. We also trouble displaying the video in opencv because Ubuntu is not fully compatible with opencv.

Accomplishments that we're proud of

We're proud that we managed to put something in a small amount of time. Because we're all first-years in college, we were inexperienced with building projects from scratch. We also used programs that we never used before in a fun and creative way.

What we learned

We learned how difficult it is to work with tensorflow and new programs in general. Also, we learned that some programs are easier to use in Windows, while others are easier to use in Ubuntu.

What's next for 20/20 Hindsight

In the future, we hope to incorporate this project into a chrome extension, so others can easily use our project.
