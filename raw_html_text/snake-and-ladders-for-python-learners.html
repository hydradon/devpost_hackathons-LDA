Inspiration : To make learning easier with games

What it does : Its a game where it tests the knowledge of a person in a particular programming language using snake and ladder game. Here, the player will be asked to answer the programming question. If he answers correctly, he proceeds in the board game by 1 step. If the player answers correctly when located in the block having a ladder, he will have the chance to climb the ladder and increase his points. If he gets the answer wrong, he will be bitten by the snake and points will be reduced. Once, the player scores 100, the game will be finished.

How we built it

We used python, sql and pygame libraries to build this prototype game

Challenges we ran into

we choose a language which none of us worked on. So, it was challenging to learn the language and build it as well

Accomplishments that we're proud of

In a short period of time, we learnt python and were successfully able to create our first python game
