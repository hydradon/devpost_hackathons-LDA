Inspiration

There has been little disparity in the housing market regarding the rent estimation which we would like to resolve.

What it does

AccuRent is a web app which predicts the property rent using Graphlap's Machine learning library.

How we built it

We used nodeJS for the API integration and UI development. The Machine learning model is built using Graphlab toolkit on the Flask(python framework).

Challenges we ran into

Feature Selection and choosing the right algorithm was little challenging

Accomplishments that we're proud of

Feature selection and algorithm implementation

What we learned

We learned the new machine learning APIs and feature selection tools.

What's next for AccuRent

We will try to improve the accuracy of the system and we would also like to utilize more features from the dataset

Built With

  • graphlab-toolkit
