Inspiration

We chanced upon so many people in school and in our daily lives but we never have the chance to talk to them.

What it does

It connects people, bring people together and forge meaningful relationships. It gives you a chance to connect.

How we built it

We built it using algorithms to match users of the bot. If 2 users like each other, they will be able to receive each other's telegram username and start chatting with each other. We extract the profile picture of each user and generate a simple profile for them, they can then be 'liked' by other users.

Challenges we ran into

Our matching algorithm and data storage of the profiles initially takes up a lot of space. However, we are able to minimise the data storage network so that it takes up significantly lesser space.

Accomplishments that we're proud of

We are able to successful implement our matching algorithm, and extract useful information such as username and profile picture through telegram, without having to store all the images.

What we learned

We learnt how to successfully deploy a telegram bot, how to extract information through scripting and how to make use of database to efficiently store data.

What's next for ChanceBot

We are planning to add group/server to partition users. This will act as a filter and only allow people in the same group/server to view and match each other.
