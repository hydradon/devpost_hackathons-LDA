Inspiration

Sometimes at home we want to play table tennis, but there's no one to play with. Or, we're looking to practice but humans get bored doing multiball for others. T3* allows us to do all this by ourselves!

What it does

T3* allows us to train our table tennis strokes, forehand, backhand, sidespin, lobs, low hits, etc. It allows us to play table tennis alone.

How I built it

We used Arduino microcontroller as the brains and used dual wheel launchers as our method of throwing.

Challenges I ran into

Multiple of our 3D printing requests got rejected so our throwing wheels are not circular which are not efficient throwers. We ended up wrapping duct tape in many many layers to try to make a circle.

Accomplishments that I'm proud of

We finished the hardware and it worked other than the lack of certain parts.

What I learned

We learned how to build things and code things and use Arduino.

What's next for T3*

We need to 3D print the throwing wheel and the funnel so it will work better. Also, we need to program more functionality and add motor controllers, which we have 0 of currently.
