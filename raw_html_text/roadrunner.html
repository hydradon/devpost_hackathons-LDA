Inspiration

Bored of running on the same track and wanted to try new suggested algorithm generated new tracks

What it does

Suggests new paths for you to run at, which has minimum stoppage, based on the data collected from run history of the users in the same area

How we built it

Used Python to implement the data modeling and prediction and used Javascript to show the path to the user using google maps API

Challenges we ran into

Simulation of data as no dataset was freely available

Accomplishments that we're proud of

Finally came up with an algorithmically generated path

What we learned

Data Modeling in Python + Unity & Hololens(Experimental purposes)

What's next for RoadRunner

Integrate AR, improve the data model using real-time data instead of simulated data and improve on the UI
