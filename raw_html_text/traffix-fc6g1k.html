Inspiration

Traffic congestion problems in Brampton and the GTA.

What it does

Simulate traffic issues and urban planning.

How we built it

Based on javascript from p5.js.

Challenges we ran into

Automatically changing road types after combining multiple roads.

Accomplishments that we're proud of

Creating an algorithm (depth-first search) able to find different paths (via roads) to connect buildings to each other. Using this and taking account for population and road splits (intersections), it adds "points" to each path, revealing areas of high congestion (areas with high points).

What we learned

Coding using p5.js.

What's next for traffix

Pulling data from GTA cities and putting it into the traffix simulation.
