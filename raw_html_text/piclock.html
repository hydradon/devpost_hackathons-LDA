Inspiration

Time is infinite and so is pi

What it does

It uses the values of pi to show the current time

How we built it

We used Pygame in Python to build the GUI for the clock

Challenges we ran into

Using Pygame to display text. Not the best idea. Can't host Pygame online

Accomplishments that we're proud of

Finishing the project. Good algorithm and nice UI.

What we learned

Pygame is not good for GUIs that are not just games.

What's next for PiClock

Adding radio buttons and drop down menus to include more options of viewing the clock.
