Inspiration

What it does

Park Rangr makes it effortless to park anywhere. The app allows users to take a picture of a parking sign to determine if they can park there. The backend runs a machine learning algorithm to extract the information of the signs and sends off the information to our other algorithms for calculating things like distance from car later, or time left on parking meter. What makes Park Rangr special is the user can hold up their phone in any direction and see a pin where their car is. This is done by using augmented reality combined with mapquests current location api

How we built it

We built a native ios app that utilizes machine learning api's as well as mapquest's mapping api,

Challenges we ran into

Lots of merge issues with implementing the machine learning object recognition as well as implementing the augmented reality icons for displaying where the car is parked.

Accomplishments that we're proud of

We really accomplished what we wanted to build, the app provides a seemless way for users to find their car and it takes away the burden of parking.

What we learned

We learned how to implement the machine learning algorithms and api's as well as some augmented reality mapping. This would not be possible without the mapquest API's
