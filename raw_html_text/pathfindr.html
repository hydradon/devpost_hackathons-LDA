Inspiration

Walking too much in the cold. It's sad.

What it does

It calculates the most efficient paths between your classes.

How we built it

The distances are calculated by hand-generated data, put into a database, and then the pathfinding looks at that. That data is then put into the google maps java api and displayed.

Challenges we ran into

Nothing worked at any point, except for kinda at the end.

Accomplishments that we're proud of

It mostly works!

What's next for PathFindr

More granular pathfinding and specializations for the CMU campus, because there are some very strange features.
