Inspiration

We wanted to use Leap Motion for something fun. We were inspired by the Rock Paper Scissor Lizard Spock game on the sitcom: The Big Bang Theory. Link: https://www.youtube.com/watch?v=cSLeBKT7-sM

What it does

It is a Rock, Paper, Scissor, Lizard Spock game with a machine learning feature where the computer learns how to beat you. We have used Leap Motion to read the user's gestures. And KMP Learning algorithm to predict the user's next move and beat him/her.

How we built it

We have used Java and Leap Motion.

Challenges we ran into

We could not figure out what a suitable front end for the app would be.

Accomplishments that we're proud of

Learning to use Leap Motion.

What we learned

Some machine learning.

What's next for leapMotionRPSLS

With a kickass UI, this app could be super addictive. As the computer learns how to beat you, you are forced to be as random as possible with your moves.
