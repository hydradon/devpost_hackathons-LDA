Inspiration

What it does

Q3pay provides a payment system different from other traditional methods. It can be used to transfer money to friend on the go with just a QR code scan,it can also make secured payments on websites avoiding all the possible scams and frauds.

How we built it

We wrote everything needed for the app from scratch. We used tons of JavaScript code for all the back-end operations. We also used CSS, HTML for the front end portion,and all the graphics related designs using InkScape.

Challenges we ran into

We were having issues while trying to host the application on the raspberry pi, which took couple of hours to knock out. Also, the scanning and decoding process using the QR code wasn't efficient enough for the instant and on the go feature we were looking in the app. This was that "Click" moment for the group when we figured it out.

Accomplishments that we're proud of

We have a working web app that is ready to make transactions and we proud to see our idea turn into reality in less than 24 hours.

What we learned

We read a lot about current payment systems implemented around the world. And learned lot of technical details of these systems.

What's next for Q3pay

We are thinking about ways to make transactions in a cheapest way possible, and if we are able to make that happen Q3pay is all set to be used as an actual payment system.
