Inspiration

Inspired by solving the student loan problem, we want to build a butler that can understand your financial situation and make informed recommendations on whether certain financial related actions are viable, such as buying an expensive phone.

What it does

Alfred, the financial butler analyses all your financial transactions, income and liabilities to make a profile on your spending pattern. With that, Alfred is able to tell you the impact to your financial health for making certain unusual large purchases.

How we built it

The app is mobile based which is built in Flutter. We call Plaid's API for financial data, and the analysis model is ran in Python with Flask post the APIs to Flutter.

Challenges we ran into

We were not able to get matching data between transactions and liabilities from Plaid, so we had to improvise some transaction data in this example.

Accomplishments that we're proud of

We have made it more interactive by integrating a speech recognition tool, so that the user can speak into the phone and get the financial advice.
