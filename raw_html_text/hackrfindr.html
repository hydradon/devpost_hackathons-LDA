Inspiration

It can be difficult to find teammates for hackathons. No one wants to have a teammate that uses the wrong editor (vim master race), and sometimes you just really need someone who understands UI/UX a little better.

What it does

HackrFindr allows users to create a profile with some basic information and then "join" the HackrFindr page for different hackathons. They are then presented with the profiles of other hackers at the hackathon, and can swipe left to reject, or swipe right to accept, these other hackers. Users that mutually accept each other will be emailed with each other's contact info.

How I built it

Backend is built using Flask and Elasticsearch; frontend is simple bootstrap, jquery, and underscorejs
