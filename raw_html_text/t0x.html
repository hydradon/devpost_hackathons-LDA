Inspiration

MyEtherWallet, Shapeshift and Coinbase

What it does

It allows users to manage ERC20 token on mobile and swap them using the decentralized exchange protocol 0x.

How we built it

Used the 0x protocol for the backend swap and the Toshi platform to build a chatbot and manage funds.

Challenges we ran into

The Toshi platform is very new and buggy. We were unable to access many of the Web3.js functions that are normally available on the browser.

Accomplishments that we're proud of

We were able to allow users to manage and swap ERC20 tokens using a chatbot on mobile. Don't think thats been done before.

What we learned

How big the possibilities are in this space and how powerful everything can be when various existing ecosystems and infrastructure components are combined.

What's next for t0x

Improve the UI and add new supported tokens. Furthermore, improving the backend swap mechanism.
