Inspiration

Team formation at hackathons is always tough. As a hackathon organizer last year, I knew the struggle of trying to help foster team making at them.

What it does

We created a web app that allows users to sign up and create a profile where they can list their skills (and organize them based on skill level) and interests.

How we built it

We built the front end using HTML5, CSS and Bootstrap. The back end uses SQL to store the form queries, flask and python.

Challenges we ran into

We ran into the challenge of how to determine who to match people up with based on the skills they specified.

Accomplishments that we're proud of

Making a web app that we believe will be helpful in fostering connections between hackers and for the future of how hackathons do team forming.

What we learned

We learned more about creating a database that connects to a web application.

What's next for hacksociety

We plan on integrating more APIs (like MLH's to import a steady list source of hackathons happening) and utilizing it at BostonHacks.
