Inspiration

We were inspired by the mobile app "Rockbot" which allows users to vote on songs that play in public areas.

What it does

This project is a Discord bot that allows users to play YouTube videos over voice channels and gives them the ability to vote on which song to play next.

How we built it

We utilized the discord.py library and YouTube-dl library. The project is coded in Python.

Challenges we ran into

We ran into multiple issues with the discord.py library since it had gone through large update relatively recently that made many past code examples not functional.

Accomplishments that we're proud of

The program plays music through Discord voice channels and users can vote on songs.

What we learned

We improved our skills with Python and became familiar with the discord.py library.
