Inspiration

Whenever I am bored, I want something to do or somewhere to go.

What it does

Allows friends and the community to quickly create a social hang-out. Gives the user ability to see whats currently happening around them and their friends latest social events.

How we built it

Using Unity and C#, Gimp for design.

Challenges we ran into

Optimizing the UI with Unity.

Accomplishments that we're proud of

Getting the UI features to work properly and smoothly.

What we learned

UI masking and scrolling in Unity.

What's next for Sircle

Live feeds, range settings, feed control, friend systems, notifications, social media implementations, google calendar.
