Inspiration

We were brainstorming ideas for this HackUPC Edition until we came up with this AR game idea.

What it does

Using different markers, we instantiate a scene and multiple mirrors. A laser tower is firing a light beam that you must redirect using the mirrors so that it impacts into a magic crystal to power it up.

How we built it

We built it using Unity3D and Vuforia.

Challenges we ran into

Without a printer, it was difficult to get some markers that could do the work. Creating a refractive laser beam was quite a challenge too.

Accomplishments that we're proud of

We succeeded in using AR to create our own game.

What we learned

Positioning in augmented reality can be quite hard sometimes, and accuracy can be low.

What's next for beaconLight

It requires some more work to finish the required functionalities to make it available to users.
