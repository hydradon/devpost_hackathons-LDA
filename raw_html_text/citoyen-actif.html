Nous avons conçu une application de mobilité active. Elle récompense les usagers en leur permettant d'accumuler des points pour chaque minute de déplacement actif qu'ils font. Les usagers doivent trouver les endroits où ils marquent le plus de points. On peut penser au réseau cyclable, aux trottoirs, aux passages piétonniers, aux parcs ou même aux infrastructures municipales, pour n'en nommer que quelques uns.

Afin de garder l'usager motivé, nous lui permettont de gagner des trophées virtuels suivant ses accomplissements individuels. Il peut aussi échanger ses points pour un rabais chez des commerçants locaux partenaires.

Nous avons développé notre interface en utilisant les langages HTML, CSS et JavaScript. Pour les fonctionnalités d'arrière-plan, nous avons choisi Java et JavaScript. Notre hébergement est sur Amazon web service. Nous avons rencontré quelques petits problèmes habituels. À ça s'ajoute : la fatigue et le stress.

L'accomplissement dont nous sommes le plus fiers, est d'être rentré dans les délais et d'avoir pu présenter une application complète, efficace et tout en couleur. Nous avons renforcé nos compétences dans divers languages de programmation et avons perfectionné notre synergie d'équipe.

Notre vision d'avenir pour l'application, serait de faire une étude de marché dans le but de mettre en ligne notre produit. Notre objectif à court terme serait de mettre en place des partenariats avec plusieurs commerces locaux afin de stimuler l'intérêt pour le jeux et procurer un revenu.
