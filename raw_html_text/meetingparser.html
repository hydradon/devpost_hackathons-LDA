THE ISSUE

We all get frustrated at meetings when someone always disrupts others. And what about those times that you all forget what you said during fierce debates? We developed AhaMeeting to be that assistant with superb memory and totally objective analytical skills. This way, we don't have to worry about wasting time taking notes during important meetings, and just focus on discussing ideas.

OUR MAGIC SOLUTION

We used Microsoft Cognitive Services API and IBM Watson Tone Analyzer to analyze how participants talk during a meeting. Users get a clear overview of the duration, tone, language style, and emotions behind each conversation. This would help to improve the efficiency of future meetings, and provide a sense of objectivity during disputes. We used Twilio to support on-the-go meeting recordings.

HOW DOES IT WORK?

  1. Call to register voice: each participant calls a Twilio number to register voice.
  2. Call to start recording meeting: the meeting host will call Twilio to record the meeting.
  3. Report analysis: the meeting host will receive a text with URL to meeting analysis.
