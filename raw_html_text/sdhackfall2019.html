Inspiration

Plastics in the ocean is a dangerous threat to sea life. We want to teach children this fact to raise awareness, and we think the best way to do it is with a simple, fun twist on the classic game of Snake.

What it does

You are a mama turtle, and you have to navigate through the ocean to find your 20 baby turtles, while avoiding plastic litters at the same time. Once all baby turtles are collected, an exit opens up at the top. Swim there to beat the game and save your babies!

How we built it

We used Java's graphics library to code the game, and Android Studio to create the app.

Challenges we ran into

Familiarizing ourselves with Android Studio, the XML library, learning how to use Git and handle merge conflicts, drawing graphics and using threads in Java.

Accomplishments that we're proud of

That we even finished building the app.

What we learned

Software development is hard and you should start early and start often. Also having teammates and communication and delegation is very important.

What's next for Turtles

Sleeping and homework probably.
