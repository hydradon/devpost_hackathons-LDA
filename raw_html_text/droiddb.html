Inspiration

Many newly formed VEX teams do not have the necessary tools and experience at competitions to scout other teams properly.

What it does

Gives VEX teams the ability to gather data from other teams competing at robotics tournaments, from Types of Awards Won to Offensive Power Rankings, and properly scout for teams when it comes time to chose or be chosen for elimination rounds.

How we built it

Used Maven with Java to get dependencies for Google APIs

Challenges we ran into

Creating a web app using Java, and bridging the actual website to the code.

Accomplishments that we're proud of

Getting the Google Sheets API to work on our back-end.

What we learned

There is a lot more to learn in terms of implementing Java onto a website.

What's next for DroidDB

Including a profile to save user data. Ex: Sign-in with Google Using Google Cloud to implement access to foreign teams by translating the website to multiple languages.
