Inspiration

There are many paid market research tools on the internet (that charge per API call). Marketlink looks to serve a new market by allowing payment to be facilitated via cryptocurrency and the research tied to a smart contract.

What it does

  • Allows paying ethereum for doing market research on domain names.

How I built it

  • Use solidity, honeycomb, and a reactjs front end to do BI (business intelligence) searches against the Regulus API wrapper built by honeycomb.

Challenges

  • Taking the example base smart contract, deploying it, and adjusting it for doing business research and tying API results to the memory of a smart contract.

What I learned

  • How to integrate honeycomb and deploy smart contracts on ropsten.
