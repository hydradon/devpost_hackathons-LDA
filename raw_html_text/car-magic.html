What it does

Car Magic provides you a personal concierge that drives it to the dealership, and gets your car the quality care it deserves!

How we built it

We used the SmartCar API to get car location and unlock the doors so the concierges can easily access the car. Twilio provided an excellent way to give our users notifications. The majority of our project was written in a Python/Flask stack which provided a quick web framework to get us running fast. Our front end used JavaScript, Bootstrap, and Flask for easy customization and information.
