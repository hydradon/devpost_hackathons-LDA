Inspiration

This was a simple project made for fun to inspire people, and we had fun making a simple chrome extension.

What it does

Generates a motivational meme randomly

How we built it

Using html, css and js

Challenges we ran into

We faced challenges every step of the way, as chrome extensions were completely new to us. We hope that with further preparation and understanding, we can make extensions with more features

Accomplishments that we're proud of

We're proud of being able to make a simple extension, hopefully to further our learning in this scope

What we learned

Through the help of mentors we learned that simple tasks can be very fun, and we enjoyed learning about the browser extension features.

What's next for MotiMeme

Adding memes to the array automatically as opposed to manually adding them to the array of memes, possibly by linking to google images, etc.
