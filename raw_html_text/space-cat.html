Space Cat!

Because cats will colonise the planets before we do.

This interactive page was a way for our team to learn javascript. It's a new language to us and we wanted to experiment with animations.

Eventually we would like to have Space cat visit different planets, and users will learn about the Solar System. We'd aim this at high school students, with some useful facts for each planet.
