Inspiration

From the experiences of the team, walking home in an unfamiliar area can be frightening for many. In order to make people feel more secure when they are commuting, we wanted to provide people with access to data about the crime/incidents in a neighborhood.

What it does

By compiling information from police reports, local news, and reports from users, Sight provides the user with an easy to read map containing icons which categorize the type of crime logged in the nearby area.

How we built it

Android Studio

Challenges we ran into

Given the limited time and the fact that the team members were beginners, we were not able to code a fully-functioning application.

Accomplishments that we're proud of

We are proud that we were able to create a functioning application which can benefit the lives of many, especially since the team is comprised first-timers at hackathons.

What's next for Sight

We plan on integrating Google Maps with the data collected to provide a fully-functioning application for users. In addition, we plan on updating the UI.
