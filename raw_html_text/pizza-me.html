Inspiration

Maybe you're at a party. Maybe you're really hungry. Whatever the situation, you need lots of pizza, as soon as possible. Ordering from one store will take too long.

What it does

Peppero.me orders one 14-inch large cheeze pizza from every Domino's within 10 miles of your current location.

How we built it

We used a node API that can automatically create Domino's orders on the backend. The Google maps API shows you every location that the orders are placed to.

On the frontend, we used HTML5 and sass to create a cool-looking UI and button animation.

Challenges we ran into

Getting nearby locations was an asynchronous call, but we needed to wait for it to finish before loading the next screen. We solved it with a series of callbacks.

Accomplishments that we're proud of

The pizza button Getting Domino's locations

What's next for peppero.me

We plan on using the Postmates API to expand to all pizza stores, not just Domino's.
