Inspiration

What it does

It provides an online streaming service for tutorials of any subject where students can ask questions regardless of distance.

How we built it

We will use streaming services from TokBox and run the software from a HTML website.

Challenges we ran into

Accomplishments that we're proud of

What we learned

We learned how to program in HTML and utilize different functions of TokBox

What's next for Expla Nation
