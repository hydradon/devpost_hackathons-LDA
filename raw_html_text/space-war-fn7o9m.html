Inspiration

We are just beginners with coding and this was our first gaming code.

What it does

This code is specifically for a few seconds of entertainment to a user who needs a quick distraction from his or her work.

How we built it

We thought of old games like pac-man but implemented a similar piece of code that is easier to operate. (more space to move the rocket ship)

Challenges we ran into

We had a few glitches where the user could go out of bounds but the witch could not. This would defeat the purpose of the game and would be a potential cheat. We used trigonometry and geometry to come up with ways to make bounds for the user's rocket ship.

Accomplishments that we're proud of

We made a quick game!

What we learned

We can definitely do a more complicated program soon which we will work on in the next 24 hours.
