Inspiration

Fusion 360 does not have the helix functionality. So for me while creating some helical sweeps i had faced lot of issues. That was the Inspiration for me to create this app and help others who face the same problem.

What it does

It creates the helix spline with two options by angle or by pitch.

How I built it

Basic helix logic and fusion 360 api's.

Challenges I ran into

This was an easy project but valuable one..

Accomplishments that I'm proud of

I am glad that i am making this functionality available for others which would be great help for them.

What I learned

I got to lean about the fusion 360 api's. Which will be great help in future.
