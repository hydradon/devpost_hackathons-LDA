Inspiration

We Wanted to make something using a technology we didn't know. We felt that creating competition with people to clean up their community sounded like a great idea.

What it does

Uses Google's Cloud Platform and MongoDb to have user's register their data and mark places in their community that would be in need of some cleaning help

How I built it

We initially used React.js as our framework but ran into problems when integrating Google's Map API and geolocation. We then switched to a simpler template using handlebar.js but still using MongoDb as our database. We were able to then easily integrate the map.

Accomplishments that I'm proud of

Having to almost reset our project was a tough blow morally, physically, and emotionally on the group; on top of the growing fatigue. We were able to pull through and deploy to a presentable state while accomplishing our main goals.

What I learned

That in 36 hours we can do more than we have done in a week or more. Together, we were able to put our minds together, create new things with technologies we knew little about.

What's next for Pixit

It would be to include more features including the Google Map API since it is our main feature. Regional battles between communities for who could maintain the cleanest area sounds like a fun and cool way to preserve the beauty of our city and nature.
