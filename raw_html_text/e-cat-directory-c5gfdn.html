Inspiration

Most likely the current directory are dull and not easy to read. So we are making a directory with ai to provide a better experience for customer in a mall.

What it does

It includes an AI chat bot to provide a more interactive and multi language support directory for customer to shop more conveniently, and business wise, it can provide a better platform to promote and advertise.

How I built it

We used azure web bot to build the chat bot that can answer the customer. By answering customer, it will provide a suggestion which can also be used in advertising. Data will be collected according to the emotion and the clothing of the customer to provide the best promotion effect.

Challenges I ran into

The chat bot is very difficult to deploy, we tried the whole night and the latest version can not be loaded in to the web app. Luckily, we got luck in the morning and finished the job.

Accomplishments that I'm proud of

We successfully made a machine learning environment and chat bot, although face api is still in process, we still learn much from there

What I learned

How to use different tools with Microsoft Azure

What's next for E Cat Directory

Mobile app that can load the app to the phone to provide a better experience
