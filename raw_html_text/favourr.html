Inspiration

Fiverr was a major inspiration along with the common use of instant messaging for asking for favours.

What it does

The website allows you to create a favour you wish for someone to do for you in exchange for your time used elsewhere in a one way platform.

How we built it

We used C# and ASP.NET Web Forms to build the entire system and a MySQL database for storing the data.

Challenges we ran into

Originally we wanted to use the Google OAuth API but couldn't get it working with our implementation and went with just using an inbuilt login system. We also had issues with the inbuilt SQL Server implementation and went with a standard MySQL server.

Accomplishments that we're proud of

Learning how to use OAuth within C# Creating a functional website in ASP.NET

What we learned

See above.

What's next for Favourr

We want to extend Favourr and add more of the nice to have features to it and ensure that it would work in an online environment with concurrent users.
