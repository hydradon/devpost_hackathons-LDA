Inspiration

As artists, we know the difficulty of creating sounds right for each song that evolves in our minds. But on the opposite end of the spectrum, many songs uses the exact same set of samples.

What it does

We wanted to change the way sounds begin - starting as expressing emotions, thoughts, or ideas, and letting Max MSP do the heavy lifting for us.

How we built it

NodeJS server takes requests, performs analytics on watson, sends serialized data to MaxMSP server.

Challenges we ran into

Interfacing Node with Max is a hack . This was crazy difficult to get working.

Accomplishments that we're proud of

We want to use this product!

What we learned

Interfacing with Max is hard. Debugging in Max is quite difficult from a traditional programming perspective. We made great strides, but also learned that max msp is the best language to produce high quality sounds in a timely manner.
