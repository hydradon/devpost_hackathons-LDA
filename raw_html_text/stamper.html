Inspiration

I make cookie cutters that I 3D print to fund college. I want to automate the cookie cutter making process so I have to start somewhere. I decided to make a stamp maker program as a start

What it does

It take's any png or jpeg file that is 300x300 and turns it into a stl file to be printed on a 3D printer.

How I built it

I build it with python 3 and OpenSCad

Challenges I ran into

Getting to run Linux commands through python proved a challenge since the packages did not always work.

Accomplishments that I'm proud of

I made the program compile more than one file at one time

What I learned

I learned how to extract file names from directory's, run Linux commands with python library's, and use Pillows for image processing.

What's next for Stamper

The next thing to do is to move the python project to a web server.
