Inspiration

Let’s face it, forms are boring, it’s a way of the past and certainly not the way things are going to be done in 2017. Coming up with our solution, we really wanted to emphasis a smooth and friction-free UX…

What it does

For this, our concept allows users to interact with userforms and surveys through the interface of a conversational chatbot on facebook messenger. The bot intuitively walks users through the process of creating and answering surveys all in the Messenger interface.
1)Eliminates the process of walking back and forward between apps

2)A platform that everyone uses and spends a lot of time on already

How we built it

Using Microsoft Azure's bot framework, Cognitive abilities, ,NodeJS and Facebook Messenger API and Facebook Analytics API

Achievements we are proud of

Simplicity of the user-interface and complexity of backend in creating the perfect
