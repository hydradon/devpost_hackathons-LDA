Inspiration

We wanted a way to spice up the usual web browsing experience.

What it does

Meme-age is a chrome extension that with the click of a button, turns all the images on a page into 5 star quality memes for your entertainment purposes.

How we built it

We built it using html, css, and jquery. Logos were designed using GIMP.

Challenges we ran into

We originally wanted to be able to replace advertisements with memes but found it difficult to identify which elements of a webpage were actually ads.

Accomplishments that we're proud of

Creating something that works! Learning jquery and how to create a chrome web extension.

What we learned

jquery, html, javascript

What's next for Meme-age

Actually accomplish what we set out to accomplish which is replacing advertisements with memes.
