Inspiration - The daily hassle that every person undergoes at UCF was the major inspiration for coming up with this easy to handle and monitoring parking system.

What it does - Each parking spot will have a sensor that will detect whether the spot is free or not.

How we built it - Using an arduino coupled with a light sensor , send the spot information to the server and a web page will display the data

Challenges we ran into - lots of backend problems.

Accomplishments that we're proud of - A working protoype that is good to be scaled up !

What we learned - Never give up !

What's next for Knight Parking - INcreasing the scale of the system.
