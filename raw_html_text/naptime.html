Inspiration

We were to lazy to always set an alarm or countdown for the little things. Besides Siri can do it but Android doesn't have this luxury.

What it does

It utilizes the in-built speech recognition api for android. It asks the user how long he/she will like to nap and computes the desired time It goes far to hear the time arguments in any given sentence. For example, if I were to say Vincent wants to take a nap for a half hour. The application will accurately set a countdown for 30 minutes

How I built it

We used java and xml to code in android studio.

Challenges I ran into

A lot. For example, extracting the time argument from any given sentence.

Accomplishments that I'm proud of

Be able to do in two days. With lots of RedBull

What I learned

Life is hard when you're not in sync

What's next for Naptime

Be more interactive and more fun.
