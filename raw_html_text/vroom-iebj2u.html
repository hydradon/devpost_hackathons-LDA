Inspiration

The rising threat of global warming due to increased CO2 emissions in the atmosphere has a great contributing factor of transportation routes that emit a lot of these greenhouse gases.

What it does

Monitors and quantifies a user's carbon footprint using their Google Maps data.

How I built it

By implementing the Google Maps API, the routes taken by each user was extracted. Using constants from reliable online sources, the CO2 emission averages were calculated.

Challenges I ran into

Learning the Google Maps API in a short amount of time.

Accomplishments that I'm proud of

Managing to finish the prototype.

What I learned

Better time management skills are crucial for any type of hackathon. A lot of patience as well.
