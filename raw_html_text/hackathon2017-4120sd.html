Inspiration

Just seemed like a neat thing to try to do and that it might make a useful tool if done right.

What it does

Allows you to create very basic JavaScript programs and execute them using only a drag-and-drop interface.

How we built it

Eclipse was our IDE using Java as the primary language making use of the swing library.

Challenges we ran into

Drag-and-drop is not easy. Especially when neither project member is proficient in using Java.

Accomplishments that we're proud of

Getting something that vaguely resembled the idea we were going for working.

What we learned

A whole lot of swing and Java.
