Inspiration

Wanted to build a really realistic simulation of an office.

What it does

You can explore the offices build along with enjoying the realism of the environment

How I built it

We used Unreal Engine 4.15 to build it.

Challenges I ran into

Could not import assets easily so we decided to make our own.

Accomplishments that I'm proud of

We were able to build a nice office space using our limited knowledge.

What I learned

We learned a lot about how to use Unreal and how events work in Unreal using Blueprints.

What's next for Jumping To Offices

We will probably make more offices to explore.
