About

Provides an easy integration with the eeze.io authentication provider.

Installation

To install from the Marketplace , click on the "Add to Project" button and then select the project you wish to add it to before updating the project to pull in the plugin.

To install from the backend, go to Settings -> Updates & Plugins -> Install Plugins and then search for LukeTowers.EEZEAuth .

To install from the repository , clone it into plugins/luketowers/eezeauth and then run composer update from your project root in order to pull in the dependencies.

To install it with Composer, run composer require luketowers/oc-eezeauth-plugin from your project root.

Documentation

Simply follow the instructions on the plugin's settings page!
