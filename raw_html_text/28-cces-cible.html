Résumé

Accès-cible est une application web compatible mobile .

Elle permet autant aux particuliers qu'aux entreprises de trouver l'emplacement idéal à partir de l'obtention d'une cote d'accessibilité .

Cette cote est calculée grâce aux données disponibles sur les modes de transports alternatifs .

Jeux de données

Metro + Bus + Pistes cyclables de Montréal
