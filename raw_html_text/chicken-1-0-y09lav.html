Beginner Project

Inspiration

We wanted a small racecar which runs away from everything, even us!

What it does

It's a remote control racecar, hacked together with ultrasonic sensors to be able to detect its surroundings. Using this to gather information about its immediate environment, it moves away from everything it senses.

How we built it

We built this racecar using an existing remote control race car, used an Arduino with 4 ultrasonic sensors and laser cut a structure to support the arduino, sensors and battery. The code is written in C.

Challenges we ran into

Like with most automated robotics, we ran into issues when implementing and testing the racecar to move away from all objects. It was difficult to reliably react to the data gained from the sensors.

Accomplishments that we're proud of

We are very proud of both how the result of the project looks and how it is able to move away objects! The relays even make the car cluck like a chicken!

What we learned

We learnt a lot about developing on the Arduino, reading from sensors and also laser cutting designs to make out project look cool!
