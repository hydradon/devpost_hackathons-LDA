Inspiration

We might not notice it on our daily basis but we lose significant amount of time looking for parking space. And we wanted to make an app to target that issue.

What it does

It is a miniature model of a parking lot. We used an Intel Edison to detect the proximity of the vehicles on the parking space and displays the vacant spaces on the entrance display board.

How we built it

We connected the light sensors the the grove base and set up the algorithm to figure out if there are any parking spaces left on the lot.

Challenges we ran into

None

Accomplishments that we're proud of
