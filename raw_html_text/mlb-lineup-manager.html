Inspiration

We wanted to build a tool that would allow a manager to input their preferences and construct a lineup based on them

What it does

It allows a manager to input how important a few factors are to them and the recieve a lineup

How I built it

We used Vue to build the front end and MySQL on the back end with AWS Lambda for the API. We used data from the MLB Gameday API and RetroLahman

Challenges I ran into

Time, data availability

What's next for mlb-lineup-manager

Fleshing out the full functionality and getting the web app working
