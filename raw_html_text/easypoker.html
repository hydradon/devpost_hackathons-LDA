Inspiration

Weariness of being asked what the value of each chip color is.

What it does

Its intent is to allow the user to set up a basic home game of poker by keeping track of time and the chips in play.

How we built it

The back-end was developed through Node.js which is hidden by the HTML and CSS in the front-end.

Challenges we ran into

We ran into technical debt and not integrating back-ends soon enough.

Accomplishments that we're proud of

We're proud of presenting this web app.

What we learned

We learned that for tasks such as this, it would be imminent to set goals at certain times to create a more finished product.

What's next for easypoker

Hopefully becoming a fully developed web app that works as expected.
