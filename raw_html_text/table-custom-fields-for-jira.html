Inspiration

JIRA only allows 1 custom field to be displayed per row and there are various panels for date and people. There is a need to group associated fields together as well to provide support for automation. It is not easy to reconstruct the changes from the Change History tab, and it is not possible to display them in the search issue or PDF.

What it does

Provides bird's eye view by grouping associated fields together into a table, with automated update via flexible workflow post functions.
