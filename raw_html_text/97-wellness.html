Inspiration

Wanted to enable continuous monitoring of mental health for patients with Anxiety Disorders.

What it does

Takes a photo, analyzes emotions. Takes in a journal entry, analyzes sentiment. Passes all that data to therapist.

Helpful for exposure and response prevention therapy monitoring.

Also, performs music therapy through intelligent tagging using the YouTube API.

How I built it

Microsoft Cognitive Services: Face API Emotion API Linguistic Analysis Text Analysis Sentiment Analysis

YouTube API: Used intelligent tagging to play music based on mood

PHP – MySQL backend Android SDK HttpRequest Library

Challenges I ran into

Network Failures. JSON Posting. Building the backend. Getting all the APIs to work together.

Accomplishments that I'm proud of

Finishing the complete app.

What I learned

A lot.
