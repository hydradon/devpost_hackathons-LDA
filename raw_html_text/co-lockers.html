Inspiration

To make co-working facilities as convenient as private office space

What it does

Provides co-working users with access to on-demand lockers that can be used to secure personal belongings or as a delivery point for laundry, dry cleaning or online shopping.

How we built it

We built a demo using Firebase and Node and a simple web-UI. We integrated 2-way SMS communication with Twilio and integrated the Salto digital lock solution.

Challenges we ran into

We were using Firebase and Salto for the first time, so there was a learning curve - we've also not done dev for a long time :P

Accomplishments that we're proud of

Working demo, solid concept and business model. Zero UI when using the SMS solution.

What we learned

Improved knowledge of the Twilio, Salto and Firebase platforms!

What's next for Co-Lockers

Integration with different partners like co-working management systems that can offer the Co-Locker solution as an extra or as part of their standard offering to users. Deeper integration with Salto to take advantage of other features of their system. Adding a payment engine to allow users to pay. Implementation into a trial space.
