PiTer

PiTer is a new, efficient, powerful, scanning tool (including front-end collection, preprocessing and back-end image processing and text dectection).

  • Take a picture from the app.
  • Rotate it to optimal angle and upload it to AWS/Google Cloud Platform server.
  • Return the text in the picture to front-end and convenient for user to edit it.

Magic Features!

  • Process images with any angle(You don't need scan a picture vertically).
  • You will get all the text on the image directely through the App.
  • The App can also read the text on it which is friendly with people with visual impairment.

You can also:

  • Name all the scanned pictures
  • Save or delete all history pictures
  • Copy texts when you get it
  • Adjust the picture when you upload it
  • Export documents as Word and PDF
