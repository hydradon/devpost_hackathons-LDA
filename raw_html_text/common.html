Inspiration

We were inspired by Magnit Message API ;)

What it does

It connects people who have common likes on Facebook. After that, the app matches these two users and tells them that they have common likes which they have to GUESS by working in team. Both users don't see each others profile pictures, names or any other info - complete incognito. The only thing they have - the chat. With communication and will to win they will succeed and guess their common interests. After the win, both users can see each others profile.

How I built it

We used CocoaPods, Facebook SDK, Objective-C and, of course, Magnit API
