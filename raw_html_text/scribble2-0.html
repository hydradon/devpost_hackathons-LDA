Inspiration: We wanted to build something to catch quick ideas easily and efficiently for creative types.

What it does: Saves your digital memos with the option to share them or keep ideas and entries private.

How I built it: HTML and CSS.

Challenges I ran into: Taking in user input and saving entries.

Accomplishments that I'm proud of: Learning web development tools for the first time.

What I learned: HTML, CSS, how to navigate web dev.

What's next for Scribble2.0: Adding entries, login, the ability to connect with social media platforms through API use.
