What it does

It's a side-scroller written in C for the Pebble smartwatch.

How I built it

I used the Pebble SDK and the C programming language to write it atop the (very basic) Pebble Game Engine.

Challenges I ran into

This is my first time ever programming a Pebble and my first time using C in several years. Thus, I ran into many syntax errors.

What I learned

Pebbles are surprisingly easy to code for.
