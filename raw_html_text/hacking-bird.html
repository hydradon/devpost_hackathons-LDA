Inspiration

The inspiration for this project was to be able to include individuals who are visually challenged into the great world of gaming. We all know how great of an experience it is to play video games and we wanted to bring a similar kind of feeling to those who cannot experience it normally through sight.

What it does

Essentially, this application was inspired by the famous game "Flappy Bird" and it is an accessible version of it. In order to make it playable for the visually challenged, we decided to substitute the main visual points of the game, into audible ones.

How we built it

We built this project using the programming language Python and a library in Python called pygame.

Challenges we ran into

We ran into many design issues because the concept was one that none of us have ever attempted to tackle prior to this hackathon. Thus we spent lots of time brainstorming in order to come up with the best approach to accomplish this goal.

Accomplishments that we're proud of

I was proud of our team for accomplishing what we set out to do which was to make this great game accessible to more individuals who may not have the luxury of having eyesight.

What we learned

From this project, we learned how to embrace differences and how to create a product that can be used not by just one group, but by many. We learned that if we have a desire to help others feel the joy of gaming, we could make it happen.

What's next for Hacking Bird

We want to perfect this app in order to make it seamless and accessible at the same time. We want to use some sort of external indicators through the form of sensors in order to make the game feel even more immersive to the visually imapired.
