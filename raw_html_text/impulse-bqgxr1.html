Inspiration

The main inspiration for Impulse was to create a platform where impulsive consumer interactions would meet appealing personalized products. We want to bring products to the attention of consumers in a way that would facilitate ease of purchase through a fun and intuitive interface.

What it does

Impulse exposes users to a range of awesome products through an interface that utilizes a 'budget slider button' and a swipe-left/swipe-right system. We hope to use the swipe-left/swipe-right system to cater more relevant products to the user. By showing more products that are similar to ones that users swipe right on and by showing fewer products that are similar to ones that users swipe left on, the app will become personalized. For all of the items that are swiped right on, they will be saved into a favorites list, where a simple gesture would facilitate the purchasing of that product.

How we built it

Our team utilized Android Studio for the code and Sketch, Photoshop, and Invision for the design.

Challenges we ran into

As we are all beginners, understanding Android Studio was the biggest challenge of this project. Using the Amazon API proved to be more confusing than helpful, but we hope to understand it completely in the future.

Accomplishments that we're proud of

We were able to create a visual prototype that represents how the user would interact with this application.

What we learned

Mobile development, Android Studio, and how to integrate design with code.

What's next for Impulse

We hope to finish the coding portion of this project and create a fully functional application!
