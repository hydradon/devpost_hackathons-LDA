Inspiration

We wanted to create an experience that combines the physical world with the mass amount of information in the digital world.

What it does

Our application encourages people to explore the world around us, if they come across a place that the service can recognize, Tales will tell the user more about this specific place.

How we built it

Using Android Studio, with API calls to google cloud places and wikipedia!

Challenges we ran into

Android Studio is hard. API calls are so hard!!

Accomplishments that we're proud of

We finished a baseline application that works!!
