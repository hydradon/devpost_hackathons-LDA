Inspiration

Our own pets! We always enjoyed programming virtual pets, but we wanted to try and make a real pet.

What it does

Provides companionship, love, wiggles when it is petted, "barks", roams a pseudo-random path.

How we built it

We used glue, gorilla tape, an arduino sidekick box, some motors, an Altoids box, and lots of love.

Challenges we ran into

Wiring, debugging navigation code, our ultrasonic sensor (that doesn't work), and initially we had to wire our own H-bridge.

Accomplishments that we're proud of

We learned to wire our own H-bridge, we had a successful bot run, and that we came up with an idea.

What we learned

We learned to always test sensors before you put it on the bot.

What's next for Jacc Bags

A makeover, and a companion app that allows users to fully care for their virtu-real pet.
