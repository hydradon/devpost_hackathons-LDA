Inspiration

I am a university student and I met with people daily. In our discussions their are so many things in our hearts that we want to say but due to shyness or our relation with them we are unable to express our selves , so because of these things i came with an idea by chance why not make an app through which we can send them private and secret messages without disclosing our identity .

What it does

So basically what this app does is that it allows you to send messages to people, friends and relatives . This message is sent secretly on Facebook . I chose Facebook because now a days it is the most popular social media even people with no computer knowledge uses Facebook in my country. The best point is that after sending your message you can also receive amazing replies from receiver . Through this you can convey your message to people and know what they think about different things. Also it can help children secretly ask their parents and know what are their reactions on different topics and react according to that next time.

How I built it

Ask Secretly is a web based Facebook app that can run on Facebook.com as an app. So its much easier for users. I have used Facebook API , PHP5+ , Javascript , HTML/CSS and a bit of my hands :) . I have uploaded this app to my Linux hosting which supports this app easily. Made in 14- 15 days it is extremely easy to use and user friendly. My app is not yet verified by Facebook but i have applied for it 2 weeks before. Waiting for the results of this app.

Challenges I ran into and Accomplishments that I'm proud of

I was working in PHP before this but i chose to make this app totally in Javascript other than server side. I have used Facebook Javascript SDK. It was a bit confusing for me to use and connect Facebook API but at the end i was able to learn so many things and i learned how API's work and what is graph API that Facebook introduced. It was really amazing accomplishment for me.

What I learned

I have learned lots of things as mentioned above first of all i learned how you integrate your code and apps with big platforms and how you talk to these platforms using API's. I am more towards senior in Javascript because i really learned new functions and i have used DOM manipulation which was great. And i learned that when you code you really do enjoy :) .
