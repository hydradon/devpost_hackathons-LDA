Inspiration

The bugs encountered while programming in workshops.

What it does

A two-dimensional, RPG-style adventure game.

How we built it

Unity rendering engine, 2D assets (drawn by us and imported from royalty-free sites), map and level design by team members, original hand coded scripts and AI.

Challenges we ran into

Non-functioning scripts, collaboration issues with MS Sharepoint, issues with uploading to Github.

Accomplishments that we're proud of

Making a nice-looking game in 24 hours!

What we learned

The various steps in making a game (concept, designing assets, scripting, finishing touches).

What's next for Debugger

Adding more in-depth and engaging gameplay!
