Inspiration

The Bird Box Movie and the Visually Impaired community

What it does

This collision detector tool senses any object under 100cm and warns the user if any object approaches close to them by signaling audio and visually notifying the user

How we built it

We used an open source project, which had a rudimentary sonar system. We made modifications to have a compact solution for the visually impaired. We used the Arduino IDE to code the functions of the system using C, while the GUI was coded using java in the processing IDE. In addition, we integrated a beacon and a warning led to signal closer objects

Challenges we ran into

_ italics _ We ran into having trouble in configuring the Arduino to all the different components. In addition, we had trouble in scaling the GUI according to our screen. Other than that were successful in overcoming the challenges faced.

Accomplishments that we're proud of

We are proud of having the intention to help the visually impaired by a movie inspiration.

What we learned

We got hands-on experience in using the processing IDE and the Arduino mega unit. This is our first hardware hack solution in a hackathon.

What's next for Bird Box Collision Detector

We would like to see if we can integrate Bird Box into an ultra-compact solution so we can have it on wearables and accessories.
