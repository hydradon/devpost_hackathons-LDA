• A demo

Inspiration

We wanted to control the computer without touching it. We got our main inspiration from that one scene in iron man.

What it does

Controls the mouse and has a few applications using just your hand and the kinect.

How we built it

Using the kinect and pykinect library.

Challenges we ran into

Getting accurate data from the kinect as the library is not documented and there are almost no tutorials available.

Accomplishments that we're proud of

Using pygame to create a User INterface. Controlling the mouse with your hand.

What we learned

How to use Pygame to create a user interface. To use the kinect.

What's next for Kinect Computer Control

To make the entire computer controllable with just your hands.
