Inspiration

With the constantly growing number of social medias, it often gets out of hand to check and keep up with each and every single post.

What it does

ZotTrending gathers trending posts from different social media platforms using APIs, displaying them all as cards in one webpage, no more jumping from app to app checking all the posts!

How We built it

Back-end built using praw library and python-twitter on python. Front-end built using flask, bootstrap4, and html.

Challenges We ran into

-Getting an idea that everyone agreed upon -Getting APIs to work properly -Understanding documentation on different wrapper modules -Git commit issues

Accomplishments that I'm proud of

We made a working website on our very first hackathon!

What I learned

We learned to use Git properly through the command line. Flask to connect front end with back end on python.
