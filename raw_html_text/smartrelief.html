Inspiration

The process of relief funding is inefficient which slows the process of financial aid disbursement No current technology provides forecast-based financing procedures

What it does

Quick Relief uses blockchain technology to enable faster, secure, and transparent financial aid distribution for disaster relief.

How we built it

We used IBM-Watson Disaster Relief challenge and thought about ways the blockchain technology can help increase the efficiency of disaster relief help. We used Stellar for the blockchain implementation, Plaid for the financial documentation and record keeping for tax deductible purposes, and Docusign for legitimate signing and record keeping. All these mechanisms utilize an efficient and innovate way to crowdfund disaster relief operations.
