Inspiration

With Internet, we can connect to anyone almost instantly, however, no longer do we pay attention to things near us. This platform is designed to bring nearby information to you.

What it does

It is a geo-social platform. The main idea is to let you know the events around you. Instead of the address book approach, it uses your location to group messages and information.

Besides connecting you to your neighbors, it also provides a platform for:

  • lost & found
  • public announcements
  • second-hand sell
  • request maintenance to broken public facility
  • or just ask for help

How we built it

Qingfan Wu built the server with python and flask, Guoye Zhang built the iOS app with Swift.

Challenges we ran into
