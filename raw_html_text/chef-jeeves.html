Inspiration: Hunger and curiosity. Wonder what kind of different possible recipe are there while using the same ingredients

What it does: It tells you the list of recipe you can make based on the ingredients you have.

How we built it: ASP.NET MVC using C# AND SQL. HTML5, CSS3, jQuery, javaScript, mySQL, azure.

Challenges we ran into: Time constraint as we have so much idea but too little time

Accomplishments that we're proud of: Creative logo, neat design.

What we learned: ASP.NET MVC

What's next for Chef Jeeves: Extra features such as completion rate for recipes with just a few ingredients missing, calorie and nutrients calculations, more recipes, cuisine category, diet restriction,
