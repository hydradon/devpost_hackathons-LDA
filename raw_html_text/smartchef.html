Inspiration

We wanted an application that helped us out to find the cost of your favorite recipe.

What it does

It makes a sweep of all the prices of all ingredients for a particular recipe using location as a priority.

How we built it

We built it using NodeJs in the backend with MySQL and Swift and Android in the front -end.
