Inspiration

We wanted to try our hand at speech-recognition, and also at web development.

What it does

The Anny Adventure website allows users to create a choose your own adventure game, which is uploaded to the database for other users to browse and play! The game itself is controlled by voice.

How we built it

We imported annyang! for speech recognition, and speech.js for text-to-speech (for the Narrator). The website is built by mostly javascript with html. Firebase is used as the database.

Challenges we ran into

Learning to work with annyang efficiently; learning to use Firebase; building the create page. Importing libraries isn't as easy as we thought!

Accomplishments that we're proud of

Anny Adventure became a fully-fledged idea! Coding for around 30 hours and pulling through feels great.

What's next for Anny Adventure

Input sanitation to the database; allowing users to edit any submissions they have made; adding some fancier CSS stylesheets.
