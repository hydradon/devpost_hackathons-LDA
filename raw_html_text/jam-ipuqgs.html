Inspiration

Being CUNY students that drive to school, we end up being late to some of our classes because we can end up spending over an hour finding a parking spot. We came up with the idea, what if our fellow CUNY drivers can help us out by notifying us when they are leaving their parking spot.

What it does

Notifies users about soon to be vacant parking spots.

How we built it

Using google maps API, JavaScript, HTML, CSS, PHP, MySQL

Challenges we ran into

integrating the Google Maps API

Accomplishments that we're proud of

working as a team to create an application that helps solve a problem

What we learned

how to debug

What's next for JAM

integrating IBM Watson to predict soon to be vacant parking spots.
