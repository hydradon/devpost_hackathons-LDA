Inspiration: Jaylen- "I feel personally connect to this project simply because I never understood credit cards and this project made it all a lot clearer for me to understand and I had impact on how i feel about coding. I feel that it is a good learning experience on coding and credit cards."

What it does: Our program recommends credit cards based on the user's preferences given in a survey.

How we built it: It was built using Java from the the ground up using various coding techniques.

Challenges we ran into: Introducing ourselves to new techniques by learning and implementing it into our code. Losing a teammate during development.

Accomplishments that we're proud of: Sharping our skills and developing new skills while implementing it.

What we learned: How to handle stress, asking for assistance, and managing coding between multiple users.

What's next for Recommended Credit Card: Database, more diverse set of credit cards, graphic user interface, etc.
