Inspiration

Wanted to make good use of the Open NYC real time database and since we go to school in New York, we wanted to know how secure every zip code is.

What it does

It tells the user information about the zip code they input, based on fire departments stations and motor vehicle accidents.

How we built it

We used python, flask, NYC Open Data, twilio, and data science.

Challenges we ran into

Parsing and traversing data.

Accomplishments that we're proud of

Figuring out an idea.

What we learned

Python list comprehension.

What's next for HackNY-F17

More datasets, more cities, more information about each zip. A legitimate database instead of a list.
