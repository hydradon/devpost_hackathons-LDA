• Color Wheel

Inspiration

Wanted to work with Arduino and Arduino Components

What it does

Categorizes pre-loaded images according to their prominent color values. Also takes in live color data from either the computer webcam or the Arduino color sensor. In the case of the webcam, an image is sampled before our algorithm determines the prominent color. We wrote code to match the live color data with our pre-loaded images and display them all to identify the best matching artwork to go with the surroundings.
