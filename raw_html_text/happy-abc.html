Inspiration

I was inspired by a toy from my childhood.

What it does

With this skill you have two different options:

1) Learn the alphabet letter by letter including examples with corresponding audio samples.

2) start counting with Alexa. Alexa will ask you a bunch of questions and you have to give her the right answer.

How I built it

The Skill was built with PHP, JavaScript and JQuery on a self hosted server.

Challenges I ran into

Making the skill simple and easy to use for kids.

Accomplishments that I'm proud of

The use of Amazons SSML Markup Language.

What I learned

More about Amazons SSML Markup Language.

What's next for Happy ABC

Currently there are only 12 questions available. I am planning to include a lot more questions by involving the community.
