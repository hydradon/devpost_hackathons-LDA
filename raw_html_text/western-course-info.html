Inspiration

Trying to select a course to enrol into takes longer than actually trying to figure out what courses you need for the year. It’s not a simple step of just searching the course and course code into a simple data base and clicking on enrol. By the time you end up finding your course and clicking enrol the class might already be full! Our program will send text messages with stdlib's messagebird API to notify you the course availability of your interest! It also displays the course's prerequisite requirement so you are aware of it before getting kicked out of the class that you barely enrolled into.

What it does

After receiving your list of courses, it will gather information about the courses and send the results to your phone as SMS message.

How we built it

Java, eclipse, stdlib's messagebird

Challenges we ran into

making http request in java language to send SMS message;

Accomplishments that we're proud of

establishing a novel system to help student life at Western University

What we learned

how to collaborate and work in a stressful environment.

What's next for Western Course Info

implementing sms message every hour or so, in order to make checking capacity and enrolling into a course easier!
