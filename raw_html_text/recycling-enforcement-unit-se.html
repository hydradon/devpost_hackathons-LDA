Inspiration

We're tired of seeing easily recyclable items in the general waste bin, so we've created a novel solution to make people more aware of their garbage.

What it does

When an item is thrown in the general bin, it is only accepted if it is indeed non-recyclable!

How we built it

We used Kotlin to write the camera app. Also we used Firebase ML kit to classify the images using Google Cloud Vision APIs, and Arduino to program the servo motor of the bin.

Challenges we ran into

Using the cryptic Android Camera2 API

Accomplishments that we're proud of

Having built a working prototype despite the challenges that we faced

What we learned

If you ever want to use the camera on Android, don't, honestly

What's next for Recycling Enforcement Unit SE

Mass production in partnership with IKEA
