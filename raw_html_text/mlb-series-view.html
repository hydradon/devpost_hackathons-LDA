Inspiration

Baseball is SeatGeek's number 1 revenue driver, we do nothing in our UI to optimize for it

What it does

It groups baseball events on the performer page into series so that it's easier to scan the list of events

How I built it

Built on top of the existing android app's performer view

Challenges I ran into

A surprising as it might be, I actually needed to learn a new UI framework called "Epoxy" to build the views, and though it's been in the app for a few months, I haven't used it yet.

Accomplishments that I'm proud of

Getting the Epoxy models working

What's next for MLB Series View

Didn't get to the point of UI polish that I wanted for presentation, so finishing that a bit more then getting to the desired filters work would be the next step.
