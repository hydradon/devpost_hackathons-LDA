We were inspired by the popularity of Electronic Music combined with the the ability for people to use technology to connect motion to computers

While playing an audio file, the Leap takes the length of the swipes of a user and adjusts the tempo and pitch of the file based on speed.

We wrote a code in java that used LEAP to read motion.

We ran into the fact that audio is difficult to work with when using Java.

Finishing the project as well as figuring out how to work with sound in Java.

How to combine the real world with the virtual world through technology.

Ways to mix more than one song and use multiple Leaps together to mix songs in multiple ways.
