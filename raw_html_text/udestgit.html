Inspiration

A yearning for travel, and an eagerness for the present.

What it does

Based on a 100-word entry filled out by the user on our website, the program analyzes the user's personality type and mood. It then determines what travel destination would be best to improve or maintain the user's mood, and informs them of the best flight and hotel deal possible at the moment.

How we built it

Front-end of web site was based on a Twitter Bootstrap template, with additional php, javascript structure and database interfacing. Databases in MySQL. IBM's Personality Insights and Alchemy API's, Amadeus' Flight and Hotel API's called in Python.

Challenges we ran into

time constraints, API documentation

Accomplishments that we're proud of

Successful implementation of Amadeus' API after a lot of trouble with calling it in source code.

What we learned

Research independent libraries of even the most obscure API's, better commenting etiquette for efficient collaboration

What's next for Udest

Continuous improvement and branching out of the application, even far in time after Calhacks. A project we are all proud of.
