Inspiration

Navigating campus can be a difficult task. UNC Transit helps alleviates this arduous duty by allowing individuals (commuters, students, or anyone taking advantage of UNC's transit system) to quickly inform themselves about the nearest bus by merely glancing at their Pebble smartwatch.

What it does

Allows the user to select bus and stop and be provided with time until next stop.

How I built it

Using javascript and Pebble's cloud services.

Challenges I ran into

Parsing the data from the transit website.

Accomplishments that I'm proud of

Building a useful utility in a matter of hours.

What I learned

Having never programmed a Pebble application before, I learned about the pebble environment and sharpened by javascript skills.

What's next for UNC Transit

Adding realtime updates and weekend support.
