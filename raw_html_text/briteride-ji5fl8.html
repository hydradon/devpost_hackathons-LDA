Inspiration

EventBrite representative wanted a good carpooling app.

What it does

Allows users to see nearby drivers and connect with them at the driver's own price.

How we built it

Using an Android app, with Venmo transactions built-in.

Challenges we ran into

Figuring out the right transaction for us. Styling ListViews.

Accomplishments that we're proud of

What we learned

How to use Android. How to use Venmo API. How to use new Android Studio features.
