Inspiration - It takes too long to wait for answers on platforms like stack.

What it does - Allows users to quickly connect with each other and talk about certain educational topics.

How I built it - Uses a database to store the accounts and uses Websockets for live chats.

Challenges I ran into - Incorporating the database and live chats.

Accomplishments that I'm proud of building the database and live chat successfully.

What I learned how to make dynamic websites that can communicate with the server.

What's next for Blitz -> Add more topics outside of STEM, and make the website ready for more users.
