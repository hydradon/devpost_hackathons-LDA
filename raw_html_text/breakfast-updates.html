Inspiration

The main Idea is created an interactive dashboard for manager/director to keep track of business performance across area.

What it does

It can display data in different criteria, such as years, and State, and shows tables with combination figures that might be useful for a country manager

How we built it

We build it on MS Excel. If the data is large enough, we will use SQL Sever to process the data.

Challenges we ran into

To find the effective combination of data to make senses to the business

Accomplishments that we're proud of

Finish the dashboard

What we learned

New ways to import large data to SQL server and exchange data from Excel and SQL server

What's next for Breakfast-updates-minihacks
