Inspiration - We wanted to develop a social media app that links together people within the same area, that share the same passions and interests.

What it does - Groupify is an Android app that provides its users with the oportunity to create certain events in their area and allow other persons to join them. Each event has a description which states what the event is about, where it is located, when it takes place and how many people attend it.

How we built it - For building our application we used Android Studio, a Firebase project that we directly linked to our mobile application and the Google Maps API for providing an easier access to the location of an event.

Challenges we ran into - Integrating Firebase Cloud for Authentication and Database management.

Accomplishments that we are proud of - We are proud of having developed an app that is capable of making people interact with each other in a more practical manner, by creating an event room which is accesible to people in the same area. Thus, we believe that our app is the perfect solution for organizing all sorts of events easily, taking advantage of an easy-to-use interface.

What we learned - During the implementation of our idea, we discovered a lot of amazing new technologies, that we managed to integrate in order to improve the functionality of the application. We have learnt amazing new things, such as the developement cycle of a mobile app, or how to integrate a Firebase project into our Android application. But more than anything, we have learnt how to work a as team.

What's next for Groupify - Our future plans with Groupify include to extend and launch our app on the market, so we could really make an impact on people's everyday life and bring an innovation that might change the way people interact with each other in the virtual world.
