Inspiration

We wanted to better pace ourselves when working out with music.

What it does

Syncs the beat of songs with the users' heartbeats to provide a better workout experience.

How we built it

This was built using the Fitbit OS SDK, Firebase, and Spotify.

Challenges we ran into

Integrating Firebase with Fitbit was more complicated than we originally thought. As well, we were occasionally held back by seemingly untraceable changes in code that resulted in large bugs.

Accomplishments that we're proud of

Managing to overcome our challenges and not giving up as well as learning to work with new technologies.

What we learned

Perserverence.

What's next for FitBeat

A personalized playlist with no wifi for your mind and heart.
