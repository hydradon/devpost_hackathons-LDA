Inspiration

Education in Nigeria is in a bad state. Public school students still struggle with reading and writing even at primary 6 level. Private schools are expensive and don't always delivery quality teaching.

And the number of people speaking out native languages is dropping while foreigners are becoming interested in learning them.

What it does

Mini Games for learning languages and maths. The platform awards points to users and increases in difficulty as they progress. We are also integrating it with facebook messenger so users can play the games inside messenger.

How I built it

It's a MEAN stack app. I make the games with Phaser and zimjs libraries. The server to is built with node.js and database is MongoDB

I worked mostly on the front end, Thankfully zimjs has a very good documentation with lots of examples which helped me speed up prototyping and test the ideas. I also have two awesome teamates building the backend.

Challenges I ran into

Electricity goes out a lot on my country which can slow things don't wn and I have had to balance my schedule doing client work while trying to launch this platform

Accomplishments that I'm proud of

I'm happy that I was able to learn and implement complex design patterns in my code and tackle some very stubborn bugs.

My skills have definitely the improved while working on this

What I learned

The HTML canvas opens up a whole new arena for creating immersive environment for learning.

What's next for Knowing

We will put the demo in the hands of users and get feedback on the work so fat
