The Bioinformatics Toolset

The goal of this project is to develop a lightweight web application that can quickly perform bioinformatics-related tasks. This will allow those in bioinformatics-related fields to make quick calculations and obtain results with ease.

The ultimate goal of this project is to make this web application a great place to come to in order to get bioinformatics calculations done quickly, and will eventually include many different functions. For example, at launch the project will include the ability to translate DNA into RNA, count the number of nucleotides in a DNA sequence, find the reverse-complement of a DNA sequence, and determine how many of each amino acid are in a RNA sequence.
