Inspiration

There has to be more ways we can interact with data.

What it does

Helps explore data using gesture to navigate.

How I built it

Kinect senses gesture and passes gesture as input to C# application.

Challenges I ran into

Making it work with Unity VR environment to fully explore the data in virtual reality.

Accomplishments that I'm proud of

Making kinect 1 work as gesture input.

What I learned

Kinect 1 is not that great for gesture.

What's next for GestFHIR

Make the VR environment + upgrade to better gesture sensor.
