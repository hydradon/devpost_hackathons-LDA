Anti-Harassment GroupMe Bot

Created by Cole Bowers, Rishab Kaup, Alvin O'Garro, and Akash Patel

We built a a Bot for Group me that is deployed through Heroku. The bot monitors language on the chat and notifies users of offensive behavior by scanning messages with regex. It also implements sentiment analysis on the messages if the group with Microsoft Azure and plots this information with the help of Plotly.
