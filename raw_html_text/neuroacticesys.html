Inspiration

We vision a game system that go beyond mere an interactive gaming system. We want to build a neuroactive sensitive gaming system that can catch the gaming experience of the users at realtime. Also, we would like to use different sensors, controllers and display to make the game more immersive.

What it does

The NeuroActiveSys immersive gaming system creates an unparalleled gaming experience, where users' emotions and level of excitements are captured and gathered. Based on the neural activity's information collected from Muse headset, and data analysis facilitated by Nueromore, NeuroActiveSys automatically change the difficulty to adapt users actual emotional status when they are playing the game.

How I built it

We used Oculus Rift, Myo to enhance the gaming experience. We used Muse to capture the concentration and relaxation status and used neuromore software to analyze the data. And we uilitized Unity3D as our game engine. For back-end, we set up a Java server to communicate with the game.
