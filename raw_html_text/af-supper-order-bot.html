Inspiration

The supper order bot was born from a panic attack due to our lack of ideas. *Also because we order supper and find it troublesome to collate orders in a chat group.

What it does

It's a bot that allows any Telegram user to add items to a supper order, collate the total order. There's a reset function to restart the order. It can be found on Telegram (Supper Order Bot)

How we built it

It was built using python and telepot (python framework for Telegram Bot API) link

Challenges we ran into

Our team only worked with Java prior to this project, also being totally new to developing, we had a hard time learning a new language and framework. Frameworks and the tutorials attached are usually not beginner friendly. NUSIS

Accomplishments that we're proud of

We managed to self-learn and apply a new language, and created a working product.

What we learned

We found da wae.
