Inspiration

Oliver had the idea to make a cool game using his Kinect.

What it does

The game is a side-scroller with targets moving across the screen. The goal is to shoot as many targets as possible. The game ends when you miss a target and it goes off screen.

You can play the game with your mouse (like in the demo video), with body gestures, or both.

How we built it

We built the game in Unity using the plugin for Microsoft Kinect for Windows SDK 2.0. We used machine learning to "teach" the game to recognize a shooting gesture.
