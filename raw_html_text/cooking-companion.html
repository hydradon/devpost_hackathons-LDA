Inspiration

The want, the need, and the curiosity of doing this project

What it does

Uses a spy-cam to capture pictures of a receipt. Then using Matlab functions UPC(Universal Product Codes) are read off of the picture. After the UPC codes are cross referenced with a produce database a search is conducted to find recipes with the ingredients. This web page is then displayed to the user.

How We built it

We split into two groups at the start of the hack-athon. One group worked on the Matlab code while the other worked on the Dragonboard 410c interface with the Adafruit Camera.

Challenges I ran into

In order to get the dragon board to connect to the camera was a challenge as well as accessing the advanced Matlab functions.

Accomplishments that we're proud of

Getting a working receipt scanner.

What we learned

We learned that we can have fun and work as a team on in depth software projects as well as how to balance between work and play.

What's next for Cooking Companion

Perhaps stronger building material and further integration of the full computing power of the dragonboard.
