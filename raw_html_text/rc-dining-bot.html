Inspiration

There was once a telegram bot called RCMealbot, which was the solution to the problem many Residential College residents faced.

Not knowing what food there was going to be at the dining hall next.

Over the recent semesters, this great companion slowly broke down, and at this current point of time, became completely unfunctional. Hence, we decided to fix that.

What it does

  1. Check breakfast and dinner menus
  2. Customize menu displayed (hide/show chosen food categories)
  3. Inform you if the menu contains your favorite food!

How we built it

Scrape the pdfs, dump the menus, deliver to users via telegram commands

Challenges we ran into

Bugs bugs bugs. First time working with a telegram bot and a scraping tool meant a lot of time was put into figuring out how things worked. Not knowing the limitations of MongoDB disrupted a majority of our plans

Accomplishments that we're proud of

We have a working MVP! With a couple of extra features too.

What we learned

Debugger > console.log() Look into the tools we intend to use more in-depth. Probably also plan some more specific details to ensure the main ideas work out.

What's next for RC Dining Bot

Notifications Deployment so everyone can use it again Adding more features, especially the long lost login feature
