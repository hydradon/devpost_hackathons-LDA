Inspiration

We wanted to give back to the community and we thought that an app like this already existed, once we found out there were no apps like this, we decided to make one!

What it does

Keeps tracks of peoples' pills and reminds them when the time is getting close to take their pills.

How we built it

Github for version control, MongoDB and Stitch for data , Ionic for cross platform, HTML for UI, JS for functions, CSS for styling.

Challenges we ran into

Our skill sets did no overlap very much, so it was initially difficult to find a framework that we could all utilize effectively.

Accomplishments that we're proud of

We were able to effectively utilize stitch and ionic, two technologies that none of us were familiar with, and that are cutting edge.

What we learned

MongoDB, Stitch, JS, HTML, CSS, Python, Angular, JSON, JQuery,

What's next for Lifeline

Possibly getting doctors on board using it, refill reminders, integrating with pharmacies to get more accurate data, and possibly even live help.
