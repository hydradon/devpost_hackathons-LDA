Inspiration

Bandwagoning on major forum sites has always deteriorated the quality and variety in discussion, and prevented real debate and original ideas, so we set out to create a site where one is forced to interact with opposing opinions and construct meaningful arguments rather than feed into echo chambers.

What it does

Our application uses machine learning and probability to predict what posts the user will likely disagree with, prioritizing these posts to promote argument and discussion.

How we built it

We used CockroachDB for the database and IBM Alchemy Language to create an agreement metric over posts.

Challenges we ran into

Using a new database that we weren't familiar with and developing an efficient algorithm that compares user post history to synthesize potential reactions to new posts.

Accomplishments that we're proud of

Overcoming our challenges, creating an intuitive user interface that challenges the standard linear structure of most forum sites with a more dynamic layout.

What we learned

alot.

What's next for backtack

Expanding how we estimate post meaning and improving/streamlining our prediction algorithm. Improvements to user interface with features such as citations and references to other posts.
