• hackny, meet streamny

Inspiration

Streamed computing is the future of everything. imho.

What it does

Streamny is an implementation of open source code that provides live streaming as a service.

How we built it

We stitched this one together with javascript basically.

Challenges we ran into

Staying up long enough to write all this code, and venturing into rarely documented territory.

Accomplishments that we're proud of

HLS(Http(s) Live Streaming) is fairly new and a pretty complex topic. I think we learned a lot about it during this hackathon.

What we learned

Coding while sleep deprived is not good for the body/mind/soul.

What's next for streamny

1) Allowing registered users to obtain their own stream keys and pages to stream on. 2) Creating more commands for chat/stream interaction. 3) Pushing HLS to the limit and hopefully knocking down the delay time in the stream.
