We wanted to create an app that would allow students whom are hard of hearing to be able to get the full experience of a lecture or meeting without feeling any bit special. We want this application to also be of use to everyone as it record, stores and converts speech to text which anyone can take to either live caption the lecture in real time while at their smartphone or laptop. Others can take and record the lecture for later use. We built this using java in android studio to create the android app that records and coverts the speech to text using Google's API. We then send that data to fire, a Google Could service, Fire base stores the converted data which we then can access from any phone, laptop or tablet whether their on IOS, Android or Windows. We ran into several challenges such as not enough time to implement a proper website with more functionality. The application also does not look very good because we did not have enough time to make a good looking. We are proud getting android studio and fire base to interact as it should and that other platforms can access the data stored in the servers. We learned a lot about fire base and google api's. We hope to expand on scroll and create a proper more fuller application.
