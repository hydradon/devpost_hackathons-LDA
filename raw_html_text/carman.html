Inspiration

Efficiency in selling cars.

What it does

It gets information about a car by taking a picture of it - such as its make and model, and how much it can sell for.

How we built it

We built it with React Native, NodeJS, and SQL

Challenges we ran into

Making a request to the server, getting the data from the API when there were no good React Native modules, setting up IDE

Accomplishments that we're proud of

We successfully made a mobile app that works with Google Vision. It can take pictures, reset itself, and output data from an API

What we learned

Implementing an API, tons about Google Cloud, SQL, creating a back-end server

What's next for CarMan

A better UI and pitching to Autotrader for a partnership.
