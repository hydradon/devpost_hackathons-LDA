Inspiration

Creating a website for middle school and high school students to have fun and also to learn more about the basics of programming and Software Engineering.

What it does

It uses Pokémon API and the player plays quizzes relates to programming and Software Engineering to evolve and upgrade his Pokemon.

How I built it

We used Sublime-Text IDE to code in Javascript and to design and edit in html and CSS to create a website that uses Pokémon API to accomplish our task.

Challenges I ran into

We were all working on different segments of the website and the most challenging part was to integrate them all.

Accomplishments that I'm proud of

Our team had no practical experience in any web development programming languages but we worked together and made our own website within a day. We were very proud of our accomplishments.

What I learned

We learnt a lot about web development programming languages and also ho to divide tasks among ourselves, work in groups in limited amount of time and produce results.

What's next for Pokeducate

The next thing for Pokeducate is more quizzes more Pokémon and actually creating a journey of learning of software and programming concepts through an easier and fun approach that kids are fond of.
