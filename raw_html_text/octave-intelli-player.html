Inspiration

We found the octave api really interesting and wanted to make a simple ground-breaking music player.

What it does

It uses real-time weather and traffic data to predict a playlist of your favourite tracks.

How we built it

We used Angular for the web framework and used node as a backend using third-party helper apis.

Challenges we ran into

We faced some challenges in mapping traffic and weather api's to the genera using the catalog api.

Accomplishments that we're proud of

We finally made a system that make the best use of the available resources. It's not the best but we think, we pitched our ideas into the right direction.

What we learned

We learned how to map data using weather and traffic api's.
