Inspiration

What it does

Breaks down customers purchases in relation to different types of advertising

How we built it

The data was pulled using the SOLR API and then presented in intuitive way through a map and charts

Challenges we ran into

The API presented some problems. Also getting the charts to display data from a certain province was an issue

Accomplishments that we're proud of

We were able to get a good chunk of our idea into a presentable format. We ended up with something that represents the initial vision well while doing this in a small amount of time.

What we learned

We learned how to use the SOLR API and also that Google provides a good way to easily visualize data.
