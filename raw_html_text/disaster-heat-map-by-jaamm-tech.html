What it does

This website allows first responders to see a heat map of disasters in certain areas to better provide assistance to those areas. The heat maps contain information about people who need emergency help. People's help requests are read from social media.

How we built it

We pulled from Twitter's api to read tweets about disasters. We cleaned up the data. We then feed the data into a heatmap that is displayed on our website.

Challenges we ran into

Working with all the APIs was challenging. AWS had restricted our abilities due to our free tier.

Accomplishments that we're proud of

We made this in less than 24 hours.

What we learned

OAuth for API calls. How to mask urls (kinda) Some members learned HTML and JS. Heatmap.js
