Inspiration

Avoiding the pin code theft by revealing the pin code while entering it on the pin pad

What it does

It has a single button and a vibrator, the user keeps the vibrator in one hand and press and holds the button with another hand, the vibrator vibrates at RANDOM time intervals, and the user releases the button when the corresponding digit of the pin is met with the number of vibrations. Since vibration times are random and happen in the hand of the user, any adversary can learn nothing about the pin even by looking directly at user's hand.
