It's a simple device worn on the wrist in order to track movement and turn it into a score. This, along with a chrome extension which scans ones active tab to determine if it is a productive webpage, act to (attempt to) quantify how productive one is. If this value is below a threshold, the device gently sends feedback to the user to remind them to get back to work (we had intended it to be a mild electric shock, hence Electric Boogaloo)...
