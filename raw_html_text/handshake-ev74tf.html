Inspiration

Do you have a problem remembering people's names? We want to solve the issue that every person has when meeting people by providing a tool so you never forget a person's name.

What it does

Handshake remembers each person that you "shake" hands with, allows you to save met individuals, dismiss those you do not want to meet, and alerts you their name and photo next time you meet. That's it.

How I built it

We used Swift and Objective-C built with the watchOS Kit and backed up our information using parse.

Challenges I ran into

Watch OS Kit is very new right now so their is little documentation and features we can access. Detecting Watch to Watch was one of the main challenges we faced.
