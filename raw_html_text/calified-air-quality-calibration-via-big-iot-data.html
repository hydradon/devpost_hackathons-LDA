The innovation project is targeting using BIG-IoT marketplace and Hopu Sensor Calibration technology

The calibration of air monitoring systems is a big challenge, Mainly because the calibration requires specialised equipment and the measurements and metrics vary form device to device.

At Hopu the calibration lab is a Oxido-Reactive based laboratory and the calibration equipments is following standards as AQI.

The process guarantees some level of accuracy but in air quality sensors. Particularly for outdoor sensors the comparison with reference calibrated sensors measurements is required.

If the proper calibration measurements can be collected forma marketplace like BIG-IoT. The process of matching calibrated measurements, with Hop calibration system can facilitate the optimization of new sensors deployment and data quality validation based on emerging standards such as IEEE PAR2510.

The problem of using only one calibration lab as reference is that is not feasible to adapt it for deployment in other side of the world. For that reason, the evolution/innovation using multiple sources of calibrated/non-calibrated sensors

BIG-IoT marketplace includes offering from calibrated metrics from distributed locations, and for that reason BIG-IoT marketplace is used as source of data and information(meta-data) from calibrated sensors
