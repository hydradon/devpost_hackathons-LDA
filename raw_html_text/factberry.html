Inspiration

I wanted to give people a cool skill to turn to every morning, so I came up with Factberry.

What it does

Factberry tells you a random fact every time you ask it.

How I built it

I used Node.js, AWS Lambda, and Amazon's new APL.

Challenges I ran into

Since APL is fairly new, it was difficult getting information on how to do certain things.

Accomplishments that I'm proud of

Creating a dynamic APL document that can show a random fact.

What I learned

I learned many useful skills including how APL works and how to pass data to a JSON file.

What's next for Factberry

Make the UI better and add additional facts. Possibly add a picture for each fact.
