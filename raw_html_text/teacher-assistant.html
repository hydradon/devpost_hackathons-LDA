Inspiration

Lack of a dynamic and helpful assistance/education in 3rd world countries.

What it does

Teachers have access to analytical and cumulative tools they can use which helps them tailor the homework to each student to reach their full potential and exercises we provide.

How we built it

Ruby on Rails as the back-end and the MaterializeCSS framework for the front-end.

Challenges we ran into

The internet provided and cross platform programming between Linux and Mac.

Accomplishments that we're proud of

The database we implemented and the overall idea.

What we learned

We learned the availability of online open free courses and that there is a way to customize homework depending on the student.

What's next for Teacher Assistant

Machine learning and a huge dynamic database which takes in more parameters from the students.
