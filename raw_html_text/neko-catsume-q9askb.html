Inspiration

We just got a pet cat and really like cats. We also really enjoyed Neko Atsume, the mobile cat collecting game, and realized that Neko Atsume would be a great port to chrome extension. We were surprised that an app like this didn't exist, so we decided to bless cat lovers everywhere with Tabby Katz.

What it does

Tabby Katz livens browsing by adding a chance of a interactive collectible cats appearing with each page hit. Cats can be collected and viewed within the home page. Along with that, cats maintain a love meter that increases as you encounter them more often which can be viewed by clicking the top-right cat icon. There are over 30 cats to find!

How we built it

We primarily used Javascript for backend, with google storage to maintain a local database of cat encounters. Our front-end is a mix between JS, HTML, and CSS.

Challenges we ran into

Our biggest challenge was that we had minimal experience creating a web app and none of us had created a chrome extension before. It was a great learning experience!

Accomplishments that we're proud of

We made a cat collecting game to feed our unhealthy obsession towards cat. Also learning how to make a chrome extension, along with using javascript, in a weekend.

What we learned

We learned how to create a basic web app and got a taste of "full-stack" development (not really though i guess).
