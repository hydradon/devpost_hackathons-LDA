• Smart-Mirror

Inspiration

We watched an awesome Youtube video of an MIT student's own implementation of a smart mirror and it looked awesome. It was beautifully made and included voice commands and useful widgets. This inspired us to make our own lightweight version with other features we thought would be useful.

What it does

The mirror has many functions, such as show time, weather, Google Calendar, and some extras that are controlled through voice recognition.

How we built it

We used JavaScript and HTML5 to run a webpage. Then we placed a two sided mirror over an old monitor and ran the webpage.

Challenges we ran into

At first, we wanted to run it on a RaspberryPi but there were lots of add-ons that we had to download and would have taken a long time.

Accomplishments that we're proud of

The great user interactions.

What we learned

Since we had some limited knowledge of HTML, we basically all learned some HTML and a lot of JavaScript from this project.

What's next for Smart-Mirror

We are planning on making it larger and more "mirror-like" and touch screen in the future.
