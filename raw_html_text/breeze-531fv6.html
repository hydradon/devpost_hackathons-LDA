• Eazy Breezy

Inspiration

Our inspiration was DocuSign hosting their competition. We have both used DocuSign in the past to sign our lease, which turned signing a contract into a trivial 1 minute activity. However, we saw much more potential in DocuSign's technology and decided to see what we could do with it

What it does

The app helps users to find temporary workers for whatever they need, whether it be tutoring in computer science or a one time plumber. The user posts a listing to his/her profile and waits for a potential helper to contact them. The app integrates the DocuSign API's to help users feel confident in the potential helper in terms of trust and payment. A legally binding contract is more ideal for the average person to make them feel safe when meeting and paying a stranger.

How we built it

We built the backend server and database in python using flask and mongodb, while the frontend app was built in react native

Challenges we ran into

The entire process was a challenge because both of us worked on things we had never worked on before. Daniil has never touched react while Vladimir has never touched flask/mongo. However we made it work in the end.

Accomplishments that we're proud of

We are proud of both getting this done, embedding the frontend to backend and having the API's work properly in response to both

What we learned

We learned how to manage a database on a server, how to make a backend server for an app, how to integrate API into the code and how to write an app in react

What's next for Breeze

Extend the functionality of the app with possibly reviews, involve machine learning search algorithm and integrate the DocuSign signatures in the app itself, not through email
