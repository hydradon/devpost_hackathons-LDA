Inspiration

We are inspired by the goals of the Aloha+ Challenge and are striving to create a platform that can show the community's progress towards these goals.

What it does

This app will allow users to showcase events or projects that contribute towards the goals of the Aloha+ Challenge.

How we built it

We used Meteor, React, MongoDB, and Semantic UI.

Challenges we ran into

We were still learning the basics of managing databases, so it was a little difficult to set up the databases for storing events.

Accomplishments that we're proud of

We are proud of the baseline web app we have created, with the ability to add and create events in the app itself.

What we learned

We have learned a lot in terms of teamwork and project management. In terms of technical work, we have learned how to better develop user interfaces and databases.

What's next for ETS/HGG Community Engagement (600-iq team)

Up next, we would like to give better functionality and customization for users (profiles, ability to add comments).
