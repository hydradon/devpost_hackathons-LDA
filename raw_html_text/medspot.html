Inspiration

Treating epilepsy requires extensive data acquisition and analysis. However, the current processes in healthcare sector require tedious and monotonous work from multiple clinicians. For example, to get the diagnosis right, it is required to analyze data from EEG and find regions of the specific interest.

What it does

By aggregating data from multiple data sources and incorporating advanced digital signal processing algorithms MedSpot creates a rich vizualization of patient's brain activity. Measurement data representation could be adjusted to doctor's needs, which allows clinicians to make better decisions, increasing success rate of the surgery from current, which is 30-70%.

How we built it

D3.js and three,js on frontend, Python on the backend.

Challenges we ran into

Exloring the D3.js library.

Accomplishments that we're proud of

Working 3d model of the brain in the web browser.
