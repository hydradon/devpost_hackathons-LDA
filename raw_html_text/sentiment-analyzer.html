Inspiration

I wanted to see what Google Cloud APIs natural language processing could do.

What it does

Uses Google Cloud API to measure the sentiment of highlighted text in order to see if it is positive or negative.

How I built it

I used a Google Chrome extension in order to facilitate the API.

Challenges I ran into

This was my first time using a google chrome extension as well as AngularJS.

Accomplishments that I'm proud of

I'm proud of how nice the user interface works and how reactive it is to the user.

What I learned

I learned that once you learn how to code a Chrome extension it is very easy to extend and update.

What's next for Sentiment Analyzer

I would like to make it work specifically for Amazon reviews, so you can see at a glance what people are actually saying about a product.
