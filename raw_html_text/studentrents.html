Inspiration

We were inspired by the fact that it is so hard to find student housing that everyone is satisfied with! We faced this when our group of friends was looking for a place to rent.

What it does

It scrapes the web, and scans RSS feeds in order to feed data into the program on rental listings that are for students. All the data is then used to show the listings and their information, as well as give them a secret score that sorts them by the most desirable.

How I built it

Web application running on a node.js server. Using web scraping for data as well as pug, and javascript for styling.

Challenges I ran into

Scraping and parsing data from the web, and amazon web hosting was not working unfortunately. As well as getting everything to display nicely on the webpage.

Accomplishments that I'm proud of

Managing to get through all of our short term goals for the project that we set, even though it is not fully done we managed to do a lot of work and had a good time!

What I learned

Webscraping and RSS feed grabbing with node.js. Design elements. Login server requests. pug.

What's next for StudentRents

Further development and bug fixes! We still need to implement sorting, as well as pulling data from other sources. There were many goals we accomplished but many more still have to do, hopefully we can launch the service one day.

Hackathon was a great time I can't wait until my next one!
