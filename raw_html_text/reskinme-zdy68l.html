Inspiration

Professor Spear

What it does

A simple game with mechanics similar to flappy bird, but with unique collectibles and enemies. The concept is not focused on gameplay, but rather the theme of the game. Eventually we will support asset packs that will change the appearance of the game, creating unique experiences everytime!

How we built it

We built it in Java, using Android Studio, with a gradle build system.

Challenges we ran into

Git caused constant problems and was tough to work with. The library we built on top of was complex and had a lot of tools to learn in order to use it effectively.

Accomplishments that we're proud of

We have built the base of what could eventually be the South Park of mobile apps!

What we learned

How to use git better.

What's next for ReSkinMe

Generalize our implementation to allow for flexible asset packs to re-theme our game!
