• Our bot in action

Inspiration

A desire for better discord moderation.

What it does

It is a bot that uses sentiment analysis to warn users who are being offensive or negative, and issues consequences based on the desires of the server owner. In our example we banned the offensive user.

How I built it

We used the Discord API and AWS Comprehend for sentiment analysis.

Challenges I ran into

We ran into issues using AWS and authenticating our credentials. We also had difficulties using the Discord API because there were four different JavaScript APIs that are not cross-compatible.

Accomplishments that I'm proud of

It works and is a complete product.

What I learned

We learned about integrating and connecting APIs

What's next for SentiBot

It will be brought to various Discord servers including our EECS Server.
