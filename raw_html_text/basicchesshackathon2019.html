Siena Hackathon 2019

Basic Chess

Hieu Le, Zachary Goodsell, and Carmelo Emmanuele

Idea's for this project came from the fact that chess is a wildly know game across many cultures. Because of this we believe this Diversity project connects individuals or groups of various ethnicity and cultures because they all can find a common game to bond around.
