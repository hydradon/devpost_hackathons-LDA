Inspiration

We wanted to build a app to make the community a safer place.

What it does

The app fetches the most recent public crime data and displays it onto the Esri API. When the user approaches near an area that is determined to be dangerous, the app will alert him/her to be cautious. If the user screams out "HELP!" at any time, the app will automatically turn on the microphone/camera and record the user's actions. It stores them on a cloud server, and the user's closest friends and family will be able to see the recordings as well as receive the user's current geolocation data.

How we built it

We built it with Esri Map API, Amazon Web Services S3 for storing in the cloud. We also used the Native Speech Recognition API to parse language.
