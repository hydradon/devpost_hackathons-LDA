Inspiration

The main idea behind this project was the problem of code sharing between students.So, we decided to develop a project over LAN which allow student to share code and edit it in real-time with permission of author.

What it does

This project not only allow users to share code but also their files through LAN.The user can store its files as well as code.

How we built it

We built its back-end using python flask framework and front-end using html, css and bootstrap.

Challenges we ran into

The main challenges was establishing connection between users and allow only authorized person to edit the files. The other challenge was to prevent user from using fake emails.

Accomplishments that we're proud of

Thanks to ALLAH, we are proud on building this project which an solve the problems of students as well as in some organizations as well.

What we learned

We learn the concepts of data transferring. How a website works?. How to use python for web-development? We also learn html, css and bootstrap as well.

What's next for Synco

Hopefully in future, we shall extend this project by adding more features like highlighting the syntax of code, making website more beautiful, forums and Q/A section where students can ask questions and get help from other students.
