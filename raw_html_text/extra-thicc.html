FutureMap's Purpose

FutureMap is a tool for the visualization of geographically-based data through the conversion of CSV data sets into an easy-to-digest format. Rather than require users to learn a new, complicated visual library to examine and process data, FutureMap allows for simple viewing and analysis.

In addition, the format allows for the prediction of future data, and enables the same comparisons and analysis that was possible for known data.

With our example case, we have used the framework to visualize crime data across the United States. Through comparing the data values of crime between 2004 and 2014, users can clearly see changes in crime rate for various types of crime in each state.
