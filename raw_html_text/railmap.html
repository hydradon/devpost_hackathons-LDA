Inspiration

We provide a hardware sysem to allow navigation inside trains. This allows for a lot of new and personalized services like:

  • Navigate and find your friends
  • Find the right coaches with the most available space
  • Deliver services to your seat
  • Gather data about train usage

What it does

It provides the train with a beacon indoor navigation system

How we built it

Using Android SDK and a backend written in nodejs running on IBM bluemix

Challenges we ran into

It is hard to isolate the correct beacons and to synchronize over the internet

Accomplishments that we're proud of

We have a working prototype ready
