Simplicity, want to help people using flip phones. with no access to internet, GPS Big Data seems like a word far away from another planet. You can get information on requested service through test message. We used Python, Twilio, Yahoo weather, doctor API to do the cooking in the background. Integrating all these individual pieces together, getting our hands dirty in unfamiliar API territory. We are happy with what we achieved in an overnight. We would be happy if people who don't have internet access can use our service to make use of the big data benefits. Learned about Twilio, Yelp, Yahoo Weather APIs, ngrok and flask. We plan to add additional options such such as being to assist with GPS service by using a utility nearby and getting it's geo coordinates.
