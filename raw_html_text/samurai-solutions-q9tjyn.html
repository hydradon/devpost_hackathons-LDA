Inspiration

An easy way to manage all forms and be able to customize them all while being able to serve them to your clients in an easy way.

What it does

It allows office admins to create and manage all patient forms in one central location all while being connected to a client facing user interface that is on a Ipad that replaces the old paper form and pad. The Ipad has the ability to capture and save the form data and signatures are captured by using an electronic pen.

How I built it

Drupal 8 as the cms to store and manage forms with react and node js app on the front end.
