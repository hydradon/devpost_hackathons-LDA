What it does

Enables BIM Users to use QR codes in their project files with just a click

Inspiration

BIM user community requests ( https://forums.autodesk.com/t5/bim-360-ideas/bim-360-docs-qr-code-module/idi-p/7421200 ) ( https://forums.autodesk.com/t5/bim-360-ideas/bim-360-docs-stamping-a-pdf-document-with-a-qr-code-to-controle/idi-p/6844430 )

How I built it

QR Image and dock panel for Viewer screens. Resize QR code to desired dimensions using Resize QR panel. QR code can be embedded into plans as well as 3D documents.

Salient Features

Just scan the QR code and get version details. Click on the panel header with document name as title to retrieve scanned user details

Impact

It provides people in the field with latest design document details and promotes harmony with in the building community. :)
