Inspiration

This app is inspired by an actual card push up game that I do with my friends. Pick a random card and do as many push ups as the number on the card.

What it does

The app helps motivate you to work out with a friend and try to score as many points as possible.

How we built it

We used Android Studio to create java and xml files.

Challenges we ran into

We ran into many many many... errors along the way. The hardest errors we had to fix were those involving Intents. We also had problems with the UI, where the graphics would be shifted into weird positions on the screen.

Accomplishments that we're proud of

We were able to create our first app! We're very proud that we added multiple functions, such as buttons and timers.

What we learned

We learned more about app design-that sometimes less IS more-and the significance of proper naming etiquette.

What's next for CardOut

We'll take CardOut further by formatting the app closer to the API and focusing more on UI. Graphics definitely could be updated and we could add Firebase to allow rankings and leaderboards.
