Inspiration

Recently, Jacob was diagnosed with Type 1 diabetes. He's hacking his way through life.

What it does

If you want to know my current blood sugar level, text me.

How I built it

Use Dexcom Share API to get the data. Run Python code to extract info from JSON and generate messages for SMS.
Serve Twilio SMS delivery system with Flask via ngrok.

Accomplishments that I'm proud of

What I learned

Python, Twilio API, Dexcom API, Flask, ngrok

What's next for textbloodsugar

More functions: warning SMS for relatives and friends
