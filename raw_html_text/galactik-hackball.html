Inspiration

A childhood animated TV series based on space themed storyline involving superhuman football players.

What it does

Football team manager simulator.

How we built it

We've split the work into design, front-end, middleware, backend. One of the team members thought of the game mechanics, one came up with the algorithm for the scoring of the matches, another built the databases and then we integrated all our separate components into a final project.

Challenges we ran into

Linking Python to our MySQL database was rather though and learning javascript for the first time was challenging.

Accomplishments that we're proud of

Making fully randomized teams with players that have completely random stats which impact the game through an algorithm based on character's strength and winning probabilities.

What we learned

How to work with complex databases and how to tackle a game, by thinking about lots of game mechanics.

What's next for Galactik Hackball

Implementing a visual representation of each player's profile and expanding the storyline.
