Inspiration

The alarm is inspired by NFC support in our phones.

What it does

The alarm rings automatically on startup and stops when you put the NFC tag down. It starts again when you remove the tag.

How we built it

We built it using NFC and a Raspberry Pi 2.

Challenges we ran into

We didn't run into many challenges.

Accomplishments that we're proud of

The device works and has some use.

What we learned

We learned about integrating NFC support.

What's next for Somewhat Smart Alarm

We're not sure, let us know if you want to fund a feature and we'll try to make it happen.
