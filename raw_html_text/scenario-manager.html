Inspiration

When attempting to compare multiple options in Excel, data usually is copied and altered. This eventually leads to large and hard to navigate documents where linked data and formulas reference the wrong cells.

What it does

Scenario Manager is an Excel add-in that allows the user to save and track multiple scenarios or sets of data. Users can capture and store a set of data cells and inject the data into the worksheet, at the same position the cells were captured at, when needed to compare options. The applications are endless! Scenario Manager helps users track data sets and compare data faster and easier than ever before.

How we built it

It is built as a Napa Office Add-in using AngularJS as the primary framework.
