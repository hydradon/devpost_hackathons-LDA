Inspiration

Alexa Echo devices provides a new way to develop nice voice applications. The Kids are loved to played with the device. The purpose of this project is to implement an interesting application that motive kids to learn new things, Foster kids’ creative problem-solving skills and also make it fun and keep them interact with the application.

What it does

Two teams will participate in a 5 rounds quiz contest. The questions in the contest are designed to encourage kids to learn new things . The skill provides weekly fresh, interesting questions on variety topics (geography, history, math, life science, etc.) that inspire kids’ curiosity and teach problem-solving skills. The Skill plays cheering up and encouraging music for both correct and incorrect answer. Motivate kids with funny SpeechCons. Create the tension situation in the game by increasing the questions’ difficulty, the speed of speech is increasing after each round, and having the lucky star question in which a team can earn extra point. Allow teams to try the question that other team could not answer and get reward point for the correct answer. The Skill motivates team work with game for two teams, suitable for classroom environment or family event. Foster collaboration and communication between team members. Provide helpful way to improve parent-child communication. The teacher and the parent can discuss with the kids about the topics of the questions or teach them on these topics.

How I built it

The Skill is implemented using the Alexa Skills Kit SDK for Node.js. The core system is imlemented as AWS Lambda function, and the content (Question files and Music files) are stored on AWS S3 bucket, so that fresh content can be delivered on a weekly basic without changing the code

Accomplishments that I'm proud of

An interesting Alexa educational Skill for Kids that are fun and motivate kids to learn and improve team work skill.
