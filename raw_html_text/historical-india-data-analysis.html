What it does

It takes into account historical data from https://data.gov.in/ saves the data in a CSV format then acesses it from Ipython notebook uses Apache Spark within Bluemix to do analysis in it

How I built it

I used IBM Bluemix then Ipython Apache Spark Instance and then used Object storage to access the data and libraries for Python to do analysis and then convert it into a graph form
