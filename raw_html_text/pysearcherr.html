• Listing on PIP

Inspiration

Getting tired of copy and pasting errors from console to browsers to search for what it means/what exactly is causing the problem

What it does 

pySearchErr filename.py

Run the python file with the pySearchErr command and it will automatically searches for the error online (if the program runs into an error)

How I built it

Python

Challenges I ran into

Accomplishments that I'm proud of

It works! And it is available on pip for everyone to install

What I learned
