This web mapping application isa solution to users who know where they want to go, but not where to park. It will allow the user to select a final destination, and then create drive time areas which query parking lots within a certain area. Once a parking lot is selected, the user can route to the best parking lot based on the fee and the proximity to the final destination. Once the user reaches the parking lot, they can utilize the routing function to route to the final destination.
