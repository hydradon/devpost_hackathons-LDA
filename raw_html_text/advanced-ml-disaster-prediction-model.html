Inspiration

We wanted to utilize the power of near field communication technologies in this project. To do this, we used NFC card to scan and process data in the card.

How we built it

At first, we attempted to create an Alexa skill and implement the Flutter project with Firebase, however, after many issues we decided to go server-less.
