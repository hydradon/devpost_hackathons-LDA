Inspiration

We wanted to gain experience with the raspberry pi controlling something robotic.

What it does

It accepts messages from a Slack group chat in order to execute routines.

How I built it

A raspberry pi is controlling the two-wheeled car. We used the Slack API to parse through messages and find keywords which will direct the car however is determined by its users.

Challenges I ran into

Parsing through Slack's cryptic data structures surrounding its messages!

Accomplishments that I'm proud of

We parsed through Slack's cryptic data structures

What I learned

A significant more about Python, electrical wiring, the power of raspberry pis

What's next for Text-to-Drive

We would like to hook a camera up to the car and stream it onto a monitor. This could allow for it to be controlled in another room, with a goal is of making it touch a certain object
