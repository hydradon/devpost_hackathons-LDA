ScheduleVisualizer

To run: 1) Clone the github and go into the ScheduleVisualizer directory 2) If you have not yet done so, install everything in requirements.txt 3) Run source venv/bin/activate to run the virtual environment 4) Run python manage.py runserver (mySQL must be installed) 5) Go to the specified developer server to see the site
