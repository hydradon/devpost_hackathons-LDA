Inspiration

People never sort their trash because they are too lazy, this machine sorts it for them.

We wanted to create something that would put an end to overloaded trash bins and greatly simplify the everyday trash sorting process for the general public.

This project is built using python language, sophisticated backend and server automation, Google image recognition API and the hardware component was incorporated using multiple servo motors.

We have incorporated Plot.ly and Dash to plot real time graphs using the data collected by our system.
