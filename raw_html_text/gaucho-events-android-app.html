Inspiration

We are freshman at UCSB and would want an app that contains all types of events that are occurring.

What it does

This android application is developed to connect UCSB students interested in attending all types of UCSB Events.

How I built it

We used Android Studio.

Challenges I ran into

Accomplishments that I'm proud of

What I learned

We learned how to use Github and Android Studio.

What's next for Gaucho Events Android App

We are hoping to keep expanding the app and potentially have UCSB students use this app.
