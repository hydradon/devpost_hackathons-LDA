Inspiration

Growing percentage of cell phone usage among the farming population leading to more of a need for mobile development in the area.

What it does

It allows farmers to communicate with one another on a platform meant to connect them to other farmers that have similar attributes to their farms.

How I built it

The project was build with both the back and front end in different Java projects and the front end being written using Android Studio.
