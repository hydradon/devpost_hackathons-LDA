• A bunch of Thruster widgets

Inspiration

I saw this video, and thought it seemed great. I wanted to implement it, so I decided to do it at SVHacks. https://vimeo.com/232230096

What it does

Thruster is a tool for creating data visualizations. You can graph data, read CSV files, display them on a table, perform transformations on top of the data, generate log files of what's happening, and tons more.

How I built it

I used raw web tech (HTML, CSS, and JS) with the jQuery, and jQuery UI libraries.

Challenges I ran into

It was really hard to find the right framework for widget development, but once I perfected that, I was cranking out new and interesting ones like mad.

Accomplishments that I'm proud of

It works! I can really easily develop a new widget.

What I learned

I learnt how important a framework is for a good project. Without the right framework, I would never have been able to make as many widgets as I was able to.

What's next for thruster

I want to improve the UI dramatically - it needs a faceplant because I was working on the widgets. I also plan to add new widgets and and contributors to this project.
