Inspiration

While playing a game of Pokemon, I fell in love with one of the minigames, "Voltorb Flip," which had a simple and fun idea that drew me in for several hours. I decided to recreate this classic game on Java for the computer.

What it does

This is an replayable arcade-like game of the original minigame "Voltorb Flip," a mix between Minesweeper and Picross. The same art style and layout of the original game are kept, but still includes some new additions such as the info and reset buttons as well as the text box.

How I built it

I coded it with some advice from my friends and the Internet.

Accomplishments that I'm proud of

It works and I was able to harness the complicated, albeit powerful, GridBagLayout.

What I learned

Graphics are hard to work with!

What's next for Voltorb Flip

I will likely look to add a difficulty or level system for Voltorb Flip, as well as a way to track high scores.
