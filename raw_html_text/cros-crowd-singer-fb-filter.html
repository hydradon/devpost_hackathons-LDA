Inspiration

I love the idea of people interacting with the same application from their own devices and though spark ar would be a good platform to do it

What it does

Connect a SparkAR filter to a unity sound generator application

How I built it

I have a sparkar filter ( patch and code ) connected to a firebase application changing the value of the face status in real time. Locally on the machine I have nodejs server that listen changes on the database and send mesages to the unity application via socket.io

Challenges I ran into

Sound Generation in Unity Spark AR under development tools

Accomplishments that I'm proud of

Realtime connection in between a FB filter

What I learned

Spark AR Networking
