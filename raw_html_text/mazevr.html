Inspiration

VR game where you had to do some physical exercises to achieve victory.

What it does

In this game, you run around in a maze and if you find an enemy, you need to jump in real life to attack the foe.

How we built it

We used Google VR (Google Cardboard) together with Unity3d engine

Challenges we ran into

Local networking is very tricky to implement when you can test it on your phone only which doesn't have to debug.

Accomplishments that we're proud of

VR is working nad we had no prior experience.

What we learned

VR and local network multiplayer implementation in Unity

What's next for MazeVR

Single player mode, mode weapons and challenges to get powerups
