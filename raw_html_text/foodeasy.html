Inspiration

There were several queues that we are put into where we don't really need to wait. If there were more efficient systems set up in stores that can intelligently prepare for the customer, consumers would have to wait less, employers would improve efficiency, and be able to reap greater profits.

What it does

Prioritizes people waiting in a queue, to result in the fewest waiting for there order.

How we built it

Android program connected to a amazon-dynamodb

Challenges we ran into

Bugs.

Accomplishments that we're proud of

581% reduction in people waiting in queues.

What we learned

Learned how to use Dynamodb on an Android app, learned how to demo applications, and learn how to pitch.

What's next for FoodEasy

Expanding to different industries, such as queue's for waiting at doctor's offices.
