Inspiration

I have a kid who loves mathematics, so I make a game for him.

What it does

It ask you addition and subtraction operations, to improve your mathematics.

How I built it

I built it in javascript / node.js, using ask-sdk, apl and lambda functions.

Challenges I ran into

I had experience with ask-sdk, but apl has been a challenge, because it is something complete new, and I have to get the feeling. Support so many languages was a challenge too.

Accomplishments that I'm proud of

I'm proud of getting the skill looks good in all the devices, and in so many languages. It was very challenging.

What I learned

I have learned about apl and working with multiple devices, and languages.

What's next for Addition and Subtraction
