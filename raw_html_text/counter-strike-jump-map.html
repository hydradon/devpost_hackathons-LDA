Inspiration

I enjoy the game Counter-Strike Global Offensive, so I decided to make a map using in engine tools in order to see what I could create.

How it works

You enter the console commands which allow the server to run correctly, (they have not been automated in this build) and complete the jump map. It also has elements of a surf map.

Challenges I ran into

The Hammer Editor for the Source Engine is outdated and had a lighting issue that took me multiple hours to troubleshoot, until I had finally fixed the issue.

Accomplishments that I'm proud of

Even being able to make a basic plane for my character to walk on felt extremely satisfying, so even being able to make a map of this scale makes me feel pretty awesome, not gonna lie.

What I learned

Troubleshooting is a pain, but very rewarding if you stick to it. Perseverance is key.

What's next for Counter-Strike Jump Map

Extending the map in order to make it longer, as it is quite short currently. I am also looking into making a script in order to teleport your character to his last location upon hitting the floor, effectively allowing him to save his progress as he goes.
