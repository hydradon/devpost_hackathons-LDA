Inspiration

When we go to trip with our friends, each friend wants to have their customized trip plan based on their own preference. For example: longer schedule, more excursions than others, etc. It would be complicated!

We created PLANNA to share and control each itinerary.

What it does

PLANNA provide shared trip plan with customized view from each individual point of view.

What's next for PLANNA

It should have communication function to brush up travel plan among friends.
