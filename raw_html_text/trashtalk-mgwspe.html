Introduction

This project was built during the Fall 2017, HackISU hackathon; and is thus not complete; but it should be functional enough to demo.

What you need

  • A Raspberry Pi
  • A HC_SRO4 distance Sonar
  • Wires to connect the Raspberry Pi to the sonar
  • A trashcan with a lid

Setup

  1. Install XAMPP
  2. Clone the source code
  3. Configure XAMPP to point to the cloned directory
  4. Move the clones file sonar_script.py to your Raspberry Pi
  5. Make sure your Raspberry Pi is connected to the same network as your computer
  6. Change the variable server_url to your local computer's IP address and
