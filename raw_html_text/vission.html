Inspiration

Linear communication is so last century. Vission aims to bring communication into the future, by implementing a node based collaborative system that fosters the organic growth of ideas.

What it does

Users can create their own topic nodes that are able to be branched off of and expanded upon by others. Any node can be further discussed in a live text chat. Any vission is welcome.

How we built it

Built using React, SigmaJS, and Semantic-UI for the frontend. Backend comprised of Firebase and Node.js.

Challenges we ran into

Our team was new to working with Firebase and connecting Firebase to a React app without a backend framework.

What we learned

We enhanced our knowledge of integration between different frameworks and products.

What's next for vission

Expanding the service to a fully featured product, adding more features to node creation to encourage more participation.
