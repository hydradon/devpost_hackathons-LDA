Inspiration

Lack of book management systems for Android.

What it does

Manages and searches all books through Google Books API. It also lets you enter comments, rate books and get daily suggestions.

How we built it

With Android Studio and 3 layers separation.

Challenges we ran into

Non working SQL and delayed Google API requests.

Accomplishments that we're proud of

Dynamically adding daily suggestions to read.

What we learned

Never do Android again.

What's next for Bookeep

Implement user registration and login.
