Inspiration

Skyscanner API and the inability to easily book flights and hotels using the same website based on landmarks

What it does

The user chooses a landmark, city, or gig location and the website finds the cheapest flights to get them there

How I built it

Using the Skyscanner API, Python, HTML, and Amazon-web-services

Challenges I ran into

Turning the Skyscanner API output into something useful and readable

What I learned

CSS and web hosting take longer than expected
