What it does

You enter whatever symptoms you are feeling and we will match your symptoms to the most probable illness. With that knowledge, we help you find the nearest doctor to get treatment.

How I built it

We used Python, JavaScript, HTML/CSS/JS and a few APIs (Open Data NYC, ipstack, apimedic)

Challenges I ran into

Getting the API to work, not having the APIs connect to the front-end

Accomplishments that I'm proud of

We were able to get the location of the user's device

What I learned

how to use APIs in python
