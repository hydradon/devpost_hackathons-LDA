Inspiration

Over 65% of the population are visual learners, and the improved technology of smartphones and AI can facilitate a whole new way for translating and learning a foreign language

What it does

It lets you quickly and visually translate objects around you. You can also save those translations for learning retention.

How I built it

The app is built for Android in Java and Android SDK. For Object detection in the captured image Salesforce vision API were used.

Accomplishments that I'm proud of

The app works! and I am using it to learn Japanese.

What I learned

Learn about Salesforce Vision API and the platform overall. I can imaging it can be applied to other apps to make them smarter.

What's next for ObjectTranslator

Add variety of objects for the app to detect. And add more translation languages like Chinese, Spanish
