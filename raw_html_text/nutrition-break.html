Inspiration

People nowadays really don't know what nutrients they're consuming when they're following recipes, that's how Nutrition Break was made, a web app that breaks down the nutrients of your recipe.

What it does

Nutrition Break gives users a decomposition of all the nutrients in the recipe they are about to make.

How we built it

We used SQLite for our server and Python for the actual nutrition analysis and breakdown.

Challenges we ran into

1) We weren't whitelisted to use Google's Vision API 2) Our domain (recipenutrition.org) was blocked by uWash's servers

Accomplishments that we're proud of

We got this thing to work

What we learned

Google's Vision API requires the developer to be whitelisted before they can utilize the full potential of the API.

What's next for Nutrition Break

Recipe analysis to give smart suggestions on how to cook healthier
