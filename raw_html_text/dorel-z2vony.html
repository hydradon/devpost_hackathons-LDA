Inspiration

We work each day in our office, with a lot of internal applications that solve different problems. Applications such as Redmine, Jenkins, Zidox or others. There's a lot of friction generated by using so many internal apps and we wanted to do something about it.

What it does

Currently our application is focused on improving the way you manage your spent time in Redmine and Zidox. Instead of having to remember all the issues you navigated to in order to log your time, our app allows you to quickly register the time spent and focus on what really matters - innovating!

How we built it

We wanted to be as easy as possible for our colleagues to install and use it. That's why we choose Chrome as our development platform. Chrome extensions are fast and easy to use. Because we used only web technologies such as JavaScript, HTML and CSS, anyone can contribuite to our project.

Challenges we ran into

Working with the Redmine REST Api hasn't been easy. Also limiting ourselves to implement a small number oo deliverable features was also a big challenge.

Accomplishments that we're proud of

We managed to work individually but towards the same goal - building a great app and a great experience. Team communication was our primary concern but we handled that very well.

What we learned

We learned how to develop extensions for the Chrome platform but more importantly we learned that planning is the key to success.

What's next for Dorel

Our app needs a lot more work and it is far from done. But we think that our future stands in integration with as many internal apps as possible.

Installation

Requirements: you need to have Chrome browser installed.

  1. Download the Chrome Extension from http://zit.ec/dorel
  2. Open up Chrome Browser
  3. Click on the three vertical dots (Settings) -> More tools -> Extensions (or navigate to chrome://extensions)
  4. Enable developer mode
  5. Drag & drop the extension file (dorel.crx) in the window
  6. Wait for it to be installed and then you'll be ready to go.
  7. Don't forget: you can use the extension's option by right clicking on the extension icon.
