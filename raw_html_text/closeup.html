Inspiration

My classroom experience isn't perfect, so I want to improve them with an easy-to-use system.

What it does

ClosUp.io is a web-based service that helps instructors manage their course communities. By simply pointing webcams towards their classes, instructors are able to obtain real-time feedback about class attendance, group activity, and even students’ emotions, upon a single click.

How we built it

Languages:

Python 3, HTML, SQLite Web Platform: Django

External Services:

Microsoft Azure Cognitive Service -Face API -Emotion API Slack API Twilio API

Libraries: Matplotlib, JSON, http.client

Challenges we ran into

Web app debugging is a very time-costly job to do, and it requires a lot of reading.

Accomplishments that we're proud of

We've successfully implemented most functionalities
