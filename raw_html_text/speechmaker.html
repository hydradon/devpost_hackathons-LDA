Inspiration

We wanted to make an applcation to help people do speeches, especially public speaking through the use of music to empower their speeches and make them more colourful.

What it does

It works like a karaoke machine where you need say a bunch of stuff in a certain amount of time.

How we built it

We built the whole thing using python, even the gui!

Challenges we ran into

Python is notriously bad at doing gui and modifying.

Accomplishments that we're proud of

We actually made something haha

What we learned

Don't use python for audio, otherwise it's a niche language

What's next for Speechmaker

Emotion analysis on speech and speech to text functionality. :)
