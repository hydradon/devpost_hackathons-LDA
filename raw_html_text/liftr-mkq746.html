Inspiration

-To learn more about programming and to create a super cool application that connects people in new ways like never before

What it does

-Allows people to create an account on the Liftr server to connect them with workout partners near them, a Tinder-like framework is used to connect users

How we built it

-Plunker was used to develop the front end webpages and Gitlab was used to coordinate with the team for the back end development

Challenges we ran into

-Creating the front end webpage with a proper navigation bar, creating a working carousel with buttons, and sleep deprivation

Accomplishments that we're proud of

-Creating an actually decent looking front end webpage

What we learned

-How to use bootstrap to easily build websites

What's next for Liftr

-Image upload so that users can customize their profile
