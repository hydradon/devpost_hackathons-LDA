Inspiration

Our inspiration was drawn from our team members wanting to create a game about doing something which benefits society.

What it does

It provides a hand-crafted adventure in which you must extinguish all of the fires in order to proceed to the next area.

How we built it

We remained awake for two days in order to model each object, design each level, program all camera and character controls as well as every object interaction, and create every audio file (with the exception of the royalty-free song that plays over the credits).

Challenges we ran into

Originally we intended to use VR, but the Oculus rift headsets seemed to be incompatible with our hardware. In addition to this, none of us had ever used the Unity environment before.

Accomplishments that we're proud of

Learning to code in C#, making some high-quality models, and creating a game entirely from scratch in two days.

What we learned

How to use the Unity platform, C#, and how to texture objects that have been modeled.

What's next for The Legend of Fred: Firefighter Supreme

Refining all of the rough edges, more levels, and more challenges.
