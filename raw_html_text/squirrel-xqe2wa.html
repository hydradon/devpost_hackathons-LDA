Squirrel

Squirrel is a free application to help you painlessly invest and give back, whenever you spend money.

Whenever you use your Capital One account to make purchases, Squirrel will automatically invest, donate, and/or purchase Bitcoins proportional to the amount of the purchase. Squirrel makes investing and giving back simple.

The application is called Squirrel because squirrels are cute animals that eat acorns and because squirreling means stashing something away for use later. We hope you find this project useful!

Note: The project demo is currently NOT hooked up to real data, because the project uses the Capital One Hackathon API, which does send real data.
