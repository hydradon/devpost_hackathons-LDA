Inspiration

Scrolling through 80+ Facebook posts in search of a cheap second-hand textbook and finding nothing.

What it does

It allows users to search for the textbooks they're looking for from the database with the ISBN, as well as put up textbooks for sale.

How I built it

Python, HTML, CSS, Google App Engine, and help from Shruthi Santhanam, Tyler Williams, and Anthony Fiddes.

Challenges I ran into

Sending information from HTML to Python.

Accomplishments that I'm proud of

Going between pages smoothly, figuring out the HTML and Python.

What I learned

How to transfer information from HTML to Python, and more about Google App Engine structures.

What's next for Smriti

More time spent in development and design; adding more features such as the ability to remove a listing from the website
