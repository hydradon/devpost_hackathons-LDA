squire is a smart way to become more efficient at the computer. Run the script, and just use the computer normally for a couple hours. Then the program will generate a couple of macros for you based on patterns in your user input.

This was built in an AutoHotKey script, which records all your actions and saves it to a file which is analyzed in a node.js server, and returns a couple of AutoHotKey scripts back to you which are the suggested macros to use.
