Inspiration

All of us have always loved playing video games. We all remember playing old 2D platformers like Mario when we were younger. We wanted to try and make something similar.

What it does

Play through 3 platforming levels of increasing difficulty collecting gummy bears along the way to unlock the next level.

How I built it

Nick Kenton and Brendan learned Unity to create scripts controlling events and movement. Jake explored plugins in FL studio to make the musical compostion and sound effects. Christine created the sprites in a pixel-art style using Paint Tool Sai.

Challenges I ran into

The tech team had to learn C sharp and Unity interactions with no prior experience. Unity bugs and collaboration features not working. Inspiration for asset design, music, and sprites.

Accomplishments that I'm proud of

Completing a finished project in Unity. Composing multiple background music tracks for different levels. Creating sound effects for various game interactions. Creating a cohesive theme through sprite design and background artwork.

What I learned

Time budgeting and working with deadlines. Working and deligating tasks as a team. Combining individual contributions to create a functional final project. Keeping goals realistic but not too simplistic. Setting hourly goals to keep us on track.

What's next for GhostyBoy's Spooky Adventure

Create more levels to adventure through. Fine-tune player and enemy movement. Create new enemy types/behaviors.
