Inspiration

Ibrahim did a lot of projects with regards to fractals and math, and so because we all wanted to learn how to program 3D environments we decided to go with this project.

What it does

It generates a whole bunch of mathematical objects

How we built it

We originally wrote this in C++ using OpenGL libraries, but due to much of our code not working we transferred at the last minute to a C# equivalent, using one of our teammates' old projects from high school.

Challenges we ran into

Our main issue was getting our 3D environment working, we spent most of the time learning about OpenGL rather than actually programming it due to the amount of issues we got.

Accomplishments that we're proud of

We got a 3D environment working! It took 3D and a whole lot of grit and late nights but it was managed eventually.

What we learned

OpenGL is a vast and dangerous realm to be in, it will require more than 3 days to fully learn it.

What's next for Fractal Terrain Project

Figuring out a better interface is our top priority, and getting our random noise generators down.
