Inspiration

Google Assistant and being able to use commands and access information without the internet.

What it does

Send commands to a number and get information

How we built it

Python, Flask, Twilio and a bunch of APIs that are used to obtain information

Challenges we ran into

Trying to integrate multiple APIs with the web calls in flask and filter valid information for end-users

Accomplishments that we're proud of

Being able to get an Alpha build that can complete basic functions and the application is modular which means it's easily extendable

What we learned

Using Twilio, Flask and python in conjunction with eachother

What's next for SMSTerminal

Improve functionality and more commands, possible integration of AI, ML and Databases
