Inspiration

We accidentally bought a sensor kit for an arduino, and wanted to make it work with a Google Home.

What it does

Monitors the environmental data of your home and presents it through Google home.

How we built it

We divided up the parts of the project into the arduino, the database, and the Google Home integration.

Challenges we ran into

Arduino's don't handle http requests well.

Accomplishments that we're proud of

Learning node.js and successfully developing an app within the span of a hackathon.

What we learned

Trying to make an arduino communicate with a mySQL server is much more painful than one can imagine.

What's next for Arduino Home Monitor

To be taken apart, and put back into its boxes.
