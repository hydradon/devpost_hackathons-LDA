Inspiration

What happens if a voice recognition engine is integrated into a website?

What it does

Talk with it. It will give you back intelligent results.

How I built it

Knockout.js as front-end JavaScript view library with Houndify voice API integrated. Express.js on Node.js as web server. Bootstrap for styling. Deployed on Heroku.

Challenges I ran into
