Inspiration

Desire to learn how to build a progressive web app and firebase.

What it does

Note taking app saved by user to secured cloud storage, provides syncing across devices.

How we built it

Using create-react-app basic starter, react-bootstrap for styling, and firebase realtime storage backend and authentication. Used Quill API for text editor.

Challenges we ran into

Initially tried to use Polymer PWA starter template, but found it overkill and difficult to integrate with Quill, the editor API essential to our app.

Accomplishments that we're proud of

Firebase integration was smooth to learn and implement, gained new insights and would use this product again in future.

What we learned

Progressive Web apps and Firebase are both pretty neat technologies that are fairly simple to learn and integrate.

What's next for Euca Note

Fix bugs, multi-user syncing, End-to-end encryption.
