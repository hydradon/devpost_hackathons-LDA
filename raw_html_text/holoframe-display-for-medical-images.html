In operation theatres, many of the times, surgeons might want to read the images while the surgery is in progress. In circumstances like this, it could be cumbersome for the surgeons to work with desktop medical imaging systems. In this case, a mixed reality system like holo-lens can be greatly helpful. This project is a simple example of how the user can view medical images using holo-lens.

In the project, images navigation is supported using voice commands.
"Increase Frame", "Next Frame" voice commands scroll the images in forward direction "Decrease Frame", Last Frame" voice commands scroll the images in backward direction. "Begin Loop" runs a cine through the stack of images "Stop Loop" stops the cine.

The project also used the "TapToPlace" script to move the display system in the world space.
