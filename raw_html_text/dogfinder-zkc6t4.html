Inspiration

Nothing is a better de-stressor than petting a cute dog. This is a way to connect dog lovers, or students who want to improve their mental health by petting a furry animal, to dog owners, who are willing to have their dogs pet. We were inspired by the challenge of creating a website that connects people and animals to improve health.

What it does

You can sign up as the dog owner, allowing others to track your pooch and show him some love! Can’t or don’t have a dog of your own? You can track all the furry friends around you whenever you’re in need of that puppy love! Maybe you are a dog owner and want to get your dog socialized - sign up as both!

How we built it

We used HTML and CSS to build a website, and AWS to host it.

Challenges we ran into

We are mostly beginners so it was challenging for us to learn HTML and CSS and learn the basic layout of a website.

Accomplishments that we're proud of

Publishing the website on the internet!!

What we learned

We learned how to code in HTML/CSS, how to collaborate on GitHub, how to setup a domain name, how to set up a KOA server, and how to host our site on AWS.

What's next for DogFinder

We want the website to be more dynamic and store user data. We want to incorporate APIs so that dog seekers can receive notifications when there is a dog nearby to pet. There will also be a map showing nearby dogs' locations
