Inspiration

We wanted to build something ridiculous, and we like WattPad.

What it does

It web scrapes the urls of Wattpad by picking random ID's and taking the strings from them and storing it in a file. The strings are then parsed into sentences and words and when the user inputs a word, it grabs a sentence that has the word in it and automatically responds by having a random word in the returned sentence searched for to simulate a random funny conversation.

How we built it

We used Json and Jsoup and observed the Wattpad API

Challenges we ran into

Using JavaFX, runtime for many URLs

Accomplishments that we're proud of

It functions and it does indeed return the proper strings.

What we learned

How to webscrape, java, javaFX(Marginally), graphic design

What's next for Wadventures

Completing the interface and creating a website to use it
