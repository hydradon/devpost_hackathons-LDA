Inspiration

We started this project to begin our learning of basic database systems and how to work with them.

What it does

Allows a student, professor, dean, or other individual to better understand the courses and data gathered about Brown University.

How we built it

Using tutorials we learned how to use JavaScript, HTML, and CSS to build a website to house our visualization. We learned to use JSON files along with D3 and implemented a beautiful graph onto a webpage.

Challenges we ran into

We all came into this competition with little prior knowledge so we spent a good amount of time watching tutorials and learning before we could actually begin our project.

Accomplishments that we're proud of

Our project is very general and can be reused for many different data sets.

What we learned

We all came in with little prior knowledge and so we all learned a great deal about HTML, CSS, JavaScript, D3, and teamwork!
