Inspiration

We were inspired by the projects that we had previously seen at past events and hackathons that we had been to. We were also inspired by minigames from varius games and online websites.

What it does

Our program is an offline web browser game for the new avast search engine. The game consists of multiple smaller mini-games to aid in the enjoyment and addicting nature of the brief.

How I built it

We build this collection of games using java script and HTML 5 along with CSS. We split up the seporate games to each different member and someone made the page that connects them all.

We ran into a lot of challenges throught the competition however the main one was the number of designs that we went through. This took up a lot of our time and so we did not have as much time to spend on our actual project.

Accomplishments that I'm proud of

I am proud of the design and look of the final product. I belive that it looks slick and proffesional.

What I learned
