Inspiration

We really wanted to make AI great again. To do this we decided to use the inspirational wisdom of the presidential candidate Donald j. Trump.

What it does

It allows you to communicate wit either text or an image (using Clarifai's api) and respond back. It uses Facebook's messenger platform to communicate with

How I built it

We built a website to show our project with links and video using HTML5, CSS3, JavaScript, and Jquery while the chatbot/AI was built using node.js and clarifai's image and video recognition api on the Facebook messenger platform.

Challenges I ran into

We had a lot of difficulty with an api called api text razor which ultimately failed and we then switched to clarifai's api platform.

Accomplishments that I'm proud of

This entire project which consisted of a 2,000 + line website, a chatbot with photo/video recognition, and a marketing video in 12 hours is probably one of my greatest projects, not to mention the humorous topic.

What I learned

Coding is hard work and advice from others is recommended.

What's next for TrumpBot

We will work to make the recognition library bigger and with different people maybe.
