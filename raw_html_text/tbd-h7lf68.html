Inspiration

We were inspired to build this webapp from some alarming statistics about youth voter ship. Only 51% of eligible millennial voters cast a vote in recent elections. However, a large amount of millennials use Twitter. The easiest way for one to get in touch with their local representatives who they may not even know about otherwise, is to use Twitter to send a message to them.

What it does

This project asks the user for input of their address, finds their local representative, and then asks for the username and password of the user. It logins to their twitter, and tweets the message the user wanted to write.

How we built it

This project was built with the Phone2Action API, which helped find the representative data. It then used Tweepy to tweet at the representatives, and Flask for the overall webapp.

Challenges we ran into

This project was built without knowing much about working with JSON files, and setting up a Flask app.

Accomplishments that we're proud of

We are proud of creating an app in a short amount of time while learning new things about JSON files and tweepy.
