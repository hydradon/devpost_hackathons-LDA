Inspiration

Apps which can be used to start a love story are plenty. Yet not every lover story can be "happily ever after". A lot of people after breaking up would like to remove all their ex's photos, some want to keep those memories but without his/her faces. We will take care of that for them.

What it does

The app basically:

  • Detect all possible faces on the user's phone and social account
  • Learn the subject's face from user confirmation
  • Remove/Replace all images that contain subject's face with a specified sticker.

How we built it

Phone app:

  • Android
  • OpenCV
  • Our built-in facebook library

Recognition API:

  • Python Flask
  • Tensorflow
  • FaceNet
  • Microsoft Cognitive Face API

Challenges we ran into

Facebook API does not meet our needs.
