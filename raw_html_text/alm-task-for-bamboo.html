Inspiration

I worked with many clients who were managing HP UFT (Unified Function Testing) test execution outside of CI tools.Most of the customers who use HP UFT as automation testing tool rely on windows scheduler to trigger test execution.

What it does

This plugin integrates HP UFT (formarly Quick Test Professional / QTP ) testing tool with Bamboo CI and JIRA. UFT individual scripts or driver script of UFT frameworks can be easily triggered with this plugin.

How I built it

I built it using Java and atlassian sdk

Challenges I ran into

HP UFT results files are tricky and different for each UFT version. This plugin needs to be updated for every new release of HP UFT to support different report format. This plugin has options to select specific UFT version.This list will grow with new versions of UFT as they comes in.

Accomplishments that I'm proud of

As a automation testing consultant I was always looking for solutions to have all testing tools integrated at one place. Now HP UFT can be easily integrated with Bamboo and JIRA

What I learned

It was a learning to unleash the true potential of atlassian products to make life easier for software development teams.

What's next for ALM Task for Bamboo

This plugin needs to be updated to support new releases of HP tools to support new versions
