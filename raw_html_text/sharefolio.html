Inspiration

ShareFolio is inspired by many modern personal finance and investing web applications such as WealthSimple. The main goal was to combine the ease of use and effectiveness of these applications with a collaborative experience.

What it does

ShareFolio provides a platform to build group portfolios through peer input. A user can join multiple investment groups which work together to create a weighted portfolio through voting.

How we built it

We built it during the HackWestern 3 weekend.

Challenges we ran into

The main challenge was implementing a large number of features along with a backend and ui in a very short and intense time frame. Having data update in realtime was also a challenging issue.

Accomplishments that we're proud of

e are proud of ShareFolio's core concept and direction as well as the attractive ui. For one of the team members this is their first hackathon and web application deployment.

What we learned

Learned a lot about graphing with highcharts, how to format them and make an asthetic presentation of data in a realtime web application. Also learned about implementing a backend and frontend system and linking them together so that the frontend can access data without any errors. The use of SQL in a web environment was another major focus of the project that led to greatly improving our skill.

What's next for ShareFolio

Next steps would be to add the ability to link to bank accounts, work out ui details, continuing to add features, cleaning up code, and perform rigorous testing.

Ruby, Sinatra, Javascript, Highcharts, SQL, Amazon Web Services
