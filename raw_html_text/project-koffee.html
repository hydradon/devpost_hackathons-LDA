Inspiration

Fed up with no smart coffee machines at our humble abodes.The current alternatives on the market are priced at $250+, we wanted to produced and recreate an affordable solution using the tools and supplies that were available to us.

What it does

App interacts wirelessly to produce hot coffee; on/off control, can schedule brewing time and date (work in progress).

How we built it

Hardware -> solder circuit board to raspberry gpio Software -> raspberry pi running python scripts on apache server over wifi Software -> cross platform Xamarin app used to control the settings for the coffee machine

Challenges we ran into

Limited experience with hardware! Xamarin

Accomplishments that we're proud of

Our product works! Initially, we did not know if it would even be possible to connect the raspberry-pi to the coffee maker, we are also proud of our skills on the physical hardware side.

What we learned

Hardware, soldering, Xamarin, apache, python, raspberry-pi integration, knowledge about coffee brewing. We learned how to connect to the coffee machines logic board from the raspberry-pi and how to interact with it correctly. We also learned how to use the software that was recommended/incentivized for this hackathon. We also learned valuable team communication and cooperative skills.

What's next for Project Koffee

We would like to further enhance our smart coffee machine to expand the current functionality with features such as estimated brewing time and other coffee related functions. Our bigger vision is to build a platform to wirelessly connect household appliances to create a smart environment/internet of things.
