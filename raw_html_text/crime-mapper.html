Inspiration

Hearing many stories of people being afraid to walk through the city due to crime, we decided to build an app that will show people where crime happens in their city

What it does

Display where in the city a crime occurs and which type of crime occured.

How I built it

Node.js backend, jquery & bootstrap used on the front. Google maps api for the maps visualization

Challenges I ran into

Getting Node.js to talk to the google maps api

Accomplishments that I'm proud of

Getting google maps api to work with our project.

What I learned

Need rest to program effectively

What's next for Crime Mapper

  • Reroute users away crime using Google Maps Direction API
