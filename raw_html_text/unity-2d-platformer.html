We have played a lot of 2d platformers when we were children. Therefore, we love to remake our own version of this game genera.

We encountered tremendous hardship either technical or coronating problems, but we overcome barriers and finished a game we would like to play!

This is a gender-balanced team. The female members of this team heavily contributed to the grand theme of the development project of this game. They worked hard on alpha-testing and graphic design.

Future update expected: graphic improvement, highly functional checkpoints, and better animation.
