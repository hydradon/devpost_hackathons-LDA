Inspiration

Home automation is great. But what about a larger scale? What if people could interact with large scale automation systems via Alexa. I'm thinking the Belagio Fountains, or the Mirage Volcano. The systems that Vegas is built on.

Let's start by connecting Alexa to a professional lighting system.

Marianne is the best lighting assistant I've ever worked with. I named this skill after her.

What it does

Controls professional lighting entertainment systems via Alexa.

How I built it

Typing very quickly. Alexa -> Lambda -> IOT -> Windows Laptop -> Martin M-PC (lighting control system)

Challenges I ran into

M-PC's API is limited. Ended up injecting keystrokes. Had to learn how synonyms are represented in the event coming into Lambda.

Accomplishments that I'm proud of

Successfully control professional lighting system via Alexa.

What I learned

What's next for Marianne - Personal Lighting Assistant

Flesh out functionality, fix bugs, improve UX, expand to other entertainment systems (audio, video, etc).
