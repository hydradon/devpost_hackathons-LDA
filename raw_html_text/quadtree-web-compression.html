Inspiration

Mainstream compression algorithms tend to produce a fuzzy or smudged image.

What it does

Quadtree web compression compress images in a way that is beautiful to they human eye. The library loads the compressed images from the lowest resolution to the highest in order, thereby reducing the time taken to load the first frame.

How we built it

Front end was written in Angular and React, which allows simple integration with a third party project. Back-end was written in go lang allowing the server to be easily ported.

Challenges we ran into

Iteratively loading images for each resolution caused problems. Integration of Go lang and Angular framework caused routing problems.

Accomplishments that we're proud of

Flexible and easy to integrate framework for both react and angular.

What we learned

Learned to integrate various frameworks on a common infrastructure platform such as Google cloud compute and Git-Hub.
