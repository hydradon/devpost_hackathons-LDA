Inspiration

Our inspiration comes from old flash games like "Hands of War" on websites such as y8 and miniclip.

What it does

It's a simple top-down shooter game.

How I built it

We built the game using the godot engine. The script is written using GD script which is a language unique to godot, but is very similar to python.

Challenges I ran into

As our programming knowledge is next to none, we ran into multiple challenges such as coding the AI, which was supposed to avoid obstacles, but we had to settle with zombies that ran into walls.

Accomplishments that I'm proud of

We're proud that we could actually hand something playable in.

What I learned

We furthered our knowledge with GD script and learned a lot about AI coding and menu making.

What's next for No Zombie's Land

We hope to heavily improve the code behind the game and make the zombie AI way smarter. Furthermore, we hope to add different enemy types to compliment our unique roster of playable characters. Lastly, since the Godot engine can support 3D we hope to upgrade our game to today's graphical standard.
