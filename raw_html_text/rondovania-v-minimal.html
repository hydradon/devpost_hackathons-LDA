Inspiration

Video games can be built relatively easily game-play wise. It is hard to balance art, music and gameplay.

Therefore this is a project where code is not first-class, but instead if the code is not accompanied with good art and decent music the code will be discarded.

In the end there is minimal gameplay, but at least quite visually appealing. Something that with some more polish might become perfectly acceptable for an "indie game".

Note

Clojure is not really that fit for game-dev. Heavy wrappers might be needed to successfully unleash Clojure's power.

2D animations can be hard.
