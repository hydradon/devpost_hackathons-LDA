Inspiration

The idea was to create a useful analytics platform for content creators to help them stay relevant in their field by providing them with info about trending topics and relevant news.

What it does

Using the HPE HavenOnDemand API it analyzes the user website and starts indexing relevant websites using HOD Connectors. The keywords found on the user's website are then matched with the Text Index in order to find relevant content.

How we built it

The app makes full use of the Django framework

What's next for RelevApp

The app has a lot of potential, the next step is surely to improve the website analyzing capabilities and improve the amount of data indexed from the web (it was limited by the limitations of the free HOD apis)
