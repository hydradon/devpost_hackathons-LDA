Inspiration

Nos hemos inspirado en las aplicaciones que facilitan la navegacion a pie por distintas ciudades.

What it does

Usamos el API de Google Maps e identificamos a encontrar los lugares que te puedan interesar en cierto momento mediante categorias para que puedas realizar tus pendientes esporadicamente mientras pasar por algun lugar que pueda satisfacer tus necesidades.

How we built it

Lo construimos usando el API de Google Maps, Pebble Smartwatches, PHP con Laravel para el back end, el cual tiene como objetivo ir aprendiendo de tus necesidades mandando notificaciones las cuales puedes aceptar o rechazar y tu comportamiento sera analizado.

Challenges we ran into

Jamas habiamos desarrollado para Pebble. Tuvimos algunos problemas de configuracion y una gran curva de aprendizaje.

Accomplishments that we're proud of

Pudimos integrar completamente la interfaz entre el dispositivo celular y el smartwatch.

What we learned

Aprendimos a utilizar el API de Google y a resolver nuevos errores y obstaculos con PHP.

What's next for Pebble Street Assistant

Planeamos terminar la aplicacion pronto, ya que es muy prometedora y ya se encuentra en una etapa avanzada.
