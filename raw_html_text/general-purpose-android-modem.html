What it does

Takes in arbitrary data, chunks it, encrypts, and then transmits file as a series of 8 KHz tones.

Provides a 150 bps data transfer rate, with some forward error correction.

Able to transmit over any comms path that sound travels across, i.e. cellular voice, PSTN, radio, playing loudly nearby.
