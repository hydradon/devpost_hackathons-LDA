GroupMe is widely used at my college for everything from group projects to class management. The amount of time it takes to manually add each member to a chat by hand is obnoxious, and most of the time we already have a CSV file with contact information for each member. With this tool, professors or mentors can simply upload the existing CSV, automatically create the GroupMe group, and add each member to it.

--

Right now, the CSV needs to have headers, and the columns need to be in the order of first name, last name, and then phone number.

Also, this will currently add me as the administrator of the group.
