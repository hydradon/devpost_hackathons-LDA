depict

_ MHacks 12 2019 Google Cloud Project _

When traveling the world, one may often encounter novel situations and sites where one might want to get a better understanding of the foreign culture and language.

For this purpose we created depict, an image-based classification and translation tool. Depict can identify objects within photos and allow users to select relevant words as well as the language to translate to.

Depict utilizes the Google Cloud Platform, more specifically Google Cloud Vision and Translate.

We implemented the system on Amazon AWS using Python Flask backend and ReactJS frontend to give a the user a fast and responsive interface. Over the course of this project, we learned many things about using the Google APIs from Python. We faced several challenges regarding the usage of Google APIs in conjunction with React.

We hope that depict will be a useful tool for those who want to learn more foreign languages and cultures. We believe that depict will be able to connect people from all around the world.
