Inspiration

I talked to Alex at well, and they are looking for a solution to let patients pay for Dr. appointments directly using crypto currency and tokens.

What it does

It validates a patient's Ether or Well Token (ERC20) payment with the ethereum network and sends a response back the the Well App. I built it for Well, but this can server as a general purpose crypto payment solution for any exisiting platform with minor tweaks.

How I built it

Overnight.

Challenges I ran into

Too many to count. Debugging solitidy code can take a lot of guess work because the language is still under heavy development.

Accomplishments that I'm proud of

Well, it works (pun intended)

What's next for Crypto Payment Gateway

Integrate into more features of the well ecosystem, and possibily other platforms.
