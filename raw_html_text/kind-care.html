Inspiration

William's Girlfriend is an early childhood education major. The realization that special needs kids cannot simply be placed with a usual baby sitter. These children require specific care and attention parents may struggle to find someone they not only trust but is qualified to handle the kids need if they have to leave for a work absence or wish to take a date night.

What it does

Connects Parents with Volunteers or Service Providers who are equipped to handle disabled / special needs kids. We can figure out someone in the system a volunteer nearby to the parents. As we have an SQL Database.

How we built it

Using HTML, Bootstrap, and PHP to create a database to compile the information submitted. Employed programs such as Atom to lay out the URL. We started during the Midnight Madness and went straight on till far past dawn.

Challenges we ran into

Getting the PHP to run properly, creating the website in a short amount of time, learning a new program so quickly. Just the sheer fact that two of the member Haley and I had never made a Website before left a lot of learning on the fly!

Accomplishments that we're proud of

The Sleek looking graphics and short amount of time we were able to get this up and running. From an hour after start of Midnight Madness. Configuring the SQL database with tables that show both Parents and Volunteers and can be focused on certain criteria such as city or state!

What we learned

For Haley and I how to create a website and work with programs such as Bootstrap and HTML in such a short amount of time. William that PHP is still a cruel mistress and he remembers less then he thought: so how to PHP again.

What's next for Kind Care

Perfecting it's appearance and configuring it for the publics consumption.
