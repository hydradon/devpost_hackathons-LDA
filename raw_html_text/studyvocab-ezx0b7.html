Inspiration

While having a hard time while studying SAT vocab

What it does

It allows you to set the time and study SAT vocab while checking the definitions through oxford dictionary

How I built it

Used Oxford-English-Dictionary API to retrieve vocabularies' definition Using CountDownTimer class, the app updates the UI while time is counting down.
