Inspiration

We are common users of 3D printing, so combining the ease of access of Google Home and the utility of 3D printing was an easy choice for a project.

What it does

Using MAD Print, all that is necessary to print is to tell your Google Assistant or Google Home, "print x", and it will print it.

How we built it

Using IFTTT alongside a webhook, several servers, an SQL database, and Python scripts, the printing request is converted from text to material.

Challenges we ran into

Interfacing software and hardware in ways that are not frequently done caused the majority of our difficulty.

Accomplishments that we're proud of

The fact that it works.

What we learned

IFTTT, SQL, PHP, server hosting, gcode

What's next for MAD Print

Add improved UI and features to make the system more user-friendly.
