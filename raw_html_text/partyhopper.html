Inspiration

As females, going out for a night can be scary and we often waste time and energy looking for our friends. This is a way that we can feel safer and more connected to each other as well as being able to check in.

What it does

Create groups of people to share location for one night with push notifications of when you leave the locations as well as a notification when one friend decides to go home and when they got there.

How I built it

We used XCode to build the iOS application utilizing the MapKit API.

Challenges I ran into

We attempted to use Radar.io to track user location, this proved to be a big challenge as the SDK was completely new to our entire team. We had to change our plan at the last minute which led to our application not working as expected.

Accomplishments that I'm proud of

Using brand new technology and adapting to challenges we faced throughout.

What I learned

This was our first time working with XCode, Swift, MapKit.

What's next for PartyHopper

Get the functionality up to where we want and be able to help our friends feel safer.
