What's next for RightPath

RightPath is a web app that takes some input from the user and determines the best resource from the list of resources provided. It saves people time by sorting through the links and just presenting the one that is most valuable.

All the user has to do it click Get Started and follow the instructions on screen. They may also view all the resources by clicking on a topic in the menu bar.
