• The belly of the beast

Inspiration

We wanted to make a capacitive touch keyboard.

What it does

It sends arrow key input to the connected computer as a keyboard.

How we built it

We built it with an Arduino UNO board.

Challenges we ran into

Soldering the wires and making them responsive to the capacitive touch. The lack of the Arduinos I/O was also a challenge to overcome. We were not able to use a transistor mux matrix as we are sending under 10microAmp through the resistor.

Accomplishments that we're proud of

We finished the thing and it works with our computers.

What we learned

How to use Arduino and more about electrical design.

What's next for Touch Team

Make a full keyboard not just arrow keys.
