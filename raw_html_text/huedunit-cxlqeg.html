Inspiration

Other color guessing games online where you have to precisely select colors.

What it does

Color game. A random color flashes and you have to attempt to click that color somewhere on a color spectrum.

How we built it

Eclipse and Java using Class-Model separation.

Challenges we ran into

Actually generating the color spectrum to be responsive to different screen resolutions.

Accomplishments that we're proud of

We made a game! And we constructed algorithms to calculate score.

What we learned

Planning planning planning! Always plan for the worst first so you do not run into these edge cases later on.

What's next for Huedunit

More game modes! Most user customization!
