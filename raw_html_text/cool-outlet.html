Inspiration

We wanted to work with JavaScript, and we wanted to make a game, and we wanted a map.

What it does

You play as the Onion Prince, a simple onion who is just trying to get to work. Can you help him overcome all of the obstacles and get to work on time?

How we built it

We used Phaser.js to build a game running on a Node server. It's currently deployed on Heroku.

Challenges we ran into

Neither of us have much front end or JavaScript experience, so the learning curve was a bit steep.

Accomplishments that we're proud of

We have a (mostly) working game, which is pretty cool.

What we learned

The basics of Node servers, the Phaser library, and the basics of JavaScript development.

What's next for Cool Outlet

We currently have the first three levels, but a lot more stops on the T for future ones. We also want to work on the logic for our level generation, lock levels that are farther away, and add in a leaderboard.
