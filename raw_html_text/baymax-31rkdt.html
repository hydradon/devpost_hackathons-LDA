What it does

Baymax is a health care web app that tracks your daily sleep and eating habits. Using data visualization we are able to display user's sleep patterns and caloric intake in a friendly manner for any demographic to understand. We hope that this application will help individuals start/maintain a healthy lifestyle.

Challenges we ran into

  • Learning new APIs
    • Google cloud platform (vision API)
    • Chart.js
    • Bootstrap
    • FireBase
  • Learning new languages
    • Javascript
    • CSS
    • HTML
    • Vue

What we learned

Throughout this project, we were able to learn full stack development and how to work together as a team to build a web application.
