Inspiration

Our team wanted to make a fun java game for our first hackathon.

What it does

Mouse Chase is a game where you get to be a mouse and eat all the cheese you can... you just have to avoid being eaten by the cat!

How I built it

We built it using Java on Eclipse.

Challenges I ran into

The challenge was uploading it to a webpage and just debugging the code in general.

Accomplishments that I'm proud of

We are proud of the working tunnel feature in the game, and we like the leaderboard feature that we included to keep track of high scores.

What I learned

Throughout the process, we learned a lot about Microsoft Azure.

What's next for Mouse Chase

Maybe Mouse Chase could eventually be a mobile app too.
