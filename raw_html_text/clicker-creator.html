Inspiration

Have you ever been obsessed with cookie clicker or clicker heroes or any other of the many clicker games? Now you can easily create your own clicker games and share it with friends!

What it does

A user can upload images and set game rules for their own custom clicker game. They can create achievements, store items, messages, and themes.

How we built it

Pain and caffeine.

Challenges we ran into

We had too many input fields and customization features so it was difficult to implement and debug everything.

Accomplishments that we're proud of

We're proud of our art team and UI team for the lovely interface. We also implemented a fairly complicated database structure.

What we learned

Start simple, work up.

What's next for Clicker Creator

We need to finish this project and get the front-end to respond to the server.
