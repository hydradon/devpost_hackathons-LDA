Inspiration

We started out by thinking about a way to find your parked car after shopping or another activity

What it does

Stores a location and points back to it based upon your current location

How I built it

Garmin Connect IQ plugin with Eclipse. Monkey C was the language.

Challenges I ran into

The Garmin vivoactive has very limited specs in terms of memory and processing power

Accomplishments that I'm proud of

We started with a very basic idea and started developing and then turned it into a very versatile tool.

What I learned

How to use a location API

What's next for WayPoint Finder

Storing multiple locations and improved optimization
