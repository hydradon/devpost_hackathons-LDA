Inspiration

Combined playing on lots of devices to make music great again!

What it does

It is at it's most basic level a chat room that plays music instead of people chatting!

How we built it

We built it using node.js for the server and JS/jQuery for the client side and then we used HTML audio player tag to play the music. We also used web sockets for sending the information between client side and server side.

Challenges we ran into

Getting the server to communicate with the client and get real time tracking on music files.

Accomplishments that we're proud of

The player excepts two file formats. MP3 and Note files. The room can be hot joined so if music is playing in a room you join and start listening to that point in the song.

What we learned

Node.js and JS as well as WebSockets
