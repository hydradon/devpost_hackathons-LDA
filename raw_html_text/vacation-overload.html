Inspiration

There are so many undiscovered places in the world, that it is often difficult for families to choose a single spot for vacation.

What it does

Our program first makes the user take a simple survey to personalize travel ideas before presenting a list of possible places to go.

How we built it

On BlueJ with Java. CSS was used to build a background for the program in the future.

Challenges we ran into

We were not able to figure out how to print a image of the presented locations, and were forced to leave it out.

Accomplishments that we're proud of

our presentation is beautiful

What we learned

if else statements are great

What's next for Vacation Overload
