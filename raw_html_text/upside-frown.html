Inspiration

Being online is fun, but it could be so much better.

What it does

Upside Frown finds pictures of people on websites you already use and flips their mouth upside down. In addition, it adds a bunch of other facial features like googly eyes, tongues, mustaches, and the Wilhelm scream. From our experience this makes browsing the web much more enjoyable.

How we built it

Upside Frown is a Chrome extension that uses the Google Vision API to identify faces in images on websites you visit.

Challenges we ran into

It was difficult to overlay an upside down mouth using vanilla HTML and JavaScript.

Accomplishments that we're proud of

The extension works and makes us laugh.

What we learned

We learned that you can use client-side JavaScript to do a surprising amount.

What's next for Upside Frown

More facial features!
