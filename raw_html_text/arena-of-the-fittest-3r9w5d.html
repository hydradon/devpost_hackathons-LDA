Inspiration

We wanted to play video games but also live a healthy lifestyle.

What it does

It tracks the distance the user travels and converts it into points that can be used to improve in the game.

How we built it

We used MIT's App Inventor 2 to build our app.

Challenges we ran into

Understanding the formula used to calculate the distance between two points on a sphere(the earth)

Accomplishments that we're proud of

10/10 sprites

What we learned

We learned how to structure code and collaborate with each other.

What's next for Arena of the Fittest

We are hoping to improve the movement quality in the game as well as Randy the Rat's movement patterns.
