Inspiration

There are so many posters for cool events around campus that we take pictures of and then forget to add to our camera. We wanted an app that would automatically add the event to our calendar.

What it does

It scans an image and extracts the time and date of the event.

How we built it

We built the app using Tesseract Ocr.

Challenges we ran into

The differences in the way dates and times can be written make it challenging to extract the date and time.

Accomplishments that we're proud of

We have a working demo!

What's next for Codename: Event Parser

Make the algorithm work for a wider array of posters.
