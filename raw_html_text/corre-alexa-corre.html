Inspiración

El desarrollar una aplicación que fuera simple y atractiva para el desarrollo de las habilidades de la memoria para los usuarios de cualquier rango de edad y poder lograr una aplicación que fuera tanto retadora como divertida.

¿Qué es lo que hace?

Por medio de una aplicación de voz con un dispositivo Echo podemos hacer que el usuario se inmersa en una experiencia donde con la ayuda de alexa tenga que salir de un calabozo antes que su antorcha se extinga y no pueda salir nunca.

¿Comó la construimos?

Haciendo uso de la plataforma de desarrollo para alexa proporcionada por amazon así como un código de backend Node.js para el análisis de matrices, las cuales actúan como el mapa de nuestro calabozo, utilizando

Dificultades del camino

En este proyecto uno de los elementos más complicados fue el desarrollo de una buena experiencia de usuario, dado que los dispositivos echo en algunos casos cuentan con pantallas para la visualización de datos pero no para la interacción del usuario con el sistema.

Aprendizajes

Como equipo no teníamos una experiencia con los dispositivos Echo, fue un reto especialmente interesante dado que nunca habíamos trabajado con estas tecnologías y el poder desarrollar bajo un nuevo modelo de diseño es bastante interesante
