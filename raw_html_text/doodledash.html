• DoodleDash

Inspiration

To bring people together to create art in a fun, competitive environment.

What it does

DoodleDash allows players to compete for the best doodle in an online game environment judged by your peers or by Google's Vision API.

How I built it

The website is hosted on the Google Cloud Platform using Google's Compute Engine. The back end logic is in C# and the front end was built using JavaScript, Vue.js, and HTML. The project was built in Visual Studio.

Challenges I ran into

Some of the less experienced members of the team had challenges getting familiar with the front end development.

Accomplishments that I'm proud of

Successfully integrating Google's vision API, hosting the project on GCP, and getting a multiplayer game online in 36 hours.

What I learned

I learned how to host and deploy a project on Google Cloud Platform.

What's next for DoodleDash

More doodling!
