Inspiration

Passionate on the subject and see how hard it is for people to correctly recycle

What it does

A trash bin that identifies what material the item that is being dispose is made of an points to the correct bin

How we built it

We found a dataset of trash training images with 6 classes, trained on tensorflow, and built a real-time dashboard to provide analytics of current items you threw away.

Challenges we ran into

Issue with communicating between tensor and dashboard

Accomplishments that we're proud of

Created a prototype that can identify what material a single item is with relatively good accuracy

What we learned

This is possible technology and can be used for a real project

What's next for Smart Trash

Create physical prototype to test concept Test at Cornell Tech and public school on Roosevelt Island Gain quote from Nielsen on user data for pricing Figure out rewards program for user engagement
