Inspiration

As high school students, almost everyone undergoes stress and we wanted to create something that could help them feel better during those stressful periods.

What it does

Anyone who visits the site can receive a positive affirmation, and find other resources to help cope with stress, such as a list of de-stressing activities. It also provides a place for people to share their ways of living.

How we built it

We learned html/css and a bit of javascript to create this website.

Challenges we ran into

We both came to TechTogether with little to no experience in web development. Our program often wouldn't work because of all the mistakes accumulated in a few lines of code, but we got a lot of help (haha yikes)

Accomplishments that we're proud of

We are proud that we were able to create something using code that we weren't very familiar with.

What we learned

We learned all about html/css, and how to use other resources such as GitHub and Terminal. We also learned about TechTogether, and how great of an opportunity and experience it was, especially for inexperienced coders.

What's next for Bee Happy

If there is the very slim chance that Bee Happy does succeed, we will continue to make advancements to the website to make it even better and efficient for users.
