What it does

It is a Neural Network that has been trained on over 10,000 different symptoms and previous illnesses to help athletes find out if they are at risk for certain dangerous and preventable diseases.

How we built it

We used python 3.7 in order to create a neural network, and then trained it on data found from several health organizations (including aspetar and BMJ)

Challenges we ran into

Getting the neural network to function properly and to accept an eleven-dimensional array (we had to program the entire thing ourselves)

Accomplishments that we're proud of

The UI of the site and the speed that the Neural Network can operate at

What we learned

A lot about class creation in python and several Javascript functions in order to properly parse the information into readable chunks for the Neural Network.

What's next for Asobey?

Adding more functions to help other members around the sports scene in order to provide more help for everyone!
