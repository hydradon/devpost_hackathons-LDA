Inspiration

What inspired our group was the lack of knowledge that we all had coming into the STEM fields. We wanted to create something that would get the attention of students.

What it does

We have created a website with three different games. These games provide a fun game while showing the code to learning how the game works.

How we built it

Two of our games were built on a website called Scratch. The third game was creted through JavaScript.

Challenges we ran into

Our team had a challenge of figuring out what program to use. Three of us only know how to use Scratch, while the fourth only knows how to use JavaScript. Initial we were going to build one game, but then quickly decided to design three different games. Two members worked on the JavaScript game. While the other two worked on two other games.

Accomplishments that we're proud of

We are very proud of completing three different games within 24 hours.

What we learned

We have learned so much about coding. We learned that coding is a very intricate art.

What's next for Project AG

We plan on building other STEM related games.
