Inspiration: Solar powered light sources to improve on the response time for emergency services.

What it does: It answers the prompt. This helps alert vehicles to move to different lanes to leave designated lanes accessible for emergency vehicles.

How I built it: Teamwork, HTML, CSS, and Javascript.

Challenges I ran into: Creating an interactive front-end UI for our system design.

Accomplishments that I'm proud of: Our team worked hard and worked together to create something that answered the prompt as well as something we all agreed on.

What I learned: Simplicity is sometimes better when trying to come up with a new system design.

What's next for Grizzly Bright: Hope that Grizzly Bright inspires a system for cities to use to minimize casualties during natural disasters.
