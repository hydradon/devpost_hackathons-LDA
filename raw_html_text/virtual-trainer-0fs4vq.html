Inspiration

We wanted to create something that will promote having good health and encourage people to stay in shape.

What it does

Allows users to take a picture of themselves or anyone else, have the application tell you what body type the individual in the picture is and receive helpful tips based on that.

How I built it

Trained a model with images of different body types of men and women using Clarifai's API. Swift was used to create the front-end of the iOS call and make a call to the API.

Challenges I ran into

We had trouble making the API call from within Swift, so a long time was taken to resolve that issue.

Accomplishments that I'm proud of

We're proud of learning about Clarifai's API because it a very well done and useful API

What I learned

Clarifai, Swift
