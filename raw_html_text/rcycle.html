Inspiration

Help the environment and give students an initiative to recycle

What it does

Allows UCR students to recycle plastic bottles or cans for rewards

How I built it

A raspberry pi receives input from a magnetic card reader or keypad as well as a bar code scanner. Information is sent to a secured server that contains their information and records.

Challenges I ran into

Completing the task in a short amount of time.

Accomplishments that I'm proud of

I was able to implement communication from the raspberry pi to multiples ICS.

What I learned

I learned that it is very important to communicate and work with a team in order to accomplish tasks.

What's next for Rcycle

Hopefully Rcycle can be implemented in other UC schools.
