Inspiration

Every person uses internet on daily basis so a Car Advisor would help people to shop smarter

What it does

It takes all the car Makers from the Body element of the webpage, and then finds all the models of the cars. It uses CDL API to retrieve data of MOT tests

How we built it

It is a Google Chrome Extension which is very easy to install. We built it using Javascript and Jquery and data gathering from queries to CDL API.

Challenges we ran into

Syntax of jquery and Json were complex, and the sever of CDL was down all night long.

Accomplishments that we're proud of

2 of us managed to cope with a lot of work. Background script to communicate with front end was really hard.

What we learned

Team work definitely, and new languages as well.

What's next for Car Stat

Worldwide popularity and more data gathering from cars in the future
