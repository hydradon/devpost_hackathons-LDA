Inspiration We wanted to learn about Google's machine vision API and reduce human effort on social media.

What it does

Automatically add hashtags to your to twitter posts using Google Cloud Vision API

How we built it

Android Studio, Google Cloud Vision API, Twitter API, and coffee... lots of coffee

Challenges we ran into

Social media APIs like snap-kit have strictly controlled access

Accomplishments that we're proud of

Being able to lower human effort on manually writing hashtags and generating useful and funny hashtags

What we learned

API usage and design can drastically affect your own designs

What's next for SnapTagger

Integration with Instagram
