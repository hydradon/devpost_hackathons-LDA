Inspiration

About a month ago, I was supposed to finish some homework once I got home, but I forgot. I was carried away from my work because I was socializing with my roommates. The next day before class started, I thought to myself that it would be amazing if I could create my own personalized to-do-list app.

What it does

MakeList is a to-do-list app that allows you to create a list of your daily activities.

How I built it

I used Swift to build my app.

Challenges I ran into

I had a hard time passing data from the view controller to another view controller. I also had troubles transitioning from one view controller to another using segues; I forgot to name the segue identifier.

Accomplishments that I'm proud of

I'm excited that I got the chance to learn the fundamentals of Swift, so I could build cool projects in XCode. I'm proud that I was able to build and ship my app.

What I learned

I learned how to create simple projects using Swift and how to organized my code.

What's next for MakeList

MakeList will get an update which includes bug fixes and a user friendly UI.
