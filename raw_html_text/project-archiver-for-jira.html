Clean-up your JIRA with ease - Activity tracking, archiving, freezing & bulk editing for your JIRA projects

1 - Create archiving, freezing & bulk profiles as you like

Select the schemes that you want to apply on specific JIRA projects and save this set as a profile, so you can reuse it later on. Archiving & Freezing profiles are already included - You can start right away!

2 - Search for unused projects

Search for unused projects with no activity in a specific time period, select the archive or freeze profile & finally choose the JIRA projects to archive, freeze or bulk edit.

3 - Project overview

The project overview provides you with important data like the last project activity, the number of issues, the project lead and many more. JIRA projects are tabbed by categories that you can define in your own profiles.
