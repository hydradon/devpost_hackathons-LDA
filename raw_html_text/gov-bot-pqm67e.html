Inspiration

We had learned about the issues with lack of civic involvement, especially at the state level, but what set the decision for us what the versatility and ability of the Phone2Action API and the impacts it could have on the community.

What it does

Our chat bot aims to get the silent 52% of US residents involved in local policy making by making bill information easily accessible and making contacting legislators an ease.

How we built it

We used the ProPublica API to get the recent bills. We used the Phone2Action API to get the proper legislator from the user’s address and to contact said legislator. The bot aspect of it was done using Python 3 and the fbchat API.

Challenges we ran into

Creating the logic flow and the integration of multiple apis into a single chat bot

Accomplishments that we're proud of

Managing to learn the ins and outs of two new APIs to create a piece software which has a very unique yet huge impact on the community.

What we learned

Proper use of different API’s in the Python programming language.

What's next for Gov Bot

We’d like to expand this to include real-time alerts so people don’t have to check with the bot, the bot can just check with them.
