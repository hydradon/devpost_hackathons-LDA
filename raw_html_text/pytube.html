Inspiration

a large amount of video in youtube but can't play it in background

What it does

allow too listen too youtube music in background without download the video for saving data

How we built it

Rasberry server (youtube TV server) in Python. Use official youtube remote control for smart TV in smartphone and PyTube app built with react-native in smartphone

Challenges we ran into

IOT, general challenge

Accomplishments that we're proud of

free spotify like with more music than spotify !

What we learned

First time react-native, socketIO communication, server Python

What's next for PyTube

More stability, use another player for native support of .m4a for avoiding transcode .m4a to .mp3 UI design...
