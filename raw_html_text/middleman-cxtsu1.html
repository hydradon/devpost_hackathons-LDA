Inspiration

We aimed to faciltate social meetups by finding the optimal meetup location for all members!

What it does

Calculates optimal meetup locations using the midpoint between all members' addresses.

How we built it

Using Firebase, Google Maps APIs, Node.js, Python, HTML, Heroku server.

Challenges we ran into

Integrating front-end and back-end server calls to align generated data points.

Accomplishments that we're proud of

Built and designed a user-friendly UI that operates using custom databases and servers.

What we learned

Node.js, integrating APIs between front-end and back-end, interchanging service icons, security authorization.

What's next for MiddleMan

Creating a mobile app and interacting with other mobile frameworks.
