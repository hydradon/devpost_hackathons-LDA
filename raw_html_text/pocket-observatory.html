Inspiration

There aren't many ways to view active satellites! This is a way to garner public interest in it.

What it does

It allows users to view satellites currently in orbit using a Google Cardboard or Daydream, for an immersive experience.

How we built it

Google VR SDK, Unity C#, and lots of snacks!

Challenges we ran into

We initially were going to use a live feed of real coordinate data using an API by n2yo, but it is a challenge to convert latitude and longitude to Unity coordinates, and make API requests through Unity.

Accomplishments that we're proud of

It looks very immersive and peaceful!

What we learned

The key to a good VR experience is good use of the camera and lighting.

What's next for Pocket Observatory

Live data that uses your own coordinates to give an accurate picture of where the satellites are!
