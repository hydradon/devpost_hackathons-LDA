Inspiration

We wanted a central/organized location for events happening on Swarthmore campus.

What it does

Our project scrapes data from the Swarthmore Calendar on Swarthmore website (Name of event, Time, Location) and displays that to the user using google maps API.

How we built it

Our scraper was built using lxml and requests on python, while our front-end was built using AngularJs 2.0 and the google maps API.

Challenges we ran into

Scraping data from the Swarthmore, Website development

Accomplishments that we're proud of

Kind of figuring out how to scrape HTML from the Swarthmore website and building a JS website

What we learned

The importance of planning and top-down design; how to parse web info, how to creat a web application

What's next for Swarthmore Events Map

Hopefully it will be finished sometime in the future
