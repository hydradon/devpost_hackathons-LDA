Inspiration

Student safety is a large concern at a campus that is as large as the University of Washington.

What it does

We created an application to visualize crime incident data around the UW campus area.

How we built it

We used SpotCrime public API for crime incident data and compiled it into a heat map using the Google Map API within Android Studio.

Challenges we ran into

Learning Kotlin and how to work with the Android SDK, how to make HTTP request to spotcrime API, how to work with Google Map API .

Accomplishments that we're proud of

We came in with virtually no knowledge of how to code an Android application and we were able to create a working application. Although we were unable to incorporate all of the features we planned for, we are satisfied with our progress in this project.

What we learned

We learned Kotlin, developing in Android Studio, and GitHub etiquette.

What's next for Dubhacks

We want to apply our data visualization further in order to increase safety for students and to notify students of areas of historically high incident rates.
