Inspiration

Being fairly new to the world of financial independence, we decided to make an application that would let us plan for our future finances by setting easy savings goals and ways to pay off debt

What it does

Through simple questions, figure out the best financial choices for you to do, and sets easy goals to keep up with debt

How we built it

Built with android studio, nessie and various other APIs

Challenges we ran into

Figuring out how the APIs work, not letting them conflict with each other.

Accomplishments that we're proud of

Getting all of the elements to work in the first place, a lot of conflictswith code at the beginning

What's next for OneUp

Smoother more polished functionality
