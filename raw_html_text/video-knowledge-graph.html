Inspiration -

How to find information at the right place and right time? Using "Graph Database" technology to store relationships between every type of information. Using "Visual Mapping" technology to make these relationships visible for users inside a company.

What it does -

Guides you from a known starting position to the place with the information you need. It could be compared to a street map, showing the way to videos, documents, any information the user needs.

How we built it -

"Neo4j" as native Graph Database "R" and "R-Studio" for developing code "k-infinity" for visualization Coding an use case example.

Challenges we ran into -

Setting up the database environment. How to show the impact? Creating a useful data structure.

Accomplishments that we're proud of -

Building a solution out of an idea in 20 hours. Make the code running. Getting enough sleep. Maybe.

What we learned -

How to handle a native "Graph Database". How to connect this database to another programming language. Working together in a newly built team

What's next for IntraVIZ

Making an implementation service out of that. Creating an Artifical Intelligence to feed a "Graph Database" with unstructured data.
