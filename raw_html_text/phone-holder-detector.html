Inspiration

Running out of time with no other tangible products, I decided to play around with the Android API and learn about the different sensors.

What it does

Using accelermeter and gyroscope, I attempt to figure out whether the phone is currently being held/carried or resting still.

How I built it

Using the Android API and nothing else.
