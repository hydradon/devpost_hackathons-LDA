Inspiration: We wanted to create an arcade game that we could later import to Android/iOS.

What it does: It's an endless running game where you have to match the type of shape

and color of two separate shapes.

How I built it: Using the StdLib.jar from the computer science course of Princeton Uni.

Challenges I ran into: Animating in real time.

Accomplishments that I'm proud of: Finishing the whole game.

What I learned: How to animate with java's JFrames.

What's next for Shape Shifter: Fixing all the bugs when know about now and importing this app to Android/iOS.
