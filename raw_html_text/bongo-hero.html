Inspiration

We wanted to start using Unity for the first time as none of us really knew how to make a video game. We thought that making a Guitar Hero-like game would be a good experience for all of us.

What it does

Takes a song, converts it to MIDI notes, and allows the player to play the song with the created notes.

How we built it

We used Unity to create the game.

Challenges we ran into

Our laptops were not as powerful as we imagined it to be so we had to scale back on the whole thing and just do a 2D game rather than a 3D game.

Accomplishments that we're proud of

We were able to learn the intricacies of Unity and how to play around with sprites.

What we learned

Same, we learnt how to use C# with Unity and how to deal with scenes and sprites.

What's next for Bongo Hero

We want to turn this into a proper game.
