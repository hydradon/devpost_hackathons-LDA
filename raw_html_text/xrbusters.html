Inspiration

What it does

A game to cooperatively defeat waves of ghosts attacking the player, where only one unarmed player can see the ghosts.

How I built it

Using the Unity game engine and C# we created a VR game with multiplayer connectivity.

Challenges I ran into

Creating the networking between the two clients to share data properly.

Accomplishments that I'm proud of

Creating a system to where a VR and a flatscreen client can play together without having to share the same computer as well as implamenting an automated matchmaking system.

What I learned

Multiplayer connectivity

What's next for XRBusters

Polishing up the game and possibly attempting to sell the game on a marketplace
