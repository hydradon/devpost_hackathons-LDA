Inspiration

Last minute. Our actual game (a 3D racing game on Unity) had a bug so we had to scrap it and build an RPG at the last minute.

What it does

It's a game that's based on rogue - like battling and leveling up.

How I built it

We used a program called RPG Maker VX Ace to build this game, and created/used some custom made tile sets.

Challenges I ran into

Programming the game in general. I never used this program before so it was a challenge to code everything to trigger at the right time or even get the BGM track in.

Accomplishments that I'm proud of

Creating this game within 3-4 hours.

What I learned

How to use this new program and how to make my own tile sets.

What's next for Rogue Battle TEST

Hopefully we can expand on the map even more and also bring more characters into the story.
