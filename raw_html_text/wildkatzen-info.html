Inspiration

I wanted to try out the Alexa Presentation Language and wanted to build a small educational Skill.

What it does

It gives Information and Facts about wildcats. It also shows a picture of a wildcat on the Alexa Devices Display and you can slide between the different info pages.

How I built it

I looked on Github and found a sample project on the Alexa Labs repository. I used it as reference for building my Skill.

Challenges I ran into

I never developed for the Alexa Presentation Language and also build just a few skills with the Alexa SDK Version 2 so there were a lot of new things for me.

Accomplishments that I'm proud of

I am proud that I build my first Skill for the Alexa Presentation Language.

What I learned

I leaned how the Alexa Presentation Language works and also understood the Alexa SDK Version 2 better.

What's next for Wildkatzen Info

Maybe I could add more facts and different pictures.
