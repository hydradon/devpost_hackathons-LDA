Inspiration

We wanted to implement facial recognition in a new way.

What it does

Send money to a friend by taking a picture of their face. Or pay a friend/business by having them take a picture of your face.

How we built it

FacePay is available both online and on mobile. The web app is built with node.js on the backend and React on the frontend. We built the mobile app for iOS using Swift.

Challenges we ran into

Timing. Connecting the underlying technologies. Building an API for the iOS app.

Accomplishments that we're proud of

Learning a lot of new technologies. Providing multiple mediums for using FacePay.

What we learned

Images are just long ass strings.

What's next for FacePay

Not android.
