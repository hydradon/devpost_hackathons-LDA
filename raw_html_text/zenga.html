Inspiration

We wanted to create our own virtual reality as well as experience it for the first time.

What it does

Our project, Zenga, aims to provide a fun and relaxing VR experience, specifically in the area of simple puzzles. Zenga functions as a way to have fun and mellow out, and we hope that it can improve hand-eye coordination and mood.

How we built it

We used the game engine Unity to create Zenga. None of us has ever used Unity, so it was a very fun challenge for all of us.

Challenges we ran into

Originally, we wanted Zenga to be a multiplayer game where two players would play against each other. We also had some trouble pulling/grabbing blocks.

Accomplishments that we're proud of

We made a simple, fun, and satisfying puzzle game.

What we learned

How to maneuver and use Unity. Writing scripts in C#. Working in a group environment with new people.

What's next for Zenga

We want to use Unity to create other fun games. Or we would like to create our own fantasy worlds. It would be awesome to make connect players from all over the world to play Zenga head-to-head in real time.
