Inspiration

We were concerned about older people who might fall in their homes and not be noticed until it was too late.

What it does

It uses a Nest Camera along with Microsoft Cognitive APIs to determine if someone is incapacitated on the floor.

How we built it

We used a Node.JS backend along with a ReactJS Frontend. The backend grabs images from a Nest Camera and processes them with NodeJS. The Frontend allows for monitoring and configuration.

Challenges we ran into

Getting the Nest REST API to work correctly. Getting Azure Web App online and working with no prior knowledge of Azure. Connections Issues with Camera / Software

Accomplishments that we're proud of

Getting the Nest API to talk to the Azure Cognitive APIs. Well Designed Interface That we are doing something to potentially help people.

What we learned

Azure ReactJS Node.JS HTML CSS Nest API manual REST API Calls

What's next for Granny Nanny

Improved Machine Learning for better detection of bad situations. Push notifications to the phone rather than email
