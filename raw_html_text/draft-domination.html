Inspiration

As an avid fantasy football fan, and business analyst, it's always been a goal of mine to run a data-based fantasy football blog. This hackathon, we made that happen!

What it does

A blog that displays data visualizations with companion blog posts for fantasy football statistics.

How we built it

Initial data cleaning in Excel using data derived from Draft Kings. Data visualization with Tableau. Machine learning algorithms in Python Jupyter Notebooks.

Challenges we ran into

Integrating dashboards into a website while retaining optimal performance, gathering clean data, and executing well-trained machine learning models were our biggest challenges.

Accomplishments that we're proud of

We were successful in publishing the website and creating a platform ready to post articles. We already wrote and published two articles!

What we learned

When creating a website from scratch; most of the time is spent in perfecting the details.

What's next for Draft Domination

Train and test various machine learning models to determine the best predictor of fantasy points. Update the blog regularly with blog posts.
