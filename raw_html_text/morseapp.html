Inspiration

We wanted to build a user-friendly app that allows users from diverse backgrounds to explore Morse Code.

What it does

Supports translation to and from Morse code.

How we built it

Android studio

Challenges I ran into

Working with the Android Studio camera API and ensuring consistent result by synchronizing the cameras

Accomplishments that we are proud of

Applying what we learned about Android app development and building something fun and practical.

What we learned

How to HACK, how to GIT, how to use Android Studio, how to explore different API's, and most importantly, how to work in a team to achieve a common goal

What's next for Morsify

Ideally we would also like to support sound detection and translate audio signals into Morse code.
