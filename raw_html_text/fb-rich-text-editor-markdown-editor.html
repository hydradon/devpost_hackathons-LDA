Inspiration and What It does

Introducing Rich Text Editor WYSIWYG will be a game changer for FacebooK Post Status Updates.
For now, Markdown formatting seems to be only available on Event/Note pages.
My studies suggest that, if Markdown formatting happens to be rolled out globally in all post update status, this would be the biggest real game-changer for Facebook.
Using rich text (HTML) like bold, italics, links, bullets etc. in Facebook status updates will create more rooms for engaging updates.

Styling. Italizing, Bolding , Bulleting, adding Colors, Images etc. inside a Post Status Updates can easily allow Facebook Users to convert Facebook Post into blogging 's, News Posts etc.
Markdown Editor will make your Facebook post look elegant, pretty, attractive and Professional while driving professional engagements within Facebook communities.

How to use this app:

1.) Login using Facebook
2.) Use FB Rich Text Editor(WYSIWYG Markdown Formatting) to create a post, design the post content by adding colors,bold, italize, bullets, headers etc and see how elegant your Post content will look like.
If Markdown formatting happens to be rolled out globally in all post update status, this would be the biggest real game-changer for Facebook as this will create more rooms for engaging updates.
