• BMCC Food Options

Inspiration

To help BMCC students find food options near the college area.

What it does

Alexa helps to search for food options near the BMCC area.

How we built it

We built Alexa Lambda function using javascript and prompted it to give food truck options around the area.

Challenges we ran into

A challenge that we ran into while building the lambda function was the random function that we would use to separate a variable that we created with all the food trucks along with their addresses.

Accomplishments that we're proud of

We tested the program and it works as we planned.

What we learned

We learned to build Alexa skills with Lambda function.

What's next for BMCC Food Options

The next step for the application we built is to include more food options along with the discounts that are available to BMCC students, hence why we named the invocation name BMCC Food Options. After we will accomplish this, we were thinking of expanding this skill to the residential area. Then, we would tell Alexa what ingredients that we have available to us and then Alexa would tell us what we could cook with those ingredients.
