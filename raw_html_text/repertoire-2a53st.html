Inspiration

When somebody asked me to perform a piano piece for them a few months ago, I ran into a classic problem for musicians - the pieces that I had practiced before I forgot because I had not practiced them recently, and the pieces that I was working on currently were not worthy of performance. Repertoire is here to change that.

What it does

Repertoire is here to change that. You start out by adding all the pieces you should know how to perform into a dynamically created list. After saving, you will receive a notification every day of what piece to practice, randomly chosen from that list. You get points for streaks of completing your practice and you earn a day off for every 5 day streak. By doing all this, you will be practicing one piece randomly chosen from your repertoire every day.

How I built it

The app is built using XML and Java in android studio. Back end storing on the phone is done with XML so that the list of pieces the user inputs persists from one instance of the application to the next.
