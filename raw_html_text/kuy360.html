Inspiration

Indonesia has many great scenery, from sea to forest. On top of that, Indonesia has great culinary and culture that can make tourist interest to come. But, our tourism promotion is number 108 of 187 countries (using printed media). There's a chance to improve our tourism business using VR. Therefore, we built Indonesian tourism scenery using React360. This app is called 'Kuy360', 'Kuy' means 'come' in a Indonesian slang and 360 mean we can see whole information in 360 degrees.

What it does

People can see a scenery, food, and another information in 360 degrees.

How we built it

We built this application using React360 and hosted at Netlify.

Challenges we ran into

It's hard to find good asset (especially 360 image) and we rarely using React to develop apps.

Accomplishments that we're proud of

We can built this application though we just have a little experience using React :D.

What we learned

We learn that modern problem had to be solved using modern solution.

What's next for Kuy360

We want to release this app to government so it can be used to improve our tourism industry.
