Inspiration

Mental Health impacts everyone's lives every single day. The form we have created caters to exactly that issue and topic in a shameless, selfless way. Taking care of ones mental health should be as important as a usual doctors check-up when we don't feel good. We want to change that.

What it does

miHealth allows the user to choose from two basic emotions, happy and sad. It then prompts the user to enter a journal entry. The journal entry asks user to enter the date, a more specific emotion, and the events that may have contributed to the emotion. It then saves the entry into a text file. Resources are also available to user upon clicking the "Resources" button.

How I built it

Using object oriented programming in python 2.7.
