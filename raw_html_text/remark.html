Remark is built with Blockstack User Authentication and Blockstack GAIA storage. All sounds, and upvotes are stored in the respective users storage.

Current Functionality:

  • Users can login with Blockstack
  • Users auto get a Bitcoin wallet derived from their Blockstack identity (we can't see the keys)
  • Users can upload mp3's to their preferred storage (max 20mb in size per file)
  • Users can customize their Channel Header Image and Short Descriptions.
  • Users can search for Channels by their Blockstack name.
  • Users can subscribe to each others Channels for convenience.(note: "Latest" page is still under dev)
  • Users can view the Latest content from the creators they have Subscribe to.
  • Users can like/upvote sounds uploaded by other Users (note: currently must be done via the detail page of the sound)

Upcoming Functionality

  • Ability to Like/Upvote sounds on all presentation pages (via Channel, via Popular, via Latest, not just via Detail view like currently)
  • Monetization of the Like (merge the like/upvote and tip into a single onclick)
  • Ability to Record Sound directly in app.
  • Ability to Search for Sound by Tags
