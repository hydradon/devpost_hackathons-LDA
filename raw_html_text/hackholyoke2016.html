• Flashcard

We decided to build a program to help us study for Language classes, and actually - any classes that require memorization. On our program you can create different categories for your classes and add/remove a flashcard out of each category. Create a flashcard, add it to a category and start practicing! Flip the card to see if you've memorized the definition and move on to the next words! Study easy!
