Inspiration

We were inspired by our common problems. We brainstormed and noticed that a common problem we shared was an unhealthy habit of frequently watching youtube. We decided to create a chrome extension that would filter youtube videos with an A.I.

What it does

It is supposed to filter youtube results.

How I built it

Using a collection of javascript, jquery, json, and html frameworks. We also utilized Google's api's to build the A.I

Challenges I ran into

We started this project with backgrounds in Java and Java only. Our issues were not easily solved in with a back end language like Java. Thus, we had to learn java script and jquery within 24 hours. Our project involved us using the Google api's to build a working A.I and researching an extensive network of Javascript API's. The fact that we did not sleep for 24 hours only contributed to the hardship.

Accomplishments that I'm proud of

We built an A.I that was surprisingly accurate. The A.I averaged 90 percent confidence with all the keywords we tested.
