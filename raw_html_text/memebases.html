Inspiration

Almost every TV shows, movies, and anime have funny moments, which puts a smile on many faces and can brighten up people’s day.

What it does

Given an anime clip as an input, the app identifies and selects funny moments, take screenshots, and output the pictures.

How we built it

Machine Learning A set of several hundred images each labeled with “funny/normal” to train our ML model. As a test set, 110 random anime images were used.

Ios Development Used Xcode to develop a basic UI and used coreML to integrate the model.

What's next for FunnyMoments

Making it applicable to humans. Making it into something so that the users can input a video and find something of their interest e.g. vegetables, key, etc.
