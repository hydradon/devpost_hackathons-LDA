Inspiration:

In the beginning, we wanted to develop an application that can recognize all object such as: people, car brands, background, etc. from streaming video, but we did not find the data or technology that support our idea.

What it does:

Automatically recognize logo from video, image.

How we built it:

Started with research online to find the API or data to support this application. we found a website called Clarifai that has API we need. In the developer tools, we found that the logo recognition is very interesting. Next, we wanted to learn some new stuff like node.js and react, than began to program our application.

Challenges we ran into:

Limited pictures upload to Clarifai, hard to test the program; New to node.js and react, no experience; No server.

Accomplishments that we're proud of:

Finished image and video logo recognize, learned node.js and react.

What we learned:

node.js, react structure.

What's next for iLogo:

Fix all remaining bugs, and add more css to it.
