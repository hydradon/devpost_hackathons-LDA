Inspiration

I like to play this kind of simple games with hard difficulty. That is the reason that I want to try to make it.

What it does

Main character penguin have to avoid ice-burg and one famous bird, and collect points, so that set a new records.

How I built it

I built it by unity engine, make singleton, collusion, physical elements.

Challenges I ran into

using physical engine, thinking regarding to fix to enough balance. But most important thing is it should work.

Accomplishments that I'm proud of

generating ice-burg and one famous bird separately. store previous data so that can make rank, and achievement as medal.

What I learned

Understanding actual physical engine. It was my first try to develop with physical engine such as gravity.

What's next for FlappyPenguin

I am currently working on another game that name is Slash Runner, and it is almost 60% done. I hope I can finish it before judge time in 4 hours.
