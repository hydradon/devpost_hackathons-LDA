Inspiration

What annoys everyone, universally. How can we take that and make it better.

What it does

It reroutes ads towards sustainable organizations ads.

How we built it

Used Pihole, an open source local network ad eraser, and modified parts of it's code so it would redirect ads rather than erase them. The redirection takes it to ad templates that we created.

Challenges we ran into

The raspberry pi we have had to be held onto during coding.

Accomplishments that we're proud of

Fun project, creative idea, and it was the first time working with hardware for myself.

What we learned

Network Protocols, Google's AdSense API, how advertising on the internet works, Raspberry Pi usage, network setups.

What's next for Sustainable AdSpace

To try and develop it into a "plug and play" system, that can be then presented to sustainable organizations to give as giveaways in returns for donations, adding the overall benefit.
