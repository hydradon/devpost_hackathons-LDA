Discord-Bot-Ruby

Discord bot made during Hackathon The bot needs to be run using linux or linux emulator as the plugins that are needed include: libsodium, opus, ffmpeg and discordrb. There is a lot of Waluigi memes with the bot please understand. Most of the waluigi pictures and memes come from the web series Brawl in the Family http://brawlinthefamily.keenspot.com/ You also will need discord installed with an account and you need to authorize the bot which needs this link: https://discordapp.com/oauth2/authorize?&client_id=269979125320515584&scope=bot&permissions=0 Hackbot needs to be run as a Ruby file by using hackbot.rb Bot is slightly bugged voice commands do function on this version.
