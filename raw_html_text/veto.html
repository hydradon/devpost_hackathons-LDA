Inspiration

People need to be able to directly reach their reps. And reps need to be able to address every single valid issue raised by every concerned citizen.

What it does

Provide a platform for continuous check of reps, and q&a interchange between citizens and their reps. Power to the poeple. Hence the name, VETO

How we built it

We used React Native, Firebase and Google's cloud platform for text analysis

Challenges we ran into

UI development with React was really difficult

Accomplishments that we're proud of

Algorithms for rating both reps and questions using Google's text analysis engine.

What we learned

React

What's next for VETO

Question classification and grouping using machine learning Security Better and more interactive UI Even more accurate rating algorithms Tipping the political balance
