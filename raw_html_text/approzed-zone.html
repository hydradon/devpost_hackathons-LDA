Inspiration

Everyone needs the token allowances dashboard to track recent approves. Some Dapps creates infinite approves for tokens they use and later some bugs may be discovered in these Dapps.

What it does

Allows managing the previous token approves, and even to decline all approves in just one click!

How we built it

Using infura and hosted on Arweave

Challenges we ran into

Handling errors of tokens which are not compliant with ERC20

Accomplishments that we're proud of

Completely decentralized and highly usefull

What we learned

How to perform call requests to Proxied tokens.

What's next for Approzed.zone

We'll make an SDK to allow any wallets to integrate it natively
