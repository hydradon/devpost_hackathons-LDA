mars-view

Very briefly, this website/web app shows a slideshow of imagery taken by Mars rovers. It uses a NASA API to fetch the images and uses Vue.js to display them (I started off using (and learning how to use) React but got fed up of Babel compiler errors that started appearing when I added features using ES2017).
