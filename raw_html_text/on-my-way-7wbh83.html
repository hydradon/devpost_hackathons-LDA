Inspiration

As a student at UC Merced, who wants to graduate , it's obvious that the current school provided graduation planner was not sufficient.

What it does

Has a data base of (some) majors available at UC Merced, and a planner that gives information on prerequisites, units, applicability to major requirements, etc.

How we built it

Using ReactJS, we avoided using server side code for speed purposes.

Challenges we ran into

Due to the unusual nature of unidirectional data flow in React, we had to change the way we thought about object oriented programming. We also had a hard time with accumulating all of the applicable courses into a format that's easily readable by code.

Accomplishments that we're proud of

Why, the whole project of course. :)

What we learned

We had learned React.

What's next for On my Way

More Majors, more courses updated from the original website. Multiple Majors. Tools for choosing optimal paths for courses.
