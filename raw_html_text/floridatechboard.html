Inspiration

Every day our notifications were flooded with forum mails.

What it does

Extract important information from emails and categories it into different categories like Buying, Selling, Events, Apartment rentals.

How I built it

Initially we extracted data from previous emails and gathered important keywords for different categories. We saved this data in a database and created one Android application to show this data in different categories.

Challenges I ran into

We had to read through lot of previous mails and gather important data such as keywords which took a lot of time of our development process.

Accomplishments that I'm proud of

The user interface of out mobile app

What I learned

In events like hackathon, time management is very important and be prepared for next 24 hours.

What's next for FloridaTechBoard

Write an algorithm to automatically sort the mails into different categories and gather relevant information
