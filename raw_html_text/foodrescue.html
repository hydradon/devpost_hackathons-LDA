Inspiration

Silicon Valley Food Council

What it does( Problem Statement)

There is lot of food being wasted everyday by the restaurants owners and pantries that can be made available for needy people. There is need for a smartphone networking application that can connect these two parties and help in reducing the wastage. App should be user friendly and should be able to get the nearby locations where food is accessible.

How we built it

Building an Android application with Java backend server and MongoDB database layer. Twilio for SMS messaging

Challenges we ran into

Integrating Android app with google maps to find the required locations within 10 miles

Accomplishments that we're proud of

SMS communication with app, social cause

What we learned

Team coordination

What's next for FoodRescue

Volunteering and further advancement of application
