Inspiration

We wanted to build something we would use after the hackathon, and a useful dashboard came to mind

What it does

The app allows people to make a todo list and currently also redirects to a page which will have google calendar integration

How we built it

The website is built with python and flask while the app and database have java and android studio

Challenges we ran into

We didn't have experience with Google APIs, struggled with authentication, and learned certain app dev skills as we went

Accomplishments that we're proud of

The website is live even if it doesn't have much on it, the app has basic functionality, and the project can grow features easily

What we learned

Come to hackathons with more preparations (libraries installed) and explore skills we will use

What's next for Dashbo

Finish the calendar integration to finish the tool of syncing calendar events and giving best times for a group to meet in a timeline.
