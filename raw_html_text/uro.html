Inspiration

Getting everyone to help non-profits help the community. Time for the community to give back to itself.

What it does

Applies a knowledge representation approach to deliver enthusiastic volunteers to nonprofit organizations

How we built it

We separated into groups and approached backend and frontend separately. We used Firebase to create a front-end where users can sign up for opportunities presented by organizations. In the backend, we apply a knowledge representation approach to deliver reliable volunteers to an organization's event in the scenario of a possible no-show.

Challenges we ran into

Implementing a web application without a server. Working with Firebase wasn't exactly fun. Implementing machine learning without real-world data.

Accomplishments that we're proud of

Learning and implementing new technologies, and applying them for a good cause.

What we learned

We need a lot more help and less experimentation! The backend finished well before the frontend, but by that time, the backend had no idea what is happening

What's next for URO

Successfully integrating the backend with the frontend. Currently, both work well in isolation.
