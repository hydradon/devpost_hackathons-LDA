Inspiration

Inspiration from the flash game "Shift"

What it does

A map consisting of black and white tiles where a white character can move on black tiles and a black character can move on white tiles. The characters are able to use each other as stepping stones to reach their respective goals.

How we built it

Built in Java using Jframe and Jpanel for the screen and the Java awt library to create all of our graphics and images.

Challenges we ran into

-Not understanding with learning the libraries. -Conditions concerning interactions between characters on the same tile -How to read from a file

Accomplishments that we're proud of

-We finished the project in time -Learned how to code in Java
