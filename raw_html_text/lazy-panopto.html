Inspiration

We are students that don't like to go to class. Also, sometimes it is difficult to create notes for lectures when professors talk quickly or incoherently. We wanted to create a way to transcribe their lectures on Panopto and summarize them or be able to translate them.

What it does

Transcribes the speech of professors on Panopto and transcribes the images/slides of Panopto into text, with features available such as summarization and translation.

How I built it

Challenges I ran into

We ran into the issue of incorporating all of our features together(i.e. speech-to-text, translation, summarization, and image-to-text). Also, we ran into the issue of accessing our server through https.

Accomplishments that I'm proud of

Successfully integrating the various APIs into our product!

What I learned

We learned a lot about the various google apis, namely Speech-to-Text and other Cloud Intelligence Apis. Also, we learned about using Flask as a framework for web development.

What's next for Lazy-Panopto

To create a nicer interface and integrate a browser extension for our project!
