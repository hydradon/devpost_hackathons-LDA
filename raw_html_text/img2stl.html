Inspiration

Multi-Color 3D Printers are becoming more common. We need a tool to create multi-color 3D Prints.

What it does

IMG2STL transforms a color image into an STL for 3D-Printing.

How I built it

The following were used: Flask, Python, HTML, CSS, and JavaScript.

Challenges I ran into

Working on front-end can be troublesome. Some attributes could have been cleaner. Additionally, the STLs created are a bit pixelated; more smoothing would help.

Accomplishments that I'm proud of

The fact that we have a MVP.

What I learned

Image Processing is fun and useful.

What's next for IMG2STL

Working to make the product great and viable for commercial purposes.
