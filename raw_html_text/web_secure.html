Web Secure

A easy way to test web applications for vulnerabilities

What it does

Tests web apps for vulnerabilities and exploits them whilst educating the end user on what it is and how to prevent it the following modules are included: CMS Mapping Directory Scanning Nikto Integration Form Finder Link Spider Verb Tampering Command Injection XSS Injection SQL Injection SQLMap Integration

How we built it

We built it in a modular format

Challenges we ran into

Interacting with websites, legal issues, formatting of results

Accomplishments that we're proud of

A working prototype
