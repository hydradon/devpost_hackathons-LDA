Inspiration

We were inspired by the popular #TrashTag challenge that motivates people to clean up trash around their community and take before/after pics.

What it does

EcofyMe allows registered users to view a map of their community and join or host clean up events that on locations of trash that is posted by other users on the map.

How we built it

We used Google's Firebase as our back-end solution for storing the pictures taken by users and managing the user base. Bootstrap was used to streamline the front-end development and the Google-maps JS web API was used to display the map and pin markers.

Challenges we ran into

The biggest challenges we faced were developing the back-end and hooking it to the front end and managing the user authentication tokens.

Accomplishments that were proud of

This was the first hackathon for most of us and we are proud that we were able to plan can complete a whole and working project.

What we learned

We learned how to use the Google-maps API, get location with the Geolocation API, and upload photos to Firebase

What's next for EcofyMe

Adding a point system to encourage clean ups, a system to curb abuse on the app, and streamlining the user experience.
