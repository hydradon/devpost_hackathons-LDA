Inspiration

We were inspired to create this web app as we were in search of a platform where people who do social good can interact and can come together to share their acts of kindness. The absence of that motivated us to build one of our own. We believe that those who work towards the betterment of society should feel proud about themselves as well as inspire others.

What it does

It is a social media platform where users can share updates about what they are doing to better society and can interact with like-minded individuals. For example, if you maintain a compost pit at your house you can urge others in your community to do the same.

How we built it

We built it using XAMPP to host a server, and used HTML, CSS, JavaScript, BootStrap and PHP to script the web app and used SQL to store data.

Challenges we ran into

We had many issues firstly connecting to XAMPP successfully as none of us had used it before. Additionally, we struggled with scripting in PHP as we have not used it before. While we didn't have the time to implement all the functionalities we would have liked to, we plan on moving forward with this idea and improving on it in the future.

Accomplishments that we're proud of

Successfully connecting to XAMPP and hosting our website locally, using SQL databases that update dynamically and encrypting passwords.

What we learned

Learning basic PHP, using SQL to store and query data.

What's next for En route humanity

Add more features and functionality to the website.
