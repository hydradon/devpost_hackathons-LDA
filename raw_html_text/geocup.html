Inspiration

We wanted to do something fun, and the Paulig Muki was such a fun concept that we couldn’t resist.

What it does

The application records the user’s location, and creates a map that shows where the user is drinking the coffee right now.

How we built it

The application is based on the MukiTestApp, and uses the Google Play Api to get the location data, the map is created with the Google Maps Api, and finally the maps is sent to the muki with the Paulig Muki Api.

Challenges we ran into

The MukiTestApp was not compatible with newer versions of Android, so first we had to make the app compatible with newer phones. Creating the app was overall quite hard, since nobody of us had any experience with Android development. Getting the Paulig Muki Api to work correctly was also tricky.

Accomplishments that we're proud of

Getting the application to work, with no experience with Android Development from before.

What I learned

Android Development and how to use different APIs.

What's next for GeoMug?

Being able to send the map to your friends, and log different data, such as how far someone walks while drinking coffee etc.
