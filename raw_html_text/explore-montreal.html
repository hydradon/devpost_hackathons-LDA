• Explore Now!

Inspiration

Living in Montreal, it is easy to notice that tourism is a booming business for the city. We wanted to facilitate tourists in discovering Montreal, and help them navigate the roads smoothly from one attraction to another.

What it does

It allows the user to decide which locations they wish to visit and creates a fluid navigation to go through all of them without wasting time. The route is a round-trip to the hotel that the user selected.

How I built it

We built it using Javascript and the Google Maps API.

Challenges I ran into

It was difficult implementing the GoogleMapsAPI with various Montreal Events, as well as finding and using data sets on the city's many events.

Accomplishments that I'm proud of

We're proud of the website design and the fluidity.

What I learned

We've learned a lot about git and the Google Maps API

What's next for Explore Montreal

We wish to branch out and take in various data sets to keep up to date with the various festivities of the city.
