Inspiration

A hat website.

What it does

It's a website that contacts known associates to provide domestic abuse victims with services to quickly escape an abusive situation.

How we built it

We started with an html skeleton, then got help with javascript. Under mentor advice, we created a javascript server that could interact with a client and send messages by text and email, using many APIs including hapi, twilio, and nylon.

Challenges we ran into

We didn't know any javascript coming into this project.

Accomplishments that we're proud of

We are proud of our accomplishments with javascript, such as creating a server and automated messaging.

What we learned

We learned how to use javascript with Node.js, along with the following APIs and their associated functions: hapi, twilio, nylon. Respectively, we learned how to create a server in javascript and interact with a client, then send automated text messages, and send automated emails.
