Inspiration

We often come across a lot of plain tabulated data in websites and it is hard to get meaningful insights by plainly looking at them. We wanted an easy way to quickly generate insights from tables and present before the users.

What it does

It is a Chrome plugin tool that automatically detects tabulated information in a website and offers a button. Tapping on the button would launch a playground with insights that our engine generates. Additionally, the playground also provides a voice interface powered by Amazon Alexa to which the user can ask questions pertinent to the data in the table, and Alexa can respond with responses. This makes it easy for users to get friendly responses to their questions.

How we built it

Node.js, Alexa. Data sourced from Techcrunch/Crunchbase.

Challenges we ran into
