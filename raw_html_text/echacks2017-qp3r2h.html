Inspiration

On the bus ride here we saw the full moon and thought it would be cool to know when the skies were clear.

What it does

Accesses weather data and provides rating of viewing experience and current temperature.

How we built it

Uses java to access openweathermap api and then is processed and displayed in a java form.

Challenges we ran into

initial plans were to implement as an android app however when the backend was implemented the app continually crashed and the error could not be found.
