What it does

Our app stores the users' cars' information and lets them create new trips where they can get information about how many gallons of gas they need and how much it would cost them.

How we built it

We built the backend first using Java and used Google Maps API and Shine API from Liberty Mutual Insurance. We then converted the app to work with Android Studio.

Challenges we ran into

Making the API's play nice with our code and then finally implementing the whole thing in Android studio was a great challenge that we overcame.

What we learned

We got to know about Android Studio to a whole new level and deepened our knowledge about API implementation too.

What's next for Gastimate

We want to be able to give the user various safety resources based on where and when he/she is going on their trip.
