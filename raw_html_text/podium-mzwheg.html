Inspiration

The desire to meet people around the world to discuss technology.

What it does

Allows you to find new meetups and conferences, to visit in the future, and watch archived footage.

How we built it

The backend is written in Go, and it serves the site from HTML rendered with Go's built in templating. Data is loaded from postgreqsql. We don't use any JS frameworks or CSS presets.

Challenges we ran into

Populating data is slow! We plan to let users help by eventually turning this into a wiki. In the meantime, we made pages that allow us to do that more easily.

Accomplishments that we're proud of

  • Sleek design
  • Well designed database

What we learned

CSS Grid, Golang.

What's next for Podium

Adding more content, allowing less popular categories to be more easily browsed, and allowing event organisers to manage their own data.
