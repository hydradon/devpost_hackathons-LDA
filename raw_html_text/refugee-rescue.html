Inspiration

Inspired by recent events, and we thought of what we could do to help.

What it does

Is an interactive adventure-style game that takes the user through the struggles of being a Syrian Refugee

How we built it

We used Unity, Adobe Illustrator and After Effects

Challenges we ran into

We tried to use the Gear VR in the beginning, but it was difficult to learn how to do it.

Accomplishments that we're proud of

We learned how to use unity and attended the design workshop to learn about adobe illustrator. We are happy that we were able to pull the project together.

What we learned

Unity and Adobe Illustrator

What's next for Refugee Rescue

We hope to make more chapters in the game and make it available to the public, so that we could fund money for the refugees.
