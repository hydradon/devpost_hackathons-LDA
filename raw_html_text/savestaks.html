Inspiration

Many people, including ourselves have trouble achieving long term financial goals due to their inability to know if we are on track or not. We wanted to build a tool that showcases our daily savings after purchases and guides us to saving more money.

What it does

SaveStaks displays progress of users financial goals by keeping track of what the user purchased during the timespan of their saving periods and their financial statistics such as salary and expenditures

How we built it

We used Capital One API for user accounts to access their balances and other account information. We host the site on AWS using php, MYSQLl and Javascript. We display beautiful graphics with css and the help of plotly.

Challenges we ran into

We ran into several javascript issues parsing the Capital One Nessi API.

Accomplishments that we're proud of

Building a working prototype and learning new technologies.

What we learned

How to discover bugs, the nessi api, and several javascript techniques.

What's next for SaveStaks

Continue to build and think of ways to help people save money and improve financial habits.
