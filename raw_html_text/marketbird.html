Inspiration

My good friend always talks about stocks and put me up to this challenge.

What it does

Sentiment analysis of tweets to gauge twitter feeds. User enters a ticker, and ultimately an algorithm will give a recommendation to the user whether or not to buy, stay, or sell.

How I built it

Using java and the twitter4j API as well as Stanfords NLP library to process sentiment analysis on tweets.

Challenges I ran into

Importing libraries, learning about paths, working individually was tough.

Accomplishments that I'm proud of

Not much. As fun as it was I wish I had team members. I would have been able to go much further with this algorithm.

What I learned

This was my first time using an API outside of Java's libraries. I learned how to navigate through external documentation which was pretty cool,

What's next for MarketBird

Hopefully a prize. Then maybe a job with Twitter!
