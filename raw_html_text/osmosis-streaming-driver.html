The Osmosis Streaming Driver demonstrates how to expand Brizo, and go beyond static data files. It enables it to download live data from WebSocket streams. There is a lot of value in being able to download live streams, and there are entire projects dedicated to this, such as the streamr.network . There is no reason why we could not use Brizo for this purpose, and this project demonstrates how we can leverage existing WSS APIs to provide this data to the end-user.

This is an extension of the osmosis-driver-interface . It can easily be set up to work with Barge. Please refer to the introduction video and the README for more details.
