Inspiration

We wanted to create a way to create space and make information more interactive.

What it does

A mobile application that scans for a marker and uses augmented reality to show a list of information like a bulletin board.

How we built it

Unity + Vuphoria

Challenges we ran into

Trying to export assets from Maya to Unity.

Accomplishments that we're proud of

Setting up Vuphoria to work on Unity and deploying it on our phones.

What we learned

Basics of Unity and Vuphoria.

What's next for Vued

Adding button features so that it takes you to an external site or event page.
