Inspiration

We read a case study highlighting how an online-record company made a simple chatbot to sell vinyl records. They combined a conversational interface with a simple reccomendation engine to great effect, and this inspired us to make our own.

What it does

Canna is a conversational bot that acts as the intermediary between the Dispensary and the Patient. Ordering Cannabis is only a few taps away! Leveraging on the modern technology of Twilio and Smooch, we can continue to progress towards anti-harassment technology by implementing a strike system to block toxic users.

How we built it

Teamwork, love and sweat equity.

Challenges we ran into

We encounter obstacles with insufficient resources to implement Front-End code for Canna. Further development of Front-End UI will be a plus.

Accomplishments that we're proud of

  • A functional build in under 24 hours!

  • Fully configured backend for Dispensary

  • Database => Flask Stack => Jody's database => Webflow

  • First time building a Flask stack

What we learned

We learned to implement code for Twilio and Smooch along with the first attempts at constructing a Fully Flask-based stack with Postgres, Nodejs/Express and Webflow.

What's next for Canna

Focus more on improve Natural Language Processing system, integrating workflow into a single server, and eventual monetization of Canna.
