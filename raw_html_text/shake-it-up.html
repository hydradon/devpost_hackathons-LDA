Inspiration

Too many trips to Deet's for our weekly Milkshake Monday have resulted in indecisiveness.

What it does

This app will rid you of the stress of having to decide on what type of milkshake you order. It will present you with a random combination of two or three flavors to create an entirely new flavor never before seen by human eyes.

How I built it

We built this app using Android Studio.

Challenges I ran into

Learning Android Studio and implementing Java code within it.

Accomplishments that I'm proud of

Although simplistic, our team is proud that we were able to learn some of the basics of Android Studio, and that we were able to produce a final product.

What I learned

Allocate more time for brainstorming ideas, and do not try to rush into it. Set small goals that are feasible. Don't try to tackle too many problems at once.

What's next for Shake It Up!

Include save states to allow users to save their favorite flavors. Also, implement a "Tweet this flavor!" feature.
