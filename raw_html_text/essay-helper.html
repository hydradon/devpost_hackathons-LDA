Inspiration

We wanted to make a program to warn us when we use too much of a single word

What it does

reads an input text file completely, makes a list of all the unique words used, and sorts them by how frequently they are used. A link to a thesaurus entry for that word is displayed next to it.

How we built it

We used parallel vectors to store the strings and the amount of times the words were used. We sorted them using selection sort and made sure to ignore punctuation marks next to the words.

Challenges we ran into

logic behind filling in the vectors with info from the text file

Accomplishments that we're proud of

We made a working program

What we learned

How to use and upload to github
