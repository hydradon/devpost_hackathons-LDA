Inspiration

Kyber Network is decentralized autonomous smart contract based exchange. TokenSets is decentralized autonomous token aggregation tool. Why not make them friends?

What it does

Allows users to buy/sell TokenSets for ETH in a single transaction. Decentralized.

How I built it

Web3-based Dapp with a smart contract allowing to perform multiple exchanges in single transaction. Build with latest Angular and Web3 versions. Deployed on IPFS under:

https://gateway.ipfs.io/ipfs/QmaGAxWTC1Eqe65HPZsGJUch4vMBWgMHRmH9gGXh9rdFU9/

Challenges I ran into

Debugging transaction in main net with parity traces. Get it running with Angular, PrimeNG and Web3.

Accomplishments that I'm proud of

It works as planned with small difficulties.

What I learned

Parity traces logs every external call inside transaction. How to working with Angular and latest Web3 version.
