Inspiration

As someone who does long distance travel a lot, we want to help improve the quality of long distance travel.

What it does

Proposes a sleep schedule based on the flight departure and arrival time.

How we built it

We used American Airline's FlightEngine API to generate flight data. We used React for the front-end of our application

Challenges we ran into

We had to modify the flight engine API in order to accommodate international travel. Also most of our team didn't know javascript or React.

Accomplishments that we're proud of

We have a semi-working website :)

What we learned

We learned JavaScript by working on this project. We improved our technical skills.

What's next for Rejuven8

Do more sleep research and come up with better algorithms to propose sleep schedules. Add layover times and multiple flight connections support. Also make it a mobile app because it is for travelers.
