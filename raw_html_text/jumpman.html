Inspiration

Our inspiration came from our innovative minds. We wish to create and spread happiness through the beauty of games in this 21st century. The main game mechanic came from Mario: people jump on the heads of others to deal damage.

What it does

This game runs a simulation of two people fighting it to the death for their lives. It simulates artificial violence and thrill which is what most children get entertained on these days.

How we built it

We built it using Godot, a game engine which uses gd script.

Challenges we ran into

There were a lot of challenges to run into. We were planning on adding a lot of power-ups to add individuality and flavor to the game. Most of the power-ups could not make it because of problems with Github (we could not log into Github for 3 hours). However, the most fundamental power-up -the ANCHOR- was able to make it into the game.

Accomplishments that we're proud of

We are proud of finishing the game in the 24 hours we have been given. We have a working game and something to turn in. That's good enough for us. Baby steps.

What we learned

We learned to plan ahead and prepare everything that is involved in making a game. That way, we can get the fundamentals down first before adding on special effects like power-ups.

What's next for jumpman

The next thing for jumpman is that we are taking it ONLINE! Yes, we are taking it ONLINE! We will include network features! You will be able to play for FREE!
