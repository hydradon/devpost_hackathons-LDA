Inspiration

Many high school and college students (including myself) have had the need of getting quick information about the elements in Periodic Table while doing homework or studying for Chemistry exams. What a better way of using Alexa-enabled devices that using it to keep learning and have better performance at school/college.

What it does

PeriodicPal is an Alexa skill that gives the user information about the Periodic Table. The user can ask for things like: Atomic mass or number of an element, search elements by category/classification, get information of an element by giving its atomic number or make Alexa dictate the elements before of after a specific element.

How I built it

This Alexa skill was built using the Alexa SDK for Node.JS and hosted using AWS Lambda.
