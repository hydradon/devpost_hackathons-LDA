Inspiration: We were introduced to bose's new speakers and we were thinking about programming and playing music all without having to pay attention.

What it does: It will select a song from spotify based on your "tempo"

How we built it: we build it as a atom package, using javascript, coffeescript and atom

Challenges we ran into: The api we wanted to use was discontinued by spotify

Accomplishments that we're proud of: writing my first atom package

What we learned: making comprehensive api calls

What's next for get-w-p-m: making a better algorithm for tempo detection
