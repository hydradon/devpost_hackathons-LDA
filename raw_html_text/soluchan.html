Inspiration

We wanted to create a hands-free shopping experience for people, especially t-mobile users. Alexa seemed like the perfect platform to build something like this.

What it does

Use Soluchan on Alexa to create shopping lists for recipes, optimize the price of your shopping cart. If you are a T-Mobile user, use it to create & manage your cell phone lines, to search new devices and get deals for new devices.

How we built it

We built an Alexa skill which is hooked up to an AWS-lambda instance. All the backend processing for the skill is done through a T-mobile Jazz serverless API, which is connected to the lambda instance.

Challenges we ran into

Working with AWS Lambda, scrapping shopping websites, working with Jazz Serverless.

Accomplishments that we're proud of

Having a working prototype at the end of 24 hours.
