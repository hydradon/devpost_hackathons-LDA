Inspiration

We wanted to give people a way to communicate with bystanders and get the help they need when they suffer a cardiac arrest.

What it does

It allows people at risk of heart-attack to be able to store their contact information into a QR code. Once the QR code is scanned, then a set of locations are returned to allow the user to find the nearest AED's around their location.

We built it using Python, Flask and Javascript
