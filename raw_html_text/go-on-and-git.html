Inspiration

Looking at the scenery around Houston, and on the bus ride inspired us to create this app.

What it does

Dynamic image analysis to determine whether or not the plant itself is of a poisonous nature.

How we built it

We decided to use android studios, integrated with CraftAR

Challenges we ran into

One of the greatest challenges we faced was the implementation of the craftar SDK. The challenge came from using a technology that we were unfamiliar with, a technology with little documentation except for a class reference and a slim example.

Accomplishments that we're proud of

We persisted through the struggles, and continued working on the project despite being largely unfamiliar with CraftAR, and Android Studios.

What we learned

How CraftAR can be used with Android Studios for image recognition, and augmented reality.

What's next for ARBotany

Expanding the library of plants that the app recognizes, and making it more user friendly
