Our aim was to create an app that allows the user to navigate anywhere with advanced warnings of the hazards roads and the likelihood of an accident occurring with them while allowing the user to select from a different route.

We have achieved this by using android studio to make an app that shows a heat map of England for the user to acknowledge areas of high risk. We have faced countless dead ends and loops. we have overcome the majority of issues by resolving them in a methodical manner and found ways around the majority of bugs and errors that we faced.
