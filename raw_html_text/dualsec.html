Google currently has two step verification where a SMS code is sent to your phone. However, we wanted to automate that process to where the authentication is seamless and does not require any input. As long as your mobile device is nearby, the two or three step authentication would automatically validate your login attempt.

We used a number of services including AWS Lambda, AWS DynamoDB, Google Maps API, AWS API Gateway, Twilio and Python to create this app.
