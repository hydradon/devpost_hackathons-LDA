Inspiration

The lack of solution to that kind of pain.

What it does

Simulate the missing hand by showing the mirror image of the real hand. Then, playing the virtual game will help the pain, in the short term. In the long term, we collect data to analyze and improve the user experience.

How we built it

VR, using Leap motion (hands recognition). The platform created in Unity.

Challenges we ran into

Integration between the unity and the Leap Motion.

Accomplishments that we're proud of

Succeed to create the mirror image.

What we learned

There is no satisfied treatment for the kind of pain.

What's next for Phantom relief

Wide storage of data for future improvement.
