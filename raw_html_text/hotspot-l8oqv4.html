Inspiration

When travelling to places it's sometimes hard to find a connection, making your travelling harder.

What it does

It allows consumers to give up their Wi-Fi for consumers to use in return for a monetary incentive.

How we built it

Using an IOS app and then IOTA for the backend. We used an RPI to emulate a Hotspot.
