Inspiration

  • Little Caesars challenge

What it does

First a user will sign up with their email on our website. Then before a football game, the webapp will send you an email asking how you are feeling. From that email the webapp will order a pizza (theoretically from Little Caesars) based on the sentiment given.

How we built it

We drew the idea out first. Then we split up doing different parts withing the front and back end.

Challenges we ran into

  • Little Caesars doesn't have an API to order pizza
  • We couldn't find a free API to get upcoming game times so we had to use a work around
  • Trying to mix flask with default HTML

Accomplishments that we're proud of

Actually making something that mostly works since this is our first hackathon that we have been too.

What we learned

A lot about front end and back end web development.

What's next for Mood Pizza

  • Actually ordering a pizza and not just reccommending one to the user
