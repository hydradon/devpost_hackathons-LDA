Inspiration

We like sneakers, a lot.

What it does

The Price Is Right is a game that utilizes the StockX API. Two players compete against each other for points by correctly selecting the last sold price of the sneaker.

Challenges we ran into

Originally, we wanted to use the Facebook Instant Games Platform to create 'The Price Is Right', but had trouble with the documentation and demo as they were outdated and the demo did not work.

Accomplishments that we're proud of

We improved our mint throwing skills.

What we learned

  • React
  • Creating mockups using Sketch
  • HTML5

What's next for The Price Is Right

Allow multiple games to be played at once with more than two players, different game modes, and implementing it as a Facebook game.
