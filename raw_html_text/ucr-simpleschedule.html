Inspiration

We were tired of the cluttered class schedule on our R'web. It was difficult to easily see the class instructors, locations, and meetings days/times all at once.

What it does

The user inputs their class and relevant info. Our tool generates a table with important class information clearly displayed. Rows can be deleted, and the table can be cleared. It can also be downloaded as a pdf file.

How we built it

We used HTML code and javascript, using the jquery library for downloading.

Challenges we ran into

It was really difficult figuring how to let the user download the table as a pdf.

Accomplishments that we're proud of

Making a web interface with intuitive options (like buttons and time selectors). Exporting the table as a pdf.

What we learned

Learned a lot about javascript options for interactivity and integrating it with html.

What's next for UCR SimpleSchedule

We want to be able to edit cells in the table. We also want to be able to export the table in more formats (image, csv).
