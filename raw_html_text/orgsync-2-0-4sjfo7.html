Inspiration

The current design for the UC Davis Clubs and Organizatios search tool was poorly designed and difficult to use. ( http://ucdavis.orgsync.com/search ) we decided to make it easier for users to use.

What it does

It provides an organized list of clubs and organizations on campus.

How I built it

We built it using HTML, CSS, Javascript and Python.

Challenges I ran into

We had a difficult time scrapping through data of the various clubs and organization that were already present on the old search tool.

Accomplishments that I'm proud of

We have never created a website before or combed through data. It was a difficult challenege but it was enjoyable.

What I learned

Picked up a lot of web development skills.

What's next for OrgSync 2.0

We plan to integrate a social aspect to the search tool. We plan to present which clubs are the most popular, how to get involved, who to contact and also which clubs you could be interested in based on the people you know (facebook integration maybe?) and the other clubs you are interested/subscribed to.
