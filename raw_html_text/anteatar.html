Inspiration

Tamogochi + AR

What it does

You can feed your anteater oranges and he will follow you wherever you go.

How we built it

Through Unity using the ARkit and Xcode.

Challenges we ran into

Platform switch from ARKIT and SCENEKIT using Xcode to using Unity in C# using ARKIT.

Accomplishments that we're proud of

Having a functional AR animal that follows the user around. Also, it is one of the first AR apps we have made.

What we learned

Look into platforms before you start coding and also look at lots of tutorials in anything we have questions about.

What's next for AnteatAR

A bit of debugging and adding more features to make it more life-like.
