What Inspired me? Once I was wondering that how about a portal where people could help each other request help and other willing to help can help if they want. Many times it happens that people want to help but don't get the way or idea where and how to help, So, it inspired me to give requirement to them and if its possible then they can help.

What it does? It just provides a portal to various NGO's to list a requirement for helping and the requirement will be provided by Alexa when user will ask for the requirements nearby him.

How I built it? I built it using various technologies like laravel, nodejs, aws lambda, Alexa skills kit, etc

Challenges I ran into? Giving nearest possible requirements from NGO's
