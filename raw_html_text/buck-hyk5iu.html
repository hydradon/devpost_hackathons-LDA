Inspiration

I used to have class from 11:00 till 2:30, so I will miss the school cafeteria lunch time. Only if I can just pay a few bucks, and have someone who is not busy and pick some food for me.

What it does

A buyer could post a task, like picking up laundry or run down to the walmart to pick up some tools. The seller will bid on the task, the lowest bidder wins and will do the job for the buyer.

How we built it

We used facebook api, so user could login with facebook. On the backend, we are using mongodb, and for the frontend we are using android.

Challenges we ran into

Initially, we were trying to use microsoft Azure server. We are able to write to the server, but we cannot read from it. Therefor we gonna switch to the mongodb at the last minute.

Accomplishments that we're proud of

We used a good design pattern in the backend

What we learned

How to connect java with facebook api. Also we learned how to setup a mobile service in Azure.

What's next for Buck$

Connect with paypal, and get ready to have a easier life.
