Inspiration

Two weeks ago, during a lunch, one of our team said a rough idea very similar to what we have build it during the hackathon.

What it does

It's a funny and quite safe app for making jokes with friends. Both agree to send a buzz which basically sends a funny sound to each other and from 0 to 6 hours you receive the buzz. The friend decide the time but you don't know it and viceversa.

How we built it

We built it using a server written on python using flask and with an android app using google cloud messaging and sign in provided by Google.

Challenges we ran into

We managed for the first time to play with the google cloud messaging which was a little confusing at first but then it was quite easy.

Accomplishments that we're proud of

We are proud of having finished the app in less than we thought. Also, we have had time for sleeping, enjoying the hackathon with others hackers around the world and the attendance to great tech talks

What we learned

For some members we have learned python and flask. For others we have learned a little bit of android.Finally, Google cloud messaging is new for all of us.

What's next for BuzzerApp

Definetaly, we want to finish it with proper interfaces and designs. Then we are going to publish it in the play store.
