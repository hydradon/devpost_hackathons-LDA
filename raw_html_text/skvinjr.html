How it works... skvïnjř has two main facets; an interface for creating scavenger hunts, and a separate interface for going on the hunts. When creating them, users can utilize text and picture clues. During the hunts, in addition to the clues, users see a compass direction displayed, helping them find the mark. When the proper tolerance of latitude and longitude to the set point are met, there is a slight vibration of the phone and the direction changes to a congratulations, also revealing a link to the next clue.

Challenges I ran into... we had some issues implementing the back end for it, so it is not fully functional.

Accomplishments that I'm proud of... we created all our own graphics, layouts, and designs. It was also our first time working on anything mobile, so we are really proud of how far we did make it.

What I learned... we learned a lot about the basics of android development. It was fun to be able to make changes and see them take place on a real phone almost immediately. The design and development coming together was really rewarding as well.

What's next for skvïnjř... with more development, this could become a really neat platform. Unfortunately, we are not quite ready for that right now, so this will probably end up getting shelved away..
