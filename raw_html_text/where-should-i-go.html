What it does

A neat way to explore all the places around you or in the city of your choosing by showing you random places from that city using the Google Maps API. This web app can show you places nearby that you might not have found on your own.

How we built it

This web app makes use of the Google Maps API along with the usual stack for a small web app with html5, css, js, jquery, and bootstrap with git being used for collaboration.

Challenges we ran into

The two of us knew nothing about this idea or how to use the Google Maps API going into it, so it was a learning curve to get things working properly.

Accomplishments that we're proud of

We are happy with what the app can provide! We were surprised at how we used this app to find restaurants or stores we hadn't noticed before that we are excited to try out next time we hit the town!

What we learned

We learned how to collaborate with Git, how to use the Google Maps API, and how to setup a basic website for rapid prototyping.

What's next for Where Should I Go?

We will continue development to improve the current design and add features, like showing you how to drive to that location from where you are now.
