Inspiration

Our inspiration was the ways that both music and weather can influence mood. We wanted to create an application that was a basic model of how that relationship can be effected.

What it does

It finds the user's location, gets the weather in that location, and then plays a certain artist's music based on the weather.

How I built it

We used two APIs, one to get the weather, wunderland, and then the soundcloud API to get the weather and music.

Challenges I ran into

A couple of little problems involving figuring out bootstrap, we also ran into a lot of problems deploying the webpage using git APIs.

Accomplishments that I'm proud of

This is our first hackathon and we had to learn a lot about how to use APIs and host websites that none of us knew how to do previously.

What I learned

We learned how to use azure to host websites and also how to use APIs and better how to use CSS.

What's next for Climatracks

We want to create a user authentication system to allow for personalization of the music played during different times and with different weather.
