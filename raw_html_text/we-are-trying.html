Inspiration

We are building Pixels to Palettes because we wanted our project to be a marriage of art and computer science, both being subjects that we are interested in.

What it does

It generates a color palette from a single color and saves time for artists and other designers when they are looking for inspiration.

How we built it

Using HTML and director.tjhsst.edu!

Challenges we ran into

Learning how to build a web app with only knowledge of Java under our belts.

Accomplishments that we're proud of

The display!

What we learned

To code with HTML, CSS, and JavaScript!

What's next for Pixels to Palettes

We hope to create more formulas for generating the palettes, perhaps add an editing feature for existing palettes, and make a way to save the palettes created.
