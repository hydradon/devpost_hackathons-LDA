Inspiration

Looking forward into the future, we asked ourselves what seems to be missing in the world of e-commerce that we can build utilizing Ebay’s API. After digging deep into the trends of the industry, we realised that we can do something much more than that. We realised we can build a social enterprise that provides value for both enterprises and consumers through our unique value proposition.

What it does

Pandora Boxs provides users with a mystery box that contained ten weekly items that pledgers can win when they pledge. Pandora Boxs aim to create a sustainable business through automation, gamify e-commerce while giving back to society.

How we built it

It was built with a react boilerplate and firebase backend. We utilized eBay’s Browse and Order API to get the data we required.

Challenges we ran into

We faced a few problems - Resolving OAuth authentication, providing web parameters for API calls, but managed to receive help and advice from the eBay engineers.

Accomplishments that we're proud of

To be able to integrate both the buyers' and sellers' point of view (POV) into our project within 2 days.

What we learned

To integrate and navigate our way through the problems we faced by trying out different methods that we found online or learn from one another through our past experiences with the stacks.

What's next for Pandora Boxs

To be able to integrate more verticals, bigger and better bundles, and better partnership programs
