Inspiration

The Amazon Echoes can do many things, but I was shocked to learn they did not provide watches and warnings for severe weather. As a meteorology I sought out to change this.

What it does

The Amazon Alexa provides current watches, warnings, and advisories when prompted by the user. The data comes straight from the National Weather Service and is updated about every 2-3 minutes.

How I built it

I created the script in python and transferred it to Lambda to integrate it with Alexa.

Challenges I ran into

Since I had never used Lambda or much of Amazon web services, it was a challenge to get the script working.

Accomplishments that I'm proud of

I was able to prevail with my code in the end and test it out against various counties.

What I learned

I learned a lot about AWS and coding in general while enjoying the process.

What's next for Weather Warn

The next steps would to be integrate it with counties all over the US.
