Inspiration

We were inspired to help vision impaired persons to navigate the world a little easier

What it does

This app is to help vision impaired people recognize objects, text, locations, etc.

How we built it

We built it using android studio, google cloud platform, and google vision API

Challenges we ran into

We had problems trying to figure out the voice command, how to use the api, and learning everything pretty much from scratch

Accomplishments that we're proud of

We got text recognition working, and voice commands working

What we learned

We learned how to use Google's API We learned how to develop an app in android studio We learned how to use GCP We learned how to do voice commands in an app in Android

-Object Recognition -Faces Recognition -Walking GPS Assistant -Currency Recognition
