Inspiration

Agar.io / Pico Park / Nidhogg

What it does

Control your popsicle to push and survive in the physics world!

How we built it

Node.js + socket.io + HTML5 multiplayer game running on Microsoft Azure.

Challenges we ran into

Adding physics to the game was a really hard part. There were many good client-side game & physics engines, but many of them did not work well in the node environment. Synchronization between the clients was also hard and important parts.

What we learned

JS Rules!

What's next for Popsicle Survival

Polish the design and gameplay.
