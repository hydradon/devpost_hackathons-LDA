Inspiration

I was inspired to build something that I know a lot of people would like to have for fun.

What it does

Build a pokedex, such as the ones seen in Pokemon. It would be an app that would see an object and identify whether it is a pokemon and describe the information about them. The Google cloud text to speech api would also tell the user the information.

How I built it

Python using Eclipse

Challenges I ran into

The python I was using was too updated and one of the imported libraries did not have that update. So I would either have to reinstall a past version of python (which would take even more hours to do) or wait for an update. Unfortunately I did not have enough time to finish the project due to this.

Accomplishments that I'm proud of

I was able to at least get the bing search api to work.

What I learned

I learned a lot about AI and data analytics.

What's next for Pokédex
