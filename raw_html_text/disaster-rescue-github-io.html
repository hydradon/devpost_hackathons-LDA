Our product

We offer a all-in-one disaster preparedness + relief tool. People can access free information about potential and ongoing disasters as well as giving rescue teams and aid organizations to assess needs for each specific event using Google analytics and location data.

The advantage

We have a working implementation of an IBM Watson natural language processing data pipeline that analyses real-time Twitter data to predict disaster damage better that the Federal Emergency Management Agency (FEMA).

Technology

We implemented our product with IBM Cloud technologies as well as our own Python data analysis pipeline which utilizes IBM Watson NLP.
