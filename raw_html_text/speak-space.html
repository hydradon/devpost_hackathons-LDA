Inspiration

Wanted to make a place where people can put their inspirational quotes and just say funny things to each other.

What it does

Allows the user to enter a quote and have it randomly placed on a large canvas, its accessible to everyone, they can all see it and post their own "quotes" or just fun things.

How I built it

Used JS and HTML to create a script to take words from a custom input box and add a random font and random color and then place it randomly on the canvas

Challenges I ran into

The domain.com host wouldn't let me transfer my domain so I couldn't fully implement the "Live" feature of people seeing the text as it is typed into the site. I also wanted to store the data and visualize it see what phrases/words were used most often by internet users exploring the site.

Accomplishments that I'm proud of

Self-made scripts and simple coding solutions, not proud of being unable to finish being staled by a 60-day standby date to use my website ;(

What I learned

A lot of Javascript and HTML and firebase database setups.

What's next for Speak.space

Finish adding full firebase database integrations allowing for fully "Multiplayer" connectivity allowing community members to have fun and communicate and boost each other and stay positive in life. Possibly leave running 24/7 so everyone gets a chance to try it.
