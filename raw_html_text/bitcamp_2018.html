Air DJ

Leap Motion Hackathon Bitcamp 2018 by Pranav Maniktala, Saajid Farooq, Sepehr Hajbarat, and Will Boodon

AirDJ is a unity based music development application using leap motion allowing users to make beats with their bare hands. https://www.youtube.com/watch?v=KBGZVMRyLz0

Within AirDJ is a hand sign based application called the meme machine

Built With

  • leap-motion
