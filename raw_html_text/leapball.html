Inspiration

Creating something fun with Leap Motion

What it does

LeapBall, uses the interactive hands of leap motion to manipulate the ball.

How I built it

Using Unity to create the game engine, Leap Motion as the controller and Blender to create all the Levels and objects.

Challenges I ran into

Manipulation of the Camera is still an unresolved issue.

Accomplishments that I'm proud of

-Getting Leap motion to work with the ball object. -Importing Blender objects into Unity whilst maintaining physics

What I learned

  • How to use Leap Motion
  • The Power of using Blender and Unity in unison

What's next for LeapBall

  • Refining Game Mechanics
  • Creating a Menu and level selection System
