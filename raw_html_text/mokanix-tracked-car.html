Inspiration

We wanted to map the air quality of the surroundings using an automatically controlled car that we can use over Wi-Fi and track its location.

What it does

A simple remote controller car that you can track with Google Maps

How we built it

We used a motor car kit and connected it to a raspberry pi, and hooked it up to a mokanix sim with a gsm dongle. Location data is collected via the mokanix API from an Angular webapp

Challenges we ran into

We don't have much hardware experience so building the car was quite challenging!!

Accomplishments that we're proud of

Being able to remote control the car

What we learned

Electronics basics, raspberry pi connectivity, mokanix API

What's next for Mokanix Tracked Car

Air quality sensors and mapping, data collection and storage, data analysis.
