Inspiration

An interactive game systems where students get points for getting the 'flag' by solving puzzles and minigames

What it does

Helps teach kids about cybersecurity through security games

How we built it

Firebase authentication and database storage, with a sanic server as the backend. Built modular pygames for different mini-games.

Challenges we ran into

Firebase, Javascript

Accomplishments that we're proud of

Live graph of scores! Network Builder minigame Server API fully built, so games can be developed on any platform!

What we learned

How to work with firebase, javascript, chart.js

What's next for Hack The Classroom!

More mini-games, Guess the hash, Common ports, More network builder content, Public key demo
