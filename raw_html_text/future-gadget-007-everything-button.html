Inspiration

We were inspired by lots of succsessful one-button projects

What it does

It is wireless button which can send requests to cloud service. Cloud service perform an event when click happens. Event can be adjusted by simple user-friendly constructor.

How I built it

We combined different hardware (esp8266, relays) and software (qt,c,c++,html) technoligies

Challenges I ran into

It was our first time building such complex solution, so we are proud that we succeeded working in team.

What I learned

We learned to combine technologes and work with specialists of different fields of knowledge

What's next for Future gadget #007 Everything button

We are going to develop project: attach gps and mobile internet modules, improve software and event constructor and make the final version of product.
