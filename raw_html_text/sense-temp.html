Inspiration

One of our team members won an arduino so we decided to use it and learn how to interface a temp sensor to the internet.

What it does

The arduino uses a temp sensor whose data is picked up by python and sent on to our sql database. The sql database is then updated on our website

How we built it

Lots of googling and tutorials

Challenges we ran into

The sensor not being installed properly. Mysql not being able to pick up the data it was being sent.

Accomplishments that we're proud of

learning something new for everyone

What we learned

The basics of the arduino and how we can use it in the internet of things

What's next for Sense Temp

Next we would like to build a full web interface that allows you to see the temperature and let you interface and interact with a virtual pet. We also want to include more than just a temperature sensor.
