Inspiration

This addin prevents you from making silly but disastrous email mistakes.

What it does

It checks that addressee of an email match the recipient. It warns if you reply to yourself, send to many recipients( 'reply all' potential hazard) or about to send to the wrong recipient.

How I built it

I build it using html, css and javascript. I tested it locally by running it on localhost , I started the server using gulp.js

Challenges I ran into

Understanding the Office365 API, properly catering for async tasks, designing the architecture of the addin.

Accomplishments that I'm proud of

It works as I expect it to. I'm already using it to avoid silly mistakes.

What I learned

Polished my javascript, learn to design from a user's perspective.

What's next for DoubleCheck

-Extending to email appointments. -Improve distinguishing between salutations and addressee name. -Extending to other languages when checking the addressee. It currently only works for English language.
