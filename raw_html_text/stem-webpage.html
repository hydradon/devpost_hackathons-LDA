Inspiration

We wanted to created a user friendly and child fun website which would encourage innovation.

What it does

The Website provides games and lessons for elementary and middle school age students to explore software developement and coding.

How we built it

We used html to code and build the website.

Challenges we ran into

Understanding how the target audience would see our games/website and be able to play and still learn.

Accomplishments that we're proud of

We have a functional and well designed website and a game within it. The website is user friendly and should be easy for young kids to navigate.

What we learned

We learn how to use html and how to link images. We also learned how in depth game coding is and a lot about coding in java in general. We also learned how important it is to think outside the box to meet our clients'/audience's needs.

What's next for STEM Webpage

We would like to be able to add more games, as well as lessons, tutorials for students to experiment with coding and development without having to download software.
