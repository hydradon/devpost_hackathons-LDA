Inspiration

Inspired by the board game "Mansions of Madness"

What it does

Terminal-based lovecraft-themed dungeon crawler. Incomplete so user can only encounter stationary cultist enemies who don't attack.

How we built it

C++ and the ncurses.h library for easier manipulation of terminal output

Challenges we ran into

In trying to print our fancy "MADNESS" title, we found that it mysteriously kept overwriting the window borders that were next to it.

Accomplishments that we're proud of

Teaching ourselves (some) ncurses in the time allotted to us

What we learned

How to use ncurses and putting together an unfamiliar object-oriented program on the fly

What's next for Madness

We hope to finish and fully flesh out the functionality of the game so that it is true to the original board game.
