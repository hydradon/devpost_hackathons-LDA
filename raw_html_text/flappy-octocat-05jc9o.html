Inspiration

As the theme is "Game", let's make a game.

What it does

Player tip screen to make octocat fly to avoid obstacies.

How I built it

Using Apple XCode Swift 3 and basic image editor

Challenges I ran into

Bunch of challenges, most toturial online are for swift 2. Much reading for swift 3 needed. Even spend two hours for one simple bug.

Accomplishments that I'm proud of

Faster learner, the final version of app is really fun, ps: Octocat, RedBull wall, Nyan Cat music, pixel style UI and so on. I really thought about a lot to make this game special, especially for hackers.

What I learned

Many useful skills for iOS swift I don't know be for. Like SpriteKit, lunch image setting, audio file insert and sense file.

What's next for Flappy Octocat

Also add forks to credit, user can collect starts and forks randomly. Divide touch screen to half and add player two play together. Speed up item for Octocat......
