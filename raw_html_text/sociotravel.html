Inspiration

People can contribute when they travel, invest their free time to help others. People can contribute in their leisure time to help other people by being a part of volunteering events organized by NGOs local to the destination.

What it does

So we provided a platform where people while finalizing their itiniary, they are recommended events which they can attend during their stay at a particular trip.

How we built it

We built it using Amaedeus Flight, Car and Hotel API to get information about flights, cars, and hotels which the user is suggested when we plan his trip. We also use Facebook's graph API and WeCare API to get events for the user, which he/she can attend, during their travel. Used Twilio's API to communicate to the NGO's when a person promises to attend. Connected google maps to get location of users.

Challenges we ran into

It was a good challenge to learn node.js and spent a good amount of time figuring out and debugging the code.

Accomplishments that we're proud of

We were able to finish it in the limited time. Implementing a dynamic app and creating a good UI as we didn't have much experience of UI design in the past

What we learned

We learned node.js and learned how to hack into Amaedeus APIs.

What's next for SocioTravel

SocioTravel can be improved more by connecting religiously with all the NGOs around the world. And also promoting events based on the incoming crowd.
