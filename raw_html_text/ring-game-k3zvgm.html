Inspiration

We were inspired by many of the games that Ketchapp publishes. Many of these are extremely simple, with easy to learn mechanics but hard to master. We wanted something addicting like that.

What it does

All you have to do is tap the screen when the rings match up!

How I built it

We used Unity and c#.

Challenges I ran into

Lots of the user interfacing was much more challenging than expected, as making sure everything worked on all screen sizes required a lot of hacking.

Accomplishments that I'm proud of

We were able to make this game pretty much entirely within 9 hours.

What I learned

We learned a bunch about Unity, as both of us were almost brand new to it. I am now confident that we will be able to go to future hackathons and be able to make projects that are even better. We also met a bunch of extremely helpful and generous people.

What's next for Ring Game

We will probably finish it up and make sure everything is working perfect and then submit it to the App Store for the world to use and see.
