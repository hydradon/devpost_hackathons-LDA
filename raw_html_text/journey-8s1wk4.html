Inspiration

We were inspired by our love and passion for traveling. It is a hobby that I believe does not have enough niche apps in the iOS store. So we decided to add to the community by creating an app that provides the user with itineraries for locations that they would be interested in traveling to.

What it does

This app is made to provide the user with any specified itinerary for where they plan to travel including the ability to add the itinerary to their desired "My Journey" tab.

How we built it

We built this using Swift on Xcode.

Challenges we ran into

We ran into numerous challenges that we were able to overcome: 1) Creating a gradient color background for the app home screen 2) Adding a navigation toolbar with icons 3) Text wrapping the labels 4) Trying to get the images into a slideshow on the Itinerary page 5) We accidentally deleted the entire app right before submitting so we had to find a way to recover it

Accomplishments that we're proud of

Being able to overcome all our challenges and make an app that would visually be appealing and providing the user with an experience that further enhances their travel.

What we learned

Learned how to use Swift in an environment that was targeted to mass market appeal.

What's next for Journey

We are looking forward to working on the Journey app and progressing it into an app that we can one day release onto the App Store. In order to release the app, we will have to make a few updates such as: connecting the app to a viable back-end server in order to hold account records and personal "My Journeys". Adding a search bar to the map so that users can easily track their locations as well as adding a preferences bar to execute an advanced search. Updating our expanding itinerary collection.
