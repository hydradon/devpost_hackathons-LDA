We were concerned about the safety of students in school.

E-Out is a JavaScript and Java based application. It uses Artificial Intelligence (machine learning) and image processing to determine the nearest exits during an emergency such as a fire, disaster, etc. It is mostly focused on schools, universities, public areas (shopping malls), and will soon be available for personal homes.

Email sender is built from Java. Display is from JavaScript
