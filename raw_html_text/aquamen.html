Our projects aim is to create a connection between the people who can trigger the change and the people who are in need of this help. It enables the users to collect the data concerning the contamination of the water using sensors connected to Arduino 101. Arduino as an open source platform makes it easy for the user to experiment with the code and understand what is happening due to the extensive documentation. Then data is forwarded to the application either by writing it manually or sending it by Bluetooth. The application itself gives opportunity to segregate the data and sends it to the server where the data can be analysed and visualised in the form of a map. The map helps to determine in which area the contamination of the water is increasing with time. That makes it possible to react in time.
