Setup

Android In the attached zip file there is a Dreamify-Android-App.apk file, that can be used to test the app just installing it in a Android phone. To build the project you can just import it from Android Studio.

Web To run the Web app use a console/terminal, navigate to the project folder and type 'python -m SimpleHTTPServer 8080' then open 'localhost:8080' in your web browser.
