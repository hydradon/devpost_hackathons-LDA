Inspiration

As freshmen, we have all recently experienced the struggle of making connections with other students on campus. We hope that our app will make it easy for all students to find friends and study mates that they wouldn't have found otherwise.

What it does

Our app asks the user to create an account with some information about themself, then uses that data to find other registered students with similar interests. This way, students can simplify and expedite the process of finding people on campus with similar interests.

How we built it

We built this app using the language Dart, and specifically libraries Flutter and Firebase.

Challenges we ran into

Not having much prior familiarity with the language or libraries we were using, we faced a number of challenges getting the software to perform how we wanted.

Accomplishments that we're proud of

A functioning networking app that successfully connects similarly-interested students.

What we learned

How to create a network-capable app using dart, flutter, and firebase.

What's next for BluePages

Improving on existing functionality (refining search parameters, making match algorithm more intelligent), and expanding to further capabilities and features (expanding past Brandeis, supporting matching for non-academic interests)
