Inspiration

What it does

It is an app dedicated for people who wants to automate coffee loyalty cards since nowadays it is very had to keep a hard copy of a coffee loyalty card. Integrating the barcode API and implementing a back-end, we are able to create a whole user interface of automated coffee loyalty card.

How we built it

There are three applications that we built, the app for the customer, the app for the coffee shop and the back end server. We use Android studio for the front end and a server with RESTful APIs.

Challenges we ran into

  • using server to communicate between perspectives
  • full scale android app
  • using camera & vibration sensors
  • using Restful API (Can be cross platform)

Accomplishments that we're proud of

  • being able to create three applications in the short duration of time
  • being able to learn new stuff such as working intensively with Gitlab

What we learned

  • We learned that we need a lot of planning to actually proceed with the plan
