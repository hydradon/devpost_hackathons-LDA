Inspiration

One of our team members has a fish and nut allergy, and unfortunately he had an allergic reaction at school. However he did not have benadryll or his epipen on him preventing him from recieving aid. Since, we know other people in our school and community have similar allergies we built this app to help them when they most need it.

What it does

It relies upon user aid but not from medical professionals. Instead it has everyday people who have asthma, and allergies to help others who have the same issue but forget the tools they need to combat the problem.

How we built it

We built the front end of the app using swift 4.2 on Xcode 10, then we used sketch to make the really nice designs you see here. Finally, we used node.js to build a database that allows users to reach one another, much like uber consumers reach their drivers. (There is no call feature).

Challenges we ran into

The main challenge that we as a team faced was trying to get our server database in node.js to connect multiple users, and then integrate it with the ios developer Xcode.

Accomplishments that we're proud of

Building the mapkit api that is utilized to help users find those in need and provide aid. The database that takes in and sends information much like uber does from user to drivers.

What we learned

We learned how to make an apple mapkit with reverse geocode, integrate node.js into swift, use sketch, firebase, and google app engine cohesively on one platform.

What's next for InstaAid

To make the app possible it needs a multidude of users to have the app and be able to successfully provide aid to those who need it.
