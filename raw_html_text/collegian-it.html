Hack K-State Escape

Try to escape Hack K-State in this voice-based adventure game.

This game is modeled after the style of text-based games, and includes voice commands as the controls and a webpage to show you where you are and what you can do.

Can you make your way through the KSU Engineering building and find a way to escape?

This game is built using python and node.js. It integrates AWS Lambda to run the python code based on your voice commands to the Amazon Echo. The python code then gets and posts requests to the central server, running on Heroku, that manages all the visuals and sounds. Our presentation includes a Qualcomm DragonBoard 410C to run a browser and give visual feedback for your voice commands.

To start the game, simply say "Alexa, play escape"

Commands include _ search, forward, back, left, right, reset _ as well as many more variations due to the flexibility of Amazon Alexa.
