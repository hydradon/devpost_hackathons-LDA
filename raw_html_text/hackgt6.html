Canvas Assistant

An Alexa skill for use by students who use Canvas.

Invoke this skill with "Start Canvas Assistant"

Available commands include:

  • Ask Canvas Assistant what is my name
  • Ask Canvas Assistant what classes am I taking
  • Ask Canvas Assistant what are my grades
  • Ask Canvas Assistant what assignments do I have
  • Ask Canvas Assistant what quizzes do I have
  • Ask Canvas Assistant what is my grade in English
  • Ask Canvas Assistant what assignments do I have in Physics
  • And more

Text updates

Utilizes Amazon's Pinpoint technology to send text updates on:

  • Grade Changes
  • New Assignments
  • New Quizzes

System Debugging

Utilizes NSA's Beer-Garden to provide web hosted REST calls to quickly debug the system and see the responses

This app was written by @jaustinmiles , @Ryanm14 , @michaelxiao16 , and @Rmoazzami during HackGT 2019.
