Inspiration

I wanted to solve a problem that I struggle with. I struggle with the amount of time it takes to invite people personally to a event. I like the personal touch to an invitation because people feel more obligated to go if you personally invite them. So I thought why not have a txt messaging bot handle the conversation for you.

What it does

It is a mobile app in react native meaning it compiles in android and ios! It allows the user to fill out a basic form when he or she decides to host an event. The form will contain the title, details about the event, and the people invited. Then will handle the conversation after.

How I built it

I built it with react native, node.js, express.js, and Twilio.

Challenges I ran into

I ran into the challenge of learning how to execute a express sever for the first time.

Accomplishments that I'm proud of

I was able to successfully make post request for the first time!

What I learned

I learned how a Full stack application works from react frontend to express backend.

What's next for SMSBot

We plan on continuing the project until it is complete because we think it solves an important problem.
