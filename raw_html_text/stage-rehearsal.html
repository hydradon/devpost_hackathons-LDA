Inspiration

Getting feedback in a presentation without having someone listen to your boring presentation

What it does

Listens to key words and syncs that with your time to help you prepare your presentation and to advance slides when needed

How we built it

Used an audio processing module to transcripe speech to text using microsoft's azure api, which then matched text to speaker's audio cues

Challenges we ran into

bing has limited api requests

Accomplishments that we're proud of
