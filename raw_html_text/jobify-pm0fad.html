Inspiration

We use telegram as our communication app and we really like the powerful chatbots. Also we were looking for an easy way to look for a job and we think that this bot makes it a lot easier.

What it does

It's a telegram bot that lets you control your infojobs account, some features are looking for job offers, apply to job offers, edit your CV and personal data.

How we built it

We used nodejs for the bot and the server side to connect de bot with the infojobs API.

Challenges we ran into

Connecting to infojobs API and the limitations of the public API.

Accomplishments that we're proud of

We can use almost every API call that infojobs has.

What we learned

In our team we had two first time hacker that learned how to implement a telegram bot and how to use an API, the other two members also learned new usages of the APIs.
