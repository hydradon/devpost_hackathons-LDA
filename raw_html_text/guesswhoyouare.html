GuessWhoYouAre is a Web Application to predict the personality traits of a person by integrating social media API's. Currently used API's are Facebook, Twitter and Quora API's.

Technology Used:- The Front end is developed using Bootstrap, HTML & CSS. The Backend of the website is made using Django 1.9. API's or SDK's for user content retrieval. IBM Personality Insights was used for training & predictions.

Issues Faced:- Authentication issues during user data collection for IBM Personality Insights. Trouble integrating Bootstrap to Django backend. Scarce documentation to assist in adding user features in IBM Personaloty Insights
