• Demo of the InsultBot!

Inspiration

Wanted to make something fun and learn how to make a chatbot.

What it does

It parrots whatever you say to it and adds "your mom" at the end. Try it out, message: https://www.facebook.com/insultbot1/

How I built it

I used node.js and intertwined it with Facebook messenger.

Challenges I ran into

I had never made a chatbot before so I spent a lot of time learning.

Accomplishments that I'm proud of

Creating my first chatbot!

What I learned

Node.js, I had never used it!

What's next for Insult Bot

I hope to add more insult features and comebacks in the future.
