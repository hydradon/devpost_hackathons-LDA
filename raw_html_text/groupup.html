Inspiration

It can be hard to find study groups or pick-up games within your own circle of friends, so a local aggregation of others' events and groups can allow you to meet new people to study with, play with, or hang out with.

What it does

It aggregates events posted by other local users and yourself, and shows them in a categorized, browsable list for you to look through, and RSVP to.

How We built it

Client app is built for Android devices through Android Studio, back end is hosted on Amazon Web Services, and requests for the server are handled by a Python script.

Challenges We ran into

Between all of us we have very little app dev experience, database experience, and client-to-server communication experience. We used all new technologies (for us) so most of the weekend had to be spent learning how to build what we wanted within them.

Accomplishments that We're proud of

We learned a lot about the services we're using (Android, Amazon Web Services, etc.), and were able to get most of the pieces working on their own, even if we havn't put it all together yet.

What We learned

How hard it can be to crash-course yourself through three new technologies in a weekend, then combine them all into one coherent experience. Also learned some new strategies for pouring over new APIs in a way that has a quick enough turnaround to start using them in a weekend.

What's next for GroupUp

Well hopefully finishing the app.
