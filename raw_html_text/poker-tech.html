Inspiration

Games like Jackbox Party Pack and Agar.io.

What it does

Allows people to enjoy poker together through a web-hosted service.

How we built it

API written in Java and served on a Domain.com website hosted by Google Cloud services, which is accessed by the front end JavaScript app running on a .tech domain.

Challenges we ran into

Hosting the back end API using Java environments.

Accomplishments that we're proud of

Hand recognition algorithm and general gameplay mechanics.

What we learned

The process for hosting an API web app on a free hosting service.
