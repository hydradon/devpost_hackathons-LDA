Inspiration

The need for simple education for those wanting to learn data structures without all the egregiously boring documentation online. Also, for those struggling with understanding the language online due to a language barrier, tis program provides a no-bias solution to computer science education.

What it does

Demonstrates the stack data structure through an interactive visual stack.

How I built it

Through JavaFx.

Challenges I ran into

JavaFX is not a great way to build visuals. I was recommended to use other languages but I wanted the challenge.

Accomplishments that I'm proud of

A no-compromise visual representation that speaks to the design language agreed upon by the team. Every main component \/decision of this project was made with the consensus of the team.

What I learned

The power of programming in creating educational materials from scratch.

What's next for Stackz

Proliferation through an online source to the masses looking for a fun and simple solution to learning the stack data structure.
