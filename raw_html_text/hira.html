People don't write good notes. Using web speech API and, to be fully implemented, Google Cloud's natural language processing, Hira creates a live transcript of notes which you can annotate, and uses machine learning to generate tags that help you sort what you're learning. This is perfect for professionals in meetings and especially for students in class who have a hard time listening and taking notes at the same time.
