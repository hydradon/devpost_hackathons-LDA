Inspiration

Know the current weather conditions and get a trustable forecast for our town

What it does

Utilizing our own weather station to receive local weather conditions. Combine data from professional weather stations to compute an optimized forecast for our small town, including rain probability and temperatures.

How I built it

PHP (Backend), Java (App), HTML (old frontend)

Challenges I ran into

Android App development is no fun.

Accomplishments that I'm proud of

Made it work, did not get a penny for it

What I learned

Use PhoneGap over native Android development

What's next for Dasbach Weather station

Make it more conversational, e.g. allow specific questions about the weather forecast to avoid hearing more than needed. Return more information, like the current humidity, the temperature under ground etc. Include the weather data to compute weather-dependent itineraries: http://www.tripango.de/
