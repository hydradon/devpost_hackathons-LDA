Inspiration

My teammate and me love our cats, so we decided to develop a game with the cat theme. And here we go! A cat endless runner game!

What it does

It's a endless 3D runner game, which will increase the difficulty by increasing the running speed. And we have 4 cats as our characters, each of whom has its own special ability, such as heal, and shield.

How we built it

We built the game using Unity game engine, and the scripts are written with C#.

Accomplishments that we're proud of

We are happy that we can almost finish a game in such a little time.

What we learned

We've learned a lot of things about how to design, develop and beautify a game with the help of a game engine.

What's next for Meow~Meow!GO!

We will improve our project and hopefully we will publish it on websites.
