fox

BitCoin made easy.

fox is a free bitcoin wallet that lets you make effortless transactions with friends, family, and businesses. Easily track recent fluctuations in bitcoin value with beautiful graphs at your fingertips.

Tech

fox is built with:

  • [BlockChain] - Bitcoin transactions
  • [Android] - Front-end application
  • [Node.js] - Dev Language

fox is open source with a public repository on GitHub.
