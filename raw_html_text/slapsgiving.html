Inspiration

Simon Says alone? Boring. Making it wireless? Better but not quite. The loser gets slapped? Now we're talking.

What it does

2 players compete with the same Simon code, with anyone that messes up getting a slap. If both players get it right, the winner is who did it faster.

How we built it

Two computers connect to each other through a direct TCP connection, with the entire interactive device controlled through an Arduino & build set per person.

Challenges we ran into

One of the main issues we ran into, down to the last minutes was having the computer send the key Simon code to the Arduino to display it for the players. Besides that, differing coding styles led to some compatibility issues between our different programmers.

Accomplishments that we're proud of

The TCP connection giving the option to play with a friend over long distance.

What we learned

Sleep is important.

What's next for Slapsgiving

Slapsmas, duh.
