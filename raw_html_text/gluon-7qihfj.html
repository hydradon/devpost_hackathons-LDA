Inspiration

Improving the e-commerce experience by having real time analytics using lex and recokgnition (aws).

What it does

It recognises emotions from photos using rekognition API, and analyse the behaviour of customers using real time chat bot.

How we built it

  • Emotions are recognized using AWS Rekognition API which is integrated with S3 cloud storage using lambda functions. Finally a lex bot is integrated with the above solution for the better understanding of user behaviour.

Challenges we ran into

Integrating Rekognition with the chat bot.

Accomplishments that we're proud

Final bot is built.

What we learned

AWS APIs such as Rekonition.

What's next for Gluon
