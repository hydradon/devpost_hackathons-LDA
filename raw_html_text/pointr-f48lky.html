Inspiration

Home automation services such as Lutron and Nest.

What it does

Uses the leap motion controller to translate physical gestures into Raspberry Pi functions

How we built it

We connected our Raspberry pi to a python socket server in order to send commands to the micro controller.

Challenges we ran into

Upon configuring the python server, we encountered a Trojan virus that temporarily locked us out of the Raspberry Pi and required us to install a fresh copy of Raspbian.

Accomplishments that we're proud of

Controlling individual lights by mimicking the gesture of flipping a switch in that general direction.

What we learned

The importance of changing the default username and password on the Raspberry Pi, controlling a remote system through SSH, and tracking movements through IR sensors.

What's next for *Pointr

Implement additional gestures for a wider variety of home automation tasks.
