Inspiration

We were bored coming up with an idea, and Morgan imagined a game board between us.

What it does

Using ISUHackathon laptop stickers as targets, the app displays AR tic-tac-toe and connect four, playable for two-player and single-player.

How we built it

We built it in the Unity Game Engine with the Vuforia AR platform.

Challenges we ran into

Building to Android required many installations and updates, learning to work with Vuforia wasted some time at the beginning. Overall, the project went pretty smoothly.

Accomplishments that we're proud of

Actually making AR, getting it built to mobile, and accommodating two-player and single-player gameplay.

What we learned

How to use Vuforia.

What's next for AR Games

Making more games and improving the UI.
