Inspiration

I have a strong desire and will to make paperclips. But, I sometimes don't always have the time to be making paperclips. So I wanted a robot to make paperclips for me.

What it does

This robot can make paperclips super fast. Not only that, it understands how to sell these paperclips online, it knows supply and demand and can change the price dynamically. The robot buys its own supply of wires and knows that it should be buying wires when it's cheapest. The wires that the robot buys eventually ends up to become paper clips

How I built it

I used UIPath Studio and used a bunch of flowcharts for the robot to automate the task of making paperclips.

Challenges I ran into

Learning UIPath Studio is a journey. I learned the many advantages and disadvantages of learning UIPath Studio. UIPath Studio is very different from low-level programming languages. UIPath is very easy to understand but also very easy to mess up if you are a messy programmer like me

Accomplishments that I'm proud of

I am proud of learning a new tool to help automate tasks. I am proud of making this project. I am proud to be here.

What I learned

I learned a lot about UIPath Studio and its style of programming.
