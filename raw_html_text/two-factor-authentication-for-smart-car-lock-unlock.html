Inspiration

Prizes from the sponsors and drive to do something cool and fun and we would like ✓ to forget our keys at home and able to unlock the car. ✓ to hands free unlock our car when both hands are pre-occupied with grocery bags ✓ to share our car with family and friends without worrying to hand over the keys or share the user credential

What it does

The smart system unlock or locks the car using the smartcar api and microsoft azure cognitive services. It uses face recognition and speech passphrase for authentication.

So, the user can unlock the car by looking at the camera followed by saying the user set passphrase.

How we built it

We used the python api for smartcar to get access and control the smartcar systems(with user approved privilleges) and integrate a real time face recognition and speech recognition microsoft azure cognitive services for dual authentication.

Challenges we ran into

¶ Real time face detection and recognition using the webcam ¶ Getting all things working together ¶ Sleep deprivation

Accomplishments that we're proud of

The project provides an interesting proof of concept prototype for hands(and offcourse password free) free two factor authentication system which could be ported to any Iot compatible system or maybe a mobile app with little effort (could be more based how well the job is done)

What we learned

Azure cloud service and Smartcar API usage for easy development of 'ready to deploy' service.

What's next for Two factor authentication for Smart Car Lock/unlock
