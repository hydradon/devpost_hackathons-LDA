Inspiration

Our school takes 10 minutes each day going through announcements, and most people talk through them anyways, so we never get to hear them. This is where our idea for creating an easy-to-use announcement application comes from.

What it does

We provide an interface to post and retrieve announcements through firebase.

How we built it

We used HTML and CSS to format our app, and used JS to fulfil requests

Challenges we ran into

We weren't able to implement authentication, as we had difficulty connecting it with the back end. Firebase broke.

Accomplishments that we're proud of

Planning and ideas We were able to make a demo app and a decent website showcasing out app. The whiteboard.

What we learned

Firebase is annoying to work with

What's next for Announcely

We are also planning on implementing user verification. We are planning on implementing personalised topic suggestions, and using location data to provide relevant info
