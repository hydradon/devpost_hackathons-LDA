Inspiration

We wanted to create an educational platform for children to find out more about the world around them.

What it does

It is a hands-free web application that shows you can interactive globe where you can search for different places using your voice. Click interactions are enables through eye tracking backend.

How we built it

We used the esri Javascript api and xLabs eye tracking api. All code is written in javascript, html, and css.

Challenges we ran into

Eye tracking can be unstable and we had to find a happy medium between precision and ease of use. We also had to figure out how to make different APIs compatible in a single application.

Accomplishments that we're proud of

We have a voice search implemented and have a listing of UNESCO World Heritage sites to provide cultural enrichment.

What we learned

We learned about the esri and xLabs apis and the challenges of building interactive web apps.

What's next for HackNC

We want to rotate the globe with our eyes, increase tracking precision, and enable make our interface as intuitive as possible
