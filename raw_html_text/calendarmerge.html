Inspiration

Never being able to schedule meetings with our friends due to our busy schedules, we wanted to create a webapp that would do it for us.

What it does

Merge 2 or more calenders to make an event that matches both schedules

How we built it

We boilt it using node.js, HTML, and Javascript

Challenges we ran into

Implementing Google Calendar,

Accomplishments that we're proud of

Building a website that functions and can interact with the user.

What we learned

Node.js, Cronofy (a cool calendar api!) and more HTML

What's next for CalendarMerge

Making a nicer interface with more user-friendly capabilities!
