Inspiration

We wanted to design a way for Wyomingites to interact with one another, learn about cryptocurrencies, and view real exchange data from Cryptowatch!

What it does

A Discord Plugin that allows users to get crypto data from Cryptowatch.

How we built it

We built CryptoCowboy as C# console application to display real-time logs of usage. We then connected CryptoCowboy to Discord API and Cryptowatch API to allow it to join the Discord server and get crypto data from Cryptowatch.

Challenges I ran into

Connecting the Cryptowatch API to CryptoCowboy is the most challenging part.

Accomplishments that I'm proud of

The inspiring interest of Blockchain!

What I learned

-What is the blockchain -C# -Discord API -Cryptowatch Usage and its API

What's next for CryptoCowboy

Host CryptoCowboy on an Azure, AWS, or some other cloud solution so it can be accessed by all users on Discord.
