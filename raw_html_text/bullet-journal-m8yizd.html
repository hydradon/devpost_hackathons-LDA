Inspiration

My girlfriend told me she wants to have such an app and on app store there are no apps that fulfill her requirements.

What it does

Users can create bullet journals by adding frames, images, and text on a mobile device.

How I built it

I tried to use Android studio to create the UI of the editing page for bullet journal. Since I would like to publish it on App Store, and all of my friends who applied for Hack The North didn't get the offer, it is better for me to work on this project my own instead of having other teammates during the hackathon.

Challenges I ran into

Right now it would produce an error if I click on the image button for the second time. Only one instance can be added for editing since I only create one ImageView for each button.

Accomplishments that I'm proud of

It took a while to figure out how to move and scale a given image. I though there would be some build-in functions in Android studio but there wasn't any, but finally I figure out a way to let it work. Android studio is only about the 3rd or 4th comfortable tool for me, but I can still managed to create a prototype of the app.

What I learned

When choosing topics for the next hackathon, an app with more backend activities and a tool I am most comfortable with should be chosen.

What's next for Bullet Journal

Maybe it is not well-polished, it is a good start. I will be keep on working this project after the hackathon.
