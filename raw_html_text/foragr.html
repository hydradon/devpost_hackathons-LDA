Inspiration

I live in my car. It's a prius. Before I moved in, I did a lot of research. I found out about the VanLife community. Those are people who live in their vans full time.

They tend to stick to cities for around a month each. Although some of them prefer remote work (called Digital Nomads), most prefer in person short-term gigs. Craigslist allows one to search for gigs, but only in one city at a time.

This makes it hard to plan where to live next. Foragr allows every VanLifer to look at a map to easily and quickly decide where to live next.

How we built it

Python for data scraping craigslist. HTML and CSS for the website. Swift for the iPhone App. Esri API for the maps.

Challenges we ran into

EVERYTHING.

Accomplishments that we're proud of

Made a speed reader in vr during spare time ;)

What we learned

How to use Esri api, how to web scrape with python, why to always save.

What's next for Foragr

Allow users to search for long term positions as well.
