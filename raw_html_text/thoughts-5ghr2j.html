Inspiration

Awkward situations on the bus where you forgot someone's name/don't know how to start a conversation/curious about what someone is thinking.

What it does

Reads a user's mind (pulls up their most recent tweets) from a picture of them.

How we built it

We used Kairos for image comparison and created galleries of profile url's to search through based on tweets in the app user's area to map profiles and populate a tweet list.

Challenges we ran into

Image processing in a timely manner, accuracy of image recognition services, Asynchronous Task usage.

Accomplishments that we're proud of

Use of multiple API's to accomplish a seemingly magical task.

What we learned

Making rest calls, using image recognition.

What's next for Thoughts

Sentient analysis on tweets to find a user's mood, gamifying user identification based on follower count.
