Inspiration

Inspired by our thirst for knowledge, we created a simple-to-use app that will break all barriers between teacher and student interactions.

What it does

WePass is an educational platform that connects teaching staff and students, allowing greater opportunity for open-ended learning and a convenient medium for everyone to share their own inquisitions.

How we built it

We leveraged the advantages of Ruby on Rails to coordinate with databases, while using HTML and CSS to provide an appealing user experience.

Challenges we ran into

Only one of our members is proficient in Ruby on Rails, so productivity was slow during the beginning of the creation process.

Accomplishments that we're proud of

We are proud that we've created a functional website despite our deficits in programming skill and time constraints.

What we learned

We learned that we should be more proactive in the future by downloading all the necessary tools needed to code certain languages, as well as taking the initiative to become more familiar with the languages we plan to use.

What's next for WePass

Better UI, improved functionality and additional features.
