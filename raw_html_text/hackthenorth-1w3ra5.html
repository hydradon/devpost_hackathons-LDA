Inspiration

We were inspired by photos where (for privacy reasons) entire faces are blurred out. We wondered if there was a way that both facial expressions (and by extension, emotions) and privacy could be conserved at the same time.

What it does

It detects the emotions from your face and replaces your face with the emoji most similar to your facial expression.

How I built it

Emoticonceal was built using Google Cloud Services' Vision API. The backend is in python.

Challenges I ran into

We tried to use Microsoft Azure's emotion api, but ran into trouble attempting to develop on our Apple computers. We tried using the api using both python and C# with no success.
