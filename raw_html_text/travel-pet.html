Inspiration

We all have fond memories of playing with virtual pets like Tamagotchi and Littlest Pet Shop. We decided we wanted to get rid of the pain of carrying around a device and make it accessible anywhere through the cloud.

What it does

It creates a virtual pet in the cloud that is accessible everywhere.

How we built it

Created a backend API that tracks all the statistics, and a frontend that interacts with that API

Challenges we ran into

Time crunch + exhaustion + lack of experience

Accomplishments that we're proud of

nearly completing the API and making cute pixel art

What we learned

MongoDB, Stateful React, Javascript, Git

What's next for Travel Pet

Finish frontend, make interaction between different users' pets, add more life stages
