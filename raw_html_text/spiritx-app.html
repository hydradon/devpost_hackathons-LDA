Inspiration: We decided to make an app that uses machine learning. Following workshops with the Exponents and Microsoft team, we thought of using Microsoft API that relates to the Detect Emotions in Images.

What it does

Based on a person's emotion, it will play a particular sound track. For example, when a person is happy, it will play some happy tracks, through Alexa.

How we built it

We built it by writing a node.js file that contains instructions for capturing images through an Iphone, and fed that image to the Microsoft face API. Following which, a corresponding sound track is broadcast by Alexa.

Challenges we ran into

Only 1 of the member knew about App Development, and javascript. We did have to learn new skills within a limited time frame. Neither of us knew about the handling of Alexa.

Accomplishments that we're proud of
