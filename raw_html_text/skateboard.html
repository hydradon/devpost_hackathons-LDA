Inspiration

We wanted to build an app to save money together for road trips.

What it does

Users have profiles and can create piggy banks with their friends. Users have personal balances, which can be drawn from a credit card. Personal balances can be added to piggy banks, and money can be withdrawn from piggy banks.

How we built it

We used firebase to store information. We attempted to use Authorize.net to manage transactions, but it was slow to return money to users and we didn't use it. The money in the application, then, is purely virtual.

Challenges we ran into

We couldn't find a feasible API to handle money, and there was no time to create our own. We struggled with and learned a lot from Android Studio.

Accomplishments that we're proud of

We have an application that communicates with a Firebase database in a way that resembles adding and removing funds from a piggy bank.

What we learned

We learned a lot about Android's development environment.
