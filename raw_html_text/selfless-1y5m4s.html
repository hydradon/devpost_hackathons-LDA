Inspiration

We all took horrible selfies before we started, and this idea was intended to help improve our skills.

What it does

Selfless allows for real-time selfie scoring. It allows the user to improve their selfies by providing a score for the data.

How we built it

We built the application using Android, Python-Flask, and deep learning. We used a labelled dataset of images to train our deep learning model.

Challenges we ran into

Our dataset consisted of too much data, which caused our model taking too long to train with our limited processing power.

Accomplishments that we're proud of

We ended up training the model with enough time to spare to create everything else in the application

What we learned

How we could maybe improve the model so it saves us time in the future.

What's next for Selfless

Diving deeper into our data to suggest fixing certain attributes
