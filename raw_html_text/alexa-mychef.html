Inspiration

We're two Umass Amherst students who want to spread the benefits of nutrition to as many people as we can. To that end, we decided to use this hackathon as an opportunity to make Alexa into a compact, nutrition assistant.

What it does

Alexa MyChef will allow for on the fly meal suggestions and nutritional facts based on our own preset information as well as food data from the USDA. Preempt any suggestion with "Alexa ask my chef - "

How we built it

MyChef was built node.js and clay for the coding, using amazon developer services to set up the necessary voice dependencies.

Challenges we ran into

One of the main features we desired to implement was constructing a meal from our data. Now while had all the data needed for construction, we found that creating strings for alexa to say based on the data to be broken.

Accomplishments that we're proud of

We can query food from the USDA database.

What we learned

javascript can be challenging for mundane reasons

What's next for Alexa MyChef

Eventually adding the ability for complex meal suggestions, based off of past inquiries. Scikit AI learning
