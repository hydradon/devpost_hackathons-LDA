Inspiration

In the production process, anything can happen. To solve any kind of emergency, because time is money, we created a system to solve and keep track of every event in the factory floor.

How it works

It tracks the solution time for the events in the production floor, updating in real time to Kanban, handheld devices, tablets and mobile phones.
