Inspiration

Help HR in keeping track of applicants, jobs and applications.

What it does

Allows HR to see available jobs in company, people who have applied, and application of people.

How I built it

I built it using Angular 6.

Challenges I ran into

It was difficult to set up CORS in my application, as iCims restserver does not put Access-Control-Allow-Origin header in pre-flight response. Hence, I created proxy server on my local to get api response for Angular application.

Accomplishments that I'm proud of

I have completed all functionality of web app single handedly.

What I learned

I learned creating Angular application, utilize API in the app, and setting up CORS.

What's next for icims-job-app

I will make UI more user friendly, make components to edit job positions and update application status.
