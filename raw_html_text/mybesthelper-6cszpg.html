Inspiration

A business already running + really thinking as a mom and a helper + old situation with my own web site.

  • my 1st Hackathon

What it does

Let both Helper and Family users to fill a quiz in a funny and soft way, by getting a "taste" before the signup process gets completed.

How I built it

Ionic Framework and controllers/factories using JSON, so it is easy to integrate. Having right icons in mind and my experience on similar projects.

Challenges I ran into

Provide an soft, fun and flowed navigation

Accomplishments that I'm proud of

Navigation, icons properly applied, sample of matches, validated on device

What I learned

Get clear idea of what to reach, focus and how an intuitive interaction can transform a simple process.

What's next for mybesthelper

1. Scope - check the possibility to see and navigate the actual flow to confirm the adherence of what I conceived

2. Questions - change the code to the provided questions

3. Design and Colour scheme - modify SASS of Ionic's default to adapt CSS regarding the provided Style Guide

4. Typography - modify SASS of Ionic's default to adapt CSS regarding the provided Style Guide

5. Sample Data - write collection objects and serve by using JSON-Server (controller and services already prepared for it)

6. Archetypes - write code to map the characteristics to ensure best matches

7. My profile - write a view with all the characteristics of a Helper / Family

8. User location - write code with Google Maps API to locate the user city and show count of matches on home

The idea In my words

We can develop an exciting interaction by providing a flow like this: in each step I inform data, I can see the "count" of matches decreasing untill reaching the final screen, where the best matches are presented and available!

Home: give the Helper/Family options AND show just a quantity of helpers by identifying the user location Archetype: when seeing mine, I can see how many of "me" there are AND how many families requesting a helper like me - but without having a way to get in touch). Personal info: lastly I fill my contact details and get a final message (maybe here we could show real matches - a sample. Need more information about the actual business rules, but I feel you got my point)
