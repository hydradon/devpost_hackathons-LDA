Inspiration

We saw a need in local midwestern communities and hoped to help meet it by creating this system.

What it does

Lets contacts at shelters request food, contacts at restaurants offer food, and drivers receive assignments.

How we built it

Firebase in the back, Angular in the front!

Challenges we ran into

Angularrrrrrrrrrrrrrrrrrrrrrr

What we learned

This was our first time using Firebase but we learned how simple and easy it was to use as well as how powerful the NoSQL database could be. We'll definitely be using these web development tools in the future.

What's next for glean

Finishing the product, adding features, and getting people in the public to use it!!!!
