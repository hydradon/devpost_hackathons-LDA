• Rhythm Boy

Inspiration

retro arcade games, and newer rhythm games

What it does

Its a rhythm game on the game boy. enough said.

How we built it

We used GB studio and a flash cartridge.

Challenges we ran into

GB studio was relatively new, so it was not a fun experience using a broken engine. Music was also difficult to port to the game boy.

Accomplishments that we're proud of

It works.

What we learned

We learned how to program a game in game boy, and learned to make music in 8-bit chiptune.

What's next for Rhythm Boy Color

When GB studio improves, we can definitely make the game much better.
