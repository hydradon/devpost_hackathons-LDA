Inspiration

Checkmate aims to bring families together , anyone can keep track on his dear and near ones and text them at will .

What it does

A user can either join a family or create a family . Creating one gives you a secret code which can be shared to people you want to add to the private group . Their locations are constantly updated on the server and can be viewed on maps anytime . A group messaging interface is setup simultaneously with which users can communicate with each other .

How I built it

Used trnql sdk for location mapping and parse cloud for messaging backend and push service in android and sinch messaging platform for ios .

What I learned

Trnql SDK ,parse push messaging

What's next for Checkmate

Published on playStore , ios code committed to git , will publish on appStore after dev account has been setup .
