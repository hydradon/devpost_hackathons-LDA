Inspiration

We believe that people's votes count and can make the difference. We believe everyone should see in realtime how the elections are going with the real votes (not just predictions).

What it does

The platform provides real time status of the election based on actual data that are available on big screens such as public/personal TVs, blogs/internet platforms, personal computers, but we also thought about a simpler platform for smartphones or old fashioned phones. It is eventually possible to get results via SMS.

How we built it

As UX designers, we created a prototype of the desktop platform through photoshop and HTML and CSS.

What's next for NAIJA RealTime

We aspire to strike up a partnership with blockchain in order to reduce the costs.
