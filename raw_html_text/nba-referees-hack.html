Introduction

We decided to see just how biased NBA referees are.

What it does

We gathered data from stats.nba.com and kept information on how many fouls were assessed to each team and by which referees. We then visualized our results.

How we built it

We used R to generate our graphs.

Challenges we ran into

The data given does not tell us which referees assessed which fouls. The only data we were given was which group of referees assessed certain teams the fouls.

Accomplishments that we're proud of

We were able to generate great graphs.

What we learned

We used a little bit of machine learning.

What's next for NBA Referees Hack

Creating a more accurate prediction algorithm.
