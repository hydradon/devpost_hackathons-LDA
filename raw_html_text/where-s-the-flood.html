Inspiration

The recent flooding in Houston.

What it does

Builds a model to predict the areas that are more prone to severe flooding

How I built it

In order to build a proper model, we pulled data from the Dark Sky API and nearby reservoirs to train a model based on historical measurements. Using factors like the elevation, precipitation rates, and proximity to water given a location, we are able to estimate the flooding intensity of an area.

What's next for Where's the Flood?

We hope to continue improving this model by including more variables that might affect the flooding such as lack of vegetation, soil absorption, elevation around areas, and snow and ice.
