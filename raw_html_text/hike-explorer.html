Inspiration

I wanted to make a game to enjoy the challenge of hiking with a bit of a funny twist

What it does

This is a PC game which puts you in the hands of a hiker trying to climb mountains from all around the world

How I built it

I used Unity for the Game Engine and C# for the scripting and I used Krita to create the images.

Challenges I ran into

A large challenge was ensuring the build for Unity reflected the editor build since there was a lot of problems due to the program.

Accomplishments that I'm proud of

I'm proud of putting a very small idea (having stretchy limbs) and running with it to create a fully working app.

What I learned

I learned quite a bit of Unity systems, how to debug C# code and learn the ins and out of Game Development

What's next for Hike Explorer

I want to add more levels and in the end, create a full game to be released to the masses.
