Inspiration

Standing in line every week with my publix sandwich behind people with like 30 items really makes me think I gotta be losing a couple hours by now, right?

What it does

We made an app to scan groceries via a barcode, then connected it to a web app to allow for easy and quick checkouts.

How I built it

We used the firebase/express api with an angular webapp and a react-native phone app. We pulled item data using the NCR API.

Challenges I ran into

We've never used phone cameras, and those of us working on the web apps were still fairly new. Firebase was also an unfamiliar way of hosting a database and api service.

Accomplishments that I'm proud of

We created and exposed a firebase API, finished a neat looking front end and app, and tied it all together in the end into an overall project.

What I learned

Some of us gained exposure to react-native and mobile dev, some to angular, and some to firebase.

What's next for The Nonstop Shop

As soon as we get out the store door, we're looking to cut wait times elsewhere. Unsure where we'll be hitting next.
