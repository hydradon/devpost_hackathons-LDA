Inspiration

Donkey Kong Jungle beats is without a doubt, the best gamecube game of our generation. We wanted to reuse its iconic bongo controller for greater purposes

What it does

It creates a guitar hero style game that allows the user to hit keys with bongo controller

How we built it

This project was made with Unity2d in C# Language

Challenges we ran into

Challenges occurred in keeping track of points, deleting reoccurring objects, connecting audio as a trigger for game beats to spawn

Accomplishments that we're proud of

Being able to understand and connect the bongo controllers to the game play, as well as actually being able to produce a demo-able MVP

What we learned

How to use unity and how to adapt game cube controllers to it

What's next for Bongo Hero

different levels of difficulty as well as new songs.
