Inspiration

We were inspired by our own personal experiences. The majority of the students in our computer science classes were boys and we were intrigued by FemmeHacks and their mission to encourage girls to enter STEM fields. As young aspiring programmers, we wanted to promote gender equality. In school, we are involved in clubs like STEM Club and Robotics that are predominantly male which encouraged us to found clubs like Girls Who Code and join She’s The First. As a result, we wanted to combine our passions to create a website about gender equality.

What it does

The website informs about various problems that women face today. It includes the location, the problem and how we can help overcome it.

How we built it

We built the website using HTML, CSS and javascript.

Challenges we ran into

The thumbnails would not appear on the website, depending on what browser you accessed it from. This was also our first time using Github.

Accomplishments that we're proud of

We initially had very limited knowledge on HTML, CSS, and JavaScript, but we were able to learn and grow through this experience. We are extremely proud of our product because of how much hard work we have put into it.

What we learned

Previously, we had only coded in Java and and had a very limited knowledge of HTML. However, this project helped our knowledge of HTML grow. We also learned aspects of design, that we would not have learned otherwise.

What's next for Empow[H]er?

This website would be updated to be a up-to-date on current events and improvements in women’s issues.
