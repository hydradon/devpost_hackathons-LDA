Inspiration

The nuke map, http://nuclearsecrecy.com/nukemap/ .

What it does

Allows you to create a hurricane path, gives back an estimate on damage costs and death toll.

How we built it

Tools used were HTML/CSS, Javascript and the Mapbox API.

Challenges we ran into

A couple of roadblocks when using the Mapbox API, one of them being attempting to use a deprecated library.

Accomplishments that we're proud of

Got a prototype up and running.

What we learned

GIS is software is painful to use.

What's next for Spin Damage

Introduce datasets to create models that make better predictions.
