Inspiration

Participated in a few hackathons. I noticed that judges have been using paper to score different projects. I wanted to create an app that can help make judging, especially in a large scale event, easier and user-friendly!

What it does

This application has a double-authentication login for security. Then, users can judge different projects based on preset guidelines.

How I built it

I used Swift, Microsoft's Face API, and Google's Firebase.

What I learned

I learned how to implement Microsoft's Face API for the first time! Learned a lot about Azure technology.

What's next for iJudge

Create an AR pathway for judges to follow and navigate through large crowds to find and evaluate their assigned projects.
