Inspiration

We wanted to facilitate a blockchain transaction based on real-world events for simple 1 to 1 wagers. With some expansion on this MVP, we believe that we can facilitate friendly, competitive, fun interactions.

What it does

It allows users to make simple wagers against the value of cryptocurrencies, sports outcomes, etc. With an integration from other data sources, we would like to be able to create small, near real-time wagers that

How I built it

We created a smart contract to hold an escrow of the wager until an oracle is able to return the outcome of the wager and execute the contract to pay the winner

Challenges I ran into

Learning how to implement our smart contracts with external data sources

Accomplishments that I'm proud of

Successfully returning the correct information and validating the outcome with an oracle

What I learned

How to use chainlink to retrieve off-chain data for a smart contract

What's next for weBet

Expanding the potential wagers to include other events Implementing a burner wallet to facilitate other wager games
