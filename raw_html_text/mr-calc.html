Inspiration

Many multirotor designer need to calculate and analyze many things such as weight distribution, flight time, etc. This webapp help them to do that.

What it does

It help you design and measure the component of quadcopter because the UX design of this app. Also it help you presenting your design through audience.

How I built it

I build it due to Autodesk Hackathon challenge that introduce me about Autodesk Forge API (especially model derivative API, its very handy).

Challenges I ran into

Originally it suppose to be hybrid app (mobile and webapp friendly version using Ionic 2) but i have a problem with visualize the 3D model through Autodesk View API.

Accomplishments that I'm proud of

It can analyze weight distribution and show the design of the multirotor.

What I learned

How useful Autodesk Forge.

What's next for mr-calc

Measure flight time.
