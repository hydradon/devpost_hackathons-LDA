Inspiration

Being forced to use racket in my intro to comp sci class.

What it does

Allows for uploading and downloading racket (.rkt) files while playing great music.

How we built it

Read/Write from FireBase into an html file.

Challenges we ran into

Firebase didn't like the .rkt extensions, so we used regex to remove every "."

What we learned

Basic web dev and connecting to Firebase

What's next for DropBox for Racket Files

Transforming it into a platform where you can edit and run Racket code.
