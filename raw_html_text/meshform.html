Inspiration

We like robotics competitions, but we sure hate scouting. Keeping track of teams' performances is pesky and tedious work, and is easily made more fun with MeshForm! Combined with garbage internet at every First Robotics event, we had to find a solution.

What it does

Scouting app where people record stats about other teams performances. Then, work is merged by transferring data through QR codes(no internet required)

How we built it

React native and expo.

Challenges we ran into

We wanted to do mesh networking but expo sucks

Accomplishments that we're proud of

We finished sort of.

What we learned

8 hours is not enough time to create something.

What's next for MeshForm

Integrate with Google Sheets API so we can throw the information online once you have internet.
