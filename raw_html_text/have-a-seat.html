Inspired by the diverse community in Miami and wanted to make our culture more inclusive with a unique and shared dining experience.

HaveASeat allows restaurant staff to see what tables have extra chairs and fill them up accordingly. This website allows a potential host/waitress to facilitate the exchange between individuals who are already have seats and want to share with individuals waiting. It's dining-out, turned social.

Challenges we ran into included adding more dynamic functionalities like the dialog that appears after a change is made to the table.

Proud of being able to achieve the tackle the main factors of the UI

With little experience in Javascript and mongoose, we were able to use the languages to build the application

What's next for Have a seat is changing more of the appearance via css
