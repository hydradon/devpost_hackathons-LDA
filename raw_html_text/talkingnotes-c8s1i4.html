Inspiration

A passion for music and development led us to create this action on google.

What it does

Talking notes aims to guide the user from their current skill level and adapt a curriculum to their pace of training. Each session will guide the user through first tuning their instrument, then a set of scales to match the pieces to be included in any particular session, finishing with full scores suited to the user's skill level. Users can repeat scales and pieces as desired, with future iterations to include built in practice logs.

How we built it

Using the Actions on Google Console, with Dialogflow and Firebase.

Challenges we ran into

Actions on Google has few resources available at this time which proved to be difficult, since we couldn't find solutions online for many of the issues we ran into.

Accomplishments that we're proud of

We've created a project that we're passionate about, and that we think others can enjoy and benefit from. Before UofT Hacks, neither of us had any knowledge of Actions on Google and we believe we've brought this to a considerably functional point in such a short time.

What we learned

Actions on Google is fairly new, so the development community is fairly small. We'd love to see it grow so that it is easier to develop!

What's next for TalkingNotes

There are many ways to elaborate on this project. We'd of course like to add more lesson content, and also more information for the user about the progress they've made and the lessons they've completed.
