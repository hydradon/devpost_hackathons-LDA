Inspiration

Growing up at a very academically competitive school, we wanted to create a way for students to quickly take notes without having to waste their time. Both Nitin and I, being at Monta Vista had a hard time managing school work with taking notes all the time, so we created a way to find useful and relevant data.

What it does

The program initially welcomes the user, and asks for two inputs in two text boxes: the first asking for the long text to be summarized/broken apart for relevant data and the second text box asking for important keywords the program has to look out for when parsing/summaring the text.

How I built it

We used java, eclipse ide, intelj ide, and atom text editor to build and refine our code.

Challenges I ran into

There were a lot of runtime errors, and had a lot of indexOutOfBound exceptions because of very fine details regarding sub stringing. We eventually were able to debug the code successfully by the end.

Accomplishments that I'm proud of

We are proud of implementing background ui to make a more aesthetic ui for the use.

What I learned

We learned how to append files and properly deal with runtime exceptions due to help from our mentors

What's next for QuickNote

We hope in the future we also hope to implement better ui and make a mobile version of this app.
