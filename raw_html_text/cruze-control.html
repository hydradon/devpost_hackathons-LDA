What it does

Pulls data from the longboard, such as RPM and Speed. Displays this information on a handy wireless HUD so you know this info at all times. Also includes rear lighting system which lets people behind you know whether you are speeding up or slowing down.

How I built it

Multiple Arduino UNOs, Arduino Pro Mini 5V, Bluetooth Modules

Challenges I ran into

Getting all the code to work together, getting the Arduinos to communicate to each other

Built With

  • arduino
  • autodesk-fusion-360
  • c++
