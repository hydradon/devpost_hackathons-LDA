Inspiration

I decided to start this project during Finals week, when I should have been spending more time studying for my Chemistry final!

What it does

This is an Alexa skill to encourage Kids to learn about Chemistry. This skill provides exciting information about the 118 Elements on the periodic table!

How I built it

I created the application using Amazon's Lambda Serverless architecture. It's using Python for the backend.
