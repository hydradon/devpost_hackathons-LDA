Inspiration

The ability to control systems at your fingertips.

What it does

We wanted to construct a versatile circuit design that could control multiple components such as LEDs and motors.

How we built it

We began by acquiring a glove, arduino, and a breadboard. We then designed a circuit that would allow the arduino to recognize different finger combinations. Next, we started working on a code that would take each finger combination and execute a task. To demonstrate an application of this controller we used LEDs that would create a unique pattern for each combination.

Challenges we ran into

We ran into multiple challenges throughout this project. First was the design. We created multiple circuit schematics that would eventually lead to our final design. Also, we had to be able to get the arduino to recognize each individual finger combination accurately.

Accomplishments that we're proud of

We are proud of the circuit design that we constructed for this project. This design allows us to utilize any circuit component combination without having to heavily modify the circuit.

What we learned

We learned how to implement a circuit that we constructed. Also, we learned about different features on the arduino.

What's next for Glove Controller

We are looking to create applications for the glove controller. Applications include, but are not limited to, controlling an RC car or a character in a video game.
