Inspiration

We have trouble reviewing difficult problems for 112 midterm, so we come up with the idea to visualize the problems and hopefully we can make revision for other students easier.

What it does

Use animations to present problems like list comprehension, recursion and backtracking.

How we built it

We wrote it in python and tkinter.

Challenges we ran into

We have major problem in writing animation for Hanoi Tower problem.

Accomplishments that we're proud of

We made a nice user interface and use strictly tkinter.

What we learned

We learned to write better animation with tkinter.

What's next for 112 ReVision

We can make better UI and animation with superior animation engines and think of visualizing more problems.
