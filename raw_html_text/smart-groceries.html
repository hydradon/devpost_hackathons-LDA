Inspiration

When visiting grocery stores, don’t we all wonder if we are at the grocery store with the cheapest options or the freshest foods?

What it does

We provide values to our customers in two ways, first we provide shopping lists that are tailored to the customers and their potential interest in recipes.

How I built it

Using modern clustering techniques in machine learning, we analyzed over 400 consumer’s annual spendings and categorized them into 5 types of consumers. Utilizing google-cloud vision API, users will be categorized into one of these 5 types of consumers by snipping a photo of their receipts. The backend server is built with Node.js.

Challenges I ran into

Managing and fitting APIs and a different component of the project together.

Accomplishments that I'm proud of

Data visualization using matplotlib.

What I learned

Clustering algorithms and data visualizations

What's next for Smart Groceries

Better frontend UI.
