Inspiration

The Koan is a well know sculpture on the Warwick Uni campus. The Warwick aerospace society logo is the Koan blasting off like a rocket. I haven't made many games before and I haven't made one for the Gamebuino Meta console. I wanted to make a game for it and I decided that an asteroids clone with the rocket Koan would be fun.

What it does

Currently the Koan is displayed on the screen and can be rotated through 360 degrees with the left and right buttons. When the A button is pressed the Koan's engine fires and it flies around. Basic screen boundary detection is in place to stop the Koan from flying off the screen.

How I built it

The library provided by Gamebuino for the Meta and many hours drawing pixel art.

Challenges I ran into

Pixel art is really hard, and writing the pixel values manually in to arrays to be stored in the Gamebuino flash is very tedious.

Accomplishments that I'm proud of

The Koan looks vaguely ok and does something.

What I learned

Pixel art is hard and I need more game dev practice if I'm going to be able to make a playable game in 24 hours.

What's next for Rocket Koan Game

Addition of asteroids, ability for Koan to shoot, collision detection with asteroids, Koan and projectiles. Game over detection and scores. Basically, making it a playable game not just a demo.
