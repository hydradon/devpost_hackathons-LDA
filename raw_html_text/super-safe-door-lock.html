Inspiration

We needed a project idea, and had an Arduino kit. so going though the kit we brainstormed while searching though the parts, and we came up with the idea to create a door security system senses we had the parts we needed in the kit.

What it does

This system uses an RFIO read to read a key fob that can open a door as well it has a numbered keypad. so the user has the choice to use the fob or the key code. Also there is a two step verification with a mobile phone app that has a dynamic key to open the door.

How I built it

using an Arduino mega,python code, aws database, a mobile phone app.

Challenges I ran into

hosting the database and connecting to it what challenging at first.

Accomplishments that I'm proud of

The working Arduino with pushing that data into an aws database.

What I learned

we learned a lot about the aws services.

What's next for Super Safe Door Lock

The future would be to support more than one device. for example multiple doors.
