• The Quiz Machine

Inspiration

We wanted to make an easy and fun way to educate ourselves and others about some important fact about women in computer science.

What it does

WomENcourage is a fully functional single player quiz game. In case of a correct answer, a green light turns on the audience has to applaud VERY loudly in order to proceed to the next question. In case of a wrong answer, a red light turns on and there is no applause.

How I built it

We display questions on an LCD, and the player can answer with "yes" or "no" by pressing a green or a red button, respectively.

Challenges I ran into

Keeping components in place.

Accomplishments that I'm proud of

The questions are related to women in computer science and can easily be customized.

What I learned

We learnt to use the Arduino platform.

What's next for WomENcourage Quiz

This project is challenging enough, but also fun, to use it in high-school environment with an appropriate manual.
