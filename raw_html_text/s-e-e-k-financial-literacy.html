Inspiration

We wanted to make a fun way to learn common financial topics a college student might ask concerning filing taxes, using credit, loaning money, and more!

What it does

Learn from a specially curated FAQ section (designed with college students in mind) that covers the basics to reaching financial literacy, then test your knowledge with some engaging mini-games.

How we built it

We built the website using HTML, and styled it using CSS. The games were built with JavaScript and jQuery,

Challenges we ran into

Learning to use GitHub, merging/consolidating code from four team members, making sure there were no repeats in variable names, and more.

Accomplishments that we're proud of

Adorable pixel art and mini-games, a functioning website, a comprehensive learning guide, and more, all in less than 24 hours.

What we learned

Serious collaboration skills, how to use JavaScript with HTML, intense styling with CSS, and that coffee doesn't taste bad.

What's next for S.E.E.K. Financial Literacy

Make more mini-games, add a streak function to encourage continued participation, additional content to Learn, and update with relevant news.
