Inspiration

When one comes across a Tinder profile, what one is presented with is not necessarily what one is looking for. People can pretend to be whoever they want to be, and ultimately develop an online persona that does not reflect who they truly are. This idea is reinforced by the fact that 54% of online daters have felt that someone else seriously misrepresented themselves in their profile. Thus, a platform in which genuine relationships can be achieved is necessitated.

What it does

Quinder is a dating app that allows potential soulmates to find the perfect match via answering a specific questionnaire and pairing those with similar results.

How we built it

We used the firebase API to set up a database and a login system for this web app.

Challenges we ran into

We had to manage time effectively as most of our team consists of first-time hackathon hackers.

Accomplishments that we're proud of

We were able to build a project that satisfied its purpose

What we learned

We learned the value of teamwork and how collaboration leads to successful projects and efficiency.

What's next for Quinder

A machine learning algorithm aspect to teach the computer what kind of questions are the most effective and how it corresponds with how long a match will last, thereby improving the efficacy in which people are matched.
