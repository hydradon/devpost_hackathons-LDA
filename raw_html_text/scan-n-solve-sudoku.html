What it does

It is an android app that detects a sudoku grid in image and then solves it.

How I built it

We used opencv to detect the grid by running computations like contour mapping, edge detection and thresholding. Then we used tesseract to detect digits.

Challenges we ran into

Our biggest challenge was to detect grid and make changes on it so that all digits are clearly visible. Another challenge was to break image into 81 parts and detect digits.
