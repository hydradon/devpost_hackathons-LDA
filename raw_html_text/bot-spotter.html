Inspiration

We wanted to use machine learning models to get a better understanding of customer impact of company production issues and we found that a majority of the responses that we were analyzing were coming from bots.

What it does

This model uses the twitter API to gather information about a user and then it sends it to a server that has a model that determines whether the characteristics of the user make it likely to be a bot.
