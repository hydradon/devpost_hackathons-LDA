Website-Sandbox

A website sandbox, where you can build a website while on a website, and solve coding puzzles. The sandbox has input boxes for HTML, CSS, and JavaScript.

Use WebsiteSandbox.html to use the website. Run either locally, or access online, at https://cse.unl.edu/~sclegg/WebsiteSandbox.html
