Inspiration

The web is not truly democratised until everyone is comfortable with code.

What it does

Copy and paste your code into the app, we tell you how to read it and what it means.

How we built it

We used IBM Watson's Natural Language Classifier (NLC)

Challenges we ran into

Bluemix was not running properly so we had to manually train Watson.

Accomplishments that we're proud of

Rebuilding our APIs after the system crashed. Creating a working app.

What we learned

Node Red, Bluemix, the NLC

What's next for To Be Determined
