Inspiration

I was mostly inspired by this hackathon. I have never participated in one of these before let alone create anything on an Alexa device, so I thought I'd try to make a skill using some free APIs that were available online (hence the focus on cats).

What it does

Cat Bot is a simple Alexa skill that utilizes APL to show cat pictures, tell cat jokes, or tell cat facts.

How I built it

Cat Bot was built using Node JS hosted on a AWS Lambda.

Challenges I ran into

A few challenges I ran into were setting up a decent workflow to debug my Alexa app and making the output from APL look nice when displayed on a screen.

Accomplishments that I'm proud of

I'm proud that I forced myself to see this challenge through and I'll be even prouder to see it live for anyone to download.

What I learned

As for what I learned, I feel that I could make an even better Alexa skill in the future if I was asked to do so.

What's next for Cat Bot

My immediate concern is getting Cat Bot through certification. After that, who knows?
