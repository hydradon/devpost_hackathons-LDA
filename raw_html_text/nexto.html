Inspiration

We are rethinking an organising tools for event holders and make information about events more assessable for all Reykjavík visitors.

What it does

It's built in three ways, first it help tourist and locals to find events nearby that are happening. It helps event holders to share their events to locals and tourist and especially to those that don’t have smart phone. Furthermore it helps event holders to manage their infrastructure and keep event history in good place. Finally it lead the tourist/local through Reykjavik city to the event that they want to see and explore.

How we built it

It is one database with multiple connection

Challenges we ran into

We did a market research and found out is that it is hard to have access to all information about all new and updated events happening right now from event holders. More attention and accessibility is needed to smaller pop up events then the bigger events.

Accomplishments that we're proud of

How well we were able to see our project involve through the entire time, and how deep we got in to Icelandic culture. We saw that there are many things in Iceland that tourist are not able to see and its nice to be able to make a solution to help tourist to see all the best of Iceland.

What we learned

Learned to work together. Split project, learned much about Icelandic culture, tourist attraction and a lot more.

What's next for NexTo
