Inspiration

Everywhere we go there is someone that's being bullied. When an individual is bullied he or she loses a lot. I want to alleviate this loss as much as possible

What it does

Provide a safe haven for individuals that are/was bullied

How we built it

We used azure and Wordpress. Also a bit of Python

Challenges we ran into

Deploying our machine learning algorithm

Accomplishments that we're proud of

Being able to create an application out of Microsoft azure

What we learned

Microsoft Azure provides a lot of resources, many of which are quite useful in real world applications

What's next for Bullied Anonymous

Incorporating the machine learning onto the website to have a chat Bot that can encourage and help people
