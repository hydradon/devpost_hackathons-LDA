Inspiration

In 2017, cryptocurrencies were blowing up and people (myself included) were obsessed with checking the price of Bitcoin and Ethereum every second.

What it does

It simply tells you the price of either Bitcoin or Ethereum

How I built it

The skill is rather simple, it's sends a request to a public API that provides prices of various cryptocurrencies in various fiat currencies.

Challenges I ran into

Parsing the response correctly.

Accomplishments that I'm proud of

Make it work with different kinds of utterances.

What I learned

How to use Alexa skills to hit external APIs

What's next for Crypto Price

Add support for more crypto and fiat currencies.
