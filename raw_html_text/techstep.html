Inspiration

When I'm not at hackathons or , I'm a fitness instructor. I teach a high intensity interval training class called 30/60/90 at several gyms in NYC

Gyms hire people to count attendance at every class

How it works

Tekscan Flexiforce sensors connected to an Arduino connected to a Adafruit Bluefruit breakout board communicating with an ios app which collects user info through facebook login and communicates with a node.js server using socket.io

Challenges I ran into HARDWARE, SOFTWARE

Problems with bluetooth naming and connectivity. Learning swift for the first time. Issues around hardware robustness

Accomplishments that I'm proud of

We got it working!

What's next for TechStep

Building analytics for gyms and course owners
