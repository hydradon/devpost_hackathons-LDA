Inspiration

To make emotion consistent with the message.

What it does

Uses Google's Vision API to tell the user their conversation partner's emotions.

How we built it

We used html and css for the user interface. We also used react and Google Cloud Platform for the backend.

Challenges we ran into

How to implement the Google Vision API into our project. There was a learning curve for everyone because we all used something new.

Accomplishments that we're proud of

Learning a new language to create and finish our user interface. We are proud of our idea in implementing Google Vision API. We created a team from people we just met.

What we learned

For some of us, this is our first hackathon, so we learned to work together as a team.

What's next for your project

We plan on going back to our project to learn, change, and share our ideas. We may add some new features like different rooms
