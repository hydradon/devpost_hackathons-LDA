Inspiration

We all have passions in design, art, and programming.

What it does

Takes a given image and returns an abstract art form of it.

How I built it

We built it through the image manipulating API, Processing, which uses Java.

Challenges I ran into

Merging all of our code together. Deciding what kind of art forms we wanted to represent.

Accomplishments that I'm proud of

We were able to accomplish our task of turning a given image into something more creative and interactive.

What I learned

How to use the IDE Processing. How to manipulate properties in images.

What's next for Abstraction

Combining all of our artforms and allowing for users to easily access them.
