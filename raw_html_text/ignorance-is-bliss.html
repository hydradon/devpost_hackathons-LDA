Inspiration

Tired of seeing useless or boring content on my social media feeds. Don't wanna see TV show spoilers? Hide them! Don't wanna see any more about Trump? Hide him!

What it does

It lets you choose keywords or phrases, a content script finds them on Reddit, Twitter and Facebook, and automatically hides them.

How I built it

Using Chrome API, with some prior knowledge of page scraping.

Challenges I ran into

Case-insensitive string matching, problems with finding correct Div element to hide.

Accomplishments that I'm proud of

It works!

What I learned

All about chrome extension message passing, local storage, and the attribute setup of popular social media sites

What's next for Ignorance is Bliss

Expand to more websites, better UI and UX, case-insensitive string matching.. release to chrome store?
