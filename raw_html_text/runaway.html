Inspiration

We wanted to make traveling on a budget easier and faster for people who just want to get away for a while.

What it does

Runaway's map displays jetBlue flight packages based on the user's current location. Once a user selects a destination and a package that fits his/her budget and tastes, they can view a list of all possible similar getaways and further filter them.

How I built it

We built this using mongodb, PHP, HTML/CSS/JS, and jQuery.

What's next for Runaway

Getting more/real flight data.
