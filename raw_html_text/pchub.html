Inspiration

The Peace Corps needs a hub for all the resources that a volunteer may need, including apps and FAQ's.

How it works

The website includes links to each app, a forum and chatbox to connect to members, a calendar of events, and connections to all the Peace Corps social media.

Challenges I ran into

We had used a web server that would not allow log-in capabilities without payment, so despite having a working log in system, we could not connect to our website.

Accomplishments that I'm proud of

We completed a functional, visually appealing website that we believe can be used by actual volunteers.

What I learned

We learned that PHP cannot be implemented on the free Weebly account, and that more research into the capabilities should have been done earlier.

What's next for PCHub

The website would be moved to a different server and PHP would be used to add login credentials.
