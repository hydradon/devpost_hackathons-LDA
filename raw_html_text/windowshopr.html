Inspiration

Everything these days can be found by just a swipe Left or Right. EXCEPT SHOPPING! WindowShopr’s goal is to combine aspects from popular services such as Craigslist, and Tinder to invent a new way to shop.

What it does

WindowShopr is a service where users and register to add listings to our database under provided categories. These categories can then be browsed as "windows". Users swipe left or right to go through products. The seller gets notified of the swipe lefts and can contact back the prospected buyer for a sale.

How we built it

We built the app using the Materialize Framework for all the front end UI. For the backened we used Node.JS with Express and MongoDB using the mongoose library for a database.

Challenges we ran into

A challenge we ran into was learning the Materialize Framework features to integrate them into what we planned. Another challenge we faced was merging the API functionality we wrote into the web server UI codebase.

Accomplishments that we're proud of

What we learned

We've advanced our skills in Web Design and Node.JS backend frameworking since the beginning of the event :)

What's next for WindowShopr

Integration into other store databases such as the official Craigslist app, Walmart, Best Buy, Amazon, and others.
