Inspiration

I like learning about cars, and felt that an Alexa skill could help spread knowledge about automobiles.

What it does

Upon request, Automobile Geek. tells the user a random automobile fact.

How I built it

I built Automobile Geek. with Node.js using the alexa-sdk NPM package.

Accomplishments that I'm proud of

Automobile Geek. is certified and available for download and use on Alexa-enabled devices.
