Inspiration

Social media challenge #Trashtag: The online challenge cleaning places up https://www.bbc.com/news/world-47536861

What it does

Collect water sample data using crowdsourcing

How we built it

  • brainstorming
  • define the actors (WWF/ Textile companies like H&M,Zara.. / Fashion addicts)
  • find the connection between actors ( App user: Fashion addicts / Partners: Textile companies )
  • define incentives for user & partners
  • define the functionalities for the app

Challenges we ran into

  • how to validate data
  • how to engage users

Accomplishments that we're proud of

  • implemented camera to take the picture and vote right away on this picture (dirty or clean)
  • add animation for user experience (fun & visual feedback for actions)

What we learned

  • new libraries for camera and animations
  • new framework (angular)
  • transforming web app to mobile app with android studio
  • how to use database with firebase

What's next for Water Watch Foundation

  • implement all the functionalities we imagined (send the kit to the user and analyze data with firebase machine learning)
  • run proof of concept with real user
