One day, we did not know whether to wear shorts or pants to school. Thus, the Weather Selector was born!

By inputing a simple factor such as the temperature, and checking a few simple boxes, you can find out what you should wear to stay safe and comfortable in this ridiculous weather!

Learning how to use github

I am proud that this can help someone having trouble picking out an outfit.

A next step is to use a weather API so the user will not even need to know what the weather is that day!
