Inspiration

We noticed that a lot of clubs have inconveniences with recording and tracking financial transactions, so we developed CheckCheque: a web app that allows club leaders to understand where and when transactions occur, along with their details. The benefits of using blockchain for this application is because:

  1. It is immutable - it cannot be changed or tampered with.
  2. It is decentralized - it cannot be manipulated.
  3. It is open - all respective students and faculty have access to this information.

What it does

It is a website that utilizes the idea of blockchain to facilitate university club fundings and provide robust security to club management.

How we built it

We used mainly html, javascript, and php to build it.

Challenges we ran into

One of our main problems was integration and trying to create a database for the blockchain program.

Accomplishments that we're proud of

We managed to implement blockchain program, and learned about web development.

What we learned

We learned a lot more about web development. In addition, we discovered how blockchain works and its many benefits to society in the future.

What's next for CheckCheque.tech

We plan to incorporate a system for students (members and execs) to use the platform and function differently from one another. Future developments also include security and cryptography measures.
