Inspiration

Making predicative movie analytics better

What it does

Uses text analytics to analyse scripts and logline to predict the most successful outcomes for comic book movies.

How I built it

With a little help from my friends using IBM Watson, Box Office Mojo, IMDB, and Glitch

Challenges I ran into

Coding the AI, getting the right algorithms for the system to have some predictive value.

Accomplishments that I'm proud of

Having a go at this even though it didn't completely work --yet!

What I learned

A lot about what's possible in AI and that with more work it would be possible to make something like this that had predictive value.

What's next for MarvelOS

More work, more tinkering, more movies, more inputs!!!
