Inspiration

We took inspiration from the Pokedex and thought it would be cool to make a real-life version for animals.

What it does

It recognizes and manages animals you have encountered. Additionally, it gives you information on animals you have seen. This application is in the form of a Flutter cross-platform mobile application.

How I built it

We created a Node.js backend API consumed by a Flutter frontend mobile app. The Wikipedia API was utilized to collect additional information for each animal. Google's Vision AI was used to detect animals in the images.
