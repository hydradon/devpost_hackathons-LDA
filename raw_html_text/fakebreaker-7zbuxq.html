Inspiration

Hi guys! Are you tired of being fed fake news? How many times have you had to proof check information? Our team came up with Factbreaker that makes life a little easier in avoiding fake news.

What it does

Our solution is a simple chatbot and chrome extensions that allows you to easily detect lies

How we built it

We created the working chatbot and chrome extensions, we used existing APIs like Google Cloud keyword detection and high level libraries in python that allowed us to prototype this idea really quickly.

Challenges we ran into

The main challenge was building deep learning model using LSDM, and getting right data to train this model on. We also changed the approach, and validated that ours approach should be good.

Accomplishments that we're proud of

We feel the idea is very hard, so we are proud that we done it, and it actually works

What we learned

A lot! We did a lot of algorithms we have implemented only once previously, or completely new ones.

What's next for FakeBreaker

We want to improve deep learning algorithm, and add support to multiple languages, which open us up to new market opportunities.
