Inspiration

This is a practical implementation of a voice based searching system which looks for the lowest priced groceries in your area and returns the item, price, and store location. There is always one store that gives a certain product for a cheaper price so we wanted to find it.

What it does

The google home/ voice assistant prompts the user for the product they are looking for and lets them know where the item is cheapest within their area.

How we built it

Dialogueflow is used for Voice input coding, Node.js is used for the over all programming along with firebase and python is used for scraping and saving data from websites.

Challenges we ran into

Had to play around for a while to integrate the database part.

Accomplishments that we're proud of

We have a working model that returns the cheapest price of a certain grocery item in your vicinity.

What we learned

Dialogueflow and Node.js

What's next for Local Cheaps

We want to make the data much more identifiable and we would like to expand our selection in the database, so it is of more use to the customer.
