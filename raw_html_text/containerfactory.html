Inspiration

While building a deployment service at lsq.io, we came across the problem of the enormous time and effort it takes to produce docker images.

How it works

  1. Choose your Github repo once you have authenticated in.
  2. Fill in the required registry information.
  3. Watch the image get built.

Challenges we ran into

The container factory integration is not easy.

Accomplishments that I'm proud of

We made a pretty nice website with some awesome copy, but more importantly we have made a service which will benefit many developers.

What I learned

We built the entire front end in Reactjs which was a first for us.
