Inspiration

We tend to forget who owes us money sometimes. These debts may hence simply go away, burning a hole in our pockets.

What it does

This Telegram bot (@pearlinebot) allows us to type in all transections into the chat. At the end of the month, when we will like to check on all the transections that we have, we simply type in "done" and it'll show us the whole list of transections that we have. This ensures that we do not miss out on any.

How I built it

We built it using Python and the Telegram Bot API.

Challenges I ran into

We did not know how to put our Python code into Telegram Bot API for it to work as a Telegram Bot and had to ask our friend for help.

Accomplishments that I'm proud of

The bot is actually working!! We came with close to zero coding experience and are very proud that something managed to come out of this.

What I learned

Coding is not easy and we need a lot more practice! However, coding is very fulfilling when you actually get something to work!

What's next for O$P$

Can include the function of who you borrowed money from, so you will be able to keep track of who you have not returned money to yet. Will also be useful if we can update whenever a debt has been cleared and have an updated list sent back to us via the bot.
