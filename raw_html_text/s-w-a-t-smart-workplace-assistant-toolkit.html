Inspiration

We experience empty paper towel in our office toilet area on a regular basis :(

What it does

It recognizes low supply levels of consumables like paper towels or liquid soap, notifies an administrator/facility manager or triggers another logistics process on it's own.

How I built it

Using node-red and the TI CC1350 SensorTag ;)

Challenges I ran into

Being mainly embedded developers we had a hard time using JavaScript.

Accomplishments that I'm proud of

Successfully crashed Node-Red by being able to use the same database for application and IDE data. (Because sandboxing is never a good idea)

What I learned

Everything related to JavaScript.

What's next for S.W.A.T (Smart Workplace Assistant Toolkit)

Extend it to be able to learn to differentiate several use-case and behaviors on it.
