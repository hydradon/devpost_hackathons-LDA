Inspiration

Wouldn't it be nice to know what your kids and elderly loved ones are doing and how they are ? Wouldn't it be helpful to know about there whereabouts in case of some emergency ?

What it does

Its a one stop dashboard for all of the data that can be required in an emergency. It collects user's relevant data which can be helpful in an emergency. Not only that, but also provides a dashboard, which provides all the data in a hassle free and easy to use form, which can be accessed and shared in case required by a third party or even the user.

It keeps a track of the movement of your loved ones with pin point accuracy and provides a one touch emergency response incase required.

How I built it

Used the wonderful trnql API of course. And used objective C for development along with Xcode.

Challenges I ran into

How to show the data collected in the most meaningful and helpful way. The main aim of the app is to visually depict the data to the user.

Accomplishments that I'm proud of

Judging myself by the time I took to build the app and the final outcome. I am very happy with the results.

What I learned

How to visually depict the data and save user from unnecessary trouble and boring numbers.

What's next for CareTracker

App would be live soon and depending on the response, new features will be added.
