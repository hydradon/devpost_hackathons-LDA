Inspiration

Two striking young men passionate about saving money and enjoying the finer things in life.

What it does

We communicate with the LCBO API to first find the store closest to the user, and pull the appropriate data as we search through the products. Using Firebase, we send all product searches to the ticker above the search box. As other users in your area search around you, you'll be notified about what is popular in your area.

How we built it

We learned how to consume the API over the course of the weekend, and how to take advantage of the search parameters. We built out the back-end API functionality first, and then designed around it. We provide a variety of search options to narrow down your results, whether you're on a budget, looking for a gift (i.e. the most expensive stuff), looking for a new product, or if you want to forget the long week you had. We have it all.

Challenges we ran into

Displaying the appropriate data in an orderly manner and cross-domain AJAX calls.

Accomplishments that we're proud of

Searching for alcohol products is really easy now.

What we learned

How to use mustache.js for templating, Firebase for real-time updates, and building out around the functionality of an API. 10/10 would do again.
