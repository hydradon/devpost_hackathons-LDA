Inspiration

Money management is difficult. Especially for college students. Peppy is the solution for wanton online spending.

What it does

Peppy monitors online spending and occasionally warns the user about potential budget concerns.

How I built it

A combination of JavaScript, HTML/CSS, JSON, and other chrome extension infrastructure were used.

Challenges we ran into

  • Extreme lack of experience with JavaScript (there was a steep learning curve that we eventually overcame)
  • The chrome extension infrastructure is far from intuitive

Accomplishments that I'm proud of

  • We learned javascript pretty well

What I learned

  • our original ideas were awfully ambitious

What's next for Peppy

  • actually fleshed out functionality
  • applicability to a more general plethora of websites
  • integration with bank accounts for more accurate balance info
  • an independent website for real-time balance updates
