Inspiration

We wanted to create something that's Valentine's Day inspired but with a bit of a humorous twist.

What it does

It creates romantic matches for you based on someone who's the exact opposite of everything you really want in your potential partner.

How I built it

We built out the html and css for a website and than created an algorithm in python. We than connected the two using Flask.

Challenges I ran into

Connecting our python and HTML and getting the information from our form to actually interact with the HTML.

Accomplishments that I'm proud of

The CSS turned out very nicely and finally getting Flask to work was the best thing ever.

What I learned

How to use Flask, and a lot of HTML and CSS!

What's next for Opposites Attract

Creating a log in system as well as creating functions where you get to pick the type of opposites you meet. Someone who will only hate you, someone you'll hate and your absolute worst enemy.
