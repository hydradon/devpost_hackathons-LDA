Inspiration

Since most music is on Spotify normal software can't be used for transposing audio to the desired key, esp. using smartphone.

What it does

Select your song and pitch it to the desired key in real-time for jamming along.

How we built it

Native Android App using Spotify SDK and Pitch Shifting lib from Fraunhofer IDMT.

Challenges we ran into

Getting samples from audio data in real time and manipulate them on the fly for proper playback.

Accomplishments that we're proud of

Prototpye works :)

What we learned

Using Spotify API on Android and processing audio in real time on Android.

What's next for PitchMeUp

Adding possibilty to query songs/playlists Nicer Ui Put to PlayStore Port to iOS Use time_stretching for playing to song at lower tempo
