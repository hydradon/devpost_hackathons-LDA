Inspiration

It is inspired by my dad because he is a stock trader and he needs to work with a lot of information in a relatively short time.

What it does

It finds similar patterns of stock in the history and uses the prerequisite to provide a prediction of the stock's future.

How I built it

I built is using a self organizing map and python.

Challenges I ran into

Lack of Information. As much of the information is locked up as programs like these are sold to hedge funds privately for millions of dollars to give them an advantage they are very likely to give up

Accomplishments that I'm proud of

It can find the same patterns really fast and easy without hard coding the program to do the feature selection

What I learned

How does self organizing map works and how to build it

What's next for Stock Pattern Finder

To use api to make it so that the program autonomously refreshes the data every three seconds to get more precise prediction
