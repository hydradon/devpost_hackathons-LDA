Inspiration

Wanted to do an IoT project revolving around aviation, and came across the idea of home automation, but for planes.

What it does

Monitors and tracks the status of the plane, by various components and values, and sends it to a database and front end to be viewed by a user

How we built it

built a simulator in the C# api, to model three individual flights and what would occur during each flight and the output the site

Challenges we ran into

Ran into issues with C# calculations and sending the data to the website

Accomplishments that we're proud of

Got the api and backend to simulate the flights.

What we learned

We learned how to combine C# api and angular to create a real time appication

What's next for AirFlow

Work on getting the physical hardware working and getting the front end to work properly
