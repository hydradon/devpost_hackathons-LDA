Inspiration

There is too much trouble around splitting a bill

What it does

It is supposed to receive an image of a receipt and correctly split the costs for each person by reading the text and symbols next to the text.

How we built it

Front-end: HTML/CSS/JS Server-side: Python/Flask

Challenges we ran into

Using OpenCV to recognize drawn shapes

Accomplishments that we're proud of

Implementing OCR successfully

What we learned

To have a more solid idea of a hack before starting

What's next for Easier Bill Splitter

Implementing the features that are still not quite there yet.
