Inspiration

To Combine a raspberry pi and an Arduino

What it does

Fully controllable R/C car from a browser based controller with camera feed

How we built it

Used an Arduino communicating with a Rasbperry pi that had a server running on it to host the controller webpage

Challenges we ran into

Getting Node.js to Communicate with the Arduino

Accomplishments that we're proud of

Getting live camera feed to a webpage and controlling motors on an RC car

What we learned

Server side technologies, Hardware, Network Interfacing, Network programming, and collaborate
