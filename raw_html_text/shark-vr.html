Inspiration

This game was inspired by another game, Whale-Come Home, we created at Global Game Jam 2019 couple weeks ago. Although the team members are not original members, the two original members along with a new member decided to make this game next level, VR ready.

How We built it

We built this game in VR. We start everything from scratch in Unity. We built the models in Autodesk Maya and colored with Microsoft Paint 3D.

Challenges We ran into

With only three members, we thought we won't be able to make anything. We were going to develop this game for Oculus Rift. However, the device doesn't work on the laptops we have. It wouldn't connect to the computer. We didn't start the project until the next day. It took us so long to figure out why the device isn't working, so we decided to just develop it on Oculus Go and changed the game idea.

What We learned

We learned how to make a VR game. It wasn't perfect, but there's still room for improvement.

Accomplishments that we're proud of

We got people to spread the word about our game. Some people came to our table saying "I heard this one is really cool"
