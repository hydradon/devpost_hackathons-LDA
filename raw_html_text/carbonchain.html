Inspiration

After taking organic chemistry, we found that the applications were very cumbersome and annoying to use. This will help simplify and help create a good visual element to help study. To help simplify organic chemistry molecule building.

What it does

A user inputs carbon chains like CH3CHFCHFCHFCH3 and then it will print out the line molecule version and it's IUPAC name.

How we built it

Coded it in python.

Challenges we ran into

The two of us have very little coding experience. We have a combined experience of less than a year of coding beneath us.

Accomplishments that we're proud of

Building the basic core code behind the program. Able to print very basic molecules and give them accurate names. Even incorporated in the main halogens.

What we learned

Turtle graphics and pygame.

What's next for Carbonchain

We want to add more features to it, such as ring structures, isomers and more 3-d representation. Develop it into a web app or mobile application eventually.
