Inspiration

During lunch on Saturday, we came up with the idea.

What it does

It allows people to connect through a mobile app in order to find new people to eat with.

How we built it

We used Java with Android Studio and FireBase for our database.

Challenges I ran into

Structuring the database and putting the parts of the code together through GitHub.

Accomplishments that I'm proud of

Learning from scratch a new language and creating a working application.

What we learned

Teamwork, Java, Android studio, FireBase.

What's next for Eat With Me

Implementing other APIs, creating a better interface, improving the algorithms and functions.
