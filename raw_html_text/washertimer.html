Inspiration

People forget about their laundry in the dorm washers, but setting a timer on a phone takes too long.

What it does

Easy to use timer that sends custom text message after user defined time. Users first create an account with a 4-digit PIN, and then simply punch in their pin each time they load laundry.

How I built it

Multithreaded python script with Twilio SMS service, Tkinter GUI, and MySQL database.

Challenges I ran into

Designing database, GUI in python

Accomplishments that I'm proud of

Implementing first database and GUI in python

What I learned

Databases, Twilio

What's next for WasherTimer

Make it smaller and move database to the cloud
