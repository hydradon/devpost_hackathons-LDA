Inspiration

As students, we have both gone through many to do list apps or organizers. However, none of these apps were able to schedule tasks into google calendar.

What it does

Add tasks to your to do list (name, description, due date, approx how long it will take to complete) and they will be scheduled into google calendar during homework times.

How we built it

We used HTML, CSS, and JS. Using firebase we are able to store each users' tasks.

Challenges we ran into

Trying to find empty slots on a user's calendar to create an event in.

Accomplishments that we're proud of

Having most of the to do list functionalities working.

What we learned

Google Calendar API and Firebase.

What's next for Planner

Better calendar scheduling.
