Inspiration

We were inspired by the great deal of infant deaths that occur every year due to unnoticed events in the household.

What it does

Uses MIT developed algorithms to measure the persons heart rate over a multitude of wavelenghts then compiles to a single measurement.

How we built it

Using MIT's algorithms and various online resources, we optimized a code for the 32 bit operating system on the raspberry PI and added various health features.

Challenges we ran into

Reading unoptimized and unorganized source code from other people's projects. Using web sockets to cloud compute our data.

Accomplishments that we're proud of

Developing a working final product.

What we learned

Better use of python, web sockets, visual recognition features.

What's next for Baby Face

Implementing a Node server to view data from anywhere. Acquire a better camera for more accurate(and stable) readings.
