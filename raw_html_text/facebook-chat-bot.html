The idea came from the Eliza bot. ( http://nlp-addiction.com/eliza/ )

Allows clients to create applications that can interact with users in multiple ways.

The applications are built in python, but they can be built in any language that supports Firebase. For the bot, we Node, Python and Selenium.

We had a lot of ideas for different modules, but some proved too difficult or too specific. We also had a very hard time obtaining the information from the facebook chat and posting back to it.

We are proud of learning a lot of new technologies.

I learned that handwritten OCR is a lot more complex than normal OCR and that training systems is not particularly simple.

A more robust API, and also developer tools/examples for some common types of chat bots (customer service, bookings, etc.)
