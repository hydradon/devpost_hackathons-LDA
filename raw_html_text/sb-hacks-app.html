Wanted to progress beyond our previous attempt at building something at a previous hackathon. Step outside our comfort zone and use technologies that we aren't fully acquainted with to build an unique app from scratch.

Learned much more about routing, flow of data, quasi-professional ui design.

Challenges: Staying focused for 8+ hours Sleep deprivation Hardware limitations Differences in understanding stack used

Ruby Version 2.6 RoR Version 5.2.4.1

Git for version control

Backend made with Ruby on Rails, database is PSQL. Frontend done with react, ruby, html and scss. Javascript packed with webpack.

1 table to store users, password data is encrypted with BCrypt

Upon login or account creation, navigates to a page which is populated wit nearby events and restaurants from submitting location data to FourSquare and SeatGeek APIs.
