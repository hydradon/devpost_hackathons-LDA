Inspiration

Alexa slot machine does not allow users to make money, but can give fun.

What it does

Choose a random number and give points

How I built it

Randomly select a number and get points corresponding to the selected number. The collected points increase or decrease in real time.

Challenges I ran into

Initially, in-skill charging was set as a type "consumable", but it did not go well, so switched to "one time purchase".

Accomplishments that I'm proud of

This game can give fun to users.

What I learned

One mechanism can build both educational and game skills.

What's next for Slot game in Alexa

add sounds.
