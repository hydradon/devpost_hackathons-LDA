• Margarita

Inspiration

You ever have some friends over and need someone to suggest an alcoholic drink to make? Just ask Alexa!

What it does

You can ask Alexa for a suggested drink and she will randomly pick one and show you how to make it.

How I built it

Its built with node.js and Alexa Presentation Language.

Challenges I ran into

Alexa Presentation Language is currently in beta so there are many bugs to work around.

Accomplishments that I'm proud of

I'm proud that I was able to develop a nice visual skill for Alexa in a reasonable amount of time.

What I learned

I learned how to use Alexa Presentation Language to make a visually appealing skill.

What's next for Make a Drink

Next steps will be to add more drinks to the random menu.
