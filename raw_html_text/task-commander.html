Inspiration

We plan to reduce stress in pressured teenagers by offering incentives for them to set schedules ahead of time.

What It Does

Our program enables a user to have a less stressful schedule through our reward system. Each user has the ability to create a daily schedule by adding activities and setting time slots. Once the user has completed the set activity within the allocated time, (s)he will be able to receive one point as an incentive (limit 8 points per day). When the user has accumulated enough points, (s)he will be able to redeem them for prizes.

How We Built It

We used Java and the GUI (Graphic User Interface) in order to design the layout, including the buttons and frames.

Challenges We Ran into

We were debating on what platform to code on (Android Studio or Eclipse).

Accomplishments that We're Proud of

We collaborated and remained calm throughout the whole journey. We talked through our differences and helped each other by using each of our skill sets.

What We Learned

We improved upon our collaborative skills and our ability to work under pressure. We also increased our knowledge in Java along with the tedious GUI creation process.

What's Next for Task Commander

Task Commander currently does not have any sponsors to provide rewards. If we plan to release the concept as an application, we would need sponsors along with a revenue generating plan.
