Inspiration

A huge part of success during and after college is having a good mentor and network.

What it does

It is a Tinder like app, where you can swipe left to reject or right to match with mentors and mentees.

How we built it

We build the app using Android Studio and Firebase.

Challenges we ran into

This was our first time using Google Cloud Service and Firebase, so managing the database through Firebase had a strong learning curve.

Accomplishments that we're proud of

Getting familiar with Firebase and Google Cloud Platform and its implementation through Android Studio as our second hack ever.

What's next for Mentor Match

Implementing more features such as more options for making one's profile, finishing setting up chat, and much much more!
