Inspiration

We've heard a lot about fake news floating around so we wanted to create a website to tackle this issue.

What it does

It has the most current news from all over the world and provides multiple perspectives on everything to ensure that it is unbiased. We also include an unbiased background to educate people about the issue.

How I built it

We built it using HTML and CSS and we used a text editor called Atom.

Challenges I ran into

Most of us did not know anything about HTML so we had to teach ourselves a whole new language. It was also difficult to write the background section without being biased.

Accomplishments that I'm proud of

We taught ourselves a new language and managed to make a fully functional website using our new skills.

What I learned

Our entire team learned something new. We taught ourselves HTML and we also learned more about current events.

What's next for Inform

We want to add a section where people can send in stories about events that benefit them and they can be posted on our page. We'd also like to expand and cover more events, as our current website only has information about one event. This will hopefully make the app even more user friendly.
