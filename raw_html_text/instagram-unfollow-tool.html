Inspiration

Wanted to see who haven't followed me back / unfollowed me on Instagram but there was no easy way to do this. As a result, we built this.

What it does

Compares the list of people you follow to the list of people that follow you on Instagram and presents the people that don't follow you back easily.

How we built it

We utilized Python to build this project, using GitHub to easily collaborate.

Challenges we ran into

Trying to determine which elements of Instagram's HTML to have our script scrape to navigate through Instagram.

Accomplishments that we're proud of

Building a tool that actually works within a short period of time.

What we learned

We can do a lot in a short amount of time with proper focus. Also, we became very familiar with utilizing Selenium for web scraping.

What's next for Instagram Unfollow Tool

Added images of the people that don't follow you back, a more aesthetic design, etc.
