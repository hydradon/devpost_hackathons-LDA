Blokta.com

A simple blogging platform for developers. Markdown based blog entries and a lightweight vue front end.

Blokta is a blogging platform for developers. The platform is designed to link to a Git repo, which stores markdown documents and image assets, and update the blog when you publish releases.

How I built it

Okta provides authentication for bloggers to login and authorize blokta to their git repo. Twilio provides mobile notifications to blog subscribers about new posts.

Challenges I ran into

Okta vue documentation is out of date. Had to resort to forum posts.

What I learned

How to configure the Okta deployment. I was not able to enable user signup though.

What's next for Blokta

Git powered file storage for storing the markdown files. More UI, polish, and a MongoDB backend.
