Inspiration

Wondering if my Microsoft Band could be used as metronome while I practiced trombone.

What it does

This application allows a developer to test the various haptic notifications on the band and set the frequency that these notifications are sent.

How I built it

I started with the examples in the SDK then built up the notification example.

Challenges I ran into

Lack of familiarity with tools involved.

Accomplishments that I'm proud of

My first hackathon submission ever!

What I learned

The idea is more important than flawless code. Relax, your project didn't exist 24hrs ago. So, don't worry about it being perfect. Presenting haptic technology is a little tricky. I actually had to wrap my band around the microphone so that the crowd could hear it vibrating.

What's next for Haptic Tester

Figuring out a solution for the Bluetooth delay.
