Inspiration

GPS navigation falls short sometimes. We leverage tour guides' knowledge to fill in the blanks.

What it does

We connect travelers with tour guides in real time.

How we built it

The user interface for the wanderer was built as a native iPhone app, while the guide dashboard is a React web app. To connect the guide and the wanderer, we built a Django backend and integrated with Twilio’s VOIP API for placing in-app phone calls. We used Esri’s mapping API to plot the real-time location of the wanderer in the guide dashboard, using coordinates logged from the wanderer’s phone with Verizon Thingspace. In addition, we pulled real-time travel guide recommendations from Esri’s Content API.

Challenges we ran into

It was our first time using the Twilio VOIP API, and it was a bit of a challenge to get everything running smoothly. In the end, the Twilio’s API and SDK’s were integrated on all three applications: the Javascript web app, Python backend and Objective C mobile app.
