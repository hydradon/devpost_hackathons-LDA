Inspiration

We wanted to make an app to help people in new environments find interesting activities to do near them with a group of people.

How I built it

We used MongoDB and NodeJS for the backend and a Web Front-End using HTML/Jquery. We also have an Android App to be used as a frontend

I am brand new to MongoDB, so that was an interesting and exciting challenge. I am also incredibly new to NodeJs which made the whole process a huge learning experience

Accomplishments that I'm proud of

Creating a backend/frontend using the Maps API and using technologies we were not familiar with

What I learned

I learned a ton about MongoDB and NodeJS

What's next for HangoutAndChill

More UI and enhancing the frontend to take full advantage of the backend.
