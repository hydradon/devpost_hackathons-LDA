Inspiration

People excel when they are limited. Twitter only allowed 160 characters. Snapchat used to only allow one line of caption text. The fictional app "Bro" from Silicon Valley only allowed people to send the word "bro." Now let's take that to emojis. What if you could only type in emojis? Better yet what if you had a keyboard where you could type normally and then convert to emojis? This is exactly the purpose of Emojer
