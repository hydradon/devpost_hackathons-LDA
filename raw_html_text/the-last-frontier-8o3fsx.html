Inspiration

We wanted to create an underwater game using virtual reality and some sort of interactive hardware.

What it does

Interact with your surroundings underwater while shooting at targets.

How I built it

Using the unity game engines coupled with the oculus rift and leap motions SDKs.

Challenges I ran into

Getting everything to work with one another, version crossovers were a big issue.

Accomplishments that I'm proud of

Getting a finished, working, really awesome product.

What I learned

More about hardware hacking.

What's next for The Last Frontier

Nothing.....it's The LAST Frontier.
