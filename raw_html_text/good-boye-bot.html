• The real Good Boye Bot

Overview 

A fun bot that adds some terminal-like functionality to a discord server

What it does 

Main Features:
- User management: allows for multiple permission levels to run commands
- Image hosting: store your most important memes on the bot server
- Behavior management: Gives points based on user behavior, and takes points for misbehavior

How we built it

  • JDA: Java Discord API
    • Powered by OKHttp, Apache Commons, JNA, SLF4J
