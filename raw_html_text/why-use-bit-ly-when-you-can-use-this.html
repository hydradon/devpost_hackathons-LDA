Inspiration

Bit.ly!

What it does

Instead of using bit.ly to redirect your links with a boring and predictable url, use this to generate a one-of-a-kind url that you'll definitely remember!

How we built it

We created a web app using Flask, used the favqs api, and designed our own background/images.
