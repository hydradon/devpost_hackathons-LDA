What it does

Allows users to give small loans to people in need in your surrounding area to help pay for emergency needs, equipment, or to improve thier employability standards (ie. prepare for an interview).

How we built it

We build an android app and did server side programming in php. We used Amazon Web Services for storing our database on cloud
