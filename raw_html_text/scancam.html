Inspiration

We saw something that we wanted to get and we didn't know where to get it. We wanted to search that object up.

What it does

Using your phone's camera, it identifies the object and searches it up via google.

How we built it

We used Swift and Clarifai's API.

Challenges we ran into

We had trouble with identifying the correct object.

Accomplishments that we're proud of

We created an app in one and a half hours.

What we learned

We learned about time management.

What's next for ScanCam

Update the data for the machine learning identifier so that it can detect the correct object.
