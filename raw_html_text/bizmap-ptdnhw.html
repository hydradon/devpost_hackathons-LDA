Inspiration

We wanted to create a product that's for everyone. This goes from the everyday business owner to large scale companies. By using Fiserv's SDK and API we wanted to create a simple visual representation of data. This is removing barriers between the person that wants to analyze customer data but lacks advanced analytical skills.

What it does

Our web application, BIZMAP, presents data in various graphical forms generated from the customer and transaction information obtained through Fiserv's API. IT provides the business owner a clear understanding of their transactional data. It uses information from transactions to categorize the state, city, and zip code where purchases were made. This web app can be experienced more intensively with the Oculus Rift.

How we built it

We hosted our sit on Firebase, and used Javascript, CSS, and HTML to create the web application. We used Postman to create customers and utilized Fiserv's SDK and API. Slack was used to communicate and share ideas.

Challenges we ran into

Learning new technologies in a short amount of time is challenging but we were able to prevail. We purchased a domain, www.thebizmap.org , however, could not successfully connect it to our hosting, Firebase.

Accomplishments that we're proud of

By gaining new skills, experimenting, and attending workshops, we are very proud of the progress we have made. The brand design, web design, and concept are representative of the hard work we have put in.

What we learned

Art as a medium of information that can increase the interactivity of the user and the application. We learned how to properly use an Oculus Rift, Firebase, the command prompt, JSON, Postman, and more. We also learned how to collaborate and be a succesfule development team.

What's next for BIZMAP

We designed our product to have VR capabilities. By designing for VR Experience, we set up BIZMAP for an exciting future. Our focus on visualization and interaction with the data. We have plans to experiment with Augmented Reality and creating 3D maps with our modeling skills.
