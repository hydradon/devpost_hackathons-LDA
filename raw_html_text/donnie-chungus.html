DONNIE CHUNGUS

A Terrible Hack by Jack Bishop and Zach Trefler

tired of being a BETA MALE? want to tweet like DADDY CHUNGUS himself?

Then clone this repo! Donnie Chungus is a character-level RNN that turns your LAME tweets into those of the biggest Chungus of all!

Setup

Clone the repo: git clone https://github.com/JohnBish/Donnie-Chungus.git Install virtualenv: pip install virtualenv And use it to create an environment: 

virtualenv env --python=python3.7
source env/bin/activate
pip install -r requirements.txt

Start the backend server: python backend/server.py Access the frontend at ( https://donnie-chungus.firebaseapp.com ).
