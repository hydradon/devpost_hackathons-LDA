Inspiration

Teaching children about STEM and coding is not a new idea, in fact there has been a tremendous rise in the number of opportunites out there for the next generation to learn about programming, but not nearly as much focused on the logic and hardware behing the scenes; enter Lil' Circuit!

What it does

Lil' Circuit is a fun interactive interface to help teach logic principles in the form of a game.

How we built it

It was created with the usage of a raspberry pi with a sense hat, all written in python.

Challenges we ran into

There were challenges all along the way with getting our game to be as involved as we wanted it to be, so with our next update the game would receive multiple levels of difficulty and an unlimited amount of unique gameplay experiences.

Accomplishments that we're proud of

Lil' Circuit is completely portable, which is awesome! With a bit of a charge you are able to take this fun game where you want.

What we learned

Sometimes hardware isn't the hard part.

What's next for Lil' Circuit

We would love to reduce the size of our game and give more levels to make it more suitable for small fries.
