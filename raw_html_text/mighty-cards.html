Inspiration

Alexa games and cards games are great fun, so myself and John teamed up and decided to combine the two together to create a card battling game for Alexa.

What it does

Mighty cards is a card games that allows you to use some awesome character cards to use in battle against other CPU controlled cards. Every user gets 50 cards for free, each with their own voice over and card stats. ISP allows users to purchase additional card decks to add to the game, starting with the 8 Bit themed deck which gives 30+ additional cards to use.

How we built it

Mighty cards is built using the ASK-CLI using node.js. We also use an API to load in the cards for each deck as the game is played. It also was built with lots of blood sweat & tears :P

Challenges we ran into

Content is king in Mighty Cards. We wanted every user to have lots of free content from the get go, and then have the ability to add more via ISP. Making sure all the content was ready on time, and each card having its own voice over was very time consuming, but I think the end result speaks for itself.

Accomplishments that we're proud of

Adding lots of free content, along with ISP is a big achievement. Working together as a team on this skill has also been a great experience as we have been able to bounce ideas off of one another, and we have both been able to bring different skill sets to the table that adds value.

What we learned

Deadlines are hard to hit when you want to make the best skill possible. There is so much more that we want to add to the skill, and we had to learn to compromise in order to have the skill and our submission ready on time.

What's next for Mighty Cards

Where to begin? We have two more ISP packs that we have ready to add to the skill, we will be adding different game modes, global leaderboards, tweaks to the battle mechanics, more APL, and we are looking into utilising the HTML beta to add more interactivity.
