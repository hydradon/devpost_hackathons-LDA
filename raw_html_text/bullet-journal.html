Inspiration

A calendar and journal all in one place...

What it does

A Bullet Journal exceeds the functionality of a normal journal or calendar because it keeps track of a wide range of categories including your habits, interests, goals, and much more. For example, our Bullet Journal has a monthly and weekly calendar and keeps track of your health and fitness, any restaurants you want to try out, recipes, music, movies, TV shows, books, bucket list ideas, travel destinations, and much more. The purpose of Bullet Journal is to be a clearly laid out calendar that can also record entries for each of the categories mentioned above all in one place.

How we built it

We built it using a text editor "Atom". We used scripting languages, mainly HTML and CSS.

Challenges we ran into

We tried implementing this in NetBeans and had a lot of issue setting up the Tomcat server. We also had issues with integrating our code.

Accomplishments that we're proud of

Having something that's practical, easy to use, and can be used by anyone.

What we learned

We learned that working in teams require a lot of collaboration and that front end coding is tedious if you're not familiar with HTML and CSS.

What's next for Bullet Journal

We want to expand upon Bullet Journal's features and integrate a server, so the calendar can be updated and allow for live user input.

How to run it

Our project must be downloaded to the C drive because we used relative path names.
