Inspiration

Self checkout kiosks are great for reducing long lineups at large stores. This integration of machine vision and image recognition training allows customers to check out items without the need for barcodes.

What it does

This device scans and identifies items for checkout, and provides a total at the end of the transaction. This device also allows for images to be stored for training the image recognition software.

How I built it

The software was built in Python, with integration with Azure Cloud Services

Challenges I ran into

Dealing with communication issues between Azure and laptop, getting the camera to work, parsing JSON, getting Dragonboard to work.

Accomplishments that I'm proud of

This is the first time using Python and machine vision for me.

What I learned

I learned to develop in Python and integration of Azure services

What's next for Custom Vision Quick Checkout
