Inspiration

Wanted to create a more safer city.

What it does

Help police receive, track and process crime in the city. Can be used to find patterns of crime activity and prevent future crimes.

How we built it

We deployed this on gCloud using Flask and Mongodb, which use the front end built on React, Plotly and Dash. We used Spacy for language processing.
