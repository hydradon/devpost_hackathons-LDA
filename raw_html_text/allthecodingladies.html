Inspiration

Social Good, Empowering women, education and Beyonce

What it does

Teaches coding in an easy, non-intimidating way. It also provides a one-stop access to information about our field, hands-on experience and networking opportunities.

How we built it

We broke down the bigger problem into smaller subtasks. Next, we broke our team into pairs to program different parts. We used HTML, CSS, Javascript to build the website and Mock-ups to develop the app.

Challenges we ran into

We were new to using GIT meaning that we ran into issues understanding how to use it. In deploying the website, we struggled to set-up to the server.

Accomplishments that we're proud of

As new hackers with non-tech group members, we are proud to have built a working website and MVP app design that we can demo.

What we learned

We came into the Hackathon as two separate groups, but we came together and worked well as a team even though we had just met. The new hackers with no experience learned how to use new programs that they had never seen before.

What's next for AllTheCodingLadies

We aim to build a working app that can be deployed to spread what we have learned during this Hackathon.
