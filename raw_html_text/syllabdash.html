Inspiration

We wanted a way to upload our assignments that were on our syllabus to our calendar to remind us of upcoming assignments.

What it does

Parses through the syllabus, finds the assignments in tables, and uses the Google Calendar API to upload them as events to a user's calendar.

How we built it

Using python, Django, HTML/CSS, Heroku & the Google Calendar API

Challenges we ran into

Poor documentation of the Calendar API and difficulties allowing submission of pdf files.
