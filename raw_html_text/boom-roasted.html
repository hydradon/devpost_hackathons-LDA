What it does

Reads your face, checks most prominent traits and roasts you based on previous roasts for that kind of face

How we built it

Using the school supplied linux server and an android device

Challenges we ran into

Machine learning was too complicated to implement, time is short and roasts are few

Accomplishments that we're proud of

Finished the app

What we learned

All the APIs, some more complicated and more finnicky than others
