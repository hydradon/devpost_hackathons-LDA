Inspiration

Ever experience a moment of genius? See something that moves you to tears? Watch one of those super motivational videos on YouTube? What if you wanted to track these and come back to it for future inspiration? Now you can!

Want to share an inspirational story? Want to check out what's moving other people? We aim to do just that.

What it does

We fetch information from all over the Internet using a Chrome extension and use of the respective APIs to fill our backend. We also gather information using an iPhone app that can take pictures and generate voice recordings (and throw it in the backend). Then magic happens (we hope). We try to create 'Stories' for users to use as motivation. A Story can comprise of Tweets, Instagram posts, a really cool YouTube video... Or maybe a cool speech you recorded. Or even, a really pretty place you visited during last Thanksgiving.

How We built it

  • Web (Angular, jQuery)
  • Backend (Node)
  • iOS (Objective C, Camera/Photo Upload, Speech Recognition Upload)
  • Facebook Graph API/Javascript SDK/Sharing
  • Parse
  • Instagram API
  • Twitter API
  • Chrome Extension
  • Google Speech Recognition API
  • Google Maps API
  • YouTube API
