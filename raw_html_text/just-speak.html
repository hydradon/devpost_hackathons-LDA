Inspiration

We wanted to work on a new project involving backend and not frontend which we've done at other hackatons. We also wanted to learn something new so we decided to take on the companies challenges to get out of our confort zone.

What it does

It allows you to control a Youtube playlist through voice recognition.

How we built it

We used language speech processing APIs to convert our voice command to text and with we hooked those up to control Youtube videos.

Challenges we ran into

We initially wanted to work on a chrome extension that allow you to control the youtube home page, but we ran into many hurdles when trying to implement the users microphone with the chrome extension. With this behind us we pivoted to just creating a stand alone website with similar functionalities. We also also register a tech domain but for the presentation we need to host it on a secure https domain which we did not get so we are just hosting it on GitHub. Website URL: StayChillJustSpeak.tech

Accomplishments that we're proud of

Creating a functional product.

What we learned

Learning to use Chrome extensions and speech to text APIs.

What's next for Just Speak

To figure out how to directly interface with the webpage through a chrome extension as an example.

You need to use Chrome and allow microphone permission.
