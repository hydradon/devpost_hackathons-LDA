Inspiration

Better way to track medicine usage

What it does

Gives incentive for users to take medication by offering rewards and giving points. Lets users add medication and keep track of what they need to take each week.

How I built it

Frontend technologies, calendar plugin usage, and storing data locally in the user's browser.

Challenges I ran into

Making sure data was consistent in local storage and handling user modifications to the calendar and adding medicine.

Accomplishments that I'm proud of

The app works very well and has a rewards system built in. I started very late because I wasn't planning on participating and only had around 5 hours to work on it as a 1 person team, but in the end, the web app looks very good.

What I learned

How to scale up a full functional web app in a short span of time.

What's next for Medicine Tracker

NodeJS or another backend middleware for DB integration(MongoDB most likely) OAuth user login Rewards tiers Company reward integration
