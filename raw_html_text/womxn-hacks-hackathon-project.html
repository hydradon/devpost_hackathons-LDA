Inspiration

Before we were in college, we had mentors who served a role as our older sisters. They inspired us to make this project and pay it forward.

What it does

Sisterr connects younger, more underprivileged girls with "big sisters" near them who can advise them and help them out.

How we built it

We built it using node.js, website development through html and css, and backing up our data using mongodb.

Challenges we ran into

We had to learn node.js, javascript, and mongodb through this process which was difficult but rewarding in our now expanded skillsets as programmers.

Accomplishments that we're proud of

We are proud of how many new technologies we utilized in our time here.

What we learned

We learned node.js, javascript, mongodb, and further familiarized ourselves with git.

What's next for Womxn-Hacks-Hackathon-Project

We will amp up our web app through further and more advanced technologies and launch it so young girls can have big sisters.
