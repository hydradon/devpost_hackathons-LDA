Inspiration

By the need for generating a secure password and providing easier ways to retrieve the password. I want to help people to have an easy time on password.

What it does

Using the Melody Password Manager generates the password for your service with unique methods.
It helps you retrieve back your password in an easier memorable way.

Need a quick one time password, Melody Password Manager will help you get a safer one-time password.

How I built it

Using Python and NSA Beer-Garden Open Source Tool.

Challenges I ran into

The design the project, the scalability and also the choose on the secure storage for the data generated by the Melody Password Manager.

Accomplishments that I'm proud of

I am excited about demo my idea

What I learned

Learn to use Beer-Garden, how it hosts the program in a standardized interface with effective Command and Control of systems. I also tried the design for the database, and how to have a secure storage.

What's next for Melody Password Manager

It is now trying to connect with the database [Testing on Microsoft Azure] with encryption features[plan to add that features from Microsoft Azure active directory], some analysis on category, the potential for authentication and a timestamp requirement: which means the authenticated request must be within certain time after the request has been made.
