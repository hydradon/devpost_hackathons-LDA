• Overview

Inspiration

We realized the possibilities of artificial intelligence and power that Microsoft Azure provide us which combined can unlock many possibilities. Our team was inspired to train machines to help people who deteriorate their mental health and potential by constantly staying in front of devices using profit-earning apps.

What it does

CognitiveBrain, after having a brief intelligent conversation with user collects enough data about user's sentiments ,emotional state, interests and displays media while taking care of learning requirements of the user and is intended to satisfy user's needs.

How we built it

We used Microsoft Azure REST APIs and used Node.js with Express for implementing a smart bot with a front-end written in JavaScript and EJS and other Libraries like Twitter Bootstrap.

Challenges we ran into

Collecting large dataset in a short period of time to train QnA maker.

Accomplishments that we're proud of

Training the QnA maker with the appropriate dataset for good precision and extensive use of REST API, parsed a large amount of data in short time, Creating frontend for our QnA bot without using Bot Service By Microsoft Azure (as it is not included in student free account)

What we learned

We learned to talk with Microsoft Azure API within a short time by only reading through the extensive and robust documentation.

What's next for CognitiveBrain

CognitiveBarin will be soon available on iOS and Android apps which will further extend the capabilities to live facial expression detection and additional functionality.
