Inspiration

Students do not know what modules to pick in NUS, where such a large variety of options are offered.

What it does

ModWreck is a quiz game to educate NUS students about NUS modules offered across all faculties. By playing this game, students will be able to discover interesting modules of their choice.

How we built it

We primarily used React, Redux, React Router, JavaScript, Material UI, HTML and CSS for designing the single-page web application. We used Heroku for deployment.

Challenges we ran into

Code architecture structure, flow of the game, organisational task management, making the game fun and understanding how to use React and Redux efficiently.

Accomplishments that we are proud of

We were able to complete development of the game and deploy it.

What we learned

We learned to how to use the given technologies effectively in 24 hours.

What's next for ModWreck

Optimise the game for mobile.
