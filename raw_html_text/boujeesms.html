Inspiration

We wanted to build a multipurpose bot with various functionality bringing together multiple tools and APIs one would never expect to be together.

What it does

There is a machine learning regression model that predicts toothpaste cost, a text-based chatbot to brighten your day, an alexa application to provide additional monetary assistance, and a financial application that helps individuals manage their money better with some fun!

How we built it

We each focused on different parts of the project then integrated them together to create our final projects.

Challenges we ran into

Integrating all the Different APIs proved to be very complicated.

Accomplishments that we're proud of

Learning something new in 24 hours with everything functioning!

What we learned

Many of us are first time hackers, so it was a brand new language and brand new experience for many of us.

What's next for BoujeeSMS

Better Machine Learning Classifier for Toothpaste cost and more functionality.
