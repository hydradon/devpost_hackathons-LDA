Inspiration

When I was being interviewed during my last internship, the interviewer mentioned how it's hard to determine the languages an applicant is proficient in or at least, how proficient they are with the ones they brag about. It's always a bother how recruiting side and applicant cannot get easily matched. Codoseek aims to bridge the gap by summarizing the projects with the programming languages present!

What it does

For a given github handle, the app presents the repositories with bytes of codes in respective programming language.

How I built it

Tears and Occasional breakdowns (: Literally coding with the prayers!

Kidding... its actually running Express as a server-server backend and Vue as the frontend.

Challenges I ran into
