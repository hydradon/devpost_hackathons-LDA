• Charities in need for beans recognized by IBM Visual Recognition

Inspiration :

The Tsunami of 2004 in India left my family with scare resources for weeks.

What it does

Allows users to inquire about the resources that charity needs.

How we built it

Using a simple python script we created a terminal chatbot. It makes use of IBM Cloudant for its database and IBM Visual Recognition to identify the object in the image that a user may take of the item they wish to donate.

Challenges we ran into

We had trouble linking the IBM Cloudant and IBM Visual Recognition into the Watson Assistant, so due to the shortage time we resorted to building a terminal based chatbot in python scripts.

Accomplishments that we're proud of

We are proud that we integrated IBM Visual Recognition into our application and that we learned how to work with non-relational databases like IBM Cloudant.

What we learned

We learned how to work with non-relational databases like IBM Cloudant and how to make a custom IBM Watson Assistant.

What's next for BeneBot

We plan to integrate the database and the image recognition service within IBM Watson Assistant, building a web application that displays the same information, display ranking by need, and suggest charities by location and recent disasters.
