Inspiration: It's ashame to see the Earth deteriorate, so this website serves the important function of raising environmental awareness. It allows students to learn about Earth Day and the problems facing our glorious planet today.

What it does: It allows users to learn about Earth Day. Users gain awareness of the issues facing our Earth and are reminded about why we need to protect our Earth.

Accomplishments that I'm proud of: I'm really proud that I built my first website. Considering I've never used HTML before, or, for that matter, CSS, I'm really proud I managed to build it.

What I learned: I learned how HTML works and how CSS can accompany it.
