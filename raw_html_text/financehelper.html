Inspiration

We wanted to help people learn about investment and provide inside to the different metrics used in investing.

What it does

Our mockup describes a portfolio tracking system and individual stock description page running on the BlackRock API with the potential for extension using machine learning. Our software is hosted on Azure's BlobStore storage.

How we built it

JS/HTML/Azure BlobStore

Challenges we ran into

Difficulty in using the BlackRock API, deploying, etc.

Accomplishments that we're proud of

Experimenting with Figma, use of BlackRock API

What we learned

Ease of use for Azure BlobStore

What's next for FinanceHelper

Extension with ML, smarter UI/UX design, etc.
