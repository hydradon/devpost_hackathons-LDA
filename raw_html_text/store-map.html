Inspiration

Have you ever wasted time running around a store with no associate around to help? When was the last time you had a great experience at a retail store?

What it does

A messenger chat bot that gives you instant access to store information and navigation help.

How we built it

We implemented a graphql api that allows stores to define floor plans, section maps, and product information. This is connected to a messenger chat bot that is written in Node.js and leverages the Messenger API. It uses NLP to provide a seamless user experience and Canvas to generate dynamic images.

Challenges we ran into

We wanted to auto select the store based on location, but unfortunately that feature is deprecated on messenger. We reverted to user prompts to selected the store.

Accomplishments that we're proud of

We built a store metadata api and a chat bot in less than 24 hrs!

What we learned

We learned about the messenger platform and canvas.

What's next for Store Map

Extending past stores into other large spaces like hospitals, malls, and conventions. Adding more metadata such as product/event/department details.
