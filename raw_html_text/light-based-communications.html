Inspiration

Light based technology is everywhere. Wifi and mobile data networks are faster than ever but, after decades of being in the limelight they are still not perfect. Alternatives never conventionally used before may be able to bridge this gap in digital communication. Out of curiosity to discover such an alternative we decided upon doing this project.

What it does

Two things - 1. Uses LED lights, a photoresistor and arduinos to transmit data; 2. Using visual light based camera for image processing and parallel data parsing

How I built it

  1. used image processing to detect light using python
  2. Simultaneous arduino board connections and photoresistor developement
  3. Using Sketch to code in arduino C
  4. Experiment to get some specific values and try to fix problems

Challenges I ran into

1.We found that the background noise is very troublesome for light based communications 2.Difficult to discern between minute intensity changes

Accomplishments that I'm proud of

Finished the prototypes. The lights actually work!

What I learned

Became familiar with arduino and dragonboard

What's next for Light Based Communications

  1. Smart car communications
  2. road lamp providing internet
