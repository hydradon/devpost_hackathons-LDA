When we saw the community prompt, we instantly had an idea in mind. Our plan was to build an easily accessible chrome extension and phone app that allows people in the KW area to fully engage in municipal events. The chrome extension app puts together all of the components we built. It features hyperlinks that direct the user to KW event page, the GRT website and much more. To add to this, our team developed a google maps application that highlights barrier-free washrooms near the user's location. We also built a forum that would encourage community participation as well as provide us with feedback. We built a prototype phone application that would show how the chrome extension can be used on mobile devices. We hope to start target penetrate the market with our chrome application to gain awareness and give us the momentum of the that we need to bring the service to mobile. We hope our project reaches people within the KW area and helps allow for more community participation.
