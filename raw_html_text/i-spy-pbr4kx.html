I-Spy

Learning A-Frame WebVR at QHACKS 2017

We wanted to play with the Samsung Gear VR as well as the A-Frame webVR framework to see if developing virtual reality on the web was more accessible than using more high end occulus rift to garner somewhat of the same effect.

The game is simple, just like the classic game of I Spy you are presented with a rhyme of a series of objects you have to locate before you can exit the room. Look by focusing the cursor at an object for more than 3 seconds.

Made entirely of JS and HTML, framework was super fun and easy to use. Would want to build out more robust score system and more levels :)
