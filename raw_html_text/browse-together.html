Inspiration

Browsing by yourself is just plain our boring. Now you can browse and make friends at the same time! What could be better?!

What it does

Browse together uses an extension (for firefox or chrome) to send your geolocation and current open pages to a server. This web server then allows you to see what other users nearby are doing and provides realtime suggestions to engage with them.

How we built it

We made a snazzy cross platform web extension using the webextensions API. We hooked it up to a flask webserver on heroku. The web server generates client ids on demand, and automatically clears inactive sessions, so users can come and go as they please.
