Inspiration

Business in the early stage would often face marketing issues. The inspiration for this project is coming from the current age of influencer marketing strategy.

What it does

Influx is an analytic tool for the business to find the right social media influencer to help them raise the awareness of their business product/services. For a product to sell well it is very important that the advertisement reaches to the right audience. Hence, finding the important traits of the audience is very important before marketing the product to the audience.

Our software runs a thorough analysis of these celebrity's social activities and their follower's activities to find the crucial data. Our system can find overall sentiment perception of the audience towards the celebrity and it also finds the dominant category of interest among the followers. This data is very important to decide whether that particular celebrity is fit to advertise the product or not.

How we built it

The project has three development stages 1- Data Gathering, Data Processing, and Labelling 2- Training Classification Model 3- Developing Web APP Frontend

We are using Reddit and Twitter stream to gather the labeled data. We hosted the python scripts on AWS EC2 instance which tracks the user activities 24/7. We used this data to train our classifier. We run sentiment Analysis and MultiLabel Relevance Analysis on user activities to calculate the credibility of the user. Finally, through the REST API interface, we have built the Web Application which allows the user to visualize our findings.

Challenges We ran into

Our project heavily relies on the data analysis concept to solve this problem. The complexity of the project, the learning curve, and implementing the data visualization keeps us motivated to pursue and complete this project. 1) data visualize the analytic data. 2) configuring the classifier with the right feature sets. 3) setting up the twitter stream.
