Inspiration

Being a parent of a newborn and constantly seeking nurse advice in the middle of the night for every concern about my child was time consuming.

What it does

It a virtual assistant nurse application that diagnose patient symptoms.

How I built it

It was built using nodejs, ibm watson, and twilio.

Challenges I ran into

Knowledge in the domain is very scarce when it comes to available APIS. Normally, this would require a subject matter to come up with a precise solution. Also, there are not much resources who are interested in working on this project.

Accomplishments that I'm proud of

We did it in less than a day despite our limited knowledge and also partly because we wanted to contribute on the Health Care sector.

What I learned.

What we learned from this is that, there are still so many problems that needs to be tackled on this domain. So there are a lot of improvements that needs to be done.

What's next for My Doctor and I

What’s next for My Doctor And I?

  1. Extend the Rule Engine from IBM-Watson to include images and videos. Those resources could be used to provide additional information on a patient’s diagnosis.
  2. Use more resource as a knowledge base, so that we could come up with a wide variety of diagnosis.
  3. A location API that would cover a lot more doctors with a particular specialty.
