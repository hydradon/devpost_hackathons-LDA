Inspiration

Because students don’t know about their investment options and investing seems intimidating and difficult, students often miss out on potentially valuable investments. As well, students don’t know how to budget their money for short and long-term investments.

What it does

The web app allows users to input their personal circumstances (comfort level, amount of money, time period) and provides investment options fitting their choices.

How we built it

We used JavaScript, HTML, and CSS to build the web page.

Challenges we ran into

Because there was not enough time to create a functional back-end for the website, we used JavaScript creatively to imitate a back-end and provide the investment options.

Accomplishments that we're proud of

We're proud that the website functioned despite having little experience with designing websites or web apps prior to the hackathon.

What we learned

How to design and build the front-end of a website using HTML, CSS, and Javascript.

What's next for Intelligent Investment

We want to implement a working back-end for the website by using the framework Flask with Python.
