Inspiration

We wanted to create an application that would facilitate the generation of protective orders while enabling abused women to not tip off their abusive husbands too early in the application process.

What it does

The rails app is a webforms application which allows users to submit the Protective Order form provided by Utah's self serve government website.

How we built it

Ruby on Rails, databases, javascript

Challenges I ran into

Understanding Utah's convoluted rules regarding how to find and submit forms.

Accomplishments that I'm proud of

Not quitting before submitting this web app.

What I learned

Utah's courts are years behind

What's next for Protective Order web application

a better user interface and design.
