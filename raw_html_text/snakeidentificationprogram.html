Inspiration

We are a group based in Texas, and there are a lot of snakes here. We wanted to make a program available to help people identify snakes and if they were venomous.

What it does

Our program takes in user input of snake characteristics and accesses our database, searching for snakes matching the characteristics.

How we built it

We used a Java program to get user input and call the command prompt to run the database file. The database file was created using s(ASP).
