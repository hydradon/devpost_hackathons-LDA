Inspiration

Who doesn't like music?

What it does

The miniature drum kit sends data to a web browser to play sounds and provides it with kick-ass visualization.

How we built it

Using piezoelectric sensors, an Arduino senses these vibrations and sends the serial data to a computer. Using CoolTerm, the data is captured into a text file which is then played back by a web browser. At the same time, the data is visualized to show which pads were hit.

Challenges we ran into

A lot. Especially accessing serial data from the web browser.

Accomplishments that we're proud of

We all contributed to making this project work and look great.

What we learned

We learned that teamwork is invaluable. We learned a new Javascript library and sensor debouncing techniques.
