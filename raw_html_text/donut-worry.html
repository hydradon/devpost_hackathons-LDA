Inspiration

Wanted to know all the options I had with all the ingredients available so that I could optimize my grocery spending.

What it does

It's supposed to return a list of recipes that can be made with user's input based on ingredients.

How I built it

Backend: nodejs to parse through raw json data; Frontend: HTML5 and CSS3 ; Donut Logo and background: Adobe Illustrator

Challenges I ran into

Having trouble waking up on Sunday lead to less work time on the project. In addition to that, implementing node.js and pushing through everything in 24 hours were very challenging. We have yet to add all the features we want.

Accomplishments that I'm proud of

At the very least, we made a working website with basic functionality of our goal.

What I learned

We learned that there's a whole lot of unaccounted bugs that you might run into when building a project.

What's next for Donut Worry

  • Our backend remains buggy so we must fix and improve our search function
  • Implement a "suggested" ingredients feature
  • Implement better responsive design
  • A save/favorites button to hold your recipe while searching for more recipes
  • Deleting search tags
  • Automatic synchronization of user input with backend
