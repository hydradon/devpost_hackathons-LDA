Thearg

This is the repository for Thearg, a Hello World Hackathon project. This project is an augmented reality text experience, taking place in the WALC building at Purdue University. One access the website at thearg.neocities.org, and visits the indicated locations, searching them for clues to solve ciphers which eventually reveal an email, where they can contact the authors of the project. JS by Gavin Williams, HTML by Ali Alnasser, and CSS by Daniel Raftery.

SOLUTIONS
