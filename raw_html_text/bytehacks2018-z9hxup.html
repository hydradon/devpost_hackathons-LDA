What it does

This site is a tool that offers those with limited medical knowledge the proper solutions to provide effective medical care.

What did you use to make it?

We used HTML and CSS along with Bootstrap to build our web design. We used Javascript to implement Google Map API, local storage, and forms.

What challenges we ran into

We struggled to make Bootstrap work with our format of the page (embedded Google Map) as well as storing local data.

How do you want to expand

After the hackathon, we would like to make it into a mobile app that could be used by more people.
