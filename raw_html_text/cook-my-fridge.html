Inspiration

People can easily find recipes based on the food that they currently have.

What it does

This app allows you to take a picture of any food, such as an apple, and numerous recipes that use an apple.

How we built it

We built this mainly using React-native on the front since it is a mobile app and we used mongo db and express on the server side.

Challenges we ran into

It was our first time using React Native so we had to spend a lot of time debugging errors in our code.

Accomplishments that we're proud of

Our app can connect to the Watson API and retrieve the response object for an image successfully.

What we learned

We learned how to use React Native and how to use the IBM Watson API.

What's next for Cook My Fridge

We want to use machine learning to have more relevant recipes show up first.
