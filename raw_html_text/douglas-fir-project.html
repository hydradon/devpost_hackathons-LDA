Inspiration

Both members of the team really enjoy video games, especially those with unique mechanics or stories. We wanted to try our hands at developing our own.

What it does

The Douglas Fir project allows a first person character to wander through a 3D landscape with some minimal narrative overlay

How we built it

We built the project exclusively through Unity 3D using javascript and c#

Challenges we ran into

Knowing nothing about game development Not knowing c# or javascript Learning how to manipulate physical aspects of a computer generated world (especially water/particle effects) Compiling different parts of the game

Accomplishments that we're proud of

Animating scenes, building a world that is more than a flat box, making a functioning menu screen, and getting a better understanding of particle effects in the 3D space

What we learned

Game design is really hard and animators deserve a lot more respect than they get.

What's next for Douglas Fir Project

Cleaning up textures, developing objects that the character can interact with, and making running water.
