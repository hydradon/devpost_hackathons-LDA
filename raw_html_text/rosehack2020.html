Inspiration

We both love video games and thought we should finally make one ourselves!

What it does

Puzzle Adventure Game is a game where you find yourself on a world with a key(s) scattered around that you must find to unlock the portal to get to the next world.

How I built it

We built this game by using assets from the Unity Asset Store, creating our own assets, and writing scripts in C# to get everything to work together cohesively.

Challenges I ran into

We ran into difficulties trying to get the map to layer correctly to give a correct representation of our isometric map. Only one of us knew C# so it took some time to help the other learn the language for all of the scripts we needed.

Accomplishments that I'm proud of

We are proud that we never gave up after encountering problem after problem. Making a game is not something we are familiar with so we are happy that we went out of our comfort zone and tried something new.

What I learned

From the project we now have a better understand of Unity and C# and how they both work together!

What's next for rosehack2020

We want to add more levels and detail to the levels to make it more aesthetically pleasing. We would also add more interesting and complex ways to obtain the keys and open the portals.
