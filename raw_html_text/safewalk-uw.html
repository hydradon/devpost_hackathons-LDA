Inspiration

We are all students that live on different parts of campus and noticed that walking alone in the Seattle area can be scary and even unsafe.

What it does

This site shows users what areas historically have the most crime so that they can avoid that when walking home alone.

How I built it

We used Seattle Police Department 911 Incident Response data from Seattle.gov and Google's Maps API to create this site.

Challenges I ran into

We struggled with filtering the data by time.

Accomplishments that I'm proud of

We are most excited about the concept of the site.

What I learned

The Google Maps API is fairly simple to use and learn.

What's next for SafeWalk UW

Add functionality and improve the UI of the site.
