Inspiration

People with allergies.

What it does

Checks for user inputted allergies against scanned food items.

How we built it

Android Studio, Visual Studio, and Azure.

Challenges we ran into

USDA database we pulled our data from was only about 230,000 which was not as comprehensive as we would have hoped for and limited the amount of items our bar code scanning would find.

Accomplishments that we're proud of

Everything we were able to finish since this was all of our first time making an Android app and using Kotlin.
