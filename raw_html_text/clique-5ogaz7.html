Inspiration

Trying to prevent cyber bulling in general

What it does

Blocks Tweets which are deemed as being offessive to a user

How we built it

We have a server running a Python script that detects faul tweets and either blocks it or allows it to be seen/visible to the user

Challenges we ran into

Everything about the project was challenging since we wrote our own LAD from scratch

Accomplishments that we're proud of

The fact that we managed to accomplish what we set out to do is enough reason for us to be proud of it

What we learned

NEVER try writing our own LAD with less than 12 hours left

What's next for Clique

Extend it just beyond Twitter. The plan is to have it working in other social media platforms
