Inspiration

Two of our members attend Philadelphia High School for Girls. In their experience, textbooks are often hard to come by and pricey. Classrooms do not have enough copies to go around. When students try to find the books online, they either cannot, or they are too expensive. Also, some students have books from previous semesters younger students could use.

How I built it

  1. Used Angular JS for the services and Bootstrap CSS for the styling part and hosted it on AWS S3 by uploading all the files and enabling the static website hosting.

Challenges I ran into

  1. Rendering different views in Angular application using ng-route and ng-view.
  2. Displaying the data using angular and database usage.
  3. Tried integrating the gmail API.
  4. CORS issue which we resolved by setting up a local python web server.

Accomplishments that I'm proud of

  1. Hosted the website on AWS S3.
  2. Website was responsive
  3. Great design

What I learned

  1. Learned different Angular concepts and the benefit and challenges of a single-page application.
  2. Using bootstrap for angular application.
  3. How to solve the challenge of a Cross-Origin Resource Sharing issue (and that such an error existed)

What's next for LendMeBook

  1. Integrating gmail API for sending out notifications.
  2. Adding a Facebook Login.
  3. Setting up a database.
  4. Hosting it on AWS EC2.
