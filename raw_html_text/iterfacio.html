Inspiration

Inspired by an initial Cross platform application (enabling tourists to create, personalize and/or share their own city tours based on the public transportation), we needed an universal wireless storage device which would be able to work on offline mode (White Space and Roaming problems).

What it does

It will be a full micro server on which you can choose the application you need (Secured files sharing, Media server, ...) but it also can be a secured gateway for IoT connections.

Accomplishments that we're proud of

In 2 days, we built a mesh network with 4 Raspberry Pi 3 in order to simulate our use-case: a Mesh Network to communicate with others when we are in a White Space.

What we learned

We learned the 5G principles and the Huawei's and Salesforce's APIs and how to use them.

What's next for ITERFACIO

We are developping a micro server which will work in any situation with securities based on encryption and blockchain.
