Inspiration

Being frustrated by messing up in AP Chem and other Mathematics involved classes.

What it does

Uses Wolfram Alpha to evaluate mathmatical equations, with a text box on the side to write down notes.

How I built it

Using Visual Studio Code to edit the code, with node-wolfram, javascript, css, node:js, jquery, html and npm.

Challenges I ran into

Overcoming the Mac/Windows barrier, as well as getting the various programs to cooperate. Our faith was challenged during this project.

Accomplishments that I'm proud of

Using API to connect multiple files together and separating strings using spaces.

What I learned

Coding is harder than we give it credit for; being prepared in advance and keeping on schedule is important.

What's next for Shock Checker

Making it more appealing to the eye.
