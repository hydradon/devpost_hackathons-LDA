Inspiration

To create the most annoying hack by generating random noises that will piss people off.

What it does

Try it out! Right click blocks to change direction. Left click blocks to change pitch. Right click sphere to make it able to move but make sure there is an arrow below it. Press space to create sphere. WASD to move background and mouse scroll to zoom in/out.
