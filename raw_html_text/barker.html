Inspiration

My dearly beloved dogs, Coby and Sasha may they rest in peace.

What it does

Allows Users to find dogs to adopt with efficiency and provide online interaction to better match owners with their new potential pets.

How I built it

React, Azure (CI/CD, repos, DevOps, appservices ) and Node.js.

Challenges I ran into

Ran into issues implementing azures database.

Accomplishments that I'm proud of

The User interface is engaging, relatable, and efficient at matching owners to new possible pets.

What I learned

How to use React and Azure tools.

What's next for Barker

To expand the application's matching algorithms efficiency by using other mediums besides location, as well as allowing for other categories of pets to be adopted.
