Inspiration

Have you felt an urge to stay fit and healthy? Have you ever been scared by the crazy cost to hire a personal trainer? Now you have cheaper solution! Using artificial intelligence, EvEN brings your personal coach into your cellphone.

What it does

EvEN used OpenPose to extract key information about your exercise, correct your mistake, provide extra motivation and count toward your success.

How we built it

We use OpenPose (and Posenet in earlier implementation) for pose detection. The front-end application is built with react native with expo.io. We use sound API from houndify to create interaction between user and bots, and Firebase for app communication.

Challenges we ran into

A lot of react native issues. Comparing user pose to correct pose on realtime is extremely difficult.
