Inspiration

We were inspired by open world adventure games and real world scavenger hunts. We decided to combine the two and create this game!

What it does

It is a game where the player wanders through the forest searching for various times using riddles as clues.

How we built it

We built the main game using Unity. The models were created in SketchUp and the music was composed in NoteFlight.

Challenges we ran into

We ran into challenges using the colliders and while designing the compass tool.

Accomplishments that we're proud of

We are most proud of the compass and the music.

What we learned

We learned a lot more about GUI's in Unity and just overall ease of use in the engine.

What's next for Adventure Hunt

We plan on adding different scenes with different scavengers hunts to explore.
