Configure of Speech

What it does

It helps train you in your public speaking skills and offers valuable feedback to build upon in later occurrences.

How we built it

Used React for the front end and used Node/Express hosted on AWS Elastic Beanstalk Instance with Mlab (Mongodb). Generated voice to text chunks using Web Speech API. Data is then processed by Node server to generate analytics and suggestions on your speech. Used Microsoft Azure cognitive service to generate sentimental analysis on speech.
