Inspiration

Built by a Power Rangers Fan, the current season of Power Rangers Inspired me to create this immersive experience.

What it does

You are brought into the base where power rangers are trained. Your in this world and now can watch video. View Images and search for characters

How I built it

I built it by using custom photo realistic images and created photo cards in photoshop and illiustrator. I used references from the show to create the world. I used React 360 as my framework.

Challenges I ran into

Libraries that were a part of React 360 were missing so a group of devs created a group to help one another out and share experiences

Accomplishments that I'm proud of

Pushing through all obstacles faces with developing with React VR

What I learned

I learned how to add video, use audio and build a search without having a text input library

What's next for Grid Battleforce VR

I would like to add more scenes and characters from the show. Would love to also have some animated 3d Models
