Inspiration

It was the challenge at LauzHack this year that we both loved and thought it’d be fun to try out.

What it does

On the server, it downloads, extracts and processes hospital data and outputs an statistical exerpt that is very light and easy to send to clients.

How we built it

We always loved java, so we’ve chosen to use it for everything : a web server is making sure clients can get hospital room infos, another thread is used to get new datas from the server automatically.

Challenges we ran into

The datasets were huge. Even vim had trouble opening them.

Accomplishments that we’re proud of

Processing those datasets was not the easiest of tasks. After 10 hours of failure we were so proud to see our files being decoded !

What we learned

Biomedical is hard. Very hard.

No idea. Perhaps even more scallable systems ?
