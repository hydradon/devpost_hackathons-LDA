Inspiration

Helping NYCer get fresh local food by improving the Track and Trace system

What it does

We leverage the blockchain protocol to make existing systems by farmers, distributors, and markets better by improving 1)proof of location, 2)ease of input, and 3) accountability

How we built it

We built it using an erc721 token with our own block explorer using Sonm's infrastructure for our server

Challenges we ran into

Our challenges were helped by our sponsors. Thank you to all!

Accomplishments that we're proud of

Building a working ERC721 token that gets streamlined on our own block explorer, and using sonm's infrastructure for our server

What we learned

Understand the process by which the food supply chain works and how to use blockchain technology to improve proof of location, ease of input, and accountability

What's next for FoodChainz

Get Farmers, Distributors, and Markets to start using the technology
