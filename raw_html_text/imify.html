Inspiration

We didn't know what a lot of the technologies around us were doing, so we decided to make something that did.

What it does

Uses Google Cloud Vision Algorithm to search and recognize any image given, and allows users to buy the item on Amazon, view it on Wikipedia, or find it on Instagram.

How we built it

First implemented Cloud Vision algorithm, then started with Amazon, Wikipedia, Instagram, and Facebook functionalities.

Challenges we ran into

Trying to make the Cloud Vision algorithm more efficient and then messing everything up.

Accomplishments that we're proud of

The app works!!!

What we learned

We know a lot more about image recognition software and how images are split into nodes for scanning.

What's next for Imify

Make Cloud Vision algorithm more accurate by using hundreds of point nodes to identify the object better, and implementing feeling detection based off of certain face nodes.
