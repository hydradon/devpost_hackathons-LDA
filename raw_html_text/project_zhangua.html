Inspiration

My partner Yilun Hua and I both interested in Chinese I Ching. I Ching is a famous literature to analyze people's fortune by their birthdays, born environment, and a lot of exterior factors. Nowadays, people live in a high speed environment and they experience a lot of pressure from work and family. My partner and I aim to design a program that can tell people their levels of luck to finish a task.

What it does

Tell user his/her luck by the wish he/she entered

How we built it

We build it in Java and collaborate via GitHub

Challenges we ran into

GitHub setup! We experienced situation like "one people created & pushed a java project, another people pull the project but not shown as java project", "cannot push due to url configuration failure", and etc. But our issue are solved under help with two mentors from Blue Origin, David and Tim ;)

Accomplishments that we're proud of

Our GUI application finally can be run!

What we learned

The importance of team communication. We all have good idea and different coding style. It is importance for us to learn code of coordination in this event

What's next for Project Suan Qian

We plan to build it into an Android App

How to use our prokect

Simply run our Zhan_Gua_Simple_GUI.java. Enter your wish in the top red text field and click "Start" button. To test another wish, click "Reset" button. To exit, click "Exit" button. It has straightforward operation guide and very convenient to use.
