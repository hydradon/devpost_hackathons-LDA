Inspiration

Never forget a KI labs event again. Everything available at one place.

What it does

It is a clean an easy tool for listing and adding/editing events. Included is also a reminder system which sends an email 3 days before the upcoming events. Also part of this tool is a illustration library where matching pictures are selected for each event. --> Find relational images based on the name of the event

How I built it

angularJS, html5, sass, webpack, es6, django, postgresql, rabbitmq, celery, python, docker

Challenges I ran into

  • Timing issues

Accomplishments that I'm proud of

What I learned

Planning is key. If you think of a project which seems easy at the beginning, there are always things coming up which you did not think about. We did not have a lot of time to think really detailed about every feature and what could happen during the developing because there was just time for 1 1/2 days. --> So if there are any customers project in the future. Planning and researching is key :)
