Inspiration

Many a time, I have walked through a wegmans, wandering aimlessly, nowhere to go, unsure what to do. Now with speedy snacks you can be in and out as fast as possible.

What it does

Provides you with an optimal path through wegmans to find all of your groceries

How we built it

We built it using the location feature of wegmans combined with heuristics based on wegmans layouts.

Challenges we ran into

Setting up an account system proved harder than expected. Also obtaining data from the wegmans API was a little strange because I had to try 3 different methods of sending requests because they kept returning junk information in the request body

Accomplishments that we're proud of

The real time rendering of the wegmans map with your food locations on it.

What we learned

Relearned much about programming in javascript as well as many mistakes to avoid in the future

What's next for Speedy Snacks

Eventually we think that by applying OCR technology to the layouts of wegmans stores online we could get an even more accurate layout for even faster shopping
