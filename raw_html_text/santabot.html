Inspiration

Norad's Santa Tracker started as a result of a misprint in a Sears ad for a Santa hotline. I was thinking about how kids used to call Santa from local newspaper hotlines, and realized they're not as popular as they used to be. Kids text more than they call nowadays, so I felt like the only solution was a chatbot.

What it does

Reads user input, regardless of specifics, and gives answers based on what the user asked for. This part was done with Wit.AI. Twilio handled the messaging end

How we built it

The one true dev language, node.js.

Challenges we ran into

Starting, realizing the Understand API by Twilio didn't exist yet.

Accomplishments that we're proud of

Finishing, being able to filter out swears.
