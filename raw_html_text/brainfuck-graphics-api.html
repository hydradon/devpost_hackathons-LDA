Inspiration

Brainfuck needs an upgrade if it is to continue being used in production code. and we really love Brainfuck.

In all seriousness, we're a group of individuals that have very different interests. We were able to divvy up this project in such a way where we could all learn about an area we weren't familiar with.

What it does

Basically it adds the ability to draw primitive shapes to the screen in brainfuck.

Accomplishments that we're proud of

Over the course of 24 hours our team was able to build a working brainfuck interpreter, design an API for this implementation of brainfuck, and write brainfuck code utilizing that API.
