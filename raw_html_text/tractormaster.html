Inspiration

Tractor Supply Company's data seemed like it had some fun patterns, so we thought we'd take a crack at some ML algorithms and data analysis

What it does

Creates a forecast of sales, projected onto a Google Maps heatmap based on various categories of goods and a date range

How we built it

Lots of web scraping in Python, NLP with spacy.py, stochastic gradient descent with Tensorflow, backend in Node.JS and frontend web development

Challenges we ran into

The data was limiting at times and didn't get us enough room to create something entirely comprehensive.

Accomplishments that we're proud of

Using ML algorithms Designing a full-fledged web app as just two people Coming up with creative ways to think about the data we were presented with
