• Discover Santa Barbara

Inspiration

Finding local entertainment can be an important step in familiarizing one's self to a new environment and feeling at home. This is especially the case in a smaller, suburban city like Santa Barbara. Our app, Discover Santa Barbara, is essentially a compact, mobile introductory guide to some key highlights, that a new resident in search of activities to fill their weekends would not be immediately aware of.

What it does

It functions like a mini booklet, allowing the user to flip freely between different categories of activities.

How we built it

We used Xcode, applying Swift.

Challenges we ran into

As beginner coders, we were unfamiliar with the differing commands associated with different versions of Swift. We also struggled with setting constraints on multiple elements on our ViewController.

Accomplishments that we're proud of

The whole process, and the fact that we have a final working product! We've never coded before. EVER. And are very happy to have the opportunity to begin our journey into coding at this hackathon.

What we learned

We learned the basics of Python, Swift, git and github, how Xcode works, and what data science is.

What's next for Discover Santa Barbara

Our original idea was considerably more ambitious than our current skills could successfully achieve. We plan to improve our coding knowledge and skills so we can incorporate the following features into Discover Santa Barbara: more pages/categories of activities, a customizable activity schedule, the snapchat map to let users see public stories for the activities listed, and Smartcar API so users can see if their friends are nearby via their vehicle locations, providing an opportunity to facilitate a possible rideshare and group activity.
