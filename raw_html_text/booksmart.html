Inspiration

Our parents have trouble saving and using bookmarks in commonly used web browsers, and we wanted to tackle this issue using a better system for bookmarking web pages.

What it does

This website bookmarks web pages, and visually represents the bookmarks as a pre-loaded web page in a tile on the home screen of the website.

How we built it

We used Cloud 9, IAM, and EC2 inside of AWS in order to develop this website.

Challenges we ran into

We ran into some front end challenges when connecting from a javascript file to php script to mysql database. We researched and resolved this issue, however it was not very evident how to do this.

Accomplishments that we're proud of

The website is completed up to the point of our original idea, and is fully operational.

What we learned

We learned that front end styling with websites can cause a lot of issues with back end logic for connecting to a database, and we learned how best to tackle these kind of issues by using resources like google, Youtube, etc.

What's next for BOOKSMART

We really enjoyed developing the website, and we would like to continue with this project and add more features as new ideas arise with how to improve this website.
