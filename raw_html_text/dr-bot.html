Inspiration

More than a billion people use messenger on a daily basis and we thought it would be cool to make something that would enable the users to check some things related to health right there without leaving the app.

What it does

Right now messenger bot can search the web for basic symptoms to find cures. Also, it has can search for hospitals near the patient's location.

How we built it

We used the messenger platform and javascript. We also used IBM Watson's conversation API

Challenges we ran into

Majority of the team had not worked with messenger bots before.

Accomplishments that we're proud of

Actually getting something to work. For 3 of us, this was our first hackathon

What we learned

How to make messenger bots and how to use the IBM Watson API

What's next for Dr.Bot

We plan to add more features. This is still a work in progress.
