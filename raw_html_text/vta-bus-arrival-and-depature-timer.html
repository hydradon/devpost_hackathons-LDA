VTA BUS Arrival & Departure Live Timer Powered By SMS

VTA Arrival & Departure Timer is an immediate or live real time bus notification system that uses SMS to notify VTA Passengers on the next arrival and departure time for VTA Buses.
Are you tired of waiting at the bus stop not knowing when the VTA bus is going to arrive? Or even worse, not knowing if the VTA bus is going to arrive at all? VTA Arrival & Departure Timer is here to help by letting you know when the next arrival times are for your stop/station.

The App makes a Live Call via the VTA API to Provide Bus Real Arrival and Departure Time for VTA Buses. API Call URL: http://api.511.org/transit//StopMonitoring?api_key={my-token}&agency=vta&stopCode={my-stop-code}

The use of this app is very simple, on your phone,desktop, laptop just call up the application and then make one time registration with your Mobile Number. Login into the app, next dial or text the five - six digit bus stop number. The bus stop number is viisbly displayed on the bus stop sign.

The immediate arrival times and departure times for VTA Buses will be both read to you via the application and also sent via SMS. The Displayed Information also have a Confirmation Link that will take you to VTA Live Website to verify the Bus Arrival and Departure time details based on that very Bustop Id.
No need to have a fancy smart phone as the app is compatible with any phone,Laptop,Desktop etc. that can just browse. The Application was Powered to increase VTA transit ridership or retention.

Notifications via the apps or SMS is available only in English.
VTA Arrival & Depature Timer supports Most Bustop/Station for only VTA transit agency.

Platform of Availability:

Its a Mobile Web App and thus allow accessibility on all Platform including Android, IOS, Windows, BB, Desktop, Laptops etc. Any Mobile Phone that can browse can use it.

About Bus Arrival & Depature Time Informations

The app provides you with VTA first Class Information on the Next Arrival and Depature Time for VTA Buses at Your Stop. This is because as you search for information, the app makes a direct call to VTA Live Server via the API. The Displayed Information also have a Confirmation Link that will take you to VTA Live Website to verify the Bus Arrival and Departure time details based on that very Bustop Id.

API Reference

Live API Link: http://511.org/vta/
API Call URL: http://api.511.org/transit//StopMonitoring?api_key={my-token}&agency=vta&stopCode={my-stop-code}
