Inspiration

First meeting a day prior on GH - lazy to write minutes - wish that someone could write it for us. - Decided to build - AI, summarize minute minutes for us - voice assistant - Google Home Mini.

What it does

The Google Home Mini far surpasses the role of a secretary in a meeting. Listens to the conversations in a meeting. AI summarizes the conversations, output it in a web - printable, etc.

How I built it

Voiceflow integrated with Google Home Mini. Url endpoint API as a database to store speech data. Python script AI summarizes UI, react

Challenges I ran into

Playing with an API which is fairly new in the industry. Not much knowledge and tutorial online.

What's next for Mynute

Test it out for student clubs. Receive feedback from them and continue to iterate.
