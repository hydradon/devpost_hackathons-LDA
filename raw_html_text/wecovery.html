Inspiration

We noticed that communities tend to get closer after disasters. Sometimes we want to help or ask for help but we just don't know how to.

What it does

It's an app where people can post what kind of help they can offer to their community on an easy-to-use map. They can offer help like sharing water, food, or just any service .

How we built it

We used AndroidStudio to build our app using MVC design pattern . We also used the BoxMap api .

Challenges we ran into

It was our first time using the BoxMap api so we ran into a lot of problems.

Accomplishments that we're proud of

We're proud of how fast we've completed the project. We had to do tons of research and most of our project is built around an api new to us.

What we learned

How to use BoxMap api, use markers. We're deepened our knowledge in Java and learned how to plan and think faster.

What's next for Wecovery

Take it to a larger scale and adding functionalities, like a messaging system and more detailed posts.
