Inspiration

Our experience with immigrants and tourists facing the language barrier with no real tools to overcome this problem.

What it does

Echo uses a speech recognition API to suggest correct replacement phrases for grammatically incorrect phrases.

How we built it

We built it using an API to listen to and interpret the text, then using a wide database to suggest correct expression.

Challenges we ran into

The APIs were hard to access, with multiple technical errors occurring.

Accomplishments that we're proud of

The idea's originality and basis would be extremely helpful in today's world.

What we learned

We gained valuable experience with using and applying APIs and website extensions, and designing a clean UI.

What's next for Echo

We could extend our database to include trends in grammatical errors from other ethnicities so Echo could cater to a wider range of people. We could also include a grammatical interpretation API for more accuracy in suggestions.
