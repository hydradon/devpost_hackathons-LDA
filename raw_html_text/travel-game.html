Inspiration

We were inspired to create a game that was a top-down transport simulation, inspired by Mini Metro and tycoon games. We want to create a real-time strategy which would be easy to understand, but hard to master.

What it does

You guide people to the airport by telling them when and where to use public transportation. There is also a time limit, which adds an additional challenge as you try to manage multiple people.

The game is built in Unity and coded in C#.
