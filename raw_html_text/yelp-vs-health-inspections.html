Ever wonder if restaurant reviews are related to health inspections results? Well, thanks to open data you can find out.

This app shows all of Surrey's restaurants on a map. You can search and locate a restaurant by typing in a few letters of the restaurants name. Or, use your smartphones GPS to locate you. Clicking on a restaurant will display the Fraser Health inspection results and yelp rating. Clicking on the inspection results displays the detailed restaurant inspection details. Clicking on the yelp rating launches the yelp app, or website to check out the most comprehensive set of user inspired restaurant reviews and restaurant details.

This app is built with PHP and JavaScript. API's used in the application include:

yelp API

ESRI JavaScript 4.0 API beta3

CKAN v3 API

ArcGIS Online
