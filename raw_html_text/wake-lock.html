Inspiration

Basically an attempt to improve on pre-existing smart IoT devices that don't really use OAuth or Two factor authentication.

What it does

Responsible for detecting the presence of a human, using a servo and raspberry pi to automate the door lock.

How we built it

Built using a servo motor, raspberry pi, a picamera and finally a lot of broken door locks

Challenges we ran into

The picamera, apparently borders on criminal when it comes to working in environments with moderate to low exposure

Accomplishments that we're proud of

Building this in about 10 hours, since we started near about Midnight

What we learned

That lighting is not an absolute godsend.

What's next for Wake-Lock

As stated before, using Two Factor Authentication and facial recognition to take the platform further.
