Inspiration

As freshman, we understand the struggle of navigating the bus system and getting to classes so we wanted to make that process easier.

What it does

Takes input from starting location to end location with optional input of a given time and returns the bus and nearest bus stop.

How we built it

We used google maps apis and twilios apis.

Challenges we ran into

Implementing the api and using python.

Accomplishments that we're proud of

Implementing the api.

What we learned

How to use apis and python language.

What's next for RU-BuSMS

Making it publicly available.
