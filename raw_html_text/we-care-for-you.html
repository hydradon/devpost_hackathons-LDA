Inspiration

The Face API of Microsoft and several instances of missing to carry medical reports.

What it does

This app is one stop for the user's health place. It includes taking care of insurance, Getting a doctor anytime and any place with contact details, keeping track of medical reports and being able to send his/her reports to the Doctor.

On the Doctor side, it's a good place to gain fame and reviews, study the patient's health condition. The doctor is able to get the patient's data even through by photo.

How we built it

We used Android studio and the Microsoft API for creating this project. Purely written in java with the highest efficiency.

Challenges we ran into

Keeping the Patient's docs remote and safe. To be able to do a nearby Bluetooth transfer and/or email transfer of docs.

Accomplishments that we're proud of

Creating the whole app well in time. In learning a lot about android studio which makes us proud that we used this opportunity to its fullest.

What we learned

Working with Android Studio and handling API's.

What's next for We Care For You

Improving the code to make it more efficient and robust and also to improve the file transfer process.
