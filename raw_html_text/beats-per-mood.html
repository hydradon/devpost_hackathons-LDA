Inspiration

Our inspiration came from our experience with the Spotify application. We wanted a feature on Spotify that allowed us to easily create playlists, based on mood from songs we already know. If you have an activity like walking or running we want to make a playlist (with songs we already know) that have songs with an appropriate tempo.

What it does

It gets information on the users' songs, filters the songs by tempo, and creates an appropriate tempo.

How I built it

We mainly worked in javascript and used commands with Spotify's API.

Challenges I ran into

We ran into little or no issues with user authentication. We ran into a lot of issues with obtaining the necessary tracks from the user.

Accomplishments that I'm proud of

Obtaining the necessary tracks from the user.

What I learned

We learned a lot

What's next for Beats Per Mood

We would like to be able to include more artists on the playlist, and we would probably need to include more discographies. Long term we would like to blend playlists together for multiple moods.
