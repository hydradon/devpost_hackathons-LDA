Our project aims to solve the solar split incentive by crediting property owners with a check for solar production and charging tenants for energy consumed.

We built a proof-of-concept web application complete with an explanatory home page, a quote generation example, and a property management portal.

With more time we would have integrated a backend database (which we built and had functional, but lacked the time to integrate).

Thanks to Powerhouse for this great opportunity!
