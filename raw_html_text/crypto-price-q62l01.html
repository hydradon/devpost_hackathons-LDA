Inspiration : Opening app and laptop to check the price of crypto currency is pain and time consuming.

What it does : Tells you latest crypto currency prices

How I built it : Its a skill build using aws lambda and crawling through various backend sources for latest prices.

Challenges I ran into : Sources consistency and cloud hosting.

Accomplishments that I'm proud of : This is pretty useful and i myself use it many times during the day.

What I learned : This are moving on voice command era and no one gonna type it will all be done via robots

What's next for Crypto Price : Personalizing with user selected crypto currencies so it can calculate the current status based on profit/loss and share the summary.
