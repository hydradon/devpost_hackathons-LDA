Inspiration Develop mobile app that allows quick lookup and sharing of physical constant data and conversion values.

What it does Allows quick lookup via mobile app to physical constant data and conversions.

How I built it A restful backend service was built with the microsoft azure platform leveraging webApi 2, entity framework, and sql server. Mobile app was built using phoneGap and Angular.js

Challenges I ran into : Backend rest service screen scrapped data from NIST web site.. process was too slow and prone to errors

Accomplishments that I'm proud of. Created a local data storage to hold static physical constant data for quick response.

What I learned. Move static data to client.

What's next for Physical Constants Mobile Reference App. Add additional NIST data points and UI enhancements.
