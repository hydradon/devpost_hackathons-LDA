Inspiration

In North America 300,000 plus birds die annually to windmills, becoming a threat to energy and wildlife.

What it does

takes pictures, uses computer vision to analyze picture to see if it is bird or not, sound buzzer for time interval if bird predicted

How we built it

Python, arduino code, java

Challenges we ran into

Learning computer vision tensorflow, keras. Keras installation, Unsupported RaspberryPi , using 32 bit jdk, finding female female connectors

Accomplishments that we're proud of

The finished product.

What we learned

We went into this completely blind, we learned computer vision, arduino and python compatability, java development kits, arduino code for taking pictures

What's next for WBR

Optimizations, mobility, camera improvements, live video feed, image data base for further training models, higher accuracy in prediction
