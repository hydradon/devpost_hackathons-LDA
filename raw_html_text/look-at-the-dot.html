Inspiration

I wanted to build a website that helps people improve their focus and practice mindfulness.

What it does

It displays a big dot in the center of the screen. The dot gradually changes color over time. The goal of the game is to stare at the dot as long as possible.

How I built it

HTML, CSS

Challenges I ran into

Adding our domain name to the website, hosting the website on GitHub Pages (still now we have not figured out how to host our website) and trying to figure out the syntax of HTML.

Accomplishments that I'm proud of

I created my first HTML and CSS file.

What I learned

I learned the basics of HTML and CSS.

What's next for Look At The Dot

We might make the dot move across the screen and use Microsoft's Azure API to make sure the user is looking at the dot.
