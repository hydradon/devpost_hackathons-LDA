For this hackathon we wanted to create an application that showed the interconnection between STEM and arts, which are usually considered opposites.

This application takes in your location, and analyses forecast data from the Open Weather API to provide the parameters of the song (tempo, key...). It then hashes the data and using our algorithm synthesizes a song.
