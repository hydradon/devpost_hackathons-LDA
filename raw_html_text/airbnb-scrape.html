Inspiration

I'm interning in New York this summer, and want to find a cheap and convenient AirBnB

What it does

Scrubs AirBnB listings given a set of parameters and dumps scrubbed data to a csv

How I built it

Used the selenium browser emulator to go to airbnb and view data

Challenges I ran into

Class and Id names on airbnb are not named conveniently

Accomplishments that I'm proud of

Successfully pulling data from AirBnB and dumping it into a csv file

What I learned

I learned a lot about how websites are structured

What's next for airbnb-scrape

Including more options for parameters to provide the airbnb query
