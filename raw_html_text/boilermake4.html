rminder -So you don't miss those precious moments- rminder is a multi-platform collaborative reminder software. You can use this app to set voice reminders for your yourself with a message and time to be displayed. The message will be read out to you using text-speech software You can also add friends and send them rminders rminder user interface is optimized to be as user friendly as possible to make sure users of all demographics can easily navigate through the app rminder is supported on browser, android and iphone
