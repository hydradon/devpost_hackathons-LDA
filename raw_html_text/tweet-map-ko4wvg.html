Inspiration

Various social media accounts that post interesting infographics and maps.

What it does

The intention for the app is to provide a heat map providing popularity of various Twitter hashtags. For example, if a user types Wizards a map of US would reveal the most heat around DC bc that is where most of their fans are.

How I built it

With android studio, java, and streaming API.

Challenges I ran into

OAUTH IS PAIN. I wanted to do OAuth bymyself with using other libariries

Accomplishments that I'm proud of

Everything. This is my first time using Rest API as well as Android. Pretty much everything in my app is brand new.

What I learned

How to use Rest-API, OAuth, and how to create Android apps.

What's next for Tweet.map

Polishing and publishing on Google Play Store.
