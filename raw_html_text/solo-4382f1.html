Inspiration

We wanted to motivate people to go out to each other and fight for glory!

What it does

You create your clan or join an existing one. After that you are ready to fight for area captured by clans. You should try to be as fast as possible in securing as large area as you can manage. Each clan then has its area coverage statistics.

How we built it

We built backend using python and flask. Frontend was built using react, javascript, html and css as a PWA.
