What it does

CalCam allows the user to quickly and easily find nutritional info and track their intake daily.

How we built it

We built the main app using Android Studio and the Google Vision API, the backend was built in node.js with a postgresql database.

Challenges we ran into

We ran into challenges while trying to get the Barcode scanning fully functional, we needed to overlay the information onto the live camera preview and this was not possible using the initial app we developed using the Ionic Framework

Accomplishments that we're proud of

Getting a fully-functional app ready in 18 hours

What we learned

Android Development, node.js, postgresql, Heroku

What's next for CalCam

More advanced nutrition tracking features, a possible integration with the myFitnessPal database to get more info on products.
