Features:

  • Hear your notifications and mark them as read
  • Check details of your last activity
  • Get the names of all your repositories and:
    • Get the open issues
    • Get the open pull requests
    • Get the most recent commit message
    • Get the most recent activity
  • Stalk a user to find out what they've done recently
  • Find an angry programmer's commit message

and more!

Tech:

Main server runs on AWS Lambda using Python 2.7 (sadly Python 3 isn't available on Lambda yet), while the sign up runs on Flask with Python.
