Inspiration

The healthcare industry is a field that is filled with a lot of personal and sensitive information. Many companies including Insurance corporations take advantage of centralized systems to buy health information on patients and increase premiums. We got inspired by this growing concern and created a de-centralized system that cannot be tracked.

What it does

Assists Patients in sending information on current medication to practitioners using blockchain keys. Practitioners can access this information and download a json/txt file.

How I built it

Using React as a main front-end framework and using BlockStack as a login and decentralization system. Used REST Calls and several other technologies to share information. Work was divided between 2 people.

Challenges I ran into

Coming up with the intial idea and having a streamlined approach was the most difficult part.

Accomplishments that I'm proud of

Created a working app!!!

What I learned

All about the blockchain ecosystem!!!

What's next for PresCrypto
