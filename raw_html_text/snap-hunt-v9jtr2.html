Inspiration

it all started at mcdonalds when me and my friends were looking for coupons to buy a big mac trio, my friend got a big load of napkins and so, obviously to make a good story; we wrote our idea on a napkin.

What it does

the app is sort of a game that lets user 'bury' treasures using pictures and their locations. using the clarifai api, a user is able to retrieve tags and post a 'treasure' within a location with only the tags as clues.

How we built it

we used the clarifai api to process the images we were taking and returning the tags.

Challenges we ran into

making the image upload successfully

Accomplishments that we're proud of

using the clarifai integration

What we learned

a lot, this experience was one of the best, if not the best we've had so far.

What's next for Snap Hunt

finish it and polish it for the market!
