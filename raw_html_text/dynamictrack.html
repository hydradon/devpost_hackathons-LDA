Inspiration

Eyetracking software and dynamic scrolling. Accessibility for mobility- and vision- impaired.

What it does

It lets users read normally, updating text outside their peripheral vision. This way, the next sentence is always ready when they want it, without distracting from the current sentence or causing motion sickness.

How I built it

Java. No sleep.

Challenges I ran into

Actually getting the display to work was a challenge

Accomplishments that I'm proud of

I was able to code it myself, without much experience with UI

What I learned

Hacking is anti-sleep

What's next for DynamicTrack

Adding gestures and alternate inputs. Adding peripheral text mode.
