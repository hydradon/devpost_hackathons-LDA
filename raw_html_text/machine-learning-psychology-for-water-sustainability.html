I wanted to see if I could identify a water appliance being used by the sound of its water flow. I found that I could and built a machine learning model that can accurately identify shower use and toilet use. Appliance Usage can then be converted into water usage in gallons.

The ML model can be used as the framework for future applications that help consumers save water. I modeled one such app: Water saver which notifies users on the lot water consumption and provides tools to help them reduce.
