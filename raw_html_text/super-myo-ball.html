Inspiration

Our main inspiration was trying to mix interesting hardware and intuitive gameplay. We are trying to win best hack by first time hackers, organizers choice, or the overall best hack. We wanted to make something fun while having fun doing it. We also wanted to learn new skills at the hackathon. This was our first interaction with C#, unity, and the Myo.

What it does

It uses a Myo controller to manipulate the orientation of a plane to move a ball across a level. There are multiple levels, unique challenges, and custom music.

How we built it

The game was built in Unity along with the Myo SDK. This involved writing scripts in C#/javascript and using the unity interface for level design. Music for the game was created using SonicPi.

Challenges we ran into

The main challenge was setting up and creating gameplay that feels good with the Myo. We had to design a system that takes 3D movement by the player and creates something enjoyable. This involved a lot of play testing and bringing in other hackers to try out our game.

Accomplishments that we're proud of

Gameplay with the Myo, music, and level design.

What we learned

We learned a lot about the unity engine and writing scripts, primarily in C# but with some javascript as well. We also took a lot away from working with the Myo and it's SDK.

What's next for Super Myo Ball

In the future Super Myo Ball can be easily expanded upon. What will come next is new mechanics and levels.
