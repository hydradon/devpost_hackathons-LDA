Inspiration

Trying to make the interactions between Flight Attendants and passengers less complicated and more intuitive.

What it does

This mobile app allows passengers to directly contact the flight attendant about their requests for items such as blankets, snacks, beverages...etc.

How we built it

We used android studio to create the basic framework of the two sister applications. One application for the passenger to request items and another application for the flight attendant to process the requests. To connect the applications we used Firebase to make a Real-Time Database.

Challenges we ran into

Using Firebase for the first time, we ran into problems one would expect on their maiden voyage of a database engine. One particular challenge we had was reading the data on the flight attendant side. We didn't know how to traverse the data structure that we made.

Accomplishments that we're proud of

We were successfully able to get the two applications to communicate via Firebase.

What we learned

We learned how to implement a Real-Time Database using Firebase.

What's next for Sky Buddy

Obvious UI improvements, as well as adding more features to the flight attendant application. We would like to use the airlines membership structures to automatically apply coupons to the passenger's order.
