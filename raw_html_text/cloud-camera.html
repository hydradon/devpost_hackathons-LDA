Inspiration

Take pictures on out of storage phone.

What it does

When the iOS camera doesn't let take pictures because out of storage, Cloud Camera let's you use the iPhone to take pictures, permanently store them on the phone's last storage space as buffer, then pushes it to the cloud - this allows to take pictures on an out of storage phone.

How I built it

Using Xcode, Swift, Dropbox API, and Google Drive API.

Challenges I ran into

First time using the Dropbox API and Google Drive API

Accomplishments that I'm proud of

The app works.

What I learned

A lot.

What's next for Cloud Camera

Bug fixes, beta tests, and release to iOS App Store.
