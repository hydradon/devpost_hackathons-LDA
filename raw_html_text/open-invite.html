Inspiration

It's never fun being alone or bored, and who doesn't want to meet new friends? Our website allows strangers to bond by leaving an open invite for events hosted by strangers, for strangers -- and soon to be friends.

What it does

The website allows you to mark on Google Maps a location for your event and set start and end times, as well as (stretch) allow you to view others' events in a nearby radius. The event is managed by the user, but also erases itself after the event has passed.

How we built it

The website is built exclusively with html, css, and javascript. The Google Maps API allowed us to work with locations and markers.

Challenges we ran into

Making the backing database and storing the information on a remote server is still having immense difficulty.

Accomplishments that we're proud of

Being able to use the foreign API and link it between a nice-looking front end website and the back end functionality was incredibly gratifying.

What we learned

More familiarity with front end development, use of Google Maps API, familiarity with Javascript (neither of us have used JS before!), basic premises of databases

What's next for Open Invite

1) use mongodb/firebase for database improved performance/less janky, and need to implement cross-device capability 2) implement friend lists and actual user accounts instead of just mockups 3) privacy settings for events 4) beautify inputs/prompts cuz they ug.lieeee 5) categorize events, eg. work, recreation, meals, etc 6) rsvp 7) ability to edit events by the user
