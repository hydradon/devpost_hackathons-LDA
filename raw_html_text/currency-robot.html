Inspiration

Always wondering what any country's currency translates to in another country, and other countries may not have access to the internet, but our hack does not require internet.

What it does

Translates the currency amount from one country to another.

How we built it

Using python and connecting it to a JSON server

Challenges we ran into

Parsing the string to analyze the correct information.

Accomplishments that we're proud of

Being able to write specifications for code before implementing it.

What we learned

How docstrings and comments help us to organize and plan out our code.

What's next for Currency Robot

Designing the front end to make it more visually appealing for the user.
