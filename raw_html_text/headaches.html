Inspiration

We have all had problems moving and so we came up with Headaches as a solution to that problem.

What it does

Headache is a basic Airbnb-style system built for the PennApps XII Hackathon! This service helps people find temporary storage for personal items using trusted third-party contractors.

How I built it

We used Bootstrap and several other libraries to make our website. We modified parts of the internal structure of the different libraries. We linked to Parse to develop the backend.
