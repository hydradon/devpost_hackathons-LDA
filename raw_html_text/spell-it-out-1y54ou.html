Inspiration

Speaking to industry experts and wanting to collaborate on a common interest.

What it does

Parses web pages to find acronyms and provide meanings.

How we built it

Setup a web service that utilized several databases, saving thousands of entries in a MySql databases hosted on Google Cloud that is called via API end points by a Chrome extension running on the client's browser.

Challenges we ran into

Setting up the web service on the Google Cloud, finding accurate databases to use, adjusting to the learning curve as none of the group members were proficient with the tools being used .

Accomplishments that we're proud of

Successful integration of many tools, languages, and frameworks. Building a successful product despite the lack of prior experience.

What we learned

We learned about new frameworks and tools. The team was able to experience working on the full application stack.

What's next for Spell It Out

Improving on the already spectacular product, looking to expand on the tools being utilized to develop a more efficient and user-friendly product. Check out link for more details!
