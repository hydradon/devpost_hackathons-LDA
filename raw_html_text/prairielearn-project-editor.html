Inspiration & What it does

PrairieLearn is an open source website for homework/exam assessment. Previously, course staff needs to write JSON and HTML file offline and upload them manually, which is inconvenient. We modified the source code to support online question modification and an online text-editor which supports MarkDown.

How we built it

Siwei implemented the text-editor, Pengyu worked on both front-end and back-end and Wei dealed with front-end.
