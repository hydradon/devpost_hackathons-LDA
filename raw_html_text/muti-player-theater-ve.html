There are a couple of interfaces in the Virtual Environment for watching media like images and videos but none of them have the capability of sharing those content in the same environment with friends and other companions.

Some of the difficulties faced were learning to building projects from the scratch in a relatively new device. And cutting edge technologies without much online help for problems encountered.

Technologies used are : Unity 3D , Photon Unity Network , C#

strategy used was to building a texture to play videos and host this environment on cloud based services so that it can be synchronously accessed by all players.
