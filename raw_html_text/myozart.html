Inspiration

The Myo looked cool.

What it does

It tracks your conducting skills as you conduct a short piece of music, then gives you a score based on how well you hit the beat.

How we built it

We downloaded Unity, then added the Myo plugin, which already had a baton-like example. Then, we used a bunch of other libraries to add audio and visual effects and beat detection.

Challenges we ran into

Using the internet (it was slow), coming up with an idea, using Unity (we had 0 experience).

Accomplishments that we're proud of

It works! (For the most part).

What we learned

How to use Unity, Blender, Myo. Also, if something's really hard, there's probably an easier way to do it.

What's next for Myozart

Aesthetic appeal, more songs, more complex scoring algorithm.
