How it works

We used Unity to create a virtual environment that you can interact with (and try to escape from) by using the Google Cardboard.

Challenges we ran into

  • Version control and splicing files together.
  • Learning how to use Unity.
  • Hardware inconsistencies (the Galaxy S5 has weird issues that make parts of the game awful)

Accomplishments that we're proud of

  • It works.
  • It is pretty cool.
  • great level design with avant-garde visuals
  • fun for the whole family

What we learned

  • How to use Unity
  • How to use Google Cardboard
  • How to manage scripts and assets
  • How to 3D model things
  • After hours of coding and little to no sleep, we found that we slowly descended into insanity and gravitated towards Bill Gates
