Inspiration

Wegmans' API was intriguing to us so we decided that we would create something useful and new to help those that are vegan, have Celiac's disease, or other dietary restrictions.

What it does

This would be an add-on for Instacart (Wegmans' partner for delivery services) to assist people with dietary restrictions or preferences in their online shopping experiences.

How we built it

We built this by improving upon the online and in-store functionality of the Instacart app.

Challenges we ran into

None of us were familiar with using an API in this manner, so this was a challenge. Learning about APIs from scratch took hours.

Accomplishments that we're proud of

However, we are proud of our model that we have created to replicate this aspect of the app.

What we learned

Overall, we learned some basics on how APIs work and can be called.

What's next for Good Food Good Mood

We would love to incorporate more settings in the future!
