Inspiration

We wanted a better way to schedule conference workshops to help student-led events.

What it does

Idea: Take framework information from the organizer, collect details from the workshop presenters, and then collect preferences from participants. Schedule the workshops accordingly and send text message reminders of room numbers shortly before each workshop

How we built it

User input via Google Forms Scripts; text messages via Twilio

Challenges we ran into

New to web development, text notifications, different programming languages

Accomplishments that we're proud of

Learned new skills

What we learned

Twilio, Google Apps Scripts, website hosting

What's next for Conference Scheduler

More user input categories Completing the scheduling logic and optimization
