Inspiration for this app was because it is always difficult to find events on campus to go to.

What it does, This app displays all the events/ club meetings happening on campus, filtered by majors to make it easy to find events/ club meetings you want to go to.

How we built it We learned how to program using Swift in Xcode.

Challenges we ran into We did not know how to program in swift so everything was new.

Accomplishments that we're proud of We are proud that we actually created something that is functional

What we learned We that you could really do anything if you really set your mind to it.

What's next for Eventable We hope to make it look better and finish the tableview that presents the events/ club meetings happening on campus.
