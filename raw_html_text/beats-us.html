Inspiration

Ever been to a party or road-trip with a bunch of friends? Music is so important. So far, creating a shared playlist is a highly manual process. We are changing that.

What it does

A user authenticates through Spotify and joins a playlist. Then we run an unsupervised learning model on the songs of everyone at the party and dynamically create a Spotify playlist with everyone's shared interests.

How we built it

We built the platform in python, node.js, express.js, and vanilla javascript.
