What it does

Trustify allows mods and streamers to create a more accountable community by “trusting” a user. This user is then known in the current and other streams for being an upstanding community member. The mods and streamer can use the chatbot to trust or untrust users and this will be displayed to the channel viewers in the panel below. By rewarding and recognizing members who are exceptional, we incentivise a safer community and platform.

How we built it

We used the Twitch Developer Rig to develop our display panel and Glitch to create our chatbot. We learned a lot about external libraries available to Twitch developers.

Challenges we ran into

Getting familiar with how Twitch’s example code was all interconnected to determine how to structure our code was a fairly involved process that we didn’t expect to take as long as it did. Also our team was not particularly familiar with the node modules that are used to build extensions (such as the example ones).

Accomplishments that we're proud of

We created a functioning IRC chatbot within our first Twitch extension. We learned a lot more about the different pieces of an extension and how they work together

What's next for Trustify

As seniors in college, we all have a lot of free time and the desire to learn more skills. We hope to stick with Trustify and one day turn it into a full extension that streamers can use to build their communities in an easy way.
