Inspiration

Certain documents are confusing and long. They require humans to guide you through. We made it possible virtually.

What it does

Video con-call with documents signing done together. Also document highlighting that can be synchronized between users.

How we built it

Docusign Embedded Signing API that allows us to embed the documents. WebRTC to allow video conferencing.

Challenges we ran into

Synchronized document highlighting.

Accomplishments that we're proud of

Making it easier for users around the world to sign together.
