Inspiration

To prevent cyber bullying across the globe.

What it does

Our app is designed as a filter that can block out certain words or phrases that are offensive. It also allows the user add a word or phrase and it can start filtering that out too.

How we built it

We built this app using Xcode, an ios development software.

Challenges we ran into

We still haven't been able to get the adding feature down but it's coming.

Accomplishments that we're proud of

We are proud of our filter functionality.

What we learned

We learned that it's more complicated than you think it is to make a simple database inside of an iphone app.

What's next for CyberSafe

We plan to improve the interface so that it's more user friendly. We are going to make it so that it can be integrated in social media apps. We will fix the adding functionality.
