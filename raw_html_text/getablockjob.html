My Place is a collaborative online canvas game made with pixel squares.

Inspired by reddit. Users can place blocks on an shared empty canvas, but had to wait before placing another. There are various kinds of blocks, each with various functionalities. Different mechanisms and colors gives impossible possibilities. Collaborate with friends or start a war is your choice. Special blocks like wiresblocks use algorithms to create and solve fun problems.
