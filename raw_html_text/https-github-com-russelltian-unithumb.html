Inspiration

David always can't find the right way and has a poor sense of direction

What it does

It has a thumb on the hat always pointing to the direction the user should go along with , it optimize the current navigation system by replacing confusing voice navigation and disturbing visual navigation

How we built it

We used wires to build a string pulling mechanism to control the direction of the thumb. And by using swift, we utilize Mapbox SDK to build navigation system, and sending data we retrieved from SDK to the hardware part via bluetooth

Challenges we ran into

There was noise in magnetic field in the building and compass could no work All the team members have very little of IOS dev experience, so staring from stretch We ran out of time for submission.

Accomplishments that we're proud of

Built the hardwares very efficiently. Learned Swift from nothing.

What we learned

teamwork
