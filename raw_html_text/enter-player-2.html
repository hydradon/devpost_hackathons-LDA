Inspiration

Breaking down barriers set by unfriendly exclusive application environments to bring people closer together through a shared gaming experience.

What it does

Enter Player 2 enables developers to produce scalable cross-platform games with persistent state and automatic matchmaking. The framework is simple and suitable to many genres of games.

How I built it

We used the Unity framework to develop client applications that run on multiple OSes with minimal code change. Server-side code is environment-agnostic JSON, with a straightforward communication mechanism easy for developers to implement.

Challenges I ran into

The Unity framework does not handle libraries that contain native code, and the best JSON libraries are incompatible. Additional library issues prevented deployment to Windows Phone before the end of the hackathon.

Accomplishments that I'm proud of

Getting our code working on 5 platforms working over the course of one weekend.

What I learned

Library support is as important as primary build tool support.

What's next for Enter Player 2
