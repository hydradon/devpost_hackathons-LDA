Inspiration

Many grandparents are in need of medical assistance or care giving. Since their children are often too busy or live far away, they are not able to care for their parents and are not willing to put their parents in retirement homes. This application could solve this problem.

What it does

Our application connects care seekers with caregivers within a certain location.

How we built it

We used Swift and Xcode to create the front end portion of our application and a user-friendly interface for the elderly and caregivers to use. We connected the front end to the back end, which was created using python and django.

Challenges we ran into

We initially had a little trouble starting the learning process for Xcode and swift. We also had trouble connecting the front end and back end, but we were able to make it work.

Accomplishments that we're proud of

We are proud of our working application that implements location services, permission handling, profiles, user info/databases. We are most proud of the fact that we could help others in need of assistance who may not be up to date on available resources.

What we learned

We learned how to code in swift and app development requires a thorough planning process.

What's next for iCare

We will integrate more features in the profile section, such as a profile picture. We will add a rating and certification system to make sure caregivers are trustworthy. We will also add a messaging feature for care givers and care seekers to communicate.
