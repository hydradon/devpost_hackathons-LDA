Inspiration

A boy has a dream and curiosity in building something. A boy has no expertise nor can afford to do it. How can we help him? We provide an AI system to reconstruct 3D objects from 2D sketches. We provide a handmade 3D printer to print the 3D object.

What it does

The raspberry pie captures image and sends it to the cloud. The generative Adversarial network reconstructs 3D image from 2D sketch and produces 3D voxels. The 3D printer takes the 3D voxel and prints the object.

How we built it

We used the IKEA dataset to generate 3D models from 2D sketches. We built a 3D printer using the parts from MakeHarvard with aluminium frame, 3 actuators and 1 syringe based extruder for x,y,z axis

Challenges we ran into

The methyl methacrylate solidification was not precise since we couldn't control the heat

Accomplishments that we're proud of

The 3D model is successfully generated from hand drawn 2D sketch. The 3D printer successfully used methyl methacrylate to print a chair.
