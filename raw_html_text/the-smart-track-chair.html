Inspiration:

We thought that disabled individuals who cannot use their hands should be able to use a wheelchair in an ergonomically simple way.

What it does do:

It allows for the control of a power wheelchair by movement of the operator's head.

How we built it:

We used an microprocessor board for wireless connection.

Challenges we ran into:

Hacking into the joystick controller.

Accomplishment:

We were able to successfully move the wheelchair through a wireless connection.

What we learned:

C++ programming skills, creating a wifi local network.

What's next for The Smart Track Chair:

Gettting the microprocessor completely integrated to the joystick controller.
