Inspiration: A passion for baseball and processing data using an application.

What it does: It takes a user-inputted question and answers the question by querying information from a database.

How I built it: We used PythonAnywhere to create the web app using Python, HTML, CSS, and a MySQL server.

Challenges I ran into: Taking into account possible user inputs to be able to output accurate answers and dealing with the problems with the server itself.

Accomplishments that I'm proud of: Providing an avenue for getting quick answers to these interesting questions.

What I learned: It is important to plan out each step and weigh different options to complete each step of the process.

What's next for Process IT Now! MLB Edition: Attempting to add graphs of the data to provide visual aid instead of just a simple answer and attempting to incorporate speech recognition to provide another option for user input.
