Inspiration

Every year 640,000 tons of fishing gear is lost or abandoned in our oceans. Hundreds of thousands of whales, dolphins, seals, and sea turtles are killed each year due to entrapment in lost gear. We want to help prevent this in our sensitive ocean environments.

What it does

Simple User Interface : Quickly report lost and found fishing gear Accessibility : Mobile device on boat acquires satellite location for data upload when cell phone connection returns near shore. Elegant Design : App Written in Xcode in Swift language and accessing Firebase database.

How we built it

Firebase as the central data store Swift for the iOS application Angular for the web portal All linked together in real-time!

Challenges we ran into

Finding good parseable data about fishing gear types

Accomplishments that we're proud of

Created a real-time web portal for marine domain awareness using data from the app in two days!

What we learned

Learned a lot about the severity of this problem. We'd rarely heard about this issue before and had no idea the amount of sea life that gets captured every year.

What's next for NetGain

Calculate/predict amount of lost biomass. Autodetect net types through image recognition Track aggregate net movement of the gear Manage ocean resources using Eco-analytics
