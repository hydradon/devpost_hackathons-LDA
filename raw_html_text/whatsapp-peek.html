Problem I want to solve

We all have messages we want to read but not for immediate process. For example, messages from your boss during the weekend, or messages from group projects when you are out having lunch. However, WhatsApp read receipt feature makes this almost impossible.

What it does

It captures all unread messages and display it to you without sending the read receipt, so the sender doesn't know that you have read the message!

How I built it

It is built as a chrome extension that will activate when you browse whatsapp web. Bootstrap for UI, and plain javascript for logic.

Challenges I ran into

Communication and object passing among the injected javascript, background script and popup script.

Accomplishments that I'm proud of

Tackled all the challenges! And this is my first solo hackathon :)

What I learned

More javascript and google chrome's message passing protocol.

What's next for WhatsApp Peek

Better UI.
