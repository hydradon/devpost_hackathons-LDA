Inspiration

When we were younger, we saw that entertainment and toys geared toward girls always tended to be non-technical. Compare that to the cool gadgets and technology-related gifts boys received and experienced. No wonder girls are less likely to go into the STEM field when they grow up. We wanted to create a website that encourages girls to go into the maths and sciences, that gives them confidence in this field.

What it does

This website creates a platform on which girls can express their more artistic side, as well as experience potential technical and coding-related skills.

How we built it

We used HTML and CSS for the website. We also built the games using JavaScript, along with HTML and CSS.

Challenges we ran into

It was hard balancing the feasibility and originality certain games that we thought of. We had to make sure that the game was not too big of a project to accomplish. At the same time, we wanted to be creative.

Accomplishments that we're proud of

We finished the website, and have a working product!

What we learned

It's definitely good to plan and make sure communication is very clear, and everything is not off doing their own thing.

What's next for girlsruntheworld

Many features have yet to be implemented. We would like to add these features, as well as add more games.
