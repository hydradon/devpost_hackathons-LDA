A simple plop and the desire for a cool blonde

What it does

The skill virtually fetches a bottle of beer for the user from anywhere in the apartment, house or surrounding area. Alexa comments on her experiences and impressions while searching for a bottle of beer. In the end the user always gets a bottle of beer and sometimes also a beer can. The skill can also tell toasts and jokes about beer. The skill also includes an alcohol calculator with which users can roughly calculate their alcohol level.

How I built it

I programmed the skill with node.js and accesses an external table for certain data via API Call. It is also hosted in the AWS Cloud.

Challenges I ran into

The switch from ASK SDK V1 to V and the first implication of the display interfaces that have now been or will be replaced by APL.

Accomplishments that I'm proud of

I have already deposited documents with most of the Intent APL. And with a few specific Intents, the data is fetched externally via an API call.

What I learned

With a lot of time and patience I can master any programming and design topic.

What's next for Bierflasche

I would like to further develop and graphically expand the Dialog Management for the alcohol calculator so that it is even easier for the user to understand and at the same time more accurate.
