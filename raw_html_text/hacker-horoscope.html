Inspiration

My whole group looks at daily horoscopes and we thought that the judges would like one too.

What it does

Based on your birth symbol it gives your horoscope for April 7, 2018.

How I built it

I used java code with a scanner we created ways to make the correct horoscope to appear.

Challenges I ran into

We didn't know what the horoscopes were and we wanted the judges to get the most authentic experience.

Accomplishments that I'm proud of

We are proud of not needing help from anybody and that it is simple and not very complex making it simple yet interesting to use.

What I learned

I learned that we should keep the code simple so that it does not create any problems for the user and at the same time does not create any tough problems for the developers to deal with it.

What's next for Hacker Horoscope

The next is creating a horoscope for the next day or week, or month etc.
