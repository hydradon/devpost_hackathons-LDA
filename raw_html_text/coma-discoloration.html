Inspiration

Every time my friend sees that my chrome book's screen is in a monochrome-kind of filter, they always come up with a sad story to go along with it.

What it does

Currently, the only fully implemented things in the game are your player and the Pituite ( the fire-like slime ). The demo is FULLY compatible with a Xbox controller, not any others. For the keyboard, the arrow keys are to move and the 'C' key is to sprint. For the controller, use the joystick to move and the 'A' key to sprint. Other monsters are placeholders or not fully added. If you have figured out how to sprint through the walls, nice! This is meant to happen as future planned levels will contain puzzles in which you have to escape to get to the story.

How I built it

Using Game Maker Studio (Found on steam), I was able to mess around for a bit without actually starting my project. Thankfully, I was able to code without using the Studio's built-in pre-programmed buttons! This was made with the help of my friends and my partner for making this, my sister.

Challenges I ran into

There were multiple challenges that I was faced with. For example: Xbox controllers not working, Keyboard controls not working, continuously running off of the face of the map without anywhere else to go while sprinting. All were resolved.
