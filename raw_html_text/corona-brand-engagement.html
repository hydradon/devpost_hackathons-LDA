Inspiration

I work for Constellation Brands and thought it would be cool to use Amazon Sumerian to create a mini AR brand ambassador.

What it does

Answers questions about Corona Extra. Provides recipes.

How I built it

Amazon Sumerian and using external brand assets and models from Turbosquid.

Challenges I ran into

State transition issues, audio/mic on iOS using the sample project, had problems using video in the HTML 3d entity, HTML 3d images on iOS.

Accomplishments that I'm proud of

Build a 3D interactive experience that runs on the web.

What I learned

How to use Amazon Sumerian.

What's next for Corona Brand Engagement

More support on the iOS platform.
