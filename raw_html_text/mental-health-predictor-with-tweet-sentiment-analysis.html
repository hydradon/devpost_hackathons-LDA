Inspiration

Mental health has obviously become a huge problem. I figured that Twitter would be a great place to pull data from, since it so accurately resembles a person's general sentiment.

What it does

You can enter a username or general query and the program will automatically pull the latest tweets, analyze them, return the sentiment, and give you customized quotes to help you on your journey.

How I built it

Flask backend with python, naive bayes algorithm, react front end. I have also registered a domain with this: mentalhealthpredictor.com It is not synced up yet, because I need to set up the backend with a server, but this is my next step.
