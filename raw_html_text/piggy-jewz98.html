Inspiration

We were shocked by research we did showing how poor financial knowledge is in Canada. Made the goal to solve this through creating an app meant to instil basics of saving money and finance in youth.

What it does

Allows for user to set goal to attain certain item that costs x amount of money. Every time they complete a chore or task they are able to earn money from, the hit the associated button and their "piggy bank" increases its amount.

How I built it

Used Android Studio and Java.

Challenges I ran into

Figuring out how to use Android Studio. Getting an image button to work.

Accomplishments that I'm proud of

Learning how to build an app using Android Studio for the first time.

What I learned

How to build an app using Android Studio. How to create an app with a young target market.

What's next for Piggy

Testing the current model, making fixes based on feedback. Create parental controls such as passwords to ensure child's completion of task.
