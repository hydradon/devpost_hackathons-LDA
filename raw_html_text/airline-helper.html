Inspiration

We think there are too many apps that do different things; we wanted an app that did it all.

What does it do

It is an aggregated way to check fights, book flight, check in and check flight info all in one place.

How we made it

We made it with Visual Studio in the Visual Basic .NET language with no additional libraries or APIs.

Challenges we ran into

We had issues trying to code a desktop application to look more like a phone application.

Accomplishments that we're proud of

The app is pretty cool I guess.

What we learned

Sleep is important.

What's next for Airline Helper

Kiosks everywhere.
