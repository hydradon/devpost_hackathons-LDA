Inspiration

The idea of 0neFiesta was the with OneClick games Kahoot and JackBox Games were the insperation

What it does

Runs simple games that support only 1 input (usually a click or a tap)

How we built it

The code was built with node.js and Javascript The backend was built with node using socket.io, express and defined.js (custom project) The front end was built on react, bundled with parcel using ant.design, and socket.io client

Challenges we ran into

Having enough time to right the entire event handler system, also creating enough games with the knowledge at hand made this very difficult

Accomplishments that we're proud of

That the web is very forgiving, so even though mistakes were made, it wasn't super hard to make.

What we learned

Never make one click games ever again with multiplayer functionality, though doing mobile apps may be different

What's next for 0neFiesta

Maybe improve it to support custom games, or whatever really. It was just made for fun
