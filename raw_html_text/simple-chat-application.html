I was inspired to built this app after learning about socket.io.In developing this project, my understanding of web sockets deeply improved.I learnt how server side and client side communicate with each other. I used nodeJS with express on the back end and html for the front end with some css for light weight styling.The next thing for this app would be to incorporate audio and video elements into it and also use an AI chat-bot which gives answers to users questions.
