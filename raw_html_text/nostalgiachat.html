Inspiration

We miss PictoChat. The Nintendo DS local chat application was a great inspiration for this project. The art is modeled after the application as well as some of the functionality. We thought it would be nice to have the same thing on a PC or mobile device since there does not seem to be any chat applications that allow you to easily send drawings to people.

What it does

You can send drawings and messages to your friends.

How we built it

We used WebSockets with a React application as well as some other stuff. The messages are sent and updated in real-time on all connected clients

Challenges we ran into

We had no idea what we were doing for most of this project. Most of the project was made from scratch because we did know how to do it any other way.

Accomplishments that we're proud of

We had never used most of these technologies before this project and we managed to get something that works decently.

What we learned

We learned a lot about React and javascript as well as WebSockets since we had not extensively used those tools before.

What's next for NostalgiaChat

Make it look prettier, add more features.
