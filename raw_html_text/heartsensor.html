Inspiration

What it does

It uses sensors to detect the heart rate of the user who is wearing the device

How I built it

Developed in Fitbit studio which uses javascript and provides a Fitbit simulator

Challenges I ran into

The challenges I ran into were understanding how Fitbit studio works and developing code for sending data from Fitbit to other applications. Another big challenge I tried working was using an actual Fitbit watch for development but I could not use an actual Fitbit device for application development due to its weak Wifi signal.

Accomplishments that I'm proud of

Adding a background to a Fitbit application

What I learned

I learned to do basic javascript code in the domain of Fitbit

What's next for HeartSensor

Learning to add network abilities to send heart rate data to databases and other applications in android and apple ios
