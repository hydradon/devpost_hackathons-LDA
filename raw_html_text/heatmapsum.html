This website and mobile applications will also serve as a point of reference for future increases in tuition for both in-state and out-of-state students per university.

Algorithm: The quadratic regression formula:

The formula projects future data points by using the previous data sets, A,B,C. Our logic for the formula:

a = (( Σ x2y * Σ xx ) - (Σ xy * Σ xx2 )) / (( Σ xx * Σ x2x2) - (Σ xx2 * Σ xx2))< b = (( Σ xy * Σ x2x2 ) - (Σ x2y * Σ xx2 )) / (( Σ xx * Σ x2x2) - (Σ xx2 * Σ xx2 )) c = ( Σ y / $numValues ) - ( b * ( Σ x / $numValues )) - ( a * ( Σ x2 / $numValues )) where: Σ xx = ((Σx^2) - (($Σx)^2 / n)) Σ xy = ((Σxy) - ((Σx* $Σy) / n)) Σ xx2 = ((Σx^3) - ((Σx^2* Σx) / n)) Σ x2y = ((Σx^2y) - ((Σx^2* Σy) / n)) Σ x2x2 = ((Σx^4) - ((Σx^2)^2 / n)))
