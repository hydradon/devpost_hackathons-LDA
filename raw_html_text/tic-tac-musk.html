What it does

Talk about Elon in the text box, if you're nice he may let you win the game. If not, he will be impossible to beat.

How I built it

Went through a minimax algorithm tutorial on Youtube to learn how to build the unbeatable tic tac toe game and then used sentiment analysis from the node package manager to determine if the user is saying something positive or negative about Elon.

Challenges I ran into

Integrating the sentiment analysis into the minimax script.

What I learned

Learned more about node.js and got to get my feet wet with the Naive Bayes classifer, elementary level machine learning.

What's next for Tic Tac Musk

Even though it's impossible, I think Elon could beat the minimax algorithm. I would like to see Elon play himself in Tic Tac Musk.
