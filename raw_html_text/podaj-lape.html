Inspiration

We've been inspired by one of our friends, who wanted to adopt a dog, but when she realized how long this proccess is and how many dog shelters she has to visit before she finds her perfect pet, she gave up. Recently, we've also heared many news about abandoning the animals. We wanted to solve both problems at the same time.

What it does

Podaj Łapę is a web application designed both for adopting dogs from the dogs shelters and putting them into ones if we decide we are not fit to be the owners. Users can filter the dogs in the nearest area and choose their own preferences such as age or race. While searching through the shelters they can get info about the conditions in each shelter and is there a space for their pet.

How we built it

For backend we used Java + Spring and MySQL database For frontend we used ReactJS.

Challenges we ran into

API Requests working on Postman, but we ran into many problems while using the api in REACT. We had to redesign our database structure during the project.

Accomplishments that we're proud of

We think the app looks pretty nice, it's clear and intuitive. We're proud that we managed to split the work properly and everyone learnt something new.

What we learned

Backend guys learnd some more React and fetching REST requests. Frontend guys realized how hard it is to design a proper database.

What's next for Podaj Łapę

Getting partners from dog shelters and facebook group. In the future maybe we will extend our services not only for dogs but also for other animals :)
