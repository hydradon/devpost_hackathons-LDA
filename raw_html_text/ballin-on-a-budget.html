Inspiration

Managing money is hard and young adults need a fun way to start spending money wisely!

What it does

Manages spending limits: Asks user to set monthly goals for spending Highlights areas to control spending: Lets user select which spending categories they'd like to manage better. Uses Location to remind users: Remind users of spending limits when they are in a 2 mile radius of a location where they can spend money in this category. Provides tips that prioritize: free/reduced resources, housing inequality (goverment and city assistance), and environmental impact (clean transportation, sustainability)

Provides personalized experiences: uses Bitmoji to provide user feedback.
