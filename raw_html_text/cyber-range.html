Inspiration

This CyberRange project represents the first open-source Cyber Range blueprint in the world. It simply provides a researcher with a disposable offensive / defensive AWS-based environment in less than 5 minutes.

What it does

This project provides a bootstrap framework for a complete offensive, defensive, reverse engineering, & security intelligence tooling in a private research lab using the AWS Cloud.

How I built it

Terraform, vagrant, AWS

Challenges I ran into

cross operating system compatibility

Accomplishments that I'm proud of

User account creation with access keys, vagrant scenarios, testing

What I learned

CloudPosse Setup abnormalities : Cross Operating System issue when testing on Kali Linux; Windows getting started needs update for vagrant Organizational Accounts are different from IAM users with role access to an organization - I’ve created multiple unaccessible / undeletable accounts DHCP / Static IP Conflict
