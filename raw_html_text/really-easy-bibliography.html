Inspiration

Many of our group members have English and History Essays coming up which require extensive research, cited in MLA of course.

What it does

Really Easy Bib generates links and citations off of given keywords so that users do not need to manually find research links and create citations.

How we built it

We created it using PHP, HTML, CSS, Python, jQuery, Bootstrap, and Javascript. It is a very simple and easy to use website.

Challenges we ran into

We used a python bot to search Google Scholar for research papers and citations. However, there is an issue with Google preventing the program from searching their site. This issue is currently unresolved.

Accomplishments that we're proud of

Eventually, we managed to complete the basic features. We are proud that we made it this far!

What we learned

We learned that one cannot simply search the internet using google.

What's next for Really Easy Bibliography

We hope to add many more features, such as APA and Chicago citations.

Will be hosted at www.reallyeasybib.com .
