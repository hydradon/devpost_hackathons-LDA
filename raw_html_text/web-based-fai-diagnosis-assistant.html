Inspiration

We recognize a need for tools in the accurate diagnosis of FAI (a hip disorder), and decided to apply our technical abilities to the problem.

What it does

Takes as input an anterio-posterior X-ray of the hip/pelvis, and user's estimate of location of femoral head centre. As output, the program yields the lateral central edge angle, one of the key indicators of the presence of FAI.

How we built it

Python and scikit

What we learned

A ton about image processing algorithms

What's next for Web-based FAI diagnosis assistant

The incorporation of additional criteria for more accurate diagnoses of FAI. Also more streamlined, and a better user interface.
