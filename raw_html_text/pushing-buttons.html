The inspiration

The inspiration for our project was to push a button to receive sweet, effortless, instant gratification in those moments when we need it, or provide a significant other with some gratification when we feel they deserve it.

What it does

Pushing buttons is a software and hardware solution for cheering people up. When the user pushes our button, the arduino makes an HTTP request to our node.js server, which then performs a pre-user-determined action, which can be set from our website. The action can either be to order Chipotle from Postmate, or use Twilio to send a good morning text or a picture of a cat.

How we built it

We built our project using our experience and knowledge node.js servers, python, and javascript.

Challenges we ran into

Learning how to use the hardware was our biggest challenge.

Accomplishments that we're proud of

Learning and understanding our hardware was our proudest accomplishment.

What we learned

We learned knowledge of arduinos and their applications.
