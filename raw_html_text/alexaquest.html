Inspiration

We wanted to create a novel use for Alexa - what better place than a game?

What it does

It is a game completely controlled by the Amazon Alexa voice assistant.

How we built it

The backend service is ruby on rails, running on Heroku. The game was made in Unity using C#.

Challenges we ran into

Interfacing with Alexa proved to be harder than we anticipated.

Accomplishments that we're proud of

Getting the backend to work and interface Alexa with Unity.

What we learned

  • How to use the Alexa API
  • C#
  • Unity
  • Making 3D models in blender

What's next for AlexaQuest

We plan to make the game a little more than just a tech demo.
