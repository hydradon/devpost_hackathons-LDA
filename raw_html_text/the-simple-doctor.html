Inspiration

One of us is a biomedical engineering student so the medical terminology seemed interesting and the whole idea of a chat bot seemed like a cool idea to learn how to implement.

What it does

Type in a message and get a more simplified definition of a key word.

How we built it

The Azure bot framework was used to make the bot. A java GUI was made to show the front end.

Challenges we ran into

Creating the bot and connecting the bot to the GUI were the major challenges we ran into.

Accomplishments that we're proud of

Being a team of two and learning skills we would not have learned if we were to have stayed back in Iowa.

What we learned

Started to get an understanding of Azure and learning how to implement text to speech.

What's next for The Simple Doctor

Full bot integration, making it more user-friendly (by using animation on the UI and being able to speak to it instead of having to type in the phrase), enable access to a large data source so it is able to give back more responses, possibly a nearby doctor lookup
