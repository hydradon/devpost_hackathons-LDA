Inspiration

We want consumer to have the upper hand on their finances and have control of their lives. We set out to accomplish this by linking several coupon systems together to save customers the most money possible

What it does

Links several coupon systems together in one convenient location

How I built it

Android Studio, HTML website builder, Xcode. The android app interfaces with google's cloud platform Firebase which houses the JSON packages.

Challenges I ran into

Connecting firebase to the android application to deliver us the data from the JSON files as well as web scraping the necessary coupon and savings data from multiple sites.

Accomplishments that I'm proud of

Finished 3 interfaces for our end users.

What I learned

A LOT. As a group we learned JSON packages, web scraping, web sockets, android studio, Xcode, HTML builder and setting up/registering a domain, and chrome developer tools.

What's next for Cashwrap

Full integration with vendors API (ibotta, HEB, etc) as well as realtime updates to the app for up-to-date coupons and deals to maximize savings. Improved UI/UX functionality across all three platforms.
