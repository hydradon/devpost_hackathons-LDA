Participando para el concurso del dominio: wwwpuntotech.tech

La inspiración nace en la necesidad de disminuir la contaminación y mejorar la movilidad en la ciudad de Puebla. A través de la aplicación queremos animar a los usuarios a utilizar el transporte colectivo como medio de movilidad para así dejar de ser tan dependientes de los autos que tanto daño causan a nuestro planeta.

La aplicación ofrece una interfaz amigable que con ayuda de la gamificación incentiva a la sociedad a usar transporte público. La aplicación busca ser un ayudante para los viajes en transporte colectivo guiándolos en el camino a su destino, mejorando así la experiencia del usuario.

La aplicación fue desarrollada haciendo uso de las herramientas web Vue.js y Bootstrap, además realizamos un mock up con la herramienta de diseño Adobe XD. Se bosquejo una base de datos con MySQL pensando a futuro.

Tuvimos un problema a la hora de querer usar Google Cloud para montar un servidor web. Usando la herramienta GitHub se nos presentaron varios conflictos entre ellos uno destacó que fue el merge conflict y en reiteradas ocasiones tuvimos que solucionarlo.

Estamos orgullosos de la idea que tenemos ya que pensamos que esta puede ayudar al planeta y mejorar la movilidad urbana en la ciudad de Puebla. Creemos que esta idea tiene mucho potencial ya que en un futuro el transporte colectivo será una necesidad.

Algunos integrantes aprendieron a usar Vue.js y GitHub. Al igual aprendimos el proceso de usar Google Cloud y trabajar bajo presión para entregar un resultado de calidad en poco tiempo.

Se busca implementar una base de datos a futuro para que con herramientas de machine learning poder optimizar el uso del transporte colectivo y sobre todo llevar la seguridad de los usuarios un paso adelante.
