Inspiration

Create a game for people who don't know anything about finances and how to manage their money.

What it does

It takes you through a pick your own adventure style game, starting in university. You go through and spend money on tuition, then you enter the work force. In the work force, you can use the money you have to trade stocks, and invest your money.

How we built it

Using python, we created a word-based text game

Challenges we ran into

The main challenge we ran into was learning how to debug all the code. This was definitely the biggest piece of code that I have written, since I am really new to programming. It was hard to keep on task and stay organized.

Accomplishments that we're proud of

We are proud that the game is now able to run from start to finish without any bugs.

What we learned

We learned a lot about writing efficient code, and ways to simplify processes.

What's next for Money Manager Game

Next, we would want to put the Money Manager Game on a GUI. It would create a more visually appealing interface, and be more user friendly.
