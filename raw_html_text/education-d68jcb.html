Problem

There are 53 schools in the Montgomery Public School System. With new families military and civilian moving into the area it can be hard to find a new school that fits the entire family needs.

Solution

We have created a website that filters through a database to help families find the school that best matches their wants and needs.

Features

The ability for both the school and families to have a login so they can either change their profile or favorite new schools.

How we built it

We used the Raspberry Pi to host a database and website. We also created and website using HTML and CSS and JavaScript2
