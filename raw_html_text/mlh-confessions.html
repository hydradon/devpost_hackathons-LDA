Inspiration

We were inspired to create this due to the lack of WiFi during the beginning portion of MLH Prime. We thought it would be good if hackers could give anonymous constructive criticism in order to improve future MLH hackathons.

What it does

This webapp allows hackers to anonymously tweet from the MLH_Confessions twitter account.

How I built it

We used the Twitter API to tweet what the user inputs.

Challenges I ran into

The lack of WiFi in the beginning set us back a bit, but allowed us to think about what we wanted to make some more. Another challenge we faced was users tweeting inappropriate and/or offensive content. We promptly took the webapp down and added a filter in order to keep the tweets clean.

Accomplishments that I'm proud of

What I learned

We learned how to use the Twitter API, and that we should never give hackers complete freedom over posting anonymously.

What's next for MLH Confessions

We want to implement sentiment analysis on the tweets to see the overall reaction towards MLH. We also plan to have a more comprehensive filter that not only filters out inappropriate and offensive tweets but also irrelevant content.

Note: due to inappropriate tweets, the main site has been taken down by us, but you can still check out the tweets that are on MLH_Confessions.
