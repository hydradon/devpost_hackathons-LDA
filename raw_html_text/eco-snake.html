Inspiration

We all came into this hackathon highly unexperienced - some us have had never written a line of code before. We all had two things in common: (a) we liked playing snake and (b) we all wanted our project to make a difference. That's how we came up with EcoSnake!

What it does

EcoSnake is a variation of Snake where a snake is eating up a can on grass, and after the game ends, a fact about climate change and what you can do to help.

How we built it

JavaScript, CSS, HTML
