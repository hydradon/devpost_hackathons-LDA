Inspiration

We want to help high school girls interested in STEM fields find the information they need to make an informed decision on what college to attend.

What it does

Provides information about colleges that specialize in technology and STEM fields to high school girls to help them make a decision about attending college.

How we built it

Using HTML, CSS, JavaScript

Challenges we ran into

Not having experience with JavaScript.

Accomplishments that we're proud of

Layout, color scheme, map.

What we learned

JavaScript, HTML, how to make a pitch

What's next for GirlSRC

Adding information for more colleges and including more information for each school. More blog posts will be included, and maybe a mobile app will be added in the future.
