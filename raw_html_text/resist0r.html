Inspiration

Working in labs can get frustrating when resistors are not organized properly. Reading resistor values is a long and tedious task.

What it does

Resist0r detects the resistance of the resistor by analyzing the colors seen on the resistor.

How I built it

Resist0r was coded using Python. We used the OpenCV library to detect the resistor as well as OpenCV's color recognition features to read the color pattern.
