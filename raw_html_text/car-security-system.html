Inspiration

México enfrenta una situación creciente en el robo de autopartes, al darnos cuenta de esta problemática, aunado a experiencias propias decidimos tomar acción.

What it does

Se trata de un sistema automático que se activa cuando el usuario pone en marcha la alarma de automóvil y se encuentra estático. Al momento de sentir un siniestro, el sistema notifica al propietario con la finalidad de entrar a un portal de internet que permite saber el estado del coche y tomar acciones pertinentes tales como prender un alarma o tiene la posibiidad de dar una descarga eléctrica.

How I built it

Tomamos como base una Raspberry Pi3 la cuál nos permitió desarrolllar código en python, con ello implementamos código apartir de librerias libres de GPIO para interactuar con el mundo físico, y el framework de flask para crea un servidor local. Por otra parte, la descarga eléctrica se constuyó con de manera modular con aislamiento óptico para proteger la circuitería interna. El sistema de alto voltaje esta compuesto por un chip TL-494, el cual es un multivibrador astable controlado por fibra óptica para evitar ruido en los sistemas digitales. Todo este sistema controla un transformador de 18v de entrada hasta 20kv en su salida. Posteriormente el voltaje entra a un multiplicador de ganancia 3 para aumentar la salida hasta 60kv.

Challenges I ran into.

La fallida implementación de Netcat con Named Pipes para tener una latencia pequeña que pérmita una imagen fluida en el servidor, luego otra problema que tuvimos fue la coordinación de todos los framework y subprocesos corriendo al mismo tiempo, el hardware a utilizar y por último el tiempo.

Accomplishments that I'm proud of

Logramos implementar el sistema de detección de cambios de inclinación del vehículo, además de poder enlazarlo con el framework de Flask para poder encender o apagar los sistemas, y proporcionar video en una red local.

What I learned

Se aprendió al desarrollo de partes en conjunto, además de la lección de trabajar por etapas y tiempos definidos para reducir los errores, también el desarrollo de código html en conjunto con python, y el uso del giroscopio en una raspberry.

What's next for Car Security System

Lo siguiente del sistema, es acondicionarlo a un sistema que incluya los espejos del carro, con la final de tener una defensa adicional, además de mejorar el sistema con una aplicación móvil y más amigable.
