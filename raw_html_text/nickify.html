Inspiration

People need more Nicks in their life!

What it does

Turn everybody into Nick by uploading selfies/group photos & nickifying them! Plus a few bonus transformations ;)

How I built it

  • Using a Machine Learning model to find bounding boxes of faces in photos via Clarifai API, then replacing the bounding box with newer images.
  • Implementing backend server with Python Flask.
  • Hosting server on Google Compute Engine using Terraform to automate the process.
  • Storing data and stats on Firebase and Google Cloud Storage.
