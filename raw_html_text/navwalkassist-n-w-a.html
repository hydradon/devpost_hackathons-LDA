Inspiration

Wanted to help out blind people navigate the world better and more safely

What it does

Warns the user if there is a obstacle in front of the person

How we built it

We used an Arduino UNO 101, ultrasonic sensor, speaker, stick, battery bank

Challenges we ran into

Finding the stick, getting the ultrasonic senor to work

Accomplishments that we're proud of

First Hardware Hack

What we learned

How the Arduino works, how the ultrasonic sensor works

What's next for NavWalkAssist-N.W.A

Multi-directional sensing, use an SBC like Raspberry Pi instead.
