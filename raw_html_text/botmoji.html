Inspiration

We love Slack. Slack bots make Slack easier. Therefore, we love Slack bots.

What it does

The bot takes in a user's Slack message as a String and outputs the message with emojis after a word if there is an emoji associated with it. The bot is activated using the /botmoji command.

How we built it

We used StdLib to build a serverless Slack bot. We coded the backend of the bot initially using Java and Python, then translating the code into Javascript. We coded the frontend in Javascript.
