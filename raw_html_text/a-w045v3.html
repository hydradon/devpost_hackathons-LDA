Inspiration - We enjoy listening to music for relaxation purposes. We have deep interest in learning a musical instrument and why not learn it with technology?

What it does - Recognizes hand gestures to play notes on a keyboard.

How we built it - Technologies used: Python and Leap Motion

Challenges we ran into - Identifying key strokes based on the hand gestures, changing color of keys on key press, taking care of cases where user presses same key more than once.

Accomplishments that we're proud of - We completed this in time

What we learned - How to use Leap Motion to recognize gestures

What's next for a - We would like to implement recording feature in Cayano. On higher level, we would like to introduce machine learning in Cayano. We'll train Cayano with some songs and as a result it can identify the notes itself.
