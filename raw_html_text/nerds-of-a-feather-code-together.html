Inspiration

Wanted to make a website that would encourage nerds like ourselves to be healthier by leveling up with daily tasks

What it does

Creates daily tasks for users to complete to gain a more balanced, healthier life style.

How we built it

MEAN stack

Challenges we ran into

None of us had any experience with MEAN. We wanted to gain experience with Angular and Node.js which is why we went with that stack. Most of the first day and half of the second was learning Angular and how to build the front end. Then we had to switch from SQL lite database to mongoDB because we couldn't get the front and back end to communicate properly.

Accomplishments that we're proud of

Functioning app with parts connecting to a mongo database

What we learned

Learned a lot about the MEAN stack
