This is a Domain.com domain-name entry. The domain I wanted to register is: samoasamoas.com. Unfortunately by the time I had this idea, the Domain.com coupons expired, so it's just hosted on my github pages.

I did not spend the entire hackathon working on this. Most of my time was spent with a dead-end Pebble app followed by a couple of other endeavors, none of which are in a state I'd be satisfied presenting. I thought the joke was worth spending a very short amount of time getting up and running, and I learned a little about web hosting along the way.

Inspiration

I was inspired by the domain name challenge for this hackathon. Nick said that he likes puns and I thought this name was so great that it needed a website to go along.

What it does

Push the button and another picture of a Samoa cookie appears on-screen.

How I built it

HTML and javascript. It's a very short program.

Challenges I ran into

Accomplishments that I'm proud of

What I learned

What's next for Samoa Samoas
