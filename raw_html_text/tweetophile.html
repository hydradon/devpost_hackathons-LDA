Inspiration

It all started on a cold, sleepy Saturday morning. I wanted to tweet buy my mind was blank, the cold darkness knowing there were 280 characters longing to be used.
That's when it hit me. Tweetophile.

What it does

Tweetophile uses the UK Crime statistics database as well as the Donald Trump quote API to gather relevant and up to date information, crafting it into the perfectly formed, but completely randomly generated sentence ended with only the best emoji.

How can we top this off?

How I built it

We use Native-React and Javascript to create a silky smooth iOS and Android app for our user's pleasure.
