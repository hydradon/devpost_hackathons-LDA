Inspiration

Working with HUD data gives us a great opportunity to look out for less fortunate American citizens, which is what we wanted to do.

What it does

The Federal Funding Mapper plots areas of significant homelessness populations. On clicking these areas the user can view information about how much federal funding was received and for what purposes by the areas.

How I built it

Built in Android Studio, but most of the database work was done outside the project in RStudio.

Challenges I ran into

Communicating between R and Java was a challenge and so reading the data from databases was very tedious.

Accomplishments that I'm proud of

Designing and implementing a UI in Android Studio that utilizes the Google Maps API.

What I learned

How to use the Google Maps API and quickly implement UIs and backend software

What's next for Federal Funding Mapper

Predictive analysis so we can forecast the best options for funding based on homelessness stats.
