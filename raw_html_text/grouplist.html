• Add dates and times to reminders by simply typing them

  • GroupList will detect dates and times the user types in

Inspiration

Frustration with the complexity and lack of functionality with current todo-list and reminder apps

What it does

An app that lets people organize reminders and lists for their groups in the simplest way possible.

How we built it

Swift 4 and Xcode 9 for the front-end Flask, Peewee, Python, Heroku, and MySql for the backend

Challenges we ran into

New changes and bugs in iOS 11 Managing the production database using services like AWS and Heroku

Accomplishments that we're proud of

Using NLP to provide a simpler UX than options currently available Building a scalable and secure backend with password hashing and token authentication

What we learned

Better techniques for communicating between the front-end and backend

What's next for GroupList

Add new features, improve design, and optimize functionality before we push it to the app store
