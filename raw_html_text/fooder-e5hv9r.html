Inspiration

We realized how many cases of major illnesses could be prevented by diet and what eating healthier, more diverse diets could do for society.

What it does

Fooder tracks personal grocery habits and nutritional information through user grocery purchases, highlights potential deficiencies, and suggests foods to fill the gaps to improve dietary health.

How I built it

We used the Django python web development framework, and used Bootstrap for the HTML, CSS, and Javascript code.

Challenges I ran into

This was the first time I used an open dataset for a program that I developed, and learning to integrate that into the Django web development framework during the hackathon was a challenge.

Accomplishments that I'm proud of

We are proud that we have developed a product that can save lives

What I learned

I learned that the social impact of a computer program is just as important as the technical capabilities

What's next for Fooder

In the future, the web application of Fooder will contain more features for diet management, and the product will also come in the form of a mobile application.
