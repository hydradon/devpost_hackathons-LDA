Inspiration

Mobile phone insurance is outrageously expensive, inconvenient, and difficult. Libravos aims to change this with its new mobile app!

What it does

The beginnings of a mobile phone app for interacting with Libravos phone insurance. Sign up, login, and ask questions to our chat bot.

How we built it

We built the mobile app in React Native and used Rasa to build a chat bot. The chat bot is hosted on AWS.

Challenges we ran into

General installation problems.

Getting Rasa bot communication going.

Setting up the AWS server and opening up the port for public communication.

Accomplishments that we're proud of

Getting the chat bot going. There were times where we weren't sure we would be able to pull it together.

Learning new technologies. Our team wasn't especially familiar with react-native at the beginning of the competition. With sleepless nights we were able to make progress we are proud of.

What we learned

React-native, AWS, and RASA

Team-building. Working with little sleep. Revised Libravos business model. Very enjoyable and enlightening discussion with the Intact team.

What's next for Libravos Mobile App With Chat Bot

Continue development and allow for users to subscribe to an insurance policy.
