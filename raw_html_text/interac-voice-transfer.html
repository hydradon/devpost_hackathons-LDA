Inspiration

Sending etransfers requires many steps, and sometimes your friends forget to send you an etransfer altogether. We streamline this process by allowing users to speak to their phones or Google smart home devices using natural language to send their friends requests for etransfers along with the amount of money owed.

From the recipients side, the etransfer request serves as a reminder for the recipient and allows them to immediately action the etransfer to the original sender.

How we built it

JavaScript, Node.js, Dialogflow, Firebase, Google App Engine.

Challenges we ran into

Learning how to set up the correct dialog flow, deploying our app on Firebase, Google App Engine and passing parameters to the NodeJS webhook.

Accomplishments that we're proud of

We can send a request for an etransfer using our voice.

What we learned

How to build and implement a Google Assistant action for any smart home or android mobile device.
