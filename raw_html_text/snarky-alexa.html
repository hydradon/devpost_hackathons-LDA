Inspiration

Alexa is usually so nice, we wanted something that would make things more fun, and completely useless!

What it does

It replaces Alexa's built in commands with snarky replies.

How we built it

Using the Alexa Skills toolkit, and programming it in Python 3, on amazon lambda.

Challenges we ran into

Really, mostly is that Alexa's skills are really not easy to debug, several hours were lost just because an amazon account issue.

Accomplishments that we're proud of

Alexa actually listen and replies to our commands, and sometimes is nice!
