What it does

We know that lit places exist all around us. Yet, how do we find them? Our application brings a unique solution to this problem by indicating the location of lit users. With the press of a button, the user can the alert others the lit status of the location.

How we built it

Frontend - React Native, Clojurescript, Airbnb React Native Maps API (Google Maps)

Backend - Clojure, Compojure, Korma, Java JDBC, MySQL, ring-clojure, Leiningen, ngrok
