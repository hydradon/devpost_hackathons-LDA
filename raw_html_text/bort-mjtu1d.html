Bulletin Board!

Bulletin Board (formerly Bort) is an anonymous localized messaging skill developed for Amazon Alexa, which takes advantage of a reddit-like voting system to ensure users are fed the most popular and recent messages first.

How to Run

If you do not have an Alexa-enabled device, visit the Alexa Skill Testing Tool after enabling the Bulletin Board skill on your Amazon account. State "Open Bulletin Board" to open the skill. You can work through an optional tutorial by following voice prompts, or you can ask Alexa to skip the tutorial and go directly to the main menu.

About Me

I'm a new developer who is hoping to break into the web development world. This project was created as a way to express my ability to quickly learn and work with a number of new technologies.

The Tech

This skill was written in NodeJS and takes advantage of Amazon's proprietary DynamoDB database system, as well as AWS's Lambda Function service.
