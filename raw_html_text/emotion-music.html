Inspiration

We wanted to create an app that would make it easier for people to find music that they would like at a certain time.

What it does

It reccomends music for you based on the emotion of a person in an image.

How we built it

Used Microsoft API to analyze picture then use it to determine emotion.

Challenges we ran into

Lack of hackathon experience.

Accomplishments that we're proud of

Building the website in 36 hours.

What we learned

We learned how to use StdLib.

What's next for Emotion Music

Add a feature that let's you take the picture yourself, instead of having to search up a url.
