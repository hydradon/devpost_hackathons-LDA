Inspiration

We thought that it would be a great project to learn how to work with the Unity gaming environment because it gives us experience with scripting and with UI interaction.

What it does

Our game presents a question to the user with an image and four randomly-ordered options to choose from. Multiple questions make up a round. The game tracks the user's number of correct answers out of the total number of questions answered, and displays their score and their time remaining in the round. When they click an answer to a question, they are presented with the next question. At the end of the game, the user is asked if they would like to play again.

How we built it

We used the Unity3D design engine.

Challenges we ran into

Merging multiple Unity files and debugging our project were two of our biggest challenges.

Accomplishments that we're proud of

We all worked together and tried our best!!

What we learned

We have begun learning how to use the Unity3D game engine, C#, and game design principles.

What's next for Quizzical

We plan to make it load quiz data from a remote file system or URL. We are going use this program as a quiz game to help professors learn the names and faces of their students.
