Inspiration

Heroku's wildly popular Deploy To Heroku button for Github is a fantastic shortcut that has significantly increased engagement by reducing the hurdles necessary for developers to deploy open source projects.

This new Deploy to WhenHub button provides the same shortcut for scholars and other timeline enthusiasts who wish to open source, version track, and collaborate on their work via Github.

What it does

Embed the button code in any markdown or html timeline (following this specific format ) you have in GitHub! When button is clicked by you or any visitor, a new WhenHub widget is dynamically created with your latest timeline, presenting your timeline in one of several different beautiful UI views!

Our backend server is responsible for this automatic creation and rendering in the background.
