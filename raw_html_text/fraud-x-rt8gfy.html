Inspiration

Our inspiration for creating Fraud-X was when we saw how many people are affected by credit card fraud each year. We believe that any program a consumer can access can help them to stop fraudulent behavior occurring on their accounts.

What it does

Fraud-X uses an algorithm to detect fraudulent activity within a bank account and allows the user to select whether or not they purchased the item or withdrew money.

How we built it

We built Fraud-X with Android Studio and started with the basic structure of the app and finished by working on the fraud detection algorithm.

Challenges we ran into

We had problems with using Firebase to handle logins and eventually switched to a different system.

Accomplishments that we're proud of

We are proud that we used the API of a major bank to find fraudulent patterns in customer bank data.

What we learned

We learned how to create an Android app with Android Studio, use an API, and discover the process of creating a program in each group member's first hackathon.

What's next for Fraud-X

Fraud-X can expand by using better and more sophisticated techniques to detect fraud in customer transactions.
