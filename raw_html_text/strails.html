Inspiration

Finding a good tour through a city is hard. People do not always know the notable places and landmarks in the city, and they don't know how much time it will take to get a good tour. sTrails simplifies and automates the process.

What it does

Finds tour routes through landmarks and notable locations easily using the HERE API.

How We built it

With Ruby on Rails, JavaScript, and the HERE API

Challenges We ran into

Here API and first hackathon

Accomplishments that I'm proud of

Finishing, and learning API calls and GET requests and JavaScript all in one day.

What I learned

How to interact with multiple APIs

What's next for sTrails

Hopefully production because this is an idea with a lot of potential. We want to deploy to a production version, and add more functionality such as searching for different types of places (e.g. landmarks, restaurants, etc).
