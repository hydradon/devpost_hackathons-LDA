Inspiration

I understood, that some progress in this sphere will help people to prevent lots of diseases and any bad situations

What it does

It counts calories using 3 factors - steps, sleep and temperature. This is the first app using the last factor. I created formula for this factor using some forums and medicine books

How I built it

I wrote it on Java using Here API to get weather depending on the region. It was tested on every type of screen using emulator

Challenges I ran into

These are Co.Factor Healthcare and my own track by russian hackers

Accomplishments that I'm proud of

What I learned

Docs about Here API to implement it to my app, some formulas from wiki, some new functions and usages of Android sensors

What's next for Solar Steps
