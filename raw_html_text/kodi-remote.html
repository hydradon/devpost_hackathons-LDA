Inspiration

I'd like to have a solution for browse my media library, so I've build this skill.

What it does

It allow to search in your kodi library a movie, like "tell me one dramma movie" or "tell me one movie with Brad Pitt", then you can open that movie.

How I built it

To built it I have used PHP7, Firebase, Heroku and Gitlab.

Challenges I ran into

Create the authentication mechanism, activate two different provider for connection.

Accomplishments that I'm proud of

Have a low prices and very fast architectural solution
