Inspiration

세계적으로 널리 사용되는 웹기반의 프로젝트 관리 소프트웨어인 trello를 실제로 만들어보며 강점에 대해서 분석해 보 고 싶었다.

What it does

trello의 핵심 기능인 드래그 앤 드랍 기능을 먼저 구현해 보았고 board 안에 list와 card 그리고 dialog의 CRUD 기능을 구현하였다.

How we built it

frontend는 Angular 프레임워크와 TypeScript 기반으로 프로젝트를 진행하였고 backend는 Python, Django(Rest Framework),Amazon AWS S3, RDS, EC2를 사용하였다. frontend와 backend 간 많은 회의를 통해 프로젝틀 진행하였고 api 문서를 통해 연결하였다.

Accomplishments that we're proud of

드래그 앤 드랍 기능 과 모든 CRUD를 서버와 통신하여 진행한 점

What we learned

frontend와 backend 간의 의사소통(협의)이 무엇보다 중요하다는 것을 배웠다.

What's next for Clone_trello

아직 구현하지 않은 회원가입 및 로그인 기능 그리고 초대 기능을 구현해 볼 계획이다.
