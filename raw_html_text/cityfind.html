Inspiration

Every time we go to a hackathon wee like to spend some time visiting the city. Nevertheless, a city is more than the tourist places recommended, and we wanted a fun way to discover the real city.

What it does

CityFind is a funny way to discover the city. Just enter into the platform, register a room and share the room number with your friends. Once you are ready, the creator of the room will start the game. Each of the participants will get an SMS to their mobile phone with a timer and a thing. They will have this time to take a photo with this thing. The photo will be validated using MS cognitive services. If the photo is validated, the participant will earn point. When ending the game, the participant with more points wins.

How we built it

We built the backend using python and flask. The interactive chat is created using firebase and the messages are sent through Twilio. We used Microsoft's cognitive services API to validate the photos and HTML, CSS and JS for the frontend.
