Inspiration

As teenagers ourselves, we know the challenges of learning personal finance and the struggles that await us in managing money in college. We created this app to make managing money simple and easy.

What it does

This web application takes all of your recent transactions and provides suggestions based on your most common or unnecessary purchases. It also reveals some techniques to better manage your money. The site also shows how much of your total balance that you have spent compared to your target budget.

How we built it

We used Go for the back-end and React.js for front-end. We used React components to organize stylistic elements on the webpage.

Challenges we ran into

We had challenges styling components using React.js. We also had to develop an advanced algorithm using logic with Go.

Accomplishments that we're proud of

We are proud that we were able to make this web application simple, aesthetic, and efficient while providing all of the most important information needed to manage a budget effectively.

What we learned

We explored the uses for Go for filtering JSON files and more. We also learned how to use React.js in web applications to promote a simple layout by using components in more efficient ways.

What's next for Kala

We will enhance the features of our webpage and provide more options for managing money effectively in the future.
