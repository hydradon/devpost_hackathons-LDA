Inspiration

Gaming to exercise, it's interesting right? I was very interested in this category in code Healthy with OpenShift hackathons because I regularly walking, so I've decided to do something to participate in this hackathons and challenge yourself!

What it does

This is an application that counts your steps and store the data for you to track your exercise process, you are awarded 3 coins, you'll buy the eggs, if you walking or running the eggs will hatch into chicks, then you sell chicks to hoard more coins, you will find interesting because you will compete achievements with everyone in the world!

How I built it

I designed and programmed a client application with features such as I have described, I created an application server on OpenShift online system to process the request and store the data of the user.
