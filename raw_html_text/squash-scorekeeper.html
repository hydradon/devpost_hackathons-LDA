Inspiration

I like to play squash but it's hard to remember the score and I wanted a hands free and simple solution!

What it does

Keeps a record of the score for you!

How I built it

Made the design in Html with Css stylizations backed by js classes and functions.

Challenges I ran into

The google speech to text api does not support javascript... and I didn't have enough time to make it work well :p

Accomplishments that I'm proud of

It works! And the clock works reallllly well (Kinda)

What I learned

Make sure that your api will work with your application before you start...

What's next for Squash Scorekeeper

I'm probably going to rip it down a bit, reorganize and replan and then build it up with a different speech to text api that has Javascript support!
