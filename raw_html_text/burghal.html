Inspiration

We were in D.C. one day and we couldn't find any clear directions or help to direct us to public facilities. This motivated us to create a project that helps you find your nearest facility, be it a bathroom, trash can, bike rack, or park.

What it does

You can search and add locations by rating and distance using a proprietary algorithm, which optimizes both.

How I built it

Using multiple API's and using React Native to develop a basic interface using help from google maps.

Challenges I ran into

Ran into multiple unidentifiable errors which led to major loss of time and in the end, the scrapping of major parts of the project

Accomplishments that I'm proud of

Coding the Website for our company and creating a user interface on an IOS device with multiple interactions and API's being used. Also, we custom made our logo.

What I learned

To come better prepared to a hackathon, with more knowledge on react native and background on IOS coding

What's next for Pinpoint?

We hope to still continue on developing the application as it was a pretty neat idea. with some polishing up and more time, this could definitely be a widely used app.
