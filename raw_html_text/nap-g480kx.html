Inspiration

Sometimes you want to request money from people who don't have cash on them, and maybe they are just too drunk to remember what their email is.

What it does

We developed a web app that requests money via Interac e-transfer from someone using their face. It also registers people into our system to opt-in for the service.

How we built it

  • using face_recognition API
  • using Interac e-transfer API

Challenges we ran into

  • using Interac API
  • using Azure Cognitive Services API
  • using PHP
  • creating our database
  • hosting on server

Accomplishments that we're proud of

  • using HTML to evoke live camera
  • using Google Cloud
  • integrating APIs

What we learned

  • using APIs
  • full-stack development

What's next for $NAP

  • improving UI/UX
  • hosting on server
  • adding mobile support
