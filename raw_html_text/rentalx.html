Inspiration

All of our team members love travelling, and we had a heated discussion about the annoying things that we ran into. One of the things is that when we rent cars, we never get the ones we like. Also, when we went to other states, we don't know how the parking system works so it always becomes a problem. Lastly, it always seems so dangerous to drive a long distance. We talked about how we could improve our experience in the future, and we came upon the APIs that Dubhacks 2017 offers.

What it does

We built a bot that suggest the best cars for rental places to buy, and we suggest the most suitable car for the people who rent from the place. We also added an extra feature to suggest parking space in unfamiliar places.

How we built it

We used the car database from Shine API to train and suggest the best car for rental places to buy. We also take Graph API from facebook and use APIs from Microsoft to determine the customers' references to give them the best recommendations. Lastly, for extra feature, we take parking data from Shine API to suggest parking space.

Challenges we ran into

We had problems in machine learning, synchronousness in node.js.

Accomplishments that we're proud of

The chat bot was returning data that we wanted it to.

What we learned

We learned how to use API in node.js and how to combine different APIs to make a good project.

What's next for RentalX

We're going to keep improving it until it's flawless.
