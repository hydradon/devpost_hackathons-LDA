Inspiration

Iclicker, which is used to takes quizs in the class, is expensive and inconvenient to bring to the class.

What it does

It achieves all the functions that an iclicker does via SMS and a web app, and also analyze data and store it into a Mongodb database.

How we built it

We use api from Twilio and Mongodb. We build two servers in python and javascript and develop a web using react.

Challenges we ran into

Combine two different languages, python and javascript, and make them work together in the front end.

Accomplishments that we're proud of

We solve all the challenges and bring an effective alternate of iclicker.

What we learned

How to work as a team. How to build things in limited time.

What's next for Mclicker

Keep improve the web design. Increase the database functions and size. Upload it on cloud to let more people use it.
