Inspiration

Wanted to ease out invoice generation process for merchants and let customers get it immediately and can pay the amount immediately.

What it does 

 It's a bot that asks the merchant to give customer's basic details to generate invoice. It sends a unique invoice-payment link to the customer via sms and email instantly and customer can pay it securely by visiting that link. Thus transaction finishes faster and securely.

How I built it

Android app + Microsoft Bot Framework + Instamojo api

Challenges I ran into

Many errors in connecting the APIs.

Accomplishments that I'm proud of

Project flow is awesome!

What I learned 

Microsoft Bot Framework

What's next for PayBot

Bug fixes, better functions and code improvement.
