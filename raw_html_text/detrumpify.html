Inspiration

I got inspired by the controversial tweets from Donald Trump and I wondered to myself, "I should clean out my social media from anything explicit" so I came up with the idea of making the cleaning process easier!

What it does

It analyzes your tweets and returns your most explicit tweets, making it easier to access them and possibly deleting them

How I built it

I used Twitter's API to retrieve the tweets of the user, and used Google Cloud NLP to do some sentiment analysis. All the tweets processing work was done in a NodeJS backend

Challenges I ran into

Setting up the Google App Engine to process the sentiment analysis for the tweets

Accomplishments that I'm proud of

Learning how sentiment analysis works and how to use Google Cloud

What I learned

More angular :) and how to utilize APIs

What's next for deTRUMPify

Add delete feature on app, and also instagram support using Google Vision API
