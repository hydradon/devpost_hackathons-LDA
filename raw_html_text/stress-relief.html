Inspiration

We wanted an app to help us de-stress

What it does

It can show a cat GIF, with a button to get new random GIF. It also has a chat feature, which assigns a random name to each new user. All users can see each chat.

How we built it

We used React-Native to build an app which can run on Android and iOS.

Challenges we ran into

We had a relatively slow start, as we were working with an unfamiliar language. In particular, getting the environment running was a challenge.

In addition, the time constraint kept us from fully fleshing out the design we originally envisioned.

Accomplishments that we're proud of

This is the first React-Native project for all of our members, and we are proud of learning a new language.

What we learned

In terms of design, we learned to design an app around the user.

For programming, we got hands-on experience programming with React-Native. Its components include CSS and Javascript.

What's next for Stress Relief

The next big goal is to integrate the existing functions with timers, alarms, and events. We think it would be a significant improvement if users could target times when they need to use the app.
