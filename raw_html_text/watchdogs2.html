Inspiration

Leader had the idea about smart-devices before the event so we tried to learn and build it here.

What it does

Right now it validates user input and sets the brightness of a light with user credentials

How I built it

PHP/Apache server -> Raspberry Pi script

Challenges I ran into

(1) Sanitizing user input to disallow shell execution (2) Setting up android studio to send proper http post requests to apache (3) Setting up an SSL certificate

Accomplishments that I'm proud of

We made something, and we got the brightness of the light to change over wifi.

What I learned

How to sanitize input with PHP, more competence with new languages.

What's next for WatchDogs2

Get a proper SSL certificate running on the apache server to interface with the android app without trouble.
