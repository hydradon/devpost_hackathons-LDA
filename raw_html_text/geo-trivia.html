Inspiration

We are trivia nerds and knew that the ArcGIS Platform would be a great environment for geographic gameplay.

What it does

Play geographic trivia on desktop, tablet or Xbox! Buffers display users' proximity to correct answers - the closer you are, the more points you earn! Three different basemaps determine the level of difficulty, meaning that even the most seasoned geo geeks can challenge themselves.

How we built it

We built the app for Universal Windows Platform using the ArcGIS Runtime SDK for .NET. The app has one code base and can be played on any device running Windows 10+ - even the Xbox! We used the Living Atlas and Open Street Map to source question data, authored and refined data in ArcGIS Pro, and published a hosted feature service in ArcGIS Online - making data easy to edit on the fly.

Challenges we ran into

The polygon data we needed for question content was complex! Enter ArcGIS Pro to simplify polygons and concatenate features into a single service. Building the app for multiple devices meant creating a responsive design and accounting for different methods of user input - mouse, touch, and Xbox controller.

Accomplishments that we're proud of

Getting the app working on Xbox! Buffering the answer polygons and drawing graphics in response to user input Incorporating Living Atlas data into questions Using polygon geometries for answers instead of just points

What we learned

How to use the .NET SDK on the Xbox How to use Pro to go further with your layer data

What's next for Geo Trivia

New platforms - web, iOS, Android? Multiple geometries - using points, lines and polygons in the same game Toggling between trivia categories - creating different layers for question data
