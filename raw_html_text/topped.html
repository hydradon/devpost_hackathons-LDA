Inspiration

Topped was inspired by and built for the University of Alberta's Wall Brawl .

What it does

Our goal was to build a web application that improves upon the pen-and-paper scoring system that the Wall Brawl currently uses.

How we built it

This web application was built using a React front-end and a Firebase backend.
