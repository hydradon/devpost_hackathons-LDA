Inspiration

That inspiration that started this project was to enhance the current recycling methods already implemented by using technology to educate, people of all ages, to make a greener future.

What it does

An educational app to classify waste disposal material using Google Vision API. It locates buyback programs that pay users for recycling and promotes community profiles systems in place.

How we built it

We built this app using the Proto.io platform and integrated many of the features listed above into the app. For instance, we included various pages such as a home page, a search page, a photo page, and more.

Challenges we ran into

One challenge we went into is at first we wanted to expand this globally, but realized that this was too difficult to accomplish. So instead, we decided to focus on a more specific area like Nashville, TN so that we can use better search terms to find more information.

Accomplishments that we're proud of

An accomplishment we came up with was designing an app using a prototyping platform to illustrate our concept of classifying waste disposal.

What we learned

We learned about teamwork, communication, improving our skills using Proto.io, and learned more about the design process through this project.

What's next for Have you bin recycling?

Integrating code into the program to allow for picture recognition of different products would be the next step for this project.
