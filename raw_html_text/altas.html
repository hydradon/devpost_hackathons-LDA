Inspiration

El reto fue impuesto por la misma empresa pero la manera de resolver las cosas fue de manera creativa

What it does

Pone a competir 4 modelos de regresión ya que analiza una base de datos de 10 millones de filas

How we built it

Utilizando la plataforma de watson studio y la base de datos que se nos dio, se generaron modelos predictivos para las ventas esperadas de cada dia durante cierto tiempo

Challenges we ran into

El problema mas fuerte que encontramos fue el tamaño de datos que el excel soportaba ya que solo soportaba 1/10 de lo que la base de datos era, al igual la manera de conectar el resultado de los modelos hacia la hoja de excel actualizada por estos mismos datos

Accomplishments that we're proud of

Hacer todo lo que hace un científico de datos ya que no habíamos tenido tal acercamiento a este tipo de datos

What we learned

a trabajar como científicos de datos y poder interpretar modelos matemáticos predictivos
