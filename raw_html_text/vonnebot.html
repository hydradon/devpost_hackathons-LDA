Inspiration

Kurt Vonnegut, author of slaughter house five argued that stories have shapes. To support this argument he graphed stories with the x axis representing time and the y axis representing good fortune (0, inf) and bad fortune (0, -inf).

What it does

This attempts to visualize stories the same way as Vonnegut did using sentiment analysis.

How we built it

Google cloud platform for sentiment analysis. python javascript and elbow grease for everything else.

Challenges we ran into

gcloud request limits

Accomplishments that we're proud of
