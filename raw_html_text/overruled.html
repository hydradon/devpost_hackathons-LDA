Inspiration

Ever played a game and thought, "Why can't I do this ? Why is this against the rules?"

What it does

Overruled pits you against other players in a frenzy to reach the goal. It starts off painfully simple, but gets complicated really fast. Changes to the environment, attributes of your character, etc. tack on after every round. To be the winner at the end of 10 rounds takes skill, grit, and of course, RNG!

How I built it

The scenes and objects were created with Unity, and the behaviours were programmed in C# and JavaScript.

Challenges I ran into

Unity had a steep learning curve and very specific settings to make the game look the way we imagined. Multiplayer on a network caused noticeable lag and many bugs.

Accomplishments that I'm proud of

Building a working application using software we have never used, implementing multiplayer over a network

What I learned

Game dev is time consuming, but rewarding!

What's next for Overruled

Upgrades and/or classes with skill trees for characters Support for more concurrent players MORE WACKY RULE CHANGES :)
