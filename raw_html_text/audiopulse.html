Inspiration

Thought that the perfect playlist was needed for the perfect workout

What it does

Takes songs from local storage, organizes them by genre, and creates random lists of them according to user input

How I built it

Took the songs from storage and assigned indexes to them that are used to make playlists

Challenges I ran into

Finding and Implementing an sdk that did audio processing, Working with sqlite databases

Accomplishments that I'm proud of

Built a UI, Got the database set up

What I learned

sqlite, setting up background threads

What's next for AudioPulse

Truly analyzing each song for things like BPM and frequency to make more accurate playlists Could add gps to track each run
