Inspiration

This idea came about when we were really bored in the car. And wanted a procrastination (PANIC) button.

What it does

The button has 3 different modes and a F mode. We have a useful, normal and asshole mode. This is really toned down because we originally wanted to shutdown and BSOD the computer :P Now the asshole mode would send your computer to sleep and do stupid shit like killing your chrome process instead!

Visit our website here to find out more :)

How we built it

We connected a BIG RED BUTTON to an arduino and to the laptop. We wrote the arduino code to communicate with the laptop over the COM port and a power shell listening on the COM port to run a script which does random things on the different modes.

Challenges we ran into

Setting up the wiring for the arduino, power shell, communicating over COM ports, and (cos we are really old...) being super sleep deprived.

Accomplishments that we're proud of

First time making a hardware hack.

What we learned

Hardware is pretty hard :(
