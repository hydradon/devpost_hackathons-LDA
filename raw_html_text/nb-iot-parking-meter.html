Inspiration

I don't like those big old-styled parking meters. I want to give our cops more rest by not requiring them patrolling for parking violations.

What it does

It uses ultrasonic sensors to detect cars. If the parking spot is used but not paid, a notification will be sent.

How I built it

Ultrasonic sensor: to detect the car. Twilio dev board: to send command using NB-IOT. Nodejs: handle the incoming command. Android: Get notifications for status and violation

Challenges I ran into

Challenging to make it work within 24 hours!
