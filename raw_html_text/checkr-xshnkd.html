Inspiration

Its always really difficult to find empty and quiet rooms around the campus, most of them are either really packed and or really noisy. So, whenever lecture rooms are not being used, why not use those rooms for yourself?

What it does

Provides real time updates, uses Location services to map the closest rooms to you. Maximizes UofC infrastructure.

How I built it

Using Xcode, We got the data from PeopleSoft through dndn.ethv.net, they had done the hard work of scraping people soft, and we used that data for our app.

Challenges I ran into

Parsing XML was difficult, not all classes are in correct format, some rooms have weird names and so on...

Accomplishments that I'm proud of

An actual working app that we can show off! Also alot of students we have asked have given us very positive feedback.

What I learned

Teamwork, code under stress, using XML and JSON.

What's next for checkR

Implementing MapKit, make scrapping data more efficiently. The framework is in place, all it needs is data and it can update every semester.
