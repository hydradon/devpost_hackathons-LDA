Inspiration

It was the first Hackaton for the whole team. The motivation of a new experience and improve our skill set with collaborative projects were the main inspirations.

What we learn

  • We learn the importance to define a nice communication standard.
  • How to make a webSocket client with html5 and a webSocket server with Python.
  • A collaborative workflow using git.

How we make it

  • Forming 2 teams and dividing the work into front-end and back-end.
  • Searching for Open source libraries and frameworks
  • hard work and motivation
