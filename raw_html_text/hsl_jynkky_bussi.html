HSL Bus Display

Inspiration

When using a new route, it's sometimes difficult to know exactly when to hop off, but with a map display you could more easily be aware of the surroundings.

What it does

Gets data from the bus sensors to locate it. Needs busId and route as parameters and the server does the rest.

How we built it

We've built it using javascript, leaflet and node.

Challenges we ran into

Busses didn't work. No route lines.
