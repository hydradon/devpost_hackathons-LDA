What it does

TAFi is a digital health information technology (IT) company that builds an integration software platform designed for tomorrow’s healthcare industry.

The TAFi platform seamlessly blends data from multiple sources and connects it to the EMR, without inflated custom services and uses our automated technology to create a governed layer where vendors can run analytics or other clinical decision support tools, powered by our patient-centric data model that provides a true longitudinal view.

How I built it

We built it using a derivative format that would be flexible with all data standards today.
