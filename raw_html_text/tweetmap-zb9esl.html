TweetMap :D

  • Our group really wanted to create a real time application to display real time twitter activity. We wanted to use this heat map to discover different trends and current events around the world. The application uses the twitter api to draw tweets and then use the location to draw a heat map using the google maps api. We ran into multiple challenges such as node.js server connection, azure deployment services, bing api location data, brainstorming. We asked mentors and fellow peers for help on problems that we ran into it was one of many learning experiences at DCHacks.We are proud of the end product because we put time and effort in the project, we did what we said were going to do.We learned a lot from mentors and workshops provided. The next step we are trying to take is to include data analyzation.
