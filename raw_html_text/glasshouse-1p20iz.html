Inspiration

Lewis was having problems monitoring the conditions in his greenhouse.

What it does

GlassHouse allows you to use a RESTful API to monitor conditions.

How we built it

We used node.js with express to build the web app, with the Pebble app written in Pebble.js and the desktop app written in Electron.

Challenges we ran into

MySQL, at 5am...

Accomplishments that we're proud of

A successful all-nighter, with the MySQL problem getting fixed when we were practically delirious with sleep depravation.

What we learned

Debugging databases isn't always common sense.

What's next for glasshouse

A more in-depth query structure for the API, and mobile apps.
