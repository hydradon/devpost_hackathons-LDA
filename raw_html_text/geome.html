Inspiration

People are lazy, thats cool, the world can be a dangerous place, thats life. I want to be secure when I'm online lets make that better and easier.

What it does

It uses your location(based on device of you choosing, probably your phone.) to verify you are who you say you are when your online.

How I built it

using a mobile app (at this point) to send location information to a server. the server verifies this information every time you attempt to login.

Challenges I ran into

communication with the server

Accomplishments that I'm proud of

getting everything to talk to one another.

What I learned

I came into this with only Java skills and basic HTML, so pretty much everything else.

What's next for GeoMe

There are several other ways this type of communication between end points can be used to authenticate.
