Inspiration

We as students, want to help other students by making an app to organize their study sessions.

What it does

It helps you divide your study time into time sessions.

How we built it

We incorporated a web app with the pebble smart watch. We used Meteor.js, MongoDB and Pebble.js.

Challenges we ran into

First, we started building the pebble app in C language, but we encountered a lot of problems importing the data that the pebble received from the web app. Due to this, we decided to use Pebble.js.

Accomplishments that we're proud of

Being able to merge our web app idea with the pebble watch.

What we learned

We learned more about JavaScript, how to make apps for the pebble and add new functionalities using Meteor.js.
