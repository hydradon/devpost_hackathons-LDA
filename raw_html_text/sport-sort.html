Inspiration

Our main inspiration was the difficulty in organising sport games that require a large amount of people to play that would normally only be possible for large, professional sport clubs. Sport Sort aims to provide the same matchmaking service for casual sport players

What it does

Matches up people with similar sport interests to organise a quick sport game.

How we built it

HTML, CSS, Python, JavaScript

Challenges we ran into

Inexperience in linking back-end to front-end

Accomplishments that we're proud of

Despite being extremely inexperienced, we were still able to persist in completing the project as well as we could

What we learned

A better understanding of databases and how a server communicates with front and back-end code

What's next for Sport Sort
