Problem

According to CBS News, there have been more mass shootings than days this year. The increase in risk in public areas demands smart security solutions to reduce the risk of danger.

Solution

IoT video footage devices can be connected to Rapid Security, a server on Google Cloud Platform's App Engine. The app then uses Google's Video Intelligence API to detect firearms in real time. If gun violence is detected, Rapid Security automatically notifies enforcement.

Challenges we ran into

Running React and Flask on the app engine simultaneously was difficult. The temporary solution is to run the front-end, React, locally to demo the functionalities and run the back-end, Flask on the app engine. The Google Video Intelligence API also significantly drops in object detection accuracy when bounding box is small and the video quality is blurry.
