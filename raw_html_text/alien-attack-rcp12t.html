As we were picking projects, we decided to bring a little bit of the past back to bitcamp. We were inspired by the Atari 8-bit games such as, Space Invaders and Asteroids, to make our own 8-bit alien space invaders game known as, Alien Attack! We designed an enjoyable game using Java applets and GUIs, in which the user controlled a space ship that would shoot down enemy aliens that were approaching it. As the user gained more points, the game would get progressively harder, with more types of aliens spawning at faster speeds and more lives; making them harder to kill. Reach 1,000,000 points and you win! This application was built on the presence of nostalgia and helped us to learn a lot more about creating games in java using java applets and GUIs.
