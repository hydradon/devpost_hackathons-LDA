Inspiration

Automate a lot of work that organizations currently do to help connect volunteers and organizations.

What it does

Helps people sign up and track volunteers.

How I built it

The front end is built using React. It uses React Router, Bootstrap, Sass, Recharts, React CSV, and React Big Calendar. The data is currently stored in PostgreSQL database. For the submission, the data is accessed locally using postgREST API but could be hosted on Heroku in the future.

What's next for Volunteering Portal

Figuring out how to best integrate our solution to solve organizational problems. The portal could be made more generic to allow multiple organizations to use it.
