Inspiration

We all used to play Swords and Sandals when we were playing browser games. We remembered that yesterday morning and thought it would be a great idea to see a similar battle style but instead of gladiators, the fighters would be websites!

What it does

The player chooses two websites and then puts them to the death. The website with better stats (provided by the Majestic Api) is going to win!

How we built it

We built the game in Unity3d and we used the statistics provided by the Majestic API as stats for the warrior websites in the game.

Challenges we ran into

Unity UI is very buggy sometimes so we had to redo everything related to that over 4 times. Compiling didn't work out every single time as a bug from the latest version of unity kept appearing.

Accomplishments that we're proud of

The game works as intended. We have really good music and sound effects. The majestic api integration is flawless for what we needed.

What we learned

How to use the Majestic api. How to hardcore Unity Ui elements. Getting the favicon from the internet and applying them to the head of a character.

What's next for The Web Wars

Release for the Android. Remaster the Graphics!!!
