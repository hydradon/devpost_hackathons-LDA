Inspiration

Want to make using email safer and avoid phishermen. Enable the ability to figure out which websites you have accounts with to make sure subscription emails are legitimate.

What it does

User will authenticate with google, the program will scrape their inbox, and then sort out the security of each domain.

How we built it

It took a lot of sweat, blood, and tears. We used a lot of different tools for each function we wished to implement and combined them all.

Challenges we ran into

So many, where to begin. Didn't know the languages nor apis, didn't know json nor pycurl, didn't know how to link everything. Turns out it's hard to decided if a website is safe or not.

Accomplishments that we're proud of

Making the website work and successfully authentication and scraping with Google.

What we learned

We learned everything from the challenges section and much more!

What's next for Secure Web Project

We will add the ability to insert a link (instead of just an email account) and increase how many traits we use to determine security.
