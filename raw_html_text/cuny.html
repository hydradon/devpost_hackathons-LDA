Inspiration

As a group of students from different CUNY colleges, we found that when we have some problems on campus, we don't know where to go to report them. We had a bad experience with dealing with even small problems such as lack of toilet papers or missing paper from the printers for several days. If you ask one person, he will send you to another one, and next person will send you somewhere else to solve this problem. This is painfully exhausting and time consuming process.

What it does

CUNY++ is an app that allows you to send any problem to the right department of your college to get it solved.

How we built it

For user's (Student) mobile app, we used React Native, Firebase to get it built. For department's (Admins) web app, we used JS, Jquery, Firebase, HTML, CSS and Bootstrap to get it built.

Challenges we ran into

Lack of time to make both mobile and web apps. Firebase set up.

Accomplishments that we are proud of

We are proud of working prototypes that we built.

What we learned

How to use React Native, Firebase and JS.

What's next for CUNY++

User account system.
