Inspiration

We were inspired by the challenges facing fish populations around the globe from over-fishing to climate change.

What it does

Allows fishermen and other people aboard fishing vessels to record their fishing trips and catch in order to increase the aggregation of fish data collected around the world.

How we built it

We first determined what kinds of data the fish-folk would need to collect by talking to a few experts in the field. We then developed an easy to use app that was as simple as possible to simplify the data-collecting process.

Challenges we ran into

Trying to determine which fields were necessary to be collected, and setting up a uniform data standard so that the data would be easily transferred between states, countries, and continents.

Accomplishments that we're proud of

Building an app that may help with the awful problem of over-fishing and creating a data standard that could help integrate fishing data worldwide.
