Inspiration

Learning programming in a classroom isn't as interactive as it should or can be. We go to class and listen to a lecture rather than actually interacting with the code itself. And when we do interact with the code, there is a limited amount of collaboration. The inspiration for this project stemmed from our desire to learn and our need for fun.

How we built it

We used Android Studio to build this app.

Challenges we ran into

We ran into a lot of challenge when it came to the backend. Without any knowledge in databases, we struggled building and using our database as well as handling all of the information that the user enters in.

Accomplishments that we're proud of

Our user interface was pretty much done.

What we learned

We learned a little bit about Android Studio.

What's next for codify

We will be continuing this project with the goal of overcoming all of our challenges and then some.
