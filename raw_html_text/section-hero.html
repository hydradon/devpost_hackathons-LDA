Inspiration

We were inspired by Mofi from IBM to build something that could potentially make the Section 8 application process a little better for those who need it.

What it does

Models what types of applications will get accepted for a Section 8 application

How we built it

Data analysis with Python and model with Watson Studio and Python to call the API again

Challenges we ran into

API authorization token errors and making a website. Also using the Watson Studio for the first time.

Accomplishments that we're proud of

Hopefully being able to apply the model to the real world

What we learned

There are a lot of messed up laws in America

What's next for Section Hero

Talking to Section 8 NGO's for data and building a user friendly interface
