What it does

This WeChat Smart Reply helps them to reply to their friends and families automatically, so they can focus on their works. First, we can set different modes, such as replying to friends or only replying to file transfer helper. Then, a QR code will be generated from WeChat Online Server when we run the code. After the QR code is scanned, a user can login and auto reply will start. More features can be found when people reply with specific words.
