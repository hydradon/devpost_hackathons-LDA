Inspiration

The virtual reality is the future of real estate and tourism.Its really painful when you have to visit an apartment you want to rent and need to travel or you choose via pictures but when you get there you are deceive. Virtualexpo aim to eliminate that.

What it does

The project help people experience the place they wish to visit, , an apartment you want to rent etc... via a virtual reality tour. By providing this ability they will be more confident about the place, the apartment etc...

How we built it

The project is build using the Facebook open source framework for virtual reality called React360 and other libraries such as React, flick api for getting 360 degree images, html5. css, javascript etc...

Challenges we ran into

The challenge we ran into it was getting 360 degree images as we don't yet have the equipment for that and taking react360 because it was completely new to us.

Accomplishments that we're proud of

We are really proud of what we did even though a lot is still to be done. We were able to build the first prototype and layout the foundation for a bigger project and expect that people will love it.

What we learned

We learned a new and amazing virtual reality framework and was happy and excited to work with it.

What's next for virtualExpo

We intend to push the project, getting 360 degrees camera and start a real estate startup.
