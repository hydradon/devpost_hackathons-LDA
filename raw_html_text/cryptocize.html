Cryptocize

Inspired by reward based motivational apps like Carrot Rewards, Cryptocize is a punishment based* motivational app that utilizes the Coinbase API to deduct bitcoins from users if they don't meet walking targets.

Technologies Used

  • Coinbase API
  • Android Studio
