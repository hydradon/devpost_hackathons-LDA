Inspiration

Seeing news stories about the endless possibilities of Alexa led me to build my own about space to spread my passion for space to others.

What it does

Provides space facts like the name implies to kids who may become scientists someday. This will spark their imaginations about the potential of space.

How I built it

Using python 3.6 for Amazon Lambda and the beta interaction model, I created the skill of Space Facts 50.

Challenges I ran into

The documentation for using python 3.6 on lambda is very hard to find.

Accomplishments that I'm proud of

I created an Alexa skill about space to spread my passion.

What I learned

The basics for building an Alexa skill.

What's next for Space Facts 50

More facts to increase its database as well as more intents.
