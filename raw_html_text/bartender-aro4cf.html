Inspiration

We could not find an app for making mixed drinks with Alexa.

What it does

You say "Bartender" "How do you make a Blue Margarita"

How I built it

Using python, lambda functions, and an API call to the mixed drinks database.

Challenges I ran into

Working with Alexa, setting that up, and importing requests from the lambda environment

Accomplishments that I'm proud of

It works!

What I learned

How to work with Alexa and create a new skill. Also we strengthened our python.

What's next for Bartender

More drinks! Also suggestions for other drinks.
