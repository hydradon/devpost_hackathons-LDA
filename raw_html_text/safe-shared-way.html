Inspiration

Our project "SAFE & SHARED WAY" is a security application. Parental concern to teenagers

What it does

This is a security system that allows the family knows where the user was every so often (for configuration, each 5, 10, 15, etc. minutes.) To have a starting point in case of loss

How I built it

ANDROID, JAV, JAVASCRIPT, HTML

Challenges I ran into

User privacy Battery drain That the application is "hidden" Real-time updating

Accomplishments that I'm proud of

It is a community service complements the amber alert the application is really friendly

What I learned

security vs privacy child rights vs concern of parents

What's next for SAFE & SHARED WAY
