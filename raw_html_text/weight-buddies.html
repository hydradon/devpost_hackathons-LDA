Inspiration

This project was inspired by https://www.dietbet.com/ . Instead of pitting people against each other to lose weight, we wanted to develop a system that would use positive reinforcement.

What it does

  • Partners are rewarded only if both reach their weight goals.
  • If either person reaches a weight checkpoint, it multiplies the total coins they get if they will get in the end.

How we built it

Front End: HTML5, CSS3, Sass, Bootstrap, Javascript, React, Semantics UI, Reactstrap Back End: Python 3.7, Flask, MongoDB

What's next for Weight Buddies
