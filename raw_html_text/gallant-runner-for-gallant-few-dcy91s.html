Inspiration

Any organization that raises funds to supports veterans is very important. Being able to built a simpler way to facilitate the donations and team accounts seemed like a cause worthy of our time.

What it does

It gives participants the ability to track their miles as an individual and as a team.

How we built it

We built the website using a Bootstrap template with Chart.js for the front end and a Tomcat server with a Mongo Database for the back end.

Challenges we ran into

Our main challenge was passing data from Facebook's Graph API to the server and then bring it back locally.

Accomplishments that we're proud of

We have designed a clean web interface that is extremely responsive and have a robust server/database to handle data.

What we learned

It's tough to ship a product in 36 hours.

What's next for Gallant Runner for Gallant Few

Adding additional features that had to be omitted because of time restrictions.
