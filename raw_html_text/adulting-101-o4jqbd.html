Inspiration

Teach for America keynote speaker inspired us to reach out to first generation students and help bridge the gap between transitioning from high school to college.

What it does

Interactive Webpage that connects first generation college students to resources and tools. Tries to ensure that every person in America has equal opportunities to get education.

How we built it

Used HTML, CSS, and Bootstrap for the website. Used github, js.fiddle.net, and liveweave to share code and collaborate.

Challenges we ran into

Navigating through Github and using other live code editors, had issues with using bootstrap template

Accomplishments that we're proud of

We're proud of our final product.

What we learned

  • we learn how to explore using IDEs because JSFiddle failed us.
  • we spent hours researching the issues that first generation students face in high school, college, and beyond.

What's next for Adulting 101

  • better design and functionality
  • include APIs such as Capital One's Nessie and Microsoft's Cognitive API.
