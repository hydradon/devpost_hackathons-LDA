hackTheNorthRNNlyricGenerator

Using recurrent neural network to create new lyrics of any artists.

Idea

The idea came from an article I read on Hackathon Hackers generating lyrics based on Taylor Swift's own lyrics using neural networks. I was fascinated by it and the potential of neural network in song generation.

What I did

What I did was try to replicate what I saw from that article and expand it to more artists. I built a web scraper that can either export its data to a .txt file or to CockroachDB database containing the lyrics. After I used Tensorflow to run a recurrent neural network modified a bit to suit my needs. The neural network learn from the lyric and generate a new lyric based on what it learned.

Artists used so far

  • Taylor Swift
  • Beatles
