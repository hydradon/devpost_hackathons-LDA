Inspiration

One team member (Arjun) had experience building android game before, so we decided to go into the route of building a simple android game for this hackathon.

What it does

It detects finger motion from players and have the small squares exploding from the place we tapped.

How we built it

We used android studio along with some other existing libraries like MotionEvents and StartActivity.

Challenges we ran into

Many of us are really new to java, so we spent a lot of time googling stuff instead of actually building things.

Accomplishments that we're proud of

At least we learned new stuff!

What we learned

The necessary building blocks (in terms of android) for this game.
