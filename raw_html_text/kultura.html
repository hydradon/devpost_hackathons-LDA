Inspiration

The lack of communication between people of different cultural backgrounds and beliefs.

What it does

Connects people with different cultural student organizations so that they can be enculturated

How we built it

We built it on Xcode using Swift and Storyboards. Our login and signup uses Firebase/Firestore.

Challenges we ran into

We were all pretty new to iOS development and Swift, so there was a steep learning curve.

Accomplishments that we're proud of

The sign up/login page works pretty well. The ultimate layout of the app is pretty cool

What we learned

We learned how to code in Swift and how to use Storyboards to design our project.

What's next for Kultura

We are aiming to incorporate either Apple or Google Maps into our app and add overlays to it so that people can select specific regions.
