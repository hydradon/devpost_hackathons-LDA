Inspiration

Creemos que es importante que la personas tengan control y conocimiento acerca de sus estados financieros de una forma fácil y segura.

What it does

La aplicación esta linkeada con tus datos de Konfio y te despliega toda tu información financiera de manera intuitiva y gráfica.

How we built it

A partir de sudor y horas de desvelo, modelamos los datasets que nos entregaron para pasarlos a un modelo de BD, al mismo tiempo se adapto el "design system" de konfio a nuestras pantallas, posteriormente a armar el back-end junto con las interfaces en flutter y para finalizar integrarlas a la lógica mediante servicios.

Challenges we ran into

Entender los datos para poder entregar la información necesaria, errores de compilación y ejecución de Flutter, adaptar grandes cantidades de información para que sean mostradas de manera sintetizada en la aplicación

Accomplishments that we're proud of

Lograr obtener una aplicación que trabaja nativamente y que se adapta al "design system" de Konfio

What we learned

Generar endpoints mediante node.js y el funcionamiento de una aplicación en flutter

What's next for kompás y escuadra

Lograr que se adapte a mas empresas financieras, generando una colaboración entre empresarios sana
