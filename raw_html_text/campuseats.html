Inspiration

We wanted to create an app that would allow college students to quickly find food around them.

What it does

CampusEats gathers data from your school's dining website to display dining locations, times and status'

How we built it

We started the app on Android which was built using Android Studio. Our backend was built with Amazon AWS and Python.

Challenges we ran into

Gathering information from school's websites was a challenge. We were also new to Android development, so getting ourselves exposed to a new environment took some work as well.

Accomplishments that we're proud of

Our backend can now gather information from a school's dining page and populate that into sendable data.

What we learned

We exposed ourselves to Amazon AWS as well as Android development.

What's next for CampusEats

We look to complete the initial release of this app. After that, we plan to re-design it and release a version for iOS.
