Inspiration

Connecting angel investors to communities with tech potential. Re-imagining how investors pinpoint regions to watch closely.

What it does

It parses through Google search terms and compares the amount of search terms for a specific category, where that category is most popular, to other cities in order to find out which of the other cities is an unknown and future hub of the category.

How it can be used

This project re-imagines how investors look at different regions to find companies to invest in. It is also a tool for angel investors to find regions to setup mentorship programs, and community tech spaces, as a social good venture.

How we built it

We scraped data using Python and trained it using Tensorflow and Baidu Paddlepaddle. Then, we represented the data with Google Maps API.

Challenges we ran into

Getting our model to work on PaddlePaddle

Accomplishments that we're proud of

Learning PaddlePaddle. Designing the front-end of the hack. It's pretty.

What we learned

Google Maps API, PaddlePaddle

What's next for Speculo

Creating different categories other than tech, visualizing data better, pulling larger amounts of data for more locations
