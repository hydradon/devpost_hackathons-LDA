What it does

controls a robotic arm using leap motion

How we built it

hardware side: we used 5 servo-motors, arduino , and a leap motion detector. Software side: we used javascript, leapjs and johnny-five packages

Challenges we ran into

wiring up the hardware properly, fixing the bugs in the code, implementing the inverse kinematics properly

Accomplishments that we're proud of

It works!!!

What we learned

javasript, leap motion, inverse kinematics

What's next for Leap Robotic arm

would like to find a way to use voice recognition or iot
