Country code Checker

A very simple project using the Nexmo Number Insight Standard API. User's simple supply the country and phone number they wish to check. The app will sent the info to Nexmo for verification. The Nexmo API will send back a JSON response the the phone number detail or an error if the phone number format is incorrect.
