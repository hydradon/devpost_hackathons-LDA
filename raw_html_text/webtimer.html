Inspiration

Wanted to make a hack that would limit distractions especially since social media has grown a lot

What it does

It displays timer on how long the user is browsing certain sites. It then redirects the user to a meme once the timer exceeds the allotted time.

How we built it

Used frontend web technologies such as HTML, CSS, and Javascript.

Challenges we ran into

We were not able to use the Twilio API to expand on our project.

Accomplishments that we're proud of

Building a workable chrome extension with no experience.

What we learned

Front end web development using vanilla Javascript.

What's next for WebTimer

Trying to integrate the Twilio API with our chrome extension. We plan to use the Twilio API to send messages to parents and guardians that their kids are on certain social media sites for too long.
