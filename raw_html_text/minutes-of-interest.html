Inspiration

Start learning is easy, but keeping up with it is much harder.

What it does

Our product is a customized subscription service which sends you a unique video or article on your topic of interest, based on the amount of time you plan to spend on learning.

How we built it

We built the front end user interface using Ruby. User information is taken in order to provide a customized list of learning material via email.

Challenges we ran into

SendGrid does not support large amount of email generation for testing purposes, we had to create multiple accounts to run tests. Youtube API's search function does not allow to filter on specific video duration.
