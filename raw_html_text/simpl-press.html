Inspiration

Facebook Fake News, Bi-partisan news outlets and too much information.

What it does

Uses IBM Watson's AlchemyData News and Language to get meta-data on news articles. Using

How I built it

Dropwizard Java Microservice with a React.js frontend

Challenges I ran into

Async delays, and loading of metadata is very slow.

Accomplishments that I'm proud of

It works, we can get a useful insight into an article without having to read it first.

What I learned

Time is precious

What's next for Simpl.press

Improve caching of requests to improve speed Storing popular stories in a DB Using more meta data and other sources like Esri and Facebook to visualize and map articles against sentiments and reliability User customizations and better UX
