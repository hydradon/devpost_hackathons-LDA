Inspiration

We all were drawn towards the Trellis challenge and data visualization technologies that uncover value from raw data and spearhead the discovery of new meaning. We are leveraging the Trellis API to harvest the data and use machine learning technologies to complete simple tasks just using voice recognition. Based on our research, we discovered that over 15% of Canadians use household smart devices, it is expected to project to 30% in 2022. It is essential that as technology changes, we become more adaptable. In the future, smart devices will be implemented in our workspaces just as much as they are in our households. We decided to develop a smart voice API that will propel us towards that future.

What it does

Smart Sativa will act as your normal household smart speaker. It will take voice commands as given and produce outputs. However, we will be using the Trellis database to output these commands. Users will be able to speak to our smart API to accomplish tasks without ever touching their computer. For instance, users will be able to track and log in when they are switching over batches to different rooms. Users will be able to use Trellis hands-free by using a bot instead. This bot will automate tasks typically done by sitting down at a computer. The bot will be powered by Automatic Speech Recognition (ASR) and Natural Language Understanding (NLU) capabilities, the same technology that powers Amazon Alexa. Smart Sativa bots can understand user input provided with text or speech and converse in the natural language.

How I built it

We built this using React.js. We also used the Wit.ai API to have a communication feature between the user and database. We programmed the software through the Atom IDE.
