Inspiration

We wanted to make it easier for hackers to find information about upcoming hackathons.

What it does

hackm.app shows data for hackathons including how far away they are, whether or not they allow high schoolers, location, website, etc.

How we built it

Using a web scraper, we compiled the information and used a Typescript backend to present it in a clean visualization.

Challenges we ran into

We failed at making a physical map of with LEDs. Some of the hackathon websites were still on http instead of https, which made integration hard. We resolved this by feeding them through our own process.

Accomplishments that we're proud of

Web Scraper built from the ground-up. A cool logo.

What we learned

How to integrate between scripts.

What's next for hackm.app

Add hackathons outside of MLH, labels, and filters.
