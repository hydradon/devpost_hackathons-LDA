• JetBlue- Trivia Game

Inspiration

We were inspired by JetBlue's customer-focused mission. JetBlue offers free snacks on flight and will now provide in-flight trivia games.

What it does

This Java program utilizes the JOptionPane class to ask the questions and calculate the score.

How we built it

In Eclipse, we coded in Java.

Challenges we ran into

We recognize the limitations of our coding skills. We actually taught ourselves arrays and JOptionPane today.

Accomplishments that we're proud of

We programmed a functioning trivia game.

What we learned

We gained invaluable teamwork skills to complete a project within a deadline. We learned that with a bit of motivation we can advance our skills.

What's next for jetBlue Fun-Facts

This program can be expanded with more levels, prizes, social media integration on the back-end, and better graphic design on the front-end.
