Inspiration

With one of our team members having grown up on a beef farm, we were inspired to create a tool to help farmers manage their livestock easily and cost efficiently.

What it does

LIVEstock gives farmers the ability to keep track of breeding information, medical history, and general data by simply scanning the QR code on the animal's ear tag.

How we built it

LIVEstock was created by building a mySQL database and a web dashboard built with handlebars, bootstrap, PHP, html & CSS.

Challenges we ran into

One of the challenges we ran into was getting blocked by security rules while setting up the database.

Accomplishments that we're proud of

We're proud of our finished product because it is a light-weight tool, cost effective as QR codes are free to create and scan, and extremely easy to implement on a real farm.

What we learned

Laura learned learned more extensive CSS and html skills, how to build a mySQL database on PHPmyadmin, and how to create QR codes. Dexter learned how to edit security rules- although extremely boring it is very useful and practical for future projects.

What's next for LIVEstock

LIVEstock will be definitely be implemented on a real farm one day- if not for full production then just to show farmers how simple technology can make their lives a little bit easier.
