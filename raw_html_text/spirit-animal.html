• Spirit Animal

Inspiration

I believe we all need an animal to get us through life. Maybe it is a pet, maybe it is a wild beast. I think regardless there is a lot to be learned about the animal world and how we might be guided by them to be a better person and do better things.

What it does

It assigns you a spirit animal, and tells what the animal brings to you.

How I built it

Using tutorials from Amazon, I created in the Sublime text editor, then loaded it onto Lambda. Then I created the Skill in the Amazon developer portal.

Challenges I ran into

Getting it published. You need to make sure you fill out all required fields.

Accomplishments that I'm proud of

Being named an Alexa Champion.

What I learned

That certain animals can be inspirational in different ways.

What's next for Spirit Animal

I would like to update it with more animals, and maybe fine tune it to pick based on attributes of the individual.
