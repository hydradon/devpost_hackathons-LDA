Inspiration

Our inspiration lies in the need of a simple and attractive way of collecting straight-forward answers to various polls with a encouraging response rate. Our main target is the working environment and that is way we thought about a desktop and not a mobile app.

What it does

ZiPOLL is a Windows desktop app which will be launched from system tray and will be installed by sysadmins. Each user can create poll and answer another colleague's poll.

How we built it

We are using Autoplay Media Studio (see here https://www.indigorose.com/autoplay-media-studio/ ) which offers support for any Windows version, from Windows XP to Windows 10.

Challenges we ran into

The first challenge we faced even before coming up with ZiPOLL idea: a few days before ZiHack we realised that the then-project was unfeasable. We quickly met for a brainstorming session and step-by-step, we concluded that ZiPOLL is a nice and ZiHack-able project.

Another challange was also finishing the functional project on time.
