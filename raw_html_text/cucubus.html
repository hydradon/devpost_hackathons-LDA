Inspiration

Tired of falling asleep on the bus.

What it does

The project is focused on providing the user with an alternative different from the ordinary, functioning as an alarm when falling asleep in the bus. This problem is present in the daily routine of people and acitivities. The project CucuBus It offers a way to keep track of user scheduling driving their daily routes to so inform you when this near your destination and notify you instantly through an application mobile.

How I built it

It is built with software tools for developing mobile applications Android Studio, Eclipe, programming languages like Java and use of libraries for specific use.
