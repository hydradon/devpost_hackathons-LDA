Inspiration

We've always wanted to be able to experience what it is like to be an air traffic controller, in VR.

What it does

Basically it's a watered down Air Traffic Control Sim in VR with the Alexa. You can clear planes for landing/takeoff. Prepare them for takeoff/landing. Have them enter holding patterns. And most importantly, prevent them from crashing!

How I built it

Created astar paths for the planes to follow on the runway, and for takeoff/landing. Used Alexa to change values in a Firebase database. Unity listens for the changes and acts appropriately.

Challenges I ran into

Firebase and Alexa don't play well. The desktop we brought could barely run the Vive, so Unity crashed frequently.

Accomplishments that I'm proud of

It works

What I learned

Unity, Alexa, and Firebase

What's next for Project Air

Adding more functionality, more types of emergencies, and more ways for the user to interact with the airport.
