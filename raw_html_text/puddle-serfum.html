Inspiration

Recent downpour and walkway blockages around campus due to it

What it does

Allows users to report potential pedestrian obstacles to help others

How I built it

Created an IOS app which sends requests to a webserver that processes data and contacts radar.io for positional mapping then sends data in the reverse order.

Challenges I ran into

Data formatting and storage. IOS integration and programming of an app with the Maps SDK

Accomplishments that I'm proud of

A working webserver that makes and processes API requests with a variety of functions

What I learned

How to format API data, make an API, and develop an IOS app

What's next for Puddle

Adding in more obstacles and custom obstacles for things like events in the way
