Inspiration

Assist people who have tremors or other disabilities with smart phone usage.

What it does

It offers an alternative to interact with on-screen links, objects, etc.

How we built it

We used android studio to make a mock social media app to showcase how our accessibility option works. This was all coded in java.

Challenges we ran into

Originally we had several ideas that we thought long over. It took a while to figure out if what we wanted to do was even possible. We are all fairly new to using Android development. In addition, this is the first hackathon for all of us.

Accomplishments that we're proud of

We didn't give up and got quite far with what we wanted to do.

What we learned

We learned how to use Android studio as well as new ways to code with java.

What's next for Swypr

Implementation of additional ideas we had, such as voice interaction.
