• The top 3 nearest movie cinemas based on the user's location.

  • Multiple movies with corresponding timing slots are given to the user.

  • A link to the specific movie ticket-booking site, is delivered to the user upon selection.

  • Link opens up to a site for the user to book movie tickets.

Inspiration

We want to make the process of booking movie tickets pleasant.

What it does

A telegram bot that finds the closest 3 cinemas based on the user's current location, and returns the booking link to make a movie ticket purchase.

How I built it

We built it using python-telegram-bot and Microsoft Visual Studio.

Challenges I ran into

Understanding and perfecting the syntax for Python was tricky, coming from a Javascript background.

Accomplishments that I'm proud of

Sorting the data, learning python-telegram-bot from scratch, extract data from webpages.

What I learned

Python-telegram-bot and all of their Python in-bulit functions. We learnt how to incorporate APIs such as BeautifulSoup4, PrettyPrinter.

What's next for Lazy Movies

Firstly, we want to incorporate the bot in such a way that a group can use the bot to vote for the best timing the group can watch the movie. This would ultimately allow groups of people to book tickets together in a simple manner. Secondly, we would like to present the available seats in the bot so that groups can choose their seats together. Alternatively, after a member of the group has purchased a ticket, the seat number could be announced to the group through the bot. Furthermore an e-ticket could also be sent through Lazy Movies in the near future.
