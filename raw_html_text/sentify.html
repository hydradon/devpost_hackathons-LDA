Inspiration

To analyse sentiments from Customer Feedback, Chats, Mails, Social Posts

What it does

Analyse the text and determine the sentiments of the text input by the user. In a safe and secure environment. No data is saved on App's server and user can secure the analysis and sentiments score in a decentralized environment provided by Blockstack

How I built it

I built it using AWS Comprehend with Blockstack Authentication

Challenges I ran into

This is my first ever web app using React, CORS configuration was a tough task to set up

Accomplishments that I'm proud of

Implementing Blockstack Authentication on React App

What I learned

React, Blockstack Framework, AWS Comprehend

What's next for Sentify

Create Dynamic Charts and Reports based on sentiment analysis
