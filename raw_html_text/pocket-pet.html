Inspiration

I was inspired by the Viacom presentation when the presenter talked about tamagotchi. It reminded me how much fun it was having a virtual animal to feed and take care of, so I made Pocket Pet!

What it does

You choose what kind of dog you want, name him/her, and give him food as his/her stomach gets empty. It's real time so it takes a bit for him/her to get hungry. Currently you can only buy Bones as food but there will be more food items in the future!

How I built it

On eclipse with Java. I have experience building 2D game engines

Challenges I ran into

Lots of bugs with the mouselisteners.

Accomplishments that I'm proud of

Created my own buttons and textboxes for user input. I also am proud of my settings and save handlers.

What I learned

Too much to say it all. How to make GUI components from scratch. Tricks for 2D game development. Organization. Image IO use.

What's next for Pocket Pet!

Complete the store. Have a WorkState in which you do tasks (complete mini games) to get money that you can use to buy food. Also, I want to implement Idle-RPG design to allow the user to make money off of his/her pet while he/she is offline.
