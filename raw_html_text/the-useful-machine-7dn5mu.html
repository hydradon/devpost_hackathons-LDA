Inspiration

Wanted to create something useful.

What it does

Moves a servo attached to a dry-erase marker to draw a dot, then moves another servo attached to an eraser to erase it.

How I built it

Arduino + servos + marker + eraser + tape + love <3

What's next for The Useful Machine

Long-term installation in an NYC art exhibition.
