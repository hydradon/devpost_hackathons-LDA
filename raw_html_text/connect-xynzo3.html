Inspiration

We wanted to create a platform that connects on campus organizations to local food businesses and allow them to host fundraisers easily.

What it does

It provides simple calculations for organizations to find out how much profit they could make with a certain fundraiser they plan on hosting, as well as placing an order of fundraising items directly through the webapp.

How we built it

We built it using ATOM editor with the language html, css, json, and java script.

Challenges we ran into

We had no previous experience with development of webapps, so we had to learn it from scratch and build it.

Accomplishments that we're proud of

We were able to create a webapp that has the basic function we initially hoped to create with it.

What we learned

Coding language for html, css, etc. and also how to create webapps

What's next for Connect

We would like to further embellish it through adding more functions and then create a database that allow organizations to sign in and view different information about the food businesses.
