Inspiration

Schools cost of use of scantron machines

What it will do

It uses the phone camera to scan scantrons and reads the answers that will be compared with the key.

What it does (as of now)

Photos taken from the phone are sent to our Backend PHP Server which is hosted on Google App Engine connected to a Cloud SQL Postgresql database. This images sent are then stored on Google Cloud Storage that can be processed by the Google Vision API.

Sign In functionality uses BlockStack signin javascript library.

Accomplishments that I'm proud of

seeding a bunch of quality scantron fakes.

What I learned

React-Native, working with Base64 image data and sending it over the server as JSON data.

What's next for OpenTron

Ability to scan scantrons. Funding and future development.
