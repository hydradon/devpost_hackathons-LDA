Inspiration

Virtual reality and general ignorance over the origins of laws

What it does

Guides the user through a virtual reality simulation that teaches them how a bill becomes a law and later provides a quiz on a website to test their knowledge. The website contains analytics showing the distribution of answers and uses NLP and ML to evaluate what keywords make a good bill and show predictions for the number of bills that Congress had/will pass.

How we built it

We built the VR portion using Unity, an Oculus Rift, and an Alienware computer. We created the website using NodeJS.

Challenges we ran into

We had some difficulty with coordinating Unity with the script we created for the game. In addition, we had some trouble integrating all the parts of our project into the website.

Accomplishments that we're proud of

We are proud of finishing the VR portion and combining several components seamlessly into our website

What we learned

We learned more about embedding charts, images, and gifs into websites. In addition, we learned about using voice recognition with VR.

What's next for Just a Bill

Potentially moving everything on the website to VR.
