Inspiration

Being a cricket fan, I always wanted something that provides me with instant access to cricket news, score, rankings and other information from around the globe.

What it does

This skill provides you information about current matches, score update, Rankings and news from cricketing world.

How I built it

I used Alexa API available in node.js and hosted this skill on lambda server provided by amazon

Challenges I ran into

It was hard to access fast, reliable and correct cricketing info. I had to parse multiple sources to access the best information possible.

Accomplishments that I'm proud of

I am proud to develop such helpful skill, personally I use this skill regularly and when I told about this skill to my friends they too were happy and excited about it.

What I learned

I learned about alexa skill development process, node and asynchronous programming in it.

What's next for CricScore

Next I would like to introduce fun games like predict cricket score, in this skill to increase the user interaction with this skill.
