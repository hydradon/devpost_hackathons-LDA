• Gambling Bank

Inspiration

When going through the sponsors, the idea of combining gambling games and financial services came to mind.

What it does

It allows you to gamble a percentage of your money each time you deposit money.

How we built it

We started with an example of a springboot application and worked our way through adding features.

Challenges we ran into

Started late, had to work while being very tired throughout the night, but managed to make it through. Had to deal with a member's github account being marked and he couldn't access anyone's github repository at all for some time.

Accomplishments that we're proud of

The demo application works pretty well to demonstrate the idea.

What we learned

How to use web frameworks to demo an idea in a very short time and how to priorities features vs stability when developing a demo.

What's next for Banking and Gambling

This will depend on how it is perceived by everyone else.
