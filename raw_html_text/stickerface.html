Inspiration

We figured that monitoring of human emotions in real-time can be very beneficial for everyone to pick up a few extra social skills. Knowing what other people are feeling through facial expressions can be a more accurate assessment than listening as not everyone opens their hearts freely! Also, additional cuteness in the world never hurts.

What it does

It allows users to tag and render images and videos with cute emoticons based on emotions on people's faces.

How I built it

Just a few functions written in python that uses the Microsoft Emotion API to add emoticons to faces in images, videos and webcam streams.

Challenges I ran into

I had zero experience with programmatic handling of videos; in fact, you could call me a relative newb in python. I had to learn everything from scratch and overnight in a team of only 1 person who writes code after the third never returned. Nonetheless, we managed to get to exactly the product we set out to create.

Accomplishments that I'm proud of

Refer to point above. We think that the final results are rather entertaining to watch, especially the rendered videos (see youtube)

What I learned

A ton of image and video handling tricks in python and opencv. A thorough reviewer of the code would notice my extremely incoherent use of image libraries - it will be fixed.

What's next for stickerface

Refining emotions we can detect and optimizing the code for many of the rendering parts.
