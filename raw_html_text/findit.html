Inspiration

Ever since my teammate and I arrived in Merced, we heard everyone wondering what there was to do in Merced. We ourselves pondered on the question. At first glance, there is nothing to do just as most people assume. But after living in Merced for a few years we've come to know all of the hidden gems that Merced has to offer. FindIt is our chance to share that information with anyone still asking the same question.

What it does

FindIt is an app that allows the user to locate places to go have fun on any given night. Driving down a road in search of entertainment can be tiresome and sometimes even dangerous. Therefore, we have pre-selected areas around Merced that may be attractive for tourists, locals, and students alike. All they would need to do is simply download our app and choose their location. We wanted to give the user options based off of their interests and expectations. We currently have one filtering question asking about the budget and how much money the user is willing to spend. If the user answers Low Budget, then the app will show a list of local attractions within their budget. The user then has the option to click on any location from the list and receive more information.

How we built it

First, we created a Domain which we named www.evaddas.com . We used Android Studio to create the android application. It is programmed to work on Androids running Lollipop(5.0) and above. We created a logo that we thought correctly depicted our project. We programmed the pages of the application using java code.

Challenges we ran into

Working on a project with only two people total while brainstorming ideas, creating an app, and simultaneously being on a time limit has been quite a challenge. Learning new skills using Android Studio has also been a challenge. It takes time to learn a skill and there are so many things that we want to add in our application. Android Studio was also acting up and creating some technical difficulties for us. We had to start all over from the beginning several time. But, we haven't let any of these challenges stop us. We will continue pushing through until the end.

Accomplishments that we're proud of

We are very proud of all that we have accomplished with this project. Even though we have far fewer teammates than most of the other groups, we have managed to use our time wisely. We have learned how to create aspects of activities within Android Studio that we hadn't known before. We are very proud of how far we have come.

What we learned

Amongst many other things, we've learned how to create a domain. We now have our very own domain named www.evaddas.com . We particularly learned how to make drop down menus in conjunction with buttons. Learning the little kinks are a necessity in programming and allowing the application to run smoothly.

What's next for FindIt

We have many ideas and expectations for the future of FindIt . We hope to ultimately create an application that can be used nationwide. Someone would just download the app,input their location, and add in their interests. They would instantly be directed to a personalized list of entertainment options no matter where they are nationwide. We hope that one day in the near future, the days of driving around searching for something to do will be behind us.
