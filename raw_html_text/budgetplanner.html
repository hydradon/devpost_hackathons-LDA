What inspired us to design this app are our very own budgeting problems. It is hard to keep track of spending, set clear saving goals or compare living costs across different areas.

Technology wise, we used Figma to create the interface, which was quite challenging as we never heard of this software and had to figure out how to work with it.

We faced the following challenges: Design challenges, time pressure, one teammate having to leave earlier.

We achieved a decent design and the structure, also the three features added(Budget, saving goals and living cost comparison)

Next steps for the project would be to improve on what we have, and add data to allow tracking spending across several months.
