What it does

  • It shows a bee inside a grid. On each tap, a block is inserted in a grid and the bee moves towards the nearest exit. The goal is to make a closed boundary with the block so that the bee gets trapped inside before flying out of the grid. We have used Artificial Intelligence for generating initial state to give a maximum number of moves. Also this is being applied to multiplayer to challenge friends.
