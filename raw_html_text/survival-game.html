Inspiration

What it does

It's a game(multi-player feature planned) that allows players to collaborate and survive in a post-apacalyptic

How we built it

We use Unity as the game engine. We made our own visuals.

Challenges we ran into

Accomplishments that we're proud of

The final game is pretty close to a playable game. For example, characters can move and shoot enemies. Moreover now we know more about Unity we can fix bugs and add more features in the future.

What we learned

Unity, C#, game development.

What's next for Survival Game

We have only implemented some of the most basic features of our design so the rest will be our future priority. Also we will test multiplayer in the future.
