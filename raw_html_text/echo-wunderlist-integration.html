Inspiration

Create easy lists and tasks for users to check through. Amazon Echo is pretty restrictive, so we decided to give any user more access and freedom with their schedule planners by creating this app.

What it does

By utilizing Wunderlist's API, we have achieved adding new lists, opening lists, and creating new tasks for users to check through.

How I built it

We used the Wunderlist API and java in order to write a service application for the Amazon Echo.

Challenges I ran into

Trust us, it would take longer to explain that than to actually present. Internet connection to the Amazon echo. Communication between Wunderlist and the java program.

Accomplishments that I'm proud of

We got the Amazon Echo to actually listen to us!

What I learned

Teamwork is necessary in order to complete large projects.
