Inspiration 

Our experience as freshmen at university!

What it does 

Pinpoint key locations at campus Mty that are not otherwise shown on any other map, to make sure lost students can find their way through campus.

How we built it 

 It's a mobile-based web application, designed with HTML, CSS and JavaScript. We used the GoogleMaps API in JavaScript for the use of maps and location interactions.

Challenges we ran into 

 Getting a concrete idea, defining the user-app interaction, and the technical implementation with the API, for it was very time consuming, and our site is still under construction.

Accomplishments that we're proud of 

 Achieving a "stable" web application with a lot of potential! And implementing our idea.

What we learned 

 Big ideas can become a reality if we use the right tools for it!

What's next for EncuentraTec 

 Continue completing the site with bicycle routes, more locations, and more specific details, along with a working side navigation. We also plan on transferring the information to a database for easier management and maintenance.
