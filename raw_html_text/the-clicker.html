Inspiration

We wanted to add a better way of communicating morse code to everyone. Using the visual and audio to the person to read the morse.

What it does

You interact with one rpi to sent the message other. The other rpi is fitted with led and a buzzer to produce the give sense.

How we built it

It was built with a very simple array of lights and buzzer and two RPI 3

Challenges we ran into

There was a communication problem that we used sockets to work around. The rpi did not connect to the devices

Accomplishments that we're proud of

It works

What we learned

Socket communication and python allowing me to use the RPI to its full potential

What's next for the Clicker

To build a mesh network of them so that you can communicate with different people
