Inspiration

Being big fans of Mirro.io we want to develop new valuable features for it.

What it does

STAR offers you an overview of your market position regarding salaries, performance, engagement, accomplished goals etc. so that it can help you take actions tailored to your company needs.

How we built it

Using Python, Django, React, Jenkins.

Challenges we ran into

Complex queries, using Django for the first time.

Accomplishments that we're proud of

Being able to manage new technologies.

What we learned

Python, Django, React.

What's next for Operation STAR

Integrate it as a new feature in Mirro app and promote it as a super new cool thing to have. More data, better correlations.
