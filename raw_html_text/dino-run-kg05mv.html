Inspiration

Our inspiration stemmed from wanting to challenge ourselves and the enjoyment of working with dinosaurs!

What it does

Dino Run is a 2 player game that uses the W, A, D, and Space bar for one player and the arrow key and shift key for the other player. Both the space bar and the arrow key are buttons to shoot at the evil pigs! The objective is to be the player that reaches the treasure first, all the while not losing its life in the process. Each player have 5 lives.

How we built it

We used several programmers, coding platforms, and coding languages to create this game. The program we used was Unity . We used Visual Studio Codes and Atom as our coding platforms, and the languages we used were C# , Java , and Html . Everything was built on a blank canvas.

Challenges we ran into

For the first 6 hours of the Hack-a-thon, we were unable to access Unity on Jade's laptop- the programmer for our game. We tried everything, restarting the laptop, deleting and reinstalling the app, updating the laptop to the most recent version, but it just would not work. Luckily, several individuals helped us fix the problem of installing Visual Studios Code onto Annie's laptop, and we were able to have one working programming laptop.

Accomplishments that we're proud of

What we learned

We learned how to create a game from scratch and customize the character, setting and obstacles. We also realized how a seemly simple game can be so hard to code!

What's next for Dino Run

Dino Run will be made into a multi-level game with even more new exciting obstacles that test the players ability to react fast. Not only so, exciting changes like audio and more screen display, such as how many lives are left, will be coded into the game!
