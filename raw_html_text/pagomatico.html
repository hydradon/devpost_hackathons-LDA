El sistema financiero Mexicano tiene un gran reto. ¿Como llegar al 70% de la población restante no bancarizada? Construimos tres herramientas con el objetivo de atacar este problema:

  1. Sistema de envio y recepcion de dinero via SMS.
  2. Sistema de cargos recurrentes via SMS.
  3. API para acceso a información crediticia a Instituciones Bancarias.
