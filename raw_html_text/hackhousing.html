Inspiration

As students at college, we know it's hard to find off-campus housing, so we wanted to make a website to make it easier for students!

What it does

The website currently displays a home page where you search a school, then a results page with a map and list of results with options to see reviews, make posts, and make reviews about nearby listings. The goal was to gather real > data to put into the website, but our task is incomplete.

How we built it

We used html to create template webpages, python to redirect between pages, and javascript to execute different functions, as well as css to make it pretty. We also created a map with a google api.

Challenges we ran into

We were all beginner web developers, some with zero experience, so starting off was difficult (but we had great mentors!!). Pulling the correct information was also a problem, as the code could not accurately scrape. Along the way, we also encountered various challenges we didn't know existed such as using github and api's.

Accomplishments that we're proud of

We got familiar with python, html, css, javascript, and Google!

What we learned

All the basics of web dev starting at 0; so everything we did was new!

What's next for HackHousing

We may try to get real data and get the website running online so students can actually use it.
