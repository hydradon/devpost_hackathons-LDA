Inspiration

My inspiration came from my horrible grades and messy school life! I built this so I can track all of my work for school in one place that syncs across all platforms.

What it does

Keeps track of all assignments, notes, and classes you have!

How I built it

I used Meteor to build a cross platform app to make it simple to sync across all platforms.

Challenges I ran into

Meteor and having a schema. I ended up ditching schema and having a messy one document system.

Accomplishments that I'm proud of

Using meteor, a framework I have never used again.

What I learned

Meteor can be annoying to work with for small projects, but it brings a lot of advantages and reduces the time needed to prototype.

What's next for Homework Planner

Adding notifications for the mobile apps or change the UI layer in Meteor to ReactJS + React Native. Also, I want to change the mongoDB to using GraphQL.
