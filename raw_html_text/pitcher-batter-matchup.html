Inspiration

I find the idea of pitching metrics really fascinating. I wanted to build an app that would give me more information on batters and how pitchers pitch to them.

What it does

A user can choose a pitcher and a batter, and see how a pitcher has pitched to that batter.

How I built it

Next.js with node backend

Challenges I ran into

Graphing libraries are hard! Getting the styling right was difficult. Also, massaging the data to be consumed by the API took time.

Accomplishments that I'm proud of

Being able to write a full stack app in a day.

What I learned

I learned a lot about what data sources I could use to make this app happen. It took quite a while to find the correct data sources.

What's next for Pitcher Batter Matchup

More data visualizations, and perhaps a "lookalike" pitcher feature to find players who are similar.
