Inspiration

Many were lacking in producitivity and wasting their time on useless things. So, We integrated ML technology with the application so that it automates the process and alerts the user when it notices the movement.

What it does

Our App notifies the user whether the task has been completed within the given time or not.

How I built it

I used Xamarin platform to create the UI and used C# for the functions (Backend). And, The Azure ML cognitive services helps to detect the focus of the user and alerts when he gets out of the focus.

Challenges I ran into

I challenged myself when I tried to connect Azure ML API with our app.

Accomplishments that I'm proud of

I used Azure ML for the first time and successfully integrated it with the application.

What I learned

I learned how to use Azure ML and I explored other services of Azure. And, i am going to use it for my future developments.

What's next for Marinara-Pomodoro

We train our ML model with more data set.to so that it finds more accurate of the user movements. Also, We try our best to improve our UI/UX experience of the application.
