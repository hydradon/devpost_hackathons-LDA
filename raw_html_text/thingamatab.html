Inspiration

Making boring ol' new tabs fun!

What it does

ThingamaTAB randomly loads a fun & addicting thingamajig every new tab is opened!

How we built it

We used a vector graphics scripting JavaScript framework called Paper.js to create original interactive designs.

Challenges we ran into

We had trouble making the pages work for Google Chrome extensions.

What we learned

We learned Paper.js and how to create a Chrome extension

What's next for ThingamaTAB

We plan on creating more fun pages, publishing it onto the Chrome Web Store, and creating a separate website.
