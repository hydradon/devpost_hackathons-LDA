Inspiration

Fantasy sports on the rise - foosball needs recognition as a competitive sport.

What it does

Enables you to play Fantasy Foosball with friends and other random people around the world - with real money!

How we built it

We have a RESTful API built with the Spring framework which stores and manages all of the user data and any interactions. The web app is the main user interface which allows users to create accounts, manager leagues and players, and bet real money.

Challenges we ran into

Accomplishments that we're proud of
