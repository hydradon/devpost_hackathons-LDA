Inspiration

We wanted to use IBM's bluemix for some real-time application using firebase.

What it does

The Reddit bot /u/chattrbot will reply to posts in /r/chattrbot. This conversation is visualized in real-time on An Azure hosted web-app . We use IBM's Bluemix Watson text-to-speech to convert the conversation to audio. You get to hear the conversation as it happens!

How I built it 

ChattrBot was built using a combination of IBM's BlueMix, Microsoft's Azure, and AngularJS.

Challenges I ran into 

Learning how python runs on a server was definitely interesting when you don't know anything about python to begin with.

Accomplishments that I'm proud of

What I learned 

Pyhton,  Angularjs

What's next for ChattrBot 

We would like to use IBM's BlueMix Watson Dialog to generate the responses more intelligently.
