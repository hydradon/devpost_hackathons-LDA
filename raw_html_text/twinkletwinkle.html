Inspiration

We were too hyped for Christmas and needed some twinkling lights.

What it does

Visualises how a neural network learns with the help of LEDs that light up.

How we built it

Using a raspberry pi with a server that takes requests, we send current to the LEDs that light up as the nn learns.
