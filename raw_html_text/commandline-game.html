The Story of Root

This is a simple game where by you have to use linux commandline commands to navigate a lost friends house in order to find a hidden letter which contains his whereabouts.

Inspiration

I was inspired by the text game Zork and my wish to make the commandline a more friendly environment for beginners

Challenges

The real challenge I faced was how to arrange my code and structure it

Lessons Learnt

I learnt about how python's import statement works and about the state machine design pattern

Future Plans

My future plans are to make the game more complex and randomise where the letters will be placed. I also plan to include a timer to see who the fastest solver will be
