Inspiration

I suck at math. So I wanted to solve math equations fast.

What it does

After you upload an image, the OCR program we built will process the image and get the math equation. After that, the program calls WolframAlpha's API which returns the equation's answer.

How I built it

With Javascript and Tesseract's api.

Challenges I ran into

Processing Ajax requests and using Wolfram's API.

Accomplishments that I'm proud of

Finished the whole project. There may be some bugs still.

What I learned

How to process Ajax requests.

What's next for QuickMaths

Better User Interface and more advanced Image recognition.
