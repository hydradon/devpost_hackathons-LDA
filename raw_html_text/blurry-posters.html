Inspiration

To spice up our office rooms and give project teams their own identity, the hackathon team will create a tool for generating awesome graphical and printable posters presenting a project with its team, mission, and tech in a nerdy way, e.g.:

  • A fancy generated team name
  • A crazy generated logo for the project team
  • Team members, maybe with their faces mapped on superhero bodies (deepfake? woah!) and showing their superpowers/roles/skills
  • The team's mission statement (maybe as meme?)
  • A buzzword logo wall highlighting the tech stack and tools the project uses
  • QR code that takes you to the project demo or docs
  • More cool things to show on the poster

What it does

A CLI tool with an interactive prompt that asks for basic project parameters and then generates a printable awesome team identity poster.
