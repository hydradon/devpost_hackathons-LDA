Inspiration

Twitch plays Pokemon and FTL.

What it does

It is a space shooter collaboratively controlled by Twitter mentions.

How we built it

  • Front end : JavaScript, it draws the game state to a canvas,
  • Back end : Python, it simulates the game, receiving tweets to a Twitter account, that control the player.

Challenges we ran into

  • Twitter performs rate limiting and does not allow duplicate tweets to be sent by users,
  • Scrolling the map on the front end relative to the player took some time.

Accomplishments that we're proud of

It works-ish.

What we learned

  • Unlike Skynet, Python is not self-aware,
  • Dolphins are whales,
  • Features are important.
