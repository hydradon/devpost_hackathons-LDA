Team 5:Hercules NPO:Braven

Inspiration

Braven has a manual system for interview scheduling for students with people who are currently working in their respective field. We have automated the procedure of scheduling using multiple additional features to save time.

What it does

We can schedule interview for students with professionals. We can have a feature of Zoom Video interview. Allows to give feedback.

How I built it

We built a Web application with React and Node JS with MongoDB in back-end.

Challenges I ran into

Interview scheduling algorithm according to Braven's policies. Zoom integration, Heroku integration and PayPal integration. Lack of time for multiple features.

Accomplishments that I'm proud of

We have successfully integrated Zoom, PayPal. Implemented the scheduling of interview. Feedback feature. Nice UI. Kafka for managing large number of users.

What I learned

Zoom Integration, PayPal Integration, Heroku deployment, Intelligent Scheduling System with multiple constraints.

What's next for Braven-Scheduling-System

We can add more features like recommending courses or tutorials for students based on the feedback.
