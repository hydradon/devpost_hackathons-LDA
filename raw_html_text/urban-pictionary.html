Inspiration

Urban dictionary, duh.

What it does

Urban Pictionary presents gifs to the player and waits for them to answer the slang word associated with the gifs. The user is able to ask for hints and see a definition of the slang word.

How we built it

We used a parse backend, with a front end on web using javascript and vue and an android mobile application. The user has a facebook login.

Challenges we ran into

We had to filter out a lot of terms.
