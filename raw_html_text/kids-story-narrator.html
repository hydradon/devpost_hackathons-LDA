Comprehensive Passage Reading, Listening and having question/answer of passages for kids is a common practice for every student in primary schools. So i want to make this practice available for kids in more interesting way using advanced technology so they can listen to the stories and have following QA's using multiple amazon polly languages .

So this skill has been built on Alexa, Java, iText, S3, Microsoft Language Translator and Lambda technologies. So the content is streamed dynamically from PDF's which are stored in Amazon S3 environment. So we can add many stories/passages in s3 and the skill will be able to play the all of them in many amazon polly languages without any code changes in the skill.

Adding Polly service to be available in the skill is good asset to the skill & in future we will make rich in having strong KB by adding more stories in S3.
