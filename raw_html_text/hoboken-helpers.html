Inspiration

Volunteering is beneficial to society, but people tend not to participate due to a lack of knowledge surrounding volunteering opportunities. Hoboken Helpers seeks to fix this issue locally here in Hoboken.

What it does

The site holds information about volunteering opportunities surrounding Hoboken.

How we built it

It is built using HTML/CSS, and is hosted on Firebase.

Challenges we ran into

We ran into some issues imported a json file and integrating google authentication for volenteer submissions. Due to the time constraint we were unable to fix these issues in time.

Accomplishments that we're proud of

A clean website that hopes to inform individuals around Hoboken about possible volunteer opportunities.

What we learned

We learned that time constraints can be an issue when you don't restrain your scope from the start of the project.

What's next for Hoboken Helpers

We hope to get through the challenges we ran into and have a site for both volunteers and organization surrounding Hoboken.
