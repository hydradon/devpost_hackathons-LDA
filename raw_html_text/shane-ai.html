Inspiration

We have a mutual friend that makes bad judgement when building and designing a resume. This project should help him and others like him.

What it does

Chat with a bot through Facebook messenger. The bot will then create a resume based on your information. It will also allow users to change the style, content, and order of sections on the resume based on the user's preferences.

How we built it

We used Facebook Graph API to get user messages and send them to a flask web application that parses a file or the user's message using IBM Watson Conversation service. The parsed object is then saved on Firebase.
