Inspiration

The problems that faces farmers are too large and to contribute in the OCP project of 'La maison du fermiers' we've proposed this prototype

What it does

It's a sort of drone that insecticide spray and visualise the action in a mobile application containing a chatbot that understand writting and speaking "darija"

How we built it

The project will be based on a raspberry pi system, which will contain a couple of raspberries connected to a drone that will be connected with a wifi network to the mobile application and visualise with its camera in a direct way by the UDP Protocol. This application will contain also a chatbot that can understand natural language "Darija" using machine learning tools (NLP, voice recognition...) and the user will have also the control of the actions of the drone.

Challenges we ran into

Minimize cost and production time. Protect farmers from chemicals caused by insecticide spray.

Accomplishments that we're proud of

As a group we are 2 devolopers full stuck and a data scientists we are proud of using the IA and IOT to simplify farmer live

What we learned

We learned some notions of agricultures that help as to knew the problems that a farmer can face

What's next for Smart Farmers

Deploy this application to be available for all categories of farmers in Africa.
