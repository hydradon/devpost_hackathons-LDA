Inspiration

I'm a newby using API's and i was interested to learn about bots with python (this is the first bot i've ever made)!!

What it does

The bot wait for the user to send an image, then downloads it and uploads to an Imgur server to get an URL. Then calls the resb.ai API to thread the image and wait for a response. Finally sends the response back to the telegram chat.

How I built it

I used only python and some web API's to communicate with Imgur.

Challenges I ran into

Learning bot programming from 0 to... hero??

Accomplishments that I'm proud of

The whole project. Specifically how i could learn all this stuff just in one day. I enjoyed a lot the selenium API!!

What's next for Telegram Bot that uses resb.ai IA recognition image

Maybe it could be improved to connect to some shopping page and then search for the prize of the objects the IA found. It could be useful to interior designers, just taking one pic you could know the cost of building a dining room for example!!
