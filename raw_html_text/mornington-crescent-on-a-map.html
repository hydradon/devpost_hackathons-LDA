Inspiration

Newcomers to London and to programming would really benefit from a fun exercise/puzzle that writing in Mornington Crescent presents.

What it does

It shows you the execution of your program on the iconic map

How we built it

We wrote our own interpreter and a browser based interface to show the program running

Challenges I ran into

Mornington Crescent is quite challenging by itself ;)

What's next for Mornington Crescent on a Map

Real life data from TfL, transpiler from nicer languages and many parallel executions on the same network - Tube can accommodate millions passengers daily!
