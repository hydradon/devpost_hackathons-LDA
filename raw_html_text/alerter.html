Inspiration

Sifting through many irrelevant articles in order to find information about important topics in daily life.

What it does

Alerter scrapes through news websites for you, classifying articles in order to determine their importance. You interact with Alerter through a chrome extension, allowing immediate access to a filtered news feed.

How I built it

Used Python to develop the backend API for web scraping and natural language classification. Developed the chrome extension with html/css/javascript and chrome api

Challenges I ran into

Figuring out how to write a chrome extension and a web scraper.

Accomplishments that I'm proud of

Having completed the project in time.

What I learned

How to write a chrome extension + web scraper

What's next for Alerter

Potentially adding features to personalize news tracking.
