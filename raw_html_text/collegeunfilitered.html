Inspiration

After being accepted to several colleges as young adults, our team all had a lot of questions about the campuses we were admitted to. We all had unanswered questions about specific departments, courses, and college life. We wanted to tackle this problem by providing a similar peer mentoring platform such as dildo.io by MIT.

What it does

Connect prospective students with current students in a variety of class levels and programs across college campuses.

How we built it

Using Bootstrap to create a static site with a SQL database. We added a Facebook Messenger API to make it easier for the user to connect with a mentor.

Challenges we ran into getting

Pulling the data from the server; it was a new challenge for all of us. We also attempted to to use AWS as an alternative platform.

Accomplishments that we're proud of

Getting our website up as our team's first hackathon!

What we learned

How to pull data from a database using PHP, Express.js, Flask, etc. We also were able to learn how to approach problems differently and exposing ourselves to new platforms, libraries and frameworks.

What's next for CollegeUnfiltered

Making it a real website and testing it out at our college campuses!
