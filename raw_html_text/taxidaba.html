Inspiration

In Morocco we have this issue of finding quickly a taxi cab.

What it does

This application will try to find a solution of this issue by giving a realtime interface to match passenger who need a taxi cab with taxi driver who wants to fill seats of his taxi.

How I built it

I built it using MIT App Inventor for frontend and PHP-MYSQL (Silex framework) for backend. I also build a testing tool to test interaction between taxi drivers and passengers, and to simulate gps positioning

Challenges I ran into

Discovering the geo localization Api using Google Map API.

Accomplishments that I'm proud of

I built this app in very small time, around 6 days (I learned to late about this challenge that's why)

What I learned

Many things, how to manage my time for coding, and new techs like Silex, App Inventor and Google Maps API

What's next for TaxiDaba

To make it more attractive and will make more real tests to stabilize the GPS positioning related functions on my app.
