Inspiration

One of us recently visited a hotel which used a card to turn on lights.

What it does

Tracks meeting room usage with a physical ID card. By inserting the card into the slot while the attendee is using the room, it will turn on the room lights and equipment. The stats connect to a dashboard which shows real-time room usage.

How we built it

  • NodeJS backend
  • MongoDB database
  • Dummy Data Generator
  • React, Redux frontend
  • SSL encryption
  • Custom Domain
  • Tiny Python client for RFID scanner
  • IFTTT Hosted Smart Bulb

Challenges we ran into

  • Getting RFID scanner to read properly (to recognize when a card enters and leaves the field)
  • SSL Certificates
  • Developing and Deploying on different machines

Accomplishments that we're proud of

  • Connecting the smart bulb light to the RFID reader.
  • Creating a real-time dashboard

What we learned

  • A ton!
