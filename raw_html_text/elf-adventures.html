Inspiration

We wanted to create a game similar to other games we have played.

What it does

The character must traverse through a landscape filled with spikes and turret shooters. You have lives, and lose when all of them are gone.

How we built it

We looked at the functionality of sample games, and attempted to reproduce it in our own game.

Challenges we ran into

  • Coding in C#
  • Unexpected Unity errors
  • Unexpected Github errors
  • Teamwork ## Accomplishments that we're proud of We had very little experience using Unity, but we were still able to create a viable game. ## What we learned
  • We figured out how to use Unity
  • We worked with animations and graphics
  • We learned how to develop a user interface that interacts with scripts ## What's next for Elf Adventures We would like to add more levels.
