Inspiration

Supporting the community's most vulnerable during emergency grid events

What it does

The planning tool enables the targeted deployment of low carbon DERs in sites that are most needed, offering value to low income and other vulnerable customers

How we built it

Python API for census data; geolocation data; front-end HTML 5; analyzed data from utility meters and critical facilities to identify practical opportunities using mapping tools.

Challenges we ran into

Time constraints; access to relevant customer data

Accomplishments that we're proud of

Understanding the contexts of resilience and creative options to address vulnerable communities
