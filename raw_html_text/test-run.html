Gamification is one strong word that is opening a whole world of opportunities. I really like doing games with something more to tell. Enterteinment is great, if it comes with any other purpose, like education, it's even better. So it's the idea behind Test Run. I built it using just Unity 5.

The player will be asked to select the correct signs while he/she runs against the clock. By getting it right you earn more 8 seconds. This is an endless game.
