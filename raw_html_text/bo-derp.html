Special prize: MOST ANNOYING HACK

Inspiration

We procrastinate a lot. We think the best way to end it in our lives is to be annoyed by it!

What it does

Uses social commitments on Twitter and annoying reminders to "motivate" us to get things done.

How we built it

We coded it using Python, and Tkinter for UI.

Challenges we ran into

Tkinter is very inconvenient to use.

Accomplishments that we're proud of

Managed to connect to Twitter account. Created really annoying pop-up reminders to pester the procrastinator.

What we learned

We learned how to use APIs, specifically the Twitter API.

What's next for BO - DERP

Improve on the User Interface to make it more visually appealing.
