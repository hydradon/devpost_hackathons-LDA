Inspiration

Alexa skills are very interactive and fun to use. Wanted to explore some avenues and keep the user engaged with a challenging quest. Fun time is the answer for that.

What it does

This is a quest which will ask for 10 questions. Questions where of multimodal. Those having voice-only alexa can hear the questions and answer to it. For alexa devices with visual support, the fun will be presented through images. From the images, they can derive the answer.

How I built it

I used Java, to build the skill. This code runs over the Alexa-skills-kit using APL and deployed in Amazon Lambda Cloud

What's next for Fun Time

In this quest, we followed three varieties of questions. In the future , we can introduce more types of questions, so that the fun quotient will be increased.
