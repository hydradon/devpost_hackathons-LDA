GetOut

GetOut is a mobile application geared towards socializing group fitness, which solves the problem of losing motivation while trying to achieve your goals.

It does this by :

  • Finding people near you interested in the same goals as you
  • Creating suggested events
  • Reminders and group rankings [Optional]

Time spent: 20+ hours spent in total

Open-source libraries used

  • Picasso - Image loading and caching library for Android
  • Firebase - Realtime Database Backend

The following additional features are implemented:

  • [x] Sign Up Screen
  • [x] Events Feed
  • [x] Detail Views
