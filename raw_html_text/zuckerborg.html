Inspiration

In light of recent meme culture, it was apparent that Mark Zuckerberg is not actually human. So we made zuckerborg!

What it does

It's a chat bot with the infamous zuckerborg.

How we built it

We built the site using basic html, bootstrap, javascript, and the cleverbot api.

Challenges we ran into

The cleverbot api we had initially used was inactive, and we had to search online for the updated api.

Accomplishments that we're proud of

We made a sentient zuckerborg that can both text and speak to you.

What we learned

Not all api's are created equally.

What's next for Zuckerborg

Improved css and outreach.
