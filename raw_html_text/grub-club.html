Inspiration

We all love food, we all hate figuring out boring logistics. It was time for a smarter solution.

What it does

Grub Club analyzes data from you and your friends (taste preferences, budget, recent trends) and suggests the best restaurant for all of you to eat at.

How we built it

Front-end: iOS App Back-end: ML Algorithm written in Swift, Google-places API to gather live location data

Challenges we ran into

Not knowing swift, learning XCode app-making, figuring out how to integrate the places API

Accomplishments that we're proud of

The algorithm works flawlessly

What we learned

  • Never underestimate the difficulty of front-end development
  • learn github commands
