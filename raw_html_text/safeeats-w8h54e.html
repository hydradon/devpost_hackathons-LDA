Inspiration

Like most people, I like food, I but do not like bad restaurants.

What it does

Based on user input, finds restaurants that have a health code violation and states exactly what the violation is.

How I built it

I built SafeEats with html, css, javascript, jquery, and ajax.

Challenges I ran into

Putting the addresses into an array for Google Maps to geolocate and map. Took me a bit.

What I learned

I learned more about javascript and jquery.
