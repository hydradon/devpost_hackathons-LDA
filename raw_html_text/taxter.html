Inspiration

We wanted to create a project that uses data analytics, but also be useful and unique to the general consumer. A lot of financial projects involve the stock market, however we wanted to target a different part of money management: taxes.

What it does

Our website displays various statistical models to clearly show the user what they should expect to recieve from their tax refunds. There is even a built in calculator that uses the data from the TaxAct API and used the trends in the data to find an estimate of the average tax refund.

How we built it

The website was built on html and css framework. For the implementation of the models, we used chart.js. For the calculations on the backend we used javascript, fetchapi, and taxactapi.

Challenges we ran into

The implementation of combining the frontend and the backend with displaying the data was the most time-consuming.

Accomplishments that we're proud of

We are proud that we were able to finish the entire project.

What we learned

We learned how to integrate the website with the extracted data and we also learned how to implement interactive graphs.

What's next for Taxter

We wish to improve our estimation analysis. We also wish to expand and also extract new information based off of the graphs that we displayed.
