Inspiration

The inspiration came from the interaction and my engagement to promote access to Justice combine with the wonderfull time and experience I have playing Monopoly with my friend.

While try to understand what exactly or how should I implement this project, I came across interesting quotes which motivated me more :

“Any fool can know. The point is to understand.” Albert Einstein A lack of knowledge creates fear. Seeking knowledge creates courage.” ― Candice Swanepoel “Whoever wants to understand much must play much?” -Gottfried Benn

What it does

Justice Game helps one understand Justice System, Law, human rights, his/her legal rights and duties while having fun.

The game is implemented both in Software and Hard Copy. The software can be played alone but the hard copy is to be played in group with family, freinds....

How I built it

Justice Game is built using Facebook instant game and Html5 technology. The game is hosted on Facebook Infrastructure for game.
