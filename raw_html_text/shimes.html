Inspiration

Safety in Sheffield.

What it does

Provides a heat map for where crime is committed.

How we built it

Using python to process the data and display the heatmap using Leaflet/OpenStreetMaps.

Challenges we ran into

Tried using predictive algorithm to predict crimes ,we needed more data. Couldn't get data to display from crimes outside the city center.

Accomplishments that we're proud of

Overcoming many problems as a team.

What we learned

Learned how to incorporate API and analyze data using python.

What's next for Shimes

App that gives the safest route home , using deep learning to predict crimes.
