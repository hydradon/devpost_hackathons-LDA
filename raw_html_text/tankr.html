Inspiration

Nostalgia for the great tanks game of the '90s.

What it does

You control a tank and have to eliminate other tanks while not getting hit by enemy tanks.

How we built it

Using Javascript, the PhaserJS game engine.

Challenges we ran into

Creating the enemy AI.

Accomplishments that we're proud of

Creating the map and the interaction between all the objects within.

What we learned

How to use the Phaser game engine.

What's next for Tankr

Online multiplayer. Improvements to the map.
