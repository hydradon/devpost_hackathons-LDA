Inspiration

Facebook and Google have been under much fire recently for some of their past decision making. We decided to help make a tool to keep big tech transparent.

What it does

It allows a user to take a screenshot of something they find disturbing, whether manipulative or invasive of privacy, and report it for further investigation.

How we built it

The frontend is a chrome extension built with html and CSS connected to Stitch API as a backend for MongoDB running on an Atlas instance on AWS

Challenges we ran into

Our initial project was determined to be too much for a 24 hour hackathon, so we decided to build this plugin.

The hardest challenge we faced was getting to upload all the data without running Node on the client.

Accomplishments that we're proud of

It works, can scale, and will have a real impact on the world
