Inspiration

Most people may have experienced that standing at the front door finding the door key for a long time. We make this facial detection door to let you get into your house easier.

What it does

Detect if the face is the house's owner and let the owner get into the house and send the alarm if a dangerous person is detected.

How we built it

Build with the flask to set connect between the alarm system (raspberry pi) and the camera and processing system (Mac). Use opencv and keras to train and predict human's face.

Challenges we ran into

Poor network connection makes data transmission very slow.

Accomplishments that we're proud of

Build a facial detection powered by CNN.

What's next for Smart Access Control System

Make a user-friendly web UI and add the face register system.
