Inspiration

We both have participated in online coding competitions and we realized that it was a great way to motivate us to practice coding. We want to help facilitate learning through competitions, especially in mathematics, because there is a lack of online contest resources for math.

What it does

This web app allows users to login and compete in online contests that are created by admins. All participants are able to compete in real time and see results of everyone who participated. Admins are able to create contests for the students.

Challenges we faced

We ran into many issues, but the issue of how to display a math problem correctly to the users bothered us for the longest time.

What's next for QuickMath

We will need to make the web app more visually appealing, especially after users login and fix all the potential issues. We hope to test it with a few local high schools and improve it based upon their feedbacks.
