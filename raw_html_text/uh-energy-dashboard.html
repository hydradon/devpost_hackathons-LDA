What it does

Fdascab reads and interprets csv files given to us by UH's Department of Energy and converts them into easily understandable line graphs. It supports any time interval for any given number of buildings and is dynamic and scalable.

How we built it

We used the Python Kivy package as well as matplotlib to develop this program, splitting the data interpretation and user interface up among our members.

Challenges we ran into

Many of us had never used Kivy before, and so we learnt the intricacies of the package from the ground up. We all came together to help and support each other where we needed it most.

Accomplishments that we're proud of

Creating a working, scalable program that upholds data integrity to the highest degree.

What's next for Fdascab

More advanced features such as anomaly detection, comparative analysis, and goal comparison.
