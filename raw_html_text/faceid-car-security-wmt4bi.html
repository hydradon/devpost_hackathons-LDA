Inspiration

A current trend in the convenience of FaceID in iPhones and any smart car's ability to do more than any another ordinary car.

What it does

Unlocks a car based on FaceID in a web application.

How we built it

Azure Face API to detect and verify a person and SmartCar API to get the data and access to a car.

Challenges we ran into

Having to learn javascript on the go. Connecting to the Azure API in being able to detect and verify people. Integrating various technologies together.

Accomplishments that we're proud of

Working with the various APIs and having them actually work!!

What we learned

Everything we did!!!!

What's next for FaceID Car Security

fully unlock/lock your car with FaceID in a mobile application. We want to expand so that more that one person's face may be able to unlock a car
