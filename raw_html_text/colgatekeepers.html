Challenges

Figuring out how machine learning works towards our design.
Learning web design and the ways it interacts with the back-end of our code
Designing the website to have simple but effective UI
Connecting our back-end code to our front-end code

What we learned

How to use Html to create a website
How to use Github to create a repository
Hosting our domain from Github
We used Flask to allow our back-end code to interact with our front-end code

What it does

Estimates the price of toothpaste based on the brand, country, unit size, and packaging size using data science and machine learning algorithms.

If we had more time...

If we were able to work on our project longer, we would have been able to create a better-looking UI for our program. We also could have connected our UI to our program using the web framework, Flask, since the framework would provide a way to link our front-end code to our back-end code.
