Inspiration

The original creation of this web application was aimed to pass time with a simple (yet, entertaining) game to bust boredom. This web app also sparked the creation of a recreational fitness tracking app for low end smartphones.

What it does

Boredom Buster is a clicker game that counts each click the user makes on the web app and changes color per each click. Simply click until you've busted your boredom. The Extra Mile is a collaborative, recreational fitness tracking android app used to count the number of steps taken straight from your smartphone without the need of expensive phones or devices. The user can start and stop the step counter at will. The Extra Mile also allows the user to share the amount of steps taken via the share feature on their smartphones.

How we built it

Boredom Buster was built using Javascript and HTML on Sublime Text 2. The Extra Mile was tested using Android Studio before being finalized on AppInventor with the help of Java, Kawa, and XML.

Challenges we ran into

Unable to add the game Boredom Buster to the finalized application "The Extra Mile".

What we learned

Developing collaborative projects require a lot of teamwork and dedication to achieve a common goal.

What's next for Boredom Buster

Further expansions! We plan on adding more entertaining and recreational functionalities to the applications and possibly uniting them onto one platform to spread across many.
