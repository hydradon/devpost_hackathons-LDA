What it is

A typical smartphone that we carry today can read, detect and store a lot of data about us. We thought wouldn't it be wonderful if there was this technology or an app, that like your personal autobiographer notes down everything you do, documenting every minute of your life.

Technologies & APIs used

It collates most of the offline data from the android smartphone, such as call logs, IM, contacts & Calendar. Google Fit API Google Photos Facebook API Twitter API Plaid API Uber API Soundhound's, Houndify API Two Sigma Beaker API
