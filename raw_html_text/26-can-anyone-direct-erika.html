Inspiration

Have you ever felt tired of someone and needed to get away? This game shows you just how impossible it is, because inevitably that person will always catch up to you. Seriously. It's impossible.

What it does

As the player, you use your mouse to direct the female character away from the male character, but be careful, because the longer the game runs, the faster he runs.

How we built it

Unity and VisualStudio, using C#

Challenges we ran into

Until today, nobody had ever built a game or used Unity before, so everything we did was newly learned today. It was difficult to understand Unity's game elements, capabilities, and how to implement them, so we spent most of our time researching and trying things out.

Accomplishments that we're proud of

We got a game to work!

What we learned

We learned how to implement the basics of Unity using C#, as well as how to utilize available resources and tutorials to teach ourselves. Basically, if you do anything for 12 hours, you'll learn something.

What's next for (26) Can Anyone Direct Erika?

In the future, we would like to add multiple levels and add new features like an exit and mystery boxes that will help/hurt Erika's escape. Also, we hope to make both the player and enemy characters' appearance more customizable.
