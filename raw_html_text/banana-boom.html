We chose to build a retro game like the beloved ones that we grew up with, and learned a good amount of Swift along the way.

Banana Boom is an arcade game where you have to dodge the bananas and collect coins as you go. The game is integrated with GameCenter, allowing you to post your high scores to a global leaderboard.

The game mechanics are simple, which means no learning curve, and instant fun!

We plan on adding power-ups and other vintage features in the future. As soon as that's done, we plan to launch the game to the App Store for everyone to enjoy!
