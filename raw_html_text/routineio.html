Our teammate's cousin has a learning disability. We wanted to see how we could make a positive difference in his life.

It helps young children with learning disabilities to create habits and routines that will allow them to become more independent individuals.

We used HTML, CSS, and Javascript to build the front end, and used Java to build the back end.

It was difficult integrating Javascript into the web application, and required a lot of debugging.

We are proud that we were able to develop a feasible prototype and hits all the needs that families want in an app like this.

We learned a lot about the ideation process and the importance of UI/UX design through various workshops.

We plan to continue fundraising efforts to help fund the creation of the mobile application, and to hopefully launch it within the next year.
