Inspiration

Kaggle Data Science competitons where your knowledge of algorithms can give you an edge in the competition

What it does

Train on a real and live dataset using popular machine learning algorithms while being able to control their hyperparameters

How we built it

We used Flask for the backend to processes user input to feed it into the machine learning algorithms

Challenges we ran into

Front-end design and routes between the front-end and backend. First time using machine learning in the context of the web

Accomplishments that we're proud of

Made within the last three hours of hacking. Lots of features and we learned a lot about frontend design.

What we learned

Front-end design and certain parameters for machine learning algorithms. Doing machine learning on the web

What's next for RandomWalk

Better UI design, expansion to the class-room, and possibly multiplayer support for friendly competitions.
