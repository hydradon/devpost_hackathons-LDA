Inspiration

Both of us live in the same student dorm with a big room of washing machines in the basement of each house. It's a pain in the ass when you go downstairs just to see all of the machines occupied.

What it does

It shows the state of each washing machine and dryer in realtime.

How I built it

With the Sens'it Kit we test the state of each machine. This state will be send through the SigFox Network to a backend. The client webpage will get HTTP Requests with the state of each machine and visualizes it.

Challenges I ran into

Working with a MCU without deeper knowledge of electro engineering.

Accomplishments that I'm proud of

Learning how to use the SDK and controlling all the small units inside the Sens'it.

What I learned

Basics of IoT and MCU.

What's next for BobTheWaschmeister

Pitching it in the students dorms.
