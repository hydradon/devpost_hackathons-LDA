Inspiration

Received few suggestions to enable features available in ALM task and UFT Task to other tools and frameworks.

What it does

This plugins adds action in Miscellanous tab of build in Bamboo. It lets build to take actions on test results created in Bamboo . Following are the actions available

Create JIRA bug for failed tests

Create test result links to JIRA issues

Add run comments , execution duration to custom fields of JIRA Issue

Add labels passed / failed to JIRA issues

Pass / Fail build for test failures
