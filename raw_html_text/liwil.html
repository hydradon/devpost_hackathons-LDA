Inspiration

Imagine, what if teachers could teach kids the way they ought to be taught - how will it transform the way the world learns?

What it does

We match teaching methods according to the students' personality, optimizing the way students absorb and process information.

How we built it

By analyzing text from previously written student assignments (without having to run personality tests in class), we are able to identify students' personalities into the big five dimensions - openness, extraversion, agreeableness, conscientiousness, and emotional range.

Challenges we ran into

Using the available research to implement our model to relate personalities to learning styles to teaching techniques. Combining different technologies into a single solution.

Accomplishments that we're proud of

Manage to integrate different technologies into a single solution.

What we learned

By using samples of speeches, we are able to trace personalities of historical people.

What's next for Liwil

Integration of Microsoft Office 365 API. Data mining for more accurate teaching method recommendations.
