Inspiration

We were both interested in productivity because we both used to procrastinate a lot and would like to help those who still do by giving them quick access to resources that will help their study.

What it does

Gives quick access to many school/work tools such as Google Suite and good study music choice to enhance productivity

How we built it

We used HTML and CSS for the majority of the web page design.

Challenges we ran into

We were challenged with making the enter key as an input that would register the events we were aiming for (output) like making the task enter the to-do list.

Accomplishments that we're proud of

Implementation of a pomodoro timer

What we learned

Since this was our first hackathon, we learned a lot about the rigor of making a project. We also learned that working together on the same problem, but different task can accelerate the process of a product. We also learned/reviewed different languages such as HTML, which will help our future endeavors.

What's next for Productivity Machine

Add audio and more implementations/features such as calcuator, etc
