Inspiration

After came up with many ideas, we decided to make an application to provide useful information to the people who want to go to the beaches in Chicago.

What it does

It provides statistical data about a certain beach such average temperature, wave height or wave period within a time range.

How we built it

Built on Android-studio. It parses data from the built dataset and plots it to the Google Map.

Challenges we ran into

First time using Google Map API in Android Studio. It very hard to find good dataset.

Accomplishments that we're proud of

Provide needed information to the people who would like to go to beaches.

What we learned

Learn how to use Google Map API and many functionalities of Android Studio.

What's next for Check Beach

Find better dataset and make a query for more detailed information and more beaches.
