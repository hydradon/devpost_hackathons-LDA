Inspiration

We were inspired by Virtual Reality and Game Development Club.

What it does

This is a game in which the player puts actions into a queue to help guide the game avatar.

How we built it

We used Unity and our own assets.

Challenges we ran into

Most of us were new to Unity, so we had to learn a lot along the way.

Accomplishments that we're proud of

We're proud of the assets we created for the game and the fact that while we only created two levels, we have the foundations to make many more.

What we learned

We learned a lot about using Unity.
