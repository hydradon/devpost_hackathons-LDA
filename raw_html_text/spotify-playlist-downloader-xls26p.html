Inspiration

We saw that you needed spotify premium to download a playlist and though that there was a better way.

What it does

It downloads mp3 files from a spotify playlist

How I built it

It scrapes the names of the songs from the playlist link and then uses a tool called youtube-dl to download each file from youtube.

Challenges I ran into

Getting the scraping to work and integrating youtube-dl with the java application were hard.

Accomplishments that I'm proud of

Making it work

What I learned

How to run terminal commands from java

What's next for Spotify-Playlist-Downloader

Making it more usable, because currently it only works when run from eclipse and when youtube-dl is installed in the exact right place.
