Inspiration

mTOMADY’s goal is to give everyone access to universal healthcare. The project helps people in Madagascar paying for medical treatments by providing a mobile phone billing interface. In the test stage the program focusses on pregnant women.

What the code does

During pregnancies there are a lot of necessary examinations, the most important ones being ultrasounds. We created a reminder tool that automatically sends SMS to the phone number associated with the patient-id. Our code allows seamless integration of other kinds of examinations and their recommended time frames.

How we built it

The code generates a reminder-queue including the patients' last medical appointment and automatically sets dates to remind the patient by SMS. The messages are then send using the twilio API.
