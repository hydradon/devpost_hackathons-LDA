Inspiration

We wanted to build a webapp that allows you to instantly make plans with a group of friends, and eliminate the hassle of trying to find a time that works for everybody.

What it does

It allows you to see your friend's availability so that you can find the best time to make plans.

How we built it

We built the webapp using React.js, it is being hosted on Firebase and also implements Firebase authentication.

Challenges we ran into

Getting React and Firebase to work together properly for hosting, setting up authentication with Firebase, and getting certain parts to render properly.

What we learned

Improving Git skills, Learning React, getting better with Firebase, and making something that actually works.

What's next for Freetyme

Implementation with Google Calendar and Apple Calendar
