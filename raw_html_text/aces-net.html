Education: Students can create an account, add classes to their profile (pulled from local json files from the CoLab curriculum API), search for users, send friend requests, and view friends' classes.

Made with Rails 4.2, MySQL, uses public gems, JQuery, Bootstrap, ..., runs on Apache.
