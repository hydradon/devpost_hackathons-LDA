Inspiration

YouTube-hosted hackathon and YouTube Data API. Wanted to allow uploaders to customize their video pages with their own art.

What it does

Allows custom art, colors, fonts, and icons to be used as UI for youtube videos.

How we built it

We made prototypes and wireframes for the web application. We used HTML, CSS, and Bootstrap for front-end development. We used JavaScript, JQuery, and AJAX for back-end development.

Challenges we ran into

Connecting and retrieving data from the YouTube API. Not enough time. Inexperience with JavaScript and other languages.

Accomplishments that we're proud of

Learning web development technologies. At least there's something :)

What we learned

Learning web development technologies. Working as a team. Using GitHub. Video customizations for YouTube. YouTube Data API.

What's next for YouArt

Finishing the customization feature. Adding Google APIs for image searching.
