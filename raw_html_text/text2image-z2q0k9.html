Inspiration

We were inspired to create a project that contributes to society and help those who are in need of assistance. We decided to help those who have trouble reading.

What it does

Text2Image changes the keywords from a body of text into images, to give a better understanding to those who have difficulty reading.

How I built it

We used the Indico Text Analysis API in Python in order to pull out the keywords in the text. We then used Flask and JavaScript to develop the webpage that allows the user to past in his text.

Challenges I ran into

We originally had no experience in using Flask in order to make webpages.

Accomplishments that I'm proud of

We successfully managed to incorporate JavaScript with Python in order to get the webpage working.

What I learned

We learned how to use Flask as a framework.

What's next for Text2Image

We plan on turning Text2Image into a Chrome extension to make the app more accessible and user friendly.
