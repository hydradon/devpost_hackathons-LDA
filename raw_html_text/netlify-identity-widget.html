Inspiration

It is a cool website, we wanted to contribute to open source projects that we can personally use in our programming career.

What it does

The widget is a family member of a group of Netlify Modules that allows developers like us to deploy websites with a SINGLE click of a button.

How we built it

Specifically, we contributed to the custom name placeholder inputting logic.

Challenges we ran into

Preact, one of the core dependencies of this module, aims to remain a fast 3kB alternative to React with the same modern API. This provided a unique design challenge in implementing the logic. Due to the lack of ability to import new helper modules, we were forced to use existing, light-weight modules to circumvent the challenges.

Accomplishments that we're proud of

We enabled developers and end-users to cross the language barrier when using this widget - commonly included as a part of other modules in the Netlify Family!

What we learned

Don't spend half of the hackathon trying to post memes!

What's next for netlify-identity-widget

More QoL changes, and auto-filling passwords implementation
