Inspiration

We talked to some people who volunteered with the latest Hurricane Harvey crisis and learned a big problem with today's relief efforts, volunteer coordination.

What it does

Our app uses the blockchain and mesh networking to build an interactive map that can persist without internet connectivity. It allows volunteers to upload audio snippets, and mark hazardous road conditions; and it allows victims to request help.

How we built it

Using Java, Mapquest, and Hashgraph
