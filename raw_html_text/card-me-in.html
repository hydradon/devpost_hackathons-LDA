Inspiration

Last year, we had too many meal points, and this year, we wanted more meal points. As a busy student, cooking is hard.

What it does

Users can create an account and advertise their meal points.

How we built it

We built this using Flask.

Challenges we ran into

Learning Flask took some time. Creating a finalized design was difficult.

Accomplishments that we're proud of

We learned a lot about Flask.

What we learned

We learned how to develop a web application with Flask that has different endpoints and stores user data in a database.

What's next for card-me-in

We would like to expand the functionality of our app by allowing users to see other users' ratings as well as allow communication and a method of payment between users.
