Inspiration

Watch public live streams from all over the world in a single feed

What it does

Ability to search for any public live stream, by name, category, country, city or mix. Ability to play back old live streams at a specific moment, category, city...

How I built it

I have first built the back end which includes the PHP scripts, APIs, database. Then started building the Android application, it reads the user commands and output a feed of the latest videos as result to his search. Plus a TV showing the most popular public streams at the moment the user is using the app.

Challenges I ran into

When I started developing my idea, i have built a Web app since I had no clue in Android developing. I challenged my self to have the same idea on my mobile, this was the time I have learned Android development.

Accomplishments that I'm proud of

A new "YouTube"* for Live Videos called MJTV

What I learned

I learned to fail forward and never lose hope in building something, until one day it will be a whole live system.

What's next for World Live

Build a platform to let user feel and live the moment from another place with interaction between streamer and audience, for any meeting, party, conference, free talk,...
