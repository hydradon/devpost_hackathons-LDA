We wanted to create a fun game in a Virtual Reality environment and we like battling with each other, so we compete with AR and a built in AI! We integrated the Capitol One api for a bank account menu that could also be integrated for micro payments in the future.

Due to the large size of the files in the VR environment, it is only available on github for now. Just download the Bot Beatdown_Data folder and Bot Beatdown.exe and run the exe to play on your own!
