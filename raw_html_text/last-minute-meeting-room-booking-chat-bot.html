• Chatbot Last Minute

Inspiration

Inspired by the problem to book meeting rooms discussed at T-Mobile presentation.

What it does

Easily get access to a non-occupied room by texting “Book” and “#” of the room.

How we built it

We used Twilio studio to define chatbot flow. It was connected to the phone number but can be connected chats like messenger, hangouts.

Challenges we ran into

We tried to use Jazz in order to receive webhooks from our chatbot. But we got format problems (XML vs. JSON)

Accomplishments that we're proud of

Solution chatbot is easy to use with mobile phone, no additional devices required.

What we learned

We have used platforms such a Twilio, Jazz, AWS Lambda for the first time.

What's next for Last Minute Meeting Room Booking Chat Bot

We are going to integrate it into the Outlook/Exchange/Google Calendar
