How We Made It

We leverage Microsoft’s computer vision API and its nifty caption creator.

Why We Made It

As lovers of web development, we know that conversations about accessibility often get pushed aside in favor of ones about the latest javascript library or how to make a page look awesome. It’s unfortunate, but we hope that our speedy extension might make the problem of un-tagged images a little better.

An Example

Let’s say we have an image embedded on a website like this one:

The HTML might look something like this: 

<img src="pug.jpg" alt="dog">

But with our Chrome extension, it will be updated to be this: 

<img src="pug.jpg" alt="Original caption: dog; Generated caption: a dog looking at the camera">
