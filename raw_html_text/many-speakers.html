Inspiration

Not having portable speakers to use during school athletic events.

How it works

By using a computer on the same network, each phone connected will download each song and play at the same time!

Challenges I ran into

While our original idea was to have one phone act as the server and have all of the clients connected directly to it, we continually ran into the problem of not being able to transfer packets between phones causing us to abandon that idea.

Accomplishments that I'm proud of

File browsing and filtering as well as getting the audio files to play perfectly together.

What I learned

We learned how to implement the framework for android apps at a competent level despite not having any prior app building knowledge.

What's next for Many Speakers

The first goal will be to figure out a way to completely remove the computer from the setup and allow it to be run in its entirety user's phones.
