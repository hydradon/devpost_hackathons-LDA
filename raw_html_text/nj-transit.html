Inspiration

I was inspired to build this skill because their was not another skill that would use APL with their train schedule skill.

What it does

It gives you times for the train station you ask for along with displaying popular train stations to ask Alexa for the next train time for.

How I built it

I built the skill by using JavaScript and connect the NJ Transit API to get update train information for every station.

Challenges I ran into

The biggest challenges was getting the NJ Transit API to work consistently.

Accomplishments that I'm proud of

I am proud of submitting this skill in to the Multi Modal Challenge.

What I learned

I learned the kind of questions you need to ask yourself when building a Alexa skill with a screen but having it voice first.

What's next for NJ Transit

I'm looking forward to updating the skill overtime. I'm planning to make the skill personalized based on location and the stations each users visits the most on the skill.
