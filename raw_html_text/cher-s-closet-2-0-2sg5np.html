Inspiration

Growing up, Clueless was my favorite movie. And I always envied Cher's electronic closet system that catalogued her entire wardrobe and allowed her to preview outfits so she didn't have to physically try clothes on. I can't believe that we were able to code a program like this ourselves!

What it does

The app allows you to have a catalog of your own clothes, which you can put together to create a visual of what the different pieces would look like in an outfit.

How we built it

We built it using Xcode and Swift.

Challenges we ran into

Our biggest challenge was the time constraint. We originally wanted to be able to upload our own pictures using our phone gallery but we ran out of time. Another thing that we wanted to was to be able to tag the clothes with their color or style, but again, we ran out of time.

Accomplishments that we're proud of

We made a working program and iOS app using Siwft.

What we learned

We learned how to Xcode and program with Swift in one night.

What's next for Cher's Closet 2.0

Hopefully we can try to implement the desired functionalities that we didn't have time for in the future.
