Inspiration

People often want to eat at food trucks. However, since the food trucks often move around, their only way of updating their location was through twitter or Facebook. This is not only inconvenient to the food trucker but also a hassle to the clients. Thus we came up with the idea of creating a website that food truckers could submit their location to so that clients can easily find them.

What it does

It uses geolocation to allow foodtrucks to share their locations with customers.

How we built it

We used HTML5, CSS, and Javascript to build the front end webpages and Python, SQLAlchemy, and Flask to create the backend database.

Challenges we ran into

We struggled to save information to and access information from the database.

Accomplishments that we're proud of

We successfully implemented Google Map's API and a SQLAlchemy database.

What we learned

We learned the start to finish process of creating a webpage and creating a database that interacts with a website.

What's next for Find my Foodtruck

We would like to add secure logins for foodtrucks, online hosting of our webpage, a search function, and ultimately turning it into an app to allow easy access to multiple food trucks' locations.
