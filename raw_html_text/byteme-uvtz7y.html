A mobile App focusing on analytics, forecast, trends, marketing and fraud detection. Customers will be motivated to store their receipts and to take the receipts’ surveys. Stored receipts will facilitate products’ returns and exchanges to improve user experience.

By making surveys ‘fun,’ this app will increase customer’s interest in taking and completing surveys that companies, as Walmart, have in place in their receipts.

By connecting a banking API as NCR’s, can add an extra layer of security to customers; strange transactions can be flag in a very early stage.

By connecting a transaction API as NCR’s, we can track customer behavior and print receipts more targeted towards each customer.

Also, customer's can do their login through their Facebook API to facilitate UX.

For that reason this mobile App will be a great user interface for companies who wish to add more value to their business transactions.
