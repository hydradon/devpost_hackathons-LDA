Inspiration

Our team has a passion for security and innovation. We decided to combine them into a novel way of increasing security.

What it does

There are 2 main features, 1 adds a layer of security over the normal password entering, by analyzing how the person enters the password. This means that is the password is lost to a malicious user it will significantly reduce the usefulness for accessing the account. The other is the change/reset passwords by asking the user to type a predetermined phrase, and using the speed/typing flow to analysie who is typing.

How we built it

We used python to quickly develop a system. and machine learning to set the threshold and weighting values.

Accomplishments that we're proud of

The paragraph recogniser managed a 70% accuracy with no false positive on our small dataset of about 30 attempts. The password enforcer also showed success but we did not hae the time to test its efficiency or put it into a nice format to display and use.
