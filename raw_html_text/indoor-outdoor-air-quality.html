Inspiration

We all breathe air, its one of the most essential resources we have. Therefore it is crucial to monitor air quality more actively

What it does

A portable sensor that can be used to gather air quality data across a variety of locations.

How we built it

First we built the sensor hardware, then we added a gps sensor and power unit. then we built a visualization engine, and setup redis for livestreaming data. Then we went outside to get live data. We can display this data, and any other data we collect live.

Challenges we ran into

Integration and real time updates
