Inspiration

Walked up to a convenience store to get some food, but I left my card at home... So I had to starve until I went back to my house.

What it does

recognizes your face on the database and connects with payment API to pay for things.

How we built it

React JS for front end, Firebase and microsoft azure API for backend.

Challenges we ran into

azure api was hard to use a bit

Accomplishments that we're proud of

we made it work! and it's awesome :D

What we learned

that convenience in our lives can also be used to solve a worldwide problem.

What's next for PLINK

make it into a legit platform so that people can start integrating in their business or places like homeless shelter.
