Inspiration

Being college students, we are always on the lookout for a good deal when we go grocery shopping and saving as much money as possible. Scouring sites for the cheapest necessities is a huge time drain with our schedules so we decided to automate it instead.

What it does

SwampFuud (Food Under UIPath Determination) uses robotic process automation to look for the cheapest version of the items on your shopping list, with the click of a button.

How We built it

UIPath to automate the search, Firebase backend to save user lists and data to the given user, and React frontend to set up the user profile and take in shopping list

Challenges We ran into

UIPath software is not consistent and is very buggy - restarting/breaking with little to no explanation. We had to work around our entire automation process being destroyed after one such occurrence, with many bumps along the way. It was also the first time we worked with hooking everything up to a serverless backend with FireBase.

Accomplishments that We're proud of

Figuring out how to get UIPath to work and get data from the webpages

What We learned

How to work with UIPath and how to use firebase, connecting everything from end to end

What's next for SwampFuud

Expanding automation for more stores to be supported Voice integration to get your chores done, hands free with InstaCart.
