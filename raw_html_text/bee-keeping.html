• Bee_tracking

Inspiration

I took a bee keeping class for fun at UCR. In my studies I came across the zombie fly and wondered if we can keep track of honey bees that are infected.

What it does

Tracks bees that go in and out of the hive.

How I built it

I built the program using computer vision.

Challenges I ran into

A big challenge I ran into is counting the bees because of the shadows, colors, and the time of day.

Accomplishments that I'm proud of

I was able to count some of the bees.

What I learned

After working with computer vision I learned that a better approach to this would be to use machine learning.

What's next for Bee Tracking

I want to work on this project again using machine learning. The first thing I want to do is keep track of the bees from morning until the evening. For night time I want to use heat sensing to keep track of the bees that could possibly be leaving at night. I want to further extend this project by detecting enemy's that want to enter the hive such as wasps, beetles, and moths.
