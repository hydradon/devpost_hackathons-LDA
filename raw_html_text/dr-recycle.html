Inspiration

Americans only recycle 34% of the waste we create. We wanted to do our best to help mitigate that number!

What it does

It's a Facebook Messenger Chat Bot that offers ideas resources and ways to recycle things best! By helping people find the best way to recycle things, we can help reduce items that go to the landfill.

How we built it

We used Clarifai, Wit.AI (NLP), Python and Facebook Graph API to build our application from the ground up!

Challenges we ran into

Training Efficiency of our models. Porting the default Node-JS Facebook API code to Python.

Accomplishments that we're proud of

Image Recognition, Diversity of cases we cover, User Experience

What we learned

Lot of Image Recognition and NLP stuff we weren't aware of before!

What's next for Dr-Recycle

Scale it to more options, more users, improve recognition accuracy.
