Inspiration

A family member of mine wanted an application they could use that would notify them when a listing had been created for an item they wanted on Craiglist. As a result, I made Craigslist Companion!

What it does

With Craigslist Companion, you can create your very own Companion that will search for an item you want. It will let you know when new items under your search criteria has been posted so you don't need to check constantly.

How I built it

I created a Java program that handles the companion's tasks and created a simple, but elegant website with HTML/CSS/PHP.

Challenges I ran into

I was hoping to make this a public website for everyone to access, but unfortunately that would go against Craigslist's terms of service. Basically, this is just an idea for them to potentially incorporate in their own site (or something that could be applied to another site like eBay).

What I learned

This project really allowed me to explore MySQL databases more and get more experience with interaction between PHP/MySQL and Java/PHP.

What's next for Craigslist Companion

I plan to incorporate an email function to this so users don't need to check the website every time. This defeats the purpose of the site. Adding email isn't too complicated, so I opted to spend my time working more on other elements.
