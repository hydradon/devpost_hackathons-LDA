Inspiration

I have always enjoyed listening to TCNJ's WTSR 91.3FM radio station, but there has never been a way to check the currently playing song other than listening very closely for when the DJs mention the song names or having to constantly use Shazam or SoundHound. There does exist a web player for WTSR, but it also does not provide song information. I wanted to make a WTSR web player that does.

What it does

  • WTSR 91.3FM real-time playback with playback and volume controls
  • Displays currently playing song
    • Name
    • Artists
    • Genres
    • Spotify link
    • YouTube link
    • Deezer link

How I built it

Languages and Frameworks:

  • JavaScript
  • Node.js
  • Gulp
  • HTML
  • CSS

Services:

  • Firebase
  • ACRCloud
  • Spotify

What's next for WTSR 91.3FM Web Player

Currently awaiting funding for the ACRCloud service from WTSR! Until then the website does not display current song information because the ACRCloud free trial has expired.
