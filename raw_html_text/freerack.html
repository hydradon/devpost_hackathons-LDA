FreeRack

It's a non-profit mobile and web application which helps the community identify and pick up kerbside items. It aims to solve the problems faced by people regarding disposing of the household items.

How it works?

The user has to successfully authenticate him/her before posting an ad about an item as well as get notifications based on preferences set by a particular user. The user can post an ad by clicking an image of the item he wants to giveaway and his/her location.

Future Scope

We are planning to include user analytics and pattern recognition to identify certain items available in a particular suburb/town which will advocate the user in finding an accurate product as per area in a short span of time.
