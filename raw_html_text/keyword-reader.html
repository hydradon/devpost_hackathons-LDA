Inspiration

The QR Code to join HackHershey. It inspired us because we did not have a QR reader on us to read the QR code at the time.

What it does

This program opens URLs from given keywords, either from scanning or manually typing them.

How we built it

We build it using Java, Eclipse Java IDE, and Eclipse WindowBuilder.

Challenges we ran into

Dependency issues and compiling into a jar file

Accomplishments that we're proud of

Finishing this program

What we learned

We learned how to utilize OpenCV, Eclipse WindowBuilder, development in a group, and how to utilize an OCR.

What's next for Keyword Reader

Possible improvements could be a cloud-based database so that multiple applications can share URLs. Mobile applications are another potential addition.
