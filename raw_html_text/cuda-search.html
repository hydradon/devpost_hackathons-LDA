Inspiration 

 As first-time Hackers, we wanted to build something that would empower us to continue our journey in technology. A search redirector seemed to be a great idea.

What it does

Cuda Search is a basic web page that redirects searches to Google.

How we built it

We used javascript to create the animations by some for loops, classes, functions. Then we used HTML and CSS to create the web page.

Challenges we ran into

The lack of knowledge in many new concepts and implementing multiple languages to a single project.

Accomplishments that we're proud of is coming to this hackathon. 2/3 members of our team members are attending to a hackathon for the first time. Also, many of us have started to code less than six months.

What we learned

Coming to this hackathon we learned many things. We were able to compete in a competitive environment, which is important in the tech industry. By far the most important idea is working with a team effectively, we were able to get a great start in it in this hackathon.

What's next for Cuda Search

We are planning to refine Cuda Search and have a stand alone, fully functional web search engine.
