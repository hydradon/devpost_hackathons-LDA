Inspiration

We like sharks, and we like fantasy, and we like role-playing games. Also Shayla had a ukelele and wanted to write a song about sharks.

What it does

This is a choose your own adventure that allows users to submerge into the identity of THE SHARK KNIGHT, the hero the ocean needs, but not the one it deserves. It allows you to experience the majesty of the open sea,

How we built it

Java and Photoshop and Google

Challenges we ran into

Visual layout and design - Sharks are very large, computer screens are very small. Choosing a feature set that was within the scope of our abilities.

Accomplishments that we're proud of

We created a fully featured game with graphics, branching paths and multiple endings in nineteen hours.

What we learned

Puffer fish are tricky little buggers. We figured out how to collaborate using Google drive.

What's next for THE SHARK KNIGHT

The Shark Knight will return in THE SHARK KNIGHT RISES
