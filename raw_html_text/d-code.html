• Flappy Bird D-Code Example

Inspiration

I got my inspiration from teaching my cousin to code. It wasn't until I got started that I realized that there were many aspects to java that required a large learning curve. That was when we decided that we wanted to deconstruct one of the most popular languages and make it simpler.

What it does

Rather than having to go through the struggles of learning a programming language, we provide a user friendly interface that makes the coding the experience more pleasant.

How we built it

We started off coding the pseudocode that java interprets and slowly moved on to the UI. Then we moved on to coding concrete example games using the pseudocode. After that we worked on created a descriptive and complete help menu so that the user feels comfortable using the program. Finally we created a index.html to supplement the the program. From the website you can download the program to your computer.

Challenges we ran into

We wanted to created a "mesh"/networking ability in our program but ran into difficulties making it work.

Accomplishments that we're proud of

We are proud of creating a simpler way to code and interpreting our own "language". We are also happy that the user can interact with a aesthetically pleasing html file.

What we learned

We learned that you can learn and accomplish a lot in 24 hours if you set your mind to it and work really hard.

What's next for D-Code

We hope to add networking abilities later and create a well designed website.
