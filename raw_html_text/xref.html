In today's world we depend so heavily on the internet to inform our judgements and knowledge. However with this access to information comes the difficulty of discerning the accuracy and reliability.

We worked on a Chrome extension that analyzes articles and pulls news articles with similar content to allow instant cross referencing and improve how we engage with content.

It was built using HTML, CSS, Javascript and Python (for Flask server) and was a great learning experience for us all. Some challenges included getting the Flask server to actually link to the front end and formatting data being passed to endpoints.

We used the Google Cloud NLP API and Microsoft Azure Bing Search API.
