Inspiration

Exams. Co-op. Extra-curricular activities. Hackathons. Being a millennial today can get extremely busy and making time to do what we're passionate about can be difficult. We have always been interested in investing but it is hard to find the time to research trends and watch the market for when to buy and sell. So in comes Coincise.

What it does

Coincise is an investment management platform that allows consumers to buy and sell Bitcoin, Ethereum, and Litecoin using the Coinbase API based on recommendations given by a deep q-learning agent. Our website displays charts that illustrate the bot's forecasts and compares the predictions with actual market value.

How we built it

With ingenuity, we applied UI & UX principles to create a website that would appeal to our target market. In the bankend, we use Tensorflow to build a recurrent neural network and the agent trains itself with data from GDAX using reinforcement learning.

Challenges we ran into

Gathering enough data, using new languages such as Python, and connecting the front and backend.

Accomplishments that we're proud of

Our bot can accurately predict the market trends.

What we learned

We learned how to work together as a team and manage our time effectively to finish our project while still having fun.

What's next for Coincise

We plan to implement sentiment analysis to improve the algorithm used to project future market values. In the short term, cryptocurrency prices are effected by the media and herd mentality so performing sentiment analysis would help give our consumers better recommendations.
