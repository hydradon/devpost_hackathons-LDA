Inspiration

Our project was designed to reduce the Gender/Racial/Age discrimination in the hiring process.

What it does

It takes a résumé pdf and uploads it to Google Vision API where it performs optical character recognition on the document. It then passes it to the AWS Comprehend API which uses natural language processing to categorize the personal information like name, gender, age. Then it calls the image processing API that we developed which censors the personal information on the résumé.
