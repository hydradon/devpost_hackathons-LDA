PlasticBeach

We are from Aston University and we created TrashTag. It is an app which allows you to report when you see some trash. As well as that if you want to volunteer to clean up, you can accept it and it will provide you with the location information, therefore you can go and clean it up. As well as that if the job is impossible for one person to do, you can choose how many people can volunteer.
