Inspiration

So many children are trafficked into sex industry, this serverless application aims to prevent that by reporting any child nudity being streamed live on the client's AWS kinesis resources.

What it does

So many illegal sex rings use technology to live stream child porn from webcam, Nsfw protector blurs out any nudity in live video chart and sends a report if that chart contains nudity by a minor. The report is sent to law registered organisations via email with the embedded video frame
