Inspiration

We were inspired by seeing the problems that we face in our own communities that could be resolved with more infrastructure to facilitate better cooperation amongst its members.

What it does

HyCP gives members of a community the power to create positive change in their area by giving them the tools to organize and gather funding to support local projects proposed by community members.

How we built it

We used Python for back-end and React for the front-end with a postgresql database.

Challenges we ran into

Using React framework for the first time and linking it with the database.

Accomplishments that we're proud of

We're proud of developing a much deeper understanding of the popular javascript framework, React, and developing a fullstack web application intended to benefit the community.

What we learned

How to use React framework.

What's next for HyCP

Better graphic design, user interface, and providing more utility.
