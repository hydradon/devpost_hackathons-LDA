Inspiration

We noticed that not all buildings are accessible for people with disabilities like blind people. So we thought if we could create a software that can guide people to certain areas such as the bathroom, and the entrance it would make peoples lives easier.

What it does

A Guide is a program that listens to where a person wants to go and then tells them how to get there. We can use this in universities and hotels.

How I built it

Python and Google Cloud. We mapped rooms using a robot.

Accomplishments that I'm proud of

Used Google Cloud API and used Python

What I learned

Google Cloud API

What's next for Guide

We would like to use it to fit the needs of specific groups such as Universities, Hotels, and Hospitals.
