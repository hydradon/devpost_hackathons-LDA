Air Thoughts

Inspiration

We want to identify and leverage customer sentiments for American Airlines customers in real time.

What it does

Displays sentiment analysis of live conversations. Parsing keywords to understand the conversation topic and the sentiment associated with the topic.

How we built it

React - Front end Flask - Back end MongoDB - Persistence

Accomplishments and What We Learned

Gained more experience with API development, how React/JS works, and applying database knowledge!
