Inspiration

I wanted to be able to build a bot that could take decisions for the group. I always hated that we could never agree peacefully to go to a particular place.

What it does

The bot fetches results from Google Places API and then using voting from users, it decides a place for them. You can search for food places, adventure sports, etc.

How I built it

I built it by using google places API and the documentation provided for the challenge.

Challenges I ran into

It is difficult to return the results of the polling as AWS lambda kept timing out.
