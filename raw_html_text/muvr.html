Inspiration

Realized that making music can get very expensive.

What it does

Creates Virtual equipment for music production with insignificant fixed costs.

How I built it

Used Unity Game Engine for the main development and Vuforia Unity SDK for the Augmented Reality integration.

Challenges I ran into

Using Unity for the first time, making the UI/UX intuitive, embedding AR into Unity and finally making it work on the iPhone.

Accomplishments that I'm proud of

Very basic prototype, but it feels great use. It's very new, but it already has a lot of potential use.

What I learned

Learnt that software development is not just about programming; one spends a lot of time learning about the configuration of different technologies so that they can work in unison.

What's next for MuVR

Introduce new equipment to the music platform.
