Inspiration

This is inspired by the Github Project BullshitGenerator, a web app that generates essay of thirteen thousand words to deal with the unreasonable requirement of withdrawing from a student council in China.

What it does

The web app generates chat reply with user-input key words and word limit number.

How we built it

We built it using Python Flask and run it on Google Cloud Platform.

Challenges we ran into

Learning Google Cloud Platform and Python Flask from scratch as first year student.

Accomplishments that we're proud of

What we learned

Python Flask and how to use Google App Engine
