Inspiration

We often spend too much time finding a place to park. This is a common problem in metropolitans and tourist places and it often becomes a headache to find a public parking. We aim to solve this problem by providing a platform where people can host their driveways or lawns or garages for temporary parking which can be requested by other users. Win-Win!

What it does

A user can host his/her driveways or lawns or garages for temporary parking on the application. Another user who needs to park his/her vehicle can then search for a location and get all the listed parking spaces along with their reviews. The user can then request for a parking space and the host can then approve or reject the request.

How we built it

The backend server is built on top of Python Django framework which has been hosted on Pythonanywhere. The client application is developed using ionic2 which helped us build hybrid applications for iOS and Android. We have used Google Maps to search, request and host a space. We have used domain.com to host the landing page on www.linknpark.com

Challenges we ran into

We underestimated task hours for a few of the tasks and since ionic2 is pretty hard to debug we ran into some serious issues. We were a team of two so the workload was on the higher side.

Accomplishments that we're proud of

We achieved most of the things that we had aimed for!
