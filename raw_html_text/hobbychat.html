Inspiration

We wanted to be able to converse with random people who have the same hobbies as us.

What it does

By inputting your username and hobby, HobbyChat will connect you with another user.

How we built it

We built HobbyChat using PHP, AJAX, HTML, and CSS.

Challenges we ran into

We were unable to get the window prompt values to show up in the specified html fields.

Accomplishments that we're proud of

We're happy to be able to show a deliverable at the end.

What we learned

We learned that Hackathons are stressful but fun!

What's next for HobbyChat

HobbyChat needs to be developed more to customize the user experience with emojis.
