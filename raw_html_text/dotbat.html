Dat is a method for distributed file storage with a lot of useful applications. However, it requires heavy use of the command line for it to work, which can be quite difficult for the average user. For our hack, our goal was to make dat more user friendly and accessible by integrating it into chat clients such as Slack, IRC, Facebook Messenger etc, although we were only able to implement it with IRC during the allotted time. Instead of using the command line, the user controls the dat protocol with simpler, more intuitive chat commands in their IRC client.
