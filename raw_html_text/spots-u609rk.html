Inspiration

There is so much open data in today’s world For the common person, data like this is inaccessible and difficult to view Humans inherently find that graphs and pictures are easier to view than data tables and lists �

What it does

Our app displays a wide array of data in an intuitive color-coded map scheme

How we built it

Developing on iOS platform, parsing through JSON files received from Canada Open Data (namara.io API), using SQLite to store the data and FMDB to access it from phone

Accomplishments that we're proud of

Connecting the SQLite database with the front-end user interface

What we learned

iOS development, SQLite

What's next for Spots

Implementation in multiple cities, addition of many more data categories
