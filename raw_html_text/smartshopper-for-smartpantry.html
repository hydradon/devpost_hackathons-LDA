Inspiration

Save money while shopping! Use the SmartPantry app to find deals on groceries stores near you with the new SmartShopper feature.

What it does

Conveniently finds if there are deals on groceries and shows what stores they can be found at.

How we built it

Used Python and MySQL on the back-end to find deals from the flyers of different grocery stores. Android Studio front end.

Challenges we ran into

Image processing was inconsistent and unusable for our purposes. Limited Android experience. MySQL cannot directly connect to Android-> BRICK WALL WE HIT AT 2AM Existing code on the application took time to understand.

Accomplishments that we're proud of

Learning how to create and query databases, Android apps, text recognition and python dictionaries. Although the app is not complete, we accomplished most of what we set out to do! We also worked as a team for 24 hours and were able to produce a deliverable.

What we learned

We made a lot of progress in MySQL and Android Development. We learned that design is not a linear process, and that you spend a lot of time going back and forth before getting to a real solution. Sometimes, you have to sacrifice some features to meet a deadline.

What's next for SmartShopper for SmartPantry

Adding more grocery stores to pull data from. Price calculations based on the flyer deals. Connect the android app to the backend through a server.
