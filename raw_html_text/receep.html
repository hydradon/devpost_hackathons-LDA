Inspiration

Laziness and financial issues

What it does

This scanner is a hardware hack that can scan receipts, uses OCR to read the receipt, then organize the output into various categories like groceries or beauty supplies.

How we built it

We designed a 3D model for the product in autodesk-360, as well as designed UI for the phone app.

Challenges we ran into

Due to not having access to the main scanner, the hardware was not put together. Instead, designs and plans for the device were created. A demo phone application was being made, but we were having issues with the Bluetooth protocol, and the demo app had to be discarded.

Accomplishments that we're proud of

First time using Autodesk Fusion 360 and Krita and ended with nice looking models and graphic design

What we learned

We learned the basics of CAD, as well as graphic design. We also learned about linear image sensors and laser mouse sensors.

What's next for Receep

An official hardware prototype
