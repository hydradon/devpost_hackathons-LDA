Inspiration

We wanted a more efficient way to get information from an audio file such as note taking.

What it does

It turns speech into text and retrieves key points from the text and more.

How we built it

NodeJs, Google Cloud Service, Google Natural Language API, Google Text-to-Speech API, expressJS and AJAX

Challenges we ran into

permissions with Google Cloud API

Accomplishments that we're proud of

Finishing our project with the functionality we intended it to

What we learned

How to manipulate Google API to our use

What's next for Knimbus

more thorough text analysis and summarizing | being able to put direct link from video instead of having to import from file system | when transcribing conversations between different people identify the voices of multiple users
