Inspiration

The fact that we are doing the hackathon at a gym makes us want to build something around it

What it does

A gym machine that creates energy. The data of energy amount the user creates will then be transferred into the user's game app to be converted to points. Points are used in the game app to play a game of building mall building.

How we built it

We used android studio and java to build the game and app. Generator and AD2 to create a demo of machine prototype. Javascript was used to calculate how many energy produced by the machine. HTML and css were used to create the website.

Challenges we ran into

A lot of problems with game creation and the problem of getting an initial idea

What we learned

We learned how to use html and css to build a website. We learned how to brainstorm ideas and act on it.

What's next for Bossy

Making a more comprehensive game and configuring real-time data transmission between the gym machine and the app
