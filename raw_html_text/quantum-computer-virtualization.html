Inspiration

Quantum computing is a entirely new and radical approach to computation. The purpose of this project to make it easier for beginners to start learning about quantum computing.

What it does

This project makes it easier for beginners to start learning quantum computing by visualizing quantum algorithms in the web browser.

How we built it

Python and Javascript deployed on Google Cloud Platform.

Challenges we ran into

We wanted to implement a machine learning algorithm on the quantum computer. We spent the first day of the hackathon trying to do this; until we realized that it is too early for quantum machine learning algorithms because the hardware for quantum computers is not advanced enough. It was also hard to animate a bloch sphere in the web browser based on the gates that were applied to the qubit.

Accomplishments that we're proud of

We were able to prototype a quantum virtual machine and to visualize quantum algorithms in the web browser in a single day.

What we learned

We have gained lot of knowledge on quantum algorithms and how to visualize them on a bloch sphere.

What's next for Quantum Computer Virtualization

The next step for this project is visualizing more complex algorithms like quantum entanglement.
