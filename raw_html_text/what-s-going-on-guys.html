Inspiration

Storing real-time data is always a challenge. We wanted to work on a project which stored the songs and showed the list of favorite songs of your friends.

What it does

It takes song inputs and details, and shows them to users.

How I built it

We used Firebase firestore as the primary database. Android Studio IDE was used for the backend code. It shows music added in real-time in the dashboard.

Challenges I ran into

Firebase firestore real-time implementation of inputs, making and organizing plans.

Accomplishments that I'm proud of

Getting to get a running app in 1 night! Getting a nice user interface. And of course, having fun and making new friends!

What I learned

Working better on Google Firebase Firestore. Working in a team under time constraints.

What's next for What's going on guys?

Make a notification system for recommendations for songs and making it social!
