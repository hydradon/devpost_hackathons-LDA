Inspiration

This is a chat bot called Zero (Xero) Chat Bot. Lately, chat bot is quite a popular thing. I think it will be helpful to build a chat bot which can navigate Xero API and provide user with information thru chating.

What it does

The current version is a private Xero app integrated with AWS Lex and FB Messenger. You can ask about the company name you are connected to, query about the invoice status and bill status reports.

How we built it

We built it using Xero API, AWS lambda (node.js), AWS Lex and webhook to FB Messenger.

Challenges we ran into

We did not run into any big challenges.

Accomplishments that we're proud of

Integration of Xero API, AWS Lex and FB Messenger.

What we learned

We learned with the Xero API which will be helpful to become a Xero partner.

What's next for Zero Chat Bot

More information will be provided using additional Xero API, such as various reports, bank transactions, purchase orders, etc. Actions like making payment can be added to make the chat bot more robust. Integration with AWS Alexa will be an additional item to look into.
