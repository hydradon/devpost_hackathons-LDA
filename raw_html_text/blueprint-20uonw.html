Inspiration: There is a need for a mobile device application for fishermen with various degrees of education and skills to record fishing trip report documents with text and photos.

What it does: Unique user login and in app registration form

One time entry of vessel information Verification of vessel information using GIS, GNSS,RS and PFZ. Create trip reports with verification feature Create catch events by uploading images and entering weight and size Automatically identify type of fish and species via machine learning Export trip report data in Excel/CSV format

How I built it: Using Android Studio Plateform:

Challenges I ran into: Training fishermen with various degrees of education and language skills for entering this information

Time intensive Inconsistent information due to lack of uniformity

Accomplishments that I'm proud of: Tool Development

What I learned: Fisheries

What's next for BluePrint: Emergency alert system during Tsunami and cyclones
