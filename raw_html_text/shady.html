Inspiration

Inspired by something I saw on reddit over summer, I wanted to faff around with opengl shaders but in a way that didn't involve writing opengl shaders.

What it does

A language that compiles (hopefully safely) to opengl shaders, and a live editing application that renders graphics as you edit scripts.

What I learned

Don't do macos programming in rust

What's next for shady

More language niceness, loading of pictures to use with images, more?
