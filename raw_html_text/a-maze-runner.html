Inspiration

As the technology is progressing, we need to give users a variety of options to choose the method they want to give their commands

What it does

Amaze Runner takes in voice commands from the user, and controls the movement of the character based on the commands.

How we built it

We used HTML, CSS, JavaScript and HTMLCanvas to make the structure of the maze game. Then, we used voice-flow to capture and send commands to the game.

Challenges we ran into

We are only a team of two. This is the first hackathon for one member of team. So we enjoyed the challenge of learning and teaching throughout the process.

Accomplishments that we're proud of

We have never used the voice control before, so it was exciting to try out something new.

What we learned

We learnt how to use voice control using voice-flow. It is also the first time we made a game using JS

What's next for A-maze Runner

We would like to turn this simple game to a more advanced application where we can use voice commands for more complicated movements. We would also like to implement voice control to a broader set of application.
