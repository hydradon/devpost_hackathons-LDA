Inspiration

We got this idea while looking for places to rent/buy in Boulder. We could not find any reliable reviews of Property Owners in Boulder.

What it does

Allows you to:

  • Look up reviews for a landlord
  • Rate a landlord
  • Provide feedback
  • Find landlords

How I built it

We changed tech stacks in the middle twice. We started with a Java backend and React frontend, but got lost in config hell. We switched to a Django backend with Angular frontend. WE used MySQL as a database.

Challenges I ran into

  • Configuring Spring
  • Configuring AWS
  • Getting React to play well with a backend framework

Accomplishments that I'm proud of

Getting a demoable prototype in time even after all the config issues. Also, we were able to change stacks pretty comfortably.

What I learned

Pick a simple tech stack. Get an MVP in early. Modify later. Dont waste too much time in configuration.
