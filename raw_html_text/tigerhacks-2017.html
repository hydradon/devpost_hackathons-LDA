nüz: aggregating public perceptions of current events

Jack Carmichael, Jeff Ruffolo, Jeremy Gonzalez, Luke Darrough

We aim to provide a new way to search the news. By coupling AI-curated articles with an analysis of public sentiment towards the topic, we deeper insights into the stories affecting us all every day.

The news curation is powered by IBM Watson Discovery service. We use the Reddit API to sources legitimate community opinions on news. We determine sentiment using the IBM Watson Tone Analyzer
