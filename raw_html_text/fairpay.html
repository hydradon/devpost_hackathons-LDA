Inspiration

We wanted to make something that would help advance women.

What it does

It shows how big the wage gap is between men and women in different job positions.

How we built it

We created the webpage in HTML/CSS and the charts with Chart.JS. We compiled and analyzed the data into JSON using Javascript and Python.

Challenges we ran into

We were new to using Chart.JS and had trouble trying to parse through our JSON objects. We also had debates about whether to make this a static or dynamic website.

Accomplishments that we're proud of

We made something.

What we learned

Chart.JS

What's next for FairPay

When information becomes public, we will create a database of companies that have a fairy pay between the two genders.
