Inspiration

The Equifax scandal left us worried about the personal information we store online. First and foremost, we felt that passwords are a very crucial.

What it does

DecentPass is a decentralized application that stores your passwords securely on the Ethereum blockchain. With the same private key you use to access your wallet, you can now also access a list of your passwords.

How we built it

We used the Truffle framework to build the dapp and we use Metamask as the wallet to interact with it.

Challenges we ran into

Storing passwords as strings on Ethereum proved difficult as Ethereum is still very basic and new. We had to store the strings as a byte array.

Accomplishments that we're proud of

We joined together as a team only six hours before the deadline and came up with this idea four hours before. We were able to hack together a working demo in just four hours.
