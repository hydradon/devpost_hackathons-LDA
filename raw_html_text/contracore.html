Inspiration

Contract work is never easy, almost always over budget and takes way longer than expected.

What it does

With our app a general contractor would be able to schedule tasks, automate workloads and forecast cost and time data based on their area. Flow based project management.

How we built it

Using React jsx and accompanying libraries we build a responsive web app. Backend work used Jason data and REST api's for data handling.

Challenges we ran into

JSX and npm dependancy breaks, time shortage and lack of experience in certain areas.

What's next for ContraCore

Development on personal time and possible deployment.
