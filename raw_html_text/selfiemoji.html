Inspiration

Create a fun web app of posting emojis with facial recognition technique

What it does

With the web app, you can post a generated emoji to facebook by your selfie

How I built it

We built the web app by python and flask framework, got the facial recognition data by face++ and project oxford api, stored the data to azure blob storage and shared the emoji to Facebook by Facebook api.

Challenges I ran into

Make a post request to external api with image. Store the image to azure blob storage. Determine the mood of the users by facial expression data and get the corresponding

Accomplishments that I'm proud of

Detect humans' mood by facial expression

What I learned

Image storage, image request. Decision tree. HTML layout. Call external api
