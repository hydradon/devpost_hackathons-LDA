Inspiration

We wanted to explore Matlab's vast functionalities by creeating an application that helps patients and physicians stay connected through emailing the collected data from the patient's phone.

What it does

Using the ios acceleration sensor, we detected any movements/steps as well as heart rate then sent a report of that to the physician if it showed any abnormal data.

How we built it

Matlab, lots of documentation reading

Challenges we ran into

Connecting heart rate script to the motion sensor script

Accomplishments that we're proud of

Getting the email component connected successfully

What we learned

How to use Matlab's toolbox to create a product that is relevant to everyday usage
