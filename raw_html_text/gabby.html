Inspiration

The world is coming to the point at which it needs a new thing, Siri from Apple, Echo from Amazon, Cortana from Windows. All these technologies inspired Gabby.

What it does

Gabby, talks to you. It would be like having a conversation with a real person.

How I built it

I built it using the Java console

Challenges I ran into

Gathering Mathematical statements inside of Strings given by the user and displaying the answer.

Accomplishments that I'm proud of

I've incorporated Memory, Gabby remembers the information you give her. I'm proud of giving Gabby emotion, because it really shows how talking to Gabby is not different from talking to a human after all.

What I learned

Two new methods, String.split() and String.

What's next for Gabby

Giving it a Beautiful GUI other than the console.
