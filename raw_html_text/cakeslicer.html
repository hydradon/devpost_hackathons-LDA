Inspiration

Solves a real-world need for cake slice equality!

What it does

The app shows you how to cut your cake so that all your guests get the same sized piece!

How I built it

We built it in Android Studio using Java and XML.

Challenges I ran into

1) Getting the camera to display a live feed 2) Saving an image to a file system

Accomplishments that I'm proud of

Getting the challenges to work!

What I learned

We learned how to use Android Studio, refreshed our Java skills and learned how to work around the challenges above!

What's next for CakeSlicer

Adding options for dividing different shaped cakes with different shaped slices.
