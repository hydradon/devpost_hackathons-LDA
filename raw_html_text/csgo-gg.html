Inspiration

We've all been playing cs go since high school and we always wondered if csgo games could be predicted.

What it does

It predicts per region, teams and map, which team will win and lose.

How I built it

We used frameworks such as Python pandas and numpy to analyze and clean the data. We also used flask for the backend to send data to our react frontend

Challenges I ran into

Analyzing and making a model was difficult because of our low sample size

Accomplishments that I'm proud of

Having 70% accuracy for the region of asia.

What I learned

We learnt a lot of web development and data analytics due to this project.

What's next for CSGO.GG

We hope to gather more variables for that we can better predict these games
