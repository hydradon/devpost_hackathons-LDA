Inspiration

We wanted to help kids improve their problem solving skills.

What it does

It allows kids of age 10-14 improve their problem solving skills and critical thinking.

How we built it

We used Java to code all the functions and add music. Photoshop was used to create the backgrounds. xml was used in studio to import backgrounds and buttons etc.logicprox was used to create the background music.

Challenges we ran into

Adding music and getting the buttons to function was difficult.

Accomplishments that we're proud of

Proud of finishing level 1 of the program with all the questions we needed.

What we learned

Improved our Java knowledge with stuff we never knew. Also learned how to use Android Studio better.

What's next for Mind Quest

We're going to finish the entire game along with all the other levels.
