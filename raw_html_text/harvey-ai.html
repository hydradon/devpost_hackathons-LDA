Inspiration

Hurricane Harvey :'(

What it does

Feeds public data gathered from Twitter into the Nexosis API to create an intelligent forecast of affected areas. This can give rescuers and volunteers the ability the figure out where people need help.

How I built it

With Node.js, Nexosis Machine Learning API, Twitter API
