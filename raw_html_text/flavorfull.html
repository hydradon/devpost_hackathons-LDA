FlavorFull

Summary: The project goal of our project is to save people from boring food and expose them to new meals by using an android app that utilizes IBM Bluemix and McCormick's API to search recipes you can make with the spices you own at home!

This project was built in 24 hours during HackUMBC Fall 2017 and if we had more time we would add more features such as:

Create a database for user settings for more customization such as adding favorites, Research usage of the app to better serve users, Quick search of retailers nearby selling McCormick products, Ability to add other ingredients other than spices, Finding recipes that only need a few more ingredients, Advanced searching functions such as "chicken" and "20 minute cooking time"
