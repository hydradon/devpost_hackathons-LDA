Inspiration

We were inspired to create a new and innovative way of voting that was more representative to the general populous by the youtube channel CGP Grey

What it does

Its a voting system that removes gerrymandering and allows for voters to be fairly represented.

How we built it

We used an Arduino to make the voting buttons, and used Python to make a script for the voting algorithm that determines the winner. We also created a website using HTML to demonstrate the concept.

Challenges we ran into

We ran into several problems with the Arduino, where in many cases we could not figure out how to code in the button responses.

Accomplishments that we're proud of

We managed to successfully make a prototype of this system.

What we learned

As beginners we learned many different coding languages such as CSS, HTML, Python and Arduino.

What's next for STV Voting System

We hope for implentation in small groups.
