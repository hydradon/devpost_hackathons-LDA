Inspiration

As 3 minority women who constantly take CTA transportation, we wanted to create something to help the voiceless express what they went through to potentially decrease the amount of sexual harassment.

What it does

Allows for users to post on the forum.

How we built it

Using HTML, CSS, and Javascript

Challenges we ran into

The forum was the hardest thing to create since we had to use Javascript, CSS, and HTML and we had a few errors with the input for the fields.

Accomplishments that we're proud of

We managed to complete this simple, yet functional website and it runs smoothly.

What we learned

How to use CSS and Javascript with HTML and debug

What's next for Aware

We aren't sure if we would like to expand on this project.
