TowAR

Inspiration

We wanted to create an AR app, and we decided on a game because we were inspired by simplistic but fun games such as Gang Beasts and Totally Accurate Battle Simulator.

Gameplay

The user fights versus endless amounts of soldiers and tries to get a high score. The user can add troops to their side by spending mana.

How we built it

We built it using the Unity Engine and the ARCore framework by Google.

Challenges we ran into

We ran into some merging issues with GitHub, and sometimes our work was slowed or deleted because of it.

What we learned

We learned how to use the ARCore framework with Unity.

What's next for TowAR
