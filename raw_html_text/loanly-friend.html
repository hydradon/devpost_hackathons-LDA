Inspiration

We took prompt 1 and tried implementing it as best as we could.

What it does

When a user tries to make a purchase, it recommends loan/credit offers that the user might be interested in.

How we built it

We built chrome extension using HTML, CSS and javascript. We connected to Even Financial API by using python and flask. We set up our local server using flask in which we create and query data that we saved about the user.

Challenges we ran into

This is all new stuff so we had to do a lot of self-learning.

Accomplishments that we're proud of

We are happy to have an end product. We also almost won the cup stacking contest.

What we learned

We learned how to properly google when we are stuck and when there is no outside resources.

What's next for Loanly Friend

We can improve the UI and be more user friendly. We can also implement more features that might be useful for the consumers.
