Inspiration

We want a fun way to analyze the mood of the chat that can be useful for the streamer or Twitch

What it does

Read emojis from twitch chat Makes a graph measuring emojis and its time posted

How we built it

We built it linking Python, JavaScript, React.js, and html.

Challenges we ran into

The python-twitch-stream library did not work on Windows Hard to design a good user interface

Accomplishments that we're proud of

We feel like it's an applicable application that has a wide variety of potential use cases and could be directly added on to existing Twitch analytic services. The flying emoji idea could be adopted to the chat box of the streaming videos.

What we learned

We learned how to use twitch API Analyzing & comparing data sets using graphs The history of Twitch emojis

What's next for twitchcon

We can further expand our emoji analysis by combining it with sentiment analysis, video overlays, emoji option for filter
