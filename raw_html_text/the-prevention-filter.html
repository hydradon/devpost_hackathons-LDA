Inspiration

We know people in our lives who have suffered from depression and trauma. They are people that we need to be more sensitive around and the internet is filled with triggering content that may harm them.

What it does

The filter skims through all websites accessed by the user and removes any potential trigger words to prevent any harm. The entire sentence around it is also removed in case it contains a context that is dangerous for the user.

How we built it

The program was built in python and a website built with HTML & CSS was created along with it to explain its function.

Challenges we ran into

We ran into the issue of not being able to do exactly what we initially planned. We originally wanted to create a chrome extension popup but we realised that would take up too much of our time and we wouldn't be able to complete it.

Accomplishments that we're proud of

We are proud that we created an actual program that works and that we managed to bring this idea to fruition quite quickly.

What we learned

We learned more about online platforms.

What's next for The Prevention Filter

Our next step is to turn this project into a user-friendly chrome extension.
