FNB Assistant is built with Python, Angular.js, and Natural Language Toolkit (NLTK). We're using context free grammars to parse natural language into database queries; It is an extensible and powerful system. The question domain deals with the user's personal finance, allowing the computer to inform the user on their personal finance in a conversational manner.

Example: Q: How much do I spend on groceries? A: $349
