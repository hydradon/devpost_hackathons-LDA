Inspiration

We always waste a lot of time trying to find good movies to watch. With our program, we want to help others avoid this dilemma.

What it does

It finds three great movies to watch based on a genre that the user selects.

How we built it

Using HTML, CSS, and JavaScript, we developed a website. We stored all the movie information in a .csv file and imported that to our program as well.

Challenges we ran into

We were not so familiar with JavaScript, and encountered several errors.

Accomplishments that we're proud of

We were able to get te website to work the way we wanted it to, and were able to further develop our HTML, CSS, and JavaScript skills along the way.

What we learned

We learned how to code with multiple languages within the same program.

What's next for What2Watch

We can add more features such as the platforms where the movies are available, links to those platforms, a more descriptive summary, cast information, online ratings. And lastly, we could add more movies to our database so we can suggest more than 3 movies.
