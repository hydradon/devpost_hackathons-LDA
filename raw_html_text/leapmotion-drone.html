Inspiration

We are sick of using remote controllers in the era of smartphones.

What it does

Using the LEAP MOTION SENSOR, we use human gestures to control a Parrot AR Drone.

How we built it

We used the PS_Drone api and the LeapMotion API to program it in python

Challenges we ran into

Communication between the LEAP and the computer, and between the computer and the drone is really tough

Accomplishments that we're proud of

We accomplished in making the drone fly using basic motions, manipulating the vertical speed, yaw, roll and pitch of the drone's orientation and movements.

What we learned

We learned how to use API effectively, as this is our first hackathon as a team, it was really hard at the beginning, but then we adapted and managed to finished our basic features.

What's next for LeapMotion Drone

This project is a pathway to the future, where drones, cars, and VR can be more adaptive using hand gestures, and even body gestures.
