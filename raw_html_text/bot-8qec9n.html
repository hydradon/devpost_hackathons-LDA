B.O.T.

Basic Operations Teacher

Nick Montelibano , Alex Lende , Ben Williams , Chris Hildreth

Robot that teaches basics of programming to kids through a Raspberry Pi robot

Features

14 instructions cards, featuring:

  • Move Forward
  • Move Backward
  • Turn Right
  • Turn Left
  • Open/Close Right Bumper Interrupt
  • Open/Close Left Bumper Interrupt
  • Beep
  • Wait
  • Done
  • Restart
  • Dance!

Record and Play modes, as well as a save program button Website featuring awesome gifs of actions stored on the device
