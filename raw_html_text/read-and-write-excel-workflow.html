• Read and Write Dynamic Column Name and Value in Excel Workflow

Inspiration

Reading from excel to perform certain task and writing of status to Excel after performing of the certain task are very common as part of the sequence in RPA but unfortunately there is no reusable workflow that can be use in Uipath. Hence, I have created a reusable snippet components(WORKFLOW TEMPLATE).

What it does

This workflow allows reading of any excel worksheets from your local drive. It allows selection of excel worksheet from any drive. Thereafter, it is able to determine the last column to write the status back to the same excel that was been read. This workflow allows dynamic column name and column value. It will prompt the user to enter the column name as well as the column value before writing back to the excel.

Accomplishments that I'm proud of

Uipath user is now able to incorporate 2 commonly used Excel functions by adding the workflow template in their project instead of re-writing each individual function in their project again.
