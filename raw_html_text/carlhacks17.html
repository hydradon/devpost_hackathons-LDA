Inspiration

We both love gardening and wanted this community to have to the tools to reach more people.

What it does

Northfield Garden Club's current website is unengaging, especially for young people trying to get involved in the community. This new website allows them to reach the rapidly changing college population of the town.

How we built it

This was our first big dive into web development. Using open source materials in html and css, we constructed this more dynamic and user friendly interface.

Challenges we ran into

Constructing a project on this scale in only 18 hours is not an easy task. We had lots of trouble merging our various design ideas into one fluid product.

Accomplishments that we're proud of

We are very excited to have written an entire website ourselves!

What we learned

The interactions between various components of web development were new to us, but, excitedly, we read lots of tutorials and learned a whole bunch.

What's next for New Website for the Northfield Garden Club

Ideally, we would work with the organization themselves to fully implement their content and hopes for their own community.
