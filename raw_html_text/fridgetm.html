Inspiration

The project was born one day before hackathon, so always excited to build something new

What it does

It allows users to take snapshots of their products e.g (vegetables) and the system based on AI will fetch the snapshots and suggest the user some recipes he/she could prepare.

How we built it

It was quite challenging only knowing PHP server side language, so relying on Google Vision API was a risk but, at the end the application proves to be useful and very funny.

Challenges we ran into

The greatest challenge we ran into was relying on a third party trained machine API, which we weren't quite sure if it's going to work

Accomplishments that we're proud of

We are proud of building a little AI system which proves to be useful for those who rarely cook and would like to

What we learned

We learnt to make use of Google Vision API and overcame lots of challenges regarding the app

What's next for FridgeTM

We plan on integrating our app with IoT devices and connect it to couple fridges
