Inspiration

Me and my friends discovered that we would often message each other in group chats to see if anyone wanted to hangout in 15 minutes, often spamming the group chat with invites and responses.

What it does

The purpose of the project is to provide a way to effectively update one's "status" with an event they want to go to, such as lunch at a specific campus. Then, their friends will get notified and have the options of "meeting up" with the event.

How we built it

We used MongoDB and Python for the back end, and Javascript and CSS for the front end.

Challenges we ran into

At first working with the database was a challenge.

Accomplishments that we're proud of

A nearly fully functional back end.

What we learned

How to use MongoDB.

What's next for Meet Up

Connecting users to each other by setting up the server publicly.
