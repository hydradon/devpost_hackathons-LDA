Inspiration

To be more financial literate in a fun and entertaining way, given our limited financial knowledge.

What it does

It's a visual novel that demonstrates the relevance of financial literacy in life in Singapore. Special attention is paid to visuals to make the experience more pleasing and unique.

How we built it

Javascript, HTML, CSS, python

Challenges we ran into

Stay sane without sleep :))))). State management.

Accomplishments that I'm proud of

Able to build a fun game with purely front-end technologies from scratch.

What we learned

Front-end technologies, responsive game design, teamwork, and financial knowledge.
