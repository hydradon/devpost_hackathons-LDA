Inspiration

We travel a lot as a family and we often spend a lot of time in a car. Our kids love asking us "are we there yet?!" every few minutes when they bored. This is how idea of building Alexa skills to help us came about.

What it does

It takes care of answering classic road trip question: "Are there yet?!"

How we built it

We used AWS provided Node.JS blueprints for Alexa Skills and we built on top of it.

Challenges we ran into

Build large enough database of good answers. It was fun :)

Accomplishments that we're proud of

Alex skill works and it is ready for Car enabled Amazon Alexa experience!

What we learned

We learned how Alexa works and learned a bit of Node.JS as well as AWS Lambda.

What's next for Ask The Driver

More interactive, more questions, more answers, more fun! We planning to add artificial intelligence to teach driver on how to answer most popular questions.
