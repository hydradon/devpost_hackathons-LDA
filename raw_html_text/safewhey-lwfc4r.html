Safewhey

Google Maps + Liberty Mutual Insurance API for recommending safe routes

Getting Started

Install:

$ pip3 install flask googlemaps numpy

Run Windows:

$ python server.py

Run mac os

$ python3 server.py

Open localhost:5000 on a web browser
