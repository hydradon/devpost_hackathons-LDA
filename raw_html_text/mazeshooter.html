MazeShooter is a Third Person Shotter (TPS) game made by using Unreal Engine 4. It is also my first solo Hackathon project ever + My first game project.

I decided the make a game as a hackathon project because too many people are doing Web.

Last time I touched Unreal Engine 4 was last summer and I only touched it for 1 week. So making a game less than 36 hours was a very interesting experience!

The current version of the game's achievement is simple. Shoot and kill the enemy and do it forever.

You will have infinite respawn and so is your enemy.

If you want to quit the game press 'Q'.

Accomplishments that I'm proud of

Challenges I ran into

MazeShooter - Instruction

Unreal Engine 4 Game Made by Michael Park

How to play the game.

i) clone the repository to your local machine

ii) Inside of "WindowsNoEditor" folder click "MazeShooter.exe"

iii) Enjoy

Keys:

Move: WASD

Aim: Right Mouse Button

Shoot: Left Mouse Button

Jump: Space

Quit: q.

Enjoy :)
