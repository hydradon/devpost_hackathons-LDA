Inspiration

We all shared a common goal of helping improve the quality of life for individuals in adverse circumstances.

What it does

It assists individuals who have medical conditions to manage or treat their illnesses with set meal recipes.

How I built it

We used HTML and CSS to build a website from scratch, Figma to design the layout of the website, and Photoshop to create the full website mockups.

Challenges I ran into

Most of us had very basic Python coding experience, so finding a project that would work within our capabilities for was difficult. We had many ideas, but not enough knowledge in the tech field to implement all of them.

Accomplishments that I'm proud of

Although our project is not entirely complete, we are very proud of ourselves for stepping outside our comfort zone to take on a higher-level project. As a team, we were able to build website and app prototypes, branding mockups, and create a project we are passionate about.

What I learned

How to better navigate HTML, CSS, Figma, and Photoshop, and how to manage a large project within a 24-hour time frame.

What's next for Kenkō website/app

Hopefully one day we will be able to put this website and app design into reality, but for now, we still have much more to learn and develop.
