Computational Subsidiarity

** Think local – subsidiarity, resilience & scalability **

Mindset

Sense and act locally (subsidiarity), don’t run to the cloud if not necessary in-network data aggregation local information collection and dissemination

Scenario: environmental data heat maps

  • parking spot availability
  • road conditions (icy road stretches)
  • road blocks (accident, traffic volume)
  • noise and air quality

Because: stepping stone for hierarchical aggregation (block, city, country)

Our Solution: In-Network Computing (using information-centric mechanics over 5G)

  • Condensing information from sensors, using in-network computations

  • NFN -- Named function networking a) state the result you want b) network returns the desired information (instead of raw data)
