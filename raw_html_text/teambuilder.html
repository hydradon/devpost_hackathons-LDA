• Do you need to fill your team?

When we arrive this Friday at our first Hackathon, HackUPC Winter 2017, we were two couples of hackers and wanted to make a team to be four, so we joined forces in order to gain as much as possible of this wonderful experience.

What it does

Teambuilder is an app/program that helps hackers find their most adequat partners in order to start the project as soon as possible and have fun.

How we built it

We used python in order to create the algorithm and at the same time we tried to build the app on android-studio. Afterwards, we tired to join the app to the algorithm with a database.

Challenges we ran into

It was the first hackathon for all of us, so we had to work on hardly with the algorithm and ran into many problems due to our lack of experience in the field. We also had many difficulties with setting up a data base and an app because non of us knew how to build them and we had to start from scratch.

Accomplishments that we're proud of

Improving our skills on python and android-studio. Going from scratch to the final project.

What we learned

Python, android-studio and database

What's next for TeamBuilder

Improving the algorithm, finishing the app and connected to a database.
