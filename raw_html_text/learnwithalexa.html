Inspiration

Not too many applications out there allow you to teach kids. We wanted to get parents involved and let them submit questions for their kids, without actually saying them.

What it does

LearnWithAlexa,

Multiplayer educational game that emphasizes learning through "fun" competition. Our target group is grade levels K-4. We include questions for Math, Geography, Drawing, etc.

How we built it

We built it by banging our heads on laptops until the Alexa skill worked.

The drawing/shape recognition was done with Google Cloud Vision API and was done separately in flask.

Challenges we ran into
