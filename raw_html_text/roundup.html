Inspiration

A circle was literally the inspiration that made us think of how people interact with their circles and also how different people's circles intertwine connecting everyone.

What it does

RoundUp allows you to connect with friends and also meet new friends. It keeps you connected and notifies you if you are in the same area as a friend or a group.

How we built it

We used html and css for the base architecture of the website and using node.js and firebase, we implemented a chat system for users.

Challenges we ran into

Using firebase in order to make the inner chat system tended to be the hardest part especially storing user's name from their login.

Accomplishments that we're proud of

Getting the chat system to work! After a long night of work and switching from Google datastore to firebase midway through, we finally got something works.

What we learned

We learned that social media isn't as easy as people think! But for real though, struggling with the database and overcoming that has taught us how to use the resources and tools available in order to increase our knowledge of something we were never exposed to before.

What's next for RoundUp

Going worldwide! Being able to help circles plan and coordinate meet up times and places no matter where in the world everyone in the circle is.
