Inspiration

For the past few semester, I have been adding my class schedule to Google’s Calendar to help me to keep track of them alongside other events. However, this is a very time-consuming and error-prone process.

What it does

It allows you so search for and select the classes that you are enrolled in, then generates a portable iCalendar file (.ics) that you can add to whatever calendar software you use. It includes the weekly recurrences from the day of the first class to the last class of the semester.
