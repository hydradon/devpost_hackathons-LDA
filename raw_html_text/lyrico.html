Inspiration

To create an application that is fun to use and to make.

What it does

Displays the lyrics of songs and asks the user to guess the song by selecting the correct option.

How we built it

We used Swift on Xcode

Challenges we ran into

Understanding and using Swift for the first time while writing this project.

Accomplishments that we're proud of

Successfully implemented Swift commands to create an application within 24 hours without any prior knowledge of the programming language.

What we learned

-Working systematically towards building an application. -How to use Swift on Xcode. -Team work and time management -Troubleshooting of problems.

What's next for Lyrico
