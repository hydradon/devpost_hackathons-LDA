Inspiration

Sample online tutorials, brainstorming

What it does

Alerts the user of a house break-in using a touch sensor (in substitute of a proximity sensor which was unavailable). Alerts user by playing a loud noise off buzzer, flashing alert message, and sending a text message.

How I built it

Built entirely in Arduino IDE, the text messaging service was setup through Twilio

Challenges I ran into

Setting up constant loops checking for input of intruder, getting lcd screen to change background colors, and to setup a reset function (never was able to complete)

Accomplishments that I'm proud of

Building and developing a hardware-based project from start to finish

What I learned

How to use Arduino and grove kit sensors, Twilio integration with Arduino, and how to code C/C++ within the Arduino IDE

What's next for Simple Burglary Alarm with Text Alert

Reset function so project doesn't need to be compiled every time you want to use it, integration with a mobile device application.
