Inspiration

Composting shouldn't take a ton of time and effort. We believe by giving people an easy way to get quality compost more people will participate and reduce their waste.

What it does

This repository contains Arduino code which will monitor a compost pile's overall health. If the temperature is not conducive to Microbial Life. It will send an alert via the Twilio API to a member of the compost team.
