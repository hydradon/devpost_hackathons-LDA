Inspiration

The goal is to bring visibility and success to impactful projects that are data constrained.

What it does

Cloudsource is a platform for contributing computation time to meaningful projects in a simple, easy way. Users can a lot their own Azure based resources to projects that they consider worthy. Furthermore at the end of every month we round up your usage to a flat a number and devote the difference to selected projects.

How I built it

Cloudsource is built on top of Azure and Microsoft's windows-live account system. It uses mongoDB for storage of User and Project information, cron for scheduling vm deployments, and node.js to host the api.

Challenges I ran into

Getting real-time log and data retrieval from VM deployments presented technical difficulties.
