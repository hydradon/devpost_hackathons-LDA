Inspiration: The terrible volume ui's seen at " https://www.designernews.co/stories/84443-redditors-design-worst-volume-sliders-possible "

The volume can only be controlled by single clicks. If you want your volume at 100% you have to click the button 100 times. The button but it is constantly moving so it is very time consuming to adjust volume. Also, if you want to lower your volume you have to reset it and start all over :)

Built using codepen.io

Challenges were controlling the speed at which the button moves so the user can _ actually _ click it.

Proud of completing it in less than 2 hours.
