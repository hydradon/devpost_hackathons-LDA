Dictionary-Bot

Description

The product is a bilingual bot that given a word by user, the bot returns the translated word and meaning in the targeted language. So far, we only provide English to Japancese and Japanese to Engligh.

Environment:

Code : Python API: Google Cloud Translate, Oxfard Dictionary API Client: Line (an instant messenger popular in many Asia Country)

Limitation:

Line does not allow us to send the audio of the pronounciation of the translated word, so we send a link that contains the mp3 file.
