Inspiration

After seeing the impressive work done by snake the planet we were inspired to make a snake game where the user could use the world around them to make the game unique. We were also amazed by the Myo armband's functionality and believe that it can provide an interesting way to control your hungry snake. Put these together, and you have Myo-Snake!

What it does

You either take a photo with the webcam on your computer or upload a photo to our web app. After this, we generate a map for the game of snake with walls that you need to avoid based on features of your image! you then use the Myo armband to tell your snake to turn and collect food, trying to avoid running into yourself or any walls.

How we built it

We use OpenCV with a Python backend to do the edge detection and generate which points on the playing field are available to play on. The snake game was written in JavaScript. The Myo armband's gestures were mapped to the directional keys which control the snake game.

Challenges we ran into

We were not experienced interfacing our Python backend with JavaScript frontend.
