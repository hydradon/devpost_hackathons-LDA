Inspiration

Thinking about ways internet connects people, and how we can use that to solve a problem that movie lovers face.

What it does

Gives movie recommendations based on things you like, which match up with our online database of what others, who have the same tastes as you, also like.

How we built it

We used MongoDB, Mongoose, Node JS, GenomeLink API, and Heroku to make a rest API, accessible by any device.

Challenges we ran into

Getting the database to work correctly, learning the genome API, and creating an algorithm to help find accurate recommendations.

Accomplishments that we're proud of

We think our minimalist UI is something to be proud of, as well as our implementation of advanced databases.

What we learned

How to create RestAPIS.

What's next for RecommendMe

We would like to expand to phones and make our website bug free, as well as add extra security.
