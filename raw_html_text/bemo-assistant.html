Inspiration

We wanted to make something that was equal parts useful, innovative, and helpful to the community. With the increasing usefulness of voice assistants, we decided to try to improve the educational applications of this technology.

What it does

We created an application using google cloud services, in specific, dialogue flow and natural language processing. This app alllows you to practice speeches. The quality of these pracitce speeches are evaulated using natural language processing heuristics, which we apply to our speech data using google cloud NLP api's.

How we built it

We used python and flask for our webhook development and heroku for webhook hosting.

Challenges we ran into

The details of dialogflow were challenging. Especially as it related to managing contexts for more complex dialog sequences.

Future of our hack

We think we could expand this into an extremely useful platform for education. Features we wanted to implement but weren't able to due to time contstraints included automatic user recongition via voice printing and more advanced recommendation tools for serving educational content via constructed user profiles.
