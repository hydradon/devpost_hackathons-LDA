Inspiration

In our short careers in the tech industry, we've gleaned insight into weak points in the development process. Most notably, frequently revised files, the quality of their tests, and the potential for bugs in code. This inspired us to create an analytical tool for Git repositories.

What it does

Provides info from the github API... And Ron Swanson quotes. We had big ambitions at the outset, but were let down by lack of preparation, a new tech stack, and a lot of issues.

How we built it

We had a team of 3-4 developers, all new to Angular, try to sync API calls with Firebase and design views.

Challenges we ran into

Too many of them. Angular has tough documentation, we've never integrated rest APIs into JavaScript, and the GitHub API provided MUCH less analytical information and documentation than we thought.

Accomplishments that we're proud of

Working together and pulling an all nighter to learn a new tech stack. It was fun!

What we learned

To choose React next time.
