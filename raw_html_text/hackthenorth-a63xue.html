Inspiration

Whenever we searched for videos on YouTube, many videos that are irrelevant and which should be restricted are shown in results. Even after switching ON the Restriction mode, many results are unwanted. This inspired us to make a chrome extension that can search for content within the YouTube video - be it meta-data, sub-titles, audio of the video, or even the images/video itself. It uses the Google Cloud Platform, YouTube API and HP Haven OnDemand API.

What it does

~ Best Parental Control tool in it's field. ~ The extension follows these steps:

  1. Captures any change in the YouTube URL.
  2. Starts operating on the video.
  3. Searches for negative content in the meta-data, subtitles.
  4. Applies Speech to text for the audio inside the video.
  5. Searches for restricted content in video using Computer Vision API.
  6. Applies the Sentiment Analysis on the extracted information.
  7. If the result is positive or neutral, then the user can see the video content. If the result is negative, video is blocked and the user is redirected to the YouTube page.

How we built it

We built it using the Google Cloud Platform, HP Haven On Demand API, and YouTube API.

Challenges we ran into

We ran into the following challenges:

  1. YouTube API v3 has restricted the users to use it without the proper authentication.
  2. Screening the video with API was taking a lot of time. We have overcome it though.

Accomplishments that we're proud of

  1. Able to reduce the burden of processing time which takes usually hours for heavy videos to just 2-3 seconds.
  2. Data filtering is much more efficient.
  3. Gives better results that the Native Restricted Mode on YouTube.
  4. We completed it in one night. Yeah!

What we learned
