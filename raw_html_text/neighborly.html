Inspiration

Thomas fire community. We wanted to find a way to help victims in the most realistic and altruistic method possible.

What it does

We're a platform that helps facilitate donations while inventorying the house. We used AR, computer vision, web scraping, and machine learning to make this possible.

How I built it

Swift, firebase, Google vision, etc.

Challenges I ran into

Issues with connecting web scrapers.

Accomplishments that I'm proud of

Helping the community in a way that might actually be feasible - not just a plan.

What I learned

To ask consumers questions and the value of building a product around existing behaviors.

What's next for Neighborly

We're going to make a difference in the disaster relief world. No doubt!
