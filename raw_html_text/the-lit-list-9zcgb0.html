Inspiration

As a college student that enjoys reading, in what little spare time I have, I have a steady list of new books and comics I want to read. However, as a typical broke college student I don’t always have the funds to commit to pre-ordering my favorite upcoming titles.

What it does

Enter, The Lit List. This app was created for anyone, like me, that doesn’t want to forget about new releases while not having to commit to a pre-order. With the ability to search for any book you’re looking for and set a reminder with the tap of a button, it’ll be your new favorite book app in no time!

Accomplishments that I'm proud of

I'm proud of having using a custom login/sign-up with Firebase. Also, I initially launched the app without user accounts and stored the information in Core Data. So, I'm proud I was able to release an update that added user accounts and migrated any existing information from Core Data to Firebase.

What I learned

I learned a lot about networking and how to write code as clean as possible.

What's next for The Lit List

I'm planning on adding a Discover tab, so users can see more about new books being released.
