    • Complete mobile application ### Pre-requirements

  • Install Nodejs v8.x.x ### Install guide 1) Clone project: git git clone https://github.com/thailamha/phoenixUI.git 2) Navigate your directory to project folder git cd phoenixUI 3) Install node_modules with yarn js yarn start 4) Start project yarn start or expo start
