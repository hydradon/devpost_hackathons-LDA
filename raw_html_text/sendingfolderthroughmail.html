• Here is the whole overview of SendingFolderThroughMail snippet.

Inspiration

I was inspired by existing UiPath activities.

What it does

In normal practice user will only be able file as an attachment in mail.If he wants to send folder then he has to zip it and send. So by using SendingFolderThroughMail snippet , user will be able to zip a folder and can send the same as an attachment through mail in a single shot.

How I built it

I've used Uipath tool with existing UiPath custom activities.

Challenges I ran into

The difficult part while building SendingFolderThroughMail snippet was creating a unique zipped folder path.

Accomplishments that I'm proud of

SendingFolderThroughMail snippet works perfectly fine as expected.
