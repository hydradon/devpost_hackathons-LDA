Inspiration

Wanted to make something cool for PowerShell. I think its a nice scripting environment.

How it works

Communicates with a Windows Powershell session via send HTTP requests over a TCP/IP port.

Challenges I ran into

Using Pebble C's SDK and trying to figure out a way to structure the data so that PowerShell receives commands from Pebble so that it can be executed as a PS command.

Accomplishments that I'm proud of

Learning the Pebble SDK

What I learned

A lot of C progrmming.

What's next for PowerShale

Adding more commands and tweaking the UI to make entering commands more flexible.
