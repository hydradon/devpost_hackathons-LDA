Inspiration

r/place on Reddit.

What it does

It is a live multiplayer game where users compete to see who can make the best pixel art while at the same time taking over other people's art as well/

How I built it

We used a Vue.js front end and a Node.js backend with the express framework and MongoDB as the DB. We sketched out our renders online and wrote the schemas out before hand.

Challenges I ran into

We are both complete noobs when it comes to web development. Simple tasks such as pulling data from a server took much much longer than it should have.

Accomplishments that I'm proud of

Replicating r/Place!

What I learned

Web development is really hard!

What's next for Pyxl Place

Host it on a dedicated server and open it to the public!
