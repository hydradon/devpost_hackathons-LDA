beer-app

Repository for beer app

HACK THE NORTH 2017

Authors: Bryce Sandlund, Lauren Wise, Natasha Stopa

iOS Application that integrates Google Cloud Vision API with the iPhone camera to detect text on a beer menu, and then queries beeradvocate.com to return each beer's rating.
