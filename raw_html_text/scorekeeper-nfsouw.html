Inspiration

I play table tennis at the national level and I would like an accurate way to keep the score of the match and have fun.

What it does

It records points for each player using a simple and easy to grasp method. The arrow keys record points for separate players, and control and shift allow the user to remove accidentally given points.

How we built it

We used google cloud to host the website, and we used 4 different languages (HTML, CSS, JavaScript, and Python) to code for separate parts of the code. The HTML and CSS were used to code the templates and to make the website aesthetically appealing. Python handled most of the backend of the program, while JavaScript was used for the remainder.

Challenges we ran into

We weren't sure how to transport code from HTML and CSS to JavaScript to position the text correctly.

Accomplishments that we're proud of

We learned how to edit pictures for the website using editing software.

What we learned

We learned how to manage time well and divide our responsibilities between education, assignments, and interests.

What's next for Scorekeeper

We are planning to add a score keeping function for more sports such as basketball, football, etc. Also we will make it look more professional.
