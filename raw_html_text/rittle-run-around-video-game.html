Who are we?

The people who created this project are part of a student-run organization known as the Embry-Riddle Game Development Club. We wanted to create a high-quality 2D platforming game this semester and wanted to make a playable, starting build during Hack Riddle to jump start our project.

What is our project?

We wanted to create a 2D platforming game that conveys an accurate, yet humorous depiction of being a college student at our school.

How did we create this?

We used Unreal Engine 4's 2D platforming build. The assets, music, and other aspects of the game play are our own. In order to create the game, we needed to draw up a tileset and sprites, create maps, create enemies, and edit the post processing and camera settings.

What did we learn?

We were familiar with the basics of the Unreal Engine but slowly started to understand the more advanced aspects of it.
