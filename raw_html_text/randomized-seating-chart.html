Inspiration

My chemistry/ Intro to Engineering teacher mentioned teachers needs of a program that makes randomized seating easier.

What it does

This program allows the user to add tables, and students, who can be randomly assigned to a certain seat.

How we built it

We used Eclipse as our IDE and Java. We used the Processing library to help integrate graphics into the program as well as provide common features such as moving objects, or using keyboard functions and on screen buttons.

Challenges we ran into

We found it hard to move the objects independently when colliding with other objects.

Accomplishments that we're proud of

The main accomplishment that we are proud of is the fact that we were able to make an idea into reality in 24 hours. In addition, we got more experience of how to store files and open them within the program.

What we learned

How to do input/output storing useful information and reading it back to the program.

What's next for Randomized Seating chart

We would try making a nicer GUI, as well as fixing some bugs.
