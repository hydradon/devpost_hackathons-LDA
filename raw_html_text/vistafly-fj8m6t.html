Inspiration

Our motive was to overcome the tedious tasks a passenger undergoes during the journey from the origin to the destination.

What it does

Our goal was to find a solution to ease the journey of the passenger using our app which provides the functionality such as FAQ's using chat bots, flight booking and complete information of a particular flight using a simple interface such input of flight code and flight number. The user can sign up into our app and he will receive an OTP on his mobile to verify his account. Later he can use the username to login into our app and get all details regarding the flights he is travelling. After he has arrived at his destination, he can view all the near by places which will be sorted according to types of places like food, malls, etc.

How I built it

We built a hybrid app using Ionic Framework and the concepts of AngularJs. Backend was made using Node.js. All animations were done using css or typescript. We also built a chatbot using a machine learning model.

Challenges I ran into

The challenges we faced that we had to make a complete working app with a functional backend in a limited time period which was very hectic and there was a difficulty in training the chatbots.

Accomplishments that I'm proud of

We have built a beautiful UI which we are super proud of and that too in a limited time frame. The user experience is a state of art.

What I learned

We learnt how to work as a team and manage our time and work force

What's next for VistaFly

We will integrate a cab sharing feature at the departure for passengers with similar locality and also recommend co-passengers based on the interests of the passengers according to the information available on their social media.
