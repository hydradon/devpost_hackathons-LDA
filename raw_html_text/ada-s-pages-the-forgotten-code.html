• Ada's Pages: The Forgotten Code

The year is 3016.

A treasure hunter called Hopper finds the robot known as ADA who operates in an ancient language. Pages of her manual are scattered throughout the lands. Hopper and ADA must uncover the mysteries of the past to solve the problems of the future.

An educational game to help people of all ages grow comfortable programming with Java syntax.

Built With.... HTML5's Canvas element and the Phaser.js engine.

Challenges we ran into... Routing, adjusting to relatively new programs and libraries.

Accomplishments that I'm proud of... All of it. We finished a thing and had a great time!
