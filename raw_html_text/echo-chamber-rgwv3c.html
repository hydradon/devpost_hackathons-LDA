Inspiration

We were inspired by Spotify's yearly "Spotify Wrapped", which compiles a playlist of all your favorites from the year.

What it does

Our application connects with user's Spotify accounts and based off your recent playback, song library, and playlists, will curate a personal playlist for you, made up of songs that we believe align with the tunes you've been listening to!

How we built it

We built our web application using Javascript, HTML, CSS, and Spotify's Web API.

What we learned

We figured out how to build an entire web application with only a client-side Javascript program and served it on HTML. We faced many problems since we did not technically have a server-side, and this caused us issues with authentication and API connections. Our team learned a lot about the nuances of web development from the challenges we faced this weekend!
