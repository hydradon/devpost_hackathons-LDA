Inspiration

It's hard for us to find motivation to workout, so here's our way of getting ourselves in the gym. One of our friends goes to a gym that takes an initial payment of $300 and only takes money if you don't know show up (so if you show up every day, the membership is essentially free). That's where we got this idea.

What it does

It takes an initial commitment of money from the user and uses Location Services to determine if the user works out everyday. If the user does not check in to his/her designated gym, then a fraction of money is decremented.

How I built it

Built with Swift in XCode using Location Services frameworks. Also uses UserDefaults to store User settings and some local data.

What's next for Commit

There's a lot of functionality that hasn't been implemented yet, but we want to allow users to have a specified number of passes per month and also allow them to choose their gym location from home rather than at the gym itself. These are just some of the improvements we have for the future.
