Inspiration : Legacy Bank systems are facing issues when it comes to monitoring, it is must to give them a cloud like feel

What it does : On premise bank systems are still not on cloud and hence they have to do the performance monitoring and every other task with manual intervention this would be helpful once integrated with App dynamics

How I built it : It's built using App Dynamics SAAS service, by doing tweaks to the SDK code as it was not supporting the older versions of C++ SDK

Challenges I ran into : All the technologies used in the project are quite old. Older version of Qt Creater, Older C++ compiler, challenge was to make APP Dynamics SDK compatible to support the legacy system

Accomplishments that I'm proud of : Giving a cloud like feel to legacy systems

What I learned : APM , integration to third party libraries.

What's next for Health Monitoring Tool : Making it ready to be used by our customers to make their life easier.
