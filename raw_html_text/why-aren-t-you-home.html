Inspiration

Wanted to find a way to make a fun children's game into a modernized game for everyone.

What it does

Separate into hiders (children) and seeker (mother). Mother has tools such as GPS, photos, and other clues to hunt down her children

How we built it

Built using React Native, Firebase, and Google Maps API.

Challenges we ran into

Synchronizing the game through a remote database.

What's next for Why Aren't You Home?

Fixing up the game for playability, adding the features we panned out from the beginning, and dealing with synchronizing issues.
