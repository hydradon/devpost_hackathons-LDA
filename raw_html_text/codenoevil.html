Inspiration

We want to be the catalyst that ends homelessness in St. Louis by creating a HMIS system that empowers CoC's administering care within our community.

What it does

Service St. Louis is a centralized coordinated entry program that provides resource management, digitized SPDAT, targeted services, and data analytics to improve the service delivery of care by CoC's. We accomplish this by facilitating easy entry of CoC services and SPDAT information. This, in turn, allows us to collect metrics and connect clients to eligible services though the application.

Our simple resource directory allows both a CoC or a client to find specific services for their needs. This directory can either be accessed through our intuitive web interface or by calling our phone resource system that integrates with our centralized data store.

It has never been easier to connect a client to the care they need with Service St. Louis.

How we built it

Our tech savvy engineers leveraged modern, scalable technology to create an intuitive, simple, and powerful HMIS tool.

Challenges we ran into

There were many features we wanted to implement but we decided to focus on core workflows that directly aid CoC's and their clients.

Accomplishments that we're proud of

We are all collectively proud that we were able to use our talents in the name of community service to help the homeless.

What we learned

Working together is paramount. We capitalized on everyone's strengths to come up with a creative solution in a short amount of time.

What's next for Service St. Louis

We want to scale this tool to support CoC logistics worldwide so we can prevent and remediate homelessness in our country.
