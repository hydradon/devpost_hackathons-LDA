Inspiration

Doom Black book by Fabien Sanglard

What it does

Allows the user to play doom wirelessly using motion.

How I built it

IMU + Button -> Arduino -> Node MCU -> WiFi -> PC -> Java -> Doom

Challenges I ran into

Java robot library suddenly stopped working and IMU didn't interface with Node MCU.

Accomplishments that I'm proud of

Works well for something built in under 24 hours, am especially proud of the one byte data sending.

What I learned

How to use Node MCU.

What's next for In here? Doom.

Dissasembly, it's just a novelty project. Maybe could have use in VR or AR gaming.
