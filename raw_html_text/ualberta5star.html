Inspiration

Tired of going to "bad" washroom on campus

What it does

It lists all the washrooms on campus with ratings based on community ratings

How I built it

Built it using android studio and youtube tutorial on creating clickable app cards

Challenges I ran into

Android oreo does not like clear text requests, building a base template that is versatile for multiple types data, making a backed api that connects with the database and doing the calculations

Accomplishments that I'm proud of

able to make a clickable card, and the data is populated from a api call to the database

What I learned

making an "api" and how it can be used to make data processing easier on the user device, and more secure since the user does not have access to the data directly

What's next for uAlberta5Star

Store images on the device, so it does not have to make calls to the server for image. Allow users to submit locations, and leave ratings, one vote per washroom per device.
