Inspiration

At college there have been days where life has been so crazy that we forget simple things such as brushing our teeth, hair, showering or even eating.

What it does

Allows user to sign in and select specific reminders to remind them to do simple daily tasks that are often forgotten during busy or stressful times.

How we built it

We first built the wireframe in AdobeXD, and built the frontend in Angular using Ionics framework.

Challenges we ran into

Originally our plan was to use Python for the backend but we ran into too many problems to work with this effectively (namely, the Wegman's API).

Working on a group project with git was an issue. It's pretty much as simple as that. Multiple pulls and pushes and commits do not always mix.

Accomplishments that we're proud of

Learning new skills and a new language.

What we learned

API's are difficult to implement and angular.

What's next for M.O.M. (Management Organization Machine)

Implementation of an actual log-in structure using Google's Cloud API. Also, implementing the ability for your mother to see how you are taking care of yourself (similar to a family GPS tracking device).
