Inspiration

We've been watching a lot of shark tank lately and came across this gem of a company: https://www.youtube.com/watch?v=AK26XpWyzLs&t=11s ! So we wanted to automate this business!

What it does

It takes a picture which you can take with the webcam of our laptop and is able to convert the image to a personalized cat version of you using machine learning to add features based on your emotion, etc. These personalized cat images are then converted to G code and our machine uses this data to draw a cat!
