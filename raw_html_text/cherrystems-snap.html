Inspiration

Anyone should be able to create music and share in a social setting

What it does

It automatically fits music to a user singing in a video, based on the user's choice of musical style

How I built it

I built it with my own API from Cherrystems Music. At this hackathon, I built the entire iOS app

Challenges I ran into

Parsing the audio from video, converting m4a file format to wav, using swift to call https requests

Accomplishments that I'm proud of

Making this thing work in one night

What I learned

How to use swift

What's next for Cherrystems Snap

Possibly pitch an updated version of this to large social media platforms
