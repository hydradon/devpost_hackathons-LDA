Inspiration

I am always losing my phone!

What it does

Find My Phone is a skill that allows Alexa to save your phone number and call your phone using Twilio.

How we built it

We imported Twilio's REST API into an Amazon Lambda function and used Alexa's cloud skills to call a phone number.

Challenges we ran into

We faced issues with properly importing Twilio's REST API into Lambda.

Accomplishments that we're proud of

We were able to successfully, make calls through Alexa.

What we learned

We learned how use Amazon's Lambda and apply our python skills to use Twilio.

What's next for Find My Phone

We would like to expand Find My Phone to signal the phone to do other actions such as set an alarm and turn on the phone's flashlight.
