Inspiration

How can we use historical accident data to help prevent similar situations in the future?

What it does

We compare current data to over 3 years of accident data and see if any accidents have happened under similar circumstances

How I built it

Uploaded historical data into Postgresql, created a dashboard in Grafana to visualize the information

Challenges I ran into

How to weigh different values. Weather condition is more impactful than wind direction

Accomplishments that I'm proud of

Very fluid and flexible visualization

What I learned

I learned a lot more about Postgresql

What's next for Accidont

With a neural net, it would be much easier to predict future accidents. This is already possible, but it would be much easier.
