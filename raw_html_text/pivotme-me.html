Inspiration

We wanted to make the most unique game that we could think of while still having simple and easy to learn controls.

What it does

Pivotme.me is a physics game about pivoting. Manipulate your position with a click of the mouse!

How we built it

Unity, blood, sweat, and tears.

Challenges we ran into

Working with Unity for the first time was a big issue for all of us.

Accomplishments that we're proud of

The game's aethetics turned out nicer than we imagined.

What we learned

We learned that it is more fun to make something you enjoy doing than making a project that aims for as many prizes as possible

What's next for Pivotme.me

Local Multiplayer as well as more game modes.
