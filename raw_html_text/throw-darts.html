I enjoy playing darts and wanted Alexa to do the grunt work of keeping score.

Throw Darts will keep track of the score, who's turn it is and even recommend a throw if you're playing double out.

I built the skill using Amazon's new APL language and Python

I haven't developed anything very difficult before so I had to learn about DynamoDB, APL and a little more of how the intents and slots need to be fine tuned to get the best possible experience with the skill.

I learned a lot on this project so I feel comfortable developing more skills for the echo.

In the future I might other dart games such as Cricket, Killer, Golf, etc.
