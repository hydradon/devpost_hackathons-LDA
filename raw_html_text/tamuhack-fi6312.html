• Research Buddy

Inspiration

We want to ease students academic life, hence we came up with a document analyzer that can query important links and references.

What it does

Research Buddy comes up with intelligent search results and arxiv.org journal paper references once you upload a document.

How we built it

We built it using node (angular, express), made use of Microsoft Azure services for intelligent search (bing), OCR and text analytics to extract key phrases.

Challenges we ran into

We faced a few problems with the document size limit Microsoft cognitive API can query, we had to deal with a lot of JSON parsing that required careful inspection of the detailed JSON results returned from the Azure API.

Accomplishments that we're proud of

We are happy to complete the product in a weekend, and the TamuHack'19 exposed us to several new APIs and programming techniques.

What we learned

We became better node developers and amazed at how powerful the Microsoft API services are!
