We wanted to alert others of a fall.

Detects a fall using Photon and Internet Button, sends data to the cloud, and then notifies the correct party.

It is built using Particle Online IDE, Microsoft Azure, and IFTT.

Selecting the right device was a major challenge. The configuration of Azure components was also difficult.

We are proud of the amount of work we put in in order to interface with the Photon with the cloud, and mobile devices. Also of the amount of information that we processed and learned was a huge success.

We learned how to code on Particle's IDE for photon, configuring Azure, and using IFTT. We also learned how to work with metawear device, and SDK's in Android Studio which were not used in final project.

Improvements to follow will be smaller form factor, and detection of other events such as seizures.
