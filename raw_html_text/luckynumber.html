Inspiration

Simple app to get your lucky number! A fun way to get a random number between 0 and 100 you can use for the lottery or making bets.

What it does

Ask it for a lucky number and it will return a random number between 0 and 100

How I built it

Uses AWS Lambda to host the backend. Amazon Alexa takes care of the speech to text.

What I learned

How to build an Amazon Alexa app
