Inspiration

Self gamification applications. Brainstorming activities on how to motivate kids.

What it does

Allows one user, like a parent, set up quests that correspond to real life activities for other users, that result in virtual rewards and optional physical rewards.

How we built it

We used NativeScript for its easy availability and testability. We attempted to start with Angular libraries, but then determined our group members would be more comfortable using straight JavaScript.

Challenges we ran into

Attempting to integrate separate integral parts of the application together was giving us a significant amount of difficulty.

Accomplishments that we're proud of

Thorough process and planning stage that helped us create a reasonable user interface. Some working individual parts of the application.

What we learned

NativeScript, some TypeScript and Angular, and more practice with JavaScript and HTML.

What's next for RitokuMuse

Finishing the full functionality of the basic application to include all the desired features.
