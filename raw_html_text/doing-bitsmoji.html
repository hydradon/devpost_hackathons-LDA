Inspiration

I always read Bitmoji stories on snapchat and thought it would be an interesting way for people to spend time together online.

What it does

Allows you to login using your Snapchat account, and have a conversation, or "Adventure" with a bot using Bitmoji stickers. Each Adventure revolves around a certain theme.

How we built it

We used the snap-kit API and Java with Android Studio to create a mobile app for Android.

Challenges we ran into

Filtering stickers by a certain theme and finding unique URL's for each Bitmoji sticker made it difficult to simulate a conversation with a bot as only one unique user can be signed into the device at one time.

Accomplishments that we're proud of

Being able to create specific Bitmoji stickers with a unique ID between any two snapchat users, and helping the bot to identify which stickers are relevant to the situation.

What we learned

We learned a lot about how the snap-kit API and how to implement Snapchat features into a mobile app, as well as general knowledge on making mobile apps.

What's next for Doing BitsMoji

Add functionality to allow two users to converse over a messaging service, have the theme evolve throughout the Adventure, increase the bot's intelligence, add more stickers to our library.
