Inspiration

We want to comment on youtube videos at peak points in time and express our emotions for what is happening in the video!

What it does

Users of the chrome extension can make comments and have them roll across the screen as the youtube video is playing. Also see comments made by other users of the extension

How I built it

Firebase, Javascript, CSS

Challenges I ran into

Chrome extension content script weirdness

Accomplishments that I'm proud of

Making a REAL TIME commenting chrome extension! Real time is awesome!

What's next for DankTube

Pull comments with timestamps from youtube videos and display them in addition to real time comments.
