Inspiration

Running is boring, don’t you think? We think everyone has a dream to become the real Mario once.

What it does

It will turn your running into a playful exercise. Everyday boring jogging has gone from now on!

How I built it

We develop the game based on ‘Esri’ location platform. And, we challenged that developing only using javascript client side caluclation!

Challenges I ran into

Our modularity skills are not good enough. So, many bugs are happen. We want more time!

Accomplishments that I’m proud of

We’re be able to form a team and develop a game from just a simple idea.

What I learned

There’s many useful APIs that we can use to develop many wonderful services. Calculate the geo points is very difficult and very interesting!

What’s next for test

Game is not just how many points you gain, but how challengeable the gameplay you can enjoy with. It’s more way to work with the concept that turn something to be more interesting, like we’re doing with the running exercise now.
