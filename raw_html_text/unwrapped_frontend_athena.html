unwrapped_frontEnd_Athena

We wanted to create an app that would allow normal people to easily find shops which sell package-free goods.

The app is crowdsourced, which means that normal people can also add the shops which sell package-free shops in their area for other users to find.

We wanted to tackle the 'reduce' part of the reduce-reuse-recycle chain - we really want to stop more plastic from being produced.

Our app will help with this in 3 ways:

  1. Make it easier for consumers to purchase package-free products
  2. Increase the number of people who are doing this
  3. Prove to bigger businesses eg Tesco that there is a demand for this.

The end goal is for package free to become the norm.

The app has 2 main features:

  1. Users add details about the shops which have package-free goods.
  2. Users can search for shops selling package free goods in their area - this part connects to a google maps api and also an api that we created, containing a database of shops (which other developers could also use in other eco-friendly applications)
