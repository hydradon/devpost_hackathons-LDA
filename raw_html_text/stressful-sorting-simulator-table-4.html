Inspiration

I wanted to make a VR game.

What it does

It's a VR game about sorting randomized objects and trying to get the highest score possible.

How I built it

The project was built in Unity using Microsoft's Windows Mixed Reality framework.

Challenges I ran into

VR is a lot harder than it looks

Accomplishments that I'm proud of

Getting a working game (however simple) by the end of the day was pretty satisfying.

What I learned

Game development is hard enough, doing it in VR adds a whole extra level of difficulty and complication. Luckily the framework is pretty friendly, but since the technology is still young, it's very finicky.

What's next for Stressful Sorting Simulator [Table 4]

I had a lot of ideas for thematic changes that could make the game funnier or more grounded in a story, but I didn't have time to do so. I'd love to expand on the game and grow it further by adding more obstacles, gamemodes, etc.
