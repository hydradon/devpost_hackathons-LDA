Our inspiration

We like circles.

How it works

The user controls the black circle and tries to touch the blue circle while evading the purple and red circles, which teleport the black ball randomly and decrease your score respectively.

What we used

We used javascript for the behavior of the balls, css for formatting of the site, html for the GUI, api for sending the emai, and node.js for the backend.

Challenges

Changing the velocity, direction, acceleration of the balls.

Our biggest accomplishment

We were able to send an email through javascript.

What we learned

How to use javascript to send emails and how to properly format the GUI.
