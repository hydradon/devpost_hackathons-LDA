What it does

visualizearth is an interactive data visualization that uses past/present/future WorldClim data and NOAA satellite data to display predicted trends in average mean temperature over time.

How I built it

I used R for data collection + wrangling, and HTML, CSS, JavaScript, and Mapbox for the data visualization.

Challenges I ran into

Getting the WorldClim and satellite data into geojson format that could be used in Mapbox, and figuring out how to use Mapbox.

What I learned

Learned how to use lots of new R commands and packages.

What's next for visualizearth

Utilize WorldClim data further to create a statistical model that can predict temperatures for future years.
