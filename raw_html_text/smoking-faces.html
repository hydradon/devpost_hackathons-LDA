Inspiration

Alert if person is smoking while in a car. Smoking is not good for our health and worse if in a closed and small place like a car. This tool could help remember this.

What it does

Detects people face and highlights it using blue rectangle. If person is smoking the rectangle turns red.

How I built it

Using tensorflow training a model with smoking faces images.

Challenges I ran into

Getting the images for smoking and non-smoking people.

What's next for smoking-faces

Improvements for accuracy by getting more images used for training the model.
