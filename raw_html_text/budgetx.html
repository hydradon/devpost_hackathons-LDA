Inspiration

We wanted a simpler tool to manage an annual income across various life needs (such as shopping, bills, etc.)

What it does

It allows you to make categories for your budget (such as shopping or bills) and subcategories (such as how much you want to spend in specific shops or on specific bills)

How we built it

We built it using Django and python.

Accomplishments that we're proud of

Building an original authentication system and learning to utilize databases across different accounts

What we learned

We learned how to use Django and build dynamic web apps.

What's next for BudgetX

We want to integrate an algorithm to help users decide on budgeting techniques best for them based on their income. This way people can avoid allocating in certain categories. Furthermore, we wish to add a feature to manage spending as well in addition to just planning.
