Inspiration

We were inspired by the Google Maps feature of 'places near me'.

What it does

The startup page displays a list of emergency service types. After the user chooses one, a map is displayed showing the emergency services of that type that are nearby. The list view can also be chosen by clicking the list view tab at the bottom of the page. Upon clicking any one of them, it leads them to a separate page that contains all details of the service.

How we built it

We used Android Studio to program the application

Challenges we ran into

This was the first time we did android programming, and we had to teach Java to ourselves before diving into programming the app. Learning the language, and also, learning to use the Google Maps IDE and incorporating it into our application was a major challenge.
