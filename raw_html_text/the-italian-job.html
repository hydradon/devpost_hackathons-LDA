• Rasdaman web service provides data about total precipitations for a given set of coordinates and a time range
  • Kmeans (Python + R scripts) provides information to estimate thresholds according to clusters
  • Node.js provides another web service to plot the previous thresholds on a map along with forecast information. It also alert a set of geolocalised users via text messages.
