Inspiration

I was inspired to create this project because of my sister. My sister was the editor-in-chief of The Oracle and recommended me to create an app related to the newspaper.

What it does

The app lets you easily access articles and issues provided by this high school's newspaper that are made available online.

How I built it

I built this app by using the platform, Xcode. In addition to this, I used the programming language, Swift, to develop the app. With Swift, I parsed the online website that this high school has, and made the display of this website much better for mobile users.

Challenges I ran into

I had trouble getting the articles to be viewed properly when sliding down to the text of the article, but managed to fix the problem when I realized that I did an incorrect use of an 'if' statement.

Accomplishments that I'm proud of

I am proud of the fact that I was able to create this app under a constraining time period. The app proved to be challenging, but, in the end, I overcame this problem.

What I learned

I learned how to successfully parse websites using cocoapods such as SwiftyJSON and Alamofire.

What's next for The Oracle App

I plan to add more features and update the app frequently to ensure the best mobile-friendly app version for users.
