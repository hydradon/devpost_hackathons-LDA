Inspiration

Even Prompt Number 2. We also before doing anything agreed that we wanted to learn something new and do something that we thought that we couldn't so we were striving towards those goals and arrived at them.

What it does

Our solution is a client that uses blockchain and a p2p network to store, get and amend a credit score based on an algorithm that we also designed.

How We built it

Many, many hours of research and then using the full time to code and still do even more research. Using Bootstrap and animate.css to make the site look good. We also tailored our backend specifically to our use case so it works faster, better and more secure.

Challenges We ran into

When autofilling one form that previously you had to fill out manually we had to find an out of the box solution for it.

Accomplishments that We are proud of

We are proud of the amount of complex features We were able to implement such as Blockchain, P2P Network, RSA security.

What We learned

What a Blockchain is and how to make it, what a P2P Network is and how to make it, what is RSA security and how to implement it.

What's next for SSFin

Adding more features and cleaning up backend as well as scaling and turning off presenter mode. Also increasing security by using more than just RSA keys.

Why are we better

We created our own blockchain from scratch so we don't need a third party so we specifically tailored our blockchain to our use case. Also we have a very high level of security with our RSA keys. Also Our Application is on Peer-To-Peer Network causing scalability to be much easier and more effective.
