How we built it

In our Proposed System, we replace the manual task of verification and validation of customer identification by automating the task using automation anywhere tool. Tools Used: uipath and SQL Server

Process Workflow:  Extract data from Claim Details field (Which is coming from Email)- and store in DataTable. It will go in Check Claim where is verifies the validate customer from database. It check for validation, if it is valid then go in next step Here their main validation of insurance work took place – it validated through database and uses business logic . If validate than it give popup and send email in given format . It will send both the customer type either the validate insurance claim as well as not valid insurance claim. And finally after the customer verification done, it close the application.
