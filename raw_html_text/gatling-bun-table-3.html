Inspiration

Contra, bullet-hell, Undertale, Cuphead

What it does

Player avoids enemy projectiles for as long as possible

How we built it

We used notepad, and javascript/html/css

Challenges we ran into

Using notepad, learning the language, going straight into programming without using any premade software to help us.

Accomplishments that we're proud of

Making the character move without delay, making a jump that followed physics(slow in/slow out), programming projectile formations, figuring out how to program without a game engine, etc.

What we learned

Javascript, html, front end coding, timers, keycodes, etc.

What's next for Gatling Bun (Table 3)

The enemy actually shoots carrots and not green squares, the player can actually win, the player can actually die, a end screen, a main menu, more bosses?
