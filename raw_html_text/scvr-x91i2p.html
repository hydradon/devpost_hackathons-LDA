Inspiration

All our have came to Junction from Moscow Technology University(MIREA). At the second course we were max interested in VR technology. And one idea of global social VR network come to use. From that time we step by step moving to reach this epic goal

What it does

Our project is multiplayer VR shooter with using Microsoft Mixed Reality headset.

How we built it

We built it under ASP.Net Core 2.0 and TypeScript with Babylon.js for front on Azure service.

Challenges we ran into

ENTERTAINMENT. MICROSOFT WEBVR CHALLENGE

Accomplishments that we're proud of

Our accomplishments is the combine of WebVR, Lenovo explorer Mixed Reality device and Azure service in one complex solution. We made custom multiplayer by our own from zero-point.

What we learned

First of all, we got experience of real working with Babylon.js and now we can use it like Web Engine to build VR application. Secondly we've deeper learned how to build Web Application based on ASP.Net Core 2.0. We have got deep knowledge about how to code multiplayer in modern games and tried to realize it in our application. ## What's next for SCVR Our next step's realize Voice Chat, File Server UI, Video and Image Viewer for people coloboration
