Inspiration

Pebble is cool, and Baltimore is dangerous.

What it does

Android app has map of dangerous areas and can plot route. Pebble calls emergency contact based on accelerometer motion.

How I built it

Android SDK, Google Maps API, Pebble JS API, Matlab, Baltimore Open Data.

Challenges I ran into

Communication between Android and Pebble, and map algorithm to avoid dangerous areas.

Accomplishments that I'm proud of

Pebble integration and heat map.

What I learned

Programming for a wearable, using Google Maps API.

What's next for SafeRock

Better map routing algorithm, more Pebble integration.
