Inspiration

I wanted to use my code skills to help those in need, especially those who do not have access to information dealing with rape, abuse, and sexual assault

What it does

The core features include: the ability to call and text hotlines from within the app, locate nearby hospitals, provide information on each topic without internet access, and contact 911.

How I built it

I used swift, Swifty JSON, and GitHub,

Challenges I ran into

This is was my first time coding and learning swift so it was a huge learning curve for me to understand code concepts in general

Accomplishments that I'm proud of

I'm most proud of the fact that I learned how to code within 6 weeks and created my dream app to help those in need

What I learned

I learned how to code in swift, communicate my ideas with others, find solutions through my own means, and foster work relationships with everyone in the Atlanta make school chapter

What's next for Recoup, Inc.

I want it to gain public attention and hopefully, I can add features to make it more useful
