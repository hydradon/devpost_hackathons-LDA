Inspiration

We wanted to create an app that could solve multiple problems with a generalized solution and connect neighboorhoods.

What it does

Bottl allows people to "drop" messages in a "bottle" by storing text, their name, and location data in a sql server. Then it shows them to people who walk by.

How we built it

We used XCode to make the frontend app, and nodejs to set up a sql server and restful api for backend storage. We sent and recieved requests using JSON

Challenges we ran into

Connecting the pipes and doing the json parsing and serialization. Random Interface builder bug that took 6 hours to spot.

Accomplishments that we're proud of

Successfully saving to server the morning of. Reorganizing our team after members left and slapping together a backend for the project.

What we learned

One of our team members came in with no backend experience and helped build a lot of the initial server, before we reorganized.

What's next for Bottl: Connecting Neighborhoods

publishing to the app store; finding better backend solution; changing lives.
