Inspiration: A simple way to remind family members whose turn it is to do chores.

What it does: Lights up an LED for a particular member depending on the day and reads off their chore on an LCD

How we built: it I designed the case it in Autodesk Fusion 360, 3D printed it, and used Arduino Uno hardware and software for the main body.

Challenges we ran into: The LCD continues to short circuit and won't read off English words. Also incorporating time as a function into the C++ code for Arduino was difficult.

Accomplishments that we're proud of: I am proud of the 3D printed box and the ability for both components of the Arduino to light up, if not the way I wanted.

What we learned: How to use Arduino, how to code in C++, that connections on breadboards are fragile.

What's next for The Chore Box: Fixing the LCD and finding out how to make time a function in the code.
