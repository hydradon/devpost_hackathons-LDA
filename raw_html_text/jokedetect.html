Inspiration

We wanted something entertaining that is also impressive

What it does

The neural network is trained on the transcription and the intonation, and based on those, it tries to figure out if something is a joke or not.

How we built it

We use Microsoft Cognitive Services for speech to text and Tensorflow + Keras for the neural network.

Challenges we ran into

Setting up Cognitive Services API.

Accomplishments that we're proud of

Recognizing a joke is very hard for a computer, and we're surprised and impressed that we got close to that.

What we learned

Microsoft Cognitive Services API

What's next for JokeDetect

We could have a crowd analysis that detects whether or not a joke was good, based on the reaction of the audience,
