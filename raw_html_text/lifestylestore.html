What it does

It is a e commerce website where we can shop the items. If you like the item you can add the item into the cart and you can pay for the item what you have choose with payment gateway ( We used Razor pay payment gateway). If you have any trouble or any queries you can connect with us through a chat window which is placed at the bottom right.

How we built it

This is a web application developed using html,css,bootstrap,php,mysql,twak api,razopay api.
