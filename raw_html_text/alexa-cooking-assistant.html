Inspiration

The ability to search, and make food at your own pace with both hands at all times

What it does

Allows user to search the web with for a recipe based on the ingredient, serving size, and cooking duration. Announces the next steps and required ingredients without having to pick up a mobile device or flip through a book while preparing and cooking food

How we built it

We used Alexa to process voice commands. Lambda to create the node,js application that processes the voice commands. Node.js to trigger an api call to a recipe database, and process, store and speak out the information related to the recipe.

Challenges we ran into

Creating and adding libraries to Lambda Connectivity issues with Alexa and the Techcrunch Wi-fi Storing and restoring session Identifying the cooking duration for certain recipes

Accomplishments that we're proud of

Finished a demo before the end of Hackathon

What we learned

Programming in Node.js on Lambda Programming with Alexa custom skills Using AWS service to integrate separate features of the application

What's next for Alexa Cooking Assistant

Calories counter Adjusted values for larger servings Integration with Pulse 2 (for egg timer) Integration with Watson (to improve results, and make suggestions to the user) Website (that reads out the steps like Alexa)
