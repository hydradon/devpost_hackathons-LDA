A problem that many students currently face is not having anyone to sit with when they are at dining halls. “Never Eat Alone” addresses this problem by allowing users to see who is currently eating, and where. This way, they can find friends and thus always have someone to sit with in the dining hall, café, etc. I learned a considerable amount about iOS app development throughout the process. For example, I learned how to set up my app’s backend using Firebase to authenticate users and to store and retrieve information about every user to create a social network.

How we built it

Built with Swift, and Xcode. Used Firebase as the backend of our app.
