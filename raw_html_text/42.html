Inspiration

Making preparation for tests simpler and less stressful.

What it does

It is aimed towards building an online community for test preparation. People can ask questions, start threads and answer questions. Resources and notes (.jpg, .jpeg, .png or .pdf) can be uploaded too. This creates a comprehensive community and resource for ensuring students reach best potential on any test.

How we built it

HTML5, javascript, css3. We also used Facebook login and parse APIs.

Challenges we ran into

Lots. Everything starting from learning javascript was a challenge.

Accomplishments that we're proud of

Team bonding and the amazing learning experience we had.

What we learned

Learnt coding javascript and sharpened HTML5 and CSS3 skills. Also, the web development training session explained the full skill set required for doing good web dev.

What's next for 42

To get it fully functional and open for users with 'real' data.
