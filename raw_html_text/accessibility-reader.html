Inspiration

Current internet screen readers for visually impaired people read every word on the webpage. Visually impaired people suffer from “information overload”, as a result of being unable to skim webpages.

What it does

Given keywords, a shortened version of an article is read out by the Amazon Alexa.

How we built it

AWS Lambda, Alexa

Challenges we ran into

Uploading files properly, learning Alexa skills, API calling speed, dynamic value insertion into skill slots

Accomplishments that we're proud of

Getting the API call to work!

What we learned

Python APIs, managing handlers

What's next for Accessibility Reader

Improve speed, add support for non-URL input (searching for articles on a topic, hearing options, and choosing one for Alexa to read), expanding or sharing article using Alexa’s follow-up mode
