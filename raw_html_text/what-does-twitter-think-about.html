Inspiration

We wanted a way of finding out what the average person thinks about anything and everything.

What it does

Searches Twitter for opinions on a given subject.

How we built it

It's a website which makes a request to a node.js server, which retrieves and filters tweets to find opinions.

Challenges we ran into

Twitter API issues, client-server communication

Accomplishments that we're proud of

The CSS, and the excellent functionality

What we learned

A lot of things about Node.js, JavaScript, and the Twitter search and embed APIs.

What's next for What Does Twitter Think About...

Alexa integration
