Inspiration

Un sistema de fácil acceso para jóvenes y personas adultas que mejora el sistema y agiliza los tramites.

What it does

vincula por medio de Bluetooth o codigo QR para la generación rápida de facturación y calculo de ganancias por trabajo.

How I built it

Se desarrollo un servidor, en el cual montamos una base de datos My SQL. Una aplicación en Android se conecta mediante Bluetooth o código QR con una interfaz(respectiva) con el que accede los datos para acelerar tramites.

Challenges I ran into

solo habia una persona con conocimientos basico de android, y la interacción entre todos los sistemas complico el asunto.

Accomplishments that I'm proud of

Lograr trabajar mas en equipo, y poder hacer casi todo el sistema basico en el horario establecido

What I learned

Desarrollo de software y vinculacion entre dispositivos Android-Bluetooth, Bluetooth-Arduino, Arduino-MySQL, Android-MySQL
