Inspiration

Pitt students often find themselves in search of a quiet and comfortable place to study. From personal experience we understand how difficult it is to find the right spot especially when considering needs such as available power sources and adequate space.

What it does

Allows users to search for study locations across Pitt's campus. Dynamic community driven data gives users accurate information on existing and potential study spots. Leverages additional information

How we built it

We created a responsive web app using the Flask Web Framework and various web Development tools

Challenges we ran into

Communicating between the SQLite and input from the user

Accomplishments that we're proud of

Effectively leveraging the Google Maps API.

What we learned

Front and Server side Web Development, Python object oriented Programming

What's next for Study Sight

We plan on continuing to develop Study Sight as a Web Application as well as migrating to a mobile App.
