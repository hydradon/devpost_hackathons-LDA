Inspiration

So with valentine's day coming up we thought what would be a good way for people who were linguistically challenged to tell their loved ones how much they love them. After that we thought maybe this could help with people who write haiku, poetry, music lyrics, etc. We named it Oz for the wizard of Oz giving people the gifts they truly wanted

What it does

it takes user input and parses what the user wants and checks our database for words that fit all their criteria.

How I built it

We decided to use JavaScript because we believe our application is more accessible if we host on a server and because we are currently learning it as well as to challenge ourselves.

Challenges I ran into

Synonyms -virtually impossible to do this by hand due to our limited time and our database is already too large and would have required server side knowledge that we did not have.

Accomplishments that I'm proud of

Creating an almost 5mb JSON file , helping to create this project, recovering from our disastrous beginning

What I learned

some of the neater tricks with javascript, cant recurse resources from other sites unless they specifically give you permission, relationship between syllables and variables, more organization required aswell as a thorough understanding of our ideas.

What's next for OzWords

other languages, synonyms, definitions and neater code
