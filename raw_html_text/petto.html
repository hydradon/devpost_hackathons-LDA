What it does

An adoption tracking system for a pet shop.

Adopting a pet - Check to see of the “petId” is in the range of our “adopters” Get the adopters Getting the adopted pets and update the UI Successful integration with Metamask and transacting using ETH

How I built it

Truffle, Ganache, Metamask, Torus login, Solidity, Etehreum

Accomplishments that I'm proud of

Integrating Torus login method.

What I learned

Run a fully functional ethereum app using truffle tools.

What's next for Petto

This is an MVP and can be modified for “Collectables” and for various other scenarios
