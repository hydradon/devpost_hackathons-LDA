Inspiration

We like Monzo and Alexa and we wanted to explore their facilities even further.

What it does

Users can set goals through the application or using Alexa voice commands. The transactions from the user's Monzo account are then analysed to detect expenses related to the user's goals. We are also using OCR to add receipt information to Monzo transactions.

How we built it

Application: android studio Back end: Python, Azure

Challenges we ran into

We had issues using Microsoft's API.

Accomplishments that we're proud of

A functional Android application.

What we learned

Azure and how to work with Alexa services.

What's next for Habit Tracker

Further statistics on transaction data.
