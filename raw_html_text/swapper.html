Inspiration

We were inspired by working multiple jobs that are shift based and struggling to have a good way to cover shifts with coworkers

What it does

See your weekly schedule. If the worker needs to have the shift replaced, they put their shift into the shift marketplace and it can be covered by a coworker. Then the manager or supervisor can either approve or deny the shift replacement.

How we built it

using a local server for our databases, lots of HTML, PHP and SQL statements for the implementation and CSS for its beauty.

Challenges we ran into

getting our local server up and running, securing our login information for the user.

Accomplishments that we're proud of

having a secured user logged in across pages

What we learned

-how to set up a server -access, create and modify databases -handling user login

What's next for Swapper

-from Web Application to native Android and iOS support
