Inspiration

What it does

Allows people to search for commonly needed utilities (restrooms, safe camps, electricity, etc)

How we built it

together :) using html5/css boilerplate

Challenges we ran into

time

Accomplishments that we're proud of

A map that shows places where people can use the restrooms, use wifi, charge their phone, take a shower, do laundry, drop trash A map that shows places people can camp safely Allows Providers to add themselves to the map as available support

What we learned

It's a good idea to set up your project the night before :)
