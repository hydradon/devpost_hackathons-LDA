Inspiration

What it does

A Google Assistant action that allows one to have true control over their fridge. Never again will you have to worry about what is and isn't in your fridge. We do that for you.

How I built it

We used Firebase, Dialogflow, and Actions on Google.

Challenges I ran into

The APIs we used did not return the values they promised to return.

Accomplishments that I'm proud of

Learning how to create a Google Assistant action.

What I learned

How to use Firebase, Actions on Google, and Javascript.

What's next for Ok Fridge

More functions and an mobile app.
