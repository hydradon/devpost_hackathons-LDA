My Finance 360

This is an application built for the First National Bank Code One Hackathon 2016 Event. The application provides a single portal view of the customer's financial data enabling an easy way to access the information from single source. It also contains visualizations regarding the financial behavior of the user while aggregating the data from several sources including the multiple financial institutions of the user.
