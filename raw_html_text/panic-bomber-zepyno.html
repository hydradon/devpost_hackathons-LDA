Inspiration: Playing similar games as a child

Gameplay: Two players, one using arrow keys and other using "wasd," compete to rack up points by holding a bomb. There is a built-in timer that displays once there is 5 sec left and that is when the two players try to get rid of the bomb so they don't get blown up.

Language: JavaFx

Difficulties: Had to learn how to use JavaFx (didn't have experience with it prior)

Knowledge Gained: Learned how to use JavaFx and created a working product

Future improvements: Make it a mobile app and make the game look nicer
