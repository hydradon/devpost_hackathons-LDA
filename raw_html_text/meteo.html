Inspiration

We wanted to create a automation house software, but because of the network restrictions we could not implement it.

What it does

It measures the temperature, humidity, light level and raining level and send it to thingspeak

How we built it

We build it using an ESP-32 micro-controller and some electronic components.

What we learned

We learned about ESP32 programming, wifi communication between micro-controllers and IoT coud platforms.

What's next for Meteo

The idea is implement a real version of the first idea, an automation house software, easy to configure. With our platform mostly everyone will be able to build his own automation house solution.
