Inspiration

We noticed that our grandparents have struggled with using their iphones. So we decided to create tools that would help other seniors learn how to use technology the right way.

What it does

There is a website that gives step-by-step instruction on how to text, facetime, and email on Iphone. For each step, there is a corresponding image. There is also a game that users can use to practice how to do each of those skills. They have to click the correct icon they would use in real life in order to complete the challenges.

How I built it

We used HTML, CSS, and Javascript for the website. We built the game using MIT App Inventor.

Challenges I ran into

There were a lot of screens to manage while constructing the game. We wasted lots of time managing trival things that should not have been a problem/

Accomplishments that I'm proud of

Building an impactful tool that will help many people.

What I learned

How to be an organized programmer.

What's next for Make Peace With Technology

We plan to add more use cases so people can learn to use all types of technology.
