Inspiration

I like difficult defies.

What it does

It deploys the BIRT report engine on JIRA and allows to upload, run and embed BIRT reports in JIRA seamlessly whereas all the limitations of running BIRT on a separated server are nicely overcome specially all the issues related to REST reports.

How I built it

Following the Atlassian plug-in framework patterns

Challenges I ran into

It has really been a investigation task to understand the BIRT runtime internals and the JIRA plug-in framework limits.

Accomplishments that I'm proud of

Resolve all the problems even whether they seemed impossible.

What I learned

The Atlassian's plugin framework limits and it looks like it does not have too much limits.

What's next for Birt Reports for JIRA

A version for JIRA Cloud.
