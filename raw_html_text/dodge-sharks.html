Inspiration

Shark week is coming up and we wanted to create an experience where users could interact with sharks in VR.

What it does

It's a game where users swim to dodge 3 dimensional sharks and get a score based on how long they survive.

How we built it

It's built in Aframe.

Challenges we ran into

Rendering new levels and objects proved challenging. We had to rebuild the game differently to capture the intended experience.

Accomplishments that we're proud of

Getting the 3d positioning of all the elements.

What we learned

How to create products in aframe.

What's next for Dodge Sharks

We're going to put up a new and improved version for shark week!
