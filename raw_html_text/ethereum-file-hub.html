• Ethereum.File.Hub

Inspiration

Remove 3rd party from media transfer.

What it does

Sends money directly to content creator based on how much bandwidth was used. Scenario 1): 0.5 ETH = 1 hour User watches 1 hour, 0.5 ETH transferred. Scenario 2): 0.5 ETH = 1 hour User watches .25 hours, 0.125 ETH transferred.
