Inspiration

Nobody likes forgetting their mouse for their laptop, so we got an idea to help out those miceless individuals.

What it does

Miceless is an mobile app and a desktop application, where users can control their desktop by interacting with their phone like a joystick, by tilting, or by gesturing like a trackpad.

How we built it

We built the mobile app using Google's Flutter framework, and used Electron, NodeJS, and RobotJS for the desktop app. The input data is transferred using websockets.

Challenges we ran into

We ran into a lot of challenges picking the right technology to send the input data to the desktop before finally settling on websockets. Also, configuring the desktop app to work on all operating systems was a challenge.

What's next for Miceless

Next up, we want to make it as easy as possible to connect a phone to the desktop app, perhaps through the use of printable barcodes or RFID tags.
