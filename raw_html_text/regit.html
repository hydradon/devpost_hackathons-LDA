We are a team of two developers from Krakow in Poland. We would like to present an application for managing git repositories. Git is an important part of every developer’s life and there is a lack of open source applications that lower a barrier to enter a git world. Regit is our idea to meet beginners expectations and to help them grasp the Git concepts.
