What it does

How I built it

Microsoft Azure cloud to make back end website (with Capital One Nessie API in order to charge dollar transaction), and XCode to make interface app for iOS in Objective-C

Challenges I ran into

-Website issues -Microsoft Band not responding -Project ot submitting

Accomplishments that I'm proud of
