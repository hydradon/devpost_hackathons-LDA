Inspiration

The great need of the millennial generation to track everything they do on the internet.

What it does

It keeps a record of what activities/workouts you did and a rough estimate of how many calories are burned based on the distance and time of the workout.

How we built it

We integrated Google Cloud App Engine and SQL to save the data on the server and retrieve it as needed. We used Android Studio to put together a simple but elegant user interface.

Challenges we ran into

Incredibly unhelpful documentation, our own inexperience with the technologies.

Accomplishments that we're proud of

Actually getting Java to connect to connect to CloudSQL with App Engine (very hard! ~8 hours).

What we learned

Using Snapchat's SnapKit on Android is awful. How to separate tasks so different team members can work simultaneously.

What's next for CloudFit

Supporting activities other than jogging, walking, swimming, and cycling.
