Inspiration

We are all VR advocates and wanted to create an immersive, educational experience for people that may not have access to the hardware to build robots.

What it does

TinkerXR is an app to help individuals learn to code robots.

How we built it

The WebVR component was created in Unity.

Challenges we ran into

Creating snap drop zones was really challenging. We wanted to provide the opportunity for users to design a robot and actually code it.

Accomplishments that we're proud of

Working as a team and being able to bring many technologies together.

What we learned

\We learned how to use Spark AR and we also learned how to have a Web VR interact with a React Web page.

What's next for TinkerXR

We add more programmable parts, a richer language, better user interaction and optimize the web page.
