Inspiration

A large number of people in big cities would like to enjoy sports activities but don’t have enough connections or people to play and engage with.

I personally experienced this problem when I first came to the U.S. I wasn’t able to find anyone to play ping-pong with. Even now when I want to find someone to go to the mountains to ski it’s a problem. Every time I want to involve myself in a sports situation, I don’t want to do it alone, and I keep wishing there was an app like this to find someone. And I am sure my situation is not unique, I think almost everybody faces this issue, especially if you are new in the city.

What it does

We provide a simple and useful online solution Web-based app where people can find each other based on sports interests. It’s fast, easy and safe.

How we built it

Sins we had limited time we decided to build it on WordPress and customize a theme in php. Also, we build Twitter, Facebook API bots to collect users information and engage them to register to our platform.

Challenges we ran into

Lack of time

Accomplishments that we're proud of

For 24 hours we built a fully functioned website so people can register and start looking for sports partners. A website is live now on a real domain so everybody can use it.

What we learned

We learned how to build Facebook and Twitter API on Python

What's next for Sweatmeet.com

Improving the web-based version of the platform and start working on a mobile app
