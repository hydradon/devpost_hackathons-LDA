Inspiration

We like making games where you can play with the person right next to you

What it does

It's a 2-person video game

How we built it

Developed in Java with the game development library LibGDX

Challenges we ran into

We didn't use a professional game engine like Unity, which was more fun to write and gave us more control but meant we had to write a lot more code

Accomplishments that we're proud of

Smooth animation system, multiple weapons and armor pickups

What we learned

It helps if you have a game idea before the hackathon starts :)

What's next for Minaret Game

We're going to add more weapons and release it to the public
