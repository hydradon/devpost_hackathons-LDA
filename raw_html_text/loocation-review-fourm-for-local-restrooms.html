Inspiration

Our friend notified us of a Purdue Instagram account which post pictures of good restrooms on campus. We thought that it would be a great idea to create an app that does something like that but on a large scale. The only way for locating bathrooms to be feasible on this scale would be to have it be crowd sourced.

What it does

Our application locates nearby restrooms uploaded by users, and allows you to view / posts comments about it. This is useful in determining if you should use this bathroom, or look for another one nearby.

How we built it

We used Android studio to build the application, Google Firebase and Cloud to handle the data, and Googles geolocation API to locate the users.

Challenges we ran into

Our biggest challenge we ran into was with trying to get the Google Cloud and Firebase services to communicate and send data to and from our Android app.

Accomplishments that we're proud of

We are proud of the fact that we actually finished and submitted our hack! We are also proud of our implementation of the user commenting section, so users can list specifics about certain Bathrooms.

What we learned

We learned a lot about Android studio, mainly with their material design features. We also learned how to implement components from Google Firebase into Android. We also looked into implementing features from Google Cloud into our Android Project.

What's next for LooCation

There are a lot of features that can be implemented if we had the time, such as a like/dislike rating system, a moderation system, or an interactive map of buildings. In addition, the possibilities of adding pictures.
