Inspiration

What it does

Nearby Minds is a Google Chrome extension that connects you to interesting people at the events you attend. Utilizing graph theory and Facebook API, it displays new people that you might be interested to talk to in a noninvasive suggestion box. Our main goal was to integrate this functionality in a seamless way, and by creating a Chrome extension, we have been able to achieve that.

How we built it

Nearby Minds is powered by the Facebook Graph API and Parse, while the Chrome extension is built with Javascript - with minimal performance impact.

Challenges we ran into

Because of Facebook's various optimizations, it was difficult to create the Facebook extension, but we have been able to implement it seamlessly with the Facebook experience.

Accomplishments that we're proud of

Working with Facebook API and Parse, creating the seamless experience with the extension, and, of course, the algorithm that powers it all.

What we learned

We all learned many things about web development, working with APIs, and creating web browser extensions.

What's next for Nearby Minds

Publishing our Chrome extension to the Chrome Store!
