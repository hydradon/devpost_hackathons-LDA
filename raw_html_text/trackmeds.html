Inspiration

I sometimes have trouble remembering whether I've taken my medication. I wanted a simple, electronic way to record whether I did.

What it does

Trackmeds lets users sign in and list their medications. A user can select which medication they've taken, and the system will record it with the time and date.

How I built it

Trackmeds is a simple Ruby on Rails and Materialize CSS application.

Challenges I ran into

I had some trouble getting forms to work with both Rails and Materialize CSS.

Accomplishments that I'm proud of

The site works and is deployed.

What I learned

I learned more about how Rails handles relationships between different models, and more about how Rails does routing.

What's next for Trackmeds

SSL encryption and visualizations of medication usage are planned.
