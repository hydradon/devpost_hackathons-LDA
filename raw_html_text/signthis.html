Inspiration: Some of our team members were interested in learning ASL and after brainstorming ideas we came up with SignThis

What it does: SignThis is a way to have live translation of ASL

How we built it: We used the Clarifai API, JavaScript, CSS, and HTML.

Challenges we ran into: We ran into several challenges while working on the front-end and back-end. There were times that we weren't sure how to approach the different parts of the code, and other times we had syntax errors which needed to be debugged.

Accomplishments that we're proud of: For some, our first hackathon, and creating a webpage using JavaScript and HTML for the first time.

What we learned: We learned how to use APIs and how they can be helpful and also how to develop a webpage

What's next for SignThis: Have a broader database for SignThis to use so that it can recognize actual phrases as well as have it compatible with videos as well as images.
