Inspiration

To empower people to become more conscious of their media consumption.

What it does

Based on your browsing history, it analyzes what the credibility of those sites are. By doing this, we are hoping to increase media literacy, by being able to be conscious as a media consumer.

How we built it

We created a database of media outlets and used Python to scale the data.

Challenges we ran into

Learning to build a chrome extension, as well as creating the database with advanced statistical methods with Python.

Accomplishments that we're proud of

We are proud of addressing real, tangible issues, while also exploring technologies in data analysis with Python. We're also proud of being able to create a cohesive user experience that users will enjoy.

What we learned

We became more familiar with advanced statistical methods such as scraping web pages and curating data into information.

What's next for Spectrum

As we amass more user data, we can begin to analyze trends surrounding media consumption habits, the pervasiveness of filter bubbles in our society, and preferred media publishers. By interpreting this big data, we will have access to a holistic view of the media industry and how news is consumed.
