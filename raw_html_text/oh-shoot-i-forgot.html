Inspiration

I forgot my phone charger on the way to DC Hacks and I needed to find a nearyby store to buy one.

How it works

Users enter in the desired product and on the map in the application, markers appear representing stores that sell that item.

Challenges I ran into

Implementing an appropriate search api that gave accurate data regarding stores.

Accomplishments that I'm proud of

We actually finished MVP for the android version of the application and are close to finishing it for ios.

What I learned

We learned how to use the Yelp and Google Maps api for Android.

What's next for Oh Shoot I Forgot...

A fun social media integration highlighting what users have "forgotten" over the days.
