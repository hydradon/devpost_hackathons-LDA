This is UiPath Outlook Custom Activity. This Custom Activity can be used to send Mail with embedded image or send Dynamic Number of Attachments.

There are two activities in this package:

  1. Send Outlook Mail Activity Details: This activity is used to send Mail Message from Outlook with Dynamic number of attachments at runtime. Use: If you want to send all text files from a folder.

2. Send Outlook Mail with image This activity is used to embed images in the mail body. Use : If you want to send Exception Screenshot embed the email body or Embedded Company Logo in Email Signature.
