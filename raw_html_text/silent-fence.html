Inspiration

A common problem we found at our school was students having their work copied or their privacy invaded without their knowledge due to wandering eyes landing on their Computer. We set out to solve this.

What it does

Silent Fence alerts the user when multiple faces are staring at their screen. It also allows the user to configure the program so that certain recognized faces are permitted.

How I built it

We built the project by integrating existing facial detection libraries and combining them with facial recognition.

Challenges I ran into

Installing the libraries and getting them to work properly on our different operating systems proved very challenging.

Accomplishments that I'm proud of

We are proud of how we configured the program to not only detect multiple faces and display a user-customizable warning but to allow for certain faces to be permitted as well.

What I learned

I learned how to work with opencv and use existing facial recognition libraries.

What's next for Silent Fence

Silent Fence could have many applications and we would like to configure it for mobile screens as well.
