Inspiration

Eating lunch alone is boring- a lot of students complain about how they have to do it since matching school time is hard.

What it does

Social network that allows students to find other available students and nearby restaurants.

How I built it

We used python flask for the website, Twilio for notifications and mongo as a database

Challenges I ran into

Making the site very user friendly
