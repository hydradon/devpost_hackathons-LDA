Inspiration

Before doing software I was drafter and I had to develop and provide support by creating and updating 2D & 3D technical CAD drawings requested by “Kim Lighting” brand. Some of the new assemblies required new hardware and a very common one that Fusion 360 was missing are the hex nuts. Also, I always wanted to know how CAD worked on a software level.

What it does

Simple and easy to use Metric Hex Nut Generator that allows you to switch between the most popular Metric Hex Nuts configuration, and if you want something different you can customize yours!

How I built it

By entering the Fusion 360 API world and reading all the samples that come when you download it.

Challenges I ran into

The Fussion 360 API was a little hard to navigate on, since you can not use your browser "press to go back" feature. :(

Accomplishments that I'm proud of

After a lot of reading I finally had a working product that is really simple to use.

What I learned

I now have an idea of how big CAD software might work from the inside in a software level

What's next for Metric Hex Nut Generator

I really liked this project and I think it can still have some improvements that can help CAD passionate people (like me)
