Inspiration

Sometimes, you need to do a presentation and you don't really have to time to do research and collect data. FlyNotes collects live data as you speech, so you don't need to prepare anything. Just talk, and we output credible data source for you and your audience on the screen.

Presenter and audience can join the website to see the live data of the presentation.

What it does

It uses Chrome Webspeech Api to convert the user speech to text. Then we submit the text to IBM Watson Alchemy API to get most important keyword from the sentence. We use that keyword to query data from Wolfram Alpha. Using websocket, everything happens in real time.
