Inspiration

Always been real passionate about Augmented Reality since Inception.

What it does

Imagine you move into a new Apartment and wanna buy furniture and when you go to the shop and start selecting the furniture, at that point you are not sure if that will suit your place or not. We help eliminate this problem, just put our app on the phone, select the models and place anywhere you want and check how it would look like.

How we built it

We built it using Unity, C#, Google Ar Core, Visual Studio etc.

Challenges we ran into

UI was big thing because Unity doesn't support native android UI so we had make it from scratch so that was tough

Accomplishments that we're proud of

We are proud of putting together a real app that people can use in 33 hours.

What we learned

We learned Multi-Threading in C# & sharpened our skills in unity.

What's next for Interior Design With Augmented Reality
