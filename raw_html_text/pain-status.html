Inspiration

Give the NY presbyterian patients a way to communicate with others and Nurses

What it does

Our web app is very basic and only has the ability to select a pain level

How I built it

We used firebase as our back-end and javascript. Some bootstrap so we did not have to worry about much design.

Accomplishments that I'm proud of

What's next for Pain Status

The idea was to make a web app that provided chat, games, and pain notifier. If we were to improve it, it would have many features patient could use to be entertained, talk to others, and a way for nurses to better know what is going on based on what the patient tell them in a centralized location.
