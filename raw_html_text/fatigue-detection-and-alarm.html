Fatigue driving is life-threatening.

It uses computer vision and machine learning algorithms to detect whether a person is fatigued. It sends an alarm information through the websocket to a PHP application to warn the driver with alarm and provide nearby hotel information using Google Map API.

The most challenging problems that we encountered is how to define fatigue based on the computer vision results. How to communicate with a web application from a local application. We captured and analyzed the eye blink event of the driver so detect whether he/she is too sleepy and might need some rest.

We created a functional project that can warn the fatigued driver in a very short period of time.

Github is a good thing. Computer vision and websocket communication.

Create a prototype of the system and make it available to the public.
