Inspiration

We wanted to build an app that would utilize arcGIS software to quickly locate desirable real estate on a map.

What it does

Locates available houses for rent/sale in counties that match user input.

How we built it

The majority of the app is built in AndroidStudio with Java and HTML. Use of the arGIS SDK.

Challenges we ran into

Clickable button overlay on map.

Accomplishments that we're proud of

Collaborated with three team members through GitHub to develop working software.

What we learned

How to create an arcGIS map and develop clickable overlay.
