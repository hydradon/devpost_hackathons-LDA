We got inspired by the idea of providing education to children all over the world.

EdifyConn provides an interactive teaching platform for students

We built it by using JavaScript, HTML5, CSS3, PHP, bootstrap for front-end and MySQL on the back-end

We faced challenges while integrating some of Microsoft cognitive API's.

We were able to provide secure login using facebook login API

We learned many new things on how to integrate API's, How to develop an interactive website.

For future enhancement forEdifyConn we would like to add some cognitive APIs available in the market to make learning more interesting for students.
