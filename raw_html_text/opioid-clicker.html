Inspiration

Our inspiration came from the addictive Cookie Clicker game.

What it does

It is an incremental clicking game where the user addictively clicks on the drugs and faces the consequences.

How we built it

We collaborated using an online server called SyncFiddle.net, programming in HTML, CSS, and Javascript, and having illustrations done in Adobe Illustrator.

Everything.

What we learned

We learned about each other's deep dark secrets. And also, how to program in Javascript.

What's next for Opioid Clicker

Making it better! We want to make the game get more gloomy as the user "intakes" more drugs portraying the symptoms of overdose.
