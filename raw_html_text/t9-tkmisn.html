Inspiration

We believe that access to computer is one of the fundamental thing people should have. Nothing should limit its accessibility

What it does

We have used Google Cloud Platform’s Speech to Text API to identify key phrases from the microphone. It is a python script which can be run on any platform.

How I built it

We used Google Cloud Platform’s Speech to Text API for natural voice recognition. Finally, we used the workshop Intro to Python’s guidelines to implement the project, also some libraries.

Challenges I ran into

The majority of the team was not familiar with python, which caused roadblocks during the the hack. We also had to work through a lot of python libraries. We managed to overcome them through proper team management and extensive research.

Accomplishments that I'm proud of

While also our largest challenge, the work we had to put in to learn about machine learning, was our best accomplishment. We were really happy to have completed a product from scratch with almost no prior knowledge of the tools we used to build it.

What I learned

• The usage of Google Cloud APIs and various ways to implement them.
• How to cooperate as a team and brainstorm effectively even coming from different discipline
• How to train a machine learning model
• Also, we learned about the poor advancement of technology for users with disabilities

What's next for Odyssey

If we had more time to we would have implemented more features to make it more user-friendly.
