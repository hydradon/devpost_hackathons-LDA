inspiration

This was inspired by a project called hotdog not hotdog by a character(Jian-Yang) in the HBO show Silicon Valley

What it does

It determines whether an object is a burger or not

How I built it

I made all the front end using react and the back end with Javascript

Challenges I ran into

Connecting the app to Firebase and sending a file to Google cloud storage and the Cloud Vision API

Accomplishments that I'm proud of

I was able to make a button to pick whether or not somebody could pick a hotdog or not

What I learned

How to create an app that connects to and takes advantage of the google cloud vision AI api

What's next for BurgerNoBurger

Accurately determining whether an item is a burger or not and becoming a suite of products that let people know which items are what.
