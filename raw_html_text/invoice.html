Inspiration

Smart Banks and IoT are both being used more frequently. Our program gives you the ability to check your capital one transactions from your Google home device. It would also be able to set budgets for the month and notify you once you went over, as well as send money between other capital one accounts.

What it does

How we built it

We utilised the Google cloud API as well as the capital one API to give us the required account information. The cloud API meant we could perform speech to text and vice versa more easily.

Challenges we ran into

Voice to text is hard and pyaudio can be tricky to implement.

Accomplishments that we're proud of

This is something that genuinely is not in the market, and showed a proof of concept we would like to develop upon

What we learned

How to use text to speech and the basic layout of the capital one API
