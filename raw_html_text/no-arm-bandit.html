Intro

We have created a super awesome retro slot machine - and it works! A slot machine is a type of arcade game where the player must match three of the slots to win.

Armed with only a junior hacksaw and lots of Poundland duct tape, we somehow managed to throw our project together from scrap that we found lying on the streets. We even resorted to negotiating with some local builders for items from their skip.

Amazing Features

Hands-free Operation using a wireless hand-motion sensor, a bit like flushing the toilets you find in shopping malls; Eco-friendly as it is made from reclaimed scrap materials; LED Matrix that performs routine sequence on win; Fantastic Prizes but only when the candy release mechanism works (which is most of the time to be fair).

Challenges

The main issue we faced was that we were only able to obtain 2 of the 3 servo motors required for spinning the slot wheels. We were able to overcome this by adapting the firmware to run another type of motor.

We also experienced problems with the LED matrix drawing a large amount of current which affected the accuracy of the slot positioning and calibration.
