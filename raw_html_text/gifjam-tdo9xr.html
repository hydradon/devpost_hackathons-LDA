Inspiration

We wanted to use GIFs and do something with media. We also wanted to have a somewhat comedic project while still doing something technically nontrivial. This was the result.

What it does

User requests a song and GifJam will take it and grab the audio as well as GIFs that match the lyrics. It then creates a music video of these GIFs in sync with the lyrics.

How we built it

With python and some libraries.

Challenges we ran into

Lots of I/O errors and semantic issues.

Accomplishments that we're proud of

It works.
