PiClique

This code in this repository is designed to help hospital workers better help their patients by being more aware of their vital signs.

The Python script takes in inputs for blood pressure, heart rate, temperature, and respiratory rate. It then clearly displays these values on a raspberry pi via command line app. When one of these values indicates that the patient is in danger, the screen flashes red and indicates that there is a problem.

In addition there is a website under a domain.com domain that receives real time data from Firebase database and displays it in a real-time graph. The graph displays the patients heart rate, using data from an application on an android phone that records a person's heartbeat. In the final version, the database will connect to the raspberry pi and display the user's heart rate on that.
