Inspiration

We wanted to develop an Augmented Reality game, that was simple and fun. And who doesn't like rubber duckies and zombies, we figured it was fun idea.

What it does

Using an ios device's rear facing camera, it sets an anchor point for the zombies to spawn. The zombies approach the player, while he/she is attempts to hit them with rubber duckies.

How we built it

We built the project using the Unity game engine, and the Vuforia AR library.

Challenges we ran into

Learning how to use Unity in 3D and Augmented Reality in 24 hours. A lot bugs were created, squashed, and then reappeared later on. But we eventually fixed them.

Accomplishments that we're proud of

We implemented AR technology into a game, in very short period of time.

What we learned

We learned many aspects of Unity 3D and Vuforia. Very cool stuff, and we'll definitely continue working with this technology.

What's next for AR-Zombies

Not sure, but we do plan to continue work on it.
