Inspiration

We were inspired by the prompt to create an app to save lives in the event of an emergency, such as the earthquake in innovation city.

What it does

Users can login via Snapchat and mark themselves safe or unsafe in the event of an emergency. If the user is unsafe, they can indicate their location so law enforcement can find them more easily.

How we built it

We used Angular, JavaScript, HTML, CSS, and Adobe Photoshop, and SnapKit to build our project.

Challenges we ran into

We had difficulty importing the SnapKit. We had difficulty because we did not all know how to use the necessary tools to build our app, but through trial and error and teaching each other we learned new skills. We also had difficulty figuring out how to all work on the project at the same time.

Accomplishments that we're proud of

We're proud of developing a working application at our first Hackathon! We also used new languages and worked well as a team, even though we just met.

What we learned

We learned about using JavaScript and Angular, because not everyone in the group had used it before. We also learned how to collaborate on an application with other teammates using GitHub.

What's next for SnapSafe

Our next idea is for users to be able to add their symptoms or injuries in case they are in need of immediate medical assistance. This will allow medical staff to be better prepared for treating patients.
