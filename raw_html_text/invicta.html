Invicta

Is a simple automated Web app for Encrypting and Decrypting Whatsapp messages Directly . It takes receiver name and message , authentication is done using whatsapp web authentications to encrypt messages. And it takes only senders name and authentication to decrypt the received Encrypted messages It uses custom encryption algorithm for Encryption and Decryption of messages.
