Inspiration

When reading the news, it can be hard to keep track of all the people, locations, and things. Our chrome plugin lets you view New York Times articles in a more vivid way.

What it does

Through Natural Language processing we identify important words of phrases in an article. Once these are found, we allow the user to hover over any of them to view more info on them.

How we built it

We use a server provided by Linode to run a REST api that allows our chrome extension to communicate with our Natural Language implementation in Python.

Challenges we ran into

Merging the annotated article without punctuation with the regular article.

Accomplishments that we're proud of

It works end-to-end.

What we learned

Natural Language processing.

What's next for Informed

Allow sites other than NYT to be annotated.
