Inspiration

YouTube and other online video content channels have a humongous amount of content for kids. There are millions of kids watching these videos and learning every day. However, just a video is not as interactive as a teacher. Echo on the other hand, can make the entire experience more interactive. It levels up the experience of learning.

What it does

This skill will give the kid an option to choose from various topics, what he would like to build. He can learn the following :

  1. Body Parts
  2. Animal sounds
  3. Shapes
  4. Months in a calendar
  5. Days of the week

How we built it

We built the entire skill on node.js and used skill builder portal to define the interaction model.

Challenges we ran into

Handling multiple intents for single utterance by implementing session persistence.

Accomplishments that we're proud of

We are proud of implementing session attributes successfully and giving a wholesome learning experience with really articulate interaction schema.

What we learned

We learnt various directives of alexa skill kit and the importance of session attributes. This challenge helped us explore each and every feature of Alexa. Building the same skill on Echo show and dot was a challenge, but we are happy we could handle it.

What's next for Kids Skill

We want to create a more comprehensive curriculum and teach them rhymes, fruits, vegetables and a lot more subjects using this skill. We want this skill to be the one stop solution for everything a kid needs to learn.
