Inspiration

Want to build a website.

What it does

We were trying to build a web app where images for news articles are listed, and the person chooses which article they want to read based on their image choice. When they click on the image only the text of the article is returned.

How we built it

We used html and GitHub for the website version. We tried using python, flask and python anywhere for the web app version.

Challenges we ran into

Learning new languages, html, python, java. Learning components of web programming.

Accomplishments that we're proud of

We're proud of challenging ourselves with a project that used tools which we have very little experience.

What we learned

We learned a lot about the different ways you can build a website and web app.

What's next for display_web_image
