Inspiration

I am inspired by the possibilities represented by voice-activated devices with interactive screens.

What it does

This Skill teaches the user about a specific topic using an engaging and intuitive interface.

How I built it

Integrated APL into an existing Alexa Skill

Challenges I ran into

Designing for different screen sizes and voice-only devices.

Accomplishments that I'm proud of

Developing a full working model with a visual interface, 12 chapters, and functional navigation.

What I learned

How to develop for multiple types of Echo devices and present a fully functional topic presentation with APL.
