Inspiration

The main reason that inspired this idea was the constant food waste experienced at the FIU campus by us.

What it does

We built a station that takes a picture of people's leftover food to be shared on social media and later picked up by hungry students/faculty staff for free.

How we built it

We used a Qualcomm dragonfire microcontroller to control the webcam and do the automatic Instagram posting.

Challenges we ran into

Automating the instagram posting, using a new microcontroller, and make a stand for it.

Accomplishments that we're proud of

Finding a meaningful project and being able to fully implement it.

What we learned

Most micro-controllers operate very similarly. Used python for the first time.

What's next for FoodForAll

Use machine learning to classify food and send emojis accordingly.
