Inspiration

Current hotels loyalty programs are simple, volume-based, and not very effective

What it does

Targeted Guest Loyalty

How we built it

Web based Service allowing Property Managers to login and view Guest Value, Index and reports Travel Data collector runs periodically to gather information from various sites. (Snapshot now, but Expedia and booking.com in future) Guest Value Calculator computes the GVI and keeps it ready in datastore. jQuery based UI to allow Property Managers to view and send promotion offers to valid guests.
