Inspiration

A lot of times when my group mates and I purchased products for any reason, we ended up just giving up because the whole return process was too much of a hassle.

What it does

As soon as you log in to the app. Select the item you want to return from your order history then fill in the form to give context about why you are returning the item. Once form is submitted a driver is requested to pickup and estimated time till arrival is displayed

How we built it

we built both a web app using JavaScript and a mobile app on android with android studio. our server is run with ngrok and express. And for getting location working we used the following APIs: google geolocation API(to get current location), google distance matrix API(to calculate distance from driver), google maps API(to display google map), geocode API(to convert coordinates)

Challenges we ran into

it was hard developing on two separate platforms. Android studio was also hard to work with

Accomplishments that we're proud of

We are proud of sticking with our project till the end and finishing despite all the obstacles we encountered

What's next for PostFix

We plan to partner with postal services in order to get data and integrate the customers orders .
