Inspiration

We were inspired by Big Hero Six, particularly by the various gadgets that Hiro had built for his friends in order to save San Fransokyo from the evil super villian.

What it does

It is a remote controlled robotic claw that can grab various objects.

How we built it

We used a robotic arm that contains 5 servos to simulate an arm and by the power of electrical engineering, all servos were able to function with adequate power. The arm is controlled by an arduino connected to a leap motion sensor and run by JavaScript.

Challenges we ran into

On the hardware side, the servos drew too much current from the power source. Three power sources were utilized in the end in order to solve the overdrawing of current. The Li-Po power source fluctuated voltage greatly during use so we switched it out with a 5V power supply module. Technical issues occured with the kinematic equations used to move the arm. This was later resolved by bounding the output values of the servos.

Accomplishments that we're proud of

Result: Arm runs smoothly with Leap Motion.

What we learned

JavaScript Current draw may exceed what power source can provide with required voltage.
