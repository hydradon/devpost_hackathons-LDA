ProxyHunt

Inspiration

We all had one thing in common, we didn't like waking up early for classes. So voila!

What it does

This is an app which allows students to look for other students who are willing to take their iClickers to class for attendance.

How we built it

We programmed in Java using Android Studio. We used Firebase to handle the backend of the program.

Challenges we ran into

Never using Firebase before. Deleting records in Firebase using the app.

Accomplishments that we're proud of

Started from the bottom without much knowledge of Firebase but managed to submit a working product. Kudos to us!

What we learned

Firebase, for sure!

What's next for ProxyHunt

Releasing it in the play store and making an iOS and web app for the same. (Disclaimer: no need to wake up for classes from now on) :D
