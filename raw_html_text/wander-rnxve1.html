What it does

Wander matches your likes and interests based on images with potential vacation destinations, from familiar towns to exotic places you've never heard of. In addition, Wander lets you take an instant vacation - you can book travel and hotels with the push of a button.

How we built it

We used Ionic for the front-end and Loki for a client-side database. We also used Flask for some back-end functionality.

We also use the following APIs:

  • Uber API

  • Capital One API
