Inspiration

As college students, going out to eat with friends can be a pain when it comes to splitting the bill. Thus, Split aims to simplify the process of calculating individual prices and tips.

What it does

Split scans the receipt and processes the items, quantities, and prices on that receipt. Then, it leads the user to an interactive sorting page to split the bill between a specified number of people. Finally, it calculates total price for each person.

How we built it

We used Android-Studio to build our app.

Challenges we ran into

We had trouble connecting Android-Studio with Git. This seemed like an occurring problem for many Android builders, so we had to copy and paste the code instead of using VCS.

Accomplishments that we're proud of

We are proud of being able to learn how to search for tools that could help us build this product because Android-Studio was being very buggy.

What's next for Split

We hope to increase the functionality of Split.
