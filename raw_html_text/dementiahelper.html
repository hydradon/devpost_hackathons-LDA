Inspiration

There are many mental illnesses out there and dementia is one of them. We want to create a simple but effective application for people with dementia to use.

What it does

Users are able to add an unlimited amount of locations. From their current position, users can find a path to travel to their location.

How we built it

Using Android Studio, we created the GUI and the back end.

Challenges we ran into

Because this was the first time that both of us are coding an Android application, none of us knew what we were doing at many times.

Accomplishments that we're proud of

Learning how to use Android Studio.

What we learned

We learned how to create a basic Android application

What's next for DementiaHelper

We want to polish up the app with more features to get it ready for deployment.
