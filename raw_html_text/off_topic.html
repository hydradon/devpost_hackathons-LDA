Inspiration

Have you ever wanted to go for an event but had no one to go with ? Have you wanted to grab a bite or a drink, but none of you friends were free ?

What it does

It leverages social media data to pair people with common interests and similarities. So we can match people, who are free at a particular time

How I built it

Built with JS, on the server and client http://offtopic.cloudapp.net/

Challenges I ran into

Understanding, how to recognise that GOT , G.O.T and Game of Throne is the same thing Have an efficient front end, so we could be cross platform How to get a feed of events around campus

Accomplishments that I'm proud of

We built it to work n every platform that has a browser

What I learned

I learned a lot about using the FB api, which i used for the first time,

What's next for Off_Topic

We plan to integrate our school api. it takes 4-5 days to process and get an aproved key

We plan to work on our ML and User experience
