Inspiration

Helping maximize customer cash back.

What it does

Two part application, with excel tool to help customer choose a cashback card/cards that maximizes their savings based on their spending history. Also

How I built it

VBA for coding in Excel. Ruby on Rails, HTML, and CSS for the website. Android studio and Java for the Android app

Challenges I ran into

Effectively simulating a live transaction in the Android app.

Accomplishments that I'm proud of

Complete algorithm to choose the best options for the user based solely on their transaction history and the credit cards they currently own.

Proof of concept for an app able to classify transactions in real-time and choose which card of the users to use.

Website for obtaining and analyzing results from the VBA script.

What I learned

How to use VBA to make analytic tools and solve difficult optimization problems.

What's next for StreetCredit

Improve Android app such that it communicates and pulls transaction data using Google pay.

Expand the scope of the VBA script to allow for a wider variety of problems (Corporate credit cards, Contracts etc.)

Improve website analytical tools by embedding descriptive charts/graphs
