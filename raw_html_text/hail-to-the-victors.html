Hail to the Victors!

An Amazon Echo skill that allows you to ask Alexa to chant the University of Michigan's fight song, "Hail to the Victors!" Submitted to MHacks X on September 24th, 2017

Why I Chose This Project

I chose this project primarily because I wanted to learn how to develop Skills for the Amazon Echo and gain experience using Amazon Web Services, specifically AWS Lambda. I am also a freshman Computer Science Engineering student at the University of Michigan, so it allowed me to show my pride as a Wolverine!

What's Next?

Hail to the Victors! is currently being considered for certification by Amazon to be added to the list of Alexa Skills. I intend to continue building Skills for the Amazon Echo now that I have been able to take the time to teach myself Amazon Echo Skill development and understand Amazon Web Services.
