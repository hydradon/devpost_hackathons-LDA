Inspiration

Prudential Bank provided us with a challenge to complete during HackRU. This is our attempt.

What it does

Using sample bank customer data, we created a website where you can log in with your bank credentials and it will show you your various account balances, expense vs. income data, and the ability to manage assets.

How I built it

We used Flask for python and the Yodlee API to create our webserver, and HTML/CSS/JS for the frontend.

Challenges I ran into

The biggest challenge was using the Yodlee API. We spent a couple of hours learning how it worked, but after that, it was pretty easy to catch on to.

Accomplishments that I'm proud of

Our website looks nice.

What I learned

Previously we had only worked with Node.js for web app backends, so we learned how to use Flask and gained some experience with python.
