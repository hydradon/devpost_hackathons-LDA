Inspiration

Was interested in helping people learn English as a few people on our team were not native English speakers and was intrigued by learning how to use speech recognition.

What it does

Takes voice input and tells you if your pronunciation is correct. There is also a dictionary function to define words.

How I built it

Used Android development to make an Android app with Java and XML.

Challenges I ran into

Did not know how to make Android apps before coming to HackDuke. Had to learn everything as we needed to create it.

Accomplishments that I'm proud of

Created a working Android app. Speech recognition works.

What I learned

Learned how to make Android apps.

What's next for Absolute Word

Make the application more dynamic and add extra functionality such as user login for cross platform learning.
