Inspiration

The internet is a powerful place of knowledge, connection, and entertainment. However, some people cannot access the internet without a mouse. Sermos solves this problem by having voice controlled commands!

What it does

Sermos is a chrome extension that recognizes voice commands such as opening new tabs, going to a website, closing a tab, closing all tabs, and even depositing money through the CapitalOne API. Sermos is constantly running and can be disabled.

How we built it

Sermos uses HTML5's getUserMedia's API to extract and understand the speech of the user and do what the user wants. A background script is constantly running in the background to listen to commands.
