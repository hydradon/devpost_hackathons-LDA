Inspiration

Waste sorting is always a complicated task for residents and sorting rules vary from country to country.
GOSorter aims to give the user an explicit guide on how to classify garbage according to the recycling system.

What it does

Helps sort different garbage types, both through a manually operated menu and through an automatic, smart search online based on the item's name.

The main features of GOSorter are:

  • User-friendly interface

  • Support drop-down trash selection

  • Provide a user-defined option

  • Follow the local garbage sorting policies

  • Intelligent material search online if no results found in the database

How we built it

Using Python 3.7 and Kivy as the UI. SQL is used for a simple database.

Challenges we ran into

Adapting the SQL to our needs, trash sorting, implementing smart search online.

Accomplishments that we're proud of

Working product. Eco-friendly. Smart.

What we learned

Teamwork.

What's next for GOSorter

Smarter iterations.
