Inspiration

Why waste time in line when you can quickly test your own eyesight in 30 seconds or less at home? We designed this app to allow for efficiency in vision testing.

What it does

iDoctor tests your vision by displaying random letters in decreasing size from 20/70 vision to 20/20 vision. The app asks the user if the correct letter was identified, and if not, displays the corresponding vision score.

How we built it

in XCode, using Swift 3.0

Challenges we ran into

learning how to use Swift and XCode, debugging

Accomplishments that we're proud of
