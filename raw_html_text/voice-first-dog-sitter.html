Inspiration

My dogs are the inspiration of this skill. Me as a dog owner want to keep my dogs well taken care when I am away from home.

What it does

This skill helps the dog sitter or the person that is taking care of your dog to have all the info that you know about your dogs. He just has to ask alexa how to do things for your dog and It will respond with the info that the owner previously set up in the skill.

How I built it

I used the Javascript to create a Lambda function hosted in AWS to fulfill the alexa intents. Also, I created a DynamoDB table to store the info about the dogs.

Challenges I ran into

Many. First learn about Voice design. Learn about Alexa Ask Cli. Plan and handle all the ways a user can ask for a task.

Accomplishments that I'm proud of

I am proud that I have learned this great technology which I think is the future. I am proud that I created my first Alexa skill.

What I learned

I have learned a lot about voice design. I have learned how to create an Alexa Skill and all the little tricks that only experience gives you when you start a project.

What's next for Voice First Dog Sitter

I plan to release a companion mobile app so the user can add more info to the DB from there. I plan to release more intents that the users can ask about their dogs.
