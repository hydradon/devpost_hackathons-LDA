Inspiration

To help sad or depressed people to change their thinking and their looking at things and being more mindful about beauty in life

What it does

It takes or captures an image and classifies it as sad or happy image and then displays quotes or images that help to change the person's mood. If the image has happy feeling then a smiley face is displayed and yay sound is run.

How I built it

We built it using Matlab and its deep learning toolbox. We assembled the whole code into a GUI.

Challenges I ran into

Collecting the data to train the model for classification, cleaning the data, and training the model.

Accomplishments that I'm proud of

Using deep learning to develop application that helps people to change their mood.

What I learned

We learned to make deep learning model to classify images.

What's next for BeHappy

The model will be further trained with more data to enhance its accuracy. More emotions will be added to the model. The application will analyse the image and produce a close image that has much similar contents to the input image and similar color but with a happy content to help person change his thinking into things. The application will display quotes that are relevant to the input image.
