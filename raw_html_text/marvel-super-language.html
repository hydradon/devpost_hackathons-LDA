Inspiration

Our team member Stanley wanted to build something effective and interesting using a unique API provided by Marvel, making it easy to use for kids and families alike.

What it does

Marvel Super Language is a web app for learning new languages developed with the superhero theme kept in mind. Not only will the user be able to take quizzes featuring Marvel characters within the content and the GUI, but there is also a special super-chat feature that allows you to communicate with the characters themselves!

How we built it

We built this web app on javascript, node.js, jquery, the Google-Cloud-Translate API, Dialogflow, Firebase, and the Marvel API itself.

Challenges we ran into

At first, half of our group was comfortable with web development, while the other half preferred IOS. We eventually settled for a web app built with javascript. One of our group members had no web development experience, but he adapted quickly. Building the layout of the quizzes and making our online references robust were especially tough and time-consuming.

Accomplishments that we're proud of

We are proud that in the span of nearly two days, we not only developed something unique, applicable, and fun to use, but also the additional social connections we have made and the ever-increasing exposure to the cutting-edge hacker-technologies of today.

What we learned

We have learned to use new APIs, frameworks, and even new languages that not only increases our rate of learning, but also helps us build even cooler projects in the future.

What's next for Marvel Super Language

Additional quizzes for the languages, more intuitive GUI, secure web connections, and audio features are all included within plans to improve Marvel Super Language for the future.
