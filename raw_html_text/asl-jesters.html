Inspiration

Wanting to build a program which uses new hardware, that can have a positive social impact.

What it does

The program recognizes American Sign Language (ASL) signals to help teach people how to communicate via American Sign Language.

How I built it

Used position vectors from the hand when the user does a signal and compared it to stored letter position vectors and determined the letter that is being signalled or if an invalid letter is being signed, the program would display an error.

Challenges I ran into

We faced many challenges during the building process. Challenges include:

  • Understanding Leap Motion API
  • Determining an algorithm to figure out what letter the user has signalled

Accomplishments that I'm proud of

First and foremost, I'm proud that it works. As well, this is especially fulfilling because this has been the first hackathon and hardware hack for our entire team.

What I learned

Learned how to use Leap Motion API.

What's next for ASL Jesters

Expand on current project by increasing possible signalled letters to the entire alphabet, numbers, symbols, and words. Then the project will be made into a fun and interactive game to help teach people American Sign Language.
