Inspiration

We want to connect the members of Facebook Developer Circle community and help new member connect with DevC Expert. The Power of Chat!!

What it does

DevC Chat is 1 on 1 chat application for connecting DevC Member and DevC Expert, you can ask the expert if you have a trouble or bug when you learn facebook tools

DevC Chat application based on mobile and divided into 2 applications are:

  1. DevC Chat for Member : Login with Account Kit, Expert Categories, Expert List & Review, Expert Online/Offline Status, Chat with Expert, Chat History, Rating & Comment, DevC Event, Manage Profile.

  2. DevC Chat for Expert : Login with Account Kit, Chat with Member, Chat History, DevC Point, Redeem DevC Point, DevC Event, Manage Profile.

How we built it

We built this app using several technologies:

  1. Front End : Facebook Account Kit (For Authentication), Qiscus (Third Party for build chat application), Android Native (Java) Firebase Push Notifification
  2. Back End : Laravel, REST API / GraphQL, MySQL, Firebase Push Notification

Challenges we ran into

  • Newbie in Facebook Framework (react, graphql, etc)
  • Delay Notification (Based on Our Experience)
