Inspiration

In the time leading up to this event numerous news stories have come up in relation to both the negative impacts of vaping and the swift, blunt responses of various governments. Our inspiration came about in response to this.

What it does

An innovative and elegant solution like vapecare can not only solve the issue of learning more about the health impacts of vaping in the long term, but also assist current vape users immediately in their efforts to use the device responsibly.

How We built it

We built it in 3 primary sections: hardware attachments, data infrastructure and user facing software. The hardware itself tracks the usage of the device using an accelerometer

Challenges I ran into

Project scope as our project was very ambitious but we did get alot done. We also ran into issues related to configuration.

Accomplishments that I'm proud of

We are proud that we were able to build so much in such a short time and that we were able to coordinate our efforts into a coherent product

What's next for vapecare

Ideally, we would like to improve the hardware implementation and make it more accessible. We would also like to improve the data infrastructure by ensuring proper data governance adherence and more scalable architecture. Given enough extra time we would like to improve the mobile application as well.
