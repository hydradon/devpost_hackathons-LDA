Inspiration

Our school's hack club leader's previous project utilizing stepper motors. Old code was easy to read and understand.

What it does

Draws using a pen mounted on a vehicle.

How we built it

Used two arduinos, one to control the motion of the vehicle and the other to control the pen's position.

Challenges we ran into

Didn't know anything about arduinos before the hackathon. Code debugging. Planning design and materials list. Maintenance of hot glue consistency.

Accomplishments that we're proud of

Determining what pin numbers each component of the arduino used. Creating smooth wheel motion. Creating a compact design. Mastering the use of the hot glue gun.

What we learned

Arduino coding language. Arduino wiring. Division of labor and teamwork. Managing time. Hot glue is not fun to use.

What's next for Pen Robot

Creating a more stable design, using more powerful motors, wireless control, smoother motion, and attaching external power sources.
