Inspiration

We wanted to build a new skill for Amazon's Alexa to help people to maintain a healthy lifestyle through exercise within the walls of their own home. We call it A90

What it does

Has a set of automated workout routines which can be initiated through simple voice commands

How I built it

Using Python and Amazon's AWS Lambda platform, and then made a compilation of all the code online using gitHub.

Challenges I ran into

Run-time errors, time delays in Alexa's speech, learning how to use AWS Lambda using javascript and converting the same logic into python, the language we coded in

Accomplishments that I'm proud of

The program runs!

What I learned

We learned how to use AWS Lambda and development in the Amazon world

What's next for A90X

We want to develop the voice intonation of Alexa as she says our words
