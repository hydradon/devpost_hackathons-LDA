Inspiration

The AutoNation challenge inspired us.

What it does

It allows you to take a picture of a vehicle and it give you information on it in an instant.

How I built it

Using multiple API's and vehicle data given by AutoNation.

Challenges I ran into

Building a UI over the built-in android camera, getting the Google Vision API to work with the many depreciations, and combining all the tech.

Accomplishments that I'm proud of

Getting better at Android development.

What I learned

API usage in Java.

What's next for AutoVision

Mapping out the hood of vehicles and mapping out the engine.
