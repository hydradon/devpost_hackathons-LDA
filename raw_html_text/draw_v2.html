Inspiration

My mom likes to paint, so she is always looking for some great apps for her to draw. So I decided to create this app as a birthday gift for her.

What it does

It include most of the basic functions of a drawing app. Because it's running on mobile, it provides share function.

How I built it

I built it with Android Studio, and Java language.

Challenges I ran into

How to implement drawing function, custom alertDialog, and a lots of bugs!

What I learned

I learned a lot when I was doing this project, such as storage data, etc.

What's next for Draw_v2

I would try my best to make it better and better.
