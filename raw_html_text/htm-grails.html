Tea'mup

A website to join and host local tea meetings. The map shows the location of meetings. Click on start a tea’mup to create a new meet up at your location or find a tea’mup to join an existing meet up.

We started building it in Flask, with a MySQL database. After realising we didn't know the technologies well enough, we started using Grails, which is an Apache Groovy based web framework.

We originally had plans to integrate Neximo for authenticating users. This would help prevent users who would abuse the system, and texts to remind you of the location, etc.

Having had an early on the Saturday morning, we didn't end up getting a fully functioning prototype.
