What it does

A mobile app that recommends you movies or tv series that you may like based on some search parameters.

How we built it

The front-end with react-native, the back-end with python and the database with firebase.

Challenges we ran into

Processing the huge raw information for the database, making the front end work among others.

Accomplishments that we're proud of

It does what it's supposed to do, we learned a lot about data management, the app is coded with react-native and javascript so it work both in android and iPhone and we had fun coding so we couldn't be more proud.

What we learned

Algorithm design and implementation Something about react-native, python and javascript How to work with .json and firebase Deal with big amounts of data Download and use big sets of data from IMDb A lot of python utilities for data processing How to use GitHub How to do a better selection of the data and select from all the data the important one

What's next for W2Watch

We would like to include screenings on Barcelona cinemas and check if some series or movies are available in live streaming services.
