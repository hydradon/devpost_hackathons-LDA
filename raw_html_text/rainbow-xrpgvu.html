Inspiration

We were inspired by w3school's color scheme generator and wanted to make a similar application.

What it does

It generates different types of color schemes based on a color of the user's choice.

How we built it

First we built the interactive form as a Windows form. Then, we played around with coding the different color schemes to make them look the way the different color schemes should.

Challenges we ran into

Compound and analogous color schemes were particularly difficult to figure out the RGB values for.

Accomplishments that we're proud of

Figuring out the equations for the color values for all the different types of color schemes

What we learned

Working with colors requires so much math! Also, make sure to have realistic goals.

What's next for Rainbow

Adding games you can play with all these colors
