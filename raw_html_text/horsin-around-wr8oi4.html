Inspiration

When we first met each other and talked about our backgrounds we learned that we came from two educational backgrounds- CS degrees and Engineering degrees. As we were discussing the pros and cons of both degrees, there was one constant theme dominating the conversation - the ability to problem solve elegant solutions to wicked problems. We realized that there needs to be more oppurtunity to develop this brand of thinking for people who don't have ready access to all of the resources we do. At the same time, one of out group members mentioned how at one hackathon he couldn't get something to work because AdBlock was blocking requests to an API. There was a joke about not being able to use the chrome developer tools properly and an idea was thought about to bring these two elements together.

What it does

Horsin' Around teaches outside the box thinking while developing chrome developer skills.

How we built it

We used HTML, CSS and jQuery

Challenges we ran into

Sprites are very hard to work with.

Accomplishments that we're proud of

We are very proud we got a dynamic background, cool levels built and sprites!

What we learned

Elementary game design, jQuery and how to work with sprites.

What's next for Horsin' Around

Making the website responsive, add more levels and make it a bit more user friendly!
