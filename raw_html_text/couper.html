Inspiration

We were fed up with having to search for 10-15 minutes to find a valid coupon code to get discount at our favourite online stores.

What it does

Couper is a Chrome extension that allows you to find the best coupon, to give you the biggest discount. You can automatically insert "the best coupon", or view all available coupons given your location/basket contents to select the one best for you.

How we built it

We have used a Python/Flask combination in the back-end, supported by HTML/CSS/JavaScript in the front-end. We also used the Nessie API (provided by Capital One) to display the balance of a customer.

Challenges we ran into

We found it difficult to extract information from the coupon code websites; the data is horrible!

Accomplishments that we're proud of

We can now buy cheap Domino's: quicker, easier, and more stress-free.

What we learned

Never pay full-price for Domino's.

What's next for Couper

We want to start using NLP techniques to fine-tune the voucher suggestions; so if you're buying 'toothbrushes' from Amazon, we won't suggest "£2 off Star Wars: Episode II". We're also looking to refine the suggestion algorithm for choosing the "best deal".
