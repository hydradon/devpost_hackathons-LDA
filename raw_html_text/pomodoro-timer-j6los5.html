Inspiration

We were inspired by the style of studying that Pomodoro offers, but the lack of integrating a timer with Spotify to play and pause music.

What it does

This web app will allow the user to authorize access to their Spotify, so that it can play and pause music based on the timer.

How we built it

We built it by using React for the front end, which interacts with an express server to service some Spotify API requests. This also connects to the ASP.NET backend which allows for the storing of user accounts through Firebase.

Challenges we ran into

We ran into challenges getting the Spotify API set up to correctly authenticate the user.

Accomplishments that we are proud of

We are proud of a functional app that we and other students can use to help in their studies.

What we learned

We learned about different technologies and APIs that some team members were unfamiliar with.

What's next for Pomodoro Timer

Pomodoro Timer will continue to improve on its security with the user accounts, as well as improving the UI and UX of the web app in general.
