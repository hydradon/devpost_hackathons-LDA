Inspiration

The city of Detroit is large, so it isn't a surprise that its citizens and information are rarely connected. Our goal was to hack to this challenge: to use map APIs and already existing data to connect communities and provide better and safer way-finding for users.

What it does

Internodes is a web application that scales down to mobile platforms beautifully. It can be used as both a social media invitation platform, as well as an information resource for safe way-finding.

By dropping a node, a user can invite community members to a social event (small, public, or sponsored). Other users will be able to see these dropped nodes and take part in the event. By accessing maps, drawn based specifically off of Detroit data, users can also figure out the efficient, safest way to get from Point A to Point B.

How we built it

The front-end is Javascript; the back-end is PHP/Laravel.
