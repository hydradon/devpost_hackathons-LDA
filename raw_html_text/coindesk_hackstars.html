We're building a decentralized code repository to enable developers to safely and securely collaborate. Our decentralized code repository, minimizes risks of censorship and single fault failures.

We've built a command line interface that allows developers to commit directly to CodeHero via IPFS. We've also built a developer dashboard that shows a developer their projects and each projects' recent commits.

We imagine that building decentralized version control on Ethereum could be the foundation for several web3 use cases, including 1) proportional ownership of code, 2) freelance development built directly into the repository, and 3) reducing the risk surrounding centralized code storage.
