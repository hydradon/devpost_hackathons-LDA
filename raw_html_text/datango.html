Inspiration

We were inspired by the struggles faced by non-profit agents in developing countries managing their data.

What it does

It allows users to generate a spreadsheet of data by simply taking an image of written data.

How we built it

This is a flask application. We used aws rekognition for image text processing and s3 for image storage.

Challenges we ran into

We had challenges with aws permissions and security issues.

Accomplishments that we're proud of

We were able to work cohesively as a unit to accomplished a meaning goal with real-world impact.

What we learned

We learned to use aws for collaboration (Cloud nine), object storage (s3), and machine learning (rekognition).

What's next for dataNGO

Working with more organizations to develop further and deploy the working project.
