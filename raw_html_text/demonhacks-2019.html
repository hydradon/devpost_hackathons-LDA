Purpose:

Easy access to medical information for low income communities. We answer your questions and direct you to nearest medical location.

Method:

We are using many apis, algorithms, tools...

APIs Used:

We use Flask, Python, Alexa Skill-kit Developer Tools, Twilio, SQLite, Node.js, Jupyter Notebook, Natural Language Processing, nltk, etc.

More description will be added later.

Data Usage:

We have two main data that are taken from Chicago Data Portal for flu shot locations[1] and primary care centers[2] that are csv type. Both of the tables have features such as the name of the center, address, geolocation, zip code, and some data specific entries.
