Inspiration

Memes are mainstream. Making them is hard. We wanted to make it easy for anyone to turn themselves, their friends, or any random pictures into memes.

What it does

Using Facebook Messenger, you send a picture. Then you send the meme caption you would like. BOOM! The bot spits out your dank meme.

How we built it

We used the Messenger API, Node.js, Express, and ran it on Heroku. We modified images using Node.js and ImageMagick.

Challenges we ran into

Debugging a deployed product in Messenger is hard. Also, modifying images using Node is tricky and hacky, but we got around it ;).

Accomplishments that we're proud of

We became one step closer to our path of Javascript wizardness.

What we learned

We love hackathons and Node is really powerful.

What's next for The Dank Memeifier

Making it more aesthetic.
