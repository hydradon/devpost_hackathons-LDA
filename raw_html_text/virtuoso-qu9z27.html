Inspiration

Some of our members have musical backgrounds, so we decided to do something related to instruments.

What it does

Play drums, piano, and other sound effects. It can also record musical sequences, change the tempo, and replay the recorded sequence in a loop.

How we built it

We made the visual for our program on PhotoShop and then programmed everything using Java.

Challenges we ran into

We had to figure out how to use Array Lists and it was also difficult to find out how to make tempo changes and looping for recordings (using elapsed time to keep the same rhythm).

Accomplishments that we're proud of

We're proud of the playback option in our program and the visual component.

What we learned

We used github, for example we learned how to pull and push information from a server. Also we learned more about Java and PhotoShop.

What's next for VIRTuoso

We want to add a feature where you can switch the keyboard to different sounds, like a violin, clarinet or other instruments. It would also be cool to layer different recordings to create harmonies.
