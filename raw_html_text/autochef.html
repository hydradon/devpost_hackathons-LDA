AutoChef

Made for PickHacks 2019 by Karl Ramberg, Paul Ramberg, Trenton Morgan, and Isiah Slater

What is it?

Generating recipes with machine learning since 2019.

Random food to tasty meal in no time at all!

How does it work?

AutoChef uses machine learning to generate a list of ingredients based on how much you liked previous lists. It uses relationships between the ingredients to determine how well they pair together.

What tools did we use?

Python - webserver and machine learning implementation

HTML & CSS - frontend structure and looks

JavaScript - HTTP request handling and other functionality
