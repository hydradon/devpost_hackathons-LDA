Inspiration

We want to attract more developers to FFDC platform by making testing easy. We are offering unique and smooth developer experience.

What it does

We give the possibility to easily test your web application, that was built on top of FFDC APIs.

You don't need to know how to create automatic tests, you just drag & drop predefined visual blocks and easily build the test scenario.

Then, you are able to run your test, we will handle the infrastructure and execution for you.

We will also notify you once the results will be ready.

You can focus on developing your application and the rest will be done by us.

How we built it

We build it using FFDC infrastructure, in Microsoft Azure cloud. We leverage a visual programming framework from Google called Blockly.

Accomplishments that we're proud of

Creating robust way for running end-2-end test scenarios on applications build on top of FFDC, directly from the browser with few clicks.

What we learned

We discovered how hard it is to orchestrate all services needed for end-2-end testing. We are glad that thanks to FusionTest you will not have to do it yourselves :)

What's next for FusionTest

Going to production ;) Greater and smoother integration with FFDC developer portal Integration with mobile devices
