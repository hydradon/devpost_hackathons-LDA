Inspiration

We were inspired by the Euro, which is a currency that unites major parts of Europe. We wanted to solve the challenge question, and also solve an underlying larger social/political issue. We also drew inspiration from the bitcoin, which is a virtual currency.

What it does

We collect data from the sensor network, in which each sensor is located next to a power source. We aggregated the data over the sensors and over a given time interval, and then estimated the energy surplus based on machine learning. This generates a market between counties, and we set pricing for the various energies. Pricing is based on greenness, stability, availability and distance of resource.

How we built it

We set up a sensor network and collected the data. We leveraged machine learning to estimate the surplus, and then we transformed the surplus into an energy market.

Challenges we ran into

  1. Retrieving data from the sensors.

Accomplishments that we're proud of

  1. Designing a virtual currency distribution that would be equalized between countries.
  2. Incorporating Machine Learning.
  3. Developing the business logic to convert sensor data into an applicable market place.

What we learned

We learned how to collect sensor data and store it in Predix. We also learned to create a virtual coin.

What's next for ElectroEuro

To surpass bitcoin as the next big thing in IoT, while driving a decarbonized economy!
