PyHackathon

This project is for the uOttawa python hackathon on 12 Nov 16. It's purpose is to allow the user to draw random circles of random colors to a turtle canvas

how to use

1 - program requires python3 and turtle (which should be included)

2 - run main.py to start the program

3 - click anywhere in the window to draw random circles of random colors
