CentreStyle is a unique way to plan, style, and browse for outfit inspirations when traveling to new destinations. Using the Esri ArcGIS API, a user's location is detected and displayed on a map. Then with information about the destination--such as weather conditions--we use the Zalando SE API to generate recommendations for the user's outfit while showing style inspirations from surrounding app users.
