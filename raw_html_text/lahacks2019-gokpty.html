Our goal was to use transaction information to provide helpful metrics for both vendors and consumers.

We give each user a qr code to scan at their checkout so using it is as simple as pointing and clicking. We once ingested we can use the transaction data for a couple of things.

On the consumer side:

  • Provide recommendations for future grocery lists
  • Use those ingredients sets to generate a set of recipes for the user.
  • Manage food freshness, provide updates when food is going stale

On the vendor side:

  • Get consumers to voluntarily provide their tracking data
  • Integrate our user handles with their rewards programs
