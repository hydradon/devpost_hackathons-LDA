Inspiration

We all play the game Cards Against Humanity and we really wanted to make something unique for iMessage (Backgammon was already made..)

What it does

It is an iMessage application that allows groups to play a less offensive version of Cards Against Humanity.

How we built it

We used xcode and swift for the entire project while using tons of Stack Overflow, Apple Documentation, and help from amazing mentors!

Challenges we ran into

We used xcode and swift, which we had no prior experience with, so we had to learn as we worked, which made the process more challenging.

Accomplishments that we're proud of

We are proud of being able to understand a new language.

What's next for Texts Against Humanity

We want to improve our interface and increase our repository of cards.
