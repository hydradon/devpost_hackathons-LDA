Inspiration

Many people from around the country may not have full access to an actual doctor. This type of app could potentially allow people in less fortunate areas to realize their medical issues and seek proper help.

What it does

The application allows you to take a picture of whatever medical issue you have and tells you what you have.

How we built it

We used Android Studio to create the application but our program depends on IBM's Watson API for machine learning so that the app could discern between different and similar symptoms.

Challenges we ran into

We had to figure out how to initialize the API and how to grab the full size image from the Android camera for later use.

Accomplishments that we're proud of

What we learned

Learned how machine learning works and how to start an android application

What's next for MedMD

In the future, we hope to increase the range of symptoms that the application can discern and tell the user. We hope that the machine algorithm could eventually allow it to accurately discern even very similar cuts from one another and we hope to add in details about it, such as details about the symptoms, possibilities of diseases, and cures for the injury.
