Inspiration

Both of us have been students in public schools for over a decade, and we have taken many standardized tests in the process. We believe they have not enhanced our learning and are in fact detrimental to students due to hindering creativity, increasing stress, and leading to unfair financial distributions

What it does

The application enables people to easily and quickly contact their local representatives through social media and send a message conveying their displeasure towards standardized testing. The web page also provides explanations as to why standardized testing has negative effects.

How we built it

We use the Phone 2 Action API to accept a user's location and get the social media information of their local representative.

Challenges we ran into

Initially we ran into struggles using the API to get representative information, but eventually figured it.

Accomplishments that we're proud of

Getting the API to work and being able to contact local politicians through our application was very satisfying. We are also proud of advocating for a cause we believe in.

What we learned

We learned to use multiple APIs in a single application. We also learned how to use advanced css tools to increase the webpage's aesthetic.

What's next for Ending Standardized Testing by Contacting Politicians

We would like be able to use the API to expand to Facebook and calling to reach out to more people and politicians. We would also like to come up with a method for distributing the campaign to those who would be interested in supporting it.
