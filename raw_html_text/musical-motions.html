Inspiration

We were planning to make a crash algorithm to help prevents crashes for cars, but that was too large for us to handle, so we came up with this idea after we had all of our materials and that nothing worked from our previous idea.

What it does

It plays notes and musical sounds of a piano, while being controlled through hand motions

How I built it

We built the app using Leap's API on python and the pygame library

Challenges I ran into

Being able to do multiple inputs at a time and doing multiple octaves for more accessibility in notes.

Accomplishments that I'm proud of

Creating a functional app that will be all controlled through motion and gestures.

What I learned

Using the Leap Motion's API for Python and to always have backup ideas in case something does not work.

What's next for Musical Motions

To incorporate more instruments that can be played with motion and to allow for a greater range in notes.
