Board Game Geek Alexa Skill

This is an unofficial skill for getting the top hot boardgame list from boardgamegeek.com.

I was just learning to code Alexa skills. I am a board game fan. I love all of the information available on board game geek. I noticed boardgamegeek.com has an XML API. I decided to teach Alexa how to talk about board games.
