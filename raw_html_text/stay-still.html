Inspiration

We were inspired to create a 3D model of a room to try and see how art would look on a wall or how a sculpture would look in a space.

What it does

Takes a series of depth images to create a 3D map of the environment by combining all of the depth images.

How we built it

We used an Intel RealSense camera to take the depth images, model them using PointCloud, and combing the PointClouds into a single environment recreation.

Challenges we ran into

We tried to build a robot to take the images, but we were having issues with the serial controls. We also were trying to implement the project in VR but we ran into difficulties with color and exporting a .ply file into a VR environment.

Accomplishments that we're proud of

We were able to successfully create a 3D model.

What we learned

We learned about 3D modeling methodologies.

What's next for Stay Still

To complete the project and implement the 3D model in a VR environment.
