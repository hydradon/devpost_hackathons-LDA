Doge-Invaders

What is it?

Doge invaders shows the story of Dogecoin. It shows the struggle that Dogecoin faced when competing with its older brother, Bitcoin. The game is modeled in a space invaders way, the thing that sets this game apart from space invaders is the integration of "Leap motion" software. This Leap Motion software allows the user to control the game using hand guestures

Controls

To go through the menu you must click next using your cursor

To play the actual game you place all your fingers 20cm above the leap motion sensor then move your hand either left to move left, or right to move right. To fire, you tap your index finger downwards.

This project requires a Leap Motion Controller and all development kits associated with the Leap Motion Controller product.
