Inspiration

Since many people are facing eye diseases i did this task so that even if i spent some time on this i would like to save a millions of lives who have this disease and to get cure before they get permanent blindness

What it does

It predicts the which of the 5 levels are affected in their eye

How I built it

I built this with Tensorflow and keras.I did image processing with opencv.I got this data from kaggle website

Challenges I ran into

Image resizing.Image localizations.Which features predicts the level of disease

Accomplishments that I'm proud of

I trained this image and i get an upto some level of accuracy that it predicts the disease level correctly on a not-seen image

What I learned

I learned Convolution neural network,opencv and how to preprocess the images before giving it into the training neural network

What's next for Diabetic Retinopathy Detection

Improve my accuracy score so that it predicts even more correctly
