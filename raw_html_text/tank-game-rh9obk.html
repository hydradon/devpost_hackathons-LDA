Inspiration

We like to play games a lot so our natural thoughts were to make some sort of game.

What it does

Online browser game where you spawn in as a tank and have to survive against other players. You have a turret to shoot other players and once you get shot you're done.

How we built it

We made use of the HTML canvas to draw the objects and mostly utilized Javascript and Node modules Express and Socket.io to have the multiplayer functionality.

Challenges we ran into

We had some challenges with making the tank properly rotate and have the bullets follow that angle. We also have some problems with the images flickering on the page.

Accomplishments that we're proud of

Actually making a multiplayer game that allows people to play against each other. A lot of the server-client interaction was tough to do and to see it work out makes us happy.

What we learned

Making games is hard. It requires a lot of planning to make sure no problems arise. We also learned more about how to work in a team when developing, using Git, communicating etc.

What's next for Tank Game

Actually making an end game and possibly fixing any of the bugs like the flickering. In all honestly we will probably not come back to this, but gives us new inspiration and drive to make something even better!
