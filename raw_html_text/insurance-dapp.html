Inspiration - insurance

What it does - provides members with a tool to create funds for disaster relief

How we built it - We built a web page that allows members to join a fund and a command line tool to create mutual funds on Bitshares platform and we've recorded those actions on Algorand blockchain

Challenges we ran into - we had difficulties to sign transactions on Algorand

Accomplishments that we're proud of - integrating various blockchain tools in one solution

What we learned - We learned Algorand and Bitshares blockchain features that helped us to implement our solution

What's next for Insurance dAPP keep developing our solution and create a distributed insurance platform
