Inspiration

A big contributing source of anxiety for many people is losing a beloved possession. Especially in big cities like New York, we all know how it feels when we've misplaced an item that feels like a hopeless cause in getting back. It can be extremely heartbreaking and also very stressful, taking a mental toll as a lost item usually preoccupies much of one's thoughts. That's why we created Retriever, a web application that can hopefully offer a solution to this problem.

What it does

Retriever allows users to search for lost items as well as report found items. This web application creates an online community that can return possessions to their owners and relieve the anxiety of having to physically search for it on your own. With the help of good samaritans, we aim to create an honest system of reporting lost items.

How we built it

We used Python Flask, HTML, CSS, and Javascript to create Retriever.

Challenges we ran into

There were a few hiccups integrating the Google Maps API to our project, as well as a few back-end issues that we eventually figured out!

Accomplishments that we're proud of

Every member of the team contributed equally and most of us gained knowledge on a new language/technology. We're happy with the project we created and hope it can help others.
