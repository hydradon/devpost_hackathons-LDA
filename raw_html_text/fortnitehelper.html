Fortnite Commands

Leaderboard – d@ftn leaderboard solo, duo, squad * optional - platform (if none specified defaults to PC)

This command is used to view the current top 10 players in Fortnite on a given gaming platform (xb1, ps4, or pc). This displays the players name and rank, along with their points(value). This will default to pc if nothing is input.

News – d@ftn news

This commands function is to display the current news in Fortnite such as updates.

Stats – d@ftn stats username solo, duo, squad, lifetime optional - platform (if none specified defaults to PC)

This function pulls the stats such as total wins, how many times they placed in the top 5 or top 12, win percentages, time played, and kills per minute. The parameters are the username, selecting a game mode, and optional parameter of the platform such as xb1, ps4, or pc. It will default to pc.

Status – d@ftn status Server name

This command shows the status of the server. You will be able to select Royale, PVE, or Store and display its online or offline status.

League of Legends Commands

Summoner Info – d@lol suminfo “Summoner Name”

This command is used to pull up info for the player input into the quotes. This displays the name, the Summoner ID, and the level of the player.

Champion Mastery – d@lol champmastery “Summoner Name”

This displays the total mastery points achieved on champions within the game to display the players skill.

Top Champion Mastery – d@lol topchamp “Summoner Name”

This displays the name and title of the champion, total level achieved, and total points earned on the players highest ranked champion.

Match History – d@lol history “Summoner Name”

This displays the last 3 played games. This includes the lane, champion, server, and role they played in the game.
