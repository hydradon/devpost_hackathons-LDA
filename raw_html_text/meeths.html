Inspiration

One examples where high schoolers would us this app is in order for finding teams for hackathons. A school's programming club can sometimes not have enough students, so it would be beneficial to meet other high schoolers with the same interests as you.

What it does

The app connects to facebook to understand about your interests and learn more about you. It verifies that you are a high schooler by making you sign in with your school email.

How we built it

HTML/CSS, JavaScript, Node.JS, Google Maps API, Facebook Graph API.

Challenges we ran into

Google Maps API wasn't loading quickly. Facebook Graph API API Key was not authenticating for some reason.

Accomplishments that we're proud of

We solved these issues and were able to create this app with 1 hour to spare.

What we learned

We learned more about the Graph API

What's next for meetHS

More functionality and better algorithms to find people with similar interests.
