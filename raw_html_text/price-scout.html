Inspiration

Tired of looking through catalogues, waiting for that one thing to go on sale? Wouldn’t it be nice to just get notified?

What it does

Price Scout allows the user to create agents which search through the Danish supermarket catalogues through eTilbudsavis. If matches to the agent is found, an email is sent to the user, notifying them that items they might be interested in have finally gone on sale to a good price!

How we built it

Backend developed in Python with Flask, using Firebase as the database and Heroku for hosting and scheduling nightly cron jobs, that send out emails through Mailgun.

What's next for Price Scout

Build support for advanced filters, so there’s no false positives when looking for items on sale.
