Inspiration

We were driven by the desire to gather and analyze data on users based on public information

What it does

Our program gathers users data from twitter and rates the sentiment of the tweet using googles natural language api

How we built it

using Django, google natural language api, and python twitter api

Challenges we ran into

lack of experience with django and google's api. Domain hosting issues

Accomplishments that we're proud of

the twitter sentiment analysis works very well, even with the initial issues, we are glad to have gotten the system working

What we learned

We were introduced to google's api and django and getting them to work together

What's next for Sentinel

We are planning on continuing to work on expanding the system to group individuals based on likes and dislikes
