Inspiration

I became frustrated when searching for a new home. I constantly found myself copying and pasting addresses in Google maps and Yelp to see what exactly was in the vicinity of the potential properties during my search.

What it does

hoodPeep communicates with Yelp via the API to compile the neighborhood amenities depending on the user type and search radius selected.

How I built it

The web app was built using Laravel, MaterializeCSS, MySQL and VueJS

Challenges I ran into

The major challenge was coming up with an intuitive way to display the results which would be most meaningful to certain types of users (young couple vs parents, etc.)

Accomplishments that I'm proud of

The app is integrated with Twilio for sharing via text messages. This app forced me to dive into new technologies and extensive frontend javascript work for the UI.

What I learned

This is the first time I built an application which heavily relies on a third-party API, let alone multiple APIs. It was certainly challenging and time-consuming, but my skills as a developer have grown in leaps and bounds thanks to this hackathon.

What's next for hoodPeep

The ability to pick and choose individual neighborhood amenities important to the user along with infographic generation, which will generate a nice infographic with the neighborhood's benefits for use in real estate ads and the like.
