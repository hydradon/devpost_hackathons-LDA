Inspiration

While studying Japanese, I found that there were a lack of good apps that were useful for practicing the Kanji, which can otherwise be difficult to practice.

What it does

Allows users to generate symbols which can then be used as templates to practice writing. The user simply draws their symbol and selects create, then they can practice and receive a grade on how well this symbol was written in regards to the original.

How we built it

We used Android Studio, primarily with Kotlin, to develop the structures to store each character as they are input, then compare new inputs to saved templates.

Challenges we ran into

Configuration issues with connecting a physical phone to Android Studio. Theory on how to handle scoring an input character.

Accomplishments that we're proud of

The user drawn input is displayed similarly to a fountain pen.

What we learned

Kotlin app Development, Pixel Analysis
