Inspiration

Tower defense games

What it does

Defend as you mine for more in a cave, from enemies. You will be able to set up and upgrade defenses in the form of friendly turrets, as well as use abilities of your own, to survive the waves of invading drones.

Features

  • Google Cardboard support
  • Classic Tower Defense gameplay, with the enemy lane coming 'at you', and spots to set up defenses around the cave
  • The current game theme is sci-fi
  • The current game session length is ~2 minutes
