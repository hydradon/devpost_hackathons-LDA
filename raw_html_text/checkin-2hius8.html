Inspiration

Wanting to soothe people dealing with mental health issues.

What it does

Attempts to provide people dealing with mental health issues resources for comfort.

How we built it

We used XCode and Swift 4 to make the app. We also used Python to integrate a chatbot API.

Challenges we ran into

  • Ability to detect tap down and up (necessary for rotary emoji selector)
  • Glitches related to @IBInspectable properties
  • Problems with time/date selectors -Problems integrating the pythonic chatbot into the Swift code for the iOS application. ## Accomplishments that we're proud of -Working chatbot -Working selector for the user interface. ## What we learned
  • Swift 4 coding -To use Microsoft Cognitive Services API in Python. ## What's next for CheckIn
  • Integrate our chatbot. Improve it using machine learning(possibly deep learning models) with data similar to that from the Crisistext hotline.
