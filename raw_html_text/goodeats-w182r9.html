Inspiration

I personally have trouble gaining and maintaining my weight, but If I had a structured meal plan of what to eat that is good for me then I could use it. Also since the theme for this Hackathon was diversity I think it's a great project for everyone because everyone has a diverse set of needs because we are all different, but that is good because we all have a similar struggle.

What it does

This app helps people to calculate their daily caloric intake needs based off of how active they are and their current biological attributes. This does not yet tell the user what foods to eat based off their AMR.

How I built it

I built this using Android studio, Java, xml, JSON

Challenges I ran into

I ran into getting the json to appear in the app from the key given, and also I have trouble creating a list of meals for a given meal plan

Accomplishments that I'm proud of

I am proud of how much I coded in short time, but also how much I learned from attending my first Hackathon

What I learned

I learned that there are so many ways to achieve a given result. I learned how to access Internet to allow JSON to be accessed in my app. I learned how to use spinners and radio buttons, passing data between different activities using explicit Intents.

What's next for GoodEats

Making the meal plan generate meals and also maybe incorporate other micro nutrients such as vitamins.
