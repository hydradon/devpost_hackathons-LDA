Inspiration

Scheduling a meet up for group project meetings or just to have a catch up with friends can be troublesome and difficult to plan when you do not know each others schedule. Comparing timetables can also be a tedious task especially when there are many people involved.

What it does

With NUS MeetLah!, it collates the timetables of everyone and creates a quick and easy way to find pockets of time which everyone is free!

How I built it

This website is made possible by retrieving data from NUSMods, coding with React and saving data via Firebase,

Challenges I ran into

Getting inspiration for the layout as it was difficult to find a way to display the information that is pleasing to the eyes

Accomplishments that I'm proud of

Creating a nice layout that is simple and easy to read, while still providing sufficient information

What's next for NUS MeetLah!

  • Adding user accounts, tightening up security issues
