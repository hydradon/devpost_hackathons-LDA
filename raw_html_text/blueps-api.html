What it does

Our bluePS API uses machine learning to determine a user's exact location based off the signal strength the device has between bluetooth nodes. Far more accurate than GPS and does not require data or device-native computing. Corporations and developers will utilize our API to build interactive physical environments (ex. like walking up to a car in a dealership to get information, pictures, links, etc.). We built a museum app to demo its functionality.

How we built it

API is coded using Python. UX is coded using HTML, CSS, and JavaScript

Challenges we ran into

Setting up the nodes and creating large amounts of data for our ML algorithm to adapt to. Lots of back end bugs! Bluetooth not working on 2 out of our 3 devices for half the night...

Accomplishments that we are proud of

The accuracy, simplicity, and the design.
