InterSpeak

Want to learn a language while interacting with native speakers of said language? Welcome to InterSpeak, a translation text messaging application. Use just like a regular text messaging app, but you will also receive a translated message from the foreign language speaker you are texting! A text based universal translator!

Approach Taken 

Created a database to handle multiple types of input information.
Incorporated forms for text field, send, and dropdown menu features.
Implemented a "search bar" that allows the user to search for languages.

Technologies used 

Developed with JavaScript, PostGresQL, Google Translate API, React, Node.js, Sequalize

Inspirations: Star Trek

Challenges: Figuring out how to get JavaScript to be incorporated into a GUI, at one point there was a working GUI, but it only worked on one machine.
