Inspiration

Uber and other location based applications gave us the inspiration to solve a problem leveraging the devices we keep on us at all times.

What it does

Sesh allows students to create and attend study groups based off their location.

How we built it

Leveraged Esri's ArcGIS API to create study locations and track when students are in proximity of a Sesh.

Challenges we ran into

Learning how to integrate ArcGIS API with React.

Accomplishments that we're proud of

Being able to bring a concept to life as a prototype within 24 hours.

What we learned

ArcGIS API and a few DynamoDB caveats.

What's next for Sesh

Continue refining the Sesh until it is bought out by Uber.
