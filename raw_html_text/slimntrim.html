Inspiration

These days, everyone is in a rush, and people often can't find the time to work out. Slim N' Trim is an easy way to generate exercises that you can fit into your schedule.

What it does

Based on how many minutes you have, the app randomly selects a set of exercises from the genre you pick (such as cardio or abs).

How I built it

We used Android Studio and sample code online to figure out how to get it to work. We also used our background in java for the overall structure of the code.

Challenges I ran into

Not being familiar with Android Studio, making it harder to do even simple things. We also wanted to make it a more interesting and original app, but we did not want to overcomplicate it and end up with a nonfunctional app.

Accomplishments that I'm proud of

We managed to make a working app using software we hadn't used before with minimal help from other people. We also feel that we improved from where we were last year.

What I learned

How to use Android Studio as well as improving our ability to not overcomplicate things.

What's next for SlimNTrim

We would like to add more features such as having an account and being able to save your past workouts. We would also like to add more workout options and make it more innovative and desirable.
