Inspiration

How we built it

Using unity, we divided the project into several different sections- the reality shifting mechanic, the platformer backbone, the art and design, and backup plans.

Challenges we ran into

Coming into Unity without any prior knowledge made it difficult to implement our ideas.

Accomplishments that we're proud of

We're proud of completing our game as a working 2D platformer with the shifting mechanic.

What we learned

We learned a great deal about how to program successfully in Unity.

What's next for Alternate

Improvements in fluidity, design, animation, and features (different game modes).
