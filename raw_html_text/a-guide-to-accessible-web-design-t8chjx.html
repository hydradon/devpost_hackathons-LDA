Inspiration

We were all interested in social justice and accessibility.

What it does

It provides resources for accessible web design for people with auditory/visual impairments and dyslexia.

How we built it

I coded it all in python with an imported graphics package (you can find the graphics package on GitHub)

Challenges we ran into

It was hard to find resources because often accessible web design is not prioritized in society.

Accomplishments that we're proud of

We're proud that we made a guide that may be useful in the future.

What we learned

We learned the importance of accessibility and how making accommodations makes life easier for everyone, not just for those who need to be accommodated.

What's next for A Guide to Accessible Web Design

We may turn this app into a website. We'd like to add information for every category as well.
