Inspiration

I wanted more related ideas for a certain topic google search, particularly animals and science, but also get pictures with them. The website itself is supposed to give the user inspiration.

What it does

Give related ideas for a topic, using a custom google search, limited to websites with good picture and description matches.

How we built it

Took data from a custom google search, and put it into a website form, using jQuery.

Challenges we ran into

domain.com is not so good, making learning how to upload a website kind of a pain, and also, just was unable to get it working, and had to do it all locally. Getting data from the custom google search and transferring it to the webpage in a new form took some effort. We attempted to take data from Microsoft Concept Graph to find related concepts to search and populate our page's concept map, but were unsuccessful.

Accomplishments that we're proud of

We learned most of jQuery in a night. Made something that works from jQuery, in the form of a web page, after learning it so rapidly.

What we learned

jQuery, web development, custom google search, and also domain.com is not so good

What's next for Idea Stormer

If we did continue, we would add better searching and better UI/page design.
