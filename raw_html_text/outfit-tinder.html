Inspiration

We wanted to create an easy way to get a new outfit, with out scrolling though 100s of articles.

What it does

You can like or dislike articles and based on recommendation and if you find your perfect match we generate you a matching outfit.

How we built it

We build it with react native (expo.io framework) ,nodejs as backend backed by mongodb.

Challenges we ran into

  • create a good look and feel for the app
  • finding a good recommendation algorithm

Accomplishments that we're proud of

  • we have a running and working app for iphone and android
  • we got a good development speed with regularly small improvements

What we learned

  • small personal things for each of us ;)
  • using react native as a development framework
  • having fun at a hackathon

What's next for Outfit Tinder

  • improve the app
  • improve the recommendation

Built With

  • mongodb
  • node.js
