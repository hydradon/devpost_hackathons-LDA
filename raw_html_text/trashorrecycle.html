Inspiration

We really wanted to make a game. Then we saw the EPA Hack Idea: Game challenge and thought, "why not?"

How it works

Very simple. Swipe left if the object in question should be thrown in the trash can, or swipe left it it should be recycled.

Challenges I ran into

We never created a game in Unity3D built for a mobile device. We also never used gesture recognition for swipes, taps, etc.

Accomplishments that I'm proud of

We were able to make a full fledged game

What I learned

How to make a full game in Unity3D

What's next for TrashOrRecycle

Animations, more objects to throw away, timer, compete amongst friends, push to a tweet.
