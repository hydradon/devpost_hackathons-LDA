Inspiration

Growning up as a kid, I loved guitar hero.

What it does

Myo hero is a guitar hero type game, however instead of strumming notes on a guitar, you will be performing hand gestures as the various inputs.

How I built it

I used the Myo SDK along with Xcode to develop this project.

Challenges I ran into

After some testing I learned that the Myo is horribly inaccurate . This presented a huge challenge for me as detecting these gestures in an accurate and timely manner is essential for this game.

Accomplishments that I'm proud of

I'm proud that I was able to learn and use the Myo SDK, and l also got to learn a bit about the Muse SDK as well, however I was unable to use it as all of the Muses were already being used.

What I learned

I learned about integrating hardware in my apps, more specifically the Myo and Muse; I learned how to use and manage hardware in an effort to make more immersive and connected user experiences.
