Inspiration

Finding a place to take a break, either to study, eat, hang out, or anything else, is always difficult, as places are always overcrowded.

What it does

Pact. is a tool designed to make daily lives easier and more convenient. With this interactive web app, you can figure out where to eat, where to study, or where to gym based on real-time information on how busy or packed a location is. All data is crowd sourced from the app's users, letting users contribute what they know about a location, including whether a place is quiet, loud, etc., in addition to how busy a place is.

How we built it

Pact. is built using Javascript (and the jQuery library), Bootstrap for the frontend, and Firebase for the database.

Challenges we ran into

Communicating information to all of Pact's users using a database with a pure javascript solution was most definitely a challenge in the creation of the application. However, we were able to overcome this using Firebase.

What we learned

We improved our web development abilities in all areas (HTML, CSS, Javascript), as well as working with APIs for third-party services.

What's next for Pact.

More data can be crowd sourced from Pact's users to give a more complete picture of what a place is like, helping users make a more accurate judgement of whether or not a place is up to their standards.
