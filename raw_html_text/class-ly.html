Inspiration

We saw a need to help students organize quiz dates, assignment dates, etc, from the bunch of syllabuses they got from teachers each semester

What it does

Allows users to take a picture of their syllabus or upload a photo, converts that to text, parses the input for dates and adds the dates found to the users calendar

How we built it

Android studio Optical Character Recognition
