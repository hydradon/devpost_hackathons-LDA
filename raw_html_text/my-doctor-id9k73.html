Inspiration

Amazon Echo powered doctor assistant with patient mapping

What it does

  • "Alexa, Ask my Doctor to come to my room"

  • "Alexa, Ask my Doctor what medication I need to take today"

  • Reports the exact location of the patient making the request

Technology Used

Here Maps API for the patient mapping/location

Javascript for the notification

Python's Flask Framework to create the REST API

Amazon's Alexa Skill kit
