Inspiration

Whenever we meet someone new, the intuitive thing is to add them on social media. However, the process of connecting with people is extremely time consuming and really hinders the process of forming social connections with other people. Therefore, we aim to solve this problem and help make connecting on social media with people easier.

What it does

This application allows people people to create profiles for them (e.g. Social, Business, School) which consists of different social media networks. Users can enter the profile code which is provided by the user whom they want to connect with. Based on which social networks are specified in each profile, the user can then add their connections directly from our app with a few simple clicks.

How I built it

We used MIT App Inventor.

Challenges I ran into

This was our first time creating an app and using this software, so understanding the nuances of it and getting a product working took a lot of time.

Accomplishments that I'm proud of

What I learned

We learnt how to create apps using this software.

What's next for Omniconnect
