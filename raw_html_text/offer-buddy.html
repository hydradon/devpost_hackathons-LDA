Inspiration

The need to connect users with local merchants using location and category based contextual advertisement.

What it does

  1. Vendor ability to add/update/delete multiple shops.
  2. Vendor ability to add/update/delete multiple ads for the different or same shop.
  3. User ability to view ads based on location.
  4. User ability to view ads based on category.
  5. User ability to add/delete/view an ad to cart.
  6. User ability to receive notifications for new offers posted in near vicinity.
  7. Ability to switch between user or vendor
  8. Ability to interact with the app hands-free using voice commands (while driving).

How we built it

  1. Android App built in Java.
  2. Backend service built in Java.
  3. AWS for infra services
  4. Dialogflow for machine learning based conversational flow

What's next for Offer Buddy

To enhance the experience of handsfree communication with app using voice commands to convert cars as a new marketing medium.
