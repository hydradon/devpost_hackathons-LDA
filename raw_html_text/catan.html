Inspiration

We sought out a project that would be challenging within the time allotted for the hackathon. we might have come up short, we're still proud nonetheless!

What it does

It rolls dice, gathers resources for the players based upon the dice rolls, and has the cation robber

How I built it

Using only the pygame library for Python, we started by mapping out the Catan rules, then coded our way up from there!

Challenges I ran into

Hexagonal grids and Graph Traversal where there is a multitude of cycles

Accomplishments that I'm proud of

The Hexagonal Grid implementation The texture loading

What I learned
