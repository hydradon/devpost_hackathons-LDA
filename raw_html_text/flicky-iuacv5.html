Inspiration

I got the inspiration from seeing how much Samsung sells the idea of how convenient the Samsung S7 edge panel is. In my personal experience, it was very hard to use and didn't give me that time advantage if I were to access similar apps.

What it does

Using accelerometers, it decides to open certain apps. If it is flicked with the right edge facing down, it opens one certain app. If it is flicked face down, then some other app opens up.

How I built it

Self learning Android Studio and reading a lot of documentation on how to use Android Studio

Challenges I ran into

First time I ever delt with Java or Android Studio. Never touched those two things anytime before the hackathon

Accomplishments that I'm proud of

Building a fairly good conceptual prototype of an android app without knowing how to build an android app when I came in.

What I learned

Learning the complexity of Android Studio and Java and how "Android Java" has its own quirks

What's next for Flicky

Add customization features such as changing accelerometer sensitivity and changing what each different flick direction opens/does
