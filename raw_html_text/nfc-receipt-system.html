Inspiration

I saw a lot of receipt papers being wasted and i realized how long it takes to enter an email or become a member on a store that you visit once a year for an e receipt. NFC Receipt would make things much easier, it'll save paper and make things quicker during checkout.

What it Does

Our app is compatible with Square and it uses NFC to send the final square receipt to another device

How I built it

We used the Square API and worked on Unity using the C# Language.

Challenges I ran into

The Square API is really badly organized and the specific factors that we needed to overcome were almost impossible due to the way their API is setup. Moreover their Sandbox mode isn't really a real API that we can use for testing.

Accomplishments that I'm proud of

We got the NFC part done, it reads and writes data.

What I learned

Learned payment gateway APIs and Android development through C# and Unity.

What's next for NFC Receipt System

Probably getting things to work properly and maybe having an actual software with a NFC reader(external)
