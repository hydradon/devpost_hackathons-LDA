• Talking Parrot

Inspiration

To have my kids entertain themselves.

What it does

Repeats back whatever you say. But be careful not to insult the parrot. It might insult you back.

How I built it

Using the Alexa Skills Kit and Amazon Lamda.

Challenges I ran into

Using the Amazon Literal slot type was confusing.

Accomplishments that I'm proud of

This was actually the first Alexa skill I ever created

What I learned

How to build a custom Alexa skill.

What's next for Talking Parrot

Make it more entertaining after the novelty wears off.
