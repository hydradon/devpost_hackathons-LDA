Inspiration

Inspiration from available hardware given by MLH and challenges from sponsers

What it does

This is a simple endless runner game that is controlled only via hand gestures. The speed of the game is linked to brain activity.

How I built it

The game is written using the Unreal Engine, using the leap-motion controller and muse headset as input. Interaction with the muse headset is written in C++, interaction with the leap-motion controller is written in python.

Challenges I ran into

Getting data from external API's and programs into UE4.

What I learned

Gained experience with using UE4 and using c++ api's.
