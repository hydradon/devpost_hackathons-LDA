I was inspired to develop EasyLearnML because of the difficulties that I faced as a self-taught student while researching and learning about Machine Learning online . The difficulties include access to beginner materials, a reliable mentor, a Machine Learning dictionary and a link to useful related websites.

What it does

Easy Machine Learning helps to guide interested prospective Machine Learning students on everything that they need to know about Artificial Intelligence. Users can start, monitor/track progress, request for mentors and assign roles on a Machine Learning Project. Projects can be commented upon, rated and supported by Machine Learning community members.

How I built it

The backend logic was developed using firebase for the notifications and hosting, PHP was used to process the webhhook messages from Facebook, the webhook was hosted on heroku. The front end Webview was developed using html, css, and vanilla javascript. The product is a facebook Messenger BOT.
