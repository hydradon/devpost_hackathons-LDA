Inspiration

To create a game that entertains and challenges coders in a competitive way.

What it does

"Code High” is a game league of coders solving a coding problem in real-time . The goal for the coders is to come up with the most creative way to solve the problem. Up to five people can join the real time league. After the 5 min time limit is over, players can like others’ solutions that they think is the most creative way to solve. After the league is over, each one of the competitor’s ranking is recorded and ranked. Programmers can play this game and have fun, compete with each other, while exchanging programming information.

How we built it

React.js + Node.js + Socket.io + Mongoose on Amazon Web Services

Challenges we ran into

We had to create a game that was fun and educational at the same time.

Accomplishments that we're proud of

Creating a game that is fun and informative at the same time.

What we learned

Coding can be fun.

What's next for Code High

Create other real-time games that can be played by a bigger range of players.
