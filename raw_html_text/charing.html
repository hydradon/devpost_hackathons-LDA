Charing

Charing is an android mobile application which allows charity organizations and funders to share the same platform to match the interests of the funders with the charity organizations according to the category of the donations. The funders can donate as per their choice to the organizations they want to donate to. The Android application uses XML to create a UI based as per organizations and users interest.
