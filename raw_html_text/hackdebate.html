Inspiration

The idea of this web application came from the observation regarding debate and discussion in our every day life. Many students never fully comprehend the importance of examining devil's advocate and using valid resources in academic and educational debates.

What it does

This program allows professors to conduct a debate online, so that student can participant either in a debate either live in class or as an assignment. This helps the students to interact with each others and share their knowledge and opinions. Each one has a different opinion for the same situation, and this approach tries to explore the unknown facts.

Project tagline

Web application that allows teachers to host live, educational debate and group discussions among students.

Project description (should be approximately 100-500 words)

The web application allows students to log into a debate bash board, where they can select their class. The professor would provide a debate topic, a pros and cons panels at a glance, valid source for students to reference, and a deadline for participation. The professors, while providing topic, also searches for books from Chegg API and refers these to students for their research.
