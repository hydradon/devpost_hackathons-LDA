Great UniHack 2019

Project: MOODIFY

Using footage from the Parrot Bebop 2 drone and applying Microsoft Azure API for Facial Recognition we determine the mood of a person or group of people and play music according to dominant mood.

Inspiration? We just wanted to use facial recognition and we started learning how to operate drones few days ago. Spontaneous ideas that turned out to be a fascinating project!

Learning outcome? We learned how to implement and use Microsoft Azure Face recognition API, operate Parrot Bebop 2 drone and its camera using python and select music based on emotion.

Challenges? Everything! It was challenging to take of a drone, took a footage from it (~7 hours). The calibration of the drone was off because we calibrated it on the slightly slanted floor of the building. Azure API, music selection and, of course, merging all the separate modules.
