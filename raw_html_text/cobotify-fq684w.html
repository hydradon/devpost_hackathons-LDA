Inspiration

Games are very popular and are strong tools for engaging users. I wanted to see if game-like features could be utilised to improve mundane tasks.

What it does

Presents a gamfied assistant that helps users manage or utilise their coworking space.

How I built it

The idea was to communicate a proposal for improving the existing COBOT online dashboard system.

Challenges I ran into

It took a long time to come up with an idea that had not been addressed so time was very limited to develop the idea.

Accomplishments that I'm proud of

I like the idea but it needs further development.

What I learned

Discussions with the mentors helped highlight real-world issues.

What's next for Cobotify

If the suitable opportunity exists then it would be interesting to develop cobotify further as an add-in for the existing Cobot software.
