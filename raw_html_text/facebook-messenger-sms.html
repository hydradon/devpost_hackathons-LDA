Using the SMS communication protocol to enable Facebook Messenger use in areas with limited access to mobile internet

Inspiration

Facebook has Facebook Lite, which is designed for users with low-speed connections and older devices. This is fantastic for users that have consistent access to a low-speed wireless connection, but it alienates more than 4.4 billion individuals that do not have access to mobile internet at all.

My project removes the need for mobile internet access in Facebook messenger. You can now communicate with users on Messenger entirely through SMS, enabling low resource communities to still benefit from the convenience of messenger without having access to a mobile internet connection.
