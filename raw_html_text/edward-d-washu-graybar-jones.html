Inspiration

We wanted a platform to help immigrants connect with each other and find communities to join.

What it does

Connects users trying to adapt to a new life in the United States.

How we built it

We used Firebase for the back end and angular for the front end.

Challenges we ran into

We tried using the Mac localhost on Windows. We accidentally called the google translate API 600,000 times with a cloud function and couldn't kill it. I entered the stadium on the opposite side and couldn't find my team because the orientation was reversed.

Accomplishments that we're proud of

We had almost no experience with any of these languages before we started, so we're happy anything worked at all.

What we learned

javascript, firebase, and angular.js

What's next for Melting Pot

A meme generator tailored for immigrants.
