gupshup chatbot + todolist link

_ by Kyle Marino and Kolby Pham _

Note: The chatbot's news viewing capabilities aren't available in a normal browser window, but work perfectly fine in GupShup's IDE's emulator.

The main challenge in development was trying to work around GupShup's shortcomings in external integration; sometimes, parsing/interaction would work in the IDE but not externally, and sometimes vice versa.

It runs off of an index.php, so it requires a local server to run if off of local files. Online, it can be run at: http://gupshup-kylemarino22164237.codeanyapp.com/

We find the design to be pretty intuitive. More on the chatbot will be shown in demo.
