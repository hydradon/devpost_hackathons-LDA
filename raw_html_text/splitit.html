Inspiration

Here's a classic scenario: you're having a nice night with the friends and order a pizza. Next thing you know, everyone's scratching their heads trying to calculate who owes who how much, and dollar bills are flying about. Here's an app that does all the hard work for you.

What it does

This is a group based IOU app that tracks inventory. It can be applied in the home when splitting groceries or the classroom while managing supplies. Your user profile will keep a log of who owes you and how much you owe.

How we built it

We built the app through Android Studio.

Challenges we ran into

Working to create a basic visual interface through Java. Working a slow emulator. Managing layouts, getting elements to appear on the screen, basic UI elements.

Accomplishments that we're proud of

Developing an Android app! Doing something!

What we learned

That we are not good at Java. Fundamental skills in app development. Managing dynamic elements accross activities. Creating a database with SQLite and manipulating it using queries.

What's next for SplitIt

Hopefully we will expand the database to allow for a more flexible formation of groups, user profiles, and inventory. Basically, a lot of backend.
