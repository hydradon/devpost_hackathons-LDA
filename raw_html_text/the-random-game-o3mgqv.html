Inspiration

While studying graph theory this problem came up and we thought about developing a game on it.

What it does

It creates a grid with randomly generated numbers and you have to traverse the grid gathering as many points as possible without getting stuck

How I built it

We used Java to build this project

Challenges I ran into

We had a lot of interesting ideas but they all flopped

Accomplishments that I'm proud of

The mathematics behind the game are quite interesting, employing some unique concepts from Graph Theory

What I learned

We learned determination and how it feels to run against the clock

What's next for The Random Game

We'll implement a Web App using Rust-Wasm. We also plan to make an app of the game in the future. We also plan on creating more complex map making algorithms.
