Inspiration

We were inspired by card games such as Magic: The Gathering but wished to experiment with mechanics never seen before in card games.

What it does

Our 3D program shows a battlefield on which the card game takes place, but the actual game rules were only implemented into a text-based prototype.

How we built it

We built the text-based prototype in Python and we built the 3D version in Unity.

Challenges we ran into

We had trouble moving the features from the Python prototype to Unity.

Accomplishments that we're proud of

We are proud of some of our unique game mechanics as well as our original artwork and music.

What we learned

We learned how to make a loop that manages a turn-system of a card game, as well as how to navigate scenes in Unity.

What's next for Battle of Estrania

We hope to make the 3D version of the game playable, and hopefully add many new features.
