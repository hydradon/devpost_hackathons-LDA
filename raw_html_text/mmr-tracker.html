Inspiration

Our friend group often plays Super Smash Brothers after school, getting into heated debates about who is the best player. Thus, we decided to make a program that would track matches and prove who was rated the highest.

What it does

Allows users to create groups, add players, and record matches. The server takes data from these matches and calculates how much MMR (Match Making Ranking) should be transferred to each player, which determines the leaderboard.

What's next for MMR Tracker

Next, we plan to add accounts so that players can keep track of individual statistics and groups that they are a part of much easier, a more fleshed-out MMR algorithm, and an apparatus that allows you to set up a game schedule.
