Inspiration

We didn't like the design of Rice University's website when we applied as new students, so when we had the opportunity to change it, we wanted to.

What it does

The new residential colleges webpage displays the information for individual colleges as a rollover of the college's crest.

How we built it

We re-designed the residential colleges webpage using only html and css.

Challenges we ran into

Designing the rollover without using javascript was difficult, especially in a horizontal direction.

Accomplishments that we're proud of

The website looks nice, is easy to use, and is informative. And the parallax scrolling is really cool!

What we learned

We learned a lot about how to implement html and css to design a beautiful and functional webpage.
