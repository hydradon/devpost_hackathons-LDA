Inspiration

As finals are coming up, we thought about how we use final grade calculators every semester to know what we need on our finals.

What it does

It calculates how much you need to get on your final based on the grades you find on BlackBoard.

How I built it

We used HTML and Javascript

Challenges I ran into

We did not have any experience with Javascript and limited experience with Html with the group, so we spent quite a while trying to figure it out as well as the calculations to work.

Accomplishments that I'm proud of

We're proud of getting it to work!

What I learned

We learned the syntax behind Javascript and Html and also how to calculate grades.

What's next for Final Grade Calculator

This can be applied as a widget for all classes so students can use it at the end of the semester.
