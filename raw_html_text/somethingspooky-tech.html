I was inspired by the Netflix sketch comedy show I Think You Should Leave and by my friends who are all far more experienced coders than myself. Somethingspooky.tech is a passion project of mine that allowed me to learn website design

Somethingspooky.tech is a simple website that responds to preset inputs with preset outputs. I built it from scratch with HTML and Javascript and used ZIET.co and Domain.com to publish it.

Although the website is wildly simple, it was a challenge for me since I have basically no coding experience. I'm very proud of Somethingspooky.tech, and I hope that it will prove to be memorable stepping stone as I expand my programming knowledge
