Inspiration

Make new cool dev tool.

What it does

It's a terminal application that has currently two plugins. TODO search: Function that returns all TODO's from popular open source project based on user input. Example: Keyword: Mining, Language: Python => Result: Ordered by popularity with TODO's. Emotion tracking: Tracks your mood threw programming session. In the end you can see the results of your mood and funny pictures of you while coding (funny pictures at 3am were made because of it).
