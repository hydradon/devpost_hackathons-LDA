Inspiration

We wanted to make mornings more fun! Most of us check the weather app in the mornings to decide on what to wear. We were inspired to create an app that incorporates the weather and music!

What it does

It gives the weather on an hourly basis based on zipcode, clothing recommendations based on the weather, and Spotify's top hits playlist.

How We built it

We used wunderground api to get the weather so that it updates each hour. We formatted our site through bootstrap and used ajax to get the data. We also used javascript and html
