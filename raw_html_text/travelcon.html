Inspiration

Having to register over and over for many events.

What it does

It allows bookings for lodgings and events to be made online, and checks the users face against their profile photo when they arrive at the destination to check in.

How we built it

We built it using Node.js and AWS Rekognition.

Challenges we ran into

The learning curve for the AWS services was likely the most difficult portion of this project.

Accomplishments that we're proud of

Coming together as a team to tackle different aspects of this challenge. Several of us used unfamiliar technologies for their assigned portions of the system, and learned a lot through this experience.

What we learned

We learned how to implement Amazon Web Services, web front-end using the Bootstrap framework, and some basic Kotlin.

What's next for TravelCon

Adding features such as registration timeout (checkout times), so that access is revoked after the event.
