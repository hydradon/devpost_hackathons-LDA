Inspiration

The inspiration came from the lack of a better idea.

What it does

Talloc sets timestamp markers on Youtube videos to let you know when a certain song can be found while listening to a mix, concert, album, OST, etc.

How I built it

This was built as a Chrome extension using Javascript and Youtube's API.

Challenges I ran into

The only changes we ran into were figuring out how to use regex to search for the information we needed.

Accomplishments that I'm proud of

Learning how to work with regex and making my first extension.

What I learned

Learning how to work with regex and making my first extension.

What's next for Talloc

We still need to make modifications on other cases for finding the tracklist from a video (different formats).
