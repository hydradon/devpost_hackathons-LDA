GoHealthy

GoHealthy application measure the surrounding electromagnetic field (EMF) and send warning messages to users when they are in a place with extreme high EMF or when some unusual EMF values are detected in the surroundings. GoHealthy can be very helpful for electrically hypersensitive people: select the bed location within the sleeping room with safe level of EMF, warning when they are entering a high level EMF places, warning when some unknown sources generate unusual high level EMF… GoHealthy is all about living in a healthy environment!
