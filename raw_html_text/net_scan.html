• Scan your network

Inspiration : I got tired of checking out my network manually.

What it does : it automatically gets the network information as soon as its plugged in to the network and displays it to either a phone or laptop via broadcasting wifi

How I built it : with a raspberry pi as the web server and network analyzer, we got it all connected using python and php

Challenges I ran into : one of the hardest challenges was getting the scripts to run all in conjunction and in order.

Accomplishments that I'm proud of : being able to make the project work! hooking it all up took a lot more planning that we originally thought.

What I learned : a basic understanding of how web servers work and how website information is delivered as well as how to automate processes using linux.

What's next for net_scan : more functionality! we're thinking of adding text message alerts next!
