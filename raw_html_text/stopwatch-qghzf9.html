Inspiration

I needed to have a stopwatch that kept tracking of time going forward, like a stopwatch!

What it does

It keeps track of time as a stopwatch

How I built it

AWS Amazon alexa using lambdas and dynamodb to keep track of elapsed time

Challenges I ran into

learning curve using lambdas and dynamodb API, IAM roles, async and await web calls

Accomplishments that I'm proud of

The stopwatch I have build works great and I'm already using :-)

What I learned

I've learned how to set up lambdas and dynamodb to communicate with each other well using roles, learnt about dynamodb API, IAM roles, async and await web calls

What's next for StopWatch

I will continue to add more features as I see fit.
