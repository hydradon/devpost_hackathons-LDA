• APL background

Inspiration

To try out Alexa APL and also for the prizes involved.

What it does

Soccer Game Facts provides fun the whole family can enjoy and is guaranteed to make you smarter. To start, just say "Alexa, launch Soccer Game Facts" or "Alexa, open Soccer Game Facts" to get a fact".

How I built it

I used Amazon Alexa ask-cli to build the skill and added APL documents and screens using the APL Authoring tool offered in the Alexa developer Console

Challenges I ran into

Adding APL feature to the skill. But APL resources and docs, and videos on twitch channel helped a lot.

Accomplishments that I'm proud of & What I learned

Was able to learn how to build APLs for Alexa Skills

What's next for Soccer Game Facts

I will add ISPs offering for the skill and add more user engaging features such as trivia questions every week
