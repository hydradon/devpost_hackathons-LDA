Inspiration: One of the biggest issue today is the refugee crisis - number of refugees are increasing and are barely getting basic human rights. People around the world want to help in form or kind but there's no platform for the givers and receivers to interact! hence, refugeXchange is worlds first platform where helpers and refugees could connect and mutually benefit.

What it does: helpers and help receivers create profiles. post the input of profiles - MapQwest API shows the data on a map to help connect with each other! helpers can help in form of Room, meals, basic sanitary supplies, education, professional services et. al.

How we built it: Bootstrapping . storage data thru google cloud and used mapqwest static map API to geo locate the helpers and refugees
