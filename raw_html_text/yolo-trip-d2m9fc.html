Inspiration

We would like to create an application for people who have conscious savings and likes to discover new places in the world.

What it does

It lists out where we can travel from our current location at a specified date by analyzing our savings.

How we built it

We built an iOS application using the Skyscanner API and Otp’s API.

Challenges we ran into

The biggest challenge for us was to use the APIs in swift.

Accomplishments that we're proud of

We are really proud of our demo video and that we could built an application from what we learned during this 24 hours.

What we learned

We came here as beginner programmers, so we had to learn a lot. I think we have never had such an intensive learning session before. We made progress in programing in swift and using APIs.

What's next for Yolo Trip

First, we are going to finish the design of our application because we hadn’t had time for it. Then we would like to add extra features like more specified searching possibilities.
