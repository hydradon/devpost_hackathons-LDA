Inspiration

I like Twitter bots and it seemed like an interesting idea

What it does

It's an automated Twitter bot that you can ask to look for Capital One bank branches or ATMs near a location, whether it's your current location or one you specify

How I built it

Lots of python and lots of APIs

Challenges I ran into

Google's Static Maps API is confusing

Accomplishments that I'm proud of

I made it so that the bot will only reply if you ask nicely

What I learned

I learned a lot about Nessie and several of Google's APIs

What's next for Nessie Twitterbot

I originally wanted to use Google's Static Maps API to have the bot reply with a snapshot of a Google Map centered on the results of the search. I ran short on time though, so I had to settle for just listing the addresses of the results
