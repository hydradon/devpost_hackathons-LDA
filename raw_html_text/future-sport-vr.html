Inspiration

The Inspiriation was when in the opening ceremony we saw Ashton Eaton's cruthes, and realized how innaccesible things are in life. We wanted to make something to revolutionize fan engagement, and also be helpful in making practices, engagement and other sessions more accessible.

What it does

This project is made with Oculus, Amazon Echo and Leap Motion integration with live Twitter API and SportRadar API, and our goal is to provide customized fan experience paralleled to live events.

How we built it

We built this using Oculus, Amazon Echo and Leap Motion integration with live Twitter API and SportRadar API.

Challenges we ran into

The main challenge we ran into was making virtual reality a possibility with this project. In order to work with Oculus Rift, we had to make sure we get a Desktop computer from the MLH Lab, since the oculus is now only "develop-able" with the desktop.

Accomplishments that we're proud of

We are proud to successfully integrate the amazon echo and oculus rift to provide entertainment and engagement with voice!!!

What we learned

We learned a lot about Unity developing environment and how awesome it is!
