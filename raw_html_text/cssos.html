Inspiration

Students in computer science barely can find videos or tutorials that speak in human language. We wanted to create a space for new computer science students that struggle with learning to code.

What it does

A website that directs you to information about computer science that is readable and easy to process.

How we built it

We build the website using css, javascript and html. We build it from scratch.

Challenges we ran into

Starting with coming with the idea itself we had problems collaborating. We constantly changed the idea and tried to implement more that we knew about.

Accomplishments that we're proud of

Making a website without any prior html/css/javascript expertise.

What we learned

Team work, html, css, javascript, API, people skills, networking, work under pressure.

What's next for CSSOS

More user friendly. A page that will allow students to get documentation that is easy to read. Links to more user-friendly programming pages. Find and API that grab news and displaying them in the main page. Compute science sponsors.
