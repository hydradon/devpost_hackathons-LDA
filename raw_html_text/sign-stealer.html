Inspiration

You should be able to quickly tell what another team is doing. Using machine learning maybe you can.

What it does

A person would plug in 5 signals and find out what that sign might mean

How we built it

We generated a bunch of test data of signs to simulate recording previously done signals. Then trained a tensorflow model which can predict what a series of signals might mean.

Challenges we ran into

We knew nothing about tensorflow or machine learning at the beginning of the day. But we powered through and learned a lot

Accomplishments that we're proud of

We have something that if we had real data could in theory... work

What's next for Sign Stealer

Learn more about machine learning and build something more meaningful
