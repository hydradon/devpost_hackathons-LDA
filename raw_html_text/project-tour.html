TravelMix (aka Project-Tour)

The Personal Tour Guide

How To Use

  1. Download the example or clone the repo
  2. Install it and run: npm install npm start 3. Then follow the on screen UI to build a dynamic tour route!

The Idea Behind TravelMix

TravelMix was concieved as an alternative to tried and true travel blogs and tour planners for building your touring plans. Simply choose a starting point and ultimate destination as well as some of your key interests and TravelMix will curate you your own tour plan! Leveraging Google's Search, Maps, and Geolocation APIs, TravelMix finds points of interest along your general route, prompts you to choose your favorites, and then generates the most efficient path through all the attractions.

The stretch goal of TravelMix was to use timing information about guests at each attraction to more accurately estimate tour route timing. Further, supervised learning could be used to train the model to pick better routes and destinations overtime. Our goal was to create a personal tour guide, accessible to anyone at anytime.

Technology

Travelmix is built off of a React.js fronend using Material-UI to provide styling. For both of our front-end developers it was their first React project. The backend is built off of Express.js which we utilized to handle complex logic and much of the API access heavylifting.
