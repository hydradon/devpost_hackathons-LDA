Inspiration

One of the hottest new inside jokes of the internet (often referred to as memes) has been the phrase "odd flex but ok." This is often a difficult phase to understand upon first hearing, so our goal is to be the #1 information epicenter for this culturally groundbreaking smash hit!

What it does

It simply directs you to a webpage (as all domains do!) where you can learn more about this hot meme! How much more elegant and straightforward can an educational website such as this get?

How we built it

After purchasing the domain from Domains.com, we were able to host the website under its name on Amazon Lightsail and then build the website under Wordpress.

Challenges we ran into

Our team had no experience whatsoever with any of these services, thus it was a mad dash to learn the user interfaces and how to integrate each service into a flowing, clean user experience that was kept fun yet educational. This steep learning curve was only climbable through the teamwork we shared and the willingness to contribute the each other's understanding that we showed.

Accomplishments that we're proud of

By not only choosing the best domain to indicate the topic material of our website, we actually built the website! This takes our submission of a simple domain name above and beyond the level of entry and imbues it with the character and depth that the website represents.

What we learned

Not only did we learn how to use Amazon Lightsail for hosting, Wordpress for building websites, and to use Domains.com to purchase a domain, but we were able to hone our teamwork and communication skills through needing to learn and quickly understand the completely alien user interfaces presented to our team throughout this project.

What's next for Best Domain Name Submission

The first addition would be a direct feed of all "odd flex but ok" content from Twitter, as it is often the breeding ground for the hottest new memes. The second addition would be a user-submitted content display feature in which we can share and display the best new user-generated content.
