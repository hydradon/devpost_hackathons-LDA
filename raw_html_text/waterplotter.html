WaterPlotter

City government receives aggregate data and a heat map showing flooding instances reported by civilians. The city can use this information to prioritize infrastructure repair and maintenance.

  • This can be used in any device!
    • (Desktop, Mobile, Microwave)

Built For:

  • The city and citizens of College Station

Built With:

  • Google Maps API (Geolocation and Static Street View)
  • JavaScript (vanilla but asynchronous), HTML, and CSS
