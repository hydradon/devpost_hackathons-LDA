Inspiration

Financial analysis is a huge and complex process, best way to simplify it is a use of technology. We decided to use multiply level process to achieve most accurate trust score of the company.

How we built it

We used yahoo API to get relevant news. Then we analyzed it in google sheets with and deliver right to the customer through messenger interface.

  • Used nodejs for news scrapping and lexalytics api.
  • Messenger api for interfacing with app.
  • Google sheets as a backend
