Inspiration

We wanted to build a game and we thought snake was a fun one.

What it does

We tried to make it implement multiplayer functionality so that we could have lots of people playing it.

How we built it

We used Java, experimenting with things like Sockets.

Challenges we ran into

We had lots of bugs in our code and we had to spend a lot of time trying to sort them out.

Accomplishments that we're proud of

We managed to make our computers talk to each other using Server Sockets

What we learned

We learned a lot about Java and building apps for it.

What's next for Snake

We will finish the multiplayer functionality and make it look nicer for a possible public release in the future.
