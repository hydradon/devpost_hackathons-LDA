Sermon of the Day Alexa Skill

The sermon of the day skill will respond to Alexa requests by first checking the user's current date and using that to index Desiring God's API sermon index and get a daily sermon specific for the day in the users time zone. It then streams this to the users alexa enabled device using the audio player interface which enabled starting over, pausing and resuming intents, all as voice commands.

If the device has APL enabled, a custom screen is sent made in APL to display information such as the scripture reference used in the sermon as well as the title, and two buttons pause/play and start over. These are treated the same as the voice versions.
