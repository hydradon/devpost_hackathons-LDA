Inspiration

We saw our designer in process of creating wireframes and we have decided to speed up this process.

What it does

It transfers paper wireframes into digital form using image recognition.

How we built it

We have created image recognition algorithm which transfers paper wireframes from photos to digital form like Sketch app files, which can be imported to Avocode, Invision and so on.

Challenges we ran into

Since we needed it to work offline, we had to do the recognition directly on the user's device, without any server or API.

Accomplishments that we're proud of

We have written the whole app in TypeScript, which can run both on the web and on mobile devices using React Native.

What we learned

Working in a team and a lot about image recognition and file formats. Also, we studied design process and made the app useful for real designers, which is pretty hard.

What's next for [AI] Vire

We will finish mobile apps and export to Sketch, InVision Studio, Photoshop and other tools used both by designers and developers.
