Inspiration

We know airports are scary

That's why we decided to make a simple web application for airlines to distribute to their first time flyers.

What it does

Using Angular, we made a simple web app that gives new flyers an overview of what to expect at the airport.

How we built it

We used VSCode and node.js to develop and host our application.

Challenges we ran into

Only one person on our team has done web development before, so we spent a few hours learning Angular and Node.js.

Accomplishments that we're proud of

We're glad to have had this amazing and immersive learning experience with some new frameworks that we have never seen before.

What we learned

Angular, Node.js, HTML, CSS, Bootstrap

What's next for NewFly

UI improvements, and connecting with airlines' APIs to replace our flight info page with actual information.
