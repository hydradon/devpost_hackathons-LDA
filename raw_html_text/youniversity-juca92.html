Inspiration

Enjoy games that fuel your capitalist needs? Now you can finally be the one to profit! Be in charge of your own university where you are no longer the student, but the one getting money from the students!

What it does

Build lecture halls, dorms, dining halls, gyms, pools, and roads and attract students to your university! The more students you can attract, the more money!

How we built it

We used android studio (java).

Challenges we ran into

There wasn't enough time to implement all the features we wanted, like custom tuition, reputation, save states, and more, but we have plenty more ideas to implement and make the game more enjoyable!

Accomplishments that we're proud of

We got the main features running, such as algorithms for attracting students, finances, and more, and it works well enough to run and play!

What we learned

We became more proficient in Android studio and made basic 8-bit designs!

What's next for Youniversity

Hopefully, when we get time, we can finish more important features like save states and background processing to make it more complete and continuous. We also need to add phone compatibility (currently only optimal on tablets).
