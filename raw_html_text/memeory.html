Inspiration

With the constant rotation of memes our generation generates, we thought it would be honorable to remember the memes of the past, present, and the overused. Memeory was created as a place to come find the latest memes and remember the lost memes.

What it does

Memeory is a simple website that has the option to display either a new, stale, or dead meme at the click of a button.

How we built it

We built this website with all members in mind. Many of us may have sight issues from browsing memes all the time so we created three large buttons that perform a simple action. We built this website to randomly select a meme from a database for the selected option.

Challenges we ran into

We ran into many small issues such as finding a good API, site design, and site consistency across devices.

Accomplishments that we're proud of

We’re proud that we were able to successfully implement JavaScript to parse an API response to randomly select a meme.

What we learned

We learned that memes never die. We also learned better API and JavaScript implementation.

What's next for Memeory

Grow website traffic and remember all the memes.
