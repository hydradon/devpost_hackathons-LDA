Inspiration

We work for Campusjäger – a Karlsruhe startup specialized in recruiting. There we learned about the importance of active sourcing, especially when recruiting developers. Thus we wanted to make a tool to make that process automatic for recruiters.

What it does

We parse and analyse Github profiles and give each talent a score. The score is based on over 30 parameters like code quality, popularity, social connections, and the number of contributions.

This score is then shown to the recruiter in a user-friendly way through a web interface. They can then write each talent a personalized message (generated with the Gtp-2 model) and bulk-mail people for the experience.

How we built it

We have a solid stack based on Go, GraphQL, Kubernetes, Couchbase and React.js.
