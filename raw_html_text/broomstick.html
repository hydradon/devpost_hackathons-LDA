Inspiration

Data set provided by the challenge shows a significant potential for an integrated building facility management app.

What it does

Basic facility management for building care-takers and users. Users will be able to observe free desks spaces and meeting room from the app as well as report any faults to the care taker by opening tickets. Fault report function allows user of the building facility to report any fault, with a picture from their camera. The system is also able to process video feeds from CCTVs and determine whether cleaning staff needs to be dispatched for cleaning.

How I built it

With Meteor.JS for the webapp, Python and OpenCV for image processing.
