Inspiration

Todos los retos y obstáculos que enfrentan hoy en día las personas que desean superarse e iniciarse en el mundo del emprendimiento

What it does

Facilita la comunicación entre los emprendedores y los inversionistas mediante una aplicación móvil y una página web en donde al inversionista se le desplegará un catálogo con las diferentes tipos de empresas o pymes, el inversionista puede seleccionar alguna de éstas, entablar contacto con el emprendedor y posteriormente realizar una financiación.

How we built it

Utilizamos android studio con java y el sdk de Firebase para la autentificación y para el desarrollo de la página web se utilizó PHP 7, con el framework para front-end Materialize.

Challenges we ran into

No sabíamos utilizar Firebase y jamás habíamos maquetado ninguna página web.

Accomplishments that we're proud of

Estamos orgullosos de haber estado en este hackathon y haber desarrollado una plataforma de éste tipo en tan poco tiempo, tomando en cuenta las limitaciones con las que contábamos cada uno de los miembros del equipo.

What we learned

A maquetar páginas web con materializecss y html 5, además de utilizar el sdk de Firebase para android.

What's next for 50 Inver-X

Asistir a más hackathones, prepararnos mucho más para ellos y adquirir más conocimiento, competencias y proponernos más y más retos.
