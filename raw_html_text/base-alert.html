Inspiration

We want to provide fans with the ability to get instant updates on their favorite baseball team in order to increase fan engagement

What it does

It sends someone message updates on their favorite teams upcoming games, current score and ticket prices.

How we built it

We built it using the Twilio api to send messages, the SeatGeek api to get the ticket data.

Accomplishments that we're proud of

What we learned

What's next for Base Alert

We hope to extend Base Alert by adding more functionality to the website. We plan to implement a points based system which can be used to win prizes. The points will be awarded based on how many times the user interacts with the app. We also want to add a gambling feature where users can gamble the points they have for more points even further increasing fan engagement.
