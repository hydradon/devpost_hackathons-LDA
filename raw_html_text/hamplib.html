Inspiration

The HampHack Library Challenge!

What it does

Shows a map of the building, service events, and has compiled catalog data and floor plans for the library.

How we built it

We used a variety of mapping tools, Facebook's react native framework for cross platform apps, an SQL server, and mapbox mapping API.

Challenges we ran into

Transferring between map formats is difficult and can cause lots of incompatibility. Getting all the data into a database was also difficult.

Accomplishments that we're proud of

We think we've created a solid starting point for a professional, open source, and very useful app!

What we learned

Cross platform apps are messy!
