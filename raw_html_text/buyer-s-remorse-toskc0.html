Inspiration

Spending too much money online.

What it does

This app takes your online purchasing data and calculates ways in which you could have better spent your money. My hope is that it will inspire better spending behaviors.

How I built it

I consumed the raw data from the Slice API then parsed it. I then created a data visualization.

Challenges I ran into

Figuring out the API authorization

Accomplishments that I'm proud of

Getting it done on time.

What I learned

I spend a lot of money on gadgets

What's next for buyer's remorse

Add more features and add some tests.
