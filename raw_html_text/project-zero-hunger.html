Inspiration

Many restaurants have food that ends up being wasted. Many would like to donate this food, but the logistics and other hurdles make it difficult. How can this be changed?

What it does

It allows for both restaurants and organizations for the homeless to signup. A restaurant can then make a food offer, and any nonprofits with volunteers nearby will be notified. Once one accepts the offer, a volunteer will be given the location of the restaurant to pick it up.

How we built it

It is built using Flask, Python, and PostGres on the backend, with React.JS on the frontend. We are also looking to implement an iOS app.

Challenges we ran into

Debugging errors in parsing ARGs. Figuring out PostGres, designing an iOS app programmatically.

Accomplishments that we're proud of

A deployed backend API, with models, controllers, and a React frontend connected to flask on the backend.

What we learned

Scoping, working with teammates, pari-programming, and debugging

What's next for Project: Zero Hunger

Fully implement all the features, partner with local organizations and restaurants, and build up the mobile side
