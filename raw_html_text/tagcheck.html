What it does

With TagCheck, you can take a picture of a product while browsing at your local supermarket and instantaneously check if other retailers offer better pricing.

How I built it

We used Android Studio, Google Cloud to extract text from images, and JSoup to scrape retailer sites.

Challenges I ran into

Using Android Studio to properly configure our data visualization.

Accomplishments that I'm proud of

Learning how to use Google Cloud OCR vision API and JSoup webscraping.

What's next for TagCheck

Better UI/UX design
