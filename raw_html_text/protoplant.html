Inspiration

Our plants kept dying (Dayeon even killed a cactus) and we wanted to solve that problem.

What it does

ProtoPlant uses Arduino sensors to monitor the temperature, light intensity, humidity, and soil moisture of a plant. It then sends the data to a locally hosted website that displays plant information, overview charts, and graphs.

How we built it

We built it using lots of new languages we learned on the spot (HTML, CSS, Javascript, C++, SQL, etc.).

Challenges we ran into

  • Had to fix multiple bugs and errors
  • Learn language syntax asap
  • Make each component of the project able to interact with each other

Accomplishments that we're proud of

  • Building a website in a day with barely any experience
  • Getting all the hardware to work with each other

What we learned

We learned a ton about new languages, software, hardware, and websites. It was an awesome learning experience for all of us.

What's next for ProtoPlant

  • Real-time updates for graphs and charts
  • Integrate robotics to make a plant on wheels that searches for sunlight within house
