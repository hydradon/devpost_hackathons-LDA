Inspiration

I usually do not work on frontend when I am working in a group so I took this chance to learn some JavaScript and see how it differs from the backend.

What it does

Type in monetary value and a currency. The program will return to you what that amount is in USD and what items you can purchase with said amount. It then gives you helpful links to purchase those items.

HTML and JavaScript.
