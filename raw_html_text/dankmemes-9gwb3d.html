making a tricopter with a rasberry pi and arduino to communicate with the motors

Tricopter was modeled in solidworks and parts were 3-D printed. The raspberry pi outputs a wifi network that a laptop can connect to to pull up the rpi terminal. The pi is connected to an arduino which controls the propellers through code.
