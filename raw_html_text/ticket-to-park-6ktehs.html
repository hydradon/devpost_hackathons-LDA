Inspiration

Hablamos sobre diferentes problemáticas que se tienen al vivir en un ciudad, de esta manera tratamos de encontrar una solución para alguna de ella.

What it does

Facilita el método de pago en estacionamientos con el objetivo de evitar pérdidas de tarjetas o boletos, además de ahorrar tiempo en hacer filas o desplazarse de un lugar otro.

How we built it

Primero enlistamos las funcionalidades que debia de cumplir la aplicación web, después, relizamos un diagrama con el mapa del sitio, para asegurarnos que cada funcionalidad fuera proporcionada. Por ultimo repartimos las tareas de desarrollo e integramos el pryecto. Utilizamos servidores propios para subir la pagina web. Utilizamos Adobe Muse para cuestiones de diseño. Ocupamos Dreamweaver y mysql para crear bases de datos y portales de inicio de sesión, aunque en la version final decidimos no implementarlo por cuestiones de tiempo.
