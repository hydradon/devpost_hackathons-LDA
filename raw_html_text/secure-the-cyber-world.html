Inspiration

To Teach Kids about Cyber Security and internet preparedness

What it does

An interactive world where students learn about how to be prepared for the cyber world through mini-games and an ultimate boss the Cyber Crook you must take down. Upon completion, students will receive a certification showing they are ready to tackle the internet.

How I built it

Game Design Build via Microsoft Word

Challenges I ran into

Technological knowledge to create the actual application.

Accomplishments that I'm proud of

The idea has big potential to help prevent bad actors from taking advantage of new users of the internet.

What I learned

General Cyber Security Knowledge

What's next for Secure the Cyber Land

Create the application, add more levels and make a game that is adaptable based on student age, difficulty, and learning objective.
