InvestSQ is a smartphone app that “rounds up” your spending to the nearest precision and invests that difference. You link a credit card and checking account, then InvestSQ does the rest.

How it works

If you make a debit card purchase for €2.25, InvestSQ withdraws €0.75. It invests the money in in one of three portfolio options Big Brave Bear that stands for Cryptocurrency investing, Hard-working Penguin that implies Peer-To-Peer lending and The Wise Wolf that stands for investing in bank funds and investment accounts.

You are to select rounding precision:

  • 0.51€ -> 1€ ( round half up )
  • 0.30€ -> 1€ ( round all the amounts to the the next integer)
  • 1.11€ -> 5€ ( some sort of bigger rounding, this should be defined as custom)

Select how big of a amount should be saved before we move it to the investment.

Nordea and InvestSQ will get commission (1% each) from every cashout.

Features
