• cryptoStar

Inspiration

NASA star / crater name registry.

What it does

It allows a user to buy naming rights to a star, which will be entered into the blockchain.

How we built it

nodejs backend frontend web stack Smart contract to handle the game logic

Challenges we ran into

Integrating with MetaMask. We successfully deployed a contract on the ropsten testnet that does the game logic, but having our frontend successfully interact was difficult. lack of api for accessing 3rd parties solutions

Accomplishments that we're proud of

Deployed a smart contract to ethereum. Cool frontend UI using javascript/react

What we learned

Debuging blockchain is hard.

What's next for cryptoStar

Add more features, fix the integration with the frontend. We'd like to maybe have the stars map to the real sky, and add integration for an AR system so you can own stellar real estate. Similar to Name a Star registries. We also would like the ability to add custom attributes onto your star.
