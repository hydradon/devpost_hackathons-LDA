Inspiration

One of our team members went out and bought an Oculus so we decided to experiment with it at this hackathon. We enjoyed the idea of a puzzle game and an escape room experience was the perfect way to combine the two.

What it does

It's an escape room designed to exercise your critical thinking skills.

How we built it

We built it using Unity, C#, and various free assets from the Unity Asset Store.

Challenges we ran into

We ran into some problems setting up the Rift because the environment around our hacking space was not sufficient at the start. We also had some trouble with the C# scripting because it was a new concept for our team. Unity was also a new framework for us.

Accomplishments that we're proud of

We're proud of the fact that we were able to hack together an immersive VR experience within 36 hours. Simply completing the project is a feat within itself because this entire project was a learning process for us.

What we learned

We learned how to program in C# and how Unity ties all of the elements together in the scripts. We figured out how the physics defined in Unity work with what we wanted to accomplish. We also learned how to account for UX problems such as motion sickness.

What's next for FreeEstate

Hopefully, we can add more puzzles that are a little more challenging and improve the graphics.
