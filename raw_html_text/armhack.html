ARM CAMBRIDGE HACKATHON 2017

A wireless server-based 3D scanner with website display. In-built accelerometers for sensor orientation calibration and light-based distance measure. Potential applications include landscape or maritime surveying in drones, search and rescue missions, and collaborative real-time object scanning. Created using the MBED CLOUD CONNECT BOARD during the CUES-ARM HACKATHON CAMBRIDGE 2017
