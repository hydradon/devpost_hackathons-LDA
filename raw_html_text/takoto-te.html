Inspiration

We wanted to raise funds for "Lideres del mañana" which is a scholarship program that gives 100% scholarships to future leaders; so we created an addictive game that we hope gets the money via advertising due to the high quantity of users of the app.

What it does

It is an endless game of a Ninja Taco called Takoto - Te that must evade the obstacles for him to survive.

How we built it

We used Unity, photoshop and C#.

Challenges we ran into

Nobody knew how to use Unity, cade in C# and we had to improve our Photoshop abilities.

Accomplishments that we're proud of

The final product, because we managed to build a basic application with the lack of knowledge in the tools that we used.

What we learned

The tools that we used, but most importantly we learned that with practice we can accomplish this kind of challenges and surpass obstacles with teamwork.

What's next for Takoto - Te

We will make it completely functional, because it is still in development stage.
