What it does

This game allows you to travel throughout the solar system while fighting enemies to reach the jump point at the end of the solar system before the sun collapses, returning us to the void.

How I built it

Game Maker Studio 2, GIMP, and Audacity were used in the development of this project.

Challenges I ran into

Networking was a big issue we had to deal with, as well as bullet path-following algorithms.

Accomplishments that I'm proud of

Our online leader board works, allowing for different clients to share the same leader board.

What I learned

We learned a lot about path-following algorithms for our rockets, as well as how to set up a network in GML.

What's next for Return Void

Some things we would like to implement in the future are allowing for multiple clients to connect at once, as well as adding new enemies and bosses.
