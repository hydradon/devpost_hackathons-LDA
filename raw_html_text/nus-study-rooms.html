Inspiration

The need to find vacant rooms to study in NUS can be a frustrating task. Multiple sources have to be accessed in order to find all nearby rooms that are vacant. There is a need for a convenient solution to this problem.

What it does

A telegram bot that gives the list of free rooms in the building the user is interested in.

How I built it

Using NUSmods API and NUS website as source of data and Python Telegram API to handle telegram messages.

Challenges I ran into

Familiarization to the Python Telegram API Familiarization to Python

Accomplishments that I'm proud of

Intuitive interface Extraction of relevant data from NUS website

What I learned

Python How to build a Telegram Bot
