Inspiration

Redesign of the Homi App

What it does

Homi allows students and recent grads to connect to alumni from their schools.

How I built it

First, I created paper prototypes of the mobile screens. I then designed these in Sketch 3, and created a working prototype of the design using HTML, CSS, and jQuery.

Challenges I ran into

I started coding it in Angular at first but decided to switch over to jQuery because I am more comfortable with it. The navigation animation is acting weird when the browser width is above 414px, but it works fine for mobile sizing (which it was intended for). I had some CSS difficulties when creating the info blocks, but as time went on they became easier.

Accomplishments that I'm proud of

I am proud of coding it out in less than 8 hours.

What I learned

I learned how to quickly and effectively prototype an app design using jQuery.

What's next for Homi

The next step for Homi is the implementation of the designs into the actual app.
