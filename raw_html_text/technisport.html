Inspiration

We all are involved in sports so we decided to use our interests to help others find their interests.

What it does

You can read a step by step instruction on how to kick a soccer ball and how to stop one. We also have a soccer kick demo using Arduino.

How we built it

We used Android Studio and Arduino to set up our app and demo.

Challenges we ran into

We struggled with getting the buttons to function correctly by opening another activity and implementing a tricky YouTube API system.

Accomplishments that we're proud of

We made an accurate demo using Arduino and cardboard.

What we learned

Working on something you're passionate about creates a better environment to learn in.

What's next for Technisport

We want to expand it to more sports and have different skills you can learn with ranging skill levels.
