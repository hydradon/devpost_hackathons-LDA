Inspiration

When working on enterprise level environment, it was a hassle to manage multiple resources in AWS and checking their performance without the help of a 3rd party tool. Without an external tool, a developer has to login to AWS console or cli, go to appropriate service dashboard and check the appropriate stats of the resource. I wanted to reduce these steps and help save some troubleshooting time.

What it does

Alute is an Alexa skill that helps you control and troubleshoot your AWS resources using your voice. You can manage and monitor your resources by invoking the skill.

What I learned

Learned to work with and create Alexa skills and utilize AWS endpoints.

What's next for Alute

I need to replicate similar capabilities for Microsoft Azure using Cortana as well as for Google Cloud Platform using Google Assistant.
