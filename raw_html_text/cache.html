Overview

https://s3.amazonaws.com/cacheassets/cache-ethdenver-presentation.pdf

Remember Drugwars? You may have played it on a TI graphing calculator! Cache is a web-based, ERC20 and ERC721 powered cryptocurrency trading game based on this old classic.

Instead of buying and selling drugs - you buy and sell cryptocurrencies in a cyberpunk dystopian setting.

About

CACHE is an also an ERC20 token created for powering the game. Conceptually, it can be exchanged inside or outside the game ecosystem.

The game that harnesses both in game currency as an ERC20 token, and the unique collectibles (non-fungible token) that a ERC721 token can provide. Think avatars and rare items.

For signing up, users are given an amount of the ERC20 token to first try the game. More tokens can be accumulated through successful gameplay.
