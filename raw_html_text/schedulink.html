ScheduLink is an arduino app with a django backend that allows users to dynamically set their availabilities in an interactive schedule builder, allows them to add friends (by phone numbers) and then allows them to compare schedules, in order to see who's available to hang out at any given time.

At this time, the link between the frontend and backend is incomplete, however, the functionality of each of these components individually is mostly implemented.

See the GitHub link below for access to all the source code

This app uses the Twilio API to confirm accounts!
