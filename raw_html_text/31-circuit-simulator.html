Inspiration

We've always being passionate about all kinds of technology and we love when we can inspire that passion on others, that's why we deemed suitable to create a tool to reassure this love for developing technology. And what better way to do it than using such an innovative technology as Virtual Reality to do so.

What it does

It's a circuit creator using a Virtual Reality and interactions for students and enthusiasts to learn how to build basic activities in a didactic and fun way. Also, you are able to see the circuit better as it is presented in an oversized manner.

How I built it

We used HTC VIVE technology from Valve Inc. through a plugin to implement it in an Unity3D environment (SteamVR). For the Unity development we used C# Scripting and the aforementioned plugin.

Challenges I ran into

Until this mere day we haven't had even used a VR Headset before, so learning how to develop with it was a major challenge. Getting the right VR Headset as Oculus Rift hadn't the proper support, we opted for the HTC VIVE instead.

Accomplishments that I'm proud of

Being able to build 3D worlds and objects that I can interact with using the VR Headset and the Vive controllers; it's an experience we are glad to have had the opportunity to do.

What I learned

Advanced Unity Scripting in C#, importance of networking, VR development using Unity.

What's next for 31: Circuit Simulator

Creating a fully operating proto-board that responds to interactions as closely as the "real deal". Implementing a system where you can save the circuits you've made and share it with others, as well as downloading and playing around with circuits made by others. And finally, to use it as a tool for spreading the fun and fulfilling building something from scratch can be.
