Inspiration

We were frustrated with how difficult it is currently to self-host your own email server and webmail client solution.

How it works

We provide a RESTful API service, and an AngularJS client designed for easily accessing JSON representations of your email. We hook into MailGun for sending mail, and we define a route in mailgun to receive mail.
