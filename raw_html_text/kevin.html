Inspiration

Going to hackathons and not having solid software to collaboratively work together easily on a project.

What it does

Uses firebase to use operational transformations to text editors in the web browser and uses GitHub to get files and where to save the files to.

How we built it

We uses javascript and html to make the webpages and used ajax and nodegit for the GitHub integration. Used jQuery UI for the nice looking tabs and navigational menu. Used firebase for the synchronous browser text editor.

Challenges we ran into

Integration with pulling and pushing to GitHub as well as using nodegit properly.

Accomplishments that we're proud of

Completing the multi-user synchronous Google Drive-like text editor with operational transformations.

What we learned

How to use javascript, ajax, jQuery UI with dynamic tabs, nodegit integration with GitHub as well as how to use node.js and host a local web server.

What's next for Kevin

To complete it fully and choose an actual name for our program. To complete the compiling of multiple different languages of code and producing outputs.
