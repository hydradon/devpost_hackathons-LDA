Inspiration

We wanted to create some type of assistive device for people who are not able to see (blind people)

What it does

It has two ultrasonic sensors that sense the distance to see if there is some object within 25 cm of the individual. If there is, the sound sensor will go off. alerting the user.

How we built it

We coded in Arduino to control the sound alert based on what we get from sensors.

Challenges we ran into

The inaccuracy of sensors and lacking knowledge of using the sensors.

What we learned

Using different types of sensors, using Arduino to operate, apply coding to real-life problems.

What's next for Blind Alert

We'll expand its functions and add more features, also make it more usable.
