Inspiration

Some awkward situations on dates

What it does

When you text an escape scenario to the app's twilio number, you will receive a series of timed texts that asks you to get away from the date.

How we built it

We used node.js and express to create the server that integrates twilio api to receive texts and to respond to the texts.

Challenges we ran into

We didn't know how to create delayed response to incoming SMS using TwiML, but we figured out how to parse the incoming SMS data for the phone number and to create our own outgoing messages to that number.

Accomplishments that we're proud of

We never gave up! We had to complete Twilio quests to gain access to upgraded account features so we can message users.

What we learned

How to use third-party API, Javascript setTimeOut features, Javascript callbacks

What's next for DateScape

Global launch in 2020.
