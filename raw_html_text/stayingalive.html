Inspiration

Heart diseases are one of the major cause of death, especially in western countries. Using more and more powerful and precise wearables, our goal was to build an app which allows to detect anomalies on the fly.

What it does

Using your wearable data and analyzing your heart rate and R-waves we detect anomalies and track these. Based on your risk group and data history you should

How we built it

We built a reactive-native app in order to support android and ios devices. Currently, wearable data is imported using HealthKit.

Challenges we ran into

It's always difficult to make use of data if it kept generic in order to protect privacy concerns. Using the given data sets cohort-analysis was not a realistic option within this hackathon.

What's next for StayingAlive

The next major feature is a summary and export for a doctor's usage. We want to further improve our algorithm to detect Atrial fibrillation or other anomalies. Furthermore, using patients information we want to work on cohort-analysis, in order to detect risk groups.
