Inspiration

We love the Scripps national spelling bee, and thought a live version where we could compete against friends would be a fun Alexa skill!

What it does

Compete against friends in a live spelling bee competition. The words get harder each round.

How we built it

Alexa Skill Kit on Node, Amazon Lambda and DynamoDB.

Challenges we ran into

Recognizing arbitrary-length single letter strings is not a strong skill of Alexa's right now, but we think we did a pretty good job!

Accomplishments that we're proud of

Hooking in the definition, etymology, and example phrases for words via a dictionary API makes the game more fun and more like the real spelling bee!

What we learned

Multi-player, live party games are a good use case for Alexa.

What's next for Spelling Bee

Competing against friends asynchonously in daily competitions—5 new words each day, and you can see how you stack up against friends, family, and the world!
