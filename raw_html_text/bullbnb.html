Inspiration

What inspired this web app was how tedious planning the most cost efficient vacation is. We have to search through so many different flights and Air BnB bookings, but it's still hard to find the best deal. We wanted to streamline this process.

What it does

BullBnB will accept: where you want to go or when you want to vacation, or a combination of the two. It will look at all the Air BnB options and give you a list of where and when is the best deal.

How I built it

We built this web app using flask. We chose flask because it was simple to get started and didn't require a lot of new knowledge. We also wanted out back-end to be in python since we were familiar mostly with python.

Challenges I ran into

The original idea was going to be this with flights and Air BnB, but flight APIs are only accessible thought requested keys, usually for business use. So we had to scrap that part and just do the Air BnB portion. Towards the end of writing the back-end though, we put too many requests to the Air BnB API we were using and were not able to have a functional back-end.

Accomplishments that I'm proud of

Very proud of all the work we put in and what came out of it even thought we had some major changed in the middle.

What I learned

I learned how to use flask on a basic level and to not trust APIs as much as I thought I could

What's next for BullBnB

What's next is to have a working back-end and to incorporate flight data as well.
