BE A FAN

Android and IPhone app designed to get you where your teams are for the game, concert, or performance Using Expedia’s suggestions, geo, and package search APIs we match up itineraries to get you to events

Our algorithm uses SeatGeek’s API to search for artists and teams with upcoming events. With some human learning, we select the correct performer.

With the performer we find ticket prices and trip packages for upcoming games and performances
