Two of Hearts

What it does

FinanceTool allows you to input revenue and costs with a tag for each of them, to export the data to text files (where it can be imported back in), or create a breakeven chart.

How we built it

We used Java and the JCommon and JFree libraries for graphing.

What we learned

Many methods of streamlining and making applications more efficient were learnt, alongside the use of the JCommon and JFree libraries.
