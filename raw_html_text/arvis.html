Inspiration

We were scrolling through the list of categories for this year's Technica, and competitions for the best programs during a crisis and for spreading education and diversity caught our attention. Since we were all passionate about at least one subject, we decided to create a chatbot that could educate people on the different subjects.

What it does

Essentially, the chatbot prompts the user with different buttons, each pertaining to a different subject. Based on the user's choice, the program will return information relating to the subject of the button press.

How I built it

We used react to build the chatbot, and a combination of Javascript, HTML, and CSS to build around the bot.

Challenges I ran into

There were many challenges in trying to get the bot to work using react, but help from mentors and research helped to better the challenges.
