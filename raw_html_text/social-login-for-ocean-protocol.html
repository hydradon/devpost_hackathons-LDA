Social Login for Commons Marketplace for Ocean Network

I integrated Torus with Commons Marketplace for Ocean Network. Torus allows users to log in with their Google, Facebook, Reddit, Twitch, or Discord accounts in a single step to start interacting with DApps. This allows users to download or publish dataset using their social accounts. There is no need of Metamask extension and it simplifies the onboarding experience for the Ocean Network. Using torus, users can also send or receive Ocean tokens or ETH using their social accounts.
