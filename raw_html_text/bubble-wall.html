Bubbly Inspiration

With the rise of lava lamps as decorative pieces, we wanted to create a display that was interactive and attractive.

What it does

Bubbly is a bubble wall display with a companion web app that allows you to create patterns, modify colors and interact with the water tubes in real-time.
