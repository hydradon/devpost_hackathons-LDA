Inspiration

We are committed to the spread of the dankest of memes. This plugin is increasing the dankness of the world one web page load at a time. Using tracking.js ( link ), we create a way for chrome users all around to add more dankess into their lives.

What it does

Dankify is a Chrome extension that inspects every image in the web page and overlays a meme face if the image contains a person's face.

How we built it

Dankify was made with a combination of Javascript and Jquery. It runs on Google Chrome.

Challenges we ran into

This was our team's first time using Javascript. Because of its asynchronous nature, it was very difficult to properly ensure that the code ran properly and consistently.

Accomplishments that we're proud of

We actually managed to create a working concept.

What we learned

As it was our first time using web programming languages, we had to learn a lot about the syntax of these languages. We learned how to set up a Chrome extension and how to use a Tracking.js, a facial recognition library.

What's next for dankify

We will continue to implement features that we were unable to today due to time constraints, such as increasing the size of the meme library and the ability to randomly select the meme to overlay. We will also increase the precision of the overlaying of the images so it fits better on the faces.
