Inspiration

Well we were inspired by ourselves since we were actually looking for teammates ourselves. It was really confusing to get all the incomplete teams to one room in the last minute and forming teams. This is really confusing and people end up puzzled.

What it does

So this app lets you apply as well as form teams once you have been accepted into the hackathon. Also this groups are done by mingling experienced and inexperienced people. This is done to give everyone a good experience by not leaving out any person idle or guilty for attending the hackathon.

How I built it

We used the Django module and python to create the web app. We also made sue of bootstrap,HTML and CSS to create the front end.

Challenges I ran into

Syntactical Errors Complex Django Modules Rendering Crispy forms Creating a platform to bridge front end and back end.

Accomplishments that I'm proud of

Learning CSS and bootstrap on spot to make proper use. Solving the Django Modules

What's next for Groupify

Creating new features in order to make this a part of MLH thus making group formation easy before hackathon.
