Inspiration

We wanted to build an app that impacted farmers this weekend. After hearing that the Climate Corporation was offering an exclusive API to HackIllinois participants, we knew immediately that we wanted to give it a try. We decided to emulate a popular product, Tinder, and combine it with the awesome API offered by Climate.com

What it does

Our web app allows users to sign in with their Climate.com account and import their fields. They can then view aerial imagery of other user's fields, and swipe right to "like" it, and left to "dislike" it. The user who owns the field can then view everyone who is interested in their fields.

How we built it

We built this app using Node.js, Express.js, and MySQL. We used Bootstrap and jQuery to power the frontend. This was the first time anyone on this team had used Node and Express to create a larger webapp.
