Inspiration

We wanted to give a full, clean, detailed layout of the daily UCLA dining hall meals.

What it does

Where the F*** Should I Eat? has a detailed list of the open dining halls and the food present at each, laced with a bit of profanity throughout.

How I built it

Nathan used Python, Flask, and Jinja to fetch the data and build the back end of the website. I used CSS to design the front end.

Challenges I ran into

Flask file management made some elements of the CSS not work, but we figured out how to adapt to it.

Accomplishments that I'm proud of

Making a hilariously stupid website was a defining moment for us.

What I learned

12 hours is not a long time to build something impressive.

What's next for Where the F*** Should I Eat?

Probably more profanity
