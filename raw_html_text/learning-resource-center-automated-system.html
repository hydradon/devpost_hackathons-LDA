Inspiration

The L.R.C had a very archaic system of which no student benefited

How it works

Site Managers manage the appointments, sessions, and tutors of the day. Students can access the list of classes available to the L.R.C. and see the wait list. Students however, must show up in person to make appointments.

Challenges I ran into

Organization with front-end, data base communication

Accomplishments that I'm proud of

Successful 2-way binding, overall simplicity of the front-end

What I learned

Communication with team members is an essential part to any project this expansive.

What's next for Learning Resource Center Automated System

Getting it fully set up and independent. Dependent on whether the L.R.C decides to take the next step with us.
