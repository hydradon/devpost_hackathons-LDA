Inspiration

Twang videos on YouTube.

What it does

You control a dot as it ventures through a one-dimensional space. Along the way, you encounter enemies and other obstacles that make the path difficult.

How we built it

We used Arduino language and IDE, and built a controller with a joystick and a push button.

Challenges we ran into

We couldn't figure out how to get a custom library that we wrote to work, so we had to put all of our code for each function into the same .ino file. We separated the task into sections and found it difficult to combine them all.

Accomplishments that we're proud of

We got the hero to move and change colors along with abilities. We also got the enemies and lava to kill the character when encountering them unprepared.

What we learned

How to combine analog and digital components to make a game.

What's next for One Dimensional Game

More levels, including boss levels, other obstacles and abilities for the hero.
