Inspiration

The demand to acquire knowledge or to learn has risen, thus, the present technology had gone a long way to provided means to learn, irrespective of the distance or the location of the e-learner and source of information.

What it does

An e-learning System is user-friendly and interactive software that can be used by both computer experts and non-computer experts to prepare an e- learning in any subject of interest and to present lectures to the target audience in full media. The resultant is accessed online by the target audience. The software features automated assessment and grading of students.

How we built it

Implementation languages include JavaScript, php, css, bootstrap, Grid3 Api html; we use the tools to create an e-learning system on any subject of his choice, while the software platform still remains intact.
