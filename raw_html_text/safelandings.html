What it does

Utilizes Amadeus and IMO APIs to compile public health information near travel destinations. The user provides a city of origin, for which Amadeus provides potential tourist destinations accessible by air. The user chooses a city from the possible destinations and healthcare providers' logs for contagious diseases in the area are found using the IMO mySQL database. The site then generates a graph of these diseases, ranked by number of recently reported cases.

How we built it

The backend that access the APIs and manipulates the data was built on Python. The front end utilizes HTML, CSS, jQuery, and Javascript, along with Python and Flask to interact with the user and provide the requested data in a meaningful manner.

Challenges

Effectively connecting two different APIs- outputs and inputs of two different APIs. Front end web development using Flask with dynamic user input.
