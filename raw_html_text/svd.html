Inspiration

To help with housing development for immigrants in the city of Brampton.

What it does

Gathers various data points from soil to intelligently determine the suitability of the soil for future building.

How we built it

Used soil moisture sensor, conductivity sensor to be wired to a Feather Huzzah! development board and displayed into two LCDs.

Challenges we ran into

Determining the correct algorithms to intelligently determine if the soil is suitable or not. Managing time to allocated resources for development of this device

Accomplishments that we're proud of

Accomplished what we set out to do within the time period and we are happy with the result. We were able to transmit the the result to the cloud so data could be stored and can used at the later time.

What we learned

Communication Teamwork Time management Allocating Resources Working in a stressful enviroment

What's next for SVD

Improve reliability Improve modularity Improve performance
