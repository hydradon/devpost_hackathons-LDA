Inspiration

Will Kittler's grunge hipster fashion

What it does

Shazam for clothes - By taking an in-app picture, you can find clothes similar to a clothing item that you see.

How we built it

We used the Microsoft Custom Vision API and trained our model using a large database of fashion pictures and tags. We could then feed in-app pictures into the prediction portion of the API, which returns tags that are most likely associated with the clothing item. Using these tags, the app conducts a Bing search that returns images and links to websites with similar apparel.
