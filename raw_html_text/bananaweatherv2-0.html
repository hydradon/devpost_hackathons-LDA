Inspiration

On our way to the hackathon from Charlotte, we saw a fog in the air and wondered if its safe to breathe it and how polluted it was.

What it does

it send a Text notifications warning people about polluted air.

How we built it

we used a raspberry pi 3 to run a Node.js server. we used an API to check the air quality in cities every 30 minutes and send a text warning if the air is poisoned.

Challenges we ran into

knowing how to use Node.js. never worked with APIs before and discovering what Jason is, and working on a raspberry pi for the first time

Accomplishments that we're proud of

getting the raspberry pi to send text messages

What we learned

Jason is a JavaScript Object :) when you're using a new tool, never spend too many hours trying to debug it because there are always many other tools that you can use. there are so many useful APIs that do different things

What's next for BananaWeatherv2.0

having it check for pollen and weather hazards not just air quality
