• Patients put basic health information that they would like emergency services to know about them - they can choose what information to put

  • Patients may also optionally list any medical histories they have

  • Patients register for the service with some personal information

  • Upon submission, their medical records will be entered into the database for emergency services to access in emergency situations

Inspiration- preventable EMT deaths, long wait times for the ER

What it does- keeps information on one app and allows medical professionals access to vital information

How I built it- I built it with react native which is a Facebook powered platform based on javascript. React native allows us to program for IOS and Android simultaneously.

Challenges I ran into- Some libraries used were not compatible with one another

Accomplishments that I'm proud of- Finishing the app

What I learned- I learned much more about react native and python

What's next for Talaria- patients will be able to see own information and hopefully we will implement facial identification as well as finger recognition to identify unconscious patients
