Inspiration

본 프로젝트는 신창선 스쿨 매니저님의 제안으로 시작되었습니다. 매일 스무 명 남짓의 학생들의 출석과 과제 여부를 엑셀 파일로 관리하는 일이 번거로워, 파일을 웹으로 옮기게 되었습니다. 일상 속의 작은 편리함이 삶의 질을 높여주기도 합니다.

What it does

스쿨 매니저가 관리하는 클래스 수강생들의 상세정보, 출결, 과제, 시험점수 등을 웹 애플리케이션에서 관리할 수 있다.

How I built it

  • Angular Framework로 제작
  • 3명의 팀원이 Git을 통해서 협업
  • Component를 분리

Challenges I ran into

  • 백엔드 없이 프론트엔드만으로 작업을 하려다 보니 json server을 사용하며 어려움이 있었다.
  • 서버에 대한 이해가 적다보니 기본적인 HTTP method인 get, post, patch 등만 사용해 데이터를 추가로 가공해야하는 점이 번거로웠다.

Accomplishments that I'm proud of

짧은 시간과 데이터베이스/서버의 부재에도 MVP를 구현했다.

What I learned

팀원의 소중함 협업의 중요성 좋은 프로젝트는 개인의 역량이 아니라 여러 사람이 함께 할 때 나올 수 있다는 것을 배웠습니다^^
