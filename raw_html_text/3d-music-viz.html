Headline

We made this visualizer because we like music and wanted to create a virtual reality environment for visual effects.

It responds to the audio output and animates 3D arrays of cubes which matches the beats. It was built using the Unity game engine, and can be viewed in VR through an Oculus Rift. In the future, our team could add more effects such as laser lights and implement a "point and shoot"-type controller which uses head movements to make selections in the GUI instead of gamepad buttons.
