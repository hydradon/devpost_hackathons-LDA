Inspiration

When my friends and I shop for our dorms and apartments, it's hard to visualize how all the furniture will come together, because we don't know the dimensions of the items we want to buy.

What it does

Uses our logo as a marker to render a 3-D, accurately-sized item. Users can browse and select the furniture they want.

How I built it

Vuforia is an AR software development kit integrated with Unity. I used Unity to build the UI.

Challenges I ran into

Bringing Vuforia and Unity together and creating an Android app. Delegating tasks to team members since everything was so closely linked together.

Accomplishments that I'm proud of

We created a working and visually appealing user interface.

What I learned

How to create an Android app, Unity, C#, augmented reality's applications to the future.
