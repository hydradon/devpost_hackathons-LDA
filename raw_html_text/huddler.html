Inspiration

We all have those really difficult course that we need help with. However, finding a study group can be a challenge.

What it does

Let's you create instant study groups or join existing groups in seconds. Once you create or join a group, we'll shoot you a text. Moreover, anyone signed up for a course gets a text when a new group is created.

How we built it

Front-End: React web app Services: Python-Flask API: Twilio, Facebook

Challenges we ran into

Integrating the services and APIs with the app.

Accomplishments that we're proud of

People with completely different set of backgrounds and skills working together to a working MVP!!!

What we learned

Integration is a time sink.

What's next for Huddler

Precise location search and pinpointing, more functionalities leveraging Twilio.
