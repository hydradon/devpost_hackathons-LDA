Improve your Speech

Speakr is a powerful speech analyzer that uses Microsoft Azure's speech to text API to parse your speech in real time and return data about how you talk. It's goal is to help user's improve their speaking style through tangible feedback and long term tracking.

Data Analysis

Speakr returns speech data such as speed, number of words repeated, and number of filler words spoken as well as an overall score of speech excellence. Our algorithms run seamlessly in the back end to analyse and deliver this data in a user-friendly format.

Long term tracking

All recording sessions are saved onto Firebase (database in the cloud) for long-term tracking and data retrieval.
