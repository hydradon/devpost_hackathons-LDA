Inspiration

Instagram C-list celebrity @Wildling_by_choice tends to wander off into the forest, and what would take a 6 hour man hunt now only takes a minute.

What it does

This code will receive an SMS, parse the sender's phone number and automatically sends a GPRSloc response

How we built it

It was built with a SIM808 GSM module for communication and a STM32 for all data parsing and battery management.

Challenges we ran into

Parsing SIM808 GPRSloc data into readable NMEA coordinates for Google Maps API

Accomplishments that we're proud of

Battery Life from a 2000mAh is ~1 month with 10 request per day. Deep Sleep mode allows for an estimated ~6 months with 1 request per day.

What we learned

IoT is Awesome!

You can make any[thing] into a IoT device.
