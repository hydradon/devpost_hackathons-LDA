An interactive cloud connected learning aid to help autistic children interact with the surroundings via sight, sound, and touch

How we did it? - Real time sensor data acquisition on AWS IOT platform with data processing using lambda functions. Interactive game to respond to voice commands using alexa voice services

With all the coding and hardware that we needed, we are excited that we have a prototype completed

We were overzealous in our ideas and instead of a couple bags full, we need a suitcase full of hardware. You'll never know what you are going to need.

Future would be to miniaturize the product into something more consumer friendly. Also determining how much power is needed to use the proper equipment to minimize cost.
