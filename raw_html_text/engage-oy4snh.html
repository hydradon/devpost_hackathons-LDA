Inspiration

Improving fan engagement at ballpark

What it does

Helps stadium staff make data-driven decisions in allocation cheer-leading, concession, and promotional activities. Uses machine learning on real-time video feeds of fans to measure engagement across various sections of the ballpark

How we built it

Frontend dashboard (React JS) Backend machine learning API (Google Cloud Vision API)

Challenges we ran into

Getting enough data

What's next for Engage

Future updates to enable collaboration and communication between staff A more robust intake system to consolidate multiple feeds Additional camera hardware to monitor each section of the ballpark
