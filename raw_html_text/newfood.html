NewFood is a project to help move thrown away food from big chain grocery stores to local homeless shelters. The database and code incorporate crowd-sourcing to add providers, drivers, and receivers to the mix so there is never a loss of location of the food in transit. The code utilizes MongoDB and helping the community to make feeding those in need efficient and hassle free.
