• hatchasong

UPC Hackaton 2018

Roc Granada & Ixent gallego - UPC Hackaton 2018

You can use this tool either using the app menu executing the program with the command: python hatchasong.py

Or using directly command line to pass the input parameters with the command: python hatchasong.py

modes: "single" | "multi"

enter the name of the file or de folder depending if the input is a midi file or a folder containing multiple midis.

The output will be a automatically generated song that has the same music style as the inputs but it is completely different.

Feed your inspiration and your games!
