Inspiration

Think of the doggos.

What it does

Snoop is a web application that connects prospective dog owners to shelters and pet stores that have the perfect pet for them. For the kennel owners, it provides a place to post pictures and descriptions about all of the dogs that they have available

How we built it

We used HTML, CSS, Javascript for the front end. Azure, PHP, SQL for the back end.

Challenges we ran into

Azure didn't work for a lot of the time. PHP was buggy.

Accomplishments that we're proud of

We got the back end to work and a sick page.

What we learned

To spend more time on back end.

What's next for Snoop

Acquisition by Google.
