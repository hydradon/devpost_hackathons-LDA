Inspiration

After competing in many hackathons. A common task was reading a CSV file and uploading the data to database.

What it does

ApiSheet creates an API endpoint from the uploaded CSV file to which user can send some HTTP requests.

How I built it

I created a server using express.js and mongodb and integrated my app logic.

Challenges I ran into

Creating a custom API access management system which involved allowing users to create API key and secret pairs so they can access the API endpoint.

Accomplishments that I'm proud of

Creating a custom API management system.

What I learned

How to use mongodb, nodejs and socket.io.

What's next for ApiSheet

Adding more features like

  1. Resetting the API.
  2. Allow pagination capability in API
