What it does

Learning the command line for the first time can be scary. Shellalian is a friendly shell meant to make shell commands more intuitive. Especially for beginners and anyone that enjoys a fun command line, Shellalian uses animations and UFOs to teach what happens in basic shell commands. For the hackathon, we focused on the most basic and highly used Unix commands: cd and ls. Fall into warpholes to enter child directories and get absorbed into the mother spaceship to move to the parent!
