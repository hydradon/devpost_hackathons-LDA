KP Nearby

Android and web application to find nearby places in KP

The web version is live at http://kphu-suaoc.rhcloud.com . This app has data for health units for now. We will add more data for health units as well as schools, museums, and other public places.

Uses geo-location api to find nearest 10 places. It also allows to find nearest 10 places to an arbitrary point on map.
