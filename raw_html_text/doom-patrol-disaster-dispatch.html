Inspiration

Hacktx JP Morgan challenge and the recent crisis with hurricane Harvey and Irma

What it does

Using twitter API and python libraries we analyze live tweets during a crisis to prioritize geographical locations in a given area to properly allocate relief efforts and optimize volunteer help.

How we built it

Use twitter API to scrape recent tweets with specific hashtags and looking at the geocode of those tweets, using data logging and analysis libraries we created crisis maps.
