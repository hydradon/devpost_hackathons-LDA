Inspiration

We are broke college students, and we want to find all deals in surrounding grocery stores to help us decide which store to go to.

What it does

It scrapes the weekly ads from all surrounding grocery stores, and compiles it into a central location.

How we built it

We used python as a web crawler. node.js as our background server, html & css to develop our frontend.

Challenges we ran into

We didn't know how to develop web pages, create servers, or crawl through websites.

Accomplishments that we're proud of

We we're able to design and finish a project in the span of 2 days.

What we learned

We learned to develop web pages, servers, and crawl through websites.

What's next for MyGroceryPal

Add more support for more grocery stores.
