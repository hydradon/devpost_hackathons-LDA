Inspiration

Thousands of people are stuck in traffic jam every day commuting to Prague, wasting time, polluting the air and making Prague itself a bit less nice place to live. Only if they knew they can effectively use a broad network of P+R parking places around Prague, which will make them drive less, save planet and be happier.

What it does

Park & Roll is a mobile app which helps people decide whether they could save time driving to Prague by using one of available P+R parkings. The app provides real-time data about free parking spots for all available parkings, including estimated times to get there and link for the navigation app.

How we built it

The app is powered by React-native. The backend is built on top of modern serverless engine, using both nodejs and python for data handling. For directions data, the google maps api is being used. For real-time info about available P+R, golemio public api is being used.
