Inspiration

The inspiration for this project came from our company where employees are using way too many applications and accounts for different purpose. We’ve managed to develop one single application where Human Resources, Managers and Developers will use it for different tasks.

What it does

The Crystal Hub (CH) represents our solution which brings together some tools that we are using in our company into a single one, and we are thinking of a global solution for other companies to use in this way. Using this application can save more time that we are spending on different tasks, and this is one of the most important thing.

How we built it

We found this as a main problem in our company, and we take it as a challenge, to think at a solution that could be implemented in our company. We came with the ideea and we have started to think how to improve it and how it works better in the company.

Challenges we ran into

The main challenge for the project was to do all the best for the application, to work fast, and efficient. Debugging was also a challenging part of the application because we experienced different problems which we never encountered before.

Accomplishments that we're proud of

Why we are so proud of this application? Because we start with a small ideea of implementation, and we managed to develop a complex application in a very short time thanks to Appian.

How we did the implementation? Firstly, we thought separatly and wrote on a paper the main actors and the ideas for the application. After that, we discussed which of the ideas are the best, and in the end we combined them on what you can see today.

Why we choose this idea? Because we want to use only one application for different purposes, and Appian solved this problem.

What we learned

We learned a lot working on this project; starting from Appian platform on a higher lever that comes with the newest release of 19.1 and also the best parts were how to manage a team, knowledge sharing and collaborate for a successful app.

What's next for Crystal Hub

The next step for Crystal Hub is to add more functionality and to make it even more complex. We are excited to see the upcoming version of Appian so we can improve our application.
