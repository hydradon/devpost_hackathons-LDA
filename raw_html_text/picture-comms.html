For WHACKS (Wellesley Hackathon), February 2016

By using state of the art object recognition software provided by indico, we were able to identify objects and translate them to other languages. This allows multi-lingual exchange between people who do not have a common language.

Originally we wanted to incorporate the GoogleTranslate API in our app, so that users would be able to translate the recognized object into the language of their choice.
