Going to college hackathons for the first time, our team has discovered the struggle of locating upcoming hackathons and learning about version control and technologies used at hackathons. Our Android app aims to ease this process by providing users with a ready-to-go app with all of the useful information on hackathons and useful technologies all in one place, at their fingertips.

We built the app using Android Studio to connect the UI of the app and create a seamless experience for the user to explore and Hack the Hack[athons]! We also implemented JSoup to webscrape the MLH website for upcoming hackathons and Java programming to extract the useful information, which is displayed in the "Hackathons" tab. GCP Firebase was used in the backend to create users and to store their login and profile information.

We also have a website to advertise the app!
