• Mr. Potato Battery

Inspiration

IoT devices generate a lot of battery waste. Let's see if we can build useful sensors powered by fruits and veggies! For Science!

We'll try it with some barebones Bluetooth modules. It would also be cool to get LoRa modules working, if we have enough food. Maybe send sensor data to Azure (IoTHub + Power BI)? In terms of interest or skills: Azure (Python SDK?), electronics, and creative prototyping/design.

I'll bring a potato. If you got something growing in the garden, feel free to bring it to hack day.
