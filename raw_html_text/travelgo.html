Inspiration

Itinerary planning is sometimes a long and tedious process. We want to make it more fun and interesting for our users.

What it does

Itinerary planning through integration with TripCase API, which is further augmented by using VR.

How we built it

We built it using the Unity Engine (same engine used by Pokemon Go). We also explored using the TripCase API to pull itinerary details for Attractions.

Challenges we ran into

There were some VR setup issues as we had to change the app orientation between landscape and portrait for VR use and normal use.

Accomplishments that we're proud of

Interesting user interface and nice graphics. Rich media makes a difference.

What's next for TraveGo

Test it out with actual users after further polish. Expand the test data to include other countries.

API used

Tested the TripCase Itinerary API
