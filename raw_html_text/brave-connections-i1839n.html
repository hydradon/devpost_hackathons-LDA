Inspiration

First generation students need help to succeed in job interviews. Braven is helping fulfill this need by connecting professional mentors with students.

What it does

It automatically matches students with mentor interviewers.

How we built it

The front-end was done with Angular, the middleware is express JS, and the back-end was made using mongoDB. Using Flask framework, we made the recommendation engine in python.

Challenges we ran into

Connect the database with the front-end app.

Accomplishments that we are proud of

We build an effective matching recommendation system engine.

What's next for Brave-Connections

A notifications system to let the interviewers and students know of the approaching mock interview events.
