Inspiration

MTU shutdown barkboard which is what most Tech students used to find rides for breaks. Ever since then we have needed a good program to help with this loss.

What it does

Takes info from drivers who want to provide rides. Shows info about available rides to people needing a ride.

How we built it

Used Xcode to build the iOS app. Using firebase for database.

Challenges we ran into

Everything since we are both new to iOS and Firebase.

Accomplishments that we're proud of

Getting data from the firebase and putting it on the screen. Sending data to the firebase.
