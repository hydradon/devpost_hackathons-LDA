hastea.org Pronounced Hasty

Inspiration

SLO and SB have pretty bad boba. We wanted to get boba from LA and the Bay Area. So, we created Hastea.

What it does

You can create boba runs to get boba from places. You can join runs to get boba from people doing boba runs.

How we built it

We kinda just typed and this came out of our fingers

Challenges we ran into

Our code kept getting reverted on accident and we all had to learn ruby on rails. Windows also got corrupted due to root user issues.

Accomplishments that we're proud of

We actually finished

What we learned

We learned how to break windows to the extreme and use ruby on rails I guess.
