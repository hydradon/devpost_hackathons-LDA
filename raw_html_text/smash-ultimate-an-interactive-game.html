Inspiration

Our inspirations were Black Mirror's Bandersnatch and the World Of Light Classic Mode in Smash Ultimate

What it does

We made an interactive, visual-novel in which every choice you make will affect the path you take.

How we built it

We started by brainstorming potential endings. We used Unity and developed the storyline using a flowchart.

Challenges we ran into

Since we're a beginner group, we had to learn how to use Unity on the spot. We experienced merging conflicts that we couldn't resolve, so we ended up working on one computer.
