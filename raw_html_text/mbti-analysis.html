Inspiration 

Long time passion for the Meyers Briggs Type Indicator. Currently serious lack of data analysis available on the MBTI. More analysis with give the MBTI more scientific backing

What it does 

Currently a demonstrative tool for exploratory analysis on the different types within the MBTI

How we built it 

Further expanded knowledge on html, css, and python to design webpage and collate data in easy-to-read format

Challenges we ran into 

We wanted to incorporate machine learning and public webpage access, but due to time constraints and knowledge gap, we had difficulty implementing initial goals.

Accomplishments that we're proud of 

Design of website, developed understanding of data set, and increased knowledge of utilized languages.

What we learned 

How to program in html and css with interactive capabilities with python.

What's next for MBTI Analysis 

We're going to continue working on the machine learning component and web page access elements.
