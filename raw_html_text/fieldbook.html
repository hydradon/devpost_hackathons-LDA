Inspiration

In the era of big data, we are interested in using data science to help farmers invest and farm their lands more efficiently.

What it does

It gives users the direct insight about value of different agriculture land in different counties

How we built it

We crawled officially published agricultural data from multiple governmental sources. Designed a comprehensive evaluation method for scoring land value Used d3.js to visualize our results Used Climate API to access user data to help them check their lands

Challenges we ran into

Design an appropriate evaluation method Combine different technologies/frameworks together Efficient usage of the APIs

Accomplishments that we're proud of

We designed a solid evaluation method

What we learned

A lot

What's next for FieldBook

Add more data and expand our areas of services Optimize the evaluation method Optimize data storage and retrieval speed
