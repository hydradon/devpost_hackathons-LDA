What it does

The we analyzed and visualized data from this public dataset (found on UCI's machine learning repository ) using Microsoft's R client. The data provides various readings after the detection of various types of failure of a robotic arm to grasp an object. We analyze this data for commonalities.

How I built it

Using R.

What we learned

We found that the most influential variable was force in the z direction. So in a machine learning context, the only relevant factor for predicting the execution failure of a robotic arm is this variable.
