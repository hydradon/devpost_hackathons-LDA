Inspiration

We all participate in volunteering in some way, and upon finding out that we had that characteristic in common, we thought it'd be a great project to build together!

What it does

People just text the number on our webpage to find out about specific volunteering opportunities!

How we built it

We built it using the Python coding language and by using the Twilio software.

Challenges we ran into

Our text message took a lot of debugging to display the results properly.

What we learned

We learned a lot about various coding languages from each other, as well as new opportunities for ethical hacking.
