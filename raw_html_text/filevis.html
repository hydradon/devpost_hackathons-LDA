Inspiration

I choose this project because it is related to the cyber security challenge and the visualization challenge.

What it does

The app would be able to let user create segments within a binary file and choose their decoding method, making reverse engineering a binary file format easier.

How I built it

I built it with Vue.js for the UI, and uses webpack for new javascript features.

Challenges I ran into

Building the UI is a lots of challenge. I took too much time just fixing some white space on the page that I didn't get to finish the actual features.

Accomplishments that I'm proud of

Not really getting much done, but this is one of the biggest web UI project I have attempted.

What I learned

I learned reactive programming with the Vue.js framework.

What's next for FileVIs

The project is not done. Most of the features are not done. I need to implement creating file segments, decoding algorithms, search features, etc.
