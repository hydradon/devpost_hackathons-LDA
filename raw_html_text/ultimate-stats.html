Inspiration

We're a team of ultimate frisbee players, so we wanted to make something related to that! Ultimate statistics are currently not super easy to find for high-level club teams, so we wanted to collect all these stats in one place and make it as simple to use as possible.

What it does

It provides a simple web UI to view ultimate frisbee stats.

How I built it

The frontend was built with Dash, a Python library created by Plotly, and some backend web scraping for data was done with Python as well.

Challenges I ran into

Getting the web scraper to work!

Accomplishments that I'm proud of

Actually making something that works, even if it's not super complicated.

What I learned

How to use Dash and Pandas.

What's next for Ultimate Stats

Better UI and possibly more features!
