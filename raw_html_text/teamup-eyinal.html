Inspiration

Dans la ville intelligente, les individus craignent de se pointer à une patinoire pour demander à des inconnus s'ils peuvent jouer à un match de Hockey, au grand risque de se faire rejeter ou de s'être déplacé pour rien. Avec TeamUP, les limites se dissipent pour permettre aux utilisateurs de jouer au Hockey sur n'importe quelle patinoire extérieure de la Ville de Montréal et cela, en tout temps.

What it does

  • Voir les patinoires sur la carte
  • Voir les parties en cours
  • Se joindre à une partie en cours
  • Planifier une partie
  • Se connecter à l'application par E-mail

How we built it

Nous avons utilisé Android Studio pour développer notre application et Firebase pour authentifier les utilisateurs. Étant donné le court intervalle de développement (24h), nous avons décidé d'éviter la programmation en paire. Pour construire l'application, nous avons donc séparé les tâches de la façon suivante:

  • Implanter l'authentification d'utilisateur par E-mail
  • Développer l'interface utilisateur
  • Utiliser l'API de Google Maps
  • Ajouter des patinoires sur la carte de Google Maps
  • Utiliser la base de données de la Ville de Montréal pour les patinoires

Challenges we ran into

  • Utiliser des programmes qu'on ne connaît pas
  • Modifier le projet en temps réel, avec Git
  • Utiliser Realtime Database de Firebase comme base de données
  • Utiliser l'API de Google Maps pour afficher la carte des patinoires
  • Mettre le logo en png ou jpg dans l'application
