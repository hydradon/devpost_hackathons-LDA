Inspiration The power and capability of technology to improve the lives of farmers

What it does

It completely involves the whole process of farming and the use of machine learning to improve the yield of crops.

How we built it

We have implemented python machine learning algorithms and collected some real-world data and performed these algorithms so that it helps to improve the yield of the crops.
