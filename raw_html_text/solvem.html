Inspiration

We built this app as a replacement of latex, the high-quality typesetting that scientists and mathematicians use when typing "complex" text.

What it does

It welcomes the user giving instructions on what the website does then gives a panel for the user to enter their math problem regardless of how neat or sloppy handwriting. Out knn algorithim learns overtime with more data and can recognize more and more characters with the "Input equation text field"

We built by using different IDE's, different OS', and all branching from one repository and adding different features. We built it off macosx and windows 10.
