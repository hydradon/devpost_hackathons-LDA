Inspiration In a world where natural disasters, terrorist attacks, and disease have become fairly common occurrences, a means of discovering the optimal charity to donate to in order to provide effective support would be a helpful resource.

What it does

The Twitter API is used in order to track which current disaster or situation is currently trending on twitter, find key words related to it and find the most appropriate/reputable charity

How I built it

The project was divided into three parts, an API robot meant to collect info and push it to the data base, a web component part, which featured clients side interaction and readability, and a web hosting / Database component.

Challenges I ran into

Learning firebase, all of its in and outs, and reaching certain points where we slowed down considerably.

Accomplishments that I'm proud of

Learning firebase, leaning of its hosting and storage capabilities and finally getting a feel for it

What I learned

How to send and retrieve data with APIs

What's next for CharityLamp

improving functionality and search efficiency
