PROBLEM

Aset berharga berupa data masih terisolasi, pemanfaatan data belum optimal, komunikasi data antar sistem belum maksimal sehingga operasional sistem masih dilakukan secara manual.

Dalam hal pengelolaan data kemiskinan : • belum terintegrasi dengan basis data kependudukan untuk identifikasi warga miskin / Rumah Tangga Sasaran (RTS) berbasis NIK • belum terintegrasi dengan basis data perencanaan untuk informasi program-program pembangunan terkait masalah kemiskinan • belum terintegrasi dengan basis data keuangan daerah untuk informasi realisasi keuangan program-program pembangunan tersebut

SOLUTION

Dikembangkan integrasi dan pertukaran data antar sistem. Project ini bernama : INTEGRASI NUSANTARA

HOW IT WORK

Data pada masing-masing sistem dapat di-push dalam format CSV/XML/JSON/ format lain atau disediakan webservice untuk akses masing-masing basis data tersebut.

Hal tersebut dapat dicapai bila ada :

  1. Standarisasi definisi data untuk objek (dan method) pada setiap item objek pada peraturan perundang-undangan.
  2. Standarisasi komunikasi pertukaran data antar aplikasi atau basis data.
