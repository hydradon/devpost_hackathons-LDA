Goal

Rearrange the tiles so the character doesn't kill a goblin and all the goblins kill themselves.

Controls

You can move blue arrow tiles (Conveyor Tiles) by clicking on them and then clicking any highlighted tile on the level. You can also rotate them by double clicking them. You can also change the direction of the circular spinner tiles by clicking on them.

Known Bugs

Game doesn't check for victory condition.

Inspiration

We were inspired by classic adventure RPG, as well as the theme "You have already won".

How we built it

Unity 3D for the engine. Photoshop + Illustrator for the 2D art, Blender for the 3D models and Spine for the 2D animations.

Challenges we ran into

It took us a long time to come up with gameplay.

Accomplishments that we're proud of

Making a beautiful looking game with several levels.

What we learned

Learn to use the tools before a Jam.
