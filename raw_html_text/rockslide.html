Inspiration

We wanted to control any application in the house, hands-free using gestures. We accomplished this by building a pebble app that detects gestures and passes them on to a chrome extension that is currently able to control the reveal.js slideshow application. Connection between the watch and the chrome extensions is done through socket connections to a managed VM provided by Google Cloud Platform.
