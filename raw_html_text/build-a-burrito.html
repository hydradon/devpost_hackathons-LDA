Inspiration ---> One of our members forgot to buy flowers for his girlfriend this Valentine's day which led to a breakup. We used Android Studio and coded in Java to create it.

What it does -----> Makes you wait less, spend time with the person more. Just a click away!

How we built it -----> We made the code involving credit card information and GUIs in Android Studio using Java.

Challenges we ran into ---------> Using APIs and importing packages. Storing credit card information.

Accomplishments that we're proud of --------> Creating our first ever commercial app!

What we learned ---------> Linking Android studio and the used API together, reading the elements of android studio into the code.

What's next for Instant Bouquet -------> Cryptography and databases for credit card information. Adding pictures and overall cosmetics to our app.
