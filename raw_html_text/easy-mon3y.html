Inspiration

I embarked on this project because I want to help people make smarter financial choices, by having a better sense of where to put their money.

What it does

It takes all the Bloomberg information and parses through it and then gives that information to the Machine-learning Prediction API, to give suggestions.

How I built it

I used Python to go through the data and parse through it. I then formatted the parsed data to be sent to the Prediction API on the google cloud.

Challenges I ran into

Prediction API wasn't what I expected it to be.

Accomplishments that I'm proud of

Getting the prediction API to produce actual results.

What I learned

Practice makes perfect.

What's next for easy-mon3y

Hopefully getting more data, and making it more public.
