Inspiration

Wanted to create something that looked better than a simple 2D game, possibly better than some existing Unity 3D projects out there.

What it does

Control a player who has to avoid the enemies. Enemies spawn at edges of the map at increasing frequencies as time passes.

How I built it

Created terrain, added some graphic details, then created enemy, player, and collision detection as well as scripts for damaging.

Challenges I ran into

Collision detection between enemy and trees as well as player and terrain.

Accomplishments that I'm proud of

Solved pathfinding through NavMesh, looks great with the wind and sun, also first project in Unity

What I learned

3D programming isn't as hard as I thought it would be, it's exciting to be able to do so much just over one weekend.

What's next for Lost In The Woods

Allow player to kill enemies, add a start screen.
