Inspiration

Trump was recently elected President + wanted to explore twitter fabric API and Android Text to Speech + wanted to be satire and enjoy some nice haiku.

What it does

Picks a random Trump Tweet from a textfile, and reads into a haiku form of 5 syllables-7 syllables-5 syllables, reading it out via Android Text to Speech. You can also use Twitter API to view original Tweet before it was made into a haiku and it can take you to Twitter App or Chrome.

How I built it

Used Android Studio, Twitter Fabric API, and Android Text to Speech.

Challenges I ran into

Counting syllables, as there is currently no known "perfect" algorithm to count syllables for words in the English Language. Had to make up makeshift cases and do lots of testing.

Accomplishments that I'm proud of

Getting some good syllable counts, and hooking up the Twitter API.

What I learned

How to use the Twitter API and that the English Language is pure evil.
