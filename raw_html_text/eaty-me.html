Inspiration

We are aware that DC is a very significant market for food trucks during lunch hours of the work week. We wanted to create a seamless cross-platform application for both vendors to quickly advertise specials while providing real-time location updates to consumers.

What it does

It provides live tracking of food trucks around the user, online ordering, and online payment to save time when going to buy food from a food truck.

How we built it

We made an ios app using swift, a public API to access our data and a website using react.

Challenges we ran into

Integrating map SDKs, query data from firebase based on geolocation.

Accomplishments that we're proud of

We are proud of our IOS app, our website, and we were only two working on this project.

What we learned

Working with each other, and managing to sync different languages of programming.

What's next for Eaty Me

Hopefully, we will add a rating system, pickup estimation for both the truck driver and the user.
