What it does

It takes a photo taken by your mobile device and grabs their social profile via face recognition.

How we built it

It was built from ground up, we implemented the UI in XCode and relayed backend information between Swift and Node.js using the firebase API. We utilized the facial recognition API provided by Microsoft to identify people.

Challenges we ran into

We didn't have much experience with firebase, so getting it set up took the bulk of the time.

Accomplishments that we're proud of

Project works, we believe this is a great idea, and that it should be continued on even after Hack The North.

What we learned

Firebase, Node.js, teamwork and collaboration
