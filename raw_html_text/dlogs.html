Inspiration

Web infrastructures nowadays suffers from censorships and DDoS attacks.

What it does

a Blog built using IPFS as storage and use smart contract as registry between user (address) and their IPNS

How we built it

We use Electron + React + Reflux + Nodejs + Go-IPFS-js-api + Web3.js + CSS-grid

local geth privnet

Challenges we ran into

IPNS latency is ~ 1 to 2.5 minutes, which degraded ux.

Accomplishments that we're proud of

full stack! from backend all the way to the front! no Web service provider used!
