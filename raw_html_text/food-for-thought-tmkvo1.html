Inspiration

We wanted to make a product that would be useful in real life. This helps raise environmental awareness about the different carbon emissions of various foods that people might not think are bad for the environment. For example, beef has around 5 times the carbon footprint of chicken

What it does

This app creates a recipe sustainability score for the recipe and ingredients and suggests alternatives.

How I built it

We used online data bases and assigned values to all ingredients after web scraping websites.

Challenges I ran into

We couldn't get the code to run on one members laptop specifically.

Accomplishments that I'm proud of

One of our members is learning everything on the spot as it is their first month of coding.

What I learned

We have learned much about python and flask. In addition, website building with HTML and CSS was learned.

What's next for Food-for-Thought

We hope to be able to use neural networks and machine learning to optimize for ingredients that it doesn't have an exact match for in our database. Also, we hope to be able to expand input methods to images and text.
