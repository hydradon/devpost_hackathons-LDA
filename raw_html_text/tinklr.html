Inspiration

There are a lot of trans and non-binary people who struggle to find a gender-neutral bathroom. Since their gender expression does not necessarily match their gender identity, it is hard for trans and NB people to be able to feel safe and secure with the simple act of going to the bathroom.

What it does

Using the Google Maps API and the help of user-sourced content, people can look at the site and see where there are gender-neutral bathrooms available.

How we built it

We created a website using HTML and CSS and PHP and incorporated Google Maps API.

Challenges we ran into

We don't know how to do a lot of things yet. We struggled with JavaScript and PHP. We stumbled through a lot of coding errors.

Accomplishments that we're proud of

It works! (for the most part)

What we learned

We have a lot of ideas that we can put forth and we are excited to continue to build and pursue other coding projects that are fun but also informative and helpful.
