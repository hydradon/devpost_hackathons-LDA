Inspiration

As individuals that suffered from asthma in the past, such an app would have been useful in taking preventive measures.

What it does

Our program predicts future pollutant levels so that a user may adequately prepare for said day by taking their inhaler or avoiding outings on said day.

How I built it

It was built with Azures Machine Learning module using boosted decision trees.

Challenges I ran into

Working around Azure was difficult because it was our first time using it and it wasn't very intuitive.

Accomplishments that I'm proud of

A robust model with good prediction accuracy.

What I learned

How to implement machine learning models on Azure.

What's next for Air_Quality_Control

An add on mobile application to make the program more convenient.
