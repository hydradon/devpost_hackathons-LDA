HACKING CHALLENGE: Hacking Homelessness

The government issues smartphones to the homeless people and we wanted to figure out how to communicate with the homeless in case of emergencies or if the homeless people need services.

The homeless can access this web site as a means of information on nearby shelters, events with free food, and provides them with their location on where they are located.

We used a combination of Programming and Structuring Languages using Python on Flask as our backend development and HTML, CSS, and JS as our front end design and structure. We then deployed our project up to Heroku.

Looking for a right and simple programming language that we could rely on for easier and faster web development.

Geolocation API works and renders successfully on the web page.

Python on Flask requires a few more steps to be able to deploy to the page.

Our future with this thing is to make it better, design it more beautiful and add more features that can better benefit those who need.
