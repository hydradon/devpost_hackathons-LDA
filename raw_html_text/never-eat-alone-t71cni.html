Inspiration

All of us have wanted to have lunch with someone else, so that we could meet new people outside of our friend circle.

What it does

The app brings people together to have meals or events together.

How we built it

We have both an iOS and web app. We used codePen and XCode to build the two interfaces. We implemented google firebase for backend data.

Challenges we ran into

Setting up a database, cross platform integration

Accomplishments that we're proud of

We managed to work together to finish the project, all while having to learn all the code we used as it was mostly all new to us.

What we learned

Swift, Firebase, CodePen, Javascript

What's next for Never Eat Alone

We want to publish it on the Apple store to implement throughout the campus.
