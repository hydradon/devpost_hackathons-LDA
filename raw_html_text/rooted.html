Inspiration

Mid terms are coming up and we wanted to create a fun product to help relieve stress.

What it does

The user can purchase items, drag and drop items onto the screen, and wait for visitors to come to their garden!

How we built it

HTML, CSS, Javascript

Challenges we ran into

We initially tried using Phaser, a game framework, but that didn't work out.

Accomplishments that we're proud of

We actually made something!

What we learned

Don't use a new framework that doesn't have well-written documentation.

What's next for Leaf Me Alone

More items! More fun!
