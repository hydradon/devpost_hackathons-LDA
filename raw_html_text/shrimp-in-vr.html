Inspiration

We wanted to see shrimp up close and personal so that we can closely examine them for disease and therefore save the world from world hunger.

What it does

You can put on the VR headset and see shrimp in VR.

How I built it

We used C# and Unity game engine as well as Blender to make the shrimp model.

Challenges I ran into

Learning how to do everything in VR.

Accomplishments that I'm proud of

We actually managed to get a shrimp in VR within the short amount of time we had to finish the project.

What I learned

We learned how to make stuff in VR. Also Hasan learned how to use the SCV file in python.

What's next for SHRIMP IN VR!

We are hopefully going to get even more shrimp in VR and also add some animations like moving legs on the shrimp for even more realistic effect. Together, we can save the world with Shrimp in VR.
