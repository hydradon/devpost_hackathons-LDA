Inspiration

Non-tech savvy people always stuck with problem when they interact with modern technology.

What it does

hardware Interface that helps non-tech savvy people to do some tasks which can be easily done with modern tech.

How I built it

we use raspberry pi and LEDs,buttons etc to make grocery list

Challenges I ran into

working with hardware with lots of connections to maintain and with lots of points of failure.

Accomplishments that I'm proud of

To be able to create something which can help people. Also I did learn lots of thing about hardwares, ICs, microcontrollers

What I learned

How to ideate, plan, design and execute an idea. Also meeting and working with new people is a learning in itself

What's next for SHVV

we will use this project's hardware in multiple use cases.
