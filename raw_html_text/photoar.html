Inspiration

For urban planning, the precise 3d model of the environment is usually required, however, this is usually very time-consuming and costly, we create machine learning solution, that can match a single photo to air captured lidar data to provide visualization for possible urban simulation

What it does

We use a machine to extract an approximate 3d model from a single photo

Which that can be aligned to lidar point cloud based on geodata provided.

Demo

Precise 3d reconstruction that can be made, to introduce auxiliary shapes to the scene f.e new building.
