Inspiration

Pronosticar las ventas de Cervecería Teotihuacan con el objetivo de conseguir inversionistas para arrancar el negocio.

What it does

Generar gráficas de las ventas de próximos tres meses, entrenando el programa con diez millones de datos.

How we built it

Python para el algoritmo, HTML y CSS para el display web:

Challenges we ran into

Priorización de gran cantidad de datos, muchas variables que tomar a consideración.

Accomplishments that we're proud of

Interfaz gráfica para la muestra de datos que calculamos.

What we learned

Aprendimos que la segmentación de los datos es muy importante para conseguir datos precisos.

What's next for Cervecería Teotihuacan

Cálculos más preciosos para empezar ventas.
