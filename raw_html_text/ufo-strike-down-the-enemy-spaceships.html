Inspiration

Whenever me and my best buddies go out and spots some unusual happenings around we used to say we used to say[just for fun] UFO[Unidentified flying objects].
And that's how I got the idea for this game

What it does

Its a HTML5 based fun game. User has to launch the missile[different missile with different speed] and strike down the spaceship.

Fun and difficulty:

  • Different missile has different speed. So user has to analyze the speed for all the missiles and launch it accordingly to make it strike to spaceship. Spaceships speed increases after certain point interval.
  • User will get bonus point if they can strike the spaceships with blinking UI. These blinkinh spaceships arrives in between

How I built it

I used HTML5, JavaScript, canvas to build this game. Tried to make it work on all the devices like browser, tab and mobile.

Challenges I ran into

  • Creating levels for the game, because I started working on the game just before a week. So got only 1 week on time to develop this game.
  • Maintaining the spaceship and missile speeds.
  • Handling UI for all the devices. While making it work for mobile, browser and tab

Accomplishments that I'm proud of

  • I could make it functional within this 1 week of timeline.
  • I could add different features such as bonus points, speed variations and levels.
  • Background handling for different devices
