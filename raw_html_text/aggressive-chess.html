Inspiration

We needed a project so we came up with real time non turn based cross platform chess app.

What it does

It is chess where you can move any piece at any time. Once a piece is moved it has a cool down time where it cannot be moved again until the cool down time has passed.

How we built it

We used an open source chess engine and modified it with new rules. We also worked on adding a gui with Kivy but had an absolutely awful experience with it.

Challenges we ran into

KIVY IS THE WORST! It wouldn't let us do anything. We used almost the entirety of the time trying to get that working.
