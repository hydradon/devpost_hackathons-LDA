Inspiration

Inspired by the Women in STEM issue raised by JP Morgan - we discussed ways to showcase women in STEM careers. We thought it would be nice to provide a platform which shows the women working in STEM by their area as a place to see the differences across the world and find role models. Also we thought this could include historical role models.

What it does

WowMap is a web app that uses google maps to display data on women in technology and STEM careers and their locations across the world. Using the app people who identify as women can add their details to the map to drop a pin in their location.

How we built it

We built a reactive map using the google api and meteor. This pulls information from a database (mongoDB) that contains geographic data on women in tech which places pins on the map to display them, this can then be added to through the form on the page. Each addition results in a new pin on the map.

Make it more interactive - i.e. get bios to display for each of the women (as an optional feature), have a historic search function, search by area (not just dragging around the map).
