Inspiration

We wanted to reliably measure happiness and provide scientists with the tools to conduct more measurable results in experiments. It can help improve workplace happiness, mental health, and intrapersonal awareness.

What it does

Reads the user's face to provide a real-time graph of their emotions

How we built it

Scans face with the webcam, analyzes the sentiment, sends it to the database, and then plotly (Dash) reads the data to create a visualization.

Challenges we ran into

Connecting the database with the graphing technology.

Accomplishments that we're proud of

Learning how to use so many different technologies in one night!

What we learned

Different tools for data analysis in Python

What's next for Sentimental Analysis

Fully develop our database with Firebase
