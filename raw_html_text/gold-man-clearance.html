Inspiration

Thinking about social problems, I thought of the elderly, who were the weak. Our team have a friend who serves in a nursing home and another friend whose grandparents are in a nursing home. So, we looked at the old people in social institutions. As a result, many nursing homes across the country have experienced numerous cases of improper claims and learned that the damage is significant. We wanted to solve some problem in this issue through a blockchain.

What it does

We are using the Blockchain techniques to secure Database that was stored in nursing elderly patients (Please refer to the attached presentation for more information.)

How we built it

We use ethereum API to make a smart-contract system. And also use android studio to make our own APP. (Very hard!!)

Challenges we ran into

When we started this hackathon, It feels really hard to use API that connected with blockchain, so we consider and study how to use them easily and effectly.

Accomplishments that we're proud of

First of all, the most proud thing I feel is I start study about blockchain more deeply by preparing this hackathon, and also It is first project that I made some Demo which is made through blockchain.

What we learned

We learned that the collaboration & co-working is really really important thing to make a successful Demo. And blockchain technology is more wide-use tech that we think.

What's next for Gold Man (Clearance)
