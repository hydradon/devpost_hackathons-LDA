Inspiration

New Avenues for Youth shared with us that there is not a single, straightforward location for youth and their support network to find available resources relevant to them.

We decided to create simple mobile friendly web application that can be easily filtered to find resources which are relevant to them based on location, eligibility and any determining factors.

Key features

Resources can be added and updated by anyone with internet access. Resources can be flagged with specific eligibility criteria. Resources can be filtered based on category and eligibility criteria. Users can add comments or add a quick thumbs up or thumbs down on a resource. No sensitive user information is collected or made publicly available.

How we built it

Firebase database for the database and web service. React for dynamic DOM generation. HTML and bootstrap.

Challenges we ran into

We made two attempts to fit a front-end framework to Firebase to connect out front end to the back end. We ran into difficulties with both frameworks and ran out of time.

Accomplishments that we're proud of

When we initially heard the pitch for New Avenues for Youth, we were overwhelmed and intimidated with the amount of information. We are proud that designed up with a viable, elegant product that would solve one of their problems.

What's we'd do if we had more time

Connect the front-end and the back-end. Add links to email a resource or list of resources. Add a more robust Admin console where user comments, categories and eligibility tags can be managed.

What we learned
