Inspiration

We wanted to create a smaller and more efficient way of detecting our surroundings for people who are visually impaired. We realized that they don’t have access to cheap, efficient, and accurate distancing technology. So we came up with a chest piece that allows you to hear objects like using echolocation in a way that is viable for humans.

What it does

It uses an ultrasonic array that measure distance.

How I built it

I used an HCSR04 Ultrasonic array connected to a Raspberry Pi 3 with the use of python code.

Challenges I ran into

Lack of needed equipment to be successful.

Accomplishments that I'm proud of

I did the hardware design and the coding for the project.

What I learned

Bring my own hardware.

What's next for Echo-locator HCSR04

Bluetooth connection to headphones.
