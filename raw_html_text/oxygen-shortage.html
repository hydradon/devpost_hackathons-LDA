Inspiration

This idea simply occurred to me during brainstorming, and it seemed the best idea so I went with it.

What it does

You can go around the map to find items and use those to set traps, and lie to others to cause them to run into those traps, or hope they run into them by themselves.

How I built it

I determined what I wanted to create first, and followed that plan, cutting when necessary or adding if I had time

Challenges I ran into

I occasionally would run into bugs that I couldn't seem to figure out, such as implementing a fog of war or spawning an item in the scene.

Accomplishments that I'm proud of

Solving all my bugs and implementing my core features.

What I learned

I learned how to create a game, modify it, as well as how to code it up.
