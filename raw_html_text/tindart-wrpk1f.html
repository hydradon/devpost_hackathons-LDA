Inspiration

Inspired by our love for art and our success on Tinder.

What it does

TindArt connects artists and art-lovers all around the world, and creates a hassle free art-buying process where people can simply swipe right for pieces of art they like.

How we built it

We built our website with HTML/CSS and JavaScript on the front end, and Python and Django for the back end. We also used SQlite as the database to store photos.

Challenges we ran into

Learning how to connect the front end with the back end, and also learning how to use Github.

Accomplishments that we're proud of

Learned how to use Github, learned how to use HTML/CSS, and building a database, as well as a secure login.

What we learned

We learned how to use the bootstrap framework, as well as how to use Github better.

What's next for tindart

Next we plan to develop the app for TindArt. We also hope to incorporate Machine Learning, so that the more you swipe, the better your tastes and preferences are understood, so you will get better art recommendations.
