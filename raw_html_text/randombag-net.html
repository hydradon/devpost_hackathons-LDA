What it does

randombag.net connects remotely to a real quantum computer in order to generate a truly random number, which is used to populate Amazon order with random $#!+ that is sent to your house.

How I built it

Using IBM qskit, generating random numbers is easy. Then, using zinc.io, we can build a list of products to populate our Amazon cart and place the order.

Challenges I ran into

Lack of experience in web development, particularly in front-end development.

Accomplishments that I'm proud of

Successfully generating truly random numbers with a quantum computer, along with a fun and creative project idea.

What I learned

How to deploy a web app using flask, how to use IBM qiskit.

What's next for randombag.net

We need to connect the site to a database to support user logins.
