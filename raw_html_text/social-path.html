What is this application all about ?

Socialpath is web-application running on either IOS, Android or any popular web browser available. The only thing you need is curiosity, a goal and the will to make a new community come to life.

How we built it ?

SocialPath is using the MEAN stack: angular.js, node.js, express.js & mongodb. It's the result of many hours of work, research and a couple sleepless night.

Challenges we ran into

Adaptation of the google-api in a smooth user interface, creating a web plateform which give the user almost infinite possibilities and opportunities in her or his travel choices.

What's next for Social Path

The only limit that SocialPath faces is the one that the user will set for himself.

SocialPath, c'est quoi?

SocialPath est une application web compatible avec les technologies mobiles d'iOS et Android et utilisable sur les navigateurs web les plus populaires. Une nouvelle communauté est à votre portée.

Comment l'application est bâtie?

SocialPath utilise la stack MEAN: angular.js, node.js, express.js & mongodb. C'est le fruit de plusieurs heures de travail, de recherche et de quelques nuits blanches.

Difficultés rencontrées

L'adaptation de l'API de googleMap dans une interface utilisateur renouvelée et facile d'utilisation créant une plateforme web qui donne à son utilisateur un nombre d'opportunités et de possibilités sans limite pour ces choix d’itinéraires.

What's next for Social Path

La seule limite que possède SocialPath, c'est celle que s'impose l'utilisateur.
