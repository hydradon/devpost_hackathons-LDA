Inspiration

We released that educate kids at a young age about wildlife can return with big positive eco-friendly behaviour to the wildlife.

What Zoo Zoo Does?

Increase wildlife awareness and make it understandable for layman using a mobile app with a knowledge base and gamification!

How we built it

We used React Native to create a cross-platform app that can be run on both android and ios using 1 base code!

We used adobe-xd to create design sprint
