Inspiration

Semiconductor Manufacturing Optics play a key role in structuring microchips, photomasks sometimes have production defects. These defects must be detected and visualized for the user.

What it does

Photomask Defect Detective provides a user interface to upload a tiff file which then gets transported to the python Deep Learning module where the image gets classified
