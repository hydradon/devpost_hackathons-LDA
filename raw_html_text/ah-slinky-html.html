What it does

Whenever a HTML page is rendered on Chrome, all the HTML elements will fall to the bottom and result in a mess of a HTML page.

How I built it

Built it on Chrome's extension API with vanilla javascript and HTML5.

Challenges I ran into

Integration and script-injection issues.

Accomplishments that I'm proud of

People are amused by it the first time they see it.

What I learned

Vanilla javascript scripting and HTML5 integration

What's next for Slinky HTML

Official launch on the Chrome Extension Store
