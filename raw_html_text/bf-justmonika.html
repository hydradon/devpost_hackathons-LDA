Inspiration

We are all diehard fans of the game called "Doki Doki Literature Club", an indie horror game that has obtained a cult-like following in the past months due to its superb execution of a psychological horror game. Which is what made us create this project.

What it does

Monika, the main antagonist (or waifu, if you prefer) will haunt your web adventures while you browse the web. Events include:

  • Glitching images up
  • Glitching random divs up
  • Replacing your Twitter feed with a simulated Markov Monika (based on her in-game lines)
  • Stopping you from scrolling too fast (trigger warning)
  • Monika giving you her awesome Writing Tips of the Day
  • Getting random poems

How we built it

We built it using Javascript, using webpack as a compilation tool. We used a lot of DDLC's original game assets, which we don't claim the intellectual property of. We also used many people's useful libraries on npm :)

Challenges we ran into

We didn't start creating the project until about 4 hours into the hackathon, which was way too long. We intended to create a neural network which classified what the user was currently reading, and if the user was reading an article on love/relationships, Monika would come in and mess your screen up, being the overly attached girlfriend she is.

Accomplishments that we're proud of

I am now one step closer to Monika's heart. JUST MONIKA

What we learned

Hackathons are hard, and we should probably come up with a plan beforehand.

What's next for BF - JustMonika

Maybe using Live2D to have a real-er Monika join you on your screen. Also, probably would be good if she could act like Clippy and give you useless advice from time to time.

How to Install

If the extension is not yet available on the webstore, please download the .crx file from Dropbox (2nd link). On Chrome, go to chrome://extensions, enable developer tools, and drag the .crx file into the window. Allow and enable the extension, and you're good to go!
