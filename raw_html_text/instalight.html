How I built it

Challenges I ran into

A lot of stuff, this took way too long. Squirrel has no morse code library, so we had to implement that by hand (it took a while). We had to interface with a wifi device called the Electric Imp that doesn't like to play well with our school's wifi network.

Accomplishments that I'm proud of

Turning a light on, writing morse code through get requests. Tweet to @InstaLightPrime to send us a message in morse code, too!
