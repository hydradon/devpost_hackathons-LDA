Inspiration

Parent asking for a more involved timer for multiple tasks

What it does

Allows for multiple timers to occur, generated from user created presets presets are saved on exit and loaded on program activation presets and active timers can be disabled by the button

How I built it

Use of Java and JavaFX

Challenges I ran into

using the multithreading nature of java

Future

-Multi step timers that indicate which step you should be on -Voice recogintion to allow for hands free use -Adaptive timers and suggestions to reflect the users history
