Inspiration

We wanted to create something that helps people make conscious decisions and improve sustainability efforts in Santa Barbara County.

What it does

The web app informs users whether commonly misplaced items are recyclable or should be thrown in the landfill.

How we built it

We used CodePen with HTML and CSS to configure all of the images and text, with the hover feature that distinguishes each type of item.

Challenges we ran into

It took a long time to align and resize images to make them uniform on the page, and to determine the commands to give images and text hover capabilities.

Accomplishments that we're proud of

We came into this with zero coding experience and were able to create something that can be of use to people in our community and aid in sustainability efforts.

What we learned

SO MUCH. Literally everything.

What's next for Santa Barbara Recycling Manual

Ideally we will be able to make this into a mobile-friendly webpage and eventually an interactive mobile app!
