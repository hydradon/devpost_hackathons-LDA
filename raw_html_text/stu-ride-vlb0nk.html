Inspiration

Difficulties finding a ride from college to college, or city to college. While long distance transportation are tough

What it does

It creates a database that helps connect drivers to riders, where riders can help pay for driver's gas while driver help the rider go to the same destination

How I built it

Using ExpressJS, Node, and mongo db to create the environment

Challenges I ran into

Not knowing the certain languages, environment setups

Accomplishments that I'm proud of

Being able to learn and build with new languages

What I learned

Express, MongoDB

What's next for Stu-Ride

Create a mobile version using react-native
