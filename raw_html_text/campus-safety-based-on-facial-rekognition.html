Inspiration

Lately, there have been few suspicious activities that have been reported in or around the campus and that's why we wanted to create a project that will help us to make the campus safer, catching the suspect quickly using live video stream and facial recognition.

What it does

We intend to capture live video streams from the surveillance cameras, recognize the suspects using the AWS platform and its APIs to report the last seen location of the suspect. This will provide aid to the authorities in concern to track the activities of the suspect as well as can be used to warn the citizens to be more alert.

How we built it

We have used AWS kinesis live video stream, signal handlers to take input from cameras, lamda, and rekognition for image identification. Using S3 for storage of data.

Challenges we ran into

Integration of all the components involved in the project.

Accomplishments that we're proud of

We have learned AWS features and the power of its libraries.

What's next for Campus safety based on facial recognition

Making it as a fully integrated product and scaling it into the pool of a large number of universities. Also, adding some new features that will show the geographical heatmap, crime rate, etc.
