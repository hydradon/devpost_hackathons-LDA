Inspiration

We were inspired to build it because we wanted to build something ARKit. It then struck us that AR has a lot of potential in e-commerce, so we thought that ARKit could be used to streamline the shopping process further.

What it does

The plan was to get "an amazon dash for everything," but we ended up with an application that can link barcodes to products.

How we built it

Swift, Xcode, APIs

Challenges we ran into

Integrating the Barcode API and ARKit.
