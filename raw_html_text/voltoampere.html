What it does

Future efficient home: it controls Individual devices's power consumption and predicts weather the building needs extra solar panels or not.

How we built it

we built it through a mobile app ade by Microsoft's PowerApp, we created live data feeding nodes using IoT.

Challenges we ran into

Software:

  • figuring out the power app environment.
  • integrating the system with Azure database.
  • integrate the software with the hardware.
  • using cloud data warehouses as a database
  • uploading data into Azure database using MacOX

Hardware:

  1. Dealing with Lora Radio --the connection between the transmitter and the receiver
  2. some needed parts did not exist.
  3. integrating the hardware information with the cloud.
  4. sending data into two wireless hardware nodes.

Accomplishments that we're proud of

Working on a cross-platform successfully (Power App)

What we learned

Asking for help whenever needed! Asking stakeholders what they really need this hackathon is just the beginning
