Inspiration

We were inspired by seeing the older generation struggle with their taxes and so we wanted to help fix that problem in order to help the future generations as well.

What it does

It currently uses the tax rates for different kinds of taxes in different states and calculates the net pay.

How we built it

Originally we decided to split it up into groups so different people could do different parts of the project, but we later realized that we all had very little experience in android studio, so we made it a group effort.

Challenges we ran into

We originally were planning on creating another pathway from the main page in which you could track how much you spent on each item you wanted to track. We ran out of time, especially since we took a lot more time using android studio than we originally thought it oul take.

Accomplishments that we're proud of

We are proud that we finally created an android studio app considering we had no prior experience with it and we got an app to work successfully after almost giving up on this idea.

What we learned

We learned how important planning is and we also learned how we could connect java to android studio and using that.

What's next for Finance Assistant

Next finance assistant will have a tracker that can track the amout you spend on the thing that you find necessary to track to monitor whether or not you exceeded your target for amount spent.
