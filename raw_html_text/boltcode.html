Inspiration

In software development many tedious or small programming tasks often delay development. What if there was a way to outsource the insignificant tasks to talented developers to do quickly, rather than waste your real employees' time.

What it does

our platform hosts companies and app developers can create small programming task posting for developers to take and complete for instant payout. Developers with a lot of free time or students can use the platform to make a small amount of money over a weekend.

How we built it

We used the Angular7 framework with design inspiration from the interwebs.

Challenges we ran into

We spent a lot of time on the backend of our website and linking to databases only to realize that it was an unnecessary hindrance to our team.

Accomplishments

we're proud of we have a very clean aesthetic website with a very user-friendly user interface.

What we learned

We learned enough about SQL to add it to our resume

What's next for BoltCode

The proper backend needs to be set up and a payment system needs to be added before it becomes a viable product, but we plan to continue working on it into the future as we believe it is a viable business platform.
