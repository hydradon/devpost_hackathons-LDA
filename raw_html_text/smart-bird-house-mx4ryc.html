Inspiration

20% of United States population are birders i.e. they like to observe birds in detail. The expenditure on equipment related to that is $26 billion.

What it does

It allows a bird to enter the house with food and water. The bird is detected using a sensor, and notification is sent via message to owner.

The camera takes pictures and videos of birds and emails them. Temperature and Humidity sensor record data to correlate environment with the birds.

A database stores all the data.

How we built it

Database built using mySQL. Raspberry pi used for processing. Sensors to collect data. Craft sticks to build house. Nexmo API to send messages.

Challenges we ran into

Integrating the system in a day was a challenge.
