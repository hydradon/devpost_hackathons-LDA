Inspiration

We were inspired by out grade 12 biology teacher, as she was always extremely engaging when it came to lessons about the human body.

What it does

Player must place a heart into the human body using their hands.

How we built it

Using unity and blender.

Challenges we ran into

Configuring the leap motion to work with unity.

Accomplishments that we're proud of

Managing to create interesting figures in blender.

What we learned

We learned how to use blender and unity.

What's next for Anatomy Professor

We want to create better graphics, and to add more features for the user to interact with (i.e. more organs to place).
