Inspiration

-We used the following website's code as a template/foundation for our own program. We practiced changing the text, inserting buttons, and adding more components to our design with this website. http://learninginhand.com/html-for-random-message/

What it does

The site displays a new motivational quote every time it refreshes. The button refreshes the page when clicked.

How we built it

We used a template and edited various commands, mostly the style and text. We experimented with adding code for the button and edited the various existing commands in order to see how each particular one affected the site's function. We used the variable text command to generate new quotes each time the button is clicked. The button itself simply refreshes the page and the code displays a new quote.

Challenges we ran into

Sometimes, when we would input new parts to the command, it would mess up the format of the entire website. Only some font families would work.Also, remembering the small details of the language, such as finishing the commands, with would sometimes catch us off guard.

Accomplishments that we're proud of

We walked into the event without any knowledge of coding whatsoever, so we're proud of the fact that we learned various commands and discovered a lot about coding. A lot of the edits we made to the template code was based on the concepts we learned earlier in the event on codecademy.com.

What we learned

We learned the basics of HTML, including text formatting, headings, paragraphs, ordered and unordered lists, inserting images, etc., though not every concept we learned was incorporated into the project.
