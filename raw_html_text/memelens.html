Inspiration

The whole project was inspired when one day i realized that I wanted to be able to tell my friends how I was feeling about a meme in real time.

What it does

It is an iOS application that identifies if someone is smiling and at what meme and sends a text message to a group of recipients with a link to the meme.

How I built it

Using ARKit to identify a users face and then identify if they are smiling based off of it. Then I used the Twilio API to send a text message to a user with the link to the meme after the smile was detected.

Built With

  • arkit
  • swift
  • twilio
  • xcode
