Inspiration

We love cinema and we like to play, naturally movie quizz is born.

What it does

It's a game. We listen to a movie clip and we have to guess the movie. You can play from one to four.

How we built it

We built it with alexa skill kit

Challenges we ran into

We are French so it was difficult to find film clips in English, so alexa presentation language is hard to integrate.

Accomplishments that we're proud of

In France the game works very well, and the user comments are exellent. It is important for us to offer our users a fun game Link skill in a french store

What we learned

We learned alexa presentation language

What's next for Movie Quizz

For the future we want to add more and more movies, and always more fun
