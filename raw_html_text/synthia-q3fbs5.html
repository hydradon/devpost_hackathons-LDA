• Synthia User Interface

Inspiration - emotionally driven music

What it does - Composes music in real time to match current tension or excitement

How we built it - we built the main program on java and made custom .wav files to exhibit tension through music

Challenges we ran into- long load times and working with large amounts of sounds, creating a tension algorithm

Accomplishments that we're proud of- completed the program, customization, it can be easily scaled and manipulated, this is the first hackathon for everyone on the team

What we learned- how to create our own personal autonomous DAW, we made a standalone java application, we created a customization program.

What's next for Synthia- allowing Synthia to make more musical and user given themes, be able to change notes and tension in real time, potential to use for real-time and dynamic music composition and to add to game development.
