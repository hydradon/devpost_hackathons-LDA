Inspiration

Dance party was an internal hackathon project a while back. It was a big hit. At atlassian connect week we were shown some of the capabilities of hipchat, and I figured we could clean it up and release it as a fun product.

What it does

Uses the spotify APIs to pick a song that matches the beats per minute of an emoticon

How we built it

Simple little tool. Uses Atlassian Connect Express as the base, and redis to store any emoticon overrides. Trickiest parts were trying to get react to play nice with the hipchat javascript apis.

Hosting

I also used it as an excuse to get more familiar with kubernetes and building docker images with CI.
