Inspiration

We were sick of using passwords so we decided to create FaceID for websites that provides a loginless solution.

What it does

It takes a picture of you and saves it in a database and then the next time you try logging on you just take a picture of yourself and then the Google Visual Recognition API takes care of the rest.

How I built it

I built it using Electron and Google Developers'

Challenges I ran into

How to integrate the API because of our lack of expertise.

Accomplishments that I'm proud of

Making something that works and something that we want.

What I learned

I learned how to build applications and how to use the Google Developers' website efficiently.

What's next for FaceID in websites

We want to improve its' accuracy and continue on improving the application.
