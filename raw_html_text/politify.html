Inspiration

Imagine a world where we could reduce the desire to haze over political newsletters online just because they're so abundant; to make it easier for citizens to access real-time information about their representatives is to increase the foothold of democracies in an online, noisy era.

What it does

Politify is a chrome extension with a dedicated RESTful API server that checks the browser's current DOM for names of congressmen, and highlights the name for you within the browser. Hovering over these names allows you to view a political summary of these individuals.

How we built it

Chrome extension, Flask Python RESTful API

Challenges we ran into

Natural Language Processing, maintaining information integrity
