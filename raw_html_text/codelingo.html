Inspiration

Codelingo began when one of our team members was interested in learning C++ on her own. Juggling classes, work, and extracurriculars can make picking up a new programming language difficult. Inspired by popular applications to learn spoken languages, Codelingo aims to do the same for programming languages.

What it does

Codelingo makes learning a programming language easy and fun! Our modules are perfect for beginning programmers or seasoned professionals. Practice anywhere and anytime.

Courses now available in Python, Java, and Data Structures and Algorithms.

How we built it

Codelingo is a JavaScript app supported by React and Bootstrap in the frontend and MongoDB in the backend. Codelingo.net, purchased from Domain.com, will be used to host our website.

Challenges we ran into

Hacktech was the longest hackathon our team has attended. We ran into difficultties with the backend database server due to issues parsing data. Additionally, we struggled with time management and efficiency which negatively impacted our desired deliverables.

Accomplishments that we're proud of

Codelingo is incredibly proud of our teamwork and ability to overcome technical and profressional challenges.

What we learned

Our team learned a lot about React and Bootstrap. We improved our user interface design skills. We also learned about the importance of effective project management and conflict resolution!

What's next for Codelingo

We look forward to further developing our existing portfolio of course offerings and expand to other languages as well. We aim to allow users to connect and compete with each other to improve their programming skills.
