Inspiration

Online harassment against the LGBT community is a perennial challenge on social media. The low barrier of entry for posting malicious content, compounded with the relative anonymity of user registration, result in a lack of deterrent for harassers.

What it does

This project aims to create a social media platform for the LGBT community. Its registration, posting, and user management (banning, privilege granting) processes are specially designed to deter potential harassers.

How I built it

The frontend is built using the Angular framework; theming is accomplished through Bootstrap templates. The backend components (as-yet unimplemented,) will be realized with the Express.js.

Challenges I ran into

The initial learning curve to Angular, and the upfront cost to using a complex framework such as Angular.

Accomplishments that I'm proud of

A rainbow heart as the upvote button! Beautiful theme for the website!

What I learned

Web programming skills, Angular, and team coordination.

What's next for QueerSpace

Finishing the transition for several static html pages to Angular apps.
