Inspiration

Given that we are international students abroad, a lot of friends from high school have travelled all across Europe and the world to seek higher education, and hence we very rarely get to meet due to the far away distances. A middle spot that is in equal reach of everyone would have led to memorable reunions and amazing memories being made in a new place we all have never been to before.

What it does

The app takes in a number of friends who join a group and calculates the middle point between all of them and gives them the directions to get there.

How we built it

We utilized Xcode and swift for the front end development, and developed an app that utilizes MKMapView that gets data from an SQL database and displays it on the map with directions. On the backend side, we used SQL database to hold all the users, their locations and their groups. We used pHp to connect the backend and the frontend.

Challenges we ran into

We ran into numerous challenges that include connecting the front end and back end, because swift does not have an API for mySQL database

Accomplishments that we're proud of

Amazing UI design, efficient algorithm to find the fairest place between everyone.

What we learned

Write consistent and clear code from the get go, because returning to it later would impede any progress when a bug is not clearly found.

What's next for Find A Spot

Adding more features including filtering my time and cost of travel.
