Inspiration

We were talking with Sabine and somehow ended up with this idea

What it does

Stream money to charity of your choice while browsing addictive sites like FB and Reddit so you don't feel as bad.

  1. In Coil Account Settings select charity that you like
  2. Go to the site that you feel bad browsing.
  3. Click Coil logo -> Enable Charity Mode
  4. ...
  5. Profit! (and be happy)

How we built it

Puma Browser is an iOS app, we're using Coil and Interledger Protocol to stream payments.

Challenges we ran into
