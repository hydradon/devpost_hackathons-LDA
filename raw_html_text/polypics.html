We used the Power of the amazon echo to convert our voice into an image and color. Using AWS as the echo Client and the intermediary do to the security Amazon used to protect the echo information. The main code was in Mathematica it read the data that was gathered by the echo. Then search the web for a picture which it then displayed then proceed to get the mean values for the RGB scale. Which were sent over Serial to an Arduino Nano which controlled RGB LED that displayed the colors.
