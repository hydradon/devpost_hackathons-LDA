Inspiration

Budgeting can be difficult especially for University students who have to balance school and fiance. This Webapp helps student to organize their expenses with statistics of how well they mange their experience.

What it does

Take in different type of expenses, add them up, subtract them from total net worth. An algorithm is used to calculate the budgeting ability of the student. A score will be give.

How we built it

Node.js was used to create server. HTML,CSS, Javascript for the website. Android Studio for the mobile app.

Challenges we ran into

We came in conflict with the overload of features for this app. In the end, everyone compromised and agreed upon the same ideas and features.

Accomplishments that we're proud of

We were happy with the seamless integration between mobile app and website. The Node.js server worked flawlessly

What we learned

Node.js web development and mobile development

What's next for DDCRAB

Improve on machine learning technologies to improve the budget life of a University studet.
