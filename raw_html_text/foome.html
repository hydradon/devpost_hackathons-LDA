Inspiration - Solve student problem about spending time and money on food for every week. This idea originally came in after spending hours and hours in the supermarket trying to figure out what to buy for the week.

What it does - generate a menu for the whole week after choosing certain categories

How we built it - with java Swing and java api

Challenges we ran into - we counter numerous problems while displaying the gui

Accomplishments that I'm proud of - we eventually finish demo of the app

What we learned - teamwork, interaction between codes of group individuals

What's next for FooMe - expand to using databases and webserver to generate a full set of menu
