Inspiration

Many commercial game engines have yet to take advantage of next generation graphics APIs, so we decided to get a head start and build our own!

What it does

The Y Engine provides a development platform for games using the modernized Vulkan API for next gen graphics.

How we built it

We built in in C++ and modeled it after an engine created by one of our team mates.
