Inspiration

Young students do not have enough access to coding education, especially within schools. While Scratch and Hopscotch provide a great playground, they do not provide the instruction necessary for students to pick up the fundamental concepts of computer science and programming.

What it does

The Code Gakko Platform makes it very easy and efficient for teachers to provide coding courses. Teachers can select and curate a course based on a list of cards. The cards are like an atomic lesson, complete with exercises. A dashboard makes it easy to follow student progress.

How we built it

Flask, Javascript, Python

Accomplishments that we're proud of
