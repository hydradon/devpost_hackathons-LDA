Inspiration

We have been inspired by the great possibilities that the Nexmo API has given us, as well as the idea of making a Server application that can translate language and destroy barriers between people.

What it does

It is a client to client chat server application that can translate messages from one language to another.

How we built it

We built it with java and divided the project into Client and Server application.

Challenges we ran into

Making the server multi threading was for sure the hardest challenge we faced.

Accomplishments that we're proud of

We managed to make the server Multi Threaded and you can message not only real numbers but Nexmo virtual ones as well.

What we learned

We learned a lot about sockets and APIs.

What's next for NexmoChatService

It will be great if we manage to extend and improve the GUI as it is lacking good looks at the moment.
