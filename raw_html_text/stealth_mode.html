Inspiration

Our drive for the project was to solve the problem of accidentally liking a post on Facebook.

What it does

It removes Facebook's Like Button from all posts and comments.

How we built it

We built it as a Chrome Extension which uses JavaScript, JQuery, and JSON.

Challenges we ran into

This was our first time making a Chrome Extension, so we had to learn how to use the languages specified above.

Accomplishments that we're proud of

We are proud that we finished it in a timely manner, considering all of the challenges we faced.

What we learned

We learned that JavaScript can be hard and difficult language to program it at times, but once you have the patience to sit down and understand it you can really take advantage of what it has to offer.

What's next for Stealth_Mode

Hopefully, we can use this extension for other Social Media as well as have users be able to customize it with different options. Additionally, we would like to see our idea be brought to other platforms, specifically in mobile devices.
