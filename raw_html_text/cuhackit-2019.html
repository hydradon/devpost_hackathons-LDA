Inspiration

Women's March and the lack of feminine hygiene products within bathrooms

What it does

This feminine hygiene dispenser is wifi enabled thus allowing students and visitors to check the availability of the dispensers closest to them. Maintenance is notified if the threshold reaches a low number so there is no weekly check up for regular maintenance.

How we built it

This project was built with python through the web framework django, javascript, heroku, raspberry pi, redis, postgres, Materialize css, Google Maps api, and html. For payments, it uses Stripe's payment system.

Challenges we ran into

A challenge we ran into was finding a spring for vending, as well as finding a structural demo vending box. We also ran into challenges deploying heroku and setting up redis, our caching system that we use for our websockets to tell a dispenser to dispense.

Accomplishments that we're proud of

We accomplished a fully functional dispenser box prototype, in addition to a fully functioning web interface.

What we learned

Deploy early and often. We learned how to operate heroku.
