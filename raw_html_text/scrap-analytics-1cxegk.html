Inspiration

One of the problems that industries have these day is reducing the amount of Scrap. This is because of the loss incurred from scrap and the time wasted by manufacturing the scrap. Plant Managers objective is to plan for the scrap and make necessary actions.

What it does

Application uses the historical scrap information and other parameters to train a Gradient Boosting Machine Model. Once the scrap data is trained with necessary parameters it will guide the Plant manager in predicting the Scrap. If Plant Manager has a knowledge of this before hand he can take necessary action to reduce the scrap Thus, creating a Impact

How I built it

We built it with Python and hosted on Predix with the help of available services.

Challenges I ran into

Integrating the application with the Predix services was a real challenge. Creating the necessary features for the GBM Model was the next.

Accomplishments that I'm proud of

Learning the Predix Platform in a Short time.

What's next for Scrap Analytics

Enhancing the application with Prescriptive Analytics.
