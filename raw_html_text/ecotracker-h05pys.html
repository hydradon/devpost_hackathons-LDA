Inspiration

Help people track their footprint so they are more aware of the impacts of their lifestyle on the environment.

How it works

We created a database with information about the CO2 Emissions/Water consumption/Waste creation/Land usage of multiple objects or activities. The app has a simple UI to present the total number of planet used by the user at the current date, based on his transport/housing configuration and entries (for example, bought objects). It's is also possible to get a detailed view of which type of impact is the highest for the user, as well as recommendation on what to improve to reduce their current impact. Finally, since the app has small nice to haves, like is being connected to Montreal's air quality report API and tells the user in real time what's the quality of the air.

Next steps

We'd need to actually save the data of the user so the app would be usable all year round. For now, it's more of a one use prototype than an actually working app. We could also host on our domain trouveton.space to have more visibility
