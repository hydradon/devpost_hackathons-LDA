We were inspired by the use of technology to enhance wearable fashion. Our skirt design uses two motors on opposite sides to create a gathered/cinching effect that changes the silhouette. The base layer is revealed to showcase a new, unique design. A set of buttons on each side of the skirt allow the user to gather the side seams and create a draping effect, or alternately lower the fabric and release the tension to revert to the original A-line style.
