Inspiration

People who want to find communities/events.

What it does

App to help build communities through local events. Using swiping to allow easy access to browsing. Can be used on a night out or to find a LAN party. Small businesses can also utilize this by offering free food for publicity etc.

How I built it

I built in in Android Studio using java and xml, using third party api's like google maps and geocoder etc.

Challenges I ran into

My emulator wouldn't install play services for a long time.

Accomplishments that I'm proud of

Being able to do something.

What I learned

How to use Android Studio better

What's next for Event Thing

Planned on adding filters/tags to find events you would be interested in. Connection to Facebook to further integrate community and interests.
