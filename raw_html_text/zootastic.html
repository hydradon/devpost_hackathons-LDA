Inspiration

Kids love animals and I wanted to build something that allows them to help them learn using the Amazon Echo.

What it does

Zootastic asks questions about animals to help kids learn by using context clues. It is an engaging way for kids to learn and have fun!

How I built it

I built Zootastic on the Amazon aws platform using Node.js

What I learned

This was my first time building for Amazon Alexa and I was amazed at all of the cool things you can do with Alexa and the Amazon Echo. I plan to continue building on the Alexa platform.
