Inspiration

The inspiration came from the need to have one's bank balance displayed in an easily accessible, but at the same private way, which doesn't require lengthy login forms or trips to the bank.

What it does

Our system gets data on charities through the "just giving" api along with the users account balance (using the capital-one api). It then stores the data in a json file, which is passed to the users dashboard and the pebble app and displayed in an accessible and simple format.

How I built it

We prototyped the data retrieval system using python and then converted the methods to php, so that we could host it and make requests with the pebble. The pebble app was built using javascript. Our web-page and dashboard were made using bootstrap.

Challenges I ran into

Setting up a braintree server.

Accomplishments that I'm proud of

Pebble integration and api backend.

What I learned

PHP, pebble, apis, javascript

What's next for MoneyMatters
