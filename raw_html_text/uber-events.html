Inspiration

UberEvents is inspired by problems that consumers face in their everyday lives that Uber can help solve. Common problems faced when it comes to big events are bad traffic, high surge, fewer rides, etc. This is where UberEvents comes in to make our lives easier.

What it does

Schedules rides in advance of big events for better planning.

How I built it

We built it using python, reactJS, html, CSS and the Uber API
