Inspiration

Bandwidth is limited in radio communication

4G itself is limited to its frequency ranges.

Mobile providers need to know when this will run out, and monitor its depletion.

What it will do

Monitors the RF spectrum using the SODAQ ExpLoRer board connected to a radio receiver.

Graph the Bandwidth data that it collects on the application side.

Allow mobile providers to plan their business strategies with when to implement new mobile data solutions and when to move into new Radio Frequency bands.
