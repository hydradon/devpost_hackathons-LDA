Inspiration

Problèmes de la vie quotidienne des gens qui doivent travailler et vivre à Montréal

What it does

Permet de trouver plus facilement une place de stationnement gratuite à Montréal

How we built it

Application mobile en Java sur Android construite avec une base de données géospatiale Postgresql sur Google Cloud, une base d'utilisateurs Firebase.

Challenges we ran into

Construire une app mobile dans un temps très limité, construire la base de données pour qu'elle soit facile à utiliser.

Accomplishments that we're proud of

La base de données géospatiale, avoir appris à coder une appli Android en 24h.

What we learned

Comment construire une application Android, comment utiliser Google Cloud pour faire une base de données géospatiale, comment utiliser Firebase.

What's next for ParkMe

Ajouter quelques fonctionnalités de notre MVP (minimum valuable product) que nous n'avons pas eu le temps d'ajouter, et peaufiner l'expérience utilisateur pour s'assurer que notre produit est facile et satisfaisant à utiliser.
