Inspiration

Although Mexico is one of the countries with the greatest linguistic diversity in the world and more than 68 indigenous languages ​​and 368 variations are spoken in its territory, 64 of them are at high risk of disappearing.

What it does

In this entertaining didactic game you will learn about culture and history through questions. Likewise, you will test your ability in the Nahuatl language and you will have the opportunity to choose a story mode to learn about the Aztec culture

How we built it

This project was built with Alexa skills in the Alexa developer online environment, in addition to using visual studio code connected through git, it also supported the use of amazon audio libraries

Challenges we ran into

We didn't know anything about Alexa developer until yesterday, so we had to take several online courses, as well as follow the support material. A problem to highlight was trying to give Alexa's voice different tones so we could improve user experience by making it sound more natural

Accomplishments that we're proud of

WE DISCOVERED THAT WE CAN DEVELOP A GOOD PRODUCT FROM ZERO WITHOUT HAVING GREAT NOTIONS OF AN ENVIRONMENT. THE EFFORT WAS GREAT!

What we learned

We learned about JavaScript and how objects and indents are used in addition to using amazon online libraries. Also, we improved our project management skills by learning the correct use of git

What's next for Xomotl - Conociendo mis raices

We are looking forward to adding more levels, as well as a survivor mode for when you are an enemy and a multiplayer mode for playing between two people. In the best-case scenario, we will add as many indigenous languages as we can. We would also love to get real advice from indigenous communities and experts on how we can improve this game for better promotion of the indigenous languages.
