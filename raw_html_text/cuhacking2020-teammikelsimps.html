Using Twitter APIs and Natural Language Processing libraries we analyze recent tweets to find potential production issues for RBC web and mobile applications. Made for the RBC analytics challenge.

As a team most of us have only had a very small amount of experience with web development, so we decided to go all in on a node.js project with API calls to twitter and usage of a natural language processing library to complete the RBC challenge and learn more about the web tech stack. Along the way we faced a lot of challenges, having never built a proper UI in js we had quite a lot of trouble messing around with bootstrap until we decided to just make a simple html design instead and focus on the technical aspects for our API calls and nlp library usage to select tweets that are relevant to possible issues with RBC applications.
