Inspiration

Once we learned of postmates, we thought about all the non-delivery uses for it. This was the best one.

What it does

We provide best time tracking for certain routes for postmates. We also use the twilio API to contact postmates with their performances.

How we built it

We built an API server on flask that does the tracking and a website GUI to visualize it all.

Challenges we ran into

The test postmates are all about evenly quick. A bigger problem is that they run through walls, and only 10 exist.

Accomplishments that we're proud of

The server actually survived a rather long stress test.

What we learned

We learned how to do web development.

What's next for Race Condition

Right now we're going to actually get real postmates to race for us.
