Inspiration

The golden age of the Internet is over. Its' "pipes" are full of AI spam, unpoliced political ads and shotty big-tech policies. It's time to take back control of your browser experience. Yes, it's time to censor YOUR Internet.

Check-in to the Internet with a custom profile and leverage pre-built rules that support the Free World, Climate Change, truthful political ads, etc.

What it does

The Blockchain to Browser technology injects CSS into the webpages you browse. The resilience of the network allows us to, for instance, block harmful products directly on amazon.com (ie/ Deet: B004H89KFC ).

How I built it

We chose to go with a hybrid approach. The front-end chromium browser extension talks to ethereum via web3 and sources a rotatable API endpoint. Python libraries where developed to manage the core infrastructure. This involves serving truthpipes.com and minting new rules & policies to a smart contract list manager (poor-mans' TCR). The smart contract records are browseable on the-graph.com

Challenges I ran into

I kept my scope narrow though was still hit by the extra over-head of managing a hybrid setup. But it works!

Accomplishments that I'm proud of

The web3.py library is great, but complete application design examples where missing. My Python libraries encapsulate the functions required to manage smart contracts and the security & gas required for specialized functions.

What I learned

Working in the ecosystem was easier then I thought. Though, I still believe that adoption will continue to thrive via these hybrid approaches.

What's next for truthpipes

We're currently collecting ideas and frustrations that people have with web2 (things they want banned). These will be used to formalize a set of policies that can be checked into and consequently rules will be outsourced "bounty-style" to supportive developers. Of course, there is a lot of grey-area in censorship so the governance layer is the next BIG goal.

How to use it

See github for all source code . You can add the browser extension and can immediately begin to see harmful products highlighted on amazon.com. Give it a try! Comment on truthpipes.com And thumbs up!
