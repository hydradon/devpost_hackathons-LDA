Inspiration

We wanted to create a bot that talked to people with mental health disorders that maybe didn't have the courage to speak to a real person or the money to go to a psychiatric.

What it does

It's a bot that the user talks to and, depending on the situation, the bot says some tips and certain procedures the user should consider following.

How we built it

LUIS Microsoft and BotFramework API's.

Challenges we ran into

Problems on getting the APIs' keys. If we had gotten them since the beginning the result would be much better.

Accomplishments that we're proud of

Managing to get something done after only two hours of coding without having any previous experience with these API's and the Javascript language.
