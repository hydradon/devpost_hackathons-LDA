Inspiration

Technologies presented at AUHack.

What it does

Augmented reality, mixed with tower defense. Using lego with special stickers as towers.

How we built it

Unity for the game engine, with game logic written in C#. Vuforia for the augmented reality.

Challenges we ran into

The scaling of objects when using Vuforia. Tried to implement a map scanner, so users could paint their own map, though porting the image analysis code from matlab to unity proved to be more difficult than first anticipated. We were also unable to use only legos for towers, thus we had to use more detailed stickers ontop of the legos.

Accomplishments that we're proud of

The final prototype.

What we learned

Using new tools, unity, vuforia etc.
