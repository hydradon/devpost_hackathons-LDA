Inspiration

JIRA greatly improves our team's productivity, and we want to make it even more productive.

What it does

This plugin modifies JIRA's appearance to make it easier for reading. Changes include:

  • More readable font and styles.
  • Cleaner comments section.
  • Tooltips showing issue's details.
  • Tooltips showing attached images.

Please see gallery for screenshots.

How I built it

We mainly use Web Resource Module from JIRA plugin framework.

What's next for JIRA productivity pack

We will continuously improve the plugin. If you have any idea, feel free to suggest it here.
