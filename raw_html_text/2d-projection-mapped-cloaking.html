Inspiration

Harry Potter's invisibility cloak, Tony Stark's "invisible" airplane

What it does

This is a script that uses a camera as well as a predetermined point of view to create an invisibility cloaking device.

How I built it

I used Autodesk Maya and MEL (Maya Embedded Language). I also used Google Cardboard's VR photo app.

Challenges I ran into

Complex geometry does not map well, so the UVs need to be skinned for more complex mapping.

Accomplishments that I'm proud of

I finally know some MEL

What I learned

I furthered my knowledge in projection mapping and I now know a little MEL as well.

What's next for 2D Projection Mapped Cloaking

UV skinning to interpolate texture mapping into physical LEDs/pixels
