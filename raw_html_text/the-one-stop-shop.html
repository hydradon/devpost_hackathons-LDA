• The One Stop Shop

Inspiration

Long plagued with the inability to easily buy fruit, we yearned for an easy solution to get our fruit fix. Just then it hit us: an all in one fruit online marketplace.And just like that, The One Stop Shop was born! It's safe to say that nothing will ever be the same.

What it does

The One Stop Shop offers a selections service of over 3 types of fruit. We plan on allowing the user to buy and have fruit delivered in the future, but right now the app serves as a luxury fruit price estimator.

How we built it

We painstaking wrote the HTML from near scratch, and used javascript to handle the data manipulation.

Challenges we ran into

We lacked some basic knowledge in data storage of user information ans decided to implement it another time. We also have to get help from mentors on how to structure our files to receive data from HTML into Javascript

Accomplishments that we're proud of

We have used our new found research skills * as well as our knowledge of HTML and Javascript to create a fuctional website with soem value

What we learned

We learned how to learn!! We gained a knew experience that forced us outside our comfort zone and apply our basic skills to be successful

What's next for The One Stop Shop

It will continue to be developed and well one day have a user registration ans shipping page
