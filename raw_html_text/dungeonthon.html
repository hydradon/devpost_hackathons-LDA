WE ARE TEAM GARDENLAND

Inspiration

We wanted to build a dungeon crawler similar to Diablo or Champions of Norrath.

What it does

Fully functional prototype with replayable level. Four types of enemies. Ranged and melee attacks. Pickup health items. Try to survive as long as you can.

How we built it

The game is built using Unity.

Challenges we ran into

Using unity.

Accomplishments that we're proud of

Actually having a working demo that is kind of fun to play.

What we learned

Features take a while to implement. Build up slowly. Constantly iterate.

What's next for Dungeonthon

Polish, add animations. Improve gameplay. Add more abilities and develop sophisticated combat.
