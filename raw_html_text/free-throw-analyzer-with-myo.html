Inspiration

Everyone wants to get the edge over their competition, so why not use wearable technology to collect data and analyze performance.

What it does

The desktop application allows the user to collect data via the Myo and its Bluetooth dongle, which can then be seen from a website online.

How I built it

The desktop app was built with JavaFX for implementing the UI, the JDBC driver for connectivity to a database, and PHP and JS for the website component. Additionally, we utilized the Myo SDK to interact with the Myo.

Challenges I ran into

It was fairly difficult to get the data from the Myo and present it in a simple yet effective manner.

Accomplishments that I'm proud of

We were able to create a cloud based solution for the data display.

What I learned

We learned how to interact with new hardware and face unfamiliar challenged.

What's next for Free Throw Analyzer with Myo

At the moment, only a simple optimal path is shown after analyzing the user's history of successful shots. We feel that other factors that could be analyzed and displayed for the user.
