Inspiration

The task of buying a house, and potentially settling down, is one that requires lengthy amounts of time and effort...thus, perfectplacefor.me was born!

What it does

The web app allows for the user to input their answers to pre made (and custom) metrics to find out the "perfect" pace to live.

How I built it

Stage 1: Analysis and parsing of data. Stage 2: Creation of front end. Stage 3: Data implementation Stage 4: Backend Stage 5: Finishing touches (yelp implementation)

Challenges I ran into

Implementing yelp without a backend...found that this wasn't possible.

Accomplishments that I'm proud of

Finishing the project.

What I learned

A lot of web development :)

What's next for perfectplacefor.me

Hopefully it is used by a widespread audience
