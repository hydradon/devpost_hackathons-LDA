What it does

The objective of the game is to protect, grow and harvest plants as long as possible, while defeating evil monsters. Rain, night, and day cycles affect how plants grow.

How we built it

We used Intellij, LibGDX, and a custom build light weight game engine, to develop this project.

Challenges we ran into

Figuring out how to position objects correctly in the world using the polar coordinate system was a big challenge that we are proud that we were able to work through.

Accomplishments that we're proud of

We are extremely proud that we were able to create a functioning, enjoyable game in the 24 hours we were given.
