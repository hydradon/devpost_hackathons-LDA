Inspiration

Many times we search spotify for playlists that we want to fit our mood but we can never find them

What it does

IOS and Android app that generates a playlist from spotify based on a series of moods (joy, anger, sadness, depression, and love)

How we built it

We built it using cordova js, spotify api, node js and ionic which translates html and css into app code

Challenges we ran into

Spotify playlists not generating the way we wanted to. Aesthetic problems with the app. Many others as well

Accomplishments that we're proud of

Making an aesthetically pleasing app

What we learned

How to use Ionic and Spotify API
