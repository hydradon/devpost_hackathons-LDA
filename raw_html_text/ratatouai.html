What it does and how we built it?

RatatouAI uses clairfai food API to analyze the potential ingredients in a give image. From this list of ingredients, we pass the data to a Food2Fork AI, which returns a relevant recipe and link.

What's next for RatatouAI

We envision that the next step for RatatouAI is to incorporate filters that would fine-tune recipe outputs. Such parameters would vary based on user preferences such as: low fat, low calorie, pescatarian, or even high vitamin C content. We would also like to utilize a better recipe API since Food2Fork is quite limited and does not accommodate for parameters such as those mentioned before.
