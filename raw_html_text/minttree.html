Inspiration

We're all broke, college students.

What it does

The app asks for a budget over a certain amount of time and shows the value in leaves. Each leaf represents a certain amount of money being spent. Orwell the Owl is a cute friend who encourages you to save and is saddened when the user goes over budget.

How we built it

Angular 2 (with HTML/CSS elements) for the front end and Angular 2 and a Firebase database for the backend (allowing us to use AngularFire).

Challenges we ran into

Pulling data from database and accessing specific elements in the object.

Accomplishments that we're proud of

Leaves fall as money is spent.

What we learned

Angular 2

What's next for MintTree

More interactive GUI.
