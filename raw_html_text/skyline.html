Inspiration

I have always seen distant buildings while driving along Texas and always wondered where that building was so maybe I could go visit.

What it does

Skyline uses trignometry , magneometers, acclerometers m and Google Services to identify buildings at any distance. It does this by calculating how far the building is where you are facing and then using Google to draw the position.

How I built it

I used Android Studio and Java.

Challenges I ran into

Google Play Services wouldn't work so I had to use the deprecated library for location.

Accomplishments that I'm proud of

It worked in the end.

What I learned

Lots of Android Studio, I didn't have any experience with the program prior to this event.

What's next for Skyline

Use ArcGIS to query elevation and building data to refine results and simplify user experience. Didn't do this because the license is $5,000 and the free trial requires credit card information.
