Project Goal

Develop a voice recognition system that will extract an individual’s name from certain introductory phrases, which will then pair the voice & name with the image of the person The system will then store this knowledge and regurgitate the information in the future when the voice and face is recognized once again

Real World Application

Can aid persons with Alzheimer’s disease or other memory difficulties remembering other people’s names can integrate program into an everyday item the user may use (such as glasses) to help the user remember people’s name in real time can create an online exercise that can engage the user visually and audibly to aid in retaining memory.

Why Now?
