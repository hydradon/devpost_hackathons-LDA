The Whole story:

Inspiration

We wanted to help give teachers a tool to see how well students are understanding the material.

What it does

Our project takes a pdf of a powerpoint, generates questions which the teacher can edit, and provides a simple environment for taking short quizzes through a phone app.

How I built it

We used Ionic to create a fast app on both Android and IOS as well as Angular for a website with more powerful quiz making tools for teachers.

Challenges I ran into

Converting powerpoints to pdf and then extracting the text was very difficult. In addition, none of us knew AngularJS or Ionic very well and there is definitely a learning curve for those.

Accomplishments that I'm proud of

The look and feel of the app, as well as the neatness of the database.

What I learned

Having no prior knowledge of the ionic framework, it was challenging but fun to jump right into a project and learn about the environment as we were trying to build it.

What's next for Octopus

We plan on adding more useful features for creating quizzes well as linking with the URL octopi.tech (we registered but did not get confirmation in time).
