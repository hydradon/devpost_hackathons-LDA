Inspiration

Hurricanes are an increasingly serious problem due to global warming. We want to improve response times and response execution by providing analytics to governments and aid organizations.

What it does

Satellite images are segmented by a machine learning algorithm and classified into destroyed/intact sections. We find the area of destruction and multiply by the prices corresponding per those areas at those areas.

How we built it

Web-app with back-end flask with Tensorflow.

Challenges we ran into

Not knowing front end.

Accomplishments that we're proud of

Learning some front end, creating a functioning project that can help people react to hurricanes effectively. .

What we learned

Some front end lmao. But also how to make pretty pictures/analytics and web-scraping/ML.
