Inspiration

I like the idea of users being able to purchase their boutique desires over a VUI.

What it does

Alexa will ask Jane for a user's top number of deals, and offer to tell them more about a deal in that result set.

How we built it

We built it with .NET Core, Azure, and of course, Alexa.

Challenges we ran into

It runs out that Alexa's not designed to handle dictation, so we had to change our initial design that allowed a user to ask more about a deal by name, to a numerical representation of that deal.

Accomplishments that we're proud of

We did a lot in a short period of time. We even got continuous deployment up and running within a few minutes of starting - that really helped out when testing.

What we learned

We learned that Alexa is a powerful platform! It's very easy to interface with.

What's next for Alexa Hackathon (Jane)

I'll probably talk to our product people, and see if they find any value in eventually maturing a VUI for Jane.
