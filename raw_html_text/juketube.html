• JukeTube web interface

Inspiration

Repurposing older technology for modern times: What do you think of when you hear retro? For us it was old-school diners and jukeboxes, which was the inspiration for this project.

What it does

JukeTube is a virtual jukebox, where anyone can request a song (in the form of a youtube video) to play. Songs are added into the queue and played when their turn is up.

How we built it

JukeTube runs as a web server written in python, and can be integrated with an IOS app to easily request tunes.

Challenges we ran into

Determining how to play audio. Should we use Spotify, Youtube, or something else? Determining the best way to run a web server that would control the jukebox. Dealing with awkward APIs and manipulating processes.

Accomplishments that we're proud of

Creating a functional product.

What we learned

Determining the best technology to accomplish a job is pretty difficult.

What's next for JukeTube

A fully-functional app, and more attractive front-end for requesting songs.
