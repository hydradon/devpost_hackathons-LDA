Inspiration

We wanted to help all students to be able to succeed in school, as we are ourselves. We know the difficulties of not having easy access to help for doing homework or projects, so we decided to create this website.

What it does

hWiki allows students in any grade communicate with one another to improve their skills and make deadlines in certain academic subjects.

How we built it

We used the website "Wix" to create our website.

Challenges we ran into

How to actually begin to build the website and making an outline.

Accomplishments that we're proud of

We were able to come up with a great idea quickly and were able to execute it.

What did we learn

We learned how to plan out, outline, and create a website
