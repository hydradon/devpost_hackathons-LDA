• OMeal...OK!

Inspiration

Idea of serving the customers in best possible ways and to provide optimum service.

What it does

This application allows customers to enter their search term and select the dietry requirements and restrictions if any and get back the list of recipes with list of ingridients and the instructions to prepare it.

How we built it

  1. first we developed python code to work on the API.
  2. next we built flask code integrate the python with front end website.
  3. at last we developed frontend web app with html and angular.js.

Challenges we ran into

  1. integrating flask and htmk and angular.js code.
  2. figuring out the what to offer to the customer.
  3. initial E2E design.

Accomplishments that we're proud of

  1. today we learnt flask, working with APIs and html.
  2. overcoming the initial design hurdle.

What we learned

team work, facing the challenges head-on, many technical nuances

What's next for OmealOK

further customization of user options, search optimization, user profiles, integrating with grocery shopping platform
