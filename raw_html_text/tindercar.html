Inspiration

  • There was this cool car from Timing Architects that we wanted to try modern car safety features on and that we wanted to play around with

What it does

  • It detects when there is something in front of it and keeps a safe, constant distance to its surroundings
  • You can control the car using voice-control
  • You can make it do cool tricks (drift, dance...)

How we built it

  • Using the documentation, Python and the Arduino IDE

Challenges we ran into

  • A camera with little to no useful official documentation
  • A transistor was failing
  • A completely new platform that we haven't developed for / dealt with previously

Accomplishments that we're proud of

  • Voice-control
  • A working RGB- and depth-camera
  • Collision detection and -prevention

What we learned

  • Coding in Python and C++
  • Dealing with the Arduino IDE

What's next for Tindercar

  • Autonomous driving, self-parking, a "summon"-feature
