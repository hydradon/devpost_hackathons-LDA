Inspiration

We started playing with the band and were really surprised how neat the controls were so we set out to make a project with the neat little armband. also wanted to use unity as it is a very interesting

What it does

a small game showcasing the powers of the Myo armband, showing the ability it has when combined with unity

How we built it

Using unity and the SDK for the Myo, we built upon those frameworks to develop a way to read from it.

Challenges we ran into

-very complicated (uses quatrains, unfamiliar with) -had to go in with very little knowledge of c# -lots of errors and bugs -odd controls

Accomplishments that we're proud of

-got it working -the music -nice clean code

What we learned

We learned a lot, from the framework of the Myo to c# itself. we went in with little knowledge of c# but learned a lot

What's next for MyoBall

It might become something more but, because we don't have the necessary hardware, it most likely won't be developed further
