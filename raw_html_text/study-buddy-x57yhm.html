Inspiration

A huge number of students struggle to maintain grades and revise efficiently. Our solution helps students make better use of their time to maximise learning potential.

What it does

Our website automates the tedious task of preparing question banks for revision. All that is required is that the student input their notes into the software. We take care of the rest.

How we built it

The UI and interface is hosted through python flask. Keyword extraction and question generation is also done through python using the TextRank algorithm.

Challenges we ran into

Extracting keywords was quite challenging. Additionally we have had no prior experience in developing web application and as a result found the front-end development difficult.

Accomplishments that we're proud of

Just getting basic end-to-end functionality working is something we feel we can be proud of.

What we learned

We gained fundamental experience in web development using python flask and a vanilla JS/HTML/CSS stack.

What's next for Study Buddy

We would like to implement spaced repetition functionality to further increase learning yield against the time spent in revision.
