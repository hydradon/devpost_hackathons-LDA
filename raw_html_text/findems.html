Inspiration

Every minute spent in the ER can be worth a lifetime for a critical patient. This time-sensitive, fast paced environment is no place for extensive or inefficient paper records, which hinder healthcare providers' ability to devote their time to treating the patient.

What it does

FindEMS is a web application that transmits patient information, such as demographics and vitals, in real time to hospitals. This allows the hospital staff to become acquainted with the patient record before the patient's arrival, speeding up the treatment process. This application will also allow transport information to be saved into majority of prevalent EMR softwares.

How we built it

WE utilized CSS and HTML to develop our front end, and the back end databases were developed using SQL to store patient information.

Challenges we ran into

Many of our team members were unfamiliar with CSS and HTML, and had to learn it on-site to develop the front end of our application. We also struggled with new languages and issues with integrating the different platforms we have incorporated into our application.

Accomplishments that we're proud of

We are proud of all the new skills we have acquired, and successfully surviving the weekend!
