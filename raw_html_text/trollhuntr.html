Inspiration

Ever wonder how many trolls lurk on your favorite sub-reddit? Want to know their alternate accounts?

What it does

Use TrollHuntr to hunt the trolls. TrollHuntr scrapes subreddits to find Troll, Throwaway and Alternate Accounts. Then using machine learning it tries to uncover the real accounts behind the trolls.

How We built it

TrollHuntr was built using Python. The Reddit API and Praw were used to scrape subreddits and user data. SciKit Learn was used for all machine learning aspects

Challenges We ran into

Finding a reliable method to calculate simmilarity between troll and real users.

Accomplishments that we are proud of

Learning and implementing simple machine learning during the hackathon.

What we learned

The basics of machine learning
