Inspiration

Looking for a fun twist on a classic platforming game!

What it does

A 2d platforming puzzle solving adventure where you're in control of gravity!

Learn cognitive skills and basic physics by playing Cake!

How we built it

Cake was built using Lua and the Love2d engine!

Challenges we ran into

Changing gravity was a pretty hard task, especially getting smooth rotations and changing the camera and player orientation.

What's next for Cake

We'd like to create levels collectibles, and enemies for increased gameplay experience
