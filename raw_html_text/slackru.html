Inspiration

We created a slack bot that would make the experience for hackers and mentors a bit more organized and clean.

What it does

Right now its functionality is solely based on finding available mentors for hackers to talk to when they have concerns about some hacking tool or hacking language.

How we built it

We used python to talk to the slack client and we used a sqlite database to integrate multiple mentors.

Challenges we ran into

Dealing with the slack api, jfc

Accomplishments that we're proud of

We're proud that the bot was able to create new private channels between mentors and their hackers, really cool stuff

What we learned

We learned a lot about the slack api with python, it's a lot easier to use once you've had a long lasting relationship with it.

What's next for SlackRU

This slack bot was not made just to have this mentor functionality, we intend to add a lot more relevant things for this slack bot to do, like create shoutouts to tell hackers about upcoming events, help hackers talk to other group members of the hackathon group, etc.
