Inspiration

"Alla Moda" means 'fashionable' in Italian. UCSC Hackathon 2017

What it does

Alla Mode is an app that allows you to upload your own clothes, sort them, and experiment with different outfits. It's a powerful tool for any style-conscious individual!

How we built it

We attempted to use Android Studio.

Challenges we ran into

The app is, as of now, just a simulation.

Accomplishments that we're proud of

The graphic design and concept of the app.

What we learned

We gained a lot of skills in programming in Android Studio and even Xcode.

What's next for Alla Mode

Bug fixes, and the actual code surrounding the app simulation.
