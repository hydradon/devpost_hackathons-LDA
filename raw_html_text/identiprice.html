Inspiration

The inspiration behind the project really comes from the Clarifai tech talk; they introduced us to their API and we took it from there.

What it does

You upload a picture of any object you'd like to know the price of, and the app delivers the price of that item from Amazon.

How I built it

We used Java code, GUI using Java Swing, parsing using Json, Clarifai API, Sellerlabs API.

Challenges I ran into

Unfinished Clarifai documentation and portability.

Accomplishments that I'm proud of

Actually submitting a project at my first hackathon.

What I learned

I learned how to use Clarifai's and SellerLab's API, a little bit of Python code, a little bit of Swift code, Json.

What's next for identiprice

If we're allowed to keep working with the provided API's, we'd like to port the application out to a mobile environment for more seamless experience.
