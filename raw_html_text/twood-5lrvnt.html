Inspiration

Social media is an important aspect to many people. We were interested in seeing what kind of information we could get from analysing other peoples tweets. After exploring some dataset we found the Sentiment 140 and thought of this project.

What it does

Visualizes networks between users and mutual friends, relationships, individual temporal changes in mood over twitter using machine learning.

How we built it

matplotlib and networkx for visuals, Twitter API and tweepy for gathering data, keras and tensorflow for the machine learning back end

Challenges we ran into

Time constraints, learning new APIs and techniques

What we learned

A lot about using matplotlib and using social media API's to gather data
