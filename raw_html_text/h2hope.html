Inspiration

What it does

Displays a map that shows the level of influence of different variables on shark migration patterns.

How we built it

We used the MEAN stack with Mapbox.JS to show the visualization with data parsed with Python and stored in MongoDB.

Challenges we ran into

We were working with enormous data sets, causing queries to take minutes to run and laptops to freeze. Mapbox.JS also had useful functions but difficult-to-understand examples that required extra attention to integrate.

Accomplishments that we're proud of

The front-end with AngularJS and HTML5 is beautifully organized, with background videos on every page. While difficult, finding and working with the huge amounts of data definitely gave us a perspective on what kinds of constraints data engineers deal with.

What we learned

We learned a lot about working with big datasets and workarounds for sprawling projects. And plenty of Git tips and problems.

What's next for H2Hope

Integrating all the tools into the backend and using the newly understood data to make actual predictive models for migration.
