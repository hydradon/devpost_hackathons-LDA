Inspiration

As initially awkward kids, it would have been nice to have a platform where we could find people that were strangely like us. This platform aims to solve that.

What it does

The user fills out a questionnaire and is then matched with a person of similar personality based on an algorithm.

How we built it

Used html/css for front end. We used firebase to handle the database and store our profile.

Challenges we ran into

We were almost able to implement IBM's personality insight API, had it working in the terminal itself but it was not feasible to finish it with the time we had left.

Accomplishments that we're proud of

We were able to implement firebase without having much exposure to it.

What we learned

We learned to have various back up options since some features will ultimately be scratched by the end of the hackathon.

What's next for Link Blue

We are hoping to completely implement IBM's personality insight.
