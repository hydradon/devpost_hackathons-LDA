Inspiration

I have a ton of friends who stream on Twitch and they've always mentioned the difficulty of configuring bots for moderating their Discord server.

What it does

Allows Twitch streamers to login and configure a bot on their Discord server entirely through a web-application. All GUI-based

How I built it

Using JavaScript, I was able to create the web-app using Express.JS and Node.JS.

Challenges I ran into

I had a difficult time remembering the classic CSS properties; took me longer to configure the GUI.

Accomplishments that I'm proud of

I now know how to use Express.JS and now have a better grasp of creating GUI applications.

What I learned

How to use Express.JS to create web-apps.

What's next for Glance

I am going to continue to develop Glance gradually so that the Twitch community has an easier way to configure Discord bots.
