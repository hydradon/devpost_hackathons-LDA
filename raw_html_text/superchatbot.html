Inspiration

A Super chat bot which can fulfill multiple task for daily productivity and needs for the user and which will not save any user data in the application . For first step SuperChatBot shall provide Translator Bot to provide translation in to multiple language.

What it does

SuperChatBot provides translator bot in its first release on decentralised environment. It can detect input language automatically and translate text into following languages: english, french, german, italian, portugese, russian, spanish and turkish

How I built it

AWS Lex bot has been integrated into React framework . AWS Lex bot is written using Python .

Challenges I ran into

Integration of AWS Lex and React without putting security credentials into browser

Accomplishments that I'm proud of

It is a flexible framework and Multiple Bots can be integrated as part of SuperChatBot. then it can be used as Google Assistant for multiple tasks

What I learned

React Framework, Blocstack Authentication, DAPP
