Inspiration

Motive to find a person's social account after a meeting.

What it does

Users take pictures of people they meet at meetups, conferences, hackathons. With the image / person's name as input the application finds social profiles of the person in the image. We started with LinkedIn for now.

How we built it

An Android frontend for a Django based https server.

Challenges we ran into

No LinkedIn API's to search for a person's profile. So we created a work around.

Accomplishments that we're proud of

We were able to search LinkedIn without a default API, and import the data into our app

What we learned

Django!

What's next for FLMatcher

An app that can find all social profiles of a person in a meetup, maybe with just facial recognition.
