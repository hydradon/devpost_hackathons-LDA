Inspiration

Tired of always having to open up a different app just to figure out what the weather or time is in another city.

What it does

Uses the openweathermap api, maps.google api and stdlib's slackbot integration to determine the time and weather in other cities using a slackbot. Also has a pretty neat feature for people who love SpaceX and rockets in general.

How we built it

Looked up tutorials on using stdlib and its slack functionality. As well as researching the format openweathermap api and map.google api accept inputs and return outputs.

Challenges we ran into

Ran into quite a few errors that are typical with api's, such as format issues for the inputs and outputs.

Accomplishments that we're proud of

It works just the way we planned!

What we learned

How to use stdlib, openweathermap api , and maps.google api to create useful chatbots.

What's next for Slack Weather and Time Utility

Have Elon Musk add our bot to the SpaceX Slack workspace, so they can use the /spacex slash command
