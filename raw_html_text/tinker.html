• Tinker

What it does

Tinker is a "personal assistant" for telegram.

How we built it

We used The telegrams bot api and CLI as the platform and node.js for the "server-side".

Challenges we ran into

Firstly we had to learn how to use Node.js and understand the async nature of the language as part of the async nature of JS we had problems with understanding how to work with the DataBase. Then we had to understand the Telegram's BOT-API and the CLI library and how to compile.

Accomplishments that we're proud of

A working Bot that for now has the next features:
-Weather telling
-List management
-Messaging management

What we learned

We have learned the node.js language and its asunc nature, the Telegram's BOT-API and CLI.

What's next for Tinker

More Features to make him an actual power tool.
