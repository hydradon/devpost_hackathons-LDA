Inspiration

Our mentor along with our parents spend most of their commute stuck in traffic, and get home much later from work or arrive at a later time than their schedule.

What it does

fastr provides options for the average user to select a convenient route to help them arrive to their destination faster.

How we built it

We used java, kotlin, in addition with the google maps api.

Challenges we ran into

We had to change our idea numerous times because it was very time consuming and required a lot of work which was unable to do with 12 hours. So we had to juggle between more ideas and finally found one that was suitable for us.

Accomplishments that we're proud of

Learning how to write complex code which was an addition to the basics we learn in school. Being able to try various methods taught us to utilize different structures to complete the product.

What we learned

Asynchronous tasks, how android makes https request, what an api is, learned about github and multi-threading. json file formats, and much more!

What's next for fastr

Adding different data sets (construction work) and expanding the region beyond Brampton.
