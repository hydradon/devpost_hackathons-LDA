Inspiration

We thought it would be fun to create a game, and then we just kept creating more!

What it does

Runs 4 different games, Simon Says, Bop-It, musical "instrument, and a puzzle.

How we built it

Using Arduino, LED, LCD display, and joystick.

Challenges we ran into

Managing the large amount of code that comes with 4 different games.

Accomplishments that we're proud of

Created games that we ourselves enjoyed playing.

What we learned

How to code Arduino, how to use joystick.

What's next for 4 Arduino Games

Adding more settings to each game.
