Inspiration

The confusion the Alzheimer’s patients have in remembering the names of the persons they meet. The methods and processes they use for remembering their names, like tagging or writing down their names is quiet time consuming.

What it does

ALZHEI CARE is a website where you can upload a photo of a person with their name. That information would be saved to the database. The website would use facial recognition to identify the name of the person.

How I built it

The website is coded using HTML and Javascript. Kairos API is used for facial recognition.

Challenges I ran into

  • I wanted to build this tool as an iOS app. Kairos did not have a good API service for iOS, so I switched to building a website.
  • It was really hard to send an image in a HTML post request.
  • It was not easy to convert an image to base64string.

Accomplishments that I'm proud of

I am proud that I was successfully able to build a tool to aid Alzheimer's community.

What I learned

I learned how to code in HTML and Javascript. It was my first time creating a website using HTML and Javascript. I also learned how to use and implement APIs.

What's next for ALZHEI CARE

  • Right now, the software just gives the name of the person, in the future I would improve the software by adding more information of the person. -In the future, I would create an app for ease of use and portability.
