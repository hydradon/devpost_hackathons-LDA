What it does

Scoliometer is a web app that measures distortions within the spine. Scoliometer can indicate early distortions within the spine so treatment for Scoliosis can easier for the patient than be harder to treat in the future.

How we built it

We built the Scoliometer as a web app using javascript, jquery, and html5. We designed the app's UI with photoshop and adobe-illustrator.

Challenges we ran into

We wanted to 3D print the phone case to have a precise prototype, but MLH did not provide 3D printers at this hackathon so we had to create a prototype out of cardboard.

Accomplishments that we're proud of

We're proud of being able to finish the project before the deadline.

What we learned

We learned how to use the phone's gyroscope and accelerometer with a web app.

What's next for Scoliometer

We plan to update Scoliometer to the Android and Ios Store in the near future. Everyone will be able to make their own Scoliometer phone case to accurately measure their spine's to find any abnormalities in shape.
