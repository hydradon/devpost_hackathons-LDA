Inspiration

Software engineering module CS2113T which inspired us to make chat bots.

What it does

Our project aims to build a product named GrandDuke, a Personal Assistant Chatbot that helps a person to keep track of various things. The name Duke was chosen as a placeholder name, in honor of Duke, the Java Mascot.

How we built it

The application is written in Java. AWS Lex and Polly provides GrandDuke with the intelligence it needs to communicate with the user.

Challenges we ran into

Lots of challenges. Example, poor wifi, troubleshooting.

Accomplishments that we're proud of

The product itself does not need require internet to use its basic functions.

What we learned

Tensorflow APIs

What's next for GrandDuke

Unite all chat bots by various organizations, under one great chat bot that manages all of them using certain credentials.
