Inspiration

I don't have a car, and have to use the buses to get around. I got annoyed with looking up the pdfs of timetables and reading them, so I thought this would be something fun to do.

What it does

You select a bus stop, then it populates with the buses that run though said stop, and then it will tell you the times that the bus stops there

How I built it

CloudPebble, Pebble's online IDE with an emulated Pebble

Challenges I ran into

Class abstraction in JS and the error logs aren't easily accessable

Accomplishments that I'm proud of

I got the tables and lookup functions working

What I learned

I learned a lot of JS syntax

What's next for CatTracks for Pebble

I'd like to finish it, maybe even publish (with approval)
