Inspiration

Amazon Alexa is a great tool for productivity, but is also a shining gem in modern entertainment. I wanted a fun skill out there on the store that can give people the excitement of exploring the world through difficult questions.

What it does

After being asked something like "Tell me an unanswered question," Alexa will generate one of the worlds many unanswered questions to make the user have an existential crisis! This skill is for entertainment.

How I built it

I built it using AWS servers and Lambda.

Challenges I ran into

My test questions were incorrect during publication, so I had to republish. Other than that, besides for a few minor coding challenges, everything ran smoothly!

Accomplishments that I'm proud of

This is my first skill published!

What I learned

How to code and publish an Amazon skill. I now look forward to developing more after this!

What's next for Existential Crisis - Unanswered Questions

More updates with more commands and more unanswered questions!
