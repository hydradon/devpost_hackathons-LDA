• Using the DJ Disc Jockey

Inspiration

We were inspired by multiple retro themes from Disc Jockeys to the Nintendo Powerglove. All of these ideas culminated into the idea to make retro-inspired DJ equipment with modern day technology.

What it does

It allows you to control the music volume, skip songs, go back to songs, and have lights match the tempo of your song.

How we built it

We programmed in Leap commands for 'raise volume,' 'mute,' 'skip song,' and 'play last song.' Additionally we made an arm band that allows the user to have lights blink to match the tempo of the song being played.

Challenges we ran into

We had a lot of trouble finding libraries to support what we wanted to do, but in the end of the day we were able to create workarounds for every problem we fixed.

Accomplishments that we're proud of

The Leap motion commands, and we also parsed HTML with Regex.

What we learned

Pyserial and coding using Spotify.

What's next for DJ Desk Jockey

More Leap commands and a cooler looking glove.
