Inspiration

We all love the Beltline in Atlanta! It's a way to get out and engage with the community, but sometimes it's too busy in certain spots to bike or skate or just walk.

What it does

Our hardware detects when people walk by and the ambient sound to find the real time and past data for Beltline traffic.

How I built it

Arduino + host of sensors --> (via bluetooth) Python Flask --> mongodb --> web

The page is built with basic HTML/CSS + Javascript and uses Google Map's API.

Challenges I ran into

Bluetooth was very difficult to set up, as was deciding on how to visualize the data and what we could do with it.

Accomplishments that I'm proud of

'Creating a meaningful visualization + getting real time data!

What I learned

Arduino and hardware components to software.

What's next for SquiggleDoob Beltline

Create a usable API for future users.
