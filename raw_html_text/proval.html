Inspiration

In Pakistan especially there is every basic product available in pretty much plenty of standards and

copies; there is no way to tell where the products in markets are substandard or counterfeit.

Especially substandard medicines are products whose composition and ingredients do not meet the

correct scientific specifications and which are consequently ineffective and often dangerous to the

patient. Thus a system like this, which provide a platform to verify the products is necessary and a

need.

The impact of this project on the industry is that its users will reject the substandard products and

the industries behind those products will have to stop it and be more ethical in their business and

provide genuine stuff rather than counterfeit.

It will bring a positive change in society and people can get what they will pay for, instead of getting

copied and fake products. Existing solutions to check the validity of the products is to check the product labels, packaging and

trademarks but somehow this is not an accurate way. For illiterate people this method doesn’t work

at all, and there is no much information on the packaging nor do the labels etc. show if they are valid

and genuine products. The other existing solution is to call the helpline of the product manufacturer

and verify the product, but that is time consuming and requires high resources.

What it does

For identifying the validity of counterfeit products we proposed the PRODUCT VALIDATION SYSTEM. The goal of

this system is to create an item verification software which will offer to approve only the valid and genuine

products. This framework will safeguard whether the item is counterfeit or not.

The presence of mobile phones is in abundance, even in low income, developing or third world countries, so this

system will provides an opportunity to consumers to check the authenticity of product using SMS text

messaging or via GPRS services.

This SMS-based product verification method relies on printing a unique one-time identification code having a

series of numbers on product packs, in form of bar code or QR code. SMS is sent which consist of code to server;

a message is sent almost instantly received specifying whether the product is authentic or fake.

This SMS based user friendly system which will also provide some critical information related to the product i.e.

ingredients, manufacturing date, expire date, details about manufacturer, certification of that product, warning

and health hazards related to that product. There will be two major modules of this project:

  1. Web based application

  2. Smart phone android based application User interface of Android application

Interface of this Android application will allow user to scan the BAR/QR code or enter the code to validate the

product. Camera will scan the code and OCR will process the code from it. It will also provide user will more

detail related to product upon user demand.

Validation of products

Android Smart phone Application that scans the product’s ID from its Bar Code or QR Code, and sends the

product unique code to server to validate the product and also retrieve product’s related information upon user

demand.

Object character reader (OCR)

Android application will also use OBJECT CHARACTER READER (OCR). Camera of android phone will scan the

Bar/QR code and OCR will extract code from that picture. That code will be sent to server to validate the

product.

SMS via GPRS

If internet connection is available, the application directly fetches data from server, in case of no internet

connectivity, the app sends product ID via SMS, and receives the relevant information via SMS.

IVR:

For ill literate users there will the option of IVR .Interactive voice response is an innovative terminology that

permit computer to interact and respond accordingly through the use of voice. users can check the validation of

the product by receiving a voice message that whether the product is valid or not.
