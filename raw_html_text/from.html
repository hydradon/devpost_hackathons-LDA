From

Inspiration

Whenever you pursue a large goal, research shows that you will do better if you work with a partner or team. We simply strive to be better around our peers and family. You are more likely to be succesful by going on a diet or training for a marathon with a friend, why can't you do other things with the same idea?

Functionality

From for iOS provides the tools for peolple to actively and consistently engage in teamwork. It provides the ability to set alerts for other people, send short messages, and send pings, which report location and emotion through a maps display and emoji.

Tools

We use Laravel (a PHP framework) for the web backend with Redis and a mySQL database. We also use Node.js for handling status updates. We then used Swift to build the front end.

Challenges

Building a user interface (UI) that felt intuitive and easy to use was especially challenging.
