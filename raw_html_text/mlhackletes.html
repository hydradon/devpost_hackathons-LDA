Inspiration

Tbh neither of us really connected with this theme, so we wanted to come up with a way to empower women in computer science. MLHackletes came out of a late night brainstorming session after we were left feeling like the underdogs in this Hackathon due to our gender.

What it does

It allows women to create profiles to display their proudest hacking moments, meet other female hackers, and discover new hackathons!

How we built it

We built it using PHP, HTML, CSS, and Bootstrap

Challenges we ran into

It took us a while to get the Google Cloud up and running our virtual machine and learning how to use Apache and mysql.

Accomplishments that we're proud of

We got a database up and running!

What we learned

We learned how to use Google Cloud platform, Google Cloud storage, and Material design.

What's next for MLHackletes

Back to St. Louis to continue our mundane lives.
