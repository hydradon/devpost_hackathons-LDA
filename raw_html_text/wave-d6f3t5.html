• Wave

Inspiration

We thought the Chirp concept was really cool and that it could be used with sound to "hide" messages and advertisement.

What it does

It picks a spotify playlist and inserts chirps in between songs which can be picked up by the devices of people in the room/queue/line.

How I built it

I used a github project which would let me interact with spotify from the cli and then we used the chirp sdk for python-

Challenges I ran into

Getting Spotify to work properly. Had to make a hackish solution by waiting the time of the song before pausing and chirping before playing the next song. Want it to run through JS in the browser natively.

Accomplishments that I'm proud of

It's kind of working :p

What I learned

More about HTML and JavaScript, also how to use sdk/api and just in general hacking.

What's next for Wave

Sky's the limit!
