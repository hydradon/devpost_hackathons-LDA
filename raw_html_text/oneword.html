Inspired by Clarifai's machine learning API that is able to identify subject matter of photos.

It provides the user with a daily word in which they are challenged to find in their surroundings and photograph it.

It's built using the Clarify API which processes the image and compares it to the current word of the day. If the match is successful the photo is uploaded to a functional account system with Parse as a backend

Maintaining source/version control. Learning the various intricacies of iOS development in Swift (especially the optional system). Parse is slightly buggy at times.

The camera is fully functional and accurate compares the users photo to the word of the day. As well as the nicely design UI elements

iOS development is challenging.

Publishing to the iOS App Store
