Inspiration

Every time we decide to plan a trip we need to lookup through several different blogs, talk to people that already been to places, and we usually waste a lot of time.

What it does

It facilitates the planning of your trip in a fun way.

How I built it

It was built using a Node.JS server that communicates with Sabre APIs, Yelp!, Google APIs and with our Android application.

Challenges I ran into

Performance during the fetch of several API's. We fixed that pre-caching our data.

Accomplishments that I'm proud of

We almost finished the entire app. It was the most amazing app we've done in a 24-hour hackathon.

What I learned

We've learned how to use ES6 generators and new APIs

What's next for Travll

Build it into a successful app. We plan to publish in the stores.
