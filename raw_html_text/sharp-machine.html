This is our HackRPI 2016 Project, The Sharp Machine.

Samad Farooqui and Ashwin Krishna worked on the front end with a focus on HTML5, CSS, Bootstrap, and JavaScript, while Saswat Bhattacharya and Toshi Piazza will worked on the machine learning part using Pyhton and set up a server as well.

The point of this website is for people to be able to use machine learning easily, with a goal of having no previous knowledge of machine learning. The user can input their data, select an algorithm to use to process that data, and see the results of their data.
