Inspiration

Kahoot and old star shooters

What it does

It is a multiplayer game in which players connect to server via android app. The game is shown on the website. Players can move their ships using thier phones.

How we built it

The server is built using node.js which runs on dedicated VPS. The server communicates with the javascript on the website and app on smartphones.

Challenges we ran into

Definietly the most challenging was creating connection between node.js and android.

Accomplishments that we're proud of

Communication between node.js and android and explosions.

What we learned

That we have to work quickly and the duties sharing is really import in group work.

What's next for Tranquility

Diversify of gameplay adding more obstacles and bonuses.
