Equity_Initiative

Equithon 2018 project for equality

Since hearing about this hackathon, our group agreed that social media is a popular medium these days and that we could use this idea to spread inspiration and stories of perseverance.

Inspire is our idea of a social media in which users share their stories of facing adversity, whether that be finding success despite a physical disability, facing discrimination or dealing with abuse.

Users will be able to communicate with people who inspire them by searching by either category or by name. They can post comments on other's posts, reply to specific comments and much more.

With this site, we aim to spread positivity and uplift those who use this site.

See the github link for more information!

This site mainly uses Ruby on Rails for the backend and to communicate with the mysql database. Html and CSS made up the front end of this project.
