Inspiration

From past personal experience ER wait times were extremely long and we wanted to combat this pressing issue our health care system faces

What it does

By creating a database that acts as a buffer between patients and the hospitals, we eliminate the percentage of patients that seek treatment at the ER that could've gone elsewhere to be served more efficiently

How we built it

We used python to create a mock database that imitates how the program would run

Challenges we ran into

Trying to make an original program that the health care industry hasn't already made, ironing out details of consent for patients health records, trouble shooting with python, working with Invision - an unfamiliar program

Accomplishments that we're proud of

Creating the mock website using Invision, collaborative work, finishing the code on python

What we learned

How to build new relationships, how to apply technical knowledge to real world problems, never give up :)

What's next for Quick Care

Creating a real website, widening the database
