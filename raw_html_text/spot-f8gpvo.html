Inspiration

Parking is hard to find in the cities. It'd be nice if it were easier to find reliable parking from others who have private spots but no car to fill those spots.

What it does

We made an web app to make it easier to find and sell a parking spot that you own. Users can post spots or find available spots and request them. It then sends an email to fill a DocuSign contract for that spot to formalize an agreement.

How we built it

We used React.js to create a frontend and Firebase to store our data of spots and users.

Challenges we ran into

Not knowing how to use React.js

Accomplishments that we're proud of

Successfully integrating the database code with the front end code.

What we learned

React.js components and how to use Firebase.

What's next for Spot

Enhancing user experience and including more key features that we haven't had time to include.
