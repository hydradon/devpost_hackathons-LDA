Inspiration

Now days, people stare at their phones too much. We wanted a way to decrease phone usage.

What it does

The Smart Lite can read messages from various mediums. It will then process the text with sentiment analysis to generate a positivity coefficient. Then, it uses an algorithm to process the coefficient into a color of light.

How I built it

We used various languages, api's, and hardware to build the Smart Lite. Twitter -> Python -> Indico -> Serial -> Arduino -> Neopixel

Challenges I ran into

Use python to make arduino read serial data. Use twitter api.

Accomplishments that I'm proud of

Cross language integration.

What I learned

We learned every language, api, and hardware from scratch.

What's next for Smart Lite

Turning the Smart Lite into a real product that can be used by consumers.
