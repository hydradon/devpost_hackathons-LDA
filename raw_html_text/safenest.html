Inspiration Ayudar a la comunidad, el bien común

What it does Sistema de asistencia remoto para domótica mediante Amazon Echo para que las personas de la tercera edad sean autosuficientes y se encuentren seguras sin la necesidad de estar siendo vigilados

How we built it Utilizamos Apps enlazadas, como IF, Amazon Alexa, Nest, Amazon web services para agregar "skills" a "Alexa", además de enlazar un Smartwatch Pebble para el control de la temperatura

Challenges we ran into No sabemos programar... y la indisposición de los servidores de Amazon para Lambda

Accomplishments that we're proud of Logramos conectar Alexa, el Pebble, y Nest al Celular y lograr programar nuevos skills para Alexa, pero desgraciadamente AWS no se activó a tiempo y no pudimos subir las lineas de código a la base de datos.

What we learned Como enlazar diversos positivos a un asistente remoto, aprender lo "básico" de ruby y Git.

What's next for SafeNest Programar de manera correcta y con el equipo adecuado para llegar al objetivo inicial, dar un mejor servicio al usuario.
