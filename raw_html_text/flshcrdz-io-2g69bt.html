Inspiration

Deep in the midst of mid-term season, drowning in calculus formulas and biopsychology vocab, inspiration struck–– FLSHCRDZ.IO utilises IBM's Watson API to extract the most important keywords in a textbook and present them to you to help study.

What it does

FLSHCRDZ.IO first converts a PDF to a friendlier text format and then utilizes IBM's AlchemyAPI in order to extract the most relevant words from your text.

How we built it

We built it using IBM's APIs with express.js.

What we learned

We learned everything from the ground up about developing a web app. Coming in with no experience of node.js and other technologies we used, this was a huge challenge.

What's next for FLSHCRDZ.IO

Accurate flashcards with more relevancy and definitions.
