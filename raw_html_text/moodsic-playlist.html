Inspiration

Looking over the indico documentation and we wanted to do something that incorporates reading a person's mood.

What it does

It takes a picture of your face, then plays music depending on what mood it thinks you're feeling

How we built it

We used the indico API for emotion recognition.

Challenges we ran into

Playing audio files and taking a picture with the webcam using python

Accomplishments that we're proud of

Getting the webcam to work

What we learned

How to write python

What's next for Moodsic Playlist

Incorporating it with a music streaming service, building a user interface, getting better emotional recognition
