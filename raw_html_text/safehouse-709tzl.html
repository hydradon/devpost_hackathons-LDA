Inspiration

Safe house is a disaster relive website which has all the resources for every possible type of disaster in one place Flood, fire, hurricane, earthquakes, volcanoes, tornado tsunami This was done to make it user friendly in the sense that they don’t have to keep going from one website to another, they have all the resources without necessarily going to bunch of websites, sniffing through the pages to find information
This website is easy to use, it saves time and a whole lot of frustrations, plus it gets to be long term This website not only serves the victims but also creates a platform where everyone can get involved from professionals rendering their services, to those without any skill set opening their homes as mean time shelters. To people who can’t do any of the above, get to involved by donating whatever amount however small it is.

All the resources for this website was gotten from vetted sources like redcross , fema, ready.gov,
