What it does

The website provides basic information about the growing carbon emission problem. It also allows for the user to calculate the carbon footprint they leave after using different types of transportation.

How we built it

We built the framework for the website using HTML and CSS. The carbon emission calculator works through JavaScript.

Challenges we ran into

We weren't proficient in JavaScript, so we had to learn as we coded.

What's next for Carbon Emission

Our goals are to expand the calculator to calculate the carbon emission that a person has overall in one day. This would extend beyond transportation into AC, heating, lights, etc. We would then also like to get sponsors for eco-friendly options to decrease the carbon emission.
