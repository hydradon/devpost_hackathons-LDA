Inspiration

Inspired by Charles Schwab to participate. No pesky passwords which need to be remembered and can be easily hacked

What it does

Uses robust 3-way security which is not inconvenient by integrating voice biometrics, picture passwords, and 2-factor auth to give a fast and robust sign-in option for critical accounts without inconveniencing users

How we built it

Using python, Node.js and various APIs by Auth0, and few Microsoft Cognitive Services.

Challenges we ran into

Very tricky API integration and poor documentation were main hinderances

Accomplishments that we're proud of

Finishing a product that we genuinely would like to use

What we learned

A lot about the great variety of ML technologies offered by various companies but perfected by big data companies like Microsoft and Google and Amazon

What's next for PasswordLess Auth

We want to make it big and make it more robust. Will continue to work on free time and get more user input to refine it and maybe one-day selling it
