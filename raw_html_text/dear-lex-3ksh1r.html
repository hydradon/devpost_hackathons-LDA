Inspiration

Keeping a journal is hard. I always forget to journal, but being able to speak it out is a really great solution. I hope that Dear Lex will make it easier for others like me!

What it does

Dear Lex takes your voice-recorded journal entries and stores them in google drive, so that you can always look back and see your journal entry with the date attached.

How I built it

I used Javascript, DynamoDB, and the Google APIs in order to build this skill. Additionally, the Cloud Guru video by Oscar Merry was extremely helpful in learning about developing with Alexa.

Challenges I ran into

I tried a few other journal services before I using Google Drive and had a lot of difficulty with their authentication services. Additionally, it took some time to familiarize myself with the different aspects and versions of the Google Drive API.

Accomplishments that I'm proud of

I am so happy that the skill works!

What I learned

A lot. I learned a lot more about the intricacies of authentication and integrating with an API like Google Drive.

What's next for Dear Lex

Next, I want to be able to have Alexa read previous journal entries aloud. Additionally, I would like to change the way that the recording of journal entries work for greater user privacy.
