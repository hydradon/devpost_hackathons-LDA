What it does :

This App serves as the self creation portal for Connected Apps with much user friendly UI.

How we built it:

We built it using Node.js, Angular.js and jsforce

Challenges we ran into:

Salesforce Connection issues, Captcha issues

Accomplishments that we're proud of :

Achieved Basic working model for the connected app creation. (Work around for captcha : used salesforce authentication)

What we learned

Explored various tools and its usage Got familiar with Node.js, Angular.js and jsforce

What's next for Connect-O

Right now its only a Connected app creation portal, the next step would be managing the connected Apps using the portal. Improving the UI Making it more secure by including the Captcha feature. Can be enhanced to control the duplicate creation of connected Apps for similar purpose.
