Inspiration

What it does

When complete, it allows professors to share documentation with students. The students should be able to see all their courses and which paperwork they have for each one

How we built it

We built it using xcode and swift. We connected the app to an online database, allowing paperwork to be viewed by different devices

Challenges we ran into

Impossible to work with docusign API. It was also challenging to understand how to create an online database to hold the data and connect to the app. Because it was every team member's first hackathon, we also had trouble finding way to share code with each other and work independently.

Accomplishments that we're proud of

Sending and retrieving to/form database. Making the tableview of our app look nice. UI does not show aparant errors.

What we learned

We learned how to work with firebase, xcode, and swift. We also learned that, in the future, it would be bette to start with a small project and build on it, rather than attempting a challenging project right away.

What's next for Sine

Finding a way to make docusign API work, or find another API that allows for the same functionalities. Finish the platform so the app can reach a usable state
