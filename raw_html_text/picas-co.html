Inspiration

We wanted to visualize data with art -- create something beautiful while delivering useful content to a user.

What it does

Represent quantity of milk with cows in an artistic manner

How I built it

Mostly Javascript for the graphics, .CO for domain with source linked to Github.io, Node.js and Bloomberg API to retrieve data

Challenges I ran into

Learning how to use the Bloomberg API, Git, HTML, Javascript, Node.js

Accomplishments that I'm proud of

Made aesthetically pleasing cows that are representative of real world data points

What I learned

HTML, Javascript, Node.js, Git, how to incorporate APIs

What's next for Picas.co

Expanding our horizons -- working with different data sets and seeking creative ways of visualization Add animations and user-interaction for better user experience
