Inspiration

To better help individuals meet their fitness goals, eat healthier and live longer.

What it does

Assistant chat service that understands your fitness goals and provides personalised meal recommendations accordingly.

How we built it

We built it using Angular on the front-end, MongoDB for the database and Node.js for the back-end and luis.ai for Natural Language Processing.

Challenges we ran into

Getting started with Microsoft Azure Bot and picking up Angular.js and Node.js from ground zero.

Accomplishments that we're proud of

Learned Angular and Node.js from scratch.

What's next for nutri.ai

Macronutrient balancing for faster results. Filtering options for users with specific allergies, sensitivities.
