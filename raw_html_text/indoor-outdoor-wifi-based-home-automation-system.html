Inspiration

We want to build an home automation system ; simple to use; integrate as many sensors as possible.

What it does

Collect data from the outdoor node, transmit the data to the cloud. Also inform and triggers alarm if required to the interior home automation system

How we built it

we build it using Photon board. We integrated flame sensor, motion sensor, voice sensor in the inner home automation system. In the outdoor unit, we integrated distance sensor to detect the entry of a person/ car in the outdoor area and open the inner gate. Also indicate about the outside temperature and humidity to the inner node. Also we integrated a water level sensor which indicates high water level to the indoor unit and activated an alarm. Also communications happen through WiFi and also can be accessed on your phone or computer from anywhere.

Challenges we ran into

We had the challenge of communicating the outdoor node to the interior node and we implemented it through WiFi

Accomplishments that we're proud of

We are happy that we implemented a fully autonomous battery powered connected home automation system using WiFi

What we learned

We learned working on a new board, also we leaned about communicating through cloud.

What's next for Indoor-Outdoor WiFi-based Home Automation System

Deploying it to real environment which will ease mankind and make him comfortable.
