Inspiration

I m a graduate student at Lindenwood university. My main interest is in computer graphics, including image processing and mixed reality. I am currently studying VR.

What it does

The application is a game that you can play on the desktop browser or in the Oculus. I have only tested the Oculus Quest and Chrome web browser in Windows.

How I built it

This was built with Typescript and WebVR using THREE.js. I attempted to use Facebook's React 360 but it didn't work for what I was trying to accomplish here.

Challenges I ran into

  1. You can't lock the mouse cursor in the browser which makes FPS games hard to create.
  2. I had to optimize the game's physics system to run smoothly on the Oculus browser.

Accomplishments that I'm proud of

I wrote this in about two weeks and it was tough but I learned a lot about Oculus and VR in the process.

What's next for Laser Wing VR

I'm going to use this technology in my portfolio.
