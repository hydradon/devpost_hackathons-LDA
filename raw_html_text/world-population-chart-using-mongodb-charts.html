Inspiration

To Create "World Population in 2019" Dashboard using MongoDB Charts

What it does

It will show Dashboard for World population in 2019. Dashboard is build using different MongoDB Charts. These charts includes Number Text Chart, Grouped Column Chart, Circular Chart, Grouped Bar Chart, Table Text Chart etc.

How I built it

I had import Data using mongoimport command in MongoDB Atlas and Created WorldPopulation Database Build different MongoDB Charts using that Database. Embed those charts in Web page and host that in firebase

Challenges I ran into

To Customize Charts that fit in Dashboard.

Accomplishments that I'm proud of

Created Dashboard using MongoDB Charts and embed them in web page.

What I learned

MongoDB Atlas, Command line for MongoDB, MongoDB Charts
