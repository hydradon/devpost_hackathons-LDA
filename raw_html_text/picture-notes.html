Inspiration

There are times when classes don't allow for computers and also times where you forget to bring your computer. Situations like these would result in you needed to hand-write your notes then needed to retype them to a online document. We want to make the process easier for people like us.

What it does

The user uploads a photo which then we use Google Vision API to translate image to text.

How I built it

We used React and JavaScript and Google vision API to build Picture Notes

Challenges I ran into

Challenges we ran into were incorporating the API and figuring out a algorithm to correctly read boundarys for different handwriting

Accomplishments that I'm proud of

We are proud that we can create a product that had a real world application in which we can use ourselves.

What I learned

We learned shortcuts for React as well as incorporating different google APIs.

What's next for Picture Notes

We can use Natural language and Machine learning to better determine messy handwriting
