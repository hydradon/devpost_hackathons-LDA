Inspiration

We all know what it feels like to get your photo of a landmark blocked by moving tourists. Photo editing techniques already allow you to stitch multiple images together to get a clean shot, but can be tedious and not easily accessible.

What it does

Removio allows you to take a clean photo with moving objects removed. The experience is streamlined and can be done with two taps on your mobile device.

How we built it

We build an Android app which takes a series of images and then uses OpenCV to process the images.

Challenges we ran into

  • Getting the image stabilization algorithm to run in a reasonable amount of time.
  • We all had minimal Android app development experience so it took a bit of time to put the app together.

Accomplishments that we're proud of

  • It actually works

What we learned

  • Android development and OpenCV

What's next for Removio

  • Improve the user experience and further optimize the image processing algorithm
