Inspiration

We've all had moments where we go to our fridge and see a lot of food but have no ideas for what to eat. We wanted to make something which would help quicken that process by providing a list of recipes the food in the fridge can be used for!

What it does

What's In Your Fridge takes in a picture of you fridge (or your cabinet, or wherever you have food) and gives a list of possible recipes which use the pictured ingredients!

How we built it

Using Google's Cloud Vision API, we can give a list of possible objects which exist in an image!

Challenges we ran into

We found it hard to efficiently organize our time.

Accomplishments that we're proud of

We bounced back

What we learned

We learned how to implement Google's Cloud Vision API and Firebase systems.

What's next for whatsinyourfridge

Maybe we'll add functions to the recipe recommendation system!
