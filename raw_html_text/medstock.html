Inspiration

We became inspired when we heard Dr. Peter Pronovost speak at the opening ceremony of MedHacks. He spoke about how 1/3 of a nurse's time is spent looking for supplies, and not caring for the patient.

What it does

MedStock allows hospitals to cheaply set up a qr-based inventory system for all of their supplies. Hospital employees can then search for inventory and scan to check in or out supplies.

How we built it

We built it using Apple's swift, and using an apache server running PHP and SQL to send and receive data.

Challenges we ran into

We ran into challenges figuring out how to make the most effective system at a low cost for hospitals. We also ran into problems figuring out how to store the data correctly.

Accomplishments that we're proud of

We're proud that we got it up and running on time. We're also proud that the prototype is ready to be deployed in a medical setting.

What we learned

Throughout the process of making this product, we learned that huge problems don't always have to be solved with expensive solutions. We can use the tools that we already have, such as iPhones and use them for solving these problems.

What's next for MedStock

Improving the interface and adding more features, such as the ability to print off new labels directly from the app.
