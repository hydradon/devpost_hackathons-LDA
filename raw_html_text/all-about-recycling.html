Inspiration

We wanted to create a program that would benefit a whole community of people. One of the largest, outstanding problems in our worldwide community is lack of education on what waste item goes where.

We wanted it to be:

  • Informative
  • Fun
  • Engaging
  • Educational

How we built it

We used the website building platform Wix to create our website.

We used an application called Greenfoot to code our interactive game for children (used Java language)

Challenges we ran into

We faced challenges when it came to the game because there was no driver class so we weren't able to control multiple objects one by one Our methods wouldn't allow both deleting and adding of objects together so we changed the project around

Accomplishments that we're proud of

Able to complete the code, although we removed the infinite looping until game over, we are still able to have a fun an interactive game for children We have gotten the bare bones of the game done, and with a couple of changes, we will be able to have a finalized version

What we learned

We learnt about a new software that we had only a little prior experiences is We learnt in what situations GreenFoot is practical and useful We learnt more about the layout and design process of creating websites

What's next for All About Recycling

Continuing to improve our game and making more games that educate children about the environment helping raise awareness and educating people on recycling and composting
