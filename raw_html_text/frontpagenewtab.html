FrontPageNewTab

Pretty print Gmail emails. My most popular Chrome Extension to date.

Once installed, open up the email thread you want to print, and press Alt + P. That's it. You'll get a print view devoid of annoying Gmail logos, sender/receiver information, etc.

Alternatively, if you hate shortcuts, click on the orange printer button next to the address bar to print.

If you want to print a single email in a thread, open the message you want to print, click the down arrow next to the reply button, at the top-right of the message, and select Print (from Gmail Help). From v1.2 on, single emails are prettified as well.

GitHub: https://github.com/ShivanKaul/PrettyPrintGmail

Icon credits: Plainicon from www.flaticon.com .

Currently works for Mac, Windows, ChromeOS, and Linux.

NOTE: Everything is client-side Javascript i.e. there's no communication with any external machine. We know nothing about you. Read the source code if interested!
