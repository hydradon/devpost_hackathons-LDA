Inspiration

For example on house parties somebody connects his phone to the local hifi system. But most of the other party visitors don't like the playing playlist. And because of this reason we'd like to design a system to make a fair playlist for everybody.

What it does

It provides the visitors an interface to vote and add songs to the playlist.

How we built it

Using GitLab for group development and an dev aws server. The backend is running on node.js with an connection to an mongodb server. The frontend is written in Angular

Challenges we ran into

Learning Typescript and Angular

Accomplishments that we're proud of

Using our system with our friends.

What we learned

Be concentrated while developing and invest more time in learning the necessary skills.

What's next for VukeBox

Rewrite the frontend codebase and finish the backend api calls. Later deploy it as on an kubernetes cluster.
