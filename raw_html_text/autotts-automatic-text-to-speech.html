Inspiration

I've always loved Text-To-Speech (TTS), since the days of Star Trek, and wanted to make it easy to add quality TTS to mobile apps.

How it works

A simple URL request generates an MP3 file using TTS. That file can be downloaded and used as desired.

Challenges I ran into

Tricky to test for special cases, like apostrophes, quotation marks, etc.

Accomplishments that I'm proud of

High quality TTS, quickly and easily added to any web or mobile app

What I learned

Need to scale server if demand increases

What's next for AutoTTS: Automatic Text-To-Speech

Add login/password or API Key to enable authentication
