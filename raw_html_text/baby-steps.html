Inspiration

We wanted to find a way to give back to the community and we're all momma's boys so we decided to create a webpage specifically designed to provide information to mothers/parents about to start a family.

What it does

It is a web application that serves to give expectant mothers a guide to their pregnancy, post-pregnancy, and child's development. It assigns tasks to look at to the user who has logged in and keeps their progress on those tasks.

How I built it

We build this web application using HTML5, CSS3, Bootstrap, and Angular for Front-End and PHP and mySQL for Backend.

Challenges I ran into

We ran into making the the web application responsive, functionality

Accomplishments that I'm proud of

A fully functional website that runs as expected and is aesthetically pleasing.

What I learned

Integration of Front-end and back-end in a web application, making sure everything worked correctly from page to page.

What's next for Baby Steps

More information and activities for the expectant mothers providing them possible push-notifications from popular baby/mom blogs. Adding location and suggestions for places to shop/visit (Health Clinics, Doctors, stores, parks, etc.)
