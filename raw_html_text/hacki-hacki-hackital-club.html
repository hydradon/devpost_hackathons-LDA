Inspiration

The inspiration for this project was our experience as volunteers at Hackital

What it does

The project is an interactive visual novel that you can play for fun.

How we built it

We built the project using Renpy link which is a visual novel creator written in python. It was built in our free time when not helping other people at the registration test.

Challenges we ran into

The main challenge that we ran into was time, as we were busy helping others. We also had a few issues with handing off code between shifts.

Accomplishments that we're proud of

We were happy to be able to add real images from Hackital this year into our game to make the experience more specific and realistic.

What we learned

We learned a new game engine for creating visual novels. We also learned that visual novels are an effective way to communicate a message and are quicker to develop than a full video game.

What's next for Hacki Hacki Hackital Club

Update for next year! Make more different visual novels!
