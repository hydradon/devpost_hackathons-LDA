Inspiration

I wanted to explore free unlimited storage alternatives.

What it does

It's a desktop application that compresses, encrypts, and uploads files to Github and fetches it for download. Github offers unlimited storage on their repositories (with small caveats), which can be utilized to provide unlimited free file storage.

How I built it

Used Electron for the desktop application. This app forked processes that would handle local git repositories for each file. Files were pushed to git on uploads and fetched + checked out on downloads.

Challenges I ran into

It's a solution that hasn't been explored before.

Accomplishments that I'm proud of

It works very well and uploads very fast.

What I learned

Desktop application level permissions enable you to do so much
