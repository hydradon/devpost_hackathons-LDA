Inspiration

We believe in the future, people will try to spend more time together and to help each other. This is why we've been thinking about a way to bring people together and help each other easily.

What it does

It link people who need help with other who can help them. It s a responsive web app available on laptop and mobile phones

How we built it

We built our code from scratch using angular 2 and typescript

Challenges we ran into

Bugs, use og google maps, design with css...

Accomplishments that we're proud of

The final result is good for people not used to use css

What we learned

we learned how to develop an application with an important features in a less time How to focus on the important, how to mock fastly an idea

What's next for HelpIt

we might launch it if people like it
