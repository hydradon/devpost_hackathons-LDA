Inspiration

Geocaching

What it does

Allows users to create scavenger hunts, hide QR codes in the real world for people to try and find.

How we built it

PHP REST server backend, MySQL database, JQuery client backend to fetch and send to the REST server, HTML5/CSS3 with materialize for the front end.

Challenges we ran into

None of us had a clear idea of what we wanted until more than half way through.

Accomplishments that we're proud of

Pretty neat REST architecture, clean code for a hackathon.

What we learned

Sleep is underrated

What's next for Go Dig It

We couldn't finish it in time to actually have anything to demonstrate, so we might finish it later on and see what happens.
