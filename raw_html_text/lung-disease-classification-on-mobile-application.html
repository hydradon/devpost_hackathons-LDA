Inspiration

Reading and diagnosing chest x-ray images may be a relatively simple task for radiologists but, in fact, it is a complex reasoning problem which often requires careful observation and knowledge of anatomical principles, physiology and pathology.

What it does

Platform for accommodate doctors to diagnose and ask for opinion on x-ray image any time, anywhere.

How we built it

Nodejs Tensorflow

Challenges we ran into

Accuracy of the model is not so satisfying at the moment

Accomplishments that we're proud of

Modify InceptionV3, a pretrained neural network widely use for one class classification to predict multiclass classification problem

What we learned

Connect users and machine learning logic on a platform

What's next

  • Cover more diseases
  • Integrate with health record

Built with
