Inspiration

Every company runs there business off spreadsheets. It's great simple database for everyday users. Sheety app allows you to upload a spreadsheet and create an app from the data.

What it does

Create Data Filtering Apps, Shopping Apps,

How I built it

I broke the app up into an ETL module, a transformer and reducer, and a few user interface components.

Challenges I ran into

It's hard to write a generic general purpose apps on top of generic spreadsheets

Accomplishments that I'm proud of

It works! And I use it in my daily life

What I learned

You can upload and tranform excel spreadsheets all client side in the browser. No SERVER NEEDED

What's next for Sheety App

Create a store from an excel spreadsheet and pay with Lightning
