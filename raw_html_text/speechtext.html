Inspiration

This was the first hackathon for everyone in our group, so we decided to look for a simple solution to a common problem. We thought to ourselves "how might we improve the lives of college students" and decided to improve the efficiency of study sessions

What it does

We created an application that uses sound to refocus groups when conversation strays from the subject of choice.

How we built it

We used Python and implemented Google text-to-speech and Google NPL. We used Python's GUI software, TkInter

Challenges we ran into

Had to navigate Google's APIs. A number of sections of code did not work as Google said.

Accomplishments that we're proud of

Used text-to-speech and Natural Language Processing and built a GUI at our first hackathon!

What we learned

How to use Python. Google APIs. GUI design.

What's next for SpeechText

Use in business meetings and for those with ADD/ADHD. Listen for abusive language in the workplace.
