Inspiration

Making the campus more accessible for differently abled people

What it does

Locates user and points out accessible entrances and exits. Also displays paths.

How I built it

We mostly used javascript and the Google Maps API

Challenges I ran into

Understanding/Implementing the API, getting current location, combining code, working with JSON and KMZ files

Accomplishments that I'm proud of

Trying really hard, not giving up, learning tools we'd never really used and being able to make something

What I learned

Making a functional program takes lots of time, patience, and hard work
