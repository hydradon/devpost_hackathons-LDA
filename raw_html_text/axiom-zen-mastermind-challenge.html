What it does

This API implement a multiplayer Mastermind game when a user can create a challenge and wait for another user to try solve it.

How we built it

I've built this API using the new Ruby on Rails 5 as framework and PostgreSQL as database. I've tried develop this challenge using NodeJS, but I don't have expertise in it and I changed to RoR5. The NodeJS attempt repository is on the links below.
