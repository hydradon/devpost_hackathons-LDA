Inspiration

This MASH game is inspired by the "Rewind" theme of the hackathon, as many of us have fond memories of playing the game as children.

What it does

MASH is a game that is intended to predict one's future. This game consists of several categories, each of which represent a part of a future life, including Home, Partner, Number of Kids, Job, and Salary. The user will be presented with each category, one at a time, and asked to enter 4 possible outcomes for them. After the user has completed their personalization, they will be given a random number and the game will predict their future!

How we built it

We built this game as a C++ console application.

Challenges we ran into

Since the game as it is traditionally played entails eliminating every nth choice in a list of all choices among all categories, it was a challenge creating an algorithm to mimic this process. It was also a challenge to select the last choice in each category for the future prediction at the end.

Accomplishments that we're proud of

We were able to figure out the process of elimination and selection of the final choices from each category.

What we learned

How to work on a collaborative project remotely using Github.
