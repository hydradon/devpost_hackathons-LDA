Inspiration

Our goal is to help students to connect with professionals working in the vocational fields they are interested in. Professionals who are willingly available to mentor students can register through the website and answer questions students may have regarding their professional career. Our service aims to help these students find mentors.

What it does

Students can type in a search query and the query becomes a tag used to access the profiles of professionals within the students’ field of interest.

How I built it

We used django which is a Python framework and using sqlite as the database source.

Challenges I ran into

Initially tried using MySQL but forgot the root password so we switched to sqlite for simplicity and ease.
