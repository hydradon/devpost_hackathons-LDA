Inspiration

Using Python to Code by Voice

A Vision of Coding, Without Opening your Eyes

What it does

*A general purpose voice user interface for IntelliJ IDEA. Some use cases: blind and RSI users.

How we built it

Kotlin, Gradle, IntelliJ Platform

Challenges we ran into

Speech is hard. Fixed grammars don't work well.

Accomplishments that we're proud of

Accessibility, productivity, usability.

What we learned

Kotlin, AWS Java API. Need to make speech commands more discoverable.

What's next for Idear

Support for more intents and broader actions.
