Purpose of this project

Looking at just another map app during walking spoils travel experience. Therefore, our purpose of this project is to make a service that traveler can focus on the local landscape with an efficient, automatic, characteristic guide.

What it does

Dochu navi is the AR app that guide you to the destination when you travel in Japan. A virtual samurai character guide you to the destination. Virtual overhead screen displays PR videos related to the destination (this screen is for the moment user don’t walk such as waiting for a traffic light).

How to use

After launch the app, type a destination name. When you confirm the destination in map view, tap "Let's go" button. After that, you can see a virtual samurai. Tapping display triggers the start of the guidance. You can watch PR video related to the destination with this overhead virtual screen (just look up!).

What's next for Dochu Navi

We will add the function of travel plan recommendation to Dochu navi, using Amazon Machine Learning or other service in AWS.
