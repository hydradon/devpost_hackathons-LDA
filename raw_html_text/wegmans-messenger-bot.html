Our inspiration

Mission:

Our Facebook Messenger bot allows you to find a local wegmans store, deals on your favorite items, as well as recipe ideas!

What We Used

We built our bot using the Wegmans API, the Facebook API, as well as the Google Cloud Platform. We used Google Cloud Platform to deploy our app using Facebook Messenger webhooks integrated Google App Engine. Also, we used Google Cloud Platform's DialogFlow SDK to use machine learning and natural language processing to recognize and understand the intent of the user's messages. Therefore, we can have a personalized customer experience for each user.

Challenges

Some challenges that we ran into were gathering the data from the Facebook bot so that we could use it with the Wegmans API to provide the information to the user.

Accomplishments that we're proud of

We are proud to have been able to use so many tools to create a quality product, as well as build something to help people in their every day lives.

What we learned

We learned that it is important to take problems one step at a time, as well as learn that things end up coming together in the end.
