Inspiration

We were frustrated with the Student Link. It's so ugly and gives limited options for filtering. We wanted to make a better version for searching for classes.

What it does

Allows user to choose filters and displays classes. Friendly UI. Working on adding features like building schedule

How I built it

Python scraper to scrape all course info that stores course info into mongodb db. TypeScript webserver with node that queries db based on request. TypeScript frontend using react.

Challenges I ran into

Hosting was very hard. We got the backend up and running but struggling with hosting frontend at the moment.

Accomplishments that I'm proud of

Happy to have a somewhat polished and useful project. It provides an actual good UX

What I learned

Learned about TypeScript, better understanding of web stack, gained skilled with GCP and AWS

What's next for BU courses

Host frontend and adding more services.
