• Delivery on MapQuest

Inspiration

The display of restaurants information on delivery.com is not intuitive. I need to a better way to review all the restaurants around me.

What it does

It will display the restaurant search result on map provided by MapQuest.

How I built it

I built a web application that provides end user search function, and then I call delivery API based on address information provided by user. And then render the result on map.

Challenges I ran into

Learn how to use delivery.com and mapquest api. Especially, some result from API is not what I expected.

Accomplishments that I'm proud of

I finally finished the demo application.

What I learned

I learned how to use delivery.com API and mapquestion API.

What's next for Delivery on MapQuest

I want to continue my application and add more functionalities into it.
