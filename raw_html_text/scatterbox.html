Inspiration

What it does

Creates an abstract representation of given sentences using NLP to determine word salience (this determines the size of the box) and the word topic (this determines the box color)

How we built it

We built most of it using anvil

Challenges we ran into

Getting used to anvil. The complexity of wordnet in the NLTK library

Accomplishments that we're proud of

We finished it and accomplished our vision

What we learned

How to easily create a full webapp
