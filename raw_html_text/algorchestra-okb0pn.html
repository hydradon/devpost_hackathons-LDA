Inspiration

An algorave is an event where people dance to music generated from algorithms, often using live coding techniques, and short for "algorithmic rave."

What it does

Algoraves are normally coded on a single machine. We build a tool to make it real-time, collaborative and distributed for multiple conductors and musicians.

How we built it

  • Pusher API to synchronize all clients
  • Lissajous lib for sound creation
  • ReactJS for frontend

Challenges we ran into

Realtime sync of sound

What we learned

  • Pusher API
  • ReactJS

What's next for Algorchestra

  • More diverse instruments
  • Multiple track channels
