Inspiration

We were inspired to develop such a program by the recent craze around bitcoin and digital currencies alike.

What it does

The program retrieves news articles from the past 10-days and performs a sentiment analysis using Google Cloud Platform's Natural Language API. The application is meant to act as an additional resource to understanding the digital currency market and to evaluating the confidence in a particular currency.

How we built it

The program incorporates the Google Cloud Platform Natural Language API and the Google News API. It is put together using the Express.js framework. Additional market data is retrieved via web-scraping the coin market-cap website. The data is presented in a graph drawn using Plotly.js.

Challenges we ran into

We had to learn how to integrate the Google Cloud Platform and the Google News API. We had several errors related to programming asynchronously which were frustrating to debug.

Accomplishments that we're proud of

We are particularly proud of being able to produce a finished product for our first Hackathon.

What we learned

We learned how to take advantage of the services of the Google Cloud Platform.

What's next for Crypto-Graph

We need to figure out how to perform regressions on the plotted data to determine trends in the public's general sentiment towards cryptocurrencies. We could work with more historical data and incorporate data from additional sources, such as Twitter, to provide a more accurate determination of the people's attitude.
