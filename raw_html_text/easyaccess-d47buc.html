Inspiration

Many times at the airport, my grandma faced issues with wheelchair assistance. She would wait on the phone for 30-45 minutes to get a representative to help her, because of her leg problems. Many elderly people face similar obstacles in today's society.

What it does

EasyAccess is a mobile application which allows users to have an insight on places that are efficiently accessible.

How we built it

Using React-native platform, we designed the interface and most of the front-end coding. We used Google Firebase for data authentication and database management. We also used Google's DialogFlow and ML APIs for the speech assistant part of our project.

Challenges we ran into

The wifi was really on and off which was very annoying, because we would have to sign back into the WiFi every time our computer turned off. This would take up a lot of time at crucial moments. There were many unforced errors in our code that sometimes took sometime to debug, but other than that the project ran pretty smoothly.

Accomplishments that we're proud of:

Our user interface is very clean and user friendly. It also contains many features that are unique solutions to obstacles in today's society.

What we learned:

We became more efficient in using GitHub as well as Firebase. We also looked into DialogFlow and other voice assistant platforms in order to incorporate it into the app. There were many errors we experienced in our code so by debugging those errors we learned how to deal with it when they would pop up again.

What's next for EasyAccess:

Through active sponsors we aspire to add hardware to strengthen the structure of our app, like easy access wheelchairs that are connected to the app when needed. Many countries in the world are facing this issue where easy accessible assistance is hard to find, because they expect physical labor force to go and aid the people in need, but the problem with that is that the people who need help outweigh the labor force. So we plan to expand our resources around the world fighting this issue.
