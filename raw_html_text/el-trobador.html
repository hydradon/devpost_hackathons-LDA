Inspiration

As university students, we usually find it difficult to get to know a teacher that wants to supervise our Final Degree Project or feel lost when deciding which topic we want to work on. For that reason we found interesting to implement a platform to make this process much easier for both students and teachers.

What it does

It works like a marketplace of Final Degree Projects where both Students and Teachers to propose and accept them. Users can modify proposals, watch it's changes historial, filter by tags and subscribe to several topics.

How we built it

A backend developed with Node.js and sqlite3 and a frontend with html5, javascript and css3.

Challenges we ran into

Learning new technologies and following a limited temporal planification. Complex design for a big system
