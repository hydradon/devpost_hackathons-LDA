Inspiration 

  Prior research project in which Hieu researched the Blue Dragon Foundation

What it does 

  The website provides information and contact information for the Blue Dragon Foundation, an organization that helps poor or abused children in Vietnam.

How we built it 

We used Wix development platform to build the website, and used styling elements on the platform to customize it.

Challenges we ran into 

   There were some technical issues in saving the website, or with the internet connection.  In addition, at times the tools didn't work in the exact ways we meant them to such as when we tried to use columns for the background on the main page.

Accomplishments that we're proud of 

 Accomplishments we're proud of include modifying the layout of the pages, and creating a submission form.

What we learned 

 We learned about some of the hardships faced by Vietnamese children, and how to use the Wix development platform.

What's next for Blue Dragon website 

 Adding more information about future projects, more pages to better separate the information, and inclusion of success stories.
