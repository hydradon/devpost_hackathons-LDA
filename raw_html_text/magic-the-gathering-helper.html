Inspiration

It's my first hackathon, so I wanted to make something that had to do with an interest of mine and something that is doable.

What it does

It provides a mobile interface to search for magic cards and get information on them. The idea was to also have it be able to keep track of decks (collections of cards) but I ran low on time.

How I built it

I just drew a few pictures of what I thought the app should look like and how it should work, then tried to recreate it. The logic was pretty basic, so that was no trouble.

Challenges I ran into

SWIFT DOESN'T LET YOU SUBSCRIPT STRINGS AND IT MAKES BE SO SAD. Also, the storyboard is nice, but rather glitchy, and using constraints has a learning curve. I ended up spending a bit more time figuring out the user interface than coding.

Accomplishments that I'm proud of

I learned Swift kind of.

What I learned

I learned Swift and how to find APIs and resources to help me.

What's next for Magic: The Gathering Helper

I'll probably implement the deck builder.
