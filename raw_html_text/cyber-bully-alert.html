What it does

It notifies the user of potential cyberbullies on the respective website, bringing awareness to the problem.

Challenges we ran into

We tried using Google Cloud to store the data in a database but the API was too complicated to understand in such a short time.

What we learned

We learned the basics of creating a browser Add-On and some of the features provided in Google Cloud.

What's next for Cyber Bully Alert

--Enabling google cloud interaction --Using machine learning have an algorithm that checks the images of cyberbullying instances and decides whether it is valid. --Implement an algorithm that notifies the user when they are interacting with a cyberbully in a certain website.
