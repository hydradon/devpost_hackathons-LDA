Inspiration

To quantify emotion through machine learning models and apply it in a fun way

What it does

"You Said What?!" is a web app that lets you know what haters are saying about you. It scrapes Twitter for tweets directed to you. These tweets are analyzed using Google Cloud's API for Natural Language Processing to analyze the sentiment of a tweet. Any "moderately negative" charged tweet will be displayed.

How I built it

I built it by leveraging Google Cloud's Natural Language Processing API

Challenges I ran into

No previous experience with this technology led to getting stuck often

Accomplishments that I'm proud of

I have wanted to do something like this for a long time but never dare and today I did.

What I learned

I learned how to use Google Cloud's APIs

What's next for You Said What?!

Increase Computation to give results faster. Do a more extensive search of tweets
