We wanted to create an interface that permits hackathon attendees and staff to read and post relevant current information about a hackathon. Some of the features of this interface will be: 1) Users can ask questions to mentors 2) Users can see what music is currently playing 3) Users can see how much time is remaining in the hackathon 4) Users can see the agenda on the hackathon.
