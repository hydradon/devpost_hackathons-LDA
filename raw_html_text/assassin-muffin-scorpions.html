Inspiration

The marching band and all of its ISUCF'V'MB glory. The game Assassins.

How it works

Hosting website that allows users to join the game of Assassins in order to monitor their status ingame. There is also a database backend to hold the information and allow for a manager to make admin-level decisions about the game.

Challenges I ran into

Started with Google Sites, but that fell through pretty hard. Implementing the firebase api into the program since it has a different system than anything else.

Accomplishments that I'm proud of

This hackathon.

What I learned

Firebase, javascript, Java, html, how to work with different programs together

What's next for Assassin Muffin Scorpions

To make Java programs work together easily in the future, and to allow for people to use our application.
