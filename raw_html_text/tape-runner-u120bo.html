Inspiration

Initially we wanted to make a car that would go to the end and find a rubiks cube, but after considering the time we came up with the tape runner idea.

What it does

It follows a tape path that is laid down, with only 90 degree turns.

How we built it

We had a webcam on the front with the edison car, that used image recognition and normal commands that ran the edison

Challenges we ran into

Trying to get the webcam to register images

Accomplishments that we're proud of

Creating 3 http servers, and auto correction, as well as image recognition

What we learned

Image recognition
