Inspiration

Climate change and Animal habitat loss.

What it does

It provides eco-friendly flight options. And provides games for the most eco-friendly users.

How we built it

We used the skyscanner api and coupled it with a Django database to create a user system and a point system with the leader board. We also used Twilio for 2 factor authentication and security.

Challenges we ran into

Splitting the work load was the biggest problem.

What's next for Plant the Planet

Better Graphics and more time to polish the website.
