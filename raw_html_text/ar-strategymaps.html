Inspiration

We wanted to create 3D map that visualizes the biomes and heights for a military use.

What it does

Map AR detects images that is stored in the database and projects it to 3D objects to the user. User can move around the images to customize map.

How We built it

We used Vuforia and Unity to scan image and project it as a 3D object.
