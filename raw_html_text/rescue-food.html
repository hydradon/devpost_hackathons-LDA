Inspiration - At school cafeteria, we saw a lot of food wastage. This got us thinking that we need to solve this problem.

What it does - We created a webapp, which upon taking a leftover food picture, provides waste estimates in $ and national resources.

How we built it - We used the standard HTML5, CSS3, JS, and PHP to develop the webapp. We used the Clarifai API for the image processing.

Challenges we ran into - We had to research how to execute a cURL from PHP as none of us knew how to do it.

Accomplishments that we're proud of - Its an easy to use product that will help analyze leftover food and tells you the wastage in terms of dollars and national resources like labor, fuel, and water. With this we expect to sensitize students to reduce the wastage.

What we learned - We learned how to execute cURL in PHP, how to take a picture using the webcam, how to code in Javascript, and how to run a local apache server.

What's next for Rescue Food - In the future, we want to expand the functionalities by adding user accounts and history to provide trends and deeper analytics over time. Create an iOS and Android app too.
