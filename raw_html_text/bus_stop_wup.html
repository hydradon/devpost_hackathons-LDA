Inspiration

What could we do with a huge screen?

What it does?

Main functions: – show timetable of the transport for the current bus stop; — show city map with other bus stops and information about these bus stops; — show current time and weather; — show advertisement and important news for tourist and locals; — provide "stop button", which future passenger has to press to indicate that he needs the closest bus to stop. The provided light show will make sure that you'll never miss a buss ever again, because the bus driver didn't see you!

How I built it

The application is built on Windows Universal Platform. Two WebView elements are used to display data from web pages. The reittiopas.fi service was used to provide information on the local bus stop (with map and transoprt information). Weather data is fetched from http://openweathermap.org/ and served to the application through an Azure instance.

Challenges I ran into

We didn't have much experience with web development so in order to be able to create something concrete we ended up creating a native Windows application with web controls.
