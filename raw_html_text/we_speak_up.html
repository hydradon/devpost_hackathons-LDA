Inspiration

Due to recent events surrounding survivors of sexual abuse, we wanted to provide a safe, non-judgmental space for survivors of abuse to have access to resources for helping survivors and a place to speak out.

What it does

This application allows you to submit posts and comment on other posts. They can also edit and delete posts and comments. It allows users to be completely anonymous.

How we built it

Ruby-on-Rails backend with React frontend. Redux middleware is used to store data. PosgreSQL is used for the database.

Challenges we ran into

Debugging certain problems took some time, but we eventually figured those out after working through all the steps our code followed.

Accomplishments that we're proud of

Building the application in such a short time and fully deploying it online

What we learned

A lot about the fundamentals of React, an intro to Redux, and how to use them together. Also some review on Ruby-on-Rails. We also reviewed basic HTML/CSS.

What's next for we_speak_up

We would like to add some more functionality later, possibly an authentication system.
