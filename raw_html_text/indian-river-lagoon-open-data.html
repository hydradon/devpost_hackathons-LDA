Inspiration

Inspired by https://data.cityoforlando.net/ this project intends to provide a directory for searching for different datasets organized by onelagoon.org's Path Forward.

What it does

Provides a searchable directory of datasets.

How we built it

Hosted in github pages, running vue.js

What's next for Indian River Lagoon Open Data

By design, this project is a github repo. For others to add their datasets or recommend data sets, contributors can simply make a pull request, or for non-technical users, create an issue ticket.
