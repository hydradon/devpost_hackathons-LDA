Inspiration

Going to a large university such as Purdue or University of Illinois, you often realize you cannot get help at office hours due to the large amounts of people present. We need an open source tutoring locator where students can find tutoring sessions and host their own.

What it does

HelpTag allows you to see a map of nearby tutoring locations and and their times, so that a student can easily incorporate times to get help into their schedules. There is an option for a "student" and a "teacher" role so that a user can either create student or teacher hosted workshops. We also incorporated a chat feature so that users can communicate with each other one-on-one.

How we built it

We used a React Native frontend and a Firebase backend.

None of us had ever worked with React before, so this whole project was an experience for us. Also, we encountered some authentication problems with Firebase for which the only solution we could find was restructuring the entire database.

Accomplishments that we're proud of

We're proud of actually making something we can see for once, as opposed to working on a machine learning problem like we have at past hackathons.

What we learned

We learned the hard way to not scale the project too fast when we began feeling rushed midway through the hackathon.

What's next for HelpTag

We'd like to streamline the "chat" and "add workshop" features.
