Inspiration

The main inspiration for this game was implement physics engine. We wanted to challenge ourselves and build something fun with HTML5 Canvas and Matter.js .

What it does

Provides a fun experience for the user.

How we built it

This game was developed by using Visual Studio Code as the primary IDE. HTML5 Canvas and Matter.js were used for rendering and adding functionalities to our game.

Challenges we ran into

Implementing the library as per our idea, managing our schedule to work on this project.

Accomplishments that we're proud of

Every developer must have dreamt of creating a game at some point in their life. We're most proud that we came through with a fun game for all ages. We were also able to collaborate and work seamlessly to successfully develop this game.

What we learned

Matter.js was a fairly new territory for us. We learnt to use the components and features provided by this library and integrate it into the HTML5 Canvas.

What's next for Triplets

Continuous development to the current version of the game by adding more levels and integrating image sprites into the game, and also enhancing the overall experience of Triplets .
