Inspiration

Most of our impetus came from following through on creating something based on a pun made in a hazy mindset. Moore’s Law. Morse Law. Get it? Beyond that, our inspiration is a lifetime of disappointment.

What it does

It converts Kentucky state law into BOTH the typed and auditory forms of Morse Code. Groundbreaking.

How we built it

We leveraged the Flask framework to create the service as a webapp. The conversion itself is handled by: a) A dual set of javascript and python Morse Code conversion libraries. b) A “pdf to plaintext” python library.

Challenges we ran into

JavaScript machine broke. Often.

Accomplishments that we're proud of

We have the sheer force of will to follow through on something this unnecessary and mildly frustrating.

What we learned

Morse Code is far less efficient than reading.

What's next for MorseLaw

We’re probably going to implement a feature which allows users to modulate the speed of the audio playback. Because why not. Also, it would be nice to scale it to pull from the user’s state legislature, Kentucky or otherwise.
