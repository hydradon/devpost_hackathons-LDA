Inspiration

Our inspiration was our own experience with planning for college and attending ACC as dual credit students.

What it does

What our website does in is inform the viewer with information about the 60x30 TX plan, different colleges (associates, bachelors, master's, etc..), the pros and cons of attending college, and information given on different educational programs (that may be best suited for you).

How I built it

We built our website by using Thimble, html and css.

Challenges I ran into

Challenges we ran into included making the quiz using 4screens.net, a website we were not familiar with, and adding the quiz into the code while still making it functional.

Accomplishments that I'm proud of

We are proud of successfully adding the quiz into the code. We are also proud of our whole code being functional, and working as a group.

What I learned

We learned how to add the quiz into the code and how to make a successful pitch to an audience.

What's next for How to Select Your Education

We can learn how to add a chatroom onto our website.
