Inspiration

We wanted a centralized platform to organize study groups and ultimately improve habits

What it does

Allows users to input times, dates, and material for others to join in and study together

How we built it

We created a front-end website in HTML, CSS, and Javascript. We used multiple pages for login/register and incorporated bootstrap's CSS classes.

Challenges we ran into

Many of us never created a full application so setting up GitHub and learning how to contribute was a big task. It was also difficult because not all of us knew how to use HTML, CSS and Javascript

Accomplishments that we're proud of

Everyone learned much more about the process to create something useful. After a lot of back and forth for ideas and troubleshooting, we proud that we were able to produce something together.

What we learned

We learned that GitHub has a huge learning curve and it's really difficult to learn Javascript on a four-hour car ride. Ultimately, each member of the team was about to confidently contribute through the skills they learned

What's next for HackPro

We need to create a backend and update much of the front end. We also need to incorporate a full database. Everyone is really excited to make this project the best it can be.
