Inspiration

A while ago I was in the mall of a cinema and I realized that people were forming a line to pay for parking in that very same moment we discovered that there were more people waiting to pay for parking because the machine solo acepta monedas so we decided to make an application and an adapter for any kind of coin machine so that payments are made wirelessly

What it does

Coinless allows payments wirelessly to any type of machine that is payment currency, the customer simply install the application on your phone and via Bluetooth 4 smartphone reads the nearest machine and send the money to it

How I built it

We used cross compilation tools to allow one single building and then we are able to deploy to iOS and android. The hardware part is an iBeacon device which is the key part in letting the user use the wireless method

Challenges I ran into

How to convert all coin machines into digital payment machines, the hardware part was the trickiest

Accomplishments that I'm proud of

The UX and UI are very easy to use and understand

What I learned

I learnt a lot about the payments done in parking lots and malls

What's next for Coinless

Establish a supply chain with nearest parking lots
