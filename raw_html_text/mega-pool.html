Inspiration

Brainstorming ideas for BrickHack

What it does

It's just a fun game which combines pool and first person shooting

How we built it

We built it on Unity throughout the course of the BrickHack, starting with basic movement and terrain and moving onto the complicated controls.

Challenges we ran into

We had troubles with implementing the rules of pool during the time period, so we decided to stick with the basics and polish the core gameplay.

Accomplishments that we're proud of

This is our first actual 3D game and it was a great experience, learning from the ground up and seeing things come together, and the gameplay being genuinely fun. Also, we finally got controller support right after a lot of time spent on it.

What we learned

There is just so much stuff we learned. Besides the obvious coding and design experience, we also learnt how to manage our expectations and get the best product out in the limited amount of time we had.

What's next for Mega Pool

We definitely plan to polish the game more and add more rules and features to it. The game has the potential to be really fun and it is simple enough that it could be run on a large diversity of platforms.
