Inspiration

Learn how to use Trilio, as I have had little experiece working with API's

What it does

BAsic driectional control of a Rasberry PI based Zumo Robot. The robot was previosly constructed for my cyrrent MSC robotics project, a crop row traversing robot.. THe robot can be controlled via a Twilio SMS number by texting simple commands, eg l = left, b = backward.

How I built it

Python - Programing language for Rasberry Pi gpioZero - Motor control Flask - Rasberry Pi Python web application server Ngrok - Network tunnel for Flask server to provide public URL Twilio - Recive text directions from text messages to registerd number. Serve commands to robot via webhooks and PUSH requests.

Challenges I ran into

Most of 24H taken up with learning how to use Twilio to send SMS, first with JavaScript, then python and Twilio grahical programing studio.

Accomplishments that I'm proud of

Development of a Flask server capable of pocessing prapmmeters for a POST request.

What I learned

Learnt how to use Ngrok to allow a Rasberry Pi or local desktop server to be accessed via a public URL for testing perposes.
