• On The Block

Inspiration

Poor property purchase experiences with money held in escrow for over a year.

What it does

Ethereum Smart Contract Backed Escrow for Property Purchases.

How we built it

Solidity Smart Contract, Node.JS Backend, Angular.JS Frontend

Challenges we ran into

Debugging issues with Web3.JS server not running correctly. Scrapping data from government websites regarding property ownership.

Accomplishments that we're proud of

Creating a project with an actual value proposition and could be developed into an actual business. Being on the cutting edge of technology with blockchain and smart contracts.

What we learned

How to incorporate smart contracts into a web application.

What's next for on-the-block

Commercialization.
