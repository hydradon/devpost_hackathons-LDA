Inspiration

Our inspiration to create MemeMote began with the desire to tinker with Google facial recognition technology. We then wanted to create something that can help bring communities together.

What it does

MemeMote uses Google's facial recognition technology to detect the emotion that the user experiencing whilst viewing a meme. That data is then used to rate the meme, giving it a point in a, joyful, neutral, angry, sorrowful, or surprised emotion.

How we built it

We built it using the MERN stack which incorporates Google's facial recognition API.

Challenges we ran into

Learning a new API and integrating the many technologies that made it possible.

Accomplishments that we're proud of

We're very proud that our application utilizes Google's technologies very smoothly. We're also incredibly happy with the end result; it achieves what we set out to do.

What we learned

We gained more experience using MERN stack as well as reading and researching technology documentation.

What's next for MemeMote

We set up a year long domain for it and so we will continue to make more improvements and add more functionality.
