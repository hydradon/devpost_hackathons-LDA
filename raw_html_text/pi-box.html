Inspiration

Relying on Dropbox to access files can be hassle. Whether Dropbox is undergoing maintenance or you cannot access the Internet, we wanted a cloud-like solution for this problem.

What it does

The RaspberryPi is able to copy all files from Dropbox, so you can accesss them if needed. Basically a backup of Dropbox.

How we built it

Built using Dropbox API/SDK, Python, Supervisor.

Challenges we ran into

Everything!

Accomplishments that we're proud of

Overcoming the lack of thorough documentation of the DropBox API. An unfamiliarity with Python made the project even more challenging, but just as fun.

What we learned

We learned many important concepts when writing programs in Python. We also learned how to directly work with an SDK or API

What's next for Pi Box

We will create documentation with a configuration file, and fix the hosting by Supervisor as a Daemon.
