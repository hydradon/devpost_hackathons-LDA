Inspiration

My inspiration comes from the previous disaster we just had. One of the problems was getting and receiving help and having a reliable source to connect to.

What it does

It allows for users to connect and request for essential stuff such as food, water, and shelter and allows user to give help to the users who needs it by sharing phone number and location.

How I built it

I use React Native for the mobile application with an ExpressJS backend with MongoDB.

Challenges I ran into

I ran into problems with React Native and some deprecated libraries.

Accomplishments that I'm proud of

This is my first solo Hackathon event.

What I learned

I learned how to develop on the backend a bit more.

What's next for WeUnite

A better UI.
