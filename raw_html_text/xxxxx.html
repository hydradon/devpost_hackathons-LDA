Inspiration

We had a desire to make it possible to live together with Masamune Date, a popular character in the smartphon game “Ikemen Sengoku: Romances Across Time” developed by our company.

What it does

We feel that the change of the background in the four seasons and the time zone is effective for giving the impression that Masamune Date and the user live together.

How we built it

Until now, it was sufficient to control a single-way transaction, but this time there were two processing origins: “Voice input on Alexa” and “Touch interface on WEB”, so control was not done at the same time. We devised it to do so.

Challenges we ran into

We had a hard time solving the memory leak problem.

Accomplishments that we're proud of

We are impressed that Alexa has been able to create a 2D moving Masamune Date, which has not yet been implemented in the “Ikemen Sengoku: Romances Across Time” smartphone game developed by our company.

What we learned
