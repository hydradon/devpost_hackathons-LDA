Inspiration

Having two of our team member being hurdled by their stutters. We wanted to make their lives and the lives for people who face the same problem easier. Specially when it comes to presenting and public speaking.

What it does

It records an audio sample from the user and then uses machine learning to analyze syllables on which the user stutters. From there, it consumes different text highlighting all the words with high probability of stuttering and offers synonyms and replacements for it.

How we built it

We built all of the front end and most of the back end using node.js. We also used python when it came to implementing the machine learning part.

Challenges we ran into

Wiring different frameworks together(python and node). Sending different types of data over http requests.

Accomplishments that we're proud of

Integrating machine learning into our project. Learning new frameworks (some of us didn't know node before this weekend) finding a way to post blobs over http

What we learned

Using node.js using different python libraries.

What's next for stutterio

Improve on the machine learning engine and deliver a better service.
