Inspiration

I've always though that sometimes you need to share your code with a team member to find a better solution together. But I as many others want to work in the environment I love and know to stay as productive as possible. Sockscode is here to help because when it comes to pair programming it should be as easy as wear a pair of socks.

What it does

Sockscode is a set of extensions(add-ins) for your favorite editors that enables you and your buddies to create textual content together. It's better suited for programming scenarios but works for everything else too. Currently, only these platforms are supported: Web: https://sockscode.azurewebsites.net/ ( https://github.com/sockscode/sockscode-server ) Visual Studio Code: https://marketplace.visualstudio.com/items?itemName=shyykoserhiy.sockscode-vscode , ( https://github.com/sockscode/sockscode-vscode ) Microsoft Word(Windows, Mac and Web), OneNote(Web) ( https://github.com/sockscode/sockscode-office ) *(more like will it work pre-pre-pre-alpha)
