What it does

Analyze crash reports from autoSense. It features:

  • Impact time point calculation
  • Impact force calculation
  • Fully dynamic impact angle rendering

How we built it

No blockchain involved, AI not necessary (not even a lot of ifs), augmented reality is just an illusion and wasn't used here either.

However, it uses Python (which basically is Machine Learning) in the backend and web stuff in the frontend.
