Inspiration

We like to play grid-based board games, and it's fun to create our own, so we wanted to make it easy for students to do the same while encouraging an interest in programming.

What it does

Creates and renders grid-based board games using our convenient Python API.

How I built it

Django & Paper.js

Accomplishments that I'm proud of

Because of the web-IDE, testing changes to your board game is extremely easy. Also, we implemented multiplayer support, so people can share the games they built with other people and play them live, from anywhere in the world!

What's next for Latticode

Game notation and timers, more built-in games (Go, Othello, etc.), and improved API documentation
