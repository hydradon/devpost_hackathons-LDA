Inspiration

For the cyclists and a social reason on mind, to help them in the day by day fighting against traffic controversies and save them from accidents.

What it does

It helps cyclists communities to make connections.

How I built it

Was meant to be done like an hybrid app, to be able to be ported to any platform. We used HTML5, CSS3, JavaScript and Parse (Cloud back-end).

Challenges I ran into

How to make some calculations and predict some stuff to solve some problems.

Accomplishments that I'm proud of

We built a reasonable and functional demo app to show.

What I learned

We learned a lot about google maps API and parse JS with back-end and front-end.
