Inspiration

This game was largely inspired by Pony Island and previous hackathon entries Project Underline and Escape .

What it does

A paranormal puzzle game experienced through the medium of a (demon-controlled) notebook app, Notes requires the player to free former users who lost their souls through in-app purchases and to ultimately return the app to a demon-free state. It is not a note-taking app.

How I built it

Built with Android Studio.

Challenges I ran into

Figuring out how to best tell the story through an app's features. Actually integrating these features.

Accomplishments that I'm proud of

Thinking up the idea and medium for this game. Creating a UI using very simple elements that surprisingly doesn't look too terrible.

What I learned

How to use toasts and dialogs.

What's next for Notes

More puzzles (likely transferred from Escape). More plot. More corrupted files. More metafictional elements. More fights with demons. But no actual note-taking capabilities. ~I'm sorry.~
