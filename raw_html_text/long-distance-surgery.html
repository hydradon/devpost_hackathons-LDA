Inspiration

We wanted to use the power of AI to directly improve someone's quality of life.

What it does

hEar is a Google chrome app that allows the user to hear a description of scenes in videos to allow them to experience online content.

How I built it

hEar was built using Google Cloud's Vision APIs.

Challenges I ran into

Getting Google Cloud's Vision API to run on my computer.

Accomplishments that I'm proud of

Building something that will truly help someone else.

What I learned

How easy it is to apply the power of AI.

What's next for hEar

Bringing it to the masses!
