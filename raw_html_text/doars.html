Inspiration

Rick and Morty, Dr. Strange, a variety of TV shows and movies all feature portals to another universe. So let's use AR to bring that into real life!

What it does

Phone application that runs Unity to download different "doors" and "rooms". Users can upload on the website, and find their file and experience a portal to that room.

How we built it

Used HTML, CSS, Javascript for the website, Unity for the game engine, ARcore for the AR things, Django for the backend

Challenges we ran into

Major versioning issue with ARCore's camera, where its parent wouldn't follow its child through a filter to proc the spawning of AR objects.

Accomplishments that we're proud of

Fully finished the essentials of the entire project from front end to back end.

What we learned

How to use ARCore, how to link front end and back end to servers.

What's next for doARs

Waiting for ARCore to be compatible with stereoscopic lens so it can be ported AR headsets.
