Inspiration

We were inspired by the overwhelming resources that were available for veterans as well as some personal connections our team members had to veterans.

What it does

It consolidates a majority of local resources that make it easier for veterans to find assistance.

How we built it

We used Android Studio to get the ball rolling.

Challenges we ran into

Recycler View was incredibly difficult to implement and pass arguments into. It was also challenging to take said value and pass it to another Activity.

Accomplishments that we're proud of

We succesfully compiled local resources, implemented a Search Bar, Speech to Text integration, Google Maps Integration, and a Recycler View.

What we learned

We learned how working in a team environment can help push a goal forward.

What's next for NewLifer

Much like the new life that will be instilled in our veterans, New Lifer can experience all kinds of growth. These changes include working with a website/database integration for easier access to local resources, the ability to message local resources for assistance, and schedule appointments for medical needs.
