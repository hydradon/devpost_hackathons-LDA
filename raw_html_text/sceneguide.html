Inspiration

We wanted to create an application with social impact. Learning about Clarifai's API gave us the perfect opportunity.

What it does

It allows clicking of photos or recording of videos and then describes the images using some tags in form of speech.

How we built it

We built it using Android Studio and Clarifai's API.

Challenges we ran into

We ran into challenges while implementing the Camera module from Android and trying to integrate the capturing image and recording video functionality together.

Accomplishments that we're proud of

Succesfully using Clarifai's API and implementing Text to Speech.
