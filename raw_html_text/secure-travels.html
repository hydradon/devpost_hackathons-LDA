Inspiration 

As a team, we were really enthusiastic about learning new technologies and we decided to try building an application involving cryptocurrency and fintech in general!

What it does 

To provide a platform that tracks and records immutable transactions between different parties. In our application, we wanted to apply this to travel planning to ensure a secure process throughout!

How we built it

Used solidity to build the back-end and html/js for the front-end and tried to use Web3 to hook them up.

Challenges we ran into

Running the program and connecting to it from a front-end. This is difficult since Blockchains usually have many people mining on it (making things happen, otherwise code isn't executed).

Accomplishments that we're proud of

We got the hard part! (Blockchain, I think XD)

What we learned

How BlockChain technology and Cryptocurrencies work. Why they're effective.

What's next for Secure Travels

We plan to incorporate our own cryptocurrency within the system.
