Inspiration

To be able to create an immersive audio experience anywhere.

What it does

It's an app that allows artists and creators to drop interactive audio pens anywhere and allows fans to meet up at the location to experience the event.

How we built it

XCode, Swift, Bose AR SDK

Challenges we ran into

Integrating BoseAR with ARkit, IMU Calibration, and implementing CoreLocation.

Accomplishments that we're proud of

integrating BoseAR into ARkit and getting gestures to work.

What we learned

A lot about the flexibility of the Apple AV Foundation and Core services.

What's next for Sound Scene

Being able to place an object with the BoseAR glasses using gestures. Using gyroscope to position the object. Place pens in real locations using CoreLocation.

Refinement for TestFlight.
