Inspiration

The war zone has now moved to cities and urban areas

What it does

Provides accurate direction and angle of incoming fire so that a precise counter attack can be planned and ambushes can be prevented Also, provide emergency evacuation for injured personnel and provide materials required urgently to the areas of unrest

How we built it

We built it using multi layer laser gris systems, fpgas and a cpu

Challenges we ran into

The issue we faced is how multiple laser sensors can be interfaced with a controller or fpga in a cost effective and efficient manner

Accomplishments that we're proud of

Creating a system which can accurately detect the direction of bullets fired onto our bot which can be used practically in military bots/vehicles alike
