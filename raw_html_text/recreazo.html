• Couverture de présentation

Inspiration

Avec le manque de communication de certaines villes concernant les activités offertes, nous avons voulu remédier à ce problème.

What it does

L'application permet aux utilisateurs de retrouver facilement des activités ou loisirs près de chez eux.

Les jeux de données utilisés

  • Ville de Québec ACTIVITÉS LOISIRS PAYANTES https://www.donneesquebec.ca/recherche/fr/dataset/vque_23
  • Ville de Québec ACTIVITÉS LOISIRS PAYANTES (STANDARDISÉ) https://www.donneesquebec.ca/recherche/fr/dataset/vque_71

How we built it

Nous avons réalisé l'application à l'aide de Flutter, le nouveau framework de Google pour créer des applications natives android et iOS.

Challenges we ran into

Nos connaissances limitées dans les technologies existantes nous ont causées beaucoup de problèmes.

Accomplishments that we're proud of

Nous avons réussi a créer une application complète qui comporte une présentation soignée.

What we learned

Nous avons appris le langage Dart et le framework Flutter. Nous n'avions jamais travaillé avec ces technologies.

What's next for RécréaZone

On aimerait implémenter un plus grand éventail de ville et permettre aux utilisateurs de créer leurs propres activités.
