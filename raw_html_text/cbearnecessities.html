Inspiration

This project was inspired by the need for data analysis in the industry today.

What it does

Our project, "CBearNecessities", is a data visualization tool. It allows for customers to visualize clients on a map and conducts valuable data analytics stuff using charts.

How we built it

We built this using the MEAN stack(minus AngluarJS).

Challenges we ran into

We ran into problems getting data from the provided API and passing our data around.

Accomplishments that we're proud of

We are proud to have built something using this tech stack since it was our first times developing with database type software.

What we learned

We learned a lot about using database and big data sets. We also learned a lot about making websites.

What's next for CBearNecessities

We plan on continuing to improve with database stuff and will continue web dev in the future.
