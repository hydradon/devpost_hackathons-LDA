Inspiration

I found the Zillow API and wanted to build something with it.

What it does

With the user's permission, it pulls zip code and finds the mean pricing for studios, one bedroom, and 2 bedroom apartments.

How I built it

NodeJS, Javascript and the Zillow API

Challenges I ran into

Will users enable zip code permissions?

What I learned

I learned how to integrate an API into an Alexa skill and ask users for permissions.
