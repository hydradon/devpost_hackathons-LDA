Inspiration

Wanted to build it for a long time. MongoDB World Hackathon was the perfect excuse.

What it does

Teaches people to learn how to write code in a playful environment. Guide Buggy the robot around and learn to how to code.

How I built it

Buggy was built using Unity3D as well as React.js and mongoDB.

Accomplishments that I'm proud of

That I actually did it. Managed to build it although I was late to the hackathon and we had to build it in 2 days.

What's next for Buggy

More scripting langues, public release as mobile app for iOS and Android, database interaction levels (bot reads from a database to which player has to write), better multiplayer mode
