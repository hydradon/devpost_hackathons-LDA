Inspiration

The concern of modern politics and misinformation tactics.

What it does

Takes a base URL and keyword from a user and returns the links on the following page listed with the estimated political bias of the writer based on the keywords returned by the search.

How we built it

We used a flask web framework and the Beautifulsoup4 python library.

Challenges we ran into

Git merge conflict, recursively scraping links from websites, CSS alignment issues and configuring the SQL Server.

Accomplishments that we're proud of

Most of our work with web frameworks have been focused on simple blog functionality or other concepts closely associated with popular tutorials. For the first time, we were able to break out of tutorial purgatory and create an app that had unique functionality.

What we learned

Beautifulsoup, CSS Grid, Flask-requests & Resolving git issues.
