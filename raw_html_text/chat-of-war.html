Inspiration

After going to the Microsoft Tech Talk, I wanted to learn how to code a project using Node.js and network programming.

What it does

Play Tug of War by typing in u or d.

How I built it

Visual Studio Code, Node.js, Azure

Challenges I ran into

Complications trying to make harder games, such as Asteroids Online, and TagPro Soccer Edition. Having the correct automated outputs in chat, such as winner and timer.

Accomplishments that I'm proud of

Getting the code to run on the server.

What I learned

Focus on projects simple at first. If you go for a too complex project, you probably will run out of time. Tech Talks are good resources to get ideas for projects/resources.

What's next for Chat of War

Scroll Bar should stay at the bottom,, Assigned Teams, Username, Graphics for the Rope.
