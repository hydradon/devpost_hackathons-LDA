Inspiration

Finances are the main reason students don't complete college.

What it does

Manage student financials with their monthly fixed income. Helps students to learn about managing the expenses.

How I built It

We used Node.js, express.js, passport.js and mongoose. We used bootstrap to support the front-end.

Challenges I ran into

Taking the backup of the code and maintaining the revisions regularly.

Accomplishments that I'm proud of

Successfully built the first prototype that can be enhanced.

What we learned

Learned usage of callbacks in Node.js. Learnt MongoDB.

What's next for Student Financial Tracker

Linking the bank account to the application using Plaid or Mint API to get the transactions and perform the analysis where the student can save more which will benefit him. Adding a chrome extension, so that whenever the purchases cross the "spending limit" a push notification is alerted to the student. Getting the financial aid or student loan details so that they can efficiently pay their interests.
