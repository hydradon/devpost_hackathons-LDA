Inspiration

Seeing friends become increasingly reckless with money transferring because of the ease of Venmo

What it does

Uses Venmo's API to retrieve financial data about the user and presents the data clearly and concisely. Also, analyzes data and gives insight into the user's spending habits, and offers recommendations for the future.

How I built it

Using objective-C for client-side code, PHP for server-side code, and SQL for a database

Challenges I ran into

Getting all the information Venmo's API has to offer, linking PHP to Objective-C, and analyzing the financial data

Accomplishments that I'm proud of

Successfully integrating Venmo's API, creating a functional User Interface, and creating a fully-functional SQL database

What I learned

Plenty of Objective-C skills, auto-layout skills, and some server-side skills

What's next for Lily

Get the data analysis side of the project to be fully operational, and create better graphics to represent the data
