Inspiration

I wanted my kids to be able to simply know some facts about French history. Since reading a history book is not very attractive for them, I created this Alexa skill

What it does

You just ask Alexa to tell you a fact about France history, and you get an historical fact.

How I built it

It was first using the Alexa SDK V1. I recently updated it to use the Alexa SDK v2 and APL to display more rich content.

What's next for Histoires de France

I would like to add more historical facts, and maybe an English translation
