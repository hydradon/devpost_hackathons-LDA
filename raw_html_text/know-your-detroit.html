Inspiration

Take the local data and put it with ease at the fingertips of users. Starting with design and simplicity as the main focus.

What it does

It builds data on top of a geographical map and allows user to interact with both two and three dimensions.

How I built it

Using web technologies I created a platform and hosted it using a web service that points to a godaddy dns server with the name knowyourdetroit.com. Each iterative build focused on user experience and access of data points.

Challenges I ran into

I ran into challenges working with the large government data sets.

Accomplishments that I'm proud of

Creating a wonderful product that is useful and entertaining.

What I learned

Ask for criticism and suggestions, build for the users and never stop iterating.

What's next for Know Your Detroit

Marketing, more web development and expansion of the data sets.
