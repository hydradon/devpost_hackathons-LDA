Inspiration

Starting to explore different formats in 3D printing technology. Found out the this format supports colors and textures.

What it does

It exports a 3MF file from Fusion 360 with a simple file save dialog provided to the user.

How I built it

I started researching on different 3D printing supported formats. This format is combination of STL and XML within a ZIP file. It to simple and understandable.

Challenges I ran into

This formats do not supports assembly structure or any hierarchy. To properly export the model with correct transformation was a challenge.

Accomplishments that I'm proud of

Finally I supported the assembly export with proper transformations. The model is exported as it is to 3MF.

What's next for Write to 3MF

To provide user specific options to redefine the plugin and make cope up with users requirement.
