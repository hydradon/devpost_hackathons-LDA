Inspiration

Suggest consistent useful resources to veterans by using data collected by phone calls with veterans!!!!!!!!!!!!!!!

What it does

Veterans call our number (+1-323-814-2445) and answer questions to allow call managers and data to make suggestions for them.

Call today!!!

How we built it

We used dialogflow to set up the questions and store answers. We used HTML, CSS, and JavaScript.

Challenges we ran into
