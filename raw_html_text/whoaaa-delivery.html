Inspiration

Greta Thunberg inspired us. There are thousands of deliveries all around the world and actually not much of them care about the environment but just about profit. We do BOTH. Using gamification we are forcing our clients to use electro vehicles. If the user chooses EV, bike or pedestrian for a delivery, he will get points. Using the points user can:

  • share it with friends and show how environment-friendly he is;
  • use points to get a discount for delivery.

What it does

  1. The client chooses pickup and destination points.
  2. Using Agents and TomTom API we are picking the best routes for the delivery.
  3. To force the client to pick environment-friendly delivery we grant them points.

How we built it

  1. For the backend, we used python and flask. There we are using TomTom API
  2. For the frontend, we are using react and leaflet and TomTom SDK.
