Inspiration

We wanted to find an interesting way to use Wegmans's API.

What it does

Put the recipe you want, scan the bar code of the ingredients that you get at home. Then, our program will tell you what ingredients you are missing in order to cook the recipe.

How we built it

We used IntelliJ, PyCharm, and Postman as our IDE. We parsed JSON file and processed it so that it can work.

Challenges we ran into

  • No JSON, Machine Learning experience
  • Accessing data was also challenging since we never use the API

Accomplishments that we're proud of

  • We have learnt new stuffs and been able to implement it

What we learned

  • JSON, Postman API, Barcode Scanner, Machine Learning

What's next for Wegmans Scanner

Front-end, mobile app, and further commercial development
