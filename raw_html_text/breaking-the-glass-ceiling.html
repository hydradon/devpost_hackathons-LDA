Inspiration

In a group of all women, we wanted to do a hack that was centered around feminist ideals.

What it does

It is a game where you are a woman in an entry level position at a tech company, and you slowly earn promotions and gain respect in the workplace through completing programming assignments in the form of mini games to eventually work your way up to CEO.

How we built it

We built this using python and pygame

Challenges we ran into

We did not have time to make the mini games ourselves, and it was difficult to find enough code for them to make our game interesting.

Accomplishments that we're proud of

We were able to utilize our pygame skills, and we are excited that we were able to get a working model of our game.

What we learned

We learned about timers, gitHub, pygame, and object oriented python

What's next for Breaking the Glass Ceiling

We hope to program more mini games ourselves, and get the higher levels more bug free.
