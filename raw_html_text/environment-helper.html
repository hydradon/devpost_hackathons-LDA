Inspiration

I wanted to give tips to people on how to help the environment. There are many simple ways to make a positive impact in this world. Everyone can make a difference.

What it does

Environment Helper helps give you tips on how to help the environment in everyday life. Simply ask it for something to do, and it will help you help the environment.

How I built it

I built it using the Alexa skills kit and amazon web services lambda. I went through the tutorials in order to learn how to utilize these tools.

What's next for Environment Helper

We will constantly add new tips to environment helper in order to keep raising awareness.
