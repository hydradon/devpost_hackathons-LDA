Inspiration

We wanted a better way to quickly gather online data into short, presentable snippets easy to spread.

What it does

Our app summarizes articles in a few sentences using machine-learning techniques to detect impactful words.

How I built it

We used Google Cloud and Cisco sparkbot tech.

Challenges I ran into

We were having problems working with Google Cloud functions, so we had to work with Google engineers to fix the issue.

Accomplishments that I'm proud of

We're really proud of setting up the backend successfully!

What I learned

I learned how to successfully work in a team.

What's next for Simpl

We're interesting in adding sentiment analysis so as to gather more useful data from online articles.
