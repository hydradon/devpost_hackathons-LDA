Inspiration

To create an app that lets you go on cheap vacations with your friends.

What it does

Do you have a friend that lives far away? Well our app solves that problem! We find the cheapest and best place for you and your friends to go on vacation.

How I built it

It was built using Expedia's API and ASP.net.

Challenges I ran into

Low amount of tutorials for ASP.net.

Accomplishments that I'm proud of

Most of the project is done, but it has a few minor glitches.

What I learned

Wrapping API's in C#, using ASP.net, Debugging apps in Google Chrome, using Azure, and Visual Studio.

What's next for FlightBook

Finishing the application for the real-world.
