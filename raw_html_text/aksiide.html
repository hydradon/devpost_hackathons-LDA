What it does

AksiIDE is a Simple and Light Code Editor for Developer . Multiplatform, running in Windows, Linux and Mac environment.

With this editor, developer can easy to create new project. just select menu new project, than choose project wizard that they need

With this editor, developer can easy to move/switch to project, just using click in project explorer. just note, some developer have many project that must handled in same time.

Features

  • Auto Complete
  • Code Completion
  • Code Snippet
  • Project Wizard
  • Live View Browser
  • Project Management
  • Class Generator
  • Integrated With Database Explorer (mysql + postgresql)
  • Database Browser
  • FTP Explorer
  • File Explorer
  • Code Explorer with Function List
  • PHP Source Minifier
  • PHP Source Formatter
  • Syntax Checker
  • Multi Workspace Layout
