Problem

Indonesia merupakan Negara berkembang dengan kondisi Infrastruktur jalan yang memprihatinkan, berdasarkan data dari dinas Pekerjaan umum pada tercatat 21.313 km jalan kabupaten dan 2.468 jalan provinsi rusak dan harus di perbaiki, demikian pula perbaikan yang dilakukan sangatlah lambat karena minimnya partispasi masyarakat dalam penyampaian aspirasi mengenai infrastruktur jalan, buruknya kondisi infrastruktur itu berdampak pada perekonomian masyarakat. Serta tidak efektifnya birokrasi Pelaporan kerusakan infrastruktur yang ada dipemerintah dalam menampung aspirasi masyarakat membuat sulitnya pendataan kondisi infrastruktur jalan yang berakibat pada tidak tepatnya pengambilan keputusan mengenai penangan kerusakan infrastruktur jalan.

Solution

Kami membangun sebuah system informasi pelaporan dibidang kerusakan infrastruktur jalan berbasis mobile yang menampung aspirasi masyarakat dan terintegrasi dengan system tanggap laporan pihak terkait seperti bappenas dan dinas pekerjaan umum, system akan secara otomatis melaporkan kondisi jalan memanfaatkan sensor accelerometer pada android pengguna sehingga data yang dikumpulkan cukup besar, dengan tujuan membantu mempercepat pengambilan keputusan pemerintah, serta memastikan aspirasi atau keluhan yang masuk di tindak lanjuti, sehingga perbaikan jalan segera mungkin dilakukan

How it works 

• Pengguna mengirimkan data kerusakan jalan secara realtime. Baik partisipasi user secara pasif(otomatis) ataupun active.
• Kumpulan data yang telah dikirimkan oleh pengguna akan disimpan di cloud server. Kemudian dianalisa sesuai krieteri pelaporan.
• Keluaran dari system digunakan oleh pemerintah, user, developer dan Industri.
• System Pelaporan terintegrasi dengan system tanggap laporan di pihak terkait dalam hal ini bappenas dan dinas pekerjaan umum untuk memastikan aspirasi di tindak lanjuti
• Aplikasi diharapkan sustainable dengan memberikan layananan :
• Pengambilan Kebijakan terkait tanggapan terhadap aspirasi.
• Berbagi informasi untuk pengguna
• Menyediakan open data  API untuk Developer
• Dan Analisa Data untuk Industri
