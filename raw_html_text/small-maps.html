Inspiration

Lost people need directions, and event staff can't be everywhere.

What it does

The Small Maps website, smallmaps.net, uses Dijkstra's shortest path algorithm to find the quickest route between strategic locations throughout the Science Building.

How we built it

We first wrote it in out most fluent programming language,Java , then converted it into JavaScript. We wanted to attempt the Domain.com challenge and used HTML to create a website to hold our program.

Challenges we ran into

Converting from Java into JavaScript for the website. Specifically retrieving user data from a text box.

Accomplishments that we're proud of

We built a website!

What we learned

How to build a website with HTML and integrate JavaScript into it.

What's next for Small Maps

User friendly node placement for organisations, improved interface,and location tracking.
