To complaint a particulate problem in these days is a big serious and tough issue because no one have enough time to go to the concern department and submit a complaint is a big issue.

The app will provide a verity of complains types and user will select the appropriate compaint type then App will provide a Map service to the user which shows the current location of the user, User will select his location by taping on Map then a dummy message according to the complaint will be displayed to the user. User can also change the messge or can type a custom message.Location of the user and complaint type will also be shown to the user. then app will allow the user to take the pictures of the complaint/incident/problem etc and then the app will send all these information using the email client of the user device to the concern department. and the copy of the complaint will also be stored in device storage for records.

We build this application in 4 months only for android platform because of the huge percentage of the android users. The tools we used was Eclipse, wamp server, Sqlite DB, Photoshop. Letter on the app was transferd to ANDROID Studio

We gave a good, helpful and easy way for a normal person to complain about any problem.

This app was for Peshawar only but it can be extend to the all Pakistan users.
