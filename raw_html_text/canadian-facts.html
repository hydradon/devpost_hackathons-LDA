Inspiration

As a proud Canadian, I wanted to create a fun tool where kids could learn more about this great country

What it does

The Alexa Skill allows users to ask for and recieve facts about Canada by asking "Alexa, open Canadian Facts"

How I built it

I used the Amazon Developer Console and AWS Lambda. Using the facts template, I tailored it to create my own questions.

Challenges I ran into

Figuring out how the invocations were connected to the code.

Accomplishments that I'm proud of

Creating something that works

What I learned

How to create a basic Alexa Skill

What's next for Canadian Facts

To add more facts and make it interactive
