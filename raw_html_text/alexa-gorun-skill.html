Inspiration

I always like running. Whenever, I get stressed or have to take a decision, i always went for a Run and all my stress is gone. Apart from that, running has many more benefits like :

  1. Strengthens your lungs
  2. Helps prevent high blood pressure.
  3. Strengthens immune system.
  4. Increased joint strength and stability. ... etc.

Apart from that, taking parts in events like Marathons, etc can raise huge amount of money for social good and charity. So, my main motivation and inspiration for developing this project was to encourage and assist many avid runners to go out and take part in different running events in India.

What it does

The skill basically lists down different running events according to the filters set by user like event type, city, and preferred date.

How I built it

I built it using Alexa Developer portal with the help of AWS Lambda as my backend. Also, I created a web scrapper for scrapping data from a website, I deployed that scrapper on Heroku instance.

Challenges I ran into

Sending emails to the user of the info. I will integrate this functionality in future releases.

Accomplishments that I'm proud of

I used many latest tech stack like Heroku and created and designed a Web scrapper using python and beautifulSoup first time

What I learned

many awesome things :)

What's next for Alexa GoRun Skill

features like notifications and emails to users.
