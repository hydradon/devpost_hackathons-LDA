• Got plans?

Inspiration

Traveling to new places around the world sounds like an amazing adventure. But sometimes you just don't know what to do and the transition could be a little harsh.

What it does

Our web application allows users to create plans for a specific area, bringing an opportunity for people to connect and get to know the best things to do around the area.

How we built it

We used the new tool Wix code and Google Maps.

Challenges we ran into

Wix code was completely novel to use so we had a very steep learning curve.

Accomplishments that we're proud of

We were able to complete a fully functional web app using only wix code. We even integrated their database service.

What we learned

These 36 hours were about time optimization, sometimes what works best is using tools with preloaded and ready to go resources such as Wix, instead of going for a fully flexible application.

What's next for gotaplan?

We want to integrate some other features such as Calendars APIs, some social media and a chat for users that want to participate in the same event.
