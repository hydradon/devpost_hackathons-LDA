Inspiration

We heard about the JP Morgan Chase code for good challenge and wanted to create a project that inspires young women to go into STEM.

What it does

The user shoots at different stereotypes about women and after they obtain a certain amount of points they go to the next level and receive a fact about women in STEM. As the game progresses and the user goes from one level to the next the stereotypes get worse and the game gets a little bit harder.

How we built it

We used HTML5 and javascript.

Challenges we ran into

We had some issues with debugging javascript and were not able to stop rendering to the canvas.

Accomplishments that we're proud of

We are proud of being able to use a language that we don't regularly use and make a project out of it.

What we learned

We became more familiar with how to use javascript and HTML5.

What's next for BiasBuster

We could improve the graphics and have more levels, more choices for the user, and different game modes.
