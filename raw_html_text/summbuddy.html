Inspiration

Generally, students need to make notes from any textual content, summarize the content in order for quick revision. Also, for effective study, a student needs to find possible questions related to the text. SummBuddy is the result of the inspiration to minimize user work of summarizing and generating questions from the text

What it does

SummBuddy takes an input image that contains text, converts the image text into normal text and summarizes the text. It also generates questions related to the text alongwith the answers for that questions

How I built it

Used the following APIs 1) Google Cloud Vision API - Optical Character Recognition 2) IBM Watson - Natural Language Processing 3) Custom Algorithm for generating questions and answers
