• English --> HTML/CSS using finite closure reduction

This isn't just another Pregasen NLP la la land project.

My first cousin wants to be a web developer but he has Osteoporosis Imperfecta, and a particularly serious type. While he is 20, he is still the size relative of an infant. However, he goes to college via an electric wheelchair and wants to become a web developer; the grandest challenge, however, is typing up the grander structure of the code since his fingers aren't sized proportional to the keyboard. Tools like this, when paired with voice to text tools, allow such people to write a template of code efficiently without much physical strain.
