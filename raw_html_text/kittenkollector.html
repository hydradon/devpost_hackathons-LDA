Inspiration

We were inspired by Pokemon Go at first, and starting brainstorming from there. We first thought about using robot pictures and travelling to collect codes. However, we didn't like moving around, and robots were boring.

So we built a tool for scavenger hunts, large or small, that can be easily created using a website.

What it does

It's a site that can be used to generate kittens. These kittens each have a code, which can then be entered by anyone into the website so that they can collect the kitten!

How we built it

We divided the project into the server and the client. We made the server in python and flask, and the client using HTML and CSS and some javascript (plus a little bootstrap).

Challenges we ran into

  • Registering a domain name - Domain.com has a few problems and bugs and so as of now, we don't actually have the domain despite having registered it 15 hours ago. Yay.
  • Google Cloud Platform - Google Cloud Platform is quite neat. There's lots of things you can do. However, it's confusing to learn and very difficult to use with SQL.
  • Lack of experience with web stuff - Not having done much work with websites before made developing one a little tricky. LocalStorage API, Fetch API, etc.

Accomplishments that we're proud of

WE HAVE SOMETHING THAT WORKS! And it looks pretty neat as well.

What we learned

  • General web stuff.
  • The localStorage API!
  • Google Cloud Platform.
  • Registering domain names is not fun. At all.
