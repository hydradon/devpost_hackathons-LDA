Inspiration

Writing programs for industrial robots is not an easy task. The idea is to provide a user an interface where they would be able to create programs from scratch and defining their own interfaces.

What it does

At the moment, it only allows a user to run selected program on the robot.

How I built it

Frontend: HTML5, CSS3, AngularJS

Challenges I ran into

The system turned out to be too complex and we ran out of time.

What's next for Clickotic

  • Allow a user to write and save a program in the system.
  • Share written programs with other users.
