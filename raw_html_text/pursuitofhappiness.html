Inspiration

We wanted to build some sort of a hands-free simulation.

What it does

Simulates basic behaviors of a person left alone in the woods, including psychological state.

How we built it

We built the logic with python and used pygame to render primitive graphics.

Challenges we ran into

Tuning the parameters of how things happen. Path finding without A*.

Accomplishments that we're proud of

Our beautiful graphics.

What we learned

How to use pygame.

What's next for PursuitOfHappiness

More states, behaviors, environmental factors, creatures, A* path finding, smart environmental features generation, colonies of multiple people simulated together.
