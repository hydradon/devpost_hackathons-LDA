Inspiration

We were thinking of things that are a hassle in our lives and organizing group work came to us

What it does

It looks at two peoples schedules that want to do a group project together and checks to see when they are both available

How we built it

We built it using Android Studio and java, with no separate apis

Challenges we ran into

Comparing the users schedule proved to be the most difficult

Accomplishments that we're proud of

Being able to use firebase to allow anyone with wifi to connect and use our app.

What we learned

Using firebase's api and calendar api in android studio

What's next for Time Table

More features to allow for time management for solo use.
