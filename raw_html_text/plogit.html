Inspiration -- Plogging is a combination of jogging with picking up litter.

What it does -- An app on Fitbit that tracks how much trash you have picked up

How we built it -- Using the built in accelerometer in the Fitbit API to log every time a trash pickup movement was completed (made in JavaScript), front end displaying said information was created in HTML and CSS

Challenges we ran into -- Getting algorithms to work, completing front end interface for application

Accomplishments that we're proud of -- Collaboration, accelerometer correctly logging and displaying data

What we're learned -- How to create Fitbit application using the Fitbit web API, how JavaScript, HTML, and CSS work together to generate a functional app, developing for hardware and wearables

What's next for PlogIt -- Turn into a startup!
