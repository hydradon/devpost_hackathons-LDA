Inspiration

Wanted to make a fast way to view health statistics in your area.

What it does

Allow the viewer to see trends in health conditions in certain areas using google maps and cerner FHIR API

How I built it

Created a web application with a log in system to grant access to a page where the user can view health conditions and statistics within a certain area.

Challenges I ran into

Utilizing the cerner API with PHP to retrieve data.

Accomplishments that I'm proud of

Able to integrate google maps with the cerner API data.

What I learned

We learned how to manipulate data with PHP and query with mysql inside of php.

What's next for Health SMACK

The database provided by Cerner can be expanded upon. More search parameters can also be implemented to narrow down results. This can be also utilized with other data to show companies where they can better market their products.
