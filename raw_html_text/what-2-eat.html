Inspiration

It is always a struggle to choose which restaurant to eat with your friends. We created this app to let a group of friends vote for which nearby restaurant to eat.

What it does

One person will generate a random generated QR code, and the other users will each scan the QR code to join the voting group. Each person will then vote on what they want to eat from a list of nearby restaurants generated using the Google-Places API. The voting result will then show once everyone finish voting.

How I built it

We used Android Studio with Firebase as the backend database.

Challenges I ran into

Integrating the database into the app functionality.

What's next for WHAT-2-EAT

  • We can integrate with Instagram by showing pictures of the food for each restaurant.
  • We need to integrate with using the Android device's GPS.
  • We can filter the restaurants by name/distance/ranks
