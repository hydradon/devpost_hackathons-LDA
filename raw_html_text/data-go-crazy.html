New York faces great issues with its traffic situation and it reflects in the data of recent road collisions and deaths. Unfortunately, there's no data visualisation and monitoring platform to address this issue so that it can be resolved or avoided in real time. This project retrieves the data from different new york public APIs and analyze them using Google Cloud Compute Engine and visualize for the user in Google Cloud DataLab for better understanding as well as provides real-time tracking using the Here.com Map API, which updated every 5 seconds. Now the big thing is the whole system is getting monitored using the data control system dashboard and monitoring system of DataDog, which gives us a second-degree flexibility to process this crucial data and address the issues without any drawbacks.
