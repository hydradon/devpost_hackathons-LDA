Inspiration

This week I have two midterms and a project. All of that stress almost made me skip UGA Hacks II. I realized I needed a way to get rid of my stress.

What it does

This app is a simple game that allows the user to explore the scenery of a calming area.

How I built it

I built this app using Unity3D and programmed the back-end scripts in C#.

Challenges I ran into

I had no idea how to make a mobile application or how to export a Unity game to an android app.

Accomplishments that I'm proud of

This is the first mobile app that I have developed.

What I learned

Through this project, I learned how to create simple Unity games and export them as mobile applications.

What's next for Calm Yourself

With more time and resources, I would expand the terrain and add more scenery. I may also split the scenes and allow the user to choose where they want to go.
