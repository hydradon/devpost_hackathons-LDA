Inspiration

Frame fitting can be done via machine learning. Why not measure visual acuity through technology?

What it does

Based on our algorithm and input analysis, we can determine the visual acuity of an individual.

How we built it

We used HTML, JS, R-Studio, CSS, and Firebase to create a website where users can use their phone as controllers to input sequences of characters that can be used to identify their visual acuity.

Challenges we ran into

Algorithm to determine visual acuity. Web requests, and database systems.

Accomplishments that we're proud of

Connecting browsers, building a viable demo.

What we learned

How visual acuity works, database management, JSON and AWS, and I learned how to integrate across platforms.

What's next for 20/20

Determine distance via facial recognition, refining and improving algorithm and recognizing optical conditions through technology.
