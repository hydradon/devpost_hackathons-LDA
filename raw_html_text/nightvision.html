Inspiration

We like looking at stars.

What it does

We wanted it to be a complete social star-gazing platform. So far we just have an interactive map that generates star visualizations based on position and time. Oh well.

How we built it

Backend with Flask and the skyfield Python library; frontend with D3; hosted on Azure.

Challenges we ran into

Running Azure. Relearning D3. Vector math.

Accomplishments that we're proud of

Teamwork makes the dream work.

What we learned
