Inspiration

Since the network was down, we started collecting some data as a joke and we wanted to do something cool with it. Eventually, it grew to something even more cool

What it does

Sensei is a platform where you can collect and sell your data. That data can be virtually anything; from number of times you opened the fridge to your heart rate and it’s also compatible with various platforms.

How we built it

We've built it on several platforms: Web, Android, Wearables and Littlebits. Each platform plays it's own part in Sensei.

Challenges we ran into

Collecting the data itself, Internet connection , parsing the data into something meaningful and displaying it on the screen in different forms. Last, but not least, the hackathon itself is sort of a challenge.

Accomplishments that we're proud of

Collecting the data through multiple hardware, Android Wear app implementation, synchronizing the data between different platforms and displaying it on the end

What we learned

Teamwork and time management skills, developing Android Wear applications, implementing hardware hacks with Littlebits, calibrating the sensors for picking up the data etc.

What's next for FireBreathingRubberDuckies---Sensei

Winning the challenge and serious code refactoring :)
