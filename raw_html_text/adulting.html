Adulting

A web application that promotes the growth of youth and create a resource to spread information that is normally not taught in academia.

Inspiration

We wanted to make an application that helps college graduates learn adulting .

What it does

Provides education modules for relevant topics adults need to know.

How we built it

  • ReactJS
  • NodeJS
  • Mongodb
  • Google Cloud Platform (OAUTH2)

Challenges we ran into:

  • Connecting each article's data to the website.
  • Deciding on functionality of the website.
  • Deciding on what's important.

Accomplishments:

Finishing a majority of the issues and tasks that we set out to accomplish at the beginning and being able to implement something as a functional team.

What we learned:

We learned our capabilities as a team when it comes to splitting up work and trying to be more productive when it comes to agile development .

What's Next?

  • Including more modules and a required items page for each skill.
  • Medal system as an incentive to complete more modules.
