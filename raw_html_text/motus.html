Inspiration

As someone who has Aspergers, sometimes it is very hard to understand the emotions being conveyed being text. We aimed to solve this problem using our hack.

What it does

It determines emotions conveyed over text and also displays how confident it is about that rating

How we built it

We built using Python for backend, Javascript to handle events, Twilio to facilitate IP messaging, and IBM BlueMix to analyze tones.

Challenges we ran into

BlueMix sometimes has hiccups and will give wrong ratings on very obvious emotions.

Accomplishments that we're proud of

That we were able to finish on time and deliever both a product and user expeirence that we can be proud of.

What we learned

Mainly Python, lots and lots of Python.

What's next for Motus

An iMessage application that can review and rate text messages on the spot.
