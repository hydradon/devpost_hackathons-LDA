Inspiration

Robotic voices suck. Elderly people sometimes get lonely, and having this more human connection is better.

What it does

Allows the elderly to upload images, documents, letters, etc.. and gets someone to record an audio file of it, making it more personable than a robotic voice.

How we built it

Used Flask as a web framework and used javascript and css and html to make a dynamic front-end

Challenges we ran into

We ran into complications connecting Flask with MongoDB

Accomplishments that we're proud of

We figured out how to use Flask, as all of us were relatively new with it

What we learned

Flask, Web Development, User Experience

What's next for ReadAloud

Perfecting the database and dynamic posts
