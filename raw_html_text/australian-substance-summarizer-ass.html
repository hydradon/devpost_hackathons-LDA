My program, the Australian Substance Summarizer (aka ASS) takes a txt file and simplifies it using the TextRank algorithm (a redesigned version of Googles PageRank) to decide what the more important sentences are, then makes it more 'colourful' by either adding compliments or swear words using NLP from the Python NLTK to recognise verbs and nouns and accordingly sprinkle the text.
