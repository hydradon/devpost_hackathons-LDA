Inspiration and how we build it

Growing Community of Students, Lecturers, Teachers, Artist etc. on Facebook

Introducing whiteboard Collaborations as Post Status Updates on FaceBook...

Students(eg. Mathematical Students, Science Students),Lecturers, Teachers, Artist, etc. has been looking for a way to share their realtime Mathematical and Scientific questions, Updates, drawings etc as Post Status Updates on Facebook.

I have a friend who created a Mathematical Group on Facebook. His problem is how do he write those mathematical Equations and then post it on Facebook as realtime updates for all her group memmbers to see and learn.

The other friends says, I have a Chemistry Class Group, Biology Class. All this Facebook groups requires writing realtime Chemical Equations, Drawings and structural labelings etc.

This app is an answer to that as it let Facebook Users Write any Mathematical Equations, Chemistry Formulas, Realtime drawings, structural Labeling etc. and then share it as post status updates on Facebook.

How to use It.

1.) User Login using Facebook.
2.) Click on Share Whieboard
3.) Enter Post Title, Category and Description of what you want to write or draw on the Whiteboard.
4.) go to board, Write and draw on the whiteBoard using diferrent colored Markers of your choice . Clean Board with Eraser when dirty and Share Whiteboard as Post Status Updates on Timeline.
5.) Like and Comment on the Whiteboard Post
6.) The User can Share any Post to her FaceBook Timeline as well as to her specified Facebook Friends in HIPAA Compliance Mode.

FaceBook Technology Used:

1.) Facebook Login Graph API: Used to authenticate Users
2.) Facebook Shared Button: used to allow user to share content to her Facebook Timeline
3.) Facebook Send Dialog Button: used to allow user to share content to a specific user in HIPAA Compliance Mode.
