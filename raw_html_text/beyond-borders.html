Inspiration

The outlash against Syrians following the Paris attacks.

What it does

Provides Syrians a variety of language tools to help them adapt into the new country.

How I built it

Using xCode and Swift 2.

Challenges I ran into

REST API call issues, as many API's were provided by small developers with minimal documentation.

Accomplishments that I'm proud of

Being able to create a functional App.

What I learned

How much research is involved in a socially aware app.

What's next for Beyond Borders

Adding multiple language functionality, adding a volunteer area where volunteers can talk to those in need Increasing language tools and learning basics.
