Inspiration

_ we were inspired to do this as we wanted to be able to secure data with a facial recognition system _

What it does

This consists of 3 tools, a web ui that allows you to take a picture and save it to the cloud, a face isolater that pulls the image from the server and a real time line drawing tool that allows you to ceate basic maps of peoples faces

Whats next?

I will unify these in to a single UI and allow you to compare line drawings to authenticate users on a system.
