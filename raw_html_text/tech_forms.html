Inspiration

To help both client side and Service side to make the process better and faster

What it does

Converts a pdf into an encrypted blockchain that only you and the employer can access to transfer that pdf in a faster way.

How we built it

We divided the task in two parts, making sure the pdf is user friendly and that the blockchain is viewable on both ends

Challenges we ran into

Working with pdf code has been very complicated, in both the pdf side and the blockchain conversion side

Accomplishments that we're proud of

The blockchain component works and it sends the encrypted message that it was suppose to do

What we learned

That pdf is a tricky source to work with but we in the end got it working.

What's next for Tech_Forms

The next step is to make it mobile compatible.
