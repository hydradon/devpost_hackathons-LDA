Inspiration

We wanted to create a project that tested both the user's knowledge and our knowledge on the climate change crisis that effects every human, plant, and animal on the planet. Quizzes are a great way to test basic knowledge.

What it does

Raises awareness of climate change by testing the user on the glaring issues that climate change presents, as well as what it even is.

How I built it

We used a shared repl.it file to work on the html, css, and javascript code. We created the project from scratch, using our knowledge of the languages and available references.

Challenges I ran into

Since I never worked on something like this before I ran into a few challenging problems. One problem we ran into was sending a variable through the .html files in order to display the user's score. Initially we tried to quickly teach ourselves PHP since we knew it was capable of such a task. We soon realized this was unnecessary as HTML already provided the syntax, it just took some digging into various forums. Another problem we ran into was creating the checkbox answering system. Before we used javascript to ensure only one checkbox could be clicked per answer, the user could check more than one by trying to switch their answer.

Accomplishments that I'm proud of

I am personally proud of the fact that we were able to create the website within the time frame. I have never attempted a website like this since I am more comfortable with micro-controllers and math based programs.

What I learned

I learned a lot about the issues about climate change. I never knew just how horrible the extent of damage was done to the planet by industrialization in just the past 200 years.

What's next for Climate Change Quiz

If we were to continue the website, we would add more pages that list and explain how the user could help the planet.
