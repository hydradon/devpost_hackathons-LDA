(Inspiration) Do you ever go out and totally forget how smashed you got last night? Well we don't because we're not 21 but if we were, we'd love this app.

Keeps track of drinks and BAC as the night goes on

Built in Android-Studio, basic android application implementation

Challenges: Android Studio was awful to download on Linux, Intents were divas to work with

Accomplishments: Developed first app

Proud Of: Making a fully functional app thats not totally awful

What's next? Implementation of a more interactive UI through the use of the camera (drunk selfies which get saved throughout the night) and GPS functionality so you can be tracked by your sober friends as well as Data Analytics Section for analysis of BAC/drinking trends
