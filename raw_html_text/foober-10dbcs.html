Inspiration

Wanting food, but not wanting to to go get it yourself

What it does

Lets you post "orders" to a mobile site, where people can accept the order and deliver it to you for a small fee

How we built it

Using the Meteor framework allowed us to host the server for the site, and the API's provided a lot of functionality. The Locu API allowed us to find local restaurants and menu items to choose, while the Stripe API allowed us to manage how payments worked for the food delivery service.

Challenges we ran into

Using inconstant API information, linking multi user payments, and learning how the Meteor framework operated

What we learned

We learned how to implement the Meteor framework to build a website

What's next for Foober

Increasing stability, security, and real account information after a lot of polishing and cleaning up code.
