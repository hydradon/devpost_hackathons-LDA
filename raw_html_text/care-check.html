Inspiration

My parents Since I was studying in a different city from my parents, so I could not check their medicine use status. They usually put the medicine inside a box and remember the time taking it for themselves. Hence, I decided to develop an app that serves both as a pill reminder for elderly which is easy to use and also a monitor for me to see if they missed any medicine today.

What it does

The elderly will only need to enter simple medicine information and set the waking hours and the app will calculate all the notifications for them. For the care taker mode, you only have to enter the pair code given by the elderly to see all their medicine information and today status.

How I built it

I use firebase to store all the medicine information.

Challenges I ran into

The most difficult part is scheduling notifications and reedit the scheduled notifications.

Accomplishments that I'm proud of

I have successfully implemented firebase database, app structure and UserDefaults eventually.

What I learned

UserDefaults, UserNotificationCenter, Firebase

What's next for Care Check

Let one user to be both a medicine user and a care taker.
