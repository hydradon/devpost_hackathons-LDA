The inspiration came from the small amount of hardware we brought and acknowledging that we wanted to build something physical.

What it does

An ultrasonic distance sensor checks distance is in range, and if not sends push notification to a phone.

How we built it

We have different ways to run our system. The preferred method is by Particle Photon and external battery. This meets the qualifications as an IOT device. Alternatively, the device can be ran from a Raspberry Pi 3 through SSH.

Challenges we ran into

Learning front end and back end languages such as HTML and Python.

Accomplishments that we're proud of

As new Hackathoner's, we are proud to have a project to submit.

What we learned

We all enjoyed the presentations, specifically the Web Apps and Machine Learning. While hacking, the importance of front end development was brought to our attention, as well as understanding what an IOT device is and basic circuitry.

What's next for Home Security System

A housing, so the unit can be robust and functional for further development.
