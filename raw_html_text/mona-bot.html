• Mona-Bot

¿Cuanto tiempo tardas buscando algún producto? Mientras buscas de que dispositivo se trata, donde lo venden y que proveedor lo tiene a mejor precio, ya te llevaste un buen rato. ¿y si te digo que Mona-Bot hace todo eso un menos de un minuto?

Mona-Bot es un servicio de Google Assistant que se encarga de buscar cualquier producto, si no conoces como se llama lo que buscas, toma le una foto, Mona-Bot lo busca, lo encuentra y te despliega las mejores ofertas entre todas las disponibles en la web

Mona-Bot esta construido utilizando diferentes servicios de Google Cloud Platform (Cloud Storage, Cloud Vision, Cloud Functions, Clud Pub/Sub, etc.)

Este proyecto realmente represento un reto increíble, se aprendió demasiado y estamos orgullosos de lo que se logro.
