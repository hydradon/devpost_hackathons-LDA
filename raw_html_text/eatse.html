Inspiration

Help Chef Try New Dishes Safely Low Startup Cost Create New Foods Pre-Order Reduce Food Waste Help Local Chef Open New Restaurants in Local Neighborhood

What it does

Accomplishments that we're proud of

We found a way to work on wix on multiple computer in parallel.

What we learned

We learned to set up database and dynamic pages on Wix.
