Inspiration:

Fraud Detection

What it does:

It is distributed credit bureau blockchain platform.

How we built it:

A blockchain API was created using springboot which is consumed by the RESTful web application based out of python. It was tested on randomly generated test data(which contains SSN, type of transaction, DOB etc.)

Challenges we ran into:

Storing data in memory in a peer-to-peer network and establishing communication in this network.

Accomplishments that we're proud of:

We were successfully able to build a distributed blockchain based fraud detection system and thereby using data to detect or cross-verify an out-of-pattern transaction thereby suggesting an unusual activity.

What we learned:

We learned blockchain, challenges in building blockchain and the current limitations. We can use this to tackle problems in other domains.

What's next for Distributed Credit Bureau based on the blockchain:

To create a prototype to see how it works in the real world.
