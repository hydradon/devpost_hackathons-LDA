Inspiration

I spend a lot of time choreographing and dancing, and wanted a simple way to create dance formations.

What it does

Allows for real time collaboration on dance formations. Ability to save and view multiple formations. Nice animations when changing formation.

How we built it

Used Firebase for real time collaboration functionality.

Challenges we ran into

Challenging project overall.

Accomplishments that we're proud of

Animations were a nice touch.

What we learned

Polishing a web app, using Firebase.

What's next for Formations

More casework, such as updating additional formations, etc.
