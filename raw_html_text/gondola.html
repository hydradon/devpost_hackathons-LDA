Inspiration

Two popular games, Smash Hit and Beat Saber, were the fundamental inspiration points for this game.

What it does

Our game puts you onto a virtual gondola on rails, and hurls you through five levels of fun, exhilarating action.

How we built it

We utilized Unity and SteamVR for the development of this project, as their built in tools and models expedited our development process given our time constraint.

Challenges we ran into

The biggest issue we ran into was the learning curve of picking up how to develop on Unity. We were all foreign to the development environment, so that and the bugs that naturally show up in any development process were the most notable challenges.

Accomplishments that we're proud of

Even though we only had 36 hours to develop this game, we built it to the point where we feel we can release it to the public given some more polishing.

What we learned

We learned many things about Unity and how it works, and how game development as a whole works. Learning how the internal components of everything that shows up in the game meshes together was also a great experience.

What's next for Gondola

Now comes the process of refinement. We would like to make new game modes and add more levels. We would also like to add more items like different types of bows, arrows, objects that can be thrown, etc.
