Inspiration

We were inspired by numerous experiences when buying a new smartphone at mobile stores.

What it does

It's a bot based app that based on AI can answer to questions more likely human. It has the ability to learn on the run an develop itself.

How I built it

We used IBM Watson Assistant and have integrated it using node.js into a web application.

Challenges I ran into

All technology we have used was new to us, so it was all a great challenge

Accomplishments that I'm proud of

Building a great team and spending time together while succeding to make the chatbot work.

What I learned

I learned many things about the ibm cloud framework, about how to establish connection between client-server-database and team work experience is a big plus.

What's next for Project void

We want to integrate new technologies(e.g. Speech Recognition) into our chatbot to make it more remarkable. We want also to get better at developing chatbots.
