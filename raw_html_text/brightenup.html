Inspiration

I wanted to help those who have no one to turn to when they are feeling down.

What it does

It provides a beautiful interface to help users communicate and provides tips to feel better.

How I built it

Ionic, Angular 2, HTML/CSS, TypeScript, What's-Next API

Challenges I ran into

Figuring out how to process the text input of the user

Accomplishments that I'm proud of

Figuring out how to create a more modern user interface.

What I learned

Front end design, working with APIs.

What's next for brightenUp

Provide more tailored responses, provide a way to connect users with each other to have someone to talk to.
