Inspiration

The idea to integrate hardware and software to work together in a way that benefits people, especially in the classroom is an idea that we all wanted to execute. Even when it seemed impossible, we kept going with this idea.

What it does

Alexa Classroom Integration connects Alexa with a text file through "If This Then That". A java program works to execute any commands or programs classroom related.

How we built it

We used IFTT to interface the Alexa input to a text file. We then created a java program that executes the desired program or command.

Challenges we ran into

Alexa is relatively closed with outputting what the user is saying to it, so finding a reliable way to get this was the largest challenge we ran into.

Accomplishments that we're proud of

Creating a program that is able to interface with another system seamlessly was a really big achievement that we're proud of.

What we learned

We learned the basics of all the cloud platforms that were given to us, before determining that it would not be the best idea for this project for the time being. We also learned how to interface with Alexa and create different "Skills". Finally, we also learned how to work with built-in Windows tools.

What's next for Alexa Classroom Integration

Continue to add more features and eventually move all the processing to the cloud rather than in a local machine.
