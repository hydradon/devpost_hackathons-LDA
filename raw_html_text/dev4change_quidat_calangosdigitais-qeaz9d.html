Inspiration

Qui Dat - Aquele que Doa. Um chatbot para conectar pessoas com a causa do GRAACC.

What it does

O QuiDatBot é um chatbot que disponibiliza de forma fácil e rápida informações sobre o GRAACC através do Telegram, e também apresenta as formas de doação, facilitando o ato de doar, já que hoje tudo pode ser feito através de um smartphone. Também podemos conferir como é agendada uma visita, e, caso deseje, marcar pelo próprio chatbot.

How we built it

Utilizamos a API do Telegram para integrar o chatbot usando o framework node-telegram-bot-api link . Esse framework é executado em um container do docker usando o node.js como servidor. Para a funcionalidade do e-mail, utilizamos o framework nodemailer link . Todos os frameworks são open source.

Challenges we ran into

  • Aprender uma API nova;
  • Descobrir projetos open source para resolver os problemas;
  • Conseguir realizar a configuração correta do docker com o node.js e os frameworks na máquina virtual da AWS.

Accomplishments that we are proud of

  • Escrever um chatbot em menos de 40 horas;
  • Integrar diferentes frameworks para atingir um objetivo;
  • Aprender novas configurações do docker para automatizar o processo de deploy.

What I learned

  • Escrever um chatbot é mais complexo do que os tutoriais mostram;
  • Encarar desafios e tomar risco em áreas desconhecidas podem ser muito gratificantes;
  • Não ter medo de errar, mas sim ter medo de permanecer errado;
  • Doar seu tempo e seu conhecimento para fazer o bem não tem preço.

What's next for dev4change_quidat_calangosdigitais

  • Integrar o chatbot com uma base de dados para buscar informações;
  • Integrar a funcionalidade de agendar visita com uma agenda virtual.
