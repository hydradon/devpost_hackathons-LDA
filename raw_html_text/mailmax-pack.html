Inspiration

Formed at the intersection of different platforms, we hoped to utilize present utilities and services to empower the average user with a more powerful integration between phone, cloud, and email

What it does

It comprises of 5 activities. The first one being a utility which sends calls the user as a notification specific to the user's keyword in the subject of the email.The second being a utility which sends the user a notification via SMS specific to the user's keyword in the subject of the email. The third utility allows the user to search their Box storage to retrieve the requested file and add it to their email as an attachment within seconds. A fourth utility is an email command which allows the user to send the email as an SMS to any requested number. The last but not least utility searches through the list of hackathons in 2017 and allows the user to send a specific hackathon's details to another user within seconds.

How we built it

We have made many Mixmax backslash commands and the Twilio API to allow the user to better integrate with their phone, cloud storage, and email.

Challenges we ran into

We were having a hard time getting the authorization for the user's cloud storage for Google Drive and One Drive. We were unable to oauth.

Accomplishments that we're proud of

We are proud that we were able to add so many features in 36 hours and integrate so many different functionaities in them.
