Inspiration

We were inspired to create FlexiBook as an easier way to book flights and plan vacations. Vacation planning is difficult and stressful.

What it does

The focus of FlexiBook is flexible booking for plane tickets. The search allows the users to choose multiple stops and connecting flights through different airlines on the same website. This allows people to stop in as many places as they want while traveling.

How we built it

We used html/css to build the websites, python to build the servers, and swift to build the iPhone app.

Challenges we ran into

We had some trouble with the server connecting to the websites, so we created an app and a website

Accomplishments that we're proud of

We made both an app for iPhones and a website

What we learned

We learned how challenging it can be when creating separate parts of a project to combine them

What's next for FlexiBook

We hope to continue to improve both the FlexiBook app and website with more links and helpful travel hacks
