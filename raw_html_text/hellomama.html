Inspiration

The zeal to provide solutions to problems in the community led to the development of hellomamabot. It was built for social good to help reduce the rate of child mortality.

What it does

Hellomamabot sends age and stage based message tips to pregnant and new mums to reduce the rate of child mortality. It allows mums to ask questions pertaining to their pregnancy or child birth from real life doctors for free. It also provides an online HTML5 quiz platform to test mums understanding of message tips sent weekly.

How we built hellomamabot

Hellomamabot was initially built on telegram using telegram bot API. During the course of this hackathon we built hellomamabot on Facebook messenger using the Facebook messenger API, PHP, MySQL and redis. The quiz engine was built using HTML5, CSS and JavaScript.

Challenges we ran into

We had difficulty getting approval from Facebook to opt in to use the subscription messaging to send messages outside the 24-hour standard messaging window. We eventually got approval after several attempts.

Accomplishments that we're proud of

We're happy hellomamabot is live on Facebook messenger and mums are already benefitting from it for free.

What's next for Hellomama

Hellomama currently serves mums in Nigeria. We hope to expand to other countries in the nearest future.
