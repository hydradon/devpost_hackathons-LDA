Inspiration

We wanted to create an app that allowed for easy collaboration between classmates. Especially as freshman, it is tough to find study groups on a new campus like this.

What it does

Study buddy is a social media platform connecting students with study groups. It has a home dashboard with relevant groups for the user and the groups they are already in. Users can search for groups based on their schedules, class, and specific questions. They can friend other users to keep track of people's study groups if they are in similar classes. There is a log-in feature, and users are only able to see others from their own university.

How we built it

We used Android studio to build the majority of the app. We tried to use Ruby on Rails to implement the backend, but ultimately we were unable to successfully link the backend framework with our app.

Challenges we ran into

The lack of ability to link the backend with the app made completing this project quite difficult.

Accomplishments that we're proud of

We are happy with the extensive UI we developed. Two of the three members of our group have never used android studio before, and so it was great to see our skills develop.

What we learned

We learned android studio, and even though we were ultimately unsuccessful, learned about how to integrate backend frameworks into android apps.

What's next for StudyBuddy

Hopefully a working backend, as well as multi-platform availability.
