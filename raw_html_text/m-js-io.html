Inspiration

As hackathon goers, we want to capture the Hackathon awesomeness with our hack and, this time, we decided to do so by making a language for it!

What it does

It converts your code to M.JS.IO, which can be run later, as well as converts the BlackRock API to our language and utilizes Twilio to send a SMS message with the M.JS.IO code that sent it

How I built it

Converting the six characters in JSF*** to "M", "HACK", ...

Challenges I ran into

Half our team ditched us during MHacks so this is what we had come up with during the last day

Accomplishments that I'm proud of

Actually completing a hackathon project to the point of being able to say "I've done something appreciable"

What I learned

Replacing characters with words makes things so much cooler

What's next for M.JS.IO

To make appearances at future hackathons but with the theme of those hackathons instead (Penn.JS.IO, etc.)
