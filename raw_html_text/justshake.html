JustShake

It is a shaking game developed in MHack X. Do you saw this before? https://youtu.be/H4wOmJ5TULc

We want to make every mobile phone become controllers and enjoy games (or concerts) with everyone. So we need to extract the sensor information by Android Apps. Send the data to the server and finally interactive with a web page with a big screen.
