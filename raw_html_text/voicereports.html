VoiceReports skill allows to generate voice based status reports from the business application to the user. Business Applications send the voice reports through REST API to the users voice reports folder. Once voice reports are arrived to user's folder user then can ask alexa to read the voice reports. VoiceReports also facilitates to read the voice reports based on date. In order to utilize the VoiceReports skill user must link Alexa account to the Voice Reports user account. Once the account is linked business application can send the voice reports in the via the REST API to the user.

This application is build using Node JS and Alexa Skill Kit.

Voice Reports can be extended to read the voice reports based on keyword or the title of the reports.
