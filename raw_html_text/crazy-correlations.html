Inspiration

SMBC's Conception Connection

What other crazy correlations can we create? This web application pulls all the tweets of people the user is following and applies machine learning analytics to see what topics are trending among the user's friends. Those topics are then matched with a current event that occurred on the same day, possibly related to the topics of the tweets. Coincidence? You decide!
