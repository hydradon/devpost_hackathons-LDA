Inspiration

We wanted to see if we could generate a heatmap and render it on a site if we supplied the data, and we could! So we used that ability to make a heatmap of vacant houses to be able to better visualize the different neighborhoods of Baltimore.

What it does

It renders a heatmap of vacant houses in Baltimore.

How we built it

We have a simple HTML webpage with a Javascript portion that gets the info and renders the heatmap

Challenges we ran into

We wanted to further implement NodeJS and React frameworks but time constraints only allowed us to learn about them.

Accomplishments that we're proud of

We managed to get something hosted and working!

What we learned

We learned how to host websites on GitHub, how to use HTML and Javascript, and we started learning how to use NodeJS and other Javascript frameworks.

What's next for Vacant-House-Heatmap

Perhaps we could add information for other cities, or compare two heatmaps for different statistics in order to draw correlations about the Baltimore community. https://dendygar.github.io/Vacant-House-Heatmap/
