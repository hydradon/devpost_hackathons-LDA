Inspiration

As preferências culturais de cada consumidor são uma fonte de inspiração para a decisão sobre suas futuras viagens

What it does

O aplicativo é um quiz que sugere destinos de viagem e épocas para realizá-la em função dos filmes e séries preferidos do usuário

How I built it

Com a API da Sabre para consulta de passagens, quiz desenvolvido com PHP, HTML e CSS

Challenges I ran into

A construção do sistema de ranking dos filmes e séries e o limite de tempo para executar a proposta

Accomplishments that I'm proud of

A solução visual, a usabilidade e o inter-relacionamento de duas bases de dados diferentes: filmes/séries e destinos turísticos

What I learned

Adaptar o escopo do projeto ao limite de tempo proposto
