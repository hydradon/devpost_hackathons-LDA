Inspiration

Frustrated by long airport taxi lines and unwilling to inconvenience family or friends, the Port.ly team set out to find an easier solution. Simply input your flight number and be automatically chauffeured to and from the airport.

What it does

Using live flight tracking data and the Uber API, Port.ly knows when to pick you up to get you to the airport on time and has a car waiting for you when you land, regardless of delays.

How we built it

Front-end: Objective-C / Uber API Back-end: Node.JS / Python / Azure / Uber API
