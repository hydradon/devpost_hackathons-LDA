Inspiration

To help prevent exercise related injuries

What it does

Uses the Clarifai API to tell if you are doing an exercise correctly

How we built it

We used Android Studio as the IDE, java as the programming language, and Clarifai API

Challenges we ran into

Clarifai not recognizing certain picture sizes

Accomplishments that we're proud of

One click away from having the website up and running, and the website name: "monopose.com"

What we learned

We learned how to collaborate with complete strangers and accomplish much more than we thought we would

What's next for Monopose

We already have a gym that is willing to put up the QR codes on the workout machines
