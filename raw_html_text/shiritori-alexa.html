What it does

You can play the word game on alexa to help train or develop vocabulary.

How I built it

It's built on a lambda function using the alexa skills kit package.

Challenges I ran into

Intent schemas didn't work as well as I planned, and responses were triggering the wrong intents, which caused slots to be wrong.

What's next for Shiritori Alexa

Will add more classifications for words as well as giving the word a dictionary definition by hooking it up to an English dictionary. This being implemented within the backend.
