Inspiration

We were inspired by the opening ceremony speech about wild life in Egypt and were guided by some posts and articles from international journals.

How we built this?

We built this project using some kind of technologies such as python programming, Pyqt for GUI, Database, Deep learning models (binary classifiers), Adobe Xd for UI/UX and Node MCU.

Challenges

One of the challenges we ran into is collecting dataset for the deep learning models to train and the evaluate the model on sea turtles and plastic bags images. And YES! we did it we accomplished a demo for our project and having a clear idea in just 24 hours only is a thing that we are proud of. We have learned how to manage our time and focus on what we can do and get the best use of it.
