Inspiration

After using sites like Craiglist and the facebook page UT buy sell trade we were annoyed on how they were set up. The UT buy sell trade was difficult to search since there was no overarching organization. Craglist also seemed a bit sketchy since anyone could sign up and make posting.

What it does

It provides a site similar to Craiglist in the fact that you post things that you want to sell and make a meeting with a person. However you are required to sign up with a valid college email.

How we built it

We used react, javascript, html to create the front end. We hosted our front end for testing on webpack and created the backend using sql and php.

Challenges we ran into

We original planed to use angular but we couldn't figure out set up directives how we wanted to. We also ran into issues setting up the sever connection and deleteing items that took up space.

Accomplishments that we're proud of

We have a useable site and got the database working.

What we learned

We learned how to create a database in sql, use javascript in react, and some quirky methody in javascript like email js.

What's next for buymyjunk

We want to change up the ui to make it look nicer and add more search functionality
