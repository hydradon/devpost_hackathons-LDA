OLGA

Open-source Language Guided Assistant
This project is meant to be a substitute for services like Alexa or OK Google. These other services collect your data, and don't tell you when and how they use it. This project will bring the data back into the hands of the users. These services also have little to no customization. OLGA also solves this problem by allowing users to create their own personal extensions to add functionality to OLGA.
