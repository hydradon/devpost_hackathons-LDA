Inspiration

Online gaming is becoming very popular in the Blockchain space so leveraging a service like Crypto Kitties is a no brainer.

What it does

Allows you to play bingo with other CryptoKitty enthusiasts, using to the Cattributes of the displayed Kitties for mark up your board.

How I built it

Built using a web3 architecture with the application being on multiple blockchains in addition ethereums.
