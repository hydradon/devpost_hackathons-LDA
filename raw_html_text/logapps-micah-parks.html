What it does

The user selects a file through the GUI to be parsed for Natural Language Processing (NLP) data. After the file is chosen, two windows will pop up, one after the other, displaying the specified parts of the challenge. Part 3 was omitted.

How I built it

With Python and only two dependencies: NLTK and easygui

Challenges I ran into

Accomplishments that I'm proud of
