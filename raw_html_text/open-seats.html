Inspiration

Students spend many hours refreshing a registration page in the hopes to get an open seat. We wanted to automate that.

What it does

Automate the class checking process.

How we built it

We built a simple front end that will allow the user to add in his/her information. Then the script sends a get request from our heroku server, which will scrape the website with the user's input and will send a text to the number if it is available. If the class is not available it will keep running the request on the cloud.

Challenges we ran into

Different schools have different catalogs, therefore we had to use a specific school as an example.

Accomplishments that we're proud of

It works and it deploys. It actually texts.

What we learned

How to use twilio and deploy to heroku.

What's next for Open Seats
