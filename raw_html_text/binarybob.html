Implementing a 2 player game with binary questions that supports learning of different number systems.

Generating and showing questions to players. Getting and proceeding input. Takes hardware input and shows game status over LEDs.

Running in browsers using JS and calls to firebase database.

Problems with generating input over different devices.

Super nice design and reaction time.
