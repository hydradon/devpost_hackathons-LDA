Inspiration

The tragic events that have transpired in 2018 (i.e. Parkland School Shooting)

What it does

In the case of urgent situations where it is harder to get the administrations attention, students can use the app to give information to the administration who can then take the action necessary to ensure the safety and well-being of the students

Credentials for testing are located in the README

How we built it

created rest API used Alamofire to allow app to communicate with server and database used Xcode to develop UI
