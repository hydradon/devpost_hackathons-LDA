Inspiration

We wanted to make everyday life easier by removing friction from computer control.

What it does

AirMouse allows you to control your mouse through hand gestures. Use the pointing branch of the repository for touch screen-esque control (Recommended). Use the master branch for joy stick-esque control.

How we built it

Programmed using PyUserInput and the LeapMotion API in Python

Challenges we ran into

Calibrating the mouse motions to feel natural was the largest challenge.

Accomplishments that we're proud of

Being able to draw art in MSPaint using the pointing branch.

What we learned

We learned multiple algorithms for clicking and moving.

What's next for AirMouse

AirMouse can become more functional with the introduction of more gestures to access more commands.
