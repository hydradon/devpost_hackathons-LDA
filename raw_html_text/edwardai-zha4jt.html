What it does

Its a chatbot that uses artificial intelligence to learn how to best use your insurance policies! "Serverless app deployed using AWS, including S3 Buckets for static file hosting, API Gateway combined with Lambda Functions for the API, and Cognito Pools to deal with authentication. DynamoDB used as a DB."
