Inspiration

We wanted to make a CAD application using VR and finger tracking, for export to 3D printers

What it does

Some stuff with cubes and spheres - it's actually quite cool but it's definitely no CAD application (just yet)

How we built it

Using the Unity and the Leap Motion SDK

Challenges we ran into

Unity cannot be learnt in a few hours (we expected it to be easy!). Also the leap motion documentation is quite bad (the SDK is in beta)

Accomplishments that we're proud of

We managed to get started but we were consistently blocked by problems with Unity

What we learned

Finger tracking isn't ready for the primetime yet

What's next for Oculus Leap

Finishing it! Using Oculus Touch would be better than leap
