Inspiration

A cool game built from dots

What it does

You can choose to either dodge dots or collect them, the objective for collecting is 10 dots, and the objective for dodging is 20. For each game, you have 5 lives

How I built it

I coded it using javascript blocks

Challenges I ran into

Checking for pressed buttons while running the other dots

Accomplishments that I'm proud of

I made a pretty good game out of nothing

What I learned

How to use the Microbit

What's next for The Dots

Upgrade to a larger LED screen with colors, and different difficulties.
