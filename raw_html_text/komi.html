Inspiration

We build this application named Komi, Komi is coming from community, by community, and to community. We solve those problem that are really common for organizing the community such as, manual process, registration, hard to find meetup location, find event, etc

What it does

Komi help us to easier manage communities, we can look for member, event, or freebies

How I built it

I build using messenger bot, because user doesn't need to install another app. Just send message to bot and get the information!

Challenges I ran into

How to make Komi simple and easy to use

Accomplishments that I'm proud of

I learn to build it in hackathon

What I learned

Learn about messenger bot implementation

What's next for Komi

More feature to connect between communities
