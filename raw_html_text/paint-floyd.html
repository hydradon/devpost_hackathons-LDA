• Paint Floyd before its prime

Inspiration

We want people to accept the childlike imagination blooming inside each of their hearts. One of the best ways we do this is by painting! So we want to make that process easier for people by mixing the colors for them automatically.

What it does

The user simply inputs their desired color and Paint Floyd mixes the color from five basic colors, CMYKW.

How we built it

We designed and built an 80-20 frame, a, automatic, cylindrical dispensing system using syringes and linear rod actuation, and controlled our device using motors and a photosensor for color feedback.

Challenges we ran into

Figuring out how to make any color given standard RGB, RYB, and CMYK color mehcanismsDriving stepper motors! Smooth linear actuation without bearings!

Accomplishments that we're proud of

Building a beautiful compact design and finally getting the stepper motors to work for a final working product!

What we learned

Stepper motor driving should be locked down and done before anything else.

What's next for Paint Floyd

Compiling lots of data to make RYB color creation possible based on from an ML algorithm.
