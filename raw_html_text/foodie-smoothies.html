Inspiration

We are always struggling to decide what to make with the food that we have, so we decided to make an application to give us suggestions on what we can do with the food that we have.

What it does

We are building a web application where users can take pictures of their pantry, refrigerator, and freezer. The application will keep track of the food they have and make cooking recipe recommendations based on the ingredients the user has.

How we built it

We built it using google's web app platform.

Challenges we ran into

We had difficulty in setting up the tutorial and getting it to actually work. Other challenges that we came across were being able to communicate between Google's cloud buckets and our web app, and heavy eye strain.

Accomplishments that we're proud of

We are proud that we got the web app to work, and can read at least two fruits as once.

What we learned

We learned how to use Google Cloud Platform to build a web application, and we learned how to use Google Cloud Platform Vision. Even though we struggled through it, we definitely learned a lot. But most importantly, we learned how to stay positive when nothing is working, and we learned to read things carefully the first time, so we don't have to go back and read it.

What's next for Foodie Smoothies

We want to be able to read multiple fruits of the same kind, and be able to read any food to make all recipes, not just smoothies
