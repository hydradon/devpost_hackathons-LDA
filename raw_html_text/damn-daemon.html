Inspiration

The name is inspired from the Damn Daniel Vine meme.

What it does

It simplifies the tedious tasks needed for developers to deploy their apps as daemons so they're always running.

How I built it

I used a combination of python and bash scripting to effectively speed up as well as simplify the steps to make a daemon with an app on a Linux system.

What's next for Damn Daemon

More interaction with daemons including logging, start/stop, and more!
