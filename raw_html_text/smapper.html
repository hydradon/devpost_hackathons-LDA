Shmapper

The best Shmapper app in da land!

Shmapper is a cool and new way to send information from your dragonboard, website, app, etc to interface with your 3D data.

Using Node.JS to host the server, we seen data via HTTP POST request and also UDP packets where the server then formulates it and sends it out to the front end via WebSockets. This allows you to real time interact with 3D data built with Three.js.

This is mainly a showcase of 3 first time hackers and what can be done from knowing nothing at all about servers, graphics, or web dev in the course of 36 hours... also the domain shmapper.com is ours so who is the real winner after all?
