• Welcome to Shopby

Inspiration

Facebook's Messenger introduced a new feature called chat bots at the same time as Shop.com introduced their Hackathon. Combining the 2 seemed like a natural peanut and jelly sandwich. Rather than building a mobile app this made a lot more sense, Messenger runs on all devices (desktop and mobile) and Messenger already has 900,000,000 users.

What it does

In Messenger type in "ShopbyBot', select it, you now have an interface into Shop.com's product line.

How we built it

Created a custom API to interface between Facebook, Shop.com and a BaaS (Backend-as-a-Service). Lisa also created custom graphics to help the users feel comfortable.

Challenges we ran into

Messenger's interface is very limited as opposed to that of a mobile app or webpage.

Accomplishments that we're proud of

Just pulling it all together is the accomplishment.

What we learned

Chat bots have a big future along with Artificial Intelligence

What's next for ShopByBot

Hopefully to build pass the demo mode to a real product.
