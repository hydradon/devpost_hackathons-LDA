Inspiration

We were inspired by a group member's experience of having limited resources but being unable to use those resources optimally.

What it does

Alpha Tech Squared uses an algorithm to allocate the listed resources in a way that would maximize consumption and efficiency.

How we built it

We used an HTML front-end to display the user interface for our web app, and coded the algorithm using PHP and AJAX to bring it all together.

Challenges we ran into

The algorithm was difficult to code since it required efficiency while simultaneously eliminating extraneous solutions. Meanwhile, the UI had to be easily understood while preserving functionality. Finally, AJAX had to be coded to successfully convert PHP to HTML so the front and back ends could communicate

Accomplishments that we're proud of

We are proud of having code that adequately performs its task to accompany the clean and minimal UI. Most significantly, we stuck with the idea that we were passionate about despite the obstacles and hardships that stood in our way.

What we learned

We learned how to ration our time between tasks and streamline our workflow to increase efficiency while meeting the deadline. We learned that not all coding problems can be approached the same way, and that different mindsets must be respected in order to find the correct angle to find the solution.

What's next for Alpha Tech Squared

We plan to implement priorities for tasks as well as scale up the website to meet the manufacturing needs of large companies such as the pioneers of Silicon Valley.
