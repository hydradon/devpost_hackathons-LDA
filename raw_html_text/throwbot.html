Inspiration

We were inspired by the terminator movie. Because it has robots.

What it does

It throws projectiles at a reasonable distance.

How we built it

We built it using arduino and java, and a bit of tears and suffering. Along with our custom made 3D printer.

Challenges we ran into

Fatigue, and learning new things that we have never learned in class.

Accomplishments that we're proud of

We managed to implement serial communication between java and arduino, made a GUI to control to parameters of the throwing motion, printed and built the throwing arm during the competition and we made the stepper motor move using the arduino IDE.

What we learned

We learned that serial communication is finicky, time is much needed for big projects. We all tackled new challenges during the hackathon.

What's next for ThrowBot

Not much actually.
