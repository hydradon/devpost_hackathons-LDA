Inspiration

Winter is coming. Our team thought that people would like to know about the weather in the morning before they made their daily commute to work or school. We decided design a device that somebody could keep beside their bedside like an alarm clock but displayed the weather instead of the time.

What it does

Displays the weather forecast when the user gestures over the box.

How I built it

Using an arduino and an lcd display

Challenges I ran into

Getting the motion sensor to work

Accomplishments that I'm proud of

Learning to code in arduino

What I learned

How to code in arduino

What's next for Arduino Weather Box

More functions e.g. time
