Inspiration

I'm inspired by a natural user interaction and not downloading another app somewhere.

What it does

You can ask the bot to tell you what desks/meeting rooms are free. Demo functionality considering available desk distance from a person. People are considered to all be on the 7th floor for this hackathon.

How I built it

  • nodejs,
  • wit.ai
  • facebook messenger api

Challenges I ran into

  • Messages you post in quick succession to fb messenger api will not necessarily show up in order in your chat window.
  • branching conditionals in wit.ai new and without many demos.

Accomplishments that I'm proud of

Setting up my first bot to be able to answer some basic useful questions

What I learned

Learned how to setup and start working to build interactive messenger bots.

What's next for TietoBot

  • Users can ask about lunch
  • find a desk with minimal noise level
  • checking who is in a current meeting
  • check if certain people/groups are in the office
  • check parking lot status (spaces available)
