Inspiration

All the experiences of what we wanted to eat, but never knew where to go.

What it does

Input any fast-food item that you want to eat and we generate the best/closest place to go to.

How we built it

Using a lot of research, we configured the google-maps-api to geolocate any user and pinpoint their best restaurant location choices along with basic html, javascript, and css.

Challenges we ran into

google-maps-api not working on a secure server, working perfectly on a local machine, but not on a web browser.

Accomplishments that we're proud of

learning 3 languages in 36 hours.

What we learned

How complicated html mixed with javascript can be.

What's next for KHE2016

Continuously coming out to more and more hackathons; this being our first.
