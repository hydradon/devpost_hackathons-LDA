Inspiration

to help tourist to know what's the correct price for an item

What it does

It can take a picture of the item and then upload it on a forum to ask people for the correct price of it. If the tourist do not know what item they are taking picture with, the app automatically generates a label for that image.

How we built it

we built this ios app with Swift, we used the Google Vision API to help us the identify what is the item in the picture

Challenges we ran into

how to use database with MongoDB

Accomplishments that we're proud of

We were able to separate the project and work together

What we learned

we learned how to use Google Vision API
