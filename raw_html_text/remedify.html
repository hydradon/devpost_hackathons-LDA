Inspiration

While struggling to identify problems in the world that still persist today, we decided to tackle expensive healthcare and make finding affordable health solutions a viable option by making easy-to-make natural remedy recipes accessible to all. The REST API can be accessed by developers anywhere that are willing to incorporate the collective data in any way they seem the best fit to provide this information to those that need it.

What it does

The REST API provides access to the database we created with affordable natural remedy recipes that many family households typically have access to.

How we built it

We developed a parsing script with Python and Beautiful Soup. We scraped data for our database from natural remedy recipe blogs, articles, and other sources. Remedify API was built with node.js, express, and mongoDB.

Challenges we ran into

Figuring out how to make traditional health remedies accessible to developers so they can utilize the data and further propose solutions.

Accomplishments that we're proud of

Being able to deploy the REST API, and setting up the API which was very difficult and made us reconsider our careers.

What we learned

How to build the API, "crawl" through the web and acquire data for a database, and successfully deploy a REST API.

What's next for Remedify

Having developers access the API and expand on the open source that can eventually be integrated into different technologies such as Alexa and others.
