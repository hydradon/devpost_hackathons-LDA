Inspiration

The idea was made by Mohammed Wahdani. He thought that paying for college can be a pain and thought of a way to have students help each other in this regard. If a student needs help with something, they can pay another student to help, hence the name Double Help.

What it does

It creates account and saves them. It also logs the accounts in if they match a saved account. After that, the user can etiher search for available jobs, which are also saved, create a job, or just sign out.

How I built it

I used JavaFX to create this project since JavaFX is GUI-based. As for the rest of the non-GUI functions, I just used conventional java coding to make the project.

Challenges I ran into

I had to get somewhat of a refresher on JavaFX while coding, and as a result, I ran into several errors.

Accomplishments that I'm proud of

I am proud to have made a project of this magnitude and having to code alone in about 8-9 hours of actual coding.

What I learned

I learned that I need to find more people who can code alongside me to made programming much easier.

What's next for Hacklahoma-2018

If anyone becomes interested in the idea, I plan on expanding to code to make it an official website.
