Inspiration

We wanted to inform the users where to dispose or recycle an item.

What it does

The user takes a picture with your phone of an item and it will show you the nearest location where you can recycle or dispose of it.

How we built it

We used React Native and expo to build the item scanner. We then used the Google Cloud Vision to identify the scanned items.

Challenges we ran into

We faced lots of server-side challenges as we were new to react native and node js.

Accomplishments that we're proud of

We were able to use Google's machine learning API to show what an object is based on the pictures taken.

What we learned

We learned new technologies and how to use APIs along with React Native and node js.

What's next for Recycle Map

We want to fully automate this and polish the app.
