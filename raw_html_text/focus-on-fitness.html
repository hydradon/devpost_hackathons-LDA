Inspiration

We wanted to combat the obesity epidemic in today's world because it is a massive issue for the general public. While the internet is a fantastic place for innovation and connectivity, it does facilitate being idle, and this is a large proponent to the issue of obesity.

What it does

We are in the process of developing a web blocker that can be connected to a person's smartphone. For one hour, the user will be blocked from online distractions on their computer, and while they work out, they can track activity from their smartphone.

How we built it

We used HTML5, CSS, JavaScript, and the Chrome web development kit to make an chrome extension.

Challenges we ran into

Only one team member knew how to code, so we needed to start from scratch and learn Javascript. We also were unable to learn Apple's coding software in the time provided at HackNYU, so we were only able to focus on the chrome extension. Finally, we were still unable to get the program to block websites with our limited coding knowledge

Accomplishments that we're proud of

We were able to learn the basics of javascript and build our knowledge of HTML and CSS through the project, and we have a greater understanding of how some code works

What we learned

Javascript programming, HTML, how to make a timer in Javascript, how to make a chrome plugin

What's next for Focus on Fitness

  • Develop the IOS App
  • Use Firebase to have an account service so people can track their exercise
  • Upgrade the Chrome extension
  • Enable the blocking of websites
