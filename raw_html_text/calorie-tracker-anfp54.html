Inspiration

We wanted to provide nutrition facts to users in a simple and systematic manner, and allows users to visuallize their caloric intake

What it does

This android app keeps track of the food you consume and will provide you with nutrition info about them, as well as how much calories you've consumed.

How we built it

We used Android Studio as the IDE for building the android application, and we used the FatSecrets REST API to provide information about the various foods. For database, we connected our application with Firebase

Challenges we ran into

The FatSecrets API was very hard to use, and setting up HTTP requests was a pain. The UI was also a struggle

Accomplishments that we're proud of

What we learned

We learned a lot about the organization of REST API, as well as database querying and mobile application development in general.

What's next for Calorie Tracker

We plan to add machine learning algorithms that keep track of users' eating habits and will provide recommendations as to which foods to consume.
