Inspiration

Our friend group would never know what everyone was up to at night.

What it does

Tracks what a friend's/friend group's plans are for the night.

How I built it

We made a RESTful Tomcat API on AWS-EC2 that abstracted the Uber, Twilio, and Capital One Nessie API's. We have the main app hosted on Azure. It is a pure php/html/js implementation.

We plan to use manifold.js to create native experiences for squawc.
