AirMed

AirMed is an all-purpose healthcare app built on the Windows Universal Platform. AirMed leverages Microsoft Cortana and Clarifai's machine-learning image recognition to create an easy-to-use yet incredibly accurate app for diagnosing and treating illness. AirMed is able to handle textual, audio and photo inputs, which in combination with geographical information, like local disease outbreak and population health trends, make for a better diagnosis than online medical apps. AirMed is easy enough to use that people without access to healthcare professionals can still receive proper treatment.
