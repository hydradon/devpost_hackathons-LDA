Inspiration

The wide amount of miscommunications between legislators and citizens about changes made to areas of interest.

What it does

RepChat will allow its users to post petitions in a website similar to a forum. After these petitions get a certain amount of up-votes, RepChat will post a tweet containing the petition and its details, tagging the query handle of the representative of choice.

How we built it

We made RepChat in two stages: 1) HTML Website structure, containing the frame and petitions waiting to be up-voted. 2) Integration of the Twitter API so we could make an impact on our surrounding communities

Challenges we ran into

Building a successful search system, applying a proper submission system, formatting, finding a reliable database, and searching through the selected database

Accomplishments that we're proud of

Some accomplishments we're proud of include using FIREBASE by Google successfully, learning Java Script and html, and overcoming restarting our project with strong teamwork.

What we learned

We learned advanced Website building, HTML, Java Script, and how to use FIREBASE by Google.

What's next for Hack TJ project

We would integrate the twitter system and the website into one live domain, as well as including a refined searching system
