Inspiration

Desperation... but also we like FPS games.

What it does

Two futuristic pods are pitted against each other, both equipped with bouncy exploding grenades. The battle takes place in a futuristic shipyard full of vibrant colors.

How we built it

Desperation, inspiration, perspiration, and perseverance. And a lot of caffeine. We used a game engine as well as some C++ coding.

Challenges we ran into

Building the map, creating the bullets, controlling the play, connection for multiplayer, and working with a 3D engine.

Accomplishments that we're proud of

We made something. It's multiplayer and you shoot exploding grenades.

What we learned

We learned TEAMWORK and how much sleep the human body requires, as well as how hard it is to work with a 3D engine and create the logic of a FPS.

What's next for Ricochet

A dedicated server, map expansion, map packs, skins for the play models, new weapons, more futuristic stuff ;)
