Using the EveryBlock API, we wanted to create an app that lets users know of recent criminal activity in their proximity. By using this app, users can avoid any possible danger save them from becoming another victim of theft, murder, etc. We are especially proud of the fact that we had no previous experience in iOS app development/ Swift, but we're still able to produce a decent app. Hopefully in the future, we can add more features to improve the app.
