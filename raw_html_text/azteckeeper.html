Inspiration

The lack of alternatives to the current highly centralized internet.

What it does

Allow users to send zero knowledge proofs in web3 transactions through a proxy IP address without having to use a vpn.

How We built it

We used a lora mesh library https://github.com/nootropicdesign/lora-mesh to create a network of Pycom Arduino devices to relay signed Ethereum transactions to a node obfuscating their origin. The devices are programmed in C++. To Generate Zero Knowledge Proofs we used the Aztec Protocols javascript libraries and solidity smart contracts. We integrated the Aztec browser extension into a react front end allowing users to generate proofs which are then sent to an express server that forwards the proof data to the mesh network.
