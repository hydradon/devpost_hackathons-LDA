Inspiration

Using the premise of Khan Academy to assist senior citizens in learning different common technologies.

What it does

Technior Ed is a library of videos that show senior citizens, in a clean and concise way, tutorials to different technologies that are relevant and useful to them.

How we built it

A standard HTML/CSS website with JavaScript and JQuery functionality, as well as a PHP and MySQL backend to store user login information.

What's next for Technior Ed

Adding our own content and interactive material to help users practice the material that they've learned.
