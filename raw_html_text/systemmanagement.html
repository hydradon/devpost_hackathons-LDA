Inspiration

We were initially inspired to create a tool to help manage our time because of times we felt overwhelmed in assignments and studying. This is a tool that could help reduce the stress and feeling of not having enough time to do everything we need to.

Challenges

We lost a lot of time debating between different ideas contemplating over the needed logic for the code. The hardest part of this project is handling all the use cases with the different priorities
