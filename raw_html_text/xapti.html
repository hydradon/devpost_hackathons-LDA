Inspiration

Sometimes I just want to share a file, but don't want to jump through a lot of hoops. This is for easy one time file sharing.

What it does

You can upload a file and specify download and time limits. It either works as a web app or desktop app that communicates with a server.

How we built it

A whole lot of javascript and Node.js (Express). Very vanilla, plain stuff, didn't want to deal with very complicated modern javascript libraries and the like.

Challenges we ran into

Not many, pretty straightforward project

Accomplishments that we're proud of

We got it to work!

What we learned

File verification and checksums, some pretty cool stuff!

What's next for Xapti

Desktop context menus, full scale release on the web, better HTML design.
