Inspiration

While preparing and organizing our trip to Bremen we had a lot of problems with finding the most convenient path, we used lots of flights search engines and there was always some missing features that we needed.

What it does

It helps travelers to find the best and optimal route, in both budget and time.

How we built it

We used Firebase for the authentication, Django Framework for the backend, SkyScanner API, ML algorithms.

Challenges we ran into

Getting familiar with the SkyScanner API. We are only two persons in the team. Time is too short to discover and use the API and for implementing our frontend mockup.

Accomplishments that we're proud of

Adding new features that don't currently exist such as: finding the cheapest or shortest route using smart layovers, choosing the optimal itinerary by first specifying the destination and then the application will list all possible departure points.

What we learned

To Use SkyScanner AP.

What's next for FlyoTrip

Finishing the implementation of all the features.
