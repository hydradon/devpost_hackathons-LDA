How we built it

Building this app was the most exciting part of our lives! Here are the steps we used to build Bleep AI Built a chrome extension that embedded javascript on the pages that we wanted. Developed a backend that downloaded the video processed it (in python and flask) , and served it to the frontend We also created a caching store using cosmosdb, so that we don’t have to do unnecessary processing. To perform Ml calculations we used the Media Services API from Azure to obtain granular speech-to-text and the Sentiment Analysis API to generate the sentiment gradient visualization right below the video.
