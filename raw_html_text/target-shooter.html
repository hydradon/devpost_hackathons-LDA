Inspiration

Wanting to create arcade games.

What it does

Shoots ping pong balls into a target

How we built it

Two cans of pringles forming a T-Shaped shooter, using motors, a push button, a wheel, an arduino and a power supply.

Challenges I ran into

Infrared sensors would not detect interference, making the scoring system non-functional

Accomplishments that I'm proud of

The shooter was successfully completed and operational.

What I learned

How to work better with Arduino and building hardware from scratch using recycled materials.

What's next for Target Shooter

Completing the target to keep a score count and a bottle knockdown
