Inspiration

The only ones capable of communication during a power outage are radio enthusiasts. How many radio enthusiasts do you know?

What it does

Our software aims to create a distributed internet in areas that have had their connection to the outside world destroyed by natural disaster. It uses drones to carry messages back to loved ones and from rescue personnel.

How we built it

Ad-hoc networks and web servers.

Challenges we ran into

Ad-hoc networks on linux are difficult to implement.

Accomplishments that we're proud of

We got the whole system working.

What we learned

We learned a lot about how wifi works.

What's next for SkyHook

We would want to better package the app so that it is more transferrable.
