Inspiration

We wanted to make a changeable model of a city without starting from scratch, so we looked at Google Earth as a means to achieve the model. We wanted to build a tool that allowed us to export parts and pieces of Earth as a .obj file. We then have the capability to modify that file however we wanted. We ended up created an interactive tour of a part of the city within Google Daydream.

What it does

Pick a place in Google Earth and it will make a 3D model of it for you.

How I built it

Using python and recap360

Challenges I ran into

NO API NO Community Support because the new google earth was released ~2 weeks ago. Doing the math to circle around the place taking screenshots Convert the screenshots into 3D images

Accomplishments that I'm proud of

We built a simple VR game using the automated process of building a 3D model from Google Earth as a reference.
