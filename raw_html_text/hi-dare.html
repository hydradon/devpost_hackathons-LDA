Inspiration

"Ice Bucket Challenge" and other kind of social movements attracted a lot of funds into charities. We thought there is a gap waiting to be filled by our app, which just gives to the society.

What it does

By Challenging your friends, you put a price on those dares. When they complete them and provide proof, you need to donate a sum of money to a charity, with the help of JustGiving API

How We built it

The App is built on two servers, one PHP acting as a client and serving the pages, and also validating the data, while the other, written on node.js, performs the API calls and database connections. The front-end is made using bootstrap CSS3 framework and basic HTML and JavaScript.

Challenges we ran into

We had some problems with our node.js hosting, not being able to pass some links.

Accomplishments that we're proud of

We managed to produce a full working prototype of the application.

What I learned

We had the opportunity to increase our knowledge in web application building.

What's next for hi-Dare

We plan to continue development, and also port the application on native mobile, with the help of Ionic Framework.
