Inspiration

We wanted to reduce carbon emissions while engaging and educating children through an app.

What it does

Our program tracks your carbon emissions through timing how long your lights in your household is on for.

How we built it

Using an arduino circuit and the RXTX library in eclipse.

Challenges we ran into

Getting the input from the arduino to use in the java program.

Accomplishments that we're proud of

Successfully connecting the arduino to the java program

What we learned

How to use the RXTX library

What's next for Watt a Waste

Create the program on a grander scale
