Inspiration

Voice control is all the rage, so I wanted users to be able to control anything easily with their voice

What it does

It interfaces with the Tasker Android automation app so that it can control anything Tasker can.

How I built it

I wrote a plugin for Tasker to allow Tasker to be controlled via voice commands

Challenges I ran into

It's hard to figure out the way people mostly want to use voice control. Usually people want to use it via "ok Google" on their phones but there's no open API for it.

Accomplishments that I'm proud of

I was able to get AutoVoice working with natural language on Google Home and the Amazon Echo

What I learned

A lot about android, chrome, Google Home and Amazon Echo APIs
