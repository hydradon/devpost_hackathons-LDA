Inspiration

We wanted to live a healthier lifestyle and understand the health benefits of the food we make.

What it does

Using the Amazon Echo's Alexa interface, we provide a quicker and hands-free access to knowledge about the food that people eat. Alexa Nutribot is a voice activated system that provides nutrition analysis and humor to your daily lifestyle. The Nutribot web application is a nutrition management tool to calendar consumption of food.

How we built it

We built this application using Alexa's Amazon Echo and firebase.

Challenges we ran into

We ran into challenges incorporating firebase's console database with the Amazon echo.

Accomplishments that we're proud of

We are proud of creating our own skill within the Amazon Echo, developing a standalone database in firebase, and rapping with Alexa.

What we learned

We learned how to overcome the technology barrier. One night is not enough time for learning but we did it. We learned the Alexa. We learned the Firebase. We learned how to do chrome extensions. Although we didnt end up using half of these, it was still a great experience.
