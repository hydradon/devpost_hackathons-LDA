Inspiration

We were inspired by the UVA Sustainability and their pitch to make UVA a greener school.

What it does

It allows different dorms to compete by showing users the dorm's overall water, electricity, and energy usage. The dorm with the best overall reduction in usage wins.

How I built it

We built it using Java and Android-Studio.

Challenges I ran into

Trying to get Json to work so we can get real-time data using UVA's API.

Accomplishments that I'm proud of

Getting the app to display values.

What I learned

Java programming in Android-Studio.

What's next for Green Dorm

Getting the API fully integrated with the app and getting data in real time.
