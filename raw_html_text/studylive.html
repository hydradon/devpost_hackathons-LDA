Inspiration

We are introducing a real-time community where students have the opportunity to study with other students who are in the same class.

What it does

StudyLive lets you register with other students from the same school as you. After your logged and enter all your classes in your schedule, StudyLive will show you students that are also in your class that want to study. You can then decide who you want to study with.

How we built it

We used GitHub as our server to launch our website. We then proceeded to use Glitch as our platform to create and share code to build our website. For our language, we mainly used Bootstrap as the backend of our website as its huge library of prebuilt programs using HTML, CSS, and Javascript, we were able to learn much quicker and efficiently on how to code. Namecheap provided us with a free .me domain because we are students.

Challenges we ran into

We have never used GitHub to host our website before which was exciting. Also to be able to use a first-class domain instead of GitHub's own domain required some work to get working properly In terms of coding, most of us never learned Bootstrap or Glitch before and using these was a really surreal experience. Challenges such as formatting and positioning were quickly resolved as we see each other's code real-time and were able to ask for feedback immediately rather than the old fashion having one person with the laptop do it.

Accomplishments that we're proud of

We were able to successfully launch a test website using GitHub into the internet in only a few hours which gave us more time to refine our website.

What's next for StudyLive

Major improvements and innovation to come.
