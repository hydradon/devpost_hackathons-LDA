We all wanted to learn and work with VR and unity so we decided to create a fun game that can help us learn and interact with Oculus, Leap and Unity

An entertainment app that allows you to defend yourself against asteroids being hurled at you

We used unity assets and we learned about Oculus leap on the spot.

A lot of time went into setting up the hardware

First VR game we ever made

Some of us learned unity more, and we all learned Virtual Reality for the first time

Possibly leaderboard to play with friends, and online storage of leaderboards. We could also easily add levels by modifying number of asteroids and speed.
