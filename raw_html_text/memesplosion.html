• This is the title page of our game.

We built this game because we like dank memes. We thought it would be cool if we could incorporate memes into an interactive game. Some of us learned how to use Construct2, while others learned learned how to better use Photoshop. In the beginning, no one new how to code, but we did not let this become an obstacle for us, as we inspired each other to learn more and work harder. From here, we hope to continue the Memesplosion series, and let the dank memes live on forever.
