Inspiration

Enter the realm of virtual reality

What it does

Allows the user to navigate around using a leap motion and pick one of the books kept on the bookshelf

How we built it

Built several graphical assets in the unity game engine to create the environment. Integrated the use of Leap motion for navigation and interaction with objects

Challenges we ran into

No one in the team had any experience with the unity engine or leap motion sdk

Accomplishments that we're proud of

The learning curve was huge and we enjoyed every part of it

What we learned

A lot can be achieved in 24 hours

What's next for TamuHack

More brainstorming. More ideas. More networking.
