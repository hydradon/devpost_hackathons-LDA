Inspiration

As students, we have been taught the importance of getting help as soon as possible when it is necessary. This is why we wanted to create a website for people to find the help they need, no matter what problems they may have.

What it does

This website allows users to effectively navigate to a sources in which they can find a support group for themselves or their friends.

How we built it

We built this website using html and java script.

Challenges we ran into

We were not familiar with the languages needed to create this website, so we had to spend time learning. We also had to manage our time properly so that we could fit in the many steps needed to create our website.

Accomplishments that we're proud of

We are proud that even though we were not familiar with our coding language, we persevered and continued to learn in order to make a successful output.

What we learned

We learned how to use html and javascript to create a fully functioning website.

What's next

In the future, we plan on expanding this website and spreading its outreach to anybody who may need help.
