• App being developed my Xcode iOS SDK

Inspiration was given when presented with a Challenge. Whenever a guest comes to your home/Business they are forced to go through various steps to access Local Wi-Fi. We were Inspired to build this solution so guests can Safely access your home Wi-Fi while you are anywhere on the Planet.

What it does this system grants access to Local Wi-Fi using a Wi-Fi access point and Cloud connected application to authenticate

How we built it (still in development) Three Phased solution using hardware ,mobile application development and the Azure cloud to bring a Simple, Safe and Secure Internet experience to any Home guest

Challenges we ran into were mostly the hardware based given it was unable to create a full Wireless solution without adding extra hardware such as a secondary wireless signal

Accomplishments, I think the concept phase is what I am most proud of.

What we learned programming and implementation requires well thought out processes and true inspiration.

What's next for Wi-Fi Shield bringing all phased solutions together to create a Minimum Viable Product. If successful we would like to integrate social networking and machine learning to the products to create a more streamlined authentication process.
