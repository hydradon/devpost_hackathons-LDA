Brush Pen Robot ✍️

X, Y, and θ motion allow us to vary the thickness of lines to create beautiful lettering.

Check out our GitHub for code!

Technical Details

  • Python program converts font bezier curves to GCODE
  • Arduino running GRBL GCODE interpreter does motion control
  • Aluminum frame uses nylon slides for linear motion, with laser cut and 3d printed mounting hardware.
