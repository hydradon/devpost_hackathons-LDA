The Power of the Crowd!!!!

This app organizes cheers at Badger football games. Users are able to submit and vote on cheers for the crowd to do. It updates in real time so votes by all others appear immediately on everyone's phone.

We used Ionic framework and Firebase.

Angular is a problem, as always. It was also difficult to get the Firebase configured. Once it worked it was great!

This was our first Ionic app that connects to a database.

We learn a lot about Ionic, angular, CSS, HTML and Firebase.

In future we would like to add users, accepted chants, and chant expiration.
