Problem

We were aiming to solve two problems: making engineering challenges more accessible to enthusiasts and making recruitment process less painful.

What it does

SpaceY provides the infrastructure for companies to post engineering challenges aimed at students. In turn students can develop their interests at the industry standard while competing at competitions they are passionate about.

How we built it

We built the web platform using Django framework, which we connect to a makeshift model rocket using Firebase cloudless instance.

Challenges we ran into

Building the Rocket

  • Establishing communication across different microcontrollers — solution: change from Photon Wi-Fi module to NRF24.

  • Creating a frame without the 3d printer — solution: using Pringles cans.

Building the Platform

  • Creating flexible database system for various challenges — getting the data from Firebase instance.

Accomplishments that we're proud of

Persisting through problems throughout the night.
