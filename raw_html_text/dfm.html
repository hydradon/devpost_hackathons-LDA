DFM - decentralized finance manager

DFM is a decentralized application built on Blockstack for Blockstack "Can't Be Evil" hackathon.

Website: dfmapp.com

Using DFM you can add/update/delete your friends and add/update/delete yours and your friends' debts. All the data is autmaticaly synced with Blockstack gaia storage. You should be authenticated in Blockstack for being able to use DFM.

Summing all up currently DFM is a decentralized analog of splitwise. But in future it will have more features, such as

  • Shopping lists
  • Debt reminders
  • Automated payments (using blockstack account)
  • Finance planning (accounting of incomes and expenses)
