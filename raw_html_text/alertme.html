Inspiration

In the wake of recent school shootings, we developed an app that allows students to notify others if they sense suspicious behavior.

What it does

AlertMe! introduces an application that allows users to quickly report a school safety issue including school shootings, drugs, weapons, bomb threats, and fights.

How we built it

We built this app using MIT App Inventor.

Challenges we ran into

Some challenges we ran into was the type of platform/software we wanted to use for this app. Also, we had trouble setting up notifications if any suspicious behavior was received.

Accomplishments that we're proud of

We are proud of accomplishing an app that has a potential to save lives and increase student safety throughout the country.

What we learned

We learned how to use MIT App Inventor as well as various teamwork and problem solving skills.

What's next for AlertMe!

We plan to make AlertMe! available for IOS users in addition to Android users. We would also like to create push notifications for suspicious behavior within a 5 mile radius and create school and student profiles that can be used to increase the safety of students.
