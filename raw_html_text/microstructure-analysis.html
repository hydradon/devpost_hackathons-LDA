What it does

Requires an image of the microstructure of a material with a light and dark phase

  • this tool will filter out all precipitates and poor lighting on image
  • determines fraction of microstructure that is in each phase
  • calculates and plots histogram of size of grains in microstructure

How we built it

  • Image Processing Toolbox in MATLAB used to filter, sharpen, and determine phases in image

Challenges we ran into

  • dealing with poor lighting in some images
  • intersections of grain boundaries being counted towards the dark phase

What's next for Microstructure Analysis

  • develop better algorithms for filtering out shading
