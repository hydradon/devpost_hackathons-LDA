Inspiration

As college students who were not aware of the new environment that were thrown into when we started, we felt that a centralized map of the activities, clubs, organizations, events, etc. would have helped with the transition to college life.

What it does

The app requests your current location or a university and centers and places a pin there. It then places markers on other events and attractions happening in the area.

How we built it

We built it using vanilla html, css, and javascript along with the Google Maps API.

Challenges we ran into

None of us had any real experience with any of the technologies we used, and we never used git to collaborate. We also spent a long time trying to decide on an idea that would be achievable with our skill set in the short amount of time allotted for this project.

Accomplishments that we're proud of

We are proud of the fact that we managed to start and make a demo of a web app from scratch with no prior experience about web, git, APIs, or hackathons.

What we learned

We learned a tremendous amount including syntax, structuring a web project, learning about APIs, pushing and pulling to git and github as well as merge conflicts.

What's next for FindMore

Next we plan on migrating the data for the markers onto a database and running the app on a server. We also plan on creating an option for the user to add markers onto the map with more descriptions of the event as well as make the app more mobile-friendly.
