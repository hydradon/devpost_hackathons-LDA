Inspiration 

We were inspired by one of the teammates job in teaching coding to children. We were also inspired by another game called _Keep Talking and Nobody Explodes_.

What it does 

This is a game that tells kids how to respond in an emergency situation. It also teaches communication and teamwork. One person is playing the game while the other person is reading a guide on how to proceed through the game.

How we built it 

We were originally going to make the game in java script, we ran into difficulties however and we wrote it in scratch. The pdf guide was written in latex.

Challenges we ran into 

We were having some issues with java script so we wrote the game in scratch instead.

Accomplishments that we're proud of 

We are proud that we have a game in working order. It may be a simple idea, however it can help educate children in an engaging way.

What we learned 

We learned to optimize our productivity by distributing our work to the areas of which each of us are proficient at.

What's next for Survival Friends 

We plan to provide expansions for Survival Friends to broaden the immersive experience.
