Inspiration

Twitter has been used to express all kinds of opinions goods and bads, and as users, we want to follow people that don't spread hate, racism, discrimination, and all these negative things. We want to get inspired by the people we follow, so we build a positivity tweets checker.

What it does

Analyzes Twitter users by the sentiment of their tweets so we can have some idea of how positively they tweet

How we built it

We used the Twitter API and the Microsoft Cognitive Services powered by Node.js for the backend and Angular for the frontend.

Challenges we ran into

We had some problems setting up all the backend and styling the frontend.

Accomplishments that we're proud of

We have a really cool demo to show!

What we learned

We learned how Microsoft Cognitive Services works

What's next for Positweet

Update the UI with more data from the paid Twitter API so we can get better results with more tweets to analyze.
