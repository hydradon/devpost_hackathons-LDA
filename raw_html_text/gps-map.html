Inspiration

Our inspiration was we wanted to play with something new

What it does

It works by passing a GPS cordinate into an sql database which comunicates with our website and ouputs a pin with the location from the dragonboard

How we built it

We built using a dragonBoard, google maps api, PHP, HTML, JavaScript, and Rust.

Challenges we ran into

We ran into problems getting the dragon board set up. We also struggled with having google maps api with setting the location for pins.

Accomplishments that we're proud of

We are proud of the fact that we tried something new! We used a hardware we had never used before and we learned how to use an api.

What we learned

We learned how to use a dragon board, how to use rust, and how to use google maps api

What's next for GPS Map

We would like to expand it so the boards GPS can communicate more efficiently with the database and store multiple pins at once.
