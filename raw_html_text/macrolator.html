Inspiration

We created this calculator because we care about nutrition.

What it does

Based on user input, it tells the user the recommended calories on any given day. It also outputs the macro-nutrients; protein, carbohydrates, and fat.

How we built it

We created the code in Intellij, and created the user interface in Android studio.

The .java files uploaded are the terminal version, while the image is a screenshot of the user interface created. The UI was not fully completed.

Challenges we ran into

Android studio is a very difficult software to use.

Accomplishments that we're proud of

All of the outputs were correct.

What we learned

We learned that using new software can be difficult.

What's next for Macrolator

We hope to finish the user interface and merge the code from Intellij in order to create a fully working app.
