Inspiration

Helping breach the socioeconomic divide of people not being able to reach proper medication.

What it does

It connects patients and doctors and pharmacies, letting each group have information about the other, such as medication, and preparing for that medication before hand.

How we built it

It is a web app with a google maps api built in dreamweaver.

Challenges we ran into

Formatting the pages and getting google maps to cooperate was a tough time.

Accomplishments that we're proud of

Actually completing the website.

What we learned

What's next for PH.IN.DR

Potentially in security and other branches, database integration, not just pharmacies.
