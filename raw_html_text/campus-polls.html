• CampusPolls

Inspiration

I've always enjoyed seeing results from twitter polls and I wanted to see if I could replicate that in a way that helps students.

What it does

It allows students to create or take polls that other students have made specifically for their school.

How I built it

I built it in Node.js using Firebase as my database and Alexa Presentation Language to build the view.

Challenges I ran into

Firebase was difficult to work with on lambda and I should have used DynamoDB instead.

What I am proud of

I am proud of being able to pick up new technologies and make some cool things for students both with Alexa and mobile development

What I learned

How to use APL to build dynamic user interfaces.

What's next for Campus Polls.

Would like to improve the UI, allow for touches on the Alexa Show, and make the skill more stable.
