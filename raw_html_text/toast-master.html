• Toast Master

Inspiration

I had a very charismatic friend who always knew the perfect thing to say when raising a glass. Wanting to have my own witty and clever things to say or practice saying next time I'm asked to a propose a toast is why I built Toast Master.

What it does

You ask Toast master to make you a toast and it will pull from a database of provide you with on.

How I built it

I built it using the node trivia skill blueprint project on AWS lambda.

What's next for Toast Master

Adding more toasts!
