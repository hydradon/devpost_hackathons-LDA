What it does

Facebook Genius Challenge connects players on facebook to unleash their knowledgeable skills fully associated with Facebook. Players become aware that they are a genius on Facebook usage when they unleash a degree of achievement in the game. The game requires a player to simply give a response to a questionable task relative to the question in the game. The aim of the game is that through the game Facebook users who are also players in the game will become entertained, educated and informed on the proper use of Facebook and its features which covers security, friends , wall, timeline, reactions, messages and inbox, notifications, groups, Events, Market Place, Notes, Places, Platforms, Photos, Videos, Facebook paper, Facebook mentions, Facebook Credits, Graph Search, IPV6, Listend with friends, Live streaming , Mode faces, Poke, Tickers, Subscribe, Like, Share, URL Shortener, Verified Accounts, Hash tags, Say thanks, Impressum, Call to actions buttons, Facebook Ads, and Pages.

How I built it

I used a quiz software that supports multi player to create the questionable task in the game. I further research Facebook features to create a challenge out of it. I further design some graphics to spice up the splash screen. I compiled everything in an HTM5 framework and display in a Wordpress web page.
