Wearable/Home Automation App, Wildcard/General Health App

SolitairE monitors acceleration and orientation, while optimizing for energy consumption, using the Android API. By monitoring activity in the pocket of a user, even when the phone is asleep, SolitairE is a new breed of at-home protection system utilizing passive mobile sensors and cloud synchronization.

OpenShift host the MongoDB database RESTful routing API endpoint, which is updated on the phone's network when any alarm occurs. Checkout the video on YouTube here: https://youtu.be/JMbFM1zXN98

Note: The app is a alpha debug version, made for demonstration. For example, the app rechecks mobility every 15s instead of every 30min (120X's battery usage). Due to difficulty testing cross-device functionality, some features may be unavailable, but should not cause app to quit.
