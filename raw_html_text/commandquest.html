Inspiration

We want to make the process of learning programming more enjoyable for students.

With this goal in mind, we incorporate the love of gaming with an educational twist. Inspired by 70's ASCII gaming, we bring HTML basics to the fingertips of our gamers - CommandQuest.

What it does

Teaches users the basics of programming in HTML in a Roguelike experience.

How we built it

We used HTML, CSS, Jquery, blood, sweat, and tears.

Challenges we ran into

Building the game itself was very difficult as this was our first time ever building an ASCII game.

Accomplishments that we're proud of

Many of us learned Jquery for the first time. We also quickly learned the process of coming up with an idea, organizing the process, testing the program, and debugging. Diligently staying up all night, we were excited to complete and incorporate our game to our website.

What we learned

How to make a Rougelike ASCII Game with only HTML, CSS, and Jquery.

What's next for CommandQuest

Incorporate more levels, add a function to answer questions in real-time, have a scoring system, leaderboard and ability to save your progress. We would also like to incorporate 3D art for characters, map, and actions. Add accessibility to more programming languages such as Python, Javascript, CSS, Ruby on Rails, etc.
