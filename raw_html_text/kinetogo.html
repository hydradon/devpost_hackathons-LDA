.KineToGo

Web application developed in order to monitor physiotherapy process and generate evolution path. Physiotherapy is a long and difficult process which does always continue outside the hospital in everyday life. Considering this fact, a web application designed in order to provide on-line interaction between patients and doctors and an established evolution plan would represent an appropriate way to make this journey more secure and easy.
