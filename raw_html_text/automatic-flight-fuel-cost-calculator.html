Inspiration

Airliners spend a lot of money on jet fuel that isn't needed. These cost reductions will allow ticket prices to become cheaper and makes traveling more affordable. As a world traveler myself, the benefits of this program will support me and thousands of others alike.

How it works

The program takes the data of the flight path from a remote server through a JSON request. Then the program calculates the difference in distance between the GPS coordinates and figures out the appropriate fuel cost for that specific flight.

Challenges I ran into

There was a complicated formula and getting the data was difficult.

Accomplishments that I'm proud of

The program functions as I envisioned it to be.

What I learned

How to program in python, use JSON for data collection, and the importance of fuel savings or airliners.

What's next for Automatic Flight Fuel Cost Calculator

Deployment into an airliner database so each flight is tracked and monitored automatically.
