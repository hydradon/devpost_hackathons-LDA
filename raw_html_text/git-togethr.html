Inspiration

As students beginning their journey in software development, we are very excited to learn and get involved in the tech community. However, getting started can be difficult, especially in finding projects. With our project, everyone can get get involved and "Git-Togethr" to create and hack.

What it does

Git-Togethr connects tech teams with software developers who are looking for a team or open-source project to join. Think of it like LinkedIn meets StackOverflow meets GitHub. This web app will join tech lovers in an inclusive community.

How we built it

Started with Django and added Bootstrap and React
