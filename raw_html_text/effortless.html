Inspiration

A lot of senior citizens go through the hassle of visiting a pharmacy after doctors prescribe them medication that they use regularly. We wanted to eliminate this time-consuming process in order to make life easier for senior citizens and patients in general.

What It Does

We created a program where the doctor can prescribe medicine that will be sent to a vending machine located at the doctor's office. This will in turn allow the patient to enter a code and retrieve the medicine, skipping the need to go to an actual pharmacy.

How We Built It

Through the use of Python and Softheon workshop UI, we were able to create a program which allows the doctor to send a code to a "vending machine," which would be used to identify and dispense the medicine.

Challenges We Ran Into

  • The UI has a lot of white space that we feel we need to add to, most likely with more informational text or navigational features.

  • Saving information to a data base in one language and then reading it using another language.

    Accomplishments that I'm Proud Of

  • Creating the UI despite our inexperience in UI development

  • Creating a functional part of our idea in 38 hours

  • Being able to focus on the main idea of the project and not the extra effects

    What We Learned

  • Softheon usage and developing a coherent UI

  • How to split the work evenly amongst my team

  • Integrating code together, especially ones we used our code together in one
