CliniQ

NTU HACKATHON

Problem statement

  1. Crowd information from nearby clinic to visit the least crowded one
  2. Old people can’t wait long
  3. Small clinic’s cannot afford to have in-house database
  4. Effective planning in the clinic, based on the crowd.

Solution Outline

An AI face detector to detect the faces in the clinic waiting area, to find the crowd capacity in the clinic. The crowd capacity is updated realtime to the cloud storage and fetched using a Python Rest API to return to the users based on their current location.

code Outline (repo's)

  1. face detect is the code for face detection using opencv which updates the firebase with the number of people information
  2. python server is a flask server, acts as a REST API
  3. Ionic app is the front end application.

technologies used

  1. Python (Flask, Opencv)
  2. Ionic
  3. Firebase
  4. Tableau
