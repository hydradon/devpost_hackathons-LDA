Inspiration

I used to be a poor English communicator, and had a lot of trouble writing emails which were appropriate for the messages I wished to deliver. I created this tool to help remind people of the most important things when writing a formal/semi-formal/informal email.

What it does

Email bro is a personal email buddy that helps you write emails. Simply ask it for help and it will prompt you through the process to find the information you need! See the demo video.

How I built it

The entire project was completed using Amazon's alexa skill kit and aws lambda with node 6.10. The process was smooth, efficient, and simple. This is a great platform for experimental development!

What's next for Email Bro

The future for email bro is an expansion and an improvement in terms of specificity of the information it can provide to users. For example, a user should be able to ask how to complete a certain sentence in a formal email in the future.
