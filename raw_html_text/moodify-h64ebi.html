Moodify is a colorful cube that lets users scroll through a RGB spectrum with a Nuimo controller and select a hue which defines a mood. The cubes color and intensity is controlled via an api which controls 2 led light inside the box. When a color is selected a mood is associated to it and used to retrieve track suggestions from the Musimap Api and then calls the Spotify Api to get a playlist of songs to fit this mood. It then adds and plays this playlist through a bluetooth speaker located inside of the box.
