What it does

Integrates multiple public transport companies on a single platform.

Company side Offers full integration package:

  • NFC readers
  • web admin dashboard
  • integration with payment system

Customer side Offers:

  • possibility to access all integrated transport systems worldwide with one device (phone, card)
  • payment by card directly on platform
  • access to buy tickets/subscriptions worldwide
