• Prewitt-Sobel Hybrid Edge Detection for Improved Pathology Reporting in Gastroenterology Cases

Inspiration

My inspiration was getting to know the possible improvements that could be made in pathology reporting.

What it does

It applies Prewitt-Sobel hybrid edge detection methods upon histological imagery.

How I built it

I utilized Python, numpy, matlib, and skiimage.

Challenges I ran into

My major challenges were understanding the relative differences between Prewitt and Sobel kernels and researching the most optimal hybrids between the two.

Accomplishments that I'm proud of

I am proud of seeing this project work within the context of Spirochaetosis identification.

What I learned

I have learned that hybrid edge detections can be heavily utilized within pathology reporting (of gastroenterology cases), often with very little cost.

What's next for Prewitt-Sobel Edge Detection for Gastroenterology Cases

I hope to use the final Prewitt-Sobel hybrid mask as input to convolutional neural network (cNN) feature extraction. I would also like to apply cNN's directly on the masked images.
