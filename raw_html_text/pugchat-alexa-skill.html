• Echo Show/Spot Support

Inspiration

We have 3 rescued pugs and many weeks we have to miss #pugchat! It can be a bummer to miss all the cute pictures and updates so we wanted a way to catch up via Alexa.

What it does

The skill provides you with each of the #pugchat questions as well as a sample of the answers that were shared by others. The skill also accesses adoptable pugs listed with Petfinder.

How I built it

The skill is built with a combination of Azure Functions and an Azure Logic App.

Challenges I ran into

While this wasn't my first Alexa skill, it is my first with a continuous session and a design for the Echo Show/Spot.

What's next for PugChat Alexa Skill

I hope to add support for finding out who won pug chat that week, maybe even some updates on gotcha days and featured adoptable pugs.
