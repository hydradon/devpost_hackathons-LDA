Inspiration

A very popular mobile game called "Smashy Road: Wanted"

What it does

The purpose of the game is to run away from the police cars (represented by rectangles)

How we built it

I guess we used our coding skills, brains and strained fingers to try to get this game into working condition

Challenges we ran into

Side scrolling took up way too much time, once done, side scrolling created many mapping/collision problems which could not be solved in the given time.

Accomplishments that we're proud of

We got past our biggest hurdle of the side scrolling.

What we learned

We learned how to save the most time, and see through our tears. We also learned how to use a mapping application called "Tiled"
