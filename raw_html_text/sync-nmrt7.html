Inspiration

Multiple browser ecosystems across different operating systems and devices create fragmentation of web history.

How it works

We sync previous history and currently-open tabs between browers, so you can maintain a persistent personal environment wherever you go. This also separates your browsing history from any particular program, so you don't have to rely on Google, Mozilla, etc. to store your web history. You control that data now.

Challenges we ran into

Confusing browser APIs and difficult documentation, RESTful API for inter-application communication

What's next for Sync

Supporting more browsers (Safari, IE) and more platforms (Android).
