Inspiration

There is a lack of good vampire dating sims and we aim to change that.

What it does

It allows you to go through a haunted house and attempt to date vampires.

How we built it

We used Unity and C# to create the UI and base classes to allow different types of interaction with the vampires and keep track of properties.

Challenges we ran into

We had trouble merging the different Unity projects and had to spend a lot of time trying to figure out how to fix what got messed up.

Accomplishments that we're proud of

It looks really nice and has potential. I'm really proud of the main introductory interaction and I think we can make a pretty awesome game once we add some interactions in the other locations!

What we learned

We learned how to use C# and the power of vampire friendship!

What's next for Voracious Vampires

More interactions! More character reaction sprites! Shia LaBeouf secret ending! Regular endings in general! And also fixing the UI from the merge
