Inspiration

We wanted to play chess together, and we wanted to make something cross-plataform

What it does

We made a chess that can be played through two Messengers, Facebook's and Slack.

How we built it

We created a Node.js Server for the messengers requests and a PHP server to form the db.

Challenges we ran into

Technical difficulties as they were expected

Accomplishments that we're proud of

Managing to have a playable product

What we learned

Several forms that chess can be translated to, and there are many ways to play a game.

What's next for ColabChess

More messengers, more game modes, AI bots
