Inspiration

People need express help in their neighborhood, e.g. grocery shopping in passing, asking current situation in a remote place and picking up packages.

What it does

Neighbor Express helps you to make request and send your request to everyone near you. As a person who wants to help, it is an easy way to earn some tips.

How I built it

Using Xcode with swift

Challenges I ran into

Server/iOS interaction, API Integration.

Accomplishments that I'm proud of

A app that every one can use and get benefit of

What I learned

iOS development

What's next for Neighbor Express

improve the payment feature and improve the UI
