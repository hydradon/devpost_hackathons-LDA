Inspiration

We can do more to make co-working spaces more productive, interactive and enjoyable for everyone. Genesis takes a few unique approaches to introduce new solutions to make this happen.

What it does

Genesis provides an interactive dashboard.

The interactive dashboard will provide with: Collaborative Hive

  • Utilising the collective knowledge base of the co-working space. Sentiment Analysis via "Moodometer"
  • Asking the user for responses at key milestones to track their emotional state. Dimplomatic Gamification
  • A points based system with a new style of rewards
