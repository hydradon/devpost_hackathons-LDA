Inspiration

To develop a web portal that helps to assess NPOs

The initiative taken by mille-center has been our inspiration to develop a solution for them to solve their data inconsistency problem. It also helps the accelerator in finding NPOs.

What it does

It provides a solution to represent the data of NPOs graphically in an efficient manner. A user-friendly portal with advanced reporting and analytical tools which utilizes standard object model to collect and display unstructured data in a rich and intuitive manner

How we built it

We built it using HTML, Javascript, PHP, CSS, Bootstrap., Google Charts, PHP-ML, Classification Algorithms.

Challenges we ran into

Classification of data metrics

Web Crawlers

Algorithms for Impact Index

Accomplishments that we're proud of

We have developed an end-to-end product that standardizes the varied data of different NPOs.

Developed a supervised learning system to classify NPOs.
