Inspiration

Some things that inspired us are the fact that our communities aren't perfect, we all have little faults. So we made this app just so that we can help people deal with these problems and to get help to solve them.

What it does

Using our program users in a community can report problems, sign petitions, and create petitions. There is a login page that you have to make an account or sign in. So this way, there aren't anonymous pranks or maybe even false tips.

How we built it

We used python ide Pycharm to code our project. We included many other third party modules such as tkinter.
