Inspiration

Nous avons décider d'améliorer les applications déjà existantes de localisation de stationnement dans Montréal. Trop de gens perde beaucoup de temps à se chercher du stationnement.

What it does

C'est une application qui permet un partage d'information entre les agents de stationnement et le public. en plus de savoir le prix des stationnement extérieur et intérieur, et les rues non payantes a Montréal, on souhaite utiliser les données de parcomètre afin de localiser précisément les stationnement disponible. De plus, l'application permet d'entrer la localisation d'une voiture en infraction qui dérange les citoyens afins d'aider le travail des agents de stationnement

How we built it

Nous voulions au départ utiliser Android Studio afin de faire une application Android, en utilisant l'API de Google Map. Par contre, nos connaissances étant limité, nous ne sommes pas arrivés a développer une application fonctionnelle. Nous avons donc utiliser Java sur Netbeans pour simuler les améliorations que nous voulons apporter aux applications existantes.

Challenges we ran into

Nos connaissances limités en programmation nous empêchant d'apprendre et de développer une application en 24h. De plus, nous n'avions pas accès au données les plus importantes, soient les données des agents de stationnement.

Accomplishments that we're proud of

Nous sommes quand même fier d'avoir travailler un peu avec Android Studio et d'avoir développer une interface graphique la plus précise possible avec les temps qui nous restait.

What we learned

Qu'il faut se préparer un peu plus d'avance et exploiter les compétences de chacun pour obtenir le projets le plus finis possible.

What's next for Parking clairvoyant

Developpement en Android, communication avec la ville de Montréal, amélioration de la vie des citoyens.
