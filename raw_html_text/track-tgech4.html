Inspiration

We wanted to create an app that will assist athletes in the preparation of athletic events and competitions.

What It Does

The app will generate an optimal workout routine for the user specifically designed to help them succeed at their desired event.

How We Built It

Using python and various libraries available, we created a basic desktop app.

Libraries Used:
  • tkinter
  • notify-run
  • mongodb
Software Used:
  • Photoshop
  • Python 3.7.2
  • Windows CMD

Challenges We Ran Into

  • Notifications currently only work on certain devices and while the app is running

Accomplishments That We're Proud Of

  • Creating a functional and effective User Interface
  • Developing an app with a lot of potential for future growth

What We Learned

  • notify-run
