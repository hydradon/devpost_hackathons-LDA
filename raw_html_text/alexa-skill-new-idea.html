Alexa Skill "New Idea" allows you to generate ideas! Combining words we can get an idea for any case. For example, for the Halloween holiday, you need to come up with a new superhero costume, then just ask:

  • "Alexa, ask New Idea generate for man" and Alexa will start generating ideas:
  • "Cup man"
  • "Arrow man"
  • "Smile man"
  • ...

This way of inventing ideas has been used since ancient times by writers and inventors. Only instead of voice-controlled intelligent personal assistant service, they used books. Taking any book they opened it on a random page and chose a random word. But using Alexa Skill "New Idea" discovery of new ideas has become much easier. For example, to come out with a theme for a school essay, just say:

  • "Alexa, ask New Idea start for summer" and Alexa will start to fantasize:
  • "Architect summer"
  • "Kimono summer"
  • "Escape summer"
  • "Ring summer"
  • ...

So artificial intelligence helps a person to be creative. For example, to invent a new chair model, simply say the magic words:

  • "Alexa, ask New Idea generate for chair" magic of the 21st century:
  • "Zebra chair"
  • "Case chair"
  • "Dance chair"
  • ....

The process of generating ideas can be paused and resumed with the following commands:

  • "Alexa pause"
  • "Alexa resume"

For references use:

  • "Alexa help"

To cancel and return in main menu:

  • "Alexa cancel"

To exit from skill:

  • "Alexa stop"

So, welcome to the world of technical magic and successful discoveries.
