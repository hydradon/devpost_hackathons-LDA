Inspiration

We were highly frustrated by existing note-taking services like OneNote, Sticky Notes, and Google Keep

What it does

It's a collaborative note-taking service - files are created and then shared through flask.

How I built it

Built with the cooperation of teammates, using Flask, Python, and TKinter

Challenges I ran into

  2. Creating a database
  3. Implementing Flask

Accomplishments that I'm proud of

  2. Learned a bit more about the functionings of python.

What I learned

  1. The operation of Flask and TKinter
  2. Certain technicalities of Python
  3. The use of SQLite3 Databases
