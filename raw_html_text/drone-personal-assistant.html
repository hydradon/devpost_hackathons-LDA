Inspiration

Need for automated personal assistant for everybody's needs.

What it does

Wave at the Drone it comes near to you.

How we built it

Using Python and OpenCV as platforms.

Challenges we ran into

New SDK, First Timer usage of Programmable Drones.

Accomplishments that we're proud of

It can run around using predefined tracks, wave at the drone and it will be able to start and come nearer to you.

What we learned

How to use Drones, newly learn SDKs in short span, and coming up with an innnovative solution.

What's next for Drone Personal Assistant

Later It identifies its owner and able to get smaller weighted things to the owner.
