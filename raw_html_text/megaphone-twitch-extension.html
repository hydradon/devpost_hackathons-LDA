What it does

Let your viewers use bits to send custom text to speech messages that will play on stream. Configure how many bits you want to charge per message. There is support for multiple text-to-speech languages and voices.

How I built it

React JS frontend for extension, Backend uses the following AWS services: API Gateway, AWS Lambda, DynamoDB, S3

Sengage Twitch Extension: https://www.sengage.io/twitch-extensions
