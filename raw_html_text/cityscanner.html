Inspiration

This is a problem we always face when we want to explore a new city with a cheap option for a weekend.

What it does

CityScanner is a tinder for cities. It offers destinations with the traveling cost, so that users can choose better options.

How we built it

First, we divided the tasks with teammates. The main tasks were graphic design, web development, back-end development, and algorithm design.

Challenges we ran into

Team members did not know each other well before. So, the most challenging parts were ideation and adaptation of team members.

Accomplishments that we're proud of

We managed to code and design simultaneously.

What we learned

Some improvements on flask, react js, arduino and ui/ux design.

What's next for CityScanner

This is just MVP, we are planning to make it more personalized.
