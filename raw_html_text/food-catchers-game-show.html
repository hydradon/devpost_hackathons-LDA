• Food Catchers Game Show is all the rage for learning to make healthy food choices.

Inspiration

People make bad choices when it comes to food. It would be great if there was a way to help create positive reinforcement for choosing better foods.

What it does

The Alexa skill simulates a game show where two food items are thrown over a wall. The goal of the player is to try and the better food item to catch. The better the food item, the higher the score. Fruits and water rank highest while candy and sugary drinks rank lowest. After a player scores 500 points, they are awarded a larger food catalog to have thrown over the wall. This can make choosing which side to catch more difficult.

How I built it

I built it in Python using an AWS Lambda function. The persistence is from Dynamodb. It uses one table for User data and another for Conversation data.

I would like to add more levels to include calorie guesser and pick that vegetable. These rounds would make the game richer and thereby making it more fun for users.
