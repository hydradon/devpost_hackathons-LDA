Inspiration

We were driven by the current administration's lack of attention to the environment. We were also fortunate enough to be allowed to experience the VR world in our science class during middle school, so we decided to target the game towards grade schoolers to share this experience. This would allow the students to be immersed in the environment and get excited about learning.

What it does

A virtual reality environment that allows the player to interact with animals affected by climate change.

Challenges I ran into

It was difficult learning how to create text in virtual reality due to the split screen for each eye and our lack of experience in making games in virtual reality.

What I learned

We learned a lot about the intricacies of coding in virtual reality.

What's next for Virtual Earth

To improve the game, we would like to add movement for greater interaction with the environment, add more animals, ecosystems, and information about climate change, and enhance the environment with additional details, such as grass.
