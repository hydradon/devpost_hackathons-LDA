Inspiration

IBM had a coding challenge using their ibm-watson APIs. We wanted to make use of their API to make an application to help people with visually difficulties.

What it does

By taking a picture on the android app, the app will analyze it and read out the information back through text to speech.

How we built it

We used android studio as our main platform. We implemented IBM's Visual Recognition API and their Text-to-speech API.

Challenges we ran into

We had to learn from scratch android studio and how to use iBM's APIs.

Accomplishments that we're proud of

We finished what we wanted.

What we learned

We learned how to make a simple app using android studio and how to implement APIs.

What's next for DoYouKnowTheWay

Adding more feature, cleaning the code, having a better structure
