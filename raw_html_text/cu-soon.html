Inspiration

What it does - CU Soon attempts to make events around campus, especially club meetings other smaller events, more accessible to Cornell students by providing a list of events, and an accompanying map of locations, that are available each day.

How I built it - We used a Meteor framework and constructed the web application using google maps and codi data.
