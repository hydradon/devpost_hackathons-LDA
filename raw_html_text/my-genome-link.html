Inspiration

I think it will be beneficial for people to find out their DNA based vitamin profile.

What it does

Just connect to your Genome Link account and a vitamin report with your scores and descriptions will be presented.

How I built it

I use Genome Link Node SDK. The node example has been very helpful.

Challenges I ran into

Getting the app hosted on Glitch is not super hard, but I did spend something getting it works.

Accomplishments that I'm proud of

Making the app works!

What I learned

I have learn something new about Glitch, JavaScript and Genome Link SDK.

What's next for My Genome Link

Use or build an AI model based on genome data to build an illness prediction app which can tell you how likely you will get a particular type of health problems.
