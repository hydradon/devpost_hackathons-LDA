Inspiration

We came up with this idea, because two of the people in our group didn't know how to use Java and this was a way to teach them the basics of how it works.

What it does

Its an application that takes users through the Java programming language.

How we built it

We built it by using JavaFX and CSS.

Challenges we ran into

We ran into many problems, but the two main ones were how to change scenes and how to run and compile the code.

Accomplishments that we're proud of

We are proud of the progress we made with the certain amount of time and learning JavaFX from scratch.

What we learned

Two of the people in our group learned the basics of Java and the other two learned how to use JavaFX.

What's next for JavaLearners

We are going to expand the amount of lessons and at sometime we might add a test that reviews the lessons.
