Inspiration

We decided that we had a unique opportunity to submit a project that combines aspects of coding and finance, since we came with both coders and business majors.

How it works

SmartStocks pulls information from the Yahoo finance API in real time and aggregates the data on our server, displaying it in a usable format on a custom iOS app and a pebble app. Parse was utilized as a back-end. FINRA, Pebble, and Bookholders challenges

Challenges I ran into

iOS libraries displaying charts, IDE updates, connecting the pebble with the iOS app.

Accomplishments that I'm proud of

Importing a .csv file using a http get request directly to the parse server using solely javascript in cloud code.

What I learned

Always have fun!
