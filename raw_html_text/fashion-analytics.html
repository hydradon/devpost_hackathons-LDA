Inspiration

We noticed the lack of efficient tool for clothes designers to help them follow the fashion very closely so we decided to provide them with this kind of tool.

What it does

It meant to scrapp data from instagram and by using image recognition to be able to really know what people actually wear.

How we built it

We built it using the NewMiip solution so it's basically a NodeJS app wich is using a mysql database.

Challenges we ran into

We had to learn how to use NewMiip and to try to enter in the mind of a designer to figure out which piece of data he really needs.

Accomplishments that we're proud of

I believe that we have been quite successfull when it cames to understand the need of the target audience.

What we learned

We learned obviously how to use NewMiip but we also learned about how we can add interactivity between clothes designers and customer to help the firsts to create clothes that are really what people wants.

What's next for fashion analytics

The next steps would be to actually scrapp datas from instagram and use image recognition. An another step would be to process the data in a way that it will be easier to understand for designers and even closer to their needs.
