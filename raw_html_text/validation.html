Inspiration

Human beings seek validation all the time! They want to know what others think about them and they want to see if they are on the same page with everyone. Use this for instant gratification! Just type in whatever's bothering (or not) you and see what everyone else "feels" about it - and get scored!

What it does

This is a web app that let's you type in a query or a name or anything really. The app then calls the twitter search api and aggregates tweets. Then we do some sentiment analysis on the tweets to quantify how people "feel" about this particular topic and shows you a nice histogram and other visualizations so that you get to see if you're aligned with other people - thereby gaining drumroll VALIDATION!

How I built it

Vue.js on the frontend, Django for the backend and we used spaCy and TextBlob for the sentiment analysis.
