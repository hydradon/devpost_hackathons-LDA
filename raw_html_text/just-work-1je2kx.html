Inspiration: We wanted to make an app that could keep track of how much work everyone did in a team and what work they did or tried to do

What it does: Tracks the time spent on tasks and can be viewed by anyone with the admincode

How we built it: We built this app using Flutter as the framework and dart as the language. This lets us use run this app on both ios and android. We used firebase and firestore to store and live stream our data

Challenges we ran into: Dart kept breaking and it refused to run the app sometimes so we had to roll back to an hour or more.
