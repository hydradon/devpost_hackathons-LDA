Inspiration

All of us were intrigued by the police data that was offered by Clarity Partners during the opening ceremony. We were interested in how we could utilize it.

What it does

This website will pull up records of specific officers and display times when emergency calls had the worst response rate

How we built it

We used the API provided, and Flask to connect with JavaScript.

Challenges we ran into

How to meaningfully extract information from a million plus records without the time to process each one.

Accomplishments that we're proud of

We were able to display data in multiple formats depending on user preference.

What we learned

Flask, UI design, jQuery and Javascript proficiency

What's next for Copturing Data

Add more formats to display the data in, going more in depth with crime data and performing a little more preprocessing on the data and create metadata.
