Inspiration

We want to make online education more accessible to people in developing countries around the world by reducing the file size of video lessons.

What it does

By utilizing the unique format of Khan academy videos we were able to reduce the video to a number of simple shapes. By saving each shape as a collection of bits, we were able to make very small videos.

How we built it

We built a file format around data density by focusing on bit necessity.

Challenges we ran into

Getting our Android app to work with our file type. Making our shapes as small as possible.

Accomplishments that we're proud of

Our strategy to implement data structures in pure binary.

What we learned

We learned how to develop for the Android OS. We learned how to work effectively and collaboratively in a time sensitive setting.

What's next for bitWise

Making a platform to create the lessons in our file format easily and intuitively.
