Inspiration

For those times when you have trouble getting into major classes, it would help if UCSC had the tools to analyze all the course pathways and class sizes to ensure we can graduate on time.

What it does

Simulates the graduation pathways and provides analytics that UCSC could utilize to improve class sizes and counseling
