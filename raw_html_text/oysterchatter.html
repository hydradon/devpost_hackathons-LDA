Inspiration

I ultimately want to create a portable time-lapse module for when I go camping. This project is just to learn the skills and expand my programming abilities.

What it does

Takes 3 pictures and automatically posts them to twitter. When the raspberry pi is turned on, it starts the python script and waits for the push button to be pressed before taking pictures.

How I built it

Interface with twitter app and take pictures in Python. Autorun python script when pi is powered with bash. Raspberry pi hardware.

Challenges I ran into

Didn't get USB Wifi working yet.

Accomplishments that I'm proud of

Learning python and setting up rasp pi with op system.

What I learned

How to use an API and make a twitter app.

What's next for OysterChatter

I ultimately want to have it create time-lapse videos for when i go camping.
