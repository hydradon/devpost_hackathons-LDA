Inspiration

When thinking about a project to make, i came across an idea to create a card game that could be fun to play and have some element of skill to it.

What it does

It a console base game where you select cards that go along with a color to see if you can take down the opponent health down before they can.

How I built it

It was built on repl.it a online IDE

Challenges I ran into

A challenge i encounter was deciding weither or not to use a vector or an array to hold integers that must be remove and added in at will. Another would be remembering the rules i made for the game to make sure the game play along with the rules.

Accomplishments that I'm proud of

Im proud to be able to complete a game i created and stuck with the rules i made to keep it from being to chaoitc.

What I learned

I learn the benefits of vectors have over arrays and how to use them.
