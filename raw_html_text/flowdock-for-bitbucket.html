Inspiration

Colaborate with the comunication across distributed teams, making the code changes visible in a easy/fast way to the whole team at real-time.

What it does

The plugin integrates Bitbucket server with Flowdock, sharing the Bitbucket activities with the Flowdock flows. In that way, the whole team will know what is going on at real-time.

How I built it

It was built using Atlassian java SDK and Flowdock Push API (REST).

What's next for Flowdock for Bitbucket

The next step is to integrate the Bitbucket push messages with the Flowdock Tags, so that the user can add the Flowdock Tags in the commit messages. It is currently being developed and will be available in the Atlassian Marketplace soon.
