Inspiration

Two things inspired me to create Skwadpoll. The first thing was my love for opinionated debate and the second thing was the struggle of not being able to get quick and reliable data when I need it.

What it does

It allows users to create quick and interactive polls to friends and family.

How we built it

Sketch & Flinto

Challenges we ran into

Accomplishments that we're proud of

Created a full prototype that incorporates a beautiful user interface.

What we learned

We learned how to delegate tasks.

What's next for Skwadpoll

I will start to work on an Android application.
