Street Farter

Play as Goku or Sailor Moon in a butt fighting game.

You have 2 attack buttons, but you have to press them in a certain order to execute attacks. Each attack is made up of a 3-stringed input combo. Maximize your attack frequency by learning the inputs!

For PC, download the data folder and the.exe under the "Finished Build" folder and run the .exe. For MAC, download the package under the "Finished Build" folder and run the .app.
