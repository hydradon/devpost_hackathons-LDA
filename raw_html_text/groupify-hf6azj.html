Inspiration

Sometimes you need to gather a huge group to do some activity, like if you want to play soccer, or maybe you want to gather a group to study for a certain exam, but your friends are not interested? Then just create a one-time event in Groupify, set limit of people who can attend and then wait for them to join.

What it does

It allows people to create events and set the limit for people to join. Then people nearby will see it and have the possibility to join. Once the right amount of people joined, the event will disappear and participants will be able to communicate with each other in a chat.

How I built it

We used swift for frontend and we used Nodejs and mongoDB as a backend server and database. We containerized them using docker and we will run a Kubernetes cluster on Azure services.
