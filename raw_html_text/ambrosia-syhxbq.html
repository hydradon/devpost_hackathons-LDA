Inspiration

I was inspired by the challenges set forth by the various sponsors here at UB Hacking. I've always wanted to make a recipe app, and luckily that matches the expectations here today perfectly.

What it does

It allows you to search for recipes, favorite them, add them to a planner, count your calories, get nutrition facts, etc.

How I built it

I built it with React Native.

Challenges I ran into

Starting up was hard, I was unsure about what I should've been doing a lot during the beginning.

Accomplishments that I'm proud of

I've never used voice recognition or TTS APIs and libraries before, getting them functioning is a proud point.
