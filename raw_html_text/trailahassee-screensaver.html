Inspiration

I recently switched screensavers to the aerial shots on the Apple TV, but I figured that having a screensaver of images from parks around Tallahassee would be good!

What it does

It does the following:

  1. Scrapes the trailahassee.com API in order to get the park names
  2. Converts those names into a "tag"
  3. Scrapes Instagram with those tags
  4. Displays 12 randomly selected images!

How I built it

  • Ruby on Rails
  • Objective C
  • Instagram API
  • Trailahassee API
