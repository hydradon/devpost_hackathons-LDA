Inspiration

We wanted to help create a more accomodating workplace for minorities, i.e. women in tech and help combat imposter syndrome.

What it does

Allows co-workers to give each other recognition for their work, and facilitate consciousness raising exercises through anonymous submissions.

How we built it

We used HTML/CSS and javascript to implement the web application.

Challenges we ran into

As with all hackathons, we faced a time crunch.

Accomplishments that we're proud of

We're proud that we can help diversify the workplace in any way possible.

What we learned

We learned how to use a framwork called Vue.js

What's next for Shoutout

We plan to continue expanding its functionality with more features.
