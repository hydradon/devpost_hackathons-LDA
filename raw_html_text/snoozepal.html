Inspiration

The inspiration for this story is to avoid being late to important and events as well as just being on time for everyday schedules.

What it does

It sets up alarms at a 5 minute interval while sending an automated message to a specified individual through SMS.

How we built it

This project was built with Java through Android Studio.

Challenges we ran into

Challenges we ran into were learning how to program each feature given that we had no experience with Android Studio.

Accomplishments that we're proud of

Accomplishments that we're proud of are the design and being able to finish the project in time.

What we learned

We learned how to code projects within Android Studio, while learning the basics of it as well.

What's next for SnoozePal

We plan to improve the overall function and design of the app.
