Inspiration

We are TAs for the Computer Science department. We are frustrated by the way of our office hours executed. Therefore, we want to make it better

What it does

There are two options: student and instructors. Students can add entries into the queue and wait for their turn to be helped. The intructors have the option of poping the queue and add student back in the queue in case of that student is not done being helped.

How we built it

We built it using android studio with a database using php.

Challenges we ran into

We have to deal a lot of networking and database which required internet and the wifi at Bitcamp is not so nice recently.

Accomplishments that we're proud of

We successfully communicate with the database and transfer data back and forth.

What we learned

We learned a lot about the Java with Android and database.

What's next for QueueList

We will expand it to many classes and maybe to other schools as well.
