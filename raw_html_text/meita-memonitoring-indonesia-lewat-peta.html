PROBLEM

Kurangnya portal untuk melihat visualisasi data publik milik negara / pemerintah

SOLUTION

Membangun portal untuk melihat visualiasi data publik milik negara / pemerintah yang menarik, informatif, dan interaktif

HOW IT WORKS

Berikut adalah langkah-langkah yang kami tempuh untuk membangun aplikasi ini:

  1. Mengambil data mentah (*.csv / *.json / *.xml) dari tempat yang kredibel
  2. Membangun API, dengan cara:
  3. Mengolah data mentah, terutama nama kota dan/atau propinsi, menjadi koordinat Long. dan Lat. dengan menggunakan webservice dari OpenStreetMap
  4. Me-return hasilnya dalam bentuk JSON
  5. Membangun pemetaan dengan menggunakan CartoDb.js
