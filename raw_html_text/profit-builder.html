Inspiration I am passionate about using analytics and Programming skills to build tools and solutions which address Commercial and Social needs.

What it does Predict the amount the customer will be paying when a customer comes to the locations by giving details like type of food, type of eating space.

How I built it Used NCR API to extract transaction information which contains Eating places transaction. using Data analytics and Machine Learning(Python, Cloud) to build a model.

Challenges I ran into- Data is not rational(Not structured). It's in Documentation form so if you want to convert it into rational table it takes a lot of time effort.

Accomplishments that I'm proud of Using NoSQL, Unstructured data to build a machine learning model which successfully predicts the price with the accuracy of 72%

What I learned Working with Document data, Pushing the boundaries, trying innovative methods.
