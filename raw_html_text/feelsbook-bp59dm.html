Inspiration

Recently I've started journaling. I wanted to track trends on my entries over time.

What it does

Performs sentiment analysis on notebook entries and changes the background color based on the analyzed moods.

How we built it

React, IBM Watson API.

Challenges we ran into

CORS errors, making API calls, front-end problems.

What we learned

Learning css, html, react, using APIs

What's next for Feelsbook

Adding an emotion sidebar. Making the background change color automatically as the user types. Adding a backend.
