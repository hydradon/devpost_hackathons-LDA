Challenge 4

Inspiration

We want to simplify the process of taking care and tending to sick animals. In particular, we aim to help farmers quickly diagnose and quarantine pigs with African Swine Fever (ASF). ASF is a deadly viral disease affecting both domestic and wild pigs. There is no vaccine or cure, so contracting the disease is a death sentence. Currently, Asia and Europe are most impacted.

What it does

Our website educates people about ASF and provides farmers with a personalized database to track their pigs' symptoms.

How we built it

We used HTML and CSS to build the website, SQL to create the database for the pig information and login credentials, and PHP to make everything dynamic. The website icon and homepage background were created digitally using Adobe Illustrator.

What's next

We want to translate the website into multiple languages to reach a greater audience. We plan on analyzing data to better under the symptoms of ASF. The website will be updated to reflect geographical susceptibility - locations of ASF outbreaks will be marked on a map. Doctors and researchers will be able to create accounts to access information and help further ASF prevention and containment studies. We will also add a function that allows a farmer to add data and track symptoms for multiple pigs.

Challenges we ran into

Linking login to the database, learning PHP, HTML, SQL... thanks Youtube!
