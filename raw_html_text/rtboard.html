Inspiration

We came into the hackaton hoping to be inspired by some of the Challenges proposed by the hackaton's sponsors. Itequia's made us think of the great possibilities such a software could have

What it does

Our web allows the user to connect to a lobby (identified by a code), and draw and send emojis in a given canvas.

How we built it

We used HTML, CSS and JavaScript for the whole web, and node.js and socket.io for the connection to the server

Challenges we ran into

We had no experience in any of the aforementioned subjects, so we had to learn it all from scratch

Accomplishments that we're proud of

The lobby management is something we are really proud of, as well as the general feel of the web

What we learned

We learned a lot about HTML, CSS and JavaScript, but more importantly, how to manage a project with other people as it was the first hackaton for some of us

What's next for RTBoard

Use timestamps in order to execute processes in the order they were sent, not received Add text input Increase the number of available emojis Multiple canvas available to a same room
