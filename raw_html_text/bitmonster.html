Inspiration

An easy web-based platform for Bitcoin payments centered around anonymous aliases (with reputation!)

What it does

Allows users to conduct Bitcoin transactions under anonymous aliases, each with its own reputation. Track and bump transactions between members of the anonymous community. Add a humorous message to your transaction.

How we built it

We built the web app using React.js and Python Flask, hosted on Heroku.
