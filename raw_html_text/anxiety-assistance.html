Inspirations

A lot of people face anxiety and we wanted to help reduce anxiety for anyone who needs help during a crises or just a time of high anxiety

What it does

It connects people to hotlines or specialists. Also provides a forum to share stories and post positive comments. There's also a mood tab to distract and relax the user

How I built it

We used HTML and CSS

Challenges I ran into

When we merged files, we all did it from different styles and it wasn't merging well.

Accomplishments that I'm proud of

We all learned something new and we were all able to contribute to this project

What I learned

I learned to use the same style sheet for all the files and to possibly put it on github earlier so that we can see the updated changes and not have to keep sending each other updated files

What's next for Anxiety Assistance

We were thinking of putting on some interactive games. And technically the live chat and forum doesn't work right now but that would be a future step.
