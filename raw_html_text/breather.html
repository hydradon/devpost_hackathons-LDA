Inspiration

As we keep using chrome for a long period, our brain could get exhausted and thus influence our efficiency and health. We want to regularly take a break after a period of use of chrome to refresh our brain and be more energetic on works.

What it does

It is a reminded chrome extension that allows you to take a break like meditation, stretching, and other mindful stuff to relax after a period of use of chrome.

How we built it

We used Javascript and HTML

Challenges we ran into

The logic of Javascript and HTML when making a chrome extension

Accomplishments that we're proud of

Everything we have built.

What we learned

Most of us hardly used HTML and Javascript before, so this is a good chance for us to learn them. We also have got the basic ideas of a chrome extension

What's next for Breather

We'll further develop Breather to make a more complete and useful extension. Also, with the skills we developed here, we can make more interesting extensions in the future
