Inspiration

Who doesn't love it when their driving is scrutinized?

What it does

DriveMom is a an app that uses the openxc OBD device to collect data on the driver's driving style. This data is then analyzed to give the driver a score based on their performance on every trip.

How I built it

This project uses a native Android frontend and a Javascript backend on node and mongodb. The data simulator is from an openxc flask app.
