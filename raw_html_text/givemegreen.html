Inspiration

Many environmental projects do not get the funding and attention that they need to be successful.

What it does

It allows people to invest in environmental projects and for them to post their own projects to be funded.

How we built it

We created an IOS mobile application that uses the Coinbase API to make transactions between investors and project creators.

Challenges we ran into

We had trouble communicating values from our JavaScript functions to our server due to the asynchronous nature of JavaScript.

Accomplishments that we're proud of

Successfully using the Coinbase API to manage and create wallets for numerous projects.

What we learned

Asynchronous languages are difficult to use with servers.

What's next for GiveMeGreen

A feature that allows a user to donate an amount of his choosing to a particular project rather than a fixed amount that has already been set ($1).
