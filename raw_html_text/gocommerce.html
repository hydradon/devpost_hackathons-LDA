Inspiration

We wanted to learn a new programming language, and we were like why not Go lang. Also, we were interested into a financial hack and contributing to the open source community.

What it does

Gocommerce is a go-based API for static e-commerce sites. It handles payments and orders, including the international trading market.

How we built it

We used go-lang as the scripting language and heroku to deploy the backend server.

Challenges we ran into

Getting the dependencies of heroku to install without crashing on us.

Accomplishments that we're proud of

We have overcame uncertainty in the financial spectrum of the Gocommerce platform by increasing the precision of calculations.

What we learned

It takes a long time for deployments to take place. You just can't deal with RAM! :P

What's next for gocommerce

Make the financial transactions more transparent for users.
