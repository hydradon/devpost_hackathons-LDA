Inspiration

We have always been struggling to find parking in a mall or city. We want to improve the finding of parking spaces with IOT and our app.

What it does

Our system would have the following features.

An app would be provided to the customer so it can register and have best use of the parking.

The user would have to register to it and all of it´s cars

When the users enters a mall with our system, using the security cameras it will take a picture of the plate and recognize it, open a the gate and start a time counter so the customer can be billed later.

Using the security cameras on the mall, it will count the empty spaces and tell them where there are trough our app.

How we built it

We designed our mockup with Affinity and Photoshop, our backend, python for our image recognition.
