Inspiration

This inspiration came from looking to do something new. I had never done a hackathon and let alone done on my own. I had created a application that runs as a multi platform build with MongoDB Clusters and Stitch.

What it does

It logs in and is hosted in Google Play store to become a integrated platform of kinds.

How I built it

Made with React-Native from the ground up in application certified through Google Store implementations. Design in Windows Paint 3D and Paint for display promo images.

Challenges I ran into

The challenges I ran into were waiting for building the application when working with a slower computer to be handling.

Accomplishments that I'm proud of

Accomplishments that I'm grateful for are the ground up application creation with no Expo just Android Studios as an Unemployed Developer. I'm proud of running it and maintaining it during development until production.

What I learned

I learned that even if you work quickly, there is a possibility that something that can slow you down in development is a slower even after maintaining disk cleanups.

What's next for Risen Radiance Mobile Application

Possiblities are endless. ARCore foundations have been complimented already. Store Handling in transactions to be placed? Whatever the need maybe, it's a React Native application created and hosted upon request.
