Inspiration

I'm constantly told by my mom throughout the day during my off time to get up stretch, go for a run, _ etc _ while i'm gaming with friend on my pc

What it does

Messages you every so while you're playing to remind you to get up, get water, eat, stretch, etc

How I built it

Using the discord.py module and discord bot api

Challenges I ran into

learning asynchronous execution, event listeners, and decorator functions, understanding what a snowflake is and how to use it.

Accomplishments that I'm proud of

Beginning to expand this program with a website. I'm using the Bulma Front-end Framework to make a static website for now that will soon be the page to automate the bot to join servers.

What I learned

Asynchronous execution, Event Listening, Decorator functions, Linux Epoch time, CSS styling, static web hosting

What's next for Llama Mama Discord bot

Generating more tips for the user, sending periodic links to users about health and wellness and a quick link to have the bot join server using oAuth.
