Inspiration

Not a lot of apps allow to search around your area for specific food categories, and easily see information.

What it does

Locates and gives information about local restaurants of a certain type of food category. It also allows you to Snapchat the information over to a friend.

How I built it

Everything is ran on Android. It utilizes a REST API for restaurant data.

Challenges I ran into

Networking on Android is very restricted and required careful consideration. Photos from the API were not able to be acquired via an API or web scraping.

Accomplishments that I'm proud of

Learning the Snapchat API and building an Android app.

What I learned

Pretty much the same as my accomplishments!

What's next for FoodFinder

I want to be able to create multiple Snapchat stickers for the various food items that are available at found restaurants.
