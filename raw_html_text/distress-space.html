Inspiration

911 emergency calls.

What it does

An app where refugees can request help with a single click of a button in distressed situations and sends their location and name to a refugee agency who can provide an emergency response.

How we built it

A react app on using a firebase backend and sends messages using the twillio api

Accomplishments that we're proud of

Getting firebase and twilio api to work nicely together and getting geolocation working

What we learned

The twilio api is easy to use

What's next for distress.space

Connecting to the refugee and refugee agency community.
