We as a team wanted to create an app and a website to ease the access to surroundings for people with special accommodations. Our app and website track all the places that have been built in a way that is not accessible by people who need special accommodations (For example- no lifts, no ramps). We built this app and website using React and HTML5. The primary challenge we faced was trying to learn and use Google Maps API within a few hours and implementing it on our mobile application and website. The next step would be to promote this app to make clear any bugs that were not fixed.
