Inspiration

As commuters attending Rutgers University, we deal with the troubles of keeping up with the various rules of parking on campus in order to avoid parking tickets. However, we all make mistakes and sometimes park in an unauthorized spot. This app will tell a student with a parking pass, whether he or she is in the correct parking space.

What it does

Uses geofence API, a google location service, along with information about the parking pass holder, particularly which zone the parking pass is, in order to alert one of an incorrect spot.

How I built it

Used android studio and useful google location APIs

Challenges I ran into

Beginner knowledge of android studio.

Accomplishments that I'm proud of

Completing the many parts of the app - writing the logic for alerting the user of incorrect parking, implementing a location API, developing a complex UI, and handling events.

What I learned

Stuff about Android Studio and how to implement APIs.

What's next for RULots

We will improve upon its functionality by being able to test it. We will add functionalities that are useful over time.
