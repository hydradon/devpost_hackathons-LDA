Inspiration 

 The inspiration for this was to have a user be able to connect with friends by bonding over different achievements they have earned with their pets. Everyone shares moments on Instagram and Facebook by having fun utilizing features such as filters. We wanted users to have their own pets in a virtual space on the go and when they are home. Social media has a way for allowing introverts to be themselves and interact with others by connecting through having virtual pets.

What it does 

  Within AR the user will utilize facial expressions to have the pet (Dragon) respond to the user. If the user lifts their eyebrows the pet will wiggle side to side. If the user opens their mouth food will appear and the pet will respond by moving towards the user's mouth. With in Oculus the user can pet, feed and play with the pet (Dragon)

How we built it 

 The way we built our VR environment was with the use of Unity Game Engine as for our AR portion we utilized Spark AR Studio and used a mix of the Patch editor and Scripting.

Challenges we ran into 

Some Challenges that we ran into was the learning curve of Spark AR. We tried to create the VR experience using React 360 but due to the internet problems we were not able to implement the VR using the react 360 framework. Lastly connecting to a database to have the AR and VR data to by synced whenever the user interacts with the pet in either reality

Accomplishments that we're proud of 

 We are proud that we were able to create two applications. Being able to use VR and AR work hand in hand. Having a short window to create our vision and be able to have the projects functioning is an accomplishment within itself.

What we learned 

 We learned how to use Spark AR by blending scripts and patch editor. We also learned how to utilize unity with allowing the user a VR experience.

What's next for XR Pets 

 Next we would like to have the VR and AR experience blend. The user will be able to choose and enjoy their pet in AR and possibly messenger. Then when the user gets home they can enter the VR space and interact with their pets and go on quest. Lastly once Facebook Horizon is launched the user can have their pet within that world
