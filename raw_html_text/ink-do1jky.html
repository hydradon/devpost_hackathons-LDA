Inspiration

Childhood memories of etch a sketch coupled with social presence.

What it does

After downloading and opening Ink, the user is prompted is prompted to allow Ink to to modify the lock screen. Afterwards, the user is able to access Ink through an icon (like a widget on the lock screen) and send drawings/sketches to contacts and vice versa.

How we built it

Firebase was primarily used in constructing the backend. The functionality and all was created in Java for Android.

Challenges we ran into

Notifications for user receiving a drawing/sketch (Firebase does not support this).

Accomplishments that we're proud of

Using Firebase to send data from one endpoint to another (one user to another).

What we learned

How to use Firebase as a backend service.

What's next for Ink

Snapchat levels of greatness!
