Auto work log is design to help employers to track the employees working hours, and calculate their hour based salary

Each employee needs to install the app on his / hers smartphone. The app is using Neura to collect location data about the worker's working hours. This data will be sent to the employer. The company server will receive a daily working hours report for each worker, based on the information provided by Neure.

  • The company's HR manager will be able to calculate the workers salary easily and accurately.
  • The employee will be able to keep track and monitor the salary easily, using the app.

No more manual log apps, no missing data, and it's all done automatically without the need for human interaction.
