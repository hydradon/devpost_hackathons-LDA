Inspiration

We believed this was a small but powerful tool in the struggle against online web harassment.

What it does

The extension replaces all instances of profanity with a kinder alternative, or simply the word "pie".

How we built it

We wrote the basic logic in JavaScript and integrated it into Chrome's JSON file format.

Challenges we ran into

We did not know how to make Chrome Extensions or had worked with JSON before.

Accomplishments that we're proud of

We finished our first Chrome Extension.

What we learned

This was our first experience in using JSON, as well as using JavaScript to modify large parts of the web page.

What's next for Curse of the Pie

In the future we can have users add to the dictionary of words to censor, as well as add a notification for when words are censored, along with a counter of the number or words censored.
