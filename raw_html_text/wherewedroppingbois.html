Inspiration

We thought we should do something related to web development, and we ended up making a html clicker game.

What it does

It is a choice based RPG game with a bit of random chance. You goal is to kill as many orcs as you can before you fall.

How we built it

Using html, css, and javascript.

Challenges we ran into

The biggest challenge we had was to integrate all the different team members work together. Also, we dreamt too big at the beginning and had to scale down dramatically given the time limits of the hackathon. Also, we had to learn the language before we started (as none of us really knew HTML Javascript before we started).

Accomplishments that we're proud of

Coming out with a working product. Working together as a team. Having fun.

What we learned

Web development skills. Github skills.
