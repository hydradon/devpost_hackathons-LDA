Inspiration

Many Americans are depressed, have anxiety, have been victims of sexual harassment, and have been abused. Therapy is often cited as the cure; however, therapy can rack up large bills. We hope to find another way to help these people.

What it does

Lighthouse allows collaborate drawing.

How we built it

We used HTML and JavaScript to format the website. We also used node.js to set up the server for collaboration.

Challenges we ran into

It was difficult to set up the server; in addition, showing one user another's drawings, when the other user used different brush sizes, was challenging to implement.

Accomplishments that we're proud of

We are proud of the formatting and layout.

What we learned

We learned about the large number of things possible through HTML and JavaScript.

What's next for Lighthouse

We hope to make Lighthouse better with more canvasses.
