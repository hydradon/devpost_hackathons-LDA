How I built it

I used Corsair's CUE SDK to develop a "flappy bird" inspired game where keyboard keys are used as both the display and user input.

Challenges I ran into

This was my first time working with C++ so it took some time to setup visual studio and dll libraries for the sdk.

Accomplishments that I'm proud of

The game is fun and I learned C++.
