Inspiration

We are three rookies that wanted to create video games.

What it does

It's a minigame where the player controls a 2d rocket. The goal is to land on top Valley-H Planet, but only on top. If you tried to penetrate the planet from underneath, you would explode. Oh, one more thing, there are angry magnet birds on the way to Valley-H Planet, so watch out for those.

How we built it

We built the game on Unity3d and C#.

Challenges we ran into

We have a lot of difficulties generating borderless map since we only have one destination. Therefore we built teleport mechanism that created the illusion of infinitely wide map.

Accomplishments that we're proud of

This is the first working game that we ever built, so it was a huge step for all of us.

What we learned

Each one of us learned a lot that we did not know before. Cumulatively we have learned: coding in C#, using libraries and tricks in Unity3D, and many photoshop techniques.

What's next for Soylent Express

We want to finish the game, add in few more functionalities and skins (few ideas in mind), and hopefully publish it on some minigame platforms.
