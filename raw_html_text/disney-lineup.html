Inspiration

When going to an amusement park what do you want to do? Waste time in line? NO! You want to go on all the rides is what you want to do. With this web app your time will be managed more efficiently.

What it does

Uses the themeparks API to get the wait times of the rides, which would then be used to help plan the persons day accordingly in order to spend less time waiting in lines. Also has a chat bot that interacts with the user to get/give information

How we built it

Built with JavaScript, HTML, CSS, BootStrap and many other things.

Challenges we ran into

Team members were not experienced in all the languages and tools we used, so we all had a first time experience with them.

Accomplishments that we're proud of

We were able to come up with an idea although we didnt finish it completely we still learned from this experience.

What we learned

Learned new languages that we have not been exposed to yet.

What's next for Disney LineUp

Implementing the things we were not able to get to, such as a chat room where people can connect with each other while waiting in line, by connecting possibilities such as chatting and playing mini games.
