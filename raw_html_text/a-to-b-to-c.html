Inspiration

Going on a road trip across the country using an electric car is not feasible. It takes up to 4 hours for a tesla to charge from 0 to 100%. This rideshare service is a solution for those who want to travel green without making long stops

What it does

navigates to places and show nearest charging station for a person

How I built it

Using swift, SmartCar API, Here Maps API

Challenges I ran into

Backend development and building server to talk to SmartCar backend

Accomplishments that I'm proud of

Learning about mobility security
