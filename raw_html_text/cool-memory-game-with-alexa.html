Inspiration

Alexa is the future of voice assistant and we wanted to contribute

What it does

Its a memory game where Alexa tells you 5 numbers and asks you if you remember

How we built it

We used Amazon Webservices, Lambda, Python, Flask and Jinja Template and Yaml files

Challenges we ran into

The the communication relay from echo to the Alexa skill set console to the developer console where we had written our lamda function and then back with a response was difficult because of configuration setting errors.

Accomplishments that we're proud of

Alexa is built in a way to address when its called by its name "Alexa". So something very cool that we did was making Alexa talk back without invoking or calling "Alexa" explicitly once we have submitted response or spoken back. So our system was able to build conversations with Alexa without having to call back by its name.

What we learned

How much clear communication between team members make a difference, how valuable version control is and you should always start configuration settings early.

What's next for Cool Memory Game with Alexa

Scale it to industry level where Alexa can remember which conference rooms are booked by whom and which are empty. Or find best resources for a particular jobs based on their qualities and skills.
