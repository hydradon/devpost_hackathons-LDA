Inspiration

We wanted to create a game that combined the challenge of learning something new with making something creative. Stackity combines attributes from Tetris, World of Goo, Getting Over It, and Bubsy the Bobcat to create a unique, non-stress game.

What it does

The objective of this game is to get Zipper the Raccoon as high as possible by clicking on the screen to spawn trash. There are 6 different types of trash, each corresponding to a color (Orange, Blue, Purple, Red, Green, Yellow), if four trash objects of the same type touch, they'll disappear. Watch as Zipper gets higher and higher by climbing the weird and whacky items that pile up and enjoy the puns!

How we built it

We used Unity to build the game while the assets were created with Procreate and Piskel.

Challenges we ran into

We had some difficulty from Unity. It's an engine we've never used before so there were some features we weren't aware that were available or that would affect the physics or collision of objects. There were a few times we coded something that Unity already included. However, this helped us learn new things about Unity and gave us extra practice we wouldn't have gotten.

Accomplishments that we're proud of

50 Unique Sprites for Droppable Items! 50 Unique Puns for each Item!

We're proud that all of our assets are custom made. There are a total of 50 trash assets, background, and Zipper's sprite and animation. We were also able to create a minimum viable product game on Unity, an engine we've seldom used before until today.

What we learned

We learned how to use unity along with creating a large amount of assets in a short amount of time.

What's next for Stackity

We hope to add more game modes along with more assets and puns!
