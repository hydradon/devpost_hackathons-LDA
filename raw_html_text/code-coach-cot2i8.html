Inspiration

I'm moving off to college soon, and my sister won't be able to have me help her learn to program. I build CodeCoach to eliminate the friction for her.

What it does

It's creates a one-on-one coding session that allows a student and a teacher to collaborate on a coding lesson in the same document while also providing the benefit of face-to-face communication with WebRTC.

How I built it

I used Node.js to handle the backend and takes requests to the site, Redis to keep the data persistent for each unique url, and a combination of different frameworks and hacks to keep everything live on the page.
