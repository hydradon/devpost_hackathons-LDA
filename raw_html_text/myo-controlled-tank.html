Inspiration

The Myo wristband is a wonderful product, allowing a really dynamic range of gestures to be picked up. Using this fact and a convenient API, we created (or programmed, rather) a robot tank that is controlled by Myo gestures.

What it does

Moves around, allows for someone either incapable of using a remote control or someone who just hates it.

How we built it

Using Myo API for python, python sent UDP packets via a local network we set up, which transmits the packets to an Arduino with WiFi shield that controls the robot.

Challenges we ran into

Arduino WiFi dongle is archaic so many trials and tribulations came with that and other hardware/software issues.

Accomplishments that we're proud of

Getting it to work! Each interface was a unique accomplishment on its own though.

What we learned

A lot about APIs and how easy python is to do otherwise obnoxious stuff.

What's next for Myo Controlled Tank

Cameras streaming, more sensors, more power and speed. Although the tank itself is Raytheon's.
