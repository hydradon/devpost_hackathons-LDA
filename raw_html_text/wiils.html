Inspiration

The main inspiration for this project comes from the difficulties the bicycle users have in order to travel within the city.

What it does

It can be used to calculate the best routes to travel in, based on many factors, such as the bikeway routes, safe zones, presence of other cyclists, and so on.

How I built it

We are on the designing phase of the project. We used Adobe Illustrator in order to make the design feasible and then we prototyped the app on a Wix server.
