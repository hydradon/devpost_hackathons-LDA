What it does

This is an api to convert motion control into keyboard input, which can be used to control games

How we built it

We used JavaCV, and OpenCV to perform vision processing on a webcam's input. It searches the image for hands, and does keyboard input based on hand position

Challenges we ran into

It turns out that actually simulating key input can be very messy, and the timing can be unpredictable

Accomplishments that we're proud of

To be honest we came in here expecting nothing to work. The project ended up being more reliable than we expected

What's next for Air Guitar Hero

As we said, you can play more than just guitar hero with this. Hopefully in the future, we can expand this to more games
