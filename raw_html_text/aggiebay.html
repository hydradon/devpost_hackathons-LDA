Inspiration

We wanted to address the following issues that vets commonly face when transitioning from active duty:

  • information on services to help vets is difficult to obtain
  • the large amount of information out there is overwhelming
  • some veterans don't want to ask for help and assistance
  • the process of getting to know veterans and the services that they're eligible for takes a long time

What it does

We created a website and corresponding app that is meant to be an open source, one stop shop for all programs that exist to help veterans.

How we built it

We built the website with html, css, javascript, and bootstrap. We began to create a database as well with MongoDB, but it was difficult to accomplish within the time period. The application was built with java.

Challenges we ran into

The time and technology was a challenge. We were only able to create a basic skeleton for the MongoDB database. There was just so much information to sift through that we weren't sure what was most important to add to the tables. So, we had to remove the database and the traces of it's implementation in order to get the website to launch. We also weren't able to implement the website scrubber that we created because of technical issues. We created a website scrubber that would be able to go through websites of nonprofits and pull important information that would be useful to veterans in their search for helpful resources, and would add this information to the database and therefore the website.

Accomplishments that we're proud of

We are proud to have a solution to present and proof of all the aspect of the solution that we weren't able to accomplish within the time period.

What we learned

We learned a lot about all the innovative technologies that we were ambitious enough to research and try to implement to bring more value to our solution.

What's next for AggieBay

We want to continue our school year strong. We want to learn much more about different technologies that are out there to use in the world. Some of use hope to go on to win more hackathons (we're not sure if we'll always be on the same team, but we're all friends and hope for the best for each other).
