Inspiration

My brother always wanted to learn coding but he never could because it was too difficult for him. I decided to make coding a lot easier.

What it does

It allows people to code with a custom .ez file.

How I built it

I took a custom made .ez file, read through the contents inside, and convert the ez code into C++ code.

Challenges I ran into

Not having enough space inside of vectors, not having enough time to implement new ideas.

Accomplishments that I'm proud of

Building it and having it work.

What I learned

Language building is hard.

What's next for EZ

New features, such as control flow.
