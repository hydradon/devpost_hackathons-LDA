Inspiration

We wanted an easy way to track an historic journey or chronological series of events. Our original aim was to make a study tool.

What it does

It allows a user to create a map of chronological events leveraging Google Maps API seamlessly. Information about each event can be added including date, general description an image and a youtube video.

How we built it

We built it using Flask server framework, hosted with Amazon Web Services and served over the internet with Apache and Nginx. We built our front end with Bootstrap, Javascript and html. Interfacing with Google Maps was done with Javascript.

Challenges we ran into

Certain elements of working with the Google Maps api, including coloring the arrows. Creating reactive pages to post from our page when creating the linked chronological 'tours'.

Accomplishments that we're proud of

The whole thing. First successful use of a Google API. It's also a tool that we could see ourselves using and sharing with our friends and family.

What we learned

That we can successfully use Google APIs! That we can always come up with something to demo, even if it is held together with proverbial duct-tape.

What's next for HereInHistory

More Interactivity, ability to search for tours created by others and better looking front end.
