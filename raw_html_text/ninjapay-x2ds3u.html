Payment so easy with social impact!

Make Payment so easy that even our 'Grandmother' can do it with a click! Also, make a social impact at the same time while you make payment.

Our 'NinjaPay' app makes possible global payment possible just using the contact number. NinjaPay app abstracts all the bank details behind the contact number.
Additionally with every payment done we add smiley points to your account (just like the loyalty points) which can be exchanged for any chosen Social cause.

Addon features includes the following

1) Scan and Pay: System is capable enough to scan a QR code in order to pay any bill

2) Skip The Line: System helps user to scan barcode of any product of super market(AH,Jumbo,Lidl etc) and connects with supermarket's pricing and inventory api , and also facilitates user to pay via QR code scanning mechanism so that user does not need to stand in the queue .

3) Split In The Group: Admin can create a new group and can invite other users . Users can enter their expenses, and once admin finalizes the settlements , all other users in the group will be notified how much he/she needs to pay to others. Clicking on a single button everyone in the group are being paid.

It is a global app which also prompts you for exchange rates and guides you for the best rates.

We built this app on the android platform. We used 'Firebase' for authentication. We used Payment API from Finastra for facilitating the payment. We also build app server (Ninja Payment server) for facilitating the payment and interfacing with Finastra API. Also we built the functionality to add 'Smiley Points' with every exchange.
