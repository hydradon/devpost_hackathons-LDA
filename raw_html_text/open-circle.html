Inspiration We want to develop a community for female programmers.

What it does It allows women to gather as a community and nurture the growth of programming among women.

How we built it We used github to gather our code as we worked on separate parts divided among ourselves while helping each other at small bumps along the way.

Challenges we ran into A big problem was organizing all of our ideas into one concrete and uniform website. A second big problem is the networking, trying to figure out a way to organize our clients data onto a database and access it throughout the site.

Accomplishments that we're proud of The collaboration and the amazing ideas we have all come up with has been amazing.

What we learned

How to implement a template How to use bootstrap About github and how to use it How opensource works How to think about problems and how to approach them.

What's next for Open-Circle Finishing the our site, Open-Circle
