Inspiration

Looking for a project with a social impact we believed that limiting ourselves to a single cause is an error. We decided to create a platform to give visibility to all kinds of social projects.

What it does

4Society connects people that want to change the world, allowing them to create projects or joining and supporting some already created, materially and in with human help.

How we built it

Using Golang and Javascript for Backend/Frontend.

Challenges we ran into

ThoughtWorks: Code what matters.

Accomplishments that we're proud of

We have developed a functional project that in the future will be able to help people to mobilize crowds and make a better world.

What we learned

Working better together and organize time.

What's next for 4Society

We want to continue the project and develop a final product that people can use and try to make a better world.
