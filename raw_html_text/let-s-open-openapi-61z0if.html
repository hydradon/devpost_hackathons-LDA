Inspiration

Spark day about Developer Portal for FFDC which show as how hard is to use APIs from API Catalog and what effort is needed to run it.

What it does

Enables an easy way to look through the Open APIs available in FFDC Portal. Additionally, it allows executing request from it without any additional software/applications.

How I built it

We used Powershell to get all information about available APIs and use it to produce valid API request. Powershell is also used as a webserver which with Angular based UI allowed to provide a well-designed interface to perform API requests.

What's next for Let's OPEN openAPI !!!

Implement our project into FFDC platform as improvements of existing API Catalog or as API Catalog 2.0.
