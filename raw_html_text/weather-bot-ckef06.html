Inspiration

To build a simple telegram bot that has practical uses.

What it does

The telegram bot takes in location queries from users and responds with the weather and temperature of the location.

How I built it

We used Python and the Telegram and Open Weather Map APIs.

Challenges I ran into

We struggled to understand the Google API, hence, we had to change to using the Open Weather Map API due to time constraints.

Accomplishments that I'm proud of

We have built a telegram bot which is for practical use and taps on existing APIs. This is our first time working with APIs so it was a very good learning experience.

What I learned

We have learnt to operate the Open weather Map APIs and how telegram bot works with the server and users.

What's next for Weather Bot

We want to extend it to a web front end by using the common back end. As for now, our telegram chat bot is working on the local server, we want to use Heroku to allow the bot to be working even when we are not running it locally on our computer.
