Inspiration

When trying to learn techniques for judo promotions it was frustrating to have to look up a technique every time I couldn't remember a name. I'd have to find a description and video each time.

What it does

Conveniently gathers all of the different techniques into one app that has a description of each as well as an embedded YouTube video of the technique being demonstrated. It also contains information for exercises such as stretching.

How we built it

We used android-studio and android emulators to develop the app. Our code was written in java and xml.

Challenges we ran into

We initially struggled with creating the application since we had never created an app before even though we have a decent amount of coding experience. Due to this we had to spend a lot of time researching how to do different things that we wanted to, as well as how to work with Android Studio.

Accomplishments that we're proud of

Having successfully completed our first app and having all of the main information we wanted to include actually be in it was cool.

What we learned

How to create an android app and work with android studios. How to embed YouTube Videos into apps and work with the YouTube API.

What's next for Judo Exercise App

Moving forward we may add more exercises not just those related to judo as well as improve the UI. We made sure that the styling wasn't completely terrible but it definitely wasn't as aesthetically pleasing as we could have made it.
