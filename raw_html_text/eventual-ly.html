Inspiration

Reduce The Clutter Of To-Dos In Your Camera Roll

How it works

Eventual.ly is an automated solution for users, extracting information from images. Take a picture of a poster and the event will be created on your calendar with the correct date and time.

Challenges I ran into

Integrating API for OCR was challenging along with comparing two results from two OCRs to improve accuracy. and using stanford webprotege online service for concept mapping using ontology network.

What's next for Eventual.ly

  • Seamless sharing with friends
  • Event managers can monitor poster exposures
  • Developers can create powerful connections with users via Eventual.ly e.g. link to ticket vendors on Songkick via Eventual.ly
