Inspiration

We don't like spending time looking for a free parking space

What it does

Finds optimal parking spaces for cars driving through the city in real time.

How we built it

We combined navigation system with a dash cam. We create a network of IoT cameras which report the state of available parking space in real time.

Challenges we ran into

Data collectionm without users we had to generate a synthetic dataset using probabilitic programming.

Accomplishments that we're proud of

I'm proud of my teammates ;)

What we learned

There are still many steps to achive a true smart city

What's next for carsense

As more drivers start to use carsense, the accuracy of our predictions will improve exponentially. We will integrate an efficient video classification machine learning algorithm to register stationary cars as other users drive by.
