Inspiration:

how to simplify parking lot management?

What it does:

it allows access to the parking lot for specific cars.

How we built it:

We ran an image processing algorithm to recognize the licence plate of the vehicle, the licence plate is sent via a text file to web application that checks if the licence plate is registered in the parking lot; if yes the gate opens, if not the access is denied.

Challenges we ran into:

extracting the licence plate from the image.

Accomplishments:

proud to participate in my first Hackathon and made some new relationships.

What I learned:

image processing and segmentation.

What's next for parking management system:

adds another security feature by checking both the licence plate and the driver's face to grant access to the parking lot.
