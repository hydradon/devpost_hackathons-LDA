Inspiration

We wanted to create a fun way to discover new places and attractions in the city.

What it does

This iOS application allows users to browse through nearby places and attractions in their city and plan things you would like to do and places you would like to go to. Adventure Now allows users to easily create their plan for their day, making it a great tool for tourists and local groups of friends.

How we built it

We started by designing the basic layout of the app and then modelling out the main features as well as integration of the API using Apple’s Xcode using the Swift programming language. Next we designed the app in Sketch and finished off the UI for the app.

Challenges we ran into

We discovered this hackathon a little over a week before the submission due date, so the project had to be developed in a very short time. We have largely achieved what we set out to do in that time frame.

Accomplishments that we're proud of

We are proud of how efficient we were able to be and how well we worked together in what ended up being a very well rounded and effective team. And we learned lots through this process.

What we learned

With persistence and dedication anything is possible.

What's next for Adventure Now

We are continuing our work on the app and are planning to deploy it to the App Store soon. We have a long roll-out plan of cool features that we want to release through updates over the next year or so.
