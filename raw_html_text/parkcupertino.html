Inspiration

We are always getting stuck in traffic going to school and other places in Cupertino. We wish to make the process easier and more organized.

What it does

The program includes a map that pinpoints parking lots, reviews whether a street or parking lot is crowded, and displays events. It can also calculate the number of parking spots based on any parking spot.

How we built it

We started with using javascript for the front end and java for the back end.

Challenges we ran into

It was difficulty to get the java code changed to javascript.
