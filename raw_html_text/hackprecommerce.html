Inspiration

We wanted to help small businesses by giving them new options of payments to innovate the way they currently do business.

What it does

Provide more Payment options to small businesses in Puerto Rico.

How we built it

We built an eCommerce Website and incorporated ATHMovil's API for processing our payments. We used Django Framework, JavaScript and HTML.

Challenges we ran into

Working with the cart logic and how we were going to present the payments was a few of our many challenges.

Accomplishments that we're proud of

Being able to integrate our app with ATHMovil's API

What we learned

We learned more about the great things that offer Django Framework and new functionalities using Javascript
