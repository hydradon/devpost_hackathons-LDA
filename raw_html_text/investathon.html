Inspiration

We were looking to understand the process of how small businesses and larger corporations/investors help them grow when we realized how there is no real platform where investors.

What it does

This platform allows for Small Businesses to find connections with investors in their work-sector. Small Businesses can adjust keywords and services they want to be associated with, and contact investors to create new market opportunities.

How we built it

We built this using HTML, CSS, JS, with a python-flask backend running on a google cloud server.
