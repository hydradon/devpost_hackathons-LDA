Aurora Metrics: The Face of the Future. Interested in making sense out of the massive data generated in social media

Built as a web app using javascript, PHP and Python to handle calculations, we built on a Language-independent Bayesian sentiment mining of Twitter method and added a weighted average formula to asses two measures on a good to bad scale.

Integration, the limits on twitter requests and not giving up. We ran into the issue of integrating everything with all the other pieces. Troubleshooting and testing provided difficult with the Twitter API limiting how many requests users have.

We created a product that is viable and polished. Although there are more features we are excited to add, Aurora is at a good early stage.

Twitter API, natural language processing, and integrating smaller modules into a larger system.

We hope to continue adding different features in the future; other social media integration, more mathematical data crunching, and explore other language processing tools.
