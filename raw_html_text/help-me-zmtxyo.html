Inspiration

I have developed a bot that makes it easier for users to access the discord chat platform. I wanted to make it easier to communicate through this bot.

What it does

discord provides very basic functions, such as text chat and voice chat. This includes joining the group, setting the name, auto-granting the member role, and asking for easy help if you have a problem. Easy access to the helper.

How I built it

Based on node.js and using npm discord.js

Challenges I ran into

There is not much example information. I started from the beginning one by one. This allows many features to be implemented.

Accomplishments that I'm proud of

It still works without errors.

What I learned

increased ability to use discord.js

What's next for Help-Me

I would like to add more help features.
