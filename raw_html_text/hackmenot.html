Inspiration

Lots of people face cyber security risks due to ignorance. We want to change that.

What it does

Educates users on cyber security and provides a password generator.

How we built it

We coded in html5, javascript, and css using text editors and online ide's.

Challenges we ran into

Unfamiliarity with the languages we were working with, styling the site with css, and coming up with a succinct random number generator.

Accomplishments that we're proud of

Coming up with a finished project in time. Our very cool intro sequence.

What we learned

We learned javascript, improved css skills, and got insight into html 5 structure.

What's next for HackMeNot

Creating a news feed with the latest cyber security information. A UI overhaul with more css knowledge. A more fluid simulation which includes more interactive features. Website design and refinment
