Inspiration

Many veterans are not up to date with the high tech of today or just prefer old fashioned processes. Ulysses can be used in public libraries and connects vets to a filtered set of benefit services.

What it does

It filters veteran benefit services according to their own reported needs.

How we built it

A survey on the front end created by JavaFX sends information to the back end which filters the data in our database and displays results to the front end.

Challenges we ran into

The intricate details of java halting the program and our development process. Scrapping or revising code we just spent a significant amount of time on, slowing the development process.

Accomplishments that we're proud of

Creating Ulysses in 24 hours. Using JavaFX when we had little previous experience.

What we learned

We learned a more of the intricate details of the Java language and JavaFX.

What's next for Ulysses

Adding more features such as a resume builder and make the filtering of the data more efficient.
