Inspiration

I wanted to create a skill that helped people to learn about geometry. As I saw that amazon had implemented a new APL interface, making it easier to integrated visuals into alexa skills, I had to create a skill. I decided on creating a skill that helped education. By having visuals, we can easily teach various shapes. I want this skill to enrich and help develop knowledge of mathematics everywhere.

What it does

This skill provides various shapes and descriptions. It allows a user to click the screen to get more shapes.

How I built it

Built using APL and AWS Lambda.

Challenges I ran into

There were no tutorials on APL, so I had to learn it from the documentation.

Accomplishments that I'm proud of

I got it to work, and the UI is good.

What I learned

I learnt how to use APL

What's next for 3D Shapes

I'm planning on adding more shapes, and more tibits on knowledge.
