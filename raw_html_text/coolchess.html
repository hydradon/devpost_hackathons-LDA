Inspiration

Our original hackathon project was not working. Midnight was approaching, and we were stressed. One team member took out a piece of paper and started drawing this smaller version of chess. From then on, we decided to build Arrow Chess

What it does

Condenses the traditional game of chess.

How we built it

We used HTML, CSS, and JavaScript. The project was done in the front end. With modern advancements in browser technology, we were able to completely forgo a back-end.

Challenges we ran into

Our first project was a failure because the most important library for the project was not compatible with the rest of the project.

Accomplishments that we're proud of

We finished our project.

What we learned

We learned how to drag and drop in HTML and how to use git to work on the project together.

What's next for ArrowChess

Turn it into a mobile app and add more rules such as promoting the pawn into a new piece and winning by advancing the king to the enemy's side.
