Inspiration

We are a group of friends, interested in technology and new challenges.

What it does

This project offers a different way of interaction within a Virtual Reality enviroment through Computer Vision and and intercomunication between diferent components of the system.

How we built it

The Project is built over to different platforms: The first one is Unity , a 3D Game Engine with the integration of the HTC Vive and the Logitech VRInk . The second one is Python and OpenCV , to handle computer vision. Both platforms are connected with a database, specificaly Firebase , using the REST endpoints it provides.

Challenges we ran into

The first and obvious challenge was the performance of the system . We had to use two different computers to launch each of the parts to solve this problem. As a result we faced another problem, which was the communication between computers . We have had to give performance and precision in exchange of an easy way of perform that communication.

Accomplishments that we're proud of

We are really proud of being able of integrating many different areas and technologies to offer an intuitive and different way of interaction in a attractive enviroment.
