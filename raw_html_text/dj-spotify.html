Inspiration

We wanted to compose a way for party hosts to allow their guests to choose the mood of the night!

What it does

Our app allows guests to collaboratively compose a spotify playlist and automatically have the songs ordered based on our algorithm

How we built it

We used a laravel back end that communicates with the spotify API to authenticate a user with Spotify and play a list of guest selected music

Challenges we ran into

We ran into a couple blocks when were attempting to figure out how to determine when a song has finished. The spotify api does not provide a way around this.

Accomplishments that we're proud of

We were able to get a list of the users saved playlists and en-queue them to a collaborative list.

What we learned

We learned more in depth about how laravel works on a back end and how to communicate with spotify via their API. we also learned how to create a front end UI

What's next for DJ Spotify

We could implement more features and functionalities such as guest voting, host blacklists and more
