Inspiration

A lot of people, especially in big cities, live in small living spaces. A wardrobe takes a significant amount of space in every household. The main reason of the big space a wardrobe takes is because people have to open it in order to select their clothes. We have designed a system that will no longer require that. Our system will eliminate the large space wardrobes take and give everyone a little bit more livable space.

What it does

Our system has a screen on top a wardrobe. In this system, a facial recognition tool runs to recognize the person. Then, the clothing inventory of that person is taken for the person to see. Our system can recommend clothing (based on the occasion that you are dressing for and the current weather). You can also select your own clothes through a select option. Our system also supports multiple people since we understand that there are usually more than one people living in a household.

How we built it

We have used Python to build a face recognition (microsoft face api,opencv) , get live weather data (open-weather-api) based on geolocation, clothing/apparel recognition (opencv, clarifai), a flask server for the front end of the application.

We have created a web-app using HTML,CSS and Javascript so that we can have a front end for our application.

We have used mongoDB stitch for the backend and database of the application.
