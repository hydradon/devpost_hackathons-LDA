Inspiration

We built this web app because a lot of us have formed friendships because of similar music tastes, and wish to discover more people who have similar music tastes to us.

What it does

This web app uses your Spotify statistics in order to analyze your affinity to other people using Spotify based on music taste. It analyzes the audio analytics that the Spotify web API provides and intersections in top tracks listened to in order to show your compatibility with those on your followers list.

How we built it

We used the Spotify web API, Javascript, and HTML.

Challenges we ran into

None of us have used javascript before, so this was an interesting project to help learn how to use APIs and work with different functions.

Accomplishments that we're proud of

Being able to visualize how the Spotify API connected to our web app and understand different javascript functionalities.

What we learned

We learned how to access and use the Spotify web API and display it using HTML and CSS.

What's next for Consonüance

We want to help pair a user with their followers and provide them recommendations.
