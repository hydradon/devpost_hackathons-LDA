Inspiration

We want fast food to actually be fast. (and lightly influenced by the asian tendence to centralise everything in one app => 'app life' concept)

What it does

Web app which centralises food delivery companies on one platform powered by Algolia.

How we built it

We used Casper JS to scrape companies' websites data, displayed with angularjs and Algolia for super cool search.

Challenges we ran into

Started off using Chrome Headless for scraping and it didn't work too well.

Accomplishments that I'm proud of

-We made something that works (yaaaaas!) -I made new friends (yaaaaaaaaaaaaas * 2!)

What we learned

Human beings don't need that much sleep... And one or two things about web development and scraping

What's next for FoodFinder

A beer, some sleep aaaaaaaaaaaand ... some fast food (using FoodFinder obviously) We're not sure if it's completely legal... so watch this space to find out!
