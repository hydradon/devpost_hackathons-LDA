Inspiration

I like to see an overview of a repository (especially what's been going on recently) on one page in Bitbucket instead of navigating around to different pages within the application.

What it does

Shows a list of recent branch activity and pull request activity all on the repository overview page in Bitbucket.

How I built it

With Javascript and the the great Bitbucket REST API documentation.

Challenges I ran into

Restricting the scope of the project and not trying to do it all "the right way" and over-engineer it. Processing the REST API data into a useful format.

Accomplishments that I'm proud of

Getting it built!

What's next for Recent Activity for Bitbucket

Improved UI and better error handling. We'll see what feature requests I get as well :)
