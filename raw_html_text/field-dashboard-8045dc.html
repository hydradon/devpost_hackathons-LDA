What it does

It allows a user to see how much money their fields can generate depending on the harvest from the Climate Corporation and gives a metrics dashboard to analyze crop prices from the wall street journal.

How I built it

Used Climate Corporation's API to retrieve field data and used Quandl's API to retrieve commodity prices. Used google charts to display statistics
