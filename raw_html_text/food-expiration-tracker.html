Inspiration We care about the environment, and food waste harms it.

What it does The tracker allows you to input food from your fridge and counts down the days until it expires. The website portion of the project provides information about food waste and expiration dates.

How I built it Tracker built with python. Website built with html and css.

Challenges I ran into Both of us are still new to python so it was hard to build the tracker.

Accomplishments that I'm proud of Utilized my resources (google!)

What I learned I learned how to code with python and how useful it could be.

What's next for food expiration tracker Adding graphics to python so we don't need a separate html website.
