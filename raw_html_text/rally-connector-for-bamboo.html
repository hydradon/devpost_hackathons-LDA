Inspiration

Makes users able to measure the Bamboo build health using Rally.

What it does

Enable teams that use Rally to see the build status for the projects they are working on.

How I built it

Using the Atlassian SDK and Rally Web Services API.

What's next for Rally Connector for Bamboo

Makes users able to track which Rally Artifacts were delivered into a specific Bamboo build through the Rally Build Traceability dashboard.
