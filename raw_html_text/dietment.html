Inspiration

When we found that there are a lot of people who don't like to take medicine when they get sick, we think whether there might some other way to help these people. In traditional Chinese medicine, food is also a kind of medicine. There are some natural chemistries which can cure people's illness. Not only vitamin, but also some other chemistry can make people feel better. We believe that some food can replace medicine in some ways when people get ill. We won't try to say every pill can find its perfect substitutes. However, foods can help most of normal sickness. We believe that if you don't like pills, some fruit, vegetables, can help you to go through uncomfortable time.

What it does

In the Dietment, users can take a photo or search in the search bar to what medicine he or she took. We recognize the picture though machine learning. In the function pages, Dietment will tell you medicine ingredient, a specific description, authority sources and what food you can take to make you feel better.

How we built it

We use following techiques to build this project: Google Cloud Platform, Microsoft Azure, Mysql, asp.net, c#, css3, html, javascript. On the cloud platform, we use Azure Custom Vision Service to recognize the picture, and run the WebApp on to to provide the user service. On Google Cloud Platform, we use the SQL database and Virtual Machine to run our searching backend. We tried to use Storage or Blob Storage to store our picture, but we did not completely finish this part.
