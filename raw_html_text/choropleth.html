Choropleth

Create chorolpeth maps (also known as thematic maps) in Excel.

  • Supports any number of data points
  • Supports both numeric and non-numeric data
  • For numeric data, higher values are shown in darker colors and lower values are shown in lighter colors

Uses the following APIs:

  • Bing Maps API
  • Spatial Data Services Module
