Inspiration

Games like Candybox and Zork.

What it does

Run based on user-inputted text.

How we built it

We put all the code into Cloud9 and then used Github to help publish it as a .tech domain.

Challenges we ran into

Getting it to run as a .tech domain, formatting the ASCII art, formatting the webpage, syntax errors.

Accomplishments that we're proud of

Getting the ASCII art to look nice, getting the webpage to look nicely formatted, figuring out how to solve errors using revision history or by talking together.

What we learned

How to use Github and Cloud9. How to code in Javascript, HTML, and CSS. How to setup a domain. How to work together as a team to solve problems.

What's next for Dungeon Diver

Having the character select actually affect the story. Having multiple pages instead of just one main homepage. More ASCII art. Having multiple dungeons to choose from and maybe a deeper storyline. Centering the "start" button.
