Inspiration

Lean Agile Scrum changed my life, and I have been dreaming of ways to bring these concepts to more people through social media.

What it does

This extension allows streamers to create projects around the content they stream. By adapting the Kanban board and borrow some Agile Scrum concepts streamers and viewers can communicate about whats up next, whats going on now, and what was recently accomplished. Streamers create projects that is little more than a "todo list", with a markdown description and a checklist.

How we built it

JS/HTML, AWS JS SDK, AWS Cognito, AWS DynamoDB
