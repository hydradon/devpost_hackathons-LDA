Inspiration

We thought of this idea while talking to a friend of ours who lives in another country and finds it difficult to identify objects in other places.

What it does

You can use our app to learn new languages from your surroundings. Not sure, how to say, "Traffic Lights" or "Butterfly" or just about any physical object, that you may not know in one language, now you can know it in multiple languages.

How we built it

Processes an image using the google visualization API which is then translated using the Google Translate API. All done using STD Lib as back end infrastructure and React Native as front end.

Challenges we ran into

As many of us were beginners in React Native, this was definitely a challenge.

What's next for BluePrint

Implementation as an online game with goals in larger areas.
