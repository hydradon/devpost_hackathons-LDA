kappapastabot

A Twitch chat bot to send popular copy pastas when prompted with commands. Built at SD Hacks 2016.

How to Use

  • Type [!copypasta] in Twitch chat without the brackets to have a bot output a random copypasta.
  • Type [!copypasta streamername] in Twitch chat to get a copypasta specific to a certain streamer.
  • Type [!copypasta help] in Twitch chat to get a list of streamers that are supported.
