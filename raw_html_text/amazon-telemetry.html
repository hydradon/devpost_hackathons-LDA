• Amazon Telemetry Workflow

What it does

"Amazon" Telemetry is a framework that assigns a phone number to any Amazon Alexa skill (Both pre-existing and new), effectively enabling companies to quickly and easily create customer service chatbots using the Alexa Skill Kit. In addition to general Alexa responses, Amazon Telemetry can also forward calls based on issue priority.

For instance, if you call the Citi example chat bot @ 888-659-0585 you can ask the following:

"I want to change my PIN", which is a low priority response and the response will be answered directly.

Whereas:

"I want to cancel my account." is a high priority response, so the call is automatically forwarded to a customer service agent.

Companies can easily assign issue priority through keywords in the Alexa Skill Kit
