Inspiration

As teleprompters haven't gotten better or expanded over the years, we decided to expand their functionality by making an Android app.

What it does

Smart teleprompter that also analyzes tone in a speech. Users can input their speech in a text file.

How we built it

Java, Android, and IBM Watson's Tone Analyzer API

Challenges we ran into

How to use IBM Watson's Tone Analyzer API, how to dynamically print text to the screen, and how to take input from a file in Android

Accomplishments that we're proud of

Learning the IBM Watson APIs

What we learned

How to use the IBM Watson Tone Analyzer API and how to automatically scroll through text in an Android app.

What's next for Prompt Talker

Expanding functionality to help users prepare for their speech and adding support for more file types.
