Inspiration

Want to go somewhere where the weather is cold? Or hot? Or just okay? And you don't know where?

What it does

This is a website and an iOS application. The project takes in the user input of the month and the range of temperatures they would like to go on vacation during. Then according to the user's temperature preference it presents a list of some popular cities that the user might enjoy visiting.

How we built it

This was built with several APIs and using Xcode for the IOS app. Javascript, HTML, and CSS was used for the development of the website.

Challenges we ran into

It was our first time building an iOS app and it was a steep learning curve to understand swift and Xcode. The free weather API we were using hit the limited amount of calls, so we had to look for a different API and be mindful of the number of calls the app makes.

What we learned

We learned that the process of building our website and mobile app could be more efficient with better planning and design.
