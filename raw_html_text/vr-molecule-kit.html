Inspiration

One of our members is a biochemistry major, and he thought of what it would be like to manipulate molecular structures in VR.

What it does

This project allows users to spawn atoms and bonds in VR to build molecules. They can also enter a path to a JSON file that was downloaded from PubChem.

How we built it

We built it using Unity. We utilized the Oculus Integration package to get the Oculus headset to work with the simulation. We also used the PubChem database to load information about molecules.

What we learned

We learned how to integrate Oculus into Unity!
