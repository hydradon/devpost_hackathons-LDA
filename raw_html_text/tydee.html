Inspiration

We wanted to see how we could help the world. We saw a lot of praise for trashtag, and thought we could get more people to join this effort.

What it does

A react native app that collects pictures and information about trash near a user (as advised by the user), evaluates the pictures to find a score for users.

How we built it

We split into teams of two. Both focusing on separate functionalities of the app. Two on front end and two on backend. We then merged our changes at the end.

Challenges we ran into

TensorFlow turned out to be more challenging that we thought. Our lack of expertise in TF limited us from truly executing our vision for this hack.

Accomplishments that we're proud of

We were able to implement a functioning application in React Native. This is quite an accomplishment when considering the fact that our team was new to React Native.

What we learned

We learned how to use EXPO, React Native, and a few libraries for taking pictures, finding locations etc.

What's next for Tydee

Implementation of a backend that leverages machine learning.
