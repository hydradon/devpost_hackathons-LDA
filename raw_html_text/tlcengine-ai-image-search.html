Inspiration - Searching for properties takes took long, Having gone through lots of photos. Also, there needs to be a way to Look at what Empty spaces look like with real furniture.

What it does - It tags all the photos in the RLS system and allows you to sort by Phototype. If the place is vacant we will use AR to put real furniture into the photo.

How we built it - We used Restb.ai API to process images and Tag them. We build a React Front end to sort the RLS feed. Then used API from Box Brownie to do teh AR

Challenges we ran into - Bad photos with poor lighting

Accomplishments that we're proud of - A working prototype

What we learned - There are a lot of great API out there that can be put together in a new way to help in the Real Estate Search.

What's next for TLCengine - AI Image Search - To Commercialize the product to sell to Agents.
