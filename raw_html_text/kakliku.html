Inspiration

What it does

Our program allows user to take a picture of , what is supposed to be an image text, and transforms it into an editable and searchable text. It currently recognizes over than 600 languages, but in this version, we are only using 2 - English and Georgian.

How we built it

We built it using an enormous and difficult library called tesseract along with leptonica within an android studio.

Challenges we ran into

This library had no support for Georgian language and we had to implement it into it ourselves. This part took the most time of our Hackathon.

Accomplishments that we're proud of

IT WORKS! (with over 95 % accuracy of course)

What we learned

We learned a lot about android and ndk, which enabled us to use c++ code. We also learned that there is a library which needs 30 minutes to compile...

What's next for Kakliku

Kakliku won't be forgotten, There lies a great path of development in front of it. More functionalities, more languages, real time support.. etc.
