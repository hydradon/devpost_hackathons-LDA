Inspiration

We only have one planet and we want to preserve our natural resources. We also wanted to help us understand the global water crisis.

What it does

The sensor reads the water's Ph and the surrounding soils water content and automatically uploads them over wifi to a database which is displayed on a front end.

How we built it

We used React-native, mysql, google cloud and ESP32 with arduino framework.

Challenges we ran into

We initially intended to have a raindrop sensor which would measure the frequency of rainfall but the sensor was shorted. Server hardware connectivity and sending the data to the database was challenging. Voltage levels adjusting for the different sensors and the MCU
