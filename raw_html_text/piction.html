What it does

The user submits a picture to the Piction server. This server then analyzes the picture and generates the suggested hashtags to be used.

How we built it

Piction was built with React Native for the client side, Azure as the main API for the back-end which is where all the data and ML algorithms were hosted.

Challenges we ran into

There were a lot of difficulties while setting up the Machine Learning model as well as connecting the mobile app with the server to show useful information to the user.

What I learned

It was the first time that many of us used React-Native. This was quite challenging at first but we eventually got used to it and now understand this framework a lot more.

What's next for Piction

Piction aims to not only suggest hashtags, but also suggest Captions for social media as well as Filters to be used on the picture.
