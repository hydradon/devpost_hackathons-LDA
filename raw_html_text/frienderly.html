Inspiration

There are lots of elderly living alone and no one cares about their daily life.

What it does

Frienderly can help the elderly to improve their daily life and monitoring their health status so problems can be figured out quickly.

How we built it

We build it using azure cloud platform and cognitive services.

Challenges we ran into

We don't have sufficient health data to train the machine learning model in order to make a precise decision.
