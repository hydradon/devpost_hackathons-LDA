Inspiration

What it does

Rocket Roller is a game where you are a ball that has different abilities and you have to get to the end of a cources. Along the way, you will run into a maze, enemies and difficult jumps

How we built it

We built Rocket Roller in unity 3d

Challenges I ran into

We ran into challenges with the camera not wanting to rotate or stick with the ball

Accomplishments that I'm proud of

We are proud of the fact that we actually got the game into a working state that is somewhat enjoyable to play

What I learned

We learned how to make a working game in unity. We also learned that even the simplest game can be enjoyable

What's next for Rocket Roller

Rocket Roller will probably sit in the unity cloud so we can download it and show it off every once and awhile
