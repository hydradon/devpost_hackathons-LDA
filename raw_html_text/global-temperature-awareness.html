Inspiration

Natural disasters are on the rise and are becoming more frequent around the world. In particular the number of citizens dying from heat strokes is rising in the US and the rest of the world. As well as heat strokes, hurricanes and storms are becoming more prevalent and deadlier than ever. We are inspired to create a tool than can help people make long term decision based on the risk associated with living in regions that are expected to become increasing difficult to inhabit as a result of global temperate rise.

What it does

The purpose of this web application is to spread the awareness and the increase of temperatures in regions around the world. Global warming is a real problem that needs to be prevented and the only ones that can make the changes are the people around the world. The predicted time(years) to live at an inhabitable temperature is purposely done to send a shocking message and show global warming is real.

How we built it

We built with JavaScript on React framework. We used Google Maps API and Meteostat API (Historical weather data).

Challenges we ran into

The meteostat API was unstable at the end of our day 2 and had to rely on waiting on it's API calls for hour limit.

Accomplishments that we're proud of

We are proud of our ability to continue seeking new knowledge and challenges in all areas related to programming and web design.

What we learned

We learned some API providers are unreliable and issue arise that are unforeseeable could occur in times least expected.

What's next for Global Temperature Awareness

The next step is projecting the awareness of global warming and encourage preventative actions to stop global warming, e.g. decreasing pollution.
