Inspiration -

This game was a favourite of mine as a child and I realised that there was no skill currently like it on the Alexa skill store.

What it does -

This is a simple game where the objective is to find the odd one out. You will be given 10 questions each with three items, you have to decide which one is the odd one out.

How I built it

Challenges I ran into -

When trying to test it I ran into a problem of not being able to invoke the skill when the langue was set to English(US), I fixed this by also setting the language to English(UK)

What's next for Odd One Out Game -

There are many more categories on the way for the game along with extra items in the existing ones, and possibly the addition of picture round for the echo show when I get a chance to play around with one.
