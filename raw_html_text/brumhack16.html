Git-Pizza

Lookup pizzas from Pizza Hut and Dominos based on your postcode to find you the best prices

The python server scrapes the two websites to obtain data on the pizzas available from your local store, using Python's requests to keep track of the cookies from the websites. A JSON object is returned containing the pizzas, which can be filtered, sorted and searched from the website.

Ideally, I would have included deals available too, but Domino's website required me to restart my server after every request to their deals details which made it more complex to do, and Pizza Hut used images to show details, which would have required computer vision to solve this, and after spending over 14 hours on the web scraping, I had little time to do this.
