What it does

Allows users to take pictures of bunnies, upload them, and get points based on how similar the bunny looks to a real bunny (According to Values returned by Watson). Firebase was intended to be used to record past bunny sightings and continuously update clients with new sightings.

How we built it

We tried to use Android Studio with Watson Image Recognition API and Firebase.

Challenges we ran into

Watson didn't go well with Android Studio because of ASynchronous network errors. Feelsbadman.

Accomplishments that we're proud of

We got access to the camera at least

What we learned

Don't procrastinate and wait till the last minute to implement core features
