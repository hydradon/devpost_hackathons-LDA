Inspiration

My dad has a shellfish allergy. He ate a dish once, that happened to have some very well-disguised crab. Riding with him to the hospital was the scariest thing that ever happened to me.

What it does

DietAR is a one-step app that displays nutritional and allergen information for prepackaged foods and restaurant dishes.

How we built it

For the prepackaged food, we scanned the universal product code and pulled nutritional and allergen information from a variety of databases. For the restaurant dishes, we used Clarifi's api to scan a dish and list its ingredients.

Challenges we ran into

Using AR in react native, getting clarifi's api to work, uniting the databases, performance

Accomplishments that we're proud of

our final product- a working, streamlined, user-friendly app that displays allergen and nutritional information.

What we learned

A lot about allergens, how to use, and how not to use, clarifi's api, using ar in a non-intrusive manner

What's next for DietAR

increase our database and get more nutritional information, and polish
