Inspiration

Virtual reality is very close to being a reality, and it is a very interesting subject to play around with. That and everybody loves zombie games.

What it does

Uses a myo to control the main character. Each arm and each leg are controlled separately. The Oculus Rift was intended to be used to provide an immersive experience, but there were some technical difficulties.

How we built it

Three myos, the Unity game engine, and the *Oculus Rift. The Unity game engine provided the basic game functionality, which was then extended to take input from the myos.

Challenges we ran into

The Oculus Rift could be used to view the environment but only as a second monitor and not as a an immersive 3D viewer. The Unity game engine was very buggy, and didn't play well with our computers.

Accomplishments that we're proud of

The animation of the hand being closed.

What we learned

The Oculus Rift is not very compatible with lower budget computers as it needs a more powerful graphics card to run. It also isn't compatible with certain older versions of Unity.

What's next for Hunted

Magic. Also the Oculus Rift being successfully integrated. Better UI. Better AI. More expansive environment.
