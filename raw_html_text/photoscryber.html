Inspiration

My dog was diagnosed with dog cancer in Russia. There were thousands of dog cancer doctors, but because I could not read the Russian signs, I could not bring my dog to a doctor in time :(

What it does

Detects text in an image and translates it to English

How we built it

Chrome extension posts image url to a server, which gets the image data and translation using Google's Cloud API. This data is sent back to the extension, which displays the translation in a popup
