Inspiration

People nowadays struggle to find a common time that they can gather together. We'd like an app that integrates with the social networks and let people find a time that they are all free and can share a moment of joy. We integrate several functions including real time schedule viewing and editing, money collecting (paying AA), and eatery selection.

What it does

The app lets you create an online scheduler session that other users can edit together. It supports real time viewing and editing of the schedule across multiple devices, and also event customization and money collection. It makes people's life a lot easier. Our app also integrates closely with WeChat, the most popular chat app in China.

How we built it

We used a minimum amount of open source APIs, (excluding WeChat, Parse), and build some great custom UI elements and animations.

Challenges we ran into

Creating UI elements and animations using only code is always a hard time to go through.

Accomplishments that we're proud of

When the creator of the event receive an request from one user to join the event session, the app on that user will automatically detects when the request is accepted and pop out the scheduler editing session.

What we learned

A lot of iOS APIs and Swift coding techniques.

What's next for Project Tic Toc

Add details to the current session to display the event name, descriptions and time labels. Add support for more social apps and payment methods.
