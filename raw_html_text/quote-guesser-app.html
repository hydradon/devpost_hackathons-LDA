Inspiration

Wanted to make a short quick game that actually tests knowledge and is not based on a single skill based game mechanic.

What it does

Tests users knowledge about historical figures and leaves them with extra knowledge of the wider world.

How we built it

We built this app using Java in Android Studio.

Challenges we ran into

We faced many challenges along the way, these included not being able to do an if statement and not having buttons do anything.

Accomplishments that we're proud of

We are proud we finished the app in time, and in its current state. Despite not knowing any java.

What we learned

We learnt to program android apps using java with Android Studio.

What's next for Quote Guesser App

More rounds of quotes, with increased levels of difficulty. The GUI will be improved.
