Inspiration

Cake Mania and similar time management games.

What it does

You can go to the oven and choose a blade you want to craft. Unfortunately many of this game's jaw-dropping features such as HTC Vive support, a 90-hour campaign, and any sort of progression had to be delayed.

How we built it

We built a map with some graphics tiles and added sprites sourced from the internet. Then we added more functionality with Phaser and our previous JavaScript experience.

Challenges we ran into

We had to reformat most of the spritesheets, and dealing with graphics generally took a lot of our time.

Accomplishments that we're proud of

The ability to use a framework to build a project.

What we learned

While Phaser is great for one-off minigames, it is probably not a commercially viable engine.

What's next for Gear Shop

Making a fully-functioning level
