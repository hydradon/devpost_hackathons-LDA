Inspiration

If you're tired of eating the same leftovers, why not switch with a friend?

What it does

It provides a way for people to browse, meet up, and exchange food in their local area.

How we built it

Using Flutter, Dart, Android Studio, and Firebase.

Challenges we ran into

The firebase packages for flutter were outdated, so we had to use the REST API to access our database.

Accomplishments that we're proud of

Learning how to use Android Studio with flutter and making a fully functional application that is able to dynamically load information.

What we learned

How to make a dynamic and responsive app with flutter and Android Studio.

What's next for Bread and Barter

User authentication to log in to the app, and a way to like food posts and let others know you'll be attending a meetup.
