Inspiration

Being a college student

What it does

The end goal is to track homework and exams by crowd-sourcing from other students in the same class. Users can be notified of upcoming deadlines or exam dates for that class. So far we managed to setup a basic interface where the user can add a class based on a dummy list of classes, and add homework and exams for that class. Currently, the data is only stored locally on the app.

How we built it

In swift using Xcode 7.

Challenges we ran into

Attempted to make a backend server but having little knowledge to accomplish. Getting familiar with swift and Xcode.

Accomplishments that we're proud of

Integrating several different View Controllers. Being able to develop a functional app after all the app crashes that we ran into

What we learned

Different parts of making an iOS app (front end, back end) The complications of making a server The importance of communication between the front end and the back end

What's next for Procrastinot

Improving design by deciding on a color scheme and adding a splash screen. Implementing notifications for homework and exam. Adding user access to implement crowd-sourcing. Building a backend server that can store data of each class and notify students in the same class of any updates.
