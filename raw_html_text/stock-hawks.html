What it does

An app that tells a user the best stock to buy based off how much money they are willing to spend.

How we built it

We took stock quotes from Yahoo Finance API and parsed it through an algorithm which rated the stocks from a point scale of 1-100. From 1 as the lowest rating and 100 as the high rating. The three highest rated stock were recommended to the user.

Challenges we ran into

-API issues -We could not reliably use the API data with JavaScript. -We had problems dealing with programming the different languages we encountered.

Accomplishments that we're proud of

We finished the product!

What we learned

-Every language has it's own purpose

What's next for Stock Hawks

-Create a more accurate algorithm -Figure out how to integrate in real time the data from Yahoo Finance API -improve UI
