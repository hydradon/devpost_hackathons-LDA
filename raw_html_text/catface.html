Inspiration

Will to produce something challenging but entertaining with low user complexity.

What it does

Transforms uploaded picture into vast mosaics of cats

How we built it

The large number of cat pictures needed were downloaded/scraped from various websites/APIs using matlab toolboxes for image processing. The website and mosaic creation software were built in node.js and html.

Challenges we ran into

Finding enough pictures and processing the images fast enough.

Accomplishments that we're proud of

Producing a usable system within the time frame, learning many new skills in programming and creating a simple and clean user interface.

What we learned

Many new programming skills, and time/project management.

What's next for CatFace

DogFace, FishFace, PastaFace......(Vary scale-able business)
