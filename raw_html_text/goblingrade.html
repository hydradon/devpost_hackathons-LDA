Inspiration

A love and passion for video games and their communities

What it does

Displays user information such as games owned and hours spent. It then calculates calculates a user score based on games owned, achievements earned, and hours spent.

How we built it

This site is built off Python working with JSON. It receives user info from Steam API, then calculates the retrieved data.

Challenges we ran into

Mixing Steam API with Python lead to complications and performance issues.

Accomplishments that we're proud of

Overcoming all of our problems.

What's next for GoblinGrade

We hope to develop it further such as adding a leaderboard and integrating it with Google.
