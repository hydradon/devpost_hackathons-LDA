• The mighty controller

Inspiration

Built off the succsess of doom controller, ARC MC is a significantly more advanced controller for Minecraft.

What it does

Controls Minecraft using a motion controller and dance mat.

How we built it

Node MCU sending accelerometer data to a neural network that classifies each movement into a pickaxe swing, shovel dig, sword hit and axe chop. This data then interfaces with a custom Minecraft mod that preforms the appropriate actions.

Challenges we ran into

We originally tried to send the data from the accelerometer to the PC via TCP but this proved unsuccessful and so we instead opted for UDP which we managed to get working.
