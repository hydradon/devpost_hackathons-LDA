Inspiration

After listening to the hardships of the refugees and how difficult it is to keep track of their data due to their circumstance, we wanted to create create a software that solves this problem and allows them to get better health care.

What it does

This software keeps a universal fingerprint of the refugees to keep track of their health care data.

How we built it

We used psql to keep track of the data of the refugees. Following that we used python for building the front end as well as send and retrieve data from psql.

Challenges we ran into

Lacked knowledge of how to start the project.

Accomplishments that we're proud of

Even though we lacked knowledge, we were able to learn on the way and finish the product.

What we learned

We learned more about python and psql as well as how to build a full stack program.

What's next for Refugee tracker

If this idea picks up, we would like to create a startup to assist better healthcare treatment of refugees.
