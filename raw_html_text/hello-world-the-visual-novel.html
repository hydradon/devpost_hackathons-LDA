How we built it

We used Ren'Py, a visual novel engine that uses Python at it's core with several built in tools to make the process easier.

Challenges we ran into

We tried to build the game with java and JavaFX at the start, but could not get it working in time. To solve this, we used Ren'Py to build the game. Also, the .zip file of the game was too large to directly upload to github, so we had to move it to our repository and commit it to get it on github.

Accomplishments that we're proud of

Creating a branching narrative revolving around fun characters with unique traits using an unfamiliar toolkit.

What we learned

How to create variables, methods and eventually entire programs in Ren'Py.

What's next for Hello World: The Visual Novel

Additional characters with longer routes, to flesh out both the personalities and the world the game is set in.
