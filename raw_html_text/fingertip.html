Inspiration

Given the limited time, we thought it would be fun to develop a little game.

What it does

It's a interesting party game. Basically it's given a keyword, one person draw and others guess what the keyword is based on his/her drawing. The most fun part of the game is the drawings of the players. Due to limited time and drawing space, most of the time the drawing can be quite funny as well as illustrative

How I built it

We build it using Swift and Socket.io, a websocket library for iOS

Challenges I ran into

Bugs, it's really hard to finish the bug-free game in one night.

Accomplishments that I'm proud of

We finished it in the end

What I learned

Socket.io and websocket, which can be used for real time communication. In our game it is the drawing will show on every player's phone simultaneously

What's next for Fingertip

We will continue to add more functionality to the game to make it more complete, and then published.
