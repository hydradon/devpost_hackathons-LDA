Inspiration.

Substance abuse and drug overdose fatalities are some of the major problems plaguing the country.

What it does

Provides information about substance abuse, treatment methods, and informers the user on where to get help.

How I built it

Our project was created with Python programming.

Challenges I ran into

The 10 hours was not sufficient time to include all the feature we originally planned to incorporate. If had more than 12 hours, then we could make our project a lot more sophisticated. Because we used embedded while loops, debugging became also complicated.

Accomplishments that I'm proud of

We were able to include open data and provide users with valuable information about a topic that has been unaddressed and overlooked in our society.

What I learned

We learned about the various preventive measures for substance abuse, some navigating URL techniques in the Python programming language, and much more!

What's next for Capping Substance Abuse

Our next step is to convert our program into a database software that includes more information about substance abuse and better help affected users find adequate treatment.
