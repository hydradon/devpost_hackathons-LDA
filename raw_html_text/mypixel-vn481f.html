Inspiration

Filters are widely used to make something appear as perfect and numbers are used as a measurement of beauty. These are the problems with social media today. Our website, MyPixel, offers the contrary.

What it does

MyPixel is a web platform connecting people using a simple, yet beautiful, retro design to let you experience what social media would have been back in the 80s.

How we built it

Utilizing Bootstrap and NES.css as front-end frameworks, PHP as backend, we were able to create a few of the most important features of social media, while also keeping it simple.

  • Log In Page
  • Sign Up Page
  • Posts Page
  • Profile Page
  • Chat Page

Challenges we ran into

  • Twilio API authentication
  • Connection between domain.com and web host

Accomplishments that we're proud of

  • Successfully connecting to the Twilio API for the first time and sending a welcome message to the users' telephone

What we learned

We learned to use APIs, Bootstrap and NES.css framework and database management.
