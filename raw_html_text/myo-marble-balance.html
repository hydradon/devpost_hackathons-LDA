Inspiration

An easter egg in a popular game called Spl-t

What it does

You use the Myo sensor and wave to move a tilting platform and try to balance a marble.

How I built it

We designed a 3d printed platform and used a Sevro motor to tilt it. Then wrote an Arduino program to turn the motor depending on what input you get. Then we built a Javascript program to communicate the Myo's input to the Arduino.

Challenges I ran into

Dealing with physical calibration issues with the Myo.

Accomplishments that I'm proud of

Having completed this project while leaning Javascript and Aurdino along the way.

What I learned

Aurdino and Javascript.

What's next for Myo Marble Balance

Possibly a bigger version with pneumatic tubes.
