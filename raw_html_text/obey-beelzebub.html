Inspiration

Creating a clean and elegant productivity web application with social aspects.

What it does

Provides an easy-to-use UI to create action plans for your personal goals.

How we built it

Spring Boot and Hibernate for relational database mapping in the backend, Thymeleaf for the template engine, and HTML, CSS, and Javascript for the frontend.

Challenges we ran into

The torture of programming on Windows.

Accomplishments that we're proud of

Full user log in/log out system, and discovering the tasks of other users on the site.

What we learned

You should always listen to Beelzebub. Also that creating a full stack web application requires constant amounts of planning and adapting as you may come across more unforseen challenges.

What's next for Obey Beelzebub

User profiles, messaging system, sharing system, follow system, email recovery system.
