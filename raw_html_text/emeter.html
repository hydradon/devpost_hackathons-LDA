Inspiration

Emotions in text messages can be misleading

What it does

iMessages extension takes input from the user, analyzes it and returns sentiment value

How we built it

We build the extension using Swift and an API from Microsoft cognitive services to analyze the text

Challenges we ran into

We had problems sending the JSON format to the cognitive services

Accomplishments that we're proud of

We all learn to use Swift in two days

What we learned

JSON conversion is hard to do in Swift

What's next for eMeter

We want to allow the user to analyze the text from messages received and and use it in other applications like Facebook Messenger, Gmail, or Dating apps
