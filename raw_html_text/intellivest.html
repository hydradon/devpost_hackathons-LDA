Inspiration

We have recently observed that modern definition of capitalism has become based on profit maximization, rather than its intent of a more humanistic approach. This can be observed in recent headlines such as, Amazon employees' work-life balance and Martin Schkerli's overpriced EpiPen. Our solution puts the power back into the hand of the investors and lets them communicate how they value seven social metrics.

What it does

Allows people access to social responsibility indices for various companies to allow them to invest and consume responsibly.

How we built it

We used Bootstrap and PHP for the front end and Python for the backend. Data was stored in a .csv file, and Pandas was used to process the data. We used iexfinance to retrieve stock performace data.

Challenges we ran into

The server was bad. We don't have an all encompassing data set.

Accomplishments that we're proud of

Website looks nice. We made a nice CSS gradient fill for the background

What we learned

How to use bootstrap to make a website look visually pleasing.

What's next for IntelliVest

Next steps include acquiring a larger data set on the a larger variety of companies. Performace enhacement: using a database instead of reading from a csv. More publicity the more people use it the more the economy shifts to a human-centered capitalism model.
