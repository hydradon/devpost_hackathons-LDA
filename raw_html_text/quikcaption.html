Inspiration

It takes a long time to type the specific hashtags for your image social media post, so why not make it simpler with suggestions?

What it does

QuikCaption takes the photo you select or take and then gets some suggested hashtags that you can select and then post to your social media.

How we built it

We used the amazing Clarifai api to generate the tags. For the iOS app we used Objective-C and for the Android app we used Java.

Challenges we ran into

When we were first creating this app, we found out that Instagram (our main idea for posting these tags) didn't let us push custom captions to it on Android. So instead it is copied to the user's clipboard and they can paste it in the caption area. On iOS the Clarifai api wrapper for Objective-C was somewhat difficult to manipulate the returned data. Instagram was also an issue on iOS just trying to get the image and caption sent via a hook.

Accomplishments that we're proud of

We made two native mobile applications in 24 hours. We also overcame the aforementioned challenges.

What we learned

The Clarifai api is really fast and insanely accurate. A little iOS/Android programming on each end that didn't know the other language.

What's next for QuikCaption

Publish the apps on the App Store and the Play Store.
