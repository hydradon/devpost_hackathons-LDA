Inspiration

In a busy campus like CMU, everyone's time is valuable. Therefore, we wanted to build an app to save people's time.

What it does

This app is designed for students to make requests asking for delivery and to help other people by delivering items for them.

How we built it

We used Django and materialized

Challenges we ran into

We had to learn multiple new languages and new applications that were beyond the scope of 112.

Accomplishments that we're proud of

We were able work with maps using API and working with a lot of new stuff that we learned in a day.

What we learned

Django, materialized, api, html

What's next for DeliveryHero

Hopefully, we can keep upgrading the app.
