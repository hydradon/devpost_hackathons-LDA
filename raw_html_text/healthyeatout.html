Inspiration

Eating too much of bad fast food.

What it does

Finds the food from nearby restaurants menu and tells you which foods are healthy for you at this time of the day and which are not.

How we built it

We built it using iOS and Firebase. Lots of analytics is provided by the app along with standard security as provided by Google.

Challenges we ran into

Time. We could have deployed it in the Appstore if we got 12 more hours.

Accomplishments that we're proud of

A fully functional and usable iOS app.

What we learned

Team work and Love.

What's next for HealthyEatOut

Restaurant advertisements + Better UI + Seamless addition of data from already existing fitness tracking apps/devices,
