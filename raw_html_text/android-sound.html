Inspiration

Trying to create an app to learn more about android studio while having some fun.

What it does

We have two apps, one records and outputs anything the use says in the mic. The second displays the use how loud their environment is in decibels.

How we built it

Android Studio

Challenges we ran into

Getting the microphone to respond properly and creating the right equation to get decibels since the mic simply outputs random numbers.

Accomplishments that we're proud of

Finishing and finalizing two apps in a short amount of time.

What we learned

How android studio works with using native hardware on the phone

What's next for Android Sound

Creating a more friendly user interface and adding a graphical representation of the decibels.
