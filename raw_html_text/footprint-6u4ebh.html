Inspiration

Bolster a environmentally friendly lifestyle by tracking your carbon footprint during the day.

What it does

Automatically tracks activity data in the background, recognizes pattern of activity, and classifies it into different categories.

How we built it

Using Adobe Xd as prototyping tool we have built a mock-up app prototype. CoreMotion activity tracking using swift is also implemented.

Difficulties

Implementing UI and app functions that we designed in under 2 days has really been a challenge for us.

Accomplishments that we're proud of

Fast prototyping using Adobe Xd.

What's next for Footprint

Implementation of all the app functionalities, web backend, and more future updates.
