• Good Neighbor Nav

  • Report an Incident Form (Geolocation/upload image)

  • Incidents Map (Google Maps API with voting system in info window)

  • Reported Incidents with Description and Uploaded Image

  • Police Departments (Grabs data from LongBeach API and mapped into our application)

Inspiration

As many crimes and incidents that get reported, there is still a large gray area of situations and people that go unnoticed. Our app aims to promote a safer community by engaging the community itself to bring light those concerns no matter how big or small a concern may be.

What it does

Brings the community together by giving people the chance to report incidents or do good deeds by helping someone in need near them. Users will also be able to upvote or downvote an incident near them.

How we built it

With a team of developers using react, asp.net, sql

Accomplishments that we're proud of

We were able to create a google map that has the incidents added dynamically with all the info and voting capabilities within an info window. This was built with a team of 5 developers in 3 days working fullstack.
