Inspiration

I have a super messy notebook and it's literally impossible to find formulas I wrote down from lecture, so we invented a way to fix that.

What it does

The Formula(s) for Success allows the user to enter, view, use, save and load formulas.

How we built it

Using Electron API and the electron-storage API

Challenges we ran into

Creating a coherent system that is able to handle many different equations. Finding a way to save formulas instead of losing them when the application is closed.

Accomplishments that we're proud of

Handling variables and string functions First Hackathon Project!

What we learned

GitHub, javascript, Electron API, HTML and CSS

What's next for The Formula(s) For Success

Take into consideration Units Solve 2 sided equations Cloud storage
