Bloch_Sphere_Three

Render a Bloch Sphere in a website with three.js.

From my experience from studying Quantum Computing, there are very poor visualization tools for representing super positions. The Bloch Sphere notation is a common notation for describing single bit super position states. As such, I have made an open source project for displaying this notation as a web element. I hope that by the use of this project, people can better understand these states.
