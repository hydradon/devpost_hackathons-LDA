Inspiration

Our inspiration for this project was to attempt to complete a hack that would actually be used to help people in some way.

What it does

This web app created is a multiplayer quizlet app that attempts to bring the competitiveness out of the users through challenging questions and elimination. Our app has a few different difficulties, Easy for those who want easier questions or those who want to solve some of the harder questions but not be timed or threatened with a knockout, Medium is our difficulty where we introduce the player to our timer and elimination feature, we try not to make it that hard on the player so less eliminations overall and generous timers. Hard difficulty is where the timer and elimination systems actually becomes obstacles for the players, timers low enough to question whether this game is fair, and an elimination system so generous with the boot you'd think it was a soccer player.

How we built it

We built this web app using html,css, and javascript on our front end, node js and firebase for back end and data storage

Challenges we ran into

Our original plan involved using react for our front end, but we realized a little too late that react was a tool that was way overkill for what we wanted to accomplish. So we scrapped the whole front end and switched to something much simpler and easier to work with.

Accomplishments that we're proud of

I feel like we accomplished alot and learned alot more. We actually got the landing page finished and have started implementing logic into it. We also made alot of progress with our back end logic.

What we learned

We learned that sometimes a framework or a library is just too much for a project and that keeping it simpler is much better and can lead to faster prototyping if you have a good grasp on it.

What's next for LeafueOfTEMS

Well if time permits, we hope to finish this project and host it on something like heroku or digital ocean so we may have people to play it, and see if the app can actually inspire individuals to learn more about STEM topics.
