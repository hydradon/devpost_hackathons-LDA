Inspiration

We felt the need to create an application that could provide us with the perfect playlist depending on the Temperature outside.

What it does

It identifies the location of the device, uses that and openweather api to identify the temperature, which it the uses to provide the perfect playlist for that temperature. It then opens that playlist on spotify and displays all the data collected.

How we built it

We used Android-Studio along with Java to build this application.

Challenges we ran into

Connecting Spotify and openweather Api's to the application, Adding the correct text outputs and staying up for the entire 24 hours to build it.

Accomplishments that we're proud of

That our application actually works without any errors. It displays the correct real time information.

What we learned

Using Api's, incorporating them in our code. Working for long duration's on massive projects.

What's next for TempTunes

We are planning to add real time traffic data and incorporate that into the algorithm by using neural networks and machine learning thus deciding which songs to play depending upon the situation. We will also permanently change its name from TempTunes as we add more and more functionalities of a Map.
