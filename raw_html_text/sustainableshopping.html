README

Jessica and I are passionate about combating climate change, but also love shopping and clothes.

We decided to make an application to help us and other environmentally conscious folks find items that are fair-trade, sustainably made and reusable. We also hope this application will encourage others to shop in a sustainable fashion.

This is a Ruby on Rails application using the NCR API. We filled a catalogue with a few products, with the intention that organizations will use the NCR API to have their products on our application.

We return products based on a search with merchandise categories that use keywords such as renewable , fair-trade and recyclable .

The search looks for the keyword in the shortDescription (few words describing the item) and returns the product with the shortDescription and longDescription .

This application can be run using ruby bin/rails server and going to localhost::3000`.
