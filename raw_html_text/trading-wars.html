Trading Wars is an educational web app that helps players learn how to optimize their trades in the stock market.

Inspiration

We wanted to create an educational web application that helps teach people how stocks work by allowing them to compete against our stock prediction bot. We were inspired by the idea of using regression analysis to predict stock prices alongside exciting new learners about investment.

What it does

Allows players to compete against each other on a weekly basis (for prizes/ monetary wagers?). Winners will be judged by quality of investments and overall growth of investments.

How we built it

Ruby on Rails web server Blackrock API for portfolio information Google Firebase for database

Challenges we ran into

Adjusting existing neural network model to represent stocks
