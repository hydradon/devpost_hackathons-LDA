Inspiration

Before coming to St Andrews, we didn't have much information about what to expect - especially the different types of accommodation.

What it does

Displays reviews for the university accommodation - written today by real first year students. Also displays distances from the halls to different places in town

How we built it

We wrote it in Java, using the Google APIs for maps and calendar integration and Java Swing for GUI

Challenges we ran into

The GUI turned out to be too complex to get working in the end so the program was left as a command line application

Accomplishments that we're proud of

Gathering data using google forms Importing this data into the program Integrating google maps
