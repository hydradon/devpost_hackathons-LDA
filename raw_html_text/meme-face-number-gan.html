Meme-Face-Number-GAN

A Generative Adversarial Network that creates memes, faces, and numbers from random numbers. Here is our Logo:

An example of a meme:

All of us enjoy memes, faces, and numbers, so we decided to make a tensor flow implementation of Ian Goodfellow's Generative Adversarial Network to create images of memes, faces, and numbers, from an array of random numbers using a generator and discriminator neural network. We also used 3 giant databases to train the discriminator.

See the Jupyter Notebook for the code and explanation on GitHub.
