Inspiration

We are teens, and we have a lot going on. We're constantly stressed and are at a very real risk of burnout.
* Burnout is a state of emotional, physical, and mental exhaustion caused by excessive and prolonged stress. * Teens need to learn how to BreakFree from stress.

What it does

BreakFree allows the user to manage their time by allowing them to input work-related assignments and approximate how long they will take. A timer feature helps the user prioritize tasks. The user receives a notification every 30 minutes, urging them to take a break. They can delay the break if necessary, but delaying 3 times required the user to take a break. The user can choose to go through a box breathing exercise, which helps relieve tension. Another option is exercise, such as 2 reps of 10 lunges or 3 reps of 5 squats. A feature called "Dance Party" generates a song for the user to jam out to. The social butterfly option reminds the user to talk to friends and family. When the time the user specified to sleep by occurs, the user receives a sleep alert. Whew! A great day of stress-free work!

How we built it

We built a prototype in Balsamiq. We used xCode and Swift for the timer, and Thunkable for the take a break features.

Challenges we ran into

Both of us (there are 2 members in this team) are new to xCode and Swift, so it was challenging to learn how to use them. Thunkable was glitchy, which unfortunately took a chunk of our time.

Accomplishments that we're proud of

We created a professional, detailed design and prototype on Balsamiq, which is a software we never used before the hackathon. We learned about Swift and its features even though we were new to it.

What we learned

We learned how to make a complete, user friendly mockup with emphasis on usability. We learned the basics of Swift and experimented with xCode. We also learned how to manage our own time during the hackathon and make concise presentations. We learned the importance of time management and the need to focus on hgh priority tasks.

What's next for BreakFree

We hope to fully develop the app, and we have more features in mind to improve usability and functionality. While our current focus is on teens, we hope to expand to a greater audience.
