Inspiração

Nos inspiramos nos problemas de representatividade que ocorreram no IME durante o primeiro semestre para desenvolver um projeto que aumentasse a importância do centro acadêmico (CAMat).

Funcionalidade

Plataforma-web com leve integração ao Telegram que agiliza as relações entre os centros acadêmicos e os alunos.

Desenvolvimento

Utilizamos HTML , CSS e Bootstrap para desenvolver a página web e Python e a API do Telegram para desenvolver o bot do respectivo aplicativo.

Desafios

O desenvolvimento do bot e sua integração ao site mostrou-se, apesar de aparentemente simples de implementar, um difícil desafio. Além disso, por sermos calouros, tivemos que aprender muitas coisas on the fly .

Conquistas

Belo design da página e bot que funciona.

Conhecimentos adquiridos

Desenvolvimento front-end e expressões regulares, uso de bibliotecas do Python e comunicação software-web.

Próximos passos

Expansão das funções do bot para votações e outros e implementação de ferramenta que permite a adição de outros centros acadêmicos com facilidade. Implementação de contato direto com os membros do CA e os alunos.
