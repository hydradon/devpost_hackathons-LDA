Inspiration

Disorganized lifestyles

What it does

Uses google's api to sync your calendar and allows you to add events, see events and see upcoming events.

How we built it

Hosted index.html and server.js on a nodejs server.

Challenges we ran into

Originally tried to use Angular to run google's api but had a lot of trouble with setting it up. As angular is a language developed by google we assumed it would be easy to integrate their api but nobody at the booth could assist. So we pivoted to a basic webpage.

Accomplishments that we're proud of

We successfully used the google calendar api and configured a web app.

What we learned

We learned a lot about using google's api

What's next for Organizer

We are hoping to complete this webapp that displays events in your life. It will also know when your available and will help you schedule study times. We ideally would like to track study times as well to alert people if they are under allocating time for their tasks.
