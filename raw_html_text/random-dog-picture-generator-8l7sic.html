Inspiration:

We always watch dog videos and look at cute dog pictures on instagram.

What it does:

It generates a random picture of a dog when you click the button.

How we built it:

Using HTML and Javascript on Visual Studio Code

Challenges we ran into:

  • Putting our code on a host server: We got a domain: doggosparty.online, but couldn't find a free hosting site and get everything together in time.
  • Learning HTML and JavaScipt: Neither of us use HTML or JavaScript normally so we had to code the entire thing in unfamiliar languages.

Accomplishments that we're proud of:

It works! We're newbies to CS so we're proud to have coded something in an unfamiliar language even though it's probably a simple project for most. And the dogs are super cute.

What we learned:

  • Basic coding in HTML, CSS, JavaScipt
  • Familiarity with VS Code
  • How to connect a coded website to a domain (although we didn't get the chance to)

What's next for Random Dog Picture Generator:

  • More dog pictures (so there's a low chance of repeats)
  • Two buttons: one for doggos and another for cattos (cats)
