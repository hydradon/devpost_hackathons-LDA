Inspiration

After watching upperclassmen build miraculous websites, we decided it was time we taught ourselves the beginnings of web development at our first hackathon.

What it does

Users can play an I-Spy game with 4 levels of increasing difficulty. There is a progress bar at the top tracking the percentage of the game completed, along with an increasing amount of hidden links in each level that result in automatic failure!

How we built it

HTML & CSS

Challenges we ran into

-Hiding the links to the next levels within the images, and getting the exact coordinates to align with the picture. -Finding interesting and complex images to use for levels. -Staying awake! -Styling with CSS to create an aesthetic experience.

  • ...Web development was truly uncharted territory for us ;)

Accomplishments that we're proud of

-Creating our first multi-page website! -Being first-years and still demoing a project!

What we learned

The fundamentals of HTML & CSS More specifically, how style sheets have precedence over one another.

What's next for iSpy Citrus Hack

The implementation of JavaScript, more levels, and more fun!
