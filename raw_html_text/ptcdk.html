Inspiration

Inspired by the recent media layoffs, I decided to build a tool to help freelance writers pitch editors more effectively. The trouble with freelance is that when pitching to editors, it is hard to know what they want. By allowing journalists to anon post their experience with various editors, freelance journalists should have an easier time creating better pitches and writing more articles.

What it does

Allows people to post comments on editors from various outlets that they dealt with.

How I built it

JavaScript (Node.js) Express combined with Google Cloud SQL's PostgreSQL database.

Challenges I ran into

Google Cloud SQL is a pain to connect. Documentation for node postgres is bad.

Accomplishments that I'm proud of

More postgres work. Cloud database.
