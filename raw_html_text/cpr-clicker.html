Inspiration

Ketchapp Mobile games do really well in their market. What if we did the same thing but for education?

What it does

A simple yet addictive game that subconsciously teachs kids how to perform CPR!

How we built it

Unity! LMMS! Krita!

Challenges we ran into

Debugging timing Figuring out a way to make medical-related things entertaining

Accomplishments that we're proud of

Working game! The game concept is fun, yet teaches players CPR! The cool background music that we made! The cool art that we made!

What we learned

When creating a game there needs to be a focus on the balance between simplicity and fun and education.

Coordinate multiple tasks at once in parallel to finish on time.

What's next for CPR Hero

New games each highlighting another medical process
