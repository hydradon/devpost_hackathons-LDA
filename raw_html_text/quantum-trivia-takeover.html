Inspiration

Wanted to learn quantum computing basics, and make our first game.

What it does

You play rounds of trivia, and the winner of each round gets to place a colored block on a 4x4 grid. The placement can be normal, or be quantum; if it's quantum, the player gets to place on two spaces, but it will be 50% likely that either space is actually taken when measured at the end of the game.

How we built it

Angular + Firebase front end, Python + Flask + Qiskit library backend.

Challenges we ran into

Angular is still a little new, and using Qiskit meant learning how to use quantum registers.

Accomplishments that we're proud of

The game is visually appealing, works, and shows quantum properties!

What we learned

How to do data binding better in Angular, and how to use IBM's quantum simulator and live system.
