Inspiration

We were inspired by the ability to use scraping and Amazon Alexa to build something that will make the campus meal searching easier to anyone.

What it does

Our program informs the user about the campus cafeteria menus.

How we built it

We used python to scrape the data from the MSU cafeteria website. We then built a data set and incorporated it into the developer console for Amazon Alexa.

Challenges we ran into

We had difficulty formatting the data into our data set.

Accomplishments that we're proud of

Our largest accomplishment is that we all learned a great amount this weekend and we were able to make something.

What we learned

We learned how to scrape and how to use Node.js within the Amazon Alexa developer console

What's next for MSU Cafeteria Search

Next we will add more features like hours of operation, nutrition information, and meal search.
