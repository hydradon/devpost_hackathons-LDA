Inspiration

We wanted to use structure interface with a language none of us have never used before (python with flask integration). Thought it would be neat to use Google maps API( none of us have ever used an API before), somewhat of a learning experience overall.

What it does

Shows popular places around the globe and puts you in the street where you can pan, move, and observe your surroundings, allowing you to "walk" the earth

How we built it

Structure web development with flask python and html, Google maps API

Challenges we ran into

Developing an initial idea, learning the bit of python necessary to proceed. Lack of experience across the board also made startup difficult.

Accomplishments that we're proud of

Developing a working web-page with a language none of us have ever used before.

What we learned

Humbling experience, realized that there are a lot of people much more knowledgeable than us. Also learned that it isn't very difficult to implement an unfamiliar language or API even if it looks daunting at first.
