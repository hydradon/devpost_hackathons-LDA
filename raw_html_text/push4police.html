Inspiration

Each day, the news highlights several unfortunate stories regarding fatalities due to the fact that medical personnel could not help on time. We decided to attack this problem by creating a platform which allows bystanders to contact emergency services without wasting any time. We have been inspired to help thousands of people across the world!

What it does

This app contacts designated emergency services with the click of a button.

How we built it

Using Android Studio, P4P was carefuly crafted to provide an efficient, and user-friendly UI.

Challenges we ran into

Our primary struggle was implementing the Google Maps API and displaying the longitude and latitude.

Accomplishments that we're proud of

This was our very first time working with API's. We carefully chose the most appropatire and suitable API for our project. Ultimately, we managed to make full use of the API's capabilities.

What we learned

This was our first interaction with the Android Studio IDE. Our past work included strictly web development and web applications.

What's next for Push4Police

Next, we would like to develop this application for iOS. iOS is home to millions of users. Through the release of the app on a second platform, we can spread this novel solution to ensure future safety for everyone.
