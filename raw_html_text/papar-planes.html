Inspiration

I wanted to try to use apples new augmented reality framework, ARKit

What it does

It presents a set of (virtual rings) for which the user must design, construct, and fly a paper airplane through to pass

How I built it

The rings were set up in arkit and the tracking of the plane was (attempted) using opencv

Challenges I ran into

I really struggled using opencv as it was my first time using it

Accomplishments that I'm proud of

I continued working even after my first group and project didn’t work out

What I learned

ARKit and OpenCV

What's next for PapAR Planes
