Inspiration

We always need a way to manage invitations

What it does

You can create events, find locations with google maps

How I built it

I use LM to build the Models Process Files to create pages then i use html and javascript i call the endpoints to create, list and update

Challenges I ran into

Try to connect the LM endpoints with a single page app Try to add authentication

Accomplishments that I'm proud of

Build a responsive site with a lot of features that developers use like html css and javascript

What I learned

LM is a great tool to built sites with minimal efforts, if you don't want to create a site you can use the endpoints and call it in current webs applications. Table rules is amazing
