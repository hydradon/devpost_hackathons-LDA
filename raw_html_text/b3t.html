Inspiration

We wanted to practice our skills by doing a real challenge. We read all the challenges and the Swissquote one was the best option for us. We have been purchasing cryptocurrency and we enjoy this topic.

What it does

We create a bot that analyzes the price of BitCoin and Buy and Sell it in the market.

How we built it

We used the Swissquote API and we create a python code that estimates the right moment to buy and sell the BitCoins.

Challenges we ran into
