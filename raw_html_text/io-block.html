Inspiration

It's hard to manage rent manually between roommates, so why not automate it?

What it does

Our project uses Ethereum to record transactions and create contracts, which can be re-used for monthly recurring payments!

How I built it

We used React, Mongo, Express, Ethereum.js, and Solidity to build a quick product.

Challenges I ran into

Setting up a test blockchain and contracts was extremely tricky. We spent all of Friday (Blockchain) and Saturday (Contracts) building it.

Accomplishments that I'm proud of

We're proud of creating an outstanding front-end that makes blockchain transactions!

What I learned

We learned how to use GraphQL, build contracts and generate blockchains in a development environment! :D We also learned how to use Ethereum Gas, similar to bitcoin's transaction fees.

What's next for IO Block

Generate a production-ready product, and go to market.

If we generate consumer demand we'll build an LLC!
