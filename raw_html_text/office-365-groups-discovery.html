Inspiration

Had a need to gather information about office 365 group usage

What it does

Uses office graph to get office 365 groups and groups stats.

Enumerates all the groups, adds the groups stats and uses the teams PowerShell module to determine is a group is a team

How I built it

Using powershell and office graph

Challenges I ran into

The office graph report for office 365 teams does not contain the group I'd only the group name and first owner.

Accomplishments that I'm proud of

It works! And I use it every day

What I learned

How to automate office graph

What's next for Office 365 Groups Discovery

Use the data it collects to help office 365 group migrations between tenants.
