Inspiration

Many people tend to silence their alarms. Instead, this app will help those types of people get out of bed with a challenge.

What it does

The app will send a text message to the input number with a complex string. It will then continuously call this number until the user sends a text with the exact same complex string.

How we built it

We used the Twilio API through python along with Java and Android Studio.

Challenges we ran into

Managing the out and in-going calls along with setting an actual designated alarm was challenging.

Accomplishments that we're proud of

The code overall works on local and actually functions.

What we learned

Learning to use Android Studio and the Twilio API.

What's next for Wake Up Call

Minor fixes in phone call situations along with more app development
