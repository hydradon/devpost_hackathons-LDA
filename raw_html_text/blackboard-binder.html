Inspiration

Blackboard can be a struggle, so we wanted to make it easier to gather your notes from different classes.

What it does

Sign into your Blackboard account, and the app takes it from there! It will go through your classes on Blackboard and extract the notes your instructors have posted. All of the files are tied up neatly into a single zip file and downloaded onto your computer.

How we built it

The application was made using Flask in Python.

Challenges we ran into

Some of us were starting from scratch. For example, we had to learn absolute basics in order to keep building off of it to make a great final project!

What we learned

We each worked on something that we don't have intensive knowledge in. This included html, javascript, python, etc.

What's next for Blackboard Binder

It would be great to put the website online so that it doesn't have to be run locally.
