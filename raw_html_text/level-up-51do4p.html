Inspiration

We wanted to provide personalized course plans to women with different income levels and circumstances. Online resources are abundant but the prices and qualities vary a lot.

What it does

This is a platform, aiming at finding available online computer science books for users.

How we built it

We built this website by using JavaScript, CSS, and scraping data from online book stores.

Challenges we ran into

At first, we planned to use Api when collecting data, but we faced the challenges in 1) get access from those Api; 2) debuting the unexpected bugs.

Accomplishments that we're proud of

We are proud of our ideas in providing women chances to embrace more knowledge related to computer science; and we are proud of our devotion in working together in hackXX;

What we learned

1) We learnt how to build a web based on JavaScript; 2) We learnt how to use online Api to get data (and this is our goal to improve our website);

What's next for LEVEL UP!

Using Youtube Data Api from Google Cloud Platform to add more online courses playlists into our websites;
