Inspiration

We were inspired by our own lack of mental health so we wanted to create something that we could use and others could also use

What it does

It lets users track and log photos and add descriptive comments. Then an AI will analyze the photos and give data/statistic to help you understand your possible mental health during the weeks.

How I built it

Used C# to get the Google Vision API in Google Cloud Shell and built the app in Android Studio

Challenges I ran into

Google Cloud Services does not fully support Android, which meant we could not fully incorporate it into the app

Accomplishments that I'm proud of

Creating the app in Android Studio

What I learned

How to use Android Studio

What's next for How you Bean?

We hope to fully incorporate Google Vision API into our app and polish it so that people may use it to help themselves.
