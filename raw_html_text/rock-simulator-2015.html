Inspiration

I was inspired by the recent trend of simulator games, and decided to parody them, but with above-par graphics.

How it works

It runs in Unreal Engine 4

Challenges I ran into

I had to learn how to use the engine, and then deal with massive textures and models eating up my entire computer.

Accomplishments that I'm proud of

It looks beautiful, and I was able to use Unreal for the first time.

What I learned

I learned how to use Unreal Engine to create maps and use the node editor for surprisingly complex control structures

What's next for Rock Simulator 2015

I hope to possibly add more levels and environments to the game, and then progress with Unreal to make games that have a goal.
