Inspiration

We always wanted to know who would win in a fight between Donald Trump and Barack Obama.

What it does

Gathers data using Wolfram Alpha's api on each celebrity.

How we built it

Using the api and a proxy node server which must be running on the machine it is accessed from (still a work in progress)

Challenges we ran into

Bypassing the lack of a cross origin resource sharing policy that Wolfram Alpha have by using the node proxy.

Accomplishments that we're proud of

Barack Obama beats Donald Trump

What we learned

Barack Obama beats Donald Trump

What's next for Celebrity Smackdown

Add celebrity images and winner image (lack of time)
