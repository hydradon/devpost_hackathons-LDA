Inspiration

Wanted to try build a web app

What it does

Helps locals bond with nearby people with similar sport hobbies

How we built it

We started by exploring react and considered using it with firebase, we eventaully ran into enough problems where we decided to simply use the database React Studio provides until we can get figure out the logistics of the login screen and complete enough UI to worry about backend functionality. We eventually ran out of time with little time to explore the backend functions of our app.

Challenges we ran into

Having no one on our team who had experience with web developement (front end and back end) was a really big challege and we had to figure out a lot of the basics before we even began.

Accomplishments that we're proud of

Experimenting with the google map api

What we learned

We learned a lot about web developement and what exactly you need to build a successful app. Particularly the importance of backend code databases in web apps.

What's next for SportsMeet

Completion of the backend database, which would allow us to give the user more freedom in interacting with the site.
