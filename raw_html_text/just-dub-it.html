Inspiration

We have traveled overseas and found that translation services are not always an accurate representation of language.

What it does

'Just Dub It' compares translation quality between two leading online services, Google and Bing Translate. After entering in a phrase in your native language, Just Dub It will return side-by-side translations from the two platforms, allowing the user to determine the more accurate translation. It also demonstrates the internal accuracy of each translation service by translating a phrase to a "middle man" language before translating back into the original input language. By comparing user input and the return phrase, Just Dub It demonstrates the scale of reliability for each platform.

How we built it

We used a stack of Javascript, HTML, CSS, Bootstrap, PHP, Google Translate API, Microsoft Azure Translate API.

Challenges we ran into

Documentation for the new Azure API was somewhat confusing to implement and we were not able to authenticate the API token.

Accomplishments that we're proud of

We were able to create a website in a short span of time that utilized Google and Microsoft API's.

What we learned

We learned that both translation services differed syntactically, though largely without a loss in meaning. We also determined that both translation services were not particularly adept at recognizing and translating idioms specific to a language.

What's next for Just dub it.

We want to fix our Microsoft Azure API call and look into providing an idiomatic translation service.
