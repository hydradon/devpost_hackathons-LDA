Inspiration

A game I played as a kid called Overball.

What it does

It let's the user play as a ball that is controlled by the mouse. The user is then supposed to collect all the flowers and crystals to complete the level. Along the way, enemies will try to kill the user which the user must avoid and kill using powerups.

How I built it

The game is built and designed using Unity game engine.

Challenges I ran into

Challenges I ran into were the time it took to create the level as well as how to setup colliders between objects in unity. Another problem was sharing variables between scripts in Unity since Unity doesn't let you have global variables.

Accomplishments that I'm proud of

Accomplishments I'm proud of are creating a fully fledged game using my own level design. I'm happy I was able to design the game since the old game I used to play as a kid was really fun.

What I learned

I learned how to create a game in Unity.

What's next for Underball

I'd say to have it released on steam for free.
