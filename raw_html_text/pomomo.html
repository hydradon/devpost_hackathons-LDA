Inspiration

We were inspired by the Pomodoro Technique, which is a efficient method of studying, as it implements a perfect balance between work and rest.

What it does

Our application is essentially a productivity timer which time your work and rest sessions. There are also built-in background noises that are proven to increase productivity and focus.

How we built it

We used the Netbeans IDE to code our program, and used libraries like java.util and java.applet.Audioclip.

Challenges we ran into

Initially, we began with the idea of creating an android alarm clock app with Android Studio. However, we soon found that we could put a spin on this idea and implement features of our original idea into a productivity app for desktop.

Accomplishments that we're proud of

We are proud of aesthetics and completeness that we were able to accomplish in a mere 22h. We were able to overcome big roadblocks wherein our code was not functioning as we wished it to do so. Additionally, it's very user friendly and intuitive in design.

What we learned

We learned how to manage time effectively and use online resources when faced with uncertainty.

What's next for Pomomo

In the future, we hope to implement Pomomo as a mobile or web application to reach a wider variety of users.
