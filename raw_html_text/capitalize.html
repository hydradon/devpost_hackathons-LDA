The idea of this app was to utilize the Capital One API in a unique and interesting way. We decided to incorporate our different interests into one project. This project uses Capital One's API in the background as a way to keep track of accounts and customers. Since we spent time learning a little about HTML canvas, we wanted to utilize it somewhere in our project. This is how we came up with creating a more secure way of logging in to your bank through signature checking with machine learning. This app also utilizes Google's Firebase's Firestore and Auth to store past signatures and other login information. This is still heavily a work in progress but it was amazing to incorporate all of our ideas into one concise project and learn something along the way.
