What it does

This website asks to take the users location and gets a random restaurant within a 10 mile radius of the user. It also gives the user some information about that restaurant.

How we built it

We used the Google Maps to retrieve the users location and the Google's Places API to retrieve information about restaurants in the area's and their information.

Challenges we ran into

Lots of issues with using Googles tools as they are not flexible with requests.

Accomplishments that we're proud of

We all learned a little bit of javascript . We are proud that we got the google places api functioning with parameters.

What we learned

We learned so much because we were not familiar with javascript at all nor any previous experience using google's tools.

What's next for Restaurant Roulette

We hope to expand this into having better parameters and adding the option to add filters such dietary restriction and price.
