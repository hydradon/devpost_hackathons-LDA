Inspiration

We wanted to work on something that would require a variety of roles/parts, and this was our best idea.

Our Goal

Students can post pictures of the classroom board(e.g. powerpoints, white/blackboards), and the best pictures can be upvoted. The best pictures can be downloaded as a package from the website.

How we built it

Using Django, HTML and CSS, we created a web application that was made up of subcomponents that functioned together.

Challenges we ran into

It was our first time using Django for many of us, and our first time of a project of this magnitude for all of us. This came with a lot of challenges, since we were often times initially unsure how to handle roadblocks. Though, with some perseverance and google-fu, we managed to overcome most obstacles.

Accomplishments that we're proud of

This was our first time handling a project like this, so we're happy that we got this far.

What we learned

Watch out for missing parentheses.

What's next for Wat Is Class

Refactoring code, fixing voting issues
