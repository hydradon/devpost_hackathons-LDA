Inspiration

We decided to make this project based on an issue that many companies have of figuring out which vendors to sell to given a limited inventory

What it does

The user changes the inventory and enters what their customer list is. The program does the work and decides which customers to sell to in order to make the most money.

How we built it

We used Pandas and Python in order to go through the data and figure out the ideal customer list.

Challenges we ran into

We found it very difficult to maintain focus throughout a 24 hour period.

What we learned

We learned how to use Python and how to work as a team.

What's next for Inventory Challenge

I think we can expand this project to have a better front end that is more intuitive and interactive for the user.
