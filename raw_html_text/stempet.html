STEMPET

Sistem Pembayaran Tempat Wisata

TUJUAN:

  1. Menerapkan teknologi informasi pada sistem pembayaran di tempat wisata
  2. Manajemen sistem pembayaran tiket masuk atau tiket permainan yang terintegrasi
  3. Manajemen sistem pembayaran dagangan di tempat wisata yang terintegrasi

Ada tiga aplikasi android yang dibuat yakni:

APLIKASI PENGUNJUNG

Aplikasi ini membaca IMEI Smartphone Android dan tanggal hari ini. Gabungan data tersebut di Enkripsi menggunakan Algoritma AES 128Mode CBC. Data yang sudah di enkripsi dijadikan QR Code.

APLIKASI ADMIN

Proses pengisian saldo pengunjung, Admin menscan QR Code dari Smartphone pengunjung hasilnya dideskripsikan menjadi IMEI. Admin menambahkan saldo sesuai inputan dan disimpan di ID IMEI pengunjung. Laporan admin disimpan di ID IMEI Admin

APLIKASI TIKET/PEDAGANG

Pada aplikasi ini, petugas tiket / pedagang menscan QR Code. Data QR Code dideskripsikan dengan Algoritma AES 128 Mode CBC menjadi IMEI. Petugas Tiket / Pedagang mengurangi saldo sesuai inputan dan disimpan di ID IMEI pengunjung. Laporan Petugas Tiket / Pedagang disimpan di ID IMEI Petugas Tiket /Pedagang.
