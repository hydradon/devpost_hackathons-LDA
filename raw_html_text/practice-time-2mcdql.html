Inspiration

We noticed too many high school students cramming for auditions, competitions, exams, etc. We wanted to help students better plan out their practice so that they can avoid cramming and be more successful.

What it does

Students enter in the date of their audition, competition, exam, etc. and select a difficulty level (1, 2, or 3) and Practice Time tells them how much they should practice each day.

How we built it

We used HTML and CSS in Sublime.

Challenges we ran into

We had to figure out how to make the calculator work, spent a very long time fixing margins, mistakes, etc., and learning how to code (as we had never coded our own project before).

Accomplishments that we're proud of

We are proud of the design and all of the code behind the project, along with completing our first project.

What we learned

We learned about coding in HTML, CSS, and Javascript (which we didn't end up using).

What's next for Practice Time!

Next time, we would like to include a calendar and a page that gives practice tips.
