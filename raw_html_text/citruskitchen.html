Inspiration

Reduce food waste while helping the hungry

What it does

Communicates location and quantity/description of food that restaurants and grocery stores would otherwise throw away to homeless shelters

How we built it

CSS and HTML for the front end; Python and Flask for the back end

Challenges we ran into

Everyone was fairly new to web development

Accomplishments that we're proud of

Web development! :)

What we learned

Web development

What's next for CitrusKitchen

Expanding map API, adding more features for potential additional food "tags" for characteristics, extensive verification processes for restaurants and shelters
