Inspiration

Coming up with ways to rephrase sentences is difficult and time consuming when writing papers.

What it does

Sentence Generator takes in writing and generates all possible grammatically-correct ways of expressing the same meaning. Writers may select any one of these outputs to use in their work.

How we built it

We used Python, Flask, and the Natural Language Tool Kit (NLTK)

Challenges we ran into

Creating a lexicon, using corpora

Accomplishments that we're proud of

Successfully parsing sentences, tagging parts of speech, creating syntax trees, generating all permutations of a sentence using constituents

What we learned

We learned how to use the Natural Language Tool Kit and Flask

What's next for Sentence Generator

Adding synonyms, tone analysis using Watson
