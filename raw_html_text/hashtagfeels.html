Inspiration

We wanted to use several different APIs. We've never used the Twitter or Indico API and they seemed like a natural fit.

How it works

It pulls historical twitter posts for top hashtags and terms and calculates the sentiment for each for the posts

Challenges I ran into

The Indico server suffered from a catastrophic DDOS attack.

Accomplishments that I'm proud of

What I learned

Indico and Twitter have easy to use APIs.

What's next for hashtagfeels

Making a version for custom requests.
