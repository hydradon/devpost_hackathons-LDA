Inspiration

Making new data from old data to bring new perspectives to new data and make a better sense of old data.

What is it?

Analytics for the provided Norwell Health dataset. New dataset created from the original noisy SQL db, which is then used to train a classifier that predicts whether a customer will come for an appointment based on the distance between user and hospital, ethnicity, when the appointment was set and the age of the patient.
