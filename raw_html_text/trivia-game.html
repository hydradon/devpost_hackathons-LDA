Inspiration

We were inspired by our own curiosity and desire to learn.

What it does

Trivia Game is an app that gives the player questions to answer within 4 categories--health, science, math, and geography.

How we built it

We built it by using code in Thunkable.

Challenges we ran into

We ran into a lot of procrastination and we didn't always know what we were supposed to complete next. We persevered and created something we can feel good about.

Accomplishments that we're proud of

We are proud of the entire project because the majority of the group has never attended an event like this. With our combined skills, we were able to make this project.

What we learned

We learned the importance of hard-work and communication. These two components made our project so much easier.

What's next for trivia game

We envision a sequel with harder questions and more of them. This will hopefully be put on the App Store and/or Google Play.
