Inspiration

My first inspiration was working with fashion industry. Ability to create all kinds of App by means of AWS SDK Use my knowlage to dewelop WEB-VR applications for displaying modern style embroidery

What it does

This application show some variations of embroidery. You should use Oculus Rift. When you click on "x" on controller, you teleport to the specified location. You change color of dresses and decorations when you touch the balls.

How I built it

I created some models, textures and simulate some cloth for show and after I added functional with state machine for work with triggers.

Challenges I ran into

Optimization of models and trasfer PBR textures

Accomplishments that I'm proud of

I'm proud to be able to do low polycount models and display my national cloth

What I learned

I learned how to work with state machine
