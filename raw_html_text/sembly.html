How we came up with the idea?

Sembly was created from our desire to help people create and lead gatherings to protect the environment. We wanted to help out leaders organizing protest marches, educators leading eco-field trips, Crisis Responders helping people during disasters, and the general public all across the world convene to positively impact our environment.

What it does?

Sembly is a social cause event planner app: Users can create protest marches or eco-tours and push those routes to supporters and participants via Google Maps and Google Play Services with a point system integrated in.

How we built it?

It was built using android studios, with firebase and the google maps api.

Challenges we ran into?

Learning how to create a joint android application using the google maps api was a challenge. Giving everything function, and storage ability was a little tricky, but we enjoyed the challenge

Accomplishments that we're proud of

Overall, we're happy about our google maps api integration with our custom images in an application that has functionality. The fact that we got the maps to work by setting markers when you add an event, and that we added our own custom images for everything, from the badges to the markers to the event pictures was nice

What we learned

We learned more about how android studios works with firebase, and google maps api implementations.

In the future

Sembly could be updated to have more badges, categories, and a chat feature.
