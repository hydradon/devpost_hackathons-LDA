Inspiration

Matthew 24:28, "Wherever the corpse is, there the vultures will gather." This section in the Bible describes the signs of the end of times.

What it does

A short tile-based game featuring movement, animations, dialogue, item pick ups.

How we built it

Built with the base game framework Bitsy, with an additional hack.

Challenges we ran into

The game was initially made for a standalone framework, but time and energy restrictions prevented us from finishing this framework in time. To ensure we had something to present, we moved the CorpseEater assets over to a basic pixel game maker and used that framework instead.

Accomplishments that we're proud of

The basic UI design, certain animations, the atmosphere of the story, completing the game in time.

What we learned

We learned many javascript hacks that can expand the potential of the tiny Bitsy engine; this knowledge will be very useful in the future if we use Bitsy again. Learned techniques for effective tile-making, sprite animation, and creative abstraction for pixel art.

What's next for CorpseEater

CorpseEater's character is a concept that has stood alone from this game for some time (in our minds). We hope to at least add some more interactions and functionality to this basic game, move it over to the intended framework, and possibly expand the story.
