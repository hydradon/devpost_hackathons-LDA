Inspiration

We tried to borrow a FitBit, but it DID NOT WORK! So we decided to make a game.

What it does

This is a shmup, a side scrolling shoot-em-up.

How we built it

We used Unity2D and C# scripts to implement a game, with a main menu, a game level and a game over screen where you can restart or quit.

Challenges we ran into

Merge errors when using Git. Things that we tried to implament just flat out not working (THE HEALTH BAR)

Accomplishments that we're proud of

We made a simple working game that is actually quite difficult while be simple to understand. (Can be unfair at times)

What we learned

We learned to use Unity and C#. We learned to work as a team and how to implement game dev practices.

What's next for HackAttack
