Inspiration

What it does

Baby Yoda is a small station that users can communicate via laptop. It can add events with time to its local database and remind the user when the user asks it when is the deadline.

How I built it

I use GCP to build the voice to text on the laptop and MQTT to transfer the text to the raspberry pi. I use 3 Servo and connect them with raspberry pi

Challenges I ran into

The challenge is setting up the APIs and environment for raspberry pi and how to make it muti-thread.
