Making the world a better place to live!

What it does for the user:

  • Gives you the best and less polluted route for you to walk / drive / cycle
  • Gives your car automatic problem detection and reporting
  • Enables Wi-Fi for everyone inside your car

What it does for the car manufacturers

  • Gives you real time data from your vehicles emissions
  • Automatically reports detected problems
  • Ranks the best in class in terms of environmental awareness

How we built it:

  • We built this app using some state of the art technologies , such as angular and typescript, and TM forum's new industry standard open APIs. We also used Huawei and FIWARE APIs. And, of course, a lot of caffeine!

Accomplishments that we're proud of:

  • We are proud of giving our contribute to make the world a better place and to include in our project industry standard APIs that are changing the future of our systems.

Suggested API improvements / troubles experienced:

  • Some APIs could not be accessed through Cross-Origin Resource Sharing (CORS)
  • The trouble ticket API could have a key-value mechanism, to enrich the data
  • Huawei Customer Management API doesn't have a reference link to itself, which is needed for party management API
  • Inconsistency in naming of some of the attributes of the APIs, such as onboarding and trouble ticket APIs
  • Some of the APIs don't have a Swagger description, which would make code generation easier
  • Inconsistency in some URL parameters, headers and body structure, across different APIs
  • Security / authentication for accessing APIs is hard to handle if we have multiple APIs, since there is no standardized access scheme

What's next for AirUp: We started in Lisbon but we want to make an impact everywhere. From making people's lives better to increasing the car industry awareness , we want to make a difference that matters by striking right on the core of the problem, the car industry.
