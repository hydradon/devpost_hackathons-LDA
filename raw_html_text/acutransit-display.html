Inspiration

The inspiration came from riding transit for the first time in years. I realized that for infrequent riders like me, finding the right bus that takes me to the right place is such a hassle. I decided to create an application that gets rid of that hassle plus more.

What it does

The Acutransit display is a smartphone application for riders that makes using mass transit simple and efficient

How I built it

I built it using the GTFS data and real-time JSON data that VTA provided for this challenge.

Challenges I ran into

A couple of challenges I ran into were parsing the GTFS data, and determining how to retrieve a route based on the relations included in the GTFS
