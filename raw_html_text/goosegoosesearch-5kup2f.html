Inspiration

Our inspiration for GooseGooseSearch was to prioritize user privacy by keeping data securely from search engines.

What it does

Our searches cite Google, yahoo, and other big name search engines for you, passing no information about you.

How we built it

Using Terra and GTK+

Challenges we ran into

We were not able to implement as many features as we wanted to in a rather short amount of time.

Accomplishments that we're proud of

OS X Yosemite support!

What we learned

It was tedius for Brian personally, since he did not have much experience using Terra, but we both learned valuable commands that will end up helping us in the future.

What's next for GooseGooseSearch

Support for native search results.
