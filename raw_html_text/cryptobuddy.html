Inspiration

We wanted to create a sandboxed cryptography device that would allow people to encrypt messages and send them to messaging apps like Discord.

What it does

CryptoBuddy uses a Raspberry Pi and a Particle Photon to encrypt messages typed on a keyboard and send them to our Discord Server.

How we built it

We used a Raspberry Pi, a Particle Photon, a keyboard, and an LCD display. All the electronics can run off a single battery to ensure portability.

Challenges we ran into

Getting the circuits to work and making sure all the parts received adequate power was challenging. Also getting the Raspberry Pi to run our Python code was difficult.

Accomplishments that we are proud of

We were able to work as a team to create CryptoBuddy and create an almost viable product.

What we learned

We got better at Python, Linux, and Electronics.
