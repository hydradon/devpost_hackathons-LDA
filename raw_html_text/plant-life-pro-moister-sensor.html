Inspiration

We have seen many people suffer with over watering their plants, leading to death of those lushes greens. My mother used to tell me that if you over water it, it would cause it to die but I never actually knew its breaking point.

What it does

We wrote a program with the ardunio in which when the sensor is placed on a moist surface, it makes the LED's on the breadboard blink. When the sensor is put in water, the programs gets the numbers from the sensor, 0 meaning bone dry, and 1023 meaning extremely wet. The program then tells the arduino to light up all the lights. This product helps people restore the green in their lives.

How we built it

we used an arduino with a moisture sensor , jumper wires , LED's and resistors. We started with connecting the LED's and making sure that that works with a spare battery that we had. once that worked, we then started to hook up the arduino with the beard board with our LED s. Then we started to write the code to make the lights light up, after that worked, we started to attach the moisture sensor to the breadboard and implement the code for it.

Challenges we ran into

We had problems with the code for the sensor in which the lights were not lighting up. At won point, we dropped the sensor in the water. it did not work for sometime.
