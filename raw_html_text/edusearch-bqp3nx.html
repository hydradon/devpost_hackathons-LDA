Oposum

Oposum is a quick and easy tool to shorten the time it takes to absorb or review information in YouTube videos. It is a Chrome extension used for YouTube that auto-generates a brief summary of the video.

Inspiration

Oposum has many different applications. It is most useful in education, as students may want to find quick summaries of useful lectures without having to watch the whole twenty or sixty minutes. In addition, Oposum can serve as a tool for reviewing for exams if a student wants a brief overview of what a video is about. However, the extension can be used beyond education, since it can be applied to any YouTube video online.

What it does

When users install this Chrome extension, every YouTube video will have a 'summarize' button. When they click on the button, Oposum's backend will generate a cohesive and short summary that highlights the main parts of the video. Users can then click a button to saves that summary, which they can view later through a webpage that can be accessed through the Chrome extension.

Next steps for Oposum

What we'd like to implement in the future:

  • Save summaries onto the user's Google account so they can view them through other browsers/computers.
  • Allow users to click on a word in the summary and skip to the corresponding part of the video.
  • Allow users to adjust the length of the summary to allow users to adjust how detailed the summary should be.
