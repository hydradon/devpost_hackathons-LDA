Inspiration

Big Two is a fun and enjoyable game that has entertain multiple generations, with many more to come. With the increasing prevalence of mobile gadgets, why don't we make our all-time favorite game paperless?

What it does

A typical setup consists of a tablet which serve the purpose of a table and a smartphone for each person as game controller. Upon creating a room in the tablet, players can connect the room and start playing!
