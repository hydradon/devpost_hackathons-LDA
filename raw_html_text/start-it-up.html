Inspiration

Nuestra inspiración fue el gran numero de PyMEs que hay en México y las que aun no pueden ser desarrolladas debido a la falta de apoyo.

What it does

Brindar el apoyo necesario a los usuarios que tengan un proyecto por lo menos en fase 2 de desarrollo y que necesiten ser impulsados y/o financiados.

How we built it

HTML5, CSS, Java script y Firebase

Challenges we ran into

EL tiempo, falta de algunos conocimientos técnicos.

Accomplishments that we're proud of

Coordinación en equipo, poner en practica conocimientos de desarrollo web

What we learned

La comunicación es clave para un óptimo trabajo en equipo, desarrollamos nuevas habilidades de desarrollo y un nuevo sistema de trabajo.

What's next for Start it up!

La implementación del prototipo y difusión de nuestra plataforma
