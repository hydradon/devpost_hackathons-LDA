Inspiration

We wanted to create an application to help students study for memorization based classes. We thought Quizlet had many good functionalities, but we wanted to add features that would make it more convenient for students to remember new information.

What it does

It is used to store sets of flash cards for studying purposes.

How we built it

We used android studio to build the whole thing while using GIT to exchange and ensure the quality of our code.
