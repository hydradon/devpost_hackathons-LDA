Inspiration

As students with many small daily transactions, we found it difficult to manage our finances.

What it does

The app prompts the user for any financial question and provides financial history for the user.

How I built it

We used Dialogflow for the integration of voice, then used Android Studio to develop the app. We then used Firebase within Android Studio to integrate Dialogflow into the app.

Challenges I ran into

Integrating Dialogflow into our app was very challenging because of the intricacies involved in the process.

Accomplishments that I'm proud of

We learned how to create a functioning app that utilizes a chatbot and multiple platforms, which we have never done before.

What I learned

How to use voice integration for an application.

What's next for Blue

Implementing bank API's to retrieve specific information for clients.
