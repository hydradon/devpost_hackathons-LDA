Inspiration

Foreign-born parents who had trouble finding the best location in the US to move to for them to be successful.

What it does

When given information on an immigrant, determines where they would be best suited when moving to the United States.

How we built it

-Front-end website designs built with Webflow, back-end built with VisualStudio using JavaScript.

Challenges we ran into

-Parsing data from different sources, differing names for cities.

Accomplishments that we're proud of

-1702 cities, 35 career fields, 48 ethnicity options.

What we learned

-Front-end website design, filtering large data streams.

What's next for AILA

-More categories to determine the best location, include more cities.
