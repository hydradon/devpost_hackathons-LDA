Inspiration

We were inspired by an appreciation for SparkPost and an interest in email functionality.

What it does

Email-to-Amish allows you to email the text of a letter to a server which would print and mail the letter to the address and recipient of your choice. You can also enter the letter and recipient into a web app.

How We built it

We built it using node.js, Javascript, Html, CSS Bootstrap, and a small amount of jQuery in conjunction with SparkPost's fuse-email and lob.com's lob-node.

Challenges We ran into

We ran into a challenge formatting the letter correctly and communicating the web app with the server back end.

Accomplishments that We're proud of

We're proud of how far we came with the node/javascript side of it since we started with very minimal knowledge of how to use them.

What We learned

We learned how to connect different packages and use them to work together and accomplish a wider task than they're meant for.
