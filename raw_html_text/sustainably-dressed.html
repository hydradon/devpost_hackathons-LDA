Inspiration

The fashion industry is one that uses up tons of resources unnecessarily, and we wanted to create something that would allow people to buy clothing that leaves a smaller footprint on the environment.

What it does

It's website that searches through multiple secondhand/thrift websites so that people can have an easier time finding sustainable and affordable clothing.

How we built it

We built it using HTML and CSS for the website and built a web crawler using Python to gather the information for listings from different websites. We also had a machine learning process to identify items of clothing that people can buy more sustainably.

Challenges we ran into

We weren't that familiar with HTML, CSS, or Python, so developing and building everything took a long time to figure out.

Accomplishments that we're proud of

We're proud that we were able to pull all the necessary information from the websites we needed to, and that we trained the AI to be able to identify different types of clothing.

What we learned

We learned a lot about web scraping, web development, and machine learning and got experience using HTML, CSS, and Python.

What's next for Sustainably Dressed

We want to add more websites/companies and improve the design of the website.
