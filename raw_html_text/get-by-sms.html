Inspiration

Find an easy, full protected way to share my private files with other persons.

What it does

Using this tool you can share files without effort. How ?
Uploading the file and sharing to several phones with a few clicks. And then ?
Unlocking and downloading the file sending an SMS.

How I built it

I used several tools to build this project:

  • Base code in NodeJS
  • Realtime with Socket.IO
  • Project hosting on Heroku
  • Database using MongoLab
  • File storage Amazon
  • Url Shortener Google
  • SMS support Nexmo
  • Email support Mailgun

Challenges I ran into

I studied several APIs to integrate all of them in this project.

Accomplishments that I'm proud of

The most important accomplishment is the result, a small, clean, and secure tool to share files.
