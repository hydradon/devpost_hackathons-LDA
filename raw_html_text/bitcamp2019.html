An animated Octocat hacking away!

Made with Processing written in Java.

My team member and I have no experience in Processing but we were both interested in using Java/Processing to make animations. We decided creating a hacking Octocat would be a great start!

Octocat can type so fluidly because her tentacles stretch using the sine function.
