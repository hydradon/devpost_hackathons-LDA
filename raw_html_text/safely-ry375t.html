Inspiration

What it does

Users may log in to view any emergency notifications that are notified on Google Maps. Other users may also submit other help notifications

How we built it

Backend using Firebase, Google Maps API

Challenges we ran into

Becoming familiar with mobile app development, Android Front End work, XML, and understanding how the Google Maps/ Firebase APIs work.

Accomplishments that we're proud of

First time Android app and using the Firebase API! :) Made a lot of personal achievements and learned a lot from this project!

What we learned

This application can be heavily expanded into something that could possibly save lives!

What's next for Safely

Implement more features such as messaging system, a way to send contact information to the people that need help, and more.
