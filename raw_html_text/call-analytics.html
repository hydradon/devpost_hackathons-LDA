What it does

This application provides call analytics for small scale businesses.

How I built it

I used Azure LUIS for identifying entities involved in the conversation and their intent. Azure's speech services were used in transcribing the voice messages and analyzing the sentiments expressed in the conversation. Speech Recognition API was used to identify the individual speakers and associate corresponding the text and sentiment.

Challenges I ran into

I had a hard time enrolling the user for speaker verification.

What's next for Call Analytics

Integrating azure intents and other applications like calendar, reminder, etc.
