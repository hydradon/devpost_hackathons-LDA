Inspiration

We got inspired by wanting to have big arms..

What it does

It’s a simple clicker game that allows the user to increase the weights and based on the weight, it’ll use up more energy but it’ll increase your score and get you to a higher level quicker. Once you’re at 0 energy you lose. If you rest you regenerate energy.

How we built it

We used the JavaFX platform to build a graphical user interface and add listeners to event triggers as a way to record the user's clicks.

Challenges we ran into

Some challenges we ran into were deciding between what java libraries we should use. At first we were going to use Slick2D but we couldn’t get it to work with our IDEs so we decided to do it with JavaFX.

Accomplishments that we're proud of

We’re proud of adding images and music and listener classes to our application.

What we learned

These are things we’ve never done before but we managed to get all of it to work. We learned how to work together on GitHub and we improved on our collaborative skills since we aren’t used to working with multiple people on coding projects.

What's next for Workout Game

The next step in the game will be to add better animations. Maybe if we learn how to draw the animations will be better.
