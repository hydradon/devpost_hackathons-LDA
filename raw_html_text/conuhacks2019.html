Car Chase

A purely Java game built from the ground up without using any external libraries.

We wanted to challenge ourselves to build a video-game in 24h using Java. What we came up with is this top-down single player action game. The original plan was to make it multiplayer by having players connect to a local server but unfortunately, the implementation was very buggy and not stable.

Features

Purely Java, no external dependencies.

Our own physics engine built from the ground up.

Randomly generated maps for each play through. This includes health pickups, obstacles, and enemies.

Health bar.

Pathfinding Enemies.

Reset button for quick playing.

Very nice collision checking.

Challenges

The biggest challenge was to get the game running in local multiplayer. It worked, but unfortunately, it was too buggy to play (eg. very high ping in excess of 1000) so we switched to a single player mode.

In the future

Looking ahead, we would want to implement a:

Menu to choose singleplayer and multiplayer.

A point system and global leaderboard.

A functioning server architecture.

Get better graphics.
