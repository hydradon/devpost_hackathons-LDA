Inspiration

Everyone in our group has ran into this issue before and we just wanted to try and provide a solution.

What it does

Takes in your clipboard contents and sends them to our server and gives you an access code for you to get it on any computer.

How we built it

We used flutter and dart to build the mobile app and HTML and dart on the server side

Challenges we ran into

Encryption and backend

Accomplishments that we're proud of

Getting the Mobile working!!!

What we learned

A lot about dart as a language and flutter as a UI kit

What's next for Common Clipboard

Finish the encryption and maybe publish to the respective app stores!!
