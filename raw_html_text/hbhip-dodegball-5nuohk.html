Inspiration

The desire and ambition to explore the creativity and ingenuity of VR. Neither of us have created any programs with VR which inspired us to pursue this sort of project.

What it does

It is a computer video games that uses popular VR headsets like the HTC Vive. The player is immersed in dodgeball the entire time.

How I built it

We used Unity 3D, Windows 10 64-bit, and the HTC Vive.

Challenges I ran into

Our limitations and knowledge of C# we limited.

Accomplishments that I'm proud of

We were able to finish the game with time to spare. We believe it is a solid game.

What I learned

Game development is hard work. It takes a team to put a project together.

What's next for HBHIP Dodgeball

Some more learning of programming languages and new games for development.
