VoxBox is a portable data driven speaker that can read out messages and tweets. Applications of this product include a direct messaging or notification system for a person, or following specific people or trends on twitter. Tweets would be read out every time a new one is added. Example applications include:

  • Live twitter announcer at events, such as hackathons
  • Personal aid for the visually impared - recieve important notifications without interacting with the device
  • Use in retail - allow customers to interact with each other through a common audio message board
