Inspiration

I wanted to develop a better space to collaborate remotely.

What it does

Two webcams are facing down on the individuals desks and is streamed to our web app. The web app blends the webcam feeds to create a blended workspace. People can write and can see the transparent collaborator and work together.

How I built it

used meteor.js and icecomm.io to facilitate the platform.

Challenges I ran into

Blending the two webcam feeds and this was my first experience using meteor.js

Accomplishments that I'm proud of

This is a simple solution to make this shared workspace, but it looks great!

What I learned

Basics of meteor.js

What's next for 2D AR Studying

Adding more functionality to enable students around the world to get access to great tutors.
