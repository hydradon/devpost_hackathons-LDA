Inspiration

We wanted to have a fun and easy to play implementation of the popular game Newlywed.

What it does

One player has to guess the answers of their friend to a series of questions, chosen from a range of topics.

How we built it

The Arduino records the data from 8 buttons, that is read in Python using PySerial. The graphical user interface was developed using Pygame.

Challenges we ran into

We faced difficulties in reading continuous input from the Arduino in Python and dynamically updating the Pygame screen.

Accomplishments that we are proud of

We are excited about our functional result and our awesome team spirit! :D
