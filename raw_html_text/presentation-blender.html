Inspiration

During the hackathon's opening ceremony lots of time and effort were wasted connecting different laptops to the projector to play different presentations from different people.

What it does

Basically it allows you to play all the pretensions sequential with the ability to insert a presentation in the middle of another presentation allowing you to have like a master presentations then merge the other presentations in to it.

How we built it

It is build using a python library called pptx alongside a GUI created using JavaScript

Challenges we ran into

Finding a library that can work with .pptx files

Accomplishments that we are proud of

What we learned

.pptx files are a nightmare!
