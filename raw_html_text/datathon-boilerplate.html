Inspiration

We had in mind to create a project using Node and Angular. When we were looking for ideas we found out that there were a lot of projects to create hackathon but none to create a datathon. So we had a project.

What it does

It's a platform to create a modelling and data competition, allowing the users to upload their predictions and check their precision with their models. It also allow for user registration.

How we built it

We used the MEAN stack.

Challenges we ran into

None of us had never programmed a Mean app so we faced a lot of problems. The first day we spent about 10 hours to get everything working

Accomplishments that we're proud of

To finish it!

What we learned

How to configure MondoDB. How to create a API with express and node. How to program in Angular

What's next for Datathon Boilerplate

We would like to improve the project and let it become a complete solution for organizing datathons.
