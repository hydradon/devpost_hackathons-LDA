• Youtube Clone App

Inspiration:I am inspired by the community leaders and members ,how they help others to explore this DevC Challenge.

What it does:This application lets you to search videos through a search bar at the very top.The search item is then used to search through the videos that are available on the youtube.A related search bar is present on the right which shows the related videos to the search video.The main search video is then displayed on the iframe in the left.

How I built it

Challenges I ran into: To get the videos getting through the youtube api and then integrate into my application.

Accomplishments that I'm proud of:That i am able to display and search all the videos available on the youtube.

What I learned:To use React to integrate to youtube API ,to browse through videos.

What's next for Youtube Clone App:I am going to make the front end of this application awesome
