Inspiration

The inspiration behind this game was the idea that we can have the convenience of owning an animal in your homes. With this game itself, it could teach responsibility to users on how to take care and meet the basic needs of an animal

What it does

Provides a virtual dog in which the user is able to interact with as well as interact with their environment.

How we built it

We used Blender to develop and edit the models, and used Unity as a platform to develop the virtual environment. We used Visual Studio to create scripts in order to interact with the environment.

Challenges we ran into

Some challenges that we ran into was emulating hands in a virtual environment, adding user input to react to the Pet and the users surroundings, and writing the script to add animations to the pet.

Accomplishments that we're proud of

We are proud of being able to finish a product which is able to provide the user with an interactive pet. Also, we are proud of being able to self teach ourselves unity within 24 hours in order to create a final product.

What we learned

We learned to self teach ourselves unity within 24 hours in order to create a VR game/simulation.

What's next for Vroof

Our goal is to allow the user to choose what and how many pets he/she wants to add in the simulation.
