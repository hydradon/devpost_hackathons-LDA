Inspiration came from wanting to communicate with the people in a way that is fun and interactive

What it does is that is allows users to drop a geo tag location with an accompanying message

How we built it using React Native CLI for the mobile app simulator with the front-end in JS

Challenges we ran into integrating compatible code using different operating systems (iOS and Windows)

Accomplishments that we're proud of is that we got a map up defaulting on the campus, and message text input.

What we learned how to use React overnight for the first time.

What's next for Message In A Bottle is having a secure log in and collection page and library of found messages.
