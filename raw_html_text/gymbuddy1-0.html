Inspiration

We love going to the gym, but we don't always have someone to go with.

What it does

It links gym goers in the same university by suggesting people with the same availabilities and interests.

How we built it

Using Xcode, swift, and firebase as out database.

Challenges we ran into

User authentication using firebase was a challenge. Also, Swift was a new language to us. We also ran into github problembs.

Accomplishments that we're proud of

Getting user authentication to work and have a working UI model.

What we learned

We learned a new language and how to integrate a working database in our IOS app. We also learned how to use XCode's UI model.

What's next for GymBuddy1.0

We're going to finish the app completely and maybe even put it on the app store!
