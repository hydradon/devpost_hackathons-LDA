Inspiration

In today’s light of social issues, it seems to us that free speech has been shadowed by the Political Correctness culture. We want to create a platform where anyone can express their thoughts and art without any limits. Anything goes.

What it does

The website allows users to draw whatever they want in a communal digital canvas in real time.

How we built it

*Used node.js for the back end of the website. *Used socket.io for the real time interaction. *Used MongoDB for the database. *Used Heroku to host the website. *Used GitHub for project management and sharing.

Challenges we ran into

One of the challenges we ran into was integrating the zooming feature so the canvas will not feel like it has a border. The other challenge was almost losing the stable build in GitHub. Trying to use GCP.....

Accomplishments that we're proud of

We are very proud of the quick response time to multiple people drawing.

What we learned

We learned real time communication with socket.io, the implementation of MongoDB, and the front end features.

What's next for RITMural

We want to integrate the zooming features, Spotify features, add image files and text, and painting rewind.
