Inspiration

We wanted to find a way to help people save money whenever they go shopping and be aware of the impacts they make when they drive their vehicles.

What it does

Prixify calculates the total price when you shop for items and shows you the carbon emissions from your journey.

How we built it

We used the Google Maps API and JavaScript with code.org.

Challenges we ran into

No prior knowledge of JavaScript or APIs.

Accomplishments that we're proud of

We stayed up all night and finished our product!!!

What we learned

We learned how to implement a Google Maps API and use the distance between 2 locations and how to build an app using JavaScript.

What's next for Prixify

Implement interactive google maps, different modes of transportation, and price detection
