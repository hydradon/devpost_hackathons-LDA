What it does

Built for parents, it tracks the users internet activity (which websites visited and how much time they spent on there) It is also password protected(for parents) and an option to set up a "tracking time" is available, you want to give your children a little bit of freedom right

How I built it

Javascript and html coded in visual code

Challenges I ran into

This is the first time our team has ever used javascript and google chrome api, so there were many issues that we ran into.

Accomplishments that I'm proud of

Completing a functional app, it's not the best but it works

What I learned

Javascript, HTML, CSS, Google Chrome API

What's next for Internet Tracker

Perhaps set up a better way to send "daily logs/update" of their children usage. Google Chrome extension does not support sending API so we need to look for a better way.
