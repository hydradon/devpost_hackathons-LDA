• Citizen

Inspiration

Jame's loves graphic design and Jacob has some experience making small games in Java. We combined our skill-sets to build this game on the theme of civic engagement.

What it does

The user is controlled with the arrow keys to move the player forward, backwards, up, and down. The user must avoid hitting the left side of the screen (out of time) and hitting any busses.

How we built it

James designed all the graphics with Paint.net and Jacob wrote the code in Java

Challenges we ran into

Various challenges came in debugging. It has been a while since coding in Java so Jacob also had to remember certain syntax and methods to building each piece. Thinking of themes and what everything should look like.

Accomplishments that we're proud of

The game works well! We still have room for improvement, which we plan on working on after the Hackathon. The graphics also look really nice!

What we learned

Before deciding on the project, we both tried learning some python. We also learned how to work to our strengths in a team setting.

What's next for Citizen

We will likely make better graphics and improve the enemies and other obstacles
