What's next for MAC Address Spoofing

MAC address is an hardware embedded address in an Network Card. I have built a script in Python using nmap package which will help us to filter all the MAC address in a given Network. My script will list all the MAC address in the Network and randomly select one MAC address or the User can select any one MAC address from the list and assign it to his Network Interface Card and can do illegal stuffs in a Network. In the script we can modify in such a way that one of the MAC address can be assign automatically by the script itself to the Network Interface card.
