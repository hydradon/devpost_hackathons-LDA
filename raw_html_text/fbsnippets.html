Inspiration

As a developer when sometimes i am working in coding and face some functionalities problems in programming, i need to search in google and forums as well searching deep inside the source code in question so as to be able to develop and find solutions to the functionalty problem. these days every person have a facebook account, considering facebook to be the largest social media and community in the whole world ,so why not use it among us as developers to share knowledge of coding.

What it does

fBsnippets is a platform that allows developers to share any coding snippets and help to treat solutions about some functionalty or problematic might be encountered when programming with Facebook technologies, to make developers able to quickly find solutions to the problems.

In this platform we find :

  • Facebook Login : to connect directly with facebook accounts.
  • Social Plugins : Like, Share, Comments. to give your points of view about every snippet.

So every shared snippet is accompanied by :

  • two special buttons "it works" or "not working" to let other developers voting.
  • Facebook like and share button.
  • Facebook comments.
  • Tags

Features and functionalities :

  • Share snippets on the platform and facebook.
  • Interact with snippets using facebook Like ,Share and comments.
  • Search snippets using filtre.
  • Find most popular snippets.
  • Get latest shared snippets notifications.

What's next for fBsnippets

  • Link with github or any other similar platform to permit developpers to easily extract wanted snippets and then share them on the given platform.
  • Possibility to share other snippets in different programming languages ( other than facebook's).
  • Advanced search with several criterions to easily find your snippets.
  • Integrate messenger platform for the developers to connect freely.
