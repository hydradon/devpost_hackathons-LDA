Inspiration

My daughter loves animal sounds so I thought I would add some extra ones to Alexa so we could play while she has her baths.

What it does

Ask Alexa for an animal sounds from a library of over 70 different animals.

How I built it

I built it using javascript and lambda functions.

Challenges I ran into

It took a long time to collect the animal sound library.

What's next for Animal Sounds

With over 1000 unique users so far, I'm proud of the usage my app is getting. I've got plans to add an animal sound quiz and more animals to the library.
