Inspiration

As young kids, we always played this word game relating to the last letter of any place, and we thought it would be interesting and neat if we could bring the experience to a laptop where you could play with it.

What it does

This project is a verbal game that detects user input to play and returns output.

How we built it

We used python and google speech API.

Challenges we ran into

We used Repl.it and Sublime so we often had issues when transferring code over.

Accomplishments that we're proud of

We're quite proud of learning how to use the Google Speech API

What we learned

We managed to learn many things about how many processes work in our project, such as how to isolate letters from word imputs, and read, split up, and randomly select an item from a text file.

What's next for Atlas

The next steps are to expand our game play to beyond just simply countries, including states, cities, mountains, etc. [all other instances of places.
