What it does

Checks in for flights right as it becomes available to get the seat you prefer, then sends you a text message with a digital ticket for the flight.

How we built it

Mock flight checkin site with React, robotic process automation with UiPath.

Challenges we ran into

UiPath & Visual Basic are completely alien and feel like learning how to program all over again. Documentation and community isn't great, so a lot of trial and error were required.

Accomplishments that we're proud of

Using non-trivial UiPath features like email automation & parsing, web automation & scraping, and general UiPath automations.
