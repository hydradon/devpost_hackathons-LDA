Wind down time can be difficult to get now a days. Life's busy - but unplugging once in a while to watch a movie, travel, or read a book is necessary for our mental health! Enjoying relaxing activities like these feels great - except when we're disappointed with the experiences!

Reading Report is a tool which helps readers invest time into reading books they enjoy. The program scans PDFs for texts using ImageMagick, Tesseract OCR, and a collection of bash&Python scripts to let readers know what kind of book they will reading - positive, sad, scary, etc. These characteristics are determined by Indico's machine learning API.
