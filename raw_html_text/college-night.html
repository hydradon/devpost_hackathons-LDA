Inspiration

College has a plethora of events to explore. We wanted a way for students to stay up to date on events going on around campus.

What it does

The app allows the user to view posted events as well as post their own. the experience is completely anonymous.

How I built it

We used android studio and created our servers through firebase.

Challenges I ran into

We ran into trouble with syncing progress from different users together.

Accomplishments that I'm proud of

We are proud to have been able to build an app in this small time frame.

What I learned

We learned a lot from each other when it came to working with android studio, which i have never worked in.

What's next for College Night

We hope to streamline the app and create a better user experience.
