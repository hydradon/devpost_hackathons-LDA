Inspiration

We wanted to rebuild our childhood game, bringing in our knowledge of modern tools and technology.

What it does

A single player game that simulates a bike racing tournament and can be controlled via physical steering of the bike controls.

How we built it

Designed a canvas emulating the racing arena. Set up the server on Intel Edison and configured it to interact with the game using an accelerometer.

Challenges we ran into

Difficulty in calibration of the accelerometer.

Accomplishments that we're proud of

We win the race every time we play it. We strictly enforced a no lose policy.

What we learned

No matter how hard it is, a game can definitely be built using just JavaScript and plain HTML.

What's next for ProjectInfinity

We plan to scale this by allowing multiple players to join in (and maybe even win sometimes).
