eStats

This code creates an application on React Native for both iOS and Android systems. It is written in Javascript.

NewHacks 2019 Submission

The application shows a person's energy usage from an Arduino tracking the energy meter, and gives instantaneous feedback about usage. The app shows peak hours, energy prices, and a monthly energy bill estimation. It also allows the user to input their daily usage goal. There is also an option for push notifications if the user uses more than 80% of their daily energy usage goal. Since the user is more aware about the money they are spending each month on electricity, they will decrease their energy usage, which is beneficial for the environment.
