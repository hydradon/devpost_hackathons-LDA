Inspiration

We wanted to help people who may not have an idea of who to pick for a team. This is especially useful for first time hackers like us.

What it does

If it is completed, it would be a website that allows a hacker to input their information. It then outputs a list of hackers by how similar they are.

How I built it

We developed the front end in HTML. The back end was developed in Java using IntelliJ

Challenges I ran into

We could not figure out how to link the front end and the back end.

Accomplishments that I'm proud of

This was our first hackathon, so the fact that we were able to develop something makes us proud.

What I learned

I learned how to code in HTML.

What's next for HackMatch If we decide to work on it, we can connect the front and back end and improve the algorithm to produce better matches.
