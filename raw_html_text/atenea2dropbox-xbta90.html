Inspiration

As UPC students, we have all been stuck in front of the computer downloading file by file from the almighty Atenea, no more.

What it does

It enables the user to select which documents and subjects he wants to synchronize, and by logging in to his Dropbox account he gets them uploaded.

How we built it

We used the Dropbox API (PHP version) to get the autentification and authorization down as well as folder and document creation. Data is obtained from Moodle with a php script and temporary stored in a MySQL DB. We do not keep any password.

Challenges we ran into

MySQL will work quite unstable, we had to enlarge timeouts and such, in addition to that the whole process is still very slow.

Accomplishments that we're proud of

Having it all work together quite nicely and smoothly.

What's next for Atenea2Dropbox

Improving time efficiency.

P.S: atenea2dropbox.com was purchased with the domain.com voucher and is still not fully functional, (not accessible from every server).
