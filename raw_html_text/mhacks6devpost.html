On the Record, On the Line

A online poll used to help foster communication within community. But _ how _?

The code behind this is simple. A basic php script that takes form data from selection and displays in a text database. What the words mean is where the Social Impact is taken. Too often conflict is started because there was a breakdown in communication. The benefits of this hack are its openness. Anyone can easily vote in this form. We have no limits on the voting, in order to get as much information as possible. However, it is secure, and no one can change the underlying idea of this disruptions.

I really learned alot in php with this hack and wish everyone luck!

Find this hack live at www.varfield.co

I hope to really improve society with this proof of concept
