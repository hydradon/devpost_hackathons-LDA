Inspiration

Last.fm app when it was an internet radio.

What it does

Showing lyrics and artist bio along with similars and genres in the player window while listening. Also showing artist info, album cover, song lyrics, personal play count for songs, personal favorites and many info about the now playing song collected from Spotify, Last.fm, Genius.com. Additionally, a lot of control playback features with Spotify Premium are available: auto-skip blacklisted songs, prev/play/pause/next, auto-skip songs by maximum play count.

How I built it

100% javascript. Hosted on Heroku.

Challenges I ran into

Implemented Oauth2 client module, api clients for different music services, search functions with distinguishing genres.

Accomplishments that I'm proud of

It's a working prototype with big plans

What I learned

How to implement client-side applications combining various api.

What's next for shuffle.guru

On-fly playlist generator, Full-text search, Skip by genres/artists, support for Google.Play Music
