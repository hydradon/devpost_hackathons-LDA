#Inspiration

Like it, double tap. Love it, share it. There is a lot goes behind the scenes before reaching this stage on social media. Filters, caption, hashtags !

#WhatItDoes

Chat with our bot and send in an image. Get hashtags that are tailored, based on the visuals and the moods reflected by the image.

#HowWeDidIt

We integrated Messenger Bot API and Google Cloud Vision API , deployed on the Heroku server .

## #Hurdles#Challenges Deploying the messenger bot in the Heroku server was as challenging as integrating the Google Cloud Vision API on the server.

#Accomplishments #Take-away #Lessons

It is our team's maiden hackathon experience and dabbling with different APIs got us energised. The fact that we could integrate them all to produce the final product, was in our opinion, an accomplishment for our team! We were able use google cloud api to display web entries that can give hashtags based on information available on the internet.

#WhatsNext

To run this chat bot on the Instagram Direct Message platform would be more fitting to our theme of hashtags; and hence is our expansion plan.
