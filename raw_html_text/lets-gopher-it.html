Inspiration

We always have a really hard time making decisions.

What it does

This web app will allow you to make a simple profile and then you can get in a group with some friends. Once everyone is in that group if someone presses one of the buttons such as the food button it will find a location that matches everyone's profile using the yelp api.

How I built it

Using Go for the back end and CSS, HTML and Angular for the front end.

Challenges I ran into

Connecting the front end to the back end and integrating a database.

Accomplishments that I'm proud of

This is the first project we finished that has a really nice front end.

What I learned

We improved our Go skills and our front end development skills.

What's next for Lets Gopher it

A few finishing touches such as integrating google maps so you can get navigation straight from our site and maybe making it mobile.
