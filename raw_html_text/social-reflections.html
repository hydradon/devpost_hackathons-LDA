Inspiration

Harvard Business Review often runs articles with advice on how to be deliberate in your social interactions: https://hbr.org/2011/07/managing-yourself-a-smarter-way-to-network . This application allows for continuous input of a user's data points into a data set that many of these models can run off of.

What it does

Social Reflections helps the user keep track of their face to face interactions, and delivers insights on how they're spending their time.

How we built it

We used Django REST framework for the back end, and React for the frontend.

Challenges we ran into

We had difficulty configuring React for use with Django REST framework, and learning how Django works.

Accomplishments that we're proud of

We worked pretty well as a team!

What we learned

We need to find a faster to configure tech stack that we are more familiar with. That involves taking deliberate time before the hackathon to familiarize. #fullstackjavascriptbros

What's next for Social Reflections

Finish deploying the website itself, and then look into deeper analytics with ML/NLP.
