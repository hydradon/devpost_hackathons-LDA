Inspiration

Thinking about how to use IOT devices for easy access to peoples houses in a AitBNB scenario..

What it does

Allows users to set up their home locks and share secure keys to give people access to enter.

How we built it

Using Java backend API deployed on AWS. A mobile app for the client keys. Web app for Owner management.

Challenges we ran into

Deciding on the design and figuring out how to implement it.

Accomplishments that we're proud of

Making a full API, with web app and Mobile application.

What we learned

How to quickly integrate and work together.
