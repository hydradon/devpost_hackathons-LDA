Inspiration

We were inspired to leverage the widespread use and appeal of Apple's iphone environment as well as taking advantage of the functional dearth of non sticker apps on the app store to create a way to order an Uber right from the iMessage app.

What it does

Allows you to easily input an address into the Uber request ride widget right from the iMessage screen and order an Uber without needing to leave the app.

How we built it

We built it on Xcode using the Uber IOS SDK with Swift and the Uber API.

Challenges we ran into

We faced a many issues with pod integration of the UberRide pod files. Also, integrating the data from the uber api using node with our ios app was difficult.

Accomplishments that we're proud of

We are proud that we learned how to use and manipulate the uber API and created an imessage application.

What we learned

We learned how to create node server and pull data from the uber api. We learned how to create an uber integrated imessage application.

What's next for RideNow

Resolve xcode errors with the pod files Request access to the limited scopes to get the real time data of the rider form the Uber API.
