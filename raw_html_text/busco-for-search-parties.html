Busco is designed for search parties looking for missing persons. The app tracks members of a search party with GPS and periodically upload their locations to a central server-side list before updating all members with all previously visited places, so the search party can avoid doubling up on locations. There is also a web viewer that can display all the visited locations in a web browser.

There are three components to Busco:

  • App. Built with Android in Android Studio and using the Google Maps API.
  • Server. Built with Node.js.
  • Web viewer. Built with HTML, CSS, JavaScript, Ajax, and the Google Maps API.
