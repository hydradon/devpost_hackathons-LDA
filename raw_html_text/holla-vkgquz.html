• Preliminary Wireframing of android application

Inspiration

How many time have you forgotten to text your mom to let her know that you had reached home safely after volley-ball practice? How many times have you also forgotten to text Jennifer that you had got home in one piece after a wild night out ? Too many times! But now, we're fixing it!

What it does

Enables you to automatically send a text message to your friend, family or loved one when you get to a specified location. Even if you forget. Ensuring your friend Bobby always gets home in one piece, one text at a time.

How we built it

  • Native android development, built with android studio.

Challenges we ran into

  • Using the google API for geofencing.
  • Building a decent user experience.

Accomplishments that we're proud of

  • Great user experience
  • Seemless Integration with native android contact
  • Gorgeous user interface

What we learned

  • The value of friendship
  • Android development
