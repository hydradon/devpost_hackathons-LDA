Inspiration

I've had a racing wheel with me for a while and thought it would be really cool to try and show it on screen for things like streaming. I've already done it before for gamepads but I had never tried it before with a racing wheel.

What it does

The Racing Wheel Display shows the user inputs of the active racing wheel on screen via the web browser

How I built it

I used HTML, CSS, JS, and the Gamepad API to create this tool. I also used Adobe illustrator to create the graphics required.

What's next for Racing Wheel Display

Next is to try and polish up what I wrote and created so I can implement it fully into my other application which lets users display their gamepads on screen.
