Inspiration- The initial project idea was based on slight annoyances within our lives, mainly the ordeal of having to search up ETS schedules each time needed to use transit.

What we does

Although different from our original idea we developed an app which could identify your exact location and display random locations within the world.

Location

Besides workspace 1 entrance.

What this falls under

Android apps, built using android studio.

How we built it

We used android studio to build the app.

Challenges we ran into

Merging conflicts, using google API keys, and user input from EditText.

Accomplishments that we are proud of

Learning how to make GUI's or splash screens for the app, developing an app that worked.

What we learned

Learned to use GitHub, and basic actions within android studio.

What's next for OctoMap

If we continue on this project we'd like to finish the initial idea which consisted of an app able to design a transit route for each event in your calendar. Example: Violin practice at 5 PM, the app would display a notification at the time of departure and give directions for transit/car (based on user choice) eventually leading them to the session. This eliminates the need to search up transit schedules on ETS Trip Planner.
