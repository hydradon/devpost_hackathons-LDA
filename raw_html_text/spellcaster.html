• Spellcaster uses magic to help kids learn how to spell hard words!

Inspiration

The inspiration for this skill was my 10-year old daughter. We brainstormed how Alexa could help her and she came up with the idea to create a skill to help her learn how to spell hard words. We decided to name the skill "Spellcaster" because artificial intelligence is like magic!

What it does

Kids often have trouble spelling hard words. Spellcaster can help kids learn how to spell any word they don't know how to spell. Kids simply ask Alexa how to spell a word and then she spells it for them.

How I built it

I built this skill using a Lambda written in Python.

Challenges I ran into

I am new to Python so the syntax of the language was a bit challenging.

Accomplishments that I'm proud of

I am proud that my daughter and I partnered together to create a skill. She came up with the idea and then I implemented it. She also helped a lot with testing and identifying bugs.

What I learned

I learned a lot about the Python string functions.

What's next for Spellcaster

My daughter will present Spellcaster at her school with the intent that kids will use it to help them learn.
