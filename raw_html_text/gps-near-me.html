We wanted to create a project that fit the theme and was actually useful

It has a list of locations in Harrisburg which fit the theme of the hackathon and helps you locate them

We build it with node.js, leaflet.js, yelp-fusion.js, and html. It was hosted on repl.it with an express server.

We had to find good APIS and wrappers for them

Yelp needs better documentation and yelp.js is bad.
