Inspiration

Using friendly competition is an excellent way to motivate people. We wanted to use this to encourage people to help the environment in whatever way they can, even if it is just in small ways.

What it does

This app allows you to make posts of something you've done to help the environment. You can also view other people's posts and rate how green you think they're being. The higher ratings you get, the more your own personal tree will grow.

How we built it

We used android studio an firebase to build this application.

Challenges we ran into

Originally we were going to use google cloud datastore but we ran into issues connecting it with android studio.

Accomplishments that we're proud of

Coding for 15 hours straight during the first portion of the hackathon.

What we learned

A better understanding of how android studio works and how apps communicate with online databases.

What's next for Social Tree

Adding more features to improve user experience.
