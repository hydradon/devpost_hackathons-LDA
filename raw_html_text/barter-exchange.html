Inspiration

We were inspired by the idea of a Green initiative. We tried to think of a way to recycle that was a little outside the box, so we came up with a barter system.

What it does

Our app will allow users to connect with other users by swiping through a database of item submitted by all users. Once a user chooses an item, the selected item's owner will get a notification, and they can then verify what items the other user posted, in order to initiate an exchange.

How we built it

We build this app using the MEAN stack.

Challenges we ran into

We struggled with setting up Google Cloud and Google Vision, we had issues with some routes with mongodb, and something minor issues with Front End Design.

Accomplishments that we're proud of

We are proud of the idea we came up with, having accomplished using the MEAN stack, and the team work we had within the team.

What we learned

We learned about the MEAN stack in more detail, we learned about how to set up the Google Cloud Server, backend routes, and how to better design an Application

What's next for Barter Exchange

We only have the basic idea and barebones of the project. We would need to further embellish the design of the application, add more functionality that for now is only basic, and also apply more Artificial Intelligence, and Machine Learning libraries to improve the Application.
