Inspiration

We were inspired to create this app, because we have a pasion for basketball, and enjoy watching the NBA games. Our app allows us to get our information quickly and efficiently while displaying lots of information

What it does

Ou app is a command line interface that returns the standings, boxscores, live scores and upcoming games in the nba.

How we built it

We built our app using python with cmd and made our website with html

Challenges we ran into

We found parsing to be a very difficult and tedious task but by making good use of for loops we were able to successfully parse the various strings

Accomplishments that we're proud of

We are proud that we were actually able to finish a very big project at a hackathon

What we learned

We learned how to code a command line interface. I(Kushaun) also learned Python which I had never coded in previously

What's next for NBA CLI

We hope to expand our CLI to operate not just for the NBA but also for other sports leagues like the NFL, NHL, etc...
