What it does

Caramelcaramelcaramel is a chatbot that allows user to search amazon products inside the facebook messenger. Caramel, can also notify you when your item price drops which allows you to save more money.

How I built it

General Architecture

Features

  1. Onboarding, getting started shows how to search etc.
  2. Search amazon products
  3. Tracks items on watch list prices
  4. Notifies the user when price drops happens
  5. Image recognition - search products using image
  6. Speech recognition - search products using speech
  7. Share to friends
  8. Link directly to the item for purchase
