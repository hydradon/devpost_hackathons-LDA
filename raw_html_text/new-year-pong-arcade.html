Inspiration

We wanted to make a social game for friends to play together on the same screen.

What it does

You can play a 4 player pong on a web browser using your smartphone as a controller.

How we built it

We used unity to create the game and WebRTC for peer-to-peer connection between each player.

Challenges we ran into

Difficulty getting the peer-to-peer connection to function reliably and learning Unity.

Accomplishments that we're proud of

Multiplayer feature. Smartphone accelerometer.

What I learned

Unity, WebRTC

What's next for New Year Pong Arcade

A more polished game
