Inspiration

Being able to allow academic success to incoming freshman.

What it does

This App pulls the user's college courses and prioritizes the task by the weight of the assignment, difficultly of the assignment, and the time left to complete it.

How we built it

We used Google's Flutter app design for quick production and used Canvas's Web API to pull the users information

Challenges we ran into.

The biggest challenge we ran into was learning the new language that Flutter needs to be written in. We also had trouble initially setting up the project do to the different operating systems for each team member.

Accomplishments that we're proud of

We are happy to have the interface to work and the information of the user being displayed in prioritized order.

What we learned

We now know how to develop an App quickly.

What's next for College Task Manager

We would like to improve on it for the future because we see potential for many freshman to use it.
