Inspiration

The convergence of new technologies have finally allowed for a decentralized and permissionless peer to peer high touch health care system.

Physicians worldwide are frustrated by the lack of direct contact with patients, the growing number of middlemen, and patients are asking for cheaper solutions.

What it does

A peer to peer direct platform between physicians and patients. A borderless payment system through the blockchain to reduce unnecessary overhead.

The state of healthcare is in crisis. E-commerce serves as a solution for the bog-down caused by major corporations. H-mobile is the uncarrier that cuts the middle man out and returns the money back to the patients

How we built it

Web based audio/video chat windows built using react js and socket.io. WebAPI built using C#. ASP .NET core, hosted as PaaS in Azure. SQL Azure database save the transactions and maintain doctor and patient keys
