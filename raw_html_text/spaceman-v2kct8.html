Inspiration

I loved video games ever since a child and I believe people would want something as such for the Hackathon.

What it does

It starts the player off in a top-down arcade driven shooter with effects, music, and a score system!

How I built it

By using the Unity game engine, music that I enjoyed, sound effects I created in FL Studio, Scripts I made in C#, and my perception as a gamer!

Challenges I ran into

Mostly run-time errors that would make the entire Unity program crash. I didn't remember to save much so I would commonly lose five to ten minutes of work 2-3 times along the way. :(

Accomplishments that I'm proud of

The game has some optimization for memory space, as it deletes specific objects over time, The game's art is fantastic, This was made from scratch Unity with almost no game plan the day before, and the game runs smoothly.

What I learned

C is a useful programming language that made me consider how Java, the language I'm most familiar with, may still have some drawbacks in 2019 due to the virtual machine concept they implement rather than talking to the computer. Also that the life of a Software Developer isn't as crazy as I thought it was and I'm very comfortable coding for a 24 hour crunch time.

What's next for SpaceMan

Online leader-boards, difficulty, character customization, levels with different backgrounds, more animations, custom Sprites, I might push to Steam!
