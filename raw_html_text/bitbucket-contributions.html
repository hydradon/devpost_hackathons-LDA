Inspiration

To view the overall activity of a project there are no add-ons. The existing add-ons either are too expensive or show textual information. Bitbucket-contributions shows the overall commit activity of a project and also activity of all the contributors in beautiful graphs.

What it does

Bitbucket-contributions indexes all the commits of a Bitbucket repository and extracts information of overall commit history, individual contributor's commit history and displays it in beautiful graphs. This add-on can be used on any public repository.
