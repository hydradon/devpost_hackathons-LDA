Inspiration

Lots of people hear about storing info on a blockchain but can't use it because of it complicated. We democratize it.

What it does

https://github.com/Startup-In-One-Day/save-on-chain

How we built it

react boiler plat on the frontend, it calls stripe API for payment, after the payment complete it sends verification and info to API (hosts as Heroku app) and there backend use appropriate function to write to Ethereum or EOS. After transaction completed - it redirects the user on a blockchain explorer.

Challenges we ran into

working with payment services is a pain in the ass. Both PayPal and stripe.

Accomplishments that we're proud of

it works both of Ethereum and EOS
