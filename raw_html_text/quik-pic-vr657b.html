Inspiration

The slower the internet connection, the greater the frustration. Web pages with high resolution images can take minutes to load.

What it does

Quik pic lowers the resolution of images on a web page, allowing for a significantly faster loading time.

How we built it

Using great collaboration and time management skills, four students were able to create this masterful extension in less than 36 hours at QHacks2017.

Accomplishments that we're proud of

Creating a functional google chrome extension, that can aid users globally.

What we learned

Chrome Extension development was a key learning point of this project.

What's next for Quik Pic

Revolutionizing the extension world.
