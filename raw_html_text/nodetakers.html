Inspiration

We were inspired by including people in the education process. At Concordia, we have designated notetakers that take notes in order to help people that can make it to class, that can't focus in class or that some sort of learning disability. We wanted to help those notetakers.

What it does

This application allows a user to interpret a picture of someones notes and communicate the notes to other users. The picture is taken with the use of a phone and is then posted on a web page. Many users can access this web page and edit it's contents alongside eachother.

How we built it

We built an Android app that takes pictures and communicates with Google Vision API to interpret the text in the picture. After filtering the data, we send the text that was found to solace pubsub+ messenger who informs everyone subscribed to it. This captured text is all visible in our web application which others users can edit alongside one another.
