Inspiration

My girlfriend was in Venice navigating the streets using Google maps - one huge problem she faced is that Google maps suggests the fastest routes but they aren't necessarily safe. We set out to tackle this problem and help recommend people safe walking routes.

What it does

We aim to help people find the safest walking routes by collecting data on nearby points and businesses. We score how safe routes are by checking points on the route and tracking if businesses are open, ratings, types of attractions nearby, etc.

How we built it

We used Flask with Google Maps API for the backend and reactjs for the front end.
