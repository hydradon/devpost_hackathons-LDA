Inspiration

We all had a diverse skillet which included programming, electronics, computer networking, robotics, and web development so we wanted to create a project that incorporated all of these things. Also, we wanted to help plants because they don't have legs ):

What it does

Photo-Rover is commanded by web server over wifi containing scripts which control the motor and photo-cell sensors.... with a plant on it. The plant's objective is to drive itself to the strongest light source near by.

How we built it

We used a raspberry-pi, arudiono-nano, motor controller, two motors, 4 photo-cell sensors, two battery packs, and a flask web server containing html, python, and java-script which communicates with Photo-Rover over wifi.

Challenges we ran into

Putting together all of this different technology was our biggest challenge. It was a big, fun puzzle. We ran into time constraints on finalizing the web server into a nice interactive website where we can see real-time data of the direction and light of the robot; also it would of gave us the option to control it manually.

Accomplishments that we're proud of

Photo-Rover works!!

What we learned

We learned how to build a robot :)

What's next for Photo-Rover

Making more improvements to the point where Photo-Rover can live all on it's own for ever... as long as light exists.
