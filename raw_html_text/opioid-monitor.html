Inspiration

How to integrate opioid use and economic data and visualize these trends over time?

What it does

Visualizes various data sources for opioid use and economic indicators to integrate these various sources of information in a cohesive manner.

How we built it

Uses a Google Trends API in Python and Tableau data visualization software

Challenges we ran into

Determining which economic indicators are the most relevant to the issue of opioid use

Accomplishments that we're proud of

Visualizing many data sources together and developing a more clear picture of how they relate to one another

What we learned

The opioid crisis is a complex issue with many relevant factors, which may be more fully understood by capturing trends over time and observing trends inherent in many sources of data.

What's next for Opioid Monitor

Integrating data for more states and gathering more data sources from Pennsylvania.
