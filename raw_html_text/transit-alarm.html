Inspiration

One of our team members was working this summer and used the metro for travel. He wanted to nap in the train but was afraid he would miss his stop.

What it does

It sets an alarm that will go off when you reach close to the stop that you have to get off at.

How I built it

We use android as our platform. We used the BART api to get details of subway stations in the bay area. and then we used a fake location app to test our app.

Challenges I ran into

We were all considerably new to both android studio and git. We ran into a lot of problems with both of these as we were learning our way around them.

Accomplishments that I'm proud of

I'm proud that I created a working application for android

What I learned

I learned about making apps for the android and became a bit more comfortable using git for my projects.

What's next for Transit Alarm

Right now, it only contains stations from the bay area, but we plan to add more locations and make it available in as many places as possible.
