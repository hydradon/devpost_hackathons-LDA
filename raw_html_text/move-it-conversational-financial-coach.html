Inspiration

Its difficult to engage users within an application and Notification Centers are sometimes useful but don't gather enough interaction. I was inspired to create this by FB Messenger the new conversational paradigm.

What it does

This system integrates within the TransAmerica application to offer advice and comments on a user's account. It may alert the user about upcoming charges so they are more aware of their financial decisions or let the user know about current expenditures.

It makes it easier for a customer to transfer money between accounts to hopefully build better money habits.

How I built it

Using HTML and NodeJS I built a simple conversational system that provides a conversational notification center and customer service platform.

What's next for Move It - Conversational Financial Coach

Next up I want to implement this application in a native system on both Android and iOS.
