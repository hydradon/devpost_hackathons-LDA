Inspiration

People don't have multiple workout playlists made beforehand that matches the intensity of their workout

What it does

It reads the heart rate through the Apple Watch and generates a music playlist that can match the intensity of your workout by playing a song with similar beats per minute

How we built it

We used Sketch app for the UI, Xcode for coding. Coding language used was Swift.

Challenges we ran into

Challenges included problems trying to use the Spotify API

Accomplishments that we're proud of

We were able to successfully create both a iPhone application and Apple Watch application

What we learned

We learn how to take a UI design made in Swift and how to add functionality to the design in Xcode

What's next for Sportify

Deployment on Apple App Store as well as Android Google Play Store
