ThereminD

ThereminD is a brain wave-controlled virtual theremin. It allows people without motor control, like quadriplegics, to play a theremin with their brain instead of moving their hands.

How we built it

We used the Muse SDK's muse-io CLI to send to a local server over OCS, and this ocs_muse_server to pull data into Matlab. From here, we generated music notes using some scripts available in our github. We developed scripts to smooth the data and threshold frequencies into discrete notes.
