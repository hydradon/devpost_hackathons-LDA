As college students, productivity plays a big role in our lives. We wanted to build a resource that brings together all the resources we use while studying.

Cool Dashboard is a webapp that features various plugins - including Spotify and Quizlet.

We used python, javascript, CSS, HTML and spotipy to build it.

Towards the end, we found out that spotipy didn't work as we expected it to. To overcome this challenge, we tweaked the objective of our project.

Learning how to code in javascript and python.

We hope to add more plugins and successfully complete the python integration moving forward.
