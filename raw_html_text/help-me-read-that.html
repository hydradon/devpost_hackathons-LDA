What it does

It reads text from a chosen or a taken picture and then extracts it. it can also read web-address and open it with browser.

How we built it

it wasn't easy at all, we've used visual studio universal windows platform, c# and XAML.

What we learned

What's next for "Help Me Read That"

next for "Help Me Read That" is speech recognition, spell check and we wanted to add project oxford's emotion api too.
