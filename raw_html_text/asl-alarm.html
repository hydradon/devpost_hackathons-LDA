Inspiration

Encourage people to learn American sign language. Additionally, help them wake up in the morning by engaging their brain.

What it does

It's an alarm clock that is disabled via the American sign language alphabet.

How we built it

We built custom image models on Clarifai. We're interacting with these models via Node.js hosted on StdLib. Our browser client interacts with our Node.js application on StdLib.

Challenges we ran into

The Xesto API failed us on Saturday at 4PM, and we had to modify our application to work with Clarifai's image recognition platform last minute. This changed out entire backend.

Accomplishments that we're proud of

We built this in record time.

What we learned

A lot of Node.js, javascript, JSON and StdLib, Clarifai.

What's next for ASL Alarm

Implementing the application on a mobile device.
