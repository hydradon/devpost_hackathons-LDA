Inspiration

We built NEYO because meetings were not meeting standards for efficiency and productivity in this fast-paced world.

What it does

Using Data insights to plan, organize, facilitate and create action items for meetings focusing on the productivity of team and organizations.

How we built it

Built using a graph database, machine learning, Microsoft cognitive service and pure passion.

Challenges we ran into

Big ambitions, limited timeframe.

Accomplishments that we're proud of

Attended all workshops, pulled an all-nighter and built a product.

What we learned

New skills and life insights from mentors.

What's next for Neyo

Go big, make it open source for more collaboration and ideas. We see potential and want to scale this solution into a valuable product that improves productivity.
