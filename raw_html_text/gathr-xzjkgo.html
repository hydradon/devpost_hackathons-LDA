Inspiration

As students, we are bombarded with event invites from Facebook, emails, and word-of-mouth with no platform that could easily accumulate all of this public data into one platform. That is gathr.

What it does

gathr queries event data from multiple sources to form one event map.

How we built it

gathr is built using Ruby-on-Rails.
