Inspiration

I want to 10x the amount of people who can easily build distributed applications. Requiring a talented Frontend Engineer for every new project is an obvious bottleneck. If 5-10 Frontend Engineers build a collection of reliable and composable React/Blockchain components, we can easily empower the next layer of distributed application builders.

What it does

Drag and drop distributed application builder - the Squarespace for dApps

How I built it

Javascript.

Challenges I ran into

Time. Sleep deprivation from several weeks of travel.

Accomplishments that I'm proud of

It's the first drag and drop editor for distributed applications (as far as I know) :)

What I learned

We need more solutions to bring decentralized to the people. Engineers can't have all the fun!

What's next for MVP - Mesh Viable Platform (Drag and Drop dApps)

Build the Byzantine Battalion of Engineers to help scale out the infrastructure, so it's as easy to build a distribute application, as it is to build a website on Squarespace.
