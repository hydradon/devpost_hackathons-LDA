Inspiration

Most college matching websites simply use SAT and GPA, while important, are not the only factors in the college admission process

How we built it

We web scrapped a bunch of data from websites of colleges and prep scholar

Challenges we ran into

Colleges really don't like it when you try to go to their website a bunch and, as it turns out, not all colleges format format the website the same. So the amount of data, quality, and location, all varied widely.

Accomplishments that we're proud of

Minimum Viable Presentation

What we learned

Filling Django databases with Fixtures how to get around web scrapping polices on websites

What's next for College Finder

Better data, better data analysis.
