Inspiration:

We try to build a productivity app add first with Android studio but unfortunately, we don't have enough experience and time for it yet. So we stop at the timer.

What it does:

It is a timer that user can set time as an alarm and the flower will grow during the time. When time is up the flower will stop growing and there will be some happy music come with it.

How we built it:

Using Unity gaming engine and C#.

Challenges we ran into:

We have lots of challenge when trying to put our work together with Unity feature since Unity is quite new for us.

Accomplishments that we're proud of:

We finally create a project that works at the end of the day although we have faced lots of struggle to get to known Unity.

What we learned:

How to work together on a project and some skill with Unity and C#.
