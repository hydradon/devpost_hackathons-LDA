T2P

Text to Image converter

Inspiration

Tom Scott's "The Bubble"

What it does

It takes a phrase entered by the user, and replaces all text nodes in containing that string with randomized images. The theme of the images can be changed within the settings. The extension is published on Web Store and users can easily install it with inline installation from our .tech website hosted on AWS.

How we built it

First, we focused on making the popup page, and getting each of the modes to alter the popup's style. Then we started working on the actual scripts for the extension. You know, the stuff that actually does something. After trying multiple methods, we finally found one that allowed us to find both the phrase within the webpage and to grab the font size of the original string before replacing it.

Challenges we ran into

So, so many. Trying to get the scripts to communicate with the webpage. Inserting div node within text node without breaking the structure or introducing potential security vulnerabilities.

Accomplishments that we're proud of

Getting this to work. Seriously. We started this project 12 hours in.

What we learned

How to make a extension. Also, how to beautiful rainbow text with CSS.

What's next for T2P

We could allow users to create their own themes for the extension, and share those themes online.
