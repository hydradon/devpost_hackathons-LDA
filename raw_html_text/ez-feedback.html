Inspiration

Companies are always looking to survey their users; the marketing data they receive can be worth millions in profit! However, most of us see surveys as intrusive. We click random knobs on forms, or even flat out ignore forms altogether. Anthem´s musings about their low company response rate inspired us to create a better service that both the customer and company are happy with.

What it does

We have created a chrome extension that surveys users who have used a certain feature of a web application. Currently, users can respond via emoji. Companies can view responses to a given survey.

How we built it

Python/Django backend, Javascript, HTML, CSS, Chrome API

Challenges we ran into

Many of our decisions for the chrome extension could be more customizable in practice, but for the sake of time we could not fully implement customizability.

Accomplishments that we're proud of

Nikhil: learned a lot of the web stack from zero prior knowledge

What we learned

Teamwork, web design

What's next for EZ Feedback

We hope to continue working on unfinished features so that people will use it
