When I learned that APL lately introduced the support for video streaming, I excitedly started working on this second entry for the Alexa Modal Skill Hackathon.

This project brings a skill what every Alexa user wishes to have in their Alexa display device . Imagine watching on a big Alexa Fire TV, web cams located around the world capturing every moment from popular places like Niagara Falls, World Trade Center, Times Square, Las Vegas, etc.

With the experience I already earned from the earlier submitted APL skill, embedding the APL JSON objects into C# classes makes it easy to use the video capability in this skill.

Developing a multi modal application requires not only coding skills but also creativity in designing user interface. I am not a UI guy but it is now so easy to design display interface using APL.

With APL, you can now create more flexible, customizable, richer and reusable display templates that will make the Alexa users engage more time with the skill.

I cant wait to code more challenging skills as HTML rendering will also be supported by APL in the future .
