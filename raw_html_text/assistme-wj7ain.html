• AssistMe Website

Inspiration

We sought out to take JP Morgan's challenge and looked forward to developing something that would help people with and without disabilities. As we spent several minutes brainstorming ideas on what we could develop we came up with the idea to create something like Uber Eats and Wave which are community-driven services that help those in specific services such as deliver food or deliver to a location, but we wanted to be more open-minded and this is for people with disabilities that may need help.

What it does

Our application allows users to create posts that will appear for other users and these posts will consist of asking for any assistance needed.

How we built it

We used Python and flask along with CSS and HTML.

Challenges we ran into

We ran into the issue of how we would connect the web app and make it into an application, creating a way to post updates/content to the main 'Assist Offers' screen.

Accomplishments that we're proud of

We managed to create our web app in python when it was a first-time project.

What we learned

Web development in python, how to use Firebase, and working together in a team

What's next for AssistMe

Finalize a complete mobile application as well as integrating an incentive system to make it more tempting for people to join.
