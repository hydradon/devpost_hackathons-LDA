SPIDER is a robot designed to aid in disaster relief. Automatically paths around searching for survivors and uses face recognition to relay information back to a command center. Can also enter manual mode remotely for an operator to take control. The SPIDER allows rescuers to stay out of dangerous areas after a disaster while still searching for survivors.

Made with a Raspberry Pi. Designed in Python and MATLAB. The SPIDER uses ultrasonic sensors for navigation and a Neopixel aided USB webcam is used for facial recognition. An Xbox controller can be used remotely to control the robot. We used OPENCV and multithreading libraries in Python as well as MATLAB for an overarching master control server on a remote machine. We also developed a custom transmission protocol between the server and Raspberry Pi. The Raspberry Pi onboard the interfaces directly to the robot's movement controller, which is an Arbotix M Arduino.
