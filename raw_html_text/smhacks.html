What is Agora?

We noticed that on a daily basis, we find ourselves drowning in an endless sea of messages, trying to get the answer to that one problem that makes 0 sense. Agora aims to change this by creating a streamlined method of communication, and an easy way to check your work. The platform allows users to create and join classes, where they can upload problems with hints, and use the answer checker to check their work.

We used:

  • python (flask)
  • wolfram alpha api
  • html and css (with bootstrap and top hat themes)
  • postgresql (hosted on heroku)
  • lots of coffee
