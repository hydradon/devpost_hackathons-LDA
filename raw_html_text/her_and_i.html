Inspiration

Unable to work with the coding languages for the other challenges we put together this based on our experience with JavaScript

What it does

It's a short game with two characters that the player controls at the same time. one of the characters controls are inverted and the goal is to get as many hearts as possible.

How we built it

using the phaser.js framework for game making in JavaScript.

Challenges we ran into

controlling the attributes of the objects was a challenge that we were able to overcome.

Accomplishments that we're proud of

The implementation of a minimalist style that is present in the game.

What we learned

Don't quit on your code so easily, allow yourself to rethink, relook and see what you did wrong before thinking it won't work. We learned different state switches and framework code orientated efficiency.

What's next for Her_and_I

More user drives towards the goal of keeping both, blu and pink, alive.
