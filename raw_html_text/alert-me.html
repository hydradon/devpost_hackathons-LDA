Inspiration

In recent events, emergency personnels had trouble with getting in touch with the people living in a specific area. So, we want to come up with a solution to help notify people as quick as possible.

What it does

Alert Me is an app that sends notifications to users within a specific region about possible preparation for evaluation in the event of an emergency. First, the user can join the app by providing information like name, email, location and phone number. In case of emergency, the admin can notify people in a selected region on the map about the news in the area. When the user is notified, the user will have option to contact any volunteers for assistance.

How we built it

We used Amazon Web Services to run the website. Setup a database using PostgreSQL to store the users information. Used google maps API to allow users to pin point their location and allows the system admin to select a region.

Challenges we ran into

The main challenge we ran into is that we were unable to connect the front-end development to the back-end development.

Accomplishments that we're proud of

My team is proud that we were able to create this project with almost zero knowledge on cloud. Also, we learned how to deploy websites on Firebase.

What we learned

My team got a better understanding of cloud services and the idea of serverless.

What's next for Alert Me

The next for Alert Me is for my team to figure out a way to connect both front-end and back-end.
