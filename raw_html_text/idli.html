Inspiration

We all have different, not ideal ways of marking our tasks, from Google Keep to a paper planner. We imagined a way to visualize your tasks in a logical way, so we tried to build that.

What it does

You can add tasks, with features like notes and urgency. Then, you can see it on a calendar view, stacked up with all your other tasks.

How we built it

We used Android Studio.

Challenges we ran into

2/3 of us were new to Android, so we had to learn the basics first! Creating the calendar view turned out to be very challenging.

Accomplishments that we're proud of

We were able to get a button working that lets you choose a date using a pop-up calendar.

What we learned

We learned a lot about Android structure, xml, and Java!

What's next for tempus

We want to finish the Calendar view first, then add in features like intelligent ordering of tasks and offering user-specified color categories.
