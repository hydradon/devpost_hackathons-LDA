Inspiration

Girls with eating disorders around the world struggle with a healthy diet and need inspiration.

What it does

Bloom encourages and empowers girls to live a healthy lifestyle. It shows statistics for weight and counts calories.

How I built it

We used xcode and swift to produce our app.

Challenges I ran into

Being new to the language, we had to overcome a lot of bugs and had difficult adapting to a new challenge

Accomplishments that I'm proud of

For the first time using the language, we were able to implement a lot of features and create our first app. We turned ideas into reality.

What I learned

We learned how to pitch an idea and the basic functionalities of swift and xCode.

What's next for Bloom

We want to continue working on it and implement many new features. We hope to improve it and see where it goes from there.
