TerribleHacks2019

An "Anti-Tilting" tool to prevent you from getting too angry when you're on a losing streak. Anti-tilter plays encouraging slogans every few minutes, depending on your selected skill level.

Provides the anti-tilter (Windows Console .NET app), and a Settings UI (Windows Forms .NET app). The Settings UI is also the launcher!

To run: Run settings.exe in Settings/bin/Debug
