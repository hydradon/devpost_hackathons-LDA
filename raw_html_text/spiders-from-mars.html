What it does

This Chrome Extension integrates Expedia's API with Facebook Events. We take data from Expedia's API and create a drop down menu that gives you prices for the best package deal.

How we built it

We injected html into the Facebook page using Javascript and a chrome extension.

Challenges we ran into

We ran into problems when attempting to get the users location so that we could use the Expedia API.

Accomplishments that we're proud of

We had no knowledge of Javascript and we were able to create a program in 36 hours.

What we learned

Lot's of Javasript and HTML

What's next for Facebook Travel Extension

We need to find a way to get user's location and communicate with the Expedia API better.
