Inspiration

Currently there is no application or web that helps to rate a cover album in a fast manner.

What it does

A multi-platform product that analyzes uploaded album covers and lyrics and detect inappropriate contents. Good for record label producers to self-rate their albums before publication, also good for parents.

How I built it

Watson Visual Recognition for image content detection, Watson Tone Analyzer for lyrics analyzation, Node-RED and Loopback/Express/BlueMix Node.JS SDK for API hosting, BlueMix .Net Core for web hosting, Xamarin/Visual Studio for iOS, Android and ASP.NET development.

Challenges I ran into

We haven't worked with any of these technologies before.

Accomplishments that I'm proud of

We worked as a team obtaining tangible results such as deployed our application in different platform in a short period of time.

What I learned

All is going to be new.

What's next for musiClasifica

Create an application multi-language so we can rate lyrics in more than 1 language.
