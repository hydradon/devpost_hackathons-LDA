Inspiration

When your life gets hectic with errands and tasks piling up, good organization is the key to saving time, money, and gas. Our idea was based around the Travelling Salesman Problem.

What it does

Our app lets users input multiple locations and returns the most efficient path for them to take.

How we built it

We built Traffigo with Xcode, apple maps API and Swift.

Challenges we ran into

A consistent challenge we ran into was learning how to use Xcode as most tutorials were either too old or the programming language was Objective-C instead of Swift.

Accomplishments that we're proud of

Successfully implementing the map API, in addition to the use of changing views.

What we learned

We discovered that learning a new language during a hackathon, while fun, is quite difficult. We also learned that communication is important to prevent team members from repeating the same tasks.

What's next for Traffigo

Transit applications as well as traffic predictions with alternate routes.
