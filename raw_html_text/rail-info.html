• Rail Info

Inspiration

With railways used as a major mode of transportation in India, a skill to give rail info was a must.

What it does

Gives the list of cancelled/rescheduled trains and PNR status as well

How I built it

Built using NodeJs and PHP for backend APIs. Dependent on a third party service provider for rail info.

Challenges I ran into

Getting API access to the third party API platform for the rail information.

Accomplishments that I'm proud of

Completion of phase 1 of the skill in minimum time. Excited for phase 2 to bring in more enhancements.

What I learned

With voice, filling the gap of getting information on the go with voice commands.

What's next for Rail Info

  • Seat availability
  • Train Status
  • Trains running between stations
  • Extend the skill for more locations
