Inspiration

Cheaper learning options for students and professors

What it does

Bookerang is a question/ answer poll that tallies up responses from students and helps instructors keep track of students' progress. It is designed for mobile use.

How we built it

We used Django framework with SQLite database and used HTML and CSS in improve user interface

Challenges we ran into

Working with foreign framework and databases for the first time Handling static text in Django framework

Accomplishments that we're proud of

As beginner coders finishing our first mini project

What we learned

Working with a new framework Using command lines and terminal Static Text to customize Django

What's next for Bookerang

An app to bring students who want to swap textbooks together Improvements to site with a bigger database and hosting on a server
