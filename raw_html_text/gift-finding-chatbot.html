Inspiration

To help users find the perfect gift for that special someone

What it does

  1. It's a chatbot to help users find gifts that are relevant to the recipient in an easy manner

How I built it

  1. Gather product data
  2. Clean and organize the product features
  3. Build product vector feature table
  4. Build the chatbot to gather the features from the user and use it to query the feature table to recommend products

Challenges I ran into

  1. Building a good chatbot takes time to make it robust to user requests

Accomplishments that I'm proud of

  2. I realize I can scale the table to build better feature table to recommend better products
