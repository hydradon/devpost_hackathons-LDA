Inspiration

Nyan vs Doge draws inspiration from the game bomberman. It combines two of our most favorite memes into an improved version of a classical game.

What it does

The game allows two players to play against one another (i.e nyan cat vs doge) with mechanism similar to that of the (in)famous bomberman.

How we built it

We used objectdraw to create the game with Java, using arrays to represent the field and player/obstacle/bomb.

Challenges we ran into

Since we use objectdraw library, we have very limited options. We have to resolve problems on a very conceptual level.

Accomplishments that we're proud of

We managed to figure out the proper algorithm of the game and add a cute special effect of the bombing! Also improve the graphics rendered by photoshop.

What we learned

We learned that coding with Java is exceptionally hard but user friendly and requires a decent amount of logic.

What's next for [Beginner] Nyan-vs-Doge-Hachathon-2016-Spring

We hope to create an improved version of our game. (i.e: updating bomb upgrades and all the cool stuffs)!
