I made this app in order to be helpful using ARKit and giving the best user experience with the power of Augmented Reality

What it does

An Augmented Reality Pong Game that its goal is to give a one of a kind experience to its user by going against a computer and not allowing the ball pass the the user's paddle by having to run side to side horizontally. This ball's speed will be faster as time goes by.

How we built it

It is built with ARKit. A new library by one of the newest language out there, which is Swift. It consist of a stage, 2 paddles, a ball, and walls

Challenges we ran into

Most of the challenge that we ran into is setting up the environment and making sure there is some kind of randomness on the ball's movement which will make it challenging for the user just as much as for the computer, kind of
