Inspiration

The Miiverse sketch feature from Game & Wario allowed you to post pictures based on a prompt, but we thought it would be just as fun to be able to guess from the drawings of others.

What it does

Shows random recent sketches from random Miiverse communities. The player can use his/her phone to guess the game the sketches refer to.

What's next for Miiverse Pictionary

We have created a solid foundation for the program to later develop into a more full-featured game.
