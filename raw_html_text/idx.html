Inspiration

We were inspired by cryptocurrency digital wallets which use QR codes to share unique wallet addresses and weChat, which uses QR codes for payment processing,

What it does

IDX is the world’s first QR based digital identification system. It allows users to safely store and display personal information through an unique, encrypted QR Code (QRID). With three levels of security, IDX enables users to selectively choose what information is shown, when and to whom, allowing ease of use without compromising user privacy.

How we built it

We created a website using html and css and a video intro using adobe aftereffect. We also used google slides for a power point presentation.

Challenges we ran into

Accomplishments that we're proud of

What we learned

Lots of html, css and even some python. Info about cryptography, encryption and blockchain
