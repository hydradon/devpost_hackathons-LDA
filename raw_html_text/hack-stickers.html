Inspiration

After 24hs at #HackGT, I found myself collecting stickers. Then, I found out that GitHub had more designs, and I wondered why I couldn't get it in my phone. So I decided to hack it out and find out the way of doing it.

What it does

This open-source project allows us to send GitHub stickers to our friends. Now we can share in paper and electronically.

How we built it

Using xcode and GitHub open source.

Challenges we ran into

The challenge was that we didn't know where or how to start, but YouTube was a great help and we were able to make this work.

What we learned

That curiosity can achieve great things when it comes out way, and that we can get something done in less than 40 minutes.

Scalability

It would be great if each hackathon has a package of stickers available, including sponsors' stickers and the event's own designs.
