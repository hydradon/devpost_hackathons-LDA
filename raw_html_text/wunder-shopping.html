Inspiration

Many people use Wunderlist for their shopping lists, so they may not want to use the built-in Alexa shopping list for that. Hundreds of requests in the relevant Alexa and Wunderlist forums have shown that there is a significant demand.

What it does

You can add items to your shopping list via Alexa.

How I built it

We used Kotlin and run everything on AWS Lambda

Challenges I ran into

Mostly the account linking process via OAuth that is incompatible between Alexa and Wunderlist.

Accomplishments that I'm proud of

Getting the account linking process right.

What I learned

How to use AWS Lambda; how to use voice recognition APIs.

What's next for Wunder Shopping

More features like deleting items, reading lists, have custom lists, manage generic lists (not only shopping)
