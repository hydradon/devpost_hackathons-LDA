Inspiration

Current data is convoluted and overwhelming.

What it does

Allows experts to create interactive documents, and individuals to find the documents, by category.

How I built it

We used an MVC php framework, for the structure. We used CKEditor for a base of the expert editor, and built a plugin that allows experts to create interactive documents.

Challenges I ran into

Accomplishments that I'm proud of

  1. Got the interactive equation editor to parse equations dynamically, and it is simple to use. ## What I learned

How to build a CKEditor plugin.
