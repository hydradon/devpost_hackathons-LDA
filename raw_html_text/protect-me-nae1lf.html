Inspiration

Hemos decidido hacer este proyecto porque creemos que el acoso escolar es un problema grave al que hay que ponerle remedio.

What it does

Proteger a los alumnos y que dejen de sentirse vulnerables.

How we built it

Ionic para el responsive y que sea multiplataforma, firebase para la base de datos en tiempo real y poder enviar notificaciones push, Api de google para geolocalización, Backend DJango que utiliza TensorFlow para machine learning.

Challenges we ran into

Decidir el diseño más adecuado para ofrecer un estado de confort y relajar al alumno en este tipo de situaciones.

Accomplishments that we're proud of

Realizar un proyecto de acción social para ayudar a las personas.

What we learned

Plantear la difusión de este tipo de App's. Centrándonos en la importancia del desarrollo de una idea y del diseño en el proyecto, alejándonos un poco más de la parte técnica, a la que estamos más acostumbrados.

What's next for Protect.me

Predicción de lugares y horarios con mayor acoso y patrones para automatizar el aviso a emergencias.
