Inspiration

Make School

What it does

Matches phones

How I built it

With the help of Make School instructors and using swift and phone database, I was able to build my phone matching app, Fonomatch

Challenges I ran into

Learning how to use the API Auto Layout Submitting to the App Store (Unfortunately, I will not be able to submit my complete app to the app store due to an issue with the App Store's policies)

Accomplishments that I'm proud of

Learning how to use XCode and build projects Being able to build my own project

What I learned

You definitely have to have patience

What's next for Fonomatch

Due to the App Store's policies, I have to remove a lot of features and I have to implement new features such as expanding the devices to Ipads, Smart watches, etc. I will also try to make it customized for users so that they are able to save phones they want into their own separate lists.
