Inspiration

We believe the freedom of movement is a fundamental human right. But, how many times has a visa process intimated you or a loved one?

What it does

It helps you streamline the visa process between you and decision-making stakeholders in an immutable, timely manner.

How we built it

Through Microsoft Azure, we seamlessly deploy smart contracts to create immutable identity authentication between immigration stakeholders, such as local governments/schools/public notaries/consulates, to communicate in an easy, linear decision-making process.

Challenges we ran into

We ran into the following challenges:

  1. We wanted to figure out a way to integrate biometric data to enhance authentication features that would compliment our existing structure.
  2. We wanted more time to push code to find malicious patterns for moderation.
  3. On a team level, we ran into motivational roadblocks.

What we learned

While the platform roadmap is sketched out, key partnerships trust is going to be integral for the success of our deployment.

What's next for Move

  1. Keep asking, "What can we do next?"
  2. Identifying key decision-makers for the early-adoption phase.
  3. Improving scalability through a viable business model.
