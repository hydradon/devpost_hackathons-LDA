Inspiration

We all had events on our calendar that affected other events and we wanted to find a way to optimize this.

What it does

It gives users the ability to order events based on significance, mental/physical fatigue, as well as indoor/outdoor attributes to each event.

How we built it

We used javascript, multiple calls to APIs, HTML, and CSS to build the project.

Challenges we ran into

We ran in to many challenges while trying to learn how to use these APIs.

Accomplishments that we're proud of

We are proud that we got a basic prototype working for the project as well as got API calls to work as well. We were able to build a basic Calendar application in just a couple days in a language that we previously do not have experience in.

What we learned

We learned how to use javascript, API calls, and HTML/CSS in our project.

What's next for Calendar Buddy

We want to work on expanding the project in the next couple week and release our idea to the public.
