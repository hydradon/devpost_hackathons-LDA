Inspiration

Help high schools students find the perfect college using big data analytics. Most international students do not know much about US colleges when they planned to study abroad. This website hopefully will make it easier for everyone to see which college is the best for him/her

What it does

Users enter their information such as SAT scores, preferred locations, intended major, or favorite schools to receive suggestions

How I built it

Back-end was built with Python and Flask. Front-end was built with HTML/CSS The website was hosted using AWS

Challenges I ran into

Cannot think of a good yet still available domain

Accomplishments that I'm proud of

This is the first webapp that I ever deployed.

What I learned

Big data is powerful

What's next for ThePerfectCollege.Net

The dataset still has lots and lots of features that I have not used. They can all be added to the webapp to personalize the suggestions
