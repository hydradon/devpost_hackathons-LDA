• Local Mocos

Inspiration

Helping Hawaii's economy and to help local businesses.

What it does

Businesses can search for local vendors as well as 100% locally grown or value-added products with at least 50% of inputs from Hawaii.

How we built it

We created a server with Express.js and the vendors and products database with PostgreSQL. We connected the data to the front end using Pug and styled with Sass.

Challenges we ran into

Creating a user friendly website that meets the needs of the state and the user experience.

Accomplishments that we're proud of

We created a user friendly site!

What we learned

We learned about the importance of communication and working in a group. While working on this project we also learned about the problems that the state encounters in creating an app.

What's next for Made In Hawaii (by Local Mocos)

Creating an administration function to allow and admin to approve/delete/edit posts.
