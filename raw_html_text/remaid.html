Inspiration

In emergencies on time and accurate actions save lifes. The high resolution and real-time transmission which is possible through 5G makes accurate and detailed communication between first responders and emergency doctors.

What it does

It connects an emergency doctor with a first responder.

How we built it

We use agora for video transmission, unity for the VR scene and angular to create the app frame.

Challenges we ran into

Displaying 3D actions on a 2D picture, now we use a 3D Model.

Accomplishments that we're proud of

Working in the team works really good and everybody is very motivated.

What we learned

Combine and use the tools.

What's next for Remaid

Integrate several phones at the same time to create a real 3D model.
