Inspiration

Akhil's sister has many severe food allergies, so we were inspired to solve this issue for her and the millions of other people suffering from similar allergies.

How we built it

We built it using Flutter and Google Cloud Platform. We all worked on different components of the app, and then Jason combined all the parts into a finished app.

Challenges we ran into

We ran into many issues using GCP, since we hadn't used RESTful API in Flutter prior to this. We also had issues with Github, as some of our Android Studios did not recognize the repository as a Flutter project.

Accomplishments that we're proud of

We're proud of the entire application :)

What we learned

We learned much more about Flutter and also how to better implement RESTful API.
