Inspiration

We heard at HackUPC that a sponsor was looking for this scaled to a massive event like football stadium or concerts.

What it does

It syncs mobiles to work as a display matrix using the flash or the screen.

How I built it

Each mobile (or set of mobiles) represent a pixel of the display and asks the server for it's sequence of states and the time of start running.

Challenges I ran into

We got stucked in the part of getting the data from the server in a mobile with JSON over HTTP as we don't know very much (or anything) about Android programming.

Accomplishments that I'm proud of

We managed to implement a server that works and an Android app that turns on the light for the sequence programmed early by some other client.

What I learned

We learned to program in GO and some Android programming.

What's next for SNAGLIGOSYS

Maybe there's no future for this project. However, we can continue it and accomplish it very successfully if asked.
