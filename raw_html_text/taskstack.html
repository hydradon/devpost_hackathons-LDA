Inspiration

I've been interested in decentralized apps for a while. One day I thought that a great use case for decentralized technology would be a task tracking app, sort of like for example Trello or Kanbanflow. To my surprise, I only found one such app on Dapp.com. And unfortunately, that app did not work. Since I had not yet built a decentralized app, I thought, why not try to build a decentralized kanban board?

What it does

Taskstack is a simple, decentralized kanban board. I'm working on adding more advanced features like for example a Pomodoro timer, see the heading "What's next for Taskstack".

What I learned

I learned a lot about how Blockstack, and decentralized apps in general work. This was my first time developing a decentralized app, and I'm surprised by how much it's like developing a non-decentralized web app. In fact, I'd say it was easier than developing an app where I'd have to manage authentication and data storage myself.
