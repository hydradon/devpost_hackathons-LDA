Inspiration

Zecheng is interested in the stock market and Reddit has several news sources

What it does

Predicts tomorrow's stock prices from today's Reddit posts using sentiment analysis

How we built it

Pandas, Reddit API, paddlepaddle, Python, Numpy, Microsoft Cognitive Services

Challenges we ran into

Getting the data into the right format, sentiment analysis

Accomplishments that we're proud of

Data cleaning methods

What we learned

Cleaning the data and formatting it for ML is very time consuming

What's next for Stockit

Grab more posts from more sources for a wider ranch of companies, use a wider range of stock data and train the model and tune the hyperparameters
