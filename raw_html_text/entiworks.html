Inspiration

Data logging and retrieval, done on the fly to aid in the A/E/C process.

What it does

Entiworks software uses QR technology and analytics to ensure full project communication.

The system ties users, site equipment and utilities to a dynamic ID stored in our database.

This ID generates a unique QR code that is scanned to retrieve data and log events or approvals.

Contact information, approved documents and certifications are a few of the information streams we currently have integrated.

How we built it

We started with a UML diagram and an idea that QR can easily connect multiple users and contractors on a job site. We began modifying features based on job site use, like the ability for administrative users to update the live database with approved records or by adding entities that other users can access. Once we had steady data collection we built a web dashboard for analytics and report customization.

Challenges we ran into

Adjusting the user experience to be accessible for various skill sets.

Accomplishments that we're proud of

We our proud of our systems ability to collect and display data while staying practical in a A/E/C environment.

What we learned

The complexity in a Job site with multiple entities working together.
