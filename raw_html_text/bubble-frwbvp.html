Inspiration

We've noticed that sometimes friends go out to events and can't keep track of when they come and leave. This app will help take care of that by geofencing event areas, and making sure everyone gets home safe.

What it does

This application creates user groups with a geofenced event which will make sure that people don't leave unnoticed.

How we built it

Built it with xCode and Swift.
