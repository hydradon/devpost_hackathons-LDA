Inspiration

As telegram is a platform that many teenagers use these days, we decided to create a telegram bot that will help these teenagers. Taking inspiration from a korean chatting platform, Kakao Talk, we wanted to see how we can incorporate google searches into telegram such that the same platform can be used for multiple uses.

What it does

This is a telegram bot that allows the user to search for their query directly on telegram, instead of having to switch browsers to and fro. This efficient bot will automatically send a link of the search engines (either Google, Google Scholar or Yahoo) with the query that the user has typed in. However, Goodgle requires the user to type in either 'Google', 'Google Scholar' or 'Yahoo' first, before typing in whatever they are looking to find out about.

How I built it

Firstly, we made a telegrambot by using Botfather. From there our group was able to get our bot token.

After that, our group made use of the pyTelegramBotAPI framework to code the Telegram Bot through Python
