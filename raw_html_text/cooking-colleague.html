Inspiration

When cooking people often have to look back to the recipe and even flip through to follow along. Not only can this be frustrating, but there just has to be a better way.

What it does

Cooking Colleague enables users to search for and follow along with recipes via voice and text interaction.

How we built it

We used Python for server-side code that stores and sends voice recordings to be processed. The web client offers a view of the data and is how the users interact with the application.

Challenges we ran into

Integrating the voice processing logic with the web client was challenging.

Accomplishments that we're proud of

The application looks great and is practical.

What we learned

Integrating multiple services together is not always as easy as it would seem.

What's next for Cooking Colleague

More advanced filtering methods for recipes based on ingredients available, dietary restrictions, and a more performant client.
