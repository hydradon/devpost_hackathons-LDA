Inspiration

So me and my friends had difficulties waking up with a standard alarm clock, thus we built one where it helps stimulating the brain by making the users to solve some brainteasers.

What it does

It lets the user to set the time and alarm time, and rings alarm with math questions, which only turns off after the user solves the question.

How I built it

We built it with Arduino one with Grove board, Button, LED Display and Buzzer.

Challenges I ran into

We had some difficulties trying to find the right casing for the unit.

Accomplishments that I'm proud of

We finished our coding process in Saturday afternoon, thus having enough time to review our final product and have some good sleep!

What I learned

I learned that a simple idea (that Alex had) can turn into such a great product.

What's next for The Alarm Clock

Finding a nice packaging to make it looks like a genuine clock, also making a mobile app with similar function.
