Inspiration

I wanted to make a quick way to close out completed github issues quickly without any hassle.

What it does

It's a Pebble smartwatch app that connects to your GitHub account and displays a summary of issues assigned to you across all repositories and allows you to close any issue that is assigned to you.

How I built it

Using GitHub's API and Pebble.js as the platform for the app. Also Caffeine.

Challenges I ran into

Reading the Pebble.js Documentation

Accomplishments that I'm proud of

Getting to play around with GitHub's API, a lot of cool features in it and will definitely use again.

What I learned

GitHub's API.

What's next for GitHub Issue Manager

Adding Issue creation via voice to text capabilities, adding user settings (currently is only set up to my account).
