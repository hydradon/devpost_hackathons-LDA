Inspiration

What it does

Its a smart knob meant to replace currently existing knobs on an stove top.

How we built it

We used java to build an Android App and used Python to build the system for the servo motor that was the knob

Challenges we ran into

Connecting the servo motor to the board without an analog port

Accomplishments that we're proud of

The Android app and the realtime database that it is connected with

What we learned

How to design a system from scratch that would allow us to control IOT devices through the internet

What's next for Smart Knob

Advancing the android app and polishing the system on the dragonboard
