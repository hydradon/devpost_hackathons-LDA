Inspiration

Have you ever had a day, where you're waiting for hours and hours for a single queue! Welcome Queue Planner a smart scheduler that will automatically queue you!

What it does

It gives suggestions to your schedule and checks your schedule with other's bookings to give an optimal schedule for you. It then allows you to book your schedule with all the places mentioned.

How we built it

The web app and the server was built with HTML, CSS, JQuery, NodeJS, MySQL, and Javascript. We built the mobile app with Swift!.

Challenges I ran into

We had problems with the Booking Algorithm because the algorithm would have to take into account not just your own schedule, but the schedule of the location, and of other users.

Accomplishments that I'm proud of

We successfully made a small demo.

What I learned

We all learned how to do HTML, CSS, and Javascript to work on the server and web app! SQL is tiresome.

What's next for QueuePlanner

We would like to improve the queuing and booking system!
