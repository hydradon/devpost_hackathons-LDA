Inspiration

To see the power of machine learning for a real world application

What it does

How we built it

Backend: Java, SpringBoot Framework, Python, JDBC & PostgreSQL Data Gathering and Cleaning: Python Frontend: HTML5, CSS, Bootstrap, JavaScript, Ajax

We have successfully implemented a book recommendation system based on co-purchaseing pattern analysis using data mining algorithm like FP growth.
