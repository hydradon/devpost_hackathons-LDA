Inspiration

My mom and i love watching the game TV show Deal or No Deal. That's when something hit me, i love using my alexa echo show, why not build an APL just for the echo show device. Later design it for the big screen LG TV.

What it does

Experience first hand of the real life game as if you were in the game show.

How I built it

Using the Node.js and both Authoring tool and Simulator provided by Alexa.

Challenges I ran into

Noticed that both device rendering and simulator are not the same. found a lot of issue. However, i designed this skill based on the simulator and later was going to fix it so it works on the device as well. I wish i had more time did not get a chance to finish the BANKER's offer and daily winning by the user.

Accomplishments that I'm proud of

I was able to go through the entire APL understand all it's use case and somewhat create a small working deal or no deal device.

What I learned

How to render using APL. Also Javascript :)

What's next for Deal or no deal Game Show
