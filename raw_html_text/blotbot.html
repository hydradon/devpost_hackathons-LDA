Inspiration

Gamergate and TerrenceM. Twitch chat has gone too far!

What it does

Uses machine learning to help moderators scale to thousands of twitch users.

How we built it

We used python. Scikit-learn and nltk were great for the text mining. We used a free dataset from kaggle datasets with Dota 2 chat messages.

Challenges we ran into

Training the model to be accurate was challenging.

Accomplishments that we're proud of

Can learn from the behavior of moderators in real time.

What we learned

Detecting violence and hate is difficult with the dataset we found.

What's next for BlotBot

More feature engineering and perhaps a web UI for easier administration.
