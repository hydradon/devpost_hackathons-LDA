Inspiration

We got inspiration from a construct workshop we attended

What it does

So by left clicking, we can have luigi shoot fireballs at the goombas

How we built it

We mainly used the program we made in the construct workshop as a template and worked our way up from there. We added different pictures, new global variables, and new events.

Challenges we ran into

A challenge we ran into was figuring out how to end the game when the lives were equal to 0. To solve this, we added an event which would show that you'd lose the game if lives were equal to 0.

Accomplishments that we're proud of

We're proud of getting the goombas to respawn and to rotate after reaching the border of the game level.

What we learned

We learned how to set up a basic shooter game with lives and a score
