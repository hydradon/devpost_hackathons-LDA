Inspiration

We were inspired to create this by the Stay Inside and Play and the Gene Challenge. After we had started creating MiniGame Madness, we further got inspired by the creation of the Best Design Category. The game Chimp Chase was inspired by MailChimp.

What it does

MiniGame Madness is an app that hosts 3 different mini-games that the user can play. They are Chimp Chase, Labyrinth, and Dance Floor Dodge.

How we built it

We build MiniGame Madness using the Unity3D game engine and their built in collaboration tools.

Challenges we ran into

Some challenges we ran into was making scripts to teleport the players across the map for Labyrinth.

Accomplishments that we're proud of

We are really proud of the professional and creative design of all three games and menu screen.

What we learned

How to make a multigame application in one Unity Project and create a well designed UI and game interfaces.

What's next for MiniGame Madness

The next features to add to MiniGame Madness is to add a login feature and add the ability to track the user's highscores. Also we will want to continue to add well designed and creative games.
