Inspiration

We love memes, and we love our professors.

What it does

The user selects a professor from the thumbnails, and selects a meme scenario and then that professor's face is displayed on the meme reflecting the scenario.

How we built it

Using Java and the built in libraries.

Challenges we ran into

Navigating github and only one of our team members is mildly-versed in coding languages, which required a lot of learning on the go.

Accomplishments that we're proud of

3 team members learning Java syntax in one day Fixing github issues Walking away with a completed product, that we could be proud of

What we learned

Java syntax Git commands How to delegate tasks

What's next for MemeYourProfessor

A website that includes images for all professors at UNC, many more meme scenarios, and that all students have access to.
