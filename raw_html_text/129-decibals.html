Inspiration

We wanted to build the most ridiculous thing possible using a speech api and lo and behold, here comes Larry Pinhead! Our inspiration for our name is 129 decibels is the loudest recorded human scream ever.

What it does

129 Decibels is a client server based game where players can guide two shining fighters to victory using speech recognition software.

How we built it

We used the sphinx api for speech recognition and used java's built in libraries to develop servers, implement multithreading and create a game engine to run the game.

Challenges we ran into

We began using a different API that used an old Google speech API that didn't have much documentation and gave us a lot of problems. Eventually, we found sphinx which was able to accomplish what we were looking for.

Accomplishments that we're proud of

We're very proud that we were actually able to pull it off. We came in doubtful if we can do it, but are proud that we actually accomplished everything we sought out to do and more! Beyond the speech recognition, our game has solid animation sequences and is ridiculously fun!

What we learned

We learned that buff Patrick is a very hard puncher. We also learned how to work with speech recognition API's.

What's next for 129 Decibals

We are planning on including more kickass animations, new characters and hopefully functional applications using the speech recognition software, including learning speech or maybe a cappella training.
