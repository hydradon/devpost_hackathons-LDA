Inspiration

On big college campuses, it can be surprisingly difficult to make new friends or to feel community. We wanted to make a service/app that can help students who feel that way.

What it does

Dorm based chat app.

How we built it

LAMP for the server side of things, html,css,bootstrap for the front end. Used aws for hosting.

Challenges we ran into

Learning modern web development in 36 hours

Accomplishments that we're proud of

What we learned

Learned how to use aws, learned more about bootstrap.

What's next for DormSocialzr

Total recoding of the app. We plan to use a mean stack for a web app, and possibly plan to make an iOS and Android app. Will probably use firebase as the backend for all of those for the purpose of easy connectivity.
