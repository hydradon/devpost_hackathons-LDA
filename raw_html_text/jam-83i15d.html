Inspiration

With college graduation soon approaching, the job search has gotten more and more intense. It is never fun to forget what company the recruiter you're on the phone with works for. We built a tool to help with that.

What it does

JAM (Job Application Manager) tracks the progress of different job applications. It can tell you how much the position pays, where it is, the status of your application, etc; all on a visually stunning dashboard to allow for quick comparison.

How we built it

We built it using the Wix database, Wix Code and a Wix landing page. A reactjs dashboard and some great API's.

Challenges we ran into

Learning the Wix Code API request syntax, this is a tool we are using for the first time.

Accomplishments that we're proud of

Getting it all done on a two person team.

What we learned

The Wix suite and all the great tools they offer.

What's next for JAM
