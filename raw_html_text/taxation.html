I was fascinated by the amount of data that I have seen from TaxAct. I was inspired to work on this dataset and create an app which can serve the user to fetch data in different ways. Our app may help search for the people of different age group and can be extended to do the same for different other fields. I have implemented VueJs for the 1st time and I really like working on it. It was a great event and I look forward to working on it. One of the challenges faced was to serve the 'Cross Origin request' which I found a way to implement and undersatnd how it should be implemented in the future. The app will be used in future to analyse the approximate tax to be paid by different category of users.
