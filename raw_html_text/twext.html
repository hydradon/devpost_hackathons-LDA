Inspiration

We watch a lot of twitch.

What it does

Allows users to follow channels and receive texts when they're live.

How I built it

Node.JS server running on Azure checks all user's streams and sends out texts when they're live.

Challenges I ran into

Looping an asynchronous function call is really hard.

Accomplishments that I'm proud of

It actually works

What's next for twext

Twitch's API, while rather limited in terms of functionality, provides enough data for a more extensive set of features to be implemented. The data provided can be analyzed to produce more informative and detailed texts to better improve the experience for the user.
