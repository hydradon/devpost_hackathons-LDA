Inspiration

We wanted to create a funny game that had a Binghamton inspiration and a throwback to an old school game that everyone loved.

What it does

This text-based game takes you through a demo of what the full game will look like.

How we built it

We used java on android studio to build our application.

Challenges we ran into

None of our team members had used android studio before, and some of us had very little experience with git so we had a huge learning curve to deal with.

Accomplishments that we're proud of

We managed to successfully have a team git repository and we created a lot of the features that we wanted to have for the full version.

What we learned

We learned how to use android studio, and that we can not create a complicated game in only 24 hours.

What's next for BinghamtonTrail

We will continue to build out the game and hopefully create the full working version of it so people can play it on their phones.
