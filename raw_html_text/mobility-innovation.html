Inspiration

Wir wollten ein umfassendes Web-basiertes Analyse und Simulationstool erstellen, das leicht erweiterbar und skalierbar ist. Durch Änderung des Zielwerts ist es möglich schnell das Tool für andere Berechnungen zu verwenden.

What it does

Analysefunktion anhand vergangenheitsbezogener Werte in MS PowerBI online. Prognose und Simulation mittels Machine Learning in MS Azure ML Studio online als Webservice.

Daten können im Tool angepasst werden, sodass diese Daten dann mittels Machine Learning in Azure ML berechnet und zurück übertragen werden zur Visualisierung.

How I built it

Datensammlung von diversen Quellen:

  • Kraftfahrtbundesamt (KBA)
  • Goole Trends
  • Twitter
  • Facebook
  • Mobile.de
  • Zensus mit Fraunhofer Studie
  • Bundesnetzagentur (aktuelle Ladestationen)

Challenges I ran into

Zuordnung der Orte zu Regionalschlüsseln - Lösung durch Geocoding Ort -> Übersetzung auf Längen- und Breitengrad -> Suche, in welcher Region die Koordinaten sind
