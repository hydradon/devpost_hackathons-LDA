Inspiration

The continuous rise in loan sharks businesses prompted me to think of a better way for people to access to loans

What it does

Connects individual lenders and borrowers to make micro-loan transactions.

How I built it

We built the iOS application using Objective C and used various different frameworks for the web application.

Challenges I ran into

None. Managed to learn through them and got the systems to work

Accomplishments that I'm proud of

The application in general. The fact that we managed to program the application and implement the website in the given time constraint!

What I learned

The integration of different APIs and technologies deemed challenging at first but I learned quite a lot through the process of integrating them.

What's next for LoanMe

Finalizing it and publishing it on the App Store. Also working on ways to minimize risk.
