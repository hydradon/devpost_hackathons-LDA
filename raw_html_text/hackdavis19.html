Inspiration

Breast Cancer should be eradicated

What it does

Primary: Helps doctors get a fast and accurate prediction of whether a given growth is malignant. The full application will also allow doctors to determine remission rates, and help with other diseases.

How we built it

We used a FF ANN to do a binary-classification. We then used a simple web applet as the interface into the program.

Challenges we ran into

We ran out of time synchronizing the front end web app to the back end Python NN modules.

Accomplishments that we're proud of

We were able to complete all the pieces, like front-end interface, the back-end model training, and learned about google cloud platform.

What we learned

We learned how to use js and html more efficiently using bootstrap, and how to use GCP better in the future.
