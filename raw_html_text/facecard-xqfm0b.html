Inspiration

No one enjoys foregtting someone's name, so we built something to make sure that it never happens again.

What it does

With FaceCard, you can take a picture of anyone in the database, and it will remind you what their name is!

How we built it

We built this MEAN stack web app with Node, Express, a couple different Amazon Web Services API's, and a whole lotta memes

Challenges we ran into

Amazon Web Services has some of the least organized documentation I've ever seen.

Accomplishments that we're proud of

We made it!

What's next for FaceCard

We would really like to develop the web app to account for continuous recognition through video streams, and integrate the service with SnapChat and Facebook. Also, more memes!
