Inspiration

We wanted to make a fun close combat first-person game. Ninjas have the coolest close combat weapons!

What it does

It combines a fun multiplayer game with a fun silly close combat game. So you and your friends can be the best ninjas ever.

How we built it

We use a game engine called Unity3D and programmed it in c#

Challenges we ran into

Setting up networking and running specific actions were a major obstacle

Accomplishments that we're proud of

We are proud to have made a multiplayer game that allows the user to play with their friends or family

What we learned

We learned that there are many obstacles in making a game, but you must overcome these obstacles if you want to make a great game.

What's next for Kunyomi Ninja

We plan to continue the growth of the game and to continue making it better.
