• Roboy_with_Connect4_game

Roboy is a safe and interactive robot, but it needs more applications for public use

It allows the user to play a game of connect four with Roboy

We built a robot arm that executes Roboy's moves and Roboy interacts with the user based on speech recognition and speech generation

A lot of network bugs, when using the services, communication errors with the arduino

We are able to play a complete interactive game with Roboy, untill a winner is found

How to use the Rosservices running on Roboy, how to create a Servo motor control with ROS
