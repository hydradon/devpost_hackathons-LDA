Inspiration

We wanted a site to conduct longitudinal surveys on ourselves. We wanted to be able to visualize our data using a calendar heatmap - similar to github's contributions calendar. A platform for introspection and tracking personal questions directed to ourselves.

There are many things I find interesting to keep track of in my day to day life. For example: How long was my commute today? How many smiles did I give today? How many servings of meat did I eat today? How many times did I check Facebook today?

What it does

A site for asking yourself questions and tracking your answers to your questions.

Challenges we ran into

Logistics - organizing meetings Determining which UI libraries to use Deciding on what data models to adhere to Developing a calendar visualization

Accomplishments that we're proud of

We deployed a working site that looks cleaner than it was 2 months ago!

What we learned

passport.js, express, node how to deploy a site how to query mongodb from a browser or console how to set up a hosted mongodb cluster react router

What's next for did I do x today?

We were hoping to use mongodb charts to visualize user behavior. For example:

How many questions does each user keep track of? Statistical distributions of the questions that people ask themselves? How do people's questions evolve over time i.e. Does someone asking "how much chicken did I eat today" evolve into a vegetarian and ask "how many servings of vegetables did I eat today" or "how many calories am I getting today"?
