Inspiration

Groups, especially women, need to feel safe when going out. You shouldn't wander off drunk when you planned to stay together sober.

What it does

This app gives you the ability to track when members of your created group wander off and can notify when someone walks a certain distance away from the venue and can give you their GPS coordinates.

How we built it

We used a Node.js backend to ping the Mapquest API and update users locations into a Firebase hosted database. The app itself is used to create users and groups to match up with the data in Firebase.

Challenges we ran into

Firebase can be pretty vague with it's documentation and building an app from the ground up is difficult. Thankfully Mapquest has a very intuitive API that made the interaction relatively painless.

Accomplishments that we're proud of

Creating this app from the ground up in 8 hours.

What we learned

We improved our understanding of the Android environment and the three way interaction with an API/Database/Client

What's next for SquaUp

Polish the app and add some features so users can feel safe going out.
