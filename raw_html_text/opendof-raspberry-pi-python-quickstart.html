Executive Summary

Shortcut the learning curve and push OpenDOF closer to the edge of the network by booting a ready-to-run Raspberry Pi . Use Python instead of C or Java to interact with your OpenDOF network. Make a real-world sensor connection with an XBee-based sensor and control network and build your own sensor/control boards with the included Gerber files.
