Inspiration

Martian Chronicles by: Ray Bradbury, Deforestation, Colonization

What it does

Survival game revolving around building and getting resources from your limited environment.

How we built it

We used C# for coding and blender for modeling and put that all together in Unity. We used procreate for 2d art and for the backstory of the game.

Accomplishments that we're proud of

We ended up making a lot more than we thought we would and that we would in the past. We also had some people who couldn't code or do much else revolving around game making so how much we achieved was surprising.

What's next for Colony 54-6

We plan to improve game mechanics, add a better inventory system, add a variety of npcs and other animals for ambience, improve the building system, add an oxygen system.
