Inspiration

There are so much information around us, but we do rarely gather knowledge from them. So consuming little at a time on regular basis is a better strategy to keep up with the running world.

What it does

Tidbits lets users create account and choose the topics they want updated news on a routinely basis. And for now user can get the message as text message on their mobile device.

How I built it

Tidbits is a web application built in angular js, html and css. Mongobd mLab is used to store user data and node server is to communicate between angular from client side and mongo db from server side. Twilio api is used to send the news content to the users’ phone number.

Challenges I ran into

AWS was not working so only one machine was able to make get, post request and get it displayed on the web app.

Accomplishments that I'm proud of

Making an application that fully able to host a server and send data based on what user wants!

What I learned

We gaining a better understanding of node server, AWS, and the connection between web app language angular, local server side language node and online database such as mongo bd.

What's next for TipBits

Users can choose how often they want the news update get sent on their phone; also whether they want it as text, email or voice call.
