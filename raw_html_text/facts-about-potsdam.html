Inspiration

I am currently learning how to use Amazon's APL so I needed something to try this out on. What's better than a fact app about my favourite city.

What it does

You ask the skill for a fact about Potsdam - and it delivers.

How I built it

I used Amazon's node.js Alexa framework (2.0), which already has embedded APL support.

Challenges I ran into

Even though testing without an actual device is possible, it is not the best experience. I'm looking forward to giving this skill a spin on a real device soon.

Accomplishments that I'm proud of

Using APL was actually easier than I expected, so I'm pretty happy that I had it up and running really fast.

What I learned

I was surprised how straightforward the use of the provided library is.

What's next for Facts about Potsdam

The APL interface definitely needs some tuning, so it also gets an appealing look. I'm really looking forward to leveraging the possibilities of the APL (developing a proper design, using individual colors, images, etc.)
