Inspiration

I needed an easy whiteboard for an educational environment that would be saved with Gaia and Blockstack.

What it does

It allows users to draw and write on a digital whiteboard.

How I built it

I built it using JS and HTML after experimenting with the Hello, Blockstacks tutorial.

Challenges I ran into

Some challenges I ran into were enabling CORS on my server and adding the correct redirect url in the manifest to allow Blockstack Authentication.
