Inspiration

Creating a Halloween inspired t-mobile store. We've worked on games before, but we wanted to take it to the next step using VR. This store will be more fun and interactive for users, especially children.

What it does

A halloween-inspired T-mobile store with displays to show the phones.

How we built it

we built this using Unity and C#, along with Oculus integration.

Challenges we ran into

Set up was the most difficult. We spent a combined half of the allotted time(12 hours) on setting up our environment and getting things to connect and work.

Accomplishments that we're proud of

We have a store with assets and phones. Movement of the player. User-friendly experience.

What we learned

We learned how to use Unity and VR. We also learned valuable lessons in time-management.

What's next for Spooky-mobile store

We hope to fully complete the grab and touch functionalities. As well as changing the store assets depending on the season/holiday.
