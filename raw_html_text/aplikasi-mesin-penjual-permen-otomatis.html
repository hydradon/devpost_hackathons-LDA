APP MENTOS

( M esin p E njual perme N o TO mati S )

IDE

Untuk menjual permen di sekolah sebagai sarana belajar kewirausahaan menggunakan mesin penjual permen otomatis.

CARA KERJA

Jalankan aplikasi MENTOS. Untuk melakukan deposit Scan Kartu QR Code Isi Saldo (Dapat dilakukan di koperasi sekolah). Setelah sukses menscan QR Code Isi Saldo maka saldo pelanggan bertambah. Pada mesin permen otomatis terdapat QR Code Mesin. Untuk membeli permen mentos, pelanggan melakukan koneksi Wifi mesin setelah terkoneksi pelanggan menjalankan aplikasi untuk melakukan Scan QR Code Mesin. Sukses melakukan Scan QR Code maka 1 permen mentos keluar dari mesin dan saldo pelanggan berkurang seharga 1 permen mentos.
