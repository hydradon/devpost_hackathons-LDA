Puede optimizar el flujo de coches en una interseccion dependiendo de el numero de carros de frente a cada semaforo.

How we built it

Empezo como todo, una simple idea que se enfocara a algo que contibuyera a nuestra sociedad, despues de varias opciones optamos por un sistema de semaforos que pudiera aumentar el tiempo de una luz verde si el numero de carros es mayor en ese lado de la interseccion. Usamos sensores ultrasonicos, cada sensor puede contar el numero de carros frente a cada semaforo para asi poder decidir si se aumenta o disminuye el tiempo de una luz verde o roja, ya que hay veces que el flujo de carros puede ser mucho de un lado y nulo por otro lado haciendo que las personas pierdan tiempo valioso mientras esperan a que la luz roja cambie para ellos poder seguir su camino. Nos enorgullece presentar un proyecto que fue ideado en muy poco tiempo pero que a pesar de todo logramos sacar adelante. Aprendimos mucho de esta experiencia, conocimos mucha gente y mejoro nuestra capacidad de socializar y trabajar bajo presion.
