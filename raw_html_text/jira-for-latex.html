• LaTex rendered inside JIRA

What it does

This plugin renders LaTeX anywhere inside JIRA - even on Summary. As an example, paste the following text into Summary and Description fields:

  • Summary: Solve equation //(ax^2 + bx + c = 0//)

  • Description: Solution for equation //(ax^2 + bx + c = 0//) should be {noformat:nopanel=true} //[x = {-b \pm \sqrt{b^2-4ac} \over 2a}//] {noformat}

Inline LaTex must be wrapped between //(..//) and LaTex block must be wrapped between //[..//]. For fields using Atlassian Wiki syntax, consider using noformat tag to prevent JIRA from modifying LaTeX content.
