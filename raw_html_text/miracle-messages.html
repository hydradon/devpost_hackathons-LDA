Inspiration

Miracle Messages is an organization that uses short video messages recorded and delivered by volunteer messengers to connect homeless people to their long lost loved ones. The process the organization currently takes is too long; it involves using google forms repeatedly, and the recording of videos with confusing instructions. What we made was an Android Application that consolidates all the features to your phone, so any volunteer helping out with the cause will have all the instructions laid out before him or her. We wanted to help somebody who was already helping the world become a better place.

What it does

Store user credentials, uses a custom camera to take videos with instructions for volunteers on the bottom third of the screen, and export generated mp4 files via email.

How we built it

Using Android Studio, open source software/tutorials, patience, and will power.

Challenges we ran into

Getting the custom camera to work (hardest part), getting the email to export the correct mp4 file, getting Android Studio to work at the start.

Accomplishments that we're proud of

Getting a near fully functional mobile application up and running within 24 hours, sleeping for only 1 hour.

What we learned

How to use the new google camera (previous one was deprecated since API 21), client-developer interaction, how to configure the designs for Android.

What's next for Miracle Messages

Get input from the founder of the organization, work out some existing bugs, get the file to directly upload to Youtube rather than using email due to memory constraints.
