Inspiration

When you want to go out on the town and you have to setup a text message artillery command center to find out where everyone wants to go, throw it on Pollar and find out what's cool in no time.

How it works

Users create polls on the web app, and can vote to weigh in on important and no-so-important topics.

Challenges I ran into

Parsing data from Firebase and learning how to parse it out once we got it proved difficult at first.

Accomplishments that I'm proud of

After persevering for a number of hours, we finally conquered Firebase.

What I learned

We learned Firebase, which allowed us to update our data on the web client in real time.

What's next for Pollar Social Poll

The next stop if pushing Pollar forward will be to setup a system for users so they can connect with each other and manage votes by user ids. Then clients will be rebuilt and optimized for Android and iOS.
