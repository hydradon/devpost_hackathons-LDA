Inspiration

We were inspired by the idea that parties can be anxiety inducing, and that this would significantly reduce stress

What it does

partypal Takes in a series of commands and does certain things based off them. You can call for help, alert the host of the party, and start a party session that periodically checks in on you to see how you are doing.

How we built it

We are implementing an app using react native, and the rest of our stack is python and twilio to process the text messages.

Accomplishments that we're proud of

We had to thread our tasks to get things to run in parallel, its more efficient, efficiency is good.

What we learned

We learned about react flask ngrok and development with twilio.

What's next for partypal

a full app to help combine numbers and parties and a corresponding webapp.
