Inspiration

We wanted to create a service that allowed users to be able to search for information without using a browser or cellular data.

What it does

You text what you want to search and the system returns a text message within seconds. It can find crypto-currency prices, weather, word definitions, images, stock prices, and Wikipedia summaries

How we built it

We used Twilio as a messaging service, Python , API's, and Flask

Challenges we ran into

We ran out of time, but we were going to incorporate a payment system using bitcoin, debugging errors in code, and creating a registration database.

Accomplishments that we're proud of

We created a service that may impact the world by finding information without the need of cellular data. We are proud of the service that we created, and how it may impact the world.

What we learned

We learned how to use API's in order to find out information and then return it as a message. We also learned how to work as a team and listen to each other's feedback and opinions.

What's next for TextBot

TextBot is massively scale-able, eventually we would want to incorporate news outlets, sports info, and reminders.
