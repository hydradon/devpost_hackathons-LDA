Inspiration

Help people keep track of their calorie consumption without much hassle.

What it does

It takes picture photo of your food and tells you how much calorie is present in that dish. Based on total calorie consumed by you in a day, BMR and how much time you have for workout suggest different exercise.

How I built it

We used Google vision API to fetch data name of a dish. Then we used fatsecret API to fetch calorie content for that dish. Used Workout manager API to suggest exercises.

Challenges I ran into

Finding the API for the data we needed and which was not private was hard. Needed to improvise on the idea of the project.

Accomplishments that I'm proud of

This was my first time developing a web app using flask and I am proud of building a project using it.

What I learned

Using Flask to build a Web App.

What's next for Foozam

Include Fitbit data to keep track of users calorie burned.
