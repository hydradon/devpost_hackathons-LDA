Inspiration

We are trying to make the world a safer place!

What it does

A data driven solution to providing Runs a custom routing algorithm to navigate the safest path

How we built it

Using Sheffield City Council's crime data set link we built up a map of the crime in our operating area. We have a Java backend that runs our custom routing algorithm and an HTML & Javascript front end that allows the user to easily request a route.

Challenges we ran into

This task was too computationally expensive for our Monte Carlo implementation.

Accomplishments that we're proud of

We created something that could help make people safer!!

What we learned

It is hard to implement routing algorithms.

What's next for SafeJourney

More data, use the CCTV dataset to balance the 'dangerous' areas with how well monitored they are. Generally more intelligent algorithm.
