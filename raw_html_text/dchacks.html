Inspiration

Wanted to see what politicians are typically posting on Twitter.

How it works

Backend takes in 40 of the most recent tweets from each of the 50 US Governor's verified Twitter accounts, and uses Indico's political sentiment analysis to determine the ideology behind these tweets. Javascript displays each governor's political idealogy, as determined by their tweets, onto a simple webpage featuring a fully interactive US state's map using a color coordinated key.

How to view the webpage

Run trialkeyrun.class, then open chrome with local-file-acess enabled in order to view index.html
