Inspiration

Animation, sound reactive lighting.

What it does

Pac-Man ghost changes type when microphone is tapped/clapping. -or- Pac-Man ghost scrolls across the screen

How I built it

Using a Raspberry Pi and Python, Pimoroni Scroll Phat HD for the display.

Challenges I ran into

Integrating the microphone, and the ability for the ghost to scroll across the screen.

Accomplishments that I'm proud of

Making the image of the ghost using only one color LED,being able to have it react to the microphone.

What I learned

Python, LED control, HW/SW integration.

What's next for Pac-Man Ghost Clapper

Scroll the image when tapped.
