Inspiration

I felt extremely unmotivated in the morning so I wanted to create a way to let me start the day right.

What it does

I created a python script that takes advantage of the twitter API. I then utilized the http://api.andrux.net API that allows me to send an HTTP POST request to receive the quote of the day. Then using the twitter API, I was able to post a quote when the script was run. Using the Cron tool that is built into unix machines, I was able to run the script in certain time intervals as well as start the script when the system was rebooted.

How we built it

Built it with python.

Challenges we ran into

Figuring out how to have the python script run on a time interval.

Accomplishments that we're proud of

A feeling of motivation every day I wake up.

What we learned

The endless possibilities that a raspberry pi can accomplish.
