Inspiration

We are passionate about preserving the environment and Earth's natural resources.

What it does

Benefits community through promoting environmental awareness. The user completes an interactive quiz and gains knowledge of many different topics pertaining to the condition of Earth today.

How we built it

We used html, css, and javascript to design the program. We also used github to collaborate with team members.

Challenges we ran into

The main challenge we ran into was lack of experience with the programming languages used for this project.

Accomplishments that we're proud of

We are proud of creating an interactive website from scratch. Additionally, we are very pleased with the display of the website along with enabling the user to easily navigate the website.

What we learned

As freshman, we were learning a lot while building this project. We gained exposure to all three programming languages, as well as github.

What's next for Nature Knowledge

We would want to expand the application of the website to different topics, such as poverty levels and climate change.
