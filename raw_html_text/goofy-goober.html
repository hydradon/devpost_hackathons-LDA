• Goofy Goober!

Inspiration

I think kids like to laugh like the rest of us and sometimes a little self-deprecation can be quite hilarious.

What it does

It generates zingers and wisecracks about you or someone you specify, like Dad for instance.

How I built it

I built it starting with the fact skill Alexa tutorial and extended it form there with AWS Lambda and Node.js

Challenges I ran into

I've never coded with Node.js or used AWS Lambda but I have JS experience and the learning curve wasn't so bad with Lambda on the count of the documentation and easy to follow tutorials.

Accomplishments that I'm proud of

I'm happy to have learned a few new things and services in the process of building it.

What I learned

I learned that Lambda really makes deploying things online way faster than setting up your own server. I think I might want to learn a bit more about Node in the future.

What's next for Goofy Goober

More ZINGERS!!
