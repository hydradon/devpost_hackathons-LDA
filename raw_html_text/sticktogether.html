• If you are far away from the rest of the group in a foreign land (e.g. Michigan), you can quickly navigate to another member in the group.

  • Confirmation for traveling to a specific travel member

  • Built with a familiar Google Maps API

Inspiration

When traveling with a large group in an unknown city, a group member can derail from the pack pretty easily. We built StickTogether to navigate the user to another member in the group. You can track all of your group members as points on the map in order to make sure that a child does not get lost in the city.

How we built it

Android Studio, Google Maps API and Firebase.

Challenges we ran into

  1. Google Directions API's lossy compression algorithm in the Encoded Polyline Algorithm Format .
  2. Travel group initialization from contacts.
  3. Database configuration.

What's next for StickTogether

  1. Travel group initialization from Facebook groups, Facebook group messages, and Groupme group messages.
  2. Concurrent navigation to a common destination for all members in the group.
  3. Invitation to other travel companions via SMS, Facebook, email, etc.
