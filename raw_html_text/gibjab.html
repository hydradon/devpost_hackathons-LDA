Inspiration

I wanted to create something that may be widely used and informative.

What it does

The site has a few buttons the contain news and the user may see that once they hover over the button. The user then may join a live chat with others on the site to dicuss, and learn about the current news/event

How I built it

I built it using HTML, CSS, Javascript, Firebase's web client and Jquery

Challenges I ran into

I ran in the responsive design, fatigue, and syntax error issues.

Accomplishments that I'm proud of

I am proud that this is my first website online.

What I learned

I learned more javascript and jquery and how to implement web chat into my site.

What's next for GibJab

To make it better for users by implementing logins, etc.
