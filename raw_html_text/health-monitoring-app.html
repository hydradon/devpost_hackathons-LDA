Inspiration

We are inspired by the services that many people who doesn't have access to health care can get. We want to serve as many people as possible by providing them with a platform that provides different medical services at the tip of their fingers.

What it does

Any one can register with the KaiOS empowered application and use the application on the language of their choice. Then any one can search information about some disease, get daily news alerts daily, search for information about some medicine including name, prescription information, expiry date. Patients and pregnant women can also set appointments with doctors. The appointment will be confirmed by the doctor.

How we built it

We used KaiOS API, html, css , javascript and nodejs as a platform.

Challenges we ran into

Our project needs some inital capital to be launched. We have been working on integrating the latest technologies like Artifical Intelligence and Block chain. These technologies by themseleves require good processing power which we currently can not afford.
