Students are notorious night owls

This app makes it easier to manager sleep by determining when you need to sleep based on your personalized sleep needs and your required wake up time for class. Using the Google Calendar API, we get the user's course schedule and then prompt them for feedback after each class on how alert and able to focus they were. We then use this data to determine how effective their sleep was the night before and adjust their estimated required time asleep accordingly. We then recommend a bedtime based on how much sleep is needed and when the user needs to get up to make it to class in the morning.

Technologies used React Native, Flask, Google Calendar API

Challenges As a team of two with little prior experience, the task list was quite ambitious. Our greatest struggle was configuring the endpoint on the Flask api to take requests from the mobile app.
