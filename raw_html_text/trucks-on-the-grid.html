Inspiration

My friends and I go to San Jose and San Francisco frequently and it is difficult to know where food trucks are because they change location daily

What it does

Tracks food trucks in real time by using GPS that can be turned on and off by the Truck Drivers

How we built it

Used the MERN stack to develop which includes MongoDB, Express, React, and Node.js.

Challenges we ran into

Getting information from API's like Google Maps

Accomplishments that we're proud of

Being able to deploy our web app

What we learned

Some team members learned the MERN Stack and how to code with Javascript and React. We learned to coordinate and communicate in small groups to prevent redundant code and to improve efficiency within these 36 hours

What's next for Trucks on the Grid

Mobile App on both ios and Android
