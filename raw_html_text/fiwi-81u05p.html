Inspiration

We were inspired by how students are spending more than they earn and always talk about being broke and how we can help to solve that.

What it does

The app allows the user to add their income and then add their expenses and also categorize them. It also allows them to set financial goals as well as suggest ways to save money.

How we built it

We built it using Microsoft's power apps API and using google sheets to store the data.

Challenges we ran into

Connecting to the database, building a wireframe and then get the app working according to our condition.

Accomplishments that we're proud of

We got a working prototype for the application and successfully were able to build the budget tracker part

What we learned

How to build a good financial planning application, Power Apps and how to connect them to a database

What's next for FiWi

Expand it to have the financial goals part as well as the suggestions part.
