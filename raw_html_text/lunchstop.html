Inspiration: Never go hungry again on a long road trip or walk, LunchStop will take you on the detour you need to go for a meal!

What it does: Finds restaurants between location and destination point and helps travelers save time!

How I built it: It is a web application built on Node.js, AngularJS, and HTML/CSS. We used Microsoft Bing Maps API and FourSquare API.

Accomplishments that I'm proud of: the app itself!

What I learned: A lot about working together as a team, and about software development in general :)

What's next for LunchStop: Expanding globally! We are also planning to make it mobile-friendly.
