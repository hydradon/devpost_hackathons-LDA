Inspiration

We have seen many students many students taking advantage of loopholes in attendance system apps. This results in fake attendance and often results in unfair grading.

What it does

Our program aims to prevent the exploitation of these loopholes. We compare the number of students present in the classroom using Facial Detection and compare that number against the number of students who entered the code for class attendance through the app.

How we built it

We used Azure's Face API and coded everything in Java.

Challenges we ran into

We learned everything on the fly.

Accomplishments that we're proud of

This is our first hackathon. We attended two workshops before coding this program. So, we had 3 hours to hack this program.

What we learned

We have a lot to learn.

What's next for Facial Detection Class Attendance

This is currently a scaled down version which uses pre-programmed images. We plan to use this program to recognize faces in real time and compare them against database of students.
