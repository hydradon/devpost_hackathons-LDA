Inspiration

How it works

One player sees a bird’s eye view of the entire map in 2D on a tablet. The other player wears the VR headset and navigates the world. The narrative of this game is found within the communication between the two players. As players describe this unfamiliar world to each other, we can see how each player is perceiving the environment. They learn how to use each other’s eyes in order to clear the obstacles while avoiding the danger of falling.

Challenges I ran into

Achieving a low polygon count in the model of the environment so the VR runs smoothly. Making all the moving parts move correctly.

Accomplishments that I'm proud of

A working demo with a finished look. My ability to learn something from scratch then build an entire project with it in less than 12 hours.

What I learned

Communication between Unity and outside objects is possible and something that I'd like to explore further.
