Inspiration

The ridiculous amount of notes we take and worksheets we get on a daily basis that end up being lost and useless.

What it does

Allows you to digitize both handwritten notes and typed information, search through them, check your spelling and grammar, and generate quizzes to test your knowledge of the material.

How I built it

We used the Google Cloud API to detect text in images, a string parser with the Quizlet API to generate quizzes based on the words in the text, and machine learning to give the user the best possible correct text in terms of spelling and grammar.

Challenges I ran into

We had trouble implementing the machine learning for spell-checking the user and getting the phone to process the text from the images.

Accomplishments that I'm proud of

The fact that we were able to build something that we would use on a daily basis.

What's next for Scribe

Deploying it in the Play Store and using it for my classes.
