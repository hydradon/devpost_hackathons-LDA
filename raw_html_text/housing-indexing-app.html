Inspiration

We really wanted to improve the way people in general find affordable housing such as apartments since there are so many apartment buildings that simply don't have their listings available in the more common websites like Craigslist, Zillow, etc. We wanted to create a website that was interactive and easy to navigate in order to better find housing.
