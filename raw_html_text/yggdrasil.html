The Quest for The Sacred Water is a text based RPG, akin to a choose-your-own adventure book, where the players are able to take multiple paths to complete their story. The first choice players make is what weapons they will use: 'sword and shield,' 'bow and dagger,' or 'staff and tome,' in other words, magic, combat, or stealth. This first choice determines ultimately what path the story will take and what text the player will see as they progress throughout the game.

I built the game with the python programming language, using the Atom code editor.

This game is my first programming project. It is still in the beginning stages and I have a lot more work to do to complete the story, but I really enjoyed working on this as my introduction to programming.
