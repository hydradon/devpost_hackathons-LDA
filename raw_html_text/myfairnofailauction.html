Inspiration

Make Hurricane donation auction possible via simple messenger and make it algorithmically fair long distance.

What it does

How we built it

A full distributed decentralized auction ledger with fair transaction serialization using Hashgraph then build a bridge to accept bids and show results in mobile chat and messengers using Hyphenate and nexme

Challenges we ran into

Making Hashgraph work with dependencies on messaging systems and writing a message broker.

Accomplishments that we're proud of

Idea, impact and use of new more fair ledger technology

What we learned

It can be done in a short time.

What's next for MyFairNoFailAuction
