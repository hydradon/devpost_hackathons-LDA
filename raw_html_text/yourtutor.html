YourTutor arise from a view that most student in our area seems to always prepare for one exam or the other, dying to know how to get a reliable and competent tutor to help them prepare for the exam. There are also numerous tutors who are capable of this but don't have access to these students. It was at today, YourTutor idea came up and we thought of how to make use of the existing technology in bringing these students and the tutor together.

YourTutor: This was achieved by students or guardian or parent by just sending an sms with the formatted tutor needed; e.g, "English language", "ketu". Once our intelligent messaging API receives these message, it goes on to our program and pulled from the database any of our registered tutor with the major requested and nearest to the requested location. The Web App serves as a remote search for parent or guardian to find the nearest and qualified tutor for their children. also the Android app serves the same purpose with the web app.

YourTutor was built with Bootstrap, JavaScript, PHP, HTML 5 and CSS 3 for the web app. And the database was created with the use of MySQL and Android was use for the mobile app for better user interface. The sms integration was done using Twilio.

Challenges: Numerous challenges were encountered during the course of accomplishing the project, like Networking and interfacing with web apps, pulling and fetching of data from MySQL database, writing API for the database and also difficulty in trying to make the sms api work. Problem of response as the number used by the API in sending the messages changes each time the message was sent and finally uploading github pages using custom domain (.Tech).

Accomplishment: We were able to make the search button works for all states in Nigeria and also successful implementation of the sms data search.

Learning never stops; We learnt more about UI designs and how to make an app UI user friendly, also learnt about APIs and how to consume them, we learn about how to integrate api into codes and makes it work with other codes. Also, more understanding about mySql database and the use of Github was also achieved

What next for YourTutor is to implement a fully functional mobile app on Google and Apple store.
