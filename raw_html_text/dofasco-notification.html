Inspiration

Dofasco needed a webapp to check the status of their incidents and the severity of them.

What it does

Our webapp allows dofasco to view incidents, and sort them based off date and severity

How we built it

We created a template using HTML, CSS, and JavaScript, then used Django to populate the site with data from Dofasco's API

Challenges we ran into

We had to learn to use Django, and additionally had to make assumptions about Dofasco's systems, such as what represented the different levels of severity were. We also ran into problems with AWS where we were unable to make an account, and issues with domain.com where we couldn't ftp to it.

Accomplishments that we're proud of

Responsive and minimalistic webpage

What we learned

How to use Django, better web design

What's next for Dofasco Notification

Implementation with real world data
