Inspiration

Aim is to improve the productivity and reduce the duplicate effort in maintaining same or slightly differing content at different places.

What it does

It allows to structure the content through Conditional MultiExcerpt. Conditional content (text, images, link etc) could also be used in excerpt which will be displayed only if condition is fulfilled for display. If there is no conditional part, same content will be displayed wherever Conditional Multiexcerpt is included using Conditional MultiExcerpt Include macro

How I built it

Atlassian sdk is used for developing this. Java and javascript is used mainly for this solution.

Challenges I ran into

Rendering Images and Links at any other place than original was main challenge.

What's next for Conditional MultiExcerpt

Creating conditional tag names for condition could be automated for working fast and efficiently on content creation.
