• Humerus

Inspiration

Help Radiologists to make a diagnosis in less time with greater accuracy

What it does

Auto-detection of bone fracture, focusing on the humerus, in x-ray images

How we built it

Implement image processing technique and machine learning in R using the packages, such as EBImage and MxNet

Challenges we ran into

  • Hardware Constraint
  • Massive dataset
  • False-positive results

Accomplishments that we're proud of

  • Highlight all the existing edges with Sobel operator

What's next for XR

Auto-detection for all bones of the human body

Built With
