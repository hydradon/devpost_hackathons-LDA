SFHacks2019_Swipe_Send

This program allows for input from the leap motion sensor and sends a message should the sensor pick up a swipe

Requirements 

1.Python 2.7.7 (Not python 3.X.X)
2.Orion 3.2.1 (For Leap motion Sensor)
3.The leap motion sensor (Obviously)
4.A laptop specs enough to run the leap motion sensor

Installation

Currently this program can only accept programmed inputs for the message that will be sent 

1. Run the program and turn on the leap motion sensor
2. Register a motion above the Leap motion sensor
3. The program will then send the preprogrammed message to the user
