What it does

Randify is a spotify utility to create randomized playlists of your favorite artists in a much more efficient manor than that allowed by the streamin service itself.

How we built it

We used a combination of javascript, jquery, css, html, and the spotify api to create our project.

Challenges we ran into

We had a lot of struggles with navigating the spotify api's confusing token authorization system.

What we learned

We learned more than we ever thought we would about the underlying features of the Spotify API.
