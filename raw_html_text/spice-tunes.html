Inspiration

Aakash and I are both fans of music. Whenever we hang out with our friends at home, we always have music suggestions. We wanted to create a web application that would allow us to enter a guest's gathering and add music to their playlist.

What it does

Our web application allows hosts to "Spice" up their gatherings. They can easily create an event name which they can provide for their guests. Guests can enter the provided code and add specific songs they want to play to the guests' playlist.

How we built it

We used HTML/CSS, Javascript, and JQuery to build our project.

Challenges we ran into

When getting songs from YouTube, we ran into the problem of playing the actual songs from Vevo (even the lyrical versions) because YouTube blocked us from doing that. So we had to pull covers of the requested songs.

What's next for Spice Tunes

We are hoping to add the thumbs up/down feature that allows guests to vote for requested songs. The greater the popularity of the song, the sooner it will be played. If there's a lot of dislike for a song, then it will be removed. Additionally, if there are no songs in the queue, we wanted to use ML to find songs that guests seem to like.
