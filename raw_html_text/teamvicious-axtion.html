Inspiration

Slack has become critical tool for inner communication for companies. It provides a perfect place to learn what's the atmosphere in company.

What it does

Our slackbot goes through slack messages and uses Sentimental Analysis Api to figure out the mood.

How we built it

We used nodejs, firebase, ibm bluemix, watson and some jquery charting libraries.

Challenges we ran into

Our most time was spent into correcting our value proposition.

Accomplishments that we're proud of

Finished Product Viable and Valuable vale proposition Teamwork across multiple skills

What we learned

New Technologies Better use of slack, and other tools Remote working Teamwork
