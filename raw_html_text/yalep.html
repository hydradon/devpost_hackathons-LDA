Inspiration

We wanted to give people a subtle and simple way of learning about culturally significant places near them.

What it does

Yalep is opened by clicking the "Y" tile icon on the Microsoft band. It shows culturally significant places near you (<2km radius) as a series of tiles which you can scroll through. If you click on a tile, it opens up a website on the phone with information about the place.

How we built it

The Microsoft Band is connected with an android app on your phone. The app is non-intrusive, runs in the background and will not disturb you. The app automatically gets information of culturally significant places near you from Google using the Places API. It sends a list of place names to the band. When you click on a tile in the band, a web event is triggered on your phone which opens up a website with a URL attached to the particular place.

Challenges we ran into

Getting culturally significant places on the app using the Google Places API was a challenge. We had to find our current location, get a list of places and filter them by types (museums, art galleries, temples, mosques, churches, synagogues, cemeteries, universities and libraries). We spent a lot of time resolving errors.

Accomplishments that we are proud of

We are proud of our team synergy towards a common goal and creating an app which would help people learn more about local culture

What we learned

We pretty much had to learn everything. We learned how to make an android app, use the Google Places API and code the Microsoft band

What's next for Yalep

We would like to add more features to the app and officially release it!
