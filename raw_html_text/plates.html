Inspiration

Finding good restaurants in a new city can be quite challenging. Why not literally see the food that restaurants are offering?

What it does

Using the location of the user, the phone application will provide pictures of food dishes from nearby restaurants. If the user really likes the food, then they can add the restaurant to a shortlist and get directions to the restaurant.

How we built it

Data hosted on MongoDB API handled by StdLib Server-side built on Node.js Client-side built on Android
