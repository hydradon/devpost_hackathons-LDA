Inspiration

We're all about protecting young women.

What it does

Sometimes - when you're out and about at a party - you can lose track of your friends. With this app, you'll all be connected to each other at all times. You'll be able to keep tabs on where your friends have gone off to, and if they go beyond a certain perimeter, an alert will go out to everyone connected stating how far that friend has gone.

How we built it

We built it using Android Studio and Java for Android. We also implemented the Google Maps API.

Challenges we ran into

None of us had really worked with Android Studio to any great extent before. So, it was quite the challenge to get even the simples things working.

Accomplishments that we're proud of

We were able to successfully start this app and implement the Google Map API.

What we learned

We learned a bit about Android Studio's functionality and that implementing GeoFencing can be quite the hassle.

What's next for Girls' Night

We'll definitely work on this in the future and implement the GeoFencing functionality.
