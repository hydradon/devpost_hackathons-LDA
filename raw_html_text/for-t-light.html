Inspiration

The danger of the California wildfires

What it does

Mini activities to learn different facts about how to stay safe in a wildfire

How we built it

Using the JAVA AWT Graphics package

Challenges we ran into

The AWT package was new to us and we had to learn on the way, with a time constraint

Accomplishments that we're proud of

Getting the little fires to animate, and having the fire extinguisher output be parabolic

What we learned

How to use Java AWT and integrate math in our project too

What's next for For't Light

Adding new safety minigames to cover additional facts
