Inspiration

Accessibility is a big consideration in gaming that is often overlooked. We decided to have a go at changing that.

What it does

Its a game package designed to work perfectly with eye tracking software

How we built it

We used some java code to get the eye tracking working. VoiceMacros to get voice input working. Godot to create all of our games.

Challenges we ran into

Voice input is haaaaard.

Accomplishments that we're proud of

We got 4 games working with eye tracking and voice control

What we learned

We learned a lot about the considerations that need to be made when developing games for use with eye tracking

What's next for Accessibility Bois

More games!
