Inspiration

We were just brainstorming ideas and we had this idea. It seemed feasible and useful, so we decided to go for it.

What it does

It's like a Tinder, except is matches people together who want to do similar things. For example, if you want to go to a Habs game but none of your friends don't want to go (or, if you do not have friends), you can put yourself into the database. When someone else also wants to watch a Habs game adds themself, the system will match them together.

How we built it

We used Java.

Challenges we ran into

Huge learning curve; we're all beginner programmers (we only finished one computer science course). It was hard to come up with something useful that we could make with our skills. Additionally, we wanted to include some sort of user interface, and that was something completely new.

Accomplishments that we're proud of

We learned how to create a user interface on Java!

What we learned

Perseverance: we had to switch between many languages and tried a lot of things that crashed and burnt, but in the end it worked.

What's next for CTF Matcher

Maybe a nicer user interface
