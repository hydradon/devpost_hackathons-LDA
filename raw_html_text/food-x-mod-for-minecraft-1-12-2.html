Inspiration

Inspired by a love for the game Minecraft and want to develop for the community.

What it does

Adds a multitude of food items to the game, which are all obtainable through the survival game mode.

How we built it

We used eclipse to setup files, and used a youtube series to help us with aspects we didn't know how to do.

Challenges we ran into

.json files not working modles not rendering properly incorrect file paths and names little to no experience in visual design

Accomplishments that we're proud of

Problem solving our way out of problems such as the models not rendering. Creating somewhat decent textures for items. Being able to successfully create a working mod in a short time span.

What we learned

Modding minecraft is a much more difficult and repetitive process. Older versions of minecraft require different versions of java to run. Creating textures in small size frames is very difficult to do well.

What's next for Food X Mod for Minecraft 1.12.2

More foods to create. New crafting stations for more advanced crafting recipes, and reusing old recipes to fit said crafting stations. Better models for item textures.
