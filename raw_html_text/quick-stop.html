Inspiration

We wanted to make a game for the longest time, but never got the chance to do it. Now, with this hackathon, we finally did it!

What it does

This game is simple. The object is to stop the wheel when the arrow is on the black part. You do, get a point. you don't, try again.

How I built it

We used Corona SDK and Lua to develop this app. It was surprisingly easy.

Challenges I ran into

At one point, we couldn't get the wheel to rotate, and making the button was a challenge in itself.

Accomplishments that I'm proud of

Finally having a mobile game that works!

What I learned

Lua programming code, and Corona SDK basics.

What's next for Quick Stop

A menu, and some ad support.
