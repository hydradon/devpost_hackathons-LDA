We developed a responsive Ruby on Rails application for any job seekers out there (especially college students!) who want to have a repository of all their company interests and job applications.

We used the LinkedIn API so users can easily register on JobHacks with their LinkedIn profile information. For styling, we used Sass and the Materialize CSS framework.
