Inspiration

We were inspired by Simona's suggestion for this kind of projects. The two Mary's were excited for a more hands-on project.

What it does

A skirt that has LED lights sown into it, connected by conductive wires to a micro-controller that lights up the lights depending on the distance of objects from the infrared sensor incorporated in the circuit.

How we built it

A lot of sowing. No, really. A lot of sowing. Programming

Challenges we ran into

A lot.Of.Sowing.

Accomplishments that we're proud of

Learning stuff about hardware, how hard sowing is and Arduino while managing to not kill each other.

What we learned

Using hardware stuff and Arduino to control LED lights.

What's next for Sparkle Skirt

Making it function properly.
