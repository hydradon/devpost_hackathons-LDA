Inspiration

Writing REST API's is tedious, we wanted to change that.

What it does

Generates a database model and a skeleton backend for you to fill out, and allows you to concentrate on the important things - how the users use your system.

How we built it

Python, Flask, MongoDB, PostgreSQL, blood, sweat and reees

Challenges we ran into

  • We are all backend developers, and unfortunatly there was a little bit of front end.
  • Decoding Javascript at 4AM is not easy.

Accomplishments that we're proud of

  • Debugging javascript at 4AM.
  • Dockerising everything

What we learned

Flask skills have improved. Learned how to use MongoDB

What's next for LAPI

  • More automation. Why dont we generate that front end for you too? ;)
  • Run on the cloud. Rest API As A Service anyone?
