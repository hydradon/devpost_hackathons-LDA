Inspiration

I was absent from school and the teacher did not post the notes, so I could not get a professional copy of the notes that afternoon.

How it works

It uses a camera to detect text. After finding a certain box of text, a text version will show up.

Challenges We ran into

Tesseract always created very many convoluted issues that required a lot more assistance to resolve.

Accomplishments that I'm proud of

We finished a portion of the app and can continue work from there.

What We learned

We learned how to use Android Studio to a basic to intermediate degree. We also learned how to manage our time.We also learned how to use Stack OverFlow and GitHub to assist us to solve our compile time and runtime exception.

What's next for Pic2Text

We would like to implement a larger camera so it can read many lines at once. After implementing a larger camera, we would also like to convert this image into a text document or word document.
