Inspiration

We were inspired to build this project from looking at common issues for people at the gym when seeking for motivation or guidance towards where to go next with their workout.

What it does

  • Allows to record workout repetitions and statistics
  • Ranking system with several users

How we built it

Developed by using the help of React.js and Express.js, as well as MongoDB for storing all user data.

Challenges we ran into

Understanding user authentication since we wanted to maintain users secure in regards to their data.

Accomplishments that we're proud of

  • Beautiful user interface and responsive design
  • Simple to use API for handling C.R.U.D in our databases

What we learned

  • MongoDB
  • React Hooks
  • Custom user auth
  • Javascript Fetch API
  • Google Cloud

What's next for HeapUp

  • implementing a suggestion algorithm that will allow users to receive ideas of where to focus their workouts mostly in order to move above their current rank in the list.
  • smoother animations and improved UX.
