• whm

Inspiration

To untether social media from profiles and static locations

What it does

It allows users to view and post about what's going on around them. The user can only post and see posts from their current location.

How I built it

We used Swift for the iOS development and Sketch and Illustrator for the design.

Challenges I ran into

Creating custom map items gave us some issues on the design. Further, we have a tremendous number of features we want to add to the platform, so choosing which ones proved a challenge.

Accomplishments that I'm proud of

It's exactly what we wanted.

What I learned

How to build awesome things!

What's next for whm

Keep building!
