Inspiration

The problem of mental health that affects millions of students and workers

What it does

Helps people practice gratitude, which will then make them view the world in a positive way

How we built it

We built it by using the nimbella command, which allows us to create commands that can call APIs. We use nimbella to call Firebase API.

Challenges we ran into

Connecting Slack and Firebase was the biggest challenge we came across

Accomplishments that we're proud of

We are proud of the effort we put into this project even though it's not the most complete project.

What we learned

We learned basic Javascript, nimbella, and the basics of Firebase
