Inspiration

We want to help people who cannot physically flip book page to be able to enjoy reading.

What it does

Smart page turner detect if the user has read until the end of page. Once it determines the action is valid, it then trigger the page flipping function

How I built it

We built it with OpenCV, using the existing landmark library and an algorithm published in a paper regarding eye center localisation

Accomplishments that I'm proud of

The valid trigger has already been accomplish and we work great as a team.

What I learned

OpenCV is really powerful and we can use open source project to create amazing things

What's next for Smart Pager

We are going to build the actual physical device for the page flipping function
