Inspiration

The corruption in the actual voting systems around the globe

What it does

It uses Blockchain technology to let people vote and avoid the votes to be manipulated

How we built it

For the backend we used nodejs to build the blockchain (a simple one because we've had no idea on how to make one ) and html, css for the frontEnd

Challenges we ran into

We aimed for dapp (decentralized application) using blockchain smart contract with Ethereum, W3.js API, and web. We've never wrote code for a blockchain and we had to learn how to do it from scratch (stressing but fun).

Accomplishments that we're proud of

Have a backend and a frontend working, we ran out of time to make them work "together" but separately, they work

What we learned

We learned how to build a simple blockchain from scratch in javascript :)

What's next for BlockChain Voting System
