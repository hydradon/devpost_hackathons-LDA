Inspiration

We wanted to make a project that would play to all of our strengths

How it works

You log into twitch, access the channel #swarajd, and type in certain specific commands in the twitch chat, which are parsed using an irc bot, and fire a keypress in a certain window which emits an event using socket.io, which is handled by an express server which then writes a certain set of values to a serial port based on the command used.

Challenges I ran into

We had quite a few networking issues, and we had a bit of trouble configuring the raspberry pi that we were planning to use until we switched to an arduino uno board.
