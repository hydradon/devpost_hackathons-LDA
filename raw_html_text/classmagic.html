Inspiration

When I go to class, I try to focus on what is being taught instead of jotting down the stuff on the board. I try to copy everything later but run out of time sometimes, so I end up taking picture of the board and then that picture is lost in my album. So I present ClassMagic which makes the organization of taking images in class and their notes easy.

What it does

ClassMagic allows you to take pictures, view them full-size, import pictures, add notes to pictures, and share it with your friends as an email or a text message.

How I built it

I built it using Objective-C and Swift with the MesssageUI framework and the Photos Framework and storing data in Core Data.

Challenges I ran into

Storing and receiving from Core Data, UI layout issues, Collection View.

Accomplishments that I'm proud of

Storing and receiving from/to Core Data, Email and text message sending or sharing, adding notes for each separate image which is saved.

What I learned

Core data efficiency, MessageUI Framework

What's next for ClassMagic

Haven't really thought about it. Did this one just for fun.
