Lately I was thinking about redoing my personal website. But personal websites these days look all the same. Same single page feel with flashy text and animations. I wanted something unique and original, so I went back to the basics. No unnecessary animations, no pictures whatsoever. Just plain and simple text in a command line. This resume is a homage to the old days when computers didn't have GUIs and everything was done through the terminal.
