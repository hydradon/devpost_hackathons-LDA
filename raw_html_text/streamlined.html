Inspiration

We wanted to see which candidates were most talked about in the 2016 elections.

What it does

Pulls live tweets from Twitter on candidates. Complex algorithms calculate each candidate's public approval rating and popularity compared to other candidates.

How we built it

We used the twitter api, some php, and some javascript.

Challenges we ran into

There was a tough learning curve for using the Twitter streaming API.

Accomplishments that we're proud of

Displaying our beautiful user interface and animated pie chart.

What we learned

We learned to never compromise our vision.

What's next for Streamlined

Getting seed funding and angel investors.
