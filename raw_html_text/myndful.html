Inspiration

Our team wanted to create an app that helped those with anxiety and depression. We thought about how it can be difficult for those with depression to get up and start their day. They may lack motivation to plan, get up, or do what others may consider 'normal' throughout the day.

What it does

The app offers one coping skill/activity for the user to try out every day. Once they have completed the activity, they can choose if they liked it or not. This data then goes into a radar chart plotting what category (physical, social, or mental) the user enjoys the most, as well as storing which activities the user enjoys. The app also prompts the user to choose how they feel at the beginning and end of the day. This data is stored and goes into a double line graph. The user can access both visualizations and the list of favorite activities.

How we built it

We started off ideating by talking about different ideas. We decided on one idea and started sketching an information flow. We then used adobe XD to create a prototype of this app. We also tried to create an android app using java (see challenges).

Challenges we ran into

We had a team of 5, with 3 designers and 2 programmers. However, both of our programmers had no experience creating an android app. So they learned how to program an android app during the hackathon. We also ran into the problem of how to complete user research in such a short time. To compensate, we created an app that allows the user to choose what works for them.
