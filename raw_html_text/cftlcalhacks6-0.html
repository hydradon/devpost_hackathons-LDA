Ford Focused Ford Focused is an app which uses Ford's Sync3 API to determine statistics about a driver's driving quality, while also pulling data from HERE's API to determine statistics about the area a driver is driving in. The result is an app which can help drivers adapt to changes in speed limit and other outside factors -- all built into Ford's native interface.

Usage This app was created in Android Studio for CalHacks 6.0 from 10/25 -10/27. Our app runs on Android phones with Android 4.2 Jelly Bean or higher. GPS signal on one's phone and a Ford vehicle equipped with Sync3 technology are required to fully utilize the capabilities of our app.
