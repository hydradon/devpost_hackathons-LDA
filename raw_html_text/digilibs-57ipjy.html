Inspiration

What it does

Digilibs pulls fan fiction from fanfiction.net and finds the character names within the text of the story. It asks you for a number of names and replaces these character names with the names you provide.

How we built it

Digilibs uses an XML request to retract HTML from a web page and then parses it to try to identify the names in the paragraphs. It then swaps the names with the names you provide, be it those of friends or family.
