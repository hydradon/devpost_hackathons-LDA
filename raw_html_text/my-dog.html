Inspiration

Xinyue has a cute golden retriever back home. She misses her dog so much that we decided to make a simulation for her (the dog).

What it does

It simulates a lovely dog that requests food, bath, and interaction. The hunger, cleanness, and happiness will decrease as time goes by. When the player walks the dog, the background will change according to the time of the day in the real world.

How we built it

We built it with Unity 2D.

Challenges we ran into

We did not know how to use Unity before the hackathon so we ran into lots of challenges, like how to create a clickable object and how to change to one scene from another.

Accomplishments that we're proud of

We made our first ever game, and it is playable.

What we learned

We learned the basic of Unity and scripting.

What's next for My Dog

We would like to add more interacting features in the future.
