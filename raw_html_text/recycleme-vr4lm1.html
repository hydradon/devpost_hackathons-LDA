Inspiration: We wanted to develop something in Android (Our first times), using Clairifai's API was motivation we needed to try it out!

What it does: Uses tags to identify garbage into catagories {Containers (commonly plastic and glass), paper, organic, landfill}

How we built it: We used Clairifai's API to generate tags, and correspond those to the closest waste management group.

Challenges we ran into: Android for the first time. Attempting to interact with hardware on a low level.

Accomplishments that we're proud of: A super unique idea for an interesting API with a lot of potential.

What we learned: Different benefits of Android's Activities, but the reduced freedom they may offer.

What's next for recycleMe: Introducing it to the IOT (Internet of things) through embedding the software in SMART Trashcans. Maybe we could try using Raspberry Pi!
