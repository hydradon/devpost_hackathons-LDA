Inspiration

Buying travel insurance sucks. There needs to be a more fun way so it feels less like a chore.

What it does

JetSet provides travel insurance in a fun, conversational way.

How I built it

I used node.js for the server, Express and Handlebars for the web, and React for the widget.

Challenges I ran into

I'm still getting used to React so there was a bit of a learning curve there.

Accomplishments that I'm proud of

Getting it all working with React!

What I learned

I cut some corners with React, passing methods down through props instead of using something like Redux. It's not totally wrong, but I feel in some places, it would've made more sense (and maybe even throughout the whole project).

What's next for JetSet

We can launch it out of the sandbox api. We can also extend the logic and conversation to other insurance policies like Rent and Commercial Liability.
