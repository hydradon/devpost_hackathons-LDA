Inspiration

We wanted to try and handle large amounts of data, from different sources, including the Bloomerg API.

How it works

We gather different data about the given company, for example: social media accounts, financial data and Twitter sentiment from Bloomberg, information from Wikipedia etc. This is done trough various apis using Flask(Python) for the backend.

Challenges we ran into

Updating the python version on aws... it killed everything; learning to build chrome extensions, especially POST and GET requests; front end in general ...

Accomplishments that we're proud of

Finding data which is not that obvious to find and using it. Making that extension work.

What we learned

Well, managing data, a lot about both front and back end of web development. Getting data; making chrome extensions; making pretty graphs;

What's next for TreeChart

Integrating a way to buy stocks trough the app, uploading the extension to Google.
