Inspiration

We tackled this challenge because it seemed the most interesting to us, and a common problem among a community -- one that we can relate to as well. We also found that we had a lot of creative freedom while designing the app, and the kinds of elements we could add in it.

What it does

Allows local community residents to connect and go for physical activities together.

How I built it

The design is done with XD, and Android Studio. The website was built with HTML5.

Challenges I ran into

Back-end programming, and front-end: coming up with aesthetic designs.

Accomplishments that I'm proud of

The designs that we came up with in the end looks pretty aesthetically pleasing and could be the final look.

What I learned

We learned Javascript, HTML, a bit of android studio, new things to be implemented in the code, and working with a team to accomplish a project.

What's next for Accello

Completing most of the back-end and some of the front-end
