Inspiration

We want to develop a tool using one of the most innovative in this day,the virtual reality, using this tool we want to take excersice and training to a state of the art technology giving the user the posibility to train in different places with out moving and with the retrieval of information

What it does

We have a statuc bicycle with 2 sensors attached, one for thedirection and the other one for the velocity, with this information we deploy the simulation using virtual reality

How we built it

We have two sensors atthachet, a potentiometer to measure the rotation in the bicyle drive. Also we have an infrared LED to count how many time does the wheel spin and how fast,this information is send through RFC with two arduinos to a unity scene in an Android device wich using a google cardboard or a Gear VR (or any VR system) display the simulation to virtual world modeledin Autodesk Maya

Challenges we ran into

The first sensor (a limiter) used to measure the spin pf the wheel wasnt the correct one,we have to change it The transmission of data using arduino was to slow we have to optimize the code tomake it faster The control rotation in unity of the bicycle was a a huge challenge, solved by test and error

Accomplishments that we're proud of

The control of a virtual bicycle using sensors in a real one in real time

What we learned

How to input sensors values into a unity scene

What's next for VR-Cycle

Use of force analysis from the virtual scene to the real bicycle using friction in the wheels
