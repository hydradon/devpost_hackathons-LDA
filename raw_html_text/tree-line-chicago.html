• Tree Line Chicago

Inspiration

Chicago's vibrant neighborhoods and excellent public transit system.

What it does

Gives users discounts and points for riding the CTA and getting off certain locations.

How we built it

React Native, CTA API, Mapbox API (not in final demo)

Challenges we ran into

None of us worked with react native before!

Accomplishments that we're proud of

We had a working app by the end, and we used git successfully for version control. We all learned a lot, and have an app we wish existed!

What we learned

How to use Native React, how to use git, frontend, and UI/UX principles.

What's next for Tree Line Chicago

We would love to see how the community reacts to it and how it can better serve Chicago.
