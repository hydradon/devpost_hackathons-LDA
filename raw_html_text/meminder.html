Inspiration

What it does

Automates the process of scheduling and managing of medicine or supplements Perfoms the following main tasks:

  • Automate the process of scheduling and managing of medicine or suppleemnts
  • Provides reminders to the user to take their prescription.

How we built it

We initially tried to develop the application using React Native and then as a native Android application, however due to the lack of experience, we were unable to create our envisioned minimum viable product so we developed the UI for the application using Adobe XD.

Challenges we ran into

  • Forming the initial idea
  • Having to learn the required technology stack to implement a prototype of our idea.

Accomplishments that we're proud of

  • Staying until the end

What we learned

  • Sticking to your strengths
  • Try to have basic experience on the technology stack that you plan to use

What's next for MeMinder

  • Integration with other health applications.
