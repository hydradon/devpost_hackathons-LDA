• How it Works

Inspiration

Just thought that why google a word to find its meaning. rather, I use hipchat why not an integration.

What it does

Just install the integration and type /dict word and get the meaning, synonym and antonym for the word.

How we built it

We have built it using python flask.

Challenges we ran into

We got to know about this hackathon only before last 2 days of the deadline. So this was a great challenge to come up with some idea and make a simple but useful integration

Accomplishments that we're proud of

We were successful in making an integration that is easy to use and fun to use. Its user friendly.

What we learned

We learned how to make quick integrations and learned about flask.

What's next for HipDict

We can give suggestive words and verify grammar of sentences in later releases.
