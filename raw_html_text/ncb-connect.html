Inspiration

Fueling the love for WiFi with the want for personalization and convenience.

What it does

NCB Connect incorporates social media login into the NCB WIFI so that tailored advertisements can be posted on the customer's timeline regarding services that may apply to them.

How we built it

We built this by doing A LOT of research. Bulma.io and Bootstrap provided the functions and frameworks for us to build the features.

Challenges we ran into

Facebook API is the source of getting user information and allowing them to log on. Unfortunately, it is not accessible to developers if it is being used on websites that are not published.

Accomplishments that We're proud of

  • We overcame the Facebook API hurdle by finding an alternative, "dummy" way to display the functionality of that aspect of the app.

What we learned

HTML, CSS and Javascript. We also learnt about the existence of bootstrap and bulma which makes website building a lot easier.

What's next for NCB Connect

We plan to expand outside of Facebook and allow users to access the internet using multiple social media platforms
