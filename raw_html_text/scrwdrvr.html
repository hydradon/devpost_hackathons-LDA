Inspiration

After a long day of hacking, our team is ready to party with a mixed drink.

What it does

SCRWDRVR pours, mixes, and serves a drink of your choice in which you can choose from a list of mixed drink options on our web app.

How we built it

We used peristaltic pumps to transfer different drinks to a cup that that the user will place in SCRWDRVR. Using a system of motors, a lead screw, a nut, a few bearings, and a stirrer, SCRWDRVR will then stir the user's drink. The user can select a mixed drink of their choice using our web app.

Challenges we ran into

  • making web app
  • stirring mechanism design
  • using stepper motor instead of normal motor
  • accidentally hot gluing shaft to motor

Accomplishments that we're proud of

  • web app is looking nice
  • peristaltic pumps can run at the same time and have adjusted flows to fit proportions of mixed drinks
  • stirring mechanism

What we learned

  • how to make a web app
  • more arduino
  • laser cutting

What's next for SCRWDRVR

We would like to incorporate a way to use Microsoft's Face Recognition API to see if the user is of legal drinking age, optimally by comparing ID photo and face and checking birth date assuming that the ID is real.
