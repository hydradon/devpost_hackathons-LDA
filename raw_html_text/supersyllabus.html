Inspiration

The time-consuming process of adding stuff from your syllabus to your calendars

What it does

Reads through the syllabus and takes key information such as class times and uses it to create a google calendar event

How I built it

Using java GUIs and initially python

Challenges I ran into

Combining the google calendar api with Java

Accomplishments that I'm proud of

Parsing through a PDF and obtaining only the relevant information

What I learned

How to use a Google Auth API for a web server

What's next for SuperSyllabus

A website that uses a google calendar widget to achieve full capability of the API
