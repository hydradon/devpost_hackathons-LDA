Inspiration

We were inspired by the coolness of the hololens to try and develop some interesting stuff on Unity that could potentially lead to games on Hololens.

What it does

We have four experimental environments ranging from an escape from a world of magma to a bland world of horses running off into infinity.

How we built it

We built it using Unity.

Challenges we ran into

We were learning how to use Unity while making it so we had a steep learning curve as we tried to determine what we wanted to achieve.

Accomplishments that we're proud of

We're proud of having developed some neat environments.

What we learned

We learned how to use the Unity program to create neat games and backgrounds.

What's next for T_Unassigned_Unity_Experiments

We're gonna train ourselves further in Unity to develop a nice Hololens game.
