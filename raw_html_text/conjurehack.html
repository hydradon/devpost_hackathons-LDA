Inspiration

We felt that homelessness itself is painful. Making them wait through the manual process of shelter allocation is even more painful. we would like to ease the shelter allocation by automating the shelter reservation system

What it does

Develop centralized shelter management system to provide real time information along with status and client information. Ability to track clients status and manage the shelter allocations.

How I built it

Front end is done by Node Js/Angular JS/Angular Material and Rest Angular Middleware/backend is done on API, REST Services/Spring boot Rest/Spring Boot Data/Java/Mongo DB

Challenges I ran into

Limiting the scope to a manageable chunk in a given day

Accomplishments that I'm proud of

Able to leverage disparate technologies Extensible solution that we can optimize to provide various service categories in future to enable a homeless to be independent again..

What I learned

its intriguing to see what a charitable organizations have to go through to serve a homeless.
