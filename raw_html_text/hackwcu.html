Inspiration- We did not know many programming languages in common, so we decided to create a program to convert between two languages .

What it does- You insert your code into the left text box, select which language it is from the drop-down menu, select which language to convert it into and click convert!

How we built it- We used java methods to sort through the conversions and break up each conversion into equivalent parts.

Challenges we ran into- Lots of problems with spacing issues and growing spaces.

Accomplishments that we're proud of- Managed to create a program that works for general basic programs with room to grow. Also, we stayed up all night to get the project done (no sleep for the win)!

What we learned- Never try to use GitHub to host dynamic websites. How to use Java, JSP, and TomCat to create a web app.

What's next for HackWCU- Ready to catch up on sleep: Just kidding.
