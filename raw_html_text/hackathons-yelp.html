Inspiration

We wanted to find an accessible platform where everyone can easily find the best Hackathons around and also help organizers of Hackathons get feedback to improve their event.

What it does

This site lets people review different Hackathons around.

How we built it

We built this site using node on the backend and angular on the front-end.

Challenges I ran into

Using angular to display certain things was challenging for me.

Accomplishments that I'm proud of

We made a fully-functioning basic version of our site!

What I learned

Angular JS, displaying certain angular views, modiying json data, etc.

What's next for hackathons.Yelp();

We hope to add more functionality in the future such as a rating system for specific parts of the Hackathon including food, organization, location, etc.
