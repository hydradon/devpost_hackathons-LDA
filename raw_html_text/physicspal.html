Inspiration

I was struggling with physics homework and wanted a quick and easy solution to pull up any formulas I needed.

What it does

Returns a list of formulas that are relevant to whatever concept the user specifies he/she is struggling with.

How we built it

Node.js framework connecting to HerokuPostgresql which stores all the data

Challenges we ran into

Accomplishments that we're proud of

Connecting to a database, storing data in the database, integrating both server and client side js in a project.

What we learned

What's next for PhysicsPal

Expand to other subjects, make UI cleaner, add problems.
