Inspiration

We were inspired through our experiences of being minorities in a STEM field. We want females in the stem field, especially computer science, to feel empowered by the accomplishments of others who are like them, so they know they are capable of achieving great things!

What it does

Take a picture of yourself, upload it, and you'll see what woman in stem you look like and learn about their achievements!

How we built it

We used python and TensorFlow for Poets to retrain the model using images of WIS and compare the uploaded image. We used Google App Engine to host our website. We used HTML, CSS, and Javascript to create the structure and style of the website. We used flask as a framework for our website.

Challenges we ran into

At first we struggled to host the website and link our application to our Python code. We also struggled with storing uploaded images into the Google Cloud Bucket.

Accomplishments that we're proud of

We used technologies new to all of us, including TensorFlow for Poets, Flask, and Google Cloud Platform. We used machine learning for the first time. We were also able to have functionality and were able to create our minimum viable product.

What we learned

We learned how to use TensorFlow for machine learning, Flask to implement our application, and Google Cloud Platform to host our website. We learned how to use Google Storage buckets to store the images uploaded on the application.

What's next for empowHER

We hope to allow the user to take a picture through their device, increase the dataset, and create a domain. We also hope to improve the user interface.
