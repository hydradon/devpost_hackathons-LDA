Inspiration

With access to machine learning and advanced analytics becoming widespread, it is time to put a dent in violent crime in some of the biggest cities across the globe.

What it does

This site provides insights and a predictive dashboard to the police of Chicago to prevent crime and improve resource management

How I built it

Paul built the machine learning algorithms, Randy built the website

Challenges I ran into

Data cleaning, algorithm tweeking, and limiting the number of awesome visualizations to only what was necessary for a complete understanding.

Accomplishments that I'm proud of

We created a website and algorithms that could save lives and prevent crime!

What I learned

Machine learning is the future. The difficult, frustrating, beautiful future.
