What's next for The DaVinci Code

MORE ART

...Besides deep pieces of art that no one can argue with, the DaVinci Code can play hangman with you. It receives letters and draws the hangman if the letter was wrong. The user can send their guess to the robot via text message which uses Twilio to select the most voted for letter and communicates it to the main python script. We faced various challenges, mainly on the hardware site and being able to move the robot arm to an exact position 'cause physics... However, we developed a functional robot able to move in the x,y and z directions.
Our team consisted of members with various skill sets and backgrounds, complementing the multidisciplinary nature of this project.
