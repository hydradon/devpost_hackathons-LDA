• TeraRobo

Inspiration

Nowadays Amazon try to automatize sorting operation by robotics technology, but it need more time to realize it. I try to develop the prototype of sorting machine.

What it does

TeraRobo automatize a part of sorting operation in warehouse.

How I built it

I used Processing as integrated development environment, OpenCV as image analysis, Support Vector Machine as pattern recognition, Arduino as micro computer to control hardware. I wrote the code in Java.

Challenges I ran into

Automation of sorting operation by using image analysis and pattern recognition technology.
