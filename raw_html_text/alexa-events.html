Inspiration

We wanted a way to keep up with our events on Facebook, and wanted to make a hands-free way to interact with them

What it does

Gets events that are coming up, and events that are happening in a given location

How we built it

Uses the Facebook API to get events Uses bing maps to get geocode from a query Uses SUTime natural language processing to parse time instructions (e.g. tonight, next week, before May) All this included in an Amazon Alexa Lambda, written in Java

Challenges we ran into

Facebook API keys kept expiring The Alexa API is very difficult to debug

Accomplishments that we're proud of

Created a working interface for getting events Used geocoding to get locations Used natural language processing to get times

What's next for Alexa Events

Notifications Interact with events (accept invitations etc.) Google calendar implementation
