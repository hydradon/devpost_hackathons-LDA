Inspiration

We all wanted to experiment with the oculus rift

What it does

A flight simulator with myo hand gesture controlled velocity

How I built it

Unity and Visual Studio

Challenges I ran into

We began with creating a working unity environment that was compatible with the oculus rift, and then needed to figure out how to integrate the myo api and scripts into our project. There were some issues with the myo syncing and alignment.

Accomplishments that I'm proud of

We successfully integrated the myo into our oculus rift unity project

What I learned

All of us were either new to unity or beginners, so we all learned more about how the unity engine works. Some of us also learned more git version control tricks.

What's next for https://github.com/bdatdo0601/myoculus-rift

Better terrains, packaged assets so that we don't have a million files
