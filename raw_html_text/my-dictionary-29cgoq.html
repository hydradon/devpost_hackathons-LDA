Inspiration

Personal struggles with vocabulary. Also would be useful for international students here at Waterloo.

What it does

Allows you to build your own dictionary by saving words that you want to remember/learn. Can also quiz you on your words.

How we built it

Used Visual Studio Code, HTML, CSS, and Javascript to build the website.

Challenges we ran into

Web design was mostly new to us, so we had to learn all of the basics while we were building this project.

Accomplishments that we're proud of

We successfully built the website, and we were able to implement all of the core features that we wanted.

What we learned

We learned how to use different full stack technologies, and we learned to deal with unexpected challenges such as bugs that made our code unresponsive.

What's next for My Dictionary

We would like to incorporate this with other technologies such as Google Home or Alexa. We hope that this project will grow larger so that it can help people improve their language skills and have a positive impact on communities.
