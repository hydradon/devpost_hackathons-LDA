Inspiration

We love printed photographs, but they can be costly with the current available options. We decided to make our project to print pictures for cheaper.

What it does

Takes a photo with a raspberry pi camera and prints it with a thermal printer on common reciept paper.

How we built it

We assembled the components in a cardboard box.

Challenges we ran into

The biggest challenge we ran into was powering both the raspberry pi and the printer.

Accomplishments that we're proud of

It prints!

What we learned

We learned a lot about powering the device and understand the script we used better.

What's next for BumCam

Develop the housing for the components.
