• NectarSpot

Inspiration

Digital Signage has been always one way. Using Voice we want to move from passivity to two-way activity

What it does

Provides relevant information to the user, in this example portfolio on Fire TV, Echo Show Medium and Echo Show Large and Echo Spot

How we built it

Using Node.js, AWS Lambda, JSON

Challenges we ran into

  1. Designing of voice 2.Designing of display feature 3.Coordination between voice and display
  2. Support for different screen sizes and hubs

Accomplishments that we're proud of

One of the first digital signage that can used across industries including medical clinics, spas etc.

What we learned

Voice Design can be challenging as we have to make sure important areas need to be showcase first.

What's next for NectarSpot

We will enhance the skill and also utilize the framework to provide this as a service to companies across industries including medical clinics, retail front etc.
