Inspiration

We were inspired by the recent developments in genomic technologies and exploring what's possible with them. We believe that technology similar to GenMatch will become popular in the near future with the advancement of these methods.

What it does

Our product uses the genome of its user to recommend people to meet about mental health, physical health, friendships, work partners, and relationships. We recommend users people that have different or similar genes based on their choice. We then connect them by allowing them to set up a meeting and video chat with these people.

How we built it

We used flask as a backend and pure HTML and CSS for the front end with some jquery mixed in. We used the Genome_Link API to provide data for our recommendations, the DocuSign API to provide authorization of our use of your genes and the Twilio API to notify users when someone wants to meet them and to video chat with them.

Challenges we ran into

Using the Twilio video API was challenging because it was only provided in Node.js and we had to transfer this over to Python. We also had a bit of trouble deciding at first what our project would be.

Accomplishments that we're proud of

We are proud of supplying a working product by the end of the hackathon and also of one our members who did not sleep throughout the 36-hour weekend.

What we learned

It was our first time using any of these APIs so we learned how to incorporate these into our projects. We learned and now know how to use a JSON web token and ngrok are from Corey Weathers from Twilio.
