Inspiration

We wanted to create a solution geared towards the elderly who have many prescription medications they have to take and have trouble remembering which ones to take at certain times.

What it does

It sets up reminders for taking medication using LED lights.

How we built it

We used the Java Swing framework to create a GUI. We used an Arduino to set up flashing lights in intervals that the user can input to remind them whenever they need to be reminded.

Challenges we ran into

We couldn't send data to the Arduino's through computer files (Json). There were no Raspberry pi's, which would've made things easier since those can read directly from files and extract info from them.

Accomplishments that we're proud of

Actually finishing our first hackathon.

What we learned

We learned how to use the Java Swing framework and the arduino ide.

What's next for When are my drugs?

A mobile app for portability, and actual software-hardware syncing capabilities.
