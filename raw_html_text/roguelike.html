RogueLike

This game is based on Python 3. It's Dungeon game, called "HelloWorld" Once user start the game, they will see the menu screen with a picture of computer. They can start the game, clicking the "start a new game" button. '@' mark would be a character for the user. They will fight with two kinds of errors in this game. One is a syntax error, and the other is a null pointer exception. There are four items that user can use. All of those would be randomly generated for each stage. There is no certain ending for this game, so the user can play till they die (losing all their hp). If the user want to play more games, they can do as many games as they want. Enjoy the game!
