Inspiration

Building a website for a friend who runs a new music agency, allowing complete content to be shown on one website instead of multiple.

What it does

Showcases current artists, sponsors and shows, as well allowing new musicians to send mp3 files via emails.

How I built it

Using a form in C# I first created a form that would allow a user to send an email, then created the feature that includes an attachment. After that, I created a web form in Visual Studio and began creating the form in html and then translating the existing code into the .aspx file.

Challenges I ran into

Building the button click event that would allow the user to send the email as well as reworking the code from the cs file to the aspx.cs file.

Accomplishments that I'm proud of

Completing the component that allows the user to send an email as well as working out the form in html.

What I learned

New languages such as html and c# as well as using .aspx files.

What's next for Mack Delfino Agencies Website

Tackling more web design features Creating a pages bar that links each page together Completing the pages Home, Sponsors, Artists and Shows.
