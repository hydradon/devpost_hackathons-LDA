Inspiration

Finding an easy and seamless solution to a common problem we struggle with every semester.

What it does

Scans a syllabus in PDF form and beautifully displays key due dates and assignment information.

How I built it

Node.js, Express, Bootstrap, HTML, and CSS

Challenges I ran into

Setting up the server, communication between the client and server, and regular expression algorithms.

Accomplishments that I'm proud of

Creating a solution that is not currently available on the market

What I learned

Regular expression search, Express.js, Google Calendar API, cleint-server communication, PDF processing, group communication, and idea mapping

What's next for PDF_Scanner

Connecting the functionality with Google Calendar API and perfecting the parsing algorithm to handle multiple syllabus formats
