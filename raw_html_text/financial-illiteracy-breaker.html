Inspiration

Learning is best done in a fun environment when it doesn't seem like a chore. Distractions are common when you are trying to learn something new, but if the distraction is built in to the learning it will be more effective!

What it does

This Brick-breaker game allows you to learn any terminology in an entertaining flashcard type manner. All one has to do is have a text file with the flashcards and they are good to use this fun built application.

How we built it

We used Processing as our IDE sort of thing. It is a Java compiler that is really good for making visual environments. To make the whole game look cooler we used Fragment shaders for the background. The build came from a basic idea and set up for how brick breaker is set up.

Challenges we ran into

Making the ball bounce in the perfect way and having the blocks disappear and bounce was a bit challenging to fix and make a bouncing effect. There are still a few glitches in the cheat modes but this is definitely a work in progress.

Accomplishments that we're proud of

Bringing the finished project to a state that is usable for any memorization or learning of terms environment is a big accomplishment! Took hours of work but was a worthy project that even we will use in the future. This is one of those cool math games that kids would love using to learn things.

What we learned

There is a lot to learn in this field and we only know so much; this gave us an opportunity to learn new things which compelled us to push even further. We stated off thinking of only making a game, but after seeing the Swamphack challenges we had to incorporate this flashcard part. This addition made the whole game seem more complete as well.

What's next for Financial Illiteracy Breaker

Publishing it and hopefully getting other users to like the game as much as we do and to use it on a regular basis to make their learning experiences more fun as well.
