Inspiration

I hate having to cite my sources when writing research papers. And I especially hate having to stop my flow of work to look up a source before I forget.

What it does

HiTemp automatically finds, searches, and cites the information that you are typing inside a Google Doc.

How I built it

I built HiTemp using Google Apps Script which allows me to edit Google applications using javascript and html5

Challenges I ran into

I found it very difficult to pull information from an API and actually print it inside the Google Doc.

Accomplishments that I'm proud of

I am proud to say that I finished all of the back end code. but I have yet to connect the front end.

What I learned

I learned how to use APIs for the first time. I've always understood how they work and how to implement it, but this is the first time I have done so.

What's next for HiTemp™

I actually hope to implement HiTemp into a fully functional cross-platform plugin...depending on how well I do.
