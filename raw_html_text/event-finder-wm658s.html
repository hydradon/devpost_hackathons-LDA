What it does

It finds events from incoming outlook emails and add it automatically to calendar. You can apply filter to the mails. So that mails only match the filter will be scanned for events. It also finds emails and names in you contacts in the mail and add that emails and names as attendee in event. It also tries to find place. It then automatically reply to the sender with a message consist of time availability based on previous events in the calendar

How I built it

It is built with PHP, Microsoft graph and wit.ai
