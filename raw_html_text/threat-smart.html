Inspiration

Better be safe than sorry.

What it does

Get crime statistics in the UK.

How we built it

Uses Twilio to get what data the user wants and queries UK police database API to inform user about threat level.

Challenges we ran into

Twilio documentation.

Accomplishments that we're proud of

Fully functional program.

What we learned

Using Twilio, Azure and c#.

What's next for Threat Smart

The main limitation of the program is that the UK police database is not updated regularly. With live data the app could inform the user when the person enters a high risk area. This can be easily done via user's live location. The AI can be trained further with more sample data.
