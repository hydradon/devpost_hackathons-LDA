Inspiration

Problem- A lot of girls are discouraged because they don’t see a lot of females as leaders in the world of technology. Solution- This website allows young emerging leaders to take a quiz which matches them with a female leader in tech based on their skills and interests which would motivate and encourage them to pursue a career in tech. We have also attached some great tools for young women to learn programming languages such as Code Academy, Udemy, and others.
