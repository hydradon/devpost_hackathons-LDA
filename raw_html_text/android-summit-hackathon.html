This is a sample app to help hackathon participants get started on hackathon day!

It includes example integrations with the following APIs:

  • Nessie - api.reimaginebanking.com (Capital One's Hackathon API)
  • Firebase - firebase.google.com
    • Firebase Auth
    • Firebase Messaging
    • Firebase Realtime Database
  • Card.IO - https://www.card.io/ (Credit Card scanning API)
