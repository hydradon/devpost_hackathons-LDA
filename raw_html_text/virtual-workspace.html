What it does

You put on the Oculus Rift and you are immersed in an epic experience known as virtual engineering graphic and design. You will have the ability to share all of your wonderful work in one file so that people you send it to can work with it. Like a high powered, virtual 3D google docs.

How we built it

We built it using unity, amazing C# skills, and the wonderful MLH hardware desk. You too can make it if you own 2 myo's, an oculus rift and a high powered laptop. (we are working on lowering that already low cost)
