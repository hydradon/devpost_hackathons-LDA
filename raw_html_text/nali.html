Inspiration

Often times people get lost in public areas, and often it's a hassle to try and reunite with one another. We sought a way to solve this issue.

What it does

It works as a compass that points in the direction, of the person you seek to find, and how far you're from them.

How we built it

We build it by taking advantage of Swift's many library, and the simpleness of implementing Mongo's database system.

Challenges we ran into

There were members who did not have a Mac in order to be able to run and test the Swift code for the ios application. This limited who could work on the front end of the system.

Accomplishments that we're proud of

Being able to implement a node server within a mobile app, ultimately creating it cross platform. Being able to learn Swift to the point we were able to create an application.

What we learned

Swift Language, including the process of making REST API calls in iOS development. Implementation of node servers within mobile apps.

What's next for Nali

Facebook integration, push notifications, and the capability to work for groups of people at once.
