Inspiration

Facial recognition had been something we really wanted to do for a long time, but we did not know exactly how we would implement it into our project.

How it works

We utilize the Kairos API to enroll Twitter profile pictures into our facial recognition database on the event that a Twitter user posts something about Bitcamp. When a user uses our app it uploads their snapshot to imgur and checks that picture against the ones in our database for a match.

Challenges we ran into

The biggest challenge was using facial recognition with images that did not show facial features well.

Accomplishments that we're proud of

We're proud of creating a working web app from scratch in 36 hours.

What we learned

Backup more frequently, version control is very important.

What's next for Twifr

First, we wish to expand the brand to be utilized at more events. Later on we would like to use this idea to allow recruiters to be able to pull info about potential applicants in real time.
