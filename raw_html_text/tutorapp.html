Inspiration

We know how hard classes can be, and how hard it is to get a job while you are studying. So we decided to make this app.

What it does

Sets up students with tutors so they can get help, while also making the students the tutors to help them make money.

How I built it

It's build on android 4.4 kitkat

Challenges I ran into

Time constraints and Remote Database implementation.

Accomplishments that I'm proud of

Basic layout and idea is solidly done. All that is left it do tie the knots.

What I learned

Planning before you make it is very helpful.

What's next for TutorApp
