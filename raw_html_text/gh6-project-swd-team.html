Inspiration

After doing some user research, we found that there is a big problem for homeless people to find emergency shelters close to them with available spots.

What it does

I. Match homeless people and shelters based on availability, closing time and demographics

  1. Emergency shelters can get vivid data visualization about average checkin time and demographics of people checking in
  2. Emergency shelters can have efficient checkin and checkout
  3. Provides information-sharing platform among agencies

How I built it

We worked for 3 days non-stop!!! For the frontend, we used Angularjs to get and post data to the database, and used google map API to present the map to the homeless people For the backend, we used tornado to run and host our websites

Challenges I ran into

1.Ran in lots of bugs because of incompatibility between angularjs and other languages.

  1. Unable to get data from the backend because of some format or request error.
  2. Get google map to work ## Accomplishments that I'm proud of
  3. Absolutely awesome user interface
  4. Successfully solved the problem of finding available shelter fast and get recommended arriving time to help homeless people to do decision making and make their life easier
