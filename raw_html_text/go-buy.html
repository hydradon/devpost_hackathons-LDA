Inspiration

Knowing before starting that compete in this hackaton was very difficult due to the high level, we choose a project "for learning", moreover, we have tried to fix a common problem that we sometimes face when we go out for buying. How many times we have gone to a shop exclusively for buying a product and it is gone?

What it does

Although is still a "baby project", the idea is to create a network where sellers can post their stock online, allowing the costumers to see if the product they are searching for is in the shop or not. In terms of eficiency, not only we just save time but we save fuel, too.

How we built it

Jordi took care of the back-ending part and I took care of the front-ending one.

Challenges we ran into

As a newbies, we ran into new programming objectives, such as using new API's and packages for us like jquery, bootstrap...

Accomplishments that we're proud of

Jordi and me have been working together for a few years, but never so much time in a once. It is the first project we start and finish (the others got stuck in some point), and we're very proud of.

What we learned

The use of the mentioned resources and an improvement of our programming skills.
