• Our Working Demo (a website) - Press "Snap" to take a photo with the device camera.Press "GoGoMusic" to access a Spotify playlist that fits

Inspiration

Imagine going on a road trip with friends. Have you ever experienced the amazement when the music playing from the radio just clicks with scenery around you? We’re here to bring you such moments of wonder, and more!

What it does

To find music that fits the scene, users can use our website prototype to snap a picture using the computer camera by pressing "SNAP"; then, users can press "GOGOMusic", and they will be led to a Spotify playlist with music that fits the scene in the picture.

How we built it

We built this project using scene recognition from Google Cloud Vision AI to automatically label the picture we captured and identify the most relevant feature in it. We then used Spotify API to query playlist most relevant to the feature. The user interface is a webpage built on Django framework and we're planning to develop it into Android App.

Challenges we ran into

We had a hard time integrating it into an Android App and we are all unfamiliar with the integration of the components into the website.

Accomplishments that we're proud of

We made used of Google API effectively to successfully identity relevant feature and learned a lot from combining different APIs together.

What's next for Go Go Music

We will train the model using data from ourselves and use accelerometer and location data to provide customized suggestions.
