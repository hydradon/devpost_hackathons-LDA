Are you trying to learn a new language? Are you tired using flashcards? Polyglot is fun way to learn the names of the objects around you in the language of your choice.

What it does

Polyglot detects the objects around you and labels them in the language of your choice . You can double tap on the object to know how it is pronounced.

How we built it

Polyglot was built in Android Studio . TensorFlow ObjectDetectionAPI was used for detecting the objects and Google Translate API was used for translating the object labels in the selected language.

Challenges we ran into

It tooks a while to install Bazel in Windows .Also, getting the coordinates of the identified object proved to be challenging as well.
