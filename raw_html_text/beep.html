Inspiration

Tailgate drivers behind you are annoying and dangerous.

What it does

The sensor produces an audio and visual response to an approaching vehicle.

How we built it

Arduino, LEDs, buzzer, op amp, capacitor, resistors, train set

Challenges we ran into

detecting speed, connecting to a radar sensor

Accomplishments that we're proud of

interfacing with an arduino, successfully configuring the radar sensor to act as a motion sensor

What we learned

how to program an arduino, how to use github, how to use a radar sensor

What's next for Trainwreck

speed detection
