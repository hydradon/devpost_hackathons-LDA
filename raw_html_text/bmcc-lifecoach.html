Inspiration

Our inspiration comes from personal experience with the counseling center.

What it does

BMCC LifeCoach provides anonymity for counseling to help students feel comfortable and give awareness to the service.

How we built it

It is built with nodejs, xamarin, aws lambda, aws dynamodb, visual studio, github for source control.

Challenges we ran into

Limited amount of time; Integerated Development Environment crashes oftenly; and lex chatbot didn't get implemented.

Accomplishments that we're proud of

We're able to work as a team, and were able to contribute in the limited time frame given. Most of all, the experience we got from this.

What we learned

Integrated Alexa Voice skill. AWS Lambda. AWS DynamoDB

What's next for BMCC LifeCoach

Implement chatbot and mobile platform.
