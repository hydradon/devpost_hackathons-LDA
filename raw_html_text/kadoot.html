Inspiration

We enjoy kahoot, we wanted to be able to play without having knowledge in the questions that we are being quizzed on.

What it does

It generates random numbers which have blocked out digits each round and you have to guess what the number actually is. There is no skill, just luck. Simply put, it's gambling.

How we built it

We set up a firebase realtime database and accessed it with a react front-end. We spent quite a bit of time trying to get used to using the websockets and the firebase API.

Challenges we ran into

The firebase API is quite limited for react projects as well as typescript.

Accomplishments that we're proud of

Getting the front-end to work with the database and being able to view live updates on different devices.

What we learned

We learned that typescript is not easy to work with the firebase API for hackathons.

What's next for kadoot!

Polishing up the game website.
