Inspiration

A simple calculator is a easiest thing to build in any language, so we choose to make this since we have no knowledge about building an Android app.

What it does

It calculates the CGPA and credit hours taken in college of the user. It also calculates how many credit hours still need to take. It also allows users to set a GPA goal for themselves.

How I built it

We built it by using android studio.

Challenges I ran into

This is our first time getting into Android so this is sort of like our first experiment.

Accomplishments that I'm proud of

We have finally built our own first app!

What I learned

We have learned some basics design and coding about Android.

What's next for GPA Planner

We will be getting deeper into Android .
