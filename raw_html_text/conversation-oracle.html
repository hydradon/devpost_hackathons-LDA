Inspiration

As a group, we are not the best at making small talk. We wanted to create an app to help us and others practice conversations.

What it does

It is a simple mobile friendly browser game that has linear gameplay and two sprites.

How we built it

We used create-react-app for our base web app and are hosting it with firebase. We started by discussing structure of our web app, and splitting into teams, where some focusing on the front-end, and some on the backend. We had an experience leader that led the team, setting tasks and troubleshooting bugs.

Challenges we ran into

Our first idea was to record audio and interface with an API to tell the user's emotions. Unfortunately, we could not find a suitable api that worked well with React. So instead we wanted to focus on the content shown in the web app, but still trying to see throughout how we can implement our original idea.

Accomplishments that we're proud of

Our website it clean looking and complete. The core functionality of the web app had finished, giving us extra time to work on making it look right. Our CSS Lead made the website beautiful while not over complex, and providing our icons and drawings here. We were very proud of making of the app multi-platform.

What we learned

How to work as a team and use Git Kracken, a ot learned React for the first time, and dealt with tricky API's

What's next for Conversation Oracle

Emotion reading and speech as an input.
