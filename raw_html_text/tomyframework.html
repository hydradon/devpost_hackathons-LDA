I like UiPath ReFramework when using Orchestrator Queue, it's very simple and easy to use. However, when we meet some requirement that can't use Orhcestrator, we need to modify the ReFramework, which is a bit troublesome, so I come out with an idea to create a similar framework that is simpler and based on Execl or you can easily change it to csv type.

Looping and processing data from Excel will become very easy by using this framework. The framework takes care of looping the excel, handling error, read config file.

For detail, you can find it in the video.
