Inspiration

Travelling is pretty expensive. We want to make it as enjoyable as the experience of traveling itself.

What it does

Provides vacation suggestions based on user preferences, location, and budget.

How we built it

We built it using Node, Express, React, American Airlines Provided Airport list & Google Maps API

Challenges we ran into

Working with frontend frameworks. Hooking up React with the node server. API calls taking a lot of time.

Accomplishments that we're proud of

We built our own mock flight generator API to account for flight info that is not provided. We learned to build efficient scalable full stack applications.

What we learned

We gained confidence in working with frontend frameworks. We learned to create our own APIs.

What's next for Flites

We plan to work on improving the queries with real data. We also plan to open source the mock flight generator API.
