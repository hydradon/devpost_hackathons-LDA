Inspiration

We used to believe in many stereotypes due to media and social influence. This all changed when we met and connected with individuals that fall under the stereotype on a personal level. Hence, to help people truly understand and connect with people around the world, we created "Ask the World".

What it does

Our web app allows users to ask, answer, and discuss questions with people around the world.

How we built it

We tested with different APIs, but we decided to implement the idea ourselves by building our frontend is in vanilla JavaScript which connects to our Firebase database.

Challenges we ran into

We ran into a maximum recursion depth reached error where while the database was listening to changes to edited the database. Hence, we could not break out of this loop. Afterwards, we realized that we can fix this by implementing a new Firebase method that only listens once.

Accomplishments that we're proud of

Getting the project to work perfectly!

What we learned

We learned how to connect both Javascript to Firebase!

What's next for Ask the World

We plan on releasing the MVP to schools across the world.
