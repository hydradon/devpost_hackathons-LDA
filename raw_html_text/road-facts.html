RoadFacts

Visualization and analysis of Indian road statistics based on multiple parameters.

Available visualizations:

  1. Road accidents analysis - Find out what is causing accidents on Indian roads. Analyze road accidents based on various parameters like road condition, type of vehicle, weather condition, driver responsibility and many more.
  2. Road statistics - State-wise road statistics for various categories like Highways, Rural roads, Urban roads etc.
  3. Transit assets statistics - Availability of transit assets like bicycle, scooter/motorcycle, car/jeep/van to Indian households. And how does the availability differ for urban vs rural India. Based on Census 2011 data.

Data Sources

  1. Road Accidents - https://data.gov.in/catalog/road-accidents-india-classified-according-various-parameters
  2. Road statistics - http://morth.nic.in/showfile.asp?lid=2445
  3. Household transit assets - http://www.censusindia.gov.in/2011census/Hlo-series/HH12.html

Tools/Libraries Used:

  1. Google charts for Visualization
  2. This is a completely client-side application with no server component. Google spreadsheets is used as backend datastore.
  3. Tabletop used for getting JSON data from Google Spreadsheet.
  4. JavaScript SQL Library alasql for filtering, grouping, sorting data fetched from spreadsheet.
