Inspiration

We wanted to learn Android and have fun at the same time.

What it does

Dodge different coloured balls. When same coloured balls collide, points are earned. Colliding the sprite with a different coloured ball ends the game.

How I built it

We did research and used the techniques that we learned online for our own purposes.

Challenges I ran into

Checking for collisions between multiple objects is tricky to get right. App flow is hard to understand at first. Drawing 2d graphics with only x and y coordinates was interesting.

Accomplishments that I'm proud of

Learning Android and producing a functional app with limited time.

What I learned

App flow. Programming with people. Make simple apps. 2d graphics require math.

What's next for Circles

Leaderboard. Sounds. Increasing difficulty. Fixing small bugs
