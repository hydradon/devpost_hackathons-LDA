Inspiration

Our environment is constantly being harmed by the products that we purchase. Whether it be by toxic chemicals used to make products or harmful byproducts production plants produce, the products we buy are slowly harming Earth's environment.

What it does

It shows an Amazon customer how environmentally friendly a certain product is on Amazon.

How we built it

We used Javascript, HTML, and CSS to change the source code for Amazon pages. We used an API for the environmental impact of cosmetic makeup products to hold all of our data.

Challenges we ran into

It was difficult to modify the code on Amazon pages, since we usually do not edit developed, complex web pages like Amazon product web pages.

Accomplishments that we're proud of

We are proud of publishing the chrome extension, and learning new things along the way.

What we learned

We learned how to make and publish a Google Chrome Extension. We also learned how to use Adobe Premier to make a logo for our app.

What's next for EcoZon

Adding more API's/databases to expand the amount of products EcoZon covers. We will also add a popup in the Amazon web page to display which database/organization we got the safety rating from and any other relevant information.
