Inspiration

If you are making some meeting, it can be problematic to keep update with already bought items and sending all time messages about them.

What it does

You can create groups with shopping list with items you want to buy.

Challenges I ran into

The biggest challenge was making it solo in short period of time.

Accomplishments that I'm proud of

I finished minimum valuable product working solo.

What I learned

I learned how to work under pressure and how valuable sleep can be.

What's next for shoppingLists

"Reservation system" which will prevents from buying the same things. Better design, especially for desktop devices. Change MySQL structure to allow creating more items.
