Inspiration

What inspired us to build this app;

Workspace App tagged 'Familiar with your destination before getting there or on the spot". It was a day out in a big Shopping mall, located in Lagos, Nigeria. I was overjoyed that I was just entering every available location in the mall and getting back myself, I discover that being so much enclosed that I couldn't find my way back to the entry of the mall, and it can be so embarrassing when you need to ask your way out. So that done on me that if assuring I have something that can just show me my way out, it will have to save the embarrassment of asking someone else way out. And to add to the story, If I have to know my way before coming to the mall (I mean if I have to know how to navigate my way before visiting the mall, I will have not been overwhelmed of how big the mall is and it will have gives me any ideas of how thing work. This gave birth to this workspace app build by our team.

What it does

What it does is that it allow you to explore and virtually tour an environment before you visit or check-in.

How we built it

It was built using Facebook VR technology call react-360 and some other library to make it function properly.

Challenges we ran into

Challenges we ran into are the privacy people give when we want to take the workspace picture and getting information from them on how the place can be explored.

Accomplishments that we're proud of

The accomplishment that we're proud of is being able to start this project successfully and build something out of it.

What we learned

We learned some much on this project especially in the libraries we use to build it like react-360 which is very new to us,.

What's next for Workspace Locator

Workspace Locator is going to be a revolution or a game change for virtual reality in exploring and familiarizing with places before visiting or checking. WS-Locator, as we use to called it, will be a global app that will make tour and VR things to work with.
