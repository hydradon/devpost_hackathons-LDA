Inspiration

Health is the most valuable gift. Technology can be used to facilitate in becoming more healthy.

What it does

This app gamifies the steps count from the Apple Health app in a social way. The users see the step count of the different people that they have invited.

How we built it

We built the backend in GO, the front end in flutter. Ideally we wanted to have the step count from Google Fit for android as well.

Challenges we ran into

We ran into android google fit data fetching problems. Also there was a lot needed to be done in Go just to get the basic stuff working. Type mismatch! String to int conversion! Invalid date parser in dart!

Accomplishments that we're proud of

Making a front-end and back-end with a steep learning curve!
