Inspiration

鬼畜视频近来作为

What it does

利用 Microsoft Cognitive 中的 Speech API 将用户的输入转换为文字，再通过训练好的 LUIS 模型对输入进行分类，得到对应的鬼畜源分类，再利用 Windows Presentation Foundation 按照顺序播放用户输入中涉及的鬼畜源。

How I built it

全程使用C#，其中还使用到了 Newtonoft.Json 库

Challenges I ran into

切割视频源是一件很辛苦的工作 (:з」∠)

Accomplishments that I'm proud of

最终生成的鬼畜短片十分洗脑

What I learned

使用 ffmpeg 进行音视频切割/转换，使用 LUIS 模型进行语言分类与识别

What's next for GuichuDirector

进一步加强鬼畜的拼接和素材能力，争取能够合成更有意义的短片
