Inspiration

ალბათ ხშირად გქონიათ "პრობლემები" თქვენს კომპიუტერთან... ხშირად გიფიქრიათ იმაზე რომ ის უფრო მეტად კომფორტული ყოფილიყო მოხმარებისთვის, ქონოდა თქვენთვის მოსაწონი დიზაინი, თქვენთვის სასურველი ფაილი ყოფილიყო უფრო "მარტივად მისაგნები" .

ასევე, სისტემური პრობლემების დროს, თქვენ გიწევთ 20 ლარიანი "IT Proffesional-ის" გამოძახება, რომელსაც შესაძლოა არანაირი ცოდნა არ გააჩნდეს და მხოლოდ კითხვა თუ იცოდეს...

What it does

ამ და სხვა პრობლემების გადასაჭრელად, მე შევეცადე შემექმნა სისტემა რომელიც, იქნებოდა ნებისმიერი მომხმარებლისთვის მოსახერხებელი, ექნებოდა სასურველი დიზაინი და გადაჭრიდა სხვადასხვა პრობლემებს.

How I built it

გდავწყვიტე რომ, უსაფრთხოებისთვის და ხელმისაწვდომობისთვის ეს სისტემა ყოფილიყო web-ში. მიზნის განსახორციელებლად გაოვიყენე command promt(შესაბამისად ეს სისტემა ჯერჯერობით მხოლოდ ვინდოუსზეა აწყობილი) და შემდეგი პროგრამული ენები: php, javascript, css/html.

რა ვისწავლე

ასე ვთქვათ, მე გადავგდი ახალი ნაბიჯი ჩემს სფეროში. და კიდევ თურმე რამოდენიმე საათით სახლში რომ წახვიდე და დაიძინო, დაჩლუნგებული აღარ ივლი და უფრო გააზრებულად დაწერ კოდს.

What's next for Web Windows

შემდგომში , თუ დამეხმარება მცოდნე ხალხი ვაპირებ რომ დავამატო უფრო მეტი სერვისი, მეტი შესაძლებლობა და მაქსიმალურად გავაუმჯობესო ეს პროექტი, რადგან ალბათ ბევრ ადამიანს გამოადგება ასეთი ტიპის Desktop-ი.
