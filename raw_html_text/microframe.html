MicroFrame

MicroFrame is a small Android application with exactly one feature: It displays a picture from the device in a widget frame on your home screen.

There are no other features. No ads, no promotions, no superfluity, no fluff, and no filler. If you want to do the one thing it does, it can do it for you.

Could I have added more features? Probably. Feature creep is something that one usually has too much of, rather than too little. But the faster we make our computers and our internet connections, the sloppier our use of those resources becomes. Scarcity teaches virtues that are lost in plenty. MicroFrame is deliberately designed to have the smallest fingerprint possible: it appears nowhere but the widget screen where it performs its function of adding a picture frame to the home screen, and it runs no code aside from what is necessary to create the picture frame.

Testing has only confirmed that it works on Android 7.0 so far, so if you get a crash on 6.0, get a friend with 7.0 to test it. Still working on API versioning.
