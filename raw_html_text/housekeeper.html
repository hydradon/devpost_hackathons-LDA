What inspired you

Problems between housemates on doing things such as washing, cleaning etc.

What it does

Resolves conflict by utilizing a voting system which notifies the user its time to perform a task once the rest of the house decided so.

How I built it

Using python and postgresql for back-end and web front-end.

Challenges I ran into

Updating the database and relating it to the web front end.

Accomplishments that I'm proud of

The project works along with separate user account and send out emails and sms when the user should be notified, its an interesting concept and should we continue with it after the hackathon, our friends will certainly use it.

What I learned

It is difficult to deal with python flask at 5am.

What's next for Housekeeper

Hopefully me and Ed continue to build our app.
