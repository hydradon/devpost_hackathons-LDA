Inspiration

The original idea was to get an original destination from your favorites movies, series, books, and songs.

What it does

You select the tv series you like most out of the available and using Finnair API, it searches for the cheapest flight ticket to travel in the next week

How we built it

We used Python Flask, HTML and lots of love

Challenges we ran into

Apart from the fire alarm in the last minute, we had problems with API of movies, series, and books. Basically, no good API for our purposes

Accomplishments that we're proud of

One of the integrants never used Flask before

What we learned

Flask and some basic alarm situation

What's next for NeverLand

Include books and movies
