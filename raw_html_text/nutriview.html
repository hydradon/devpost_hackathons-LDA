Inspiration

Providing easy, reliable, and accessible nutritional information.

What it does

NutriView scans in a photo of your food and runs it through AWS's rekognition api to determine the food in the photo. Searching for the nutrients of this food through USDA's nutrition api, we return an accurate nutrition label back to the user.

How we built it

We implemented our idea through a web app, using Django, bootstrap, and python. We used AWS's rekognition api as well as the USDA's nutrition api. We hosted on Google Cloud.

Challenges we ran into

Our main challenges were implementing a working camera on the mobile version, as well as requesting information from the USDA and displaying it in an aesthetic manner.

Accomplishments that we're proud of

The whole project!

What we learned

Using apis, parsing JSON, hosting a web server on an Ubuntu vm, svg animations.

What's next for NutriView

Adding accounts, saving nutrition, created displayable data statistics on food consumption, options for place of purchase/brand of food, and making it into an app.
