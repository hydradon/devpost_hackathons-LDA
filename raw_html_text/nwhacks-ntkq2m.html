Inspiration

We were looking to develop a better way for groups to collaborate on tasks where teammates can keep track of which tasks are currently in progress and which tasks have been completed. This can be applied to school projects as well as keeping up with chores around the house.

What it does

We created a simple collaborating system to facilitate team work. Our system helps team members keep track of the tasks they need to work on and who is currently working on them.

How we built it

We used Android Studio for our IDE and AWS mobile hub's provided NoSQL database for our backend.

Challenges we ran into

We had some trouble connecting to the AWS NoSQL database but eventually got the app to send data successfully. However, we were unable to complete retrieval of data in time.

Accomplishments that we're proud of

Both of us were fairly new to the Android Studio IDE so it was a good learning experience in terms of developing the front-end.

What we learned

We learned how to connect pages together on Android Studio and setting up AWS Mobile Hub services for Android.

What's next for nwHacks

We definitely want to look into expanding the project further such as tying up loose ends and adding more features.
