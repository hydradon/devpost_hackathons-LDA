Inspiration

Based off of Momentum, (Chrome Extension). Inspired by the focus on mental health and sustainability hacks in LAHacks.

What it does

Tells the time (both: analog and digital) Tells the weather (in degrees Fahrenheit) Uses the google api (Natural Language api) to analyze the sentiment of the user when he/she types their answer to the question ("How are you doing today?"). The site creates a graph of responses over time, for the user to see his/her mood over a long period of time.

How we built it

We used: flask, html5, javascript, python, and the Google API

Challenges we ran into

Running scripts on the backend, setting up a server, creating a clean design

Accomplishments that we're proud of

Learning how to use the Google API along with a weather api to get data

What we learned

Using APIs and Flask, along with improving our web skills

What's next for Steve: personal assistant

Adding more features like todo lists and getting more data from our apis
