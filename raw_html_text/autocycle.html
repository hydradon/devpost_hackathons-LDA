Inspiration

Billions of dollars wasted sorting processes that can be automated/assisted through ML.

What it does

Takes in images of objects/materials and runs them through an ML trained model to analyze. Determines materials of the object and whether they are recyclable or not.

How we built it

Android mobile application to take pictures of objects/materials and send to ML model for analysis. Express.js server for communication ML trained model to analyze objects' materials and recyclable properties.
