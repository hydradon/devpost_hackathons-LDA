Problem solved

CUNY-Resource-Communications Project and The Future Is Female.

Inspiration

Increasing the graduation rate of single mothers in college by connecting them to reliable, accessible childcare.

What it does

Establishes a community of single mothers in school who help each other by providing childcare for each other.

How we built it

We used Javascript, Google Maps API, html, css and bootstrap.

Challenges we ran into

Database systems.

Accomplishments that we're proud of

We were able to leverage each others' strengths in completing our project.

What we learned

We learned about bootstrapping a website and using Google Maps API.

What's next for BabySurf

Build a back-end relational database to track member information; build a fully-functional mobile app for members who do not have access to a computer.
