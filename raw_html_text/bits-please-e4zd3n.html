Inspiration

We think that there is already an IoT infrastructure in the major cities that is currently underused. Most of them are generating high amounts of contamination every day that are harmful to the citizens.

What it does

CityManager monitors and controls the main sources of contamination to reduce them and get cleaner cities. Also, the citizen could be aware of the condition of different areas and actively participate in the process of making a cleaner city.

How we built it

We have different components using different technologies:

  • Arduino deployment connected to a raspberry-pi that has a python server using rest and coap for communicating.
  • Cloud service running in python (connected to the raspberry-pi using coap), offering a rest api for the mobile app.
  • The mobile app is using Flutter. It connects to google maps and to our cloud service.

Challenges we ran into

As our project is covering different areas. We ran into two challenges: Nokia challenge and Ericsson challenge.

Accomplishments that we're proud of

We are proud of the prototype that we built and in the product idea that we got. We think that a product with these features could be really beneficial to our society and future.

What we learned

We learned the technologies that we are using. Most of us didn't have experience with some or most of these technologies before the hackathon.
