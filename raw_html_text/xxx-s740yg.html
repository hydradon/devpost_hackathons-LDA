Inspiration

Cryptocurrencies have become more and more popular but still, it is not mainstream. One of the main reasons that it is not being adopting widely is its high volatility. Companies address this issue by creating stable coins. But there are a variety of stable coins in the market and sometimes, we might not just want to stick with one.

What it does & How I built it

With the help of set protocol, we can now send payments with a basket of stable coins of your choice. A stable set is issued based on the proportions of the stable coins the user wants to distribute to use and act as an collateralization of the stable set.

This project is a react web app integrated with web3 metamask which demonstrates the process of issuing a stable set with 1 Stable set = 0.5 TUSD + 0.5 DAI by calling the Set protocol. All tokens are using kovan test network.

Challenges I ran into

I was blocked at one point for issuing a set, and the Set protocol team was very helpful. Also took me a while to configure the react-native app.
