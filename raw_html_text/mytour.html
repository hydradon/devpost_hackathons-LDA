• First enter a city you want to visit.

  • Enter what type of trip you are looking for.

  • See the list of tour guides and the places you will visit on each tour.

  • Then view an over view of the entire tour up close with the WRLD APi

We were inspired by the concept of making travel easier for people as we always thought sometimes being a tourist, it is hard to know what to search for to see.

My tour basically allows you to input a place you want to visit, what specifically you want to visit, all will show you a bunch of tours accommodating your request. With various tour guides and you can see tour rating,

We developed it using web development and the WRLD API.

Some challenges we ran into was connecting all the different components together.

We were proud of getting it to work.

We learned how to use the WRLD API.

We want to build more features and add even more user functionality.
