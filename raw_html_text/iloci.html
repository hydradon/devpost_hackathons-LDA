Inspiration

We were inspired by the creative method of Loci to tackle the problem of forgetfulness.

What it does

It presents an Augmented Reality model for you to visualize a list of items you want to remember by letting you add images. Each image represents an individual alphabet, making memorization easy. Once the images have been added on the camera, you can walk through the location and review an actualization of your mind palace.

How we built it

We built it using xcode, swift and Apple's Augmented Reality Kit.

Challenges we ran into

Using Swift for the first time

Accomplishments that we're proud of

Being able to add AR objects on screen based on the user's choice of letter inputs.

What we learned

Swift, xcode, using the ARKit.

What's next for iLoci

-Letting the user add words and numbers. -Letting the user upload their own images.
