WebReg-Chrome-Extension

Chrome extension that implements several functionalities into the Rutgers WebReg system. We wanted to speed up the process of registering for classes in an efficient way. The extension is able to automate checkbox clicks, directly link to ratemyprofessors, and show conflicts for classes. This is our first time making a web extension =)
