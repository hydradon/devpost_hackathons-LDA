Inspiration

As college students, we are inspired to try to save money and by making it interactive, it would make saving money fun for other people our age as well.

What it does

Users would input the goal they want to reach and whenever they spend money, the dog will be closer to death while if they save more money, the dog will live a fulfilling life.

How I built it

It is built using HTML and CSS as a design element and javascript to implement the codes.

Challenges I ran into

Making sure the images changing with every input the user enters.

Accomplishments that I'm proud of

Learning a new element in a short amount of time and being able to create a website with it.

What's next for Crazy Rich Corgi

Make it responsive and mobile-friendly so it would be more portable.
