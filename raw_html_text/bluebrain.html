Inspiration

Security is a modern issue everyone faces, why not use your brain to solve it?

What it does

Utilizes EEG feedback to lock/unlock.

How I built it

We used arduino and the wearable sensing DSI24 system

Challenges I ran into

The interface was difficult to merge together into a single project

Accomplishments that I'm proud of

We were able to better understand the complexities of an EEG system and the work required to complete a group project

What I learned

EEG signaling, Python packages, Arduino UNO BLE system

What's next for BlueBrain

Use more complex signaling through OpenVibe
