I had a strong desire to use the Reddit Api and Python and when I saw the Tone Analysis tool from Watson I knew the subreddit r/ShitTrumpSays was where I would be. I extract the most recent 250 trump quotes and run them through the analyzer then return and graph the results. I am nearing the finish of a chatterbot as well so you could talk to a Trump-bot (sounds like a horrible time) and learn more than you would ever want to know.
