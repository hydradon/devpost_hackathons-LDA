Inspiration

I created a SMS application linked to the Automatic dataset to notify drivers when and where they experienced a hard brake incident and remind themselves to practice safe driving on the road!

What it does

A user sends a text "BRAKE" to a designated telephone number. After the user types the the text BRAKE, he or she gets a text response asking if he or she wants more detail of any Hard Brake event in certain time period. After he or she texts back to a request for more information, the server returns any data associated with any hard brake incident.

How I built it

I built using the Twilio API and as well as the dummy Automatic dataset. I also used webscript.io using Lua to implement the backend of the web service for the SMS web app.

Challenges I ran into

I had a challenge integrating Automatic's Webhook Api with Twilio API and webscript.io

Accomplishments that I'm proud of

I am happy that I was to create a simple, yet useful application for the Transportation community to build on.

What I learned

I learned a lot of Automatic API and Webhook and importance this technology will have to improve smart driving for San Francisco Bay Area drivers.

What's next for Hard_Brake_Notification:

To integrate the SMS application with Automatic API (WebHook) so that when the car is turned off and driver experienced a Hard Brake event, a text message is send to him or her about the details of the incident and remind him or her to practice safe driving on the road.
