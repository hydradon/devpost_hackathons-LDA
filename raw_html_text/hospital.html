Hospital

Set up on Computer (code in quotes " ")

  1. git clone
  2. install node.js
  3. navigate to Hospital folder
  4. "npm install"

5.run server: "SET DEBUG=hospital:* & npm run devstart"

See In Browser:

  • got to: http://localhost:3000/

Use with database:

  1. download MongoDB, add program folder with executables to path
  2. restart server in another cmd window
