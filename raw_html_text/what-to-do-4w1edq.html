What it does

This web app takes a few Devpost tags and tries to come up with "semi-realistic" project ideas.

How we built it

We scraped around 4000 projects from Devpost with C# and put it into a Markov chain to generate new ideas. This was put into a web backend so that it can generate ideas given a few tags to get it started.
