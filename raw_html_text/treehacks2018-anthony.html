Inspiration

I was inspired to do this project after searching for musical advice from masters for the piano pieces that I was working on but many piano masters are European and unfortunately I do not speak any languages other than spanish or english. This is when I decided to translate the videos so I could read from them whenever I wanted to.

What it does

This project uses Houndify to read the language from youtube videos and then converts the text, using the Google Translate API, to any language.

How I built it

I built it with javascript, a java servlet, and a php server.

Challenges I ran into

The hardest challenge was reading the youtube video correctly.

Accomplishments that I'm proud of

I am proud of my successful implementation of houndify and the Google Translate API.

What I learned

I learned how to make a java backend.

What's next for Youtube Translator

Next, I will take the translated text convert to a speech MP3 and replace with the old audio of the video.
