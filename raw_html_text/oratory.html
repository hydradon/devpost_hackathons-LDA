Inspiration

We were thinking about inconveniences we face during school, and one idea that came up was speech memorization.

What it does

Oratory helps the user memorize a given speech.

How we built it

We used Android Studio, Google speech-to-text API, and Classifer4J Summarizer API.

Challenges we ran into

Putting it all together and transitioning between reading to summarization to memorization were the main points of difficulty.

Accomplishments that we're proud of

The whole app ended up working the way we envisioned it.

What we learned

We learned how to use speech-to-text API, Classifer4J Summarizer API and tricks with Android UI.

What's next for Oratory

Oratory plans on having implementation of multiple languages, better speech recognition, and a more personalized experience with machine learning.
