Inspiration

I lead a child trafficking awareness club at my school and it has been a really important topic in my life. A few months ago, I met a survivor of child trafficking who is an advocate and a voice against sex trafficking. Since then, her words and strength has inspired me to spread awareness on the topic- one way is with an educational website.

What it does

It teaches users about the fundamental aspects of child trafficking. It is an opening for the user to learn more on their own.

How we built it

We built it using Notepad++ with languages such as HTML, CSS, and JavaScript.

Challenges we ran into

It was difficult to make the slideshow on the homepage.

Accomplishments that we're proud of

We're proud that we have made an informational site for everyone to learn about child trafficking.

What we learned

We learned how to make a slideshow and navigation bar.

What's next for Child Trafficking Awareness

Hopefully we can expand on our coding abilities and build upon this website so that it can have features like a donation button and an interactive map.
