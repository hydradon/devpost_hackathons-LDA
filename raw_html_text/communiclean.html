Inspiration

We would love to create a more enjoyable urban environment to live in.

What it does

Users report issues they come across during their everyday-lives. The reports are automatically displayed to the corresponding authority as well as the public.

How we built it

We built the front end using Flutter/Dart. Additional resources for the back end include Firebase and the Google Maps API.

Challenges we ran into

We had no experience in mobile app development.

Accomplishments that we're proud of

We finally created a working MVP. Besides, we think that our product is of actual value to society.

What we learned

We were able to gain some mobile app development/Flutter/Dart experience.

What's next for CommuniClean

Further implementation of our key features and connection to social media are our main goals for the future.
