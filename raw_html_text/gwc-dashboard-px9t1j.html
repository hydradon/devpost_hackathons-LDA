Inspiration

To make it easy for corporate volunteers to find opportunities to volunteer with girls who code

What it does

This is a service that allows opportunities to be digitized and assigned to people and tracked

How we built it

We used nodejs on lambda for the backend We used mongodb as the backing store We used api gateway to connect to the lambda We used react hosted on a static s3 website as the frontend of the application

Challenges we ran into

We had some problems with the technology that we chose We lost some people at different times of the day with specific skill sets

Accomplishments that we're proud of

Coming up with a solution that we think solves a specific problem

What we learned

Priotization and time management. Quickly iterating and building a quick and dirty prototype that we improve on

What's next for gwc-dashboard

Finish building out the apis and the interface
