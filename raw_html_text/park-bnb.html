Inspiration

Inspired by complaints of residents in bustling cities around the world in getting a sufficient parking space, Parkbnb provides a breakthrough solution in providing additional parking space through lease-and-rent system with any of its community members. Aside from getting easier parking access, leasers will get extra money through leasing parking space.

What it does

Users can find and rent a parking spot from home-owners who have avaialble parking space, either by booking/schedule appointment or directly on-the-spot through QR code.

How we built it

The app was built through Android Studio in Java and incorporated with vast user database. We also provide additional interface and website development with HTML.

Challenges we ran into

Very short time, have never used Java or Android Studio before (no experience), long time for debugging and integrating between app and database.

Accomplishments that we're proud of

Developed the system that is equipped with the following features:

  • Main, specific, and confirmation pages that flows with smooth enough interface
  • QR scanner, code generator, and success response.
  • Integration of bookings and databases between users (leasers and leasees) as well as payment systems.
  • Arduino proximity sensors as additional security measures(to signal if someone trespass property).

What we learned

Coding in Java, application and front-end development skills.

What's next for ParkBnB

More integration between the different parts of the systems, large incorporation and scaling, advanced booking and database systems, more payment methods with debit and credit cards, chatbot and chat room development between leaser and leasee, data analysis through machine learning.
