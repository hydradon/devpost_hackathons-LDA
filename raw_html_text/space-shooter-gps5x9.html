Inspiration

Justin came up with the idea of making a 2D game about shooting alien aircraft in outer space.

What it does

It lets you play a classic space shooting game and keep score.

How we built it

Assets were designed in Unity. We used graphic images available in the public domain. C# scripts were written in Microsoft Visual Studio to build the game mechanics.

Challenges we ran into

Orion and I had little to no prior experience working with Unity, so a majority of our time was spent setting up the game development environment.

Accomplishments that we're proud of

Having a working prototype of the game built over the weekend. Also, playing the game using a joystick adds more fun to it.

What we learned

Each one of us are slightly more experienced with game development in Unity for 2D games.

What's next for Space Shooter

  • Add power boosters for the user player in addition to the core game functionality
  • Improve graphic elements
  • Maybe adding 3D elements to it
