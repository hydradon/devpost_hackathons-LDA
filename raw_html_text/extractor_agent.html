Inspiration

Worked on a lot of Unattended robots, but Always felt like there is more place for the expansion of the interaction of a computer user through RPA, and this can be very infeluencial on a lot dimensions.

What it does

Take Data out of PDF files simply by selecting the PDFs and pasting specific formats of this data according to the end application

How I built it

Using UiPath and Custom Activity using C#

Challenges I ran into

The Custom activity wasn't straight forward, also the techniques of attended robots wasn't my thing before creating this bot.

Accomplishments that I'm proud of

Starting from and idea that seemed that it's going to stay as an idea, but has been executed.

What I learned

More thing about custom activities and parallel keyboard monitoring

What's next for Extractor_Agent

This is the most exiting part there so much things to add, and there so much behaviors that can push further the usefulness of this robot.
