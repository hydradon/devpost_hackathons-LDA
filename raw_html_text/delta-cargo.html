• DLTA CARGO, Finding changes in your data to make a positive change

Inspiration

Wanting to solve a problem affecting a $5 billion dollar industry. Target and better take down criminals and thus reduce violence in the community.

What it does

It is a system through a mobile app, that is given the freight routes and schedules. By cross referencing these schedules with the data received from the tractor and/or trailer it is possible to determine when the truck is moving suspiciously and outside of its scheduled timing.

How we built it

We built it using a dart server, a postgreSQL database and the client application using flutter, so that it can be more accessible on both iOS and Android.
