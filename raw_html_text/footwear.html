Inspiration

We came up with this idea during our visit to a crowded shoe store. It took so much time for the worker to help us out getting the desired shoe size that we wanted. It did not stop there, next we need to face the very long line which took at least 30 minutes of our time just to pay for the shoe. It was such an waful experience and we would to find a solution to solve this problem.

What it does

Smart shelf that will make time to shop shoes more convenience and less time consuming. It allow user to pick up shoes that they want and every single information about it will be display in the computer booth. However, we do not want people to wait, so we created a mobile application, which do the same thing but instead of taking the shoes, we just need our phone to make a contact with the shoes.

How we built it

using low budget electronics such as ultrasonic sensors, arduino, raspberry pi for the hardware. For Software we use Microsoft Azure for our cloud server and table mysql for both Mobile and Web Application. We use NFC tags to let the mobile application detect specific shoes. We use HTML, CSS,Javascrirpt, and Python to create the computer booth for a store or any places.

Challenges we ran into

configuring the dimension measurement' devices and sensors to higher precision. data transfer using serial communication. Finishing a mobile application with a decent User interface and User Experience in 36 hours. Making Microsoft Azure to work with both web and mobile using one service.

Accomplishments that we're proud of

Being able to implement machine learning for higher precision measurements, using Microsoft Azure for our cloud service and make a mobile application that support NFC features.

What we learned

learning Microsoft azure API in short amount of time. Technique fort Android Mobile Application. serial communication

What's next for Footwear

higher precision measurements, better user interface. More features. Not only footwear, but all possible things that people can buy.
