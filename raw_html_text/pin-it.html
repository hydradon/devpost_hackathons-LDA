Inspiration

We were inspired by the idea of a travelling and exploring game.

What it does

Fetches all pinned data on the app with radius around 1 mile, so you would be invited to find out more pins as you go.

How we built it

We used SQL and PHP to build and interact with the the database. We used Java for the front-end app with the Android Studio IDE.

Challenges we ran into

1) Get the app to communicate with the server to download the API

Accomplishments that we're proud of

1) Applying google maps api and obtaining key to integrate it with our app. 2) Working back-end database to return nearest and top hot pins.

What we learned

We have learned how to use Git more efficiently. We have also updated our communications and networking skills as being part of a team.

What's next for Pin It

We would like to pin not only quotes and pictures, but music and even videos in the local area.
