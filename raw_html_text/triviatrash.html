It's a chat bot that asks trivia questions and you can answer them, in the end it'll give you a score. I built it using the Twilio programmable sms chat, MongoDB, Node.js and Express. I ran into a lot of challenges like retrieving information from the MongoDB using mongoose, actually making the messages come synchronously. I did this project on my own, did't have a team, so I learned a lot about databases and API integration. For now, it only asks a limited number of questions, but I would like to modify it such that it asks you randomized questions every time you text the number.
