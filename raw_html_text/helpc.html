Inspiration

One of our friends on the basketball team has color blindness.

What it does

This codes identifies different or enhances certain colors within an image, so that people with different forms of color blindness are able to differentiate between differently colored aspects within the image.

How we built it

We built it by translating java code unto object-c and then swift in order to create the app.

Challenges we ran into

Transcribing the code from one language to the other, as we had to restructure the algorithms with each translation.

Accomplishments that we're proud of

We were successfully able to complete the app. In addition, we are proud of learning and using image processing.

What we learned

The importance of the planning process, and the intricacies of image processing. `

What's next for HelpC

To further diversify the pallet of colors it is able to recognize, and building a more accurate compare algorithm trained to find the 'same' colors within the image better.
