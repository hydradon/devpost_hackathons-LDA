Inspiration

We wanted to find the safest route among the routes provided by Google Maps.

What it does

Calculates the safest route between 2 points by accounting for crime rate at different locations.

How I built it

Using azure, flask, swift.

Challenges I ran into

Debugging Google Maps SDK in ios, and using python with azure.

Accomplishments that I'm proud of

Integrating 3 separate aspects of the application.

What I learned

Download software and get an idea before arriving. Never give up.

What's next for safeRoute

Implementing it on the Apple Watch.
