Inspiration

I want to provide platform for Android user to publish there ebook and earn money by publishing there ebook on Amazon Kindle since there was no platform for Mobile users to publish there ebook provided by Amazon. So I decided to make a one and help them.

What it does

It helps mobile phone user to publish there ebook directly on Amazon Kindle.

How I built it

Since I don't know how to use framework so I decided to make it using kodular

Challenges I ran into

I was not able to good user interface to the users.

Accomplishments that I'm proud of

My application is live on Google Play store and people's are loving it. Even Amazon Kindle app has lower ranking in Playstore than mine.

What I learned

Creating web view applications. Creating buttons.

What's next for Kindle publish

I will improve user interface of my app. Provide will additional tools for my user that will help them to publish ebooks.
