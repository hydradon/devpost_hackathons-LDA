Inspiration

I was hugely inspired by a website called Bored Button but I wanted to create a website with a similar idea but instead of distracting you from the work, it helps you focus on it so you can get it done faster. (At least that's the idea)

What it does

You click a button and it will take you to a random site that will either wake you up, help you focus, or both.

How I built it

I built it in Visual Studio using HTML, CSS, and Javascript

Challenges I ran into

I am a complete beginner with coding so trying to write and build good code was a big challenge for me, and I expect this will be a challenge I will have to face for quite a while.

Accomplishments that I'm proud of

Honestly, I'm just happy the website is even working.

What I learned

I learned how to build a website, use HTML, CSS, and javascript, using visual studio and how to stay awake for more than 24 hours.

What's next for Don't Fall Asleep Moron!

A way where other people can suggest other helpful links, adding more websites, and improving the website's overall design. Having a web animation on the front page would be awesome!
