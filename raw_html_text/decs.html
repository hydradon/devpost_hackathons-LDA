Inspiration

A simple way to orient yourself in the street without the need to take out your phone, so it is more safe.

What it does

It is like a compass, but instead of pointong north, it points where you want to go in a simple and elegant way.

How I built it

Using the Fitbit API (using svg, javascript and css), using a mobile app with google maps API to set the location where i want to go, and a server built using PHP.

Challenges I ran into

The Fitbit API is very limited, the langagues are stripped of a lot of functions. Compatibility, google maps API.

Accomplishments that I'm proud of

The compass is fairily reliable. The android and iphone apps. The authentication.

What I learned

I leared to solve problems and interate multiple technolgies.
