Inspiration

Trolls on youtube hurting feelings

What it does

Takes comments on a user's youtube channel and uses dandelion sentiment analyzer to determine negativity of a comment and if it is past a threshold will delete the comment.

How we built it

Hosting an ec2 website that users log into with their google account and the script checks their channel for negative comments.

Challenges we ran into

Time constraints.

Accomplishments that we're proud of

Finishing a minimum viable product

What we learned

Use a library for OAuth, don't try to do it yourself

What's next for ribbons

Database integration for storing user info
