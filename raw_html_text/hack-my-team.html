Inspiration

Millions of Americans spend dozens of hours a week on fantasy football research. Time to eliminate that barrier to entry!

What it does

Web scrapes web data, puts it into a relational database, and automatically parses through the data to create a high level executive summary.

How I built it

Web scrape data with python, load onto a sql server, connect R to sql server to automate report creation.
