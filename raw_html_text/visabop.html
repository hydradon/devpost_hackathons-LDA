Inspiration

I was inspired by my friends who are struggling to find companies who will sponsor them. It is a huge issue in the United States and we aspire to assist the lives of many, since it is a struggle and sacrifice.

What it does

It allows users to search for jobs based off their visa and desired location.

How we built it

We built it utilizing React as the frontend, Python as the webscraping tool, Node, and Firebase/GCP.

Challenges we ran into

We ran into code communication and API problems.

Accomplishments that I'm proud of

We have a platform that can change many lives.

What we learned

We learned about new technologies and the difficulties of working with technology we were not familiar with.

What's next for visaBop

We hope to continue pursuing the website and improving its functionalities.
