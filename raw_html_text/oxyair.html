Inspiration

One of our lectures informed us about the troubles of dealing with flight companies in regards to taking on air canisters and compresses.

What it does

The website provides the functionality to search for a list of flight quotes and display data relating to their companies' policies on medical oxygen.

How we built it

Using a web scraper API to search for the policy data and stores this in a MySQL database, javascript, php and html then display this on our website oxyair.tech.

Challenges I ran into

After designing the back end around mongoDB we had major issues actually connecting to server.

Accomplishments that I'm proud of

We all worked amazingly as a team, working hard to actually deliver something that worked and putting in the time.

What I learned

We learned about web scraping, SkyScanner Api and generally things going very wrong

What's next for OxyAir

OxyAir has the potential and breadth to become so much more, it could be adapted to show more medical data such as allergy policies for the companies
