• My Birthdays

Inspiration

Elder people especially Seniors in the retirement house who have difficulties remembering the birthdays of their relatives.

What it does

Returns the upcoming birthdays of your family and friends.

How I built it

I built and run the app with PHP, Javascript and JQuery on a self hosted https server.

Challenges I ran into

How to return a result to the user even Alexa could not recognize what you are saying.

Accomplishments that I'm proud of

Using the SSML markup language by Amazon. Using the metaphone and levenshtein algorithm in order to improve the users voice(-recognition) experience.

What I learned

Make it easy.

What's next for My Birthdays

I am planning to add the Amazon Notification Interface, so user gets notified as soon as there is a new upcoming birthday.
