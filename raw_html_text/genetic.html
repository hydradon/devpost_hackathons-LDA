Inspiration

I just want to make programm which will save peoples time, who want to solve genetic problem.

What it does

it solves Genetic problems (Mendel inheritance), for any size. it works like calculator, if you've worked on genetic problems, you will probably know, that it takes time to solve, so my solver helps people not to waste that precious time.

How I built it

I came up with idea, then I just started thinking about algorithm and wrote it.
