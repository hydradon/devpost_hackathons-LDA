Inspiration

We were inspired by space,and NASA's recent accomplishment.

What it does

it lets users select any of the pictures available from NASA's publically available API for the 3 previous mars rovers and using kinect, insert themselves into that picture

How we built it

We used kinect 2 on windows, with C# API with massive help from the available samples.

Challenges we ran into

we were unable to get the UI working to a satisfactory degree using the kinect as a controller so we had to fall back to a basic UI.

Accomplishments that we're proud of

it works to a good degree .

What we learned

Kinect API is a difficult beast

What's next for Life-On-Mars

When NASA release the new Rover's pictures, adding that rover and it's cameras
