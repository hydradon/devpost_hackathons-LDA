Inspiration

Banking apps are dumb, we have so much of information available on our phones and still if you want to send money to someone or pay a bill then we have to enter account number, bank code, n number of passwords etc. We wanted a better solution. I solution which could understand us and to which we could talk.

What it does

Its smart banking assistant which can be invoked from any screen by long pressing om the home screen. The app gets the context of the screen which it is currently on and is smart enough figure if there is any financial conversation going on. It then extract the information the conversation and builds a request to fulfill the intent and we can complete request by conversing with the bot.

How we built it

We built it with Amazon Lex and Lambda, the context is extracted using Android assist API. And the whole app is built on android platform. The context less chat conversation works on web and slack.

Challenges we ran into

Figuring out how the assist API works and colleting data to build Amazon lex intents.

Accomplishments that we're proud of

Completing the product on time.

What we learned

We learned how to not give up

What's next for Evee

We will approach any banks which might be interested in the idea.
