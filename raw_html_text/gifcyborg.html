Inspiration

We wanted to combine multiple APIs to create an easy-to-use service where anyone can combine pictures to find gifs.

What it does

Combines any number of pics into a relevant gif. Text (321) 424-6539

How I built it

Used Twilio API for texting, Claraphi for image tagging, and Giphy for gif search.

Challenges I ran into

Multiple API calls can make it slow.

Accomplishments that I'm proud of

What I learned

Learned how to use APIs, had never used them before.
