Bring stories to life with geoBook, the web app that lets you explore all of the places in your favorite books, right next to the text.

We used nltk, a natural language processing library from Stanford, to build a database of locations mentioned in thousands of well-known books. We built this datbase using Azure. We built a Python backend using web.py, that communicates with a JavaScript/jQuery frontend. We used the MapBox API and the FPDF Python library to display a pdf of the book next to an interactive map with markings at the book's locations.
