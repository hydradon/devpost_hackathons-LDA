Inspiration

Jamaica is a beautiful country with a rich culture that is still unknown to many persons across the world and we want to change that.

What it does

Island Adventures: Jamaica is an desktop adventure game that uses a local taxi driver, Nigel, to shown a foreigner, Michael, the local richness of the country's culture. Michael will go on many fun quests/challenges as if he were a Jamaica himself.

How we built it

We created almost all of the asset from square one, including the characters, using Unity Game Engine, Photoshop, and C# scripts. Each team member focused on different areas to manage time as efficient as possible.

Challenges we ran into

We were unable to finish each challenge/quest to the extent that we intended, nevertheless, the main concept of the culture of Jamaica will be evident.

Accomplishments that we're proud of

We are extremely happy with how Island Adventures: Jamaica is materializing and how well we worked as a team within the time constraint.

What we learned

Teamwork makes the dream work!

What's next for Island Adventures: Jamaica

Island Adventures: Jamaica aims to deploy a mobile version, include the amazing in-app purchases, expand to other countries in the Caribbean, and ultimately, become the NEXT BIG THING.
