Inspiration

When we first entered UCSC as incoming freshmen, we found that it was difficult to locate the various resources that UCSC provides. As a result, we decided to create a website that provides easy access to important resources in a simple layout.

What it does

Slug Assist is a service that provides instant access to important student resources across campus.

How we built it

Slug Assist was created using HTML, CSS, JavaScript, and Bootstrap. We also used services such as GitHub to coordinate our project.

Challenges we ran into

Determining the app's purpose and scope, debugging display errors

Accomplishments that we're proud of

Completing a submission that can help other students succeed.

What we learned

How to format in HTML, CSS, and Bootstrap and use Github.

What's next for Slug Assist

Converting SlugAssist to a mobile app and adding more resources:

  • Contacting RAs and CSOs
  • Specific advising numbers
  • Additional contacts based on college affiliation
  • Improve UI
