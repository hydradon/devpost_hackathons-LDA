Inspiration

We wanted to build something that could potentially impact the lives of millions of people. So we focused on the one they they all needed--good health.

What it does

Currently, users are able to take a visual acuity exam and also link pictures of their eyes to see if they have eye diseases such as cataract and glaucoma. The image recognition is trained through clarifai api, which we could use in the future to also detect things such as bug bites or other noticeable diseases

How we built it

JS, HTML, CSS, NodeJS, Angular, Clarifai api for machine learning

Challenges we ran into

scaling of distance and letter

Accomplishments that we're proud of

training a machine learning to be able to detect diseases with pretty good accuracy

What we learned

how to use speech and facial recognition on a web page

What's next for Home Screen

improved machine learning to be able to detect other things such as bug bites
