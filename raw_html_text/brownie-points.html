Inspiration

We wanted to create a free cryptocurrency that was easy to use and quick to transfer, so we created brownie points!

What it does

It's a secure, smart contract solution for almost instantaneous transactions.

How we built it

We used solidarity to build the smart contract system on the ERC-20 standard

Challenges I ran into

We started to use Raiblocks for it's high transaction capability, but when we realized that we did not have the time to download the entire ledger, we had to choose another option.
