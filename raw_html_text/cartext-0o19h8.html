Inspiration

I was rear ended when trying to merge over break so we made this app in hopes that we could communicate to drivers behind us to prevent more crashes. Voice recognition was added for safety.

What it does

displays input on led screen supplied by iphone app and stored on a web server

How I built it

iphone app pushes data to SQL db on a web server. Raspberry pi monitors for incoming messages.

Challenges I ran into

how to convert speech to text

Accomplishments that I'm proud of

The fact that the iphone does not need to be in any certain range to the pi

What I learned

how to work with raspberry pi and connect multiple devices via a web server

What's next for CarText

sell it in walmart.
