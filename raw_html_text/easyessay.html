Inspiration

Writing college essays can be hard, especially trying to find credible sources quickly.

What it does

Uses Google's natural language API to analyze the contents of a google docs essay and recommends scholarly articles for references.

How we built it

With Google Natural Language API and App Script to add on functionality to Google Docs.

Challenges we ran into

The Nature API, the scholarly articles database, proved to be difficult to use along with App Script's URL fetching to retrieve the necessary information.

Accomplishments that we're proud of

There was a moment where everything worked.

What we learned

How to start working with Google Cloud Platform. We definitely improved our Javascript skills, and furthered our knowledge in connecting various API's together.

What's next for EasyEssay

Making it more robust and modern looking.
