Inspiration

We feel everyone deserves a safe and healthy environment to live their lives and Canada can offer this life to many. We are a country that is welcoming, helpful and want to see unity.

What it does

This website is a great resource that offers Syrian families a place where they can find all the important information they need to live their regular lives. This resources points to locations that meet their religious views. Places such as worship, halal restaurants along with many health care places.

How we built it

Using Wix, html, javascript and css along with lots of coffee.

Challenges we ran into

Having no previous experience with javascript made this task difficult as we had to experiment and learn as we go.

Accomplishments that we're proud of

We learned to become more efficient with reading and coding in a new language and were happy to have a finished product.

What we learned

Coding in javascript, html, and css, teamwork, communication, creativity.

What's next for Home Sweet Home

More locations, more interactivity and provide the website in Arabic.
