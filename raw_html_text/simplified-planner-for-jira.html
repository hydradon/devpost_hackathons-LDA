What it does

Is project planning becoming a nightmare for you? Uses complex tools and still missing key details?

Presenting the "Simplified Planner for JIRA" from J-Tricks.

Create simple, yet effective, project plans. Simplified planner focuses on

  • Start and End dates for JIRA tickets
  • Estimates on tickets
  • User Capacity Planning
  • Custom milestones
  • Themes for plans
  • And above all, ease of use!

Choose a filter and start planning.
