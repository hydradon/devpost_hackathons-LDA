Inspiration

Trying to search Ebay for the right item can sometimes be a pain, because the market prices of goods vary wildly and very different goods may be returned from a similar search.

What it does

It allows for one to search Ebay as they would normally, but filter results based using an interactive histogram visualizing the prices of your results.

How we built it

The service was built using Ebay's Finding API through PHP running on Apache.

Accomplishments that we're proud of

The result is relatively visually attractive and quickly returns and processes a large number of search results.

What we learned

Usage of the Ebay Finding API in addition to the application of a range of web technologies.
