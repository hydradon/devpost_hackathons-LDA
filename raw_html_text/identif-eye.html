Inspiration

We wanted to build a project that would use cutting edge technology to solve a societal issue.

What it does

The app takes an image as input and detects whether there is a wanted person (whether it be a missing person or a runaway criminal) in the image. This would give more confidence to people who suspect that they are in proximity of a criminal.

How we built it

We used the Microsoft Azure Cognitive Software Face API to detect and identify faces in an image. The identification process is trained based on an existing database of wanted people. We then query the image to see whether we are confident that there is a wanted person in the photo.

Challenges we ran into

Using the AI frameworks were confusing at first.

Accomplishments that we're proud of

We are proud of applying our technical abilities towards solving a societal problem that many face.

What we learned
