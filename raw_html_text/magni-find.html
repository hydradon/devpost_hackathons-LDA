Inspiration

We have pretty strict parents, and as a result, we know that many would be interested in this web app.

What it does

The api allows you to locate your car, and our code amplifies that by not only providing location, but speed as well (through odometer and time).

How we built it

We used the SmartCar API and then coded the web app in python using flask.

Challenges we ran into

We had trouble implementing the SmartCar api and deploying the web app. Additionally, it was difficult to reach into the variables from the API.

Accomplishments that we're proud of

Making a working product, being a team of mainly beginners, getting to access real-time information on a tesla

What we learned

We learned how to use python, heroku, github, and the SmartCar API

What's next for Magni-find

We want to make an android and IOS app, and add features to it such as: real-time speed of vehicle, GoogleMaps or Snapchat API to track location in real-time, track parking spot, a notification center, and have an appealing UI.
