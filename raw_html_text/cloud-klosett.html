Inspiration

The opportunity to compete in a category sponsered by a local startup! What could be more exciting?

What it does

Cloud Klosett is an android app that allows the user to either take/upload photos of their wardrobe, we then use computer vision to cluster the items by categories.

How we built it

We built this app using Android studio, we used Google Cloud Vision to scan the images, Google Calendars api to allow the scheduling of outfits, and we used the mindworks library to develop the closet deck, as well as many other tools

Challenges we ran into

Android studio, learning the apis and having relatively low experience with computer vision made the learning curve very large.

Accomplishments that we're proud of

The amount of work we put in as a team, we are extremely happy with our choice to take on a challenging category.
