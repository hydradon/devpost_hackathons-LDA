Inspiration

We like Nerf gun fights.

What it does

We automated the reloading and shooting process in the Nerf gun using an Arduino board and continuous RC servo. Moreover we used an Intel RealSense camera to detect faces, calculate the relative angle between the the gun and the target and shoot at the target.

How we built it

We used:

  • Arduino Genuino 101 Board
  • Continuous RC servo
  • Batteries
  • Intel RealSense camera
  • Lighthouse sensor with wifi
  • Nerf gun
  • Myorobotic arm

Challenges we ran into

  • Assembling and automating the Nerf gun.
  • Setting up Intel RealSense.
  • Mechanical issues.

Accomplishments that we're proud of

  • making Intel RealSense face detection work.
  • making the Nerf gun automatically reload and shoot when it recognizes faces.
  • setting up ROSbridge client.
