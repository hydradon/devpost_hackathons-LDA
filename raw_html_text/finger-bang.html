What it does

Using the leap motion, we detect changes in the vertical motion of your hands so that you can "shoot" out of your fingers to knock over bottles as fast as possible.

How we built it

We used Unity to easily create a 3D world and support the Oculus Rift for vision.

Challenges we ran into

It was difficult to properly detect the gesture required to shoot. It may have been easier on an older SDK, but the tracking was a lot worse so we decided it would be better to use the newer SDK and manually detect the motion.

Accomplishments that we're proud of

Creating something in Unity after not properly using it before, and effectively using the Leap Motion.

What's next for The Sheriff

We may develop this into a small full demo (Or at least make the shooting much more reliable).
