CDN Resource Mapper

About

CDN Resource Mapper implements simple possible blockchain based resource mapper which hosts files. It allows web sites to run directly from blockchain. It provides the step to the fully-decentralized internet.

Prerequisites

Install dependencies: 

npm install

Run app 

node ./main.js

Go to website: http://localhost:12345
