Inspiration

Researchers need tools to annotate data. It's not practical to load your data to crowdflower to annotate or use Python notebooks everytime. So I developed a basic framework to annotate data.

What it does

It reads twitter users, show the relevant data in a nice format and saves your annotations.

How I built it

Python, Flask, Jinja2

Challenges I ran into

Everything was hard for a data scientist that do not have frontend & backend experience.

Accomplishments that I'm proud of

It works

What I learned

Flask

What's next for Teajay Did Stuff

Turn it into a basic Twitter User Interface for researchers with live tweets and search tweets by keywords.
