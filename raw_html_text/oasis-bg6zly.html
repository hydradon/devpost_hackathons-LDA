Inspiration

Women empowerment through ensuring women safety. A safe public sphere.

What it does

Oasis is an IOS app that allows the user to take a picture and uses facial recognition to determine if the individual in the picture is a sex offender. In addition, the app will send a text if the user enters a hot spot with high rates of sex crimes.

What's next for Oasis

We hope to add additional features like a buddy system in which users are able to connect with other users in an unsafe situation. Also, having an emergency button in high-risk situations will also be beneficial.
