Inspiration

Our biggest inspiration was sharing the passion of the non-profit organization to help humans and animals alike in times of crisis.

What it does

The website allows the users to easily create surveys, store answers., and export them in one place. It also allows for visualization of inventory including where donations have come from, what categories exist, and how much there is.

How we built it

We hosted the web app using Hostinger, and built the app using Bootstrap, php, phpmyadmin, and mySQL. The data analysis portion was created in using R in an R shiny app.

Challenges we ran into

We were caught off guard by the complexity of the SPRING framework and limited knowledge of the framework. Because the problem was not well defined, we lost some time figuring out the details of the project.

Accomplishments that we're proud of

Reaffirming how to design DTOs. As well as being able to resort and pull together information from various resources in a limited time and apply what we learned to make a tangible product. We are proud that we learned a lot, since we worked with tools we didn't have much experience in.

What we learned

We learned a lot in PHP, webhosting, and database management. We have not used bootstrap before which was a challenge. We also have not had much experience in R shiny apps which was fun.

What's next for Team22-DataAnalysis

We plan to have more data analysis tools in the future using R shiny apps, and further build the website.
