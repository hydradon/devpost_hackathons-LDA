Inspiration

I was inspired by how the TJHSST library website contained the links for all of the necessary research databases and I decided to primarily base my project on the library website.

How it works

The website contains pages of science links, mathematics links, history links, and literature links. On the home page, or the first page, there are four pictures that the user can click on to access each section (science, mathematics, history, or literature).

Challenges I ran into

Some of the challenges that I encountered included the readability of the website and whether or not I needed sub-menus or not.

Accomplishments that I'm proud of

I was able to successfully engineer and create a functioning website incorporating some use of html for the first time.

What I learned

I was able to learn some aspects of html, especially styling (color, font-size, etc.), as well as how to use a website builder (IM creator) to create a website (with drag-and-drop elements).

What's next for Education Hotspot

In the near future, I plan to improve the website by incorporating more complex elements of html or constructing a website in JavaScript or another language. Using CSS in improving the website will also make the site significantly more appealing to the user.
