Inspiration

We are passionate about building people up and wanted to allow people the joy of discovering positive messages.

What it does

pickmeup is an iOS app that allows users to pin inspirational messages wherever they go and discover ones that others have left along the way.

How I built it

We used Swift to create a native iOS application. The user can take a photo and add a quote chosen from our list of pre-approved ones. Then, using their location, we associated the particular message with that spot so if other users pass by, they can view it.
