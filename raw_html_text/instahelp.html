Inspiration

Recent natural calamities like Harvey, Irma and Mexico Earthquake

What it does

It Automatically detects when you are in danger by using Phone sensors and request for help from near by hospital and Users of the app , it gives us a social forum for people who need help around you.

How we built it

It is a native Android App which uses java web services to interact with back end. Android app uses available sensors to detect that the user is in danger.

Challenges we ran into

Creating a automated algorithm to detect impact without any user input and can adapt to different Phone sensors

Accomplishments that we're proud of

our app have potential to save lives.

What we learned

its hard to work on a path of "saving lives"

What's next for smartArlington

Expanding the network and using government services
