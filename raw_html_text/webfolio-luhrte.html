Inspiration

Well many Student have difficulties in making their Portfolio or Resume to show Employers.

What it does

WebFolio can Parse a students Resume and create an online Portfolio for a student. So they can show this link to an Employer without having to spend a lot of time making the whole Portfolio from scratch

How I built it

We have 2 Backend Languages, Node and Python hosted on 2 diff Servers. One to handle manual entry and there other for Resume parsing. We needed to create a Pipeline for the servers to communicate and exchange messages.

Challenges I ran into

Communication and NLP, GitHub

Accomplishments that I'm proud of

What I learned

NLP, NODE, more tricks and shortcuts in mongoDB

What's next for WebFolio

A lot more for employers and Students in the future for matching and finding the right job. For now it just designed to help Student with no Web Development Experience to have their Portfolio built Easily.
