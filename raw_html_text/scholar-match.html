Inspiration

How can teams be formed based on overlapping interests? e.g. for course projects

What it does

Builds a user interface for team formation using a keyword for a topic input by each user and outputs the overlap of topics using connectivity diagrams and pie charts

How I built it

Obtained co-occurence matrix information from news articles and output networks based on these connections to suggest relevant pairings of classmates based on their keyword input

Challenges I ran into

Developing networks based on relevant relatedness-metrics

Accomplishments that I'm proud of

A user-friendly interface that allows users to get relevant output quickly and easily to match them with another scholar

What I learned

How to integrate information from various users using topic modeling and network visualization approaches

What's next for Scholar Match

Optimizing an algorithm for matching scholars based on their keywords and generating a ranking of possible matches.
