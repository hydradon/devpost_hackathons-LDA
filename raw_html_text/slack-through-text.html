What it does

Allows people to join a Slack conversation even if they don't have access to the web or a smartphone.

How I built it

I monitor for messages on your Slack team and send a simplified version of the message using Twilio. A reply is matched with a channel and sent to Slack. No internet is required on the phone.

Challenges I ran into

I learned how to host my own api on Heroku and use API's I've never used before (Slack and Twilio)

What's next for Slack through Text

One part of the application was done manually, the authentication of the user. I'm hoping to have a version where I can authenticate the user through only text, without access to internet.
