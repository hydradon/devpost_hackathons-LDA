Inspiration:

What it does: It is tetris game with voice control

How we built it: We build the backend as a dispatch server and there are three small application we build with voice recognition(Tetris Game2.0; Tetris Game 1.0; NotesTaker).

Challenges we ran into: First, first challenge was find the right voice library which is compatible with NodeJs web application. Second, optimizing the voice control with semantic analysis and blurry search.

Accomplishments that we're proud of: We explored the voice recognition area within short time and deploy the whole project on both our domain and heroku.

What we learned: All of these!

What's next for Voice Adventure: try Google Voice model comparing with annyang and add voice control to other game or add game modes.
