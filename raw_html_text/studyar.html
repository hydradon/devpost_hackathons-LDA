Inspiration

StudyAR wants to help students learn and study more efficiently and effectively.

What it does

50% of the students prefer visual kinesthetic learning. StudyAR combines kinesthetic and auditory learning to helps students optimize their maximum focus on course material.

How I built it

We used Vuforia-api to recognize the symbols with Android. We used Microsoft visual-basic for C# script for modeling and tweaking Camera for our specific use. And Unity.

Accomplishments that I'm proud of

Used Vuforia's api for the first time and loved it.

What I learned

learnt how to used Visual Basic.

Challenges we ran into

We did not have any Samsung compatible with Gear VR. Project is VR ready. We still were able to work it out as an AR only presentation on our HTC phone.
