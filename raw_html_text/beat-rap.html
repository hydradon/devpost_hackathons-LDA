Introduction

After reading this article link we decided to train of these network on rap lyrics and then feed it into Amazon Polly, the text to speech API which powers Alexa (some of the voices are from young children, which made the final result even more hilarious)

How we made it

We used Tensorflow and some hand written post processing algorithms to build an API returning the lyrics of a generated rap song. Remi then built an iOS client using AWS Polly. The results were quite funny
