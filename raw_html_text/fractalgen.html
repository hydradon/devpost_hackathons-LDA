Inspiration

Benoit Mandelbrot and his mathematical genius

What it does

Generates fractal shapes from regular polygons or custom shapes

How I built it

Using javascript and HTML5 canvas, with graphics from the p5.js Library

Challenges I ran into

We ran into challenges when designing the UI so it would scale properly, as well as when generating a fractal from a user created shape.

Accomplishments that I'm proud of

Finishing on time, turning user input into a fractal shape, making a program that can generate a variety of recursive shapes.

What I learned

How to make a user friendly interface, how long a real programming project can take and how much energy it takes to build one.

What's next for FractalGen

Bug fixes, more customizability for users.
