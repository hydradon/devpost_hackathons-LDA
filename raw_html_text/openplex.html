Supaplex was one of my favourite games in childhood. One of my dreams was to create similar game on my own. I came to AGHacks hackaton with plan to join some team as Java or Web Developer, but somehow I teamed up with two other Java Developers: Sławek and Maciek and we made an Open Source game inspired by Supaplex which can be developed further and thanks to used technology (LibGDX) it may be easily converted from Desktop to Mobile platforms too. We implemented simple gravity engine which allows blocks to fall in many directions thanks to which player can carve own path to get all the score points represented by sheeps. But watch out don't allow block to fall on your playable character's head, or bomb to explode nerby.
