What it does

It determines whether the object in the photo is on fire or not.

How we built it

We built this using TensorFlow and Convolutional Neural Networks. We created the web application using the Django framework.

Challenges we ran into

We had no prior knowledge of machine learning with limited prior knowledge of Django. It was difficult to adjust to the frameworks but we eventually figured things out.

Accomplishments that we're proud of

We finished the project earlier than expected.

What we learned

We learned a lot about python, TensorFlow, machine learning, convolutional neural networks, group programming, github, and Django.
