Inspiration

We have a lot of task requests from email but sometimes we forget our tasks after checking emails. JUNE is the mobile planner optimized for the effortless time management in mobile with patented drag&drop

Production video Product link

In this project we tried to connect JUNE planner with flagged emails. From this, users can manage task requests from email and other schedules in one place.

How it works

JUNE is the planner which integrates calendar, task and memo in a single place. And users can move a schedule easily between these spaces with a drag&drop.

Based on JUNE planner, users can also connect JUNE to external services. For example, calendar can be connected to Google calendar and iOS calendar, task can be connected to reminders and memo can be connected to Evernote.

In this project, we made connection between JUNE memo and flagged emails. So if users flag an email while they are checking an email, these flagged emails will automatically come to JUNE memo. And from this time on, users can move these memos to calendar or task.
