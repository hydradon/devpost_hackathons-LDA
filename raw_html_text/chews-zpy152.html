Inspiration

We spend too much time deciding where to eat especially when everyone is like "it doesn't matter, you choose". So the group ends up standing around for hours before someone decides for everyone. We wanted to make an app that can help save some time. The app will decide for you what restaurant to eat at.

What it does

It takes a list of restaurants on Franklin and generates a restaurant depending on the cuisine, price range or dine-in (or don't care) option you choose.

How I built it

We built the app using Android Studio

Challenges I ran into

We all came in with no app development experience so none of us knew how to use Android Studio. It was a challenge to learn how Android Studio worked and how to put the java code into the Android java code. ***

Accomplishments that I'm proud of

We actually created an app that works, buttons that respond.

What I learned

We learned how to use Android Studio and how tedious app development is.

What's next for Chews!

We hope to expand the list of restaurants one day, maybe adding more information about each restaurant at the end including links to the restaurant's own homepage. We also hope to be able to incorporate more pictures into the app so people would know what kind of food to expect at that restaurant.
