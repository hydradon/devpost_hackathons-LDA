Inspiration

Increasing Growth of Electric Vehicles,

What it does

Allows energy assets of a consortium blockchain to be shared outside its ecosystem using special transaction from the Dash public blockchain and Chainrider APIs

How we built it

Sugar, Spice and Everything ... Just Kidding. The Consortium Blockchain Layer uses Hyperledger Fabric Node SDK. For submitting OP_RETURN transactions dash-core lib is uses which again utilizes the node.js. Finally, Chainrider APIs are used to query the Dash networks for transactions.

Challenges we ran into

Time Constraints as well as keeping a simple design.

Accomplishments that we're proud of

A demo showing how a small scale system would work.

What we learned

Interoperability is key for free movement of assets and is a stepping stone toward mass adoption of the technology.

What's next for EV Roaming

Scaling up as the industry grows
