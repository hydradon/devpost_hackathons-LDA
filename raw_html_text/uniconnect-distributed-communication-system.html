Inspiration

the inspiration for this project is watching that the communication technique that are currently used in various academic centres are tiring, resource wasting, and not promising way of communication.

What it does

this system will enable student's, class representatives, lecturer's, university admin, and other stuffs to stay connected no matter where they are.

How we built it

in building the web system we will use HTML5, CSS3, and JAVASCRIPT for the front end design, and Laravel PHP framework, Mysql DB on the backend. on the mobile app side we will use the popular framework called React Native which will enable us to build our app for different platforms.

Challenges we ran into

we have faced challenges on how can make the the app side of the system cross platform.

Accomplishments that we're proud of

we have done a good work on the UI/UX or the frontend design of the system so far.

What we learned

we have learned a lot on the UI/UX design, Cloud Computing Technology, SMS Gateway

What's next for Uniconnect [Distributed Communication System]

building a distributed communication system that will take the way of communication in academic centres to the next level by making it fully digital based.
