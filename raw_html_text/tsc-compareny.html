Inspiration

We want to build a website using the data provided by Tractor Supply Co. to help both the corporate executives and the consumers alike to visualize the data.

What it does

Should be able to plot the graphs based on the backup embedded database.

How we built it

We created the website using the domain.com and WebsiteBuilder. For the data analytics part, we used Microsoft Excel (Comma Separated Valued File).

Challenges we ran into

We have to look up and read through many different APIs and think about the variety of different ways we could potentially implement this project successfully. Unfortunately, none of the team members are proficient in Java Script or similar scripting languages for front-end development, which proved to be quite troublesome. We also have trouble implementing using Python code (with the MatPlotLib) to generate graphs.

Accomplishments that we're proud of

For 3/4 members in our all freshman team , this is only our first hackathon, and we were able to create a simple finished product in only 36 hours.

What we learned

We went to many workshops and took notes on how to use Node.JS, Git, and Go. In addition, we also learned about different ways to create a website and carry out data analytics programs on the back-end.

What's next for TSC Compareny

More sophisticated graphic user interfaces. More secure way to store the data on our server so that real data may be used in a confidential manner. Only authorized personnel with valid login information (from corporate) will be able to access these confidential real data and customers will only get a very broad overview of the data or data that the legal teams deem appropriate to be released.
