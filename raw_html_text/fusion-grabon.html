Inspiration

When it comes to banking,marketing offers & products is very crucial.We researched behind it and built an AR interactive platform where user can get to know about the offers without much effort.

What Fusion GrabOn does

The idea is an AR interactive platform where the user in a specific location can see various offers nearby provided by bank which he can grab on a click. This would bridge the gap between bank and customers, thus helping bank to promote offers that energize existing customers and attract new customers

How we built Fusion GrabOn

  • Game interface and AR were built using Unity.
  • Then connected FFDC Consumer Profile rest API.

Challenges we ran into

  • Moving from an idea to a solution.
  • Connecting FFDC API to unity was initially a challenge.

Accomplishments that we're proud of

Building a successful solution for a relevant issue with bunch of freshers.

What we learned

  • How to incorporate Augmented Reality in a mobile application with unity.
  • How to connect FFDC API with unity.

What's next for Fusion GrabOn

  • In future using ML, offers can be customized.
  • Will consume more FFDC APIs
