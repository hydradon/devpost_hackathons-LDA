Inspiration

I was inspired by the Alexa tech for good challenge, seeing that STEM was one of the topics, I decided to create a trivia game about it.

What it does

Quite simple, the skill asks the user 5 questions related to STEM, the user has to say the number of what they think the correct answer is. If they got it right, they get 1 point, if not, Alexa will tell them what the correct answer is.

How I built it

Based on some of templates available on Alexa's github repo.

Challenges I ran into

Finding questions that are not too easy and pass the "one breath" test.

Accomplishments that I'm proud of

Creating a skill for the first time! (I'm not a software developer)

What I learned

How to create a skill.

What's next for STEM pm

Adding more questions, and perhaps raising the number of questions in a each game from 5 to 7 or 10, depending on user feedback.
