Inspiration

We got inspiration from learning about the many suicides happening in a rural community within close proximity to us.

What it does

The patient can choose to record their daily log with the help of Amazon's Alexa, this log is then sent to a database and can be viewed by doctors on a web application.

How we built it

We used Google's FireBase for the back-end database, along with React for the front-end of the web application. VoiceFlow Creator was used to make a skill for the Amazon Alexa.

Challenges we ran into

Configuring FireBase was a little challenging not to mention the many challenges faced by the front-end developers with React, Amazon's Alexa is also not the easiest thing in the world when it comes to setup.

Accomplishments that we're proud of

What we learned

Successfully learned how to get and post data from a react application to firebase. Learned how to create a skill in Amazon Alexa using VoiceFlow Creator.

What's next for Health Hub

Expansion for different cities/provinces/countries. Partnership with major pharmaceutical companies and clinics.
