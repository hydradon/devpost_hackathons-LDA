Inspiration

How awesome would it be if you could play an MMO where the world is the Internet? That was our ambitious goal when we started building Project Snow.

What it does

Simply turn on the Chrome extension and your current tab turns into an online game! Walk around by clicking on the page and use 'shift-enter' to chat with other bears! From browsing the internet with your friends to collaborating on online projects, the possibilities are endless.

How I built it

Project Snow is a Chrome extension that communicates with an Azure server running node.js. Networking is handled by socket.io.

What's next for Project Snow

The server architecture could use some work. We plan on expanding this project and adding actual gameplay in the future!
