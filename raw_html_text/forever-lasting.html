Inspiration

The name Forever Lasting came from a video game project that my friends and I had called Long Lasted while the game itself was inspired by the film Castaway and the game Stranded Deep .

What it does

This skill teaches people how to plan out their strategy for completing the game and helps people improve their memory with practical skills to survive.

How I built it

I have built the skill using javascript and node.js on AWS Lamda .

Challenges I ran into

When I had first summited the skill, I got it back to me with information on several bugs. This would go on for 2 - 3 days until I had the entire program fixed. We also couldn't test the skill on Alexa after submitting the program and the text on javascript came out much differently as I expected which has made me fix the dialog multiple times. When you submit a certification, you can not edit anything on your skill which was a huge problem.

Accomplishments that I'm proud of

I am proud of being able to increase my understanding of the Alexa program and the achievement of creating a skill that many people would enjoy was a wonderful experience.

What I learned

I have learned to accept the multiple frustrations that come into being when I create a more complicated program. I would get frustrated over minor bugs that can be easily fixed. However, I have learned to accept the bugs and was taught that fixing bugs is just a normal part of programming software .

What's next for Forever Lasting

I plan to one day add in more features into Forever Lasting such as more food, more sound effects, and recordings of people's voices to bring more realism to the game.
