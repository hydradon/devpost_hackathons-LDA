Inspiration

There are currently not many budget websites for young adults to use. Many young adults have a unique income and unique set of expenses so a personalized budget is very important to have.

What it does

Young Budget helps teens and young adults budget their money and practice responsible spending.

How we built it

It was hosted on AWS, we built it using html, css, php, and sql.

Challenges we ran into

Our login system would not operate and css would not look how we envisioned.

Accomplishments that we're proud of

We were able to get almost a fully operational website in a very limited time.

What we learned

We learned more about css, html, php, and sql. We also learned how to work in a group and achieve the same outcome.

What's next for Young Budget

We would like to make it completely functional and available for young adults to use.
