Inspiration

I want to bring rEVolution by replacing fuel based vehicles to EV.

What it does

SwachEV finds nearby Electric Vehicle charging stations and/or battery swapping station. Book a slot for charging and pay in advance. You can also ask home delivery of battery or you can visit to station to get it swapped

How I built it

I built it on Openshift container using Application runtime. I used two application runtime - nodejs and springboot. I used react, java spring Boot.

Challenges I ran into

Faced challenges while working on Redhat fuse and Redhat SSO

Accomplishments that I'm proud of

I built this app and learnt Redhat AMQ, fuse, SSO, application manager etc.

What I learned

I learnt Red hat services namely - Fuse, SSO, AMQ, Application manager etc.

What's next for E

I am going to build Android app.
