BuzzPics is a whole new way to view your BuzzFeed articles. Instead of the traditional titles and thumbnail images, BuzzPics sources and displays only the images associated with the articles. If you hover over an interesting image, you will get the articles category. This creates a more immersive experience and can be a fun new way to discover articles based on what photo catches your eye. This web app was built using a Python script to locate the images and other info off of the BuzzFeed API. Having never used Python or the BuzzFeed API, I am proud of how this web app turned out and plan to continue tweaking and adding to it!
