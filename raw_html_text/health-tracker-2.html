Inspiration: It is a big problem for patients to seek medical treatment now. Hospitals in different places have different data systems. Many times they cannot communicate the medical taboos of patients. We want to develop a health tracking app to help patients.

What it does: It is a personal health record tracking and health advisor in your pocket.

How we built it: We use a cross platform framework angular & ionic to develop the app. We use firebase & google's AI api.

Problem Solving

  1. Patients' data interact with all the kinds of hospitals safely when they need to see a doctor quickly.
  2. user can take the pictures of their doctor notes or medical record and store in the cloud which they can access all the time by themselves, patients can send to hospitals too.
  3. QR code function can provide real-time information transformation between patients , hospitals and health department effectively.
  4. Having an education section 5.Health Advisor chatbot.

Technology Pitch: One Stop Cloud Data Solution :High Security, Up to Date, Auto Scaling Cross Platform: easily implement to IOS & Android System Machine Learning : Chat bot Dynamic real-time data interaction

Challenges we ran into: How to train the chatbot. Work together effectively. Learn new skills fast.

Accomplishments that we're proud of: We connect front-end and cloud backend smoothly in a very limited time. We have implemented the QR code function. We have built a health chatbot and embed into our app.

What we learned Framework has itself limits. Too many bugs need to be fix.

What's next for Cyber Rover: Connect with FHIR . Use machine learning convert image to CCDA . Live online diagnosis Add google assistant
