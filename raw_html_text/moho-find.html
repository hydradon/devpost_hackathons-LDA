Inspiration

We have a blind friend who has trouble navigating the campus.

What it does

This app tells users where they are based on their Access Point locations on the campus' IP address.

How I built it

We started with Javascript, planning to make a website. Then we realized that in order to achieve our goal, we would have to build an app. To get the APs from the user's device, we needed to use Objective C. So, we learned it in a night...

Challenges I ran into

We had to learn an entirely new language overnight and change our plan.

Accomplishments that I'm proud of

We are SO proud of learning Objective C and making a working project. Thank you, Mentors !! :)

What I learned

Objective C and how to work in a team.

What's next for MoHo Find

More locations, route guidance.
