Inspiration

I always feel more engaged with interactive apps and websites, so I wanted to use interactivity to motivate myself to complete tasks, which I often leave for months incomplete on my to-do list.

What it does

Allows editing and creation of to-do list tasks. Completed tasks are rewarded by new plants in the virtual garden.

How I built it

I used Java and XML in Android Studio to build the app.

Challenges I ran into

Building the database was a mostly new experience, as I wasn't experienced with MS SQL or SQLite. I ended up fitting in the graphics at the very end, so they unfortunately are not what I had hoped. Unfortunately there is still a glitch where a new task is duplicated at first (but it goes away when the next task is created).

Accomplishments that I'm proud of

Creating the editable list of tasks.

What I learned

That it's really useful to plan out my time and structure the progress of a project, no matter how many people (or how few) are working on the project.

What's Next

Further updates on graphics and interactivity with the garden, as well as more variety in the plants! Animation is also a possibility.
