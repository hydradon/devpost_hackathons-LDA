Inspiration

We really like playing with the Rubik's Cube and we were thinking how we could help blind people experience the same joy that we had.

What it does

Tells the user the way to solve the cube via audio cues.

How we built it

With Python and Java programming languages.

Challenges we ran into

Improving color recognition, porting Python development to Java for Android.

Accomplishments that we're proud of

We did a major part of the project and it is ready for a demo. Our skills as a team got better. Team collaboration was awesome.

What we learned

Porting from one programming language to another one is not always an easy job.

What's next for OTI - Helping blind people to solve the Rubik's Cube

Completing OTI to a fully working state on Android and continue the development on iOS.
