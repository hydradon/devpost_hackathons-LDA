Inspiration

I got inspiration from different sites that post compliments to people who are having a bad day in an effort to reduce cyber bullying on the internet.

What it does

Generates a random compliment for people who need one.

How we built it

With lots of love and wholesomeness.

Challenges we ran into

We tried to put an upvote/downvote button but we ran out of time.

Accomplishments that we're proud of

I think it looks aesthetically pleasing. Random Colors was something that took a long time to implement due to my naivety with javascript.

What we learned

Learned a lot of jQuery

What's next for Wishing Well

We definitely want to add user input, specifically to adding user generated compliments in respect to different complaints. We have done this by preparing an ID value for the complaints, so users who submit complaints can track compliments.
