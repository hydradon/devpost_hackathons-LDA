Inspiration

The recent resurgence of 80's retro and the failure of 'plan A'.

What it does

It let's the wearer show off their individuality and their nerdy appreciation for the past.

How we built it

For the code we used the Arduino IDE. For the strap we used fabric.

Challenges we ran into

We ran into a lot of challenges mostly involving 'plan A'. Once we switched over to ORG, Original Retro Gewelry, it was smooth sailing for the most part aside from some persistent collision issues.

Accomplishments that we are proud of

We have a final working product. It works. It doesn't break. Trust us.

What we learned

Recovering from failure. Working with the TinyScreen+.

What's next for SaMiDa - Original Retro Gewelry
