Inspiration

Create an integration that will link Magento and XERO with no need of configuring and creating ssl keys . Users require only few clicks to install and configure this integration .

What it does

Magento orders are saved as invoices in Xero . Integration supports any type of tax can be applied in Woocommerce and creates that tax in XERO on the fly .
