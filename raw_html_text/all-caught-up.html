Inspiration

We experience it in our day-to-day work. We get overwhelmed with a lot of Important meetings and emails. Someone gets missed and someone gets dissapointed...sometimes _ very disappointed _.

What it does

All Caught Up shows you all of the important emails and meetings that you have for the day and categorizes them in color depending on the sentiment of the email or meeting. With this feature, it lets you know which emails or meetings you need to prioritize among all other important emails and meetings that you have

How we built it

We built it using tools such as Visual Studio Code taking advantage of the Outlook Addin platform and utilizing the Microsoft Graph API (Mail) and IBM's Alchemy API
