Inspiration

Consumers should only pay for what they consume.

What it does

It lets you pay only for how much content you consume. For example, if you listen only to the first 1 minute of a song, you only pay for that first minute.

How we built it

This was built using Solidity, Javascript, Python, and react

Challenges we ran into
