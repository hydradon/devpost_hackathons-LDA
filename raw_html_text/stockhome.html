Inspiration

We got sick of using the notes app for groceries, so what better to make as a first app than a grocery list? We wanted to make something convenient that would get us into the world of Android app development

What it does

Keeps track of a grocery list and an inventory list. When an item is checked off one, it moves to the other, allowing users to easily see both what they need and what they have.

How we built it

Android studio and java

Challenges we ran into

Literally too many to list here. This project was quite a challenge, as neither of us had any experience beforehand with android studio or xml. We ran into issues with synchronization, saving data after closing the app, and general app stability.

Accomplishments that we're proud of

Every aspect of this app was an achievement as it was our first time learning this platform. Although the app is still not fully usable/functional, we have created a base to build off of and have gained a tremendous amount of knowledge about the inner workings of mobile development

What we learned

Basic xml, basic android development skills, differences between desktop and mobile architecture, using and creating GUIs

What's next for StockHome

We'd like to polish and improve what we've built on. Adding expiration dates to items, family sharing features, and the ability to add multiple lists are all be features we plan to look into
