• Smart Surveillance

Inspiration

Off late there have been many incidents on the national highways/ secluded patches country wide involving petty theft, malpractices to big crime plots, terrorist activities, robbery and rapes. These areas, though part of important areas, lack police Surveillance and monitoring. Pinning CCTV's on these areas would not bring any emergency help but decoding on what happened the morning after.

What it does

Drones fitted with high definition cameras can span a large area and can be controlled and moved anywhere. These drones can be fitted with sound sensors/ devices which would use predix analytics to smartly analyze if there is something out of the normal happening in that area, say it can sense and decode the alarm from someone's phone or can read some sharp noises, explosion, fire or anything and report back to the nearest police stations of the area. This would not only call in for emergency services but would be very beneficial in averting the mishaps.

How we built it

1) Created microservices in predix that would interact with the hardware system to collect below data points a) Sound b) Images c)Electromagnetic Radiations/ d) Location(Longitude/Latitude)

2) Created microservices that utilizes above data points to fine tune it for further analysis 3) Created a core analysis microservice that take the fine tuned data to detect anomalies(Fire,explosion.mob gathering) with higher probabilities 4) UI that shows the near real time monitoring using the above micro services 5) Instant notification in case of anomaly detection

Challenges we ran into

1) Hardware Interaction 2) Took us some time to be comfortable with predix services
