Inspiration

Visualizing and analyzing data has always interested our team, and we thought this challenge would be a great challenge to undertake.

What it does

This project visualizes the locations of the involved people by floor, keeping track of locations. The map shows dots representing the locations of people. It is also possible to view events in text.

How We built it

We applied Python to process the data, Flask as the (micro) web framework and JavaScript to implement the movement of the people.

Challenges We ran into

We ran into challenges relating to ensuring we properly processed data, especially from doors. After close examination of the dataset, it became apparent that we had to keep track of doors opened from the inside among other intricacies.

Accomplishments that We’re proud of

We’re proud of having built together an application in such a short timeframe. Specifically, being able to process data and get the web application online.

What We learned

We learned how to deal with processing data and visualizing it. We learned more on web development and about the languages we used.

What's next for Murder Analysis

We thought this challenge helped us take on new knowledge and lessons learned, which we’ll apply in the future.
