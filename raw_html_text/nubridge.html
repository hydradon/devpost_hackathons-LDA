Inspiration

With the way data breaches happen, access control has never been more vital. We are moving pass an age where storing important in plaintext is a realistic solution. Thanks to NuCypher's technology, access control of object data has become a real solution for all. We wanted to build a interface that not only allows easy user management of NuCrypher's Decentralized KSM, but also allows different rings of permission. Welcome to the NuCypher Bridge.

What it does

With NuCypher Bridge, anyone can easily allow, adjust, and access resources their own, or that is shared to them.

How we built it

By breaking up the different components and features, we leverage React to build our interface.
