Inspiration

We wanted to create a new way for people to connect while increasing outdoor activity

What it does

It is a location-based puzzle app

How we built it

We used Python flask to create a web server, swift to create a mobile app and Javascript to create a web-based interface

Challenges we ran into

We had to create an API protocol which took a while to get working

Accomplishments that we're proud of

we are proud that we are able to combine front end, back end, and mobile technology into one system

What we learned

We learned a lot about the google maps API and rest API

What's next for RiddleMe

We want to flesh out the app and make it more fluid, we want to create riddles around the University of Maryland College Park, to use riddle me around campus
