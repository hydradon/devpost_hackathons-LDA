Inspiration

We wanted to create a website like GreatSchools that would enable the user to assess whether a neighborhood is a nice place to live or not. Currently, it's easy to obtain info on the features inside the house, but not the most important feature: the location! Our website solves this problem.

What it does

Input a ZIP code and get results on various statistics of the area.

How we built it

Using HTML, Python, and JavaScript in combination with the Domain.com website builder.

Challenges we ran into

Integrating all the different components into a cohesive unit. There was also hundreds of thousands of data points, which made things difficult.

Accomplishments that we're proud of

Managing to solve these challenges.

What we learned

Integration of different languages and building a website from scratch.

What's next for Sherlock Homes

With more time and resources, we could add much more data to our analysis, giving a fuller picture of the neighborhood.
