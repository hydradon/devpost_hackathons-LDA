Inspiration

I've always wanted to create a game in Unity, and I based this idea off of one of my favorite arcade games: Doodle Jump.

What it does

Allows the player to jump from platform to platform in an attempt to keep going up for as long as possible without falling.

How I built it

I followed some basic Unity tutorials and Googled a lot to build as many features as I could.

Challenges I ran into

This is my first time working on a project in Unity and also my first time working with C#! So I had to really take my time to understand what I was doing and debug my code.

Accomplishments that I'm proud of

I'm proud that the game works! And I had fun :)

What I learned

I learned how to write C# scripts for objects in Unity, I am now familiar with the Unity interface, and I even got some practice with graphic design tools.

What's next for Rise & Shatter

Better UI! I will be adding more sounds and animations to make the experience much more fun. I really want to add capabilities such as using Bitmojis as the players.
