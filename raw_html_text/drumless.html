Inspiration

After a long brainstorming session, we began throwing ideas at the wall. As we sat discussing ideas, one of our members starting air drumming and that's when he realized what if there could really be a beat based on his air drumming.

What it does

Drumless brings one's own auditory imagination to reality by supplying the missing beat to the air drumming one does in their downtime. By using a bracelet around one's wrist, it can determine the swing of an arm

How we built it

Drumless utilizes a raspberry pi using accelerometer/gyroscope handler software made using python to determine drum swings.

Challenges we ran into

We ran into many challenges in determining what tools and utilities to build this idea off of. A hardware base was required that could handle filesystems and music files which is something an Arduino lacked without specified code. We ended up using a raspberry pi with a few accelerometers.

Accomplishments that we're proud of

We're proud of the ability of Drumless, figuring out the proper hardware, such as the accelerometers, to use made developing this idea more interesting.

What we learned

We learned specific raspberry pi hardware libraries offered by Adafruit and how to implement them in Python.

What's next for Drumless

The next plans for Drumless are to create a smaller and more comfortable device preferably with two for the utilization of both hands.
