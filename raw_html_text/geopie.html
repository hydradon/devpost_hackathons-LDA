Inspiration

My experiences as a black software developer in tech and the challenges in being able to find opportunities.

What it does

Notifies candidate about tech job openings in a .5 mile radius while traveling in realtime

How we built it

We utilized the ESRI Javascript API and dojo along with node.js and nexmo.

Challenges we ran into

We ran into developing a way to simulate a demo and implementing this demo. The reason is due to it being location based so it required physical movement to demonstrate the notifications that ring into the phone. It is a partial augmented reality application so this made it difficult.

Accomplishments that we're proud of

We are extremely proud of coming up with an idea and implementing it overnight. We are also extremely that we have completed something that can become a value add to a major issue.

What we learned

We learned how to remain calm under time crunching pressure. We also learned a lot about new API's that are available.

What's next for GeoPie

launching to market! We are looking forward to improving and expanding the features of this application to our target audience
