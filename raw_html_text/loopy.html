Inspiration

I was singing in my room one day and was annoyed that I could not remember lyrics to my favorite songs. It was inconvenient to search for lyrics one by one.

What it does

It integrates with your current spotify account and automatically shows you lyrics of what you are listening to.

How I built it

I used a lyrics api and spotifys api to make all of this happen

Challenges I ran into

I could not deploy my web server so I had to run it locally. There is also a glitch where I have to login twice!

Accomplishments that I'm proud of

I found ways to bring API's together and make them work with each other.

What I learned

How to develop for the web in a more efficient way.

What's next for Loopy

Fix the bugs and expand functionality to search for songs etc..
