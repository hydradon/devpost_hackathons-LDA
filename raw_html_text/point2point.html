Inspiration

We felt Georgia Tech's transportation system was not satisfying our needs, so we decided to find a way to efficiently get around campus.

What it does

It is a web-based application that tracks your location, marks the bus stops around campus, and provides routes and directions from point to point.

How we built it

We built the website using html, css, and javascript. The API provided by ESRI created the map on the website.

Challenges we ran into

We had trouble creating popups for the points to describe their location, and it was a challenge connecting the API to the website.

Accomplishments that we're proud of

We managed to create a nice interface for the website, and completing the code for the map.

What we learned

We learned how to connect Javascript files to HTML files and connecting HTML files to other HTML files using APIs.

What's next for Point2Point

We would like to limit user interface so there is less searching for map points, adding more bus stops, and creating a mobile application.
