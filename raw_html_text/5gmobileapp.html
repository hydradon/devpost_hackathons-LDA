Inspiration

A better location and mobility solution to help empower people to make the best location decisions. A more advanced mobility tool to help mark transportation more efficient.

What it does

What it does more advanced map than is typically used. Our lives are composed of events, these are time and location. We help empower empower people to be in the right location at the right time, to be sure they are living their best life.

Our app helps people to better navigation with image recognition to find where a landmark photo image was taken, and then map a 3D path to that destination.
The user takes a photo, which the app can recognize uniquely, then it can find the location of the landmark.
Next the app helps the user to navigate to the destination with life-like 3D maps.

How I built it

Using Unity3D we use mapping API's to generate and load both satellite images and height maps to generate buildings at a desired lat-long.

Challenges I ran into

Generating lots of buildings came with performance optimization issues. Maxing out of API requests and optimizing that. Connecting unity 3D to IBM software
