Inspiration

I wanted to make something that would enable creators to easily make small, redeemable gift vouchers of BTC

What it does

Using the Lightning Network, Opennode and Firebase.

How I built it

Using technologies that allowed me to build without any set up: Opennode api Glitch server (Node, backend) Firebase hosting (React, frontend) Firebase database (database)

Challenges I ran into
