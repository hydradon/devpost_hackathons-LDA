Inspiration

As a group, we love to watch movies. They are a great medium of creativity and story-telling. The only issue is, there are two main problems with how movies are present to the public. Reviews are bias and subjective. Recently, there have been many scandals with critics taking bribes to rate certain movies higher, affecting the reliability of movie reviews. Additionally, the reality is that everyone has different preferences and custom features they search for in movies, making a global rating of a movie irrelevant. Our program fixes this issue.

What it does

MovieMe is a custom rating system that utilizes a revolutionary multi-dimensional clustering algorithm to improve the rating of movies based on the users requesting in the website. Every user gets a different rating for different movies based on their preferences, and the system gets better the more it is used.

How we built it

We used Vue for the front-end of the website and node.js/firebase for the back-end.

Challenges we ran into

When creating the algorithm, we spent hours fixing issues and polishing the program to make it as accurate as humanly possible.

Accomplishments that we're proud of

We believe that we have created an extremely accurate custom rating system utilizing our algorithm.

What we learned

We have learned how to connect code from front-end and back-end and how to work together as a team towards a common goal.

What's next for MovieMe

We are going to continue developing the program for an official, public, release.
