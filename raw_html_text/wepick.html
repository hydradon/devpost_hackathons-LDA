Inspiration

People spend about 11-30 minutes getting ready in the morning. I spend 10+ minutes to choose the right outfit because I live in San Francisco. In San Francisco, the weather and temperature change drastically. "Should I bring a jacket? Or simply t-shirt and shorts are enough?" Thinking about these questions takes lots of time in the morning. I wished I could have an app that understood my needs and told me what to wear.

What it does

By inputing user's comfortable body temperature, the app generates an outfit tailored to you.

How I built it

(1) Fetched data from Weather API (2) Parsed the weather and temperature to build the recommendation algorithm (3) Added Google Places to search cities

Challenges I ran into

(1) How to build a recommendation system (2) Making sure the clothing images react in time with multiple behaviors (i.e. search cities, change comfortable temperature, change outfit)

Accomplishments that I'm proud of

(1) Finished a basic recommendation system relevant to personalization (2) Created all the clothes/images in Illustrator (my first time using it!)

What I learned

(1) Implementing APIs (2) Refactoring code/writing elegant code (3) Good debugging procedure (4) Never give up!

What's next for WePick

(1) Upload user's wardrobe (2) Add texture, thickness of clothing items (3) Analyze images by incorporating Watson API (4) Make the recommendation more specific (5) Integrate Google Analytics to understand user behaviors
