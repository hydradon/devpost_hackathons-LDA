MarkerMarker

Earlier this day we tried to write on the board but we couldn't find any of the markers. Then we realized that we need a convenient way to report missing markers and erasers to facilities. So we designed this website that takes information including the location with missing stuff, the category of the missing stuff, and then generates a report table for the facilities to view. When duplicated reports happen, the latter ones would be ignored. When a missing marker or eraser is supplemented, it will be deleted from the table. If the reporter of a missing marker or eraser found the marker or eraser after reporting it, he or she can also report it as a "have", so that the former report will be canceled. Link to the GitHub repository: https://github.com/natewatson999/MarkerMarker/
