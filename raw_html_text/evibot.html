Inspiration

Often when there are major cases about sexual assault one of the first things to get questioned is the victim's credibility, especially due to an inability to recall details about the incident. The goal of this app would be to store some of those details so that a victim's credibility is no longer a big question in sexual assault cases.

What's next for EviBot

There are a couple of improvements that could be made to EviBot. The first is that we would like to be able to add multiple pictures to a report. It is unrealistic to assume that only one picture would be needed for every report but we were unable to implement a way of getting multiple images in the time that we had. The second and loftier goal would be to get the data onto remote servers so that the app could report where incidents occur frequently.
