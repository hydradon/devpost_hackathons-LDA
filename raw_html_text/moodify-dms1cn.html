Inspiration

We wanted to do some machine learning

What it does

Gives users music recommendations based on social media posts

How I built it

Scraped social media posts using a chrome extension, sends the data to ibm-watson tone identifier and then uses that data to give the end user ideas of what songs to listen to

Challenges I ran into

Setting up OAuth2 for Spotify and scraping web pages with chrome apps

Accomplishments that I'm proud of

Setting up a backend was quick and easy

What I learned

It's very difficult to scrape data off the web as it's being inputted by the users

What's next for Moodify

Probably not much since it needs to be re worked into something somewhat different
