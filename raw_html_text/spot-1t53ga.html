Inspiration

We wanted two things at the same time. We wanted a way to find new music without having to search for it, and we also wanted an app that would automatically play music that fit the right mood.

What it does

We wanted to use google awareness to track when earphones are plugged in. When it detects that earphones are plugged in, it automatically plays music based on your situation. The situation being your location, time, and weather.

How we built it

We tried using react native

Challenges we ran into

None of us knew react native, nor had any of us done mobile development before. It also took us a long time to get set up. Additionally, we didn't realize react native does not work well with google awareness until we were too deep into trying to learn react native to back out.

Accomplishments that we're proud of

We got somewhat familiar with react native. We learned some about using permissions.

What we learned

We have a lot of work to do when it comes to functional languages

What's next for Spot

Once we get it to actually do what we want, we then want to make it so that the app is always running in the background
