Inspiration

We don't know where to find the products for the best prices.

What it does

Allows users to share what items they've found at certain locations and for what prices, for the benefit of others shopping for that item.

How we built it

We used a traditional node/express web server with PostgreSQL, and Angular for the front-end application.

Challenges we ran into

Thinking too much into specific use cases. We should have attacked the general use case first, which would have been much easier to tackle.

Accomplishments that we're proud of

The UI is simple and intuitive (of course, it needs to be vastly expanded), but the user should be able to find what they are looking for quickly and easily.

What we learned

Planning is futile.

What's next for Pare

Finishing the google maps integration, stronger suggestion matching, and a business interface that would allow businesses to input their products.
