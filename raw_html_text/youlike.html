Inspiration

make computer vision program to like youtube videos by a simple thumbs up or thumbs down

What it does

you thumbs up your webcam and it likes videos on youtube

How I built it

custom chrome extension with modified GitHub repo of hand gesture recognition mixed with youtube API authorization/action

Challenges I ran into

gesture program dependencies, youtube API connection, chrome extension design and implementation, seamless integration into webpage

Accomplishments that I'm proud of

product that works, creating a chrome extension, even if it takes many steps to accomplish, connecting two concepts into one program

What I learned

more python skills, program flow analysis, computer vision techniques, chrome extension building, use of host process with browser

What's next for YouLike

make it so that all you need to do is thumbs up the camera and it will like the video within the webpage without the need to refresh, toggle animation and instant thumbs up recognition, no google account authorization
