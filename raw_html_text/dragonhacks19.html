Inspiration

A new twist on an old classic genre.

What it does

Rogue-like dungeon crawler that allows users to cast spells with gyroscopic controls.

How we built it

Started with game development and control development separate. Then merged the two so that attacks could be created from the control scheme events.

Challenges we ran into

Getting the core down so that we could focus on the more unique and enjoyable parts.
