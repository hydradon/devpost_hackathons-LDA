Inspiration

I was inspired to code this bot from all of the other bots that exist. Some are really creative, and some are useless.

What it does

This bot posts random facts and random jokes in chat, triggered by commands.

How I built it

I built it using JavaScript. The server that runs it is a Microsoft Azure VM running Windows Server 2016.

Challenges I ran into

I had some issues with missing Node.JS modules.

Accomplishments that I'm proud of

Making it work.

What I learned

A bit of JavaScript. Also how to make an array in JavaScript.

What's next for Discord JokeBot

Maybe a command like !picof [object]. This would search Google Images for the specified object, and post the first result in the channel where the sequence was triggered from.
