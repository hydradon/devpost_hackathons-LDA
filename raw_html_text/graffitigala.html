Graffiti Gala was conceptualized following Google's AR demo for Hack@Brown . While creating AR graffiti without vandalizing historic Brown university buildings is quite a feat, we decided graffiti is more fun if you can share you work with others.

We built an app that allows users to see and edit an augmented reality world, while seeing others drawings all in real time. This makes it seem like the users are editing one shared reality.
