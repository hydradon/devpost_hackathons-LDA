Our andriod application is called NYCmyTree which helps locate a NYC tree using the location setting on someone’s smartphone. The user can then access info including species & Latin name, health, status, history including why a certain species was chosen to be planted in NYC, other links & fun facts, and a “help this tree” option that lists instructions on how to care for the tree.
