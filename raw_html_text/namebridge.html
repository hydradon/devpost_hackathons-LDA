Inspiration

We wanted to work on some sort of AI intelligence, and as soon as we heard about Metamind, we knew that it was the perfect technology and we needed to use it.

How it works

We gathered data from DC open data and fed them into Metamind and trained the AI to be able to identify what category a venue name would be most likely to fit into - nightclub, restaurant, gas station, etc.

Challenges I ran into

Challengepost decided to crash and destroy all the descriptions I made. However, the most notable one was the fact that Metamind only had an api for Python and cURL, but I knew that you could make a request and receive a response in return, and did so via AJAX
