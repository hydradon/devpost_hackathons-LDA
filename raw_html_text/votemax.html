Our goal was to create a 21st century approach to applying for and voting by absentee ballot.

This challenge is focused on helping to address economic inequality as it relates to voter turnout.

How we built it

A IOS/Android application developed using React-Native, Redux, Firebase, expo.io, JavaScript, and Google Civic Information API

Challenges we ran into

Several team members were completely new to React Native. Issues with the most recent version of NPM made for some interesting moments.

Accomplishments that we're proud of.

We did it!

What we learned

Don't try to use NPM when it doesn't work.

What's next for VoteMax

Automatic parsing for photo IDs and ensure compliance with new state voting security rules.
