Inspiration

We were inspired by JetBlues core values!

What it does

A user can explore the JetBlue Quiz which calculates your next destination based on your answers. A user can interact with the web page to view an embedded twitter timeline displaying JetBlue's tweets.

How we built it

Front-end: Javascript, React, & Bootstrap Back-end: Node.JS, Express, Sequelize, PostgreSQL
