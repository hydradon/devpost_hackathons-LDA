Inspiration

Facebook messenger has an unique platform that hasn't been explored a lot. We thought it would be interesting to use Facebook messenger as an interactive story telling platform because it's also a conversational platform. We decided on using a famous twine on the internet called, "My Father's Long Long Legs", as our interactive story.

What it does

Our project bot allows people to play/read the interactive story in a Facebook messenger platform.

How we built it

We created a page on Facebook, then created a callback URL through ngrok, then wrote the server in python3.

Challenges we ran into

Unfamiliarity with the Facebook platform. We had to manually scrape and reassemble the story.

Accomplishments that we're proud of

This is my first Hackathon! I just started learning how to code and this is my very first project. I'm glad I didn't give up and turned a project in. Woot Woot!

What we learned

I learned how Facebook platforms work and how to create a callback URL through ngrok.

What's next for "My Father's Long Long Legs" Story Messenger Bot

Making more twines/ interactive stories messenger bots in the future!
