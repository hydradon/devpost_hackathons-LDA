Inspiration

We saw a really nice website and wanted to address the debate over pets.

What it does

After you ask a couple of general questions and help us get to know you slightly better, we provide the best recommendation on what pet you should get

How we built it

we designed a beautiful static web page with HTML5 and CSS3. We simply used JavaScript for our quiz logic.

Challenges we ran into

Since this was a first project for most of us, we had a hard time figuring out how to set up interactive Backend and front end suitably. We ran into a lot of logic errors and we tried to incorporate some aspects that ended up becoming completely unresponsive and unusable. We struggled a lot with working on the UI and getting the web page to display properly.

Accomplishments we're proud of

We were able to create a considerably good application with some cool UI as our first project using JavaScript, HTML, and CSS. The web page looks very nice and works relatively well.

What we learned

We learned more about development of web browsers and how to incorporate different aspects into a nice web page.

What's next for Purfect Pet

We wish to incorporate an API to improve upon the quiz and make it more diverse and complex. We would also like to get some of the aspects that we had to remove, incorporated.
