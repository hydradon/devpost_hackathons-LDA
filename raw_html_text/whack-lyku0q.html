Inspiration

In fact we want to empower locals. We believe a good business is a support of cooperation between people. We want to build an easy and accessible infrastructure for new markets to charge e-Scooters without additional expenses. Any businesses and private sectors can join the ecosystem and become a charging provider for rented e-Scooters. It allows the rental business to grow as a grass.

What it does

It allows scooter as agent to find nearest cheaper charge station, get charged on it, and pay tokens for charging. So, we have a market for electric power token.

How we built it

We made it on Fetch.Ai OEF framework and ERC20 compatible smartcontracts for economic infrastructure. Project consists of OEF stock market and tokens to pay on it.

Challenges we ran into

We found a completely new infrastructure and programming languages. It was really fun to write on python only with knowledge of indentation rules. Etch contracts wasn't too difficult, but software sometimes undocumented or behaves in different way.

Accomplishments that we're proud of

Finally it works. We defined a scope of work, budget it, and had finished in time.

What we learned

We understand the new ideas of blockchain projects interoperation. It is sad, that we used only fetch.ai, but we see the potential of this cooperation. Here is a lot of new tools in open source to make an application with it. It differs from situation we had five years ago.

What's next for whack

The main problem is to attract pythoners to development team. Then we may:

  • make better code for economics agents
  • implement hardware sensors and brains for e-scooters.
  • make an better algorithm for charge station search (ai)
  • make an fair pricing algorithm for charge stations (ai)
  • make an additional services for locals for interacting with renters
  • minting and selling tokens, taking fees
