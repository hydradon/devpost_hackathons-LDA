Inspiration

We wanted to make a game - including all of the logic, physics, and art - from scratch.

What it does

Burd is hungry. Burd is coming for you. Survive as long as possible.

Move with arrow keys. Attack with A, dimensional shift with S.

It's best played offline as a LOVE game package, but is also available online for convenient access.

How we built it

The main game is built with LOVE, a 2D game engine for Lua that is well-suited for small, self-contained games. All of the logic and assets were written and drawn over the course of 36 hours at HackWestern.

The Lua-based game was then compiled for web using Emscripten, specifically, using LOVE.js.

Challenges we ran into

Figuring out how to do collision detection/resolution, gameplay mechanics, and drawing clouds. There is only one cloud in the entire game.

Accomplishments that we're proud of

Bringing a fully playable game to life, from drawing rectangles on screen to figuring out the math to make them move around realistically.
