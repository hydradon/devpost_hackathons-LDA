Inspiration

Most people are somewhat aware of major armed confrontations that occur, but often do not know where they occur. I built this applications to hopefully alleviate this issue.

What it does

It is a map mashup of any type of armed confrontations. This means that any one can submit an entry to the map! The advantage of this is that it provides a better outlook of news, as people living in different areas have different perspectives on global armed confrontations.

How I built it

It is an android application built using the ESRI ArcGIS SDK, as well as a NodeJS backend + database solution. Initially an Arc solution was attempted, but did not work out so well for the data in this application.

Accomplishments that I'm proud of

Using the ArcGIS android SDK.

What I learned

Lots about GIS and mobile development.
