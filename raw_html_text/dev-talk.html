Features

Search : Allows developers to search and connect with local developers based on a search criteria (Locality / Programing language / Development Platform)

Stories : Allows developers to anonymously share stories about their failures .

Polling : Allows developers to recommend libraries/plugins/framework and products.

How is DevTalk different?

Search functionality : The app performs search operation based on the locality(town/city) the user inputs instead of confining the user to perform search operation within a certain radius from his/her current location.

Privacy : Since DevTalk allows anybody to signup and search for detail's (location/contact) of other users, the app implements the following to ensure that the user’s privacy isn't violated :

1) The app doesn't store or display the precise location of a user to other users because it might lead to undesirable situations.To avoid this the app stores and displays the location of a user which has only locality level precision.

2) The app shares contact details of a user with other users only if the user had enabled “Allow developers to contact me” option.
