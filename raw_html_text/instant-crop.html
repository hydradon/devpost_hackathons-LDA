Inspiration

Wanted to do an API mashup using Google Cloud Vision.

What it does

Detects faces uses Google's cloud vision engine, and automatically crops out each face into a profile picture format. Link with Google Contacts to create or update a current contact automatically.

How I built it

Software backend built with Python-Django. Uses google-api-python client libraries

Challenges I ran into

Google Contacts API hasn't been updated in 4 years. TERRIBLE documentation. Critical functions don't work.

Accomplishments that I'm proud of

Thought of this at 8am on Saturday, and group had a working prototype online in less than half a day. Lots of awesome help from the hackathon.

What I learned

Don't underestimate the power of brute force and ignorance.
