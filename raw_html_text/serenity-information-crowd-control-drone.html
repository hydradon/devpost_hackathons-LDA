• Serenity in the making

Inspiration

During a large event in Baltimore called Light City there was a frantic crowd.

What it does

It flies over a crowd of people to display a message to either direct them to safety or remind them to be clam and look of for other people.

How we built it

We used an Arduino, an AR.Drone, and a NeoPixel Matrix

Challenges we ran into

The Arduino only has one core so we have be smart with or code to allow new request to come in while an important loop ran.

Accomplishments that we're proud of

Interfacing with an web form to update the info on the NeoPixel Matrix.

What we learned

We learn how to use some arduino libraries and methods.

What's next for Serenity - Information Crowd Control Drone

We're not sure, maybe make in look nicer.
