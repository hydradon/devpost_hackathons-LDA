Inspiration

One of us enjoyed playing games online and asked the rest of us for help on an anagram puzzle. It was a challenging word, and we could not solve it. We decided to create an anagram game in which we could enter our own words and challenge each other to solve it.

How it works

The user enters a word and then the word is scrambled. The scrambled clue is displayed and the user is given chances to guess the word. Each word starts at 10 points, and its value decreases over the course of the game. If the user guesses a wrong answer, the word's worth decreases by 1, and if the user asks for a hint, it decreases by 2.

Challenges I ran into

At one point, we wanted to all guess the word, so we needed the computer to generate its own word for us to guess. We had to establish a word bank for the computer to pull from and make sure that there were no words repeated.

Accomplishments that I'm proud of

The method we used for scrambling the word needed conversions from a string to a character array, and then back again.

What I learned

There are always things to add on to the game, and there is always stuff to work on.

What's next for Anagram Game

Multiplayer capability for 1 on 1 playing.
