Inspiration

Everytime I go to Starbucks, they get my name wrong

What it does

The system tells you which names are phonetically close to your name and there is a high probability if you use those names, the Starbucks server might get your name correct.

How we built it

HTML, CSS, JavaScript, PHP, Bootstrap

Challenges we ran into

variable names and syntax as we did not sleep

Accomplishments that we're proud of

pulled this off in 2 hours

What we learned

no matter how much trouble you are in, a small joke can bring in a new idea

What's next for Name's Fake

data analysis and more mis-classifier and algorithms to get in wrong input to get right output.
