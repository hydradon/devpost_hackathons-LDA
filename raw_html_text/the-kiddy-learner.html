Inspiration

We were inspired by our love of children. Many of us have siblings and young family members whose educational development we have observed. As we were brainstorming, we decided to capitalize on this passion by creating The Kiddy Learner!

What it does

The Kiddy Learner is an educational website that displays various scientific cycles with a visual component.

How we built it

We used html, css, and javascript.

Challenges we ran into

As beginners, we ran into many syntax errors. Two team members did not know html, javascript, or css and the learning curve was a challenge.

Accomplishments that we're proud of

We finished and learned a lot!

What we learned

The two members that did not know the language we coded in, learned the basics of the languages. The whole team learned about the process involved in collaboratively working on a website.

What's next for The Kiddy Learner

We would like to learn more javascript and css to make the site more interactive. Also, we would like to be able to add more pages to the website.
