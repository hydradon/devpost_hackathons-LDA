Inspiration

We were inspired by the stories told by travellers of places they've been.

What it does

Fable allows you to listen to personal stories attached to map locations.

How we built it

We first built a mockup for this project at Yale's YHACK. There, it won the Cultural Heritage award. Now we are building a functional version with map tagging.

Challenges we ran into

The HTML5 default audio player is super super ugly and inconsistent with our style.

To fix this, we had to build our own audio player. This caused many problems.
