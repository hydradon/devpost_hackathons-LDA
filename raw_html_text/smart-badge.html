Inspiration

Easy manage and identify people on events

What it does

App loaded on iPhone or iPad will turn it into Host device. Any guest attended on event will pre-fill some "guest form" with a picture, and will supplied with some badge/band/wrist band with Beacon implemented in it. When guest pass by or approach the Host, the "guest form" with picture will popup on Host screen.

How I built it

Dmitrijs build iOS native application using Xcode. Some demo profiles preloaded on host device. Tom build remote server and host in on Amazon cloud and upload demo "guest forms" on it. In app setting user may have option used preloaded "guest forms" or used cloud version.

Challenges I ran into

How to present "guest form" on host device if more then one guest in-range.
