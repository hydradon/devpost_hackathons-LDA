Inspiration

We all see sprinklers on farms and in lawns going off when its raining. This wastes water.

What it does

Our platform uses future weather forecasts and live soil moisture readings to remotely control watering systems.

How we built it

We built this using Flask, and Arduino, and the Google Cloud Platform. We built a HTTP API to access our backend platform.

Challenges we ran into

This is our first time using Flask, or a cloud platform to deploy code. This is also our first time using Flask to create/receive requests.

Accomplishments that we're proud of

We're proud of developing a working proof of concept.

What we learned

We learned the basics of using Flask.

What's next for Smart Horticultural Irrigation Technology

Getting a full UI frontend implemented in Vue.js.
