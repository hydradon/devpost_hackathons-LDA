Inspiration

Firebase talk at HackRU IX

What it does

Users can enter hex values into the box. In real time, pixels of whatever color are added to the picture. Users can watch their image unfold in real time!

How I built it

Using Firebase, hex values are pushed to the database and immediately pulled in each local user's session.

Challenges I ran into

Handling the pixel generation was a bit of a challenge.

Accomplishments that I'm proud of

The whole thing works! Multiple people can watch pixels get added in real time!

What I learned

Firebase is awesome.

What's next for CrowdPixel

A few more buttons for easier access to primary colors, maybe.
