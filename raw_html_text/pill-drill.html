Inspiration

This project was made for all individuals with different medical needs, ranging from prescription drugs, over the counter drugs, supplements, and other miscellaneous items

What it does

User inputs data into the system which is then displayed. The website/app allows users to track their medication intake and other details.

How we built it

We built this application using repl.it, an online platform that allow all user to edit in real-time.

Challenges we ran into

All member of our team had very basic knowledge in coding, thus being able to make a tangible product given our experience was a challenge.

Accomplishments that we're proud of

Given the limited amount of time and experience we had, we're proud of all the wire framing and implementation of the application.

What we learned

We learned that creating an application is not easy and requires many individuals with different skill sets.

What's next for Pill Drill

While we created a web application, our next steps would be implementing a mobile app along with it.
