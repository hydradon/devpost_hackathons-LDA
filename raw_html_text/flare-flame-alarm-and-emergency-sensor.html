Dementia is a general term for a decline in mental ability severe enough to interfere with daily life. Memory loss is an example. We intend to help people with Dementia. We have designed a device with flame sensor and the Rasberry Pi to detect the flame and notify the patient if any flame is unattended for a long time.

It looks for the flame and is connected to a computer wirelessly through the Rockwell Gateway API. When a patient forgets to turn the cooking stove off after cooking. This device can notify the patient to turn it off

We used flame sensor LM393 to detect flame. It is connected to Rasberry Pi and then through the Rockwell Gateway API we send this information computer wirelessly.

Working with sensors and Rasberry Pi.

We were able to complete the project within the a day. This is impressive. The project will have an impact on lives of people with dementia.

We learned working through the difficulties to implement the project with a short period of time

Making is robust. Making it able to connect to an Application in a mobile device.
