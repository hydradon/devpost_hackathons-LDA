What it does

Using an algorithm, StockSee predicts future prices over one year for stocks in the Energy, Financial, Healthcare and Technology sectors.

How we built it

First, using Python, we used the Yahoo Finance API, we took in data over a period of 10 years for a stock, and then converted it into a CSV, and then into an SVG graph. Afterwards, we iterated through the CSV, to isolate the value for the 'close' for every data point, and added it into a new array. Then, we plotted each day with each future data point, which we were able to find using the average difference of close values for our past data, and using that value when determining future values. Then, we created a website, for which we displayed our information and predicted prices.
