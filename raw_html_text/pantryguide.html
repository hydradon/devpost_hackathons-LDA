Inspiration

Everybody knows this problem. You want to eat some pasta? Problem: You have forgotten to buy it. But you know, you want to have Pasta at home all the time. So why is there no reminder, when to buy pasta.

Now it exists: It's called PantryGuide. It's even better... It buys the products at Amazon Pantry automatically.

What it does

Choose your favorite products and amount, you always want to have at home. The app will remind you ,when a new Amazon Pantry order is needed and what to buy.

How I built it

Built entirely in C# using Microsoft SQL Server as database structure. For data access Entity Framework was used. ASP.NET WebApi was used for creating an API.

Challenges I ran into

Scale our algorithm to multi users. We did not have time to develop a front end.

Accomplishments that I'm proud of

The algorithm, when to order items.

What I learned

Using Amazon Web Services

What's next for PantryGuide

Develop a mobile front end
