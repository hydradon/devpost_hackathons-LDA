Inspiration

I wanted to come up with a way to make finding organizations to volunteer with more accessible.

What it does

It presents the user with tinder-style swipecards and profiles for philanthropic and service organizations in their area, and allows them to easily communicate with spokespeople of the organizations.

How I built it

The front end is all android studio in kotlin, and the database/auth/storage is handled by Firebase and GCP.

Challenges I ran into

The main challenge was getting everything done on time. Additionally the text on the messaging has a very large spacing between each message.

Accomplishments that I'm proud of

This is my first time working with android studio or any kind of database, so I am glad I was able to create a working product

What I learned

Tons about mobile development and database organization
