Inspiration

Personalized content has more value than generic content. Got inspired by Facebook and Google on how they personalize content, which has been a factor to their success.

What it does

We have developed a Personalization SDK - both REST and Android, the developer can use this minimalistic API, so that his app users can consume personalized content, and the developer gets to see the analytics of personalization and engagement.

How we built it

We developed an Android SDK and a REST interface which act as Personalization Engines.

Challenges we ran into

Coming up with the learning algorithm that can digest almost any type of raw data, and give a personalized data.

Accomplishments that we're proud of

Our first SDK! And everything just works. #TouchWood
