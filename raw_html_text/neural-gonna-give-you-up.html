What it does

Passes a song through a recurrent neural network to make it sound more like Rick Astley's wonderful song "Never Gonna Give You Up"

How we built it

We processed midi files before sending them through a recurrent neural network developed using keras and tensorflow.

Challenges we ran into

Understanding how midi files are used to represent music. This was difficult, but necessary to interpret the files.
