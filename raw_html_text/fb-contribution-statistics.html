Inspiration and What it does

Introducing FB Contribution Statistic for engaging Silent Readers on FaceBook

Thousands of Users on different Facebook Groups are just there to read only with little or no contribiution to other people's life on facebook. This app was design to engage all this Silent Readers on Facebook.

The App Leverages Users Contributions Statistics/Performance Badges to create a challenging environment to all those Facebook read only users.
For every post shared on the platform by a certain User, that users previous Contributions is displayed on the post timeline inline with the post updates for all other users reading the post to see.

Imagine when someone Shared a Post and her contributions Statistics tends to zero or 1 while He/She is not a new person on Facebook. The Person will know that he or She is not doing well and her contribution to other peoples life is highly needed on Facebook..

The App shows statistics based on the following components

1.) Posts: This is defined by total number of Post Status Updates made by that User in her entire life on Facebook
2.) People Made Happy: This is defined by total number of Likes made on all the Users Post updates so far.
3.) Interaction: This is defined by total number of comments that was commented on all the users post so far
4.) Crime: This is defined by total number of time this User was reported as Posting Illegal Contents, Frauds etc. on Facebook.
5.) Disciplined: This is defined by total number of times this User help Facebook in fighting Fake News, Illegal Posts, Ponography etc by reporting the Offensive Post to the Facebook.

Seeing all this amazing Statistics, the question to Facebook Silent Readers now is?
A..)How many Post have you contributed on facebook,
B How many people have you made happy,
C. what interactions has you triggered on facebook?.
With all this in mind, this Statistics will create a challenging environs that will let Facebook Users to know their Performance and to Read only Users to try catch up with others no matter how little their contributions might be. This will in turn tends to create more engagement ever in the history of Facebook.

This is because the more you contributes:

1.)The More your Statistical counts is perpetually raised on Facebook for an award.
2.)The More you are likely to make someone happy.
3.)The more are likely to trigger an interactions that might solve someone's Problem
4.)The more happier and Popular you will become on Facebook thus taking your credits/glory going forward

How use this App

1.) Login using Facebook.
2.) You can Share Informations through Post Status, Photos and Video Updates.
3.) Each time you Shared a Post, Your Post Count Goes up by 1
4.) Each Time You Comments on any Post, Your Interaction counts and that of the Post owner count up by 1
5.) Each time You Like someone Post, You are happy and have made someone happy so the happy count goes up by 1
6.) Each Time you report in-appropriate Post to Facebook, You become a disciplinarian and your discipline count goes up by 1
7.) Each Time Your own Post is being reported by other users as being Offensive, Your Crime count goes up by 1

8.) You can refresh the app to see your current statistics if it does not show immediately.
9.) Share on Facebook using Facebook Share Button on each Post

FaceBook Technology Used:

1.) Facebook Login Graph API: Used to authenticate Users
2.) Facebook Shared Button: used to allow user to share content to her Facebook Timeline
3.) Facebook Send Dialog Button: used to allow user to share content to a specific user in HIPAA Compliance Mode.
