Inspiration

I am a planetarian and the Berea College Planetarium and I wanted a contained source for all the information I needed to know about the current night sky for giving a sky talk in the planetarium.

What it does

Gives information about the current phase of the moon and when it will be up, what planets will be visible, how many exoplanets have been discovered and astronomers birthdays.

How I built it

Using astronomical python packages to compile and output information about the night sky

Challenges I ran into

Interfacing with astropy to communicate and retrieve all constellations that will be in the night sky.

Accomplishments that I'm proud of

The visual aspect of the website

What I learned

How to build a web app using flask

What's next for StateoftheUniverse

Hopefully a usable phone app
