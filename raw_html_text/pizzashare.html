Inspiration

Late hungry nights when we had to order a large pizza to be cost effective.

What it does

Helps you share bulk costs attached to various food deliveries (tax, delivery fee, etc.)

How we built it

Github as a repo, HTML/CSS from scratch and javascript functionality through bootstrap alongside Google Maps and Paypal API. Didn't get to finish the server through js which would host our "post order" forum.

Challenges we ran into

Since we start learning and making the project from scratch, we didn't get the chance to finish its host server.

Accomplishments that we're proud of

Learnt how to use APIs and do general web dev!

What we learned

We technically learnt everything from scratch!

What's next for PizzaShare

Probably open source it or finish it in our personal time!
