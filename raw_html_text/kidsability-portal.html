Inspiration

Create a portal to open communications with families and staff

What it does

Allows staff at KidsAbility to connect with each other on an easy online platform - reducing the need for phones

How we built it

Team work combined with react, firebase, figma

Challenges we ran into

Firebase is really hard to use on windows, a lot of us did not have enough sleep

Accomplishments that we're proud of

We learned a lot, split up the tasks nicely, everyone contributed

What we learned

Learned how to work as a team, and how to use react, firebase, figma

What's next for Kidsability Portal

Create a functional calendar, request system, support system, etc...
