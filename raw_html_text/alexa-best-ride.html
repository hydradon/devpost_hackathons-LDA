Inspiration

Prices vary so much between Uber and Lyft especially with surge pricing. There should be an easy way to figure out which service to use!

What it does

Calls both Uber and Lyft APIs to find the lowest price

How we built it

Alexa API, Java Spark Server, Uber API, Lyft API

Challenges we ran into

Figuring out how to use the Alexa API, Figuring out authentication for Lyft and Uber APIs

Accomplishments that we're proud of

We have a presentable product!

What we learned

It takes a whole lot more than 12 hours to make a finished product!
