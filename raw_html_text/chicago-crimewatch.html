When looking for houses and schools, one major factor for a lot of families is safety. We built this tool to allow people to crowdsource information to see changes in real-time. In the same way that crowd-sourcing information has helped Google Maps become as ubiquitous as it is, this tool will help people avoid areas where crimes may be higher.

Using Firebase, we keep all clients in synchronization, so that if a client reports a new crime, it appears on all other clients, notifying them that this is a recent hotspot. Further, using Firebase's hosting tools, we removed the need for a traditional server, making this app fully scalable.
