Inspiration

We all run into traffic situations in big cities. The amount of time spent in traffic could be saved and used in more productive aspect of our lives.

What it does

It automatically loads traffic information from nearest user location and provides the user with option to input address. We pull in intelligent traffic data from available mapping services.

How I built it

I designed the concept and worked with my developer to identify various services to build the app.

Challenges I ran into

The amount of time it took us to integrate to ESRI

Accomplishments that I'm proud of

Being able to eventually submit the app

What I learned

The ESRI platform provides some quite interesting feature we might be considering down for some feature development.

What's next for HighStrit

We want to provide some additional features that will make it useful to road users.
