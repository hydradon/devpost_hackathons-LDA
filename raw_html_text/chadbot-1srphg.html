Inspiration

Ever find yourself bored with the regular chatbots out there? Chad's personality is unique, unlike any of the conventional chatbots! Chad's purpose is simply to entertain. If you ever find yourself with nothing to do, talking to Chad would be a great use of your time.

What it does

Our machine can interact with humans, responding to basic phrases!

How we built it

In order to create Chad the Chatbot, we used... HTML (to build up our website structure), CSS (in order to make the website look presentable), Flask (to implement Python into our HTML), JavaScript (to input Flask and to enable the Python to be translated, displaying our output onto our website).

Challenges we ran into

We needed admin permissions in order to do most of the things we wanted, and we were unable to receive help from many of the mentors, since most of them left later on in the night-- thus, delaying our project back around 8 hours.

Accomplishments that we're proud of

We were able to survive and give Chad a website.

What we learned

It is rare to finish a project as dense as this overnight. We had very high aims, but ran into a few problems that prevented us from actually putting out chatbot onto the website.

What's next for ChadBot

We plan on making an actual live website and implementing deeper learning AI.
