Inspiration

We wanted to learn how to utilize drone-based robotics and integrate the mechanical component of the project with our Machine Learning knowledge.

What it does

Viewers can give commands to fly a drone which capture image and identifies objects in said image.

How we built it

Using the ar-drone control modules along with the API for Twitch chat and then tensorflow for image identification

Challenges we ran into

Drone ran out of battery, I.E. limited testing

Accomplishments that I'm proud of

This drone can identify anything in image net. Running three separate scripts that communicate with each other simultaneously.

What I learned

A lot about servers and sockets, Also that many people playing twitch chaotically is very fun with a somewhat dangerous machine.
