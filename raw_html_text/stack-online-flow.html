Inspiration

Facebook Is A Medium To Connect With Potential People And Solve Your Problems And For Making Productive Coders We Have Tools Like Stack Over Flaw.But Whenever We Ran Into Some Problem We Usually Look Solution In Internet Which Is Matching Our Issue.What If We Have A Tool That Can Share Our Code Editor With Our Friends Or Potentially Good Developers In The Friend Circle Of Our Facebook Account And Working On The Problem / Coding Simultaneously In Real Time Or Code With Your Team To Make Products And Sharing Words Between Them To Create Awesome Things Or Better Solutions.That Would Be The STACK ONLINE FLAW

What it does

By Clicking The Add Friend Button In The Sidebar , A Link Will Be Generated.This Link Can Be Shared To Your Facebook Friend Or Mentor.So Whenever Your Friend Open Up The Link , Your Code Editor Will Be Shared With Your Friend And Your Can Share The Problem / Issue By Clicking On Chat Icon Or Code Simultaneously To Achieve Solution Or Make Some Awesome Products.

How I built it

I Used Technologies ReactJS(By Facebook) As Major Frontend Framework , Backend In Express And Database In MongoDB.

Also I Used Other Open Source Technologies Like Codemirror And TogetherJS.

For Bundling And Packaging Webpack Is Used And Socket Programming For File Ops.

Analytics Is Managed By Using Facebook Analytics.
