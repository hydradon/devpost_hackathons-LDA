• Pictures can be taken directly through the app or selected in the gallery.

  • You get the result of your code texted directly to you!

Inspiration

We thought it would be pretty cool to be able to hand write some code and run it simply by taking a picture with a phone.

What it does

You take a picture of Python code and you receive a text with the output.

How we built it

Java for frontend client, Python for Backend, everything hosted on AWS
