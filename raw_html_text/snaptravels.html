SkyScanner - API

Inspiration

Our inspiration came from personal struggles to decide on a destination to go on holiday.

What it does

A user answers a few questions regarding the type of holiday and cuisine and a destination will be generated. Using the SkyScanner API, we can pull the cheapest flights to the destination and give users a direct link to book. We also use the flickr API to obtain images of the question options and the destinations for a more dynamic and visual holiday booking experience.

How we built it

The program is built on the LAMP stack, with the application code in PHP, a MySQL database for analysis.

Challenges we ran into

There was a huge amount of the data returned by the SkyScanner API. It was difficult to understand at first.

Accomplishments that we're proud of

Being able to build a web app in under 24 hours.

What we learned

Real-life constraints with APIs such as rate limits and request caching and the impact these have on our application

What's next for SnapTravels

We'd like to request images from Instagram for more choice, and further develop upon the SkyScanner API to get hotel and car transfer information to enhance the user experience.
