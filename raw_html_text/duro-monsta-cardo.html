Inspiration

This is an idea I had a few years ago. Now I was able to code it because of my time at Uni. Figured it would be a good idea to try it here since its my first time

What it does

You press a button and a card is pulled down from the screen with a voice over line of "DURO, MONSTA CARDO". This is the main feature. There is 2 other features such that it counts how many times the button has been clicked by the user in the current session and then on the php version it also tellls the user how many times the button has been clicked since the page was loaded

How I built it

Used simple html and CSS Used more Javascript than anything else that did:

  • the number generator to determine what card comes next
  • to play the audio
  • the move the card down and then back up
  • change the class associated with the cards so that the one coming down is always on top

Challenges I ran into

I forgot almost all my html and CSS. Simple fix was just to go over til I started to remember everthing. Another Challenge was getting the card that was coming down to be on top of the other card, It was resolved by giving the two cards classes "top" and "bot" which gave z-indexs on the cards where I used javascript to switch the classes between them to The final one was was sending data to a server without refreshing the page, I followed enough tutorials of AJAX and Jquery so that I could do that (I did not learn enough Jquery for it to replace Javascript). AJAX allowed the php file to be used constantly without refreshing the page so that the client side javascript still kept count
