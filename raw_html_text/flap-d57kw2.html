Inspiration We wanted to pioneer a new form of gaming experience by combining the Oculus headgear and MyoBands

What it does

This a third person bird simulator game that requires the user to flap like a bird to fly though obstacles

How we built it

We built it using unity3d and C# language

Challenges we ran into

We had trouble with having the game loop after the user lost and creating the random obstacles in the game.

Accomplishments that we're proud of

We are most proud getting the MyoBands to work for them game and finishing the game so everyone can enjoy

What we learned

All of us had very little prior experience using Unity and C#, so we all learned a lot about the engine and language by building this game

What's next for Flap

We hope to make it multi-player in the future to make it more social.
