Problem: malls and retailers losing in-store customers to online shopping, and in need of a solution. http://www.forbes.com/sites/timworstall/2015/01/04/the-shopping-malls-really-are-being-killed-by-online-shopping/

How it works: Watson uses Twitter social feed data and processes it according to a variety of different natural language inferencing tools - this data is then connected to local geolocation, providing for a rich highly personalized shopping concierge experience like never before seen.

Challenges I ran into: getting Watson Alchemy API to render real time.

Accomplishments that I'm proud of: producing real time sentiment data and matching it to geolocation inferences that leverage social media and watson tech to provide personalized data at a new level.

What I learned: Teamwork and new Watson tech like machine learning!

What's next for Shop Keeper: exploring the idea and how using Watson technology can transform the marketplace.
