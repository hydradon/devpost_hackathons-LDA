• The TMT team & co.

Inspiration

Helping moms around the world with their typing problems.

What it does

It is a training keyboard that given a text file will light up a single key at a time and keep track of how many mistakes you made.

How we built it

Built it by reverse engineering the provided code and some unmet need to train noobs around the world to Corsair's keyboard.

Challenges I ran into

Alot.

Accomplishments that we are proud of

Having completed this hack in time and sharing it with the world.

What we learned

C++, Corsair's SDK, how to type faster and more efficiently. Last but not least the great feel of doing all this with a mechanical keyboard.

What's next for TmT keyboard

Motivational and inspirational scrolling messages over the keyboards.
