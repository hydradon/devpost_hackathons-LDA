Inspiration

We were very inspired by the changing climate, and wanted to make a game to address it. We wanted to attack the problem from an emotional aspect, so we named the tree. The idea to have 2d sprites in a 3d space was inspired by Danganronpa. 2d sprites are easier to make and get the exact same task done as a 3d model would; communicate that there's an object there and not disrupt the scene.

What it does

You grow a tree by fast forwarding time. You can also Upgrade your tree too.

How I built it

We used Unity and C# to build the the game and blender to make the intro animation.

Challenges I ran into

converting drawings on paper to drawings in real life. Brainstorming ideas on how to play as a tree. the skill tree was difficult.

Accomplishments that We're proud of

The graphics/artStyle turned out really well We started out with really good brainstorming.

What we learned

Leadership skills, Time Management, And the difficulties that stress and sleep deprivation cause.

What's next for Tim Berr

A skill tree with more options, more trees, add game over conditions.
