Inspiration

We wanted a new and refreshing take on social media. Especially a mix of twitter with the anonymity of the glorious Yik Yak (RIP).

What it does

Message in a Bottle allows users to submit anonymous messages to the platform. The messages are then randomly delivered to other users who wish to view other's funnies. The messages have a higher chance of discovery based on the amount of claps(like likes but better) they get.

How we built it

We used react native for the front end and UI work, and we used firebase for the backend in order to hold the submitted posts. The app was built for cross-platform experience using the Expo tools and JavaScript.

Challenges we ran into

Integrating firebase to hold the post data.

Accomplishments that we're proud of

Learning a new set of tools including react native and firebase.

What we learned

How to code good.

What's next for Message in a Bottle

Take it to market. Probably pivot to 'Mason Mail'
