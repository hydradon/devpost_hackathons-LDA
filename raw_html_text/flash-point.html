• Flash Point

Inspiration

Passionate about developing project that would be potentially helpful in data visualization of complex datasets and applying machine learning to solving conventionally natural-science related problems (like flood prediction).

What it does

Utilizes machine learning techniques to analyze Big Data from multiple sources and plot a heat map of water levels over time in a specific location.

How we built it

We built it with Bootstrap and JQuery for the Front-End and node.js for the backend.

Challenges we ran into

Manipulating large dataset from multiple sources which came in with different formats was difficult to deal with.

Accomplishments that we're proud of

A notable MVP that provides evidence that machine learning does result in accurate predictions that achieve ___ accuracy when compared with real results.

What we learned

Applied machine learning and front end web development to build a potentially deployable app.

What's next for Flash Point

Continue to solve unfinished problems and bugs that remain in the web app, and introduce new datasets and locations. This may lead to the need to develop more efficient algorithms as data size becomes even larger.
