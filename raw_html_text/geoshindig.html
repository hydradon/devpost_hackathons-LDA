Inspiration

W każdym większym mieście odbywa się dużo wydarzeń o których nie mamy pojęcia. Zwykle każde wydarzenie ma swój odpowiednik w postaci wpisu na facebooku.

What it does

Nasza aplikacja wykorzystuje API Facebooka - OpenGraph do pobrania WSZYSTKICH wydarzeń związanych z danym miastem.

How I built it

Wykorzystaliśmy API OpenGraph, serwer, który pobiera wszystkie wydarzenia i zapisuje je do naszej bazy napisaliśmy w Pythonie, który pobiera wydarzenia z wszystkich dużych miast. Do stworzenia aplikacji która komunikuje się z serwerem wykorzystaliśmy hybrydowy framework IONIC. Aplikacja jest na system Android, Windows, iOS
