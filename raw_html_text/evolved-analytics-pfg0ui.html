Inspiration

• Bank's typically apply data analytics on internal data which revolves around Customer, Customer Account Balances and Customer Transactions. This provides limited insights and delay in decision making. • If the analytics application connects to external data such as Political, Social and Cultural events that trigger financial impact then the insights are wholesome. • Idea is to correlate the external events with bank's CashFlow patterns and predict the Impact.

What it does

Financial CashFlow Impact Analytics based on Political, Social and Cultural events.

How I built it

Started with Azure Cognitive Services - Bing Custom Search and Text Analytics to analyse and understand the output of various external search results. Then, we built a Data Model to hold Bank's Internal data and the various results from Azure Cognitive Text Analytics. With the help Azure ML Services, we created a Prediction Model to process and predict Bank's CashFlow. Leveraged Power BI to represent Time-Series based Charts and dashboards.

Accomplishments that I'm proud of

What I learned

Azure's artifical intelligence services: Bing Custom Search - Azure Cognitive Service to find the relevant information over the web. Text Analytics - Helps processes the content and provide Sentiment Analysis, Key Phrases and Entities. Azure ML Studio - helps create a AI workflow to process and predict CashFlow Impact based on Time Series Anomalies model. Power BI - Data Visualization tool helps create dashboards, charts and reports.

What's next for Evolved Analytics

• Improve prediction confidence level. • Based on CFO's financial parameters, proactively identify the external events and predict the impact. • Create a persona based search i.e., CEO, CFO, Branch Manager interested in and show the Impact Analysis.
