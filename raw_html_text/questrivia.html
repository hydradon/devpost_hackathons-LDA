Inspiration

We were inspired when we went to an escape room over the summer and thought it would be fun if we created an online version that offered the same experience.

What it does

It is a simulation of built-in quests and a "make your own" quest. It asks the user questions and lets them respond.

How we built it

We used Java and the software Eclipse

Challenges we ran into

Condensing out code, considering advanced skills

Accomplishments that we're proud of

Figure out how to condense our code using multiple methods in a different class

What we learned

useful to know how to code GUI/Applets

What's next for Questrivia

developing visuals to coordinate with the quests
