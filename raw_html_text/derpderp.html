Inspiration

We were bored, and like robots. Robots are fun to build, especially with limited resources and time.

What it does

Mostly it collects data about the surrounding environment, then discards it and drives into walls. bash.org

How we built it

Hot glue, caffeine, old RC truck chassis. Several hours later, it nearly moves.

Challenges we ran into

Mostly walls. Also, a trashcan. Several dev boards failed to work so we did not use them after all.

Accomplishments that we're proud of

Has not yet caught fire. The code is functional.

What we learned

Robots are hard.

What's next for Derpderp

Maybe fewer walls.
