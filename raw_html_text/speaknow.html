• speakNow

Inspiration

We wanted to make sure that everyone has the opportunity to learn! We wanted to help a handicapped person have the opportunity to learn, just like every other regular student.

What it does

We created a communication app that will obtain speech, and convert it into sign language for the user. It also outputs it in many languages, based on the languages the user can read.

How I built it

We launched a flutter app, and knowing little about the upcoming language, we began by watching tutorials and becoming familiar with the framework. We used dart for the ios and android apps, and we obtain speech and convert it to text using a web app created as well.

Challenges I ran into

We began by trying to incorporate SPRING, and Java into our tech stack. We wanted to use the real time speech-text using gcp and working around it with java, however we found a much simpler solution using a simple and efficient web app.

Accomplishments that I'm proud of

Connecting all of our experiences into making a big project that has many levels.

What I learned

We learned that working as a team with many skills is the most key component to any project. Apart from the new languages, this journey was amazing!

What's next for speakNow

We will next use words to convert sign language to text in real time (instead of letters)
