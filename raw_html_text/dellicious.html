Inspiration

Data science has become a huge field in all aspects of our society - we couldn't help but jump at the opportunity to apply our knowledge to a field so impactful on others.

What it does

Our project diagnoses patients based on medical information.

How we built it

Using the Fhir API provided by Tactio and Python's Requests library.

Challenges we ran into

We had trouble tackling the API and accessing workable data.

Accomplishments that we're proud of

Finally being able to work with the API can start seeing the results of us processing data

What we learned

The most important lesson we learned was that more data doesn't necessarily mean an easier project.

What's next for DELLicious

To improve our diagnosis accuracy in order to one day have a usable prototype.
