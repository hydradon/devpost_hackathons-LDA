Inspiration

We wanted to create a project that would allow us to learn about how Unity works, and a little bit about how the code in Unity works.

What it does

It's a game where you run from your nightmares/shoot them.

How we built it

Integrated premade Unity assets and scavenged premade code. We wrote the enemy AI and fixed over the shoulder camera movement, and player movement.

Challenges we ran into

Troubleshooting errors was challenging and writing in C# was challenging with no prior knowledge.

Accomplishments that we're proud of

It works!

What we learned

How to piece multiple things together in the Unity editor.

What's next for Purple Nightmare

Music! More sound effects! Levels! Guns! Stuff!
