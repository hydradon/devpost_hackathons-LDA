Inspiration

Since we are busy with school, we want to find a way to keep track of news regarding cryptocurrencies without having to read many news articles per week.

What it does

The application uses artificial intelligence to analyze news articles and gives the users an idea on the current cryptocurrency trends.

How we built it

We used the IBM Natural Language Understanding API to analyze the positivity/negativity of news articles URL scraped from Reddit.

Challenges we ran into

The IBM Natural Language Understanding API wasn't made for complex tasks such as analyzing news articles. The level of positivity of each article can be quite inaccurate, but when we average out the scores of multiple articles, it becomes reasonably accurate.

Accomplishments that we're proud of

We are proud of accomplishing a project about AI since it is a new kind technology for us. It is a technology that we were interested in for a while and we finally succeeded in implementing something with it.

What we learned

We learned how to implement multiple APIs at the same time. Also, we learned how to distribute the tasks efficiently since we only had 24 hours to submit this project.

What's next for CryptoTrends

The accuracy and algorithm efficiency of this project can be optimised further.
