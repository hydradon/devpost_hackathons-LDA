SeatMe

Getting Started: Fork this repository on GitHub: https://github.com/calistenson/whack_2016/blob/master/README.md

Purpose of App: This app allows you to find empty seats in a crowded movie theater for your whole party together!

Usage Note: Arduino hardware prototype required to simulate chair occupied/unoccupied status.

Created by and is maintained by Cali Stenson, Kim Asenbeck, Sam Chin, Kevin Guo, and Sung Park

Note: Link to webapp is live only when ngrok is connected.
