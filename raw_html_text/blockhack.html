Inspiration

Making a Murderer

What it does

It's a way to store evidence from crimes on the blockcain to prevent evidence tampering.

How I built it

On a user interface, user inputs text which is captured and hashed. This hashed data is pushed to the blockchain.

Challenges I ran into

Not being proficient in JavaScript, no full understanding of how solidity communicates with the blockchain.

Accomplishments that I'm proud of

We are able to successfully hash data through solidity node. Our solidity smart contract complies.

What I learned

As a team fairly new to blockchain technology, we gained a better understanding on smart contracts, solidity, html and interacting with the blockchain.

What's next for BlockHack

To further be able to build on this project, we need better JavaScript coding ability. We also need to get "our of the building", talk to problem owners and learn more about the law and order process.
