Inspiration

Anthem's challenge was appealing and our team decided to tackle it.

What it does

It allows company employees to evaluate their job satisfaction and other organization-related matters.

How we built it

We used Swift 4 and Firebase 3 to create a mobile app that allows employees the flexibility to answer at their convenience.

Challenges we ran into

Connecting the database to the code to update information in real time and implementing animations on swift 4. Furthermore, the GUI internal connections were tricky, however, we were able to make everything click before the deadline.

Accomplishments that we're proud of

Completing on deadline. Able to use swift and incorporate database. Learn as we went and make excellent progress.

What we learned

We learned to code using swift and connecting it to the cloud (firebase3)

What's next for Anthem Internal App Updated

Incorporating security, having real-time graphs, adding more incentives for using the app (Loyalty Program).
