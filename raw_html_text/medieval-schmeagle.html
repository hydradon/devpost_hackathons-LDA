• The Red Dragon in Action

Inspiration

This game was inspired by medieval times, and games like X-COM and Darkest Dungeon.

What it does

This generates a random map with units of various types spawned on it. Twp players take turns moving on a computer, trying to eliminate their opponent.

How we built it

We put this together with Unity, using things like Piskel and Github to share code and assets.

Challenges we ran into

None of us have extensive experience in Unity, especially on this type of project. We had some issues with the collaborate function as well. The biggest thing code wise was getting a map with units on it.

Accomplishments that we're proud of

The random map generator worked very well. On top of that, the issues we did have, we were able to work around. We are also proud of all of the sprites we used for this project.

What we learned

We learned more about how Unity works, and how to use Github to share all of our files. We also learned to work together as a team, and to alternate sleeping.

What's next for Medieval Skeveil

The future is uncertain for this project, as we might keep working on it or we might not. In any case, this was certainly an enjoyable experience.
