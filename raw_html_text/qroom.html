Inspiration

Blake's sister had a hearing impaired professor, and asking the professor questions was almost impossible because verbally communicating with the professor was difficult.

What it does

Receives SMS messages and displays them on screen, flashing red for each message sent in. Questions can be removed from the list after they have been answered. There is also an enable sound option for questions received.

How we built it

The frontend was built using HTML and CSS. The backend was created using node.js, javascript, and fonestorm API.

Challenges we ran into

Learning javascript

Accomplishments that we're proud of

Creating a fully functional website for the first time and incorporating an API

What we learned

Many javascript skills and how to design a website

What's next for QRoom

Possible feature implementation such as multi room usage
