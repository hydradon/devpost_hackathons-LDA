We developed our project for dotLearn, to create an app and a backend solution to map schools in Ghana. Our team created an android application where users can record internet speeds, contact information, and location of schools in Ghana, and uploads the information into a server. The app updates in real time for everyone that adds information, enabling volunteers to gather information quickly and efficiently. We also used an AWS instance for centralized data processing and analysis of the data recorded.
