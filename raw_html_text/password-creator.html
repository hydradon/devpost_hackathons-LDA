Inspiration

This started out when I was trying to think of a password for a new account. I soon tried the XKCD method, however, in that point in time it was hard to think of 6 different words. I then asked Alexa, but she didn’t help either. So I decided to build this skill for Alexa so that people in my same shoes could solve this much faster than me!
