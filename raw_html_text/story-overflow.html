Inspiration

Story overflow was inspired my the idea of multiple writers or writer enthusiasts to contribute to creating a short story. We believe that new and interesting story ideas would come out of the worlds collaboration.

What it does

Story overflow starts out with a prompt, which writer can use as the starting base for their story. Following that prompt, writers will each post and vote on the best post to continue the story. As writers gain more popularity the would be allowed to type more characters into their posts. Multiple stories can be created, viewed, and contributed to at any time.

How we built it

Using java.

Challenges we ran into

Not knowing enough about SQL/databases limited our options to local storage. Not enough time to create the user interface.

Accomplishments that we're proud of

Worked in a group project. Made amazing friends.

What we learned

We learned how to use Github to commit and push files to each other.

What's next for Story Overflow

Were going to implement a GUI and hopefully make a web application.
