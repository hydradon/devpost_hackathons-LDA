Inspiration

Using the bathroom can be a scary and dangerous experience for trans and gender-nonconforming people. Trans people are constantly faced with the threat of harassment, assault, and bathroom laws designed to prevent them from existing in public spaces.

What it does

EnbyPee gives users the five closest gender-neutral bathrooms within the area around them.

Using volunteer-compiled lists and scouting around campus, we put together a database of gender-neutral bathrooms tagged with details like wheelchair accessibility and whether they were single-person. Our app takes this database and uses the Google Maps apis to find the closest bathrooms by walking distance to the user.

How we built it

Used back-end server using python, flask, and nginx to process from the html5 front end and proxy requests to Google maps API.

Challenges we ran into

Figuring out front-end javascript details.

Accomplishments that we're proud of

Creating a workable product and coming up with a decent pitch

What's next for EnbyPee

Gathering more data, adding filtering options and improving UI.
