Inspiration

Course registration season for the winter semester.

What it does

Makes things easier when the professors' ratemyprofessor.com ratings and the number of ratings show up by the professor's name on the course registration website instead of going to ratemyprofessor.com every single time to check a certain rating

How I built it

Its a chrome extension so javascript and chrome API

Challenges I ran into

Sleep deprivation at the hackathon so couldn't add features to it

Accomplishments that I'm proud of

My first-ever hackathon project that I finished it in a single day. I started it Saturday afternoon and finished Sunday morning.

What I learned

That I could actually use my skills to solve problems and ease things for ourselves.

What's next for Rate My Boilermakers

I am going to add a lot of features to it. For instance, pop up with detailed rating from ratemyprofessor.com when hovered over the rating on the course webpage, etc.
