Inspiration

The Hackathon's deadline.

What it does

Allows the player to play as 2016 Presidential Candidate Bernie Sanders to crush world leaders and protect the all-important nuclear launch codes.

How we built it

We used the Python library RenPy, a visual novel library.

Challenges we ran into

GitHub posed a significant challenge for us, as well as the limited functionality of RenPy.

Accomplishments that we're proud of

The game is finished :)

What we learned

How to use RenPy and to be patient when coding.

What's next for Go Go Bernie Sanders!

Multi-character support and multiplayer.
