Provides an Offline POI based navigation concept, giving the user all the important information required to "survive" in a big city!

How I built it

Accomplishments that I'm proud of

Managed to extract lots of useful information using OpenStreetMap and various APIs for it. I have a deeper respect for the platform from this experience!
