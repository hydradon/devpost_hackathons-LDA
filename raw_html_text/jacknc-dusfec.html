Inspiration

After reflecting upon our lack of physical activity since arriving at university, we wanted to create a platform that would encourage others to take the steps to becoming active, or "swoll."

What it does

It is an interactive website that allows the user to observe the physical changes when one engages in exercise or "getting jacked."

How we built it

We used Visual Studio Code to code all of the website and published it online using GitHub.

Challenges we ran into

We started with little to no experience in CSS, javascript, and html but after EXTENSIVE research and trial and error, we were able to accomplish our goal of expanding our coding knowledge as well as creating a product we are proud to submit for this hackathon.

Accomplishments that we're proud of

Having the button modify the height and width variable of the image continuously was very difficult but extremely rewarding once we were finally able to code it.

What we learned

We learned what is required in basic web development and progressed in our coding capabilities.
