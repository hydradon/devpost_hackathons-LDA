Inspiration

I've fallen in love with data science and wanted to complete a project involving a lot of data!

What it does

Facial recognition software to detect during an interview or interrogation if someone is lying. Check out more on how the software works on my GitHub! ( https://github.com/OliverCollins/Face2Face )

How I built it

I strictly used Microsoft Azure's Face API and @ilkkao's capture-video-frame micro-library

Challenges I ran into

Troubles implementing Microsoft Azure Face API, loss in ideas, and running out of time!

Accomplishments that I'm proud of

Implementing my first API and figuring out how to harness all my coding skills into one project.

What I learned

How to use and implement an API

What's next for Face2Face

Not quite sure yet, really open to anything!
