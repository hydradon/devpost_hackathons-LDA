Inspiration

It hurts us so much to see people with a weak gif game. Then we asked ourselves, what is a 'good' gif?

What it does

Enables user to save any gif at any location, organize it, and use it.

How I built it

Used gfycat api to upload and create new gif that the user finds. Used gfycat api to display a gif that the user saved on gfycatnip User gfycat api to post a gif

Challenges I ran into

Authentication - spent way too much time

Accomplishments that I'm proud of

Being able to save gif from anywhere anytime.
