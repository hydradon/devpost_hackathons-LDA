WHAT IS THAT?

It is an open source wallet inside Discord. It is awesome!

Value

Discord is one of the most popular messengers. What is more important - gamers use it. With access to crypto assets you can combine games crypto collectibles and trade it.

This wallet can be customizable and provide direct access to Dapps for different Discord communities.

What we did

  1. We developed a core for Discord and we provide a usecase for this bot.
  2. We integrate Binance Dex with order book and BNB token
  3. We add Matic network too
  4. Also you can send ETH by ENS with Infura
  5. Chainlink API for exchange rates
