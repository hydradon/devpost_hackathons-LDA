Proto Builds - Flight Stick

The website: We created a website to post our prototyping guides and projects available for others to build. We write why we chose certain software and explain things step by step with pictures and videos.

The project: For this project, we created a makeshift flight stick controller to use to control a space ship and shoot bullets. We also created the game to have the controller playable on!

Hardware: Arduino Leonardo, joystick, push button

Software: Godot, Arduino

Check out our website, protobuilds.tech (under construction) for more information and how to build it yourself! Our domain name is protobuilds.tech but we didn't have time to implement it as a website.
