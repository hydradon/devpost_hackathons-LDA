Inspiration

Saw the idea from an online tutorial and we reedited it and add interesting effects.

What it does

Use QR code to direct the little copter. When it passes the home base, there would be firework to welcome it!

How we built it

Build the model with unity and capture the QR code with the API vuforia

Challenges we ran into

Why the collision cannot be detected? What's the correct code to trigger effects? How to operate unity? It's full of challenges...

Accomplishments that we're proud of

We used some math method to make the copter slightly leaning to the direction it's turning to. We made it a smoothly running IOS app. We can actually control the copter and trigger the effect as we expect.

What we learned

What's next for ARCopter

More operations, like firing bullets may be added to the copter. Other missions could also be added, such as getting energy block or others.
