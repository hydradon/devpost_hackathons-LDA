Inspiration

We wanted to build rapid onepage theme for WordPress and there is no such framework or builder exist on the market. So we decided to build a page builder for ourself that is easy to use and developer friendly.

What it does

Onepager helps developer rapidly develop onepage WordPress themes that are ready to sell in the marketplace. The API is very developer friendly. You can build, manage and deoply onepage WordPress theme in very short time.

How we built it

We used reactjs for the builder part and small use of jQuery for 3rd party libraries.

Challenges we ran into

This is the first page builder built with reactjs and there were lack of libraries our there when we built it. We took many jQuery library and re-build it from reactjs. Most challenging job was real time changing and rendering.

Accomplishments that we're proud of

First page builder for WordPress with reactjs and we featured on ProductHunt as #2 product of the day https://www.producthunt.com/posts/onepager also lots of 5 star review on Wp.org repo https://wordpress.org/plugins/tx-onepager/ with 4k+ active install.

What we learned

A lot actually. We learned in-depth of react and build another drag & drop page builder for Joomla called Quix.

What's next for onepager

Our plan is to turn onepager into SaaS model and help people build high converting landing pages with minimum effort.
