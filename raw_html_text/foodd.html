Inspiration

We wanted to play a small part in reducing food wastage through a medium that plays on what everyone is familiar with.

What it does

(The plan) Create a holistic food manager app able to take inventory and send reminders for approaching expiration dates.

How we built it

Android Studio patched with various libraries like webkit to connect with Microsoft Azure Computer Vision API, simple.JSON, etc.

Challenges we ran into

Lack of team experience with Android Studio

Accomplishments that we're proud of

The app can analyze an image, write relevant information to a JSON file on the Android device, and read from that file.

What we learned

Make sure your team members have their build tools beforehand!

What's next for foodd

Perhaps future development on our own.
