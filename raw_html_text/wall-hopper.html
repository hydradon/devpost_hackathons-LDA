Inspiration

Flappy Bird inspired us to create a similar platform style game.

What it does

It is a game where the player has to jump through pillars of varying lengths to increase their score.

How I built it

We used textpad, JFrame, and Java.

Challenges I ran into

Implementing the runnable interface to get the objects to run smoothly over the course of the game

Accomplishments that I'm proud of

The game runs smoothy and looks aesthetically-appealing to the user.

What I learned

We gained experience in JFrame and became comfortable with game design.

What's next for Wall Hopper

We hope to add more levels to the game in the future.
