Inspiration

We are travelers, we had to search around multiple websites and other platforms to get a tour guide.

What it does

This app provides the travelers the facility to search for the guides, agents, hotels and restaurants in specified region through android and web platforms.

How we built it

We developed the android prototype using figma and web prototype using html, css, javascript, php and xampp as tool. We used TPL Maps API to integrate into the prototypes.
