Inspiration

We needed a website, and we're all passionate about quality work.

What it does

Gives the user(current and potential club members) information about the club overview, officers, and events.

How I built it

We decided to build a website with GitHub pages because we could all contribute via GitHub and as a bonus, using a version control system would help us immensely if/when we faced some issues. We found a basic html, css, javascript theme and built the website from that theme to suit our club needs.

Challenges I ran into

Learning Git, merging and utilizing pull requests. Mapping custom domain names to the GitHub pages website.

Accomplishments that I'm proud of

The website looks amazing, it is easy to use and has all the pertinent information about the club and our activities.

What We learned

We learned about web development and Git/GitHub. As a team we worked to solve some issues with domain names and DNS issues.

What's next for Rowdy Gaming

Map a custom domain name to the GitHub pages and continue to utilize the website for our club's use!
