Inspiration is Requirement i got to do which included joining multiple excels. where i got input all the excels name and every time i need to use join wizard for every 2 excels. Thought of making this as utility where my friends who has same requirement would use the same to save time.

What it does is It joins multiple excels laying in single folder based on the key field provided.

Advantages: 1) No need to hardcode or pass workbook names. 2) Can get required columns on join in output excel. ( Provision to customize output columns) 3) Can handle join of any feasible number of Excels on the go.
