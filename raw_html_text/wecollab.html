Inspiration

The idea was born from the concept of letting artists collaborate with other artists based on common interests

What it does

It lets users upload their music, listen to other artists' music and build playlists. It also lets users collaborate with other users based on what is trending and recommendations.

How we built it

The demo was purely built using Bootstrap embedding the SoundCloud widget.

Challenges we ran into

The biggest challenges were how to not make this another Spotify or SoundCloud because all these apps play music and lets users create playlists.

Accomplishments that we're proud of

Conceptually, we were still able to differentiate this based on the collaboration aspect where the app would recommend users to collaborate with based on common interests.

What we learned

This was a good team building project where we learned each others skills ranging from UI development to back end development and learning new skills like: consuming APIs, using GitHub, Heroku, Bootstrap, Ruby/Rails, etc.

What's next for WeCollab

Take it from a demo to reality!
