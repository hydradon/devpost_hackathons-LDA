Inspiration

Use a cool API we aren't familiar with

What it does

Use Google's Nearby API to find nearby devices and attempt to connect with them. The goal was to establish a chat based on location for quick messaging groups

How we built it

We built the application in Android and it also used Firebase, Google's Nearby API over course, and the Facebook SDK

Challenges we ran into

Troubleshooting was a big issue. There were small problems we were running into and spent a decent amount of time trying to fix them. It was also difficult setting up the API key and SHA1 fingerprint

Accomplishments that we're proud of

It compiles

What we learned

We learned about the Nearby API, Firebase, and the Facebook SDK
