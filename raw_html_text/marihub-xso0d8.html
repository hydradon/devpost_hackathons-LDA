• Marianopolis College Crest

Inspiration

Too unorganized for peer tutoring. Too broke for a Chegg account.

What it does

Allows students to see which other students are in the room based on their specific qualifications and interests.

How we built it

Using HTML, CSS, javascript, python and Flask.

Challenges we ran into

2/3 of the team had no experience with programming coming into the project and we were all learning as we went along. Creating a server that allows cross communication was particularly hard.

Accomplishments that we're proud of

Managed to create a presentable website from scratch with no prior knowledge of any programming language. The 1/3 that knew a bit of programming managed to create his first database.

What we learned

We learned a great deal about the thought process of a developer, as well as basic html, css, and javascript.

What's next for MariHub

Going global. Having a proper login system, internal messaging, QR scanning for any room in the world to be able to turn into an academic hub.
