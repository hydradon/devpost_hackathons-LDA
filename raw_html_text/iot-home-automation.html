Inspiration

To make the control of electronic devices much easier and more convenient.

What it does

Lets you control your electronic devices at home from the internet

How we built it

We built it using an Arduino and a ESP8266 WiFi Module. The electronic devices were connected to the Wifi Module through relays, which would allow the arduino to send the high or low signal to whether turn the desired devices on or off. The project is still not complete.

Challenges we ran into

The device that was supposed to provide AC Voltage to the devices (LEDs in this case) stopped working and therefore we couldn't get the circuit to power up and test it.

Accomplishments that we're proud of

The circuit is designed right and should work theoretically as predicted, however, technical issues we ran into and a device failure prevented us from finishing the project in the given time.

What we learned

We learned to code in C language for Arduino. We learned how to use Wifi Module and send signals through it to other devices.

What's next for IoT home automation

Provide support for more electronic devices that can be controlled at once. Also, develop a smartphone app instead of web platform to control between the devices.
