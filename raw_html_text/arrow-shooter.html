Inspiration

I used to play a lot of archery games back when flash was popular. Now my teammates and I have the opportunity to build upon them with our very own game

What it does

A normal archery target practice game but as the levels progress, the more room for error! The amount of archers to control also increases leading to all kinds of situations!

How we built it

We wrote it all in Haxe and used the HaxeFlixel library to aid us. It compiles to HTML5, Flash, etc.

Challenges we ran into

We had trouble with coding the physics of the arrows as they can collide which drastically increases the complexity

Accomplishments that we're proud of

What we learned

WE learned a new approach to programming games.

What's next for Arrow Shooter

We plan to add more levels and flesh it out more
