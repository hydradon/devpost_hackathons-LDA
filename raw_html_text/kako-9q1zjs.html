Inspiration

The internet of things is one of the biggest attack surfaces in modern consumer electronics, and it has been exploited by a notoriously destructive program: Mirai. The goal of this is to create a program that will scan, clean, and secure an network against this program and strains associated with it.

What it does

The program scans the networks and secures vulnerable devices.

How we built it

The system is a Python program.

Challenges we ran into

There was a lot of research that went into actually defining the nature and presence of the malware, and figuring out exactly what mechanisms would be needed to clear it.

Accomplishments that we're proud of

We used Edisons to model small consumer IoT devices.

What's next for kako

We plan on bolstering detection mechanisms and the ability to take out various other potentially more insidious strains.
