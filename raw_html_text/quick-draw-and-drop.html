Inspiration

Our group loves card games so we wanted to bring some of them to the computer.

What it does

It allows for you to shuffle and draw and discard from a 52 card deck. It can differentiate between two player's hands.

How we built it

With Unity 2D using C#

Challenges we ran into

Using Unity for a card game instead of a more simple language like JS.

Accomplishments that we're proud of

It doesn't crash or have any bugs.

What we learned

That Unity is not meant for card games.

What's next for Quick Draw and Drop
