ShowUp is an android app designed to be widely used on college campuses. It uses a database to quickly and simply check to see if any other clubs have scheduled a meeting at the same time as you. It allows kids to actually be able to "Show Up" to a club rather than having multiple club meetings at the same time. The interface is kinda rough but the database is smooth and dynamic.
