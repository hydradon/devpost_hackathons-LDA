Inspiration

One time we found a phone and had to dig a little bit through personal info to find that person and return their phone to them. We did not like the idea of that, so that's why we built this app.

What it does

Allows a user to generate a key to write on belongings so that they can contact the owner while both sides remain anonymous.

How we built it

We used Firebase as our backend and javascript for frontend.

Challenges we ran into

Not knowing javascript.

Accomplishments that we're proud of

Making a fun idea.

What we learned

More javascript

What's next for KeyPin

If people like it, we'll put more work in it.
