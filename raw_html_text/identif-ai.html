Inspiration

We believe technology has the ability to bring about a new level of security but at the same time we want it to be transparent to the public. This is a proof of concept of allowing everyday people to contribute to the cause and Identif.ai wanted people.

What it does

We want to help people identify lost children, criminals, and other 'wanted' people.

How we built it

We built a backend API powered by Microsoft's Project Oxford

Challenges we ran into

A lot of encoding and transmission problems from Android to backend.

Accomplishments that we are proud of

It works even if the idea is super grand.

What we learned

Encoding and decoding binary files and understanding HTTP request mimetypes

What's next for Identif.ai

Make the connectivity a lot cleaner and do thorough error checking
