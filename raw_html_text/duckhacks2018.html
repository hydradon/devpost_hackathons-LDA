SMART devices can help you control them remotely, what they can't do is, remind you when you forget to switch them off.

Best feature of the app is then you don't have to use the app. Just add devices to your app and quit the app.

Based on proximity and device current status app will send you notification if required.

Code repo contains only POC which proves full feasibility of idea detail android code can be found on link in readme of repo.
