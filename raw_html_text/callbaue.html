Inspiration

An web app that works in both Android and iOS devices efficiently. The idea is that Stony Brook Students can get very personal Counseling and Psychological Services (CAPS) service.

What it does

The app has a function to talk to a psychiatrist more personally and enjoy more privacy using the online chatting system. This will allow more students to confidently use the service. There is a calling function (via cellular connectivity). The map function allows the customer to find the CAPS facility in GoogleMaps. The app has background music and a relaxing theme (with randomly generated quotes) to help the student relax immediately and create an amiable atmosphere to come forward and enjoy the service .

How we built it

We used HTML, CSS, Bootstrap, jQuery and JavaScript.

Challenges we ran into

Making the display fluid and run across all devices. Using the soundcloud API.
