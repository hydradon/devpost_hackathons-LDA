NutraSnap

A Budget Conscious Supermarket Analytics Application

Our program uses the Google Vision API to pull product ID's from grocery store receipts, using the raw information that google provides we created a process to find and pull the ID's from the google response. After getting the ID we use the Wallgreens API to gather the UPD about that product which can be inputted to a Nutritionix API. After getting data from these three APIs they store it in a user database where the user can recall the program at any time. Users at registration can input allergens, weight, height, and gender. With that info we can use the wolfram API to assign important information like allergen alerts or recommend calorie intake.

Loose pounds, not money.
