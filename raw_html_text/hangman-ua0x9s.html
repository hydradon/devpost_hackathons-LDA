Inspiration

Inclination for interactive front-end coding, wanted to make something fun

What it does

takes your submissions of letters and answers whether the letter is in the randomly generated word. The program will notify the user if the submission is not a letter, an already entered value or a multiple letters.

How I built it

Basic framework was done in HTML & CSS, then I used Javascript to make the program interactive and dynamic

Challenges I ran into

Lack of programming experience, Javascript syntax and methods, CSS positioning, need to sleep.

Accomplishments that I'm proud of

Building and interactive site, learning more Javascript.

What I learned

Everything takes longer than you think it will. Javascript. It's about the process and not the outcome.

What's next for Hangman

-Categorized words -words (answers) can be hints in a treasure hunt -competition with friends -High scores, scores, standings -scores given based on difficulty of word -use as a way to send messages then "decrypt" them
