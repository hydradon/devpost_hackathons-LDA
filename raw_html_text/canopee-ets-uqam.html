Inspiration

Nous ne savions pas ce qu'était la canopée avant de voir le jeu de données disponible sur données Québec. Étant curieux, nous avons décidé d'explorer ces données pour en apprendre davantage sur le sujet.

What it does

Notre app permet de voir la progression et la régression de la canopée en comparant les données d'années différentes pour un même territoire.

How we built it

Nous avons utilisé des librairies Python pour manipuler les données et générer des statistiques à partir de celles-ci. Ensuite, nous avons utilisé React pour présenter ces données dans une application web.

Challenges we ran into

Les données de 2007 et 2015 pour la ville de Montréal n'étaient pas exactement sous le même format. En effet, les données de 2007 étaient séparées par quartier. Il a fallu remanier la structure pour pouvoir établir une comparaison entre 2015 et 2007.

Accomplishments that we're proud of

Nous sommes contents d'avoir réussi à manipuler les données ouvertes pour en générer une représentation graphique. Nous croyons que ceci peut faire de notre un outil pour les gens qui étudient la canopée.

What we learned

Nous avons appris que les données ouvertes ne sont pas toujours faciles à manipuler. Comme mentionné précedemment, les données d'une même ville pour deux années différentes étaient présentées dans des formats différents. Il a fallu faire preuve de débrouillardise pour faire face à ce problème.

What's next for Canopée ETS UQAM

La réalisation de cette application nous a permis de se familiariser avec les données ouvertes. Si elle n'a pas d'application réelle, elle aura au moins servi à nous rendre compte de l'importance et de la pertinence de ces données.
