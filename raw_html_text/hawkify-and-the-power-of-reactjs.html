Inspiration

I wanted to showcase the power of a technology that I stumbled upon called ReactJs. Use it to solve a simple issue in an efficient way

What it does

We are able to clone Spotify and while maintaining a great amount of performance efficiency and responsiveness.

How I built it

Built using the ReactJs along with node.js to create the initial framework. Then acquire the Spotify API and manipulating it to give us access to numerous capabilities. Then to make it all come together I use bootstrap and CSS to make it a little more appealing
