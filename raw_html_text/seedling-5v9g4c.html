Inspiration

There are a lot of people struggling with mental health issues today and several that we know personally. We have seen them struggle to accomplish even the smallest daily goals, and we wanted to develop something that would aid them.

What it does

Sets up a preset list of Daily Tasks that are curated based off of the 5 areas of wellness. 2 tasks are given from each area: Social, Emotional, Occupational, Intellectual, and Physical. This process has been proven scientifically to be one of the best methods in helping individuals recover from depression/anxiety.

How we built it

We used angularjs for a web frontend to showcase the app. We built the cross-platform (iOS, Android, Windows) app using Ionic Framework for Cordova.

Challenges we ran into

The build system required configuration separately for the Linux and the OSX systems that we used to program it. In addition, we had to learn the design pattern of angular.js and its MVVM model from scratch.
