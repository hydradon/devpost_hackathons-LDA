The Team

David, Jake, Kyle, and Victor

Inspiration

We wanted to build on the statistics Steam already gives its users by adding more detail to the data available.

What it does

Takes a Steam ID and uses the Steam Web API to get the user's data and display unique statistics.

How we built it

A JS file implements the API to access Steam data, which we modify and summarize for the user.

Challenges we ran into

Accomplishments that we're proud of
