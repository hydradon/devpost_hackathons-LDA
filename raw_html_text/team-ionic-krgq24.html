Inspiration

I wanted to create a small set of mobile apps that would help HMIS systems and also assist with Emergency Relief Shelters. I believe that we don't need to reinvent the wheel when it comes to HMIS systems, but instead complement them with mobile enablement.

What it does

These apps provide several different core features:

  1. Tracking and reporting on homeless person locations.
  2. Sending notifications on homeless person location updates.
  3. Creating and managing Emergency Relief Shelters.
  4. Searching and viewing Homeless person location data.

How I built it

I used Ionic 2 and Node. Ionic 2 was used to build the mobile apps, and Node.JS + MongoDB to build the backend API.

Challenges I ran into

The google maps heatmap API.

Accomplishments that I'm proud of

Getting the heatmaps api to overlay and interface in real-time with the Google Maps widget.

What I learned

How to hack geolocation data.
