Inspiration

Finding the right charity to donate can be really hard, and we decided to make this project as a way of making things easier

What it does

It has every user build their own preferences of charities they want to donate to, then it makes it easier for the user to find new local charities that they would donate to. It also can scan the items that the user would like to donate and show charities that are accepting of the items

How we built it

We made the backend using express to handle requests and mongodb for the database. We used react-native for the front end of the app

Challenges we ran into

it was hard to properly connect the front end and the backend since we didn't have much time left after the front end and back end both finished work

Accomplishments that we're proud of

We have an extensive database of the charities and can quickly filter them out based on the user requirements.

What we learned

How to work with APIs that don't want to work.

Built With
