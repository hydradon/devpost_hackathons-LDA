• Watch Out: tracking incidents and stores near you.

Inspiration

During emergencies like snow storms or hurricanes, I would go to Mcdonalds to only find out that they are closed. Another step forward, what if near by power is out or a tree is fallen.

What it does

User's can report open or closed near by so I know where I can find food to eat. User's also can make incidents if near by activates like fallen trees, we should be aware of.

How we built it

We built it using HTML, CSS, and JavaScript. We used GitHub for team collaboration.

Challenges we ran into

Time is the most challenging process we ran into. Another challenge is vetting the system to make sure false incidents are not reported.

Accomplishments that we're proud of

We are proud of time restrictions, that we pushed out a working website.

What we learned

We learned a lot in this journey, especially disaster preparedness and project management.

What's next for WatchOut

Next steps it to make everything fully automated process.
