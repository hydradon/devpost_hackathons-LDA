Inspiration We wanted to find something that would help us as high school students to find a perfect fit for college based on our GPA and SAT score.

What it does

The program shows students what colleges they could and should look into applying based on academic achievements.

How we built it

We used html5, CSS, and javascript

Challenges we ran into

finding an idea, comparing user input to the actual real-time data from colleges

Accomplishments that we're proud of

We were able to finish in time, and add new features.

What we learned

How to implement animations, and user inputs

What's next for Texas College Finder

Expand our reach to colleges outside of Texas, using a system based on location
