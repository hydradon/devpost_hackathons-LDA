Inspiration

Nowadays tourists need a new way to visit cities. We propose a mobile system without physicals maps to plan specific routes around the towns.

What it does

The main purpose is to purpose paths to visit special monuments, as churches, historic buildings, parks and some other interesting places. The system works fine trough an easy-to-use Android application. It also disposes a webpage for personal computer users.

The app's performance works as follows: thanks to Google Maps user can locate himself on the globe. With a range selector one can select in which area he or she wants to visit specific monuments; afterwards, the monuments or places are selected click-by-click and pressing a button the app draws the shorter route that contains each item the user selected before.
