Plagiarism Avoidance Software and Graph Forgery

All You Can Cheat is a web-based project comprising two tools. The first allows the user to upload plagiarised text and performs a re-phrasing to preserve the meaning whilst making the text sufficiently different from the original that it does not flag up on plagiarism detection tools. The second tool generates natural-looking, fake empirical data based on a user's mathematical formula and chosen noise level. The tool also presents the user with a scientific plot of the data.

The project backend was written in Python 3.6 and the frontend was written with Bootstrap and JavaScript. We used Python NLTK, Stanford Dependency Parser, WordNet, Google Cloud API amongst others.
