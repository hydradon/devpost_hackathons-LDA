• Calm me down

Inspiration

Someone I know has difficulties managing their emotions and anger. I got some tips from a professional on how to help them calm down and thought this person might use Alexa as a sounding board to remind them how to self-regulate.

What it does

When the skill is opened, Alexa says (at random) one of the tips.

How I built it

I used Storyline linked to a Google Sheet containing the tips. This means I can easily edit and add to the tips over time to improve the skill and keep it interesting for users.

Challenges I ran into

I hadn't used Storyline before, but it wasn't too hard.

Accomplishments that I'm proud of

Learning to use Storyline, including making an API call to Sheets.

What I learned

As above.

What's next for Calm me down

I will add more tips. I may also look at logging when the skill is used so that the user can see if there are patterns to their feelings.
