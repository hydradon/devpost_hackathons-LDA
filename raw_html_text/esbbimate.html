Inspiration

Company SBB Cargo is facing a problem related to providing the accurate delivery time data of the cargo transported once it has been unloaded off the rail and being delivered to the end customer.

What it does

It helps SBB to accurately estimate the delivery time of the cargo and improve their logistics process. That data could be shared with an end customer and improve his experience with the service.

How I built it

We created an integrated solution using Graphcool (server and database setup), provided SBB .csv data, Angular 2, Javascript, HTML, bootstrap, CSS, Phython to process the data, native mobile application environment with a Bluetooth Smart technology (Estimote Beacon)

Challenges I ran into

Parsing the data from SBB .csv; integrating the Beacon and testing it;

Accomplishments that I'm proud of

We have provided an improvement to the existing solution, upon which future innovations could be done.

What I learned

We learned how to use Graphcool, how to parse big .csv files with Python

What's next for eSBBimate

Potential collaboration with SBB and implementing our solution within an integral platform that would gather full delivery tracking data and engage customers into the process more actively. Also we would like to employ various methods to create overall better customer experience.
