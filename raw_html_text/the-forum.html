Inspiration

People being harassed for their own political opinions

What it does

Allows people to anonymously share their opinions regarding various current political topics

How we built it

We used sql and php for the backend, while using html/css/javascript for the front end, as well as using bootstrap to make the overall aesthetic more appealing

Challenges we ran into

Creating a database that we could upload information into and get information out of

Accomplishments that we're proud of

Creating a sql database where we could essentially build a 'social-media'-like platform

What we learned

How to create and effectively use databases for creating apps

What's next for The Forum

Allow users to make comments on each others post, algorithm to rate posts higher than others and display them higher, minor bug fixes
