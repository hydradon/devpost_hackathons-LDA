Inspiration

Dimulai dari saat kesusahan mencari bubur yang buka di pagi hari untuk sarapan anak yang sakit, atau mencari yang menjual daging di malam hari ketika pagi hari ada event, saat mencari tambal ban yang dekat, dan penasaran mencari tahu yang apa saja yang dijual di pasar

What it does

  • menyediakan informasi tentang apa saja yang dijual di suatu area, misal Pasar, Pertokoan, Tempat keramaian
  • menyediakan informasi tentang barang / jasa yang di cari, yang terdekat, beserta alamat, nomor yang bisa di kontak, jadwal buka
  • memberikan kemudahan untuk penjual / penyedia jasa ditemukan / dikontak
  • memberikan kemudahan untuk penjual / penyedia jasa membuat daftar penjualan / jasa untuk di tawarkan
  • dapat diakses menggunakan halaman web ataupun aplikasi android yang umum digunakan
  • dapat memberikan data umum untuk dapat digunakan aplikasi / penggunaan yang lain

How we built it

  • membangun database design dengan mysql
  • membangun back end sistem proxy dengan rest api untuk menyediakan data dan menerima input dengan json sehingga dapat di gunakan untuk aplikasi web dan aplikasi android
  • membangun front end aplikasi web dengan codeigniter dan aplikasi android dengan android studio

Challenges we ran into

publish aplikasi lewat sosial media dan referal media dengan sistem reward untuk mencapai jumlah pengguna / toko / warung / penyedia jasa agar data yang disediakan lebih valid
