Inspiration

Missing classes due to unavoidable circumstances

What it does

Allows users to upload the photos of notes anonymously so that others can view them.

How I built it

We divided the team into half. One was working on front end, while the other back end. The duties were further divided on the basis of skill set. We used Angular.js, node.js and firebase as the main tool

Challenges I ran into

  • Selecting the appropriate database
    • Lost a member who knew most of the technologies used in the project, learning new technologies in general

Accomplishments that I'm proud of

  • Learning the whole process of web development
  • Learning the new technologies from scratch in few hours

What I learned

  • Web dev takes a lot of time
  • A lot of documentation is sparse and/or non-existent

What's next for Cognition

  • Better UI/UX
  • More powerful back end
