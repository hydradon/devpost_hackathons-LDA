Inspiration

People always hunt for the best sale products. Sometimes it's too intimidating to search for the sale products on the webshop, so we created a campaign site to make it easier and less complicated.

What it does

Salendo searches only the sale products from Zalando API.

How I built it

It was built on bootstrap with js/jquery.

Challenges I ran into

We wanted to be able to view products sorted by the sale percentage but the API doesn't allow it. We would have to iterate through the +111k products and then sort, which we did not do.

Accomplishments that I'm proud of

Being able to fetch data from the API and view it on the website, which I have never done before.

What I learned

To fetch data from the API.

What's next for Salendo

More sorting options for women, men, shoes, shirts and so on. It would also be nice to be able to sort products by the sale percentage.
