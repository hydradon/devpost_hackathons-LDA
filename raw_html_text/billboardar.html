Inspiration

Remove the gap from a billboard ad to purchase. Create a hassle-free fun AR shopping experience.

What it does

Turns any billboard advertisement into a instant purchase opportunity.

How I built it

AR: Unity Game Engine + Vuforia AR SDK AR Map with the route to the shop: ESRI JS SDK, embedded into Unity WebBrowser

Product related info: Clover API (Merchants + Inventory)

AR Advertisement campaign based:

  • Per-View and Per-Click charging model, utilizing SAP Revenue Cloud

Success Page: Github Pages + HTML + CSSS

Challenges I ran into

Condensing the buying experience into a few small panels in the AR world.

Accomplishments that I'm proud of

Interactive aspect relating to changing the product selection in the AR world.

What's next for BillboardAR

Tighter integration into the online shopping experience
