What it does

ZenCash is a web app made in JS that uses machine learning to recognize a users face as verification before a cash transfer!

How we built it

ZenCash is built using Node.js, React.js, MySQL, and OpenCV.js (Machine Learning Javascript Library)

Challenges we ran into

We originally wanted to develop a data visualization web app using Omni - Sci's data analytics software, but after a considerable time sink into creation, we decided ZenCash would suite our collective skill sets better, and pivoted our project!

Accomplishments that we're proud of

We've finally built something using machine learning! It just feels so cool to work with such cutting edge technology. That, and sleeping literally in any position. Really useful skill.

What we learned

By far, our biggest accomplishment was what we learned! Isabelle, our front - end engineer, picked up some awesome React skills. Antonio and Ricardo discovered the painful yet ultimately glorious path to machine learning and facial recognition. Carmina fell in love with React and the magic of Git. Oscar learned the joys of front end design and implementing our vision into something our users and/or investors can love.

What's next for ZenCash

Mobile! We want the capability to take a photo of a person with our phone and instantly link to their account and send cash securely.
