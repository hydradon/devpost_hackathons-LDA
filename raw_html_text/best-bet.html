Inspiration

Educating and bringing greater accessibility to the financial markets.

What it does

Best Bet incorporates data from Blackrock and other sources and predicts the best use of your money based on past trends.

How we built it

  1. Front-end - React-native with Expo.
  2. Back-end - Predictive modeling via recurrent neural networks and linear programming.

Challenges we ran into

Finding the best model to fit the data and predict accurately.

Accomplishments that we're proud of

Building a robust, high-performance, cross-platform, distributed machine learning platform using resilient data sets.

What we learned

  1. How to build cross-platform applications using React-native.
  2. How to apply machine learning to time-series cross-section data for predictive modeling.

What's next for Best Bet

Expand the asset classes to conservative (bonds) and high risk/high reward investments such as cryptocurrencies.
