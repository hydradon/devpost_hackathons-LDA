Inspiration

I wanted to build something that I could use, and something that my friends could use. There's a couple nice sites out there that let you work on practice interview problems, but none that let you do that and let other people upload problems that others can find. My project allows people to upload their submissions and tests, and lets other users give it a shot!

How I built it

This project is built with two different parts. The web app frontend is built with Laravel, and the backend build server that builds the Java code is created with NodeJS. These two series interact with each other to seamlessly compile and run code, and return feedback to the user.
