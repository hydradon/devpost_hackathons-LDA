Insta Caption

Authors: Keyan Fayaz, Francesco Virga, Raasil Khan

Insta Caption is a caption-generating tool that analyzes an image and returns relavent captions to post along with your photos. Our program uses Microsoft Azure's API in order to generate associated tags and keywords. Our application then uses those tags and keywords to generate captions it thinks you would like.

Everyone knows that thinking of captions are hard. Insta Caption makes it easy. Categories include: motivational quotes, song lyrics, puns, "selfies" and generic quotes.
