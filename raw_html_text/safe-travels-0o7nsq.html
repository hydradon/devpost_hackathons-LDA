Safe Travels

Storing a file specific hash on the blockchain that allows for users to verify whether or not they have downloaded the correct file. This also creates a way for users to prove that a file has not been modified since the time of inclusion. (deadline dates)

For file uploaders:(Small ETH requirement)

  1. Upload the file you want converted to be hashed
  2. Input a name and/or version number for the file
  3. Accept the usage of ETH through MetaMask
  4. Save the Transaction hash for your users to check the hash on Safe-Travels

For validity checkers:(No ETH required)

  1. Upload the downloaded file you want to be checked
  2. If you have the Transaction hash also input this to save time
  3. Returns : "Match" for equivalent hashes "No Match" for non equivalent hashes
