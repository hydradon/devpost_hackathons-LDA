Inspiration

We wanted to create an environment where students can build their careers.

What it does

Netly is a place where students can connect with other students and can recieve internships and scholarships from companies.

How we built it

We used a combination of React.js, html5, and css3 for front end and used firebase for back end.

Challenges we ran into

We had a problem with the firebase oauth.

Accomplishments that we're proud of

We're proud of the connection feed where students can talk to each other and the ability to connect with other users.

What we learned

We learned how to manage our time while coding.

What's next for Netly

We will polish the UI and fix all the back end bugs so that we can eventually host it online.
