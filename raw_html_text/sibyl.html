SIBYL

by Srihari Nanniyur

What is SIBYL?

SIBYL is a new, original algorithm and modern web app that predicts the future behavior of binary data sets with one hundred percent accuracy. Yes, this can be done, and I just did it.

Who can use SIBYL?

Anyone who deals with binary data sets. Electrical engineers, meteorologists, statisticians... the list goes on. Anyone who wants to accurately predict the behavior of binary data.

How does SIBYL work?

SIBYL creates a K sub N graph of the parameters and recursively combines each node into binary or ternary expressions that go into new graphs which are recursively collapsed like this until only one node is left, and that’s the final expression. SIBYL can also use Bayesian inference algorithms to increase precision and speed.

Why does SIBYL matter?

It's revolutionary. When it comes to binary data, SIBYL renders all other algorithms obsolete. It presents a new way of looking at data, and a vast new horizon of analytical and AI applications.

What is SIBYL written in?

The algorithm can be written in any programming language, but I chose Python, with HTML and Flask for the web app.
