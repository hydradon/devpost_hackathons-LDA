Inspiration

There aren't many options when it comes to graphical git managers using Linux. I wanted not only to approach this market but to try and cover the 3 main operating systems and adding gamefication as a differential feature from all other tools available.

What it does

Allows the user to clone, commit, pull and push to git repositories while interacting with a user friendly material design ui.

How I built it

I learned about Electron while using Atom, after getting AngularJS to work properly I added ng-material for the UI components.

Challenges I ran into

Learning Electron, using git from nodejs.

Accomplishments that I'm proud of

Managing to commit and push to the tool's repository using the built tool.

What I learned

Web technologies are taking over multiplatform development.

What's next for Ionberry

Completing more git tasks such as merging and branching.
