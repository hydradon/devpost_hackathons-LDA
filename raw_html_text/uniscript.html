Inspiration

Transferring between universities requires students to manually aggregate their transcripts and syllabi, an error-prone, manual, centralized system. Additionally, employers must contract background verification companies to thoroughly vet their prospective employees.

What it does

Publish academic receipts on a decentralized, permissioned blockchain.

How we built it

Oracle Blockchain hyperledger fabric for the blockchain, Node.js web backend, Mongo DB.

Challenges we ran into

First time using blockchain -- growing familiar with Oracle platform. We ran into issues parsing JSON arrays for the student courses.

Accomplishments that we're proud of

We offer full CRU functionality for the blockchain behind an authorized web layer.

What we learned

How to write and deploy blockchain applications to the world.

What's next for Uniscript

  1. Non-university educational receipts
  2. Transferable course whitelist per university account
  3. Recommendation engine for transfer students, including "most transferable universities" and optimal course ordering.
