Inspiration

We were inspired to create this website because of how difficult it is to navigate the multiple website that BU offers when it comes to planning our courses. Ideally, we wanted a single website where we could look for BU Courses, to map out those courses for the next four years, and to keep track of the hub units that are fulfilled along the way.

What it does

When the user enters in the courses they plan on taking, the website returns the hub units that the courses fulfill. In the future, we also plan on having a feature that suggests courses to fulfill the hub units that aren't fulfilled. We also plan on allowing users to create profiles so that the courses they have already taken are saved into a database.

How we built it

We built it using Visual Studio Code (html, css) and Firebase.

Challenges we ran into

We don't know how to connect to a database to keep data for all the courses that BU offers. We also had a hard time with successfully Googling the things that we needed, with creating a box around the Login Form, and with filling the whole page behind the Login Form using a BU picture.

Accomplishments that we're proud of

We coded something. We are also proud of how some links can redirect to a different portion of the website and how user Emails/Passwords would be saved onto a database (Firebase).

What we learned

That creating a website is really hard. Otherwise, we were exposed to html and css and have become more comfortable with it in comparison to before we started the project.

What's next for BU Hub Units

We want to create a database for all the courses that BU offers so that we won't need to hard code every course and their BU Hub Units. As mentioned before, we also plan on having a feature that suggests courses to fulfill the hub units that aren't fulfilled and for users to create profiles.
