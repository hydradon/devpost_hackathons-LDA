awesomewholesomebot

Equithon 2018 Project

A Reddit bot that strives to make people happy written in python. Promotes good mental health.

In today's world, social media pervades our day to day life. A lot of times, social media can cause stress and anguish towards one's self. With awesomewholesomebot, it exudes positivity throughout social media, and in particular, Reddit!

This bot was made using the Python Reddit API Wrapper (PRAW), an API used to be able to scripts or bots on Reddit, of course.

Features:

The bot uses keywords to detect certain emotions within a user and will provide a wholesome, helpful message depending on mood.

It will direct users towards different resources for their mental health, depending on severity.

Also, there are some commands that can be used for the bot

!quote for inspirational quote !joke for a funny joke !music for a calming Spotify playlist
