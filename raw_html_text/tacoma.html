Inspirations

Introducing FB Users Contest or Electoral Page for Live Voting, Elections and Contests on Facebook

Here on Facebook, a lot of contests, elections etc. is going on. Contest and Elections is an event that can drag people out from all corners of the world this because its either a win or lose affair. It triggers status updates engagement as people will be commenting, reacting, liking as they see contestant vote performance going up or down in real-time...

After watching United State elections between President Elect. Mr. Donald Trump and Mrs Hillary Clinton. I could see the kind of wave it could create.

Unfortunately, I have to switch from one Facebook Page to another to see Users reaction on Eg. from MR A page to Mrs B Page. . The same FB Users page switching is applicable to all other elections and contest going on Facebook.

To some extent this page switching is tiresome. I need to change that. What if we can bring all the contestant on just a page view with each person on its rows showing his vote statistics, performance, likes and commentaries. etc. This will smooth the whole electoral and contest processes while driving users engagement to its peak...

Consequently, in Social Groups,Schools, Offices, Politics etc. , People are organizing contests, election when contesting for different positions. During Contest/elections, a lot of promises is being made like Contestant A will say, I will build good Road Network, Contestant B might say say I will build Hospital for the Needy. Okay We will like to see all this happening live on Facebook Electoral/Contestant Page .

My question now is? " How do we generate this Contest and Electoral wave impulse into Facebook to create the highest engagement ever in the history of mankind " This is where this app comes in.

What It Does

1.) It allows Students, Schools, Facebook Groups, Social Groups,Politicians, Governments etc to showcase their Electoral/Contestant Candidate Live on Facebook via FB Electoral Page System.

2.)Each Contestant will have what they say they will do during Contest if elected or win displayed on their contest timeline in form of Statistical Contributions as can be seen in the screenshot above

3.) People uses Facebook to login, Votes and comments under any Contestants where appropriately.
4.) The Voting Counts goes on live on Facebook and winner will be selected based on Vote counts and commentaries count etc.
Bring all the contest and election going on in all part of the world on this FB Users Electoral/Contestant Page will create the highest engagement ever in the history on mankind..

How to Use this App.

Contestants is already Created by me the admin
1.) Login using Facebook
2.) Read the Statistical Contribution of all the contestant (In the real word on Facebook, this should define the list of functions or things the Contestant Promise he will do if he wins)
3.) Vote and comment under any contestants that you wants
3.) Invites Facebook Friends using Facebook Shared button etc..

FaceBook Technology Used:

1.) Facebook Login Graph API: Used to authenticate Users
2.) Facebook Shared Button: used to allow user to share content to her Facebook Timeline
3.) Facebook Send Dialog Button: used to allow user to share content to a specific user in HIPAA Compliance Mode.
