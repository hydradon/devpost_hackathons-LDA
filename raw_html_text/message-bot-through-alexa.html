• Alexa Message Bot

Inspiration

Wanted to build Alexa Skill for communication using SparkPost REST API.

What it does

Can send emails and texts using our Alexa Skill powered by SparkPost and Twilio. Trivial questions are answered too.

How we built it

Using Python, Alexa Skill Set and AWS Lambda.

Challenges we ran into

Could not get proper formatted output and configuring Alexa Skill was a big deal.

Accomplishments that we're proud of

Finishing a simple alexa skill set for email and text.

What we learned

Bits and parts of Alexa services and AWS.

What's next for Message Bot through Alexa

Can integrate other apis for full-fledged communication skill set and enable other features for the skill.
