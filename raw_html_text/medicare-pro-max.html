Inspiration

Troubled by the lack of transparency in American healthcare system, we created this web app to set the baseline for transparency and negotiation.

What it does

You can look up common inpatient and outpatient to find out how much each hospital charges, the state, and national average.

How I built it

Using public data from cms.gov and the desire to change the system.

Challenges I ran into

Deciding between local storage, database (SQL / NoSQL), and calling API.

Accomplishments that I'm proud of

Being able to generate working prototype of our website.

What I learned

Calling API repeatedly on bad wifi is, surprisingly, a bad idea.

What's next for Medicare Pro Max

Add options for users to specify states and update data should the government continue to release them.
