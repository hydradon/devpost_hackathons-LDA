Inspiration

Actually, it was my wife's idea of creating an skill for Alexa that would help kids with the question: What do you want to be when you grow up?

What it does

Asks questions regarding things that kids may like, e.g. Do you like music? Do you like math? and suggests possible career paths.

How I built it

I got an existing template for a quiz type skill and I changed it to the required functionality using Node.js.

Challenges I ran into

Debugging and testing a voice app is very different than traditional software.

Accomplishments that I'm proud of

Put live my first Alexa skill for kids.

What I learned

I learned how to create an interactive dialog with Alexa and how to manage states and sessions. I also learned about kids' preferences.

What's next for When I Grow Up

Add more careers and suggestions.
