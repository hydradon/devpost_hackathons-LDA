Inspiration

We love free food, and would pretty much do anything within reason to get more free food. However, most of the time we miss out on such events because they are poorly organized, publicized, or announced at the last minute. At the same time, we wanted a way for clubs and organizations to easily increase their publicity. Thus, we created a solution.

What it does

The mobile-friendly real-time updated page keeps track of free on-campus food events sponsored by clubs or organizations or even individuals chronologically by closest event to the current date. Within that, the users can search for, add future events to the website and even add the most interesting events to a separate favorite list inside their own personalized account page.

How we built it

We used Python with Flask as a framework to code much of the back-end functionality, coding in HTML, CSS, and Javascript for the front-end functions. We then deployed it using Azure.

Challenges we ran into

We have never used many of the languages we used for the hackathon before, so learning them was difficult.

Accomplishments that I'm proud of

Finishing the project and creating a fully functional product.

What I learned

How to program in HTML and CSS, as well as improved my skills in Python.

What's next for Free Food Finder

We'll add a GPS tracking function to each event that will estimate the distance to the event from your location as well as predict if you still have time to reach the destination before the food runs out.
