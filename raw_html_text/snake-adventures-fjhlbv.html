Snake breakout

This game is mashup of snake and Atarii Breakout.

to run use python3 main.py

use arrow keys to turn the snake. the ball bounces off the walls and the snake while damaging the bricks it touches. to win, destroy all the bricks. to lose run your snake into the walls, bricks or itself.

created for hackEd 2017
