Purpose

Statera aims to help you Improve your posture by analyzing your balance when walking. Using the smartphone's gyroscope and accelerometer, we evaluate the directions your body naturally leans and presents it in a one-page dashboard.

How We Built It

Statera is hosted on Google Cloud platform. We used a NodeJs Express app as our backend server and built the client with vanilla HTML, CSS, and JavaScript. Our neural network uses a simple linear model and is run by tensorflowjs in the browser.
