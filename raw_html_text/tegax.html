Inspiration

The inspiration was to speed up the whole process of acquiring and processing multimedia data to get fast and acurate results.

What it does

Automates transcribing and translating audio (acquired directly or ripped from video files)

How we built it

We used react native framework for building a cross-platform mobile application which uses google speech and google translate APIs for automatic transcribing and translation for mobile surveys.

Challenges we ran into

Integrating google APIs in react-native was a daunting task.

Accomplishments that we're proud of

Buiding a neat and efficient UI with a working prototype that could be sclaed
