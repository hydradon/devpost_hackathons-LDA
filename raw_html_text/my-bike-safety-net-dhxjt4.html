Inspiration

Oxford is acknowledged as one of few true 'Cycling Cities' in the United Kingdom. ( oxford.gov.uk ). As students, we are conscious of the bond between a student and their bike, that's why we've created an interactive map helping identify spots, where you should not leave your bike!

P.S. Non-students are more than welcome to use it too:)!

What it does

It visualizes bike theft data from police.uk and plots it on the one and only Azure Maps. We also provide Machine Learning powered insights on bike thefts.

How we built it

It all runs on Azure and Azure only:

  • Azure App Service
  • Azure Maps
  • Azure Notebooks
  • Azure Machine Learning Studio
  • Azure Data Explorer

Challenges we ran into

  • Poorly documented Bleeding edge Azure Maps.
  • Data preprocessing (missing values)

Accomplishments that we're proud of

  • Using Azure Maps.
  • Building the heat map!

What we learned

  • data viz
  • Azure services and its... perks.
