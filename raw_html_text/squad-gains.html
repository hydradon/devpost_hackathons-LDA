Squad Gains

Team Investing

Description

Squad Gains is a platform that lets people invest as a team. Every week, members can choose and vote the portfolio type they would like to target by updating answers to the risk assessment survey.

Stack

  • Wealthsimple API (investment platform)
  • ReactJS (frontend)
  • ReactBootstrap (styling)
  • Flask (backend)
  • SQLite 3 (database)
  • Advanced Python Scheduler (scheduler)
  • Darksky (weather data)
