Inspiration

College campuses have been known for high quantities of sexual abuse. Since one of our members is in training to be a Green Dot, we decided to make a project focuses around that.

What it does

The belt buckle has a button which is connected to an alarm system with speakers imbedded into the studs of the buckle. The user presses the button twice in rapid succession in order to activate the alarm.

How we built it

We coded an alarm system on the Arduino board and built the circuits to go with it. We created mock prototypes as well as an online model.

Challenges we ran into

The circuit board stopped working after awhile.

Accomplishments that we're proud of

We have a working code as well as the schematics for the actual circuitry for the alarm.

What we learned

That we have a lot to learn about programming and engineering

What's next for Alarm Belt

Hopefully getting a working sample with a microchip. This will allow us to actually imbed the software into the belt buckle to make it less noticeable.
