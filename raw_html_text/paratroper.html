Inspiration 

 We like books.

What it does 

 We take a list of books the user has read and enjoyed and find a book with similar tropes to those shared by the inputted books by scraping DBTropes/TVTropes pages.

How we built it 

 Teamwork and Caffeine

Challenges we ran into 

 Attempting to use and load the relational n-tuple database from DBTropes. We changed the project to xml parsing to overcome this difficulty but cut down features and scope due to the slow speed.

Accomplishments that we're proud of 

 Basic functionality.

What we learned 

 Basic Javascript and Ajax; SPARQL queries (unused)

What's next for paraTroper 

Implementing the ability to add movies you like and search for either a movie or a book
