Inspiration

"Smart" appliances and "Smart Home" devices are expensive; we thought we might be able to add a little bit of smartness with some less expensive accessory devices.

What it does

Warns user when they leave home and sensor readings (such as stove temperature) are out of range.

How we built it

Using Android Studio for the user-facing Android app, and and Edison for Arduino for collecting sensor data.

Challenges we ran into

Sending data over a bluetooth connection on the Edison.

Accomplishments that we're proud of

Create data simulator.

What we learned

Some of our team members used C for the first time, others used Android APIs for the first time.

What's next for You Left The Stove On

  • Communication from the Edison isn't working
  • Support multiple sensors with configurable ranges
  • App should warn when connection is lost when reading is out of range
  • A hosted service to convey warnings to users when away from home
