Inspiration

Nobody is interested in trying to resolve public problems found around. Mainly the reason to do so is the path to file at least a complain about it is way to hard and in fact nobody know exactly where they need to complain. This platform will allow the user to simply file in complains without any other headaches. While filing, user need to select concerned department if he knows, or we will take care of it.

How it works

User files one complain. It goes to admin. Admin verifies the genuinity. Once verified, a complain email is fired to concerned department.
