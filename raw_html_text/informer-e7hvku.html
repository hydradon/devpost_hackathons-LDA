Office add-in Informer provides the user a quick access to messaging statistics, events and notes. The application shows a brief information about messages and events. The message can be opened in a single click, which increases the user productivity. Also, a funny animated cat is designed in the application to show that there is a new message.

To be informed means to be productive.

Used technologies:

  • Office Add-in
  • Fabric UI
  • Microsoft Graph
    • User API
    • OneNote API
  • OAuth 2.0 authorization
  • AngularJS
  • WebGL + BlenderWeb
