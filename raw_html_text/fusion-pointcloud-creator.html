When I heard the name of this hackthon about Fusion 360. It went a spark on my mind because I have used Fusion 360. And I always looked for this type of events.

This application creates point cloud file(PLY) of Fusion 360 model

It build in c++ using fusion 360 APIs.

There are challenges like understanding and implementing point cloud library which should be in c++.

I am satisfied that I have accomplished the application in expected timeline with successful implementation of point cloud.

It tough me very deep knowledge of c++ and it's interface.

I will be working to support more and more point cloud format in future.
