Inspiration

We were inspired by seeing inappropriate LinkedIn photos or other photos people used on professional networking sites.

What it does

Our platform analyzes peoples profile images and uses classifications from the googles-vision platform to determine if the proper criteria.

How we built it

We built it using the google vision platform which uses google AI to detect keywords

Challenges we ran into

We initially had difficulty authenticating with the google api, and we also didn't realize we were writing code for the depreciated version of google cloud "gcloud". We then converted out application to use "google-cloud"

Accomplishments that we're proud of

Getting the authentication to work with the google api.

What we learned

We learned is that we were unable to train the google cloud vision platform in the same way we could train the IBM Watson vision platform. The google vision platform does work really well for object detection though.

What's next for Profile Checker

Next we plan on trying to use another platform capable of tracking more emotions which can be used to check profile photos for dating and other social media websites as well.
