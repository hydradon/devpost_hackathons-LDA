What it does

EduSpace is a multi-lingual logical query system that makes learning interactive.

How we built it

We used Deep Learning to create an intelligent system that can extract relevant information from a given text. It is trained on wide range open-source dataset.

We have a working demo and we got very good positive reviews from the test users we showed our product to.

What we learned

We learned that the world cannot be changed overnight but we are proud to build the first step.
