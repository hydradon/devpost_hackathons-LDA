Inspiration

We wanted to experiment with Virtual Reality.

What it does

This is a VR Cardboard game where you fly around, explore and find hidden glowing balls.

How we built it

We used Unity with a Cardboard headset on Android.

Challenges we ran into

We were inexperienced with Unity and VR.

Accomplishments that we're proud of

We made a complex world that can be navigated.

What's next for Snow Volcano Whale

We hope to make it even bigger and with more features and whales.
