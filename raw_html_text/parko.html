Inspiration

What it does

I grabs the location of empty parking and show on the app. User can book the location. Park !!!

How I built it

We used Facebook APIs for login authentication then We use Google Maps for showing empty parking spots. We used Intel Edison + Bluetooth + Ultrasonic Range Finder to simplify registration and booking process.
