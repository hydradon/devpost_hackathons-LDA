Inspiration

It is always a problem to remember to take pills on time, so we need a better way to track pill taking.

What it does

The user can input the number of times he has to take the medicine and also provide safe intervals for each time he has to take these pills. Subsequently, every time you take a pill you should press the button. This will send request to the cloud server to store the data. Then, we check in regular intervals whether there is an interval and provide messages to the user about his pill taking.

How we built it

We build it using Node.js and AXA wireless button.

What's next for PillTracker
