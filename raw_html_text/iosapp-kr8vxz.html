Kayla and I decided to build this IOS application because we liked the idea of being able to select an event through a dating app style. If you selected the event it would save the details and remind you of your upcoming adventure. In this demo we added mock events and descriptions. Given more time (and experience) we would have liked to implement dating app style "its a match" swiping and an integrated search function to find the users perfect adventure. We wanted to encourage the user to explore their city by attending local events. Kayla is began this app with no experience and I started with just barely the basics. We completely corrupted our code multiple times and had to restart. Other then that we enjoyed this experience and we hope you enjoy the small demo.
