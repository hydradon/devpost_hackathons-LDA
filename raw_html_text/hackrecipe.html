Inspiration

Have you ever wondered what to have to eat? Get home from work/uni and just can't decide?

We have so we made a web app to decide for you

What it does

The app picks a random recipe from the database Displays the recipe and other information to the user Provides a link for the user to view the full recipe online

How we built it

The backend is built via an SQL database with recipes grabbed from a recipe API The database stores what ingredients have been liked/disliked The web app calls for a new recipe to be delivered and displayed

Challenges we ran into

Sorting the ingredients was challenging due to a lack of normalisation Building a dynamic webpage

Accomplishments that we're proud of

Randomly generating meals Parsing the original database

What we learned

Basics of building a flask web app How to manipulate web APIs

What's next for HackRecipe

Recommending meals containing preferred ingredients AI learning to recommend recipes based on the ingredients you tend to like/dislike using a weighting system
