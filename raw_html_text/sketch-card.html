This app is a baseball app that allows users draw on a scorecard from an upcoming/ongoing game. We used the SportsRadar api to get current game data. We used the javascript canvas to create a scorecard. While we have some of the components of our app working, we overscoped the project for the amount of time we had. We enjoyed working on this project and intent to continue to build it out.
