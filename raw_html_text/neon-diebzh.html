bold NEON PROJECT

Inspiration

We all ended sadly on a hospital. Unfortunately we saw the lack of efficiency of some paperworks. And we are watching Scrubs so we got an idea on how hard it is to make a diagnosis. That's why we wanted to do something to help theses issues.

What it does

The idea is to keep all our medical data on our cellphone (on hard of course). Like that, we would be able to transfer by NFC all our data to doctor or medical professionnal. Like that, even if the patient is unconscious, the doctor would be able to gather all needed information to make a more accurate diagnosis and take les risks.

How we built it

It was built base on the technology NFC to communicate the data from the cellphone (coded on Java for Android) to a Nucleo F7.Then the flow of data is given in UART for the main PC which displays the data thanks to QT.

Challenges we ran into

We wanted to apply to Capgemini project and a bit for the Bloomberg subject.

Accomplishments that we're proud of

We are so proud succeeding on making an Android App, gathering the stored data on NFC and display it on a screen in only 24Hours (23 actually xD). And everything works as expected.

What we learned

NFC protocol, cross platform coding, management of time and project.

What's next for NEON

More improvements as appointment application, chat bot, etc...
