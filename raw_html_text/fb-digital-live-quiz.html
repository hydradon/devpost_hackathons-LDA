How to use this app:

1.) Login using Facebook
2.) Select Create Live Quiz
3.) Enter Quiz Title, Category and Quiz Description(Enter this just once and it will get propagated)
4.) Enter the time you want the quiz to elapse(Enter this Once. The quiz time will get propagated throughout the quiz questions and answers that you will create)
once the time is over, The Quiz gets submitted immediately and score the user
5.) User can create unlimited number of Questions(Eg. Up to 50 Questions) along with their Corresponding Anwsers and Options
6.) For each Questions and answers and options enter click Save Quiz Button to save it.
7.) Once you finished creating your quiz questions(Eg. Up to 3- 50 Questions and above), then click on Publish Quiz button to post it on timeline.

8.) Friends, Members etc can take your on any Post directly from the Post Status timeline. The quiz is timed and User get scored based on her performance.
9.)Once the Users finish taking the quiz, Users can see Reviwed Quiz button which when click will show them questions they get right and wrong.
10.) User can Like and comment on the quiz
11.)The User can Share any Post to her FaceBook Timeline as well as to her specified Facebook Friends in HIPAA Compliance Mode.

FaceBook Technology Used:

1.) Facebook Login Graph API: Used to authenticate Users
2.) Facebook Shared Button: used to allow user to share content to her Facebook Timeline
3.) Facebook Send Dialog Button: used to allow user to share content to a specific user in HIPAA Compliance Mode.
