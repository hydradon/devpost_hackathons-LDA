Inspiration

A lot of people do not consider all the facets of travelling, when they do so. We decided to build a web app that would help remedy this problem that many travelers face.

What it does

It reports the average temperature and humidity of the location for the next 5 days, after given an input of an address or region.

How we built it

We utilized OpenWeatherMap API and Google Maps API to construct accurate and up-to-date information.

Challenges we ran into

We found it particularly challenging to use Microsoft Azure to host our backend python code.

Accomplishments that we're proud of

We have working API's.

What we learned

We learned how to use Flask to contruct API-endpoints, as well as how API's function.

What's next for Just Another Travel App

We plan on incorporating more features, such as nearby hotels and airports, as well as transportation in the area.
