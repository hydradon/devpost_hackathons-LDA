Inspiration

We wanted to create something that would get users to interact with each other.

What it does

An in chat game of hot potato with a bot named "Hot Botato" where users have to type in commands to pass the bot around. If left idle too long, the bot will kick the user out of the discord chatroom who is in hold of the "Hot Botato".

How we built it

Primarily javascript and node.js

Challenges we ran into

Implementing features while learning the API on the go!

Accomplishments that we're proud of

All the gifs we put in this bot

What's next for Hot Botato

Next implementation will have a "death count" for players.
