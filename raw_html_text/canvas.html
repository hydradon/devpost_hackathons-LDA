Inspiration

I wanted a sharp looking report that detailed information about my Twitter account. Rockegraph was the perfect platform to build this on.

What it does

Using the Twitter API and Rocketgraph SDK the report pulls together information and displays it in a useful way.

How I built it

I broke down the process into blocks and focused on each separate block. In the end I was able to pull it all together and generate a sharp looking report.

Challenges I ran into

Time, it slips away quickly. I had to postpone several features that I look forward to implementing shortly.

Accomplishments that I'm proud of

Dusting off the php knowledge and turning it into something useful.

What's next for Canvas

Refine the report and make it a tool many can use to track Twitter account stats.
