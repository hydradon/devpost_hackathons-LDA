Inspiration

I have an interest in coding since 13. and a few days back I got an email from MLH about local hack day. When I came t know about it, I just had to join! So I freed up all the works I had in 1st December and started preparing for it. So in this hack day, I decided to make a tasks web app which runs in the browser and uses local storage as a state management system. I have a very limited time to complete the project, so I skipped all server-side part, instead used local storage as state management.

It helps to keep track of Tasks.
