Inspiration

Somos estudiantes y como estudiantes afrontamos el problema diario del transporte público de la ciudad de Durango, el cuál es muy tardado y con pocas unidades por ruta. Lo cuál causa que el perder un camión, causa una espera de 30 minutos promedio. Es por eso que quisimos resolver esto.

What it does

Es una aplicación en la cuál le dices la dirección a la cuál quieres llegar o un lugar y te dice que ruta es la más adecuada tomar. Después muestra en tiempo real el movimiento de la unidad de transporte en un mapa, permitiendo así visualizar donde está.

How I built it

Lo construímos con Android Studio, utilizando las plantillas de Material Design compatibles con versiones anteriores. Permitiendo así brindar un diseño actual para todos. El core de la app es Google Maps API 2 para Android, el cuál brinda una gigantesta base de datos de direcciones y posibilidades para dibujar en el mapa además de ser actualizados.

Challenges I ran into

Nos faltaban las herramientas necesarias para comenzar a desarrollar y el internet no permitía conseguirlas, pero fuimos a pedir prestado internet a una casa cerca y pudimos descargar las herramientas para comenzar el desarrollo. Además no sabíamos nada sobre Google Maps.

Accomplishments that I'm proud of

Estoy orgulloso de lograr una animación que nos tomo toda la noche y que terminamos lo planeado antes del tiempo. Además congeniamos muy bien como equipo y no hubo malentendidos.

What I learned

Aprendí el manejo de la API de Google Maps para Android haciéndome ver que es una de las herramientas indispensables para esta plataforma.

What's next for BUSca

Planeamos terminarlo y lanzarlo al público.
