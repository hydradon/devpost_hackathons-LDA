Inspiration - Our parents who might enjoy video games if they could just yell at them instead of button mashing

What it does - Throws pizza at the enemy chef when player yells "throw", player ducks when user yells "duck" and jumps when the user yells "jump" in English, Arabic, Spanish

How we built it - Built the game in Unity and wrote scripts in VS Code, activated google voice api with python

Challenges we ran into - connecting voice activation to unity

Accomplishments that we're proud of - completing a game in 25 hours

What we learned - we can do a lot in 24 hrs

What's next for Food Fight! - more languages, multiplayer
