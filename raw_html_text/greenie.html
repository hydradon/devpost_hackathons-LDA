• Greenie

Inspiration

Climate is Changing, since there is no B Planet, we have to promote Environment Best Pratices to protect it for all human beings and future generations.

What it does

Greenie Alexa custom skill provides message to educate end users for selective sorting, recycling and promote advice to protect better the planet

How I built it

Greenie Alexa custom skill is built with Node JS, AWS Lambda

Challenges I ran into

Since the Alexa skill addresses global planet/environment issue, I decide to design multi language/countries skill everywhere Alexa is accessible :

  • EN : english (US, UK, CA, IN, AU)
  • FR ; french
  • DE : german
  • IT : italian
  • ES : Spanish from Spain and Mexico

The challenge is to use only Alexa simulator to test the custom skill service in languages like IT, DE, ES, since I am not living in those countries, but the skill has been certified successfully, after few failed attempts ;) I highly recommend the usage of AWS Amazon translate service to help translate in multiple languages.

Accomplishments that I'm proud of

Custom Skill in multiple languages/countries

What I learned

Multiple Langues Skill Experience (Interaction Model,...)
