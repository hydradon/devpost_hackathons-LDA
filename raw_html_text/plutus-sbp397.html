Inspiration

The theme of this hackathon: a solution that you can bank on

What it does

Plutus is a multi-platform chatbot that you can talk to your bank account including making payment, asking about balance and spendings. You can access the chatbot through Facebook messenger, Slack or website. Plutus makes it easier and more fun to manage our banking account.

How I built it

End user -> Slack app/Facebook Messenger/Web demo -> stdlib handles events and messages -> Google Assistant+Google Cloud for database(dialogflow)

Challenges I ran into

This is my first time using stdlib and I am already amazed by how easily it can create commands and functions with this many platforms

Accomplishments that I'm proud of

The lucky cat is pretty cute I think

What I learned

Learning new things is fun

What's next for Plutus
