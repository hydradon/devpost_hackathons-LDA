Inspiration

We enjoy Escape Rooms and virtual reality so we wanted to combine the two and have it be reeeaallllyyy spoookyyyyy.

What it does

It scares you, gives you a little motion sickness and allows you to walk around and interact with a few object in our virtual escape room.

How we built it

We used the Oculus DK2 integration tools within Unity to allow the use of the headset. The rest of the game was just made in Unity.

Challenges we ran into

We are all starters in using Unity as well as the Oculus rift so it took about four hours to just get the rift working in Unity.

Accomplishments that we're proud of

We are proud of everything we have managed to accomplish today.

What we learned

We learned a lot about Unity as well as the nuances of coding for the Oculus.

What's next for Escape Room VR

We are going to add a heart rate monitor, polish it, and then publish it.
