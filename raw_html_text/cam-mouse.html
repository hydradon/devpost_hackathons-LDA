• Cam Mouse

Inspiration

I wanted to work on a project involving image processing to help increase the amount of ways that we can interact with our computers.

What it does

This program uses image recognition to determine your hand position and gestures to interact with your computer.

How I built it

Cam Mouse is built using python and opencv. It utilizes deep learning algorithms to detect a hand and determine the number of separate fingers on the hand as a rudimentary gesture system.

Challenges I ran into

Working with new software such as opencv and libraries such as dlib

Accomplishments that I'm proud of

Getting this project working after changing ideas multiple times.

What I learned

opencv, image processing, and deep learning

What's next for Cam Mouse

Implement more ways to interact, such as facial recognition, eye tracking, and blink recognition.
