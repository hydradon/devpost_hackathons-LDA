Inspiration

Coming from another city, we know what it feels to be lost in a new place. We created a way to make custom maps to localize esential places in an area, and made a tool for anyone to make custom maps.

What it does

Make custom maps

How I built it

Polymer based Webapp, used google maps api. An android client app was made to access this maps via a custom code.

Challenges I ran into

I didn't knew how to use maps api.

Accomplishments that I'm proud of

Learn to use maps api and make an app in under 1 day, use git with other people.

What I learned

Maps api for js and android, firebase storage.
