Inspiration

We thought it would be great to have a chatbot handy to answer any of your financial questions.

What it does

Our financial chatbot gives users the information they need to make informed decisions about their financial matters.

How we built it

We used botkit, which is a tool for creating conversational automation, to manage the chatting interactions. Then we used BlackRock APIs to get financial information based on current markets.

Challenges we ran into

The most significant challenges came from connecting the chatbot to the API. We were not familiar with the way we were supposed to call the API; however, once we got the API working, we were happy with our product.

Accomplishments that we're proud of

We're proud that we tried out some new APIs and made a cool, useful product.

What we learned

We learned more about chatbots and javascript.

What's next for Financial Chatbot

We would like to increase the feature set and make the chatbot widely available for those interested in getting useful financial information.
