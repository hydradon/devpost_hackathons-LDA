Inspired by Gojek to connect Ojek Community, we build mangkal.id . Mangkal.id is an platform to connect street hawkers community to their costumer. We Build mangkal.id with Vue.js, google api login and firebase in a month. We challenges to build Street hawkers be large community and easy connect to costumer and be a bridge to make online orders for street hawkers. Before we start develop, we have submit idea on Bali Startup Camp 2017 and we win first place, after we hear about this challenge, we very proud to start develop this app. Next project mangkal.id will build in app chat and free call.
