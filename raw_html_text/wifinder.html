Inspiration

To be able to track any moving objects with WiFi signals. Since WiFi is everywhere!

What it does

Able to track and create a profile of certain people, for passive identification. Able to differentiate between humans, animals, and objects.

How we built it

Blackboards to synthesize the idea, Python, Keras(TensorFlow), for back-end, Bootstrap (HTML, CSS, JavaScript), for front-end.

Challenges we ran into

Bridging the gap between back-end and front-end.

Accomplishments that we are proud of

Completing the project.

What we learned

Learning the new libraries, such as bootstrap, and Keras(TensorFlow).
