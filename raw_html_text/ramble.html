Inspiration

Given the huge amount of cultural events and amazing places to visit, we decided we wanted to give a tool to travellers and people in order to let them know what happens around them.

What it does

The app uses the GPS in the iPhone to accurately find the user's position. Then, once the user has selected his interests he will get notified of events that are relevant to his choices, or the user can also search for popular events around his current location.

How we built it

We decided to create the database using Laravel, because it is a framework easy to use which can help us create a good API which would help any other developers work with our tools. For the application, we focused on iPhone development using Swift. Finally, to host our web service we used AWS to deploy a server.

Challenges we ran into

Time was probably the main challenge. Also we used some relations in Laravel that we were not used to. When deploying the server in AWS we had to configure it in order for it to work online and to publish a public api.

Accomplishments that we're proud of

Our whole work and specially teamwork, this was the first we 4 worked a s a team and we were amazed on how well we worked.

What we learned

Each one of us learnt a lot of things, most of us we were not familiarized with Laravel and we also managed to get to work some amazing effects in our application.

What's next for Ramble

We hope to create a nice, web admin control panel. Where promoters of events and places could add and notify users interested in their activities.
