Inspiration

What it does

Detects loudness or softness of sounds and the frequencies of the pitch that is being played that will write to a guitar tab.

How we built it

Using an Arduino UNO R3 board and the software that comes with it

Challenges we ran into

We all had very little knowledge of how to code with Arduinos

Accomplishments that we're proud of

Creating the decibel meter part of the Guitar Protagonist by learning Arduino on the spot

What we learned

We learned how to use the Arduino Hardware and Software
