Inspiration

The one app through which you can share all your social media information through one QR code and keep your information updated with all your friends.

What it does

Based on a given input of different social media account, the app offers you an interface through which you can share the selected social media platforms and share it with your friends which can get all the links to the most used social application out there.

How we built it

The user interface is implemented with React Native for both Android and iOS. We used Heroku as a host for our server and database which is accessed by requests sent by our own implemented API. We managed the workspace using Expo for running the app on devices and Github for project management.

Challenges I ran into

API calls to and from the database.

Accomplishments that we're proud of

Implementing a functional QR scanner and generator. Connecting to a real-time Database. Creating a smooth UX.

What I learned

React-Native, APIS, Working with APIs, UI/UX design and Heroku.

What's next for Pong

Push notifications, add more social media platforms, increase usability and publishing it.
