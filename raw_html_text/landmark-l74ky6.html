Landmark is a directions service that gives you directions the way people do. Streets are useful but landmarks and buildings make traveling a lot easier than squinting at street signs. This is how we do it. First, Landmark using the Bing maps API for routing between addresses. Then we use the coordinates of the decision points (like intersections to turn at) and the Yahoo Flickr API to find nearby geotagged pictures. Finally, we run those images through the Bing Computer Vision API to get descriptions of the photos.
