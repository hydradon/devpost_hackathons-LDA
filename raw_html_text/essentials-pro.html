Inspiration

I was inspired to make an app that has a large customer base on Alexa devices, and with the growing interest of essential oils this seemed like a great opportunity.

What it does

You can ask the application what uses of commonly used essential oils. It responds with the uses and then suggests another oil to search for.

How I built it

I built it using the recipe template provided by amazon, making changes as necessary.

Challenges I ran into

I had issues with making the recommended next oil to make sure it does not recommend the same oil twice in a row.

Accomplishments that I'm proud of

Using AWS as a backend of the application.

What I learned

How to use AWS and linking it with outside software.

What's next for Essentials Pro

I have already created a quiz app people can take to test their knowledge in the area. Next would be go back to the drawing board and rework this original app to be more what the people want.
