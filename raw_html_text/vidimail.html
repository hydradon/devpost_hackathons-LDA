Inspiration

A completely new fun way to voicemail

What it does

Reroutes voicemail to send video messages.

How I built it

Using meteor and pycharms platform.

Challenges I ran into

Integrating the twilio post twml response with meteor.

Accomplishments that I'm proud of

Finished the rerouting of the voicemail and sent the video message successfully.

What I learned

To dynamically use ngork make a page. Also, needed to quickly get through errors.

What's next for Vidimail

Set up accounts properly change the url to the video message. Adding video editor inside, saying video straight to the platform using google api. Email to check video messages, and normal text messages. if received text needs to be able to translate the text input to different languages.
