Inspiration

How often are you with a group of friends and someone has the idea to play poker, but alas no one has the cards or poker chips to play with. For us, it happens all the time. That's why we decided to make a fully digital poker game that only requires computer/smartphone and internet access to play.

What it does

The whole point of the game is to be played with a group of people in the same room (or it could also be streamed via Twitch). One person needs to run the Unity interface, which contains the majority of the table state including the card pool. Anyone who wants to join the game acquires the room code, and then joins via a web browser on any device (meant for smartphones). The group of people then proceed to play a game of poker, with their "face down" cards seen by them on their individual phones.

How we built it

All code was done by Ian, all artwork was done by Matt. Main interface was done in Unity/C#, individual player controller was done using php so as to be accessible by web browser. Communication is done through SQL database communication (mainly due to lack of time)

Accomplishments that I'm proud of

This project was a lot of work, and still has a long ways to go to be fully playable, but to achieve what we have in just this weekend is pretty incredible if you ask me.
