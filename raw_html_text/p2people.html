Inspiration

Bringing accessibility to everyone. We are here to help the disabled and those in need to access to public transportation.

What it does

Three main functions:

  • Planning the most accessible path for people with disabilities.
  • Live Assist. One button for help from those around you.
  • Social Community for locating and notifying other about obstructions.

How we built it

We built a working iphone app using xcode and implementing the P2PKit framework.

Challenges we ran into

Bugs and No sleep. It was amazing.

Accomplishments that we're proud of

We have a working app! That can really make an impact.

What we learned

Teamwork, Perseverance, Swift Coding, Staying Positive.

What's next for p2People

We may build this into a huge multinational conglomerate one day.
