Inspiration

What inspired us was the never ending customer obsession for rogers, and it was inspiring to see an organization that is so relentless in their chase for customer perfection.

What it does

What it does - A 24/7 unmanned pop-up store that provides a personalized and experiential environment to discover phones

How we built it

We built a full-stack app using a node express server, with html, css, bootstrap and jquery for dynamic page loading and reloading. We mocked database with rogers data and displayed it in the app. We also created a 3D model from ground up with Sketchup

Challenges we ran into

Unfamiliarity with languages, Javascript isnt our best language. We faced Feature creep, we had too many ideas at the start but not enough time to finish all of them. We decided to go towards an MVP with minimal features, but with the proof of concept.

We had no experience in 3D modelling, but have to design a 3D model to express our idea and vision.

We also faced problems with time management. We spent too long being unsure of what API we were going to use, and which frameworks would fit the project the best.

Accomplishments that we're proud of

We dared to dream big and worked on this project until 7am yesterday to make it come true. We are extremely proud of the effort we put into this project and the idea. Apart from that, we learned the 3D modelling program, Sketchup, from scratch.

What we learned

Empathy is important while designing a product. We also wanted to dare to dream towards big ideas. With programming and how integrated we all are with technology, the possibilities are endless

What's next for Rocket Raccoons

Exploring, learning and growing together here in the amazing city of Toronto
