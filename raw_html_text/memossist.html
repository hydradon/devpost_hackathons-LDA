Inspiration

Help students memorise texts and lessons

What it does

extracts keywords from given texts or words with Machine text processing and helps you memorize them in three different ways:

  • visualize them with images or graphs
  • acronym the keywords to make them easy to link
  • make rhymes with them to be easy to remember

How we built it

a mobile app(ideally with Xamarin) to enter or scan the texts and pushing it to Azure text ML engine for extracting key ideas, searching for images or acronym words, and ryhme forms

Active use cases

useful for many students, for actors and lecturers

Built With

  • azure
  • c#
  • xamarin
