Inspiration

We are inspired by evolving blockchain-related technologies and decided to build platform to simplify usage of BigchainDB.

What it does

It provides REST API to communicate with BigchainDB, and shows one of many usages in Android app.

How we built it

We considered many options of databases to suit our needs, and decided to create microblogging app for Android system. We separated our work to backend (create API), mobile (create Android app), devops (configure whole ecosystem) and UX (prepare graphical design and test some UX solutions). However, every team member participated in more than one role.

Challenges we ran into

Build ecosystem for Flask Python library with Docker.

Accomplishments that I'm proud of

Setup blockchain ecosystem and create mobile app very fast.

What we learned

We learned how to work in a agile-like team and every member of a team learned something new about software engineering.

What's next for TEAM Halveringstid - BlockDeepChainGraph.it

Currently we are planning to continue development of BlockDeepChainGraph.it project, for we see interesting challenges coming with it.
