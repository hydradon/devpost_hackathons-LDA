Inspiration

We were inspired by one of the challenges at CUHacking 2017

What it does

The project is two-fold. First it allows clients to submit their resume to jobs. Second, it allows employers to find candidates they want by filtering by key words.

How we built it

We built this project using JavaScript, HTML, CSS, React, and RethinkDB.

Challenges we ran into

Sever side session stores, handling state and react, storing PDFs, retrieving text from PDFs

Accomplishments that we're proud of

The fact we were able to make a decent product

What we learned

React, Rethinkdb

What's next for ITripleM

More projects, maybe try using new technologies for cool features
