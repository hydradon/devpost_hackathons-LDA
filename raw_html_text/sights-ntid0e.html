Inspiration

Our team is from UNC-Charlotte, we almost died on the road getting to State

What it does

When using map directions, we alert the user if they doze off from looking into the road for more than a few seconds.

How we built it

Used android studio to build app, google mobile vision to track facial features and RESTFUL protocols to send and receive location tracking information

Challenges we ran into

Started off with iOS but thad compatibility issues. Another issue was finding the face angle relative to the phones position in car.

Accomplishments that we're proud of

We were able to make the app work with mobile vision and take the issue of distracted driving

What we learned

Alot about google apis, backend frameworks, fullstack architecture.
