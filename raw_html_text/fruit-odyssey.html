Inspiration

This is a fun application for children to increase their memory power while engaged in a VR experience in a fruit orchard.

What it does

The user navigates through the fruit orchard picking up fruit in a sequence presented at the beginning of every challenge.

How we built it

We used 3D-Unity Engine with Oculus Rift using Collab feature(collaborative development in Unity Environment). Scripts were written in C#.

Challenges we ran into

There was a steep learning curve for the majority of the team since we were new to VR concepts and Unity programming.

Accomplishments that we're proud of

Beautiful Orchard Overcoming integration hurdles to establish an engaging experience

What we learned

C# Unity(3D Physics Engine)

What's next for Fruit Odyssey

We plan to introduce level based gaming experience with larger sequences and different themes.
