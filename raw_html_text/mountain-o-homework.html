Inspiration

I find it useful for myself to give myself a set amount of time to work, but I also get distracted by my phone, this ideally solves both these issues

What it does

I creates a timer and a climber climbs the mountain while you do your work, incentivising you to not use phone, so the climber can reach the top of the mountain

How I built it

I built it using android studio and using whatever tutorials/information was available in the standard android references

Challenges I ran into

i have used android studio once before, many months ago, so most of my time was spent learning how to make an android app and what features were available to me. One major hurdle was figure out how to animate the mountain climber

What I learned

many, many things about android, and how terrible it is to make sprites and to use sprite animation.

What's next for Mountain O' Homework

I would like to add more features, such as resetting the mountain climber if the app is exited and a score keeping systems.
