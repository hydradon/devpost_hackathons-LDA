Inspiration

We want to make crypto-currencies accessible to everyone.

What it does

We created a crypto-currency that mines for crypto-creatures with unique features.

How we built it

Challenges we ran into

We had difficulty working with the different sizes of the svgs, and learning how to add features to the website. We had difficulties getting the blockchain up on the server and ensuring that everyone would be able to access it.

Accomplishments that we're proud of

We're most proud of the working blockchain and how much progress we made on the website given this was our first time making one.

What we learned

We learned a lot about graphic design tools and how to implement them within a webpage. We also learned a lot about building a website, it was the first time we had used Flask, a Python microframework.

What's next for Crypto Creatures

We want to add player transactions involving the blockchain and animals. Eg. an animal trading system. We also want to implement items and fighting mechanics with the animals as well as a functioning market to buy and sell the animals.
