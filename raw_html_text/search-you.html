Inspiration

Although we live in a virtually hyper-connected world, sometimes we loose the human touch . Everyone has heard older generations explain how the world was before the internet arrived and how all the values have evolved.

So, what is stopping us from enabling people get in touch, face to face , with the help of technology? There are already some successful solutions, but we believed we could go a step further. Imagine you are representing your company at a conference: this app could help you know the other attendees and create solid professional relationships.

What it does

In a nutshell, our project allows anyone to access someone else's data by just taking a picture, and so making it easier to engage in a conversation.

Our focus during the HackUPC has been on developing an API where everyone can publish their images and add personal information, which could include the name, hobbies, a description, Twitter username, etc. Although the endpoint to add this information has not been created, we have added this information manually on the database. This way, by sending an image to the API, an algorithm recognizes the person and returns their information.

With our Android showcase app we have tried to show a basic example. It allows users to take and send images and get the information from the API.

How we built it

At the frontnend side we have developed and Android app that allows the user to scan people faces in order to get his name, hobbies, etc. The app is built using Android Studio and Java.

On the backend side we have depeloped an API that connects the Android device to Microsoft Face Recognition system and our user DB. The system is developed using Node js with Typescript, built on top of docker containers.

Challenges we ran into

In many futuristic movies, people walk around with digital goggles that display people's name on real time next to that person face. One of our objectives was to build an app that would simulate this feeling, but because of the lack of time and resources, the app is quite basic.

Accomplishments that we're proud of

We completed our objectives and we've made a prototype of the app with real functionality.

What's next for Search You

In a world driven by data, we believe that having all the information of every registered user could allow us to predict compatibility between people. This way, one could find highly similar people
