Inspiration

Halloween

What it does

The user has to use the mouse and arrow keys to move the character around. The goal is collect as much candy one can get before the zombies can attack and kill.

How I built it

We built it through Unity and Javascript.

Challenges I ran into

We wanted to add sound when the candy is hit by the user part of the zombie would be in the wall, all disembodied

Accomplishments that I'm proud of

We got the zombie to follow the user Making the game feel creepy

What I learned

How to use the Unity API And the first person motion How to make things move As newbies, we learned that we can do anything that we set our minds to

What's next for Candy Hunt

Add power ups like cars and plants, where the plants would give life and the car would give the user speed.
