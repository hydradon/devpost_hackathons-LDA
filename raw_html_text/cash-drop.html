Inspiration

Tired of sending money to the wrong people and tired of entering long usernames

What it does

Finds local users and allows them to connect and initiate a financial transaction

How I built it

Android Studio with user data stored through Google Firebase Authentication and Database System

Challenges I ran into

Biggest challenge is implementing a working payment system without a dedicated server

Accomplishments that I'm proud of

Finding local users through GPS location

What I learned

Corey: I learned how to use Android Studio and how not sleeping makes you feel sick, Melissa: I learned how to stay up longer

What's next for Cash Drop

Bluetooth implementation for finding local users, group payments, improved UI with floating bubbles for users, improved system security, and of course, public release.
