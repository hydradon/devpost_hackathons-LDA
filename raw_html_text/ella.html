Inspiration

I always wanted to develop such an app but did not have enough motivation to do it.

What it does

The app make taxi sharing much faster, safer, easier and efficient for both taxi drivers an passengers.

It uses a very sophisticated algorithm to determine the best ride for the both riders and taxi drivers.

The user can to request to :

  • Ride alone or with others.
  • Ride with men only, women only or both.
  • Can refuse to take a specific rider, the request must be accepted by all riders.

While driving, the drivers does not need to interact with the app. the app will only guide him to the next location.
