Inspiration

Our inspiration started with a really small problem: An easier way to take attendance. We wanted to use facial recognition technology to automize taking attendance in the classroom. However, then we deviated into the security aspect of the project which led to the beginning of differentiating between a criminal and a normal person.

What it does

Its a program that compares an image with features of a face with another set of pictures and tells you whether that face matches with a set of bad people, or "criminals"

How I built it
