Inspiration

  • We have seen rural people transportation challenges.
  • In urban areas we ourselves have experienced many challenges everyday while commuting.
  • Cause public transport is the future.

What it does

  • Our system tracks vehicles which are used for public transport systems to inform users/commuters where the desired bus/vehicle is currently located.
  • It helps drivers find passenger without much hassle.

How we built it

  • Cause this app is also used in rural area we kept in mind non-functional requirements so it was built using feature phone running on KaiOS.
  • We also integrated google map API for navigation.

Challenges we ran into

  • Some challenges that we faced were integrating maps API.
  • We were also very new to KaiOS development.

Accomplishments that we're proud of

  • We have successfully integrated maps API.
  • We also have finished most of the user interface.

What we learned

  • We have learned how to integrate maps API.
  • We have learned how to build a KaiOS app.

What's next for Yet? app

  • We want to further develop this app cause we believe that our app solves a critical problem.
  • We also want some sponsors who are willing to fund our app.
