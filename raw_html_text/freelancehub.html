Inspiration

Very hard to find internships in the real world as we do not have enough experience/skills and it may be a source of side income once you become very good

What it does

Once a client post their job listing on our website, it will automatically be sent into a telegram channel with the relevant details for the free lancer to see whether they are qualified/interested for the job. (All information are archived in SQL server)

How I built it

Front-end includes bootstrap with html5 and css, Back-end for the web was built with Express.js, microsoft azure cloud for SQL telebot was build with javascript library connected to back-end server

Challenges I ran into

No experience with web developing, had to learn the backend from scratch

Accomplishments that I'm proud of

learn the backend from scratch

What I learned

full stack web development

What's next for FreelanceHub

Polishing as we do not have time for some features that were initially planned.
