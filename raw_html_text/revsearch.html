Inspiration

Everyone has spent time searching for the perfect item--this extension helps you narrow down your options and save time!

What it does

From an initial search on Amazon.com and a key word describing your perfect item, this Chrome extension searches through the top search hits' reviews and lets you know which products have the most reviews matching your taste!

How I built it

Used JSoup to web crawl and web scrape through Amazon.com. The Chrome extension used JavaScript and HTML in the front end, PHP and Java in the back end.
