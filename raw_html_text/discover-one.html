Inspiration

We wanted to create a web app to support local small businesses and startups, because the more capital these companies have, the stronger and healthier the overall community is. Banks would benefit from strong companies by way of increased business investments.

What it does

The search results then populate a list and corresponding markers drop on the map using the Google Maps API. The user can click on the link to add that list item to "My List". The user can enter a store or restaurant name in the input box and add that to "My List" without preforming a search. The user can then either delete the list item or add personal notes.
