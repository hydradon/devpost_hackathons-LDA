Inspiration

Our inspiration to create this app is the constant struggle with falling asleep in class because some professors are just soo. boring.

What it does

It helps you pay attention in class and stay awake by having achievements to accomplish and set an all-time high score!

How we built it

We built this app using Android Studio and java.

Challenges we ran into

We had to learn how to use Android Studio. Creating and aligning the buttons was the most challenging part for us.

Accomplishments that we're proud of

The accomplishment we are most proud of is learning how to use Android Studio in half a day.

What's next for Classroom Achievements

We want to add subject-specific achievements and a high-score system. In the long run, we'd like to implement a multiplayer version so classmates can see who gets the highest score.
