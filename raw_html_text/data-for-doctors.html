Inspiration

We wanted to create a tool to help the community and assist people in finding healthcare resources.

What it does

Provides clearer data using data analytics to doctors.

How we built it

The data has been analyzed using Python, and the website was built using HTML/CSS/Javascript.

Challenges we ran into

The data analyses was often difficult due to inaccuracies, and the UI techniques used to build the website are not commonly seen.

Accomplishments that we're proud of

We are proud of implementing the UI techniques like parallax scrolling in our website, improving the accuracy of our models, and displaying the data in a clean way.

What we learned

We learned new techniques for managing data and new UI techniques. We also learned Django in order to create a backend that could handle the interactive parts of the website.

What's next for Data for Doctors

Next, we will contact doctors and policymakers to advocate for further improvement of our community.
