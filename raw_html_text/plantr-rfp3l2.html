Inspiration

In the Portland office we have a number of indoor plants that require different levels of care, which is a project in itself. This app will make it easier!

What it does

Using a DIY-connected moisture sensor and indoor maps, plant location and care history will be easier to manage. This is an entry for the Consumer Category.

How we built it

We started with a concept about integrating a Arduino moisture sensor and indoor location tracking. We leveraged the IndoorAtlas SDK to track indoor locations. This required us to create a floor plan using Indoor Atlas tools. The floorplan was essentially a way to calibrate our indoor location tracking sensors (using a device's accelerometer, gyroscope, wifi, and compass). GPS was not used. In addition, we digitized the floorplan in ArcGIS Pro and added features representing conference rooms, offices, and other office structures. An Android library was constructed to leverage the Arduino sensor and integrated into the app. Repo is here : https://devtopia.esri.com/sand8529/plantr
