Inspiration

Gradually, increasingly autonomous technology in this world.

What it does

It creates basic text and image-based webpages using simple, conversational chat and vocal commands like "Add a title to my webpage."

How we built it

We built it using HTML, CSS, JavaScript, and jQuery. The images we used were edited using GIMP.

Challenges we ran into

Too many to explain... Lol. The learning curve was generally very high for the both of us. Styling was probably the most difficult aspect of this project.

Accomplishments that we're proud of

The design of the bot and the extensiveness of its capabilities, considering we were both beginner web developers/designers.

What we learned

Responsive JavaScript, graph algorithms in web development, and basic AI.

What's next for Webbo

Webbo has a lot of potential. We plan on expanding its neural networks so that it can do more advanced things like working with many types of HTML tags, and possible even with embedded CSS and JavaScript.
