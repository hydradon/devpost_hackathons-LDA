Inspiration

Wanting to create the perfect party playlist that matches everyone's tastes

What it does

Create a "master" playlist based on the playlists entered by user(s)

How we built it

Working with the Spotify API, utilized node.js and express.js to create the client and server HTML and CSS were used to create the appearance of the website.

Challenges we ran into

Working with the Spotify API

  • Authentication
  • Old documentation

Accomplishments that we're proud of

Creating a web application Working with an API

What we learned

How to work with an API How to send data between the client and server using JSON objects

What's next for Ensemble

Increasing functionality of web app

  • Create play button on web app that plays created playlist
  • Create a playlist by genre
  • Get a genre specific playlist from Spotify instead of creating a combined playlist
