Inspiration

We wanted to build something using one of Google's amazing variety of resources on its cloud platform.

What it does

This app takes in any English word or phrase, then randomly selects one of the many languages Google translate offers, and translates it to that language. It can be used as a game to guess the random language it translates to, or use it to communicate with other people in a cryptic way.

How I built it

We built this using node.js using Google's translation API, according to a workshop that Google hosted. We used Google's App Engine to host the web app.

Challenges I ran into

I have never used Javascript before, so I had to learn it. Thanks to the mentors, I was able to grasp a basic understanding of promises.

Accomplishments that I'm proud of

This is the first time I have submitted anything at a hackathon. And I also started learning a new language.

What I learned

I learned that attending hackathons will give me valuable experience, and is a learning process. I know I will improve every time attend one.
