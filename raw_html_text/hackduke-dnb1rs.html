The Team

Inspiration

We believe that everyone in the world deserves an easy and fun way to learn python programming!

What it does

allows users to input python code into puzzles in a 3d world to allow navigation through a maze.

How we built it

Unity engine using ironpython to compile the user inputted code.

Challenges we ran into

merge conflicts, unity version issues, object instantiation issues, dealing with insane user inputted code.

Accomplishments that we're proud of

Python compiles! The dancing man who asks the puzzles.

What we learned

how to deal with horrible merge problems in asset files, how to write a full game in unity.

What's next for Roomly

Expand the maze to have a wide variety of puzzles for the user to solve
