Inspiration

We wanted to make an asymmetric game that pits mobile players against desktop players.

What it does

Desktop players get to control the guns and lasers of a massive gunship, while mobile players pit their thumbs against the frantic onslaught of colorful bullets, trying their best to survive and defeat the juggernaut!

How I built it

We built the game using Unity and a free space shooter asset package. The server is hosted on a VM in Google Cloud Platform's Compute Engine.

Challenges I ran into

Lack of sleep was the main challenge. Navigating Unity's multiplayer capabilities and the Google Cloud Platform was also an interesting learning experience.

Accomplishments that I'm proud of

Implemented multiple weapon systems!

What I learned

Multiplayer games are complicated.

What's next for Dimensional Dodgers

More ships! More bullets! More polish!
