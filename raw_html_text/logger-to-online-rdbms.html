Logger-To-Online-RDBMS

Manipulating Log file through API about customer data and create graphical report based on data collection. It collects data from site about user interactions, links he visited and links he failed or succeded to open. It then uses this data (Log files) to create customer interaction report. It note the congestion of servers and it give report about which server is facing higher number of request, when it's failing, how many servers needed to cure the high request congestion. All the information are sent in online DB and then it's showed in graphical representations.
