Inspiration

We read about ORB-SLAM, a in a computer vision research paper earlier this year. We thought this paper was great and were very excited when the code was released to the public, we wanted to be able to run ORB-SLAM from our Android phones and decided to create TWO3D (ROScloud)

What it does

It streams video from the android device to our AWS server which builds a 3D model from the video. model is then sent back to a screen.

How I built it

We started by building the server, then we created the app and finally we started streaming data between the devices

Challenges I ran into

Using ROS, getting the data streaming
