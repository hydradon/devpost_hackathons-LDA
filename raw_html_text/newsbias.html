Inspiration

Our team was inspired by the current political environment to create an app that would let readers quickly discover how biased a news article was.

What it does

Our application scrapes data from news articles and analyzes the text for key works to determine how bias the news article is, in either a negative or positive way.

How we built it

We used android studios as the environment and used XML and Java to create it. Using some APIs like jsoup, gson, and OkHttp.

Challenges we ran into

It was tough finding an API to do the job we needed while it was also difficult to actually create a function that would parse the text and store the data along with it.

Accomplishments that we're proud of

Were proud to have submitted something at all. We are a team of mostly newcomers, so we learned a lot of new techniques using APIs and webcrawers.

What we learned

We learned how to parse a websites text and scrub the URL using Android studios.

What's next for NewsBias

We're going to take on the world of public news and expose the media for being bias.
