Inspiration

Rhythm Master

What it does

A real-time multi-player music and dancing game on multi-platform.

How we built it

  • Backend: node.js
  • Frontend: paper.js
  • SDK: Kinect for Windows
  • Embeded server: Raspberry Pi
  • Development platform: Mac OS, Windows, Linux ## Challenges we ran into
  • Not familiar with C#, Kinect SDK, paper.js,etc.

Accomplishments that we're proud of

  • Play the same game across many screen with different size.
  • Real-time interaction among Kinect player and game player.
  • Real-time music generation by dancer using Kinect.

What we learned

  • New programming languages, framework, SDK.
  • Team work ## What's next for Aviate
  • More songs of course!
