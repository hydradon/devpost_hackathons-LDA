agolo-twitterbot

A Twitter bot that summarizes URLs

If you tweet @ it with a URL, it will reply with an Agolo summary of its contents.

What is Agolo?

Agolo is a summarization startup located in Midtown Manhattan. Its core functionality is to take one or more documents, summarize it down to the most essential points, and serve it up in a readable format.

Agolo has exposed its API in a private beta. This Twitter bot makes use of the API to scrape the contents of the given URL and summarize it into 5 bullet points.

To circumvent the fact that Twitter only allows 140 characters, this hack converts the 5 bullet points into an image and uploads it to Twitter.
