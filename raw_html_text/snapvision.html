Inspiration

My intention is to remove some of the linguistic barriers between communities and individuals.

What it does

This applications allows the user to take pictures of signs, flyers, and written text and translate the text into a message in their native language

How I built it

I used android studio and the google vision and translation APIs

Challenges I ran into

I'm very much a newbie when it comes to using the GCP, so it found it quite challenging to make use of the Google APIs

Accomplishments that I'm proud of

I was happy I managed to finished this project by myself! I'm used to working with people during hackathons but this time I wanted to go solo.

What I learned

I learned a ton about the GCP

What's next for SnapVision

I would like to improve the accuracy of the machine learning algorithms and extend the number of available languages it can translate.
