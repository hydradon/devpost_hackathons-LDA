Inspiration

We wanted to utilize new technology to create an experience that would both help people while teaching them.

What it does

The player interacts with a character through Watson voice recognition and gets to know her.

How we built it

We built it using iClone, Unity, Substance Painter, Blender SDKS include FinalIK, IBM Watson, Unity Dialogue System Simple REST API built with Nodejs

Challenges we ran into

Getting everything to run on mobile, especially not having the test device. Getting wireless streaming from our PC running well.

Accomplishments that we're proud of

Building simultaneous HD and Lightweight versions of our projects with Unity, getting the Watson SDK to work, and mastering Unity's animation controller system.

What we learned

How to utilize Watson to communicate with a character, how to master animation masks and conversation trees in Unity

What's next for GFE (The Girlfriend Experience)

We plan on continuing building upon the experience by integrating character arcs, different scenarios, and really building out the full relationship the Player has with the Character. We also plan on integrating more genders and LGBTQ aspects.
