What it does

The Extractor delivers the summarized article We are creating a Google Assistant application called THE EXTRACTOR: A unique tool that summarizes any reading material such as news articles, course notes using Beautiful Soup which is a html parser, as well as Microsoft Azure's image recognition API which incorporates OCR to convert images to text.
