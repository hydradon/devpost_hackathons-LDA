Inspiration

I've always felt there needed to be an easy way to look at weather and time of the place I'm traveling to without reaching into my pocket and using my phone.

What it does

Provides time and weather data of the city

How I built it

Used Pebble.js, openweathermap api

Challenges I ran into

This was our first time working with javascript and pebble so we really had to use every hour of the hackathon to build an app. Since this was my first hackathon, I also couldn't focus sometimes due to the "swag" the sponsor companies give out.

Accomplishments that I'm proud of

I'm very proud I was actually able to complete a project on my first hackathon.

What I learned

I learned even if you've never done something before, you can always achieve your goal if you dedicate yourself.

What's next for Travel Time

I want to make the UI more user-friendly.
