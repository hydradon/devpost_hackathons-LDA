Inspiration

Jacob's Idea, the inspiration is pretty obvious; parking is always hard and it would be nice to have a Waze-like app to mitigate that stress.

What it does

Allows users to update the number of available parking spots in each lot on campus.

How we built it

Using react-native (tuned toward IOS)

Challenges we ran into

Converting google maps KMZ files to something usable.

Accomplishments that we're proud of

Making an app with detailed maps of campus parking!

What we learned

parsing kml and xml is not impossible.

What's next for ParkMate

Other college campuses!
