Inspiration

Mostly wanted to learn how to create a skill to begin with. The idea for this project was because I felt this could help boost someone's morale when they may need it or even just to start their day with a good attitude.

What it does

This skill will give you a random inspirational quote.

How I built it

Built off of the fact skill template using node.js

Challenges I ran into

The most challenging part was hand picking quotes that I felt had value.

What I learned

It's easy to work off of the templates that the developer kit includes.

What's next for Inspire Me

Integrate with an api to include a wider selection of quotes, maybe also adding category specific selection.
