Inspiration

We were too lazy to get out of bed and search what was being served at the cafe, so we decided to program a new skill for Alexa instead because that was easier.

What it does

The Spartan Menu skill tells the user what is being served at a specific cafe and can tell you the location of that cafe at any time using live menu data.

How we built it

Used node.js and an external library to convert an RSS feed to JSON so we could program it for Alexa to use.

Challenges we ran into

None of us knew how to program in Javascript or to build skills for Alexa, so we had to learn how to do that.

Accomplishments that we're proud of

It works

What we learned

Node.js and how to create skills for Alexa
