Inspiration

Driving up to HackDartmouth, there was no way for us to see how much the tolls will cost. In previous experiences, I was charged with $80 in tolls and I didn't even realize it until I checked my E-Z Pass account! Hopefully this will help with the guesswork.

What it does

This app calculates tolls and using Google Maps API.

How I built it

Using Google Maps API returns polylines which we have to interpret.

Challenges I ran into

No toll information is accessible via an API, we had to scrape the web and manually make the .json files.
