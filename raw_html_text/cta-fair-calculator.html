Inspiration

No price calculator for CTA trips exist

What it does

Given a start and end location, it calculates the expected amount given a certain time frame including all transfers for both train and bus

How I built it

used jupyter notebook and python files

Challenges I ran into

learning google's directions API learning how to navigate json files inexperienced with python no experience with tkinter

Accomplishments that I'm proud of

Managed to complete the base of an app that has actual functionality

What I learned

Apart from the new Software and APIs, we learned that there are many edge cases when it comes to trying to estimate CTA cost.

What's next for CTA fair calculator

Being able to factor in more parameters to better estimate the cost. For example, a wheel chair user may not be able to transfer within the 2 hour window if a train stop is too far away or inaccessible from a previous bustop. This could change the cost of the overall trip
