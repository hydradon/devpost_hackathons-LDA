Inspiration

We wanted to create an application that lets users who lost something easily find it.

What it does

It automatically tags the photo you upload so others can easily find the post.

How we built it

We used the Microsoft computer vision api to allow us to automatically tag the uploaded photos. We used mongoDB for our database. We used html and javascript for our front end interface. We used Firebase for authentication of the user login.

Challenges we ran into

One of your biggest challenges was connecting all of the separate working parts together.

What we learned

It is a good idea to brainstorm and plan your program's control flow.
