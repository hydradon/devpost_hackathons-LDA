Inspiration

We wanted to make a fun hack while also learning new things. Dad Simulator (Dadgotchi) was born from the love for Tomagatchi.

What it does

Provides the player with a virtual Dad in his room. You need to keep him happy and healthy with your allowance by using the Pop's Pop Up Pop Shop. He'll reward with Dad Jokes ya lil rascal.

How we built it

All of it is client side, using Phaser as the game engine and storing data in cookies.

Challenges we ran into

Some minor CSS bugs and general JavaScript and game engine issues. Also making pixel art.

Accomplishments that we're proud of

Making pixel art and the whole thing.

What we learned

A lot about game engines and Phaser.

What's next for Dad Simulator

More dad jokes and fun.
