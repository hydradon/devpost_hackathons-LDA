Introduction:

This Package is having two activities.

  1. GenerateImage – To create new Wallpaper with your favarite images or solid color and text.
  2. ChangeWallPaper – To set windows wallpaper.

What it does

  1. GenerateImage

To create new Wallpaper with your favarite images or solid color and text. Output wallpaper size is following your screen size. GenerateImage Activity has 5 input parameters, no output parameter.

・BackGroundColor: Input the color you want to set as the background. This property is optional and the default value is Color.Black .

・BackGroundImageFilePath Input the Image File Path you want to set as the background. Keep the image size and center it on the screen. This property is optional and the default value is Nothing.

・OutputFilePath Input the Output File Path. This Property is required.

・Text Input the text you want to set as the background. This property is optional and the default value is Nothing.

・TextColor Input the text color you want to set as the text forecolor. This property is optional and the default value is Color.White .

The simplest use is to specify the BackGroundColor and Text, then generate a wallpaper image file With text. GenerateImage Activity can also specify an image.

  1. ChangeWallPaper

As the name suggests, change the wallpaper of Windows.

・OutputFilePath Input the Output File Path. Keep the image size and center it on the screen. This Property is required.
