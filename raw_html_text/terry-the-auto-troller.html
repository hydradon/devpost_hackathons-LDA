Inspiration

The need to troll celebrities fast.

What it does

Open the target celebrities page, hit ctrl+alt+shift+r, a "roast" is selected based on the celebrities name and bio, and is copied to your clipboard, ready to be spammed.

How we built it

We used autohotkey to call a python script that takes an input of the twitter handle, and produces a roast.

Challenges we ran into

Connecting auto hotkey with python.

Accomplishments that we're proud of

What we learned
