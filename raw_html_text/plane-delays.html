Inspiration

We found the Kaggle dataset of plane delays ( https://www.kaggle.com/giovamata/airlinedelaycauses/data ).

What it does

Our app analyzes and predicts plane delays.

How we built it

We built Plane Delays with Python, Altair, and a few bottles of Soylent + Yerba Mate (drink responsibly).

Challenges we ran into

We experienced constraints when working with the large dataset.

Accomplishments that we're proud of

We played with Dash (Python) and a new dataset!

What we learned

  1. Amount of data cleaning up necessary > Amount of data cleaning up expected
  2. We also learned about the Git LFS (Large File Storage) to manage our data.

What's next for Plane Delays

We can hopefully use more data from more recent years for better insights.
