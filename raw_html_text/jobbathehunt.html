Inspiration

I was inspired to do this project because I want an easier way to apply for jobs.

What it does

This is a Facebook chatbot that helps you find and apply to jobs that are posted on LinkedIn.

How I built it

I build the chatbot part using Node.js and the LinkedIn part using python

Challenges I ran into

One of the hardest parts was getting the chatbot and the python script to work together.

Accomplishments that I'm proud of

I'm proud of finishing this chatbot and turning something in, even though I wasn't at the hackathon the whole time!

What I learned

I learned a lot about selenium and python web-scraping.

What's next for JobbaTheHunt

What's next is to make the chatbot conversation feel more natural for the user.
