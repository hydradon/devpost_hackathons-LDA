• Contracts page for Consult X, Client perspective

Inspiration

Many members of the team faced firsthand struggles when it came to gather workplace experience in such a competitive industry. We wanted to create an application that would allow students facing similar situations to be able gather experience through freelancing work.

What it does

Clients are able to sign up for an account on the site and create contracts. Consultants, who must also create accounts, can view available contracts and apply to postings. Clients can then approve applications, allowing the consultants to get to work. Once the project is completed, the client is able to mark the contract is completed.

How we built it

We built this application using the Django-Python web framework. Our application also uses Bootstrap for a pleasing user experience, and web scraping techniques to grab data such as Github repositories.

Challenges we ran into

Biggest challenge was using git and github for collaboration due to the various moving parts. Furthermore, learning the Django-web framework within 24 hours proved to be a difficult task.

Accomplishments that we're proud of

We are very proud of our confidence with using project management systems such as Git. Moreover, the application boasts a clean UI and fulfills a plethora of functions

What we learned

We learned the importance of familiarizing ourselves with effective collaboration techniques and cohesive teamwork. We also recognized that learning new frameworks from scratch takes significant patience.

What's next for Consult X

We would definitely like to continue the app's development and incorporate more features, such as LinkedIn API configurations and an improved UI experience.
