Inspiration

To showcase IoT technology!

What it does

Reads RFID UID of various Mifare tags. Displays via locally hosted web server, and through Blynk mobile application. Able to control LED via both the web server and the Blynk mobile application. RFID is displayed on both ends as well.

How we built it

ESP32 Thing communicates via SPI with RN532 RFID Reader. LED on breadboard. External AC to DC power supply (5 V / 2A) to power the RFID Reader; this is due to insufficient voltage from MCU.

Challenges we ran into

Power supply limitations that we realized later on.

Accomplishments that we're proud of

Not giving up.

What we learned

How to code the Front End of the web server hosted by the ESP32. Implementing Blynk. Power consumption issues from various devices caused failure.

What's next for IoT - RFID

Incorporating more sensors!
