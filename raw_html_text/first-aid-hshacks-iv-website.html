Inspiration

After attending the Red Cross First Aid & CPR training course, we learned that it was a lot of information to take in, and the average, normal citizen most likely wouldn't know the information, much less than be able to recall it in an emergency. As a result, we created the First Aid Emergency Guide to help those wanting to learn more about what to do in emergencies and to help spread knowledge about emergencies.

What it does

The First Aid Emergency Guide is a simplistic, but effective, website that has all the necessary basic information needed to act quickly and efficiently in an emergency. We opted for a minimalist design, so in an emergency, where there is no time to waste, the user can navigate the website quickly.

How I built it

We built the website on repl.it using HTML, CSS, and a bit of JavaScript.

Challenges I ran into

During this project, we were tasked with creating a very simple but appealing design. Additionally, since none of were particularly skilled at HTML or CSS, we had to learn the language along the way.

Accomplishments I'm proud of

Upon completing this project, we were proud of being able to learn how to code in HTML and CSS and create a fully-functioning website in the allotted time.

What I learned

We learned how to create a design for a website and how to execute it without any bugs or mishaps.

What's next for the First Aid Emergency Guide

We plan on adding more information to the guide and perhaps turning the website into a more accessible and easy-to-use app.
