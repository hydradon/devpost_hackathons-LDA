What it does

Provides a fun, authentic balloon popping experience for users of all ages.

How we built it

We built the app using CoronaSDK and created builds for iOS and Android. (Also available for Windows, and OSX)

What's next for Balloop

-A game mode which counts points for people who need more than aimless tapping to sate their boredom. -A challenge mode which continuously increases balloon velocity for a more difficult popping experience
