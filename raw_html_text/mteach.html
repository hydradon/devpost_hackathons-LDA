Inspiration

We were all interested in applying our skills in web development and programming to education. As MHacks progressed, we also wanted to improve access to quality resources during valuable hackathon minutes.

What it does

It connects a community with collections of courses, tutorials, and other learning resources in order to sort out the highest quality educational materials from all over the web.

How we built it

We used AngularFire to link a Firebase backend with AngularJS.

Challenges we ran into

We ran into a lot of syncing problems with Firebase, but with the help of the Firebase mentors we were able to get past several of these hurdles.

Accomplishments that we're proud of

We were able to create a very sleek, beautiful website with most of the features we had set out to create as well as many more.

What we learned

We've learned a lot about AngularJS, Firebase, and other web technologies. Also about the intensity and sleep-deprivation of a hackathon. But most importantly, we've learned to work as an effective team.

What's next for MTeach

MTeach needs a community to create, curate, and filter better and better content. We'd love to be able to expand our reach in order to find those who love learning and give them the best chance to learn.
