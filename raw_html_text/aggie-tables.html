Inspiration

We have all personally been annoyed by getting our food at one of the dining halls and trying to find a seat for so long that our food gets cold.

What it does

A button would be at each dining table and this button will be green for open, yellow for semi-open, and red for full. The page will then show in real time which tables are open based on their buttons.

How we built it

We have used C++, a ti-launchpad, and a breadboard to create a button that would be at each dining table.

Challenges we ran into

transferring our code from Python into HTML and reading data from button.

Accomplishments that we're proud of

learning html, flask, and to read data from button into computer.

What we learned

Better programming skills
