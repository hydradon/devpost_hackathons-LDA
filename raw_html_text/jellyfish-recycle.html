Inspiration

In honor of Earth Day, we wanted to find a fun way to promote environmental awareness to the children.

What it does

2D Top-scrolling game with tilt controls to gather CA CASH REFUND recyclables for predict the money earned.

How we built it

Unity game developing software using C#, Adobe Illustrator, Adobe Photoshop

Challenges we ran into

Github cohesion problems, new language and platforms for most team members

Accomplishments that we're proud of

A polished product that is both educational and fun.

What we learned

New language, new platform, github

What's next for Jellyfish Recycle

More features, pause, more characters
