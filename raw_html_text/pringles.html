Inspiration

We wanted to create a tool that provided valuable information on someone's emotions.

What it does

A machine learning algorithm analyzes your facial expression to determine your emotions.

How I built it

We used ARKit to display the results of a sentiment analysis machine learning algorithm.

Challenges I ran into

Combining AR with machine learning was more difficult that we anticipated.

Accomplishments that I'm proud of

This is the first IOS app that we've worked on.

What I learned

We learned how to combine AR with machine learning sentiment analysis.

What's next for Prongles

Improving the machine learning algorithm by training it with more photos to increase its accuracy.
