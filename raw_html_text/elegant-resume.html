Elegant Resume (Beta)

A simple, elegant resume builder web app inspired by BootRésumé and Stack Overflow Developer Stories .

What is Elegant Resume?

Elegant Resume turns long winded, bullet heavy résumés into streamlined static webpages. Small words to describe yourself and colorful tags to highlight your skills & accomplishments.

Not a Tech guy? Don't worry, we got you covered. We have a super user-friendly dashboard for you to automate the resume building.

Features

  • Simple and Elegant Resume Webpage
  • Easy to use Dashboard
  • Responsive and mobile friendly
