Inspiration

We imagined how people feel standing at the bus stop and tried to create an ideal interface for them.

What it does

There is a map that shows buses' path and a schedule to see which bus will come first. You can also find a places that are currently nearby and watch a commercial.

How we built it

We used php for back-end and html/css+bootstrap for front-end.

Challenges we ran into

We had a great idea but no time to implement everything. That is why our project is more like a concept than a finalized work.

What we learned

That it is really difficult to manage the work.
