Inspiration

For our knowledge because the online security systems are working poorly nowadays, requiring a little to no confirmation for the identity and there is room for improvement

What it does

It uses three algorithms (OCR, Facial Recognition and Anti-Spoofing) for a secure identification of a person

How we built it

Using Python with Flask framework and OpenCV, numpy, pytesseract, C++ with OpenFace library

Challenges we ran into

Building the functional server and client, because of the need of communication between each other

Accomplishments that we're proud of

We are proud of the algorithms implemented - OCR, Facial Recognition and Anti-Spoofing

What we learned

Operating with different frameworks, libraries, languages; Working in a team; Using Git for a shared project

What's next for True Identity

Mobile Application for Android and iOS
