Inspiration We wanted to create a site where getting information on web privacy, online harassment, and even the transparency of local politicians. The whole team felt strongly about these topics and wanted to find a way to improve on these topics!

What it does

This site gives tips and information to people who want to know more about keeping their online identity safe, avoiding harassment, and how to decode the truth behind what people online tell you. This site also has a feature dedicated to local politicians and determining how trustworthy they are. This feature is somwhat expirimental, but we believe has great potential for growth.

How we built it

Using PHP, CSS, MySQL, and HTML5!. Mostly our amazing programmer, Spencer Colton, knew exactly how to combine and code all of these languages to make our site come together.

Challenges we ran into

Creating an algorithm for politicians trust factor, site design, and incorporating Cycle (a new technology for us).

Accomplishments that we're proud of

We are so proud that this project is complete and presentable. We are proud we made something to help the community and that we could work together and get this done!

What we learned

We learned a lot about containers, web building, site design, and teamwork. This project was a combination of all of our ideas and we are happy we could form it into a cohesive project.

What's next for Sketchy Site

We hope to expand the sections on privacy and info bias as well as really develop the local politician platform!
