Inspiration

Finding the perfect color scheme for your text editor takes a while. Why not use machine learning to create one that fits your tastes?

What it does

CoolorsAI allows you to train a neural network to generate the perfect color scheme based on your preferences.

How I built it

The UI is built using Vue.js and the neural network was created using Brain.js.
