Inspiration

Bluetooth devices are getting smaller and easier to loose

What it does

Keeps track of your bluetooth devices and minimizes your chances of loosing them.

How we built it

We built it using Android studios for the front end, Node.js with Amazon web services to host the backend server. Used Twilio to send and receive SMS notifications.

Challenges we ran into

Interfacing the Twilio SDK with Android Studios.

Accomplishments that we're proud of

Finish the product and polishing it.

What we learned

We learned a lot about server side technologies.

What's next for TraQ

To be able to connect more devices in parallel. Implementing a better querying system.
