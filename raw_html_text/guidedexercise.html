Inspiration:

We all know how difficult it is for people with health issues or recently met with accidents to get back to doing their normal day to day activities. As well, brave people who go in space need to exercise, but it is not fun to exercise alone for hours in space with no visual stimulation. We would like to present a immersive environment for these people to better enjoy exercising.

What it does:

It takes you to an immersive environment you are unable to currently enjoy in the real world and have a pet coach you through exercises in the proper way. The first phase is rowing exercise where the user wears the virtual reality headset and has the controllers in his/her hands and rows the boat. (Video on GitHub)

How I built it:

We used Unity 3D to build the environment using standard assets and implemented an immersive exercise activity. We used GitHub for version control. We used Paint3D to generate custom 3D assets and Paint.net to make 2D assets.

Challenges I ran into:

Unity3D 2018 LTS currently has issues with the Valve Index (VR headmounted device released in June 2019) and the provided hardware (Oculus Rift) experienced technical difficulties. We faced challenges working with Unity3D to move the 3D objects in a custom environment.

Accomplishments that I'm proud of:

We have a virtual environment that is aesthetically pleasing and we have a boat! And it looks promising for future development.

What I learned:

Unity3D in 24 hours. Working with VR integration is strenuous and massive amount of time, research, and computing power.

What's next for GuidedExercise:

  1. Integration with Virtual Reality headmounted devices like Oculus Rift, HTC Vive, and the Valve Index
  2. Include more exercises and environments
  3. More interactive coaches
  4. Sound effects
  5. Tracking log and personal records
  6. Competitive playing with friends
