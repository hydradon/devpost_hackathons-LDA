Inspiration

As high school students we realized that finding events to volunteer is a really hard process. Not only this but this app will bring together communities and allow people to help those around them whenever they want to.

What it does

This app is a volunteer-based platform where you use a map to find places to volunteer or you can post a volunteer event for people to apply to.

How we built it

We built two versions of the app for Android and iOS using Android Studio and XCode. We also used Firebase to hold the account information for all the people that register.

Challenges we ran into

Some challenges that we ran into was integrating the firebase platform with our app and actually being able to push and retrieve information from the server.

Accomplishments that we're proud of

We are proud that we accomplished to make an app on both major platforms of the world in one day while integrating a cloud based database on both versions.

What we learned

We learned how to implement new kinds of code and software to programs that we haven't used before. We also learned how to work together in a big group.

What's next for Volunteer Check

Volunteer check could expand into a bigger non-profit which connects people in communities together through volunteering opportunities.
