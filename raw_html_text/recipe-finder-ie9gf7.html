Inspiration

I've always had niche one-time use ingredients and herbs that I didn't know what to do with, so I decided to build a tool that would help me find other ingredients that would go well with it, as well as a recipe to tie it all together.

What it does

Takes one ingredient as input, returns as many other ingredients as needed, and downloads a video of a recipe.

How I built it

Wrote a script in python with he assistance of 2 large databases.

Challenges I ran into

Databases were so large that run time became an issue.

Accomplishments that I'm proud of

Finishing it! Being able to download appropriate videos.

What I learned

Hard work and perseverance pays off.

What's next for Recipe Finder

Better GUI and a more robust recipe library.
