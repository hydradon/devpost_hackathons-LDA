Inspiration

At the end of 2014 Ecuador start to issue electronic documents, two months ago we realized that all this information can help people to save money and improve their economy.

How it works

Currently the system records one or more email accounts per user. Automatically extracts and indexes all electronic documents in mailboxes added both new and old.

Challenges I ran into

  • WebHooks for incoming mails.
  • Lightweight index system.
  • Live notifications of received documents.
  • Standardize all formats of electronic documents.

Accomplishments that I'm proud of

  • Apache Solr integration.
  • High system performance.
  • Intuitive UI.
  • Multiple mail providers.
  • Live receipt of documents.
  • Pusher integration.
  • Use both ContextIO API (2.0, Lite)

What I learned

  • Facebook API.
