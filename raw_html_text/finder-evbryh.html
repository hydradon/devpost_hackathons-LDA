Inspiration

No fun or interactive way to discover new music

What it does

Allows users to rank and discover new songs and styles with a Tinder-style swipe interface

How we built it

Broke it into a distinct frontend and backend, building a whole react native application for the client side and a custom API to serve the data and recommendations

Challenges we ran into

Working with React Native and finding a secure and sustainable way to host an API

Accomplishments that we're proud of

Successfully created a comprehensive recommendation API and a intuitively-designed application interface

What we learned'

Learned a bunch about API development as well as the ins-and-outs of React Native

What's next for Finder

Improving the recommendation accuracy and adding new features such as sessions and social media integration
