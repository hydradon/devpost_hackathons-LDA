Inspiration

We wanted to make music

What it does

It takes user's hand movement to trigger a audio synthesizer

How we built it

We used a leap motion usb adapter to read the information from a user's hand movement and turn it into music

Challenges we ran into

Conflicts between apis

Accomplishments that we're proud of

We managed to log the data coming from the device and test events

What we learned

How to work with the leap motion api and organize a large amount of code

What's next for Project Noisey Hands

We are planning on creating a ui for a web application that will allow you to create and record music online
