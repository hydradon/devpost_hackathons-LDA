Inspiration

Losing information forever is a thing of the past, even after deletion!

Now, information will be safely kept.

Blockchain is on the rise and is the main contributing factor in this project.

All history is guaranteed maximum security and ensures that it is all accurately saved.

What it does

Our webpage preserver stores any public web page provided by a user.

Q: What makes you special from about other Wayback machines?

A: Other Wayback machines can not be fully trusted to govern these machines.

With Blockchain’s help, the service provider is no longer governing the machine.

The machine governs itself. This is what makes our machine so secure.

How we built it

-first day, formed a team of four, brainstorming
-had four ideas
-game, 51%, notarchive.org, blockchain reliable service provider
-notarchive was most appealing
-second day, managed to breakdown the project into four parts; backend, frontend, scraper, exporter
-had to use blockchain, javascript, html,react,phantomjs,node.js,ipfs,solidity,css
-backend: created the protocol to use blockchain
-frontend: create the website look, provides functionality of the ui/ux
-scraper: create a webscraper to read website’s HTML and img files, send files to exporter
-exporter: creates smart contract for files to pass through, gives files to backend
