Inspiration

Recently I have noticed that there aren't really any good websites/apps to form a group, so I decided it would be a good idea for a hackathon. Specifically, I play an instrument in an orchestral setting and was wondering if there was any program for finding people nearby to form an ensemble.

What it does

Users can create an account and from there can create their own group or join an existing group. From there, there is an application process if a group has that setting activated or a user can join an open group.

How I built it

I used React Native to make an app interface.

Challenges I ran into

Lots- This was my first time working in React-Native, so that was very new to me. There were a lot of difficulties dealing with data and data propagation, designing an app that is styled to fit on a phone (I generally do general websites)

Accomplishments that I'm proud of

I'm proud of the fact that I was able to do so much by myself in so little time. Setting up a whole app with at least basic functionality is quite impressive.

What I learned

Mostly just React-Native basics

What's next for Group With Me

I'm not sure whether or not I will continue the project since I have been very busy lately.
