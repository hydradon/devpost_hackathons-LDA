Inspiration

There is a lack of small party games that are currently available for PC so we wanted to explore new ideas that we had for this genre.

What it does

You play as a ball on the platform and try to knock other players off. Over time, the platform gets smaller and smaller until only one ball can remain.

How I built it

We built it using Unity and Blender.

Challenges I ran into

We wanted to challenge ourselves by making a multiplayer game. Making multiplayer game always has challenges and we were able to overcome most of them.

Accomplishments that I'm proud of

We got multiplayer to work pretty well.

What I learned

We learned about how to quickly implement multiplayer into a Unity game.

What's next for SuperBallB

This is a prototype for a game we plan to release on Steam.
