Inspiration

With the abundance of fake news detectors, we thought we'd go against the flow and make a fake news generator.

What it does

Takes a regular article and uses natural language processing to generate a satirical version of the article inspired by The Onion.

How I built it

Flask backend linked to a React frontend. We use Azure for analytics.

Challenges I ran into

Azure is a little finicky when it comes to using it for NLP.

Accomplishments that I'm proud of

Finishing with both a great backend and a great frontend.
