What it does

It allows a user to see how crowded areas are on a map by generating a heat map of the general congestion of an area.

How we built it

In order to use the app, the user must provide data, which is anonymous, given by location services. We then aggregated the data to get information for our server to process. After our server processes the data, we are able to send back the data to the user and produce a heat map of the crowd levels in the area.

Possible functionalities for CLUSTR

  • Add a "search by address" feature in order to find specific locations
  • Connect with Facebook in order to find precise locations of events (or areas of high congestion)
  • View historical data to see population trends throughout time
  • Integration with emergency and/or natural disaster dispatch systems
  • Allows big event organizers to minimize bottlenecks
