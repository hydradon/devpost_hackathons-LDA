Inspiration

Make job hunting faster and easier.

What it does

Submits your information to Indeed.com easily apply-able jobs.

How we built it

The front end is made with jquery, HTML, CSS. The backend is made with flask, selenium, scrapy, beautiful soup, and python. The backend scraped indeed for easily apply-able jobs then clicked the buttons until the fillable form is available. The front end gathers information from the user so that the backend can search Indeed.

Challenges we ran into

Integrating and passing data between the front end and back end. Executing python scripts that needed special command line arguments.

Accomplishments that we're proud of

None of us had web dev experience, but we were able to create this app. We were able to integrate many subsystems in able to get the app to work correctly.

What we learned

We leaned how to web scrape.

What's next for Resume Bot

Allow users to upload a resume and then use that file to apply to jobs instantly.
