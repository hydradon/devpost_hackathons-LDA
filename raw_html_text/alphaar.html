Inspiration

We love everything that has anything to do with augmented reality, and we'd like to share why we love it by making it more accessible to everyone.

What it does

Allows people to host AR sessions, whereby people who connect to the session are able to view augmented reality 3D models.

How we built it

  • Models are made through Blender, Mobile App and UI made through unity and Vuforia.
  • Web App made through React

Challenges we ran into

  • IOS to Android Compatibility
  • Windows and Mac sharing the same repository
  • Unity and React asynchronous calls
  • Connecting to Firebase
  • Blender animatioB
