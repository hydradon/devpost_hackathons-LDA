Inspiration

We got the idea from the board game Life and decided to make a card variant.

What it does

Teaches players about how to manage money, loans, credit as well as the advantages and disadvantages.

How we built it

We used Java to program/create the game.

Challenges we ran into

We had to make objects that had decks and be able to make sure each person/players' deck was the same as everyone else.

Accomplishments that we're proud of

Finishing our first project ever.

What we learned

We learned that there are many things to consider with loans, credit, etc. That things must be talked about first before coding.

What's next for Credit Crasher!

Try to finish all its planned features.
