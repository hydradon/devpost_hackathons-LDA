Inspiration

To intertwine the technologies of web development, embedded systems, and mobile application.

What it does

It starts off with the mobile application, that uses OCR to parse your grocery receipt to pass to the web to look for recipes based on what you bought.

It then sends recipes back to the mobile application, so it can forward the recipe the user selected, to the recipe scouter to instruct the user on how to cook the recipe, hands free.

How we built it

We built a website, android mobile application, and physically built a scouter with an arduino, 3D printed parts, mirror lens and plastic film.

Challenges we ran into

We were able to get everything to function separately but unfortunately we did not have enough time for them to communicate with each other. In addition, the AWS free service did not get back to us so we were not able to use their services for our website. We were not able to get a physical receipt of a grocery list, so we used a Digital Ocean card instead to demo the functionality.

Accomplishments that we're proud of

We are proud that we were able to get that much work finished in a given allotted time.

What we learned

We learned alot about different technologies and how to manage time more efficiently. We also learned ham + turkey subways sandwiches is the best.

What's next for Recipe Scouter

We plan to have them completely communicate with each other
