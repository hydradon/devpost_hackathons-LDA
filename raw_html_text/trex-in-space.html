Inspiration

The avenue where the hackathon took place.

What it does

Plays a simple side-scroller shooter style game.

How we built it

Using Unity 5, C# scripts in Unity 2D.

Challenges we ran into

Scripting with C#, which neither of the two of us really know too well. Learning Unity's interface with old tutorials. Sprite creation. Unity integration with Github.

Accomplishments that we're proud of

We made a animated Trex that shoots lasers.

What we learned

Some C# scripting such as how to move objects with vectors and the frustrating aspect of integrating assets into the project expecting them to work properly.

What's next for Trex in Space

More levels, better animations, and better enemy AI.
