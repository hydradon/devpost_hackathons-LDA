Inspiration

I was inspired by PrideBot . And the theme of the hackathon!

What it does

When you connect your phone number to Twilio and input a zip code as a text message to a Twilio phone number, the bot sends a message with the nearest 5 pizza places near the zip code entered and underneath the 'pizza' category on Yelp's API.

How I built it

I build a server in Node that takes requests to Yelp's API.

Challenges I ran into

I was attempting to get the output to be sent as SMS messages.

Accomplishments that I'm proud of

I am proud of being able to implement Yelp's API to my server and going to my first hackathon.
