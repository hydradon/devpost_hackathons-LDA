Inspiration

I didn't have a real idea, but I really wanted to participate.

What it does

I did two things:

  1. Set up a custom domain email
  2. Create keyboard shortcuts to toggle my monitor input between my computer and my Xbox.

How I built it

For the custom domain email setup, I used this medium article: https://blog.alexlenail.me/i-want-to-send-emails-from-my-google-domains-email-through-gmail-992bb3eae4c9

For the monitor stuff, I used this repo: https://github.com/kfix/ddcctl

Challenges I ran into

Running this command (below) crashed my computer :( ddcctl -d 1 -i ? I really wanted to debug this, but I also didn't want to have to repeatedly restart my computer.

Accomplishments that I'm proud of

I'm just proud of the fact that I accomplished something, and it's actually useful.
