Inspiration

We wanted to create a retro game that was open-world like but also had some new-school elements like AR. Open World AR was created by allowing you to complete quests in different worlds.

What it does

It is a game that has creative puzzles using AR, and is an open world to explore.

How I built it

Built in Unity with C#, and ARCore for AR.

Challenges I ran into

ARCore installation is only on Unity 2017.x NOT Unity 2018.x

Accomplishments that I'm proud of

We made the open world as big as possible.

What I learned

Graphics are hard. and AR can be mildly challenging.

What's next for Open World AR

More open world.
