Inspiration

collect data from remote places where the user are not at

What it does

collect sensor data and send text notification if humidity over limit. Send humidity and a number of other sensor data to a dashboard.

How I built it

photon particle, weather shield, node js, twilio

Challenges I ran into

getting the particle photon board to work. Board driver didn't work, and Serial connection didn't work. Had to connect using WiFi.

Accomplishments that I'm proud of

new to Node js and getting it to work

What I learned

the data backend for a IoT project
