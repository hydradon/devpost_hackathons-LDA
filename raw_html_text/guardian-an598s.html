The goal of Guardian is to guide users to their destination by using a route that is the safest. We are using QML and Javascript for developing the app in Appstudio. We use ArcGIS PRO for doing geospatial analysis, for processing crime data, creating a network dataset, and integrating crime data with the network dataset. Safe Travels! :)
