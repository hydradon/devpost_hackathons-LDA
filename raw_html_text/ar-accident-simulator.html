Inspiration

We talked to the HackUTA sponsors and they told us what they are in need for.

What it does

It uses Augmented Reality to create and simulate a crash from the different inputs using the inputted speed of the cars. This way the insurance adjustors could potentially determine a quote within minutes of the accident.

How we built it

We were completely new to Unity and C#. After watching some tutorial videos and working non-stop we were able to create an Augmented environment that simulates a crash based on the inputted speed of the user.

Challenges we ran into

1 - No prior experience. 2 - The Physics behind the crashes. 3- Working on different platforms. 4- Trying to understand how unity works. 5- Debugging annoying C# bugs. 6- How to make iOS and Android app

Accomplishments that we're proud of

We are really proud of this project because we had no prior experience using Unity or C#. We had to learn how they work and they use Augmented Reality to create a simulation.

What we learned

1- Unity 2- C# 4- How to work with iOS and Android SDK's 5 - How to use multiple platforms 6- How to make Augmented Reality iOS and Android apps

What's next for AR Accident Simulator

1- Expanding the Simulator so it can take different inputs, such as (speed, velocity, distance, etc). 2- Different types of collisions. 3- Integration with google maps for more accurate mapping 4- Take data from Allstate and simulate the accidents within the Augmented Reality Application.
