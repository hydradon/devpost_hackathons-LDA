This video shows the power of FizzBuzz As A Service. This was created by only using a small python script, youtube-dl, opencv, and calling a get request to request a FizzBuzz array of size 1 million.

Every 3rd, 5th, and 3rd * 5th frame has been replace successfully with Fizz, Buzz, and FizzBuzz.

FizBuzz As A Service

Overview

fbaas is a next generation, web scale, distributed, atomic, big data, angular.js, no-ops saas solution for providing all the Fizzes and the Buzzes you will ever need.

mobius-tubez

Install

Install pip3: 

sudo apt install pip3

Install ffmpeg: 

sudo apt install ffmpeg

Requirements: 

pip3 install -r requirements3.txt

Windows Install

First install ffmpeg and scipy

for scipy use: 

pip install *filename*.whl

Then install requirements: 

python install -r requirements.txt
