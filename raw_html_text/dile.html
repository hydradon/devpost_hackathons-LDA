Inspiration

Hace unas semanas se empezo a implementar el 911 como numero de emergencia, este numero acotaria almenos 8 lineas de atencion para diferentes apoyos; bajo estas medidas se abre la oportunidad de reducir el tiempo de contacto entre las personas involucradas en la emergencia o delito (espctador o afectado) y las autoridades competentes.

What it does

Buscamos reducir el tiempo de atencion a emergencias, agilizar la busqueda de afectados mediante la ubicacion exacta de los afectados, reducir el numero de llamadas osciosas, atencion de emergencias en una sola linea.

How we built it

Mediante el uso de programacion estructurada en Arduino, el uso de App inventor y personas con conocimientos y habilidades de programacion y electronica para hacer el ensamble del prototipo.

Challenges we ran into

La vinculacion entre Arduino y Android, dado que no teniamos conocimientos fuertes con los software

Accomplishments that we're proud of

Lograr hacer una interfaz funcional, ademas de que el hardware implementado (en Arduino) y del software (hecho con App inventor) fuesen funcionales

What we learned

A tener una fuerte comunicacion entre los miembros del equipo, el lenguaje de los distintos tipos de software

What's next for Dile

Desarrollarlo hasta su maximo potencial y lograr que sea una util herramienta cotidiana para la sociedad
