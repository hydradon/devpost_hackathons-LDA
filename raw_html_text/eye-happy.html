Inspiration:

With the growing use of technology, our team aimed to provide an everyday solution to help those suffering from mental health illnesses that they may/may not be aware of. End the stigma.

What it does:

A computer vision application that analyzes the eyes of the user to determine their level of stress and anxiety. Mental wellness is enhanced by sending the user positive, encouraging notifications that correlate directly with their observed level of anxiety.

How we built it:

We used Indigo API for computer vision. The application was coded in python.

Challenges we ran into:

Trying to compile all code together and testing it until it runs smoothly.

Accomplishments that we're proud of:

We are first time hackers and we are proud that were able to create a new application using computer vision APIs. Some of the team was able to learn a new language (python), while others were able to learn how to use the computer vision APIs in a new creative way.

What we learned:

We learned a bit more about APIs and about computer vision application.

What's next for eye calm:

To expand into other areas of the face to help better determine their level of stress anxiety. Also a side code was developed for detecting how close a user is to their screen of the computer by outputting frequency waves.
