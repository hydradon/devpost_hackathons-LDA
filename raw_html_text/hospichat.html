Inspiration

We have all had a hospital visit where we felt bored or distraught. We all use social media on a daily basis to communicate our lives with others.

What it does

This website is simply a social media for hospital patients. Each hospital will have a code that will unlock its own chat box. The site offers a chat box for patient communication within the hospital and a description of each hospital. It's overall goal is to provide a safe haven for each and every patient.

How we built it

  • Webpages are built with CSS3 and HTML5
  • Login system created with JavaScript
  • Chat box was created using HTML5, CSS3. and Java

Challenges I ran into

  • Creating a fully functional chat bot
  • Storing data

Accomplishments that I'm proud of

  • Finishing a project we are proud of in the time we had
  • Fully functioning login system without the use of a server

What we learned

  • How to create a javascript login system
  • How to build a chatbot

What's next for HospiChat

We need to create a server for more hospitals to expand upon our current demo of 3. We want to enhance the chat box for better user communication. We also hope to improve upon the design
