Knxt

Knxt is a simple implementation of a peer to peer payment dApp. Inspired by Connext state channel hub and spoke model, we are able to create a decentralized version of this.

Using orbit db, we are able to take the hub and spoke model mostly off-chain on ipfs, avoiding the need for staking through a smart contract on-chain.

Created for the #ETHSanFrancisco Hackathon.
