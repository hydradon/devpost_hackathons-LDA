Inspiration

The age of data should enable us to better understand ourselves

What it does

A journaling interface to track when, where, and what you're thinking as you go about your day. A dashboard to visualize what you write

How we built it

A node/express/mongo app wired together in a simple MVC pattern. jQuery to make the front end run smooth. Mapbox to bring our data to life

Challenges we ran into

So many features, so little time

Accomplishments that we're proud of

Visualizing a user's history with geolocation tells powerful stories

What we learned

How to write lean, maintainable web-apps with node.js, work with geoJSON data, use mapping APIs

What's next for Glimpse

More data! More features!
