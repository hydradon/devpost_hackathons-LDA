Inspiration

Getting people go to out more often, but also making sure the conditions are safe.

What it does

It uses sensors that collects data for temperature, rain, soil moisture, and crowd levels.

How we built it

Using an Arduino to collect data from all the various sensors and creating an algorithm that creates unique messages on the LCD.

Challenges we ran into

Putting data on a server and understanding the larger impact of our system.

Accomplishments that we're proud of

Getting five individual sensors to submit accurate data and merging code successfully.

What we learned

Used a lot of new hardware that we have never been exposed to and learned a lot about Arduino IDE.

What's next for TrailSafe

Creating a smoother system that works on a much larger platform like a park.
