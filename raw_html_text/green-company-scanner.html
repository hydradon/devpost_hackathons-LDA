Inspiration - How hard it can be to find how environmentally friendly any company you buy products are. So the goal is to make it easier for people to be able to buy more ethically.

What it does - Our broad idea was to have a barcode scanner that returned feedback on how green a company was. However, being a team of, effectively, complete beginners to any coding, we only made a small step on that path. Our program simply uses a computer's connected webcam to read a presented barcode and store it in an excel file in the same folder.

How we built it - Solely python, using imutils and cv2

Challenges we ran into: Installation problems, and generally overestimating our scope. Our team was fairly unaware of eachother's abilities, so it was difficult to get a start. A few of the modules required some additional installation which were somewhat confusing.

What we learned: How to install a wide variety of packages in python, use of webcam in python applications
