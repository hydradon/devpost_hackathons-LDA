Inspiration

We wanted to get more familiar with JavaScript and the concept of Block Chain and thought this was a great opportunity to broaden our knowledge in those areas.

What it does

Our hack makes use of the block chain api provided by Merck, which allows us to track all the places a specific product was scanned and displays markers on Google Maps.

How we built it

We built this project using Node.js, html, css, and the block chain api provided by Merck to be able to use the data.

Challenges we ran into

The challenges we ran nto included trying to access the data on a local server, and trying to get the server running.

Accomplishments that we're proud of

Being able to query data.

What we learned

We learned more about the block chain structure, and we also utilized node.js.
