Inspiration - The game is inspired by bad games that are supposed to be funny because they are so bad.

What it does - Its a very simple game that simulates the campaign of Donald Drumpf. It only takes around 3 minutes to finish.

How we built it - we built it using a language called processing.js

Challenges we ran into - Making the game worse than it actually is.

Accomplishments that we're proud of - The sprites

What we learned - We learned its harder than we think to make bad games (seriously, it took 5 hours)

What's next for Drumph Simulator - Not much honestly, if we people like this game today (which they probably won't) we will try to give the game better graphics and make it more complicated.
