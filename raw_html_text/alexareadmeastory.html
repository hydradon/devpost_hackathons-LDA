AlexaReadMeAStory

HACKTECH 2017 PROJECT

Goal: Have Alexa read a picture of a document for the user

Primary Use Case: User needs audio of a document(homework, instructions, notes, etc)

With the integration of Alexa into the home, daily routines for the average person are constantly being rewritten. One such change we hope to see is Alexa's involvement in academic and other similar activities. Alexa's ability to interact with the web and vocalize information makes the work environment fluid, an ideal we hope to further with this project.
