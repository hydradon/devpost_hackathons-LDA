What it does

It's a system that allow parents to control,limit the use of websites for kids and redirect them to an educational game or quiz to gain more browsing time.

How we built it

Dashbord for parents and a google chrome extension on kids chrome browser.

Challenges we ran into

How to controle and limit the use of some website according to parent's choices,

What's next for CEL

In fact CEL is not only a control system but it could be more especially by including home works for kids and the filtration of contents.
