Inspiration 

   As three seniors nearing the end of our high school careers, we feel as if it is important for us to understand the types of paths we can take next.

What it does 

  TJHSST Alumni fill out a google form with relevant information about their education, career and contact information. TJHSST students are easily able to search through this information and find useful contacts.

How we built it 

 We used Big Parser to store the data and easily search for keywords. We used jQuery to interact with the API, and HTML5 to display the information.

Challenges we ran into 

We had issues getting a domain as well as figuring out how to use the Big Parser API.

Accomplishments that we're proud of 

We were able to learn jQuery in a shortened amount of time as well as interact with the Big Parser API successfully.

What we learned 

 We learned how fast we can learn something new.

What's next for Colonial Connect 

 We hope to extend our application so it has a broader reach and extends to other high schools.
