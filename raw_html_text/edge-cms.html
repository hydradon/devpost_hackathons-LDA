Inspiration

We don't like updating sites for people years after we made them

What it does

It let's non-technical people edit their site in a more natural way

How we built it

We used Firebase to store the inner HTML of elements that we let logged in users edit using Medium Editor. We verify ownership of a site by monitoring if they can add a meta tag to the source, secured through verification on a serverless microservice because we're hipster (but mostly cheap).

Challenges we ran into

API Gateway was changing my POST data from json to plain text and I still don't know why. I love it anyways <3

Accomplishments that we're proud of

We managed a decent design that we are proud of given our lack of design experience

What we learned
