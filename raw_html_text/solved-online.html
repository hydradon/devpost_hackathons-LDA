Inspiration

We wanted to build a gaming dapp with innovative technology but also easy to use and fun.

What it does

This is a traditional jigsaw puzzle game. We created over 50 puzzles to be solved and then created an adventure feature that could be expanded in the future.

How I built it

Starting with modern HTML5, PHP, and javascript technology and using Blockstack as the auth system. We are also using Blockstack and Gaia for storing progress.

Challenges I ran into

Technical design was the biggest challenge. Adding complex functionality with fine mouse control in HTML and integrating it with Blockstack to create a seamless user experience. Browser compatibility was also a challenge. Many advanced techniques were not possible due to a lack of support in some browsers.

Accomplishments that I'm proud of

We have a solid base to build an incredible dapp. All of the core features have been tested in several modern browsers for compatibility.

What I learned

This is our first venture into creating dapps. There was an initial learning curve with terminology but we adapted our design approach to pull off the integration.
