Inspiration

I was inspired by other shooter games and my mother's ideas for a fun, yet educational game that taught the users math.

What it does

The user uses the arrow keys to move and space bar to shoot. Gain points by shooting the sharks. Every few seconds, a math problem appears that the user has to solve to continue. The user must complete the math problem quickly because remaining enemies can still shoot them.

Challenges I ran into

I did not run into any difficult challenges, but it took a long time to make the enemy sharks disappear when they were hit.

What I learned

I learned about how to work quickly and ignore distractions around me. I also learned that it is important to look at code around the perceived problematic code, and not just those few lines.
