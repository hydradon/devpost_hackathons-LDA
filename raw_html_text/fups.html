Inspiration

I was sick and tired of the UPS guy running away with my package because I wasn't at the door to sign it. There needed to be a better solution than for me to physically be in the presence of the UPS guy.

What it does

FUPS allows users to remotely sign packages so that the UPS guy can just leave them outside.

How I built it

React Native x Docusign rest api

Challenges I ran into

Push notifications are hard.

Accomplishments that I'm proud of

I got push notifications to kinda work.

What I learned

I learned how to create Docusign envelopes and request their recipient view URL.

What's next for FUPS

It'll probably get buried in future hackathon projects.
