Inspiration

We're unhealthy first-year CS majors that dream of becoming parkour masters/assassins. Combining the best of two worlds, we created an app that will allow us to virtually exercise while mastering the secrets of assassins. Someday, we'll leave the depths of the dungeon.

What it does

You can freely run and climb around in a desert city located near a port. Show off your parkour abilities by climbing (literally) anything.

How we built it

We used C# to write the scripts, Unity as the game engine, and Unity's Asset Store to download open source assets for our project.

Challenges we ran into

As we are inexperienced in Unity and C#, we had a lot of trouble getting off the ground. As a result, we dedicated the majority of our time to learning the basics of Unity and C#. With a lot of help from mentors, workshops, and online tutorials, we were able to get started.

Accomplishments that we're proud of

We're proud of the fact that we were able to build a working VR application within 36 hours with essentially no experience.

What we learned

We learned how to utilize Unity's framework, program and debug in C#, and address the needs of the user by accounting for common user experience problems.

What's next for AssassinTraining

We plan on adding corgis (and other entities) for the user to interact with in other ways. We also plan on addressing some minor bugs in the movement and climbing scripts.
