• Count Defenders

Inspiration

Motivate children to learn basic mathematics.

What it does

Game that takes place in space and the user has to solve simple math problems by shooting down the correct answer.

How we built it

We used: --html, javascript, jQuery, photoshop , canvas and css.

Challenges we ran into

-Couldn't put separate images on same canvas -Couldn't find the correct numbers that represent keys such as "w" , "a" , "s", and "d".

Accomplishments that we're proud of

-Animated "stars" background to run smoothly.

What we learned

-How to make a simple game with collisions, movement, and other techniques. -Some javascript and jquery

What's next for Count Defender

Implementing a system so the numbers that are "enemies" , are randomized to make levels more challenging.
