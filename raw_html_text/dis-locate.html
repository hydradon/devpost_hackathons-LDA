Inspiration

The all seeing eyes of the social media will help us pinpoint any kind of disaster that might have struck a city in the world.

What it does

It uses the twitter scraper api to scrape the tweets and then create a database that would finally predict the top 5 location where a disaster may have struck.

How we built it

We used a twitter api to create a back-end and then created a front-end using html, css and javascript to create a front end
