Inspiration

Previous accidents in which this device we created could have saved the lives of many.

What it does

It sends immediates ignals to authorities as soon as a vehicle's vitals crash. It displays previously inputted insurance information to the authorities, as soon as a vehicle's vitals crash. It also sends alerts to emergency contacts regarding the accident.

How we built it

It was programmed on Android studio as a mobile app.

Challenges we ran into

Using the Google Maps API

Accomplishments that we're proud of

Being able to resolve issue of remote car wrecks efficiently.
