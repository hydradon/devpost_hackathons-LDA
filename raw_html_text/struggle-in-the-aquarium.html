Inspiration

We wanted to add a fish and starfish to play with each other.

What it does

Plays as fish to defeat the starfish army.

How I built it

Using JavaScript and p5.js.

Challenges I ran into

Coding hitboxes and character animations.

Accomplishments that I'm proud of

I used my knowledge in data structures to keep track of all the dynamic particles and creatures in the environment.

What I learned

I learned the importance of data structures and object-oriented programming.

What's next for Struggle in the Aquarium

Add scrolling (i.e. side scroller) and power-ups.
