We were very interested in machine learning and the ability for computers to connect with people

It is a chat bot that you can strike a simple conversation with and a budgeting bot that can manage money.

We used the recommended software (Dialogflow) to build our bot and we used node.js to connect the bot on a network so the bot could be communicated with on multiple computers.

Working with node.js and learning machine learning

Being able to forward data to a network so the bot could be used on different computers and not locally

Nothing is easy; something that could take an experienced programmer an hour took us several hours

We plan on implementing more features such as a weather function or a list maker
