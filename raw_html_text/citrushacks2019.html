Inspiration

If everyone just picked up one piece of litter, everyday, the world would quickly become a much cleaner place. We often pass by littered areas, telling ourselves that we were too busy to clean it up. Hoping to find a solution to these problems, we developed ReclaimEarth.

What it does

ReclaimEarth is a web based application that allows its users to mark where they see littering, rating it on a scale from 1-5. Other users can see these marks along with a short description and a picture and can choose to help clean it up, gaining points in the process. The user's avatar will grow as they acquire more points as the earth becomes a cleaner place to live in.

How we built it

We built ReclaimEarth with HTML, CSS, and JavaScript with Flask.

Challenges we ran into

We ran into multiple challenges along the way, including syncing our login system with Google's OAuth and the database for storing all of the information from our users.

Accomplishments that we're proud of

We are proud that we managed to deploy the app onto Heroku, allowing everyone to use it on a public domain at https://reclaimearth.herokuapp.com .
