Technology used

  • Google Vision API
  • Google Translate API
  • iOS + Swift

Inspiration

We wanted to explore new, engaging ways to learn a new language.

What it does

SnapLingua allows you to snap a photo of an object and then get the name of that object in a different language. This is simpler said than done. It leverages Google's Vision API to identify objects and then uses Google's Translate API to translate the name of the objects to the language of your choosing.
