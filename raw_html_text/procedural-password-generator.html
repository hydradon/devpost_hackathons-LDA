Inspiration

Passwords managers suck. We don't suck. Thus our password manager won't suck.

What it does

Generates passwords procedurally rather than storing things on disk. It also provides a means of end to end encryption for sending the final password to the user from the cloud.

How we built it

C#, Python, VPS, Soylent

Challenges we ran into

Users would refuse to use a terminal application, so we had to figure out a user interface that the mass public would not mind using.

Accomplishments that we're proud of

We built the most secure password manager in the world.

What we learned

We needed to learn more about transmission of secure information over the web.

What's next for Procedural Password Generator

Web App and world domination.
